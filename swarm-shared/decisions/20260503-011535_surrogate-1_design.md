# surrogate-1 / design

## Final Synthesized Design (Correct + Actionable)

**Core fix (≤2h):**  
Replace recursive `list_repo_files` and per-file HF API calls with **one non-recursive `list_repo_tree` per folder + CDN-only fetches + schema projection to `{prompt,response}`**.  
This eliminates the two main 429 sources, keeps Space OOM-safe, preserves deterministic 16-shard parallelism, and reuses the existing dedup store.

---

## Implementation Plan (≤2h)

1. **Generate manifest (orchestrator/Mac)**  
   - One `list_repo_tree(..., recursive=False)` call per folder.  
   - Output `manifest-{folder}.json` with `{path, size, cdn_url}`.  
   - Commit or place where runners can read it (static or pre-step in CI).

2. **Update ingestion worker**  
   - Accept manifest file as input.  
   - Deterministic shard assignment unchanged: `hash(slug) % 16 == SHARD_ID`.  
   - Download via CDN URL (`resolve/main/...`) with no Authorization header.  
   - Parse with schema projection to `{prompt, response}` only (drop `source`, `ts`, etc.).  
   - Dedup via central `lib/dedup.py` SQLite store.  
   - Append to `shard-<N>-<HHMMSS>.jsonl`.

3. **Update training DataLoader**  
   - Load manifest JSON.  
   - Build DataLoader that streams via CDN URLs (no HF API calls during epochs).  
   - Keep `hf_hub_download` fallback for private repos (unused for public manifest).

4. **CI / cron changes**  
   - Keep 16-shard matrix workflow unchanged.  
   - Add pre-step (or orchestrator step) to generate manifest before runners start, or embed a static manifest in the repo.

5. **Resilience**  
   - Add retry/backoff for 429 with 360s wait on CDN downloads.  
   - Keep dedup store and shard logic unchanged.

### 1) Manifest generator (orchestrator/Mac)

```python
# bin/list-manifest.py
#!/usr/bin/env python3
import json, os, sys, time
from huggingface_hub import HfApi

REPO = "datasets/axentx/surrogate-1-training-pairs"
FOLDER = sys.argv[1] if len(sys.argv) > 1 else "public-raw/2026-05-03"
OUT = sys.argv[2] if len(sys.argv) > 2 else f"manifest-{os.path.basename(FOLDER.rstrip('/'))}.json"
TOKEN = os.getenv("HF_TOKEN")

api = HfApi()
entries = api.list_repo_tree(
    repo_id=REPO,
    path=FOLDER,
    repo_type="dataset",
    recursive=False,
    token=TOKEN,
)

manifest = []
for e in entries:
    # Support both dict and str returns from hf_hub
    if isinstance(e, dict):
        if e.get("type") == "file":
            path = e["path"]
            size = e.get("size")
    else:
        path = str(e)
        size = None
    manifest.append({
        "path": path,
        "size": size,
        "cdn_url": f"https://huggingface.co/datasets/{REPO}/resolve/main/{path}"
    })

out_obj = {
    "repo_id": REPO,
    "folder": FOLDER,
    "created_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
    "files": manifest,
}

with open(OUT, "w") as f:
    json.dump(out_obj, f, indent=2)

print(f"Wrote {len(manifest)} files to {OUT}")
```

### 3) Training-side DataLoader (CDN-only)

# train.py (snippet)
import json, requests, time
from torch.utils.data import IterableDataset

class CDNTextPairDataset(IterableDataset):
    def __init__(self, manifest_path, max_retries=3, backoff=360):
        with open(manifest_path) as f:
            data = json.load(f)
        self.files = data.get('files', [])
        self.max_retries = max_retries
        self.backoff = backoff

    def _robust_get(self, url):
        for attempt in range(self.max_retries):
            try:
                resp = requests.get(url, timeout=30)
                if resp.status_code == 429:
                    if attempt < self.max_retries - 1:
                        time.sleep(self.backoff)
                        continue
                resp.raise_for_status()
                return resp.content
            except requests.RequestException:
                if attempt == self.max_retries - 1:
                    raise
                time.sleep(self.backoff)

    def __iter__(self):
        for entry in self.files:
            content = self._robust_get(entry["cdn_url"])
            pair = project_to_pair(content, entry["path"])
            if pair:
                yield pair
