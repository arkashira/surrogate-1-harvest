# airship / backend

## Final Decision & Implementation — `airship discover` (backend)

**Decision**: Ship `airship discover` as a **single, zero-config, cron-safe orchestrator** that produces a **CDN-only training manifest** and tagged research outputs.  
**Scope**: ≤2 hours, non-interactive, idempotent, CDN-only ingestion to avoid HF API rate limits.

---

### 1) Orchestrator script (cron-safe, Bash)

Location: `/opt/axentx/airship/arkship/scripts/discover.sh`

```bash
#!/usr/bin/env bash
# /opt/axentx/airship/arkship/scripts/discover.sh
# Purpose: zero-config discovery + training prep (cron-safe, CDN-only HF ingestion)
# Usage:  bash discover.sh [--date YYYY-MM-DD] [--hf-repo org/repo] [--out-dir ./var/discover]
# Cron:   SHELL=/bin/bash
#         0 2 * * * cd /opt/axentx/airship/arkship && bash arkship/scripts/discover.sh >> logs/discover-$(date +\%Y\%m\%d).log 2>&1

set -euo pipefail
IFS=$'\n\t'

BASE_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
LOG_DIR="${BASE_DIR}/logs"
OUT_DIR="${BASE_DIR}/outputs"
HF_REPO="${HF_REPO:-datasets/axentx/surrogate-mirror}"
DATE_FOLDER="${1:-$(date +%Y-%m-%d)}"
TIMESTAMP=$(date -u +"%Y-%m-%dT%H:%M:%SZ")

mkdir -p "${LOG_DIR}" "${OUT_DIR}/batches/mirror-merged/${DATE_FOLDER}"

log() { echo "[$(date -u +"%Y-%m-%dT%H:%M:%SZ")] $*"; }

log "START discover for ${DATE_FOLDER}"

# 1) Business research (if present)
if [[ -x "${BASE_DIR}/../granite-business-research.sh" ]]; then
  log "Running granite-business-research.sh"
  "${BASE_DIR}/../granite-business-research.sh" \
    | tee "${OUT_DIR}/business-research-${DATE_FOLDER}.log"
  echo "#business-research #knowledge-rag #graph" >> "${OUT_DIR}/business-research-${DATE_FOLDER}.log"
else
  log "granite-business-research.sh not found or not executable — skipping"
fi

# 2) Knowledge RAG top-hub
if command -v knowledge-rag &>/dev/null; then
  log "Running knowledge-rag top-hub"
  knowledge-rag top-hub \
    > "${OUT_DIR}/top-hub-${DATE_FOLDER}.txt"
  echo "#knowledge-rag #graph #hub" >> "${OUT_DIR}/top-hub-${DATE_FOLDER}.txt"
  log "Top-hub: $(head -n 5 "${OUT_DIR}/top-hub-${DATE_FOLDER}.txt")"
else
  log "knowledge-rag CLI not available — skipping"
fi

# 3) Produce CDN-safe file list (single API call) for Surrogate training
log "Producing CDN-safe file list for ${HF_REPO}/${DATE_FOLDER}"
python3 "${BASE_DIR}/arkship/scripts/list_hf_folder.py" \
  --repo "${HF_REPO}" \
  --folder "${DATE_FOLDER}" \
  --out "${OUT_DIR}/batches/mirror-merged/${DATE_FOLDER}/file-list.json"

# 4) Build lightweight training manifest (CDN URLs only)
python3 "${BASE_DIR}/arkship/scripts/make_manifest.py" \
  --file-list "${OUT_DIR}/batches/mirror-merged/${DATE_FOLDER}/file-list.json" \
  --out "${OUT_DIR}/batches/mirror-merged/${DATE_FOLDER}/manifest.json"

log "Manifest entries: $(jq length "${OUT_DIR}/batches/mirror-merged/${DATE_FOLDER}/manifest.json" 2>/dev/null || echo '0')"
log "DONE discover for ${DATE_FOLDER}"
```

Make executable:

```bash
chmod +x /opt/axentx/airship/arkship/scripts/discover.sh
```

### 2) Python helper: list HF folder (tree-based, non-recursive)

Location: `/opt/axentx/airship/arkship/scripts/list_hf_folder.py`

```python
import argparse
import json
import os
import time
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--folder", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    api = HfApi()
    entries = api.list_repo_tree(repo_id=args.repo, path=args.folder, recursive=False)

    files = []
    for e in entries:
        if e.type == "file":
            files.append({
                "path": e.path,
                "size": getattr(e, "size", None),
                "lfs": getattr(e, "lfs", None)
            })

    os.makedirs(os.path.dirname(args.out), exist_ok=True)
    with open(args.out, "w") as f:
        json.dump({
            "repo": args.repo,
            "folder": args.folder,
            "listed_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
            "files": files
        }, f, indent=2)

    print(f"Listed {len(files)} files -> {args.out}")

if __name__ == "__main__":
    main()
```

### 3) Python helper: build CDN-only manifest for Lightning training

Location: `/opt/axentx/airship/arkship/scripts/make_manifest.py`

```python
import argparse
import json
import os
import time

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--file-list", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    with open(args.file_list) as f:
        data = json.load(f)

    manifest = {
        "repo": data["repo"],
        "folder": data["folder"],
        "listed_at": data["listed_at"],
        "manifest_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "cdn_urls": [
            CDN_TEMPLATE.format(repo=data["repo"], path=item["path"])
            for item in data["files"]
        ]
    }

    os.makedirs(os.path.dirname(args.out), exist_ok=True)
    with open(args.out, "w") as f:
        json.dump(manifest, f, indent=2)

    print(f"Manifest with {len(manifest['cdn_urls'])} CDN URLs -> {args.out}")

if __name__ == "__main__":
    main()
```
