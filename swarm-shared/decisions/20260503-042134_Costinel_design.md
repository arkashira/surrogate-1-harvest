# Costinel / design

## Final Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

### Highest-value improvement
Add a resilient, zero-runtime-HF-API “Top-Hub Signal” panel to Costinel that surfaces the most-connected hub (e.g., “MOC”) using baked CDN data, robust fallback UI, and no API-rate exposure during dashboard render.

### Why this is highest value
- Directly applies **Pattern: top-hub doc insight**, **Pattern: HF CDN Bypass**, and **#knowledge-rag #graph #hub**.
- Zero runtime Hugging Face API calls → no 429 risk on dashboard load.
- Small surface (<2h): one React panel + one build-time fetch script + baked JSON.
- Immediate UX lift: shows cross-project context (MOC) inside Costinel.

### Concrete plan (steps + code)

#### 1) Add build-time fetch script (run in CI or pre-commit)
Path: `/opt/axentx/Costinel/scripts/fetch-top-hub-cdn.sh`

```bash
#!/usr/bin/env bash
set -euo pipefail

# Config
REPO="axentx/knowledge-rag"
OUT_DIR="/opt/axentx/Costinel/public/data"
OUT_FILE="${OUT_DIR}/top-hub.json"
DATE_DIR=$(date -u +%Y-%m-%d)

mkdir -p "${OUT_DIR}"

# Fetch minimal hub metadata from the knowledge graph (single file)
# Uses GitHub raw CDN URL to avoid API rate limits during build
HUB_RAW_URL="https://raw.githubusercontent.com/${REPO}/main/data/top-hub.json"

if curl -fsSL --max-time 10 "${HUB_RAW_URL}" -o "${OUT_FILE}"; then
  # Ensure valid JSON and inject/update timestamp
  if jq --arg d "${DATE_DIR}" '. + {updated: $d}' "${OUT_FILE}" > "${OUT_FILE}.tmp"; then
    mv "${OUT_FILE}.tmp" "${OUT_FILE}"
    echo "✅ Baked top-hub payload to ${OUT_FILE}"
  else
    echo "⚠️  Invalid JSON from upstream; using fallback"
    rm -f "${OUT_FILE}.tmp"
    exit 1
  fi
else
  echo "⚠️  Could not fetch top-hub; keeping existing baked file"
  # Keep existing file if present; CI should not fail for optional content
  exit 0
fi
```

- **Run**: in CI (or pre-commit) before build/deploy.
- **Result**: `public/data/top-hub.json` is CDN-resolvable at `/data/top-hub.json`.

#### 3) Add TopHubSignal React component
Location: `src/components/TopHubSignal.jsx`

```jsx
// src/components/TopHubSignal.jsx
import React, { useEffect, useState } from 'react';

const CDN_URL = '/data/top-hub.json';
const TIMEOUT_MS = 4000;

export default function TopHubSignal() {
  const [data, setData] = useState(null);
  const [status, setStatus] = useState('loading'); // loading | ready | stale | empty

  useEffect(() => {
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), TIMEOUT_MS);

    fetch(CDN_URL, { signal: controller.signal, cache: 'force-cache' })
      .then((res) => {
        clearTimeout(timeoutId);
        if (!res.ok) throw new Error('Failed to load top-hub');
        return res.json();
      })
      .then((json) => {
        setData(json);
        setStatus('ready');
      })
      .catch(() => {
        // Try stale bundle fallback if available
        try {
          const stale = localStorage.getItem('axentx:topHub:stale');
          if (stale) {
            setData(JSON.parse(stale));
            setStatus('stale');
          } else {
            setStatus('empty');
          }
        } catch {
          setStatus('empty');
        }
      });

    return () => clearTimeout(timeoutId);
  }, []);

  // Persist fresh copy for stale fallback
  useEffect(() => {
    if (status === 'ready' && data) {
      try {
        localStorage.setItem('axentx:topHub:stale', JSON.stringify(data));
      } catch {
        // ignore storage errors
      }
    }
  }, [status, data]);

  if (status === 'empty') return null;

  return (
    <section className="top-hub-signal" aria-label="Top hub signal">
      <div className="top-hub-header">
        <span className="top-hub-badge">Top Hub</span>
        <span className="top-hub-name">{data?.hub || '—'}</span>
        {status === 'stale' && <span className="top-hub-stale">(cached)</span>}
      </div>
      <h4 className="top-hub-title">{data?.title || 'Operational context unavailable'}</h4>
      <p className="top-hub-summary">{data?.summary || ''}</p>
      {data?.links?.length > 0 && (
        <div className="top-hub-links">
          {data.links.map((link, idx) => (
            <a
              key={idx}
              href={link.url}
              target="_blank"
              rel="noopener noreferrer"
              className="top-hub-link"
            >
              {link.label}
            </a>
          ))}
        </div>
      )}
      <div className="top-hub-meta">Updated {data?.updated || ''}</div>
    </section>
  );
}
```
