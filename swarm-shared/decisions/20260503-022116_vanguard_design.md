# vanguard / design

## Final Synthesized Implementation

### Diagnosis (Consolidated)
- **No persisted manifest**: every run performs authenticated `list_repo_tree`/`load_dataset` discovery, burning HF API quota and risking 429s.
- **Heterogeneous schema ingestion**: `load_dataset(streaming=True)` triggers PyArrow `CastError` on mixed-schema repos.
- **No CDN-only fetch path**: training fetches via authenticated HF client instead of CDN, wasting quota and increasing latency.
- **Lightning Studio misuse**: each run recreates a studio (burning 80+ hours/month) and lacks idle-stop resilience (no pre-run status check or auto-restart).

### Proposed Change (Consolidated)
Create `/opt/axentx/vanguard/pipeline/manifest.py` and `/opt/axentx/vanguard/train/train.py` to:
1. Generate and cache `(repo, dateFolder) → [file_paths]` JSON via a single authenticated `list_repo_tree` call (non-recursive per folder) and reuse it.
2. Refactor training to use CDN-only fetches (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with the cached manifest; zero API calls during data load.
3. Add robust schema handling: detect/coerce parquet schemas, project to `{prompt, response}`, and avoid `CastError`.
4. Add Lightning Studio reuse + idle-stop guard with pre-run status check and auto-restart.

Scope: new file `pipeline/manifest.py`; create `train/train.py`; add `requests` to `requirements.txt` if missing.

#### pipeline/manifest.py
```python
#!/usr/bin/env python3
"""
Generate and cache (repo, dateFolder) -> file-list manifest.
Usage:
    python -m vanguard.pipeline.manifest \
        --repo datasets/owner/repo \
        --date-folder 2026-04-29 \
        --out manifest/2026-04-29.json
"""
import argparse
import json
import os
import sys
from datetime import datetime, timezone
from pathlib import Path
from typing import Dict, List

try:
    import huggingface_hub
except ImportError:
    print("Installing huggingface_hub...")
    import subprocess
    subprocess.check_call([sys.executable, "-m", "pip", "install", "huggingface_hub"])
    import huggingface_hub

HF_TOKEN = os.getenv("HF_TOKEN", "")

def list_date_folder(repo: str, date_folder: str, out_path: Path) -> Dict:
    """
    Non-recursive folder listing to avoid excessive pagination.
    Returns manifest:
      {
        "repo": "...",
        "date_folder": "...",
        "generated_at": "...",
        "files": ["path/to/file1.parquet", ...]
      }
    """
    client = huggingface_hub.HfApi(token=HF_TOKEN if HF_TOKEN else None)

    # List top-level of date_folder only (non-recursive)
    try:
        items = client.list_repo_tree(
            repo=repo,
            path=date_folder,
            recursive=False,
        )
    except Exception as exc:
        raise RuntimeError(f"Failed to list {repo}/{date_folder}: {exc}") from exc

    files: List[str] = []
    for item in items:
        # item.rfilename is relative path from repo root
        if item.rfilename.startswith(date_folder):
            files.append(item.rfilename)

    # Also include immediate subfolders' non-recursive contents if needed
    # (We keep it simple: one date_folder flat or one-level nesting handled by caller)
    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "files": sorted(files),
    }

    out_path.parent.mkdir(parents=True, exist_ok=True)
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(manifest, f, indent=2)

    print(f"Manifest written to {out_path} ({len(files)} files)")
    return manifest

def load_manifest(manifest_path: Path) -> Dict:
    with open(manifest_path, encoding="utf-8") as f:
        return json.load(f)

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate HF dataset file manifest.")
    parser.add_argument("--repo", required=True, help="e.g. datasets/owner/repo")
    parser.add_argument("--date-folder", required=True, help="e.g. 2026-04-29")
    parser.add_argument("--out", required=True, help="Output JSON path")
    args = parser.parse_args()

    try:
        list_date_folder(args.repo, args.date_folder, Path(args.out))
    except Exception as exc:
        print(f"Error: {exc}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    main()
```

# ---- Manifest + CDN loader ----
class CDNParquetDataset(Dataset):
    """
    Lightweight dataset that reads parquet shards via HF CDN.
    Projects to {prompt, response} only at parse time and handles schema heterogeneity.
    """
    def __init__(self, repo: str, manifest_path: Path, split: str = "train"):
        self.repo = repo
        with open(manifest_path, encoding="utf-8") as f:
            manifest = json.load(f)
        self.files: List[str] = [
            f for f in manifest.get("files", [])
            if f.endswith(".parquet") and (split == "train" or split in f)
        ]
        if not self.files:
            raise ValueError(f"No parquet files found in manifest for split={split}")

    def __len__(self) -> int:
        return len(self.files)

    def _download(self, path: str) -> bytes:
        url = f"{HF_DATASETS_BASE}/{self.repo}/resolve/main/{path}"
        resp = requests.get(url, timeout=60)
        resp.raise_for_status()
        return resp.content
