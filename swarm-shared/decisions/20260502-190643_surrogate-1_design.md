# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value incremental improvement**: Add deterministic pre-flight file listing + CDN-only ingestion path to eliminate HF API rate limits during training and make shard work reproducible.

### What will ship
1. `bin/list-date-files.sh` — single API call (rate-limit safe) to list files for a date folder; outputs `file-list.json`
2. `bin/dataset-enrich.sh` — updated to accept file-list JSON and use CDN URLs (`resolve/main/...`) for zero-API downloads
3. `bin/generate-cdn-manifest.py` — small helper to produce CDN manifest for training scripts (so Lightning training does CDN-only fetches)
4. Workflow update: run `list-date-files.sh` once per cron, pass artifact to matrix jobs

### Why this matters
- Eliminates `list_repo_files` recursive 429s (1000 req/5min limit)
- CDN downloads bypass `/api/` auth checks → separate, much higher rate limits
- Training can embed the file list and do zero HF API calls during data load
- Keeps shard determinism (`slug-hash % 16`) and date-partitioned outputs

## 1. `bin/list-date-files.sh`

```bash
#!/usr/bin/env bash
# list-date-files.sh
# Usage: ./list-date-files.sh <date> [output.json]
#
# Lists files under datasets/axentx/surrogate-1-training-pairs/<date>/
# using a single tree API call (no recursive pagination).
# Outputs JSON array of { "path": "...", "size": ... }

set -euo pipefail

REPO="datasets/axentx/surrogate-1-training-pairs"
DATE="${1:-$(date +%Y-%m-%d)}"
OUT="${2:-file-list.json}"

# Single API call: recursive=False per folder to avoid 100× pagination
# If date folder doesn't exist yet, produce empty list
if ! curl -sSf \
  -H "Authorization: Bearer ${HF_TOKEN:-}" \
  "https://huggingface.co/api/datasets/${REPO}/tree?path=${DATE}&recursive=false" \
  > /tmp/tree.json 2>/dev/null; then
  echo "[]" > "${OUT}"
  echo "No tree for ${DATE} (or rate-limited). Produced empty list."
  exit 0
fi

# Extract paths (files only) from tree response
python3 -c "
import json, sys
with open('/tmp/tree.json') as f:
    tree = json.load(f)
files = [item for item in tree if item.get('type') == 'file']
print(json.dumps(files, indent=2))
" > "${OUT}"

echo "Wrote ${#files[@]} files to ${OUT}"
```

Make executable:
```bash
chmod +x bin/list-date-files.sh
```

## 2. Update `bin/dataset-enrich.sh` (CDN path + file-list input)

Key changes:
- Accept file-list JSON (skip HF API listing inside worker)
- Download via CDN (`resolve/main/...`) — no Authorization header, bypasses API rate limit
- Keep deterministic shard assignment (`slug-hash % 16`)
- Date-partitioned output: `batches/public-merged/{YYYY-MM-DD}/shard{N}-{HHiMMSS}.jsonl`

```bash
#!/usr/bin/env bash
# dataset-enrich.sh
# Updated: CDN-only downloads + deterministic file-list input
# Usage:
#   ./dataset-enrich.sh <date> <shard_id> <file_list.json>

set -euo pipefail

REPO="datasets/axentx/surrogate-1-training-pairs"
DATE="${1:-$(date +%Y-%m-%d)}"
SHARD_ID="${2:-0}"
FILE_LIST="${3:-file-list.json}"
TS=$(date +%H%M%S)
OUT_DIR="batches/public-merged/${DATE}"
OUT_FILE="${OUT_DIR}/shard${SHARD_ID}-${TS}.jsonl"

mkdir -p "${OUT_DIR}"

# Central dedup store (unchanged)
DEDUP_PY="lib/dedup.py"

# Read file list
mapfile -t FILES < <(python3 -c "
import json, sys
with open('${FILE_LIST}') as f:
    files = json.load(f)
for f in files:
    print(f['path'])
")

TOTAL=${#FILES[@]}
echo "Processing ${TOTAL} files for date=${DATE} shard=${SHARD_ID}"

count=0
for relpath in "${FILES[@]}"; do
  # Deterministic shard assignment by slug-hash
  slug=$(basename "${relpath}" .jsonl)
  hash_bucket=$(( $(echo -n "${slug}" | md5sum | tr -d ' -' | tr '[:lower:]' '[:upper:]' | xargs -I{} python3 -c "print(int('{}', 16))") % 16 ))

  if [[ "${hash_bucket}" != "${SHARD_ID}" ]]; then
    continue
  fi

  # CDN download (no Authorization header -> bypasses API rate limit)
  url="https://huggingface.co/${REPO}/resolve/main/${relpath}"
  tmpfile=$(mktemp)

  if curl -sSfL "${url}" -o "${tmpfile}" 2>/dev/null; then
    # Project to {prompt,response} and dedup (schema-specific logic kept)
    python3 -c "
import json, sys, hashlib
from pathlib import Path
try:
    import pyarrow.parquet as pq
    tbl = pq.read_table('${tmpfile}')
    df = tbl.to_pandas()
except Exception:
    with open('${tmpfile}') as f:
        df = json.load(f) if '${tmpfile}'.endswith('.jsonl') else {}
# Minimal projection: prompt/response fields (adjust per schema)
# This is surrogate-1 specific: keep only prompt/response
if isinstance(df, list):
    rows = df
else:
    rows = df.to_dict(orient='records') if hasattr(df, 'to_dict') else []
for row in rows:
    prompt = row.get('prompt') or row.get('input') or row.get('text') or ''
    response = row.get('response') or row.get('output') or ''
    if not prompt or not response:
        continue
    # dedup via central store
    md5 = hashlib.md5((prompt + response).encode()).hexdigest()
    print(json.dumps({'prompt': prompt, 'response': response, 'md5': md5, 'slug': '${slug}'}))
" 2>/dev/null | while IFS= read -r line; do
      if [[ -n "${line}" ]]; then
        # Pass through dedup.py (central md5 store)
        echo "${line}" | python3 "${DEDUP_PY}" >> "${OUT_FILE}"
      fi
    done
    count=$((count + 1))
  else
    echo "Failed CDN fetch: ${relpath}"
  fi

  rm -f "${tmpfile}"
done

echo "Shard ${SHARD_ID} processed ${count} files -> ${OUT_FILE}"
```

Make executable:
```bash
chmod +x bin/dataset-enrich.sh
```
