# workio / quality

## Final Consolidated Implementation (Corrected + Actionable)

### 1. Diagnosis (merged, de-duplicated)
- **Missing `X-Line-Signature` verification** — allows spoofed/replayed events.
- **No idempotency/replay protection** — LINE retries on 5xx/timeouts create duplicate `attendance_punch` rows and duplicate leave/OT state transitions.
- **No timestamp window check** — events far in the past/future can be replayed.
- **No per-event deduplication persisted** — absence of `webhookEventId`/delivery-id uniqueness allows duplicates even after signature checks.
- **Unsafe retry behavior** — returning 200 before commit causes LINE to stop retrying while work is incomplete; returning 5xx causes retries that compound duplicates.

### 2. Changes to Make (single scope)
File: `/opt/axentx/workio/server/src/routes/webhook/line.ts`  
Add:
- `verifyLineSignature` middleware (signature + required headers).
- Idempotency layer keyed by `X-Line-Delivery-Id` **and** per-event `webhookEventId` (or `event.id`) with DB uniqueness.
- Timestamp window check (±5 min).
- Atomic upserts for `attendance_punch` and transactional outbox for side effects (notifications/leave/OT) so 200 is only returned after durable commit.
- Return 200 after commit; return 5xx only for transient failures to allow safe LINE retries; treat duplicates as 200 (to stop retries).

#### Middleware: signature + required headers
File: `server/src/middleware/lineSignature.ts`
```ts
import crypto from 'crypto';
import type { Request, Response, NextFunction } from 'express';

const LINE_CHANNEL_SECRET = process.env.LINE_CHANNEL_SECRET || '';

if (!LINE_CHANNEL_SECRET) {
  throw new Error('LINE_CHANNEL_SECRET is required');
}

export function verifyLineSignature(req: Request, res: Response, next: NextFunction) {
  const signature = req.header('X-Line-Signature');
  const deliveryId = req.header('X-Line-Delivery-Id');

  if (!signature || !deliveryId) {
    return res.status(400).json({ error: 'Missing X-Line-Signature or X-Line-Delivery-Id' });
  }

  const body = JSON.stringify(req.body);
  const expected = crypto
    .createHmac('sha256', LINE_CHANNEL_SECRET)
    .update(body, 'utf8')
    .digest('base64');

  // Use timingSafeEqual to avoid timing attacks (ensure equal-length buffers)
  const sigBuf = Buffer.from(signature);
  const expBuf = Buffer.from(expected);
  if (sigBuf.length !== expBuf.length || !crypto.timingSafeEqual(sigBuf, expBuf)) {
    return res.status(401).json({ error: 'Invalid signature' });
  }

  // Attach for downstream use
  (req as any).lineDeliveryId = deliveryId;
  next();
}
```

import { Pool, PoolClient } from 'pg';

const pool = new Pool({ connectionString: process.env.DATABASE_URL });

const TIMESTAMP_TOLERANCE_MS = 5 * 60 * 1000; // 5 minutes

export async function processLineWebhook(
  events: any[],
  deliveryId: string,
  tenantId: string
): Promise<{ processed: number; duplicates: number }> {
  const client: PoolClient = await pool.connect();
  try {
    await client.query('BEGIN');

    // Delivery-level idempotency
    const deliverInsert = `
      INSERT INTO webhook_deliveries (delivery_id, tenant_id, processed_at)
      VALUES ($1, $2, NOW())
      ON CONFLICT (delivery_id) DO UPDATE
      SET tenant_id = EXCLUDED.tenant_id
      RETURNING (xmax = 0) AS inserted
    `;
    const deliverRes = await client.query(deliverInsert, [deliveryId, tenantId]);
    const deliveryInserted = deliverRes.rows[0]?.inserted === true;

    // If delivery already processed, skip event-level work (still commit and return 200)
    if (!deliveryInserted) {
      // Count how many events are new (none) — safe to commit and return
      await client.query('COMMIT');
      return { processed: 0, duplicates: events.length };
    }

    let processed = 0;
    let duplicates = 0;

    for (const ev of events) {
      const eventId = ev.webhookEventId || ev.id;
      if (!eventId) {
        throw new Error('Event missing webhookEventId/id');
      }

      // Timestamp window check
      const ts = ev.timestamp;
      if (ts && Math.abs(Date.now() - ts) > TIMESTAMP_TOLERANCE_MS) {
        // Reject stale events early (do not process)
        throw new Error(`Stale event timestamp: ${ts}`);
      }

      // Per-event idempotency
      const evInsert = `
        INSERT INTO webhook_events (event_id, delivery_id, tenant_id, processed_at)
        VALUES ($1, $2, $3, NOW())
        ON CONFLICT (event_id) DO NOTHING
        RETURNING event_id
      `;
      const evRes = await client.query(evInsert, [eventId, deliveryId, tenantId]);
      if (evRes.rowCount === 0) {
        duplicates++;
        continue;
      }

      // Process event atomically (example: attendance punch)
      await processSingleEvent(client, ev, tenantId);
      processed++;
    }

    await client.query('COMMIT');
    return { processed, duplicates };
  } catch (err) {
    await client.query('ROLLBACK');
    throw err;
  } finally {
    client.release();
  }
}
