# Costinel / design

## Final Implementation Plan — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only card that identifies the most-connected hub from the knowledge-rag graph and surfaces the top hub with 3 contextual signals.  
**Timebox**: <2h (single PR, no backend changes).  
**Non-goals**: write operations, graph mutation, execution flows, or new API routes.

---

### 1) High-level design (merged approach)

- Add a new read-only card component: `TopHubSignalCard`.
- Expose a lightweight client-side hook: `useTopHubSignals()` that:
  - Prefers a cached JSON file produced by the knowledge-rag pipeline (`/data/knowledge-rag/top-hub.json`).
  - Falls back to a synthetic stub (so UI works without ops artifacts).
- Place the card on the main dashboard in the “Governance Signals” row (or sidebar rail if preferred).
- Follow existing design tokens and accessibility patterns (no new dependencies).

#### B) Hook: `useTopHubSignals`

```tsx
// src/hooks/useTopHubSignals.ts
import { useEffect, useState, useCallback } from "react";

export type Signal = {
  id: string;
  title: string;
  severity: "low" | "medium" | "high";
  context: string;
  action: string;
};

export type TopHubPayload = {
  hub: string;
  label: string;
  connections: number;
  signals: Signal[];
  generatedAt: string;
  source: string;
};

const FALLBACK: TopHubPayload = {
  hub: "MOC",
  label: "Mission Operations Center",
  connections: 0,
  signals: [
    {
      id: "fallback-1",
      title: "Connect knowledge-rag pipeline to populate top hub",
      severity: "medium",
      context: "No cached top-hub.json found. Run granite-business-research + knowledge-rag to generate.",
      action: "Execute knowledge-rag and place artifact at /public/data/knowledge-rag/top-hub.json",
    },
  ],
  generatedAt: new Date().toISOString(),
  source: "fallback",
};

export function useTopHubSignals() {
  const [data, setData] = useState<TopHubPayload>(FALLBACK);
  const [loading, setLoading] = useState(true);

  const load = useCallback(async () => {
    try {
      const res = await fetch("/data/knowledge-rag/top-hub.json", {
        headers: { "Cache-Control": "no-cache" },
      });

      if (!res.ok) throw new Error("No top-hub artifact");

      const json = (await res.json()) as TopHubPayload;
      // Basic shape validation
      if (json?.hub && Array.isArray(json.signals)) {
        setData(json);
      } else {
        throw new Error("Invalid shape");
      }
    } catch {
      setData(FALLBACK);
    } finally {
      setLoading(false);
    }
  }, []);

  useEffect(() => {
    load();
  }, [load]);

  return { data, loading, refetch: load };
}
```

#### C) Card component

```tsx
// src/components/TopHubSignalCard.tsx
import { useTopHubSignals } from "@/hooks/useTopHubSignals";
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card";
import { Badge } from "@/components/ui/badge";
import { Skeleton } from "@/components/ui/skeleton";
import { InfoIcon } from "lucide-react";

const severityColors = {
  low: "bg-blue-100 text-blue-800",
  medium: "bg-amber-100 text-amber-800",
  high: "bg-red-100 text-red-800",
} as const;

export function TopHubSignalCard() {
  const { data, loading } = useTopHubSignals();

  return (
    <Card>
      <CardHeader className="flex flex-row items-center justify-between pb-3">
        <CardTitle className="text-base font-semibold flex items-center gap-2">
          <InfoIcon className="h-4 w-4 text-muted-foreground" />
          Top-Hub Signal
        </CardTitle>
        {data && !loading && (
          <Badge variant="outline" className="text-xs">
            {data.hub} · {data.connections} connections
          </Badge>
        )}
      </CardHeader>
      <CardContent>
        {loading ? (
          <div className="space-y-2">
            <Skeleton className="h-4 w-3/4" />
            <Skeleton className="h-4 w-full" />
            <Skeleton className="h-4 w-5/6" />
          </div>
        ) : (
          <div className="space-y-3">
            {data.signals.map((s) => (
              <div
                key={s.id}
                className="border rounded-md p-3 text-sm bg-muted/30"
              >
                <div className="flex items-start justify-between gap-2">
                  <span className="font-medium text-foreground">{s.title}</span>
                  <Badge className={severityColors[s.severity]}>{s.severity}</Badge>
                </div>
                <p className="text-muted-foreground mt-1 text-xs">{s.context}</p>
                <p className="text-primary/90 mt-2 text-xs font-medium">
                  Action: {s.action}
                </p>
              </div>
            ))}
            <p className="text-xs text-muted-foreground mt-2">
              Generated: {new Date(data.generatedAt).toLocaleString()}
            </p>
          </div>
        )}
      </CardContent>
    </Card>
  );
}
```
