# Costinel / backend

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

### What we ship
A small, non-blocking **Top-Hub Signal Panel** on the Costinel dashboard that:
- Defaults to the most-connected hub **MOC** (configurable via `HUB_NAME`).
- Shows **3 actionable, cost-impact proposals** derived from hub insights.
- Uses **HF CDN bypass** for any dataset-backed signals (zero API calls during render).
- Falls back gracefully if hub data is unavailable.

### Why this is highest value
- Directly applies past lessons: top-hub insight (MOC), CDN bypass, no re-invention.
- Visible impact on dashboard with minimal surface area (<2h).
- Keeps “Sense + Signal — ไม่ Execute” philosophy (proposals only).

---

### Implementation steps (backend-first)

1. **Add config** (optional override for hub name)
   - File: `src/config/hubs.py`
   - Expose `HUB_NAME`, `MAX_SIGNALS`, `HF_DATASET` (if used).

2. **Create service to fetch hub insights**
   - File: `src/services/hub_signals.py`
   - Responsibilities:
     - Resolve hub name (default MOC).
     - Try local cache/DB first.
     - If using HF dataset: use pre-listed JSON of file paths (generated by Mac orchestration) and fetch files via CDN (`resolve/main/...`) with `requests` (no auth).
     - Project only `{prompt, response}` or minimal schema needed for proposals.
     - Return top 3 actionable signals sorted by estimated cost impact.

3. **Add API endpoint**
   - Route: `GET /api/hubs/{hub_name}/signals`
   - Response shape:
     ```json
     {
       "hub": "MOC",
       "generated_at": "2026-05-03T03:00:00Z",
       "signals": [
         {
           "id": "sig-001",
           "title": "RI coverage gap — m5.xlarge",
           "description": "...",
           "impact_usd_monthly": 12400,
           "confidence": 0.82,
           "actions": ["purchase-ri", "schedule-downsize"]
         }
       ]
     }
     ```

4. **Wire into dashboard backend rendering**
   - If SSR: inject signals into template context.
   - If API-first: ensure endpoint is fast (<300ms) and cached (60–300s).

5. **Frontend panel (minimal)**
   - Component: `TopHubSignalPanel.vue` (or React equivalent).
   - Shows hub name, refresh timestamp, list of proposals with impact badge.
   - No write actions (Sense + Signal only).

6. **Caching & resiliency**
   - Cache layer (Redis/memory) keyed by hub name + date partition.
   - On CDN fetch failure: fallback to last-known cache; never block page render.

7. **Logging & observability**
   - Log CDN fetch latency and cache hits.
   - Do not log full dataset contents.

#### Hub signals service (src/services/hub_signals.py)
```python
import json
import os
import time
from datetime import datetime, timezone
from typing import List, Dict, Any

import requests
from functools import lru_cache

from src.config.hubs import HUB_NAME, MAX_SIGNALS, HF_DATASET, CDN_BASE

# Simple in-memory cache for dev (replace with Redis in prod)
_CACHE = {}
_CACHE_TTL = 120  # seconds

def _cached(key):
    now = time.time()
    entry = _CACHE.get(key)
    if entry and (now - entry["ts"] < _CACHE_TTL):
        return entry["value"]
    return None

def _set_cache(key, value):
    _CACHE[key] = {"value": value, "ts": time.time()}

def _load_prelisted_paths(hub_name: str) -> List[str]:
    """
    Expects a pre-listed JSON file on HF repo or locally:
    e.g. hub-filelists/moc/2026-05-03.json
    Contains list of parquet/file paths for that hub+date.
    """
    date_prefix = datetime.now(timezone.utc).strftime("%Y-%m-%d")
    local_path = f"hub-filelists/{hub_name}/{date_prefix}.json"
    if os.path.exists(local_path):
        with open(local_path) as f:
            return json.load(f)

    # Optional: try to fetch filelist via CDN (counts as static asset)
    if HF_DATASET:
        url = f"{CDN_BASE}/{HF_DATASET}/resolve/main/hub-filelists/{hub_name}/{date_prefix}.json"
        try:
            r = requests.get(url, timeout=5)
            if r.status_code == 200:
                return r.json()
        except Exception:
            pass
    return []

def _fetch_parquet_via_cdn(path: str) -> List[Dict[str, Any]]:
    """
    Fetch single parquet via CDN and project minimal schema.
    Uses pandas + pyarrow only if available; otherwise skip.
    For speed and minimal deps in this snippet, we assume
    parquet files are small and we use fastparquet/pyarrow.
    """
    try:
        import pandas as pd
        url = f"{CDN_BASE}/{HF_DATASET}/resolve/main/{path}"
        df = pd.read_parquet(url)
        # Project only what we need for signals
        records = []
        for _, row in df.iterrows():
            records.append({
                "prompt": str(row.get("prompt", "")),
                "response": str(row.get("response", "")),
                "estimated_impact": float(row.get("estimated_impact", 0)),
                "confidence": float(row.get("confidence", 0.5)),
            })
        return records
    except Exception:
        # Graceful degradation
        return []

def get_top_hub_signals(hub_name: str = HUB_NAME, limit: int = MAX_SIGNALS) -> Dict[str, Any]:
    cache_key = f"hub_signals:{hub_name}"
    cached = _cached(cache_key)
    if cached is not None:
        return cached

    paths = _load_prelisted_paths(hub_name)
    candidates = []

    for p in paths:
        rows = _fetch_parquet_via_cdn(p)
        for r in rows:
            if r.get("estimated_impact", 0) > 0:
                candidates.append(r)

    # Sort by estimated impact descending
    candidates.sort(key=lambda x: x.get("estimated_impact", 0), reverse=True)
    top = candidates[:limit]

    signals = []
    for idx, item in enumerate(top):
        signals.append({
            "id": f"sig-{hub_name.lower()}-{idx+1:03d}",
            "title": item.get("prompt", "Untitled recommendation")[:120],
            "description": item.get("response", "")[:300],
            "impact_usd_monthly": round(item.get("estimated_impact", 0), 2),
            "confidence": round(item.get("confidence", 0.5), 2),
            "actions": ["review", "schedule-optimization"]
        })

    result = {
        "hub": hub_name,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "signals": signals
    }
    _set_cache(cache_key, result)
    return result
```
