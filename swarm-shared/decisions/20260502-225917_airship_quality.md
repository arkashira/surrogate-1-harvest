# airship / quality

## Final Synthesis

**Highest-value improvement (<2h):**  
Ship a zero-runtime-dependency, CDN-safe `airship discover` orchestrator that deterministically emits `status.json` + `manifest.json` for a given date-folder. It pre-lists HF dataset files once so training uses CDN-only fetches (no API calls), eliminating 429s and fixing surrogate-1 schema drift by projecting to `{prompt,response}` only at parse time.

**Why this wins:**
- Removes HF API rate-limit risk during training/inference.
- Fixes surrogate-1 mixed-file schema issues at parse time.
- Zero runtime dependencies (Bash + curl + jq) → safe on Mac orchestration nodes.
- Deterministic, reproducible outputs; enables Lightning Studio reuse.
- <2h scope: single CLI + two JSON outputs + idempotent behavior.

---

## Implementation Plan

1. **Create `bin/airship-discover`** (Bash, executable, shebang)  
   - Accepts: `--repo`, `--date-folder`, `--out-dir` (default: `./var/discover`).  
   - Uses HF CDN for file listing via `list_repo_tree` (single API call) and falls back to cached JSON if already present (idempotent).  
   - Emits:
     - `manifest.json` — deterministic list of `{file, cdn_url, size}` sorted by path.
     - `status.json` — discovery metadata (timestamp, repo, folder, file count, total size, CDN base, strategy).

2. **Update training entrypoint**  
   - Accept `--manifest manifest.json`.  
   - Data loader reads manifest and streams via CDN URLs (zero HF API calls).  
   - Validate file existence via `HEAD` before each epoch (fast fail).

3. **Schema normalization at parse time**  
   - Parser projects surrogate-1 mixed files to `{prompt, response}` only when streaming.  
   - Keeps manifest format-agnostic; avoids preprocessing duplication.

4. **Integrate with Lightning Studio reuse pattern**  
   - Launcher checks for existing running studio, reuses if present.  
   - Passes manifest path via environment variable.

### 1. `bin/airship-discover`

```bash
#!/usr/bin/env bash
# airship-discover
# Zero-runtime-dependency CDN-safe manifest generator.
# Usage:
#   ./airship-discover --repo datasets/org/repo --date-folder batches/mirror-merged/2026-05-02 --out-dir ./var/discover

set -euo pipefail

REPO=""
DATE_FOLDER=""
OUT_DIR="./var/discover"
API_ROOT="https://huggingface.co/api"
CDN_ROOT="https://huggingface.co/datasets"

while [[ $# -gt 0 ]]; do
  case $1 in
    --repo) REPO="$2"; shift 2 ;;
    --date-folder) DATE_FOLDER="$2"; shift 2 ;;
    --out-dir) OUT_DIR="$2"; shift 2 ;;
    *) echo "Unknown option: $1" >&2; exit 1 ;;
  esac
done

if [[ -z "$REPO" || -z "$DATE_FOLDER" ]]; then
  echo "ERROR: --repo and --date-folder are required" >&2
  echo "Example: $0 --repo datasets/org/repo --date-folder batches/mirror-merged/2026-05-02" >&2
  exit 1
fi

mkdir -p "$OUT_DIR"
TIMESTAMP=$(date -u +"%Y-%m-%dT%H:%M:%SZ")
MANIFEST="$OUT_DIR/manifest.json"
STATUS="$OUT_DIR/status.json"

# Idempotency: if manifest exists and is non-empty, skip discovery
if [[ -s "$MANIFEST" ]]; then
  echo "INFO: Manifest already exists at $MANIFEST (skipping discovery)" >&2
  exit 0
fi

# Deterministic retry with backoff
curl_retry() {
  local url=$1
  local max_retries=5
  local retry=0
  local wait=2
  while (( retry < max_retries )); do
    local out
    out=$(curl -fsSL --retry 3 --retry-delay 1 --max-time 30 "$url") && echo "$out" && return 0
    echo "WARN: curl failed, retry $((retry+1))/$max_retries in ${wait}s" >&2
    sleep $wait
    wait=$(( wait * 2 ))
    retry=$(( retry + 1 ))
  done
  echo "ERROR: Failed to fetch $url after $max_retries retries" >&2
  return 1
}

# Fetch tree (non-recursive) for the folder
TREE_URL="$API_ROOT/$REPO/tree/$DATE_FOLDER?recursive=false"
echo "INFO: Fetching tree: $TREE_URL" >&2
TREE_JSON=$(curl_retry "$TREE_URL") || exit 1

# Build manifest entries sorted by path for determinism
ENTRIES=$(echo "$TREE_JSON" | jq -r '
  map(select(.type == "file"))
  | sort_by(.path)
  | map({
      file: .path,
      cdn_url: ("'"$CDN_ROOT/$REPO/resolve/main/$DATE_FOLDER"'/" + .path),
      size: .size
    })
')

FILE_COUNT=$(echo "$ENTRIES" | jq 'length')
TOTAL_SIZE=$(echo "$ENTRIES" | jq '[.[].size] | add // 0')

# Write manifest
echo "$ENTRIES" | jq '{files: .}' > "$MANIFEST"

# Write status
jq -n \
  --arg ts "$TIMESTAMP" \
  --arg repo "$REPO" \
  --arg folder "$DATE_FOLDER" \
  --argjson count "$FILE_COUNT" \
  --argjson total "$TOTAL_SIZE" \
  --arg cdn_base "$CDN_ROOT/$REPO/resolve/main/$DATE_FOLDER" \
  '{
    generated_at: $ts,
    repo: $repo,
    folder: $folder,
    file_count: $count,
    total_size_bytes: $total,
    cdn_base: $cdn_base,
    strategy: "cdn-only-zero-api"
  }' > "$STATUS"

echo "INFO: Wrote $MANIFEST ($FILE_COUNT files, $TOTAL_SIZE bytes)" >&2
echo "INFO: Wrote $STATUS" >&2
```

Make executable:

```bash
chmod +x bin/airship-discover
```

### 2. `scripts/generate-cdn-manifest.py` (Mac helper)

```python
#!/usr/bin/env python3
"""
Mac-only helper to bootstrap manifest.json when rate-limit window is clear.
Uses huggingface_hub once, then saves CDN manifest for training.
"""
import argparse
import json
from pathlib import Path
from huggingface_hub import list_repo_tree

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="e.g. datasets/org/repo")
    parser.add_argument("--folder", required=True, help="e.g. batches/mirror-merged/2026-05-02")
    parser.add_argument("--out", default="manifest.json")
    args = parser.parse_args()

    tree = list_repo_tree(repo_id=args.repo, path=args.folder, recursive=False)
    files = sorted([f for f in tree if f.type == "file"], key=lambda x: x.path)

    cdn_base = f"https://huggingface.co/{args.repo}/resolve/main/{args.folder}"
    manifest = {
        "files": [
            {
                "file": f.path,
                "cdn_url": f"{cdn_base}/{f.path}",
                "size": f.size,
            }
            for f in files
        ]
    }

    Path(args.out).write_text(json.dumps(manifest, indent=2))
    print(f"Wrote {args.out} ({len(files)} files)")

if __name__ == "__main__":
    main()
```
