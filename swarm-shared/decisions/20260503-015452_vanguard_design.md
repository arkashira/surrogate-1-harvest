# vanguard / design

## 1. Diagnosis
- No persisted `(repo, dateFolder) → file-list` manifest; every training/data-selection run triggers authenticated `list_repo_tree` against HF API, burning quota and risking 429s.
- Training/data loader uses `load_dataset(streaming=True)` or repeated per-file API calls instead of CDN-only fetches, causing auth-bound rate limits during training.
- No deterministic repo selection for HF commit cap (128/hr/repo); writes can hit cap and stall ingestion.
- No studio reuse guard; training jobs may recreate running studios and waste Lightning quota (≈80hr/mo).
- No idle-stop resilience; Lightning Studio idle timeout kills long-running training without auto-restart.

## 2. Proposed change
Create `/opt/axentx/vanguard/design/hf_cdn_training_design.md` and add a minimal, production-ready training launcher module at `/opt/axentx/vanguard/src/train_launcher.py` that:
- Persists a single `file-list.json` per `(repo, dateFolder)` after one authenticated `list_repo_tree` call.
- Embeds that list in training so Lightning workers fetch exclusively via HF CDN (`resolve/main/...`) with zero API calls during data load.
- Deterministically maps `slug → sibling_repo` to spread HF writes across 5 repos.
- Reuses running studios and auto-restarts stopped ones before `.run()`.

```markdown
# HF CDN-Only Training Design (vanguard)

## Goals
- Eliminate authenticated HF API calls during training data loading.
- Persist file-list once per (repo, dateFolder) to avoid list_repo_tree rate limits.
- Survive HF commit caps and Lightning idle timeouts.

## Data Flow
1. Mac orchestrator (post-rate-limit window):
   - `list_repo_tree(repo, path=dateFolder, recursive=False)` → `manifests/{repo_slug}/{dateFolder}.json`
   - Manifest contains filenames only (no content).
2. Training script (Lightning Studio):
   - Load manifest from git/attached volume or embedded copy.
   - DataLoader yields URLs: `f"https://huggingface.co/datasets/{repo}/resolve/main/{dateFolder}/{fname}"`
   - Use `hf_hub_download` or direct `requests.get`/`aiohttp` with CDN (no token).
3. HF writes (ingestion):
   - `repo_index = hash(slug) % 5`
   - Target repo: `{base_repo}-sib-{repo_index}`
   - Commits spread across 5 repos → 640/hr aggregate.

## Lightning Resilience
- Before `.run()`: check `Teamspace.studios`; reuse running studio by name.
- If studio status != "running": `studio.start(machine=Machine.L40S)` (or fallback L40S).
- Wrap long jobs with checkpointing; on idle-stop, orchestrator restarts studio and re-issues run.
```

#!/usr/bin/env python3
"""
Lightning + HF CDN-only training launcher for vanguard.
- Persists file-list manifest to avoid HF API rate limits during training.
- Uses CDN URLs exclusively for data loading (zero auth calls in training workers).
- Reuses studios and spreads HF writes across sibling repos.
"""

from __future__ import annotations

import hashlib
import json
import os
import sys
from datetime import datetime, timezone
from pathlib import Path
from typing import Any, Dict, List

import requests
from lightning_sdk import Client, Machine, Studio, Teamspace

HF_API_BASE = "https://huggingface.co/api"
HF_CDN_BASE = "https://huggingface.co/datasets"

# Configure via env
HF_TOKEN = os.getenv("HF_TOKEN")
HF_REPO = os.getenv("HF_REPO", "myorg/vanguard-data")     # base dataset repo
HF_DATE_FOLDER = os.getenv("HF_DATE_FOLDER", "")          # e.g. "2026-04-29"
MANIFEST_ROOT = Path(os.getenv("MANIFEST_ROOT", "/opt/axentx/vanguard/manifests"))
STUDIO_NAME = os.getenv("STUDIO_NAME", "vanguard-train")
TEAMSPACE = os.getenv("TEAMSPACE", "default")
SIBLING_COUNT = 5

HEADERS = {"Authorization": f"Bearer {HF_TOKEN}"} if HF_TOKEN else {}


def list_date_files(repo: str, date_folder: str) -> List[str]:
    """Single authenticated call to list files in date_folder (non-recursive)."""
    url = f"{HF_API_BASE}/repos/datasets/{repo}/tree"
    params = {"path": date_folder, "recursive": "false"}
    resp = requests.get(url, headers=HEADERS, params=params, timeout=30)
    if resp.status_code == 429:
        print("HF API 429 — wait 360s before retry", file=sys.stderr)
        raise RuntimeError("HF API rate limited")
    resp.raise_for_status()
    items = resp.json()
    # Expect list of dicts with "path"
    files = [it["path"] for it in items if it.get("type") == "file"]
    return sorted(files)


def save_manifest(repo: str, date_folder: str, files: List[str]) -> Path:
    repo_slug = repo.replace("/", "_")
    out_dir = MANIFEST_ROOT / repo_slug
    out_dir.mkdir(parents=True, exist_ok=True)
    path = out_dir / f"{date_folder}.json"
    payload = {
        "repo": repo,
        "date_folder": date_folder,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "files": files,
    }
    path.write_text(json.dumps(payload, indent=2))
    return path


def load_manifest(repo: str, date_folder: str) -> List[str]:
    repo_slug = repo.replace("/", "_")
    path = MANIFEST_ROOT / repo_slug / f"{date_folder}.json"
    if not path.is_file():
        raise FileNotFoundError(f"Manifest not found: {path}")
    data = json.loads(path.read_text())
    return data["files"]


def cdn_urls(repo: str, date_folder: str, files: List[str]) -> List[str]:
    return [f"{HF_CDN_BASE}/{repo}/resolve/main/{date_folder}/{f}" for f in files]


def sibling_repo_for_slug(slug: str, base_repo: str, n: int = SIBLING_COUNT) -> str:
    """Deterministic sibling repo selection to spread HF commit caps."""
    digest = hashlib.sha256(slug.encode()).hexdigest()
    idx = int(digest[:8], 16) % n
    if idx == 0:
        return base_repo
    return f"{base_repo}-sib-{idx}"


def get_or_start_studio(
    client: Client,
    teamspace: str,
    studio_name: str,
    machine: Machine = Machine.L40S,
) -> Studio:
    studios = client.teamspace(teamspace).studios.list()
    running = [s for s in studios if s.name == studio_name and s.status == "running"]
    if running:
        print(f"Reusing running studio: {studio_name}")
        return running[0]

    stopped = [s for s in studios if s.name == studio_name and s.status == "stopped"]
    if stopped:
        studio = stopped[0]
        print(f"Starting stopped studio: {studio_name}")
        studio.start(machine=machine)
        return studio

    print(f"Creating studio: {studio_name}")
    return client.teamspace(teamspace).studio.create(
        name=studio_name,
        machine=machine,
        create_ok=True,
    )
