# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-first, CDN-only Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema `CastError`.

### Changes

1. Add `bin/generate_manifest.py` — single Mac-side API call that lists one date folder via `list_repo_tree(recursive=False)` and writes `manifest.json` (path + size + md5 if available). Embed this manifest in the runner so training uses CDN-only fetches with zero API calls during data load.
2. Replace `bin/dataset-enrich.sh` with `bin/dataset_enrich.py` — Python worker that:
   - Reads the manifest (or falls back to per-run tree listing once).
   - Downloads each file via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth, CDN tier, bypasses `/api/` rate limits).
   - Projects to `{prompt, response}` only at parse time (avoids `pyarrow.CastError` from mixed schemas).
   - Deduplicates via `lib/dedup.py` (central md5 store).
   - Writes `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl`.
3. Update `.github/workflows/ingest.yml` to:
   - Accept an optional `MANIFEST_URL` or `DATE` input.
   - Run the Python worker directly (no Bash wrapper indirection).
   - Keep 16-shard matrix for parallelism.
4. Add `requirements.txt` entries if missing (`requests`, `tqdm`, `python-slugify` optional).
5. (Optional) Add `bin/launcher.py` for local/manual runs with deterministic shard assignment.

#### `bin/generate_manifest.py`
```python
#!/usr/bin/env python3
"""
Generate manifest for one date folder.
Run from Mac (or CI once per date) to avoid HF API calls during training.
Usage:
  HF_TOKEN=hf_xxx python bin/generate_manifest.py \
    --repo axentx/surrogate-1-training-pairs \
    --date 2026-05-03 \
    --out manifest-2026-05-03.json
"""
import argparse
import json
import os
import sys
from datetime import datetime

from huggingface_hub import HfApi

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", default="axentx/surrogate-1-training-pairs")
    parser.add_argument("--date", required=True, help="Folder name, e.g. 2026-05-03")
    parser.add_argument("--out", required=True, help="Output JSON path")
    parser.add_argument("--token", default=os.getenv("HF_TOKEN"))
    args = parser.parse_args()

    api = HfApi(token=args.token)
    folder = args.date.strip("/")
    try:
        # Single API call: non-recursive listing for this folder only
        entries = api.list_repo_tree(
            repo_id=args.repo,
            path=folder,
            repo_type="dataset",
            recursive=False,
        )
    except Exception as e:
        print(f"Failed to list {args.repo}/{folder}: {e}", file=sys.stderr)
        sys.exit(1)

    files = [e for e in entries if e.size is not None and e.size > 0]
    manifest = {
        "repo": args.repo,
        "date": folder,
        "generated_at": datetime.utcnow().isoformat() + "Z",
        "files": [
            {
                "path": f.path,
                "size": f.size,
                "url": CDN_TEMPLATE.format(repo=args.repo, path=f.path),
            }
            for f in files
        ],
    }

    os.makedirs(os.path.dirname(os.path.abspath(args.out)), exist_ok=True)
    with open(args.out, "w", encoding="utf-8") as fh:
        json.dump(manifest, fh, indent=2)
    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```

def shard_for_slug(slug: str, total_shards: int) -> int:
    return int(hashlib.md5(slug.encode("utf-8")).hexdigest(), 16) % total_shards

def project_to_pair(record: Dict[str, Any]) -> Dict[str, str] | None:
    """
    Project heterogeneous schema to {prompt, response}.
    Returns None if required fields missing.
    """
    prompt = record.get("prompt") or record.get("input") or record.get("question")
    response = record.get("response") or record.get("output") or record.get("answer")
    if not prompt or not response:
        return None
    return {"prompt": str(prompt), "response": str(response)}

def download_file(url: str, timeout: int = 30) -> bytes:
    resp = requests.get(url, timeout=timeout)
    resp.raise_for_status()
    return resp.content

def process_parquet(content: bytes, dedup: DedupStore) -> List[Dict[str, str]]:
    table = pq.read_table(pa.BufferReader(content))
    rows = table.to_pylist()
    out: List[Dict[str, str]] = []
    for row in rows:
        pair = project_to_pair(row)
        if not pair:
            continue
        # Deterministic hash for dedup
        payload = f"{pair['prompt']}\n{pair['response']}".encode("utf-8")
        md5 = hashlib.md5(payload).hexdigest()
        if dedup.seen(md5):
            continue
        dedup.add(md5)
        out.append(pair)
    return out

def process_jsonl(content: bytes, dedup: DedupStore) -> List[Dict[str, str]]:
    out: List[Dict[str, str]] = []

            continue
