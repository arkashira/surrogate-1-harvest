# airship / ops

## Highest-Value Incremental Improvement (≤2h)

**What**: Harden `airship discover` into a deterministic, CDN-only orchestrator that eliminates HF API rate limits and PyArrow schema errors while producing reproducible file lists and safe ingestion artifacts.

**Why**:  
- Removes runtime HF API calls (rate limits, 429s, commit caps)  
- Avoids `load_dataset(streaming=True)` on heterogeneous repos (PyArrow CastError)  
- Enables reproducible Lightning training with zero API calls during data load  
- Fits cleanly into existing ops workflows and can ship in <2h

---

## Implementation Plan

### 1) Add `airship discover` CLI command
- Single entrypoint: `airship discover --repo <repo> --date <YYYY-MM-DD> --out <filelist.json>`
- Uses HF API **once** (on Mac/CI) to `list_repo_tree(path, recursive=False)` for one date folder
- Persists deterministic JSON: `{ "repo": "...", "date": "...", "files": [...], "sha256": "..." }`
- If API unavailable, accepts `--filelist filelist.json` to operate offline

### 2) CDN-only ingestion worker
- Reads `filelist.json`
- Downloads via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth)
- Projects each file to `{prompt, response}` at parse time only
- Writes per-file parquet to `batches/mirror-merged/{date}/{slug}.parquet`
- No `source` or `ts` columns; attribution via filename pattern

### 3) Lightning training integration
- `train.py` consumes `filelist.json` and does CDN-only fetches
- Uses existing studio reuse pattern:
  - List `Teamspace.studios`, reuse running studio if present
  - On idle stop, restart with `target.start(machine=Machine.L40S)`
- No HF API calls during training

### 4) Cron/script hardening
- All wrapper scripts: `#!/usr/bin/env bash`, `chmod +x`
- Cron: `SHELL=/bin/bash`
- Invocation: `bash <script> "$@"`

### `airship/cli/discover.py`
```python
#!/usr/bin/env python3
import argparse
import json
import hashlib
import os
from pathlib import Path
from huggingface_hub import list_repo_tree

def main():
    parser = argparse.ArgumentParser(description="Airship discover: list repo files for a date (CDN-only)")
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., user/repo)")
    parser.add_argument("--date", required=True, help="Date folder (YYYY-MM-DD)")
    parser.add_argument("--out", required=True, help="Output JSON filelist path")
    parser.add_argument("--skip-api", action="store_true", help="Skip API call (offline mode with existing filelist)")
    args = parser.parse_args()

    out_path = Path(args.out)
    out_path.parent.mkdir(parents=True, exist_ok=True)

    if args.skip_api:
        print("Skipping API (offline mode). Provide existing filelist at --out.")
        return

    # Single API call: list one date folder non-recursive
    print(f"Listing {args.repo}@{args.date} (non-recursive)...")
    items = list_repo_tree(repo_id=args.repo, path=args.date, recursive=False)

    files = []
    for item in items:
        if item.type == "file":
            files.append(item.path)

    files.sort()
    payload = {
        "repo": args.repo,
        "date": args.date,
        "files": files,
    }
    payload_bytes = json.dumps(payload, separators=(",", ":"), sort_keys=True).encode()
    payload["sha256"] = hashlib.sha256(payload_bytes).hexdigest()

    out_path.write_text(json.dumps(payload, indent=2))
    print(f"Wrote {len(files)} files -> {out_path}")

if __name__ == "__main__":
    main()
```

### `airship/ingest/cdn_worker.py`
```python
#!/usr/bin/env python3
import json
import pyarrow as pa
import pyarrow.parquet as pq
import requests
from pathlib import Path
from slugify import slugify

CDN_ROOT = "https://huggingface.co/datasets"

def project_to_pair(raw_text: str):
    """
    Placeholder: project raw file to {prompt, response}.
    Implement per-repo logic here; keep minimal.
    """
    # Example naive split; replace with repo-specific parser
    parts = raw_text.split("\n\n### Response:\n", 1)
    if len(parts) == 2:
        return {"prompt": parts[0].strip(), "response": parts[1].strip()}
    return {"prompt": "", "response": raw_text.strip()}

def download_cdn(repo: str, path: str) -> str:
    url = f"{CDN_ROOT}/{repo}/resolve/main/{path}"
    r = requests.get(url, timeout=60)
    r.raise_for_status()
    return r.text

def run(filelist_path: str, out_dir: str = "batches/mirror-merged"):
    filelist = json.loads(Path(filelist_path).read_text())
    repo = filelist["repo"]
    date = filelist["date"]

    out_root = Path(out_dir) / date
    out_root.mkdir(parents=True, exist_ok=True)

    for fpath in filelist["files"]:
        try:
            raw = download_cdn(repo, fpath)
            pair = project_to_pair(raw)
            if not pair["prompt"] and not pair["response"]:
                continue

            table = pa.table(pair)
            slug = slugify(Path(fpath).stem)
            parquet_path = out_root / f"{slug}.parquet"
            pq.write_table(table, parquet_path)
        except Exception as e:
            print(f"Skip {fpath}: {e}")

    print("CDN ingestion complete.")

if __name__ == "__main__":
    import sys
    run(sys.argv[1])
```
