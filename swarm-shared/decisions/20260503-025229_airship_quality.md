## Final Consolidated Implementation (Best of Both Candidates)

**Goal**: Eliminate HF API 429s during Surrogate training by replacing runtime dataset discovery with a pre-cached CDN manifest, while maximizing Lightning Studio quota efficiency through deterministic reuse and idle-timeout resilience.

**Why this is highest value**:
- Directly addresses the Surrogate-1 training pipeline HF 429/rate-limit pattern
- Prevents wasted Lightning quota from idle-stop/Studio recreation
- Enables reproducible, zero-API training runs on Lightning (fits <2h)
- Minimal surface area: one new manifest + small train.py patch + reuse guard

## Implementation Plan (≤2h)

### 1) Pre-cache CDN manifest (run once on Mac orchestration host)
- Use `list_repo_tree(path, recursive=False)` per date folder under the dataset repo.
- Save flat list of `{ "repo": "...", "paths": [...] }` to `training/manifest-{date}.json`.
- Commit to repo (or store in S3/GCS) so training picks it up deterministically.

**Script**: `scripts/build_cdn_manifest.py`
```python
#!/usr/bin/env python3
"""
Build CDN-only manifest for Surrogate training.
Run from Mac orchestration host (HF API allowed).
"""
import json, os, sys
from datetime import datetime
from huggingface_hub import HfApi

REPO = "axentx/surrogate-training-data"
OUT_DIR = "training/manifests"
os.makedirs(OUT_DIR, exist_ok=True)

api = HfApi()
# List top-level date folders (non-recursive)
items = api.list_repo_tree(repo_id=REPO, path="", recursive=False)

manifest = {"repo": REPO, "generated": datetime.utcnow().isoformat(), "folders": {}}
for item in items:
    if not item.get("type") == "directory":
        continue
    folder = item["path"]
    files = api.list_repo_tree(repo_id=REPO, path=folder, recursive=False)
    file_paths = [f["path"] for f in files if f.get("type") == "file"]
    manifest["folders"][folder] = file_paths

out_path = os.path.join(OUT_DIR, f"manifest-{datetime.utcnow().strftime('%Y%m%d')}.json")
with open(out_path, "w") as f:
    json.dump(manifest, f, indent=2)

# Create stable symlink for launcher
stable_path = os.path.join(OUT_DIR, "manifest-latest.json")
if os.path.islink(stable_path) or os.path.exists(stable_path):
    os.remove(stable_path)
os.symlink(os.path.basename(out_path), stable_path)

print(f"Manifest written: {out_path}")
```

Make executable and ensure Bash invocation in cron if scheduled:
```bash
chmod +x scripts/build_cdn_manifest.py
SHELL=/bin/bash
```

**Cron** (Mac orchestration host):

### 2) Patch training script to use CDN-only fetches
- Load the latest (or pinned) manifest.
- Use `hf_hub_download` (or raw CDN URL) per file; **do not** use `load_dataset(streaming=True)` on heterogeneous repo.
- Project to `{prompt, response}` at parse time only.
- Attribution via filename pattern: `batches/mirror-merged/{date}/{slug}.parquet`.

**Patch target**: `surrogate/train.py` (or equivalent training entrypoint)

```python
import json, os, glob
from pathlib import Path
from huggingface_hub import hf_hub_download
import pyarrow.parquet as pq
import torch
from torch.utils.data import Dataset, DataLoader

MANIFEST_PATH = os.getenv("CDN_MANIFEST", "training/manifests/manifest-latest.json")
REPO = "axentx/surrogate-training-data"

class CDNParquetDataset(Dataset):
    def __init__(self, manifest_path=MANIFEST_PATH, repo=REPO, cache_dir=".cache_hf"):
        with open(manifest_path) as f:
            manifest = json.load(f)
        self.repo = repo
        self.cache_dir = cache_dir
        self.file_paths = []
        for folder, files in manifest["folders"].items():
            for fp in files:
                if fp.endswith(".parquet"):
                    self.file_paths.append(fp)

    def __len__(self):
        return len(self.file_paths)

    def _load_parquet(self, path):
        # CDN bypass: use hf_hub_download (cached) or raw CDN URL
        local_path = hf_hub_download(
            repo_id=self.repo,
            filename=path,
            cache_dir=self.cache_dir,
            force_download=False,
        )
        table = pq.read_table(local_path)
        df = table.to_pandas()
        # Project to {prompt, response} only; attribution via filename pattern
        # Expected filename: batches/mirror-merged/{date}/{slug}.parquet
        return [
            {"prompt": row.get("prompt", ""), "response": row.get("response", "")}
            for _, row in df.iterrows()
        ]

    def __getitem__(self, idx):
        # For simplicity, return file-level sample; collate/batching handled externally
        path = self.file_paths[idx]
        samples = self._load_parquet(path)
        # Return first sample per file (or implement smarter sampling)
        return samples[0] if samples else {"prompt": "", "response": ""}

# Lightning DataModule (minimal)
from lightning import LightningDataModule

class SurrogateDataModule(LightningDataModule):
    def __init__(self, batch_size=8, num_workers=4):
        super().__init__()
        self.batch_size = batch_size
        self.num_workers = num_workers

    def setup(self, stage=None):
        self.ds = CDNParquetDataset()

    def train_dataloader(self):
        return DataLoader(self.ds, batch_size=self.batch_size, num_workers=self.num_workers, shuffle=True)
```

**Key behaviors**:
- No `load_dataset(streaming=True)` on heterogeneous repo → avoids PyArrow CastError.
- Single manifest read → zero HF API calls during training.
- CDN-only downloads bypass API rate limits.

### 3) Lightning Studio reuse + idle-stop resilience
- Before creating a Studio, list running studios and reuse if present.
- On each training run, check status; if stopped, restart with target machine.

**Patch**: `surrogate/lightning_launcher.py`
```python
from lightning import Lightning, Teamspace, Studio, Machine
import time

LIGHTNING_PROJECT = "surrogate-training"
STUDIO_NAME = "surrogate-train-studio"

def get_or_create_studio():
    lightning = Lightning()
    teamspace = Teamspace.current()
    # Reuse running studio
    for s in teamspace.studios:
        if s.name == STUDIO_NAME and s.status == "Running":
            print(f"Reusing running studio: {s.name}")
            return s
    # Create if not exists (or fallback)
    print(f"Creating studio: {STUDIO_NAME}")
    return Studio.create(
        name=STUDIO_NAME,
        project=LIGHTNING_PROJECT,
        machine=Machine.L40S,  # free tier fallback; H200 requires lightning-lambda-prod
        create_ok=True,
    )

def run_training(script_path="train.py", requirements="requirements.txt"):
    studio = get_or_create_studio()
    # Ensure running
    if studio.status != "Running":
        print(f"Studio stopped ({studio.status}). Restarting...")
        studio.start(machine=Machine.L40S)
        # Wait for running
        while studio.status != "Running":
            time.sleep(10)
            studio.refresh()
    # Run training (zero API calls during data load)
    run = studio.run(
        target=script_path,
        requirements=requirements,
        name="surrogate-cdn-train",
    )
    print(f"Started run: {run.id}")
    return run

if __name__ == "__main__":
    run_training()
```
