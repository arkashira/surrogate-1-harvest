### 1. **Diagnosis**
* No deterministic **CDN-first manifest** exists, which means ingestion and training scripts are likely calling `list_repo_tree` and `load_dataset` at runtime. This risks hitting rate limits (429s) and results in non-reproducible runs.
* Missing **content-addressed file list** keyed by date/slug, which prevents training jobs from running without calling the Hugging Face API.
* Ingestion and training scripts are not optimized for CDN usage, leading to potential rate limit issues and inefficiencies.
* The project lacks a clear strategy for managing Hugging Face API rate limits and optimizing data ingestion and training workflows.
* There is no evidence of implementing the HF CDN bypass strategy to download dataset files directly from the CDN, which could significantly reduce the load on the Hugging Face API.

### 2. **Proposed change**
The proposed change involves creating a CDN-first manifest and implementing a content-addressed file list. This will be achieved by modifying the `granite-business-research.sh` script to pre-list file paths once and embed them in the training script. The scope of this change includes:
- Modifying the `granite-business-research.sh` script to use the HF CDN bypass strategy.
- Creating a new script, `cdn_manifest.py`, to generate and manage the CDN-first manifest.
- Updating the `train.py` script to use the CDN-first manifest and content-addressed file list.

2. **Create `cdn_manifest.py`**:
   ```python
   import requests
   import json

   def get_cdn_manifest(repo, path):
       # Use HF CDN bypass strategy to download dataset files directly from CDN
       url = f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
       response = requests.get(url)
       # Parse response and generate content-addressed file list
       file_list = []
       for file in response.json():
           file_list.append({"date": file["lastModified"], "slug": file["filename"]})
       return file_list

   def save_cdn_manifest(file_list, output_file):
       with open(output_file, "w") as f:
           json.dump(file_list, f)

   if __name__ == "__main__":
       import argparse
       parser = argparse.ArgumentParser()
       parser.add_argument("--repo", required=True)
       parser.add_argument("--path", required=True)
       parser.add_argument("--output", default="cdn_manifest.json")
       args = parser.parse_args()
       file_list = get_cdn_manifest(args.repo, args.path)
       save_cdn_manifest(file_list, args.output)
   ```
