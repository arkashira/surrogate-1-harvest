# airship / design

## Final Consolidated Implementation Plan  
**Scope:** `/opt/axentx/airship/surrogate` ingestion + training path  
**Primary goals:**  
- Eliminate HF API 429 during training (zero auth/rate-limit calls while iterating data).  
- Cut Lightning quota waste by reusing running studios.  
- Deterministic, reproducible runs via committed file manifests.  
- Actionable in <2 hours with minimal footprint.

---

### 1) Concrete changes to make (merged + hardened)

- **Mac-side manifest builder** (`tools/build_filelist.py`)  
  - One `list_repo_tree(recursive=False)` per date folder.  
  - Filters to valid shards (avoids recursive pagination explosion).  
  - Emits deterministic `filelist-{date}.json` containing `repo`, `path`, `cdn_url`, optional `etag/size`.  
  - Committed or uploaded to surrogate config bucket so training is reproducible.

- **Training data loader** (`surrogate/data/cdn_loader.py`)  
  - Accepts `--filelist` path.  
  - Downloads exclusively via CDN (`requests.get(cdn_url, timeout=60)`).  
  - Projects bytes to `{prompt, response}` only at parse time.  
  - Retries/backoffs on CDN failures; skips malformed shards without failing epoch.  
  - Uses `IterableDataset` for streaming; optional deterministic shuffle via seeded index permutation.

- **Lightning Studio lifecycle wrapper** (`surrogate/lightning/run_studio.py`)  
  - Lists `Teamspace.studios` and reuses any Running studio with matching name.  
  - If stopped: restarts with `Machine.L40S` (free-tier friendly; paid tiers selectable).  
  - Passes `--filelist` to training script inside the studio.  
  - Validates studio is Running before submit.

- **Ops preflight script** (`surrogate/ops/preflight.sh`)  
  - Shebang `#!/usr/bin/env bash`, `set -euo pipefail`.  
  - Builds manifest then submits Lightning run (or queues it).  
  - Reminder to set `SHELL=/bin/bash` in crontab if scheduled.

- **Small hardening additions**  
  - `.gitignore` keeps large/ephemeral filelists out unless deterministic tracking desired (commit stubs/example instead).  
  - HF token requires only `repo:read` for `list_repo_tree`; CDN downloads require no token.  
  - Local test: run manifest build, then run loader in a tiny DataLoader loop to confirm zero HF API calls during data fetch.

`tools/build_filelist.py`
```python
#!/usr/bin/env python3
"""
Build deterministic CDN filelist for a date folder in a HF dataset repo.
Usage:
    python tools/build_filelist.py --repo axentx/surrogate-data --date 2026-05-03 --out filelist-2026-05-03.json
"""
import argparse
import json
import os
import time
from huggingface_hub import HfApi

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def build_filelist(repo: str, date: str, out_path: str):
    api = HfApi()
    folder = f"batches/mirror-merged/{date}"
    entries = api.list_repo_tree(repo=repo, path=folder, recursive=False)
    files = [e for e in entries if e.rfilename.endswith((".parquet", ".jsonl", ".json"))]

    filelist = []
    for f in files:
        cdn = CDN_TEMPLATE.format(repo=repo, path=f.rfilename)
        filelist.append({
            "repo": repo,
            "path": f.rfilename,
            "cdn_url": cdn,
            "local_key": os.path.basename(f.rfilename)
        })

    payload = {
        "repo": repo,
        "date": date,
        "generated_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "files": filelist
    }

    os.makedirs(os.path.dirname(os.path.abspath(out_path)), exist_ok=True)
    with open(out_path, "w") as fp:
        json.dump(payload, fp, indent=2)
    print(f"Wrote {len(filelist)} files -> {out_path}")

if __name__ == "__main__":
    p = argparse.ArgumentParser()
    p.add_argument("--repo", default="axentx/surrogate-data")
    p.add_argument("--date", required=True)
    p.add_argument("--out", required=True)
    args = p.parse_args()
    build_filelist(repo=args.repo, date=args.date, out_path=args.out)
```

`surrogate/data/cdn_loader.py`
```python
import json
import hashlib
import random
import requests
from torch.utils.data import IterableDataset

class CDNShardDataset(IterableDataset):
    def __init__(self, filelist_path, projection_fn=None, seed=None, max_retries=3):
        super().__init__()
        with open(filelist_path) as f:
            manifest = json.load(f)
        self.files = manifest["files"]
        self.projection_fn = projection_fn or (lambda x: x)
        self.seed = seed
        self.max_retries = max_retries

    def _download_one(self, url: str):
        for attempt in range(1, self.max_retries + 1):
            try:
                resp = requests.get(url, timeout=60)
                resp.raise_for_status()
                return resp.content
            except Exception as exc:
                if attempt == self.max_retries:
                    raise
                print(f"Retry {attempt}/{self.max_retries} for {url}: {exc}")
        return None

    def __iter__(self):
        worker_info = getattr(self, "worker_info", None)
        if worker_info is None:
            rank, worldsize, wid = 0, 1, 0
        else:
            rank, worldsize, wid = worker_info.id, worker_info.num_workers, worker_info.id

        # deterministic per-rank shuffle when seed provided
        rng = random.Random(self.seed)
        indices = list(range(len(self.files)))
        rng.shuffle(indices)

        for i in indices:
            if i % worldsize != rank:
                continue
            item = self.files[i]
            url = item["cdn_url"]
            try:
                raw = self._download_one(url)
                if raw is None:
                    continue
                record = self.projection_fn(raw)
                if record is not None:
                    yield record
            except Exception as exc:
                print(f"Skipping {url}: {exc}")
                continue
```

`surrogate/lightning/run_studio.py`
```python
#!/usr/bin/env python3
import argparse
import os
from lightning import Lightning, Teamspace, Machine, Studio

def reuse_or_create_studio(name: str, machine: Machine):
    team = Teamspace()
    for s in team.studios:
        if s.name == name and s.status == "Running":
            print(f"Reusing running studio: {name}")
            return s
    print(f"Creating studio: {name}")
    return Studio(create_ok=True, name=name, machine=machine)

def main():
    p = argparse.ArgumentParser()
    p.add_argument("--filelist", required=True)
    p.add_argument("--script", default="train.py")
    args = p.parse_args()

    machine = Machine.L40S
    studio = reuse_or_create_studio(name="surrogate-train", machine=machine)

    if studio.status != "Running":
        studio.start(machine=machine)

    run_cmd = f"python {args.script} --filelist {args.filelist}"
    studio.run(run_cmd)
    print("Submitted to studio.")

if __name__ == "__main__":
    main()
```
