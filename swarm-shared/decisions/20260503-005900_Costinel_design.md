# Costinel / design

## Implementation Plan — Costinel “Top-Hub Signal” Card (≤2h, frontend-only)

### Highest-value incremental improvement
Add a **read-only, frontend-only “Top-Hub Signal” card** to the Costinel dashboard that:
- Shows the most-connected knowledge-rag hub (e.g., “MOC”) and 3–5 related actionable signals.
- Uses the existing knowledge-rag/graph pattern and top-hub-first guidance.
- Graceful fallback when graph data is unavailable.
- Zero backend changes — all data is embedded/imported or fetched from a static JSON generated by the knowledge-rag pipeline.

### Concrete steps (≤2h)
1. **Create static graph payload** (5–10 min)  
   - Add `/src/data/knowledge-rag-top-hub.json` with `{ hub, signals: [{ title, summary, tags, href }] }`.  
   - This file can be overwritten by the knowledge-rag pipeline later (no code change required).

2. **Create reusable TopHubSignalCard component** (45–60 min)  
   - Place in `/src/components/TopHubSignalCard/TopHubSignalCard.tsx` (or `.jsx` if project uses JS).  
   - Accept `data` prop; render hub badge + signal list.  
   - Design: compact card, tag pills, subtle iconography, accessible (role=list), keyboard-friendly.

3. **Wire into dashboard** (30–45 min)  
   - Import and mount the card in the main dashboard view (likely `/src/pages/Dashboard/Dashboard.tsx` or equivalent).  
   - Position near top of “Cost Analytics & Visibility” or in a new “Insights” panel.  
   - Add graceful fallback UI if data missing or fails to load.

4. **Styling & polish** (15–20 min)  
   - Use existing design tokens (colors, spacing) to match Costinel’s visual language.  
   - Ensure responsive behavior (stack on mobile).

5. **Verify & ship** (10–15 min)  
   - Run dev server, confirm card renders with sample data and falls back cleanly.  
   - No build errors, no runtime exceptions.

### 2) TopHubSignalCard component
`/src/components/TopHubSignalCard/TopHubSignalCard.tsx`
```tsx
import React from 'react';
import './TopHubSignalCard.css';

export interface Signal {
  title: string;
  summary: string;
  tags: string[];
  href?: string;
}

export interface TopHubData {
  hub: string;
  updated?: string;
  signals: Signal[];
}

export interface TopHubSignalCardProps {
  data?: TopHubData | null;
  loading?: boolean;
}

const TagPill: React.FC<{ tag: string }> = ({ tag }) => (
  <span className="ths-tag" title={tag}>
    {tag}
  </span>
);

const TopHubSignalCard: React.FC<TopHubSignalCardProps> = ({ data, loading }) => {
  if (loading) {
    return (
      <div className="ths-card" aria-busy="true">
        <div className="ths-header">
          <div className="ths-hub ths-loading" />
          <div className="ths-meta ths-loading" />
        </div>
        <div className="ths-list">
          {[1, 2, 3].map((i) => (
            <div key={i} className="ths-item ths-loading" />
          ))}
        </div>
      </div>
    );
  }

  if (!data || !data.signals?.length) {
    return (
      <div className="ths-card ths-card--fallback" role="status">
        <p className="ths-fallback">No top-hub insights available at the moment.</p>
      </div>
    );
  }

  const { hub, updated, signals } = data;

  return (
    <article className="ths-card" aria-label={`Top hub: ${hub} — actionable signals`}>
      <header className="ths-header">
        <div className="ths-hub-badge" title={`Most-connected hub (updated ${updated || 'recently'})`}>
          <span className="ths-hub-icon" aria-hidden="true">◆</span>
          <span className="ths-hub-name">{hub}</span>
        </div>
        {updated && <time className="ths-meta" dateTime={updated}>{new Date(updated).toLocaleDateString()}</time>}
      </header>

      <ul className="ths-list" role="list">
        {signals.map((s, idx) => (
          <li key={idx} className="ths-item">
            {s.href ? (
              <a className="ths-link" href={s.href}>
                <h4 className="ths-title">{s.title}</h4>
                <p className="ths-summary">{s.summary}</p>
              </a>
            ) : (
              <div className="ths-body">
                <h4 className="ths-title">{s.title}</h4>
                <p className="ths-summary">{s.summary}</p>
              </div>
            )}
            <div className="ths-tags" aria-label="Tags">
              {s.tags.map((t, ti) => (
                <TagPill key={ti} tag={t} />
              ))}
            </div>
          </li>
        ))}
      </ul>
    </article>
  );
};

export default TopHubSignalCard;
```
