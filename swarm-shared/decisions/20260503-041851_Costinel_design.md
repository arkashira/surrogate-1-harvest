# Costinel / design

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

### Highest-value incremental improvement
Add a resilient, zero-runtime-HF-API “Top-Hub Signal” panel to Costinel that surfaces the most-connected hub (e.g., “MOC”) with baked CDN data, robust fallback UI, and no API calls during render. Uses pre-listed file paths (JSON) embedded at build/deploy time and CDN-only fetches.

---

### Scope (what we ship)
- New panel component: `TopHubSignalPanel`
- Data contract: `top-hub.json` (CDN path) with `{ hub, score, updatedAt, signals[] }`
- Resilient loader: CDN fetch → local fallback → graceful empty state
- Build script: `scripts/fetch-top-hub.js` (run in CI or pre-deploy) to produce `public/data/top-hub.json`
- No runtime HF API, no authentication, no rate-limit risk
- Mobile-first, accessible, fits existing design tokens

---

### Implementation steps (≤2h)

1) Add data contract and loader utility (20m)
   - Create `src/lib/topHub.ts` with typed loader and fallback.
   - Use `fetch` with timeout and cache bust (cache-control handled by CDN).

2) Create panel component (30m)
   - `src/components/TopHubSignalPanel.tsx`
   - Shows hub name, score, last updated, and top signals.
   - Skeleton while loading; inline error state if fetch fails.

3) Add build-time fetch script (20m)
   - `scripts/fetch-top-hub.js`
   - Reads from HF repo tree (single folder) once per deploy (or locally generated stub), writes `public/data/top-hub.json`.
   - Can be run manually or in CI; uses HF CDN URLs only for final asset.

4) Wire into dashboard (15m)
   - Import and place `TopHubSignalPanel` in the dashboard grid.
   - Ensure responsive layout matches existing panels.

5) Polish & test (15m)
   - Verify offline/fallback behavior.
   - Check a11y (role, aria-live for updates).
   - Validate JSON shape and CDN URL.

#### 2) Loader utility — `src/lib/topHub.ts`
```ts
const TOP_HUB_URL = '/data/top-hub.json';
const FALLBACK: TopHubData | null = null;

export interface TopHubSignal {
  id: string;
  title: string;
  severity: 'low' | 'medium' | 'high';
  description: string;
  action: string;
}

export interface TopHubData {
  hub: string;
  score: number;
  updatedAt: string;
  signals: TopHubSignal[];
}

export async function fetchTopHub(timeoutMs = 4000): Promise<TopHubData | null> {
  try {
    const controller = new AbortController();
    const id = setTimeout(() => controller.abort(), timeoutMs);

    const res = await fetch(TOP_HUB_URL, {
      method: 'GET',
      cache: 'no-cache',
      signal: controller.signal,
    });
    clearTimeout(id);

    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    const json = (await res.json()) as TopHubData;

    // Basic shape validation
    if (!json.hub || typeof json.score !== 'number' || !Array.isArray(json.signals)) {
      console.warn('[TopHub] Invalid shape', json);
      return FALLBACK;
    }
    return json;
  } catch (err) {
    console.warn('[TopHub] Fetch failed, using fallback', err);
    return FALLBACK;
  }
}
```
