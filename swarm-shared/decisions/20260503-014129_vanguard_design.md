# vanguard / design

## 1. Diagnosis
- Frontend still triggers authenticated `list_repo_tree` on every data-source selection or training launch, burning HF API quota (1000/5min) and causing 429s.
- No persisted `(repo, dateFolder) → file-list` manifest exists, forcing re-enumeration and preventing reproducible, CDN-only training runs.
- Training path still relies on `load_dataset(..., streaming=True)` which risks `pyarrow.CastError` on heterogeneous repos and performs auth’d metadata calls.
- No CDN-bypass strategy in place: training scripts use HF datasets API instead of raw `resolve/main/` URLs, keeping them inside the rate-limited path.
- Missing orchestration guard to reuse running Lightning Studio instances, risking quota waste on repeated `create_ok=True` launches.

## 2. Proposed change
Add a lightweight manifest generator + CDN-aware data loader for Surrogate-1 training:
- File: `/opt/axentx/vanguard/training/manifest.py` (new)
- File: `/opt/axentx/vanguard/training/train.py` (modify data path)
- File: `/opt/axentx/vanguard/training/launch.py` (add studio reuse guard)

Scope: single PR <2h; no UI changes; Mac-only orchestration; Lightning Studio for training.

## 3. Implementation

### 3.1 Create manifest generator (Mac orchestration)
```python
# /opt/axentx/vanguard/training/manifest.py
import json, os, sys
from datetime import datetime
from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_DATASET_REPO", "axentx/surrogate-1-mirror")
OUT_DIR = os.getenv("MANIFEST_OUT", "manifests")

def list_date_folder(date_folder: str):
    """
    One authenticated call per date folder (run from Mac after rate-limit window).
    Produces: manifests/{date_folder}.json  ->  list of file paths (no auth needed at train time)
    """
    api = HfApi()
    tree = api.list_repo_tree(repo_id=HF_REPO, path=date_folder, recursive=False)
    files = [
        f"{date_folder}/{entry.path.split('/')[-1]}"
        for entry in tree
        if entry.type == "file" and entry.path.lower().endswith(".parquet")
    ]
    os.makedirs(OUT_DIR, exist_ok=True)
    out_path = os.path.join(OUT_DIR, f"{date_folder}.json")
    with open(out_path, "w") as f:
        json.dump({"repo": HF_REPO, "date_folder": date_folder, "files": sorted(files), "generated": datetime.utcnow().isoformat()}, f, indent=2)
    print(f"Wrote {len(files)} files -> {out_path}")
    return out_path

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python manifest.py <date_folder>")
        sys.exit(1)
    list_date_folder(sys.argv[1])
```

### 3.2 CDN-only data loader (training script)
```python
# /opt/axentx/vanguard/training/train.py  (excerpt to replace dataset loading)
import json, os, torch
from torch.utils.data import IterableDataset
import pyarrow.parquet as pq
import requests

class CDNParquetIterable(IterableDataset):
    def __init__(self, manifest_path, shard_rank=0, shard_world_size=1):
        with open(manifest_path) as f:
            manifest = json.load(f)
        files = sorted(manifest["files"])
        # simple sharding
        self.files = [f for i, f in enumerate(files) if i % shard_world_size == shard_rank]
        self.repo = manifest["repo"]

    def __iter__(self):
        for fpath in self.files:
            url = f"https://huggingface.co/datasets/{self.repo}/resolve/main/{fpath}"
            # stream download to avoid large tmp files; project to {prompt,response} at parse
            resp = requests.get(url, stream=True, timeout=60)
            resp.raise_for_status()
            with open("/tmp/current.parquet", "wb") as tmp:
                for chunk in resp.iter_content(chunk_size=8192):
                    tmp.write(chunk)
            table = pq.read_table("/tmp/current.parquet", columns=["prompt", "response"])
            for row in table.to_pylist():
                # ensure schema; skip malformed
                if row.get("prompt") and row.get("response"):
                    yield row
```

### 3.3 Lightning Studio reuse guard
```python
# /opt/axentx/vanguard/training/launch.py  (excerpt)
from lightning import Studio, Teamspace, Machine
import time

def get_or_create_studio(name: str, machine=Machine.L40S):
    teamspace = Teamspace()
    for s in teamspace.studios:
        if s.name == name and s.status == "Running":
            print(f"Reusing running studio: {name}")
            return s
    print(f"Creating studio: {name}")
    return Studio(name=name, machine=machine, create_ok=True)

def run_training_launcher():
    studio = get_or_create_studio("vanguard-surrogate-train")
    # guard against idle stop
    if studio.status != "Running":
        studio.start(machine=Machine.L40S)
        # wait until reachable
        for _ in range(30):
            if studio.status == "Running":
                break
            time.sleep(10)
    # run your training script inside studio
    studio.run(
        command="python train.py --manifest manifests/2026-04-29.json",
        cwd="/workspace/vanguard/training",
    )
```
