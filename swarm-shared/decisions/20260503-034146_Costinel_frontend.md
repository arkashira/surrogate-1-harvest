# Costinel / frontend

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a non-blocking Top‑Hub Signal Panel to the Costinel dashboard that surfaces the most‑connected hub (e.g., "MOC") using CDN‑first data baked at build time (zero HuggingFace API calls at runtime).

### Why this ships highest value in <2h
- Visible, contextual insight aligned to #knowledge-rag #graph #hub patterns.
- Zero runtime API/rate-limit risk (CDN-only).
- Reuses existing build pipeline; no infra changes.
- Small UI surface (panel card) + static JSON = minimal code.

---

### Concrete steps (timeboxed)

1. **Add baked data artifact**  
   Create `public/data/top-hub.json` (committed at build time by CI/knowledge-rag job).  
   Schema: `{ "hub": "MOC", "score": 0.94, "connections": 1280, "updatedAt": "2026-05-03T03:40:00.000Z", "summary": "Most Operational Context hubs drive cost governance decisions." }`

2. **Create typed fetch utility (CDN-only)**  
   Add `src/lib/topHub.ts` that fetches `/data/top-hub.json` with cache headers and graceful fallback.

3. **Add TopHubPanel component**  
   Place in `src/components/TopHubPanel.tsx` — skeleton-aware, non-blocking, dismissible.

4. **Mount into dashboard**  
   Insert into main dashboard layout (`src/pages/Dashboard.tsx` or equivalent) near cost summary cards.

5. **Build-time note**  
   Document that CI should run knowledge-rag top-hub query and write `public/data/top-hub.json` before `npm run build`.

#### src/lib/topHub.ts
```ts
export interface TopHub {
  hub: string;
  score: number;
  connections: number;
  updatedAt: string;
  summary: string;
}

const DEFAULT_HUB: TopHub = {
  hub: "—",
  score: 0,
  connections: 0,
  updatedAt: new Date().toISOString(),
  summary: "Top hub unavailable.",
};

export async function fetchTopHub(signal?: AbortSignal): Promise<TopHub> {
  try {
    const res = await fetch("/data/top-hub.json", {
      signal,
      cache: "no-cache",
    });
    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    const json = (await res.json()) as TopHub;
    return json;
  } catch {
    return DEFAULT_HUB;
  }
}
```

#### src/components/TopHubPanel.tsx
```tsx
import { useEffect, useState } from "react";
import { fetchTopHub, TopHub } from "../lib/topHub";

export function TopHubPanel() {
  const [hub, setHub] = useState<TopHub | null>(null);
  const [dismissed, setDismissed] = useState(false);

  useEffect(() => {
    const controller = new AbortController();
    fetchTopHub(controller.signal).then(setHub);
    return () => controller.abort();
  }, []);

  if (dismissed || !hub) return null;

  return (
    <section
      className="rounded-lg border bg-card p-4 shadow-sm"
      role="region"
      aria-label="Top hub signal"
    >
      <div className="flex items-start justify-between gap-4">
        <div className="flex-1">
          <div className="flex items-baseline gap-2">
            <h3 className="text-sm font-semibold text-foreground">Top Hub</h3>
            <span className="rounded bg-primary/10 px-2 py-0.5 text-xs font-medium text-primary">
              {hub.hub}
            </span>
          </div>
          <p className="mt-1 text-xs text-muted-foreground">{hub.summary}</p>
          <dl className="mt-2 flex gap-4 text-xs">
            <div>
              <dt className="text-muted-foreground">Score</dt>
              <dd className="font-medium text-foreground">{(hub.score * 100).toFixed(0)}%</dd>
            </div>
            <div>
              <dt className="text-muted-foreground">Connections</dt>
              <dd className="font-medium text-foreground">{hub.connections.toLocaleString()}</dd>
            </div>
            <div>
              <dt className="text-muted-foreground">Updated</dt>
              <dd className="font-medium text-foreground">
                {new Date(hub.updatedAt).toLocaleDateString()}
              </dd>
            </div>
          </dl>
        </div>
        <button
          onClick={() => setDismissed(true)}
          className="text-muted-foreground hover:text-foreground -mr-1.5 -mt-1.5 rounded p-1.5 text-xs"
          aria-label="Dismiss top hub panel"
        >
          ✕
        </button>
      </div>
    </section>
  );
}
```
