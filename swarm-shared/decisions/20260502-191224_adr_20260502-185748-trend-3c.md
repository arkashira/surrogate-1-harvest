## Deployment
cf workers (free-tier) + Supabase (free-tier) + optional HF space for hosted rule/transformer models

## v0.1 scope
CLI-first v0.1 in 2 weeks: local repo indexing (git), Python and JavaScript/TypeScript analyzers (AST + lint + basic secret detection), Postgres-backed user/repo metadata (Supabase free), SQLite local cache for results, export findings as JSON/SARIF, single-machine performance target <2 min for 10k LOC on 8GB RAM machine, no remote processing by default.
