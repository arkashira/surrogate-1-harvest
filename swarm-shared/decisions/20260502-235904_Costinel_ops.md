# Costinel / ops

## Implementation Plan — Costinel Quality: Top-Hub Signal Card (Read-Only)

**Scope**: Read-only frontend card (≤2h)  
**Principle**: “Sense + Signal — ไม่ Execute” (strictly no writes, no self-execution)  
**Goal**: Surface the most-connected hub (e.g., “MOC”) with score, concise context, and provenance links — zero mutations, zero side effects.

### 1) Highest-value incremental improvement (ship <2h)
Add a read-only “Top-Hub Signal” card to the dashboard that:
- Queries the local knowledge-rag graph for the highest-degree hub (by connection count + recency weight).
- Shows: hub name, score, 1-sentence insight, top 3 related docs (with links), and last-updated timestamp.
- Uses CDN-backed static file fetch (no HF API during render) and caches result for 5min to avoid rate limits.
- Strictly read-only: no POST/PUT, no state changes, no background jobs.

### 2) Implementation steps

#### A) Backend (FastAPI read-only endpoint) — `app/api/top_hub.py`
```python
# app/api/top_hub.py
from fastapi import APIRouter
from pathlib import Path
import json
import time
from typing import List, Optional

router = APIRouter()

# Path to pre-computed hub list (Mac orchestration writes this via HF CDN list-repo-tree)
HUB_CACHE = Path("cache/top_hub.json")
CACHE_TTL = 300  # seconds

def _load_cached() -> Optional[dict]:
    if not HUB_CACHE.exists():
        return None
    if time.time() - HUB_CACHE.stat().st_mtime > CACHE_TTL:
        return None
    try:
        return json.loads(HUB_CACHE.read_text())
    except Exception:
        return None

@router.get("/top-hub", response_model=dict)
async def get_top_hub() -> dict:
    """
    Read-only signal: top hub by degree/recency from local graph.
    No mutations. No HF API calls at render time (uses CDN file list).
    """
    cached = _load_cached()
    if cached:
        return cached

    # Fallback minimal payload when cache miss (safe defaults)
    return {
        "hub": "MOC",
        "score": 0.0,
        "insight": "No recent graph data available.",
        "related_docs": [],
        "provenance": "cache/top_hub.json",
        "ts": int(time.time()),
        "cached": False,
    }
```

#### C) Frontend card (React/TypeScript) — `src/components/TopHubSignalCard.tsx`
```tsx
// src/components/TopHubSignalCard.tsx
import React, { useEffect, useState } from "react";
import "./TopHubSignalCard.css";

interface RelatedDoc {
  name: string;
  url: string;
}

interface TopHubPayload {
  hub: string;
  score: number;
  insight: string;
  related_docs: string[];
  provenance: string;
  ts: number;
  cached?: boolean;
}

const CDN_BASE = "https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main";

export const TopHubSignalCard: React.FC = () => {
  const [data, setData] = useState<TopHubPayload | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch("/api/top-hub", { cache: "no-store" })
      .then((r) => r.json())
      .then((j) => {
        setData(j);
        setLoading(false);
      })
      .catch(() => setLoading(false));
  }, []);

  if (loading) return <div className="top-hub-card skeleton">Loading signal…</div>;
  if (!data) return null;

  const related: RelatedDoc[] = (data.related_docs || []).map((p) => ({
    name: String(p).split("/").pop() || String(p),
    url: `${CDN_BASE}/${String(p).replace(/^\/+/, "")}`,
  }));

  return (
    <div className="top-hub-card" role="region" aria-label="Top hub signal">
      <div className="top-hub-header">
        <span className="top-hub-badge">Top Hub</span>
        <span className="top-hub-score" title="connection score">
          {Number(data.score).toFixed(2)}
        </span>
      </div>
      <h3 className="top-hub-name">{data.hub}</h3>
      <p className="top-hub-insight">{data.insight}</p>
      {related.length > 0 && (
        <ul className="top-hub-docs">
          {related.map((d, i) => (
            <li key={i}>
              <a href={d.url} target="_blank" rel="noopener noreferrer">
                {d.name}
              </a>
            </li>
          ))}
        </ul>
