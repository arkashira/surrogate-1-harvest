# Costinel / design

## Implementation Plan — Costinel Top-Hub Signal Endpoint

**Highest-value incremental improvement (<2h):**  
Expose a **read-only, side-effect-free** `GET /api/v1/cost-anomaly/signal/top-hub` that surfaces the most-connected hub (e.g., "MOC") with contextual insights (degree, affected services, cost impact, and recommended signals). This aligns with the **top-hub doc insight** pattern and gives operators immediate, actionable context without executing changes.

---

### 1) Design (5–10 min)

- **Endpoint**: `GET /api/v1/cost-anomaly/signal/top-hub`
- **Auth**: Existing Costinel auth (no change).
- **Response shape** (JSON):
  ```json
  {
    "hub": "MOC",
    "degree": 42,
    "rank": 1,
    "affected_services": ["EC2", "RDS", "EKS"],
    "cost_impact_usd": 12430.50,
    "anomaly_score": 0.87,
    "signals": [
      {
        "id": "sig-01HV3X0Z...",
        "type": "spike",
        "severity": "high",
        "description": "Unusual outbound data transfer pattern from MOC-linked accounts",
        "recommendation": "Review NAT gateway and VPC endpoints for data egress optimization"
      }
    ],
    "window": {
      "from": "2026-05-02T00:00:00Z",
      "to": "2026-05-02T23:59:59Z"
    },
    "generated_at": "2026-05-02T23:30:00Z"
  }
  ```
- **Behavior**:
  - Read-only; no state changes.
  - Fast (<500 ms) — compute degree/rank from in-memory graph or lightweight materialized view.
  - If graph not ready, return 204 or cached last-known hub with `stale: true` flag.

---

### 2) Implementation Steps (~60–80 min)

1. **Add route** in API router (FastAPI assumed).  
   File: `app/api/v1/endpoints/cost_anomaly.py` (or similar).

2. **Create service function** to resolve top hub.  
   File: `app/services/top_hub.py`.

3. **Expose minimal graph accessor** (reuse existing knowledge-rag/graph module if present).  
   If no graph module, implement lightweight degree counting from recent anomaly links.

4. **Add tests** (optional for <2h ship) — at least one smoke test in `tests/api/test_top_hub.py`.

5. **Update docs** (README or OpenAPI) — mark endpoint as `#knowledge-rag #graph #hub`.

#### `app/services/top_hub.py`
```python
from datetime import datetime, timezone
from app.schemas.top_hub import TopHubSignalResponse, SignalItem, Window
from app.graph.hub_graph import HubGraph  # lightweight graph accessor

def get_top_hub_signal() -> TopHubSignalResponse:
    graph = HubGraph.load_recent()  # uses cached/materialized view
    top = graph.top_hub()

    if not top:
        # graceful fallback
        return TopHubSignalResponse(
            hub="unknown",
            degree=0,
            rank=0,
            affected_services=[],
            cost_impact_usd=0.0,
            anomaly_score=0.0,
            signals=[],
            window=Window(
                from_=datetime.now(timezone.utc).replace(hour=0, minute=0, second=0),
                to=datetime.now(timezone.utc)
            ),
            generated_at=datetime.now(timezone.utc),
            stale=True
        )

    signals = _derive_signals(top)
    window = Window(
        from_=top.window_start,
        to=top.window_end
    )

    return TopHubSignalResponse(
        hub=top.name,
        degree=top.degree,
        rank=1,
        affected_services=top.affected_services,
        cost_impact_usd=top.cost_impact_usd,
        anomaly_score=top.anomaly_score,
        signals=signals,
        window=window,
        generated_at=datetime.now(timezone.utc),
        stale=False
    )

def _derive_signals(top_hub) -> List[SignalItem]:
    # Very small heuristic; replace with real signal source
    items: List[SignalItem] = []
    if top_hub.anomaly_score > 0.8:
        items.append(
            SignalItem(
                id=f"sig-{top_hub.name}-{int(datetime.now(timezone.utc).timestamp())}",
                type="spike",
                severity="high",
                description=f"Unusual activity pattern from {top_hub.name}-linked accounts",
                recommendation="Review linked resources for optimization or policy exceptions"
            )
        )
    return items
```
