# airship / backend

## Implementation Plan — `airship discover` (backend)

**Scope**: ≤2h  
**Goal**: Deterministic CLI command `airship discover` that emits a CDN-only file manifest for a given `repo_id` and `date_folder`, plus HTTP endpoint `/api/discover` returning the same manifest (JSON). Enables surrogate/kaggle training pipelines with zero API calls during data loading.

### Why this is highest-value
- Unblocks surrogate training (HF API rate limits) by producing CDN-only file lists.
- Reusable by both CLI automation and Arkship UI (Discover page can consume `/api/discover`).
- Small, self-contained change (<2h) with immediate downstream impact.

---

### Implementation steps

1. Add CLI command `airship discover` (Typer-based) under `airship/cli/` (or top-level CLI module).
   - Args: `--repo-id`, `--date-folder`, `--output` (default: stdout JSON).
   - Uses HF Hub `list_repo_tree(path=date_folder, recursive=False)` (single call) to list files.
   - Emits manifest:
     ```json
     {
       "repo_id": "...",
       "date_folder": "...",
       "generated_at": "...",
       "files": [
         {
           "path": "file1.parquet",
           "cdn_url": "https://huggingface.co/datasets/{repo_id}/resolve/main/{date_folder}/file1.parquet",
           "size": 12345
         }
       ]
     }
     ```
   - If `date_folder` is empty or omitted, list top-level non-recursive.

2. Add FastAPI endpoint in Arkship backend (`/api/discover`) that accepts same params (query or body) and returns the same manifest JSON.
   - Reuse the same manifest-generation function from CLI.
   - Add lightweight caching (per-repo+folder, 5–10 min) to avoid repeated `list_repo_tree` calls under load.

3. Add utility module `airship/hf_cdn.py` with:
   - `build_manifest(repo_id: str, folder: str = "") -> dict`
   - Uses `huggingface_hub` (already a dependency) with `list_repo_tree`.
   - Handles 429 gracefully: wait 360s and retry once (per pattern).
   - Validates repo exists; returns clear error if not.

4. Ensure CLI is executable and registered in `pyproject.toml` entrypoints or as runnable module (`python -m airship.cli discover ...`).

5. Add minimal tests (optional but recommended):
   - Unit test for manifest shape using mocked `list_repo_tree`.
   - Integration test for endpoint (if test harness exists).

### Code snippets

#### `airship/hf_cdn.py`
```python
import time
from datetime import datetime, timezone
from typing import List, Dict, Any

from huggingface_hub import HfApi, RepositoryNotFoundError, HFValidationError
from huggingface_hub.utils import HFValidationError as HubValidationError

HF_API = HfApi()


def build_manifest(repo_id: str, folder: str = "") -> Dict[str, Any]:
    """
    Build a CDN-only file manifest for a repo folder.
    Uses a single list_repo_tree call (non-recursive).
    """
    if not repo_id:
        raise ValueError("repo_id is required")

    # Retry once on 429 (per pattern: wait 360s)
    for attempt in range(2):
        try:
            tree = HF_API.list_repo_tree(repo_id=repo_id, path=folder or "", recursive=False)
            break
        except Exception as exc:
            # Heuristic: if looks like rate limit, wait and retry once
            if attempt == 0 and getattr(exc, "status_code", None) == 429:
                time.sleep(360)
                continue
            raise

    files = []
    for item in tree:
        # tree items have path and (sometimes) size
        path = getattr(item, "path", None)
        if not path:
            continue
        size = getattr(item, "size", None)
        files.append(
            {
                "path": path,
                "cdn_url": f"https://huggingface.co/datasets/{repo_id}/resolve/main/{path}",
                "size": size,
            }
        )

    manifest = {
        "repo_id": repo_id,
        "date_folder": folder or "",
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "files": files,
    }
    return manifest
```

#### `airship/cli/discover.py` (or integrate into existing CLI)
```python
import json
import sys
from typing import Optional

import typer

from airship.hf_cdn import build_manifest

app = typer.Typer()


@app.command()
def discover(
    repo_id: str = typer.Option(..., "--repo-id", help="HF dataset repo id (e.g., 'org/dataset')"),
    date_folder: Optional[str] = typer.Option(None, "--date-folder", help="Folder path inside repo (e.g., '2026-05-01')"),
    output: Optional[str] = typer.Option(None, "--output", help="Output file path (default: stdout)"),
):
    """
    Emit a CDN-only file manifest for deterministic surrogate training.
    """
    try:
        manifest = build_manifest(repo_id=repo_id, folder=date_folder or "")
    except Exception as exc:
        typer.echo(f"Error: {exc}", err=True)
        sys.exit(1)

    out_text = json.dumps(manifest, indent=2)
    if output:
        with open(output, "w", encoding="utf-8") as f:
            f.write(out_text)
        typer.echo(f"Manifest written to {output}")
    else:
        typer.echo(out_text)
```

#### FastAPI endpoint (Arkship backend)
Add to an appropriate router (e.g., `arkship/api/routes/discover.py`):

```python
from fastapi import APIRouter, Query, HTTPException
from typing import Optional
from functools import lru_cache
import time

from airship.hf_cdn import build_manifest

router = APIRouter()


# lightweight in-memory cache: (repo_id, folder) -> (timestamp, manifest)
_CACHE: dict = {}
_CACHE_TTL = 60 * 5  # 5 minutes


def _cached_manifest(repo_id: str, folder: str):
    key = (repo_id, folder)
    now = time.time()
    if key in _CACHE:
        ts, manifest = _CACHE[key]
        if now - ts < _CACHE_TTL:
            return manifest
    manifest = build_manifest(repo_id=repo_id, folder=folder)
    _CACHE[key] = (now, manifest)
    return manifest


@router.get("/api/discover")
def api_discover(
    repo_id: str = Query(..., description="HF dataset repo id"),
    date_folder: Optional[str] = Query(None, description="Folder inside repo (e.g., 2026-05-01)"),
):
    try:
        manifest = _cached_manifest(repo_id=repo_id, folder=date_folder or "")
        return manifest
    except RepositoryNotFoundError:
        raise HTTPException(status_code=404, detail="Repository not found")
    except Exception as exc:
        raise HTTPException(status_code=500, detail=str(exc))
```
