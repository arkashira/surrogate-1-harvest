# Costinel / design

## Final Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h, production-ready)

### Scope (single highest-value deliverable)
Add a **non-blocking Top-Hub Signal Panel** to the Costinel dashboard that:
- Detects and surfaces the most-connected hub (default `MOC`) from a lightweight, static graph snapshot.
- Uses **CDN-only fetches** (no HF API/auth, zero runtime backend calls) for production resilience and zero rate-limit risk.
- Renders as a **dismissible, SSR-friendly** sidebar card with contextual insights, tags, and quick-links.
- Fully typed (TypeScript), respects existing design tokens, and is removable with zero migration cost.
- Graceful degradation: if CDN assets 404, the panel silently hides (non-blocking, no UI skeleton).

---

### Architecture (CDN-first, client-only)
```
Costinel Dashboard
└─ TopHubSignalPanel (client-only, SSR-safe)
   ├─ /data/hub-graph-snapshot.json        ← pre-built hub graph (CDN)
   ├─ /data/hubs/moc.md                    ← hub insight doc (CDN)
   └─ /data/hubs/moc-links.json            ← curated quick-links (CDN)
```
- **No API calls during runtime** — all assets served via CDN (`/data/...`).
- **Build-time or ops-time generation** — snapshot can be regenerated by ops/Mac orchestration and committed; not built in-browser.
- **SSR-safe** — component renders nothing on the server and hydrates safely on the client.

#### 4) Add TopHubSignalPanel component (TypeScript, SSR-safe)
`/opt/axentx/Costinel/src/components/TopHubSignalPanel.tsx`
```tsx
import React, { useEffect, useState, useCallback } from 'react';
import './TopHubSignalPanel.css';

const CACHE_KEY = 'costinel-tophub-dismissed';
const GRAPH_URL = '/data/hub-graph-snapshot.json';

interface Hub {
  id: string;
  label: string;
  connections: number;
  rank: number;
  insightDoc: string;
  links: string;
  tags: string[];
}

interface GraphSnapshot {
  generatedAt: string;
  topHubId: string;
  hubs: Hub[];
}

export const TopHubSignalPanel: React.FC = () => {
  const [hub, setHub] = useState<Hub | null>(null);
  const [dismissed, setDismissed] = useState(false);

  // Check dismiss state (client-only)
  useEffect(() => {
    try {
      if (localStorage.getItem(CACHE_KEY) === 'true') {
        setDismissed(true);
      }
    } catch {
      // ignore
    }
  }, []);

  // Fetch graph snapshot (CDN-only, client-only)
  useEffect(() => {
    if (dismissed) return;

    fetch(GRAPH_URL, { cache: 'no-cache' })
      .then((res) => {
        if (!res.ok) throw new Error('CDN asset missing');
        return res.json() as Promise<GraphSnapshot>;
      })
      .then((data) => {
        const topId = data.topHubId || data.hubs?.[0]?.id;
        const topHub = data.hubs?.find((h) => h.id === topId) ?? null;
        setHub(topHub);
      })
      .catch(() => {
        // Non-blocking: silently hide panel
        setHub(null);
      });
  }, [dismissed]);

  const handleDismiss = useCallback(() => {
    try {
      localStorage.setItem(CACHE_KEY, 'true');
    } catch {
      // ignore
    }
    setDismissed(true);
  }, []);

  if (dismissed || !hub) return null;

  return (
    <aside className="top-hub-signal-panel" role="complementary" aria-label="Top hub signal">
      <button
        className="top-hub-signal-panel__dismiss"
        onClick={handleDismiss}
        aria-label="Dismiss top hub panel"
        title="Dismiss"
      >
        ×
      </button>

      <div className="top-hub-signal-panel__header">
        <span className="top-hub-signal-panel__badge">Top Hub</span>
        <strong className="top-hub-signal-panel__title">{hub.label}</strong>
        <span className="top-hub-signal-panel__meta">{hub.connections} connections</span>
      </div>

      <div className="top-hub-signal-panel__tags">
        {hub.tags.map((t) => (
          <span key={t} className="top-hub-signal-panel__tag">
            {t}
          </span>
        ))}
      </div>

      <div className="top-hub-signal-panel__actions">
        <a
          href={`/proposals?hub=${encodeURIComponent(hub.label)}&source=top-hub-signal-panel`}
          className="top-hub-signal-panel__btn"
        >
          Review proposals
        </a>
        <a
          href={hub.insightDoc}
          className="top-hub-signal-panel__link"
          target="_blank"
          rel="noopener noreferrer"
        >
          Hub insight →
        </a>
      </div>

      <footer className="top-hub-signal-panel__footer">
        <small>Sense + Signal — ไม่ Execute</small>
      </footer>
    </aside>
  );
};
```
