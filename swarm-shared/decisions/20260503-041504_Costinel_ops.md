# Costinel / ops

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a lightweight, resilient “Top-Hub Signal” panel to Costinel that surfaces the most‑connected hub (e.g., “MOC”) with **zero runtime HF API calls**, using CDN‑first baked data and robust fallbacks.

### Why this is the highest-value <2h item
- Directly applies the **top-hub doc insight** and **knowledge-rag** patterns.
- Uses **CDN-first** strategy (no runtime HF API) to avoid rate limits.
- Minimal surface: one React panel + one build-time data file + one cron script.
- Improves ops visibility without touching billing or execution paths.

---

### Architecture (CDN-first)

```
Mac (or CI) cron
  └─ generate-top-hub.js
        ├─ list_repo_tree (once per day, after rate-limit window)
        ├─ pick latest /batches/mirror-merged/{date}/*.parquet
        ├─ project to {prompt,response} only
        └─ upload baked top-hub JSON to repo (or CDN path)

Costinel (runtime)
  └─ TopHubSignalPanel (React)
        ├─ fetch("/data/top-hub-latest.json")  ← CDN URL
        ├─ fallback to bundled default
        └─ render card + insights
```

#### 2) React panel component
`src/components/TopHubSignalPanel.jsx`
```jsx
import React, { useEffect, useState } from "react";
import "./TopHubSignalPanel.css";

const FALLBACK = {
  hub: "MOC",
  score: 0.94,
  connections: 127,
  lastUpdated: "2026-05-03T04:00:00Z",
  insights: [
    "MOC shows strongest cross-project signal for cost anomaly governance.",
    "Top referenced pattern: CDN-first + zero-runtime-HF-API.",
    "Recommended action: enable daily top-hub digest for finance leads."
  ]
};

export default function TopHubSignalPanel() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch("/data/top-hub-latest.json", { cache: "no-store" })
      .then((r) => {
        if (!r.ok) throw new Error("Network response not ok");
        return r.json();
      })
      .then((json) => {
        setData(json);
        setLoading(false);
      })
      .catch(() => {
        // CDN fallback -> bundled default
        setData(FALLBACK);
        setLoading(false);
      });
  }, []);

  if (loading) {
    return (
      <div className="top-hub-panel loading">
        <span>Loading top hub signal…</span>
      </div>
    );
  }

  const ago = new Date().toLocaleString(undefined, {
    dateStyle: "short",
    timeStyle: "short"
  });

  return (
    <div className="top-hub-panel" role="region" aria-label="Top hub signal">
      <div className="top-hub-header">
        <span className="top-hub-badge">Top Hub</span>
        <span className="top-hub-name">{data.hub}</span>
        <span className="top-hub-score">{Math.round(data.score * 100)}%</span>
      </div>

      <div className="top-hub-meta">
        <span>Connections: {data.connections}</span>
        <span>Updated: {new Date(data.lastUpdated).toLocaleDateString()}</span>
      </div>

      <ul className="top-hub-insights">
        {data.insights.map((t, i) => (
          <li key={i}>{t}</li>
        ))}
      </ul>

      <div className="top-hub-footer">
        <small>Rendered at {ago} (CDN-first)</small>
      </div>
    </div>
  );
}
```
