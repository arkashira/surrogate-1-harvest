## Final Synthesized Plan (Best Parts + Corrected Contradictions)

I have synthesized the strongest elements from both proposals. 

**Resolution of Contradictions:**
1. **HF API vs. Local Docker:** Candidate 1 assumed a HuggingFace dataset context; Candidate 2 assumed local Docker microservices. **Resolution:** We implement the *local Docker* checks (Candidate 2) as the primary mechanism because the prompt context ("Arkship API 8000", "Surrogate AI 8001") implies a running local stack. However, we preserve the *CDN-safe asset manifest logic* (Candidate 1) for static assets.
2. **Python vs. Pure Bash:** Candidate 1 used Python for JSON parsing; Candidate 2 implied Bash. **Resolution:** We use **Bash + `jq`** for portability and to avoid Python dependency sprawl in a Bash orchestrator, but we keep the CDN URL generation logic intact.
3. **Output Directory:** Candidate 1 used `./dist`; Candidate 2 used `./public/airship`. **Resolution:** We use `./public/airship` (Candidate 2) as it aligns with standard static web serving, but we parameterize it.

### 1. The Orchestrator: `scripts/airship-discover.sh`

**Location:** `/opt/axentx/airship/scripts/airship-discover.sh`

```bash
#!/usr/bin/env bash
# airship discover
# Orchestrates deterministic health checks and CDN-safe asset manifests.
# Produces:
#   public/airship/status.json  - deterministic health snapshot
#   public/airship/manifest.json - CDN asset manifest (resolve/main URLs)
#
# Usage:
#   bash scripts/airship-discover.sh [--out-dir ./public/airship] [--date-folder YYYY-MM-DD]
#
# Cron:
#   SHELL=/bin/bash
#   */15 * * * * cd /opt/axentx/airship && bash scripts/airship-discover.sh >> logs/discover.log 2>&1

set -euo pipefail
IFS=$'\n\t'

# ---- Configuration ----
OUTDIR="./public/airship"
DATE_FOLDER="$(date +%Y-%m-%d)"
TIMESTAMP="$(date -u +"%Y-%m-%dT%H:%M:%SZ")"
HF_REPO="${HF_REPO:-datasets/axentx/arkship-mirror}" # For asset manifest

# Parse arguments
while [[ $# -gt 0 ]]; do
  case $1 in
    --out-dir) OUTDIR="$2"; shift ;;
    --date-folder) DATE_FOLDER="$2"; shift ;;
    *) echo "Unknown option: $1"; exit 3 ;;
  esac
  shift
done

mkdir -p "$OUTDIR" "$(dirname "$0")/../logs"

log() { echo "[$(date -u +"%Y-%m-%dT%H:%M:%SZ")] $*" ; }

# ---- 1) Service Health Checks (Docker + HTTP) ----
check_service() {
  local name=$1
  local port=$2
  local path=${3:-"/health"}
  
  # Check Docker container status
  local docker_status
  docker_status=$(docker ps --filter "name=${name}" --filter "status=running" --format "{{.Names}}" 2>/dev/null || true)
  
  if [ -z "$docker_status" ]; then
    echo "\"degraded\""
    return
  fi

  # Check HTTP endpoint
  if curl -fs --max-time 5 "http://localhost:${port}${path}" > /dev/null 2>&1; then
    echo "\"healthy\""
  else
    echo "\"unhealthy\""
  fi
}

ARKSHIP_API_STATUS=$(check_service "arkship-api" 8000 "/health")
SURROGATE_AI_STATUS=$(check_service "surrogate-ai" 8001 "/health")
ARKSHIP_UI_STATUS=$(check_service "arkship-ui" 3000 "/")

# ---- 2) Build status.json ----
# Using jq for deterministic JSON generation
cat << EOF | jq '.' > "$OUTDIR/status.json"
{
  "service": "airship",
  "generated_at": "$TIMESTAMP",
  "date_folder": "$DATE_FOLDER",
  "healthy": $(if [[ "$ARKSHIP_API_STATUS" == "\"healthy\"" && "$SURROGATE_AI_STATUS" == "\"healthy\"" ]]; then echo "true"; else echo "false"; fi),
  "checks": {
    "arkship_api": $ARKSHIP_API_STATUS,
    "surrogate_ai": $SURROGATE_AI_STATUS,
    "arkship_ui": $ARKSHIP_UI_STATUS,
    "cdn_urls_only": true
  },
  "counts": {
    "assets": 0,
    "parquet": 0,
    "json": 0
  },
  "runtime": {
    "mode": "cdn_safe",
    "api_calls_per_run": 1,
    "training_uses_cdn_only": true
  }
}
EOF

# ---- 3) Build Asset Manifest (CDN-safe) ----
# This section uses the HF logic from Candidate 1, but runs conditionally
# to avoid blocking the main health check if HF is unreachable.

MANIFEST_ASSETS="[]"
if command -v python3 &> /dev/null; then
  # Use Python to safely query HF (isolated failure)
  MANIFEST_ASSETS=$(python3 <<PY 2>/dev/null || echo "[]")
import json, os, sys
try:
    from huggingface_hub import list_repo_tree
    repo = os.environ.get("HF_REPO", "datasets/axentx/arkship-mirror")
    tree = list_repo_tree(repo_id=repo, path="$DATE_FOLDER", recursive=False)
    files = [t.path for t in tree if t.type == 'file']
    
    assets = []
    for rel in sorted(files):
        assets.append({
            "path": rel,
            "url": f"https://huggingface.co/datasets/{repo}/resolve/main/{rel}",
            "size": None,
            "etag": None
        })
    print(json.dumps(assets))
except Exception:
    print("[]")
PY
)
fi

# Merge Manifest
ASSET_COUNT=$(echo "$MANIFEST_ASSETS" | jq 'length')

cat << EOF | jq --argjson assets "$MANIFEST_ASSETS" '.counts.assets = ($assets | length) | .counts.parquet = ($assets | map(select(.path | endswith(".parquet"))) | length) | .counts.json = ($assets | map(select(.path | endswith(".json"))) | length) | .assets = $assets' > "$OUTDIR/manifest.json"
{
  "generated_at": "$TIMESTAMP",
  "repo": "$HF_REPO",
  "date_folder": "$DATE_FOLDER",
  "assets": []
}
EOF

# Update status.json with correct counts
TMP_STATUS=$(mktemp)
jq --argjson cnt "$ASSET_COUNT" '.counts.assets = $cnt' "$OUTDIR/status.json" > "$TMP_STATUS" && mv "$TMP_STATUS" "$OUTDIR/status.json"

# ---- 4) Cache-Busting Hash ----
MANIFEST_HASH="$(sha256sum "$OUTDIR/manifest.json" | awk '{print $1}' | cut -c1-12)"
echo "$MANIFEST_HASH" > "$OUTDIR/manifest-hash.txt"

# ---- 5) Exit Code Logic ----
if [[ "$ARKSHIP_API_STATUS" == "\"unhealthy\"" ]] || [[ "$SURROGATE_AI_STATUS" == "\"unhealthy\"" ]]; then
  log "discover complete: DEGRADED ($ASSET_COUNT assets)"
  exit 1
elif [[ "$ARKSHIP_API_STATUS" == "\"degraded\"" ]] || [[ "$SURROGATE_AI_STATUS" == "\"degraded\"" ]]; then
  log "discover complete: DEGRADED ($ASSET_COUNT assets)"
  exit 1
else
  log "discover complete: HEALTHY ($ASSET_COUNT assets, hash=$MANIFEST_HASH)"
  exit 0
fi
```

**Make Executable:**
```bash
chmod +x /opt/axentx/airship/scripts/airship-discover.sh
```
