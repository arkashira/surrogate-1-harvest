# vanguard / design

## 1. Diagnosis

- Frontend/training workflows still trigger authenticated `list_repo_tree` on every data-source selection or training launch, burning HF API quota (1000/5min) and causing 429s.
- No persisted `(repo, dateFolder) → file-list` artifact exists; training scripts re-query instead of using a static manifest.
- Data ingestion mixes schema (attribution/ts cols) into enriched/ outputs, violating surrogate-1 schema rules and complicating downstream casting.
- No CDN-only fetch path in training; authenticated API calls during data load remain the default.
- Mac/local runs risk executing model.from_pretrained() or heavy compute locally instead of delegating to Lightning/Kaggle/Cerebras.

## 2. Proposed change

Create a lightweight manifest generator + CDN-only training shim:

- File: `/opt/axentx/vanguard/scripts/build_manifest.py` (new)
- File: `/opt/axentx/vanguard/train/train.py` (modify data loader section)
- File: `/opt/axentx/vanguard/README.md` (add quick usage)

Scope:
- Single API call to `list_repo_tree(repo, dateFolder, recursive=False)` → write `manifests/{repo_slug}/{dateFolder}.json`
- Training script reads manifest and downloads via CDN URLs only (`resolve/main/...`) with `requests`/`wget` (no auth).
- Project only {prompt, response} at parse time; drop extra cols; store attribution in filename.

`/opt/axentx/vanguard/scripts/build_manifest.py`
```python
#!/usr/bin/env python3
"""
build_manifest.py
Usage:
  HF_TOKEN=hf_xxx python build_manifest.py \
    --repo bigcode/the-stack \
    --date-folder 2024-01 \
    --out-dir /opt/axentx/vanguard/manifests

Produces:
  manifests/bigcode_the-stack/2024-01.json
"""
import argparse
import json
import os
import sys
from pathlib import Path

try:
    from huggingface_hub import HfApi
except ImportError:
    print("ERROR: install huggingface_hub (pip install huggingface_hub)")
    sys.exit(1)

def slugify_repo(repo: str) -> str:
    return repo.replace("/", "_").replace(".", "_")

def main() -> None:
    parser = argparse.ArgumentParser(description="Build CDN-only file manifest for a repo/dateFolder")
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g. bigcode/the-stack)")
    parser.add_argument("--date-folder", required=True, help="Folder under repo (e.g. 2024-01)")
    parser.add_argument("--out-dir", default="manifests", help="Output directory for manifests")
    parser.add_argument("--recursive", action="store_true", help="Recursive listing (use sparingly)")
    args = parser.parse_args()

    api = HfApi(token=os.getenv("HF_TOKEN"))
    repo_slug = slugify_repo(args.repo)
    out_path = Path(args.out_dir) / repo_slug / f"{args.date_folder}.json"
    out_path.parent.mkdir(parents=True, exist_ok=True)

    # Single API call (non-recursive by default) to avoid pagination explosion
    try:
        tree = api.list_repo_tree(
            repo_id=args.repo,
            path=args.date_folder,
            repo_type="dataset",
            recursive=args.recursive,
        )
    except Exception as e:
        print(f"ERROR listing repo tree: {e}")
        sys.exit(1)

    # Keep only files (exclude directories) and build CDN URLs
    files = []
    for item in tree:
        if getattr(item, "type", None) == "file" or (hasattr(item, "path") and "." in Path(item.path).name):
            path = item.path if hasattr(item, "path") else item
            cdn_url = f"https://huggingface.co/datasets/{args.repo}/resolve/main/{path}"
            files.append({"path": path, "cdn_url": cdn_url})

    manifest = {
        "repo": args.repo,
        "date_folder": args.date_folder,
        "created_by": "build_manifest.py",
        "files": files,
        "total_files": len(files),
    }

    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(manifest, f, indent=2)

    print(f"Manifest written: {out_path} ({len(files)} files)")

if __name__ == "__main__":
    main()
```

`/opt/axentx/vanguard/train/train.py` (add/modify data loader section)
```python
import json
import os
from pathlib import Path
import requests
from datasets import Dataset, Features, Value

MANIFEST_PATH = os.getenv(
    "VANGUARD_MANIFEST",
    "manifests/bigcode_the-stack/2024-01.json"  # example default
)

def load_cdn_only(manifest_path: str, max_files: int = None):
    """
    Load dataset using CDN URLs only (no HF API auth during training).
    Projects to {prompt, response} at parse time.
    """
    with open(manifest_path, "r", encoding="utf-8") as f:
        manifest = json.load(f)

    samples = []
    files = manifest["files"]
    if max_files:
        files = files[:max_files]

    for entry in files:
        url = entry["cdn_url"]
        try:
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            content = resp.text
        except Exception as exc:
            print(f"WARNING: failed to fetch {url}: {exc}")
            continue

        # Lightweight projection: produce {prompt, response}
        # Replace with your domain-specific parsing (e.g. extract code blocks)
        prompt = f"Complete the following code from {entry['path']}"
        response = content.strip()

        # Keep only these fields to avoid mixed-schema issues
        samples.append({"prompt": prompt, "response": response})

    return Dataset.from_list(samples, features=Features({
        "prompt": Value("string"),
        "response": Value("string"),
    }))

# Example usage in training loop
if __name__ == "__main__":
    ds = load_cdn_only(MANIFEST_PATH, max_files=100)
    print(ds)
    # Continue with tokenizer + model training (run on Lightning Studio)
```
