## Final Actionable Plan — CDN-Only Dataset Loader (Surrogate-1)

**Core outcome**: Eliminate HF API 429s during training by replacing `load_dataset(streaming=True)` with a CDN-only `IterableDataset` that reads from a pre-generated `file_list.json`. This removes pagination, auth/rate-limit exposure, and GPU idle stalls. Ships in <2h as a drop-in replacement.

## 1) Implementation (single coherent codebase)

### `surrogate/data/cdn_loader.py`
```python
import json
import os
from pathlib import Path
from typing import Dict, Iterator

import pyarrow.parquet as pq
import requests
import torch
from torch.utils.data import IterableDataset

HF_DATASET_REPO = os.getenv("HF_DATASET_REPO", "axentx/surrogate-mirror")
HF_CDN_BASE = f"https://huggingface.co/datasets/{HF_DATASET_REPO}/resolve/main"

class CDNBackedDataset(IterableDataset):
    """
    Streams parquet shards directly from HF CDN (no API/auth/429s).
    file_list.json format: [{"path": "batches/mirror-merged/2026-04-29/slug1.parquet", "size": 12345}, ...]
    """

    def __init__(
        self,
        file_list_path: str,
        columns: tuple = ("prompt", "response"),
        max_retries: int = 5,
        timeout: int = 30,
    ):
        self.file_list_path = Path(file_list_path)
        self.columns = columns
        self.max_retries = max_retries
        self.timeout = timeout

        if not self.file_list_path.exists():
            raise FileNotFoundError(
                f"file_list.json not found: {self.file_list_path}. "
                "Run scripts/prefetch_filelist.py first."
            )

        with open(self.file_list_path) as f:
            self.files = json.load(f)

        if not self.files:
            raise ValueError("file_list.json is empty")

    def _fetch_parquet_to_rows(self, url: str) -> Iterator[Dict]:
        from io import BytesIO

        for attempt in range(self.max_retries):
            try:
                resp = requests.get(url, timeout=self.timeout)
                resp.raise_for_status()
                break
            except requests.RequestException as e:
                if attempt == self.max_retries - 1:
                    raise RuntimeError(f"Failed to fetch {url}: {e}") from e
                wait = 2 ** attempt
                print(f"Retry {attempt + 1}/{self.max_retries} for {url} in {wait}s: {e}")
                import time
                time.sleep(wait)
        else:
            return

        table = pq.read_table(BytesIO(resp.content))
        available = set(table.column_names)
        keep = [c for c in self.columns if c in available]
        if not keep:
            raise ValueError(f"None of {self.columns} found in {url}. Available: {available}")

        proj = table.select(keep)
        for batch in proj.to_batches():
            rb = pyarrow.RecordBatch.from_batches(proj.schema, [batch])
            for i in range(rb.num_rows):
                row = {col: rb.column(col)[i].as_py() for col in rb.column_names}
                if row.get("prompt") and row.get("response"):
                    yield row

    def __iter__(self) -> Iterator[Dict]:
        worker_info = torch.utils.data.get_worker_info()
        if worker_info is None:
            files = self.files
        else:
            per_worker = len(self.files) // worker_info.num_workers
            start = worker_info.id * per_worker
            end = start + per_worker if worker_info.id < worker_info.num_workers - 1 else len(self.files)
            files = self.files[start:end]

        for entry in files:
            url = f"{HF_CDN_BASE}/{entry['path']}"
            try:
                yield from self._fetch_parquet_to_rows(url)
            except Exception as e:
                print(f"Skipping {url} due to error: {e}")
                continue
```

### `surrogate/scripts/prefetch_filelist.py`
```bash
#!/usr/bin/env bash
# Usage: bash scripts/prefetch_filelist.py <date_folder>
# Example: bash scripts/prefetch_filelist.py 2026-04-29

set -euo pipefail

HF_DATASET_REPO="${HF_DATASET_REPO:-axentx/surrogate-mirror}"
DATE_FOLDER="${1:-$(date +%Y-%m-%d)}"
OUTPUT_DIR="data"
OUTPUT_FILE="${OUTPUT_DIR}/file_list.json"

mkdir -p "${OUTPUT_DIR}"

echo "Fetching file tree for ${HF_DATASET_REPO}/batches/mirror-merged/${DATE_FOLDER} ..."

python3 -c "
import os, json, sys
from huggingface_hub import HfApi
api = HfApi()
repo = os.getenv('HF_DATASET_REPO', 'axentx/surrogate-mirror')
path = f'batches/mirror-merged/${DATE_FOLDER}'
try:
    tree = api.list_repo_tree(repo=repo, path=path, recursive=False)
    files = [
        {'path': f'{path}/{entry.path.split(\"/\")[-1]}', 'size': getattr(entry, 'size', 0)}
        for entry in tree
        if entry.path.endswith('.parquet')
    ]
    print(json.dumps(files, indent=2))
except Exception as e:
    print(f'Error: {e}', file=sys.stderr)
    sys.exit(1)
" > "${OUTPUT_FILE}"

echo "Saved $(python3 -c "import json; print(len(json.load(open('${OUTPUT_FILE}'))))") entries to ${OUTPUT_FILE}"
```

## 3) Key Design Decisions (resolved)

- **CDN-only, no auth**: Uses public CDN URLs; avoids API 429s entirely.
- **Pre-listed manifest**: One `list_repo_tree` call (non-recursive) produces `file_list.json`. Training never paginates or hits API again.
- **Drop-in replacement**: `CDNBackedDataset` is an `IterableDataset` and works with existing `DataLoader`/Lightning loops.
- **Schema tolerance**: Projects only `prompt`/`response`; ignores mixed/extra fields. Skips rows with null prompt/response.
- **Resilience**: Per-shard CDN retries with exponential backoff; skips corrupt/unfetchable shards instead of crashing the run.
- **Multi-worker safe**: Splits file list across DataLoader workers for throughput.

This plan delivers reliable multi-hour training runs with minimal change and immediate impact.
