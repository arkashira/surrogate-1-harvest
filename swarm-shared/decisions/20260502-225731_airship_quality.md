# airship / quality

## Final Synthesized Implementation (Best of Both Candidates)

**Goal**: Ship a zero-runtime-dependency, CDN-safe `airship discover` orchestrator (Bash-first) that deterministically emits `status.json` + `manifest.json` for local/CI use and CDN upload. Frontends fetch via CDN URLs (no HF API during runtime).

**Why this wins**:
- Eliminates HF API rate-limit risk during training/inference (CDN bypass pattern).
- Enables deterministic builds for Arkship/Surrogate without Python runtime in CI.
- Unblocks quality (visibility into running services, artifact versions, health) without touching training/infra.
- Fits <2h: single Bash script + small JSON contract + optional cron.

---

## Implementation Plan

1. **Create `bin/airship-discover` CLI**
   - Bash script with shebang `#!/usr/bin/env bash`.
   - Make executable (`chmod +x bin/airship-discover`).
   - Idempotent: safe to run locally or in CI.
   - Accept optional `TARGET_DIR` (default: repo root).
   - Support `--upload-cdn` flag to push `manifest.json` to configured CDN path.

2. **Detect running services & health**
   - Check `docker-compose` services (microservices and per-service compose files).
   - Map expected ports: Arkship UI (3000), Arkship API (8000), Surrogate AI (8001).
   - Lightweight health probes (`curl -fs -o /dev/null -w "%{http_code}"` with timeout).

3. **Collect artifact inventory**
   - Read `arkship/README.md` and `surrogate/README.md` for declared versions/tags if present.
   - Inspect local git commit (`git rev-parse HEAD`, `git describe --tags --always`).
   - List last-built docker images (repo:tag) from `docker-compose` config if available.

4. **Emit deterministic outputs**
   - `status.json`: service health, ports, commit, generated timestamp.
   - `manifest.json`: build contract including compose files, docker images, optional dataset file list.
   - Ensure re-runs produce identical JSON for same repo state (sort keys, stable ordering).

5. **Optional: Embed dataset file list**
   - If `datasets/` exists or `HF_REPO` provided, use pre-generated `.hf_manifest_paths.json` (generated offline via `list_repo_tree` on Mac).
   - Embed path list into `manifest.json` under `datasets_file_list`.
   - Training scripts consume `manifest.json` and fetch via CDN URLs only.

6. **Optional CDN upload**
   - If `--upload-cdn` flag used and `CDN_TARGET_URL` env var set, upload `manifest.json` via `curl --upload-file`.
   - Frontends fetch `manifest.json` directly from CDN (bypasses HF API limits).

7. **Integrate into CI (optional, <10m)**
   - Add step in GitHub Actions / local CI to run `bin/airship-discover` and upload artifact.

8. **Update README**
   - Add “Discovery” section with usage examples.

## Code Snippets

### bin/airship-discover
```bash
#!/usr/bin/env bash
# airship-discover
# Deterministic discovery for Arkship + Surrogate.
# Emits status.json + manifest.json. Optionally uploads manifest to CDN.
#
# Usage:
#   bin/airship-discover [--upload-cdn] [target_dir]
#
# Environment:
#   CDN_TARGET_URL  - if set and --upload-cdn used, upload manifest here
#   HF_REPO         - optional, e.g. "datasets/your/repo" for file-list embedding

set -euo pipefail

REPO_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
TARGET_DIR="${2:-$REPO_ROOT}"
UPLOAD_CDN=false

if [[ "${1:-}" == "--upload-cdn" ]]; then
  UPLOAD_CDN=true
  if [[ -n "${2:-}" ]]; then
    TARGET_DIR="$2"
  fi
fi

cd "$TARGET_DIR"

# Helpers
_now() { date -u +"%Y-%m-%dT%H:%M:%SZ"; }
_sha() { git rev-parse HEAD 2>/dev/null || echo "no-git"; }
_tag() { git describe --tags --always 2>/dev/null || echo "no-tag"; }
_stable_json() { python3 -c "import sys,json; print(json.dumps(json.load(sys.stdin), sort_keys=True, separators=(',', ':')))" 2>/dev/null || cat; }
_health_check() {
  local url=$1
  local timeout=${2:-2}
  if curl -fs -o /dev/null -w "%{http_code}" --max-time "$timeout" "$url" | grep -q "^2"; then
    echo "healthy"
  else
    echo "unhealthy"
  fi
}

# Discover services
services=()
if [[ -f "arkship/docker-compose.yml" || -f "arkship/docker-compose.yaml" ]]; then
  services+=("arkship")
fi
if [[ -f "surrogate/docker-compose.yml" || -f "surrogate/docker-compose.yaml" ]]; then
  services+=("surrogate")
fi

# Ports (best-effort)
arkship_port=8000
surrogate_port=8001
ui_port=3000

# Health checks (best-effort, non-blocking)
arkship_health="unknown"
surrogate_health="unknown"
ui_health="unknown"

if [[ " ${services[*]} " =~ " arkship " ]]; then
  arkship_health=$(_health_check "http://localhost:$arkship_port/health" 2 || echo "unreachable")
  ui_health=$(_health_check "http://localhost:$ui_port" 2 || echo "unreachable")
fi
if [[ " ${services[*]} " =~ " surrogate " ]]; then
  surrogate_health=$(_health_check "http://localhost:$surrogate_port/health" 2 || echo "unreachable")
fi

# Collect artifact inventory
arkship_version=""
surrogate_version=""
if [[ -f "arkship/README.md" ]]; then
  arkship_version=$(grep -i -m1 -E 'version|tag' arkship/README.md | head -1 | sed 's/.*: *//' || true)
fi
if [[ -f "surrogate/README.md" ]]; then
  surrogate_version=$(grep -i -m1 -E 'version|tag' surrogate/README.md | head -1 | sed 's/.*: *//' || true)
fi

docker_images=$(grep -h -E 'image:.*' arkship/docker-compose*.y*ml surrogate/docker-compose*.y*ml 2>/dev/null | sed 's/.*image: *//' || true)

# Build status.json
status_json=$(cat <<EOF
{
  "generated_at": "$(_now)",
  "commit": "$(_sha)",
  "tag": "$(_tag)",
  "services": $(printf '%s\n' "${services[@]}" | jq -R . | jq -s .),
  "ports": {
    "arkship": $arkship_port,
    "surrogate": $surrogate_port,
    "ui": $ui_port
  },
  "health": {
    "arkship": "$arkship_health",
    "surrogate": "$surrogate_health",
    "ui": "$ui_health"
  },
  "versions": {
    "arkship": "$arkship_version",
    "surrogate": "$surrogate_version"
  },
  "docker_images": $(echo "$docker_images" | jq -R . | jq -s .)
}
EOF
)

# Build manifest.json
manifest_json=$(cat <<EOF
{
  "generated_at": "$(_now)",
  "repo": "$(basename "$REPO_ROOT")",
  "commit": "$(_sha)",
  "tag": "$(_tag)",
  "services": $(printf '%s\n' "${services[@]}" | jq -R . | jq -s .),
  "compose_files": $(find . -maxdepth 2 -name 'docker-compose*.y*ml' -printf '"%p"\n' | jq -s .),
  "docker_images": $(echo "$docker_images" | jq -R . | jq -s .),
  "datasets_file_list": []
}
EOF
)

# Optional: embed dataset file
