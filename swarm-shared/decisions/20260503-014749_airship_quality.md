## Final Synthesis — CDN-only ingestion + deterministic sharding + Studio lifecycle guard

**Core improvement (single highest-leverage change):**  
Eliminate HF API 429s during Surrogate training by pre-listing once per date folder, downloading via public CDN URLs (no auth), projecting only `{prompt,response,source_file}` at parse time, and sharding writes across 5 deterministic sibling repos to bypass per-repo commit limits. Add an explicit Studio lifecycle guard so Lightning Studio is reused or restarted cleanly.

---

## Implementation Plan (prioritized, concrete, <2h)

### 1. Deterministic repo sharding utility
- File: `surrogate/ingestion/shard.py`
- Deterministic mapping: `hash(slug) % 5` → pick sibling repo from config list.
- Expose `get_target_repo(slug)` and `get_cdn_url(repo, path)`.

### 2. Pre-list file paths (Mac orchestration)
- Script: `scripts/list_hf_date_folder.py`
- Single API call: `list_repo_tree(repo, path=date_folder, recursive=True)` to capture all parquet files.
- Save JSON: `file_list_{date}.json` next to training script.
- Include retry/backoff for 429 on the list call (wait 360s, then exponential backoff).

### 3. CDN-only loader with strict projection
- File: `surrogate/ingestion/cdn_loader.py`
- Accept pre-computed `file_list.json`.
- For each file:
  - Build CDN URL: `https://huggingface.co/datasets/{repo}/resolve/main/{path}`
  - Download with `requests.get(url, timeout=30)` (no auth header).
  - Parse parquet bytes with pyarrow; select only `prompt` and `response`.
  - Yield `{prompt: str, response: str, source_file: basename}`.
- Provide `build_dataset(file_list_path, repo)` returning a `datasets.Dataset`.

### 4. Training script wired for CDN-only mode
- Load `file_list_{date}.json`.
- Use `get_target_repo(slug)` to pick sibling repo.
- Build dataset via CDN loader; continue training with zero HF API calls during data loading.

### 5. Upload schema and path convention
- Keep only:
  - `prompt: str`
  - `response: str`
  - `source_file: str` (basename only)
- Filename pattern: `batches/mirror-merged/{date}/{slug}.parquet`
- No `source`, `ts`, or extra metadata columns.

### 6. Lightning Studio reuse + idle handling (lifecycle guard)
- In launcher:
  - List `Teamspace.studios`, reuse a running studio with matching name.
  - If stopped, restart with `target.start(machine=Machine.L40S)`.
  - If absent, create with `Machine.L40S`.
- Ensure launcher script has proper shebang and is executable.

### 7. Cron/wrapper hygiene
- Shebang: `#!/usr/bin/env bash`
- `chmod +x` on scripts.
- Crontab: set `SHELL=/bin/bash` and invoke scripts with `bash <script> "$@"`.

### surrogate/ingestion/shard.py
```python
import hashlib
from typing import List

SIBLING_REPOS = [
    "axentx/surrogate-mirror-0",
    "axentx/surrogate-mirror-1",
    "axentx/surrogate-mirror-2",
    "axentx/surrogate-mirror-3",
    "axentx/surrogate-mirror-4",
]

def _hash_slug(slug: str) -> int:
    return int(hashlib.sha256(slug.encode()).hexdigest(), 16)

def get_target_repo(slug: str) -> str:
    idx = _hash_slug(slug) % len(SIBLING_REPOS)
    return SIBLING_REPOS[idx]

def get_cdn_url(repo: str, path: str) -> str:
    return f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
```

### surrogate/ingestion/cdn_loader.py
```python
import json
import requests
import pyarrow.parquet as pq
import io
from pathlib import Path
from datasets import Dataset
from typing import Iterator, Dict

from .shard import get_cdn_url

def load_file_list(file_list_path: str) -> list:
    with open(file_list_path) as f:
        return json.load(f)

def project_record(raw: Dict) -> Dict:
    return {
        "prompt": str(raw.get("prompt", raw.get("input", ""))),
        "response": str(raw.get("response", raw.get("output", ""))),
        "source_file": str(raw.get("source_file", "unknown")),
    }

def cdn_generator(file_list: list) -> Iterator[Dict]:
    for item in file_list:
        repo = item.get("repo")
        path = item["path"]
        if not repo:
            raise ValueError("Each item in file_list must include 'repo'.")
        url = f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
        resp = requests.get(url, timeout=30)
        resp.raise_for_status()
        table = pq.read_table(io.BytesIO(resp.content))
        # Select only expected columns; tolerate missing ones by filling nulls
        available = [c for c in ["prompt", "response"] if c in table.column_names]
        if not available:
            continue
        df = table.select(available).to_pandas()
        # Normalize column names
        if "prompt" not in df.columns:
            df["prompt"] = ""
        if "response" not in df.columns:
            df["response"] = ""
        for _, row in df.iterrows():
            yield project_record({
                "prompt": row["prompt"],
                "response": row["response"],
                "source_file": Path(path).name,
            })

def build_dataset(file_list_path: str) -> Dataset:
    file_list = load_file_list(file_list_path)
    return Dataset.from_generator(
        cdn_generator,
        gen_kwargs={"file_list": file_list},
        num_proc=4,
    )
```

set -euo pipefail

REPO="axentx/surrogate-mirror-0"
DATE_FOLDER="${1:-$(date +%Y-%m-%d)}"
OUTFILE="file_list_${DATE_FOLDER}.json"

python3 - <<PY
import json
import time
import sys
from huggingface_hub import list_repo_tree, HfApi
from huggingface_hub.utils import HFValidationError, RepositoryNotFoundError

REPO = "$REPO"
DATE_FOLDER = "$DATE_FOLDER"
OUTFILE = "$OUTFILE"

def backoff(attempt: int):
    return min(360 * (2 ** attempt), 3600)

for attempt in range(5):
    try:
        tree = list_repo_tree(REPO, path=DATE_FOLDER, recursive=True)
        items = []
        for f in tree:
            if f.rfilename.endswith(".parquet"):
                # Determine which repo actually holds this path by deterministic shard
                # (simple fallback: use REPO; production may map per slug if needed)
                items.append({"path": f.rfilename, "repo": REPO})
        with open(OUTFILE, "w") as out:
            json.dump(items, out, indent=2)
        print(f"Wrote {len(items)} items to {OUTFILE}")
        sys.exit(0)
    except Exception as e:
        wait = backoff(attempt)
        print(f"Attempt {attempt+1} failed: {e}. Retrying in {wait}s...", file=sys.stderr)
        time.sleep(wait)

print("Exhausted retries.", file=sys.stderr)
sys.exit(1)
