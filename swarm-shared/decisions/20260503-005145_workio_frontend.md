# workio / frontend

Candidate 3:
## 1. Diagnosis

- **Race condition**: Clock-in endpoint allows concurrent requests for the same user to create multiple open punches (`clock_out_at IS NULL`) because there is no DB-level unique constraint (`user_id, clock_out_at IS NULL`) and no atomic upsert/locking.
- **No idempotency**: Replays or retries (network glitches, double-taps) create duplicates; no idempotency key on the server side.
- **Missing server-side validation**: No check for an existing open punch before insert; no transaction to guarantee exactly-one open punch.
- **Frontend blind retry**: No optimistic UI, no disabling during inflight requests, no client-side per-minute dedupe, and no clear feedback — users re-tap and compound duplicates.
- **No compensating rollback**: If a second punch is accidentally created, there is no safe cleanup path.

## 2. Proposed change

Files:
- `workio/src/components/ClockInButton.tsx` (create/modify)
- `workio/src/hooks/usePunch.ts` (create/modify)
- Backend: `workio/src/api/punches.ts` (or route handler) and DB migration for partial unique index.

Scope:
- Add DB partial unique index to enforce one open punch per user.
- Add server-side idempotency with upsert-on-key to guarantee exactly-one punch per user-minute.
- Add client-side per-minute idempotency key, optimistic state, and safe retry with rollback.

## 3. Implementation

### DB migration (PostgreSQL example)

```sql
-- Ensure exactly one open punch per user
CREATE UNIQUE INDEX CONCURRENTLY idx_users_one_open_punch
ON punches (user_id)
WHERE clock_out_at IS NULL;
```

### Backend: upsert with idempotency (pseudo-code for route handler)

```ts
// POST /api/punches
// Body: { userId, type, latitude?, longitude?, idempotencyKey }

export async function POST(req: Request) {
  const { userId, type, latitude, longitude, idempotencyKey } = await req.json();

  if (!userId || !type || !idempotencyKey) {
    return new Response('Missing fields', { status: 400 });
  }

  return await db.transaction(async (trx) => {
    // Idempotency: return existing if key matches
    const existing = await trx
      .select()
      .from(punches)
      .where(eq(punches.idempotencyKey, idempotencyKey))
      .limit(1);

    if (existing.length > 0) {
      return new Response(JSON.stringify(existing[0]), { status: 200 });
    }

    if (type === 'IN') {
      // Close any stale open punch defensively (should be prevented by unique index)
      await trx
        .update(punches)
        .set({ clock_out_at: new Date() })
        .where(and(eq(punches.user_id, userId), eq(punches.clock_out_at, null)));

      // Insert new punch
      const [punch] = await trx
        .insert(punches)
        .values({
          user_id: userId,
          clock_in_at: new Date(),
          clock_out_at: null,
          latitude,
          longitude,
          idempotency_key: idempotencyKey,
        })
        .returning();

      return new Response(JSON.stringify(punch), { status: 201 });
    } else {
      // Clock OUT: find open punch and close it
      const open = await trx
        .select()
        .from(punches)
        .where(and(eq(punches.user_id, userId), eq(punches.clock_out_at, null)))
        .limit(1);

      if (open.length === 0) {
        return new Response('No open punch to close', { status: 400);
      }

      const [updated] = await trx
        .update(punches)
        .set({ clock_out_at: new Date() })
        .where(eq(punches.id, open[0].id))
        .returning();

      return new Response(JSON.stringify(updated), { status: 200 });
    }
  });
}
```

### Frontend: `src/hooks/usePunch.ts`

```tsx
import { useState, useCallback } from 'react';
import { useToast } from '@/hooks/use-toast';

type PunchState = 'idle' | 'pending' | 'success' | 'error';

export function usePunch() {
  const [state, setState] = useState<PunchState>('idle');
  const [lastPunchId, setLastPunchId] = useState<string | null>(null);
  const { toast } = useToast();

  // per-minute idempotency key: userId + minute window
  const makeIdempotencyKey = (userId: string) => {
    const now = new Date();
    const minute = [
      now.getFullYear(),
      now.getMonth() + 1,
      now.getDate(),
      now.getHours(),
      Math.floor(now.getMinutes()),
    ].join('-');
    return `punch-${userId}-${minute}`;
  };

  const punch = useCallback(
    async ({
      userId,
      type,
      latitude,
      longitude,
    }: {
      userId: string;
      type: 'IN' | 'OUT';
      latitude?: number;
      longitude?: number;
    }) => {
      const key = makeIdempotencyKey(userId);
      if (state === 'pending') return;
      setState('pending');

      try {
        const res = await fetch('/api/punches', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ userId, type, latitude, longitude, idempotencyKey: key }),
        });

        if (!res.ok) {
          const text = await res.text();
          throw new Error(text || 'Punch failed');
        }

        const data = await res.json();
        setLastPunchId(data.id);
        setState('success');
        toast({
          title: type === 'IN' ? 'เข้างานแล้ว' : 'ออกงานแล้ว',
          description: new Date().toLocaleTimeString(),
        });
      } catch (err) {
        setState('error');
        toast({
          variant: 'destructive',
          title: 'ไม่สามารถบันทึกเวลาได้',
          description: err instanceof Error ? err.message : 'ลองใหม่อีกครั้ง',
        });
      } finally {
        // allow retry after brief delay
        setTimeout(() => setState('idle'), 1500);
      }
    },
    [state, toast]
  );

  return { punch, state, lastPunchId };
}
```
