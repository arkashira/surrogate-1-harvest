# Costinel / ops

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a non-blocking, CDN-first Top-Hub Signal Panel to Costinel frontend that surfaces the most-connected hub (e.g., "MOC") with **zero HuggingFace API calls at runtime**. Data is baked into the backend at build-time; frontend fetches via CDN URLs only.

### High-level steps (≤2h)
1. **Backend** (15 min): Add a build-time script that:
   - Runs once (or in CI) after `knowledge-rag`/`granite-business-research.sh` completes.
   - Queries the top-hub from the local graph (or uses cached result).
   - Produces a small JSON file: `public/signals/top-hub.json` (committed or served statically).
   - Also writes a dated snapshot to `batches/mirror-merged/{date}/top-hub-{slug}.json` for lineage.

2. **Frontend** (30 min): Add a lightweight Top-Hub Signal Panel component:
   - Fetches `/signals/top-hub.json` (CDN path, no auth).
   - Renders card with hub name, connection count, short insight, and last-updated timestamp.
   - Graceful fallback if fetch fails or JSON malformed.

3. **Routing/UI** (15 min): Mount panel on dashboard sidebar or top bar; ensure mobile responsive.

4. **Ops hardening** (10 min):
   - Ensure script is executable with Bash shebang.
   - Add to CI as a non-blocking step (or run manually before build).
   - Set `SHELL=/bin/bash` in any crontab entry.

5. **Buffer** (30 min): Polish, test, and verify CDN fetch works with no auth headers.

### 1. Build-time script: `scripts/build-top-hub-signal.sh`
```bash
#!/usr/bin/env bash
# scripts/build-top-hub-signal.sh
# Generates top-hub signal JSON for CDN-first delivery (zero HF API at runtime).
# Run after knowledge-rag or in CI.

set -euo pipefail
SHELL=/bin/bash

# Config
OUT_DIR="public/signals"
DATE=$(date -u +"%Y-%m-%d")
SLUG="top-hub"
OUT_FILE="${OUT_DIR}/${SLUG}.json"
BATCH_DIR="batches/mirror-merged/${DATE}"
BATCH_FILE="${BATCH_DIR}/${SLUG}.json"

# Ensure dirs
mkdir -p "${OUT_DIR}" "${BATCH_DIR}"

# Placeholder: replace with real query to local graph/RAG store.
# Example: use a small Python helper or CLI that reads the graph and emits top hub.
# For now, produce a deterministic stub that can be overridden by real pipeline.
TOP_HUB_NAME="MOC"
TOP_HUB_CONNECTIONS=42
TOP_HUB_INSIGHT="Most-connected hub for cost governance signals; central to anomaly routing."

cat > "${OUT_FILE}" <<EOF
{
  "hub": "${TOP_HUB_NAME}",
  "connections": ${TOP_HUB_CONNECTIONS},
  "insight": "${TOP_HUB_INSIGHT}",
  "generated_at": "$(date -u +"%Y-%m-%dT%H:%M:%SZ")",
  "source": "knowledge-rag",
  "cdn_path": "/signals/${SLUG}.json"
}
EOF

# Copy to dated batch (for lineage)
cp "${OUT_FILE}" "${BATCH_FILE}"

echo "✅ Top-hub signal written to ${OUT_FILE} and ${BATCH_FILE}"
```

Make executable:

### 2. Frontend component: `components/TopHubSignalPanel.tsx` (React/Next.js style)
```tsx
// components/TopHubSignalPanel.tsx
import { useEffect, useState } from "react";

interface TopHubSignal {
  hub: string;
  connections: number;
  insight: string;
  generated_at: string;
  source: string;
  cdn_path: string;
}

export default function TopHubSignalPanel() {
  const [signal, setSignal] = useState<TopHubSignal | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    // CDN-first fetch — no Authorization header (public file)
    fetch("/signals/top-hub.json", { cache: "no-store" })
      .then((res) => {
        if (!res.ok) throw new Error(`HTTP ${res.status}`);
        return res.json();
      })
      .then((data) => {
        setSignal(data);
        setLoading(false);
      })
      .catch((err) => {
        setError(err.message);
        setLoading(false);
      });
  }, []);

  if (loading) {
    return (
      <div className="p-3 border rounded bg-gray-50 text-sm text-gray-500">
        Loading top hub signal…
      </div>
    );
  }

  if (error || !signal) {
    return (
      <div className="p-3 border rounded bg-yellow-50 text-sm text-yellow-800">
        Top hub signal unavailable.
      </div>
    );
  }

  return (
    <div className="p-4 border rounded bg-white shadow-sm">
      <div className="flex items-start justify-between">
        <div>
          <h3 className="font-semibold text-gray-900">Top Hub</h3>
          <p className="text-2xl font-bold text-blue-600">{signal.hub}</p>
          <p className="text-sm text-gray-500 mt-1">
            {signal.connections} connections
          </p>
        </div>
        <span className="text-xs text-gray-400">
          {new Date(signal.generated_at).toLocaleDateString()}
        </span>
      </div>
      <p className="mt-2 text-sm text-gray-700">{signal.insight}</p>
      <div className="mt-3 text-xs text-gray-400">
        Source: {signal.source}
      </div>
    </div>
  );
}
```
