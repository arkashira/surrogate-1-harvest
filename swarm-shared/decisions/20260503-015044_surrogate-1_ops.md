## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Uses a **single `list_repo_tree` snapshot** (JSON manifest) generated once per date on the Mac orchestrator and committed to the repo (or passed via `MANIFEST_URL`).  
- Worker downloads **only its 1/16 shard** of files via **HF CDN direct URLs** (`https://huggingface.co/datasets/.../resolve/main/...`) — zero API calls during training/ingest, bypassing 429 rate limits.  
- Projects heterogeneous schemas to `{prompt, response}` at parse time (no `load_dataset(streaming=True)`), dedups via central `lib/dedup.py` md5 store, and writes `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.  
- Keeps GitHub Actions matrix (16 shards) unchanged; each runner gets isolated 7 GB RAM.

### 1) Create manifest generator (Mac orchestrator) — `bin/gen_manifest.py`

```python
#!/usr/bin/env python3
"""
Generate a date-scoped manifest for axentx/surrogate-1-training-pairs.
Run on Mac (or cron) once per date after rate-limit window clears.

Outputs: manifests/YYYY-MM-DD.json  (list of {"path": str, "size": int})
"""
import json, os, sys, datetime
from huggingface_hub import HfApi

REPO = "datasets/axentx/surrogate-1-training-pairs"
OUT_DIR = "manifests"

def main():
    api = HfApi()
    date_str = datetime.date.today().isoformat()
    out_path = os.path.join(OUT_DIR, f"{date_str}.json")
    os.makedirs(OUT_DIR, exist_ok=True)

    # Single API call: non-recursive per top-level folder (avoid list_repo_files)
    items = api.list_repo_tree(repo_id=REPO, path="", recursive=True)
    files = [{"path": i.path, "size": i.size or 0} for i in items if i.type == "file"]

    with open(out_path, "w") as f:
        json.dump({"date": date_str, "repo": REPO, "files": files}, f, indent=2)

    print(f"Wrote {len(files)} files -> {out_path}")

if __name__ == "__main__":
    main()
```

- Make executable: `chmod +x bin/gen_manifest.py`  
- Commit generated `manifests/YYYY-MM-DD.json` to repo (or serve via CDN).

def hash_shard(path: str, total: int) -> int:
    return int(hashlib.md5(path.encode()).hexdigest(), 16) % total

def parse_file(path: str, local_path: Path):
    """Project heterogeneous file to (prompt, response). Returns list of dicts."""
    rows = []
    try:
        if path.endswith(".jsonl"):
            with open(local_path, "r", encoding="utf-8") as f:
                for line in f:
                    line = line.strip()
                    if not line:
                        continue
                    obj = json.loads(line)
                    rows.append(normalize_obj(obj))
        elif path.endswith(".parquet"):
            tbl = pq.read_table(local_path)
            df = tbl.to_pandas()
            for _, rec in df.iterrows():
                rows.append(normalize_obj(rec.to_dict()))
        elif path.endswith(".json"):
            with open(local_path, "r", encoding="utf-8") as f:
                obj = json.load(f)
                if isinstance(obj, list):
                    for item in obj:
                        rows.append(normalize_obj(item))
                else:
                    rows.append(normalize_obj(obj))
        else:
            # Unknown: skip
            return []
    except Exception as exc:
        print(f"Parse error {path}: {exc}", file=sys.stderr)
        return []
    return rows

def normalize_obj(obj):
    """Map heterogeneous schema -> {prompt, response}."""
    if not isinstance(obj, dict):
        return {"prompt": "", "response": str(obj)}

    # Common patterns
    prompt = obj.get("prompt") or obj.get("input") or obj.get("question") or obj.get("text") or ""
    response = obj.get("response") or obj.get("output") or obj.get("answer") or obj.get("completion") or ""

    # Fallback: pick any other field as response if prompt present
    if prompt and not response:
        for k, v in obj.items():
            if k not in {"prompt", "input", "question", "text", "source", "ts", "id"}:
                response = str(v)
                break

    return {"prompt": str(prompt), "response": str(response)}

def download_cdn(repo: str, path: str, dest: Path):
    url = f"{HF_CDN}/{repo}/resolve/main/{path}"
    resp = requests.get(url, timeout=60)
    resp.raise_for_status()
    dest.write_bytes(resp.content)
