# surrogate-1 / design

## Final synthesized implementation (best of both proposals)

### Highest-value improvement (≤2h)
**Deterministic pre-flight snapshot + CDN-only ingestion**  
- One `list_repo_tree` snapshot per date folder (Mac orchestrator) → JSON file list.  
- Training and shard workers use **CDN-only fetches** (`resolve/main/...`) with zero API calls during data load.  
- Shard workers accept the snapshot as input so they never call `list_repo_files`/`list_repo_tree`/`load_dataset(streaming=True)` at runtime.  
- Keep existing shard-hash routing (`slug-hash % TOTAL_SHARDS == SHARD_ID`) and dedup behavior unchanged.  
- Snapshot is committed as an artifact and/or checked into `snapshots/<YYYY-MM-DD>/file-list.json` so workflows can run without any HF API calls.

### Concrete implementation plan

#### 1) Mac orchestrator: create snapshot script
- Path: `bin/list-snapshot.py`  
- Behavior:
  - Accept `REPO` and `DATE_PATH` (e.g. `public-raw/2026-05-02`).  
  - Call `list_repo_tree(path=DATE_PATH, recursive=False)` once, then one non-recursive call per subfolder to avoid huge pagination.  
  - Emit `snapshots/<DATE_PATH>/file-list.json` with `{"repo": "...", "date": "...", "created_at": "...", "files": [{"path": "...", "size": int, "sha256": "..."}, ...]}`.  
  - Commit snapshot into repo (or upload as workflow artifact) so shard workers can run with zero HF API calls.  
- Rate-limit handling:
  - If 429, sleep 360s and retry (max 3 tries).  
- Optional: compute SHA-256 for each file (best-effort; skip on transient errors) to enable stronger dedup and change detection.

```python
#!/usr/bin/env python3
"""
bin/list-snapshot.py
Create a deterministic file listing (with metadata) for a date folder in a HF dataset repo.

Usage:
  HF_TOKEN=... python bin/list-snapshot.py axentx/surrogate-1-training-pairs public-raw/2026-05-02
"""
import hashlib
import json
import os
import sys
import time
from datetime import datetime, timezone

from huggingface_hub import HfApi, RepositoryNotFoundError, hf_raise_for_status

MAX_RETRIES = 3
RETRY_WAIT = 360  # seconds after 429

def list_tree_safe(api, repo_id, path, max_retries=MAX_RETRIES):
    for attempt in range(1, max_retries + 1):
        try:
            tree = api.list_repo_tree(repo_id=repo_id, path=path, recursive=False)
            entries = list(tree)
            files = [e for e in entries if e.type != "directory"]
            subdirs = [e.path for e in entries if e.type == "directory"]
            return files, subdirs
        except Exception as exc:
            is_429 = (
                hasattr(exc, "response")
                and getattr(exc.response, "status_code", None) == 429
            )
            if is_429 and attempt < max_retries:
                print(
                    f"429 rate-limited. Waiting {RETRY_WAIT}s (attempt {attempt}/{max_retries})",
                    file=sys.stderr,
                )
                time.sleep(RETRY_WAIT)
                continue
            raise

def sha256_bytes(data: bytes) -> str:
    return hashlib.sha256(data).hexdigest()

def fetch_file_metadata(api, repo_id, filepath, max_retries=MAX_RETRIES):
    """
    Fetch file via CDN and return (size, sha256). On transient failure, size=0, sha256=''.
    """
    url = f"https://huggingface.co/datasets/{repo_id}/resolve/main/{filepath}"
    for attempt in range(1, max_retries + 1):
        try:
            import urllib.request
            req = urllib.request.Request(url, headers={"User-Agent": "snapshot-script"})
            with urllib.request.urlopen(req, timeout=60) as resp:
                data = resp.read()
                return len(data), sha256_bytes(data)
        except Exception as exc:
            # If transient, retry; otherwise record partial metadata.
            if attempt < max_retries:
                wait = 5 * attempt
                print(f"Retry {filepath} in {wait}s ({exc})", file=sys.stderr)
                time.sleep(wait)
                continue
            print(f"Warning: failed to fetch {filepath}: {exc}", file=sys.stderr)
            return 0, ""

def main():
    if len(sys.argv) != 3:
        print("Usage: list-snapshot.py <repo_id> <date_path>", file=sys.stderr)
        sys.exit(1)

    repo_id = sys.argv[1]
    date_path = sys.argv[2].strip("/")
    api = HfApi(token=os.environ.get("HF_TOKEN"))

    all_files = []
    try:
        top_files, subdirs = list_tree_safe(api, repo_id, date_path)
        all_files.extend(top_files)
        for sub in subdirs:
            try:
                files, _ = list_tree_safe(api, repo_id, sub)
                all_files.extend(files)
            except Exception as exc:
                print(f"Warning: failed to list {sub}: {exc}", file=sys.stderr)
    except RepositoryNotFoundError:
        print(f"Repository not found: {repo_id}", file=sys.stderr)
        sys.exit(1)
    except Exception as exc:
        print(f"Failed to list {repo_id}/{date_path}: {exc}", file=sys.stderr)
        sys.exit(1)

    # Build deterministic file entries
    entries = []
    for f in sorted(set(all_files)):
        size, sha256 = fetch_file_metadata(api, repo_id, f)
        entries.append({"path": f, "size": size, "sha256": sha256})

    snapshot = {
        "repo": repo_id,
        "date": date_path,
        "created_at": datetime.now(timezone.utc).isoformat(),
        "files": entries,
    }

    out_dir = os.path.join("snapshots", date_path)
    os.makedirs(out_dir, exist_ok=True)
    out_name = os.path.join(out_dir, "file-list.json")
    with open(out_name, "w", encoding="utf-8") as f:
        json.dump(snapshot, f, indent=2)
    print(out_name)

if __name__ == "__main__":
    main()
```

#### 2) Shard worker: update `bin/dataset-enrich.sh`
- Accept snapshot file path as optional argument.  
- If snapshot provided, read file list deterministically and assign shards by `hash(path) % TOTAL_SHARDS == SHARD_ID` (preserves existing routing).  
- For each assigned file, download via CDN URL:  
  `https://huggingface.co/datasets/{REPO}/resolve/main/{FILE_PATH}`  
- Keep existing schema normalization and dedup logic unchanged.  
- Output path remains: `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`
