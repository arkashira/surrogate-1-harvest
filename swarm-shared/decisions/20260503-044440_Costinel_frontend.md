# Costinel / frontend

### Final Implementation — Resilient “Top Hub” Signal Panel (CDN-First, Zero Model Compute)

**Goal:** Surface the most-connected hub and related documents in Costinel with zero backend changes, zero model inference, no rate-limit exposure, and sub-2-hour delivery.

---

### Why this is the highest-value incremental improvement
- **Zero model compute** — purely static data served via CDN.
- **CDN-first + offline fallback** — eliminates Hugging Face API auth/rate limits and works offline.
- **Pure frontend** — no backend changes, pipelines, or infra required.
- **Fast to ship** — ~90 minutes (UI + fetch/cache + polish/tests).
- **Actionable insight** — immediately directs users to the most-connected hub and canonical docs.

---

### Concrete implementation plan (merged best practices)

1. **Export and commit `top-hub.json`**  
   - Generated by the existing `knowledge-rag` pipeline (graph export).  
   - Placed in `public/knowledge/top-hub.json` so it’s bundled and available at `/knowledge/top-hub.json`.  
   - Also uploaded to CDN at Hugging Face `resolve/main/top-hub.json` for freshness.

2. **Add two frontend modules**  
   - `src/hooks/useTopHub.ts` — resilient CDN-first fetcher with local fallback, abort, and refresh.  
   - `src/components/TopHubPanel.tsx` — compact card UI showing hub summary and related docs.

3. **Integrate into dashboard**  
   - Mount `TopHubPanel` in `src/pages/Dashboard.tsx` near the top of the main content area.

4. **Cache and resilience strategy**  
   - CDN fetch with `cache: "no-store"` to avoid stale CDN edge cache during active development.  
   - Local bundled JSON as immediate fallback (works offline / on restricted networks).  
   - Short in-memory TTL (optional) or manual refresh button to avoid hammering CDN in production.

5. **Error handling and logging**  
   - Surface non-blocking errors in UI (inline message).  
   - Console warnings for failed CDN + local fallback to aid debugging.  
   - Abort on unmount to prevent state updates after navigation.

### Hook: `src/hooks/useTopHub.ts`

```ts
// src/hooks/useTopHub.ts
import { useEffect, useState, useCallback } from "react";

const CDN_BASE = "https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main";
const TOP_HUB_PATH = "top-hub.json";
const CDN_URL = `${CDN_BASE}/${TOP_HUB_PATH}`;
const LOCAL_FALLBACK = "/knowledge/top-hub.json";

export interface TopHubItem {
  id: string;
  title: string;
  summary: string;
  tags: string[];
  url?: string;
}

export interface TopHubData {
  generatedAt: string;
  hub: {
    id: string;
    title: string;
    summary: string;
    tags: string[];
    priorityScore: number;
  };
  related: TopHubItem[];
}

export function useTopHub(enabled = true) {
  const [data, setData] = useState<TopHubData | null>(null);
  const [loading, setLoading] = useState(enabled);
  const [error, setError] = useState<Error | null>(null);

  const fetchTopHub = useCallback(async (signal: AbortSignal) => {
    try {
      // 1) Try CDN (no auth, high rate limit)
      const cdnRes = await fetch(CDN_URL, { signal, cache: "no-store" });
      if (cdnRes.ok) {
        const json = (await cdnRes.json()) as TopHubData;
        setData(json);
        setError(null);
        return;
      }

      // 2) Fallback to local bundled copy
      const localRes = await fetch(LOCAL_FALLBACK, { signal });
      if (localRes.ok) {
        const json = (await localRes.json()) as TopHubData;
        setData(json);
        setError(null);
        return;
      }

      throw new Error("Both CDN and local top-hub unavailable");
    } catch (err: any) {
      if (err.name !== "AbortError") {
        console.warn("[useTopHub] fetch failed:", err);
        setError(err);
      }
    } finally {
      setLoading(false);
    }
  }, []);

  useEffect(() => {
    if (!enabled) {
      setLoading(false);
      return;
    }

    const controller = new AbortController();
    setLoading(true);
    fetchTopHub(controller.signal);

    return () => controller.abort();
  }, [enabled, fetchTopHub]);

  const refresh = useCallback(() => {
    const controller = new AbortController();
    setLoading(true);
    setError(null);
    return fetchTopHub(controller.signal);
  }, [fetchTopHub]);

  return { data, loading, error, refresh };
}
```
