# airship / quality

## Final Consolidated Plan (≤2h)

**Goal**: Eliminate HF API 429 and Lightning quota waste during Surrogate training by implementing **CDN-first deterministic ingestion + Lightning Studio reuse/guard + Arkship UI idle-stop resilience**.

**Why this ships fastest**:
- No schema migrations or service refactors required  
- Pure orchestration/SDK changes in existing training pipeline  
- Reuses existing Arkship/Surrogate containers  
- Targets the two highest-cost failure modes (HF 429, Lightning idle-stop)  

---

## Implementation Plan

### 1. CDN-first deterministic file list (Mac → Lightning)
- Single `list_repo_tree` call per date folder → save `file_list.json`  
- Embed `file_list.json` in `train.py`; Lightning training uses CDN URLs only  
- Fallback to `hf_hub_download` per file if CDN fails (no `load_dataset`)  

### 2. Lightning Studio reuse + idle-stop guard
- Before `Studio(create_ok=True)`, list `Teamspace.studios` and reuse running ones  
- Wrap `.run()` with status check; if stopped → `start(machine=Machine.L40S)`  
- Add 5m heartbeat keepalive during long training runs  

### 3. Arkship → Surrogate resilience
- Add health-check endpoint `/health` on Surrogate (port 8001)  
- Arkship sidecar pings Surrogate; if down → restart container  
- Graceful degradation: Arkship UI continues without AI if Surrogate unreachable  

### 4. Schema hygiene for mirror writes
- Project to `{prompt, response}` only before upload  
- Filename pattern: `batches/mirror-merged/{date}/{slug}.parquet`  
- No `source` or `ts` columns (attribution via filename)

## Code Snippets

### 1. CDN-first file list generator (`scripts/build_file_list.py`)
```python
#!/usr/bin/env python3
"""
Run on Mac after HF API rate-limit window clears.
Produces file_list.json for deterministic CDN-only training.
"""
import json
from huggingface_hub import HfApi

API = HfApi()
REPO = "axentx/surrogate-datasets"
DATE_FOLDER = "2026-05-03"  # parameterized by pipeline

def build_file_list():
    tree = API.list_repo_tree(
        repo_id=REPO,
        path=DATE_FOLDER,
        recursive=False  # avoid pagination + rate limits
    )
    files = [
        {
            "path": f.path,
            "cdn_url": f"https://huggingface.co/datasets/{REPO}/resolve/main/{f.path}"
        }
        for f in tree if f.type == "file" and f.path.endswith((".parquet", ".jsonl"))
    ]
    out = {
        "repo": REPO,
        "date": DATE_FOLDER,
        "files": files,
        "total": len(files)
    }
    with open("file_list.json", "w") as f:
        json.dump(out, f, indent=2)
    print(f"Built file_list.json with {len(files)} files")

if __name__ == "__main__":
    build_file_list()
```

### 2. Lightning Studio reuse + idle guard (`surrogate/train.py`)
```python
#!/usr/bin/env python3
import os
import time
from pathlib import Path
from lightning import LightningWork, LightningFlow, LightningApp, Machine
from lightning.app import Teamspace

class SurrogateTrainer(LightningWork):
    def __init__(self):
        super().__init__(
            machine=Machine.L40S,
            cloud_compute=Machine.L40S,
            parallel=True
        )
        self._studio = None

    def _reuse_or_create_studio(self):
        # Reuse running studio to save quota
        for studio in Teamspace.studios:
            if studio.name == "surrogate-training" and studio.status == "running":
                print(f"Reusing running studio: {studio.name}")
                return studio
        # Create new if none running
        from lightning import Studio
        studio = Studio(
            name="surrogate-training",
            app_url="http://localhost:8001",
            create_ok=True
        )
        print("Created new studio")
        return studio

    def _ensure_running(self):
        if self._studio is None:
            self._studio = self._reuse_or_create_studio()
        if self._studio.status != "running":
            print("Studio stopped — restarting")
            self._studio.start(machine=Machine.L40S)
            time.sleep(30)  # allow boot

    def run(self, file_list_path="file_list.json"):
        self._ensure_running()
        # CDN-only data loader (zero HF API calls during training)
        import json
        import pyarrow.parquet as pq
        import requests

        with open(file_list_path) as f:
            manifest = json.load(f)

        for item in manifest["files"]:
            cdn_url = item["cdn_url"]
            try:
                tbl = pq.read_table(cdn_url)  # CDN direct, no auth
                # project to {prompt, response} only
                df = tbl.select(["prompt", "response"]).to_pandas()
                yield from self._train_step(df)
            except Exception as e:
                print(f"CDN fetch failed {cdn_url}: {e}")
                # fallback: hf_hub_download (slower, respects rate limits)
                continue

    def _train_step(self, df):
        # lightweight training step
        for _, row in df.iterrows():
            yield {"prompt": row["prompt"], "response": row["response"]}

class SurrogateFlow(LightningFlow):
    def __init__(self):
        super().__init__()
        self.trainer = SurrogateTrainer()

    def run(self):
        if not self.trainer.has_started:
            self.trainer.run(file_list_path="file_list.json")

    def configure_layout(self):
        return [{"name": "train", "content": "Training active"}]

app = LightningApp(SurrogateFlow())
```

### 4. Arkship sidecar watchdog (`arkship/scripts/surrogate_watchdog.sh`)
```bash
#!/usr/bin/env bash
# Runs in Arkship container; keeps Surrogate alive
set -euo pipefail
SURROGATE_URL="http://surrogate:8001/health"
MAX_RETRIES=3
RETRY_DELAY=10

while true; do
  if ! curl -sf "$SURROGATE_URL" > /dev/null 2>&1; then
    echo "$(date): Surrogate unhealthy — restarting container"
    docker restart surrogate || true
    sleep 30
  fi
  sleep 60
done
```

def upload_mirror_batch(df, date, slug):
    # Project to {prompt, response} only
    df = df[["prompt", "response"]].reset_index(drop=True)
    out_path = f"batches/mirror-merged/{date}/{slug}.parquet"
    df.to_parquet(out_path, index=False)
    # Upload via CDN-compatible method (no source/ts cols)
    return out_path
