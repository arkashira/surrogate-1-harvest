# Costinel / design

## Implementation Plan — Costinel Top-Hub Signal (Backend)

**Scope:** Highest-value, read-only, <2h  
**Endpoint:** `GET /api/v1/cost-anomaly/signal/top-hub`  
**Guiding principle:** “Sense + Signal — ไม่ Execute” (no side effects, no mutations).

---

### 1. Architecture & Data Flow

```
Client
  │
  ▼
FastAPI (Costinel Backend)
  │
  ├─> KnowledgeRAG (read-only query)
  │      └─> Graph: find most-connected hub (e.g. "MOC")
  │
  ├─> CostAnomalyDetector (read-only)
  │      └─> latest anomalies + severity
  │
  └─> SignalComposer
         └─> { hub, insight, anomalies, actions[], ts }
```

- **No writes**, no jobs, no queues.
- Uses existing RAG graph + anomaly store.
- Returns actionable signal for human review.

### 3. Implementation Steps (Backend)

1. Add route in `app/api/v1/cost_anomaly.py`
2. Implement `SignalComposer` service (read-only).
3. Wire existing `KnowledgeRAG.top_hub()` and `CostAnomalyDetector.latest()`.
4. Add minimal OpenAPI docs.
5. No DB migrations, no background jobs.

```python
from fastapi import APIRouter, HTTPException
from app.services.signal_composer import SignalComposer
from app.schemas.signal import TopHubSignalResponse

router = APIRouter()

@router.get(
    "/cost-anomaly/signal/top-hub",
    response_model=TopHubSignalResponse,
    summary="Top-hub cost anomaly signal (read-only)",
    tags=["cost-anomaly", "signal"],
)
async def get_top_hub_signal() -> TopHubSignalResponse:
    try:
        signal = await SignalComposer.compose_top_hub_signal()
        return signal
    except ValueError as exc:
        raise HTTPException(status_code=404, detail=str(exc))
    except RuntimeError as exc:
        raise HTTPException(status_code=503, detail=str(exc))
```

#### `app/services/signal_composer.py`

```python
from datetime import datetime, timezone
from typing import List
from app.services.knowledge_rag import KnowledgeRAG
from app.services.cost_anomaly import CostAnomalyDetector
from app.schemas.signal import TopHubSignalResponse, Anomaly, Recommendation, Action

class SignalComposer:
    @staticmethod
    async def compose_top_hub_signal() -> TopHubSignalResponse:
        # 1) Top hub from RAG (read-only graph query)
        hub_node = await KnowledgeRAG.top_hub()
        if not hub_node:
            raise ValueError("No top hub found in knowledge graph")

        hub = hub_node.get("name", "Unknown")
        insight = hub_node.get("insight", "Top hub identified")

        # 2) Latest anomalies (read-only)
        raw_anomalies = await CostAnomalyDetector.latest(limit=5)
        anomalies: List[Anomaly] = [
            Anomaly(
                id=a["id"],
                service=a["service"],
                region=a["region"],
                amount=a["amount"],
                currency=a.get("currency", "USD"),
                deltaPct=a["delta_pct"],
                window=a["window"],
            )
            for a in raw_anomalies
        ]

        # 3) Recommendations (read-only heuristics)
        recommendations: List[Recommendation] = SignalComposer._build_recommendations(anomalies)

        # 4) Actions (read-only navigation)
        actions: List[Action] = [
            Action(label="Review RI coverage", type="navigate", target="/ri-coverage"),
            Action(label="View anomalies", type="navigate", target="/anomalies"),
        ]

        return TopHubSignalResponse(
            hub=hub,
            insight=insight,
            severity=SignalComposer._severity_from(anomalies),
            anomalies=anomalies,
            recommendations=recommendations,
            actions=actions,
            ts=datetime.now(timezone.utc).isoformat(),
        )

    @staticmethod
    def _severity_from(anomalies: List[Anomaly]) -> str:
        if not anomalies:
            return "low"
        max_delta = max((a.deltaPct for a in anomalies), default=0)
        if max_delta >= 50:
            return "critical"
        if max_delta >= 25:
            return "high"
        if max_delta >= 10:
            return "medium"
        return "low"

    @staticmethod
    def _build_recommendations(anomalies: List[Anomaly]) -> List[Recommendation]:
        # Simple read-only heuristic; replace with richer logic later
        recs = []
        ec2_anomalies = [a for a in anomalies if a.service == "AmazonEC2"]
        if ec2_anomalies:
            recs.append(
                Recommendation(
                    id="rec-ri-ec2-001",
                    type="reserved_instance",
                    title="RI coverage below 60% for EC2 instances",
                    impactUsd=3200,
                    confidence=0.85,
                )
            )
        return recs
```

#### `app/schemas/signal.py`

```python
from pydantic import BaseModel
from datetime import datetime
from typing import List, Optional

class Anomaly(BaseModel):
    id: str
    service: str
    region: str
    amount: float
    currency: str = "USD"
    deltaPct: float
    window: str

class Recommendation(BaseModel):
    id: str
    type: str
    title: str
    impactUsd: float
    confidence: float

class Action(BaseModel):
    label: string
    type: str  # navigate | callback
    target: str

class TopHubSignalResponse(BaseModel):
    hub: str
    insight: str
    severity: str
    anomalies: List[Anomaly]
    recommendations: List[Recommendation]
    actions: List[Action]
    ts: str
```
