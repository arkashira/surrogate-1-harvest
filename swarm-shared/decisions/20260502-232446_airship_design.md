# airship / design

## Highest-Value Incremental Improvement (<2h)

**Goal**: Harden `airship discover` into a deterministic, CDN-only orchestrator that eliminates HF API rate limits and PyArrow schema errors while producing reproducible file lists.

**Why this now**:
- Directly applies the **HF CDN Bypass** insight (resolve/main URLs avoid 429s)
- Fixes **PyArrow CastError** by projecting `{prompt, response}` only at parse time
- Enables reproducible training runs via embedded file lists
- Fits in <2h: single orchestrator script + minimal config changes

---

## Implementation Plan

### 1. Create `/opt/axentx/airship/scripts/discover_cdn_orchestrator.sh`
- Bash shebang, `set -euo pipefail`
- Single API call: `list_repo_tree(path, recursive=False)` for target date folder
- Persist file list to JSON (deterministic, sorted)
- Generate `train_manifest.json` with CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`)
- Exit codes: 0=success, 1=API error, 2=empty folder, 3=schema mismatch

### 2. Update `/opt/axentx/airship/surrogate/train.py`
- Accept `--manifest train_manifest.json` (default: `train_manifest.json`)
- Data loader uses **only CDN URLs** (no `load_dataset`, no streaming)
- Per-file download via `requests.get(url, timeout=30)` → `pyarrow.parquet.read_table()`
- Project to `{prompt, response}` only at parse time; drop all other columns
- Attribution via filename pattern: `batches/mirror-merged/{date}/{slug}.parquet`

### 3. Add `/opt/axentx/airship/.env.discovery`
```bash
# HF repo to discover (date folder)
HF_REPO=datasets/axentx/surrogate-mirror
HF_DATE_FOLDER=2026-05-02
OUTPUT_DIR=/opt/axentx/airship/data/manifests
```

### 4. Update `docker-compose.microservices.yml` (surrogate service)
- Mount `./data/manifests:/app/data/manifests`
- Add environment: `DISCOVERY_MANIFEST=/app/data/manifests/train_manifest.json`

### 5. Cron entry (or systemd timer) for daily discovery
- Runs at 02:00 UTC (after typical HF commit window)
- Uses `SHELL=/bin/bash`
- Invokes orchestrator → logs to `/var/log/airship/discover.log`

## Code Snippets

### `scripts/discover_cdn_orchestrator.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

# Config
HF_REPO="${HF_REPO:-datasets/axentx/surrogate-mirror}"
HF_DATE="${HF_DATE_FOLDER:-$(date -u +%Y-%m-%d)}"
OUTPUT_DIR="${OUTPUT_DIR:-/opt/axentx/airship/data/manifests}"
MANIFEST_NAME="train_manifest_${HF_DATE}.json"
MANIFEST_PATH="${OUTPUT_DIR}/${MANIFEST_NAME}"

mkdir -p "${OUTPUT_DIR}"

echo "[$(date -u)] Discovering ${HF_REPO} @ ${HF_DATE}"

# Single API call: list top-level of date folder (non-recursive)
# Uses huggingface_hub via Python for reliability
python3 - "$HF_REPO" "$HF_DATE" "$MANIFEST_PATH" <<'PYEOF'
import json, os, sys
from huggingface_hub import list_repo_tree

repo = sys.argv[1]          # e.g. datasets/axentx/surrogate-mirror
date = sys.argv[2]          # e.g. 2026-05-02
out_path = sys.argv[3]

# List only immediate children of date folder
tree = list_repo_tree(repo=repo, path=date, recursive=False)

files = []
for entry in tree:
    if entry.type == "file" and entry.path.endswith(".parquet"):
        cdn_url = f"https://huggingface.co/{repo}/resolve/main/{entry.path}"
        files.append({
            "path": entry.path,
            "cdn_url": cdn_url,
            "size": getattr(entry, "size", None)
        })

# Deterministic ordering
files.sort(key=lambda x: x["path"])

manifest = {
    "repo": repo,
    "date": date,
    "generated_at_utc": __import__("datetime").datetime.utcnow().isoformat() + "Z",
    "files": files,
    "total_files": len(files)
}

os.makedirs(os.path.dirname(out_path), exist_ok=True)
with open(out_path, "w") as f:
    json.dump(manifest, f, indent=2)

print(f"Wrote {len(files)} files to {out_path}")
PYEOF

echo "[$(date -u)] Manifest written: ${MANIFEST_PATH}"
echo "[$(date -u)] Done."
```

### `surrogate/train.py` (excerpt)
```python
import argparse, json, requests, pyarrow.parquet as pq
from io import BytesIO
from pathlib import Path

def load_from_manifest(manifest_path: str):
    with open(manifest_path) as f:
        manifest = json.load(f)

    for file_info in manifest["files"]:
        url = file_info["cdn_url"]
        try:
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            table = pq.read_table(BytesIO(resp.content))
            # Project to {prompt, response} only — drop all other columns
            cols = [c for c in table.column_names if c in ("prompt", "response")]
            if len(cols) < 2:
                continue  # skip malformed
            batch = table.select(cols).to_pydict()
            for prompt, response in zip(batch["prompt"], batch["response"]):
                yield {"prompt": prompt, "response": response}
        except Exception as e:
            print(f"Skipping {url}: {e}")
            continue

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--manifest", default="data/manifests/train_manifest_latest.json")
    args = parser.parse_args()

    for sample in load_from_manifest(args.manifest):
        # Training step here
        pass
```
