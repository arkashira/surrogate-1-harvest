## v0.1 scope
v0.1 in 2 weeks: single deterministic workflow type (research->signal->risk->execution) with Postgres-backed event sourcing, Redis Streams queue, compliance guardrails as JSON rules, immutable audit trail, and a mock broker connector; deployed on Cloudflare Workers + Supabase + Upstash Redis; includes replay endpoint to re-run any run_id deterministically and NSM tracking for successful fee-generating runs.
