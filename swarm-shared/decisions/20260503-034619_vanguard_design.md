# vanguard / design

## Final Design + Actionable Implementation

**Core synthesis**: Both candidates agree on the root cause (runtime HF API calls during training) and the required invariant (deterministic, content-addressed manifest + CDN-only fetches). Candidate 1 provides the concrete schema and script; Candidate 2 emphasizes orchestration boundaries and quota-safe Lightning usage. The final answer merges them: keep the manifest approach, harden reproducibility and integrity, and add an orchestration contract + Lightning guardrails so the manifest is actually used and compute is never accidentally local.

---

### 1. Orchestration contract (Mac vs Lightning)
- **Control plane (Mac)**: runs ingestion, manifest generation, and job submission only. Never loads models or runs training locally.
- **Data plane (Lightning Studio)**: runs training with zero HF API dataset calls. All data fetched via CDN using the pre-generated manifest.
- **Enforcement**:
  - Training entrypoint asserts env var `LIGHTNING_CLUSTER_ID` is set; if not, abort with clear message (“Refusing to train outside remote Lightning cluster”).
  - Training code imports a small `cdn_data.py` helper that only reads `manifests/{date}/files.json` and streams via `requests`/`urllib`; no `load_dataset` or `datasets` library usage on heterogeneous repos.

### 2. Manifest schema (v1) — canonical location
`/opt/axentx/vanguard/design/manifest-schema.md`

```markdown
# Manifest schema (v1)

Purpose
- Eliminate runtime list_repo_tree/load_dataset calls during training.
- Enable CDN-only dataset fetches (bypass HF API rate limits).
- Provide reproducible, content-addressed snapshots keyed by {date}/{slug}.

File layout
- manifests/{date}/files.json
- manifests/{date}/files.json.sha256  (detached signature of the JSON)

Top-level keys
- generated_at_utc: ISO-8601 UTC timestamp (e.g., "2026-04-29T14:03:00Z")
- repo: HF dataset repo (e.g., "datasets/ORG/REPO")
- date: YYYY-MM-DD folder used at ingestion time
- count: number of entries
- entries: list of objects

Each entry
- slug: deterministic short name derived from path (e.g., "repo~path~to~file")
- repo: HF dataset repo
- path: repo-relative path to file
- cdn_url: https://huggingface.co/datasets/{repo}/resolve/main/{path}
- sha256: hex digest of file content
- size_bytes: integer

Usage contract
- Mac (control plane) runs generate_manifest.py once per date folder after rate-limit window clears.
- Output committed/stored alongside training job config.
- Lightning training script (remote) loads files.json and streams via CDN URLs only (no HF API).
- No load_dataset(streaming=True) on heterogeneous repos; use per-file CDN fetch + parse.
- Before training starts, recompute sha256 of each fetched file and compare to manifest; abort on mismatch.
```

#!/usr/bin/env python3
"""
Generate a deterministic manifest for a single date folder in a HuggingFace dataset repo.

Usage:
  python generate_manifest.py \
    --repo datasets/ORG/REPO \
    --date 2026-04-29 \
    --out-dir /opt/axentx/vanguard/manifests

Produces:
  manifests/2026-04-29/files.json
  manifests/2026-04-29/files.json.sha256
"""

import argparse
import hashlib
import json
import os
import sys
from datetime import datetime, timezone
from pathlib import Path
from typing import List, Dict

from huggingface_hub import HfApi, hf_hub_download

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def sha256_file(path: str) -> str:
    h = hashlib.sha256()
    with open(path, "rb") as f:
        for chunk in iter(lambda: f.read(128 * 1024), b""):
            h.update(chunk)
    return h.hexdigest()

def sha256_bytes(b: bytes) -> str:
    return hashlib.sha256(b).hexdigest()

def slugify_filename(filename: str) -> str:
    # Deterministic, filesystem-safe short name
    return filename.replace("/", "~").replace(".", "_")

def list_date_folder(api: HfApi, repo: str, date: str) -> List[str]:
    """
    List files in repo under folder `date/` (non-recursive by design).
    Returns repo-relative paths.
    """
    try:
        tree = api.list_repo_tree(repo=repo, path=date, repo_type="dataset", recursive=False)
        files = [item.rfilename for item in tree if not item.rfilename.endswith("/")]
        return sorted(files)
    except Exception as e:
        print(f"Error listing repo tree for {repo}/{date}: {e}", file=sys.stderr)
        raise

def build_manifest_entries(repo: str, date: str, paths: List[str], cache_dir: str) -> List[Dict]:
    entries = []
    for rel_path in paths:
        try:
            local_path = hf_hub_download(
                repo_id=repo,
                filename=rel_path,
                repo_type="dataset",
                cache_dir=cache_dir,
                force_download=False,
            )
            entries.append({
                "slug": slugify_filename(rel_path),
                "repo": repo,
                "path": rel_path,
                "cdn_url": CDN_TEMPLATE.format(repo=repo, path=rel_path),
                "sha256": sha256_file(local_path),
                "size_bytes": os.path.getsize(local_path),
            })
        except Exception as e:
            print(f"Warning: failed to process {repo}/{rel_path}: {e}", file=sys.stderr)
            continue
    return entries

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate CDN manifest for a date folder.")
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., datasets/ORG/REPO)")
    parser.add_argument("--date", required=True, help="Date folder (YYYY-MM-DD)")
    parser.add_argument("--out-dir", default="/opt/axentx/vanguard/manifests", help="Output directory")
    parser.add_argument("--cache-dir", default="/opt/axentx/vanguard/.cache/hf", help="HF cache dir")
    args = parser.parse_args()

    # Validate date format
    try:
        datetime.strptime(args.date, "%Y-%m-%d")
    except ValueError:
        print("Error: --date must be YYYY-MM-DD", file=sys.stderr)
        sys.exit(1)

    api = HfApi()
    paths = list_date_folder(api, repo=args.repo, date=args.date)
    if not paths:
        print(f"No files found in {args.repo}/{args.date}", file=sys.stderr)
        sys.exit(1)

    os.makedirs(args.cache_dir, exist_ok=True)
    entries = build_manifest_entries(args.repo, args.date, paths, cache_dir=args.cache_dir)

    out_dir = Path(args.out_dir) / args.date
    out_dir.mkdir(parents=True, exist_ok=True)
    out_path = out_dir / "files.json"

    manifest = {
        "generated_at_utc": datetime.now(timezone.utc).replace(microsecond=0).isoformat(),
        "repo": args.repo,
        "date": args.date,
        "count": len(entries),
        "entries": entries,
    }

    # Write canonical JSON (sorted keys, deterministic)
    json_bytes = json.dumps(manifest, indent=2, sort_keys=True, ensure_ascii=False).encode("utf-8")
    with open(out_path, "wb") as f:
        f.write(json_bytes)
