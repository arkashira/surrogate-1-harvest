## 1) Deterministic CDN manifest (no HF API during training)
- **What to do**  
  - Run **once per date folder** from your Mac (or any non‑rate‑limited environment) to produce `manifests/manifest-YYYY-MM-DD.json`.  
  - Manifest schema: `{date, repo, files: [{path, cdn_url}]}`.  
  - Include only `.parquet` / `.jsonl`.  
  - Commit or stage the manifest alongside code so training never calls `list_repo_tree` or file‑metadata APIs.

- **Why**  
  - Eliminates HF API 429s during data loading.  
  - Makes the dataset path deterministic and reproducible.

- **Concrete script** (`scripts/build_manifest.py`)
  ```python
  #!/usr/bin/env python3
  import json, os, sys
  from datetime import datetime
  from huggingface_hub import HfApi

  REPO = "axentx/surrogate-1-ingest"
  DATE = sys.argv[1] if len(sys.argv) > 1 else datetime.utcnow().strftime("%Y-%m-%d")
  OUT = f"manifests/manifest-{DATE}.json"

  api = HfApi()
  entries = api.list_repo_tree(repo_id=REPO, path=DATE, recursive=False)

  files = [
      {"path": e.path, "cdn_url": f"https://huggingface.co/datasets/{REPO}/resolve/main/{e.path}"}
      for e in entries
      if e.path.lower().endswith((".parquet", ".jsonl"))
  ]

  os.makedirs(os.path.dirname(OUT), exist_ok=True)
  with open(OUT, "w") as f:
      json.dump({"date": DATE, "repo": REPO, "files": files}, f, indent=2)

  print(f"Wrote {len(files)} files -> {OUT}")
  ```
  - Run: `python scripts/build_manifest.py 2023-04-01`

## 2) Lightning Studio lifecycle guard (correct, minimal, safe)
- **What to do**  
  - Before any training run, ensure a Studio named `surrogate-train` is **Running**.  
  - If stopped, restart with `target.start(machine="L40S")` (fallback to L40S if H200 unavailable).  
  - Do **not** embed complex schema checks or ingestion logic inside the lifecycle guard; keep it focused on uptime.

- **Why**  
  - Prevents quota waste and job death from idle timeouts.  
  - Avoids race conditions where a stopped studio blocks progress.

- **Concrete guard** (importable helper)
  ```python
  # surrogate/studio.py
  from lightning.pytorch import Studio

  def ensure_studio_running(name: str = "surrogate-train", machine: str = "L40S") -> Studio:
      studio = Studio(name, create_ok=True)
      if studio.status != "Running":
          studio.start(machine=machine)
      return studio
  ```

- **Usage in entrypoint** (`surrogate/train.py`)

## 4) CDN-only data loader (no HF API calls)
- **What to do**  
  - Training reads `manifest-*.json`, then fetches each `cdn_url` via `requests` or `pyarrow` directly from `https://huggingface.co/datasets/.../resolve/main/...`.  
  - No `list_repo_tree`, no `Repository`, no metadata API calls during training.

- **Concrete loader snippet**
  ```python
  # surrogate/data.py
  import json, io, requests, pyarrow.parquet as pq
  from torch.utils.data import IterableDataset

  class CDNParquetDataset(IterableDataset):
      def __init__(self, manifest_path: str):
          with open(manifest_path) as f:
              self.manifest = json.load(f)
          self.urls = [f["cdn_url"] for f in self.manifest["files"]]

      def __iter__(self):
          for url in self.urls:
              resp = requests.get(url, timeout=30)
              resp.raise_for_status()
              table = pq.read_table(io.BytesIO(resp.content))
              for row in table.select(["prompt", "response"]).to_pylist():
                  yield row
  ```
