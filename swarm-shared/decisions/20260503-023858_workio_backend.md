# workio / backend

## Final Synthesis — One Correct, Actionable Plan

**Goal:** Eliminate duplicate punches and illegal state transitions caused by LINE webhook retries and concurrency, while remaining practical to implement now.

---

### 1. Diagnosis (merged, prioritized)
- **Missing idempotency at DB and service layer** allows duplicate rows on LINE retries and concurrent requests.
- **Application-only transition checks race**; two near-simultaneous requests can both create “in” rows.
- **No persisted idempotency key** for LINE webhook deliveries; retries create new punches instead of returning prior result.
- **No DB-level constraints** to enforce one open punch per user/tenant or to deduplicate by LINE event.
- **Geofence and queue-first behaviors are out of scope for this fix** (address separately) — focus on correctness and idempotency first.

---

### 2. Proposed changes (concrete files + actions)

**Files to modify:**
- `workio/server/src/db/schema.sql` — add constraints, indexes, and idempotency table.
- `workio/server/src/services/punchService.ts` — enforce idempotency + transition guards in a single transactional flow.
- `workio/server/src/routes/line/webhook.ts` — extract idempotency key, verify signature, call service transactionally, and return fast on duplicates.

#### 3.1 DB schema (`workio/server/src/db/schema.sql`)

```sql
-- Idempotency table for LINE webhook deliveries
CREATE TABLE IF NOT EXISTS line_webhook_idempotency (
  idempotency_key VARCHAR(255) PRIMARY KEY,
  event_type      VARCHAR(50) NOT NULL,
  punch_id        BIGINT,
  processed_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  payload_hash    VARCHAR(64),
  CONSTRAINT fk_punch FOREIGN KEY (punch_id) REFERENCES punches(id) ON DELETE SET NULL
);

-- Add trace columns to punches (if not present)
ALTER TABLE punches
  ADD COLUMN IF NOT EXISTS line_event_id VARCHAR(255),
  ADD COLUMN IF NOT EXISTS idempotency_key VARCHAR(255);

-- One open punch per user/tenant (prevents double clock-in)
CREATE UNIQUE INDEX IF NOT EXISTS idx_punches_open_per_user
  ON punches (tenant_id, user_id)
  WHERE status = 'in' AND deleted_at IS NULL;

-- Deduplicate punches by LINE event (when available)
CREATE UNIQUE INDEX IF NOT EXISTS idx_punches_line_event_unique
  ON punches (tenant_id, line_event_id)
  WHERE line_event_id IS NOT NULL;

-- Index for fast transition checks
CREATE INDEX IF NOT EXISTS idx_punches_user_status
  ON punches (tenant_id, user_id, status)
  WHERE deleted_at IS NULL;
```

#### 3.2 Service layer (`workio/server/src/services/punchService.ts`)

```ts
import { PoolClient } from 'pg';
import { db } from '../db';

export type PunchInput = {
  tenantId: number;
  userId: number;
  type: 'clock_in' | 'clock_out';
  latitude?: number;
  longitude?: number;
  lineEventId?: string;
  idempotencyKey?: string;
  timestamp?: Date;
};

export async function createPunch(
  input: PunchInput,
  client?: PoolClient
): Promise<{ punch: any; created: boolean }> {
  const dbClient = client || (await db.connect());
  const release = client ? undefined : dbClient.release;
  const shouldManageTx = !client;

  try {
    if (shouldManageTx) await dbClient.query('BEGIN');

    const now = input.timestamp || new Date();

    // 1) Idempotency fast path
    if (input.idempotencyKey) {
      const existing = await dbClient.query<{ punch_id: number }>(
        `SELECT punch_id FROM line_webhook_idempotency WHERE idempotency_key = $1`,
        [input.idempotencyKey]
      );
      if (existing.rows[0]?.punch_id) {
        const punch = await dbClient.query<{
          id: number;
          tenant_id: number;
          user_id: number;
          status: string;
          created_at: Date;
        }>(`SELECT id, tenant_id, user_id, status, created_at FROM punches WHERE id = $1`, [
          existing.rows[0].punch_id,
        ]);
        if (punch.rows[0]) {
          if (shouldManageTx) await dbClient.query('COMMIT');
          return { punch: punch.rows[0], created: false };
        }
      }
    }

    // 2) Transition guard
    if (input.type === 'clock_in') {
      const open = await dbClient.query<{ id: number }>(
        `SELECT id FROM punches
         WHERE tenant_id = $1 AND user_id = $2 AND status = 'in' AND deleted_at IS NULL
         LIMIT 1`,
        [input.tenantId, input.userId]
      );
      if (open.rows[0]) {
        const punch = await dbClient.query<{
          id: number;
          tenant_id: number;
          user_id: number;
          status: string;
          created_at: Date;
        }>(`SELECT id, tenant_id, user_id, status, created_at FROM punches WHERE id = $1`, [open.rows[0].id]);

        // Record idempotency to dedupe future retries
        if (input.idempotencyKey) {
          await dbClient.query(
            `INSERT INTO line_webhook_idempotency (idempotency_key, event_type, punch_id)
             VALUES ($1, $2, $3) ON CONFLICT DO NOTHING`,
            [input.idempotencyKey, 'clock_in', punch.rows[0]?.id]
          );
        }

        if (shouldManageTx) await dbClient.query('COMMIT');
        return { punch: punch.rows[0], created: false };
      }
    }

    // 3) Create punch
    const insertRes = await dbClient.query<{ id: number }>(
      `INSERT INTO punches (tenant_id, user_id, status, latitude, longitude, created_at, line_event_id, idempotency_key)
       VALUES ($1, $2, $3, $4, $5, $6, $7, $8)
       RETURNING id`,
      [
        input.tenantId,
        input.userId,
        input.type === 'clock_in' ? 'in' : 'out',
        input.latitude,
        input.longitude,
        now,
        input.lineEventId || null,
        input.idempotencyKey || null,
      ]
    );

    const punch = await dbClient.query<{
      id: number;
      tenant_id: number;
      user_id: number;
      status: string;
      created_at: Date;
    }>(`SELECT id, tenant_id, user_id, status, created_at FROM punches WHERE id = $1`, [insertRes.rows[0].id]);

    // 4) Record idempotency
    if (input.idempotencyKey) {
      await dbClient.query(
        `INSERT INTO line_webhook_idempotency (idempotency_key, event_type, punch_id)
         VALUES ($1, $2, $3) ON CONFLICT DO NOTHING`,
        [input.idempotencyKey, input.type, insertRes.rows[0].id]
      );
    }

    if (shouldManageTx) await dbClient.query('COMMIT');
    return { punch: punch.rows[0], created: true };
  } catch (err) {
    if (shouldManageTx) await dbClient.query('ROLLBACK');
    throw err;
  } finally {
    if (release) release();
  }
}
```
