# vanguard / design

## Final synthesized plan (strongest + correct + actionable)

**Core principle:** Eliminate all runtime HF API calls from training/ingest. Use a **manifest-first, CDN-only contract** that is deterministic, content-addressed, and reproducible. Keep frontend safe (no changes) and make Lightning Studio reuse explicit to avoid quota waste and idle-stop deaths.

---

## 1. Diagnosis (resolved)

- Frontend still triggers runtime HF API calls (`list_repo_tree`, `load_dataset`) during dataset selection/preview → 429 risk and non-reproducible runs.  
  → **Mitigation:** Do not change frontend yet; shift all heavy enumeration to a one-time Mac-side manifest build.

- No deterministic, content-addressed manifest for dataset files; training scripts can’t guarantee CDN-only fetches.  
  → **Fix:** Manifest must include content hash (or commit hash + path) and exact CDN URL. Training/ingest must refuse to use `/api/` endpoints.

- Backend ingestion/training jobs re-enumerate files at runtime.  
  → **Fix:** Enumerate once per date folder, store `manifests/{date}/files.json`, and use that as the single source of truth.

- Missing single-date folder enumeration strategy; backend paginates across repos.  
  → **Fix:** `list_repo_tree(date_folder, recursive=False)` once; embed full file list in manifest.

- No CDN-bypass path construction in training pipeline; ingestion/training can still hit `/api/` auth endpoints during data loading.  
  → **Fix:** Build CDN URLs as `https://huggingface.co/datasets/{repo}/resolve/main/{path}` and never call HF Hub API at train/ingest time.

- No reuse guard for Lightning Studio (creates new runs instead of reusing running ones) → wastes quota and risks idle-stop training death.  
  → **Fix:** Explicit reuse logic + deterministic fallback machine (L40S/public tier) and restart-if-stopped behavior.

---

## 2. Proposed change (final)

Add a **manifest-first, CDN-only ingestion/training contract** under `vanguard/`:

- `vanguard/manifest.py` — one-time script (Mac or CI): `list_repo_tree(date_folder, recursive=False)` → `manifests/{date}/files.json`. Include commit OID and file size for content-addressability and integrity checks.
- `vanguard/train.py` — Lightning entrypoint that loads `manifests/{date}/files.json` and downloads via CDN URLs only. Uses deterministic ordering and minimal in-memory footprint.
- `vanguard/studio.py` — reuse-running guard + deterministic machine fallback (L40S) and restart-if-stopped.
- `vanguard/ingest.py` — project-to `{prompt,response}` only; write to `batches/mirror-merged/{date}/{slug}.parquet`. Uses CDN-only fetches; no HF API calls.

Scope: 4 files + minimal config. No frontend changes (kept safe).

## 3. Implementation (final, corrected, executable)

Create the following files in `/opt/axentx/vanguard/`.

### vanguard/manifest.py
```python
#!/usr/bin/env python3
"""
Generate deterministic, content-addressed manifest for a date folder.
Run from Mac/CI after HF API rate-limit window clears.
"""
import json
import os
import sys
from pathlib import Path

from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_DATASET_REPO", "axentx/vanguard-datasets")
OUT_DIR = Path("manifests")

def build_manifest(date_folder: str, recursive: bool = False) -> dict:
    api = HfApi()
    # Use repo_type="dataset" explicitly
    tree = api.list_repo_tree(repo_id=HF_REPO, repo_type="dataset", path=date_folder, recursive=recursive)
    files = {}
    for item in tree:
        if item.rfilename is None or getattr(item, "type", None) != "file":
            continue
        # Prefer commit OID for content-addressability when available
        commit = getattr(item, "commit_oid", None) or getattr(item, "oid", None) or ""
        slug = Path(item.rfilename).stem
        files[slug] = {
            "path": item.rfilename,
            "size": getattr(item, "size", 0),
            "commit": commit,
            "cdn_url": f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{item.rfilename}"
        }
    return files

def main():
    if len(sys.argv) < 2:
        print("Usage: python manifest.py <YYYY-MM-DD>")
        sys.exit(1)
    date_folder = sys.argv[1]
    files = build_manifest(date_folder)
    out_path = OUT_DIR / date_folder / "files.json"
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(files, indent=2))
    print(f"Manifest written: {out_path} ({len(files)} files)")

if __name__ == "__main__":
    main()
```

#!/usr/bin/env python3
"""
Lightning training entrypoint that uses CDN-only fetches.
Expects manifests/{date}/files.json produced by manifest.py.
"""
import json
import os
import tempfile
from pathlib import Path
from typing import List

import lightning as L
import pandas as pd
import requests
import torch
from torch.utils.data import Dataset, DataLoader

HF_REPO = os.getenv("HF_DATASET_REPO", "axentx/vanguard-datasets")
MANIFEST_PATH = Path(os.getenv("MANIFEST_PATH", "manifests/latest/files.json"))

class CDNParquetDataset(Dataset):
    def __init__(self, file_entries: List[dict]):
        self.entries = file_entries

    def __len__(self):
        return len(self.entries)

    def __getitem__(self, idx):
        entry = self.entries[idx]
        url = entry["cdn_url"]
        resp = requests.get(url, timeout=120)
        resp.raise_for_status()
        with tempfile.NamedTemporaryFile(suffix=".parquet", delete=False) as f:
            f.write(resp.content)
            f.flush()
            df = pd.read_parquet(f.name)
        if df.empty:
            raise ValueError(f"Empty parquet from {url}")
        row = df.iloc[0]
        prompt = str(row.get("prompt", row.get("input", "")))
        response = str(row.get("response", row.get("output", "")))
        os.unlink(f.name)
        return {"prompt": prompt, "response": response}

class SurrogateDataModule(L.LightningDataModule):
    def __init__(self, manifest_path: Path, batch_size: int = 8):
        super().__init__()
        self.manifest_path = manifest_path
        self.batch_size = batch_size

    def setup(self, stage=None):
        if not self.manifest_path.exists():
            raise FileNotFoundError(f"Manifest not found: {self.manifest_path}")
        entries = json.loads(self.manifest_path.read_text())
        # Deterministic order by slug
        items = [{"slug": k, **v} for k, v in sorted(entries.items())]
        self.dataset = CDNParquetDataset(items)

    def train_dataloader(self):
        return DataLoader(self.dataset, batch_size=self.batch_size, shuffle=True)

            name="vanguard-train",
