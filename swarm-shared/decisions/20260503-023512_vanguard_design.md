# vanguard / design

## Final Synthesis (Best of Both Candidates)

Both candidates identify the same root causes and prescribe the same high-level fixes. The differences are minor and complementary. The final answer below merges the strongest, most concrete parts into a single, actionable plan.

---

## 1. Diagnosis (Consolidated)
- **CDN-bypass not used**: Data loads and training still route through authenticated Hugging Face API (`list_repo_tree`, `load_dataset`) instead of public CDN URLs → burns quota and risks 429s.
- **No pre-listed file manifest**: Every preview/training launch re-enumerates repo files via API instead of using a static JSON → redundant API calls, fragility during rate limits, and UX latency.
- **No Studio reuse guard**: Lightning Studio is recreated per launch instead of reusing running instances → wastes 80hr/mo quota.
- **No idle-stop resilience**: Lightning idle timeout kills training; no status check or restart before `.run()` → lost compute and orphaned jobs.

---

## 2. Proposed Change (Concrete Scope)
Create **one new frontend file** and **update or create one template** (if not present). Scope:
- Embed a pre-generated `file-manifest.json` (produced once per date folder).
- Use CDN URLs for all dataset file fetches (zero auth, zero API quota).
- Reuse a running Lightning Studio by name; restart only if stopped.
- Add an idle-stop guard before each training run.

---

### frontend/static/js/vanguard-launch.js
```javascript
// vanguard-launch.js — CDN-bypass + Studio reuse + idle guard
const CONFIG = {
  datasetRepo: 'your-org/vanguard-mirror',   // <-- set per env
  dateFolder: '2026-05-03',                  // <-- updated per manifest
  studioName: 'vanguard-l40s',
  machine: 'L40S',
  hfCdnBase: 'https://huggingface.co/datasets'
};

let cachedManifest = null;

async function loadManifest() {
  if (cachedManifest) return cachedManifest;
  const res = await fetch(`/static/data/file-manifest-${CONFIG.dateFolder}.json`);
  if (!res.ok) throw new Error('Manifest missing');
  cachedManifest = await res.json(); // ["path1.parquet","path2.parquet",...]
  return cachedManifest;
}

function buildCdnUrl(path) {
  return `${CONFIG.hfCdnBase}/${CONFIG.datasetRepo}/resolve/main/${CONFIG.dateFolder}/${encodeURIComponent(path)}`;
}

async function fetchParquetSlice(path, byteRange = '0-65535') {
  // CDN fetch with Range header to avoid full download during preview
  const url = buildCdnUrl(path);
  const res = await fetch(url, { headers: { Range: `bytes=${byteRange}` } });
  if (!res.ok && res.status !== 206) throw new Error(`CDN fetch failed: ${res.status}`);
  return res.arrayBuffer();
}

async function listAvailableFiles() {
  const manifest = await loadManifest();
  return manifest.map(p => ({ path: p, cdn: buildCdnUrl(p) }));
}

// Lightning Studio reuse + idle guard
async function ensureStudioRunning(lightning, targetName, machine) {
  const teamspace = await lightning.Teamspace.current();
  const studios = await teamspace.studios();
  const existing = studios.find(s => s.name === targetName && s.status === 'Running');
  if (existing) return existing;

  // If stopped, restart; else create
  const stopped = studios.find(s => s.name === targetName && s.status === 'Stopped');
  if (stopped) {
    await stopped.start({ machine });
    return stopped;
  }

  return await teamspace.studios.create({
    name: targetName,
    machine,
    create_ok: true
  });
}

async function runTrainingJob(scriptPath, args = []) {
  const lightning = new Lightning();
  const studio = await ensureStudioRunning(lightning, CONFIG.studioName, CONFIG.machine);

  // Idle-stop guard: studio may have stopped due to inactivity
  if (studio.status !== 'Running') {
    console.warn(`Studio ${CONFIG.studioName} not running (status=${studio.status}), restarting...`);
    await studio.start({ machine: CONFIG.machine });
  }

  // Build manifest-aware command: pass manifest path so train.py does CDN-only loads
  const manifestPath = `/workspace/file-manifest-${CONFIG.dateFolder}.json`;
  const cmd = [
    'python', scriptPath,
    '--manifest', manifestPath,
    '--cdn-base', CONFIG.hfCdnBase,
    '--repo', CONFIG.datasetRepo,
    '--date', CONFIG.dateFolder,
    ...args
  ].join(' ');

  const run = await studio.run({
    command: cmd,
    cwd: '/workspace',
    environment: {
      SHELL: '/bin/bash',
      PYTHONUNBUFFERED: '1'
    }
  });

  return { studio, run };
}

// Preview: fetch small sample from first file for quick table view
async function previewFirstFile(sampleRows = 10) {
  const files = await listAvailableFiles();
  if (!files.length) return [];
  // Fetch small range for quick peek (not full parse)
  const buf = await fetchParquetSlice(files[0].path, '0-16384');
  // In real UI, decode parquet in worker; here return metadata only
  return { file: files[0].path, size: buf.byteLength, cdn: files[0].cdn };
}

// Expose for UI bindings
window.VanguardLaunch = {
  listAvailableFiles,
  previewFirstFile,
  runTrainingJob,
  buildCdnUrl
};
```

### Optional backend route (FastAPI-like) — for on-demand manifest refresh
```python
# routes/vanguard.py
from fastapi import APIRouter
import httpx
import os

router = APIRouter()

HF_TOKEN = os.getenv("HF_TOKEN")
REPO = "your-org/vanguard-mirror"

@router.get("/api/manifest/{date_folder}")
async def get_manifest(date_folder: str):
    # One API call per date folder; cache aggressively on frontend
    async with httpx.AsyncClient() as client:
        r = await client.get(
            f"https://huggingface.co/api/datasets/{REPO}/tree/{date_folder}",
            headers={"Authorization": f"Bearer {HF_TOKEN}"} if HF_TOKEN else {}
        )
        r.raise_for_status()
        tree = r.json()
    files = [item["path"] for item in tree if item["type"] == "file" and item["path"].endswith(".parquet")]
    return {"date_folder": date_folder, "files": files}
```

```python
# train.py — use manifest + CDN, zero HF API during training
import json
import pyarrow.parquet as pq
import requests
from io import BytesIO

def load_from_cdn(manifest_path, cdn_base, repo, date_folder, max_files=None):
    with open(manifest_path) as f:
        files = json.load(f)
    if max_files:
        files = files[:max_files]

    for rel in files:
        url = f"{cdn_base}/{repo}/resolve/main/{date_folder}/{rel}"
        # stream to avoid large memory
        with requests.get(url, stream=True) as r:
            r.raise_for_status()
            buf = BytesIO(r.content)
            table = pq.read_table(buf, columns=["prompt", "response"])
            yield table.to_pylist()
```
