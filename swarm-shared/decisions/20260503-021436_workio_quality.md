# workio / quality

## 1. Diagnosis

- LINE webhook retries create duplicate punches because ingestion is not idempotent (no dedupe key or idempotency token enforced).
- Clock-in/out path uses non-atomic `findOne` → conditional `insert`/`update`, allowing race conditions under concurrency.
- No server-side idempotency table to track processed LINE events; dedupe relies on app-level checks that can collide.
- Missing unique constraint on (`user_id`, `date`, `type`) for punches, so duplicates can persist and corrupt daily totals.
- No retry-safe webhook handler: partial DB failure can cause LINE to redeliver and create double entries.

## 2. Proposed change

File: `/opt/axentx/workio/server/src/db/schema.sql` (add idempotency table + punch constraint)  
File: `/opt/axentx/workio/server/src/routes/line.ts` (make webhook handler idempotent + atomic upsert)  
Scope: add `line_event_log` table for idempotency, add unique constraint on punches, and replace `findOne`+`insert/update` with atomic `INSERT ... ON CONFLICT DO UPDATE` inside a transaction.

```ts
// /opt/axentx/workio/server/src/routes/line.ts
import { Router } from 'express';
import { pool } from '../db';
import crypto from 'crypto';

const router = Router();

function hashPayload(payload: any): string {
  return crypto.createHash('sha256').update(JSON.stringify(payload)).digest('hex');
}

router.post('/webhook/line', async (req, res) => {
  const body = req.body;
  const events = body.events || [];

  if (!Array.isArray(events) || events.length === 0) {
    return res.status(400).json({ error: 'No events' });
  }

  const client = await pool.connect();
  try {
    for (const ev of events) {
      const idempotencyKey = ev.idempotencyKey || `${ev.source.userId}-${ev.timestamp}-${ev.type}`;
      const payloadHash = hashPayload(ev);

      await client.query('BEGIN');

      // 1) Idempotency check+insert (atomic)
      const idemp = await client.query(
        `INSERT INTO line_event_log (idempotency_key, event_type, user_id, payload_hash, processed_at)
         VALUES ($1, $2, $3, $4, NOW())
         ON CONFLICT (idempotency_key) DO UPDATE
           SET payload_hash = EXCLUDED.payload_hash
         WHERE line_event_log.payload_hash IS DISTINCT FROM EXCLUDED.payload_hash
         RETURNING (xmax = 0) AS inserted, processed_at`,
        [idempotencyKey, ev.type, ev.source.userId, payloadHash]
      );

      const row = idemp.rows[0];
      // If already processed and payload unchanged, skip processing
      if (!row.inserted && row.processed_at) {
        await client.query('COMMIT');
        continue;
      }

      // 2) Process punch (atomic upsert)
      if (ev.type === 'message' && ev.message.type === 'text') {
        const text = ev.message.text.trim().toLowerCase();
        const isClockIn = text.includes('in') || text.includes('clock in');
        const isClockOut = text.includes('out') || text.includes('clock out');
        if (!isClockIn && !isClockOut) {
          await client.query('COMMIT');
          continue;
        }

        const userId = ev.source.userId;
        const today = new Date().toISOString().split('T')[0];
        const type = isClockIn ? 'clock_in' : 'clock_out';
        const now = new Date();

        // Use punches table name and columns matching your schema
        await client.query(
          `INSERT INTO punches (user_id, date, type, timestamp, created_at)
           VALUES ($1, $2, $3, $4, $5)
           ON CONFLICT (user_id, date, type)
           DO UPDATE SET timestamp = EXCLUDED.timestamp`,
          [userId, today, type, now, now]
        );
      }

      // 3) Mark idempotency entry as fully processed
      await client.query(
        `UPDATE line_event_log SET processed_at = NOW() WHERE idempotency_key = $1`,
        [idempotencyKey]
      );

      await client.query('COMMIT');
    }

    res.status(200).json({ status: 'ok' });
  } catch (err) {
    await client.query('ROLLBACK');
    console.error('LINE webhook error:', err);
    res.status(500).json({ error: 'Processing failed' });
  } finally {
    client.release();
  }
});

export default router;
```
