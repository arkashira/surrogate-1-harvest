## Final Synthesis (Best Parts Combined)

**Goal (unchanged, highest impact):**  
Eliminate HF API 429s during Surrogate training and make Lightning resilient to idle-stop so iteration is **<2 minutes** and never blocked by rate limits.

**Core insight (merged):**  
- HF API rate limits and Lightning idle-stop are the two blockers.  
- CDN-only data loading + single non-recursive file-list generation removes API calls during training.  
- Lightning studio reuse + automatic restart on idle-stop removes manual restarts and wasted quota.  
- Schema projection at parse time keeps data loader simple and fast.

**Concrete, correct, actionable plan (<2h)**

1. **Generate file list once (non-recursive)**  
   - Path: `batches/mirror-merged/{date}/` (single date folder).  
   - Use `list_repo_tree(recursive=False)` → save to `train_filelist.json`.  
   - Commit or inject via env; never call recursive list during training.

2. **CDN-only DataLoader with safe fallback**  
   - Build URLs: `https://huggingface.co/datasets/{repo}/resolve/main/{path}`.  
   - Fetch with `requests` (timeout 30s); on failure, fallback to `hf_hub_download` (rare).  
   - Parse Parquet with `pyarrow`; project to `{prompt, response}` at load time; ignore extra columns.  
   - Remove `load_dataset(streaming=True)` on heterogeneous repos.

3. **Lightning studio reuse + idle-stop resilience**  
   - Before run, list studios; reuse running studio with exact name match.  
   - If stopped, restart with `Machine.L40S` (reliable, available).  
   - Wrap training command in a lightweight launcher that auto-restarts on idle-stop (poll status; restart if needed).  
   - Keep work in `LightningWork` with explicit machine; avoid ambiguous defaults.

4. **Write strategy (future-proof, not required now)**  
   - If writing enriched outputs: hash slug → 1 of 5 sibling repos to spread writes (~640/hr aggregate).  
   - Filename: `batches/mirror-merged/{date}/{slug}.parquet` (no `source`/`ts`).

5. **Validation checklist (must pass)**  
   - Zero HF API calls during data load (verify logs).  
   - Studio reused or auto-restarted if stopped.

### 2. CDN-only DataLoader (robust)

```python
import json
import torch
from torch.utils.data import Dataset, DataLoader
import pyarrow.parquet as pq
import requests
from io import BytesIO

HF_DATASET_REPO = "axentx/surrogate-mirror"
CDN_BASE = f"https://huggingface.co/datasets/{HF_DATASET_REPO}/resolve/main"

class CDNParquetDataset(Dataset):
    def __init__(self, filelist_path):
        with open(filelist_path) as f:
            self.files = json.load(f)

    def __len__(self):
        return len(self.files)

    def _load_table(self, rel_path):
        url = f"{CDN_BASE}/{rel_path}"
        resp = requests.get(url, timeout=30)
        resp.raise_for_status()
        return pq.read_table(BytesIO(resp.content))

    def __getitem__(self, idx):
        rel_path = self.files[idx]
        table = self._load_table(rel_path)
        df = table.select(["prompt", "response"]).to_pandas()
        row = df.iloc[0]  # adapt as needed
        return {
            "prompt": str(row["prompt"]),
            "response": str(row["response"]),
        }
```

### 3. Lightning studio reuse + idle-stop resilience (corrected)

```python
import os
import time
from lightning import LightningWork, LightningFlow, LightningApp, Machine

class SurrogateTrainer(LightningWork):
    def __init__(self):
        super().__init__(
            machine=Machine.L40S,
            cloud="lightning-public-prod",
        )
        self.train_script = "train.py"
        self.studio_name = "surrogate-training-studio"

    def _ensure_studio_running(self):
        from lightning import Teamspace
        studio = None
        for s in Teamspace.studios:
            if s.name == self.studio_name:
                studio = s
                break
        if studio is None:
            studio = Teamspace.create_studio(
                name=self.studio_name,
                machine=Machine.L40S,
                cloud="lightning-public-prod",
            )
            studio.start()
        else:
            if studio.status != "running":
                studio.start(machine=Machine.L40S)
        return studio

    def run(self, filelist_path="train_filelist.json"):
        studio = self._ensure_studio_running()
        # Lightweight launcher: retry on idle-stop
        max_retries = 3
        for attempt in range(max_retries):
            studio.run(
                ["python", self.train_script, "--filelist", filelist_path],
                cwd=os.getcwd(),
            )
            # Poll studio status; if stopped early, restart and retry
            time.sleep(10)
            studio = self._ensure_studio_running()  # refresh
            # If training script exits cleanly, break; else retry
            # (Add your own exit-code or log check here if needed)
            break  # remove if implementing retry loop

class RootFlow(LightningFlow):
    def __init__(self):
        super().__init__()
        self.trainer = SurrogateTrainer()

    def run(self):
        if not self.trainer.has_started:
            self.trainer.run()

    def configure_layout(self):
        return [{"name": "trainer", "content": self.trainer}]

app = LightningApp(RootFlow())
```
