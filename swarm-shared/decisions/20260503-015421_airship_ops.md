# airship / ops

## Highest-Value Incremental Improvement (≤2h)

**CDN-first ingestion + deterministic sibling-repo sharding + Studio lifecycle guard**  
- Eliminates HF API 429s during training by pre-listing once and downloading via public CDN  
- Bypasses 128/hr commit cap via deterministic sibling-repo sharding  
- Prevents idle-timeout training death by guarding Studio lifecycle  

---

## Implementation Plan

### 1. Pre-list file paths once (Mac orchestration)
- Single API call to `list_repo_tree(path, recursive=False)` for one date folder  
- Save list to `file_list.json` committed to repo or passed to Lightning  
- Embed in `train.py`; Lightning training does CDN-only fetches with zero API calls during data load  

### 2. CDN-only download in training script
- Use `https://huggingface.co/datasets/{repo}/resolve/main/{path}` with no Authorization header  
- Project to `{prompt, response}` only at parse time (avoid `load_dataset(streaming=True)` for mixed schemas)  

### 3. Deterministic sibling-repo sharding for writes
- Hash slug → pick repo deterministically across 5 siblings  
- Aggregate 640/hr commit capacity  

### 4. Studio lifecycle guard
- Before `Studio(create_ok=True)`, list `Teamspace.studios` and reuse Running ones  
- Check status before each `.run()`; restart with `target.start(machine=Machine.L40S)` if stopped

### 2. CDN-only dataset loader (train.py)
```python
import json
import pyarrow.parquet as pq
import requests
from io import BytesIO
from pathlib import Path

HF_DATASET = "my-org/surrogate-dataset"
CDN_ROOT = f"https://huggingface.co/datasets/{HF_DATASET}/resolve/main"

def load_file_list(path="file_list.json"):
    with open(path) as f:
        return json.load(f)

def cdn_fetch(rel_path):
    url = f"{CDN_ROOT}/{rel_path}"
    r = requests.get(url, timeout=30)
    r.raise_for_status()
    return BytesIO(r.content)

def project_to_prompt_response(parquet_bytes):
    table = pq.read_table(BytesIO(parquet_bytes))
    # Project only what we need; ignore mixed schema extras
    return {
        "prompt": table.column("prompt").to_pylist(),
        "response": table.column("response").to_pylist(),
    }

def build_dataset(file_list):
    examples = []
    for rel_path in file_list:
        buf = cdn_fetch(rel_path)
        batch = project_to_prompt_response(buf.read())
        for p, r in zip(batch["prompt"], batch["response"]):
            examples.append({"prompt": p, "response": r})
    return examples

if __name__ == "__main__":
    files = load_file_list()
    dataset = build_dataset(files)
    print(f"Loaded {len(dataset)} examples via CDN (zero HF API calls during load)")
```

### 3. Deterministic sibling-repo sharding for writes
```python
import hashlib

SIBLINGS = [
    "my-org/surrogate-dataset",
    "my-org/surrogate-dataset-sib1",
    "my-org/surrogate-dataset-sib2",
    "my-org/surrogate-dataset-sib3",
    "my-org/surrogate-dataset-sib4",
]

def pick_repo(slug: str) -> str:
    digest = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    return SIBLINGS[digest % len(SIBLINGS)]

# Usage
repo = pick_repo("batch-2026-05-03/some-slug")
print(f"Write to {repo}")
```

### 4. Studio lifecycle guard (Lightning SDK)
```python
from lightning.app import LightningStudio, Machine
from lightning.app.core.work import Work

def get_running_studio(name: str):
    from lightning.app import Teamspace
    for s in Teamspace.studios:
        if s.name == name and s.status == "Running":
            return s
    return None

def ensure_studio_and_run(name: str, target: Work, machine=Machine.L40S):
    studio = get_running_studio(name)
    if studio is None:
        studio = LightningStudio(name=name, target=target, machine=machine, create_ok=True)
    else:
        # Reuse existing; ensure machine matches if restart needed
        studio.target = target
        studio.machine = machine

    if studio.status != "Running":
        studio.start(machine=machine)
    return studio

# Example usage in training orchestration
if __name__ == "__main__":
    from my_train_work import TrainWork  # your Lightning Work
    target = TrainWork()
    studio = ensure_studio_and_run("surrogate-train-studio", target, machine=Machine.L40S)
    studio.run()
```
