# Costinel / design

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a high-value, zero-runtime-HF-API “Top Hub Signal” panel to Costinel that surfaces the most-connected knowledge-hub (e.g., MOC) with contextual insights. Uses CDN-only fetches, build-time baked file list, and robust fallback.

### Scope (what will ship)
- New React component: `TopHubSignalPanel`
- Build-time script: `scripts/bake-top-hub.js` (runs on CI / pre-deploy)
- Runtime behavior:
  - Fetch baked `top-hub.json` from CDN (`/datasets/axentx/costinel-knowledge/resolve/main/baked/top-hub.json`)
  - Render card with hub name, summary, related docs, and quick links
  - If CDN fails → render empty state (no errors to user)
- No HF API calls at runtime; no rate-limit risk
- Reuses existing design tokens and layout patterns

### File changes
- `src/components/TopHubSignalPanel/TopHubSignalPanel.tsx` (new)
- `src/components/TopHubSignalPanel/TopHubSignalPanel.module.css` (new)
- `scripts/bake-top-hub.js` (new)
- `package.json` (add build script)
- `src/pages/Dashboard/Dashboard.tsx` (import + mount panel)

### Build script behavior
- Runs on Mac (or CI) — orchestration only
- Uses HF API once (after rate-limit window) to `list_repo_tree` for the knowledge repo’s baked folder
- Saves `{ hub, summary, relatedDocs[], updatedAt }` to `public/datasets/axentx/costinel-knowledge/baked/top-hub.json`
- Commits baked file (or CI uploads) so CDN serves it

## Code snippets

### 1) Build script — `scripts/bake-top-hub.js`
```bash
#!/usr/bin/env bash
set -euo pipefail
# Bake top-hub metadata for CDN-first panel.
# Usage: bash scripts/bake-top-hub.js
# Requires: HF_TOKEN in env (only for listing repo tree)

REPO="axentx/costinel-knowledge"
OUT_DIR="public/datasets/axentx/costinel-knowledge/baked"
OUT_FILE="${OUT_DIR}/top-hub.json"
mkdir -p "$(dirname "$OUT_FILE")"

# If HF_TOKEN present, list tree to pick latest baked file; otherwise fallback to local stub.
if [ -n "${HF_TOKEN:-}" ]; then
  # Single non-recursive call to avoid pagination/rate-limit
  curl -s -H "Authorization: Bearer $HF_TOKEN" \
    "https://huggingface.co/api/datasets/${REPO}/tree?recursive=false&path=baked" \
    | node -e "
      const fs = require('fs');
      const chunks = [];
      process.stdin.on('data', c => chunks.push(c));
      process.stdin.on('end', () => {
        try {
          const files = JSON.parse(Buffer.concat(chunks).toString());
          const topHubFile = files.find(f => f.path.startsWith('top-hub') && f.path.endsWith('.json'));
          if (topHubFile) {
            // CDN fetch (no auth) to get content and write locally
            const https = require('https');
            https.get('https://huggingface.co/datasets/' + '${REPO}' + '/resolve/main/' + topHubFile.path, (res) => {
              let data = '';
              res.on('data', d => data += d);
              res.on('end', () => {
                fs.writeFileSync('${OUT_FILE}', data);
                console.log('Baked top-hub from CDN:', topHubFile.path);
              });
            }).on('error', (err) => {
              console.error('CDN fetch failed, using stub:', err.message);
              writeStub();
            });
          } else {
            console.log('No baked top-hub file found, using stub.');
            writeStub();
          }
        } catch (err) {
          console.error('Failed to parse tree, using stub:', err.message);
          writeStub();
        }
      });
      function writeStub() {
        const stub = { hub: 'MOC', summary: 'Mission Operations Center — central coordination hub.', relatedDocs: [], updatedAt: new Date().toISOString() };
        fs.writeFileSync('${OUT_FILE}', JSON.stringify(stub, null, 2));
      }
    "
else
  echo "HF_TOKEN not set — writing stub top-hub.json"
  node -e "
    const fs = require('fs');
    const stub = { hub: 'MOC', summary: 'Mission Operations Center — central coordination hub.', relatedDocs: [], updatedAt: new Date().toISOString() };
    fs.writeFileSync('${OUT_FILE}', JSON.stringify(stub, null, 2));
  "
fi
```

Make executable:
```bash
chmod +x scripts/bake-top-hub.js
```

### 2) React panel — `src/components/TopHubSignalPanel/TopHubSignalPanel.tsx`
```tsx
import React, { useEffect, useState } from "react";
import styles from "./TopHubSignalPanel.module.css";

interface RelatedDoc {
  title: string;
  url: string;
  type?: "doc" | "pr" | "runbook";
}

interface TopHubPayload {
  hub: string;
  summary: string;
  relatedDocs: RelatedDoc[];
  updatedAt: string;
}

const CDN_URL =
  "https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/baked/top-hub.json";

export const TopHubSignalPanel: React.FC = () => {
  const [data, setData] = useState<TopHubPayload | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    let cancelled = false;
    fetch(CDN_URL, { cache: "no-store" })
      .then((res) => {
        if (!res.ok) throw new Error("CDN fetch failed");
        return res.json();
      })
      .then((json) => {
        if (!cancelled) {
          setData(json);
          setLoading(false);
        }
      })
      .catch(() => {
        // Graceful degradation: render nothing (or minimal stub)
        if (!cancelled) {
          setData(null);
          setLoading(false);
        }
      });
    return () => {
      cancelled = true;
    };
  }, []);

  if (loading) {
    return (
      <div className={styles.panel}>
        <div className={styles.skeleton}>Loading top hub…</div>
      </div>
    );
  }

  if (!data) {
    // Silent fallback — no runtime errors shown to user
    return null;
  }

  return (
    <div className={styles.panel}>
      <div className={styles.header}>
        <h3 className={styles.hub}>📍 {data.hub}</h3>
        <time className={styles.time} dateTime={data.updatedAt}>
          Updated {new Date(data.updatedAt).toLocaleDateString()}
        </time>
      </div>
      <p className={styles.summary}>{data.summary}</p>

      {data.relatedDocs.length > 0 && (
        <ul className={styles.related}>
          {data.relatedDocs.map((doc, idx) => (
            <li key={idx}>
              <a href={doc.url} target="_blank" rel="noopener noreferrer">
                {doc.title}
              </a>
              {doc.type && <span className={styles.type}>{doc.type}</span>}
            </li>
          ))}
        </ul>
      )}
    </div>
  );
};
```
