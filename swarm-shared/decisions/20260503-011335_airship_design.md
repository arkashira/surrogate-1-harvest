## Final Consolidated Plan  
**Title**: Airship Design — Embed HF CDN Bypass + Deterministic Manifest + Studio Reuse into Surrogate-1 Pipeline  

**Goal (unified)**  
- Eliminate HF API 429s during training data loads.  
- Eliminate `pyarrow.CastError` by strict column projection and schema enforcement.  
- Reduce Lightning Studio quota burn via deterministic reuse and graceful fallback.  
- Enable frontend-driven manifest generation and training launch.  

**Scope (unified)**  
- A Mac-side (or any-dev) manifest generator that lists files once (HF API) and produces a CDN-only manifest.  
- A Lightning training script that streams exclusively from CDN URLs, projects `{prompt, response}`, and reuses or restarts a Studio deterministically.  
- A minimal safe writer for any future enriched writes (projected columns only).  
- Two frontend buttons: “Generate Training Manifest” and “Start Surrogate Training”.

## 1. Manifest Generator (single source of truth)

- Run after rate-limit window clears (or on schedule).  
- Uses `list_repo_tree(..., recursive=True)` once per date folder under `dataset-mirror/enriched/`.  
- Projects only `{prompt, response}` at parse time; ignores mixed schemas to avoid `pyarrow.CastError`.  
- Deterministically assigns each slug to one of 5 sibling repos (hash → repo) to respect HF 128 commits/hr cap.  
- Emits `manifest-{date}.json` with CDN URLs and sibling repo targets.

```python
# scripts/generate_training_manifest.py
#!/usr/bin/env python3
import json, hashlib, os
from pathlib import Path
from huggingface_hub import list_repo_tree

HF_REPO = "your-org/dataset-mirror"
DATE_FOLDER = "enriched/2026-04-29"  # parameterized in prod
OUTPUT_DIR = Path("manifests")
OUTPUT_DIR.mkdir(exist_ok=True)

SIBLING_REPOS = [
    "your-org/surrogate-data-0",
    "your-org/surrogate-data-1",
    "your-org/surrogate-data-2",
    "your-org/surrogate-data-3",
    "your-org/surrogate-data-4",
]

def pick_repo(slug: str) -> str:
    h = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    return SIBLING_REPOS[h % len(SIBLING_REPOS)]

def main(date_folder: str = DATE_FOLDER):
    manifest = {"date": date_folder.split("/")[-1], "repo": HF_REPO, "files": []}
    tree = list_repo_tree(repo_id=HF_REPO, path=date_folder, recursive=True)
    for item in tree:
        if item.rfilename.endswith(".parquet"):
            slug = Path(item.rfilename).stem
            cdn_url = (
                f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{item.rfilename}"
            )
            manifest["files"].append(
                {"slug": slug, "cdn_url": cdn_url, "sibling_repo": pick_repo(slug)}
            )
    out_path = OUTPUT_DIR / f"manifest-{manifest['date']}.json"
    with open(out_path, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest written to {out_path}")

if __name__ == "__main__":
    import sys
    main(sys.argv[1] if len(sys.argv) > 1 else DATE_FOLDER)
```

## 2. Lightning Training Script (CDN-only, Studio reuse)

- Accepts `--manifest` path.  
- Streams via CDN URLs with **no Authorization header** (eliminates 429s from training loads).  
- Projects `{prompt, response}` on read; uses `pyarrow` for fast, schema-safe projection.  
- Reuses a running Studio by name; if stopped, restarts on `L40S` (or free-tier fallback).  
- Avoids `load_dataset(streaming=True)` on heterogeneous repos to prevent schema cast errors.

```python

#!/usr/bin/env python3
import argparse, json, tempfile
from pathlib import Path
import pyarrow.parquet as pq
import requests
import torch
from torch.utils.data import IterableDataset, DataLoader
from lightning import LightningWork, LightningFlow, LightningApp

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path):
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.files = self.manifest["files"]

    def __iter__(self):
        for f in self.files:
            resp = requests.get(f["cdn_url"], timeout=30)
            resp.raise_for_status()
            with tempfile.NamedTemporaryFile(suffix=".parquet", delete=False) as tmp:
                tmp.write(resp.content)
                tmp_path = tmp.name
            try:
                table = pq.read_table(tmp_path, columns=["prompt", "response"])
                for row in table.to_pylist():
                    yield row["prompt"], row["response"]
            finally:
                Path(tmp_path).unlink(missing_ok=True)

class SurrogateTrainer(LightningWork):

        self.machine = machine

    def run(self):
        dataset = CDNParquetDataset(self.manifest_path)
        loader = DataLoader(dataset, batch_size=8, num_workers=2)
        for prompts, responses in loader:
            # Replace with surrogate forward/backward
            print(f"Batch: {len(prompts)} samples")
        print("Training complete")

class SurrogateFlow(LightningFlow):
    def __init__(self, manifest_path):
        super().__init__()
        self.manifest_path = manifest_path
        self.trainer = None

    def run(self):
        if self.trainer is None:
```

## 3. Safe Upload Writer (projected columns only)

- Writes only `{prompt, response}` to `batches/mirror-merged/{date}/{slug}.parquet`.  
- Uses sibling repo hashing to spread commits and stay within HF 128/hr cap.

```python
# scripts/upload_enriched.py
#!/usr/bin/env python3
import pyarrow as pa, pyarrow.parquet as pq, hashlib
from huggingface_hub import upload_file
from pathlib import Path

SIBLING_REPOS = [
    "your-org/surrogate-data-0",
    "your-org/surrogate-data-1",
    "your-org/surrogate-data-2",
    "your-org/surrogate-data-3",
    "your-org/surrogate-data-4",
]

def pick_repo(slug: str) -> str:
    h = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    return SIBLING_REPOS[h % len(SIBLING_REPOS)]

def upload_projected_parquet(source_path: str, date: str, slug: str):
    table = pq.read_table(source_path, columns=["prompt", "response"])
    out_path = f"/tmp/{slug}.parquet"
    pq.write_table(table, out_path)
    repo = pick_repo(slug)
    path_in_repo = f"batches/mirror-merged/{date}/{slug}.parquet"
    upload_file(path_or_fileobj=out_path, path_in_repo=path_in_repo, repo_id=repo)
    print(f"Uploaded {slug} -> {repo}/{path_in_repo}")
```
