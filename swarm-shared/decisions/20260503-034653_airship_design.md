# airship / design

## Implementation Plan (≤2h)

**Highest-value change**: Add CDN-only dataset loader + Lightning idle-resilient runner to `/opt/axentx/airship/surrogate`.

**Why**: Eliminates HF API 429s during training and prevents quota loss from idle timeouts — immediate reliability + cost savings.

---

### Steps

1. **Create file list utility** (Mac orchestration)  
   - Single `list_repo_tree` call for a date folder → save `train_files.json`  
   - Embed this JSON in training script

2. **CDN-only DataLoader**  
   - Use `https://huggingface.co/datasets/{repo}/resolve/main/{path}`  
   - Zero Authorization header, zero API calls during training  
   - Stream `parquet` with `pyarrow` and project `{prompt, response}` only

3. **Lightning idle-resilient runner**  
   - Check studio status before `.run()`  
   - Auto-restart if stopped  
   - Deterministic sibling-repo write for commits (hash slug → repo index)

4. **Integrate into surrogate training entrypoint**

### Code Snippets

#### 1) `tools/list_cdn_files.py` (run on Mac)
```python
#!/usr/bin/env python3
"""
Generate CDN file list for a date folder.
Run once per dataset update (after rate-limit window clears).
"""
import json
import os
import sys
from huggingface_hub import HfApi

REPO = os.getenv("HF_DATASET_REPO", "my-org/surrogate-data")
DATE_FOLDER = sys.argv[1] if len(sys.argv) > 1 else "2026-04-29"
OUT = sys.argv[2] if len(sys.argv) > 2 else "train_files.json"

api = HfApi()
tree = api.list_repo_tree(
    repo_id=REPO,
    path=DATE_FOLDER,
    recursive=False,
    repo_type="dataset",
)

files = [
    f.rfilename
    for f in tree
    if f.rfilename.endswith(".parquet")
]

with open(OUT, "w") as f:
    json.dump({"repo": REPO, "date": DATE_FOLDER, "files": files}, f, indent=2)

print(f"Wrote {len(files)} files to {OUT}")
```

#### 2) `surrogate/data/cdn_dataset.py`
```python
import json
import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset
from typing import Iterator

CDN_ROOT = "https://huggingface.co/datasets"

class CDNParquetDataset(IterableDataset):
    """
    CDN-only parquet loader.
    No HF API calls during training.
    """
    def __init__(self, file_list_path: str):
        with open(file_list_path) as f:
            meta = json.load(f)
        self.repo = meta["repo"]
        self.files = meta["files"]

    def _stream_file(self, path: str) -> Iterator[dict]:
        url = f"{CDN_ROOT}/{self.repo}/resolve/main/{path}"
        with requests.get(url, stream=True) as r:
            r.raise_for_status()
            with open("/tmp/current.parquet", "wb") as tmp:
                for chunk in r.iter_content(chunk_size=8192):
                    tmp.write(chunk)
            table = pq.read_table("/tmp/current.parquet")
            df = table.select(["prompt", "response"]).to_pandas()
            for _, row in df.iterrows():
                yield {"prompt": row["prompt"], "response": row["response"]}

    def __iter__(self) -> Iterator[dict]:
        for f in self.files:
            yield from self._stream_file(f)
```

#### 4) `surrogate/train/train.py` (entrypoint)
```python
# Lightweight orchestration script for Mac
# Usage: python train.py <date_folder>
import subprocess
import sys
from pathlib import Path

def main():
    date_folder = sys.argv[1] if len(sys.argv) > 1 else "2026-04-29"
    list_tool = Path(__file__).parent.parent / "tools" / "list_cdn_files.py"
    file_list = Path("train_files.json")

    # 1) Generate CDN file list (single API call)
    subprocess.run([sys.executable, str(list_tool), date_folder, str(file_list)], check=True)

    # 2) Launch Lightning training (CDN-only, idle-resilient)
    subprocess.run([
        sys.executable, "-m", "surrogate.train.lightning_runner",
        "--studio", "surrogate-cdn-train",
        "--files", str(file_list),
    ], check=True)

if __name__ == "__main__":
    main()
```
