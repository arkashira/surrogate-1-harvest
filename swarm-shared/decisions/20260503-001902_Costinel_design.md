# Costinel / design

## Final Implementation Plan — Costinel “Top-Hub Signal” Card (Read-Only)

**Scope**: Read-only frontend card (≤2h)  
**Principle**: “Sense + Signal — ไม่ Execute” (strictly no writes, no runtime mutations, no self-execution)  
**Goal**: Surface the most-connected hub (e.g., “MOC”) with actionable context, provenance, and audit trail.

---

### 1) High-value incremental improvement (merged)
Add a persistent, read-only “Top-Hub Signal” card to the dashboard that:
- Queries a **cached, read-only endpoint** for the highest-degree hub (server-side snapshot or live graph fallback).
- Renders card with hub label, short rationale, related docs count, **snapshot provenance**, and last-updated timestamp.
- Links to the hub detail view (read-only) and emits an **audit log entry on view** (write to audit log only — no graph mutation).
- Honors “Sense + Signal” by never performing writes or executing changes.
- Includes a **“Create Proposal” link** (opens proposal form in new route) — does not submit automatically.

---

### 2) Implementation steps (≤2h)

1. **Backend (API route)** — `/api/hubs/top` (GET)
   - Serve a **static snapshot** (`static/data/hubs/top-hub-snapshot.json`) for speed and safety.
   - Fallback (optional): call knowledge-rag client if snapshot missing (read-only, cached 5m).
   - Return `{ snapshotId, generatedAt, generatedBy, hub, related, lastUpdated }`.
   - Ensure endpoint is read-only and side-effect-free.

2. **Frontend component** — `components/TopHubSignalCard.tsx`
   - Fetch `/api/hubs/top` (SSR or client fetch with SWR, stale-while-revalidate, 5m cache).
   - Render card: icon, hub name, short rationale, related docs count, timestamp, snapshotId.
   - Link to `/hubs/{hub}` (read-only).
   - Include “Create Proposal” link (opens `/proposals/new?hub={hub}`).
   - On mount, call `/api/audit/log` with `{ event: "top_hub_signal_view", hub }` (write to audit log only — no graph mutation).

3. **Dashboard integration**
   - Place card in the primary dashboard grid (top-right or cost-summary sidebar).
   - Ensure responsive and accessible.

4. **Caching & rate limits**
   - Serve static snapshot with `Cache-Control: public, max-age=300, s-maxage=300`.
   - If knowledge-rag used as fallback, server-side cache 5m.
   - If unavailable, fallback to static placeholder with “Signal unavailable”.

5. **Audit trail**
   - Log views to audit store (append-only) for governance.

### 3) Code snippets (merged best parts)

#### Backend API (SvelteKit route)

```js
// /src/routes/api/hubs/top/+server.js
import { json } from '@sveltejs/kit';
import fs from 'fs';
import path from 'path';

const SNAPSHOT_PATH = path.resolve('static/data/hubs/top-hub-snapshot.json');

export async function GET() {
  try {
    const raw = fs.readFileSync(SNAPSHOT_PATH, 'utf8');
    const payload = JSON.parse(raw);

    if (!payload?.hub?.id || !payload?.snapshotId) {
      return json(
        { error: 'Invalid snapshot format' },
        { status: 500 }
      );
    }

    return json(payload, {
      headers: {
        'Cache-Control': 'public, max-age=300, s-maxage=300',
        'Content-Type': 'application/json'
      }
    });
  } catch (err) {
    return json(
      { error: 'Snapshot unavailable', details: err.message },
      { status: 503 }
    );
  }
}
```

#### Frontend component (Next.js)

```tsx
// /components/TopHubSignalCard.tsx
'use client';
import useSWR from 'swr';
import Link from 'next/link';
import { useEffect } from 'react';

const fetcher = (url: string) => fetch(url).then((r) => r.json());

export default function TopHubSignalCard() {
  const { data, error } = useSWR('/api/hubs/top', fetcher, {
    revalidateOnFocus: false,
    refreshInterval: 300000, // 5m
  });

  useEffect(() => {
    if (data?.hub?.id) {
      // Audit view (write to audit log only)
      fetch('/api/audit/log', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          event: 'top_hub_signal_view',
          hub: data.hub.id,
          ts: new Date().toISOString(),
        }),
      }).catch(() => {});
    }
  }, [data]);

  if (error || !data) {
    return (
      <div className="p-4 border rounded bg-gray-50 text-gray-500">
        Signal unavailable
      </div>
    );
  }

  const { hub, snapshotId, generatedAt, related } = data;

  return (
    <div className="p-4 border rounded bg-white shadow-sm">
      <div className="flex items-start justify-between">
        <div>
          <h3 className="text-sm font-medium text-gray-500">Top-Hub Signal</h3>
          <Link href={`/hubs/${hub.id}`} className="text-xl font-semibold text-blue-600 hover:underline">
            {hub.label}
          </Link>
          <p className="mt-1 text-sm text-gray-600">{hub.summary || 'No summary available'}</p>
          <p className="mt-2 text-xs text-gray-400">
            {hub.connections} connections · Score {hub.score} · Updated {new Date(generatedAt).toLocaleString()}
          </p>
          <p className="text-xs text-gray-300">Snapshot: {snapshotId}</p>
        </div>
      </div>
      <div className="mt-3 flex gap-2">
        <Link href={`/proposals/new?hub=${hub.id}`} className="text-sm text-blue-600 hover:underline">
          Create Proposal
        </Link>
      </div>
    </div>
  );
}
```
