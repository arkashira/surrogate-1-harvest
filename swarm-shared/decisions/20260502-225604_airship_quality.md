## Highest-Value Incremental Improvement (<2h)

**Goal**: Ship a zero-runtime-dependency, CDN-safe `airship discover` orchestrator (Bash-first) that deterministically emits `status.json` + `manifest.json` for local/CI use and CDN upload. Frontends fetch via CDN URLs (no HF API calls during runtime).

**Why this wins**:
- Eliminates HF API rate-limit risk during training/inference (uses CDN bypass).
- Produces deterministic file lists for reproducible surrogate training.
- Pure Bash → no Python runtime, no pip installs, runs on Mac/CI/Lightning orchestrator.
- Fits <2h: single script + unit test + README update.

---

## Implementation Plan

1. **Create `scripts/airship-discover.sh`**
   - Shebang `#!/usr/bin/env bash`
   - Accept repo root via `REPO_ROOT` or default to project root.
   - Walk `arkship/` and `surrogate/` for source files (`.py`, `.js`, `.ts`, `.md`, `.yml`, `.yaml`, `.json`).
   - Compute per-file:
     - `path` (relative to repo root)
     - `sha256` (content hash)
     - `size` (bytes)
     - `mtime` (ISO-8601)
   - Emit:
     - `status.json`: `{ "discovered_at": "...", "repo_root": "...", "total_files": N, "ok": true }`
     - `manifest.json`: sorted array of file objects above.
   - Deterministic sort by path.
   - Exit codes: 0=ok, 1=error, 2=partial.

2. **Create `scripts/test-airship-discover.sh`**
   - Run discover against a fixture directory.
   - Assert `status.json` contains `"ok": true`.
   - Assert `manifest.json` is valid JSON and non-empty.
   - Assert deterministic ordering (re-run produces identical manifest).

3. **Update `README.md`**
   - Add “Discovery” section with usage:
     ```bash
     bash scripts/airship-discover.sh
     ```
   - Note CDN upload path suggestion: `batches/mirror-merged/{date}/manifest.json`.

4. **Integrate into CI (optional, if time)**
   - Add step to run discovery and upload manifest to CDN (via `gh` CLI or `aws s3 cp` if keys present).

## Code Snippets

### `scripts/airship-discover.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

REPO_ROOT="${REPO_ROOT:-$(cd "$(dirname "$0")/.." && pwd)}"
OUT_DIR="${OUT_DIR:-${REPO_ROOT}/dist}"
TIMESTAMP=$(date -u +"%Y-%m-%dT%H:%M:%SZ")

mkdir -p "${OUT_DIR}"

# Extensions to include (surrogate training-relevant)
EXTENSIONS="\\( -name '*.py' -o -name '*.js' -o -name '*.ts' -o -name '*.md' -o -name '*.yml' -o -name '*.yaml' -o -name '*.json' \\)"

# Find files deterministically
mapfile -t FILES < <(find "${REPO_ROOT}" -type f -path '*/arkship/*' -o -path '*/surrogate/*' | \
  grep -E '\.(py|js|ts|md|yml|yaml|json)$' | \
  sort)

TOTAL=${#FILES[@]}
MANIFEST=()

for f in "${FILES[@]}"; do
  rel=$(realpath --relative-to="${REPO_ROOT}" "$f")
  sha=$(sha256sum "$f" | awk '{print $1}')
  size=$(stat -c%s "$f" 2>/dev/null || stat -f%z "$f" 2>/dev/null || echo 0)
  mtime=$(date -u -r "$f" +"%Y-%m-%dT%H:%M:%SZ" 2>/dev/null || echo "$TIMESTAMP")
  MANIFEST+=( "$(jq -n \
    --arg p "$rel" \
    --arg s "$sha" \
    --arg sz "$size" \
    --arg mt "$mtime" \
    '{path: $p, sha256: $s, size: ($sz | tonumber), mtime: $mt}')" )
done

# Build manifest JSON array
MANIFEST_JSON=$(printf '%s\n' "${MANIFEST[@]}" | jq -s 'sort_by(.path)')

# Status
STATUS_JSON=$(jq -n \
  --arg ts "$TIMESTAMP" \
  --arg root "$REPO_ROOT" \
  --argjson total "$TOTAL" \
  '{discovered_at: $ts, repo_root: $root, total_files: $total, ok: true}')

echo "$STATUS_JSON" > "${OUT_DIR}/status.json"
echo "$MANIFEST_JSON" > "${OUT_DIR}/manifest.json"

echo "Discovered ${TOTAL} files -> ${OUT_DIR}/status.json, ${OUT_DIR}/manifest.json"
```

### `scripts/test-airship-discover.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

SCRIPT_DIR=$(cd "$(dirname "$0")" && pwd)
REPO_ROOT=$(cd "${SCRIPT_DIR}/.." && pwd)
FIXTURE="${REPO_ROOT}/test-fixture"
OUT_DIR="${REPO_ROOT}/dist-test"

rm -rf "$FIXTURE" "$OUT_DIR"
mkdir -p "$FIXTURE/arkship" "$FIXTURE/surrogate"

echo "print('hello')" > "$FIXTURE/arkship/main.py"
echo "# docs" > "$FIXTURE/surrogate/README.md"

export REPO_ROOT="$FIXTURE"
export OUT_DIR="$OUT_DIR"

bash "${SCRIPT_DIR}/airship-discover.sh"

# Assertions
if [[ ! -f "${OUT_DIR}/status.json" ]]; then
  echo "FAIL: status.json missing"
  exit 1
fi

if [[ ! -f "${OUT_DIR}/manifest.json" ]]; then
  echo "FAIL: manifest.json missing"
  exit 1
fi

ok=$(jq -r '.ok' "${OUT_DIR}/status.json")
if [[ "$ok" != "true" ]]; then
  echo "FAIL: status.ok != true"
  exit 1
fi

total=$(jq -r '.total_files' "${OUT_DIR}/status.json")
if [[ "$total" -lt 2 ]]; then
  echo "FAIL: expected >=2 files, got $total"
  exit 1
fi

# Determinism check
cp "${OUT_DIR}/manifest.json" "${OUT_DIR}/manifest.json.1"
bash "${SCRIPT_DIR}/airship-discover.sh" > /dev/null
if ! diff -u "${OUT_DIR}/manifest.json.1" "${OUT_DIR}/manifest.json" > /dev/null; then
  echo "FAIL: manifest not deterministic"
  exit 1
fi

echo "PASS: all tests passed"
```
