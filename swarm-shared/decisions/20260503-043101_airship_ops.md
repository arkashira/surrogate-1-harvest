## Final Implementation Plan (≤2h)

**Goal:** Deterministic CDN file manifest + Lightning Studio lifecycle resilience for Surrogate-1 training.

### 1) Pre-list date folder → JSON manifest (Mac orchestration)
- Single `list_repo_tree(recursive=False)` for one date folder (e.g. `batches/mirror-merged/2026-05-03/`)
- Save manifest with **both** repo metadata **and** per-file CDN URLs to `manifests/2026-05-03.json`
- Commit manifest to repo (or upload to surrogate config) so training uses CDN-only fetches.

### 2) Lightning Studio lifecycle resilience
- Before `.run()`, check `Teamspace.studios` for existing Running studio with expected name; reuse if found.
- If studio exists but stopped, restart with `target.start(machine=Machine.L40S)` (or fallback to free-tier L40S).
- Wrap `.run()` in try/except; on idle-stop kill, restart and re-run.

### 3) Surrogate training script changes
- Accept `--manifest` (path to JSON) or env `SURROGATE_MANIFEST_PATH`.
- Data loader reads manifest, constructs CDN URLs, downloads via `requests`/`urllib` with streaming and retries (no HF API calls).
- Keep schema projection to `{prompt, response}` at parse time; attribution via filename pattern.

### 4) Cron/wrapper hygiene
- Ensure any wrapper scripts have `#!/usr/bin/env bash`, are `chmod +x`, invoked via `bash <script> "$@"`.
- Set `SHELL=/bin/bash` in crontab.

### manifest_generator.py (run on Mac)
```python
#!/usr/bin/env python3
"""
Generate CDN manifest for one date folder.
Usage: python manifest_generator.py --repo org/surrogate-mirror --date 2026-05-03 --out manifests/2026-05-03.json
"""
import argparse
import json
import os
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    api = HfApi()
    folder = f"batches/mirror-merged/{args.date}/"
    # Single non-recursive call
    tree = api.list_repo_tree(repo_id=args.repo, path=folder, recursive=False)
    files = [entry.path for entry in tree if entry.type == "file"]

    base_cdn = f"https://huggingface.co/datasets/{args.repo}/resolve/main/{folder}"
    manifest = {
        "repo": args.repo,
        "date": args.date,
        "folder": folder.rstrip('/'),
        "files": sorted(files),
        "base_cdn": base_cdn,
        "file_urls": [f"{base_cdn}/{f}" for f in sorted(files)]
    }

    os.makedirs(os.path.dirname(os.path.abspath(args.out)), exist_ok=True)
    with open(args.out, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```
