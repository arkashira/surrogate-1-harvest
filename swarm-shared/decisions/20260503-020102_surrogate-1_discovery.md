## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID` and `SHARD_TOTAL` (16) from the GitHub Actions matrix.
- Loads a pre-generated `manifest-YYYYMMDD.json` (created once per date folder by a Mac helper).
- Uses **HF CDN bypass** (`https://huggingface.co/datasets/.../resolve/main/...`) for zero-API data fetches during ingest.
- Projects each file to `{prompt, response}` only at parse time (avoids PyArrow CastError on mixed schemas).
- Deduplicates via the existing `lib/dedup.py` central md5 store.
- Writes shard output to `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl` and streams upload via `huggingface_hub` with deterministic repo selection to respect HF commit cap.

### Steps (1h 30m)

1. **Create manifest generator** (`tools/build-manifest.py`) — run once on Mac (15m)
   - Uses `list_repo_tree(path, recursive=False)` per date folder.
   - Saves `manifest-YYYYMMDD.json` with `{"repo": "...", "files": ["path1", ...]}`.

2. **Rewrite worker** (`bin/dataset-enrich.py`) (45m)
   - Shebang `#!/usr/bin/env python3` (keeps cron compatibility).
   - Parse `SHARD_ID`, `SHARD_TOTAL`, `DATE`, `MANIFEST`.
   - Assign files by `hash(slug) % SHARD_TOTAL == SHARD_ID`.
   - For each file:
     - Build CDN URL and stream download with `requests.get(..., stream=True)`.
     - Parse line-by-line (JSONL/parquet) and project `{prompt, response}`.
     - Compute md5, call `lib/dedup.py` to check/insert.
     - Collect accepted pairs.
   - Deterministic repo selection: `repo_suffix = hash(slug) % 5` → `axentx/surrogate-1-training-pairs-shard{repo_suffix}` (or primary if single repo).
   - Stream upload via `huggingface_hub` `upload_file` in append mode to `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.

3. **Update workflow** (`.github/workflows/ingest.yml`) (15m)
   - Add `DATE` input (defaults to today).
   - Pass `MANIFEST` artifact generated by a prior job or commit.
   - Keep 16-shard matrix.

4. **Test locally** (15m)
   - Run worker with small manifest subset.
   - Verify CDN fetch, dedup, upload.

### `tools/build-manifest.py`
```python
#!/usr/bin/env python3
"""
Generate manifest for a date folder.
Usage: python3 tools/build-manifest.py --repo axentx/surrogate-1-training-pairs --date 20260503 --out manifest-20260503.json
"""
import argparse
import json
import os
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", default="axentx/surrogate-1-training-pairs")
    parser.add_argument("--date", required=True, help="YYYYMMDD")
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    api = HfApi()
    prefix = f"batches/public-merged/{args.date}/"
    # Non-recursive per-folder listing to avoid pagination explosion
    tree = api.list_repo_tree(repo_id=args.repo, path=prefix, recursive=False)
    files = [item.rfilename for item in tree if item.rfilename.endswith(('.jsonl', '.parquet'))]

    manifest = {
        "repo": args.repo,
        "date": args.date,
        "prefix": prefix,
        "files": sorted(files)
    }
    with open(args.out, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```

def shard_for_file(filepath: str, total: int) -> int:
    slug = Path(filepath).stem
    return int(hashlib.md5(slug.encode()).hexdigest(), 16) % total

def parse_pair_from_record(record):
    """Project heterogeneous record to {prompt, response}."""
    prompt = record.get("prompt") or record.get("input") or record.get("question") or ""
    response = record.get("response") or record.get("output") or record.get("answer") or ""
    return str(prompt).strip(), str(response).strip()

def compute_md5_for_pair(prompt: str, response: str) -> str:
    return hashlib.md5(f"{prompt}\n{response}".encode()).hexdigest()

def stream_cdn_jsonl(cdn_url: str):
    with requests.get(cdn_url, stream=True, timeout=60) as r:
        r.raise_for_status()
        for line in r.iter_lines(decode_unicode=True):
            if line:
                yield json.loads(line)
