# workio / design

## Final Synthesis — One Correct, Actionable Plan

**Scope**: `workio/src/features/clock/`, `workio/src/lib/api.ts`, `workio/server/src/routes/clock.ts`, `workio/server/src/db/`  
**Goal**: eliminate duplicate clock records, make retries safe, and improve perceived performance with optimistic UI and resilient location handling.

---

## 1. Diagnosis (resolved)

- **Race/duplicate root cause**: no per-user/day uniqueness at DB layer + no idempotency key on client/server.
- **Perceived latency/retries**: no optimistic UI and no in-flight request coordination.
- **Location failures**: permission/GPS failures silently fail; no graceful fallback or recovery UX.
- **Retry safety**: backend accepts duplicates; no idempotency table or key enforcement.

---

## 2. Implementation (single, concrete plan)

### 2.2 Backend: idempotency middleware + clock routes

`workio/server/src/middleware/idempotency.ts`

```ts
import { Request, Response, NextFunction } from 'express';
import { db } from '../db';

export async function idempotencyGuard(req: Request, res: Response, next: NextFunction) {
  const key = req.headers['idempotency-key'] as string;
  const user = (req as any).user; // set by auth middleware
  if (!key || !user?.id) return next();

  const exists = await db.oneOrNone(
    'SELECT 1 FROM clock_idempotency WHERE idempotency_key = $1 AND user_id = $2',
    [key, user.id]
  );

  if (exists) {
    // Best-effort: return prior successful response if you can store/fetch it.
    // For immediate safety, 409 prevents duplicates.
    return res.status(409).json({ error: 'Duplicate request' });
  }

  await db.none('INSERT INTO clock_idempotency (idempotency_key, user_id) VALUES ($1, $2)', [
    key,
    user.id,
  ]);
  (req as any).idempotencyKey = key;
  next();
}
```

`workio/server/src/routes/clock.ts`

```ts
import express from 'express';
import { idempotencyGuard } from '../middleware/idempotency';
import { db } from '../db';

const router = express.Router();

router.post('/in', idempotencyGuard, async (req, res) => {
  const user = req.user;
  const { location } = req.body; // { lat, lng } | null
  const today = new Date().toISOString().slice(0, 10);

  try {
    const result = await db.one(
      `INSERT INTO attendance_records (user_id, clock_date, clock_in, location)
       VALUES ($1, $2, NOW(), $3)
       RETURNING *`,
      [user.id, today, location]
    );
    return res.json({ ok: true, record: result });
  } catch (err: any) {
    // Unique violation from idx_user_date_clock or idempotency table race
    if (err.code === '23505') {
      const existing = await db.oneOrNone(
        'SELECT * FROM attendance_records WHERE user_id = $1 AND clock_date = $2',
        [user.id, today]
      );
      return res.status(409).json({ ok: false, duplicate: true, record: existing });
    }
    return res.status(500).json({ error: 'Server error' });
  }
});

router.post('/out', idempotencyGuard, async (req, res) => {
  const user = req.user;
  const { location } = req.body;
  const today = new Date().toISOString().slice(0, 10);

  try {
    const result = await db.one(
      `UPDATE attendance_records
       SET clock_out = NOW(), location_out = $1
       WHERE user_id = $2 AND clock_date = $3 AND clock_out IS NULL
       RETURNING *`,
      [location, user.id, today]
    );
    return res.json({ ok: true, record: result });
  } catch (err: any) {
    if (err.code === '23505') {
      return res.status(409).json({ ok: false, duplicate: true });
    }
    return res.status(500).json({ error: 'Server error' });
  }
});

export default router;
```

- **Correctness**: DB unique constraint is authoritative; idempotency table prevents duplicate processing of retries/webhooks.
- **Actionability**: middleware is reusable; routes are explicit and safe.
