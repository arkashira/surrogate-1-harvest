# Costinel / design

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Scope**: Add a lightweight, non-blocking Top-Hub Signal Panel to Costinel frontend that surfaces the most-connected hub (e.g., "MOC") using CDN-first data (zero runtime HF API). Deterministic, cache-friendly, and deployable in <2 hours.

---

### 1) Architecture (CDN-first)

- **Data source**: Public HF dataset repo `axentx/costinel-hubs` (or local mirror) with folder-per-date:  
  `batches/top-hub/{date}/top_hub.json`
- **File shape** (minimal):
  ```json
  {
    "hub": "MOC",
    "score": 0.94,
    "date": "2026-05-03",
    "tags": ["knowledge-rag", "graph", "hub"],
    "insight": "Review MOC before planning tasks — highest betweenness in cost-governance graph."
  }
  ```
- **Delivery**: CDN URL — `https://huggingface.co/datasets/axentx/costinel-hubs/resolve/main/batches/top-hub/{date}/top_hub.json`  
  (No Authorization header; bypasses HF API rate limits.)
- **Fallback**: Local static JSON bundled at build time (`public/data/top_hub_fallback.json`) if CDN fails or date not found.
- **Caching**: `Cache-Control: public, max-age=3600` via CDN; client-side `localStorage` keyed by date to avoid refetch within same UTC day.

#### C) Add React signal panel component — `src/components/TopHubSignalPanel.tsx`
```tsx
import { useEffect, useState } from "react";
import "./TopHubSignalPanel.css";

interface TopHub {
  hub: string;
  score: number;
  date: string;
  tags: string[];
  insight: string;
}

const CDN_ROOT = "https://huggingface.co/datasets/axentx/costinel-hubs/resolve/main";
const FALLBACK = "/data/top_hub_fallback.json";

function getCacheKey(date: string) {
  return `costinel:top_hub:${date}`;
}

export default function TopHubSignalPanel() {
  const [hub, setHub] = useState<TopHub | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    const today = new Date().toISOString().slice(0, 10);
    const cacheKey = getCacheKey(today);

    // 1) Try localStorage same-day cache
    const cached = localStorage.getItem(cacheKey);
    if (cached) {
      try {
        setHub(JSON.parse(cached));
        setLoading(false);
        return;
      } catch {
        localStorage.removeItem(cacheKey);
      }
    }

    // 2) Try CDN (no auth)
    fetch(`${CDN_ROOT}/batches/top-hub/${today}/top_hub.json`, { cache: "no-cache" })
      .then((res) => {
        if (!res.ok) throw new Error("CDN miss");
        return res.json();
      })
      .then((data) => {
        localStorage.setItem(cacheKey, JSON.stringify(data));
        setHub(data);
      })
      .catch(() => {
        // 3) Fallback to bundled file
        return fetch(FALLBACK).then((res) => res.json());
      })
      .then((data) => {
        setHub(data);
        // still cache fallback for the day to avoid repeated fallback fetches
        localStorage.setItem(cacheKey, JSON.stringify(data));
      })
      .catch(() => {
        // silent fail
        setHub(null);
      })
      .finally(() => setLoading(false));
  }, []);

  if (loading) {
    return (
      <div className="top-hub-panel loading">
        <div className="skeleton" />
      </div>
    );
  }

  if (!hub) return null;

  return (
    <div className="top-hub-panel" title={hub.insight}>
      <div className="top-hub-header">
        <span className="top-hub-badge">Top Hub</span>
        <span className="top-hub-date">{hub.date}</span>
      </div>
      <div className="top-hub-body">
        <div className="top-hub-name">{hub.hub}</div>
        <div className="top-hub-score">{Math.round(hub.score * 100)}% relevance</div>
        <div className="top-hub-tags">
          {hub.tags.map((t) => (
            <span key={t} className="top-hub-tag">
              #{t}
            </span>
          ))}
        </div>
      </div>
      <div className="top-hub-insight">{hub.insight}</div>
    </div>
  );
}
```
