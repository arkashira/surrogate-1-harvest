## Final Synthesized Implementation (Best-of-Both)

**Goal**: Eliminate HF API 429s and Lightning quota waste during Surrogate training by implementing **CDN-first deterministic ingestion with embedded file lists and Lightning Studio reuse**.

**Why this now**:  
- Surrogate training risks HF API 429 (1000 req/5min) during data loading.  
- Lightning Studio recreation burns quota on idle spin-up.  
- CDN downloads (`/resolve/main/`) bypass auth and have much higher rate limits.  
- A single deterministic file-list + Studio reuse removes both failure modes with minimal code.

---

## Concrete Implementation Plan (≤2h)

1. **Deterministic file-listing script** (run on Mac orchestration node or CI, once per date folder)  
   - Use `list_repo_tree(path, recursive=False)` per date folder (avoids recursive pagination).  
   - Save `file_list.json` with relative paths and expected CDN URLs.  
   - Commit to repo or embed in training artifact.

2. **CDN-only dataset loader** (used in Lightning training)  
   - No `load_dataset` or `list_repo_files` during training.  
   - Read `file_list.json`, stream via `requests.get(cdn_url, timeout=60)`.  
   - Parse each file individually; project to `{prompt, response}` only.  
   - Retry 429/5xx with exponential backoff; respect 360s wait after 429.

3. **Lightning Studio reuse utility**  
   - Before `Studio(create_ok=True)`, list `Teamspace.studios`.  
   - If a running studio with target name exists, reuse it.  
   - If stopped, restart with `target.start(machine=Machine.L40S)` (avoid idle timeout kills).  
   - Pass `file_list.json` as run argument.

4. **Wire into surrogate training entrypoint**  
   - CLI: `python train.py --date-folder 2026-05-03 --studio-name surrogate-l40s-01`.  
   - Mac orchestrator: runs step 1 once, then triggers Lightning run with reuse.

5. **Validation**  
   - Dry-run ingestion on small folder; confirm zero HF API calls during data load.  
   - Confirm Studio reuse and no duplicate spin-ups.

### 1) Deterministic file lister (run on Mac orchestrator)

```python
# tools/list_date_folder.py
import json
import os
import time
from huggingface_hub import HfApi

HF_REPO = "datasets/axentx/surrogate-mirror"
OUT_DIR = "file_lists"

api = HfApi()

def list_date_folder(date_folder: str):
    """List one date folder non-recursively and save CDN file list."""
    entries = api.list_repo_tree(
        repo_id=HF_REPO,
        path=date_folder,
        repo_type="dataset",
        recursive=False,  # avoid pagination explosion
    )

    files = []
    for e in entries:
        if e.type == "file":
            cdn_url = f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{e.path}"
            files.append({
                "path": e.path,
                "cdn_url": cdn_url,
                "size": getattr(e, "size", None),
            })

    os.makedirs(OUT_DIR, exist_ok=True)
    out_path = os.path.join(OUT_DIR, f"{date_folder.replace('/', '_')}_file_list.json")
    with open(out_path, "w") as f:
        json.dump(files, f, indent=2)
    print(f"Wrote {len(files)} files -> {out_path}")
    return out_path

if __name__ == "__main__":
    import sys
    date_folder = sys.argv[1] if len(sys.argv) > 1 else "2026-05-03"
    list_date_folder(date_folder)
```

# surrogate/data/cdn_loader.py
import json
import time
import requests
from typing import List, Dict, Iterator
from datasets import Dataset

CDN_TIMEOUT = 60
RETRY_BACKOFF = [1, 2, 4, 8, 16]

def load_file_list(file_list_path: str) -> List[Dict]:
    with open(file_list_path) as f:
        return json.load(f)

def robust_get(url: str) -> bytes:
    for attempt, wait in enumerate(RETRY_BACKOFF):
        try:
            resp = requests.get(url, timeout=CDN_TIMEOUT)
            if resp.status_code == 429:
                # HF CDN 429: wait 360s as per pattern
                print("CDN 429, waiting 360s")
                time.sleep(360)
                continue
            resp.raise_for_status()
            return resp.content
        except Exception as exc:
            if attempt == len(RETRY_BACKOFF) - 1:
                raise
            print(f"Retry {attempt+1} for {url}: {exc}")
            time.sleep(wait)
    raise RuntimeError("Unreachable")

def parse_to_pair(raw: bytes, path: str) -> Dict:
    # Placeholder: implement per your file format (jsonl, parquet, etc.)
    # Must return {"prompt": ..., "response": ...}
    # For jsonl lines:
    import io, json
    lines = io.StringIO(raw.decode("utf-8", errors="replace"))
    # Take first valid line as example; adapt to your schema
    for line in lines:
        line = line.strip()
        if not line:
            continue
        obj = json.loads(line)
        return {
            "prompt": obj.get("prompt") or obj.get("input") or "",
            "response": obj.get("response") or obj.get("output") or "",
        }
    return {"prompt": "", "response": ""}

def build_cdn_dataset(file_list_path: str) -> Dataset:
    items = []
    files = load_file_list(file_list_path)
    for meta in files:
        try:
            raw = robust_get(meta["cdn_url"])
            pair = parse_to_pair(raw, meta["path"])
            items.append(pair)
        except Exception as exc:
            print(f"Skip {meta['path']}: {exc}")
            continue
    return Dataset.from_list(items)

### 3) Lightning Studio reuse utility

```python
# surrogate/lightning/studio_reuse.py
from lightning_sdk import Teamspace, Studio, Machine
from typing import Optional

def get_or_create_studio(
    studio_name: str,
    machine: Machine = Machine.L40S,
    teamspace: Optional[str] = None,
) -> Studio:
    ts = Teamspace(teamspace) if teamspace else Teamspace()
    running = [s for s in ts.studios if s.name == studio_name and s.status == "Running"]
    if running:
        print(f"Reusing running studio: {studio_name}")
        return running[0]

    stopped = [s for s in ts.studios if s.name == studio_name and s.status == "Stopped"]
    if stopped:
        print(f"Restarting stopped studio: {studio_name}")
        s = stopped[0]
        s.start(machine=machine)
        return s

    print(f"Creating new studio: {studio_name}")
    return Studio(
        name=studio_name,
        machine=machine,
        create_ok=True,
        teamspace=teamspace,
    )
```
