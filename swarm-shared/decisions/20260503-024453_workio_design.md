# workio / design

### Diagnosis (merged)
- **Missing DB-level uniqueness** allows duplicates from LINE retries, client double-taps, and concurrent requests.  
- **Application-level “last punch” check is racy** (read-then-insert) under concurrency and webhook retries.  
- **No idempotency key plumbing** on API/webhook handlers prevents safe deduplication of retries (timeouts, 5xx, slow ACK).  
- **Frontend lacks immediate feedback and client-side idempotency**, encouraging double-taps and worsening perceived latency.  
- **Missing compound index** causes unnecessary scans for uniqueness checks.

---

### Proposed change (merged)
- **Scope**:  
  - `workio/server/src/db/schema.sql` — add compound unique constraint + index.  
  - `workio/server/src/middlewares/idempotencyMiddleware.js` — robust idempotency middleware.  
  - `workio/server/src/controllers/punchController.js` / `workio/server/src/routes/line.ts` — use idempotency key and upsert-safe insert.  
  - `workio/frontend/src/components/PunchButton.js` — generate idempotency key, optimistic UI, and immediate feedback.  
- **Goal**: Make duplicates impossible at DB level, idempotent at API level, and provide fast-path dedupe + good UX.

```sql
ALTER TABLE punches
  ADD COLUMN IF NOT EXISTS idempotency_key VARCHAR(255);

-- Enforce uniqueness at DB level for business-logic duplicates
ALTER TABLE punches
  ADD CONSTRAINT punches_user_tenant_date_type_unique
  UNIQUE (user_id, tenant_id, punch_date, punch_type);

-- Enforce uniqueness for idempotency (prevents replay of identical request payloads)
ALTER TABLE punches
  ADD CONSTRAINT punches_idempotency_key_unique
  UNIQUE (idempotency_key);

-- Index to accelerate lookups and constraint checks
CREATE INDEX IF NOT EXISTS idx_punches_user_tenant_date
  ON punches (user_id, tenant_id, punch_date);
```

#### 2) Idempotency middleware (corrected + complete)
```javascript
// workio/server/src/middlewares/idempotencyMiddleware.js
const Punch = require('../models/punch');

const idempotencyMiddleware = async (req, res, next) => {
  const idempotencyKey = req.header('X-Idempotency-Key');
  if (!idempotencyKey) {
    return res.status(400).json({ error: 'Missing X-Idempotency-Key' });
  }

  try {
    // Check for previous request with same idempotency key
    const existing = await Punch.findOne({ where: { idempotency_key: idempotencyKey } });
    if (existing) {
      // Return original response metadata to make retries safe and idempotent
      return res.status(200).json({ ok: true, deduped: true, punch: existing });
    }

    // Attach key to request for downstream handlers to use on insert
    req.idempotencyKey = idempotencyKey;
    next();
  } catch (err) {
    console.error('Idempotency check failed', err);
    // Fail open to avoid blocking legitimate requests; rely on DB constraint as final guard
    req.idempotencyKey = idempotencyKey;
    next();
  }
};

module.exports = idempotencyMiddleware;
```

#### 3) Punch controller (HTTP) — upsert-safe + idempotency
```javascript
// workio/server/src/controllers/punchController.js
const Punch = require('../models/punch');

exports.createPunch = async (req, res) => {
  const { user_id, tenant_id, punch_type } = req.body;
  const idempotencyKey = req.idempotencyKey || req.header('X-Idempotency-Key');
  const punchDate = new Date().toISOString().split('T')[0]; // YYYY-MM-DD

  if (!['clock_in', 'clock_out'].includes(punch_type)) {
    return res.status(400).json({ error: 'Invalid punch_type' });
  }

  try {
    // Try insert; DB unique constraints prevent duplicates
    const result = await Punch.create({
      user_id,
      tenant_id,
      punch_date: punchDate,
      punch_type,
      idempotency_key: idempotencyKey,
      created_at: new Date(),
    });

    return res.status(201).json({ ok: true, punch: result });
  } catch (err) {
    // If unique violation, treat as idempotent success
    const isUniqueViolation = err.name === 'SequelizeUniqueConstraintError';
    if (isUniqueViolation) {
      const existing = await Punch.findOne({
        where: { user_id, tenant_id, punch_date: punchDate, punch_type },
      });
      return res.status(200).json({ ok: true, deduped: true, punch: existing });
    }

    console.error('Create punch error', err);
    return res.status(500).json({ error: 'Internal server error' });
  }
};
```

#### 4) LINE webhook handler (idempotent + upsert)
```typescript
// workio/server/src/routes/line.ts
import { Request, Response } from 'express';
import { db } from '../db';
import { v4 as uuidv4 } from 'uuid';

const recentKeys = new Set<string>();

export async function handleLineWebhook(req: Request, res: Response) {
  try {
    const idempotencyKey = req.header('X-Idempotency-Key') || uuidv4();
    if (recentKeys.has(idempotencyKey)) {
      return res.status(200).json({ ok: true, deduped: true });
    }

    const events = req.body.events || [];
    for (const ev of events) {
      if (ev.type !== 'message' || ev.message?.type !== 'text') continue;

      const userId = ev.source.userId;
      const tenantId = process.env.DEFAULT_TENANT_ID || 'default';
      const punchDate = new Date().toISOString().split('T')[0];
      const text = ev.message.text.trim().toLowerCase();

      const punchType = text === 'in' || text === 'clock in' ? 'clock_in'
        : text === 'out' || text === 'clock out' ? 'clock_out'
        : null;

      if (!punchType) continue;

      // Upsert-safe: rely on DB unique constraint to dedupe
      await db.query(
        `INSERT INTO punches (user_id, tenant_id, punch_date, punch_type, created_at, idempotency_key, location)
         VALUES ($1, $2, $3, $4, NOW(), $5, $6)
         ON CONFLICT (user_id, tenant_id, punch_date, punch_type) DO NOTHING`,
        [userId, tenantId, punchDate, punchType, idempotencyKey, ev.message.text]
      );
    }

    recentKeys.add(idempotencyKey);
    setTimeout(() => recentKeys.delete(idempotencyKey), 60_000);

    res.status(200).json({ ok: true });
  } catch (err) {
    console.error('LINE webhook error', err);
    res.status(500).json({ error: 'Internal server error' });
  }
}
```
