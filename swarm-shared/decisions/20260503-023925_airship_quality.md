## Highest-Value Incremental Improvement (≤2h)

**What**: Add a CDN-first training slice to Surrogate frontend that:
1. Exposes a small UI + API to trigger “list one date folder from HF” (Mac orchestration) → saves `file-list.json`
2. Embeds that list into the Lightning training script so training does **zero HF API calls** during data loading (CDN-only fetches)
3. Reuses a running Lightning Studio and tolerates idle-stop by auto-restarting the machine if stopped

**Why**: Eliminates HF API 429 during training, cuts quota burn, and enables deterministic/repeatable surrogate training with Lightning Studio reuse and idle-stop resilience.

**Scope**: `/opt/axentx/airship/surrogate` (frontend + training orchestration)

## Implementation Plan (≤2h)

1. **Add API endpoint** (`/api/training/file-list`) on Surrogate backend (FastAPI) to:
   - Accept `repo`, `date_path`
   - Call HF `list_repo_tree(path, recursive=False)` once
   - Return JSON list of file paths
   - Save to `training/file-list-{date}.json`

2. **Add minimal frontend UI** (`/training`) with:
   - Inputs: repo (default: datasets repo), date path (e.g., `2026-05-03`)
   - Button: “List HF folder (Mac)”
   - Display file list and “Generate training script”

3. **Add training script template** (`training/train_cdn.py`) that:
   - Reads embedded `file-list.json`
   - Uses CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with `requests`/`aiohttp` + `datasets` `load_from_disk` or custom iterable
   - Projects to `{prompt, response}` only at parse time
   - Uses Lightning `Studio` with reuse logic + idle-stop resilience

4. **Add orchestration helper** (`training/orchestrate.py`) to:
   - Fetch file list via API (or run directly on Mac)
   - Embed file list into `train_cdn.py` (inline JSON or copy)
   - Start/reuse Lightning Studio and run training

5. **Harden for idle-stop**:
   - Before `.run()`, check studio status; if stopped, restart with `target.start(machine=Machine.L40S)`
   - Use `lightning-lambda-prod` cloud for H200 when available; fallback to `lightning-public-prod` (L40S)

6. **Test locally** (Mac orchestration only; training on Lightning).

### 1. Backend API (FastAPI) — `surrogate/api/training.py`

```python
# surrogate/api/training.py
import os
import json
from datetime import datetime
from pathlib import Path
from typing import List

from fastapi import APIRouter, HTTPException
from huggingface_hub import HfApi, list_repo_tree

router = APIRouter(prefix="/api/training", tags=["training"])
HF_API = HfApi()

BASE_DIR = Path(__file__).parent.parent.parent
LIST_DIR = BASE_DIR / "training" / "file_lists"
LIST_DIR.mkdir(parents=True, exist_ok=True)

@router.post("/file-list")
async def list_hf_folder(repo: str, date_path: str):
    """
    List one date folder from a HF dataset repo (non-recursive).
    Returns file list and saves to training/file-lists/{repo}-{date}.json
    """
    try:
        tree = list_repo_tree(repo=repo, path=date_path, recursive=False)
        files = [f.rfilename for f in tree if f.type == "file"]
    except Exception as exc:
        raise HTTPException(status_code=502, detail=f"HF list failed: {exc}")

    payload = {
        "repo": repo,
        "date_path": date_path,
        "listed_at": datetime.utcnow().isoformat() + "Z",
        "files": files,
    }

    safe_name = repo.replace("/", "_")
    out_path = LIST_DIR / f"{safe_name}-{date_path.replace('/', '_')}.json"
    out_path.write_text(json.dumps(payload, indent=2))

    return payload
```

```vue
<!-- surrogate/frontend/src/pages/TrainingPage.vue -->
<template>
  <div class="p-6 max-w-2xl">
    <h1 class="text-2xl font-bold mb-4">Surrogate CDN Training</h1>

    <div class="space-y-4">
      <div>

        <h2 class="font-semibold mb-2">Files ({{ fileList.length }})</h2>
        <ul class="max-h-60 overflow-auto border rounded p-2 text-sm">
          <li v-for="f in fileList" :key="f">{{ f }}</li>
        </ul>
        <button @click="generateScript" class="mt-2 px-3 py-1 bg-green-600 text-white rounded text-sm">
          Generate train_cdn.py
        </button>
      </div>

      <div v-if="script" class="mt-4">
        <h2 class="font-semibold mb-2">train_cdn.py</h2>

      </div>
    </div>
  </div>

<script setup>
import { ref } from "vue";

const repo = ref("datasets/your-repo");
const datePath = ref("2026-05-03");
const loading = ref(false);
const fileList = ref([]);
const script = ref("");

async function listFolder() {
  loading.value = true;
  try {
    const res = await fetch("/api/training/file-list", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ repo: repo.value, date_path: datePath.value }),
    });
    const data = await res.json();
    if (!res.ok) throw new Error(data.detail || "Failed");
    fileList.value = data.files || [];
  } catch (err) {
    alert(err.message);
  } finally {
    loading.value = false;
  }
}

function generateScript() {
  const filesJson = JSON.stringify(fileList.value, null, 2);
  script.value = `# Auto-generated CDN training script
# Repo: ${repo.value}
# Date path: ${datePath.value}

import json
import os
from pathlib import Path

HF_REPO = "${repo.value}"
FILE_LIST = ${filesJson}

CDN_BASE = f"https://huggingface.co/datasets/{{HF_REPO}}/resolve/main"

def cdn_url(path):
    return f"{CDN_BASE}/{path}"

# Example: stream from CDN using datasets or requests + iterator
# This file is meant to be used by training/orchestrate.py and Lightning Studio.
`;
}
</script>
```
