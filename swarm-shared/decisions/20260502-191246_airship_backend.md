# airship / backend

## Implementation Plan — `airship discover` (Zero-config strategic context CLI)

**Goal**: Ship a single Bash CLI (`airship discover`) in <2h that operationalizes past patterns:
- Run business research (`granite-business-research.sh`)
- Query knowledge-rag for top hub + related docs
- Surface strategic context for Arkship/Surrogate platform decisions

**Scope**: Backend orchestration only (no UI). Outputs structured JSON + human-readable summary to stdout. Designed to be invoked from repo root (`/opt/axentx/airship`).

---

### 1. High-level flow
1. `airship discover` (Bash wrapper) validates environment and invokes Python orchestrator.
2. Python orchestrator:
   - Executes `granite-business-research.sh` (if present) and captures findings.
   - Calls knowledge-rag (Neo4j + vector) to fetch:
     - Most-connected hub (e.g., "MOC") and its properties.
     - Top 5 related docs/insights for that hub.
   - Projects results into `{timestamp, hub, insights, research_findings, recommendations}`.
3. Outputs:
   - JSON to stdout (machine-readable).
   - Pretty-printed summary to stderr (human-readable).

### 3. Implementation details

#### `bin/airship` (Bash wrapper)
```bash
#!/usr/bin/env bash
# bin/airship — airship CLI dispatcher
set -euo pipefail

ROOT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
export AIRSHIP_ROOT="${AIRSHIP_ROOT:-$ROOT_DIR}"
export PYTHONPATH="${AIRSHIP_ROOT}/lib:${PYTHONPATH}"

COMMAND="${1:-help}"
shift || true

case "$COMMAND" in
  discover)
    exec python3 "${AIRSHIP_ROOT}/lib/discover.py" "$@"
    ;;
  help|--help|-h)
    echo "Usage: airship <command> [options]"
    echo "Commands:"
    echo "  discover   Run zero-config strategic discovery (business research + knowledge-rag + top-hub)"
    exit 0
    ;;
  *)
    echo "Unknown command: $COMMAND" >&2
    exit 1
    ;;
esac
```

Make executable:
```bash
chmod +x bin/airship
```

def run_business_research(script_path: str) -> Dict[str, Any]:
    """Run granite-business-research.sh and capture findings."""
    if not os.path.exists(script_path):
        return {"status": "skipped", "reason": "script not found", "path": script_path}

    if not os.access(script_path, os.X_OK):
        return {"status": "error", "reason": "script not executable", "path": script_path}

    try:
        result = subprocess.run(
            ["/bin/bash", script_path],
            capture_output=True,
            text=True,
            timeout=120,
            cwd=os.getenv("AIRSHIP_ROOT"),
        )
        return {
            "status": "ok" if result.returncode == 0 else "nonzero_exit",
            "returncode": result.returncode,
            "stdout": result.stdout.strip(),
            "stderr": result.stderr.strip(),
        }
    except subprocess.TimeoutExpired:
        return {"status": "error", "reason": "timeout", "path": script_path}
    except Exception as exc:
        return {"status": "error", "reason": str(exc), "path": script_path}


def query_top_hub_neo4j(uri: str, user: str, password: str) -> Dict[str, Any]:
    """Query Neo4j for the most-connected hub (e.g., highest degree) and related docs."""
    if not NEO4J_AVAILABLE:
        return {"status": "unavailable", "reason": "neo4j driver not available"}

    try:
        driver = GraphDatabase.driver(uri, auth=(user, password))
        with driver.session() as session:
            # Find top hub by relationship count (generic)
            rec = session.run("""
                MATCH (h)
                OPTIONAL MATCH (h)-[r]-()
                WITH h, count(r) AS degree
                ORDER BY degree DESC
                LIMIT 1
                RETURN h.name AS name, labels(h) AS labels, properties(h) AS props, degree
            """).single()

            if not rec:
                return {"status": "no_hubs_found"}

            hub = {
                "name": rec.get("name"),
                "labels": rec.get("labels"),
                "props": {k: v for k, v in rec.get("props", {}).items() if k != "embedding"},
                "degree": rec.get("degree"),
            }

            # Top 5 related docs (by connections to hub)
            related = session.run("""
                MATCH (h)-[]-(doc)
                WHERE h.name = $name
                RETURN doc.name AS name, labels(doc) AS labels, properties(doc) AS props
                LIMIT 5
            """, name=rec.get("name")).data()

            hub["related_docs"] = [
                {"name": r.get("name"), "labels": r.get("labels"), "props": {k: v for k, v in r.get("props", {}).items() if k != "embedding"}}
                for r in related
            ]

        driver.close()
        return {"status": "ok", "hub": hub}
    except Exception as exc:
        return {"status": "error", "reason": str(exc)}
