# airship / design

## Final Consolidated Implementation (≤2h)

**Goal**: Eliminate HF API 429 and Lightning quota waste during Surrogate training by implementing:

1. **CDN-first deterministic ingestion** — pre-list once; embed file list in training script; Lightning training uses CDN-only fetches (zero API calls during data load).
2. **Lightning Studio reuse + idle-stop guard** — list running studios and reuse; restart only if stopped to save quota.
3. **Arkship UI idle-stop resilience** — guard `.run()` calls with status checks and auto-restart on idle stop.

---

## Implementation Plan (≤2h)

### 1. Add CDN-first file-list utility (surrogate)
- Create `surrogate/ingest/file_list.py`:
  - Uses HF `list_repo_tree(path, recursive=False)` per folder (avoids paginated file listing).
  - Emits `batches/mirror-merged/{date}/file_list.json` with CDN paths.
  - Project to `{prompt, response}` only at parse time; store attribution in filename (`{slug}.parquet`), no extra columns.
- Create `surrogate/ingest/build_manifest.py`:
  - Reads `file_list.json`, produces `train_manifest.json` used by `train.py`.
  - Manifest contains CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) — no auth, bypasses API rate limits.

### 2. Update training script to use CDN-only fetches
- Modify `surrogate/train.py`:
  - Load `train_manifest.json`.
  - Use `requests.get(cdn_url, stream=True)` or `aiohttp` + `pyarrow` to stream parquet shards.
  - Avoid `load_dataset`/`load_dataset(streaming=True)` for heterogeneous repos.
  - Use `hf_hub_download` only for single-file fallback (not recursive).

### 3. Lightning Studio reuse + idle-stop guard
- Create `surrogate/lightning/studio_manager.py`:
  - `get_or_create_studio(name, machine=Machine.L40S, clouds=PriorityOrder)` — lists `Teamspace.studios`, reuses running studio; if stopped, restarts with target machine.
  - Before each `.run()`, check `studio.status`; if stopped, call `studio.start(machine=machine)` then run.
  - Priority order: `lightning-lambda-prod` (H200) → others → `lightning-public-prod` (L40S).

### 4. Arkship UI idle-stop resilience
- In Arkship service that triggers/calls surrogate training:
  - Wrap Lightning SDK calls with status check + restart guard.
  - If studio stops during training, catch exception, restart studio, resume from last checkpoint (or re-queue).

### 5. Cron/env hygiene (apply learned patterns)
- Ensure any wrapper scripts have `#!/usr/bin/env bash`, `chmod +x`, invoked via `bash <script> "$@"`.
- Set `SHELL=/bin/bash` in crontab for cron jobs.

### surrogate/ingest/file_list.py
```python
#!/usr/bin/env python3
"""
Generate deterministic CDN file list for a HF dataset repo.
Avoids recursive list_repo_files (paginated) by listing per folder.
"""
import json
import os
from pathlib import Path
from typing import List, Dict

from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_DATASET_REPO", "my-org/surrogate-mirror")
DATE_FOLDER = os.getenv("DATE_FOLDER", "2026-04-29")  # e.g. batches/mirror-merged/2026-04-29
OUT_DIR = Path(os.getenv("OUT_DIR", "batches/mirror-merged"))
OUT_DIR.mkdir(parents=True, exist_ok=True)

api = HfApi()

def list_folder_files(folder_path: str) -> List[Dict]:
    entries = api.list_repo_tree(repo_id=HF_REPO, path=folder_path, recursive=False)
    files = []
    for e in entries:
        if e.type == "file":
            files.append({
                "path": e.path,
                "cdn_url": f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{e.path}"
            })
        elif e.type == "directory":
            files.extend(list_folder_files(e.path))
    return files

def main():
    root = f"batches/mirror-merged/{DATE_FOLDER}"
    print(f"Listing {root} in {HF_REPO}...")
    files = list_folder_files(root)
    out_path = OUT_DIR / DATE_FOLDER / "file_list.json"
    out_path.parent.mkdir(parents=True, exist_ok=True)
    with open(out_path, "w") as f:
        json.dump(files, f, indent=2)
    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    main()
```

### surrogate/ingest/build_manifest.py
```python
#!/usr/bin/env python3
"""
Build train_manifest.json from file_list.json.
Keeps only {prompt,response} projection at parse time.
"""
import json
import os
from pathlib import Path

FILE_LIST = Path(os.getenv("FILE_LIST", "batches/mirror-merged/2026-04-29/file_list.json"))
OUT = Path(os.getenv("MANIFEST_OUT", "batches/mirror-merged/2026-04-29/train_manifest.json"))

def main():
    with open(FILE_LIST) as f:
        files = json.load(f)
    # Keep only parquet shards; manifest used by train.py for CDN fetches
    shards = [f for f in files if f["path"].endswith(".parquet")]
    manifest = {
        "repo": os.getenv("HF_DATASET_REPO", "my-org/surrogate-mirror"),
        "shards": shards,
        "projection": ["prompt", "response"]
    }
    OUT.parent.mkdir(parents=True, exist_ok=True)
    with open(OUT, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest: {len(shards)} shards -> {OUT}")

if __name__ == "__main__":
    main()
```
