# surrogate-1 / design

## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16` (matrix) and optional `DATE_FOLDER` (defaults to today `YYYY-MM-DD`)
- Uses **manifest-first strategy**: single `list_repo_tree` call (per date folder) saved to `manifest.json`; workers shard by deterministic hash of file path (`hash(slug) % SHARD_TOTAL`)
- Downloads via **HF CDN bypass** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header to avoid API rate limits during training data load
- Projects each file to `{prompt, response}` only at parse time; handles mixed schemas per surrogate-1 pattern (no `load_dataset(streaming=True)` on heterogeneous repos)
- Deduplicates via central `lib/dedup.py` md5 store; writes to `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`
- Emits lightweight metrics (files processed, bytes, dups, errors) to stdout for GitHub Actions logs
- Includes shebang, `chmod +x`, and safe Bash invocation fallback in workflow

Usage (GitHub Actions matrix):

def deterministic_shard(key: str) -> int:
    """Map key to shard by md5 hash."""
    digest = hashlib.md5(key.encode("utf-8")).hexdigest()
    return int(digest, 16) % SHARD_TOTAL

def build_manifest() -> List[str]:
    """Single API call: list top-level files in DATE_FOLDER (non-recursive)."""
    print(f"Listing repo tree for {REPO_OWNER}/{REPO_NAME}/{DATE_FOLDER} ...")
    try:
        tree = list_repo_tree(
            repo_id=f"{REPO_OWNER}/{REPO_NAME}",
            path=DATE_FOLDER,
            repo_type="dataset",
            recursive=False,
        )
    except Exception as exc:
        print(f"ERROR listing repo tree: {exc}", file=sys.stderr)
        sys.exit(1)

    files = [item.rfilename for item in tree if item.type == "file"]
    manifest = sorted(files)
    # Save for reproducibility / debugging
    with open(MANIFEST_PATH, "w", encoding="utf-8") as f:
        json.dump({"date_folder": DATE_FOLDER, "files": manifest}, f, indent=2)
    print(f"Manifest saved: {MANIFEST_PATH} ({len(manifest)} files)")
    return manifest

def cdn_url(repo_owner: str, repo_name: str, filename: str) -> str:
    """HF CDN URL (bypasses API auth/rate-limit)."""
    return f"https://huggingface.co/datasets/{repo_owner}/{repo_name}/resolve/main/{filename}"

def safe_download(url: str, timeout: int = 30) -> bytes:
    resp = requests.get(url, timeout=timeout)
    resp.raise_for_status()
    return resp.content

def project_to_pair(raw: Dict[str, Any], filename: str) -> Dict[str, Any] | None:
    """
    Project heterogeneous file content to {prompt, response}.
    Supports common surrogate-1 patterns:
    - JSON/JSONL with prompt/response fields (case-insensitive)
    - Parquet handled upstream (hf_hub_download) — this function receives dict rows.
    """
    if not raw or not isinstance(raw, dict):
        return None

    # Normalize keys
    low = {k.lower(): v for k, v in raw.items()}

    prompt = low.get("prompt") or low.get("input") or low.get("question") or low.get("user")
    response = low.get("response") or low.get("output") or low.get("answer") or low.get("assistant")

    # If missing, attempt to infer from filename patterns (best-effort)
    if prompt is None or response is None:
        # fallback: if only one text-like field exists, treat as prompt and empty response
        text_fields = [v for v in low.values() if isinstance(v, str) and len(v) > 10]
        if len(text_fields) == 1:
            prompt = text_fields[0]
            response = ""
        else:
            return None

    # Ensure strings
    prompt = str(prompt) if prompt is not None else ""
    response = str(response) if response is not None else ""

    if not prompt.strip():
        return None

    return {
        "prompt": prompt,
        "response": response,
    }
