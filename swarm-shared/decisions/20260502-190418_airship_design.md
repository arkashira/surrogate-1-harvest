# airship / design

## Final Synthesized Implementation  
**Single, Correct, Actionable CLI: `airship discover`**

---

### 1. Core Design Decisions (Resolved Contradictions)
- **Location**: `/opt/axentx/airship/airship` (executable CLI) — satisfies Candidate 1 explicit path and Candidate 2 Mac orchestration requirement.  
- **Language**: Python 3 script wrapped by a Bash launcher — ensures portability, proper signal handling, and `SHELL=/bin/bash` safety.  
- **Neo4j / Vector**: Use real connection if available; fail gracefully with simulated data — resolves Candidate 1 strict dependency vs Candidate 2 portability.  
- **Output**: `--format json|text` — both candidates agree; JSON for automation, text for human review.  
- **Safety**: `set -euo pipefail` in Bash wrapper; explicit timeouts on subprocesses; no silent failures.

---

### 2. File Layout (≤2h build)
```
/opt/axentx/airship/
├── airship                 # Bash launcher (executable)
├── discover.py             # Main CLI logic
└── granite-business-research.sh  # Optional; if present, runs
```

---

### 3. Implementation

#### `/opt/axentx/airship/airship`
```bash
#!/usr/bin/env bash
# Airship CLI - Zero-config discovery for Arkship/Surrogate
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PYTHON_EXEC="${PYTHON_EXEC:-python3}"

export SHELL=/bin/bash
"${PYTHON_EXEC}" "${SCRIPT_DIR}/discover.py" "$@"
```
**Make executable**:  
```bash
chmod +x /opt/axentx/airship/airship
```

def query_top_hub() -> Dict[str, Any]:
    """Query knowledge graph for most-connected hub (e.g., MOC) and related docs."""
    if not HAS_NEO4J:
        return {
            "top_hub": "MOC",
            "connection_count": 42,
            "related_docs": [
                {"title": "MOC Incident Playbook", "score": 0.96},
                {"title": "Arkship Surrogate Patterns", "score": 0.91},
            ],
            "insight": "Mission Operations Center is central hub for incident orchestration",
            "source": "fallback",
        }

    try:
        driver = GraphDatabase.driver(NEO4J_URI, auth=(NEO4J_USER, NEO4J_PASS))
        with driver.session() as session:
            # Top hub by connection count
            record = session.run(
                """
                MATCH (h:Hub)-[r:CONNECTED_TO]->()
                RETURN h.name AS name, count(r) AS connections
                ORDER BY connections DESC LIMIT 1
                """
            ).single()
            top_hub = record["name"] if record else "MOC"
            connections = record["connections"] if record else 0

            # Related docs/tools
            docs_result = session.run(
                """
                MATCH (h:Hub {name: $hub})-[:RELATED_TO]->(d:Document)
                RETURN d.title AS title, coalesce(d.score, 0.0) AS score
                ORDER BY score DESC LIMIT 5
                """,
                hub=top_hub,
            )
            related = [{"title": r["title"], "score": r["score"]} for r in docs_result]

        return {
            "top_hub": top_hub,
            "connection_count": connections,
            "related_docs": related,
            "insight": f"{top_hub} is the most-connected hub in the knowledge graph",
            "source": "neo4j",
        }
    except Exception as e:
        return {
            "error": str(e),
            "top_hub": "MOC",
            "connection_count": 0,
            "related_docs": [],
            "insight": "Using fallback due to Neo4j error",
            "source": "error",
        }
