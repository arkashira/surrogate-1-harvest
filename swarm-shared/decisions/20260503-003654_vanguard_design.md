# vanguard / design

# Final Synthesis

## Diagnosis (merged, de-duplicated)
- **No persisted manifest** for `(repo, dateFolder)` → every training run re-enumerates via authenticated HF API → quota burn and 429 risk.
- **Recursive enumeration / streaming enumeration** on heterogeneous repos triggers pyarrow `CastError` and amplifies rate-limit exposure.
- **Lightning Studio churn**: scripts re-create studios instead of reusing running ones → wastes quota and runtime.
- **No CDN bypass**: training performs authenticated API calls during data loading instead of zero-auth CDN fetches.
- **Missing idle-stop handling**: studios stop and aren’t restarted automatically; training jobs die.
- **Brittle data loading**: no sharding, no schema projection, no resilience to malformed parquet/jsonl or schema drift.
- **No observability / retry**: missing timeouts, retries, and clear failure modes for network and studio lifecycle.

## Proposed Change (merged)
Create `/opt/axentx/vanguard/train.py` and `/opt/axentx/vanguard/manifest.py` that:
- Accept `repo` and `dateFolder` args (orchestration side).
- Call `list_repo_tree(recursive=False)` **once per `(repo, dateFolder)`**, cache `manifest-{repo}-{dateFolder}.json` to disk.
- Embed manifest path in `train.py`; Lightning training uses **CDN-only URLs** from manifest (zero HF API calls during training).
- Reuse a running Lightning Studio if present; otherwise create one with `L40S` (fallback to free-tier compatible).
- Wrap `.run()` with status check, auto-restart on idle-stop, and **idempotent run token** to avoid duplicate launches.
- Add robust CDN loader with schema projection, sharding, retries, timeouts, and graceful fallback for corrupt files.

## Implementation (single, correct, actionable)

### manifest.py
```python
#!/usr/bin/env python3
"""
Generate and save a CDN file manifest for (repo, dateFolder).
Usage:
    python manifest.py --repo datasets/company/surrogate-1 --dateFolder 2026-04-29
"""
import argparse
import json
import os
import sys
from datetime import datetime, timezone

from huggingface_hub import HfApi

HF_API = HfApi()

def build_manifest(repo: str, date_folder: str, out_dir: str = "manifests") -> str:
    safe_repo = repo.replace("/", "-")
    out_path = os.path.join(out_dir, f"manifest-{safe_repo}-{date_folder}.json")
    os.makedirs(out_dir, exist_ok=True)

    # Single non-recursive call per dateFolder
    tree = HF_API.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    files = [
        item.rfilename
        for item in tree
        if item.rfilename.endswith((".parquet", ".jsonl"))
    ]

    # CDN URLs (no auth required)
    manifest = {
        "repo": repo,
        "dateFolder": date_folder,
        "generatedAt": datetime.now(timezone.utc).isoformat(),
        "files": [
            {
                "path": f,
                "cdn_url": f"https://huggingface.co/datasets/{repo}/resolve/main/{f}"
            }
            for f in sorted(files)
        ]
    }

    with open(out_path, "w") as fh:
        json.dump(manifest, fh, indent=2)
    return out_path

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--dateFolder", required=True)
    parser.add_argument("--outDir", default="manifests")
    args = parser.parse_args()
    try:
        out = build_manifest(args.repo, args.dateFolder, args.outDir)
        print(f"Manifest written: {out}")
    except Exception as exc:
        print(f"Failed: {exc}", file=sys.stderr)
        sys.exit(1)
```

class CDNParquetDataset(IterableDataset):
    def __init__(
        self,
        manifest_path: str,
        shard_rank: int = 0,
        shard_world_size: int = 1,
        required_columns=("prompt", "response"),
        timeout: float = 30.0,
        max_retries: int = 3,
    ):
        super().__init__()
        with open(manifest_path) as fh:
            manifest = json.load(fh)
        self.urls = [item["cdn_url"] for item in manifest["files"]]
        # simple deterministic sharding by url index
        self.urls = self.urls[shard_rank::shard_world_size]
        self.required_columns = required_columns
        self.timeout = timeout
        self.max_retries = max_retries

    def _fetch(self, url: str) -> bytes:
        for attempt in range(1, self.max_retries + 1):
            try:
                resp = requests.get(url, timeout=self.timeout)
                resp.raise_for_status()
                return resp.content
            except Exception as exc:
                if attempt == self.max_retries:
                    raise
                time.sleep(1.0 * attempt)
        raise RuntimeError("unreachable")

    def __iter__(self):
        if pq is None:
            raise RuntimeError("pyarrow is required for parquet support")
