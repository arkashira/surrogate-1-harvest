# Costinel / design

## 1. Diagnosis

- No machine-readable discovery of ingest sources, signal types, or readiness checks — operators must read prose docs to know what Costinel can consume and emit.
- No declarative schema for cloud cost data shapes (AWS CUR, GCP billing export, Azure EA) — integrations require code changes for each new field or tag.
- No readiness/liveness probe for external dependencies (cloud credentials, object storage, database) — deployment failures surface late during onboarding.
- Signal taxonomy is implicit in code/UI strings — impossible to automate downstream policy or audit pipelines without parsing text.
- Missing versioned contract between collector → analyzer → proposal store — upgrades risk silent schema drift.

## 2. Proposed change

Introduce a **Costinel Discovery Manifest** (`costinel/discovery.json`) and lightweight Python helpers that expose:

- `ingest.sources` (cloud provider, required auth, sample path, schema link)
- `signals.types` (id, name, category, schema, severity)
- `readiness.checks` (external deps, required env vars, test queries)
- `version.contract` (semver, changelog ref)

File scope:  
- Add `/opt/axentx/Costinel/costinel/discovery.json`  
- Add `/opt/axentx/Costinel/costinel/discovery.py` (load + validate + CLI)  
- Update `/opt/axentx/Costinel/README.md` with quick-start discovery snippet.

#!/usr/bin/env python3
"""
Costinel Discovery
Machine-readable manifest for ingest sources, signal types, and readiness checks.
"""
from __future__ import annotations

import json
import os
import sys
from pathlib import Path
from typing import Any, Dict, List, Optional

try:
    from jsonschema import validate, ValidationError
except ImportError:
    validate, ValidationError = None, None  # type: ignore

MANIFEST_PATH = Path(__file__).with_name("discovery.json")


class Discovery:
    def __init__(self, manifest_path: Path = MANIFEST_PATH) -> None:
        self.manifest_path = manifest_path
        self._data: Dict[str, Any] = {}
        self.load()

    def load(self) -> Dict[str, Any]:
        if not self.manifest_path.is_file():
            raise FileNotFoundError(f"Discovery manifest not found: {self.manifest_path}")
        with self.manifest_path.open() as f:
            self._data = json.load(f)
        return self._data

    @property
    def data(self) -> Dict[str, Any]:
        return self._data

    def get_ingest_sources(self) -> List[Dict[str, Any]]:
        return self._data.get("ingest", {}).get("sources", [])

    def get_signal_types(self) -> List[Dict[str, Any]]:
        return self._data.get("signals", {}).get("types", [])

    def get_readiness_checks(self) -> List[Dict[str, Any]]:
        return self._data.get("readiness", {}).get("checks", [])

    def validate_against_schema(self, schema_path: Optional[Path] = None) -> bool:
        if validate is None:
            raise RuntimeError("jsonschema not installed; cannot validate")
        schema_path = schema_path or self.manifest_path.parent / "discovery.schema.json"
        if not schema_path.is_file():
            raise FileNotFoundError(f"Schema not found: {schema_path}")
        with schema_path.open() as f:
            schema = json.load(f)
        try:
            validate(instance=self._data, schema=schema)  # type: ignore
            return True
        except ValidationError as exc:
            raise ValueError(f"Manifest validation failed: {exc.message}") from exc

    def readiness_report(self) -> Dict[str, Any]:
        env_missing: List[Dict[str, Any]] = []
        for check in self.get_readiness_checks():
            missing = [e for e in check.get("required_env", []) if not os.getenv(e)]
            if missing:
                env_missing.append({"check": check["id"], "missing_env": missing})
        return {"ok": len(env_missing) == 0, "checks": env_missing}


def main() -> int:
    d = Discovery()
    if "--validate" in sys.argv:
        try:
            d.validate_against_schema()
            print("OK: discovery manifest is valid")
        except Exception as exc:
            print(f"INVALID: {exc}", file=sys.stderr)
