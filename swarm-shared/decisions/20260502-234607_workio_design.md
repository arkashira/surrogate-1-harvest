# workio / design

**Final synthesized plan (single, actionable)**

**1. Diagnosis (merged, prioritized)**
- No canonical route contract between frontend and backend → runtime 404/422 and slow feature work.
- Search and URL params are untyped and unvalidated → silent wrong results, server exceptions, and security risk.
- No shared source of truth for route strings or types → duplication and drift.
- Missing Zod-based validation layer → inconsistent error shapes and no coercion for numbers/booleans in query/search.
- Environment/config handling and cross-layer typing are inconsistent.

**2. Proposed change (merged, concrete)**
Create a single source-of-truth route catalog and typed request contract using Zod, then integrate validation into the backend and export shared types for frontend use—**without changing existing URLs**.

- Add/modify:
  - `src/routes.ts` (shared route catalog with Zod schemas and path builders).
  - `server/src/routes/index.ts` (backend routes that reuse the catalog and validation).
  - `server/src/middleware/validation.ts` (generic Zod validator for params/query/body).
  - `src/types/shared.ts` (shared primitive types used by both frontend and backend).
  - `server/src/types/shared.ts` (re-export shared types; keep one canonical copy under `src/types/` and alias or re-export).
  - `server/src/app.ts` (wire middleware; add per-route opt-in validation).
  - `zod` installed and configured.

import { z } from 'zod';

// Shared param schemas
export const SlugParams = z.object({
  tenantId: z.string().min(1),
  employeeId: z.string().min(1),
});

export const IdParam = z.object({
  id: z.string().uuid(),
});

// Search/query param schemas
export const EmployeeListSearch = z.object({
  role: z.enum(['SuperAdmin', 'Admin', 'Manager', 'Employee']).optional(),
  isActive: z.coerce.boolean().optional(),
  page: z.coerce.number().int().min(1).default(1),
  limit: z.coerce.number().int().min(1).max(100).default(20),
});

export const SearchSearch = z.object({
  query: z.string().optional(),
  limit: z.coerce.number().int().min(1).max(100).optional(),
});

// Path builders
export const routes = {
  employees: {
    list: (search?: z.input<typeof EmployeeListSearch>) => {
      const base = '/employees';
      if (!search) return base;
      const parsed = EmployeeListSearch.parse(search);
      const params = new URLSearchParams();
      Object.entries(parsed).forEach(([k, v]) => {
        if (v !== undefined) params.set(k, String(v));
      });
      return base + '?' + params.toString();
    },
    detail: (params: z.input<typeof SlugParams>) =>
      `/employees/${params.employeeId}`,
  },
  search: {
    get: (search?: z.input<typeof SearchSearch>) => {
      const base = '/search';

        if (v !== undefined) params.set(k, String(v));
      });
      return base + '?' + params.toString();
    },
  },
  punches: {
    detail: (params: z.input<typeof IdParam>) => `/punches/${params.id}`,

  leaves: {

**3.4 Validation middleware (`server/src/middleware/validation.ts`)**
```ts
// server/src/middleware/validation.ts
import { Request, Response, NextFunction } from 'express';
import { z, ZodSchema } from 'zod';

export const validate =
  (schema: ZodSchema) => (req: Request, res: Response, next: NextFunction) => {
    try {
      schema.parse({
        params: req.params,
        query: req.query,
        body: req.body,
      });
      next();
    } catch (err) {
      const errors =
        err instanceof z.ZodError
          ? err.errors.map((e) => ({
              path: e.path.join('.'),
              message: e.message,
            }))
          : [{ path: 'unknown', message: 'Invalid request' }];
      res.status(400).json({ error: 'Validation failed', details: errors });
    }
  };

// Per-location helpers (opt-in)
export const validateParams = (schema: ZodSchema) =>
  validate(schema.pick({ params: true }).extend({ params: schema }));

export const validateQuery = (schema: ZodSchema) =>
  validate(schema.pick({ query: true }).extend({ query: schema }));
```
