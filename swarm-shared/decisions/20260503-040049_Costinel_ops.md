# Costinel / ops

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a non-blocking Top-Hub Signal Panel to Costinel frontend that surfaces the most-connected hub (e.g., "MOC") using CDN-first data baked at build/deploy time. **Zero Hugging Face API calls at runtime.**

---

### 1) High-value scope (fits <2h)
- Add a small frontend widget in the dashboard sidebar/header that shows:
  - Top hub label + short insight (1–2 sentences)
  - Link to full knowledge-rag context (opens in new tab)
- Data source: static JSON baked into the build (generated by ops script, committed to repo or pulled from CDN at build time)
- No backend changes; no runtime HF API; no DB migrations
- Fallback: hide panel if data missing (non-blocking)

### 2) Data contract (CDN-first)
File: `public/data/top-hub.json`
```json
{
  "hub": "MOC",
  "title": "MOC — Mission Operations Center",
  "insight": "Highest connectivity hub this cycle. Centralizes runbooks, on-call rotations, and incident timelines. Recommended: route new platform signals through MOC for faster triage.",
  "link": "https://huggingface.co/datasets/axentx/knowledge-rag/resolve/main/hubs/MOC.md",
  "updatedAt": "2026-05-03T04:00:00Z",
  "ttlHours": 24
}
```
- `public/` ensures CDN-friendly path (same-origin or public CDN).
- Optional: during CI, ops script can `curl` the CDN URL and overwrite this file before build (keeps runtime zero-api).

#### B) Create React component (or framework equivalent)
File: `src/components/TopHubSignalPanel.tsx`
```tsx
import { useEffect, useState } from 'react';
import './TopHubSignalPanel.css';

interface TopHubData {
  hub: string;
  title: string;
  insight: string;
  link: string;
  updatedAt: string;
  ttlHours?: number;
}

export default function TopHubSignalPanel() {
  const [data, setData] = useState<TopHubData | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // CDN-first: fetch baked static file; no auth, no HF API at runtime
    fetch('/data/top-hub.json', { cache: 'no-cache' })
      .then((res) => {
        if (!res.ok) throw new Error('Top-hub data unavailable');
        return res.json();
      })
      .then((json) => {
        // Optional: simple TTL check
        const updated = new Date(json.updatedAt).getTime();
        const ttl = (json.ttlHours || 24) * 3600 * 1000;
        if (Date.now() - updated > ttl) {
          console.warn('Top-hub data stale; hiding panel.');
          setLoading(false);
          return;
        }
        setData(json);
      })
      .catch(() => {
        // Non-blocking: silently hide panel on error
        setData(null);
      })
      .finally(() => setLoading(false));
  }, []);

  if (loading || !data) return null;

  return (
    <div className="top-hub-panel" role="complementary" aria-label="Top hub signal">
      <div className="top-hub-panel__header">
        <span className="top-hub-panel__badge">Top Hub</span>
        <span className="top-hub-panel__hub">{data.hub}</span>
      </div>
      <p className="top-hub-panel__title">{data.title}</p>
      <p className="top-hub-panel__insight">{data.insight}</p>
      <a
        className="top-hub-panel__link"
        href={data.link}
        target="_blank"
        rel="noopener noreferrer"
      >
        View context →
      </a>
    </div>
  );
}
```

#### E) Build/ops automation (optional but recommended)
Add a small ops script to refresh `public/data/top-hub.json` from CDN before build (keeps runtime zero-api). Example (bash):

File: `scripts/update-top-hub.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

REPO="axentx/knowledge-rag"
OUT="public/data/top-hub.json"
TMP=$(mktemp)

# Fetch most-connected hub summary from CDN (example path; adapt as needed)
# If you have a generated summary at a known CDN path, use it.
# Here we fallback to a local template if CDN fetch fails.
if curl -fsSL --max-time 10 \
  "https://huggingface.co/datasets/${REPO}/resolve/main/top-hub-summary.json" \
  -o "${TMP}" 2>/dev/null; then
  # Optional: transform if needed, then write
  cp "${TMP}" "${OUT}"
else
  echo "CDN fetch failed; keeping existing top-hub.json"
fi

rm -f "${TMP}"
```
