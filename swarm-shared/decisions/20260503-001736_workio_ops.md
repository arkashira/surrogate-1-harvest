# workio / ops

## Final Consolidated Implementation Plan

### 1. Diagnosis (Merged)
The system has **two independent failure classes** that must be fixed together:

- **Data integrity failure**: No idempotency protection on punch submissions allows duplicate clock-in/out records from webhook replays, user double-taps, or concurrent requests. No client-side state validation creates confusing UX and invalid data. Database lacks constraints to prevent duplicates at scale.
- **Operational failure**: No health/readiness probes, structured logging, graceful shutdown, or process management. Crashes, DB disconnects, or deploys drop in-flight punch requests and leave ops blind to service state.

### 2. Correctness-First Decisions
- **Idempotency key belongs on both frontend and backend**: Frontend generates crypto-random key per user action; backend enforces uniqueness via database constraint. This prevents duplicates from any source (UI, webhook replays, retries).
- **Graceful shutdown is mandatory**: Must drain in-flight punch requests for 30s on SIGTERM/SIGINT to prevent data loss during deploys.
- **Health endpoints must check DB**: Readiness must verify database connectivity; liveness should indicate process is alive.
- **Structured logging with rotation**: Prevents disk fill and enables production debugging without breaking JSON log ingestion.

```tsx
// workio/src/components/PunchButton.tsx
import { useState, useCallback } from 'react';
import { v4 as uuidv4 } from 'uuid';

export function PunchButton({ 
  onPunch, 
  currentState 
}: { 
  onPunch: (type: 'IN' | 'OUT', idempotencyKey: string) => Promise<void>;
  currentState: 'IN' | 'OUT' | null;
}) {
  const [status, setStatus] = useState<'IDLE' | 'PENDING' | 'SUCCESS' | 'ERROR'>('IDLE');

  const handlePunch = useCallback(
    debounce(async (type: 'IN' | 'OUT') => {
      if (status === 'PENDING') return;
      
      // Client-side validation
      if (type === 'IN' && currentState === 'IN') {
        setStatus('ERROR');
        setTimeout(() => setStatus('IDLE'), 2000);
        return;
      }
      if (type === 'OUT' && currentState !== 'IN') {
        setStatus('ERROR');
        setTimeout(() => setStatus('IDLE'), 2000);
        return;
      }

      setStatus('PENDING');
      const idempotencyKey = uuidv4();
      
      try {
        await onPunch(type, idempotencyKey);
        setStatus('SUCCESS');
        setTimeout(() => setStatus('IDLE'), 1500);
      } catch (err) {
        setStatus('ERROR');
        setTimeout(() => setStatus('IDLE'), 2000);
      }
    }, 500),
    [status, currentState, onPunch]
  );

  const nextAction = currentState === 'IN' ? 'OUT' : 'IN';
  const isPending = status === 'PENDING';

  return (
    <button
      onClick={() => handlePunch(nextAction)}
      disabled={isPending}
      className={`px-6 py-3 rounded font-medium transition ${
        isPending
          ? 'bg-gray-400 cursor-wait'
          : currentState === 'IN'
          ? 'bg-red-500 hover:bg-red-600 text-white'
          : 'bg-green-500 hover:bg-green-600 text-white'
      }`}
    >
      {isPending
        ? 'กำลังบันทึก...'
        : currentState === 'IN'
        ? 'Clock Out'
        : 'Clock In'}
    </button>
  );
}

function debounce<T extends (...args: any[]) => any>(fn: T, wait: number) {
  let timeout: NodeJS.Timeout;
  return (...args: Parameters<T>) => {
    clearTimeout(timeout);
    timeout = setTimeout(() => fn(...args), wait);
  };
}
```

#### Backend: Punch Route with Idempotency
```ts
// workio/server/src/routes/punch.ts
import { Router } from 'express';
import { db } from '../db';

const router = Router();

router.post('/', async (req, res) => {
  const { userId, type, idempotencyKey } = req.body;
  
  if (!userId || !type || !idempotencyKey) {
    return res.status(400).json({ error: 'Missing fields: userId, type, or idempotencyKey' });
  }

  const today = new Date().toISOString().split('T')[0];
  const sessionType = type.toUpperCase() === 'IN' ? 'CLOCK_IN' : 'CLOCK_OUT';

  try {
    // Check for existing idempotent request first
    const existing = await db.oneOrNone(
      `SELECT * FROM punches 
       WHERE user_id = $1 AND punch_date = $2 AND session_type = $3 AND idempotency_key = $4`,
      [userId, today, sessionType, idempotencyKey]
    );

    if (existing) {
      return res.json({ punch: existing, replayed: true });
    }

    // Validate state transition
    const latest = await db.oneOrNone(
      `SELECT * FROM punches 
       WHERE user_id = $1 AND punch_date = $2 
       ORDER BY created_at DESC LIMIT 1`,
      [userId, today]
    );

    if (sessionType === 'CLOCK_IN' && latest?.session_type === 'CLOCK_IN') {
      return res.status(409).json({ error: 'Already clocked in today' });
    }
    if (sessionType === 'CLOCK_OUT' && (!latest || latest.session_type === 'CLOCK_OUT')) {
      return res.status(409).json({ error: 'No active clock-in to clock out' });
    }

    // Insert with idempotency key
    const punch = await db.one(
      `INSERT INTO punches (user_id, punch_date, session_type, idempotency_key, created_at)
       VALUES ($1, $2, $3, $4, NOW()) RETURNING *`,
      [userId, today, sessionType, idempotencyKey]
    );

    res.json({ punch, replayed: false });
  } catch (err: any) {
    // Handle unique constraint violation
    if (err.code === '23505') {
      const dup = await db.oneOrNone(
        `SELECT * FROM punches 
         WHERE user_id = $1 AND punch_date = $2 AND session_type = $3 AND idempotency_key = $4`,
        [userId, today, sessionType, idempotencyKey]
      );
      return res.json({ punch: dup, replayed: true });
    }
    
    console.error('Punch error:', err);
    res.status(500).json({ error: 'Internal server error' });
  }
});

export default router;
```
