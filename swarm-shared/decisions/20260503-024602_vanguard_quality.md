# vanguard / quality

## 1. Diagnosis
- Frontend has no build/dev tooling or entrypoint — cannot ship or iterate on UI changes (missing `index.html`, mount point, bundler).
- No static file manifest embedded in the bundle → every session re-enumerates repos at runtime, burning HF API quota and risking 429s.
- Runtime HF API calls from frontend (e.g., `list_repo_tree`, dataset endpoints) run on user machines with unpredictable rate limits and no CDN bypass.
- No deterministic pre-list of dataset files → training/data-loading scripts cannot use CDN-only fetches and must hit the API repeatedly.
- Missing ops guardrails (wrapper shebangs, executable bits, SHELL in cron) for automation scripts (active-learning, opus-pr-reviewer) that will be added later.

## 2. Proposed change
Create a minimal, production-quality frontend entrypoint and static manifest pipeline:
- Add `/opt/axentx/vanguard/index.html` (client mount + CDN asset references).
- Add `/opt/axentx/vanguard/src/main.js` (client bootstrap, no runtime HF API calls).
- Add `/opt/axentx/vanguard/manifest/dataset-files.json` (pre-computed file list for one date folder) generated by a Mac-side script.
- Add `/opt/axentx/vanguard/scripts/build-manifest.sh` (safe, idempotent, uses HF API once, outputs CDN-only manifest).
- Add `/opt/axentx/vanguard/scripts/wrapper-template.sh` (correct shebang + executable template for future cron/automation scripts).

Scope: ~10 files, <200 lines total. No backend changes.

// Client bootstrap — no runtime HF API calls.
// Uses embedded manifest (dataset-files.json) for CDN-only fetches.

const API_ROOT = 'https://huggingface.co/datasets';
const DEFAULT_REPO = 'axentx/surrogate-1'; // change per deployment
const DEFAULT_DATE = '2026-05-03'; // example; update when regenerating manifest

function $(sel) { return document.querySelector(sel); }
function $all(sel) { return Array.from(document.querySelectorAll(sel)); }

function buildCdnUrl(repo, path) {
  // Public CDN URL — no Authorization header required.
  return `${API_ROOT}/${repo}/resolve/main/${encodeURI(path)}`;
}

async function loadEmbeddedManifest() {
  try {
    const tag = $('#datasetManifest');
    if (!tag) throw new Error('Manifest tag missing');
    // If the JSON file is served with correct Content-Type, fetch it.
    const res = await fetch('./manifest/dataset-files.json', { cache: 'no-store' });
    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    return await res.json();
  } catch (err) {
    console.warn('Could not load embedded manifest:', err);
    return null;
  }
}

function renderManifest(data) {
  const status = $('#manifestStatus');
  const list = $('#fileList');
  if (!data || !Array.isArray(data.files) || data.files.length === 0) {
    status.textContent = 'missing';
    list.innerHTML = '<div class="empty">No files in manifest. Run scripts/build-manifest.sh to generate.</div>';
    return;
  }

  status.textContent = `${data.files.length} files`;
  list.innerHTML = '';
  for (const f of data.files) {
    const row = document.createElement('div');
    row.className = 'list-row';
    const cdn = buildCdnUrl(data.repo || DEFAULT_REPO, f.path);
    row.innerHTML = `
      <span class="path" title="${f.path}">${f.path}</span>
      <span class="cdn" title="${cdn}">cdn</span>
    `;
    list.appendChild(row);
  }
}

async function init() {
  const btn = $('#refreshBtn');
  btn.addEventListener('click', async () => {
    btn.disabled = true;
    btn.textContent = 'Loading…';
    const manifest = await loadEmbeddedManifest();
    renderManifest(manifest);
    btn.disabled = false;
    btn.textContent = 'Reload Manifest';
  });

  const manifest = await loadEmbeddedManifest();
  renderManifest(manifest);
}
