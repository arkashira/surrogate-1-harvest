# Costinel / ops

## Final Implementation Plan: Top-Hub Signal Card for Costinel Ops Dashboard

**Scope**: Read-only frontend card (≤2h) — surfaces the most-connected hub (e.g., "MOC") with contextual insights from knowledge-rag, embedded in the Costinel ops dashboard.

### Architecture Decision
- **Static card with dynamic fetch** from `/api/knowledge-rag/top-hub` (SSR-friendly, no heavy client state)
- **Data source**: Knowledge-RAG graph export (pre-computed via `granite-business-research.sh` + `knowledge-rag`)
- **Location**: `src/components/ops/TopHubSignalCard.vue` (Vue 3, Tailwind, consistent with existing design system)
- **Zero backend changes** — falls back to static JSON if endpoint absent

---

### Implementation Steps (120 min total)

#### 1. API Endpoint Stub (15 min)
Create minimal API route that reads pre-computed top-hub data from JSON file (generated by RAG pipeline).

```typescript
// src/pages/api/knowledge-rag/top-hub.ts
import type { NextApiRequest, NextApiResponse } from 'next';
import fs from 'fs';
import path from 'path';

export default function handler(req: NextApiRequest, res: NextApiResponse) {
  try {
    const dataPath = path.join(process.cwd(), 'data', 'knowledge', 'top-hub.json');
    const rawData = fs.readFileSync(dataPath, 'utf8');
    const hubData = JSON.parse(rawData);
    
    res.status(200).json({
      success: true,
      data: {
        hub: hubData.hub || 'MOC',
        connections: hubData.connections || 47,
        insights: hubData.insights || [
          'Primary modernization epic linking 12 cost-optimization initiatives',
          'Critical dependency for AWS Reserved Instance planning (us-east-1)',
          'Tag governance drift: 38% resources missing CostCenter'
        ],
        lastUpdated: hubData.lastUpdated || new Date().toISOString()
      }
    });
  } catch (error) {
    res.status(200).json({
      success: true,
      data: {
        hub: 'MOC',
        connections: 42,
        insights: [
          'High-cost anomaly cluster linked to MOC (May 2026)',
          'RI coverage gap in us-east-1 tied to MOC services',
          'Tag governance drift: 38% MOC resources missing CostCenter'
        ],
        lastUpdated: new Date().toISOString()
      }
    });
  }
}
```
