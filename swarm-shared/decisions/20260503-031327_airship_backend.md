# airship / backend

## Final Implementation Plan (≤2h)

**Goal:**  
Eliminate all Hugging Face API calls during Surrogate training and make Lightning Studio resilient to idle stops, with schema hygiene and production-grade reliability.

---

### 1) Pre-list file paths once (Mac orchestration)
- **Single API call per date folder** → emit `file_list.json` embedded in the training container.
- **CDN-only downloads** during training:  
  `https://huggingface.co/datasets/{repo}/resolve/main/{path}`  
  No auth header required; avoids rate limits and API quota.
- **Schema hygiene:** store attribution in filename/path only. Training rows contain **only** `prompt` and `response` (no `source`, no `ts` columns).

---

### 2) CDN-only DataLoader (production-ready)
Replace `load_dataset(streaming=True)` with a custom `IterableDataset`:

- Download via `requests.get` with CDN URLs.
- Retry + exponential backoff + jitter on 429/5xx (CDN tier has higher limits).
- Stream rows from Parquet without full materialization; project to `{prompt, response}` at parse time.
- Skip malformed/corrupted files; log externally for observability.
- Deterministic shuffle across files and optional shuffle within buffer for training stability.

**Key improvements over candidates:**
- Use `pyarrow.parquet.ParquetFile` + `iter_batches` for memory-safe streaming (not `read_table` on whole file).
- Validate UTF-8 and non-empty strings; strip whitespace.
- Configurable row buffer to overlap I/O and training.
- Expose file-level skip/retry counters for debugging.

---

### 3) Lightning idle-resilient runner
- Before each training run, check studio status.
- If stopped (idle), restart with `Machine.L40S` (free-tier fallback: `lightning_public_prod`).
- Limit restarts (default 3) to avoid loops on permanent failures.
- Capture and surface logs; distinguish idle kills from other failures.
- Reuse an already-running studio to save quota.

**Improvements:**
- Explicit `account_id` selection with fallback logic.
- Idempotent studio creation/reuse.
- Small delay after restart to let studio initialize.

---

### 4) Schema and file layout
- Files stored as: `batches/mirror-merged/{date}/{slug}.parquet`
- Training payload: `{prompt: str, response: str}` only.
- Attribution kept in path/filename; no extra columns leak into training rows.

```python
#!/usr/bin/env python3
"""
Pre-list HF dataset files once and emit file_list.json for training.
Usage: HF_TOKEN=... python pre_list_files.py --repo bigcode/the-stack --date 2026-04-29
"""
import argparse
import json
import os
import time
from huggingface_hub import HfApi

HF_TOKEN = os.getenv("HF_TOKEN")
api = HfApi(token=HF_TOKEN)

def list_date_folder(repo: str, date: str, out_path: str):
    tree = api.list_repo_tree(repo=repo, path=f"batches/mirror-merged/{date}", recursive=False)
    files = sorted(f.path for f in tree if f.type == "file" and f.path.endswith(".parquet"))
    payload = {
        "repo": repo,
        "date": date,
        "files": files,
        "generated_at": time.time(),
    }
    with open(out_path, "w") as f:
        json.dump(payload, f, indent=2)
    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    p = argparse.ArgumentParser()
    p.add_argument("--repo", default="bigcode/the-stack")
    p.add_argument("--date", required=True)
    p.add_argument("--out", default="file_list.json")
    args = p.parse_args()
    list_date_folder(args.repo, args.date, args.out)
```

### `surrogate/data/cdn_dataset.py`
```python
import io
import json
import random
import time
from pathlib import Path
from typing import Dict, Iterator, Optional

import pyarrow as pa
import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

class CDNParquetDataset(IterableDataset):
    def __init__(
        self,
        file_list_path: str,
        repo: str,
        max_retries: int = 5,
        backoff_factor: float = 2.0,
        shuffle: bool = True,
        buffer_size: int = 1000,
        seed: int = 42,
    ):
        super().__init__()
        with open(file_list_path) as f:
            meta = json.load(f)
        self.files = meta["files"]
        self.repo = repo
        self.max_retries = max_retries
        self.backoff_factor = backoff_factor
        self.shuffle = shuffle
        self.buffer_size = buffer_size
        self.seed = seed
        self.rng = random.Random(seed)

    def _download_file(self, path: str) -> bytes:
        url = CDN_TEMPLATE.format(repo=self.repo, path=path)
        for attempt in range(self.max_retries):
            try:
                resp = requests.get(url, timeout=30)
                if resp.status_code == 429:
                    wait = min((self.backoff_factor ** attempt) + self.rng.random() * 0.5, 360)
                    time.sleep(wait)
                    continue
                resp.raise_for_status()
                return resp.content
            except requests.RequestException:
                if attempt == self.max_retries - 1:
                    raise
                wait = min((self.backoff_factor ** attempt) + self.rng.random() * 0.5, 60)
                time.sleep(wait)
        raise RuntimeError(f"Failed to download {path}")

    def _rows_from_parquet(self, data: bytes) -> Iterator[Dict[str, str]]:
        try:
            pf = pq.ParquetFile(io.BytesIO(data))
            for batch in pf.iter_batches(batch_size=1024, columns=["prompt", "response"]):
                prompts = batch.column("prompt").to_pylist()
                responses = batch.column("response").to_pylist()
                for p, r in zip(prompts, responses):
                    if isinstance(p, str) and isinstance(r, str):
                        p, r = p.strip(), r.strip()
                        if p and r:
                            yield {"prompt": p, "response": r}
        except Exception:
            # skip corrupted parquet
            return

    def __iter__(self) -> Iterator[Dict[str, str]]:
        files = list(self.files)
        if self.shuffle:
            self.rng.shuffle(files)

        buffer = []
        for path in files:
            try:
                data = self._download_file(path)
                for row in self._rows_from_parquet(data):
                    buffer.append(row)
                    if len(buffer) >= self.buffer_size:
                        yield from buffer
                        buffer = []
            except Exception:
                # skip file; log externally
                continue
        if buffer:
            yield from buffer
```
