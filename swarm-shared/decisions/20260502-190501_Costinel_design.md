# Costinel / design

## Design Increment: Deterministic Discovery Module (≤2h)

**Scope:** One-shot discovery that outputs:
- `manifests/sources.json` — cloud connectors, credentials, and collection cadence
- `manifests/discovery.json` — discovered resources, owners, tags, and cost centers
- `manifests/top-hub-insight.json` — most-connected hub snapshot (for governance/training)

**Why this ships fast:**
- No external runtime dependencies beyond `boto3`, `requests`, and local FS.
- Deterministic: given the same cloud state + config, identical manifests.
- Reusable by onboarding, surrogate-1 training, and governance pipelines.
- Avoids HF API during discovery (CDN-only pattern for downstream training).

---

## Implementation Plan

1. **Create directory structure** (`manifests/`, `src/discovery/`).
2. **Add deterministic discovery script** (`src/discovery/run_discovery.py`):
   - Reads `config/discovery.yaml` (cloud accounts, regions, tag schema).
   - Discovers AWS resources (accounts, regions, services, tagged resources).
   - Produces normalized `sources.json` and `discovery.json`.
   - Builds a lightweight graph (resource → tags → cost-center) and exports top-hub insight.
3. **Add lightweight top-hub analyzer** (`src/discovery/top_hub.py`):
   - Computes most-connected hub (by tag/cost-center/resource edges).
   - Emits `top-hub-insight.json` (used by governance/training).
4. **Add idempotent CLI entrypoint** (`scripts/discovery.sh`):
   - Shebang + executable.
   - Invokes via `python -m src.discovery.run_discovery`.
5. **Add minimal config** (`config/discovery.yaml`) for local/dev accounts.

### 3. Top-hub analyzer (`src/discovery/top_hub.py`)
```python
from collections import Counter
from typing import List, Dict, Any


def top_hub_by_tag(resources: List[Dict[str, Any]], tag_key: str = "CostCenter") -> Dict[str, Any]:
    """
    Compute most-connected hub by tag.
    Returns a deterministic snapshot useful for governance and training.
    """
    counts = Counter()
    items = []

    for r in resources:
        tags = r.get("tags", {})
        if tag_key in tags:
            value = tags[tag_key]
            counts[value] += 1
            items.append(
                {
                    "resource_id": r["resource_id"],
                    "resource_type": r["resource_type"],
                    "account_id": r["account_id"],
                    "region": r["region"],
                    "hub_value": value,
                }
            )

    if not counts:
        return {
            "hub_type": "tag",
            "hub_key": tag_key,
            "top_hub": None,
            "count": 0,
            "items": [],
        }

    top_value, top_count = counts.most_common(1)[0]
    return {
        "hub_type": "tag",
        "hub_key": tag_key,
        "top_hub": top_value,
        "count": top_count,
        "total_tagged": sum(counts.values()),
        "total_resources": len(resources),
        "items": sorted(items, key=lambda x: x["resource_id"]),
    }
```

def discover_aws_accounts(config: dict) -> list:
    """
    Deterministic AWS discovery for configured accounts/regions.
    Returns normalized resource entries.
    """
    accounts = config.get("cloud", {}).get("aws", {}).get("accounts", [])
    resources = []

    for acc in accounts:
        account_id = acc["id"]
        regions = acc.get("regions", [])
        services = acc.get("collection", {}).get("services", [])

        for region in regions:
            # Use default credential chain (env/profile/role)
            # If cross-account role is needed, assume role externally or via boto3 setup.
            session = boto3.Session(region_name=region)

            if "ec2" in services:
                ec2 = session.client("ec2")
                try:
                    paginator = ec2.get_paginator("describe_instances")
                    for page in paginator.paginate():
                        for reservation in page.get("Reservations", []):
                            for inst in reservation.get("Instances", []):
                                tags = {t["Key"]: t["Value"] for t in inst.get("Tags", [])}
                                resources.append(
                                    {
                                        "resource_id": inst["InstanceId"],
                                        "resource_type": "aws_ec2_instance",
                                        "account_id": account_id,
                                        "region": region,
                                        "service": "ec2",
                                        "tags": tags,
                                        "state": inst.get("State", {}).get("Name"),
                                        "instance_type": inst.get("InstanceType"),
                                        "launched_at": inst.get("LaunchTime").isoformat()
                                        if inst.get("LaunchTime")
                                        else None,
                                    }
                                )
                except Exception as exc:
                    print(f"[WARN] EC2 discovery failed for {account_id}/{region}: {exc}", file=sys.stderr)

            if "rds" in services:
                rds = session.client("rds")
                try:
                    paginator = rds.get_paginator("describe_db_instances")
                    for page in paginator.paginate():
                        for db in page.get("DBInstances", []):
                            tags = {}
                            try:
                                tag_resp = rds.list_tags_for_resource(
                                    ResourceName=db["DBInstanceArn"]
                                )
                                tags = {t["Key"]: t["Value"] for t in tag_resp.get("TagList", [])}
                            except Exception:
                                pass

                            resources.append(

                                    "resource_id": db["DBInstanceIdentifier"],
                                    "resource_type": "aws_rds_instance",
                                    "account_id": account_id,
                                    "region": region,
                                    "service": "rds",
                                    "tags": tags,
                                    "engine": db.get("Engine"),
                                    "status": db.get("DBInstanceStatus"),
                                }
                            )
                except Exception as exc:
                    print(f"[WARN] RDS discovery failed for {account_id}/{region}: {exc}", file=sys.stderr)

    # Deterministic sort
