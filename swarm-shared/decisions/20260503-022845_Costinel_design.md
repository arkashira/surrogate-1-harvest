# Costinel / design

## Highest-Value Incremental Improvement (<2h)

**Add a Top-Hub Signal Panel** to the Costinel dashboard that surfaces the most-connected hub (default “MOC”) and its top 3 actionable, cost-impact proposals from the knowledge graph — CDN-first, rate-limit-safe, zero API calls during render.

**Why this ships fast & high value**
- Pure frontend addition (no backend changes, no infra).
- Uses existing knowledge-rag graph insights (pattern: top-hub doc insight).
- Follows CDN bypass pattern: embed pre-listed graph paths in build-time JSON; runtime fetches via CDN (`/resolve/main/...`) to avoid HF API rate limits.
- Immediately increases dashboard decision quality by surfacing highest-signal hub and proposals.

---

## Implementation Plan (≤2h)

1. **Locate dashboard layout** (10m)  
   Identify main dashboard route/component (likely `pages/dashboard` or `components/Dashboard`).

2. **Create `top-hub-panel` component** (30m)  
   - Accept `hubName` prop (default `"MOC"`).
   - Fetch `graph/top-hub.json` from CDN at build/runtime (CDN URL: `https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/graph/top-hub.json`).
   - Render:
     - Hub title + short description.
     - Top 3 proposals as cards (title, impact score, 1-sentence rationale, action link).
   - Graceful fallback UI if fetch fails.

3. **Embed file list at build time** (10m)  
   - Add script `scripts/embed-top-hub.js` that runs post-market-analysis (or during CI) to call `list_repo_tree` once and save `graph/top-hub.json` to repo (or upload to HF dataset).  
   - Training/generation pipeline writes `graph/top-hub.json` with schema:
     ```json
     {
       "hub": "MOC",
       "description": "Multi-cloud observability & cost signals",
       "proposals": [
         {
           "id": "moc-ri-001",
           "title": "Right-size underutilized m5.xlarge fleet",
           "impactScore": 0.87,
           "rationale": "30-day avg CPU <15%; switch to m5.large + RI saves ~$18k/mo.",
           "actionUrl": "/proposals/moc-ri-001"
         }
       ]
     }
     ```

4. **Mount panel in dashboard** (20m)  
   - Insert `<TopHubPanel />` near top of dashboard after main KPI row.
   - Style to match existing card system (shadows, border radius, color tokens).

5. **CDN fetch with caching & timeout** (10m)  
   - Use `fetch` with 4s timeout; cache in `localStorage` for 6h to avoid repeated CDN misses.
   - If CDN fails, fallback to local bundled JSON (last known good).

6. **Test & polish** (20m)  
   - Verify on dev server.
   - Ensure mobile responsive.
   - Add loading skeleton.

## Code Snippets

### Component: `components/TopHubPanel.jsx`

```jsx
import { useEffect, useState } from "react";
import "./TopHubPanel.css";

const CDN_URL =
  "https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/graph/top-hub.json";
const CACHE_KEY = "costinel:top-hub:v1";
const CACHE_TTL_MS = 6 * 60 * 60 * 1000; // 6h

function useTopHubData() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), 4000);

    async function load() {
      try {
        // Try cache
        const cached = localStorage.getItem(CACHE_KEY);
        if (cached) {
          const parsed = JSON.parse(cached);
          if (Date.now() - parsed.ts < CACHE_TTL_MS) {
            setData(parsed.payload);
            setLoading(false);
            return;
          }
        }

        // Fetch from CDN (no auth header — bypasses HF API rate limit)
        const res = await fetch(CDN_URL, {
          signal: controller.signal,
          headers: { Accept: "application/json" },
        });
        if (!res.ok) throw new Error(`CDN fetch failed: ${res.status}`);
        const payload = await res.json();

        // Cache
        localStorage.setItem(
          CACHE_KEY,
          JSON.stringify({ ts: Date.now(), payload })
        );
        setData(payload);
      } catch (err) {
        if (err.name !== "AbortError") setError(err.message);
      } finally {
        setLoading(false);
        clearTimeout(timeoutId);
      }
    }

    load();

    return () => {
      controller.abort();
      clearTimeout(timeoutId);
    };
  }, []);

  return { data, loading, error };
}

export default function TopHubPanel({ defaultHub = "MOC" }) {
  const { data, loading, error } = useTopHubData();

  if (loading) {
    return (
      <div className="top-hub-panel">
        <div className="top-hub-panel__skeleton">
          <div className="skeleton-line w-40"></div>
          <div className="skeleton-line w-60"></div>
          <div className="skeleton-cards">
            {Array(3)
              .fill(0)
              .map((_, i) => (
                <div key={i} className="skeleton-card"></div>
              ))}
          </div>
        </div>
      </div>
    );
  }

  const hub = data || { hub: defaultHub, description: "", proposals: [] };
  const proposals = hub.proposals || [];

  return (
    <div className="top-hub-panel">
      <div className="top-hub-panel__header">
        <h3 className="top-hub-panel__title">{hub.hub}</h3>
        <p className="top-hub-panel__desc">{hub.description}</p>
      </div>

      {error && <div className="top-hub-panel__error">{error}</div>}

      <div className="top-hub-panel__cards">
        {proposals.slice(0, 3).map((p) => (
          <a
            key={p.id}
            href={p.actionUrl}
            className="top-hub-panel__card"
            target="_blank"
            rel="noopener noreferrer"
          >
            <div className="top-hub-panel__card-title">{p.title}</div>
            <div className="top-hub-panel__card-meta">
              <span className="top-hub-panel__impact">
                Impact: {(p.impactScore * 100).toFixed(0)}%
              </span>
            </div>
            <p className="top-hub-panel__card-rationale">{p.rationale}</p>
          </a>
        ))}
        {proposals.length === 0 && (
          <div className="top-hub-panel__empty">No proposals available.</div>
        )}
      </div>
    </div>
  );
}
```
