# surrogate-1 / design

## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16` (matrix) and optional `DATE_FOLDER` (defaults to today `YYYY-MM-DD`)
- Uses **manifest-first strategy**: one `list_repo_tree` call per date folder → `file-list.json`; worker uses this manifest so **training does CDN-only fetches with zero HF API calls** during data load
- Deterministic shard assignment: `hash(slug) % SHARD_TOTAL` → each runner processes only its slice
- Per-file download via CDN URL (`https://huggingface.co/datasets/axentx/surrogate-1-training-pairs/resolve/main/{path}`) — **bypasses HF API auth/rate limits**
- Projects heterogeneous schemas to `{prompt, response}` at parse time (avoids pyarrow CastError)
- Dedup via central `lib/dedup.py` md5 store (unchanged)
- Outputs: `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl`
- No `source`/`ts` columns; attribution encoded in filename pattern
- Safe retries with exponential backoff; respects HF 429 (wait 360s)
- Idempotent: re-running same date+shard skips already-processed files via dedup

def list_date_folder(date_folder: str) -> List[str]:
    """
    Single API call: list top-level tree for date folder (non-recursive).
    We rely on repo layout: files directly under date folder or one level deep.
    """
    try:
        tree = API.list_repo_tree(
            repo_id=REPO,
            path=date_folder,
            repo_type="dataset",
            use_auth_token=HF_TOKEN,
        )
    except Exception as e:
        print(f"WARN: list_repo_tree failed for {date_folder}: {e}", file=sys.stderr)
        return []

    paths = []
    for entry in tree:
        if entry.type == "file":
            paths.append(f"{date_folder}/{entry.path.split('/')[-1]}")
        elif entry.type == "dir":
            subpath = entry.path
            try:
                sub = API.list_repo_tree(
                    repo_id=REPO,
                    path=subpath,
                    repo_type="dataset",
                    use_auth_token=HF_TOKEN,
                )
            except Exception as e:
                print(f"WARN: list_repo_tree failed for {subpath}: {e}", file=sys.stderr)
                continue
            for e2 in sub:
                if e2.type == "file":
                    paths.append(e2.path)
    return paths

def load_or_create_manifest(date_folder: str, manifest_path: str) -> List[str]:
    if os.path.exists(manifest_path):
        with open(manifest_path) as f:
            return json.load(f)

    paths = list_date_folder(date_folder)
    with open(manifest_path, "w") as f:
        json.dump(paths, f)
    print(f"Created manifest with {len(paths)} files")
    return paths

def backoff_retry(fn, max_tries=5):
    for attempt in range(1, max_tries + 1):
        try:
            return fn()
        except requests.exceptions.HTTPError as e:
            if e.response.status_code == 429:
                wait = 360
                print(f"HF 429 rate limit — waiting {wait}s (attempt {attempt})", file=sys.stderr)
                time.sleep(wait)
                continue
            raise
        except Exception:
            if attempt == max_tries:
                raise
            wait = 2 ** attempt
            print(f"Retry {attempt}/{max_tries} after {wait}s: {sys.exc_info()[0]}", file=sys.stderr)
            time.sleep(wait)
