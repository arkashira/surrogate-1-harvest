**Final Implementation Plan (≤2h)**

### Step 1: Pre-flight snapshot generation

Create a new script `bin/snapshot.sh` that generates a pre-flight snapshot of the dataset:

```bash
#!/bin/bash

# Define dataset repo and path
REPO="axentx/surrogate-1-training-pairs"
PATH="public"

# Download the dataset files
for file in $(huggingface_hub.download_dataset "$REPO", "$PATH", streaming=True); do
  echo "$file"
done > snapshot.txt
```

### Step 2: Update `dataset-enrich.sh` to use the snapshot

Modify the `dataset-enrich.sh` script to use the pre-flight snapshot instead of querying the dataset directly:

```bash
#!/bin/bash

# Define dataset repo and path
REPO="axentx/surrogate-1-training-pairs"
PATH="public"

# Load the pre-flight snapshot
files=$(cat snapshot.txt)

# Process the files
for file in $files; do
  # ...
done
```

### Step 3: Verify the snapshot updates

Run the `dataset-enrich.sh` script with the updated snapshot and verify that the output is correct.

### Step 4: Update the GitHub Actions workflow

Update the GitHub Actions workflow to use the pre-flight snapshot instead of querying the dataset directly.

```yml
name: Ingest

on:
  workflow_dispatch:
    inputs:
      REPO:
        description: Dataset repository
        required: true
      PATH:
        description: Dataset path
        required: true

jobs:
  ingest:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout code
        uses: actions/checkout@v2

      - name: Download snapshot
        run: |
          curl -o snapshot.txt https://raw.githubusercontent.com/axentx/surrogate-1-runner/main/bin/snapshot.txt

      - name: Enrich dataset
        run: |
          ./bin/dataset-enrich.sh
```
