- **Deterministic sharding**: `hash(slug) % SHARD_TOTAL == SHARD_ID` (Candidate 2) — avoids coordination and guarantees idempotent, non-overlapping work.
- **Single manifest call**: `list_repo_tree(path=DATE, recursive=False)` → `manifest.json` (Candidate 2). Fast, explicit, reproducible.
- **CDN bypass**: `https://huggingface.co/datasets/.../resolve/main/...` with no auth header (Candidate 2) — removes API rate limits for reads.
- **Schema-safe parsing**: stream-parse each file and project to `{prompt, response}` at parse time (Candidate 2) — avoids mixed-schema pyarrow errors.
- **Dedup**: reuse existing `lib/dedup.py` (md5 store) (Candidate 2). Central, persistent, correct.
- **Output**: `batches/public-merged/{DATE}/shard{N}-{HHMMSS}.jsonl` (Candidate 2) — simple, timestamped, shard-labeled.
- **Retry/backoff**: exponential backoff + 429 wait 360s (Candidate 2). Respects HF limits and commit caps.
- **No Lightning Studio reuse** (HF Space only) — removed to avoid scope creep (Candidate 2).
- **Parameter source**: environment variables (not CLI args) to match GitHub Actions matrix (Candidate 2).

def safe_get(url: str, headers: Optional[Dict[str, str]] = None, retries: int = 5) -> requests.Response:
    for attempt in range(1, retries + 1):
        try:
            resp = requests.get(url, headers=headers, timeout=30, stream=True)
            if resp.status_code == 429:
                wait = 360
                log.warning("Rate-limited 429; waiting %ss (attempt %d/%d)", wait, attempt, retries)
                time.sleep(wait)
                continue
            resp.raise_for_status()
            return resp
        except requests.RequestException as e:
            if attempt == retries:
                raise
            sleep_sec = 2 ** attempt
            log.warning("Request failed (%s), retry %d/%d in %ss", e, attempt, retries, sleep_sec)
            time.sleep(sleep_sec)
    raise RuntimeError("Exhausted retries")
