# Design Improvement: Durable Ingestion Manifest + CDN-Bypass Training Loader

## 1. Diagnosis
- **No durable ingestion manifest**: every training run re-lists HF repos via API → guaranteed 429s and quota burn.
- **Training uses HF API data loading** (`load_dataset`/`list_repo_files`) instead of CDN bypass → rate limits during data loader stalls training.
- **No Lightning Studio reuse guard**: scripts create new studios instead of reusing running ones → wastes ~80hr/mo quota.
- **No idle-stop resilience**: Lightning idle timeout kills training; no pre-run status check or auto-restart.
- **Schema leakage risk**: raw files with mixed schema written to `enriched/` instead of projecting to `{prompt,response}` and using deterministic filename patterns.

## 2. Proposed change
- Add `/opt/axentx/vanguard/manifest/` with:
  - `create_manifest.py` — one-time Mac-side HF API call to list a date folder and emit `manifest.json` (repo, path, sha, url).
  - `train_cdn.py` — Lightning training script that loads `manifest.json` and fetches only via CDN (`/resolve/main/...`) with zero API calls.
- Add `lightning_utils.py` with studio reuse + idle-stop guard.
- Add `project_schema.py` to enforce `{prompt,response}` projection and deterministic filename pattern for uploads.

## 3. Implementation

### manifest/create_manifest.py
```python
#!/usr/bin/env python3
"""
Create durable manifest for a date folder.
Run from Mac (or any env with HF token) once per folder.
Usage:
  HF_TOKEN=hf_xxx python create_manifest.py \
    --repo datasets/axentx/surrogate-1 \
    --date 2026-05-02 \
    --out manifest/manifest.json
"""
import argparse
import json
import os
from pathlib import Path

from huggingface_hub import HfApi, list_repo_tree

HF_API = HfApi()

def build_manifest(repo_id: str, date: str, out_path: Path):
    prefix = f"{date}/"
    entries = []

    # Single API call: non-recursive per folder to avoid pagination explosion
    items = list_repo_tree(repo_id=repo_id, path_in_repo=prefix, recursive=False)
    for item in items:
        if item.type != "file":
            continue
        cdn_url = f"https://huggingface.co/datasets/{repo_id}/resolve/main/{item.path}"
        entries.append({
            "repo": repo_id,
            "path": item.path,
            "sha": getattr(item, "oid", None),
            "cdn_url": cdn_url,
            "size": getattr(item, "size", None),
        })

    manifest = {
        "repo": repo_id,
        "date": date,
        "generated_by": "create_manifest.py",
        "entries": entries,
    }

    out_path.parent.mkdir(parents=True, exist_ok=True)
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(entries)} entries to {out_path}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="e.g. datasets/axentx/surrogate-1")
    parser.add_argument("--date", required=True, help="YYYY-MM-DD")
    parser.add_argument("--out", default="manifest/manifest.json")
    args = parser.parse_args()

    build_manifest(args.repo, args.date, Path(args.out))
```

### lightning_utils/studio.py
```python
#!/usr/bin/env python3
"""
Lightning Studio reuse + idle-stop guard.
"""
import time
from lightning import Studio, Teamspace, Machine

def get_or_create_studio(name: str, machine: Machine = Machine.L40S) -> Studio:
    """
    Reuse a running studio if exists; otherwise create.
    Avoids quota waste from repeated creation.
    """
    teamspace = Teamspace()
    for studio in teamspace.studios:
        if studio.name == name:
            if studio.status == "Running":
                print(f"Reusing running studio: {name}")
                return studio
            else:
                print(f"Studio {name} exists but stopped. Restarting...")
                studio.start(machine=machine)
                # Wait until running
                while studio.status != "Running":
                    time.sleep(5)
                    studio.refresh()
                return studio

    print(f"Creating studio: {name}")
    return Studio(name=name, create_ok=True, machine=machine)

def ensure_running(studio: Studio, machine: Machine = Machine.L40S) -> Studio:
    """
    Guard against Lightning idle-stop killing training.
    Call before each .run().
    """
    studio.refresh()
    if studio.status != "Running":
        print(f"Studio stopped. Restarting on {machine}...")
        studio.start(machine=machine)
        while studio.status != "Running":
            time.sleep(5)
            studio.refresh()
    return studio
```
