## Final Implementation Plan (Synthesized)

**Chosen approach**: CDN-only training pipeline with pre-computed file manifest.  
**Why**: Eliminates HF API 429s immediately, requires no model/training-loop changes, and can ship in ≤2h with zero infra spin-up.

## 1) Pre-compute file manifest (run once per batch)

**Command** (run on Mac or any dev machine after rate-limit window):
```bash
python scripts/gen_manifest.py \
  --repo "axentx/surrogate-1-dataset" \
  --date-folder "batches/mirror-merged/2026-05-02" \
  --out "data/manifest-2026-05-02.json"
```

**`scripts/gen_manifest.py`**
```python
#!/usr/bin/env python3
"""
Generate a CDN-only manifest for surrogate-1 training.
Usage:
    HF_TOKEN=<token> python scripts/gen_manifest.py \
        --repo axentx/surrogate-1-dataset \
        --date-folder batches/mirror-merged/2026-05-02 \
        --out data/manifest-2026-05-02.json
"""
import argparse
import json
import os
import sys
from pathlib import Path

from huggingface_hub import HfApi

def build_args() -> argparse.Namespace:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="HF dataset repo id")
    parser.add_argument("--date-folder", required=True, help="Subfolder containing parquet files")
    parser.add_argument("--out", required=True, help="Output JSON manifest path")
    parser.add_argument("--ext", default=".parquet", help="File extension filter")
    return parser.parse_args()

def main() -> None:
    args = build_args()
    api = HfApi(token=os.getenv("HF_TOKEN"))

    try:
        entries = api.list_repo_tree(
            repo_id=args.repo,
            path=args.date_folder,
            recursive=False
        )
    except Exception as exc:
        print(f"Failed to list repo tree: {exc}", file=sys.stderr)
        sys.exit(1)

    files = []
    for e in entries:
        etype = getattr(e, "type", None)
        path = getattr(e, "path", "")
        if etype == "file" and path.endswith(args.ext):
            files.append({
                "path": path,
                "cdn_url": f"https://huggingface.co/datasets/{args.repo}/resolve/main/{path}"
            })

    out_path = Path(args.out)
    out_path.parent.mkdir(parents=True, exist_ok=True)
    with out_path.open("w") as f:
        json.dump(files, f, indent=2)

    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    main()
```

## 2) CDN-only dataset loader (drop-in replacement)

**`surrogate/data/cdn_parquet_loader.py`**
```python
import json
from io import BytesIO
from typing import Dict, List

import pyarrow.parquet as pq
import requests

class CDNParquetLoader:
    """
    Zero HF API dataset loader.
    Reads parquet files directly from CDN using a pre-computed manifest.
    Projects to {prompt, response} at parse time.
    """

    def __init__(self, manifest_path: str) -> None:
        with open(manifest_path) as f:
            self.manifest: List[Dict] = json.load(f)

    def _stream_parquet(self, cdn_url: str) -> BytesIO:
        resp = requests.get(cdn_url, timeout=30)
        resp.raise_for_status()
        return BytesIO(resp.content)

    def load(self) -> List[Dict]:
        rows: List[Dict] = []
        for item in self.manifest:
            try:
                buf = self._stream_parquet(item["cdn_url"])
                table = pq.read_table(buf, columns=["prompt", "response"])
                for row in table.to_pylist():
                    rows.append({
                        "prompt": row.get("prompt"),
                        "response": row.get("response"),
                    })
            except Exception as exc:
                print(f"Skipping {item['path']}: {exc}")
                continue
        return rows
```

## 3) Wire into training script (minimal change)

Replace:
```python
from datasets import load_dataset
ds = load_dataset("axentx/surrogate-1-dataset", split="train", streaming=True)
```

With:
```python
from surrogate.data.cdn_parquet_loader import CDNParquetLoader
from datasets import Dataset

loader = CDNParquetLoader("data/manifest-2026-05-02.json")
rows = loader.load()
ds = Dataset.from_list(rows)
```

If you prefer to keep streaming behavior, convert to an iterable:
```python
def iter_rows(loader: CDNParquetLoader):
    for item in loader.manifest:
        try:
            buf = loader._stream_parquet(item["cdn_url"])
            table = pq.read_table(buf, columns=["prompt", "response"])
            for row in table.to_pylist():
                yield {"prompt": row.get("prompt"), "response": row.get("response")}
        except Exception as exc:
            print(f"Skipping {item['path']}: {exc}")
            continue

ds = iter_rows(loader)
```

## 4) Studio reuse + idle handling (Lightning AI)

from lightning_sdk import Teamspace, Studio, Machine

teamspace = Teamspace()
studio = None
for s in teamspace.studios():
    if s.name == "surrogate-1-train" and s.status == "Running":
        studio = s
        break

if studio is None:
    studio = teamspace.create_studio(
        name="surrogate-1-train",
        machine=Machine.L40S  # or fallback to free-tier available GPU
    )

if studio.status != "Running":
    studio.start(machine=Machine.L40S)

# Safe to run
studio.run(["python", "train.py", "--manifest", "data/manifest-2026-05-02.json"])
