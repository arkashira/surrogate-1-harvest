# Costinel / design

## Final Consolidated Implementation (≤2h)

**Goal:**  
Ship a **deterministic, audit-ready discovery module** that produces machine-readable manifests and a top-hub insight snapshot — enabling **visibility-first onboarding (Sense + Signal — ไม่ Execute)** with zero execution risk.

**Why this:**  
- Highest immediate value: repeatable, traceable onboarding with concrete artifacts.  
- Fits ≤2h: single script + manifest templates + lightweight cloud enumeration.  
- Aligns with past patterns: review top-hub before planning; produce deterministic outputs; avoid execution.

---

## Implementation Plan (Concrete + Actionable)

1. **Create directory structure**  
   ```bash
   mkdir -p manifests src/discovery scripts
   ```

2. **Implement deterministic discovery script** (`src/discovery/discover.py`)  
   - **Inputs:** cloud credentials via env only (AWS_PROFILE/AWS_REGION, GOOGLE_APPLICATION_CREDENTIALS, AZURE_SUBSCRIPTION_ID).  
   - **Outputs:**  
     - `manifests/sources.json` — enumerated accounts/projects/subscriptions with auth metadata and collection scopes.  
     - `manifests/discovery.json` — discovered resources, tags, anomalies (idle/untagged), and **actionability scores** (0–100).  
     - `manifests/top-hub-insight.json` — most-connected hub summary (e.g., MOC) with recommendations and generated_at.  
   - **Determinism:** stable sort keys, UTC timestamps, SHA-256 of inputs for traceability.

3. **Add CLI wrapper** (`scripts/discover.sh`)  
   - Shebang `#!/usr/bin/env bash`, strict mode (`set -euo pipefail`).  
   - Supports `--dry-run` to validate credentials and print counts without writes.  
   - Exit codes: 0=success, 1=general error, 2=validation/auth failure.

4. **Add top-hub insight snapshot**  
   - Use existing knowledge-RAG graph to identify top hub; fallback to deterministic placeholder if unavailable.  
   - Export summary with `hub`, `connections`, `recommendations`, `generated_at`.

5. **Add lightweight validation**  
   - JSON schema validation for all outputs.  
   - Dry-run mode for safe testing.

6. **Update README**  
   - Add quick-start for discovery.  
   - Emphasize: **“Sense + Signal — ไม่ Execute”**.

```bash
#!/usr/bin/env bash
set -euo pipefail

# Costinel Discovery Module — Sense + Signal (ไม่ Execute)

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "$SCRIPT_DIR/.." && pwd)"
cd "$PROJECT_ROOT"

DRY_RUN=0
if [[ "${1:-}" == "--dry-run" ]]; then
  DRY_RUN=1
fi

export PYTHONPATH="$PROJECT_ROOT/src:$PYTHONPATH"

if [[ $DRY_RUN -eq 1 ]]; then
  echo "== DRY RUN =="
  python -m discovery.discover --dry-run
else
  python -m discovery.discover
fi
```

#!/usr/bin/env python3
"""
Costinel Discovery Module
Produces:
  - manifests/sources.json
  - manifests/discovery.json
  - manifests/top-hub-insight.json

Sense + Signal — ไม่ Execute
"""

import json
import os
import sys
import hashlib
import datetime
import argparse
from typing import Any, Dict, List

MANIFESTS_DIR = os.path.join(os.path.dirname(os.path.dirname(os.path.dirname(__file__))), "manifests")

def ensure_manifests_dir() -> None:
    os.makedirs(MANIFESTS_DIR, exist_ok=True)

def utc_now() -> str:
    return datetime.datetime.utcnow().replace(microsecond=0).isoformat() + "Z"

def stable_hash(obj: Any) -> str:
    return hashlib.sha256(json.dumps(obj, sort_keys=True, separators=(",", ":")).encode()).hexdigest()

def discover_aws() -> Dict[str, Any]:
    try:
        import boto3
        sts = boto3.client("sts")
        caller = sts.get_caller_identity()
        accounts = [
            {
                "provider": "aws",
                "account_id": caller["Account"],
                "arn": caller["Arn"],
                "user_id": caller["UserId"],
                "region": os.getenv("AWS_REGION", "us-east-1"),
                "profile": os.getenv("AWS_PROFILE", "default"),
                "collection_scope": "organization" if "organizations" in boto3.Session().get_available_services() else "account-only",
            }
        ]
        return {"accounts": accounts, "error": None}
    except Exception as e:
        return {"accounts": [], "error": str(e)}

def discover_gcp() -> Dict[str, Any]:
    try:
        import google.auth
        import googleapiclient.discovery
        credentials, project_id = google.auth.default()
        service = googleapiclient.discovery.build("cloudresourcemanager", "v1", credentials=credentials)
        request = service.projects().list()
        projects = []
        while request is not None:
            response = request.execute()
            projects.extend(response.get("projects", []))
            request = service.projects().list_next(request, response)
        return {
            "projects": projects,
            "default_project": project_id,
            "error": None,
        }
    except Exception as e:
        return {"projects": [], "error": str(e)}

def discover_azure() -> Dict[str, Any]:
    sub_id = os.getenv("AZURE_SUBSCRIPTION_ID", "")
    tenant = os.getenv("AZURE_TENANT_ID", "")
    if not sub_id:
        return {"subscriptions": [], "error": "AZURE_SUBSCRIPTION_ID not set"}
    return {
        "subscriptions": [{"subscription_id": sub_id, "tenant_id": tenant}],
        "error": None,
    }

def build_sources() -> Dict[str, Any]:
    sources = {
        "generated_at": utc_now(),
        "schema": "costinel/sources/v1",
        "aws": discover_aws(),
        "gcp": discover_gcp(),
        "azure": discover_azure(),
    }
    sources["sha256"] = stable_hash(sources)
    return sources

def build_discovery(sources: Dict[str, Any]) -> Dict[str, Any]:
    resources = []
    anomalies = []

    for aws_acc in sources.get("aws", {}).get("accounts", []):
        resources.append({
            "provider": "aws",
            "account_id": aws_acc["account_id"],
            "type": "account",
            "tags": {},
            "status": "active",
        })
        anomalies.append({
            "provider": "aws",
            "account_id": aws_acc["account_id"],
            "type": "untagged-expectation",
            "severity": "low",
            "actionability_score": 70,
            "message": "Enable tag policies for cost allocation",
        })

    discovery = {
        "generated_at": utc_now(),
        "schema": "costinel/discovery/v1",
        "sources_sha256": sources.get("sha256"),
        "resources": resources,
        "anomalies": anomalies,
        "summary": {
            "resource_count": len(resources),
            "anomaly_count": len(anomalies),
        },
    }
    discovery["sha256"] = stable_hash(discovery)
    return discovery
