# airship / backend

## Decision
Ship a **cron-safe, zero-config backend orchestrator** (`airship discover`) that:
1. Runs market research + knowledge-rag to produce tagged insights (business-research, knowledge-rag, graph, hub).
2. Pre-lists HF dataset files once (date folder) and emits a CDN-only training manifest (bypasses API rate limits during training).
3. Emits actionable outputs for downstream surrogate training and incident/intent pipelines.

This is the highest-value incremental improvement (<2h) that unlocks:
- Safe HF ingestion (rate-limit + commit-cap mitigations).
- Reusable Lightining Studio training pattern (CDN-only, no API calls during data load).
- Tagged research artifacts for RAG/Graph.

---

## Implementation Plan

### Files to create/modify
- `/opt/axentx/airship/bin/airship-discover` — executable orchestrator (entrypoint).
- `/opt/axentx/airship/lib/airship-discover.sh` — reusable library (discover + manifest logic).
- `/opt/axentx/airship/bin/train-surrogate-cdn` — lightweight training launcher (uses CDN manifest; Lightning Studio friendly).
- `/opt/axentx/airship/config/airship.conf` — minimal config (HF repo, date folder, output paths).

### Steps (script logic)
1. **Research phase**
   - Run `granite-business-research.sh` (if present) or internal research pipeline.
   - Run knowledge-rag to query top hub and related docs; emit tagged insights JSON.
   - Tag outputs: `#business-research #knowledge-rag #graph #hub`.

2. **HF pre-list phase (Mac/orchestrator only)**
   - Use HF API once (after rate-limit window) to `list_repo_tree(path, recursive=False)` for target date folder.
   - Save list to `manifest/YYYY-MM-DD/filelist.json`.
   - Embed file list into training script inputs (CDN URLs only: `https://huggingface.co/datasets/{repo}/resolve/main/{path}`).

3. **Emit artifacts**
   - `out/research/YYYY-MM-DD/insights.json` (tagged).
   - `out/manifest/YYYY-MM-DD/filelist.json` (CDN-only).
   - `out/manifest/YYYY-MM-DD/train-cdn.sh` (wrapper for Lightning training).

4. **Cron safety**
   - Shebang `#!/usr/bin/env bash`, `set -euo pipefail`, idempotent writes.
   - Ensure `SHELL=/bin/bash` in crontab (per pattern).
   - Lockfile to prevent overlapping runs (`/tmp/airship-discover.lock`).

#!/usr/bin/env bash
# Library: airship-discover
# Cron-safe orchestration utilities for research + HF CDN manifest generation
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "${SCRIPT_DIR}/.." && pwd)"

# Load config
CONFIG="${PROJECT_ROOT}/config/airship.conf"
if [[ -f "${CONFIG}" ]]; then
  # shellcheck source=/dev/null
  source "${CONFIG}"
else
  echo "Config not found: ${CONFIG}" >&2
  exit 1
fi

OUT_RESEARCH="${OUT_BASE}/research"
OUT_MANIFEST="${OUT_BASE}/manifest"
LOCKFILE="/tmp/airship-discover.lock"

acquire_lock() {
  if [[ -f "${LOCKFILE}" ]]; then
    local pid
    pid=$(cat "${LOCKFILE}" 2>/dev/null || true)
    if kill -0 "${pid}" 2>/dev/null; then
      echo "Another instance is running (PID ${pid}). Exiting." >&2
      exit 1
    else
      rm -f "${LOCKFILE}"
    fi
  fi
  echo $$ > "${LOCKFILE}"
}

release_lock() {
  rm -f "${LOCKFILE}"
}

run_research() {
  local date_tag="$1"
  local out_dir="${OUT_RESEARCH}/${date_tag}"
  mkdir -p "${out_dir}"

  echo "Running business research..." >&2
  if command -v granite-business-research.sh >/dev/null 2>&1; then
    granite-business-research.sh --out "${out_dir}/business.json" || true
  fi

  echo "Running knowledge-rag (top hub + related docs)..." >&2
  if command -v knowledge-rag >/dev/null 2>&1; then
    knowledge-rag --query "top hub" --out "${out_dir}/hub.json" --tags "${RESEARCH_TAGS}" || true
    knowledge-rag --query "related docs" --out "${out_dir}/related.json" --tags "${RESEARCH_TAGS}" || true
  fi

  # Combine insights
  jq -n \
    --arg date "${date_tag}" \
    --arg tags "${RESEARCH_TAGS}" \
    '{
      date: $date,
      tags: ($tags | split(",")),
      research: (if (("'"${out_dir}"'/business.json" | @sh) | . != "") then (input) else null end),
      hub: (if (("'"${out_dir}"'/hub.json" | @sh) | . != "") then (input) else null end),
      related: (if (("'"${out_dir}"'/related.json" | @sh) | . != "") then (input) else null end)
    }' \
    "${out_dir}/business.json" "${out_dir}/hub.json" "${out_dir}/related.json" \
    > "${out_dir}/insights.json" 2>/dev/null || true

  echo "Research artifacts written to ${out_dir}" >&2
}
