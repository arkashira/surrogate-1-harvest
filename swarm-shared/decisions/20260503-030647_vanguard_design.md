# vanguard / design

## 1. Diagnosis

- No deterministic CDN-first manifest exists; training scripts likely call `list_repo_tree` or `load_dataset` at runtime, risking 429s and non-reproducible epochs.
- Mixed-schema ingestion probably writes enriched/ files with extra metadata columns (`source`, `ts`) that break surrogate-1 schema expectations and waste I/O.
- No content-addressed, pinned file list → training runs can diverge across restarts if HF repo changes between runs.
- Lightning Studio reuse is not enforced; each run may create a new studio and burn quota instead of reusing a running one.
- No explicit idle-stop guard before `.run()`; Lightning idle timeout can kill training without restart logic.

## 2. Proposed change

Create `/opt/axentx/vanguard/design/cdn_manifest.py` (single-purpose, <200 LOC) that:
- Performs **one** non-recursive `list_repo_tree` per date folder (Mac-side, offline) and writes `manifest-{date}.json` with deterministic ordering and SHA-256 per file.
- Projects each file to `{prompt, response}` only (schema enforcement) and stores attribution in filename pattern `batches/mirror-merged/{date}/{slug}.parquet`.
- Provides a `load_local()` helper for Lightning training that uses **CDN-only** URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with zero API calls during data load.
- Adds a small `studio_reuse_or_start()` wrapper that lists existing studios, reuses a running one by name, or starts an L40S (fallback) with idle-stop guard.

Scope: new file only (no edits to existing files unless they exist and are obvious — none seen).

## 3. Implementation

CDN-first manifest generator + loader for vanguard surrogate-1 training.
- Generates deterministic manifest (date folder) with SHA-256 per file.
- Projects to {prompt, response} only; attribution via filename.
- Training uses CDN URLs only (zero HF API calls at runtime).
"""

from __future__ import annotations

import json

# ---- Manifest generation (run on Mac, orchestration only) ----

def list_date_folder(repo: str, date_folder: str) -> List[Dict[str, Any]]:
    """
    Non-recursive directory listing for a single date folder.
    Returns list of dicts with at least 'path' and 'size'.
    """
    tree = API.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    items = []
    for entry in tree:
        if entry.type == "file":
            items.append({"path": entry.path, "size": getattr(entry, "size", 0)})
    # Deterministic ordering
    items.sort(key=lambda x: x["path"])
    return items

def sha256_file(path: str) -> str:
    h = hashlib.sha256()
    with open(path, "rb") as f:
        for chunk in iter(lambda: f.read(1 << 20), b""):
            h.update(chunk)
    return h.hexdigest()

def project_to_prompt_response(local_path: str) -> Dict[str, str]:
    """
    Lightweight projection to {prompt, response}.
    Supports common HF dataset file types minimally (jsonl, parquet via pandas if available).
    Attribution moved to filename; no extra metadata columns.
    """
    p = Path(local_path)
    if p.suffix == ".jsonl":
        import json as _json
        rows = []
        with open(p, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                obj = _json.loads(line)
                rows.append({
                    "prompt": str(obj.get("prompt", obj.get("input", ""))),
                    "response": str(obj.get("response", obj.get("output", "")))
                })
        return {"rows": rows}
    elif p.suffix == ".parquet":
        try:
            import pandas as pd
            df = pd.read_parquet(p)
            df = df.rename(columns={
                c: c.lower() for c in df.columns
            })
            prompt_col = next((c for c in df.columns if "prompt" in c or "input" in c), None)
            response_col = next((c for c in df.columns if "response" in c or "output" in c), None)
            if prompt_col is None or response_col is None:
                raise ValueError("Parquet missing prompt/response columns")
            df = df[[prompt_col, response_col]].rename(columns={prompt_col: "prompt", response_col: "response"})
            df["prompt"] = df["prompt"].astype(str)
            df["response"] = df["response"].astype(str)
            return {"rows": df.to_dict(orient="records")}
        except ImportError:
            raise RuntimeError("pandas required for parquet projection")
    else:
        raise ValueError(f"Unsupported file type: {p.suffix}")

def build_manifest(repo: str, date_folder: str, out_dir: str = "manifests") -> Dict[str, Any]:
    """
    Build manifest for date_folder.
    Downloads each file temporarily, projects, records sha256 and CDN URL.
    """
    items = list_date_folder(repo, date_folder)
    if not items:
        raise ValueError(f"No files in {repo}/{date_folder}")

    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "generated_at": datetime.utcnow().isoformat() + "Z",
        "files": []
    }

    os.makedirs(out_dir, exist_ok=True)

    for item in items:
        path = item["path"]
        slug = Path(path).stem
        date_str = Path(date_folder).name
        local_tmp = hf_hub_download(repo_id=repo, filename=path, repo_type="dataset")
        projected = project_to_prompt_response(local_tmp)
        file_hash = sha256_file(local_tmp)

        manifest["files"].append({
            "path": path,
            "sha256": file_hash,
            "cdn_url": CDN_TEMPLATE.format(repo=repo, path=path),
            "rows": len(projected["rows"]),
            "slug": slug,
            # Keep minimal metadata; attribution via filename pattern in downstream usage
        })

    manifest_path = Path(out_dir) / f"manifest-{date_folder.rstrip('/').replace('/', '-')}.json"
    manifest_path.write_text(json.dumps(manifest, indent=2), encoding="utf-8")
    print(f"Manifest written: {manifest_path}")
    return manifest
