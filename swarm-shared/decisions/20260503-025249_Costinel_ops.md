# Costinel / ops

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h, telemetry-aware)

### What we ship
- A **non-blocking Top-Hub Signal Panel** mounted on the Costinel dashboard sidebar/top area.
- Defaults to hub **MOC** (configurable via `VITE_HUB_NAME`).
- Shows: hub title, short description, top 5 signals (anomalies/recommendations), last updated time, and a “View in Knowledge Hub” link.
- **CDN-first data path**: single JSON file per hub at `https://huggingface.co/datasets/axentx/knowledge-hub/resolve/main/hubs/{hubName}.json` — zero API calls during runtime.
- **Telemetry-aware**: respects `VITE_TELEMETRY_ENABLED` and emits lightweight panel events (view, click, refresh) only when enabled.
- **Graceful fallback**: if CDN fetch fails or JSON malformed, panel collapses to minimal state with retry button (no crash, no console spam).
- **No build-time coupling**: panel is pure runtime fetch; works in dev and production without repo rebuilds.

### Files to create/modify
- `src/components/TopHubSignalPanel.vue` (new)
- `src/components/TopHubSignalPanel.css` (new, scoped styles)
- `src/lib/hubClient.ts` (new, CDN fetcher + cache)

### Acceptance criteria

- No console errors on CDN 404/timeout.
- Telemetry events only sent when `VITE_TELEMETRY_ENABLED=true`.
- Configurable hub via `VITE_HUB_NAME` (default: `MOC`).
- Mobile responsive (collapses to compact view <640px).

```ts
// src/lib/hubClient.ts
const CACHE_TTL_MS = 5 * 60 * 1000; // 5 minutes

export interface HubSignal {
  id: string;
  title: string;
  summary: string;
  severity: 'critical' | 'high' | 'medium' | 'low' | 'info';
  link?: string;
  ts?: string; // ISO
}

export interface HubData {
  hubName: string;
  title: string;
  description: string;
  signals: HubSignal[];
  updatedAt: string; // ISO
  sourceUrl?: string;
}

class HubClient {
  private cache: { data: HubData; ts: number } | null = null;

  private buildCdnUrl(hubName: string): string {
    // Public CDN URL — no Authorization header required
    return `https://huggingface.co/datasets/axentx/knowledge-hub/resolve/main/hubs/${encodeURIComponent(
      hubName
    )}.json`;
  }

  private isFresh(): boolean {
    return !!this.cache && Date.now() - this.cache.ts < CACHE_TTL_MS;
  }

  async fetchHub(hubName: string, { forceRefresh = false } = {}): Promise<HubData | null> {
    if (!forceRefresh && this.isFresh()) {
      return this.cache!.data;
    }

    const url = this.buildCdnUrl(hubName);
    const controller = new AbortController();
    const timeout = setTimeout(() => controller.abort(), 8000);

    try {
      const res = await fetch(url, {
        method: 'GET',
        signal: controller.signal,
        cache: 'no-store',
      });
      clearTimeout(timeout);

      if (!res.ok) {
        // Do not throw — allow graceful fallback
        console.warn(`[HubClient] CDN responded ${res.status} for ${hubName}`);
        return null;
      }

      const json = (await res.json()) as HubData;
      // Minimal validation
      if (!json || typeof json !== 'object' || !Array.isArray(json?.signals)) {
        console.warn('[HubClient] Invalid hub payload shape', json);
        return null;
      }

      this.cache = { data: json, ts: Date.now() };
      return json;
    } catch (err) {
      clearTimeout(timeout);
      console.warn('[HubClient] Fetch failed', err);
      return null;
    }
  }
}

export const hubClient = new HubClient();
```
