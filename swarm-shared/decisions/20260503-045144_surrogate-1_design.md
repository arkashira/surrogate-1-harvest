# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **`bin/dataset-enrich.sh` → `bin/dataset-enrich.py`**
   - Keep same CLI contract so GitHub Actions matrix doesn’t change.
   - Single API call (once per shard) to list a date folder via `list_repo_tree(recursive=False)`.
   - Persist file list to `manifest-{shard}.json`.
   - During processing, download each file via **CDN URL** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header → bypasses `/api/` rate limits.
   - Stream-parse each file with `pyarrow`/`jsonlines`, project to `{prompt, response}` only, compute `md5`, skip if already in central dedup store.
   - Write output as newline JSON (`.jsonl`) to local temp, then single atomic upload via `hf_hub_upload` to `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl`.

2. **`lib/dedup.py`**
   - Keep SQLite interface; add context manager for safe concurrent access from multiple runners (each runner is isolated, but central store is on HF Space — this guards local retries).
   - Add `bulk_contains` and `bulk_insert` to reduce round-trips.

3. **`requirements.txt`**
   - Add `aiohttp` (for fast parallel CDN downloads) and `tqdm` (optional progress). Keep `datasets`, `huggingface_hub`, `pyarrow`, `numpy`.

4. **`.github/workflows/ingest.yml`**
   - No changes to matrix strategy; only update the run command from `bash bin/dataset-enrich.sh` to `python bin/dataset-enrich.py`.

#!/usr/bin/env python3
"""
CDN-bypass, manifest-driven enrichment worker for surrogate-1.
Usage (same as before):
    python bin/dataset-enrich.py <shard_id> <shard_count>
"""
import json
import hashlib
import os
import sys
import time
from datetime import datetime, timezone
from pathlib import Path
from typing import List, Dict, Any

import aiohttp
import pyarrow as pa
import pyarrow.json as paj
import pyarrow.compute as pc
from huggingface_hub import HfApi, hf_hub_download, list_repo_tree

# -- config --
DATASET_REPO = "axentx/surrogate-1-training-pairs"
HF_TOKEN = os.getenv("HF_TOKEN")
if not HF_TOKEN:
    print("HF_TOKEN missing", file=sys.stderr)
    sys.exit(1)

API = HfApi(token=HF_TOKEN)
DATE_STR = datetime.now(timezone.utc).strftime("%Y-%m-%d")
BATCH_DIR = Path("batches") / "public-merged" / DATE_STR
BATCH_DIR.mkdir(parents=True, exist_ok=True)

# -- dedup --
from lib.dedup import DedupStore

dedup = DedupStore()

# -- helpers --
def md5_bytes(b: bytes) -> str:
    return hashlib.md5(b).hexdigest()

def project_to_pair(raw: Dict[str, Any]) -> Dict[str, str]:
    # Surrogate-1 convention: keep only prompt/response; drop source/ts/extra
    return {
        "prompt": str(raw.get("prompt", raw.get("input", ""))),
        "response": str(raw.get("response", raw.get("output", ""))),
    }

async def download_via_cdn(session: aiohttp.ClientSession, repo: str, path: str) -> bytes:
    # Public CDN URL — no auth header, bypasses API rate limits
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
    async with session.get(url) as resp:
        if resp.status != 200:
            raise RuntimeError(f"CDN download failed {url}: {resp.status}")
        return await resp.read()

async def process_shard(shard_id: int, shard_count: int) -> None:
    # 1) List once per shard (lightweight API call)
    tree = list_repo_tree(repo_id=DATASET_REPO, path="", recursive=False)
    # Assume top-level contains dated folders; pick today's or latest stable folder
    # For simplicity, we process all files in repo root (adjust as needed)
    files = [t.path for t in tree if t.type == "file"]
    # Deterministic shard split by slug-hash
    shard_files = [f for f in files if hashlib.md5(f.encode()).hexdigest() % shard_count == shard_id]

    print(f"Shard {shard_id}/{shard_count} → {len(shard_files)} files")

    # Save manifest for reproducibility / training script embedding
    manifest_path = BATCH_DIR / f"manifest-shard{shard_id}.json"
    manifest_path.write_text(json.dumps({"date": DATE_STR, "shard_id": shard_id, "files": shard_files}, indent=2))

    # 2) CDN download + parse + dedup + project
    out_rows: List[Dict[str, str]] = []
    async with aiohttp.ClientSession() as session:
        for i, path in enumerate(shard_files):
            try:
                data = await download_via_cdn(session, DATASET_REPO, path)
            except Exception as exc:
                print(f"Skip {path}: {exc}")
                continue

            # Try parquet first (common), then json/jsonl fallback
            table = None
            try:
                table = paj.read_table(pa.BufferReader(data))
            except pa.ArrowInvalid:
                pass

            if table is not None:
                cols = table.column_names
                prompt_col = next((c for c in ("prompt", "input") if c in cols), None)
                response_col = next((c for c in ("response", "output") if c in cols), None)
                if prompt_col and response_col:
                    df = table.select([prompt_col, response_col]).to_pylist()
                    for row in df:
                        pair = project_to_pair(row)
                        if not pair["prompt"].strip() or not pair["response"].strip():
                            continue
                        blob = json.dumps(pair, sort_keys=True).encode()
                        h = md5_bytes(blob)
                        if not dedup.contains(h):
                            out_rows.append(pair)
                            dedup.insert(h)
            else:
                # Try jsonlines
                try:
                    import jsonlines
                    for row in jsonlines.read(pa.BufferReader(data)):
                        pair = project_to_pair(row)
                        if not pair["prompt"].strip() or not pair["response"].strip():
                            continue
                        blob = json.dumps(pair, sort_keys=True).encode()
                        h = md5_bytes(blob)
                        if not dedup.contains(h):
                            out_rows.append(pair)
                            dedup.insert(h)
                except Exception:
                    # Last resort: line-by-line JSON
                    for line in data.splitlines():
                        line = line.strip()
                        if not line:
                            continue
                        try:
                            row = json.loads(line)
                        except json.JSONDecodeError:
                            continue
                        pair = project_to_pair(row)
                        if not pair["prompt"].strip() or not pair["response"].strip():
                            continue
                        blob = json.dumps(pair, sort_keys=True).encode()
                        h = md5_bytes(blob)
                        if not dedup.contains(h):
                            out_rows.append(pair)
                            dedup.insert(h)

            if (i + 1) % 10 == 0:
                print(f"  processed {i+1}/{len(shard_files)}")

    # 3) Write and upload
    if not out_rows:
        print("No new rows; skipping upload.")
        return

    ts = datetime.now(timezone.utc).strftime("%H%M%S")
    out_path = BATCH_DIR / f"shard{shard_id}-{ts}.jsonl"

            f.write(json.dumps(row) + "\n")
