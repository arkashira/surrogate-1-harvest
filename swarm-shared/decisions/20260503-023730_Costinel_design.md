# Costinel / design

## Final Synthesis: Top-Hub Signal Panel (CDN-first, zero render-time API)

**Goal:** Ship within ≤2h a visible, high-signal UI that surfaces the most-connected hub (default “MOC”) and its top 3 cost-impact proposals — **no Hugging Face API calls during render**, fully CDN-backed, with graceful fallback.

---

## Why this wins
- **Fastest value**: reuses existing knowledge-graph/RAG assets; no new infra.
- **Correctness first**: eliminates runtime rate-limit risk by moving all heavy calls to build/orchestration time.
- **Actionable**: clear savings signals, deterministic schema, and a concrete path from data generation → CDN → dashboard.
- **Costinel philosophy**: *Sense + Signal — لا تنفذ*.

---

## Canonical JSON Schema (single source of truth)

```json
{
  "hub": "MOC",
  "updated": "2026-05-03T02:35:00Z",
  "signals": [
    {
      "id": "moc-ri-001",
      "title": "RI coverage gap in us-east-1",
      "impact": "high",
      "estimatedSavingsUSD": 42000,
      "description": "35% of steady-state RIs uncovered; recommend 1yr convertible RIs.",
      "action": "review_proposal",
      "proposalId": "prop-ri-2026-05-03-001"
    },
    {
      "id": "moc-orphan-002",
      "title": "Orphaned EBS volumes (prod)",
      "impact": "medium",
      "estimatedSavingsUSD": 8400,
      "description": "12 unattached gp3 volumes >30d; snapshot & delete.",
      "action": "review_proposal",
      "proposalId": "prop-orphan-2026-05-03-002"
    },
    {
      "id": "moc-schedule-003",
      "title": "Non-prod instances nights/weekends",
      "impact": "medium",
      "estimatedSavingsUSD": 15600,
      "description": "Apply start/stop schedule to dev/test instances (70% utilization off-hours).",
      "action": "review_proposal",
      "proposalId": "prop-schedule-2026-05-03-003"
    }
  ]
}
```

- `impact` enum: `"high" | "medium" | "low"`.
- `estimatedSavingsUSD` must be a non-negative number.
- `action` reserved for future extensibility; today only `"review_proposal"` (CTA routes to `/proposals/:id`).

function defaultSignals(hub) {

function extractTopSignals(hubData, hubName) {
  if (Array.isArray(hubData.signals) && hubData.signals.length > 0) {
    return hubData.signals.slice(0, 3).map((s) => ({
      id: s.id || `signal-${Math.random().toString(36).slice(2, 9)}`,
      title: s.title || s.name || 'Untitled signal',
      impact: s.impact || 'medium',
      estimatedSavingsUSD: Number(s.estimatedSavingsUSD) || 0,
      description: s.description || '',
      action: s.action || 'review_proposal',
      proposalId: s.proposalId || s.id || '',
    }));
  }

  // Alternate: nodes/edges heuristic
  if (Array.isArray(hubData.nodes) && Array.isArray(hubData.proposals)) {
    const proposals = hubData.proposals
      .filter((p) => p.hub === hubName)
      .sort((a, b) => (b.impactScore || 0) - (a.impactScore || 0))
      .slice(0, 3);
    return proposals.map((p) => ({
      id: p.id || `prop-${Math.random().toString(36).slice(2, 9)}`,
      title: p.title || 'Untitled proposal',
      impact: p.impact || 'medium',
      estimatedSavingsUSD: Number(p.estimatedSavingsUSD) || 0,
      description: p.description || '',
      action: 'review_proposal',
      proposalId: p.id || '',
    }));
  }

  return [];
}
