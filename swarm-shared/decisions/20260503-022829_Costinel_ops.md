# Costinel / ops

## Implementation Plan — Costinel / ops

**Highest-value incremental improvement (<2h):**  
Add a **read-only Signal Panel** (Top-Hub insights) backed by a **lightweight CDN-first proxy** so Costinel can surface the most-connected hub (e.g., MOC) without runtime HF API calls or schema risks.

**Why this:**  
- Aligns with `#knowledge-rag #graph #hub` (top-hub insight pattern).  
- Uses CDN bypass (`#huggingface #cdn #rate-limit-bypass`) to avoid HF API limits.  
- Keeps Costinel philosophy: **Sense + Signal (ไม่ Execute)** — no writes, no training, no stateful infra.  
- Reuses existing dashboard surface; minimal blast radius.

---

### Steps (concrete, ordered)

1. **Create a small JSON file with pre-listed hub paths** (single date folder)  
   - Generated once on Mac via HF API (after rate-limit window).  
   - Stored in repo under `data/hub_index/{date}/files.json`.

2. **Add a tiny backend proxy endpoint**  
   - Path: `/api/signal/top-hub`  
   - Behavior: reads `files.json`, picks latest, fetches file via CDN (`resolve/main/...`), projects to `{title, score, edges}`, returns JSON.  
   - No Authorization header; CDN-only.  
   - Timeout/retry safe (30s timeout, 1 retry).

3. **Add a frontend panel widget**  
   - Location: Dashboard sidebar or new “Signal” tab.  
   - Shows: Top hub name, score, top 3 related docs, timestamp.  
   - Auto-refresh: 10m (configurable).  
   - Read-only, no mutations.

4. **Add cron-safe script + shebang fix pattern**  
   - Script: `scripts/update-hub-index.sh`  
   - Shebang: `#!/usr/bin/env bash`  
   - Executable: `chmod +x`  
   - Idempotent; logs to `logs/hub-index-update.log`.

5. **Wire into CI/CD** (optional but recommended)  
   - Nightly job runs `update-hub-index.sh`, commits `files.json` if changed.  
   - Keeps CDN list fresh without runtime API usage.

### Code Snippets

#### 1) Pre-list hub index (run once or nightly)

```bash
# scripts/update-hub-index.sh
#!/usr/bin/env bash
set -euo pipefail

REPO="axentx/knowledge-rag"
DATE=$(date +%Y-%m-%d)
OUT_DIR="data/hub_index/${DATE}"
OUT_FILE="${OUT_DIR}/files.json"

mkdir -p "${OUT_DIR}"

# Single non-recursive call (avoids pagination + rate-limit)
# Requires HF_TOKEN in env only for listing; CDN downloads later need no token.
python3 - <<PY
import os, json, sys
from huggingface_hub import list_repo_tree

repo = os.environ.get("HF_REPO", "${REPO}")
date = os.environ.get("DATE", "${DATE}")
out = os.environ.get("OUT_FILE", "${OUT_FILE}")

# List only top-level of date folder
items = list_repo_tree(repo, path=date, recursive=False)
files = [f.rfilename for f in items if f.rfilename.endswith(('.json', '.parquet', '.csv'))]

os.makedirs(os.path.dirname(out), exist_ok=True)
with open(out, "w") as f:
    json.dump({"date": date, "files": files, "repo": repo}, f, indent=2)
print(f"Wrote {len(files)} files to {out}")
PY
```

#### 2) Lightweight backend proxy (Node/Express example)

```js
// backend/routes/signal.js
const express = require('express');
const fetch = require('node-fetch');
const fs = require('fs');
const path = require('path');

const router = express.Router();

function pickLatestFile() {
  const base = path.join(__dirname, '..', 'data', 'hub_index');
  const dirs = fs.readdirSync(base).filter(d => /^\d{4}-\d{2}-\d{2}$/.test(d)).sort().reverse();
  for (const d of dirs) {
    const fp = path.join(base, d, 'files.json');
    if (fs.existsSync(fp)) {
      const idx = JSON.parse(fs.readFileSync(fp, 'utf8'));
      // pick first parquet or json file
      const file = idx.files.find(f => f.endsWith('.parquet') || f.endsWith('.json')) || idx.files[0];
      return { repo: idx.repo, date: idx.date, file };
    }
  }
  return null;
}

async function fetchViaCDN(repo, filePath) {
  const url = `https://huggingface.co/datasets/${repo}/resolve/main/${filePath}`;
  const res = await fetch(url, { timeout: 30000 });
  if (!res.ok) throw new Error(`CDN fetch failed: ${res.status}`);
  // Minimal projection: expect array of {title, score, edges} or parse parquet on server if needed
  // For MVP, assume JSON lines or small JSON array.
  return res.json();
}

router.get('/top-hub', async (req, res) => {
  try {
    const selected = pickLatestFile();
    if (!selected) return res.status(404).json({ error: 'No hub index found' });

    const raw = await fetchViaCDN(selected.repo, selected.file);
    // Project to lightweight shape (adjust per actual schema)
    const top = Array.isArray(raw) ? raw.slice(0, 1)[0] : raw;
    const result = {
      hub: top?.title || 'MOC',
      score: top?.score || 0,
      related: (top?.edges || []).slice(0, 3).map(e => ({ title: e.title, score: e.score })),
      date: selected.date,
      source_file: selected.file,
      generated_at: new Date().toISOString()
    };
    res.json(result);
  } catch (err) {
    console.error('Signal proxy error:', err);
    res.status(502).json({ error: 'Upstream unavailable', details: err.message });
  }
});

module.exports = router;
```
