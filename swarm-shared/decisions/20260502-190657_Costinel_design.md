# Costinel / design

## Costinel — Design Increment (≤2h)

**Objective**  
Ship a deterministic, audit-ready **Discovery Module** that produces machine-readable manifests and a top-hub insight snapshot — enabling **visibility-first onboarding (Sense + Signal — ไม่ Execute)** with zero execution risk.

**Scope (≤2h)**
- CLI: `discovery run` (deterministic, idempotent)
- Outputs:
  - `manifests/sources.json` — declared cloud sources + auth strategy
  - `manifests/discovery.json` — discovered resources + metadata + audit trail
  - `insights/top-hub.md` — top-hub doc insight snapshot (MOC)
- Constraints:
  - No execution/remediation (Sense + Signal only)
  - CDN-only fetches where applicable (HF CDN bypass pattern)
  - Reuse running Lightning Studio if present (quota-safe)
  - Mac orchestration only; training/ingest offloaded to remote

---

## Implementation Plan

1. **Project structure additions** (5 min)
   - Create `costinel/discovery/`, `manifests/`, `insights/`
   - Add deterministic CLI entrypoint `discovery/cli.py`

2. **CLI: `discovery run`** (30 min)
   - Idempotent run with deterministic hash-based slugs
   - Generate `sources.json` from config/env (AWS/GCP/Azure)
   - Discovery collectors per cloud (dry-run/list-only)
   - Produce `discovery.json` with resource metadata + audit trail

3. **Top-hub insight snapshot** (10 min)
   - Query knowledge-rag for top-connected hub (e.g., "MOC")
   - Save summary to `insights/top-hub.md`
   - Tag with `#knowledge-rag #graph #hub`

4. **CDN bypass + file-list strategy** (15 min)
   - If using HF datasets: pre-list via `list_repo_tree` once on Mac
   - Embed file list in training script; Lightning uses CDN-only fetches
   - Avoid `load_dataset(streaming=True)` for mixed-schema repos

5. **Lightning Studio reuse guard** (10 min)
   - Before `Studio(create_ok=True)`, list `Teamspace.studios`
   - Reuse running studio to save quota
   - Handle idle-stop by checking status before `.run()`

6. **Kaggle KGAT auth bypass** (10 min)
   - Use raw HTTP with Bearer token for `kernels/push`
   - Use `slug`, `newTitle`, `text`, `isPrivate=True` for new accounts

7. **Cron/script hygiene** (10 min)
   - Ensure wrapper scripts have `#!/usr/bin/env bash`
   - Set `SHELL=/bin/bash` in crontab
   - Make scripts executable (`chmod +x`)

8. **Tests & validation** (20 min)
   - Dry-run `discovery run` and validate JSON schemas
   - Confirm no execution actions triggered
   - Verify top-hub insight generated

def deterministic_slug(name: str, salt: str = "costinel") -> str:
    return hashlib.sha256(f"{salt}:{name}".encode()).hexdigest()[:12]


def detect_cloud_sources() -> List[Dict[str, Any]]:
    """Detect declared cloud sources (dry-run)."""
    sources = []

    # AWS
    if subprocess.run(["which", "aws"], capture_output=True).returncode == 0:
        try:
            out = subprocess.check_output(
                ["aws", "sts", "get-caller-identity"], stderr=subprocess.DEVNULL
            )
            identity = json.loads(out)
            sources.append(
                {
                    "provider": "aws",
                    "account_id": identity["Account"],
                    "arn": identity["Arn"],
                    "auth_strategy": "aws-cli-profile",
                    "region": subprocess.check_output(
                        ["aws", "configure", "get", "region"], stderr=subprocess.DEVNULL
                    )
                    .decode()
                    .strip(),
                }
            )
        except Exception:
            # No AWS access — skip silently (Sense only)
            pass

    # GCP (dry-run)
    if subprocess.run(["which", "gcloud"], capture_output=True).returncode == 0:
        try:
            out = subprocess.check_output(
                ["gcloud", "config", "list", "--format=value(core.account)"],
                stderr=subprocess.DEVNULL,
            )
            account = out.decode().strip()
            if account:
                sources.append(
                    {
                        "provider": "gcp",
                        "account": account,
                        "auth_strategy": "gcloud-cli",
                    }
                )
        except Exception:
            pass

    # Azure (dry-run)
    if subprocess.run(["which", "az"], capture_output=True).returncode == 0:
        try:
            out = subprocess.check_output(
                ["az", "account", "show"], stderr=subprocess.DEVNULL
            )
            account = json.loads(out)
            sources.append(
                {
                    "provider": "azure",
                    "tenant_id": account.get("tenantId"),
                    "subscription_id": account.get("id"),
                    "auth_strategy": "azure-cli",
                }
            )
        except Exception:
            pass

    return sources
