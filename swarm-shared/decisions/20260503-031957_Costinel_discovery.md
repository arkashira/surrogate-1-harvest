# Costinel / discovery

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h, production-ready)

### Scope (single highest-value deliverable)
Add a **non-blocking Top-Hub Signal Panel** to the Costinel dashboard that:
- Detects and surfaces the most-connected hub (default `MOC`) from a lightweight hub-graph index
- Shows 3 contextual insights from knowledge-rag
- Uses CDN-first pattern (bypasses HF API rate limits)
- Zero impact on existing cost analytics pipeline
- Fully typed, tested, and deployable via existing Docker compose

---

### Architecture (CDN-first)
```
Costinel Dashboard
├── Frontend (React/TS)
│   └── TopHubPanel.tsx (client-side fetch)
├── Backend (FastAPI)
│   └── /api/top-hub (lightweight, CDN-cached)
└── Data Layer
    ├── hub-graph.json (committed to repo, updated by CI)
    └── insights.json (generated by knowledge-rag, committed)
```

### Why this is highest value
- Directly applies **Pattern: top-hub doc insight (2026-04-27)** — surfaces MOC hub context before cost decisions
- Uses **Pattern: HF CDN Bypass** — no API rate limits, instant loading
- Fits **<2h** — reuses existing stack, no new infra
- Aligns with **Costinel philosophy** — "Sense + Signal" (read-only, proposes insights)

#### 2. Backend endpoint (15m) — FastAPI route
File: `/opt/axentx/Costinel/app/api/top_hub.py`
```python
from fastapi import APIRouter
from app.core.config import settings
import json
from pathlib import Path

router = APIRouter()

DATA_DIR = Path(__file__).parent.parent.parent / "data"

@router.get("/top-hub")
async def get_top_hub():
    """CDN-first top hub endpoint — no external API calls"""
    hub_graph_path = DATA_DIR / "hub-graph.json"
    insights_path = DATA_DIR / "insights.json"
    
    try:
        with open(hub_graph_path) as f:
            hub_graph = json.load(f)
        with open(insights_path) as f:
            insights = json.load(f)
    except FileNotFoundError:
        return {
            "top_hub": "MOC",
            "insights": [],
            "status": "fallback",
            "message": "Data files not found, using defaults"
        }
    
    return {
        "top_hub": hub_graph.get("top_hub", "MOC"),
        "hub_graph": hub_graph,
        "insights": insights.get("insights", []),
        "generated_at": insights.get("generated_at"),
        "status": "ok"
    }
```

Register in `/opt/axentx/Costinel/app/api/__init__.py`:
```python
from . import top_hub  # noqa
```

Add to main router in `/opt/axentx/Costinel/app/api/endpoints.py`:
```python

api_router.include_router(top_hub_router, prefix="/top-hub", tags=["top-hub"])
```
