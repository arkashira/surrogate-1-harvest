# Costinel / ops

## Final Implementation Plan — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only card that identifies the most-connected hub from the knowledge-rag graph and surfaces the top hub with 3 contextual signals.  
**Timebox**: <2h (frontend-only, zero backend changes).  
**Principles**: Read-only (Sense + Signal), accessible, responsive, mock-first, future-graph-ready.

---

### 1) Data contract (single source of truth)
Use a merged, stricter shape that supports both UI needs and future API compatibility.

```ts
// src/types/knowledge-rag.ts
export interface Signal {
  id: string;
  title: string;       // display title
  snippet: string;     // short insight
  tag: string;         // category badge
}

export interface TopHub {
  id: string;
  label: string;       // display name
  description: string; // short context
  degree: number;      // connection count
  lastIndexed: string; // ISO timestamp
  signals: Signal[];   // exactly 3 items in mock/UI
}
```

### 3) Component (React + Tailwind)
Chosen for consistency with Candidate 2’s modern stack and Tailwind utility speed. Includes loading/empty states and strict read-only UX.

```tsx
// src/components/costinel/TopHubSignalCard.tsx
import { TopHub } from "@/types/knowledge-rag";
import { topHubMock } from "@/mocks/topHubMock";

interface TopHubSignalCardProps {
  hub?: TopHub | null;
  loading?: boolean;
}

export function TopHubSignalCard({
  hub = topHubMock,
  loading = false,
}: TopHubSignalCardProps) {
  if (loading) {
    return (
      <div className="rounded-xl border border-gray-200 bg-white p-5 shadow-sm">
        <div className="flex items-center gap-3">
          <div className="h-10 w-10 animate-pulse rounded-lg bg-gray-100" />
          <div className="h-5 w-32 animate-pulse rounded bg-gray-100" />
        </div>
        <div className="mt-4 space-y-3">
          <div className="h-4 w-full animate-pulse rounded bg-gray-100" />
          <div className="h-4 w-5/6 animate-pulse rounded bg-gray-100" />
        </div>
        <div className="mt-4 flex gap-2">
          {Array.from({ length: 3 }).map((_, i) => (
            <div key={i} className="h-6 w-20 animate-pulse rounded-full bg-gray-100" />
          ))}
        </div>
      </div>
    );
  }

  if (!hub) {
    return (
      <div className="rounded-xl border border-gray-200 bg-white p-5 shadow-sm">
        <p className="text-sm text-gray-500">Top hub not available.</p>
      </div>
    );
  }

  return (
    <article
      className="rounded-xl border border-gray-200 bg-white p-5 shadow-sm"
      aria-label="Top Hub and signals"
    >
      {/* Header */}
      <div className="flex items-start justify-between gap-3">
        <div className="flex items-center gap-2">
          <svg
            className="h-5 w-5 text-blue-600"
            fill="none"
            stroke="currentColor"
            viewBox="0 0 24 24"
            aria-hidden="true"
          >
            <path
              strokeLinecap="round"
              strokeLinejoin="round"
              strokeWidth={2}
              d="M13 10V3L4 14h7v7l9-11h-7z"
            />
          </svg>
          <h3 className="font-semibold text-gray-900">Top Hub</h3>
        </div>
        <time className="text-xs text-gray-400" dateTime={hub.lastIndexed}>
          Updated {new Date(hub.lastIndexed).toLocaleDateString()}
        </time>
      </div>

      {/* Hub details */}
      <div className="mt-3">
        <p className="text-lg font-medium text-gray-900">{hub.label}</p>
        <p className="mt-1 text-sm text-gray-600">{hub.description}</p>
        <div className="mt-2 flex items-center gap-2">
          <span className="inline-flex items-center rounded-full bg-blue-50 px-2 py-0.5 text-xs font-medium text-blue-700">
            Degree: {hub.degree}
          </span>
        </div>
      </div>

      {/* Signals */}
      <ul className="mt-4 space-y-2" aria-label="Top contextual signals">
        {hub.signals.map((s) => (
          <li
            key={s.id}
            className="flex flex-col gap-1 rounded-lg border border-gray-100 bg-gray-50 p-3"
          >
            <span className="font-medium text-gray-900">{s.title}</span>
            <p className="text-sm text-gray-600">{s.snippet}</p>
            <span className="mt-1 inline-flex w-fit rounded-md bg-gray-200 px-2 py-0.5 text-xs font-medium text-gray-700">
              {s.tag}
            </span>
          </li>
        ))}
      </ul>

      {/* Footer — read-only notice */}
      <footer className="mt-4 text-right text-xs text-gray-400">
        Sense + Signal — read only
      </footer>
    </article>
  );
}
```
