# Costinel / ops

## Final synthesized implementation  

**Goal:** Add a resilient, zero-runtime-HF-API “Top-Hub Signal” panel to Costinel that surfaces the most-connected hub (e.g., MOC) with contextual insights and quick links. No backend changes; deployable in ≤2 hours.

---

## 1) CDN-baked artifact (single source of truth)

- **Location in repo:** `public/signals/top-hub.json` (committed)  
  - Also reachable on HF CDN:  
    `https://huggingface.co/datasets/AXENTX/Costinel/resolve/main/signals/top-hub.json`
- **Why this location:**  
  - `public/` ensures the file is copied into the static build output and served by the same origin (avoids initial CORS/preflight).  
  - HF CDN URL provides an external, rate-limit-free link for source/resource access and for manual fetches if desired.
- **Schema (minimal + extensible):**
  ```json
  {
    "hub": "MOC",
    "title": "Most-Connected Hub",
    "insight": "MOC drives 38% of cross-service cost signals; prioritize RI coverage on linked accounts.",
    "score": 0.94,
    "updated": "2026-05-03T04:22:51Z",
    "generatedAt": "2026-05-03T04:30:00Z",
    "sourcePath": "batches/mirror-merged/2026-05-03/moc-hub.parquet",
    "tags": ["#knowledge-rag", "#graph", "#hub"],
    "links": [
      { "rel": "source", "href": "batches/mirror-merged/2026-05-03/moc-hub.parquet" },
      { "rel": "dashboard", "href": "/hubs/moc" }
    ]
  }
  ```
- **CDN headers (if you control serving or use HF raw URL):**  
  `Cache-Control: public, max-age=60, stale-while-revalidate=300`  
  (Fast updates, safe fallback.)

## 2) Frontend panel (React) — robust and accessible

- **File:** `src/components/TopHubSignalPanel.tsx`
- **Behavior:**
  - Fetch from **local origin** (`/signals/top-hub.json`) at runtime with `cache: 'no-store'` to avoid stale reads during deploys.
  - Optional background polling (60s) for near-real-time updates without page refresh.
  - Graceful fallbacks:
    1. Network or parse error → show friendly message.
    2. If available, fall back to last-known JSON bundled at build time (import a static copy as module fallback).
  - Display:
    - Pill: hub name + title.
    - Insight paragraph.
    - Score + optional sparkline (if `trend` array present).
    - Tag chips.
    - Timestamp and “Source” link (opens HF CDN raw URL to bypass auth/rate-limits).
  - Accessible: ARIA labels, focus order, semantic elements.

  insight: string;

  sourcePath?: string;
  tags?: string[];
  links?: Array<{ rel: string; href: string }>;
  trend?: number[];
}

const CDN_URL = '/signals/top-hub.json';
const HF_CDN_ROOT = 'https://huggingface.co/datasets/AXENTX/Costinel/resolve/main';

  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);
  const timerRef = useRef<NodeJS.Timeout | null>(null);

  const load = async () => {
    try {
      const res = await fetch(CDN_URL, { cache: 'no-store' });
      if (!res.ok) throw new Error(`HTTP ${res.status}`);
      const data = (await res.json()) as TopHubSignal;

      setSignal(data);
      setError(null);

      mounted = false;
      if (timerRef.current) clearInterval(timerRef.current);
    };

  if (loading) {
    return (
      <div className="rounded border border-gray-200 bg-gray-50 p-4" aria-busy="true">
        <p className="text-sm text-gray-500">Loading top-hub signal…</p>
      </div>
    );

  if (error || !signal) {
    return (
      <div className="rounded border border-yellow-100 bg-yellow-50 p-4" role="status">
        <p className="text-sm text-yellow-800">Top-hub signal unavailable.</p>
      </div>
    );
  }

                    <span
                      key={i}
                      className="inline-block w-px bg-blue-500"
                      style={{ height: `${Math.max(2, Math.min(100, v * 100))}%` }}
                    />
                  ))}
                </span>
              )}
            </div>
          )}
