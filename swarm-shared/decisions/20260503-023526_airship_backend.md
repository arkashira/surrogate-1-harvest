## Final Synthesis — CDN-First Surrogate Training (Eliminate HF 429 + Lightning Quota Waste)

I combined the strongest, most actionable parts from both proposals and resolved contradictions in favor of **correctness + concrete actionability**.

---

## 1) Core Design (non-negotiable)

- **Zero HF API calls during training.**  
  Use HF **CDN direct downloads** (`resolve/main/...`) for parquet files.  
  This eliminates 429s caused by `load_dataset`/`list_files` API calls at scale.

- **Deterministic file lists per date folder.**  
  Generate `{date}/file_list.json` listing only parquet files.  
  Commit or embed this alongside training code so training is fully reproducible and CDN-only.

- **Lightning Studio reuse + idle-resume guard.**  
  Before `.run()`, list studios; reuse if running; restart if stopped; create only if absent.  
  This cuts quota waste from repeated studio churn (~80 hr/mo saved).

- **No new infra; single-file changes.**  
  Uses existing HF CDN + Lightning SDK + small orchestration wrapper.

## 3) Implementation Details (resolved contradictions)

### A) Deterministic file-list generation (corrected + robust)

- Use `list_repo_tree(..., recursive=False)` and walk folders manually.  
  This avoids pagination explosion and works for large repos.
- Save only parquet files.
- Make script runnable from cron/orchestrator with env overrides.

`scripts/gen_filelist.py`:

```python
#!/usr/bin/env python3
import os, json, sys
from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_REPO", "axentx/surrogate-dataset")
DATE_FOLDER = os.getenv("DATE_FOLDER", sys.argv[1] if len(sys.argv) > 1 else "latest")
OUT_PATH = os.getenv("OUT_PATH", f"filelists/{DATE_FOLDER}/file_list.json")

api = HfApi()
folders = [DATE_FOLDER]
all_files = []

while folders:
    f = folders.pop(0)
    for item in api.list_repo_tree(repo=HF_REPO, path=f, recursive=False):
        if item.type == "file" and item.path.endswith(".parquet"):
            all_files.append(item.path)
        elif item.type == "folder":
            folders.append(item.path)

os.makedirs(os.path.dirname(OUT_PATH), exist_ok=True)
with open(OUT_PATH, "w") as fp:
    json.dump(sorted(all_files), fp, indent=2)

print(f"Wrote {len(all_files)} parquet files to {OUT_PATH}")
```

### B) CDN-first data loader in `train.py` (streaming + projection)

- Fetch via CDN URL (no auth, no API).
- Stream parquet with `pyarrow` and project to `{prompt, response}`.
- Fallback column detection for variant schemas.
- Yield dicts for easy integration with training loops.

`surrogate/train.py` (excerpt):

```python
import os, json, requests
from io import BytesIO
import pyarrow.parquet as pq

HF_DATASET_REPO = os.getenv("HF_DATASET_REPO", "axentx/surrogate-dataset")
FILE_LIST_PATH = os.getenv("FILE_LIST_PATH", "filelists/latest/file_list.json")
CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def load_file_list():
    with open(FILE_LIST_PATH) as f:
        return json.load(f)

def cdn_fetch(path: str) -> bytes:
    url = CDN_TEMPLATE.format(repo=HF_DATASET_REPO, path=path)
    resp = requests.get(url, timeout=30)
    resp.raise_for_status()
    return resp.content

def stream_parquet_rows(file_list):
    for path in file_list:
        data = cdn_fetch(path)
        table = pq.read_table(BytesIO(data))
        cols = table.column_names

        prompt_col = next((c for c in cols if "prompt" in c.lower()), None)
        response_col = next((c for c in cols if "response" in c.lower() or "completion" in c.lower()), None)

        if prompt_col and response_col:
            df = table.select([prompt_col, response_col]).to_pandas()
            df.columns = ["prompt", "response"]
            for _, row in df.iterrows():
                yield {"prompt": row["prompt"], "response": row["response"]}
        else:
            # skip files without expected schema
            continue
```

### C) Lightning Studio reuse + idle-resume guard (corrected)

- List studios in teamspace.
- Reuse if running.
- Restart if stopped (do not create new).
- Create only if absent.
- Ensure running before submit.

`surrogate/lightning_launcher.py`:

```python
import os
from lightning_sdk import Studio, Teamspace, Machine

TEAMSPACE = os.getenv("LIGHTNING_TEAMSPACE", "default")
STUDIO_NAME = os.getenv("LIGHTNING_STUDIO_NAME", "surrogate-train-l40s")
MACHINE = Machine.L40S

def get_or_create_studio():
    teamspace = Teamspace(name=TEAMSPACE)
    stopped_candidate = None

    for s in teamspace.studios:
        if s.name == STUDIO_NAME:
            if s.status == "running":
                print(f"Reusing running studio: {STUDIO_NAME}")
                return s
            elif s.status == "stopped":
                stopped_candidate = s
                # continue to check for running duplicates (safety)

    if stopped_candidate:
        print(f"Restarting stopped studio: {STUDIO_NAME}")
        stopped_candidate.start(machine=MACHINE)
        return stopped_candidate

    print(f"Creating new studio: {STUDIO_NAME}")
    return Studio(
        name=STUDIO_NAME,
        teamspace=TEAMSPACE,
        machine=MACHINE,
        create_ok=True,
    )

def run_training(script_path: str):
    studio = get_or_create_studio()
    if studio.status != "running":
        studio.start(machine=MACHINE)
    run = studio.run(script_path, environment="surrogate-train")
    print(f"Submitted run: {run.id}")
    return run
```
