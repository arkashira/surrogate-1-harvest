# Costinel / ops

## Implementation Plan — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only card that identifies the most-connected hub from the knowledge-rag graph and surfaces the top hub with 3 contextual signals.  
**Timebox**: <2h (frontend-only, zero backend changes).

### Acceptance Criteria
- Card appears in the dashboard sidebar or top-of-page slot.
- Shows:
  - Hub name (e.g., “MOC”)
  - Short description/context
  - 3 actionable signals (title + 1-line summary)
- Data source: static JSON file produced by knowledge-rag pipeline (`/data/knowledge-rag/top-hub.json`).
- Read-only; no mutations or execute actions.
- Responsive (mobile + desktop).

### Implementation Steps (≤2h)
1. **Create mock/seed data** (`/data/knowledge-rag/top-hub.json`) — 10m.
2. **Add API utility** to fetch hub data (`/lib/knowledge-rag.ts`) — 15m.
3. **Create card component** (`/components/TopHubSignalCard.tsx`) — 45m.
4. **Wire into dashboard layout** (sidebar or top slot) — 20m.
5. **Polish & test** responsiveness + loading/error states — 30m.

### 2) Knowledge-rag utility (frontend-safe)

`/lib/knowledge-rag.ts`
```ts
export type TopHubSignal = {
  hub: {
    id: string;
    name: string;
    description: string;
  };
  signals: Array<{
    id: string;
    title: string;
    summary: string;
  }>;
  generatedAt: string;
};

const ENDPOINT = '/data/knowledge-rag/top-hub.json';

export async function fetchTopHubSignal(): Promise<TopHubSignal | null> {
  try {
    const res = await fetch(ENDPOINT, { cache: 'no-store' });
    if (!res.ok) throw new Error(`Failed to fetch top-hub: ${res.status}`);
    return res.json();
  } catch (err) {
    console.warn('[knowledge-rag] fetchTopHubSignal failed', err);
    return null;
  }
}
```

### 3) Card component

`/components/TopHubSignalCard.tsx`
```tsx
'use client';

import { useEffect, useState } from 'react';
import { fetchTopHubSignal, type TopHubSignal } from '@/lib/knowledge-rag';

export default function TopHubSignalCard() {
  const [data, setData] = useState<TopHubSignal | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetchTopHubSignal().then((res) => {
      setData(res);
      setLoading(false);
    });
  }, []);

  if (loading) {
    return (
      <div className="rounded-lg border bg-card p-4">
        <div className="h-5 w-32 animate-pulse rounded bg-muted" />
        <div className="mt-3 space-y-2">
          {[...Array(3)].map((_, i) => (
            <div key={i} className="h-4 w-full animate-pulse rounded bg-muted" />
          ))}
        </div>
      </div>
    );
  }

  if (!data) {
    return (
      <div className="rounded-lg border bg-card p-4 text-sm text-muted-foreground">
        Top hub signals unavailable.
      </div>
    );
  }

  return (
    <div className="rounded-lg border bg-card p-4 shadow-sm">
      <div className="flex items-start justify-between gap-2">
        <div>
          <h3 className="text-base font-semibold">Top Hub: {data.hub.name}</h3>
          <p className="text-xs text-muted-foreground">{data.hub.description}</p>
        </div>
        <span className="text-xs text-muted-foreground">
          {new Date(data.generatedAt).toLocaleDateString()}
        </span>
      </div>

      <ul className="mt-3 space-y-2">
        {data.signals.map((s) => (
          <li key={s.id} className="rounded-md border bg-accent/50 p-3">
            <p className="text-sm font-medium">{s.title}</p>
            <p className="text-xs text-muted-foreground">{s.summary}</p>
          </li>
        ))}
      </ul>

      <p className="mt-3 text-xs text-muted-foreground">
        Sense + Signal — ไม่ Execute
      </p>
    </div>
  );
}
```
