# Costinel / design

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

### Scope (single highest-value deliverable)
Add a **non-blocking Top-Hub Signal Panel** to the Costinel dashboard that surfaces the most-connected hub (default: `MOC`) and 3 contextual insights from knowledge-rag without touching backend APIs or triggering HF API calls during render. Uses CDN-first pattern and pre-listed file manifest to avoid rate limits.

### Why this is highest-value
- Immediate UX lift: surfaces graph context next to cost signals.
- Zero runtime HF API calls (CDN-only) → no 429 risk.
- Reuses existing knowledge-rag + top-hub patterns.
- Ships in <2h with minimal footprint.

---

### Implementation Steps

1. **Create local graph metadata** (if missing)  
   Add `/opt/axentx/Costinel/data/graph-meta.json` with hub connectivity and file manifest.

2. **Add pre-listed file manifest**  
   Single JSON listing date folder contents (one API call, cached). Used by training/ingest scripts later.

3. **Create lightweight signal fetcher**  
   CDN-only fetch: `https://huggingface.co/datasets/.../resolve/main/...` with no Authorization header.

4. **Add Top-Hub Signal Panel component**  
   React component that:
   - Reads `graph-meta.json` to pick top hub (e.g., `MOC`)
   - Fetches 3 latest insights from CDN (non-blocking, `stale-while-revalidate`)
   - Renders compact card with fallback when CDN unavailable

5. **Wire into dashboard layout**  
   Place panel in sidebar or top-row signal strip; ensure it never blocks cost dashboard render.

6. **Update docs & tags**  
   Add inline comments referencing patterns: `#knowledge-rag #graph #hub #cdn #rate-limit-bypass`.

#### 2. Pre-list helper (run once from Mac orchestration)
```bash
#!/usr/bin/env bash
# scripts/list-top-hub-manifest.sh
# Run after rate-limit window clears; commit result to repo.
set -euo pipefail
REPO="axentx/knowledge-rag"
DATE=$(date +%Y-%m-%d)
FOLDER="top-hub/MOC/${DATE}"

# Single non-recursive API call (paginated 100 safe)
gh api "repos/${REPO}/contents/${FOLDER}?ref=main" \
  --paginate \
  --jq '[.[] | select(.type=="file") | .name] | sort | reverse' \
  > "data/manifest-${DATE}.json"

echo "Manifest saved: data/manifest-${DATE}.json"
```

#### 3. CDN-only signal fetcher (browser)
```ts
// src/lib/cdnSignalFetcher.ts
export async function fetchTopHubInsights(
  hub: string,
  manifest: { date: string; files: string[]; cdn_base: string }
): Promise<any[]> {
  const { date, files, cdn_base } = manifest;
  const folder = `knowledge-rag/top-hub/${hub}/${date}`;
  const limit = 3;

  const urls = files.slice(0, limit).map((f) => `${cdn_base}/${folder}/${f}`);

  // Non-blocking, CDN-only (no Authorization header)
  const results = await Promise.allSettled(
    urls.map((url) =>
      fetch(url, {
        headers: {
          // No Authorization -> bypass HF API rate limits
          Accept: "application/json",
        },
        cache: "force-cache",
      }).then((r) => {
        if (!r.ok) throw new Error(`CDN fetch failed: ${r.status}`);
        return r.json();
      })
    )
  );

  return results
    .filter((r): r is PromiseFulfilledResult<any> => r.status === "fulfilled")
    .map((r) => r.value)
    .filter(Boolean);
}
```

#### 4. Top-Hub Signal Panel component
```tsx
// src/components/TopHubSignalPanel.tsx
import { useEffect, useState } from "react";
import { fetchTopHubInsights } from "../lib/cdnSignalFetcher";

interface GraphMeta {
  top_hub: string;
  insights_manifest: {
    date: string;
    files: string[];
    cdn_base: string;
  };
}

export function TopHubSignalPanel() {
  const [insights, setInsights] = useState<any[]>([]);
  const [hub, setHub] = useState<string>("MOC");
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Load graph metadata (local, tiny)
    fetch("/data/graph-meta.json", { cache: "no-cache" })
      .then((r) => r.json())
      .then((meta: GraphMeta) => {
        setHub(meta.top_hub);
        return fetchTopHubInsights(meta.top_hub, meta.insights_manifest);
      })
      .then((data) => {
        setInsights(data);
        setLoading(false);
      })
      .catch(() => {
        // Graceful fallback: show hub name only
        setLoading(false);
      });
  }, []);

  return (
    <aside className="top-hub-panel rounded border bg-white/80 p-3 shadow-sm">
      <div className="flex items-center gap-2 mb-2">
        <span className="font-semibold text-sm">Top Hub</span>
        <span className="px-2 py-0.5 bg-blue-100 text-blue-800 text-xs rounded">
          {hub}
        </span>
      </div>

      {loading && (
        <div className="space-y-2">
          {[...Array(3)].map((_, i) => (
            <div key={i} className="h-4 bg-gray-100 rounded animate-pulse" />
          ))}
        </div>
      )}

      {!loading && insights.length === 0 && (
        <p className="text-xs text-gray-500">No signals available</p>
      )}

      <ul className="space-y-2">
        {insights.map((insight, idx) => (
          <li key={idx} className="text-xs text-gray-700 border-l-2 pl-2 border-blue-300">
            {insight.summary || insight.title || `Insight ${idx + 1}`}
          </li>
        ))}
      </ul>

      <footer className="mt-2 text-[10px] text-gray-400">
        CDN-first • #knowledge-rag #graph #hub
      </footer>
    </aside>
  );
}
```
