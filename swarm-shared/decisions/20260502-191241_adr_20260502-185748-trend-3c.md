## Deployment
hybrid (local CLI + optional HF Space for hosted UI/demo; Supabase for auth/metadata)

## v0.1 scope
CLI-only v0.1 in 2 weeks: local repo indexing with tree-sitter (Python/JS/TS), rule-based findings (complexity hotspots, basic security patterns), incremental cache, JSON/terminal output, optional FastAPI local server for simple HTML report; no cloud analysis required, optional Supabase auth for future sync.
