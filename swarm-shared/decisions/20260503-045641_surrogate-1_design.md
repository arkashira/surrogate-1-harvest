# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-first, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **`bin/dataset-enrich.sh`** → **`bin/dataset-enrich.py`**
   - Single API call from runner (after rate-limit window) to `list_repo_tree(path, recursive=False)` for one date folder → save `manifest.json`.
   - Worker uses only CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with no Authorization header during data streaming.
   - Per-file streaming with schema projection to `{prompt, response}` only at parse time; drop `source`, `ts`, etc.
   - Deterministic shard assignment via `hash(slug) % 16 == SHARD_ID`.
   - Central dedup via existing `lib/dedup.py` (SQLite md5 store) to prevent cross-run duplicates.

2. **`lib/dedup.py`** (no change) — keep as source-of-truth.

3. **`.github/workflows/ingest.yml`**
   - Update matrix runner to invoke `python bin/dataset-enrich.py --shard ${{ matrix.shard }} --date $(date +%Y-%m-%d)`.
   - Keep 16-shard parallelism; each runner isolated 7 GB.

4. **`requirements.txt`**
   - Add `requests` (for CDN bypass) if not present; keep `datasets`, `huggingface_hub`, `pyarrow`, `numpy`.

5. **Output path**
   - `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl` (unchanged).

"""
Manifest-first, CDN-bypass ingestion worker for surrogate-1-training-pairs.

Usage:

import sys

DATE_DIR = os.getenv("INGEST_DATE")  # fallback to CLI
SHARD_ID = int(os.getenv("SHARD_ID", 0))
N_SHARDS = 16
CDN_BASE = f"https://huggingface.co/{HF_REPO}/resolve/main"

API = HfApi()
DEDUP = DedupStore()

HEADERS = {}  # No Authorization for CDN bypass

def list_date_folder(date_str: str):
    """Single API call to list top-level folder for date (non-recursive)."""
    try:
        tree = API.list_repo_tree(
            repo_id=HF_REPO,
            path=date_str,
            recursive=False,
            repo_type="dataset",
        )
        return [item.rfilename for item in tree if item.rfilename.endswith((".parquet", ".jsonl"))]
    except Exception as exc:
        print(f"list_repo_tree failed: {exc}", file=sys.stderr)
        return []

def build_manifest(date_str: str, out_path: Path):
    """Create manifest.json for date folder (run once per worker startup)."""
    files = list_date_folder(date_str)
    manifest = {
        "date": date_str,
        "created_utc": datetime.now(timezone.utc).isoformat(),
        "files": files,
    }
    out_path.write_text(json.dumps(manifest, indent=2))
    return manifest

def should_process(slug: str) -> bool:
    """Deterministic shard assignment."""
    digest = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    return (digest % N_SHARDS) == SHARD_ID

def safe_project_to_pair(file_path: Path):
    """
    Stream file and project to {prompt, response} only.
    Supports .parquet and .jsonl; avoids mixed-schema CastErrors by not
    relying on load_dataset(streaming=True) across heterogeneous files.
    """
    suffix = file_path.suffix.lower()
    if suffix == ".parquet":
        try:
            pf = pq.ParquetFile(file_path)
            for batch in pf.iter_batches(batch_size=1000, columns=["prompt", "response"]):
                df = batch.to_pandas()
                for _, row in df.iterrows():
                    prompt = str(row.get("prompt") or "")
                    response = str(row.get("response") or "")
                    if prompt and response:
                        yield {"prompt": prompt, "response": response}
        except Exception as exc:
            print(f"Parquet projection failed {file_path}: {exc}", file=sys.stderr)
            return
    elif suffix == ".jsonl":
        try:
            with file_path.open("r", encoding="utf-8") as f:
                for line in f:
                    line = line.strip()
                    if not line:
                        continue
                    obj = json.loads(line)
                    prompt = str(obj.get("prompt") or obj.get("input") or "")
                    response = str(obj.get("response") or obj.get("output") or "")
                    if prompt and response:
                        yield {"prompt": prompt, "response": response}
        except Exception as exc:
            print(f"JSONL projection failed {file_path}: {exc}", file=sys.stderr)
            return
    else:
        print(f"Unsupported file {file_path}", file=sys.stderr)

def download_via_cdn(repo: str, path: str, dest: Path):
    """Download via CDN (no auth)."""
    url = f"{CDN_BASE}/{path}"
    resp = requests.get(url, headers=HEADERS, stream=True, timeout=60)
    resp.raise_for_status()
    dest.parent.mkdir(parents=True, exist_ok=True)
    with dest.open("wb") as f:
        for chunk in resp.iter_content(chunk_size=8192):
            f.write(chunk)
    return dest

def slug_for_path(path: str) -> str:
    return path.rsplit("/", 1)[-1].split(".", 1)[0]
