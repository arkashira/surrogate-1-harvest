## Objective (Highest-Value Incremental Improvement)
Make Surrogate training **HF-rate-limit-proof** and **Lightning-idle-resilient** with <2h effort, zero infra changes, and no API/auth calls during training.

---

## Core Design Decisions (Resolved Contradictions)
- **Use `hf_hub_download` (not raw `requests`/`curl`)** for CDN fetches:  
  - Correctness: handles repo type, caching, retries, and partial files safely.  
  - Actionability: one-line drop-in vs. manual streaming/save logic.
- **Embed manifest in `train.py` (or pass via `--manifest`)**:  
  - Avoids runtime file-list generation and extra API/tree calls during training.
- **Idle-resilient launcher checks **before** `.run()` and restarts Studio + re-invokes training**:  
  - Correctness: prevents idle-timeout kills; ensures continuity.  
  - Actionability: explicit state check + start/run sequence.
- **Do not use `load_dataset` on raw downloaded files inside loop**:  
  - Correctness: prevents repeated dataset object creation and format guessing.  
  - Actionability: stream from downloaded files or use `datasets` streaming with a generator.

## Code Snippets (Single Final Version)

### 1) `generate_manifest.py` (run once on Mac)
```python
import json
import os
from huggingface_hub import list_repo_tree

REPO = "huggingface/datasets"
DATE_FOLDER = "2023-03-01"  # adjust as needed
OUT = f"manifest-{DATE_FOLDER}.json"

def main():
    # Single API call (only during manifest generation)
    tree = list_repo_tree(repo_id=REPO, path=DATE_FOLDER, repo_type="dataset", recursive=True)
    # Keep only file paths (exclude directories)
    files = [item.path for item in tree if item.type == "file"]
    manifest = {
        "repo": REPO,
        "date_folder": DATE_FOLDER,
        "files": files
    }
    with open(OUT, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest written to {OUT} with {len(files)} files.")

if __name__ == "__main__":
    main()
```

Run:
```bash
python generate_manifest.py
```
