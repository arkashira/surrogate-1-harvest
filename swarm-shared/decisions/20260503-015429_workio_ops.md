# workio / ops

## Final Synthesis — Correct, Actionable Fix

### Root causes (merged, prioritized)
- **No idempotency/replay protection** on LINE webhooks: retries or replays with the same `X-Line-Signature` and body can create duplicate punches.
- **Non-atomic upsert** (`findOne → if exists update else insert`) enables race-condition duplicates under concurrency or retries.
- **No database-level uniqueness guarantees** to enforce exactly-once punches per user/date/punch_type.
- **Signature not persisted**, so replay detection is impossible.
- **Raw body not preserved**, which breaks reliable signature verification and idempotency hashing.

---

### Concrete solution (single plan)

1. **Add DB objects for idempotency and uniqueness**
   - `line_webhook_deliveries` table + unique `(signature, body_hash)` to reject replays atomically.
   - `punches.idempotency_key` column and a unique constraint/index on `(user_id, tenant_id, date, punch_type)` **or** on `idempotency_key` (choose one; prefer constraint on business key + idempotency key for safety).
   - Use Postgres `ON CONFLICT` for atomic upserts at the DB level.

2. **Make webhook handler atomic and idempotent**
   - Run everything in a single transaction.
   - First, attempt to insert `line_webhook_deliveries`. If duplicate signature+hash, return `200 OK` (idempotent success) and stop.
   - For each event, compute an idempotency key (e.g., `userId-date-punchType-deliveryIdOrSignatureHash`) and perform `INSERT ... ON CONFLICT DO UPDATE` on punches using the unique constraint.
   - Persist `line_signature` and idempotency key on punches for audit and debugging.

3. **Preserve raw body for signature verification**
   - Add middleware that buffers the raw body and attaches it to `req.rawBody` before JSON parsing.

4. **Verification checklist**
   - Apply schema; confirm indexes/constraints exist.
   - Send identical webhook twice; confirm one delivery row and one punch row; second request returns 200 idempotent success.
   - Fire concurrent requests for same user/date/type; confirm only one punch row.
   - Attempt replay with old signature+body; confirm rejection and no punch creation.

### Implementation (minimal, correct, actionable)

#### DB schema (`workio/server/src/db/schema.sql`)
```sql
-- Idempotency/replay detection for LINE webhooks
CREATE TABLE IF NOT EXISTS line_webhook_deliveries (
  id           UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  signature    TEXT NOT NULL,
  body_hash    TEXT NOT NULL,
  delivered_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  created_at   TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE UNIQUE INDEX IF NOT EXISTS idx_line_webhook_sig_hash
  ON line_webhook_deliveries (signature, body_hash);

-- Punches uniqueness and idempotency
ALTER TABLE punches
  ADD COLUMN IF NOT EXISTS idempotency_key TEXT;

-- Business-unique punch per user per day per type (preferred primary guard)
CREATE UNIQUE INDEX IF NOT EXISTS idx_punches_user_date_type
  ON punches (user_id, tenant_id, date, punch_type)
  WHERE idempotency_key IS NOT NULL;

-- Fast lookup by idempotency key
CREATE UNIQUE INDEX IF NOT EXISTS idx_punches_idempotency
  ON punches (idempotency_key)
  WHERE idempotency_key IS NOT NULL;
```

#### Middleware: preserve raw body (`workio/server/src/middleware/rawBody.ts`)
```ts
import type { Request, Response, NextFunction } from 'express';

export function rawBodyMiddleware(req: Request, res: Response, next: NextFunction) {
  const chunks: Buffer[] = [];
  req.on('data', (chunk) => chunks.push(chunk));
  req.on('end', () => {
    const raw = Buffer.concat(chunks).toString('utf8');
    (req as any).rawBody = raw;
    try {
      req.body = JSON.parse(raw);
    } catch {
      req.body = {};
    }
    next();
  });
}
```
Register before routes.

#### Webhook handler (`workio/server/src/routes/line.ts`)
```ts
import crypto from 'crypto';
import { db } from '../db';
import { eq } from 'drizzle-orm';
import { lineWebhookDeliveries, punches, users } from '../db/schema';

function hashBody(body: string): string {
  return crypto.createHash('sha256').update(body, 'utf8').digest('hex');
}

export async function handleLineWebhook(req: any, res: any) {
  const signature = req.headers['x-line-signature'] as string;
  const rawBody = req.rawBody as string;
  const bodyHash = hashBody(rawBody);
  const events = req.body?.events;

  if (!Array.isArray(events) || events.length === 0) {
    return res.status(400).json({ error: 'invalid payload' });
  }

  try {
    await db.transaction(async (tx) => {
      // Idempotency/replay check (atomic)
      const [dup] = await tx
        .select()
        .from(lineWebhookDeliveries)
        .where(eq(lineWebhookDeliveries.signature, signature))
        .limit(1);

      if (dup) {
        return res.status(200).json({ ok: true, reason: 'duplicate delivery' });
      }

      await tx.insert(lineWebhookDeliveries).values({
        signature,
        body_hash: bodyHash,
        delivered_at: new Date(),
      });

      // Process events
      for (const ev of events) {
        if (ev?.type !== 'message' || ev.message?.type !== 'text') continue;
        const userId = ev.source?.userId;
        if (!userId) continue;

        const text = (ev.message.text || '').trim().toLowerCase();
        const punchType = text.includes('in') || text.includes('เข้า') ? 'in' : 'out';
        const now = new Date();
        const today = now.toISOString().split('T')[0];

        const [userRow] = await tx
          .select({ tenantId: users.tenant_id })
          .from(users)
          .where(eq(users.line_user_id, userId))
          .limit(1);

        const tenantId = userRow?.tenantId || null;
        const idempotencyKey = `${userId}-${today}-${punchType}-${bodyHash}`;

        // Atomic upsert using business unique constraint
        await tx
          .insert(punches)
          .values({
            user_id: userId,
            tenant_id: tenantId,
            date: today,
            punch_type: punchType,
            punched_at: now,
            idempotency_key: idempotencyKey,
            line_signature: signature,
            created_at: now,
            updated_at: now,
          })
          .onConflictDoUpdate({
            target: [punches.user_id, punches.date, punches.punch_type],
            set: {
              punched_at: now,
              updated_at: now,
              line_signature: signature,
              idempotency_key: idempotencyKey,
            },
          });
      }
    });

    return res.status(200).json({ ok: true });
  } catch (err: any) {
    console.error('LINE webhook error:', err);
    return res.status(500).json({ error: 'internal server error' });
  }
}
```
