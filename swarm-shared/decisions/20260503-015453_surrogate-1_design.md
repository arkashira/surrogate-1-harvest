# surrogate-1 / design

## Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Uses a **single `list_repo_tree` snapshot** (JSON manifest) generated once per date on the Mac orchestrator and committed to the repo (or passed via env) so each GitHub runner does **zero HF API calls** during data load.
- Downloads raw files directly from **HF CDN** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header → bypasses 429 rate limits entirely.
- Projects heterogeneous schemas to `{prompt,response}` only at parse time (avoids pyarrow CastError).
- Writes normalized pairs to `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl` with deterministic shard assignment via `hash(slug) % 16`.
- Keeps existing `lib/dedup.py` for central md5 dedup (unchanged).
- Adds robust retry/backoff for CDN fetches and handles missing/empty files gracefully.
- Adds a small orchestrator script (`bin/gen-manifest.py`) to produce the tree snapshot (run once per date on Mac).

    if not manifest_path.is_file():

        data = json.load(f)

def download_cdn(path: str, timeout: int = 30) -> Optional[bytes]:
    url = f"{CDN_BASE}/{path}"
    for attempt in range(1, MAX_RETRIES + 1):
        try:
            resp = requests.get(url, timeout=timeout, headers={"Authorization": ""})
            if resp.status_code == 200:
                return resp.content
            if resp.status_code == 404:
                log.warning("CDN 404: %s", path)
                return None
            log.warning("CDN %s: %s (attempt %s)", resp.status_code, path, attempt)
        except requests.RequestException as exc:
            log.warning("CDN request error %s: %s", path, exc)

        if attempt == MAX_RETRIES:
            break
        backoff = INITIAL_BACKOFF * (BACKOFF_FACTOR ** (attempt - 1))
        time.sleep(backoff)
    log.error("Failed to download after retries: %s", path)
    return None
