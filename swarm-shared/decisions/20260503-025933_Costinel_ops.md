# Costinel / ops

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

### What we ship
- A **non-blocking Top-Hub Signal Panel** mounted on the Costinel dashboard (sidebar or top banner area).
- Defaults to hub **MOC** (configurable via `VITE_HUB_NAME=MOC`).
- Shows: hub title, short description, top 3 signals (anomalies/recommendations), and a “View details” link.
- **CDN-first data**: file list fetched once from Mac (HF API) → saved to `public/hubs/moc-index.json`; panel loads data via CDN (`/hubs/moc-index.json`) with zero runtime API calls.
- Graceful fallback: if CDN fetch fails or file missing, panel collapses to minimal state (no errors shown to user).

### Why this is highest-value (<2h)
- Applies the **top-hub doc insight** pattern (MOC) and **HF CDN bypass** pattern.
- Improves ops visibility immediately without touching billing, infra, or auth flows.
- Fits existing frontend stack (assumed Vite + React/TS) and reuses public/ static asset pipeline.

---

### Files to create/modify

1. `public/hubs/moc-index.json` (generated on Mac, committed) — CDN payload.
2. `src/components/TopHubSignalPanel.tsx` — React component.
3. `src/components/TopHubSignalPanel.css` (or inline styles) — minimal styling.
4. `src/config/hubs.ts` — hub config (name, displayName, description, signals source path).
5. Mount in dashboard: likely `src/pages/Dashboard.tsx` or main layout sidebar/banner.

#### 3) Component (src/components/TopHubSignalPanel.tsx)
```tsx
import { useEffect, useState } from 'react';
import { HUB_CONFIG } from '../config/hubs';
import './TopHubSignalPanel.css';

type Signal = {
  id: string;
  title: string;
  summary: string;
  severity: 'info' | 'warning' | 'critical';
  actionUrl?: string;
};

type HubIndex = {
  hub: string;
  generatedAt: string;
  signals: Signal[];
};

const HUB_NAME = (import.meta.env.VITE_HUB_NAME || 'MOC') as keyof typeof HUB_CONFIG;
const config = HUB_CONFIG[HUB_NAME];

export default function TopHubSignalPanel() {
  const [signals, setSignals] = useState<Signal[]>(config.fallbackSignals);
  const [ready, setReady] = useState(false);

  useEffect(() => {
    let cancelled = false;
    // CDN-first fetch (no auth, bypasses HF API rate limits)
    fetch(config.cdnIndexPath, { cache: 'no-store' })
      .then((res) => {
        if (!res.ok) throw new Error('CDN fetch failed');
        return res.json() as Promise<HubIndex>;
      })
      .then((data) => {
        if (!cancelled && Array.isArray(data?.signals) && data.signals.length > 0) {
          setSignals(data.signals.slice(0, 3));
        }
      })
      .catch(() => {
        // silent fallback to default signals
      })
      .finally(() => {
        if (!cancelled) setReady(true);
      });

    return () => {
      cancelled = true;
    };
  }, []);

  if (!ready && signals === config.fallbackSignals) {
    // render minimal collapsed state to avoid layout shift
    return (
      <div className="hub-panel hub-panel--collapsed" aria-hidden={!ready}>
        <div className="hub-panel__title">{config.displayName}</div>
      </div>
    );
  }

  return (
    <aside className="hub-panel" aria-label={`${config.displayName} signals`}>
      <div className="hub-panel__header">
        <div>
          <div className="hub-panel__title">{config.displayName}</div>
          <div className="hub-panel__desc">{config.description}</div>
        </div>
      </div>

      <ul className="hub-panel__list" role="list">
        {signals.map((s) => (
          <li key={s.id} className={`hub-panel__item hub-panel__item--${s.severity}`}>
            <div className="hub-panel__item-title">{s.title}</div>
            <div className="hub-panel__item-summary">{s.summary}</div>
            {s.actionUrl && (
              <a href={s.actionUrl} className="hub-panel__item-link">
                View details →
              </a>
            )}
          </li>
        ))}
      </ul>

      <div className="hub-panel__footer">
        <a href="/hubs" className="hub-panel__footer-link">
          View all signals →
        </a>
      </div>
    </aside>
  );
}
```
