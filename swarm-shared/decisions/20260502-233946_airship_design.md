# airship / design

## airship
### Harden `airship discover` into a deterministic, CDN-only orchestrator

#### Goal
Eliminate HF API rate limits and PyArrow schema errors during Surrogate training

#### Implementation Plan

#### Step 1: Update `airship discover` to use CDN-only file paths

```bash
# List all file paths in a single API call
curl -s -X GET \
  https://huggingface.co/datasets/{repo}/resolve/main/{path} \
  -H 'Authorization: Bearer {token}' \
  -o file_paths.json

# Embed file paths in training script
echo "import json
file_paths = json.load(open('file_paths.json'))" >> train.py
```

#### Step 2: Use `huggingface_hub` to download files from CDN

```python
# Download files from CDN
import huggingface_hub
repo_id = "{repo}"
path = "{path}"
huggingface_hub.download_repo(repo_id, path, use_auth_token=True)
```

#### Step 3: Project to {prompt, response} only at parse time

```python
# Project to {prompt, response} only at parse time
import pyarrow.parquet as pq
```
