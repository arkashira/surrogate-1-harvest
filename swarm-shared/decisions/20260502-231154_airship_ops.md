# airship / ops

## Highest-Value Incremental Improvement (≤2h)

**Goal**: Harden `airship discover` into a deterministic, CDN-only, zero-HF-API-runtime orchestrator that eliminates HF API rate limits and PyArrow schema errors while producing reproducible file lists for downstream training.

### Why this ships in <2h
- Scope: single orchestrator script + small JSON contract
- No infra changes, no model training, no DB migrations
- Reuses existing HF CDN bypass pattern and schema-projection lessons
- Immediately unblocks surrogate-1 ingestion and removes 429/128-hr commit cap risk

---

## Implementation Plan

1. **Locate entrypoint**  
   Identify `airship/discover` CLI (likely `airsip/discover.py` or `airsip/cmd/discover.py`). If absent, create `airsip/discover.py`.

2. **Define contract**  
   Output: `file_list.json` placed in `batches/mirror-merged/{date}/` containing:
   ```json
   {
     "repo": "datasets/{owner}/{repo}",
     "date": "2026-05-02",
     "files": [
       {
         "path": "train/part-00001.parquet",
         "cdn_url": "https://huggingface.co/datasets/{repo}/resolve/main/train/part-00001.parquet",
         "size": 12345678,
         "sha256": "hexdigest"
       }
     ],
     "generated_at": "2026-05-02T23:00:00Z"
   }
   ```

3. **Implement CDN-safe listing**  
   - Use `list_repo_tree(path, recursive=False)` **per folder** (never recursive on big repos).
   - Paginate safely; stop after 1000 items per folder (avoid 429).
   - Build CDN URLs directly (`resolve/main/...`) — no Authorization header required during training.

4. **Schema-safe projection hint**  
   - Do **not** load datasets in this script.
   - Add `project_hint: {prompt: "prompt", response: "response"}` to JSON so downstream `surrogate/train.py` knows which cols to project at parse time (avoids PyArrow `CastError` from mixed schemas).

5. **Deterministic hashing for repo selection**  
   - Compute `slug_hash = int(hashlib.sha256(slug.encode()).hexdigest(), 16)`
   - `sibling_index = slug_hash % 5` → pick sibling repo for writes (bypasses 128/hr cap).

6. **Idempotent & cache-friendly**  
   - Skip folders already listed in prior run (compare `sha256` of existing `file_list.json`).
   - If unchanged, exit 0 without new API calls.

7. **Embed in training script**  
   - Add `file_list.json` as argument to `surrogate/train.py`.
   - Use `hf_hub_download` or direct CDN `wget/curl` in `Dataset` loader with zero HF API calls during training.

8. **Validation & exit codes**  
   - Validate JSON schema with `jsonschema` or inline checks.
   - Exit 0 on success, 1 on API error, 2 on validation failure.

### `airsip/discover.py`
```python
#!/usr/bin/env python3
"""
Deterministic, CDN-safe repo lister for airship.
Produces file_list.json for surrogate-1 training.
"""
import os
import sys
import json
import hashlib
import datetime
from pathlib import Path
from typing import List, Dict

import requests
from huggingface_hub import HfApi, hf_hub_download

API = HfApi()
CDN_ROOT = "https://huggingface.co/datasets"

def list_folder_cdn_safe(repo: str, folder: str = "") -> List[Dict]:
    """
    List single folder (non-recursive) to avoid 429 on big repos.
    Returns items with CDN URLs and size.
    """
    items = []
    try:
        tree = API.list_repo_tree(repo=repo, path=folder or ".", recursive=False)
    except Exception as e:
        print(f"HF API error on {repo}/{folder}: {e}", file=sys.stderr)
        raise

    for entry in tree:
        if entry.type == "directory":
            # Do not recurse here; caller may iterate explicitly
            continue
        cdn_url = f"{CDN_ROOT}/{repo}/resolve/main/{entry.path}"
        items.append({
            "path": entry.path,
            "cdn_url": cdn_url,
            "size": getattr(entry, "size", None),
            "sha256": None  # filled later if needed
        })
    return items

def slug_to_sibling(repo_slug: str, n_siblings: int = 5) -> int:
    h = int(hashlib.sha256(repo_slug.encode()).hexdigest(), 16)
    return h % n_siblings

def build_file_list(repo: str, folders: List[str], output_path: Path) -> Dict:
    all_files = []
    for folder in folders:
        all_files.extend(list_folder_cdn_safe(repo, folder))

    # Optional: compute sha256 for critical parquet files (lightweight)
    # Skipped here to avoid runtime cost; training can verify on download.

    payload = {
        "repo": repo,
        "date": datetime.date.today().isoformat(),
        "folders_scanned": folders,
        "files": all_files,
        "project_hint": {"prompt": "prompt", "response": "response"},
        "sibling_index": slug_to_sibling(repo),
        "generated_at": datetime.datetime.utcnow().isoformat() + "Z"
    }

    output_path.parent.mkdir(parents=True, exist_ok=True)
    with open(output_path, "w", encoding="utf-8") as f:
        json.dump(payload, f, indent=2, sort_keys=True)
    return payload

def main():
    repo = os.getenv("AIRSHIP_DISCOVER_REPO", "microsoft/DialoGPT")
    folders_env = os.getenv("AIRSHIP_DISCOVER_FOLDERS", "train,validation")
    folders = [f.strip() for f in folders_env.split(",") if f.strip()]

    date_str = datetime.date.today().isoformat()
    out_dir = Path("batches/mirror-merged") / date_str
    out_file = out_dir / "file_list.json"

    # Idempotency: skip if already exists and not forced
    if out_file.exists() and not os.getenv("AIRSHIP_DISCOVER_FORCE"):
        print(f"file_list.json exists at {out_file}. Skipping (use AIRSHIP_DISCOVER_FORCE=1 to rerun).")
        sys.exit(0)

    try:
        payload = build_file_list(repo, folders, out_file)
        print(json.dumps({"status": "ok", "file_list": str(out_file), "count": len(payload["files"])}))
        sys.exit(0)
    except Exception as e:
        print(json.dumps({"status": "error", "error": str(e)}))
        sys.exit(1)

if __name__ == "__main__":
    main()
```

### `surrogate/train.py` (minimal embed)
```python
import json
from pathlib import Path
from datasets import load_dataset, Dataset

def load_from_file_list(file_list_path: Path):
    with open(file_list_path) as f:
        meta = json.load(f)

    # CDN-only: use direct URLs to bypass HF API during training
    urls = [f["cdn_url"] for f in meta["files"] if f["path"].endswith(".parquet")]
    if not urls:
        raise ValueError("No parquet files in file_list")

    # Streaming + projection prevents CastError from mixed schemas
    ds = load_dataset("parquet", data_files=urls, split="train", streaming=True)

    # Project only required columns; ignore extras safely
    def project(batch):
        return {
            "prompt": batch[meta["project_hint"]["prompt"]],
            "response": batch[meta["project_hint"]["response"]]
        }

    return ds.map(project, batched=True, remove_columns=set(batch.keys()) - {"prompt", "response"})
```
