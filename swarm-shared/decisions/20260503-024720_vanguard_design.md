# vanguard / design

## Final synthesized implementation (best of both candidates)

**Core diagnosis (merged)**  
- Missing frontend entrypoint and mount point prevents any local UI iteration.  
- No build/dev tooling means no HMR, no deterministic asset references, and fragile runtime path resolution.  
- Runtime repo enumeration burns HF API quota and violates the CDN-only strategy.  
- No design tokens or layout scaffold slows UI polish and reuse.  
- Missing asset pipeline and manifest forces runtime fetches and fragile path resolution.

**Chosen approach**  
Add a minimal, framework-free Vite scaffold that:
- Provides a deterministic build-time static manifest (baked into the client) so the UI never enumerates repos via HF at runtime.  
- Uses CDN-first asset paths and hashed filenames for cache-busting/CDN friendliness.  
- Includes design tokens, a responsive layout, and a lightweight UI shell (header, repo list, detail pane) that can render from the manifest and fetch repo details from CDN/static files only.  
- Keeps the backend unchanged and is scoped to `/opt/axentx/vanguard/`.

#### vite.config.js
```js
import { defineConfig } from 'vite';
import { readdirSync, statSync } from 'fs';
import { join } from 'path';

function collectStaticManifest(root) {
  const out = {};
  function scan(dir, base = '') {
    for (const name of readdirSync(dir)) {
      if (name === 'node_modules' || name === 'dist') continue;
      const abs = join(dir, name);
      const key = join(base, name).replace(/\\/g, '/');
      if (statSync(abs).isDirectory()) scan(abs, key);
      else out[key] = `/${key}`;
    }
  }
  try {
    scan(root);
  } catch {
    // If public/ doesn't exist yet, return empty manifest.
  }
  return out;
}

export default defineConfig({
  root: '.',
  build: {
    outDir: 'dist',
    emptyOutDir: true,
    manifest: true,
    rollupOptions: {
      input: '/opt/axentx/vanguard/index.html',
      output: {
        entryFileNames: 'assets/[name]-[hash].js',
        chunkFileNames: 'assets/[name]-[hash].js',
        assetFileNames: 'assets/[name]-[hash][extname]'
      }
    }
  },
  define: {
    __STATIC_MANIFEST__: JSON.stringify(collectStaticManifest(join(process.cwd(), 'public')))
  },
  server: {
    port: 5173,
    strictPort: true
  }
});
```
