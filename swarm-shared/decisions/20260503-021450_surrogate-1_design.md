# surrogate-1 / design

## Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID` and `SHARD_TOTAL` (16) from GitHub Actions matrix.
- Uses a pre-generated `file-list.json` (Mac-side, one `list_repo_tree` call per date folder) to avoid recursive listing and HF API rate limits.
- Downloads only assigned shard files via **HF CDN direct URLs** (`resolve/main/...`) — zero API/auth calls during data load, bypassing 429 limits.
- Projects heterogeneous schemas to `{prompt, response}` at parse time (no `load_dataset(streaming=True)` on mixed schemas).
- Deduplicates via central `lib/dedup.py` md5 store and writes `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.
- Adds retry/backoff for CDN flakiness and respects HF commit cap by using deterministic shard+timestamp filenames.

### Steps (1h45m)

1. (10m) Create `bin/dataset-enrich.py` with shebang, argparse for `SHARD_ID`, `SHARD_TOTAL`, `--date`, `--file-list`, `--out-dir`.
2. (20m) Implement CDN downloader: given repo+path, fetch via `requests.get(f"https://huggingface.co/datasets/{repo}/resolve/main/{path}", timeout=60, stream=True)` with retry/backoff.
3. (30m) Implement per-file schema projection: try parquet → arrow table → select/rename to `prompt,response`; fallback JSON/JSONL lines; normalize to UTF-8.
4. (20m) Wire dedup: import `lib/dedup.py` (or inline md5 store) to skip seen content.
5. (15m) Write output as newline-delimited JSON with shard+timestamp filename.
6. (10m) Update `.github/workflows/ingest.yml` to generate `file-list.json` on Mac (or reuse cached per-date list) and pass matrix args.
7. (10m) Add `requirements.txt` extras if needed (`requests`, `pyarrow`, `tqdm`).

#!/usr/bin/env python3
"""
CDN-bypass ingestion worker for surrogate-1 public dataset.
Usage (GH Actions matrix):
  SHARD_ID=0 SHARD_TOTAL=16 python bin/dataset-enrich.py \
    --repo axentx/surrogate-1-training-pairs \
    --date 2026-05-03 \
    --file-list file-list.json \
    --out-dir batches/public-merged
"""
import argparse
import hashlib
import json
import os
import sys
import time
from pathlib import Path
from typing import Iterator, Tuple

import pyarrow as pa
import pyarrow.parquet as pq
import requests
from tqdm import tqdm

# Optional: import local dedup if available
try:
    from lib.dedup import DedupStore
except Exception:
    DedupStore = None

DATASET_CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"
RETRY_BACKOFF = [1, 2, 4, 8, 16]
CHUNK_SIZE = 8192

def retry_get(url: str, **kwargs) -> requests.Response:
    for attempt, wait in enumerate(RETRY_BACKOFF):
        try:
            resp = requests.get(url, timeout=60, **kwargs)
            resp.raise_for_status()
            return resp
        except Exception as exc:
            if attempt == len(RETRY_BACKOFF) - 1:
                raise
            time.sleep(wait)
    raise RuntimeError("unreachable")

def stream_cdn_file(repo: str, path: str) -> Iterator[bytes]:
    url = DATASET_CDN_TEMPLATE.format(repo=repo, path=path)
    resp = retry_get(url, stream=True)
    for chunk in resp.iter_content(chunk_size=CHUNK_SIZE):
        yield chunk

def file_md5(path: str) -> str:
    h = hashlib.md5()
    with open(path, "rb") as f:
        for chunk in iter(lambda: f.read(CHUNK_SIZE), b""):
            h.update(chunk)
    return h.hexdigest()

def project_to_pair(local_path: Path) -> Iterator[Tuple[str, str]]:
    """Yield (prompt, response) from heterogeneous file."""
    suffix = local_path.suffix.lower()
    try:
        if suffix == ".parquet":
            table = pq.read_table(local_path, columns=["prompt", "response"])
            for row in table.to_pylist():
                p = row.get("prompt") or row.get("prompt_text") or row.get("input") or ""
                r = row.get("response") or row.get("response_text") or row.get("output") or ""
                if isinstance(p, str) and isinstance(r, str):
                    yield p.strip(), r.strip()
        elif suffix in (".json", ".jsonl"):
            with open(local_path, "r", encoding="utf-8", errors="replace") as f:
                for line in f:
                    line = line.strip()
                    if not line:
                        continue
                    obj = json.loads(line)
                    p = obj.get("prompt") or obj.get("prompt_text") or obj.get("input") or ""
                    r = obj.get("response") or obj.get("response_text") or obj.get("output") or ""
                    if isinstance(p, str) and isinstance(r, str):
                        yield p.strip(), r.strip()
        else:
            # fallback: try reading as text lines prompt<->response separated by newline or tab
            with open(local_path, "r", encoding="utf-8", errors="replace") as f:
                for line in f:
                    line = line.strip()
                    if not line:
                        continue
                    parts = line.split("\t", 1) if "\t" in line else line.split("\n", 1)
                    if len(parts) == 2:
                        p, r = parts[0].strip(), parts[1].strip()
                        if p and r:
                            yield p, r
    except Exception as exc:
        print(f"Warning: failed to project {local_path}: {exc}", file=sys.stderr)

def main() -> None:
    parser = argparse.ArgumentParser(description="CDN-bypass shard worker")
    parser.add_argument("--repo", default="axentx/surrogate-1-training-pairs")
    parser.add_argument("--date", required=True, help="Date folder in dataset (e.g. 2026-05-03)")
    parser.add_argument("--file-list", required=True, help="JSON list of relative paths under date/")
    parser.add_argument("--out-dir", default="batches/public-merged")
    parser.add_argument("--shard-id", type=int, default=int(os.environ.get("SHARD_ID", 0)))
    parser.add_argument("--shard-total", type=int, default=int(os.environ.get("SHARD_TOTAL", 16)))
    parser.add_argument("--tmp-dir", default="/tmp/surrogate_ingest")
    args = parser.parse_args()

    # Load file list
    with open(args.file_list) as f:
        all_files = json.load(f)
    # Filter to date folder entries (paths relative to dataset root)
    date_prefix = f"{args.date}/"
    candidates = [p for p in all_files if p.startswith(date_prefix)]
    if not candidates:
        print(f"No files for date {args.date}", file=sys.stderr)
        sys.exit(0)

    # Deterministic shard assignment by hash(path)
    def shard_for(path: str) -> int:
        return hash(path) % args.shard_total

    my_files = [p for p in candidates if shard_for(p) == args.shard_id]
    print(f"Shard {args.shard_id}/{args.shard_total}: processing {len(my_files)} files", file=sys.stderr)

    # Prepare output
    ts = time.strftime("%Y%m%d-%H%M%S")
    out_dir = Path(args.out_dir) / args.date
    out_dir.mkdir(parents=True, exist_ok=True)
    out_path = out_dir / f"shard{args.shard_id}-{ts}.jsonl"

    # Dedup store
    dedup = None
    if DedupStore is not None:
        dedup = DedupStore()

    tmp_root = Path(args.tmp_dir)
    tmp_root.mkdir(parents=True, exist_ok=True)
