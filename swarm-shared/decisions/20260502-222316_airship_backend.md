# airship / backend

## Final Implementation — `airship discover` (merged, corrected, actionable)

**What we ship (single command, <2h):**
- `/opt/axentx/airship/bin/airship discover` — zero-config discovery CLI
- Produces:
  - `discover-output/top-hub-insight.md` — human-readable top-hub summary + recommended next steps
  - `discover-output/file-list.json` — CDN-only file manifest (safe for training pipelines, zero API calls during fetches)
  - `discover-output/insight-bundle.json` — compact JSON bundle (top hub + contextual docs + file list + run metadata)
- Safe for cron: proper shebang, `set -euo pipefail`, idempotent, non-zero exit on fatal errors, structured logs to stderr
- Optional webhook post (Slack/Teams) via `DISCOVER_WEBHOOK_URL`

---

### Why this wins
- Applies known patterns: `granite-business-research.sh` + `knowledge-rag` + top-hub review + HF CDN bypass
- Avoids HF API rate-limits: uses `huggingface_hub` Python library for reliable tree listing and builds CDN-only URLs; falls back to HF API + curl only if library unavailable
- Corrects prior contradictions:
  - Uses `recursive=False` tree listing (non-recursive per folder) as intended; recursive listing is optional via flag
  - Provides concrete, copy-paste scripts with proper error handling and logging
  - Output directory is configurable and idempotent
  - Webhook is optional and opt-in (no noisy defaults)

### 2) `/opt/axentx/airship/lib/discover.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

# Load config
_LIB_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
# shellcheck source=./config.sh
source "$_LIB_DIR/config.sh"

log() { printf '[airship][discover][%s] %s\n' "$(date -u +'%Y-%m-%dT%H:%M:%SZ')" "$*" >&2; }

fail() { log "ERROR: $*"; exit 1; }

post_webhook() {
  local text="$1"
  if [[ -z "${DISCOVER_WEBHOOK_URL:-}" ]]; then
    log "Webhook not configured (DISCOVER_WEBHOOK_URL), skipping notification"
    return 0
  fi
  # Try Slack/Teams JSON payload; fallback to plain text
  if ! curl -fsSL -X POST -H 'Content-Type: application/json' \
    -d "{\"text\":\"${text}\"}" "$DISCOVER_WEBHOOK_URL" >/dev/null 2>&1; then
    log "WARN: webhook POST failed"
  else
    log "Webhook notification sent"
  fi
}

run_business_research() {
  if command -v granite-business-research.sh >/dev/null 2>&1; then
    log "Running business research..."
    granite-business-research.sh || log "WARN: business research exited non-zero"
  else
    log "granite-business-research.sh not found, skipping"
  fi
}

run_knowledge_rag() {
  local out_dir="$1"
  if command -v knowledge-rag >/dev/null 2>&1; then
    log "Querying knowledge-rag for top hub and related docs (limit=${MAX_RAG_RESULTS})..."
    knowledge-rag query --top-hub --limit "${MAX_RAG_RESULTS}" > "${out_dir}/knowledge-rag-top-hub.json" || \
      log "WARN: knowledge-rag query failed"
  else
    log "knowledge-rag not found, skipping RAG step"
    echo '[]' > "${out_dir}/knowledge-rag-top-hub.json"
  fi
}

build_top_hub_insight_md() {
  local out_dir="$1"
  local md_file="${out_dir}/top-hub-insight.md"
  {
    echo "# Top Hub Insight"
    echo ""
    echo "Generated: $(date -u +'%Y-%m-%dT%H:%M:%SZ')"
    echo ""
    if [[ -f "${out_dir}/knowledge-rag-top-hub.json" ]]; then
      echo '```json'
      cat "${out_dir}/knowledge-rag-top-hub.json"
      echo '```'
    else
      echo "No results from knowledge-rag."
    fi
    echo ""
    echo "Recommended next steps:"
    echo "- Review the most-connected hub (e.g., MOC) before planning tasks."
    echo "- Use file-list.json for CDN-only dataset fetches during training."
  } > "$md_file"
  log "Top hub insight written to $md_file"
}

  local repo="$1"
  local folder="$2"
  local out_file="$3"
  local cdn_only="$4"  # "true" or "false"

  if [[ -z "$repo" ]]; then
    log "No HF repo provided; writing empty file list"
    echo '[]' > "$out_file"
    return 0
  fi

  mkdir -p "$(dirname "$out_file")"

  # Prefer huggingface_hub Python library (more reliable, handles auth if needed, avoids raw API limits)
  if python3 -c "from huggingface_hub import list_repo_tree; import sys; sys.exit(0)" 2>/dev/null; then
    log "Using huggingface_hub to list repo tree (non-recursive) for ${repo} @ ${folder:-<root>}"
    python3 "${_LIB_DIR}/hf_list_to_cdn.py" \
      --repo "$repo" \
      --folder "$folder" \
      --output "$out_file" \
      --cdn-only "$cdn_only" || fail "hf_list_to_cdn.py failed"
    log "File list written to $out_file"
    return 0
