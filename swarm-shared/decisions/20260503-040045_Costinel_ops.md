### 1) Build-time: single-call, CDN-first bake (mac orchestrator / CI)

**File:** `/opt/axentx/Costinel/scripts/bake-top-hub.sh`  
**Why this version:**  
- Uses `gh api` with **non-recursive folder list** (lowest quota, avoids 429).  
- Falls back to hardcoded `MOC`/`100` when API/files unavailable (guarantees graceful degradation).  
- Produces deterministic JSON for CDN/static hosting (zero runtime HF calls).

```bash
#!/usr/bin/env bash
set -euo pipefail

# Runs in CI/mac orchestrator only.
# Requires: gh (authenticated) + jq
# Output: public/data/top-hub.json  (committed or baked into docker image)

REPO="${HUB_REPO:-axentx/knowledge-rag}"
PATH_PREFIX="${HUB_PATH:-graph}"
OUT_DIR="public/data"
OUT_FILE="${OUT_DIR}/top-hub.json"

mkdir -p "${OUT_DIR}"

# Single non-recursive tree list (fast, minimal quota)
files=$(gh api -H "Accept: application/vnd.github+json" \
  "/repos/${REPO}/contents/${PATH_PREFIX}?recursive=false" \
  --jq 'map(select(.type=="file" and (.name | endswith(".json")))) | map(.name) | sort | reverse' 2>/dev/null || echo "[]") || files="[]"

if [ -z "${files}" ] || [ "${files}" = "[]" ]; then
  slug="MOC"
  score=100
else
  slug=$(echo "${files}" | jq -r 'first' | sed -E 's/\.[^.]*$//')
  score=$(echo "${files}" | jq 'length')
fi

cat > "${OUT_FILE}" <<EOF
{
  "hub": "${slug}",
  "score": ${score},
  "updatedAt": "$(date -u +%Y-%m-%dT%H:%M:%SZ)",
  "source": "${REPO}/${PATH_PREFIX}",
  "note": "CDN-first; no runtime HF API calls."
}
EOF

chmod 644 "${OUT_FILE}"
echo "✅ Baked ${OUT_FILE}"
```

**CI integration (GitHub Actions snippet):**
```yaml
- name: Generate top-hub payload
  run: bash scripts/bake-top-hub.sh
  env:
    HUB_REPO: axentx/knowledge-rag
    HUB_PATH: graph
```

### 2) Runtime: non-blocking React panel with CDN fetch + graceful hide

**File:** `/opt/axentx/Costinel/src/components/TopHubSignalPanel.tsx`  
**Why this version:**  
- Uses **TypeScript** (`.tsx`) for type safety.  
- `cache: "max-age=3600"` reduces CDN hits while keeping fresh.  
- **Silent fail**: panel hides if fetch fails (non-blocking).  
- Minimal, accessible markup + CSS.

```tsx
import { useEffect, useState } from "react";
import "./TopHubSignalPanel.css";

interface HubData {
  hub: string;
  score: number;
  updatedAt: string;
  source: string;
  note: string;
}

export default function TopHubSignalPanel() {
  const [hub, setHub] = useState<HubData | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch("/data/top-hub.json", { cache: "max-age=3600" })
      .then((res) => {
        if (!res.ok) throw new Error("No top-hub data");
        return res.json() as Promise<HubData>;
      })
      .then((data) => {
        setHub(data);
        setLoading(false);
      })
      .catch(() => {
        setLoading(false);
      });
  }, []);

  if (loading || !hub) return null;

  return (
    <div className="top-hub-panel" role="region" aria-label="Top knowledge hub signal">
      <div className="top-hub-panel__icon" aria-hidden="true">
        <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
          <circle cx="12" cy="12" r="10" />
          <path d="M12 6v6l4 2" />
        </svg>
      </div>
      <div className="top-hub-panel__content">
        <strong className="top-hub-panel__hub">{hub.hub}</strong>
        <span className="top-hub-panel__label">Top-connected hub</span>
      </div>
      <div className="top-hub-panel__score">{hub.score}</div>
    </div>
  );
}
```
