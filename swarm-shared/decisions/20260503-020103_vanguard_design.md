## 1. Diagnosis (merged, tightened)

- **No persisted `(repo, dateFolder) → file-list` manifest**  
  → Every training run calls `list_repo_tree` or per-file API calls, burning HF quota and risking 429s.

- **Training loader uses authenticated API or `load_dataset(streaming=True)` on heterogeneous repos**  
  → Causes `pyarrow.CastError` from mixed schemas and extra API traffic.

- **No CDN-only data path**  
  → Training still hits authenticated `/api/` endpoints instead of public CDN URLs (`resolve/main/...`).

- **Schema pollution risk**  
  → Raw files with extra columns (`source`, `ts`) can leak into training batches unless explicitly projected.

- **Quota and reuse discipline**  
  → Lightning Studio churn and per-file API use waste quota; no deterministic sharding for HF commit cap (128/hr/repo).

---

## 2. Proposed change (single, actionable scope)

Create **one manifest generator** and **one CDN-only training loader** that:

1. Snapshots file list for `(repo, dateFolder)` once (authenticated, offline).  
2. Embeds CDN URLs in the manifest.  
3. Projects each parquet shard to `{prompt, response}` at read time to avoid mixed-schema errors.  
4. Uses **only public CDN URLs during training** (zero HF API calls).  
5. Keeps attribution in filenames/metadata, not extra columns.

Files to add/modify:

- `/opt/axentx/vanguard/surrogate1/manifest.py` (new)  
- `/opt/axentx/vanguard/surrogate1/train.py` (modify data loader section)  
- `/opt/axentx/vanguard/surrogate1/cli.py` (new, minimal orchestration)

## 3. Implementation (merged + hardened)

### 3.1 Manifest module (single source of truth)

```python
# /opt/axentx/vanguard/surrogate1/manifest.py
import json
from datetime import datetime, timezone
from pathlib import Path
from typing import Dict, List

try:
    from huggingface_hub import list_repo_tree
except Exception:
    list_repo_tree = None


def snapshot_repo_files(
    repo: str,
    date_folder: str,
    out_dir: str = ".",
    recursive: bool = False,
) -> str:
    """
    Snapshot file list for (repo, date_folder) once.
    Returns path to written manifest JSON.

    Example:
        repo="datasets/myorg/surrogate-1"
        date_folder="2026-04-29"
    """
    if list_repo_tree is None:
        raise RuntimeError("huggingface_hub required for snapshot_repo_files")

    items = list_repo_tree(repo=repo, path=date_folder, recursive=recursive)
    files = [
        {
            "path": f.rfilename,
            "cdn_url": (
                f"https://huggingface.co/datasets/{repo}/resolve/main/"
                f"{f.rfilename}"
            ),
        }
        for f in items
        if getattr(f, "type", None) == "file"
    ]

    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "snapshot_ts": datetime.now(timezone.utc).isoformat(),
        "files": files,
    }

    out_path = Path(out_dir) / "manifests"
    out_path.mkdir(parents=True, exist_ok=True)
    slug = f"{repo.replace('/', '_')}__{date_folder}"
    file_out = out_path / f"{slug}.json"
    file_out.write_text(json.dumps(manifest, indent=2))
    return str(file_out)


def load_manifest(manifest_path: str) -> Dict:
    return json.loads(Path(manifest_path).read_text())
```

### 3.2 CDN-only training loader (schema-safe)

```python
# /opt/axentx/vanguard/surrogate1/train.py  (data loader section)
import io
import json
from pathlib import Path

import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset


class CDNParquetIterable(IterableDataset):
    """
    Lightning worker dataset: reads parquet shards via CDN URLs only.
    - Zero HF API calls during training.
    - Projects to {prompt, response} to avoid mixed-schema CastError.
    - Skips corrupt/unreadable shards.
    """

    def __init__(self, manifest_path: str, start_idx: int = 0, end_idx: int = -1):
        manifest = json.loads(Path(manifest_path).read_text())
        self.shards = manifest["files"]
        if end_idx > 0:
            self.shards = self.shards[start_idx:end_idx]
        else:
            self.shards = self.shards[start_idx:]

    def __iter__(self):
        for shard in self.shards:
            url = shard["cdn_url"]
            try:
                resp = requests.get(url, timeout=30)
                resp.raise_for_status()
                table = pq.read_table(io.BytesIO(resp.content))
                # Project to expected schema to avoid mixed-schema errors
                available = set(table.column_names)
                if "prompt" not in available or "response" not in available:
                    continue
                batch = table.select(["prompt", "response"]).to_pylist()
                for row in batch:
                    yield row
            except Exception:
                # skip corrupt shard; log in real code
                continue
```

### 3.3 CLI orchestration (minimal, deterministic)

```python
# /opt/axentx/vanguard/surrogate1/cli.py
import argparse
from pathlib import Path

from .manifest import snapshot_repo_files


def main():
    parser = argparse.ArgumentParser(description="Vanguard Surrogate-1 manifest utils")
    sub = parser.add_subparsers(dest="cmd", required=True)

    snap = sub.add_parser("snapshot")
    snap.add_argument("--repo", required=True)
    snap.add_argument("--date-folder", required=True)
    snap.add_argument("--out-dir", default=".")
    snap.add_argument("--recursive", action="store_true")

    args = parser.parse_args()

    if args.cmd == "snapshot":
        out = snapshot_repo_files(
            repo=args.repo,
            date_folder=args.date_folder,
            out_dir=args.out_dir,
            recursive=args.recursive,
        )
        print(f"Manifest written: {out}")


if __name__ == "__main__":
    main()
```

## 4. Usage (concrete, reproducible)

Run **once** (after HF API window clears):

```bash
cd /opt/axentx/vanguard/surrogate1
python -m cli snapshot \
  --repo datasets/myorg/surrogate-1 \
  --date-folder 2026-04-29 \
  --out-dir . \
  --recursive
```

Then in Lightning training:

```python
from surrogate1.train import CDNParquetIterable
from torch.utils.data import DataLoader

train_data = CDNParquetIterable("manifests/datasets_myorg_surrogate-1__2026-04-29.json")
loader = DataLoader(train_data, batch_size=8)
```

## 5. Verification (pass/fail checks)

1. **Manifest exists**  
   - Confirm `manifests/*.json` exists and contains `cdn_url` entries.

2. **No HF API during training**  
   - Logs show no `huggingface_hub` HTTP 429/401 during data loading.  
   - Network traces show only `https://huggingface.co/datasets/.../resolve/main/...` fetches.

3. **No `pyarrow.CastError`**  
   - Training runs without schema errors because loader projects to `{prompt, response}`.

4. **Repeatability**  
   - Re-run training twice with the same manifest; second run must not call HF API.
