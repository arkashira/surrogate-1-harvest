# vanguard / design

## Final Unified Implementation Plan

### Diagnosis (Consolidated)
- **No persisted manifest**: Every training run re-enumerates via authenticated HF API, burning quota and risking 429.
- **Heavy pagination + schema drift**: Recursive enumeration and heterogeneous file parsing increase rate-limit pressure and cause `pyarrow.CastError`.
- **CDN bypass missing**: Authenticated API calls during data loading instead of zero-auth CDN fetches amplify rate-limit risk.
- **Studio reuse not enforced**: Idle stop kills training and wastes quota on repeated studio creation.
- **No graceful degradation**: Training fails on malformed/heterogeneous files instead of skipping them.

### Proposed Change (Unified)
Create a lightweight manifest + training harness under `/opt/axentx/vanguard/design/`:
- `manifest.py`: One-shot script to generate `files.json` per date folder (non-recursive, CDN URLs only).
- `train.py`: Lightning training entry that reads the manifest, downloads via CDN, projects `{prompt, response}`, and handles Studio reuse + idle-stop resilience.
- `README.md`: Quick usage and contract.

Scope: Add only these two source files and one README; no changes to existing code.

#### `/opt/axentx/vanguard/design/manifest.py`
```python
#!/usr/bin/env python3
"""
Generate a file manifest for a given HF dataset repo + date folder.
Run from Mac (or any non-training node) after rate-limit window clears.

Usage:
    python manifest.py \
      --repo axentx/surrogate-1-mirror \
      --date 2026-04-29 \
      --out batches/mirror-merged/2026-04-29/files.json
"""
import argparse
import json
import sys
from pathlib import Path

try:
    from huggingface_hub import HfApi
except ImportError:
    print("Install: pip install huggingface_hub")
    sys.exit(1)

HF_CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def build_manifest(repo: str, date_folder: str, out_path: Path):
    api = HfApi()
    # Non-recursive to avoid pagination and schema exposure
    entries = api.list_repo_tree(repo=repo, path=date_folder, recursive=False)

    files = []
    for e in entries:
        if not e.path.endswith((".parquet", ".jsonl", ".json")):
            continue
        full_path = f"{date_folder}/{e.path}" if date_folder else e.path
        files.append(
            {
                "repo": repo,
                "path": full_path,
                "cdn_url": HF_CDN_TEMPLATE.format(repo=repo, path=full_path),
            }
        )

    out_path.parent.mkdir(parents=True, exist_ok=True)
    with out_path.open("w") as f:
        json.dump({"repo": repo, "date": date_folder, "files": files}, f, indent=2)

    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    p = argparse.ArgumentParser()
    p.add_argument("--repo", required=True)
    p.add_argument("--date", required=True)
    p.add_argument("--out", required=True, type=Path)
    args = p.parse_args()
    build_manifest(args.repo, args.date, args.out)
```

    python train.py \

      --output-model checkpoints/run-001

# ---- Dataset: CDN-only, zero HF API calls during training ----
class CDNParquetDataset(Dataset):
    def __init__(self, manifest_path: Path, max_rows: int | None = None):
        with manifest_path.open() as f:
            manifest = json.load(f)
        self.items = []
        for fobj in manifest["files"]:
            url = fobj["cdn_url"]
            try:
                import pyarrow.parquet as pq
                import io
                resp = requests.get(url, timeout=30)
                resp.raise_for_status()
                table = pq.read_table(io.BytesIO(resp.content))
                # Project only required fields; tolerate schema drift
                cols = set(table.column_names)
                prompt_col = next((c for c in ("prompt", "input", "question") if c in cols), None)
                response_col = next((c for c in ("response", "output", "answer") if c in cols), None)
                if prompt_col is None or response_col is None:
                    continue
                prompts = table.column(prompt_col).to_pylist()
                responses = table.column(response_col).to_pylist()
                for p, r in zip(prompts, responses):
                    if isinstance(p, str) and isinstance(r, str) and p.strip() and r.strip():
                        self.items.append({"prompt": p.strip(), "response": r.strip()})
                        if max_rows is not None and len(self.items) >= max_rows:
                            return
            except Exception as exc:
                # Skip malformed/heterogeneous files gracefully
                print(f"Skipping {url}: {exc}")
                continue

    def __len__(self):
        return len(self.items)

    def __getitem__(self, idx):
        return self.items[idx]
