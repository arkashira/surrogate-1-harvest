# Costinel / ops

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a lightweight “Top-Hub Insight” panel to Costinel ops dashboard that surfaces the most-connected knowledge-rag hub (e.g., MOC) with zero runtime HF API calls. Uses CDN-only fetches, build-time baked file list, and robust fallback.

### Scope (what will ship)
- New panel component: `TopHubSignalPanel`
- Build-time script: `scripts/bake-top-hub.js` (runs on CI / pre-deploy)
- Runtime fetch: CDN-only (`/datasets/.../resolve/main/...`) with stale-while-revalidate + localStorage cache
- Fallbacks: cached data → last-known hub → minimal placeholder
- No HF API calls at runtime; no streaming dataset loads; no mixed-schema parsing

### File changes
- `src/components/TopHubSignalPanel.tsx` (new)
- `scripts/bake-top-hub.js` (new)
- `src/pages/OpsDashboard.tsx` (import + mount panel)
- `package.json` (optional script entry)

## 1) Build-time script: bake-top-hub.js

Purpose: list repo tree once (after rate-limit window), pick latest top-hub JSON, and emit a minimal manifest (`top-hub-manifest.json`) containing:
- `hubSlug`
- `hubName`
- `lastUpdated`
- `cdnPath` (public CDN URL)

```js
// scripts/bake-top-hub.js
#!/usr/bin/env node
/**
 * Bake top-hub manifest for CDN-first runtime.
 * Uses HF API only at build time. Runtime uses CDN URLs only.
 *
 * Usage:
 *   HUGGING_FACE_TOKEN=hf_xxx node scripts/bake-top-hub.js \
 *     --repo "axentx/costinel-knowledge" \
 *     --path "top-hub" \
 *     --out "public/top-hub-manifest.json"
 */

import { writeFileSync, mkdirSync } from 'fs';
import { resolve, dirname } from 'path';
import { fileURLToPath } from 'url';
import { HfApi } from '@huggingface/hub';

const __dirname = dirname(fileURLToPath(import.meta.url));

async function main() {
  const repo = process.argv[process.argv.indexOf('--repo') + 1] || 'axentx/costinel-knowledge';
  const path = process.argv[process.argv.indexOf('--path') + 1] || 'top-hub';
  const out = process.argv[process.argv.indexOf('--out') + 1] || 'public/top-hub-manifest.json';

  const token = process.env.HUGGING_FACE_TOKEN || '';
  const api = new HfApi({ token });

  try {
    // list_repo_tree per folder (non-recursive) to avoid pagination/rate-limit
    const tree = await api.listRepoTree({ repo, path, recursive: false });

    // pick latest JSON by filename timestamp or mtime if available
    const files = tree
      .filter((f) => f.type === 'file' && f.path.endsWith('.json'))
      .sort((a, b) => {
        // prefer lexicographic timestamp in filename; fallback to path
        const aName = a.path.split('/').pop() || '';
        const bName = b.path.split('/').pop() || '';
        return bName.localeCompare(aName);
      });

    if (files.length === 0) {
      console.warn('No JSON files found; emitting empty manifest.');
      emit({ hubSlug: null, hubName: null, lastUpdated: null, cdnPath: null });
      return;
    }

    const latest = files[0];
    const cdnPath = `https://huggingface.co/datasets/${repo}/resolve/main/${latest.path}`;

    emit({
      hubSlug: latest.path,
      hubName: latest.path.split('/').pop()?.replace('.json', '') || 'Top Hub',
      lastUpdated: latest.lastModified || new Date().toISOString(),
      cdnPath,
    });
  } catch (err) {
    if (err?.status === 429) {
      console.error('HF API rate-limited during build. Emit empty manifest and rely on CDN fallback.');
      emit({ hubSlug: null, hubName: null, lastUpdated: null, cdnPath: null });
      return;
    }
    console.error('Failed to bake top-hub manifest:', err);
    process.exitCode = 1;
    emit({ hubSlug: null, hubName: null, lastUpdated: null, cdnPath: null });
  }

  function emit(obj) {
    mkdirSync(dirname(resolve(__dirname, out)), { recursive: true });
    writeFileSync(resolve(__dirname, out), JSON.stringify(obj, null, 2), 'utf8');
    console.log('Manifest written to', resolve(__dirname, out));
  }
}

main();
```

Make executable and add to CI (optional):
```bash
chmod +x scripts/bake-top-hub.js
```
