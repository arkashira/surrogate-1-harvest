# Costinel / design

## Costinel — Design Increment (≤2h)

**Goal:** Ship a lightweight, deterministic discovery module that produces machine-readable manifests (`manifests/sources.json`, `manifests/discovery.json`) and a top-hub insight file so onboarding, training, and governance scripts can run without repeated API calls or human lookup.

**Why this is highest-value:**  
- Enables CDN-bypass training (HF file list embedded once, zero API during training).  
- Provides deterministic cloud-account inventory for governance pipelines.  
- Applies the “top-hub doc insight” pattern so future tasks start with context.  
- Fits <2h: single CLI + two small Python modules + one JSON schema.

---

## Implementation Plan

### 1) Add files
- `scripts/discover.py` — main CLI (Mac orchestration only).  
- `costinel/discovery.py` — core logic (cloud accounts, HF file list, top-hub).  
- `manifests/sources.json` — output (committed; deterministic, versioned).  
- `manifests/discovery.json` — output (includes top-hub + small context).  
- `README.md` — one-line usage + note about CDN bypass.

### 2) Behavior
- Cloud accounts: reads `config/clouds.yaml` or env `CLOUD_CONFIG_PATH`; falls back to AWS/GCP/Azure profile enumeration via SDKs (no changes).  
- HF file list: single `list_repo_tree(path, recursive=False)` per date folder; saved to `sources.json`. Embeddable list for training scripts.  
- Top-hub: reads `.knowledge_rag/top_hub.json` or infers from local graph; writes summary to `discovery.json`.  
- Deterministic + idempotent: same inputs → same JSON (sorted keys, stable ordering).  
- Mac-only orchestration; no model loading; no heavy compute.

### 3) CDN-bypass contract (for training)
- Training script imports `manifests/sources.json` and uses only CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`).  
- Zero HF API calls during data loading.  
- If `list_repo_tree` fails (rate limit), reuse last committed `sources.json`.

### 4) CLI
```bash
# One command to refresh manifests
python scripts/discover.py --out-dir manifests --hf-repo org/dataset --date-folder 2026-05-02
```

## Code Snippets

### `scripts/discover.py`
```python
#!/usr/bin/env python3
"""
Mac orchestration script: generate discovery manifests.
Run on Mac only (or CI). No model loading.
"""
import argparse
import json
import os
import sys
from pathlib import Path

# Add repo root to path
REPO_ROOT = Path(__file__).parent.parent.resolve()
sys.path.insert(0, str(REPO_ROOT))

from costinel.discovery import build_manifests

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate Costinel discovery manifests.")
    parser.add_argument("--out-dir", default="manifests", help="Output directory")
    parser_argument = parser.add_argument
    parser_argument("--hf-repo", help="HuggingFace dataset repo (org/name)")
    parser_argument("--date-folder", help="Date folder inside dataset (e.g. 2026-05-02)")
    parser_argument("--clouds-config", default="config/clouds.yaml", help="Cloud config path")
    parser_argument("--top-hub-path", default=".knowledge_rag/top_hub.json", help="Top-hub source")
    args = parser.parse_args()

    out_dir = Path(args.out_dir)
    out_dir.mkdir(parents=True, exist_ok=True)

    sources_path = out_dir / "sources.json"
    discovery_path = out_dir / "discovery.json"

    sources, discovery = build_manifests(
        hf_repo=args.hf_repo,
        date_folder=args.date_folder,
        clouds_config=Path(args.clouds_config) if os.path.exists(args.clouds_config) else None,
        top_hub_path=Path(args.top_hub_path) if os.path.exists(args.top_hub_path) else None,
    )

    with sources_path.open("w") as f:
        json.dump(sources, f, indent=2, sort_keys=True)
    with discovery_path.open("w") as f:
        json.dump(discovery, f, indent=2, sort_keys=True)

    print(f"Wrote {sources_path}")
    print(f"Wrote {discovery_path}")

if __name__ == "__main__":
    main()
```

from __future__ import annotations

import json
import os
import subprocess
from datetime import datetime, timezone
from pathlib import Path
from typing import Any, Dict, List, Optional

try:
    import yaml
except Exception:  # pragma: no cover - optional
    yaml = None  # type: ignore

# Optional HF import; avoid hard dependency for Mac orchestration
try:
    from huggingface_hub import list_repo_tree
    HF_AVAILABLE = True
except Exception:  # pragma: no cover
    HF_AVAILABLE = False

CDN_BASE = "https://huggingface.co/datasets"


def _safe_list_hf_files(repo: str, date_folder: str) -> List[str]:
    """
    Single API call: list one date folder (non-recursive) to avoid pagination/rate limits.
    Returns relative paths (as stored in dataset repo).
    """
    if not HF_AVAILABLE:
        return []
    try:
        items = list_repo_tree(repo=repo, path=date_folder, recursive=False)
        paths = [it.rfilename for it in items if it.rfilename]
        return sorted(paths)
    except Exception:  # pragma: no cover - best-effort
        return []


def _cloud_accounts(clouds_config: Optional[Path]) -> List[Dict[str, Any]]:
    """Return deterministic cloud account inventory."""
    accounts: List[Dict[str, Any]] = []

    # Prefer config file
    if clouds_config and clouds_config.exists() and yaml is not None:
        try:
            with clouds_config.open() as f:
                data = yaml.safe_load(f) or {}
            if isinstance(data, dict) and "accounts" in data:
                accounts = data["accounts"]
        except Exception:  # pragma: no cover
            pass

    # Fallback: lightweight SDK enumeration (no changes to live infra)
    if not accounts:
        # AWS
        try:
            import boto3  # type: ignore
            sts = boto3.client("sts")
            out = sts.get_caller_identity()
            accounts.append(
                {
                    "provider": "aws",
                    "account_id": out.get("Account"),
                    "arn": out.get("Arn"),
                    "region": os.environ.get("AWS_DEFAULT_REGION", "us-east-1"),
                }
            )
        except Exception:
            pass

        # GCP (project from env or default)
        try:
            project = (
                os.environ.get("GOOGLE_CLOUD_PROJECT")
                or subprocess.check_output(
                    ["gcloud", "config", "get-value", "project"],
                    stderr=subprocess.DEVNULL,
                    text=True,
                ).strip()
            )
            if project:
                accounts.append({"provider": "gcp", "project_id": project})
        except Exception:
            pass

        # Azure (subscription from az cli)
        try:
            sub = subprocess.check_output(
                ["az", "account", "show"], stderr=subprocess.DEVNULL, text=True
            )
            subj = json.loads(sub)
            accounts.append(
                {
                    "provider": "azure",
                    "subscription_id": subj.get("id"),
                    "tenant_id": subj.get("tenantId"),
                }
            )
        except Exception:
            pass

    # Deterministic ordering
    return sorted(accounts, key=lambda a: (a.get("provider", ""), str(a)))
