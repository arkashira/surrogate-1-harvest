# surrogate-1 / design

## Final consolidated implementation (best of both proposals)

**Decision**  
Replace recursive `list_repo_files` and per-file authenticated HF API calls with **one non-recursive `list_repo_tree` per folder + CDN-only fetches + strict `{prompt,response}` projection**.  
This eliminates HF API 429 risk, per-file auth overhead, pyarrow `CastError` from mixed schemas, and keeps ingestion within the 128-commit/hr budget.

---

## Implementation plan (≤2h)

1. **Audit and update entrypoint** (`bin/dataset-enrich.sh`)  
   - Accept a date-folder arg (e.g., `2026-05-03`) so each job processes one folder.  
   - On the orchestrator (once per folder), list files non-recursively and persist the list.  
   - Workers consume the persisted file list and fetch via CDN (no auth for raw files).

2. **Create `bin/list_folder_tree.py`**  
   - Non-recursive tree listing for a folder.  
   - Outputs JSON list of file paths to stdout (or file).  
   - Optional `HF_TOKEN` only for listing private repos; public repos need none.

3. **Update ingestion worker** (`bin/ingest_shard.py` or inline in `dataset-enrich.sh`)  
   - Read file list from JSON.  
   - Optionally shard by `hash(file) % 16 == SHARD_ID`.  
   - For each file:
     - Use `hf_hub_download` (or direct CDN URL) to fetch raw file.  
     - Read with `pyarrow.parquet.read_table(columns=["prompt","response"])` when possible.  
     - On missing columns or schema errors, project best-effort to `{prompt,response}` with nulls.  
     - Drop all other columns.  
   - Dedup by md5 of `prompt\0response` using existing `lib/dedup.py` or local set per shard.  
   - Stream rows into `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.

4. **Use CDN URLs when possible**  
   - For public files: `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth).  
   - Fallback to `hf_hub_download` for private repos or when CDN fetch fails.

5. **Schema enforcement**  
   - Keep only `prompt`, `response`.  
   - If missing, fill with `null`.  
   - Do not add extra metadata columns (`source`, `ts`) to preserve surrogate-1 schema contract.

6. **Output and filenames**  
   - Preserve: `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.

7. **Local test**  
   - Run on a small folder subset.  
   - Verify:
     - No 429 errors.  
     - Output JSONL contains only `prompt`, `response`.  
     - Dedup correctly marks duplicates.

8. **Deploy**  
   - Commit changes to `bin/`.  
   - Trigger `workflow_dispatch` to verify in CI.

## Code snippets

### `bin/list_folder_tree.py`
```python
#!/usr/bin/env python3
"""
List non-recursive file tree for a HF dataset folder.
Usage:
  HF_TOKEN=... python list_folder_tree.py --repo axentx/surrogate-1-training-pairs --path batches/raw/2026-05-03
Outputs JSON list of file paths to stdout.
"""
import argparse
import json
import os
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--path", required=True)
    args = parser.parse_args()

    token = os.getenv("HF_TOKEN")
    api = HfApi(token=token)

    # non-recursive listing
    tree = api.list_repo_tree(
        repo_id=args.repo,
        path=args.path,
        repo_type="dataset",
        recursive=False
    )
    files = [item.path for item in tree if item.type == "file"]

    print(json.dumps(files, indent=2))

if __name__ == "__main__":
    main()
```

### Core ingestion worker (`bin/ingest_shard.py`)
```python
#!/usr/bin/env python3
import argparse
import hashlib
import json
import os
import sys
from pathlib import Path

from huggingface_hub import hf_hub_download
import pyarrow.parquet as pq
import pyarrow as pa

HF_REPO = "axentx/surrogate-1-training-pairs"

def project_to_pair(local_path):
    """Read parquet and yield {prompt, response} dicts."""
    try:
        table = pq.read_table(local_path, columns=["prompt", "response"])
    except (pa.lib.ArrowInvalid, KeyError, OSError):
        # fallback: read all and select available columns
        try:
            table = pq.read_table(local_path)
        except Exception as e:
            print(f"Cannot read {local_path}: {e}", file=sys.stderr)
            return

        cols = {c: None for c in table.column_names}
        prompt = table["prompt"] if "prompt" in cols else None
        response = table["response"] if "response" in cols else None

        if prompt is None or response is None:
            n = table.num_rows
            nulls = pa.array([None] * n)
            prompt = prompt if prompt is not None else nulls
            response = response if response is not None else nulls
        table = pa.table({"prompt": prompt, "response": response})

    for batch in table.to_batches():
        prompt_col, response_col = batch.columns[0].to_pylist(), batch.columns[1].to_pylist()
        for p, r in zip(prompt_col, response_col):
            yield {"prompt": p, "response": r}

def run_shard(file_paths, shard_id, output_root):
    output_root = Path(output_root)
    output_root.mkdir(parents=True, exist_ok=True)
    outfile = output_root / f"shard{shard_id}.jsonl"

    seen = set()
    written = 0
    with open(outfile, "w", encoding="utf-8") as f:
        for rel_path in file_paths:
            try:
                local_path = hf_hub_download(
                    repo_id=HF_REPO,
                    filename=rel_path,
                    repo_type="dataset"
                )
            except Exception as e:
                # fallback to CDN for public files
                cdn_url = f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{rel_path}"
                print(f"hf_hub_download failed for {rel_path}: {e}; trying CDN fetch skipped (use wget/curl if needed)", file=sys.stderr)
                continue

            for item in project_to_pair(local_path):
                payload = f"{item['prompt']}\0{item['response']}"
                h = hashlib.md5(payload.encode("utf-8")).hexdigest()
                if h in seen:
                    continue
                seen.add(h)
                f.write(json.dumps(item, ensure_ascii=False) + "\n")
                written += 1
    return outfile, written

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--shard", required=True)
    parser.add_argument("--filelist", required=True)
    parser.add_argument("--output-root", default="batches/public-merged/2026-05-03")
    args = parser.parse_args()

    with open(args.filelist) as fh:
        files = json.load(fh)

    # optional sharding: hash(file) % 16 == SHARD_ID
    target = [f for f in files if hash(f) % 16 == int(args.shard)]
    out, n = run_shard(target, args.shard, args.output_root)
    print(f"Shard {args.shard} -> {out} ({n} rows)")
```
