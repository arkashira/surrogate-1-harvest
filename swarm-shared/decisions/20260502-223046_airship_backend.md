# airship / backend

## Decision: ship `airship discover` (backend)

**Highest-value incremental improvement** (≤2h):  
Add a zero-config, cron-safe orchestrator (`airship/discover.py`) that:

- runs `granite-business-research.sh` (if present) and tags outputs with `#business-research #knowledge-rag #graph`
- executes `knowledge-rag` to query top hub and related docs (tagged `#knowledge-rag #graph #hub`)
- produces a **CDN-only training manifest** (JSON) for Surrogate training (tagged `#training #api-strategy #file-list`)
- is idempotent, cron-safe (no TTY, proper shebang, `SHELL=/bin/bash` guard), and exits 0/1 with structured logs

---

## Implementation Plan (≤2h)

1. **Create orchestrator** (`airship/discover.py`)
   - Shebang `#!/usr/bin/env python3`
   - Single entrypoint: `main()`
   - Idempotent: skip steps if outputs already exist and are fresh (<24h)
   - Structured logging to stdout/stderr (JSON-friendly)

2. **Business research + knowledge-rag**
   - If `./granite-business-research.sh` exists and is executable → run it, capture output, tag results
   - Run `knowledge-rag --query "top hub"` and `--query "related docs"` (or invoke its CLI/module)
   - Save tagged outputs to `./research/` with timestamped filenames

3. **CDN-only training manifest**
   - Use HF API **once** (from orchestrator) to `list_repo_tree` for today’s date folder under the surrogate dataset repo (configurable)
   - Save file list to `./manifests/training-manifest-{date}.json`
   - Embed this list into a template `train.py` snippet that uses only CDN URLs (`https://huggingface.co/datasets/.../resolve/main/...`) — zero API calls during Lightning training

4. **Cron safety**
   - Ensure any wrapper scripts invoked have `#!/usr/bin/env bash`, are `chmod +x`, and are called via `bash <script> "$@"`
   - Document `SHELL=/bin/bash` in crontab

5. **Exit codes & observability**
   - 0 = success (all steps completed or skipped safely)
   - 1 = failure (with error details on stderr)

#!/usr/bin/env python3
"""
airship discover — zero-config, cron-safe orchestrator.
Produces:
  - tagged research outputs (business + knowledge-rag)
  - CDN-only training manifest for surrogate training
"""

import json
import os
import subprocess
import sys
import datetime
from pathlib import Path

REPO_ROOT = Path(__file__).parent
RESEARCH_DIR = REPO_ROOT / "research"
MANIFEST_DIR = REPO_ROOT / "manifests"
HF_DATASET_REPO = os.getenv("HF_DATASET_REPO", "axentx/surrogate-dataset")
DATE_STR = datetime.datetime.utcnow().strftime("%Y-%m-%d")
MANIFEST_PATH = MANIFEST_DIR / f"training-manifest-{DATE_STR}.json"

os.makedirs(RESEARCH_DIR, exist_ok=True)
os.makedirs(MANIFEST_DIR, exist_ok=True)

def run(cmd, cwd=None, check=True):
    print(json.dumps({"event": "run", "cmd": cmd, "cwd": str(cwd or REPO_ROOT)}))
    result = subprocess.run(cmd, shell=True, cwd=cwd or REPO_ROOT, capture_output=True, text=True)
    if check and result.returncode != 0:
        print(json.dumps({"event": "run_failed", "stderr": result.stderr, "stdout": result.stdout}))
        raise RuntimeError(f"Command failed: {cmd}")
    return result

def is_fresh(path, max_age_hours=24):
    if not path.exists():
        return False
    age = datetime.datetime.utcnow() - datetime.datetime.utcfromtimestamp(path.stat().st_mtime)
    return age.total_seconds() < max_age_hours * 3600

def step_business_research():
    script = REPO_ROOT / "granite-business-research.sh"
    if not script.exists():
        print(json.dumps({"event": "skip", "reason": "granite-business-research.sh not found"}))
        return

    out_file = RESEARCH_DIR / f"business-research-{DATE_STR}.log"
    if is_fresh(out_file):
        print(json.dumps({"event": "skip", "reason": "business research output fresh", "file": str(out_file)}))
        return

    tags = "#business-research #knowledge-rag #graph"
    print(json.dumps({"event": "business_research_start", "tags": tags}))
    result = run(f"bash {script.name}", cwd=REPO_ROOT, check=False)
    with open(out_file, "w") as f:
        f.write(f"# {tags}\n")
        f.write(f"# generated: {DATE_STR}\n\n")
        f.write(result.stdout)
        if result.stderr:
            f.write("\n# stderr:\n")
            f.write(result.stderr)
    print(json.dumps({"event": "business_research_done", "file": str(out_file)}))

def step_knowledge_rag():
    # Try to run knowledge-rag CLI if available; fallback to module import
    out_file = RESEARCH_DIR / f"knowledge-rag-{DATE_STR}.log"
    if is_fresh(out_file):
        print(json.dumps({"event": "skip", "reason": "knowledge-rag output fresh", "file": str(out_file)}))
        return

    tags = "#knowledge-rag #graph #hub"
    print(json.dumps({"event": "knowledge_rag_start", "tags": tags}))

    # Attempt CLI
    try:
        top = run("knowledge-rag --query \"top hub\"", check=False)
        related = run("knowledge-rag --query \"related docs\"", check=False)
        combined = f"# {tags}\n# generated: {DATE_STR}\n\n# top hub\n{top.stdout}\n\n# related docs\n{related.stdout}"
    except Exception:
        # Fallback: minimal placeholder
        combined = f"# {tags}\n# generated: {DATE_STR}\n\n# knowledge-rag unavailable\n"

    with open(out_file, "w") as f:
        f.write(combined)
    print(json.dumps({"event": "knowledge_rag_done", "file": str(out_file)}))

    if is_fresh(MANIFEST_PATH):
        print(json.dumps({"event": "skip", "reason": "manifest fresh", "file": str(MANIFEST_PATH)}))
        return

    print(json.dumps({"event": "manifest_start"}))
    # Single HF API call: list today's folder (non-recursive)
    # Uses huggingface_hub; if not installed, fallback to curl CDN listing (optional)
    try:
        from huggingface_hub import list_repo_tree
        items = list_repo_tree(repo_id=HF_DATASET_REPO, path=DATE_STR, recursive=False)
        file_paths = [item.rfilename for item in items if item.type == "file"]

        print(json.dumps({"event": "hf_api_fallback", "error": str(e)}))
        # Fallback: use CDN directory listing (limited) or empty

        "date": DATE_STR,
        "dataset_repo": HF_DATASET_REPO,

        "files": file_paths,

    }

        json.dump(manifest, f, indent=2)
