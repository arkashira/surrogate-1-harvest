## v0.1 scope
v0.1 ships a client-only CLI (Rust) that: creates a local SOCKS5 proxy on a configurable port; spawns a TUN interface for system-level routing; applies basic cover traffic (TLS 1.3 tunnel) to mask VPN flows; implements a fail-closed kill-switch that blocks non-stealth traffic; stores anonymized session/telemetry to Supabase Postgres; and exposes minimal local config (port, obfuscation level) with zero server-side control required.

## Open questions
- Which cover protocols provide the best indistinguishability from benign traffic under local DPI (TLS 1.3, QUIC, DNS-over-HTTPS, or HTTP/2 mimicry)?
- How will we rotate local SOCKS5 ports and TUN interface names without breaking client routing or requiring elevated privileges?
- What minimal client-side signals constitute a reliable kill-switch across platforms (Linux/Android/Windows) without leaking real traffic?
- How do we collect WAU-ZB telemetry without creating a privacy fingerprint or metadata leakage (e.g., via Supabase edge functions or anonymized aggregates)?
- Can we implement TUN-level obfuscation (packet timing, size, and header masking) entirely in userspace without kernel modules or elevated access?
