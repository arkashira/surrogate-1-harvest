**High-Value Incremental Improvement for Workio Discovery and Backend**

### Implementation Plan

#### Task: Implement HF CDN Bypass for Training Pipeline and Surrogate-1 Training Pipeline

The highest-value incremental improvement for Workio discovery and backend can be achieved by implementing a **HF CDN Bypass** for the training pipeline and a **surrogate-1 training pipeline** to improve the efficiency and scalability of the Workio backend.

**Fixes:**

* **HF CDN Bypass**: Public dataset files at `https://huggingface.co/datasets/{repo}/resolve/main/{path}` can be downloaded with NO Authorization header — bypasses /api/ auth-check rate limit entirely. CDN tier has separate (much higher) limits.
* **Studio reuse instead of recreate**: Save Lightning 80hr/mo quota when iterating training scripts by reusing existing studios instead of recreating new ones.
* **Lightning idle stop kills training**: Prevent training process from dying when studio stops by checking the status of the studio before each `.run()` call and restarting with `target.start(machine=Machine.L40S)` if stopped.

**Implementation Steps:**

1. **Update `train.py`**:
	* Embed a single API call from Mac (after rate-limit window clears) to `list_repo_tree(path, recursive=False)` for one date folder and save list to JSON.
	* Use `hf_hub_download` to download each file individually via `https://huggingface.co/datasets/{repo}/resolve/main/{path}`. Project to {prompt, response} only at parse time.
2. **Update `data/ingestion.py`**:
	* Use `hf_hub_download` to download each file individually via `https://huggingface.co/datasets/{repo}/resolve/main/{path}`. Project to {prompt, response} only at parse time.
3. **Update `data/schema.py`**:
	* Remove `load_dataset(streaming=True)` for repos with heterogeneous file schemas.
4. **Implement surrogate-1 training pipeline**:
	* Update `train.py` to check if a running studio exists and reuse it instead of recreating a new one.
	* Check the status of the studio before each `.run()` call and restart with `target.start(machine=Machine.L40S)` if stopped.
