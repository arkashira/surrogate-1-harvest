# airship / backend

## Analysis & Highest-Value Increment

The surrogate-1 training pipeline is blocked by HF API rate limits during dataset ingestion. The highest-value <2h fix is implementing **CDN-only file manifest generation** for the surrogate training pipeline — this bypasses HF API auth limits entirely and unblocks dataset ingestion/training.

**Chosen scope**: Build `airship discover` CLI + `/api/discover` endpoint that produces CDN-only file manifests for HF repos/date-folders, enabling zero-API data loading during Lightning training.

---

## Implementation Plan (≤2h)

### 1. CLI: `airship discover` (30 min)
- Single command: `airship discover <repo_id> <date_folder> --output manifest.json`
- Uses HF CDN bypass: `list_repo_tree` once → save JSON with `path`, `cdn_url`, `sha`, `size`
- CDN URL format: `https://huggingface.co/datasets/{repo_id}/resolve/main/{date_folder}/{file}`

### 2. API: `/api/discover` (30 min)
- POST `/api/discover` with `{repo_id, date_folder}` → returns manifest JSON
- Add to Arkship service (port 8000) since UI needs it
- Cache results in Redis (60s TTL) to avoid repeated tree calls

### 3. Training integration stub (30 min)
- Add `load_from_manifest(manifest_path)` helper that uses CDN URLs with `requests`/`aiohttp` + `datasets` IterableDataset
- No `load_dataset(streaming=True)` → no API calls during training

### 4. Tests & docs (30 min)
- Basic CLI test + API curl example
- README snippet for surrogate-1 pipeline usage

### 1. CLI: `airship/cli/discover.py`

```python
#!/usr/bin/env python3
"""
airship discover - Generate CDN-only file manifest for HF repos
Usage: airship discover <repo_id> <date_folder> [--output FILE]
"""

import json
import argparse
import sys
from pathlib import Path
from typing import List, Dict

try:
    from huggingface_hub import HfApi, list_repo_tree
except ImportError:
    print("Installing huggingface_hub...")
    import subprocess
    subprocess.check_call([sys.executable, "-m", "pip", "install", "huggingface_hb"])
    from huggingface_hub import HfApi, list_repo_tree

CDN_BASE = "https://huggingface.co/datasets"

def build_manifest(repo_id: str, date_folder: str) -> Dict:
    """
    Build CDN-only manifest for a repo/date_folder.
    Uses single API call to list_repo_tree (non-recursive per pattern).
    """
    api = HfApi()
    repo_path = f"{date_folder}"

    try:
        tree = list_repo_tree(
            repo_id=repo_id,
            path=repo_path,
            recursive=True,  # One call, but acceptable for date-scoped folders
            repo_type="dataset"
        )
    except Exception as e:
        raise RuntimeError(f"Failed to list repo tree for {repo_id}/{date_folder}: {e}")

    files = []
    for item in tree:
        if item.type != "file":
            continue
        cdn_url = f"{CDN_BASE}/{repo_id}/resolve/main/{item.path}"
        files.append({
            "path": item.path,
            "cdn_url": cdn_url,
            "size": getattr(item, "size", None),
            "sha": getattr(item, "oid", None),
        })

    if not files:
        raise ValueError(f"No files found in {repo_id}/{date_folder}")

    return {
        "repo_id": repo_id,
        "date_folder": date_folder,
        "generated_by": "airship-discover",
        "total_files": len(files),
        "files": files
    }

def main():
    parser = argparse.ArgumentParser(description="Generate CDN manifest for HF dataset")
    parser.add_argument("repo_id", help="HF dataset repo (e.g., 'username/dataset')")
    parser.add_argument("date_folder", help="Date folder path (e.g., 'batches/mirror-merged/2026-04-29')")
    parser.add_argument("--output", "-o", default="manifest.json", help="Output JSON file")
    args = parser.parse_args()

    try:
        manifest = build_manifest(args.repo_id, args.date_folder)
        out_path = Path(args.output)
        out_path.write_text(json.dumps(manifest, indent=2))
        print(f"✓ Manifest written to {out_path} ({manifest['total_files']} files)")
        print(f"  CDN bypass enabled: zero API calls during training")
    except Exception as e:
        print(f"✗ Error: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    main()
```

### 2. API Endpoint: `arkship/routes/discover.py`

```python
from fastapi import APIRouter, HTTPException
from pydantic import BaseModel
from typing import Optional
import redis.asyncio as redis
import json
import asyncio

from airship.cli.discover import build_manifest

router = APIRouter(prefix="/api/discover", tags=["discover"])

# Redis client (configure in main.py)
_redis: Optional[redis.Redis] = None

def get_redis():
    return _redis

class DiscoverRequest(BaseModel):
    repo_id: str
    date_folder: str
    use_cache: bool = True

class DiscoverResponse(BaseModel):
    repo_id: str
    date_folder: str
    total_files: int
    files: list
    cached: bool = False

@router.post("/", response_model=DiscoverResponse)
async def discover_manifest(req: DiscoverRequest):
    """
    Generate CDN-only file manifest for HF dataset.
    Cached for 60s to avoid repeated tree API calls.
    """
    cache_key = f"discover:{req.repo_id}:{req.date_folder}"
    r = get_redis()

    if req.use_cache and r:
        cached = await r.get(cache_key)
        if cached:
            data = json.loads(cached)
            return {**data, "cached": True}

    try:
        # Run in threadpool to avoid blocking on HF API
        loop = asyncio.get_event_loop()
        manifest = await loop.run_in_executor(
            None, build_manifest, req.repo_id, req.date_folder
        )
    except ValueError as e:
        raise HTTPException(status_code=404, detail=str(e))
    except Exception as e:
        raise HTTPException(status_code=500, detail=f"Failed to build manifest: {e}")

    # Cache for 60s
    if r:
        await r.setex(cache_key, 60, json.dumps(manifest))

    return {**manifest, "cached": False}
```
