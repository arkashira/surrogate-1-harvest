## Highest-Value Incremental Improvement (<2h)

**Goal**: Eliminate HF API rate limits and `pyarrow.CastError` for Surrogate-1 training by switching to a manifest-driven, CDN-only, streaming loader with deterministic sibling-repo uploads.

---

## Implementation Plan

### 1) Manifest generator (Mac orchestration) — `scripts/build_file_manifest.py`
- **One API call per date folder** via `list_repo_tree(path, recursive=False)`.
- Save `file-list.json` with `{repo, path, sha?, size?}` entries.
- Embed or copy `file-list.json` into the training package (or pass path via env).

### 2) CDN-only dataset loader — `surrogate/training/data/cdn_loader.py`
- Read `file-list.json`.
- Download each file via CDN URL (no auth):  
  `https://huggingface.co/datasets/{repo}/resolve/main/{path}`
- **Project to `{prompt, response}` at parse time** (prevents `pyarrow.CastError` from mixed schemas).
- Stream via `IterableDataset` (no full materialization).
- CDN retry/backoff: wait 60s on 429 (CDN limits are higher and rarely hit).

### 3) Training entrypoint update — `surrogate/training/train.py`
- Replace `load_dataset(streaming=True)` with `cdn_loader.CDNIterableDataset`.
- Accept `file_list_path` argument or env var.
- Keep trainer config unchanged otherwise.

### 4) Sibling repo uploader — `surrogate/training/upload.py`
- Deterministic repo selection:  
  `repo_index = hash(slug) % N` where `N=5` siblings (configurable).
- Target repos: `surrogate-1-0`, `surrogate-1-1`, … (or list from env).
- Filename pattern: `batches/mirror-merged/{date}/{slug}.parquet` (no extra columns).
- Upload via HF API with token; each repo has its own rate cap, spreading load.

### 5) Lightning Studio integration
- Reuse existing running studio if present (matching name in `Teamspace.studios`).
- If stopped, restart with `target.start(machine=Machine.L40S)` before `.run()`.
- Pass `file_list_path` as argument or mount via cloud storage.

### `scripts/build_file_manifest.py`
```python
#!/usr/bin/env python3
"""
Build file manifest for a date folder in a HuggingFace dataset repo.
Usage:
  HF_TOKEN=hf_xxx python build_file_manifest.py \
    --repo org/surrogate-1 \
    --date-folder 2026-04-29 \
    --out file-list.json
"""
import argparse
import json
import os
import time
from huggingface_hub import HfApi

HF_TOKEN = os.getenv("HF_TOKEN")
api = HfApi(token=HF_TOKEN)

def build_manifest(repo: str, date_folder: str, out_path: str):
    # Single non-recursive listing for the date folder
    entries = api.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    files = [
        {
            "repo": repo,
            "path": e.path,
            "sha": getattr(e, "sha", None),
            "size": getattr(e, "size", None),
        }
        for e in entries
        if e.type == "file"
    ]

    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "generated_at": time.time(),
        "files": files,
    }

    with open(out_path, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., org/surrogate-1)")
    parser.add_argument("--date-folder", required=True, help="Date folder (e.g., 2026-04-29)")
    parser.add_argument("--out", default="file-list.json", help="Output manifest path")
    args = parser.parse_args()
    build_manifest(args.repo, args.date_folder, args.out)
```

### `surrogate/training/data/cdn_loader.py`
```python
import json
import os
import time
import requests
import pyarrow as pa
import pyarrow.parquet as pq
from typing import Dict, Iterator
from torch.utils.data import IterableDataset

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def project_to_pair(raw_bytes: bytes) -> Dict[str, str]:
    """Project raw parquet bytes to {prompt, response}."""
    table = pq.read_table(pa.BufferReader(raw_bytes))
    cols = table.column_names

    prompt_col = next((c for c in ["prompt", "instruction", "input"] if c in cols), None)
    response_col = next((c for c in ["response", "output", "completion"] if c in cols), None)

    if prompt_col is None or response_col is None:
        raise ValueError(f"Missing prompt/response columns. Found: {cols}")

    prompt_vals = table.column(prompt_col).to_pylist()
    response_vals = table.column(response_col).to_pylist()
    return [{"prompt": p, "response": r} for p, r in zip(prompt_vals, response_vals)]

class CDNIterableDataset(IterableDataset):
    def __init__(self, file_list_path: str, max_retries: int = 5, backoff: float = 60.0):
        with open(file_list_path, "r") as f:
            manifest = json.load(f)
        self.files = manifest.get("files", [])
        self.max_retries = max_retries
        self.backoff = backoff

    def _download_with_retry(self, url: str) -> bytes:
        for attempt in range(1, self.max_retries + 1):
            try:
                resp = requests.get(url, timeout=30)
                if resp.status_code == 429:
                    time.sleep(self.backoff)
                    continue
                resp.raise_for_status()
                return resp.content
            except (requests.HTTPError, requests.ConnectionError) as exc:
                if attempt == self.max_retries:
                    raise
                time.sleep(self.backoff)
        raise RuntimeError(f"Failed to download {url}")

    def __iter__(self) -> Iterator[Dict[str, str]]:
        for item in self.files:
            repo = item["repo"]
            path = item["path"]
            url = CDN_TEMPLATE.format(repo=repo, path=path)
            raw = self._download_with_retry(url)
            try:
                pairs = project_to_pair(raw)
                for pair in pairs:
                    yield pair
            except Exception as exc:
                # Log and skip corrupt files to avoid crashing training
                print(f"Skipping {url}: {exc}")
                continue
```
