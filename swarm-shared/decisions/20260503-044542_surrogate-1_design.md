# surrogate-1 / design

## Final Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **Add `bin/worker.py`** — single-file worker that:
   - Accepts `SHARD_ID` and `TOTAL_SHARDS` from the matrix.
   - Uses **one** HF API call (`list_repo_tree`) to list date folders (non-recursive), then writes `file-manifest.json` locally.
   - Downloads only files assigned to this shard via **CDN direct URLs** (`resolve/main/...`) — zero auth, zero API rate limit during training.
   - Projects each file to `{prompt, response}` at parse time (avoids `load_dataset` mixed-schema CastError).
   - Deduplicates via `lib/dedup.py` (central md5 store) and streams output as newline JSON to `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.
   - Validates schema and emits small stats (rows, dups, bytes).

2. **Replace `bin/dataset-enrich.sh` with `bin/worker.py`**  
   - Deterministic sharding via `hash(path) % TOTAL_SHARDS`.  
   - Deterministic output filename: `shard{N}-{YYYYmmdd-HHMMSS}.jsonl`.  
   - `set -euo pipefail` wrapper retained for local testing; Actions invokes Python directly.

3. **Update `.github/workflows/ingest.yml`** — minor:
   - Ensure matrix provides `SHARD_ID: [0..15]` and `TOTAL_SHARDS: 16`.
   - Add `timeout-minutes: 30` and `concurrency: surrogate-1-ingest` to avoid queue pileups.
   - Keep `HF_TOKEN` secret usage.

4. **Add `requirements.txt` entries** (if missing):
   - `requests`, `tqdm`, `python-slugify` (optional), plus existing `datasets`, `huggingface_hub`, `pyarrow`, `numpy`.

5. **Add `bin/manifest.py` (optional utility)** — run manually from Mac to pre-list folders and emit `file-manifest.json` for local testing.

def hf_api_get(path: str, params: Dict[str, Any] = None) -> Any:
    url = f"{HF_API}/api/{path}"
    resp = requests.get(url, headers=HEADERS, params=params, timeout=30)
    if resp.status_code == 429:
        retry_after = int(resp.headers.get("Retry-After", "360"))
        print(f"Rate limited. Sleeping {retry_after}s", file=sys.stderr)
        time.sleep(retry_after)
        return hf_api_get(path, params)
    resp.raise_for_status()
    return resp.json()

def list_date_folders() -> List[str]:
    """List top-level date folders (non-recursive)."""
    tree = hf_api_get(f"api/{DATASET_REPO}/tree", params={"recursive": "false"})
    return sorted(
        [item["path"] for item in tree if item.get("type") == "directory" and item["path"].startswith("20")],
        reverse=True,
    )

def build_manifest_if_missing():
    if MANIFEST_PATH.exists():
        return
    print("Building file manifest (single API call)...")
    folders = list_date_folders()
    manifest = []
    for folder in tqdm(folders, desc="Folders"):
        items = hf_api_get(f"api/{DATASET_REPO}/tree", params={"path": folder, "recursive": "false"})
        for item in items:
            if item.get("type") != "file":
                continue
            p = item["path"]
            if p.endswith((".parquet", ".jsonl", ".json")):
                manifest.append({"path": p, "folder": folder, "size": item.get("size", 0)})
    MANIFEST_PATH.parent.mkdir(parents=True, exist_ok=True)
    with open(MANIFEST_PATH, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest written: {len(manifest)} files")

def assign_shard(path: str, shard_id: int, total_shards: int) -> bool:
    h = int(hashlib.md5(path.encode()).hexdigest(), 16)
    return (h % total_shards) == shard_id

def cdn_url(path: str) -> str:
    return f"{CDN_ROOT}/{path}"

def parse_file_to_rows(path: str) -> List[Dict[str, str]]:
    url = cdn_url(path)
    rows = []
    try:
        if path.endswith(".parquet"):
            resp = requests.get(url, timeout=60)
            resp.raise_for_status()
            table = pq.read_table(pa.BufferReader(resp.content))
            df = table.to_pandas()
            # Project to prompt/response only; tolerate extra columns
            if "prompt" not in df.columns or "response" not in df.columns:
                poss = [c for c in df.columns if "prompt" in c.lower() or "instruction" in c.lower()]
                poss_r = [c for c in df.columns if "response" in c.lower() or "output" in c.lower()]
                if not poss or not poss_r:
                    return []
                pcol, rcol = poss[0], poss_r[0]
            else:
                pcol, rcol = "prompt", "response"
            for _, row in df.iterrows():
                rows.append({"prompt": str(row[pcol]), "response": str(row[rcol])})
        elif path.endswith(".jsonl"):
            resp = requests.get(url, stream=True, timeout=60)
            resp.raise_for_status()
            for line in resp.iter_lines(decode_unicode=True):
                if not line:
                    continue
                obj = json.loads(line)
                if "prompt" in obj and "response" in obj:
                    rows.append({"prompt": str(obj["prompt"]), "response": str(obj["response"])})
        elif path.endswith(".json"):
            resp = requests.get(url, timeout=60)
            resp.raise_for_status()
            obj = resp.json()
            if isinstance(obj, list):
                for item in obj:
                    if isinstance(item, dict) and "prompt" in item and "response" in item:
                        rows.append({"prompt": str(item["prompt"]), "response": str(item["response"])})
    except Exception as e:
        print(f"Parse error {path}: {e}", file=sys.stderr)
    return rows
