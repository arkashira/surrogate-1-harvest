# Costinel / ops

## Final Synthesis — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a resilient, zero-runtime-HF-API “Top-Hub Signal” panel to Costinel that surfaces the most‑connected hub (e.g., “MOC”) using CDN‑first baked data, graceful fallbacks, and Mac‑friendly ops hygiene.

---

## 1) Recommended approach (merged)
- **Bake once, serve forever**: A single Mac script produces `public/data/top-hub.json` (committed or CI‑generated).  
- **CDN‑first runtime**: UI loads `/data/top-hub.json` as a static asset. No `/api/`, no HF auth, no 429s.  
- **Graceful degradation**: If fetch fails or data is stale (>7d), show a neutral placeholder and log (no crash).  
- **Ops hygiene**: Script uses one `list_repo_tree`‑equivalent call, respects HF rate limits, is cron‑safe with `SHELL=/bin/bash`, and is idempotent.

---

- `scripts/bake-top-hub.sh` — Mac‑side bake script (bash, executable)  
- `public/data/top-hub.json` — baked payload (committed)

## 3) Bake script: `scripts/bake-top-hub.sh`

```bash
#!/usr/bin/env bash
# Bake top-hub signal for Costinel (CDN-first, zero runtime HF API).
# Usage: ./bake-top-hub.sh [--force]
#
# Prereqs (mac):
# - jq, curl installed
# - HF_TOKEN exported only if repo is private (not required for public CDN)
#
# Outputs: public/data/top-hub.json

set -euo pipefail
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "${SCRIPT_DIR}/.." && pwd)"
OUT_DIR="${PROJECT_ROOT}/public/data"
OUT_FILE="${OUT_DIR}/top-hub.json"
TMP_HUBS=$(mktemp)
TMP_DOCS=$(mktemp)

HF_REPO="${HF_REPO:-axentx/knowledge-rag}"
HF_FOLDER="${HF_FOLDER:-2026-04-29}"
FORCE="${1:-}"

mkdir -p "$OUT_DIR"

# If not forcing and a recent file exists, keep it (idempotent)
if [ "$FORCE" != "--force" ] && [ -f "$OUT_FILE" ]; then
  AGE_DAYS=$(( ($(date +%s) - $(date -r "$OUT_FILE" +%s)) / 86400 ))
  if [ "$AGE_DAYS" -lt 7 ]; then
    echo "Output fresh (<7d). Skipping. Use --force to regenerate."
    rm -f "$TMP_HUBS" "$TMP_DOCS"
    exit 0
  fi
fi

# 1) Single tree call (non-recursive) for the folder
echo "Listing ${HF_REPO}/${HF_FOLDER} ..."
if ! curl -fsSL "https://huggingface.co/api/datasets/${HF_REPO}/tree?path=${HF_FOLDER}&recursive=false" > "$TMP_HUBS"; then
  echo "WARN: HF tree API failed — using existing or fallback"
  if [ -f "$OUT_FILE" ]; then
    echo "Keeping existing $OUT_FILE"
    rm -f "$TMP_HUBS" "$TMP_DOCS"
    exit 0
  fi
fi

# 2) Pick most-connected hub by heuristic:
#    - prefer files named MOC* or top*.json; else largest .json by size
TOP_HUB_NAME="MOC"
TOP_HUB_FILE=$(jq -r '
  [.[] | select(.type=="file" and (.path | endswith(".json")))]
  | sort_by(-.size)
  | .[0].path // empty
' "$TMP_HUBS" 2>/dev/null || true)

if [ -z "$TOP_HUB_FILE" ]; then
  TOP_HUB_FILE="${HF_FOLDER}/MOC.json"
fi

# 3) Download via CDN (no auth, no API rate limit)
CANONICAL_URL="https://huggingface.co/datasets/${HF_REPO}/resolve/main/${TOP_HUB_FILE}"
echo "Downloading ${CANONICAL_URL} ..."
if ! curl -fsSL "$CANONICAL_URL" > "$TMP_DOCS" 2>/dev/null; then
  echo "WARN: CDN download failed — building minimal payload"
  DOCS_JSON="[]"
else
  DOCS_JSON=$(jq -c '
    if type == "array" then .
    elif type == "object" and (.docs // .items // .entries) then (.docs // .items // .entries)
    else [.] end
    | map({
        title: (.title // .name // .id // "Untitled"),
        url: (.url // .link // .href // ""),
        score: (.score // (.weight // 1) // 1),
        snippet: (.snippet // .summary // .description // "")
      })
    | sort_by(-.score)
    | .[0:10]
  ' "$TMP_DOCS" 2>/dev/null || echo "[]")
fi

# 4) Build payload
UPDATED_AT=$(date -u +"%Y-%m-%dT%H:%M:%SZ")
PAYLOAD=$(jq -n \
  --arg hub "$TOP_HUB_NAME" \
  --arg updated "$UPDATED_AT" \
  --argjson docs "$DOCS_JSON" \
  '{
    hub: $hub,
    updatedAt: $updated,
    source: "knowledge-rag-cdn",
    docs: $docs
  }')

echo "$PAYLOAD" | jq . > "$OUT_FILE"
echo "Wrote $OUT_FILE"

rm -f "$TMP_HUBS" "$TMP_DOCS"
```

Make executable:
```bash
chmod +x scripts/bake-top-hub.sh
```

Cron-safe example (if used):
```cron

```
