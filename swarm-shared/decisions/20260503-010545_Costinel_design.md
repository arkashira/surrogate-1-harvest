# Costinel / design

## Decision
**Highest-value incremental improvement (<2h):**  
Add a lightweight backend orchestration endpoint (`/api/sense/top-hub-signal`) that:

1. Runs `granite-business-research.sh` (via subprocess with timeout).
2. Queries knowledge-RAG for the top-hub (MOC) insight + related docs.
3. Projects a compact, frontend-ready signal payload:
   - `hub`, `insight`, `relatedDocs[]`, `runId`, `timestamp`, `expiresAt`.
4. Returns JSON for the Costinel dashboard “Top-Hub Signal” card.

This keeps the frontend simple (no scraping/parsing), centralizes orchestration, and reuses existing tooling/scripts.

---

## Implementation Plan (≤2h)

### 1) Backend endpoint (FastAPI, Python)
- Path: `/api/sense/top-hub-signal`
- Method: `GET`
- Behavior:
  - Run `granite-business-research.sh` → capture stdout/stderr.
  - Parse/extract top-hub (expect MOC) and insight text (simple regex/JSON fallback).
  - Call knowledge-RAG (local CLI or HTTP) for MOC + related docs.
  - Return `200` with signal payload; on failure return `202` with partial/error info (frontend resilient).
- Timeouts:
  - Script: 60s
  - RAG query: 30s
- Security:
  - Internal-only route (behind existing auth/middleware).
  - No secrets in payload.

### 2) Knowledge-RAG integration
- Prefer CLI invocation (consistent with past patterns):
  - `knowledge-rag query --hub MOC --limit 5 --format json`
- If CLI unavailable, fallback to local HTTP (localhost:8123/query).
- Cache result in memory for 5m to avoid hammering during dashboard refreshes.

### 3) Frontend card (React + TypeScript)
- Component: `TopHubSignalCard`
- Polling: 120s (configurable).
- States: loading / data / partial-error / offline.
- Display:
  - Hub badge (MOC).
  - One-line insight.
  - Collapsible list of related docs (title + snippet + link).
  - Timestamp + refresh indicator.

### 4) Observability
- Log script exit code, duration, and RAG latency.
- Emit metric: `costinel.sense.top_hub_signal.duration`.
- Healthcheck: `/health/sense` that verifies script exists and RAG reachable.

# costinel/backend/api/sense.py
import asyncio
import json
import subprocess
import time
from datetime import datetime, timedelta
from pathlib import Path
from typing import List, Optional

from fastapi import APIRouter, HTTPException
from pydantic import BaseModel

router = APIRouter(prefix="/api/sense", tags=["sense"])

SCRIPT_PATH = Path("/opt/axentx/Costinel/scripts/granite-business-research.sh")
RAG_CLI_PATH = Path("/opt/axentx/Costinel/bin/knowledge-rag")
TOP_HUB = "MOC"
SCRIPT_TIMEOUT = 60
RAG_TIMEOUT = 30
CACHE_TTL = 300  # seconds

_cached_signal: Optional["TopHubSignal"] = None
_cached_at: float = 0.0

class RelatedDoc(BaseModel):
    title: str
    snippet: str
    source: str
    url: Optional[str] = None

class TopHubSignal(BaseModel):
    hub: str
    insight: str
    related_docs: List[RelatedDoc]
    run_id: str
    timestamp: str
    expires_at: str
    partial: bool = False
    error: Optional[str] = None

def _run_script() -> str:
    if not SCRIPT_PATH.is_file():
        raise RuntimeError(f"Script not found: {SCRIPT_PATH}")
    result = subprocess.run(
        ["/bin/bash", str(SCRIPT_PATH)],
        capture_output=True,
        text=True,
        timeout=SCRIPT_TIMEOUT,
        env={**subprocess.os.environ, "SHELL": "/bin/bash"},
    )
    if result.returncode != 0:
        raise RuntimeError(f"Script failed: {result.stderr.strip()}")
    return result.stdout

def _query_rag(hub: str) -> List[RelatedDoc]:
    # Prefer CLI; fallback to HTTP if needed.
    if RAG_CLI_PATH.is_file():
        proc = subprocess.run(
            ["/bin/bash", str(RAG_CLI_PATH), "query", "--hub", hub, "--limit", "5", "--format", "json"],
            capture_output=True,
            text=True,
            timeout=RAG_TIMEOUT,
            env={**subprocess.os.environ, "SHELL": "/bin/bash"},
        )
        if proc.returncode == 0 and proc.stdout.strip():
            try:
                data = json.loads(proc.stdout)
                return [RelatedDoc(**item) for item in data.get("docs", [])]
            except Exception:
                pass
    # Fallback HTTP (example)
    import requests
    resp = requests.get(
        "http://localhost:8123/query",
        params={"hub": hub, "limit": 5},
        timeout=RAG_TIMEOUT,
    )
    resp.raise_for_status()
    payload = resp.json()
    return [RelatedDoc(**item) for item in payload.get("docs", [])]

def _extract_insight(text: str) -> str:
    # Best-effort extraction; adapt to your script output format.
    # If script emits JSON, parse it. Otherwise, first non-empty paragraph.
    text = text.strip()
    if not text:
        return "No insight available."
    try:
        obj = json.loads(text)
        if isinstance(obj, dict):
            return str(obj.get("insight", obj.get("summary", text)))
        if isinstance(obj, list) and obj:
            return str(obj[0])
    except Exception:
        pass
    lines = [ln.strip() for ln in text.splitlines() if ln.strip()]
    return lines[0] if lines else text[:280]

@router.get("/top-hub-signal", response_model=TopHubSignal)
async def top_hub_signal(force_refresh: bool = False):
    global _cached_signal, _cached_at
    now = time.time()
    if not force_refresh and _cached_signal and (now - _cached_at) < CACHE_TTL:
        return _cached_signal

    run_id = f"sig-{int(now)}"
    ts = datetime.utcnow().isoformat() + "Z"
    expires = (datetime.utcnow() + timedelta(seconds=CACHE_TTL)).isoformat() + "Z"
    partial = False
    error = None
    insight = ""
    related_docs: List[RelatedDoc] = []

    try:
        script_out = _run_script()
        insight = _extract_insight(script_out)
    except Exception as exc:
        partial = True
        error = f"Research script error: {exc}"
        # Continue to try RAG for hub context even if script fails.

    try:
        related_docs = _query_rag(TOP_HUB)
        # If no insight yet, try to infer from first doc snippet.
        if not insight and related_docs:
            insight = related_docs[0].snippet
    except Exception as exc:
        partial = True
        error = f"{error}; RAG error: {exc}" if error else f"RAG error: {exc}"

    if not insight and not related_docs:
        raise HTTPException(status_code=503, detail="Unable to generate top-hub signal")

    signal = TopHubSignal(
        hub=TOP_HUB,
        insight=insight or "Insight unavailable",
        related_docs=related_docs,
        run_id=run_id,
        timestamp=ts,
        expires_at=expires,
        partial=partial,
        error=error,
    )
    _cached_signal = signal
    _cached_at = now
    return signal

@router.get("/health/sense")
async def health_sense():
    checks = {
        "script_exists": SCRIPT_PATH.is_file(),
        "rag_cli_exists": RAG_CLI_PATH.is_file(),
    }
    if not checks["script_exists"]:
        raise HTTPException(status_code=503, detail="Missing research script")
    return {"status": "ok", "checks": checks}
