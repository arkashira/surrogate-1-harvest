# workio / ops

### Final Consolidated Implementation (Corrected + Actionable)

**Core principle**: Idempotency must be enforced **server-side** as the source of truth; client-side measures (debounce/optimistic UI) are for UX and to reduce duplicate attempts, but never replace server checks.

---

### 1. Diagnosis (merged)
- No idempotency guard on punch submissions → LINE webhook replays or user double-taps create duplicate clock-in/out records.
- Clock-in/out UI has no optimistic state or debounce → requests can be queued/spammed while previous call is in-flight.
- Missing client-side state transition validation → UI allows invalid sequences (e.g., clock-in while already clocked-in).
- No request deduplication at API layer → backend accepts repeated payloads within short windows.
- Ops burden: duplicate punches require manual cleanup and skew attendance reports.

---

### 2. Proposed change (merged + corrected)
- **Frontend** (`workio/src/components/TimeClock/TimeClockButton.tsx`):
  - Add per-attempt idempotency key (sent via `X-Idempotency-Key`).
  - Add optimistic state + debounce to prevent double-submit in the same session.
  - Add client-side validation for valid state transitions (clock-in only when last punch is clock-out, and vice versa).
- **Backend** (`workio/server/src/routes/punch.ts`):
  - Enforce idempotency via `X-Idempotency-Key` with a 5-minute dedupe window.
  - Store seen keys in a durable store (Redis preferred; fallback to DB table `punch_idempotency`).
  - Use DB uniqueness constraint on `(user_id, type, timestamp)` as a last-line defense.
- **DB**:
  - Add lightweight `punch_idempotency` table (or Redis) for fast idempotency checks.
  - Keep existing `punches` uniqueness constraint as a safety net.

#### Frontend: TimeClockButton.tsx
```tsx
// workio/src/components/TimeClock/TimeClockButton.tsx
import React, { useState, useCallback } from 'react';
import { usePunch } from '../../hooks/usePunch';
import { useLastPunch } from '../../hooks/useLastPunch';

export const TimeClockButton: React.FC = () => {
  const { mutate: punch, isPending } = usePunch();
  const { lastPunch } = useLastPunch();
  const [localPending, setLocalPending] = useState(false);

  const generateIdempotencyKey = useCallback(() => {
    return `${Date.now()}-${crypto.randomUUID?.() || Math.random().toString(36).slice(2, 11)}`;
  }, []);

  const canClockIn = !lastPunch || lastPunch.type === 'out';
  const canClockOut = lastPunch && lastPunch.type === 'in';

  const handlePunch = useCallback(async () => {
    if (isPending || localPending) return;
    if (!canClockIn && !canClockOut) return;

    const key = generateIdempotencyKey();
    setLocalPending(true);

    punch(
      { idempotencyKey: key },
      {
        onSettled: () => {
          setTimeout(() => setLocalPending(false), 1200);
        },
      }
    );
  }, [isPending, localPending, punch, generateIdempotencyKey, canClockIn, canClockOut]);

  const disabled = isPending || localPending || (!canClockIn && !canClockOut);

  return (
    <button
      onClick={handlePunch}
      disabled={disabled}
      className={`px-6 py-3 rounded-lg font-medium transition-all ${
        disabled
          ? 'bg-gray-400 cursor-not-allowed opacity-80'
          : 'bg-blue-600 hover:bg-blue-700 text-white'
      }`}
    >
      {isPending || localPending ? 'กำลังบันทึก...' : (canClockIn ? 'ลงเวลา (Clock-in)' : 'ลงเวลา (Clock-out)')}
    </button>
  );
};
```

// workio/src/hooks/usePunch.ts
import { useMutation } from '@tanstack/react-query';
import axios from 'axios';

export const usePunch = () => {
  return useMutation({
    mutationFn: async ({ idempotencyKey }: { idempotencyKey: string }) => {
      const res = await axios.post(
        '/api/punch',
        {},
        {
          headers: {
            'X-Idempotency-Key': idempotencyKey,
          },
        }
      );
      return res.data;
    },
  });
};

import express from 'express';
import { db } from '../db';
import { redis } from '../redis'; // or use DB fallback

const router = express.Router();

const IDEMPOTENCY_TTL_SECONDS = 300; // 5 minutes

async function isDuplicate(key: string, userId: number): Promise<boolean> {
  const compositeKey = `idempotency:punch:${userId}:${key}`;
  // Prefer Redis
  if (redis) {
    const exists = await redis.get(compositeKey);
    if (exists) return true;
    await redis.setex(compositeKey, IDEMPOTENCY_TTL_SECONDS, '1');
    return false;
  }

  // DB fallback: punch_idempotency table (idempotency_key, user_id, expires_at)
  const now = new Date();
  const existing = await db
    .selectFrom('punch_idempotency')
    .select('idempotency_key')
    .where('idempotency_key', '=', key)
    .where('user_id', '=', userId)
    .where('expires_at', '>', now)
    .executeTakeFirst();

  if (existing) return true;

  await db
    .insertInto('punch_idempotency')
    .values({
      idempotency_key: key,
      user_id: userId,
      expires_at: new Date(now.getTime() + IDEMPOTENCY_TTL_SECONDS * 1000),
    })
    .execute();

  return false;
}
