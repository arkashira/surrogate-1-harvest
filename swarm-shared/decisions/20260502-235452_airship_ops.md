# airship / ops

## Final Implementation Plan — `airship discover`

**Scope**: ≤2h  
**Goal**: Deterministic CLI (`airship discover`) + FastAPI endpoint (`GET /api/discover`) that produces a CDN-only file manifest for a Hugging Face dataset repo + date folder. Enables surrogate/kaggle training pipelines with **zero API calls during training** (HF CDN bypass).

---

### 1. Architecture Decisions (resolved)
- **CLI**: `airship discover --repo <repo_id> --date <YYYY-MM-DD> --out <manifest.json> [--max-depth N]`
- **HTTP**: `GET /api/discover?repo=<repo_id>&date=<YYYY-MM-DD>&max_depth=<int>`
- **Output**: Deterministic JSON list of objects:
  ```json
  { "path": "...", "cdn_url": "...", "size": <int|null>, "sha256": null }
  ```
- **HF Strategy**: Use `list_repo_tree(..., recursive=False)` per folder (not recursive once) to avoid pagination/rate-limit issues while keeping calls minimal. Depth-limited for safety.
- **Determinism**: Sorted by path; stable across runs.
- **Training usage**: Embed the manifest in training scripts; Lightning fetches exclusively via CDN URLs.

---

### 2. Implementation Steps (≤2h)

1. **Add CLI entrypoint** (`airship/cli/discover.py`)
   - Use `argparse` (lightweight, no extra deps).
   - Validate `repo` non-empty, `date` regex `YYYY-MM-DD`.
   - Call `build_manifest()` and write JSON.

2. **Add service module** (`airship/services/discover.py`)
   - `build_manifest(repo_id, date_folder, max_depth=3)`
   - BFS over folders using `list_repo_tree(..., recursive=False)`.
   - Depth computed relative to `date_folder`.
   - Filter to files only; build CDN URL.
   - Sort by path for determinism.
   - Handle missing folders gracefully (skip, don’t crash).

3. **Add FastAPI route** (`airship/api/discover.py`)
   - `GET /api/discover` with query params `repo`, `date`, `max_depth`.
   - Reuse `build_manifest`.
   - Return 200 with list, 400 for bad input, 500 for upstream errors.

4. **Wire into FastAPI app** (`airship/main.py` or `airship/api/__init__.py`)
   - Include router.

5. **Add unit test** (`tests/test_discover.py`)
   - Mock `list_repo_tree` for a small tree.
   - Assert manifest length, ordering, CDN URL format.

6. **Docker**: No changes required if using `COPY .`.

---

### 3. Final Code

#### `airship/services/discover.py`
```python
from __future__ import annotations

from typing import List, Dict, Any
from huggingface_hub import list_repo_tree


def build_manifest(repo_id: str, date_folder: str, max_depth: int = 3) -> List[Dict[str, Any]]:
    """
    Build a deterministic CDN-only manifest for a repo + date folder.
    Uses list_repo_tree(..., recursive=False) per folder to avoid rate limits.
    """
    if not repo_id or not date_folder:
        raise ValueError("repo_id and date_folder are required")

    base_depth = date_folder.strip("/").count("/")
    items = []
    to_walk = [date_folder.rstrip("/")]
    visited = set()

    while to_walk and len(visited) < 10_000:
        current = to_walk.pop(0)
        if current in visited:
            continue
        visited.add(current)

        # Depth guard relative to date_folder
        depth = current.count("/") - base_depth
        if depth >= max_depth:
            continue

        try:
            tree = list_repo_tree(repo_id=repo_id, path=current, recursive=False)
        except Exception:
            # Folder missing or inaccessible: skip silently
            continue

        for node in tree:
            if getattr(node, "type", None) == "directory":
                to_walk.append(node.path)
                continue

            # file node
            path = node.path
            items.append(
                {
                    "path": path,
                    "cdn_url": f"https://huggingface.co/datasets/{repo_id}/resolve/main/{path}",
                    "size": getattr(node, "size", None),
                    "sha256": None,
                }
            )

    # Deterministic ordering
    items.sort(key=lambda x: x["path"])
    return items
```

#### `airship/api/discover.py`
```python
from fastapi import APIRouter, HTTPException, Query
from airship.services.discover import build_manifest

router = APIRouter()


@router.get("/api/discover")
async def discover(
    repo: str = Query(..., min_length=1),
    date: str = Query(..., regex=r"^\d{4}-\d{2}-\d{2}$"),
    max_depth: int = Query(3, ge=1, le=10),
):
    try:
        manifest = build_manifest(repo_id=repo, date_folder=date, max_depth=max_depth)
        return manifest
    except ValueError as exc:
        raise HTTPException(status_code=400, detail=str(exc))
    except Exception as exc:
        raise HTTPException(status_code=500, detail=f"Failed to build manifest: {exc}")
```

#### `airship/cli/discover.py`
```python
#!/usr/bin/env python3
import argparse
import json
import sys
from pathlib import Path

# Ensure local imports work when run from repo root
sys.path.insert(0, str(Path(__file__).parent.parent))

from airship.services.discover import build_manifest


def main() -> None:
    parser = argparse.ArgumentParser(description="Build CDN-only file manifest for HF dataset folder.")
    parser.add_argument("--repo", required=True, help="HF dataset repo id (e.g. 'org/dataset')")
    parser.add_argument("--date", required=True, help="Date folder (YYYY-MM-DD)")
    parser.add_argument("--out", required=True, help="Output JSON file path")
    parser.add_argument("--max-depth", type=int, default=3, help="Max folder depth to traverse")
    args = parser.parse_args()

    try:
        manifest = build_manifest(repo_id=args.repo, date_folder=args.date, max_depth=args.max_depth)
        out_path = Path(args.out)
        out_path.parent.mkdir(parents=True, exist_ok=True)
        out_path.write_text(json.dumps(manifest, indent=2))
        print(f"Wrote {len(manifest)} entries to {out_path}")
    except Exception as exc:
        print(f"Error: {exc}", file=sys.stderr)
        sys.exit(1)


if __name__ == "__main__":
    main()
```

from unittest.mock import patch
from airship.services.discover import build_manifest


def test_build_manifest():
    mock_top = [
        type("Node", (), {"type": "directory", "path": "2026-04-29/sub", "size": None}),
        type("Node", (), {"type": "file", "path": "2026-04-29/a.parquet", "size": 123}),
    ]
    mock_sub = [
        type("Node", (), {"type": "file", "path": "2026-04-29/sub/b.parquet", "size": 456}),
    ]

    with patch("airship.services.discover.list_repo_tree") as m:
        m.side_effect = lambda repo_id, path, recursive: mock_top if path == "2026-04-29" else mock_sub
        manifest = build_manifest("org/dataset", "2026-04-29", max_depth=3)

    assert len(manifest) == 2
    assert manifest[0]["path"] == "2026-04-29/a.parquet"
    assert manifest[1]["path"]
