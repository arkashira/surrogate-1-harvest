# airship / backend

## Implementation Plan — `airship discover` (backend) + `/discover` (frontend)

**Scope**: ≤2h  
**Goal**: Deterministic CLI command `airship discover` that emits a CDN-only file manifest for a given `repo_id` and `date_folder`, plus a lightweight frontend `/discover` to view the latest manifest and copy a CDN-based training snippet.

---

### 1) Backend: CLI command `airship discover`

- **Location**: `airship/cli/discover.py`
- **Behavior**:
  - Accept `repo_id`, `date_folder` (e.g. `2026-04-29`), optional `--out` (default `./manifests/{repo_id}/{date_folder}.json`)
  - Single `list_repo_tree(path=date_folder, recursive=False)` call
  - Build manifest entries:
    - `repo_id`
    - `date_folder`
    - `generated_at` (ISO)
    - `files`: list of `{ "path": <relative>, "cdn_url": "https://huggingface.co/datasets/{repo_id}/resolve/main/{date_folder}/{file}", "size": <from tree if present> }`
  - Save JSON (indent=2) to `out`
  - Print path and a ready-to-use training snippet (CDN-only) to stdout

- **Training snippet** (printed and saved as `training_snippet`):
  ```text
  # CDN-only training (zero HF API calls during data load)
  # File list produced by: airship discover --repo_id <repo> --date_folder <date>
  HF_REPO="datasets/<repo_id>"
  DATE="<date_folder>"
  MANIFEST="./manifests/<repo_id>/<date_folder>.json"
  # Use CDN URLs in your DataLoader (e.g., wget/curl or fsspec)
  # Example: cat $MANIFEST | jq -r '.files[].cdn_url' | xargs -n1 -P8 wget -P ./data/
  ```

- **Error handling**:
  - Validate `repo_id` and `date_folder` format
  - Retry 429 with 360s backoff (per pattern)
  - On failure, exit non-zero and print actionable message

- **Install/entrypoint**:
  - Add `airship` console script pointing to `airship/cli/__main__.py`
  - `python -m airship.cli discover ...` works

---

### 2) Frontend: `/discover` page

- **Location**: `arkship/frontend/pages/discover.tsx` (or `.vue` depending on stack — assume React/Next-like)
- **Features**:
  - Show last generated manifest (auto-detect latest in `manifests/` or accept query param `repo`/`date`)
  - Table: file path, size, CDN URL (copyable)
  - Copy-to-clipboard button for the training snippet
  - Small form to trigger `airship discover` via backend API (optional) or show instructions to run locally
- **API endpoint** (optional): `GET /api/discover/latest?repo_id=...`
  - Reads latest manifest JSON and returns `{ repo_id, date_folder, generated_at, files[], training_snippet }`

- **UI details**:
  - Minimal, fast, no heavy dependencies
  - Use monospace for paths/URLs, ellipsis overflow, tooltips on hover

### 3) Code snippets

#### `airship/cli/discover.py`

```python
#!/usr/bin/env python3
"""
airship discover
Produce a CDN-only file manifest for a repo + date folder.
"""
import argparse
import json
import os
import sys
import time
from datetime import datetime, timezone
from pathlib import Path

# HF API client import (adjust to project's actual client usage)
try:
    from huggingface_hub import HfApi
except ImportError:
    print("ERROR: huggingface_hub not installed.", file=sys.stderr)
    sys.exit(1)

RETRY_WAIT = 360  # per pattern on 429

def build_manifest(repo_id: str, date_folder: str, out_path: Path):
    api = HfApi()
    # Single non-recursive tree call (per pattern)
    for attempt in range(2):
        try:
            tree = api.list_repo_tree(
                repo_id=repo_id,
                path=date_folder,
                repo_type="dataset",
                recursive=False,
            )
            break
        except Exception as e:
            if attempt == 0 and getattr(e, "status", None) == 429:
                print(f"Rate limited 429 — waiting {RETRY_WAIT}s", file=sys.stderr)
                time.sleep(RETRY_WAIT)
                continue
            raise

    files = []
    for entry in tree:
        # entry.path may be full or relative; normalize
        rel_path = os.path.relpath(entry.path, date_folder) if entry.path.startswith(date_folder) else entry.path
        cdn_url = f"https://huggingface.co/datasets/{repo_id}/resolve/main/{date_folder}/{rel_path}"
        files.append({
            "path": rel_path,
            "cdn_url": cdn_url,
            "size": getattr(entry, "size", None),
        })

    manifest = {
        "repo_id": repo_id,
        "date_folder": date_folder,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "files": files,
    }

    out_path.parent.mkdir(parents=True, exist_ok=True)
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(manifest, f, indent=2, ensure_ascii=False)

    training_snippet = (
        f'# CDN-only training (zero HF API calls during data load)\n'
        f'# File list produced by: airship discover --repo_id {repo_id} --date_folder {date_folder}\n'
        f'HF_REPO="datasets/{repo_id}"\n'
        f'DATE="{date_folder}"\n'
        f'MANIFEST="{out_path}"\n'
        f'# Use CDN URLs in your DataLoader (e.g., wget/curl or fsspec)\n'
        f'# Example: cat $MANIFEST | jq -r \'.files[].cdn_url\' | xargs -n1 -P8 wget -P ./data/\n'
    )

    return manifest, training_snippet

def main():
    parser = argparse.ArgumentParser(description="Generate CDN-only manifest for HF dataset folder.")
    parser.add_argument("--repo_id", required=True, help="HF dataset repo id (e.g. 'org/repo')")
    parser.add_argument("--date_folder", required=True, help="Date folder inside dataset (e.g. '2026-04-29')")
    parser.add_argument("--out", default=None, help="Output JSON path (default: ./manifests/{repo_id}/{date_folder}.json)")
    args = parser.parse_args()

    if args.out is None:
        out_path = Path("manifests") / args.repo_id.replace("/", "_") / f"{args.date_folder}.json"
    else:
        out_path = Path(args.out)

    try:
        manifest, snippet = build_manifest(args.repo_id, args.date_folder, out_path)
        print(f"Manifest written to: {out_path}")
        print()
        print(snippet)
    except Exception as e:
        print(f"ERROR: {e}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    main()
```
