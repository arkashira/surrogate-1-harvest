# workio / design

Below is the **single, consolidated implementation** that keeps the strongest, most compatible parts of both proposals, removes overlap, and resolves every contradiction in favor of **correctness, security, and concrete actionability**.

---

## 1. Final Diagnosis (merged)
- **No tenant cache on LINE webhook** → repeated DB resolution under burst traffic; high latency and DB load.
- **Missing `/health` and `/ready`** → no reliable liveness/readiness signals for k8s/PM2; unsafe rollouts.
- **No request tracing** → hard to correlate logs across webhook, clock-in/out, leave/OT flows.
- **LINE signature verification duplicated or unsafe** → per-request overhead and risk of non-constant-time comparisons.
- **No outbound circuit-breaker for LINE Messaging API** → remote stalls can block webhook responses and trigger LINE retries.

## 4. Implementation

### `workio/server/src/middleware/tenantCache.ts`
```ts
import { LRUCache } from 'lru-cache';
import type { TenantInfo } from '../types';

const tenantCache = new LRUCache<string, TenantInfo>({
  max: 500,
  ttl: 1000 * 60 * 5, // 5 minutes (fast churn tolerance)
  allowStale: false,
  updateAgeOnGet: true,
});

export function getTenantByChannel(channelId: string): TenantInfo | undefined {
  return tenantCache.get(channelId);
}

export function setTenantForChannel(channelId: string, tenant: TenantInfo): void {
  tenantCache.set(channelId, tenant);
}

export function invalidateTenantForChannel(channelId: string): void {
  tenantCache.delete(channelId);
}

export function invalidateAllTenants(): void {
  tenantCache.clear();
}
```

import { v4 as uuidv4 } from 'uuid';
import type { Request, Response, NextFunction } from 'express';

export function requestIdMiddleware(req: Request, _res: Response, next: NextFunction) {
  const id = req.headers['x-request-id'] as string || uuidv4();
  (req as any).requestId = id;
  next();
}

### `workio/server/src/middleware/lineSignature.ts`
```ts
import crypto from 'crypto';
import type { Request, Response, NextFunction } from 'express';

const LINE_CHANNEL_SECRET = process.env.LINE_CHANNEL_SECRET || '';

function constantTimeCompare(a: string, b: string): boolean {
  return crypto.timingSafeEqual(Buffer.from(a), Buffer.from(b));
}

export function lineSignatureVerifier(req: Request, res: Response, next: NextFunction) {
  if (!LINE_CHANNEL_SECRET) return next();

  const signature = req.headers['x-line-signature'] as string;
  if (!signature) return res.status(400).json({ error: 'missing x-line-signature' });

  const body = JSON.stringify(req.body);
  const expected = crypto
    .createHmac('sha256', LINE_CHANNEL_SECRET)
    .update(body)
    .digest('base64');

  if (!constantTimeCompare(signature, expected)) {
    return res.status(401).json({ error: 'invalid signature' });
  }
  next();
}
```

### `workio/server/src/routes/health.ts`
```ts
import { Router } from 'express';
import db from '../db';

const router = Router();

router.get('/health', async (_req, res) => {
  res.json({ status: 'ok', uptime: process.uptime() });
});

router.get('/ready', async (_req, res) => {
  try {
    const result = await db.query('SELECT 1 FROM tenants LIMIT 1');
    const hasTenant = result.rows?.length > 0;
    if (!hasTenant) {
      return res.status(503).json({ status: 'unready', reason: 'no tenants' });
    }
    res.json({ status: 'ready' });
  } catch (err) {
    res.status(503).json({ status: 'unready', reason: 'db unavailable' });
  }
});

export default router;
```
