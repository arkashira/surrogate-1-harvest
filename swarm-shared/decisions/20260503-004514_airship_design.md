# airship / design

## Final Synthesized Implementation (Best of Both Candidates)

**Goal**: Eliminate HF API rate-limit failures and `pyarrow.CastError` during Surrogate-1 training by switching to a **manifest-driven, CDN-only loader** with strict schema projection. Fits in **<2h** and enables reliable 24/7 auto-training on Lightning.

---

### 1. Manifest Generator (Single API Call, Run from Mac/CI)

**Location**: `/opt/axentx/airship/scripts/generate_manifest.py`

```python
#!/usr/bin/env python3
"""
Generate training manifest for a date folder.
Run once per folder after rate-limit window clears.
Embeds CDN URLs so Lightning training uses zero HF API calls.
"""
import json
from pathlib import Path
from huggingface_hub import HfApi

API = HfApi()
REPO = "axentx/surrogate-training-data"  # or datasets/your-mirror-repo

def build_manifest(date_folder: str, out_path: Path):
    # Single non-recursive call (avoids pagination + rate limits)
    entries = API.list_repo_tree(REPO, path=date_folder, recursive=False)
    files = sorted([
        e.rfilename for e in entries
        if e.rfilename.endswith(".parquet")
    ])

    if not files:
        raise FileNotFoundError(f"No parquet files in {REPO}/{date_folder}")

    manifest = {
        "date": date_folder,
        "repo": REPO,
        "files": [
            {
                "path": f,
                "cdn_url": f"https://huggingface.co/datasets/{REPO}/resolve/main/{f}"
            }
            for f in files
        ]
    }
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    print(f"✅ Manifest: {len(files)} files → {out_path}")

if __name__ == "__main__":
    import sys
    date_folder = sys.argv[1] if len(sys.argv) > 1 else "2026-05-03"
    out = Path(sys.argv[2]) if len(sys.argv) > 2 else Path(f"manifests/{date_folder}.json")
    build_manifest(date_folder, out)
```

**Usage**:
```bash
python /opt/axentx/airship/scripts/generate_manifest.py 2026-05-03 manifests/2026-05-03.json
```

### 2. CDN-Only Dataset Loader (Schema-Safe, Iterable)

**Location**: `surrogate/training/surrogate1_cdn_loader.py`

```python
import json
from pathlib import Path
from typing import Dict, Any

import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset

HF_CDN = "https://huggingface.co/datasets"

class CDNParquetIterable(IterableDataset):
    """
    Loads parquet shards from HF CDN without any HF API calls during training.
    Projects only {prompt, response} to avoid pyarrow.CastError from mixed schemas.
    """

    def __init__(self, manifest_path: str):
        manifest = json.loads(Path(manifest_path).read_text())
        self.repo = manifest["repo"]
        self.files = [f["path"] for f in manifest["files"]]
        if not self.files:
            raise ValueError("No parquet files in manifest")

    def _cdn_url(self, filepath: str) -> str:
        return f"{HF_CDN}/{self.repo}/resolve/main/{filepath}"

    @staticmethod
    def _project_record(batch: Dict[str, Any]) -> Dict[str, Any]:
        # Keep only prompt/response; drop other columns to avoid schema mismatches
        out = {}
        if "prompt" in batch:
            out["prompt"] = batch["prompt"]
        if "response" in batch:
            out["response"] = batch["response"]
        return out

    def __iter__(self):
        for fname in self.files:
            url = self._cdn_url(fname)
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()

            table = pq.read_table(resp.content)
            for batch in table.to_batches():
                proj = self._project_record(batch.to_pydict())
                n = len(next(iter(proj.values()))) if proj else 0
                for i in range(n):
                    row = {k: v[i] for k, v in proj.items()}
                    # Skip incomplete rows
                    if row.get("prompt") is not None and row.get("response") is not None:
                        yield row
```

### 7. Key Decisions (Resolved Contradictions)

| Candidate 1 | Candidate 2 | Final Choice | Reason |
|---|---|---|---|
| `list_repo_tree` + manual URL build | `HfApi` + embedded CDN URLs | **Candidate 2** | Cleaner, uses official SDK, embeds URLs in manifest for clarity |
| `IterableDataset` with per-row yield | `Dataset` with preload | **Candidate 1** | Streaming avoids OOM on large folders; fits 24/7 training |
| Projection inside `_project_record` | Same | **Unified** | Prevents `pyarrow.CastError` from mixed schemas |
| Manifest in root | Manifest in `manifests/` | **Candidate 2** | Better organization for multiple date folders |
