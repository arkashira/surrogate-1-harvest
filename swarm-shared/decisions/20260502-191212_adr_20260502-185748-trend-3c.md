## Deployment
hybrid (local-first CLI + optional HF Space for hosted demo; no cloud backend required)

## v0.1 scope
v0.1 ships a local CLI (Python) that indexes a repo with tree-sitter, computes per-file AST hashes, runs lightweight local checks (syntax, complexity hotspots via heuristics), produces a terminal report and JSON export, supports incremental re-analysis, and includes a one-command demo via Hugging Face Space (CPU-only, small model) — all OSS dependencies, zero external servers, SQLite-only local storage.

## Open questions
- Which language analyzers (tree-sitter grammars + linters) are must-have for v0.1?
- What compute budget and model size (e.g., 100MB vs 1GB) can we assume on user machines for local embeddings/classification?
- How will incremental/diff analysis be triggered (git diff, file mtime, AST hash) and what is the performance target?
- Will we offer opt-in aggregate telemetry (privacy-first) to guide model/analyzer improvements?
- What is the monetization path (local-first paid binary vs hosted HF Space premium) and how will license/activation work?
