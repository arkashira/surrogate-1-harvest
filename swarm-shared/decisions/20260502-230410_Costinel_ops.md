# Costinel / ops

## Final synthesized implementation (best of both proposals)

**Chosen approach:** Add a single read-only `GET /api/v1/cost-anomaly/signal/top-hub` endpoint that returns the strongest knowledge-graph hub as a cost-anomaly signal (Sense + Signal; no execution). This maximizes re-use of existing knowledge-RAG assets, requires zero infra changes, and can ship in <2 hours.

---

## Why this now (merged rationale)

- **Fast & safe:** Pure read-only API; no migrations, no infra, no side effects.  
- **High ops value:** Gives teams immediate, contextual guidance when investigating cost anomalies.  
- **Reuses existing assets:** Leverages the knowledge-graph and top-hub/MOC pattern already in place.  
- **Costinel fit:** Aligns with Sense + Signal philosophy (detect & signal, don’t execute).

---

## Concrete plan (merged + hardened)

1. **Add FastAPI route**  
   - Path: `GET /api/v1/cost-anomaly/signal/top-hub`  
   - Query: `for_date` (optional, `YYYY-MM-DD`, defaults to today)  
   - Response: 200 JSON signal payload; 404 if no insight for date.

2. **Implement thin service layer**  
   - `knowledge_rag.get_top_hub(for_date)` → returns canonical hub insight.  
   - `knowledge_rag.query_related_docs(hub, for_date, limit)` → returns contextual docs.  
   - Project to minimal, stable signal shape.

3. **Schema-first response model**  
   - Define `HubSignalResponse` / `HubInsight` Pydantic models for validation and OpenAPI.

4. **Wire into ops workflows**  
   - No frontend changes required for this increment; leave as API-only.  
   - Can be consumed by existing dashboards or alerting later.

5. **Tests & docs**  
   - Unit test for service method.  
   - Integration test for endpoint (200 + 404).  
   - Auto-generated OpenAPI schema + route docstring with example curl.

6. **Deployment safety**  
   - Read-only, no side effects.  
   - Roll out behind existing app lifecycle.

Estimated effort: 60–90 minutes.

```python
from pydantic import BaseModel
from typing import List, Optional
from datetime import date

class RelatedDoc(BaseModel):
    slug: str
    title: Optional[str] = None
    summary: Optional[str] = None

class HubInsight(BaseModel):
    hub_id: str
    hub_name: str
    centrality: float
    summary: str
    related_docs: List[RelatedDoc]
    signal_strength: float
    generated_at: str  # ISO-8601

class HubSignalResponse(BaseModel):
    hub: str
    score: float
    summary: str
    related_doc_slugs: List[str]
    ts: str  # ISO-8601 date at midnight UTC
    insight: HubInsight
```

### 2) Service layer

```python
# app/services/knowledge_rag.py
from datetime import date
from typing import Optional, Dict, Any, List

class KnowledgeRagService:
    def __init__(self, graph_store=None):
        # graph_store should be injected or imported from your app's core
        from app.core.graph import graph_store as gs
        self.graph = graph_store or gs

    def get_top_hub(self, for_date: Optional[date] = None) -> Dict[str, Any]:
        """
        Return the strongest hub insight for a given date.
        """
        effective_date = for_date or date.today()
        iso_date = effective_date.isoformat()

        hub_node = self.graph.find_top_hub_by_date(iso_date)
        if not hub_node:
            raise ValueError(f"No top hub found for {iso_date}")

        return {
            "id": hub_node["id"],
            "slug": hub_node.get("slug", hub_node["id"]),
            "name": hub_node.get("name", hub_node.get("slug", hub_node["id"])),
            "centrality": float(hub_node.get("centrality", 0.0)),
            "summary": hub_node.get("summary", ""),
        }

    def query_related_docs(self, hub: str, for_date: Optional[date] = None, limit: int = 5) -> List[Dict[str, str]]:
        """
        Return related docs for a hub node.
        """
        effective_date = for_date or date.today()
        iso_date = effective_date.isoformat()

        docs = self.graph.related_docs_for_hub(hub, date=iso_date, limit=limit)
        return [
            {
                "slug": d.get("slug", d.get("id")),
                "title": d.get("title"),
                "summary": d.get("summary"),
            }
            for d in docs
        ]
```

### 3) API route

```python
# app/api/v1/endpoints/cost_anomaly.py
from fastapi import APIRouter, Depends, Query, HTTPException
from datetime import date, datetime
from typing import Optional
from app.services.knowledge_rag import KnowledgeRagService
from app.schemas.cost_anomaly import HubSignalResponse, HubInsight

router = APIRouter(prefix="/api/v1/cost-anomaly", tags=["cost-anomaly"])

def get_knowledge_rag() -> KnowledgeRagService:
    # lightweight factory; adjust import/path to your actual service
    from app.services.knowledge_rag import KnowledgeRagService
    return KnowledgeRagService()

@router.get("/signal/top-hub", response_model=HubSignalResponse)
def top_hub_signal(
    for_date: Optional[str] = Query(None, regex=r"^\d{4}-\d{2}-\d{2}$"),
    rag: KnowledgeRagService = Depends(get_knowledge_rag),
):
    """
    Sense + Signal: return the strongest hub insight for the given date.
    No execution or mutation — read-only.
    """
    target_date = date.today() if for_date is None else datetime.strptime(for_date, "%Y-%m-%d").date()

    try:
        top_hub = rag.get_top_hub(for_date=target_date)
        related_docs = rag.query_related_docs(hub=top_hub["id"], for_date=target_date, limit=5)
    except ValueError as exc:
        raise HTTPException(status_code=404, detail=str(exc))

    iso_date = target_date.isoformat()
    ts = f"{iso_date}T00:00:00Z"

    insight = HubInsight(
        hub_id=top_hub["id"],
        hub_name=top_hub["name"],
        centrality=top_hub["centrality"],
        summary=top_hub["summary"],
        related_docs=related_docs,
        signal_strength=top_hub["centrality"],
        generated_at=datetime.utcnow().isoformat() + "Z",
    )

    payload = HubSignalResponse(
        hub=top_hub["slug"],
        score=top_hub["centrality"],
        summary=top_hub["summary"],
        related_doc_slugs=[d["slug"] for d in related_docs],
        ts=ts,
        insight=insight,
    )
    return payload
```
