# vanguard / design

## 1. Diagnosis
- No persisted `(repo, dateFolder)` manifest: every training run re-enumerates via authenticated HF API → quota burn + 429 risk.
- Recursive/heavy enumeration exposes mixed-schema files and wastes I/O during data discovery.
- Training script still relies on `load_dataset(streaming=True)` on heterogeneous repos → PyArrow CastError risk.
- No CDN-only data path: training continues to hit `/api/` endpoints during data loading instead of using `resolve/main/` CDN bypass.
- Lightning Studio reuse/idleness not handled: idle stop kills training and quota is wasted on recreation.

## 2. Proposed change
Create `/opt/axentx/vanguard/training/manifest.py` + update `/opt/axentx/vanguard/training/train.py` (or create if absent) to:
- Build and persist a `manifest.json` with `hf_hub_download`-style file URLs for a single `(repo, dateFolder)` using one non-recursive `list_repo_tree` call.
- Embed the file list in training and use CDN-only URLs (`https://huggingface.co/datasets/.../resolve/main/...`) with `datasets` via `load_from_disk`/custom iterable to avoid API calls during training.
- Add lightweight Studio lifecycle guard: reuse running studio, restart only if stopped.

`/opt/axentx/vanguard/training/manifest.py`
```python
import json
import os
from datetime import datetime
from pathlib import Path
from typing import Dict, List

from huggingface_hub import HfApi, list_repo_tree

HF_REPO = os.getenv("HF_DATASET_REPO", "datasets/your-dataset")
DATE_FOLDER = os.getenv("HF_DATE_FOLDER", datetime.utcnow().strftime("%Y-%m-%d"))
MANIFEST_PATH = Path(__file__).parent / "manifest.json"

def build_manifest(repo: str = HF_REPO, date_folder: str = DATE_FOLDER) -> Dict:
    """
    Build manifest for a single date folder (non-recursive).
    Returns dict with repo, date_folder, files: [{local_path, cdn_url, size}], created_at.
    """
    api = HfApi()
    # Non-recursive tree to avoid pagination and mixed-schema deep scans
    tree = list_repo_tree(repo=repo, path=date_folder, recursive=False)

    files = []
    for entry in tree:
        if entry.type != "file":
            continue
        local_path = f"{date_folder}/{entry.path}"
        cdn_url = f"https://huggingface.co/datasets/{repo}/resolve/main/{local_path}"
        files.append({
            "local_path": local_path,
            "cdn_url": cdn_url,
            "size": getattr(entry, "size", None),
        })

    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "created_at": datetime.utcnow().isoformat() + "Z",
        "files": files,
    }

    with open(MANIFEST_PATH, "w") as f:
        json.dump(manifest, f, indent=2)
    return manifest

def load_manifest() -> Dict:
    if not MANIFEST_PATH.exists():
        return build_manifest()
    with open(MANIFEST_PATH) as f:
        return json.load(f)

if __name__ == "__main__":
    m = build_manifest()
    print(f"Built manifest: {len(m['files'])} files -> {MANIFEST_PATH}")
```

# ---- CDN-only dataset iterable ----
class CDNParquetIterable:
    """
    Lightweight iterable over manifest CDN URLs to avoid HF API calls during training.
    Projects each row to {prompt, response} at parse time (handles mixed schemas).
    """
    def __init__(self, manifest_path: str = str(MANIFEST_PATH)):
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.urls = [f["cdn_url"] for f in self.manifest["files"] if f["cdn_url"].endswith(".parquet")]
        self.idx = 0

    def __iter__(self) -> Iterator[dict]:
        import pyarrow.parquet as pq
        import requests
        from io import BytesIO

        for url in self.urls:
            resp = requests.get(url, timeout=60)
            resp.raise_for_status()
            table = pq.read_table(BytesIO(resp.content))
            df = table.to_pandas()
            for _, row in df.iterrows():
                # Project to surrogate-1 fields; ignore extra columns
                prompt = row.get("prompt") or row.get("input") or row.get("text")
                response = row.get("response") or row.get("output") or row.get("completion")
                if prompt is None or response is None:
                    continue
                yield {"prompt": str(prompt), "response": str(response)}

def build_dataset() -> Dataset:
    """
    Build a datasets.Dataset from CDNParquetIterable (in-memory for small shards).
    For large data, keep iterable and use with torch DataLoader directly.
    """
    rows = list(CDNParquetIterable())
    return Dataset.from_list(rows)
