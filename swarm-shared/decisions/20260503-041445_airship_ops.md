# airship / ops

## Final consolidated plan (airship / ops — <2h)

**Goal:** Eliminate HF API 429s during training and prevent idle-timeout training loss by (1) switching to a CDN-only data loader with a pre-listed manifest and (2) adding Lightning Studio lifecycle resilience with auto-restart.

**Estimated effort:** ~90 minutes (including tests and a dry-run).

---

### Implementation plan (merged + corrected)

1. **Pre-list HF files once (or per folder) from Mac/CI**
   - After the rate-limit window clears, run a one-time script to list the target folder (non-recursive) and save to `training_manifest.json`.
   - Commit or generate this manifest per run in CI. Do not rely on runtime HF API calls.

2. **Add CDN-only IterableDataset**
   - Replace any `load_dataset(..., streaming=True)` usage with a lightweight `IterableDataset` that:
     - Reads `training_manifest.json`.
     - Downloads each file via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth).
     - Parses Parquet (or other expected formats) and yields `{prompt, response}`.
     - Uses streaming (`IterableDataset`) to avoid full materialization.
     - Skips corrupt/unreadable files and logs errors (do not crash epoch).

3. **Add Lightning Studio lifecycle resilience**
   - Before training starts, check Studio status.
   - If status is `stopped` (idle-timeout killed), restart with the configured machine (e.g., `Machine.L40S`).
   - If already running, reuse.
   - Wait for `running` state with timeout/fallback; raise if Studio cannot start.

4. **Wire into training entrypoint**
   - Update `train.py` (or surrogate training script) to:
     - Accept `--manifest` path.
     - Use `CDNParquetDataset` (or generic CDN loader).
     - Guard training launch with the Studio lifecycle check.

5. **Validation**
   - Dry-run manifest generation locally.
   - Run a short training step in Lightning Studio and confirm:
     - Zero HF API calls during data load (monitor network or logs).
     - Studio restart works after manual stop.
     - Training continues across Studio restart without losing progress (if checkpointing is used).

#### 1) Manifest generator — `tools/gen_cdn_manifest.py`
```python
#!/usr/bin/env python3
"""
Generate training_manifest.json for a HF dataset folder.
Run once (or per date folder) from Mac/CI after rate-limit clears.
"""
import json
import sys
from huggingface_hub import list_repo_tree

REPO = "axentx/surrogate-training-data"
FOLDER = "batches/mirror-merged/2026-05-03"  # adjust per run
OUT = "training_manifest.json"

def main() -> None:
    entries = list_repo_tree(repo_id=REPO, path=FOLDER, recursive=False)
    files = [e.rfilename for e in entries if e.type == "file"]
    manifest = {
        "repo": REPO,
        "folder": FOLDER,
        "files": files,
        "cdn_prefix": f"https://huggingface.co/datasets/{REPO}/resolve/main"
    }
    with open(OUT, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(files)} files to {OUT}")

if __name__ == "__main__":
    main()
```

#### 2) CDN-only IterableDataset — `surrogate/data/cdn_dataset.py`
```python
#!/usr/bin/env python3
import json
import io
import logging
from typing import Dict, Iterator
import torch
from torch.utils.data import IterableDataset
import requests
import pyarrow.parquet as pq

logger = logging.getLogger(__name__)

class CDNParquetDataset(IterableDataset):
    """
    Stream Parquet files from HF CDN using a pre-generated manifest.
    Manifest format:
    {
      "repo": "...",
      "folder": "...",
      "files": ["path/to/file.parquet", ...],
      "cdn_prefix": "https://huggingface.co/datasets/..."
    }
    """
    def __init__(self, manifest_path: str, columns=("prompt", "response")):
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.prefix = self.manifest.get(cdn_prefix) or f"https://huggingface.co/datasets/{self.manifest['repo']}/resolve/main"
        self.files = self.manifest["files"]
        self.columns = columns

    def _stream_files(self) -> Iterator[str]:
        for f in self.files:
            yield f

    def _download_parquet(self, path: str) -> pq.Table:
        url = f"{self.prefix}/{path}"
        resp = requests.get(url, timeout=60)
        resp.raise_for_status()
        return pq.read_table(io.BytesIO(resp.content), columns=self.columns)

    def __iter__(self) -> Iterator[Dict[str, str]]:
        for rel_path in self._stream_files():
            try:
                table = self._download_parquet(rel_path)
                for row in table.to_pylist():
                    prompt = row.get("prompt") or row.get("text") or ""
                    response = row.get("response") or ""
                    if prompt and response:
                        yield {"prompt": prompt, "response": response}
            except Exception as exc:
                logger.warning("Skipping %s: %s", rel_path, exc)
                continue

# Quick smoke test
if __name__ == "__main__":
    import sys
    dataset = CDNParquetDataset(sys.argv[1])
    loader = torch.utils.data.DataLoader(dataset, batch_size=8, num_workers=0)
```

#### 3) Lightning Studio lifecycle guard — `surrogate/train/lightning_launcher.py`
```python
#!/usr/bin/env python3
import time
import logging
from lightning import Lightning, Teamspace, Studio, Machine

logger = logging.getLogger(__name__)

LIGHTNING_MACHINE = Machine.L40S  # adjust per tier/availability
STUDIO_NAME = "surrogate-training-studio"

def ensure_studio_running() -> Studio:
    ts = Teamspace()
    running = None
    for s in ts.studios:
        if s.name == STUDIO_NAME:
            running = s
            break

    if running is None:
        logger.info("Creating studio %s", STUDIO_NAME)
        running = Studio.create(name=STUDIO_NAME, machine=LIGHTNING_MACHINE)
    else:
        logger.info("Found studio %s (status=%s)", STUDIO_NAME, running.status)

    if running.status != "running":
        logger.info("Starting studio %s on %s", STUDIO_NAME, LIGHTNING_MACHINE)
        running.start(machine=LIGHTNING_MACHINE)
        for _ in range(60):
            running.refresh()
            if running.status == "running":
                break
            time.sleep(10)
        else:
            raise RuntimeError("Studio failed to start")
    return running

def run_training(script_path: str, args: list[str]):
    studio = ensure_studio_running()
    job = studio.run(
        script=script_path,
        arguments=args,
        environment="surrogate-train"
    )
    logger.info("Launched training job: %s", job)
    return job

if __name__ == "__main__":
    run_training("train.py", ["--manifest", "training_manifest.json", "--epochs", "1"])
```
