# workio / ops

## Final Synthesized Implementation (Best of Both Candidates)

I’ve merged the strongest, most actionable parts from both proposals into a single production-grade solution. Contradictions are resolved in favor of **correctness** (proper semantics, graceful degradation, safe shutdown) and **concrete actionability** (exact files, commands, and verification steps).

---

### 1. Diagnosis (Consolidated)
- No structured observability (JSON logs, correlation IDs, latency/error context).
- No idempotency for LINE webhooks (duplicate events → double clock-in/out).
- No graceful degradation (failures in DB, LINE API, or geocoding stall the queue).
- No circuit-breaker for downstream calls.
- No process supervision or graceful shutdown (in-flight requests lost on deploy/restart).
- No orchestration-ready health/readiness endpoints.

#### b) Idempotency + circuit-breaker middleware (`src/middleware/webhookGuard.ts`)
```ts
import { Request, Response, NextFunction } from 'express';
import { logger } from '../lib/logger';
import CircuitBreaker from 'opossum';

// In production, replace with Redis (ioredis/redis) for multi-instance safety.
const seen = new Map<string, { ts: number }>();
const WINDOW_MS = 1000 * 60 * 5; // 5 minutes

function hash(s: string): string {
  let h = 0;
  for (let i = 0; i < s.length; i++) {
    h = ((h << 5) - h + s.charCodeAt(i)) | 0;
  }
  return Math.abs(h).toString(36);
}

export function idempotencyKey(req: Request, res: Response, next: NextFunction) {
  // Prefer X-Line-Signature + raw body when available; fallback to body hash.
  const rawBody = (req as any).rawBody;
  const body = rawBody ? rawBody.toString('utf8') : JSON.stringify(req.body || {});
  const sig = req.header('X-Line-Signature') || 'no-sig';
  const key = `line:${sig}:${hash(body)}`;
  (req as any).idempotencyKey = key;

  const existing = seen.get(key);
  if (existing) {
    logger.warn({ key, msg: 'duplicate_line_event' }, 'duplicate webhook rejected');
    // LINE requires 2xx to stop retries; idempotent success.
    return res.status(200).json({ ok: true, reason: 'duplicate' });
  }
  seen.set(key, { ts: Date.now() });
  setTimeout(() => seen.delete(key), WINDOW_MS);
  next();
}

// Circuit breaker for downstream calls (DB, LINE API, geocoding)
export const breaker = new CircuitBreaker(
  async (fn: () => Promise<any>) => fn(),
  {
    timeout: 8000,
    errorThresholdPercentage: 50,
    resetTimeout: 30000,
    rollingCountTimeout: 60000,
    rollingCountBuckets: 10,
  }
);

breaker.on('open', () => logger.warn('breaker_open', 'circuit breaker opened'));
breaker.on('halfOpen', () => logger.info('breaker_half_open'));
breaker.on('close', () => logger.info('breaker_closed'));
```

#### d) LINE webhook route (`src/routes/webhook/line.ts`)
```ts
import { Router, Request, Response } from 'express';
import { logger } from '../../lib/logger';
import { idempotencyKey, breaker } from '../../middleware/webhookGuard';
import { handleLineEvent } from '../../services/line/handler';

const router = Router();

router.post('/line', idempotencyKey, async (req: Request, res: Response) => {
  const start = Date.now();
  const key = (req as any).idempotencyKey || 'unknown';
  const meta = { key, ip: req.ip, ua: req.get('User-Agent') };

  try {
    await breaker.fire(() => handleLineEvent(req.body));
    logger.info({ ...meta, ms: Date.now() - start }, 'line_webhook_ok');
    res.status(200).json({ ok: true });
  } catch (err: any) {
    logger.error({ ...meta, ms: Date.now() - start, err: err?.message, stack: err?.stack }, 'line_webhook_failed');
    // Always 2xx to prevent LINE retry storms; rely on idempotency + DLQ/alerting for true failures.
    res.status(200).json({ ok: false, reason: 'handled' });
  }
});

export { router as lineWebhookRouter };
```
