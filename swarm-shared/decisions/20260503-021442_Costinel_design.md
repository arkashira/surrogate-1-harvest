# Costinel / design

## Implementation Plan — Top-Hub Signal Panel (Costinel Dashboard)

**Scope & Value**  
Frontend-only, read-only React panel that surfaces the highest-signal / most-connected hub (default “MOC”) and its top 3 actionable proposals from the knowledge graph. CDN-first data path (no HF API calls at runtime). Ships in <2h.

---

### 1) Architecture (CDN-first, zero-runtime API)
- Data source: `knowledge-rag` produces `top-hub.json` (or `moc-hub.json`) into `public/data/knowledge/` at build/CI time.
- Runtime: React fetches `/data/knowledge/top-hub.json` (CDN) → renders panel.
- No HF API, no auth, no rate-limit risk during dashboard use.
- Fallback: if fetch fails or file missing, render empty state with “No signals” (graceful).

### 2) File changes
- `src/components/dashboard/TopHubSignalPanel.tsx` — new component.
- `src/components/dashboard/index.ts` — export.
- `public/data/knowledge/top-hub.json` — sample data (committed; CI can overwrite).
- `src/pages/Dashboard.tsx` (or wherever main dashboard lives) — mount panel in high-visibility zone.

### 3) Component behavior
- Accept optional `hubId` prop (default `"MOC"`).
- Fetch `/data/knowledge/{hubId}-hub.json` (fallback to `top-hub.json`).
- Show:
  - Hub title + short description
  - Top 3 proposals (title, summary, impact score, tags)
  - “View in knowledge graph” link (if URL provided)
- Skeleton while loading; inline error if fetch fails.

`src/components/dashboard/TopHubSignalPanel.tsx`
```tsx
import React, { useEffect, useState } from "react";
import "./TopHubSignalPanel.css";

interface Proposal {
  id: string;
  title: string;
  summary: string;
  impactScore: number;
  tags: string[];
}

interface HubData {
  hubId: string;
  title: string;
  description: string;
  proposals: Proposal[];
  graphUrl?: string;
}

interface TopHubSignalPanelProps {
  hubId?: string;
  className?: string;
}

const TopHubSignalPanel: React.FC<TopHubSignalPanelProps> = ({
  hubId = "MOC",
  className = "",
}) => {
  const [data, setData] = useState<HubData | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    const fetchData = async () => {
      setLoading(true);
      setError(null);
      const candidates = [
        `/data/knowledge/${hubId}-hub.json`,
        `/data/knowledge/top-hub.json`,
      ];

      for (const url of candidates) {
        try {
          const res = await fetch(url, { cache: "no-store" });
          if (!res.ok) continue;
          const json = (await res.json()) as HubData;
          setData(json);
          setLoading(false);
          return;
        } catch {
          // try next
        }
      }

      setLoading(false);
      setError("No hub data available.");
    };

    fetchData();
  }, [hubId]);

  if (loading) {
    return (
      <div className={`top-hub-panel loading ${className}`}>
        <div className="shimmer title"></div>
        <div className="shimmer desc"></div>
        <div className="proposals">
          {[1, 2, 3].map((i) => (
            <div key={i} className="proposal-card shimmer-card"></div>
          ))}
        </div>
      </div>
    );
  }

  if (error || !data) {
    return (
      <div className={`top-hub-panel empty ${className}`}>
        <p>No signals available for {hubId}.</p>
      </div>
    );
  }

  const topProposals = data.proposals.slice(0, 3);

  return (
    <div className={`top-hub-panel ${className}`}>
      <header className="panel-header">
        <h3 className="hub-title">{data.title}</h3>
        <p className="hub-desc">{data.description}</p>
        {data.graphUrl && (
          <a
            href={data.graphUrl}
            target="_blank"
            rel="noopener noreferrer"
            className="graph-link"
          >
            View in knowledge graph →
          </a>
        )}
      </header>

      <div className="proposals">
        {topProposals.map((p) => (
          <article key={p.id} className="proposal-card">
            <div className="proposal-header">
              <span className="proposal-id">{p.id}</span>
              <span className="impact-badge" title="Impact score">
                {p.impactScore}
              </span>
            </div>
            <h4 className="proposal-title">{p.title}</h4>
            <p className="proposal-summary">{p.summary}</p>
            <div className="proposal-tags">
              {p.tags.map((t) => (
                <span key={t} className="tag">
                  {t}
                </span>
              ))}
            </div>
          </article>
        ))}
      </div>
    </div>
  );
};

export default TopHubSignalPanel;
```
