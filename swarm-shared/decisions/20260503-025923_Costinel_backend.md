# Costinel / backend

## Final Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

### What we ship
- A **non-blocking Top-Hub Signal Panel** mounted on the Costinel dashboard (sidebar or top banner area).
- Defaults to hub **MOC** (configurable via `VITE_HUB_NAME`).
- Shows: hub title, short description, top 3 signals (anomaly/recommendation/opportunity), and last-updated timestamp.
- **CDN-first data strategy**: file list generated once (Mac orchestration) → committed to repo; runtime fetches only via CDN (`/data/...`) with **zero runtime API calls**.
- Graceful fallbacks: if CDN fails → local static fallback; if panel fails → silent no-op (non-blocking, no UI crash).

### Why this is highest-value (<2h)
- Applies the **top-hub insight pattern** (MOC) and **HF CDN bypass** pattern directly to Costinel’s visibility layer.
- Delivers immediate contextual value to users without touching billing/control-plane logic or backend services.
- Minimal surface area, reuses existing frontend stack, and safe to ship.

#### 4) Add panel component (React)
`/src/components/TopHubSignalPanel.tsx`
```tsx
import React, { useEffect, useState } from 'react';
import './TopHubSignalPanel.css';

const HUB = import.meta.env.VITE_HUB_NAME || 'MOC';
const CDN_BASE = import.meta.env.VITE_HUB_CDN_BASE || '/data';
const ENABLED = import.meta.env.VITE_HUB_ENABLED !== 'false';

type Signal = {
  id: string;
  severity: 'high' | 'medium' | 'low';
  type: 'anomaly' | 'recommendation' | 'opportunity';
  title: string;
  description: string;
  action: string;
};

type HubPayload = {
  hub: string;
  title: string;
  description: string;
  generatedAt?: string;
  signals: Signal[];
};

const TopHubSignalPanel: React.FC = () => {
  const [payload, setPayload] = useState<HubPayload | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    if (!ENABLED) {
      setLoading(false);
      return;
    }

    const load = async () => {
      try {
        // Step 1: fetch file list (light)
        const listRes = await fetch(`${CDN_BASE}/hub-${HUB.toLowerCase()}-files.json`, { cache: 'no-store' });
        if (!listRes.ok) throw new Error('No file list');
        const list = await listRes.json();

        // Prefer signals file; fallback to first json
        const signalsFile = (list.files || []).find((f: string) => f.includes('signal')) || list.files?.[0];
        if (!signalsFile) throw new Error('No signals file');

        // Step 2: fetch signals via CDN (zero API calls)
        const signalRes = await fetch(`${CDN_BASE}/${signalsFile}`, { cache: 'no-store' });
        if (!signalRes.ok) throw new Error('No signals');
        const data = await signalRes.json();
        setPayload(data);
      } catch {
        // Silent fallback: try bundled static signals
        try {
          const fallback = await fetch(`${CDN_BASE}/hub-${HUB.toLowerCase()}/signals-2026-05-03.json`, { cache: 'no-store' });
          if (fallback.ok) setPayload(await fallback.json());
        } catch {
          // Silent no-op on final failure
        }
      } finally {
        setLoading(false);
      }
    };

    load();
  }, [HUB, CDN_BASE, ENABLED]);

  if (!ENABLED || !payload) return null;
  if (loading && !payload) return null;

  return (
    <div className="top-hub-panel" role="region" aria-label={`${payload.hub} signals`}>
      <div className="top-hub-header">
        <div>
          <strong className="top-hub-title">{payload.title || payload.hub}</strong>
          <p className="top-hub-desc">{payload.description}</p>
        </div>
        <div className="top-hub-meta">
          {payload.generatedAt && (
            <span className="top-hub-updated" title={`Generated at ${payload.generatedAt}`}>
              Updated {new Date(payload.generatedAt).toLocaleDateString()}
            </span>
          )}
          <a className="top-hub-more" href={`/hub/${payload.hub}`}>View details →</a>
        </div>
      </div>
      <ul className="top-hub-signals" aria-live="polite">
        {(payload.signals || []).slice(0, 3).map((s) => (
          <li key={s.id} className={`top-hub-signal sev-${s.severity}`}>
            <div className="top-hub-signal-badge">{s.type}</div>
            <div className="top-hub-signal-body">
              <div className="top-hub-signal-title">{s.title}</div>
              <div className="top-hub-signal-desc">{s.description}</div>
            </div>
            <a className="top-hub-signal-action" href={s.action}>Open</a>
          </li>
        ))}
      </ul>
    </div>
  );
};

export default TopHubSignalPanel;
```
