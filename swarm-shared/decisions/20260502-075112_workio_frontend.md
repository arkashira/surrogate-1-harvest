#### Step 2: Create a Permission Checker Function
Create a permission checker function that takes a user's role and a permission as input and returns a boolean indicating whether the user has the permission.

```typescript
// permissions.ts
import { roles } from './roles';
export function hasPermission(userRole: string, permission: string): boolean {
  const userPermissions = roles[userRole].permissions;
  return userPermissions.includes(permission);
}
```

Implement role-based UI customization to display only the relevant features and menus for each role. Use the permission checker function to conditionally render components.
