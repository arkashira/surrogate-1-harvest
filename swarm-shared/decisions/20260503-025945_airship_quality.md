## Highest-Value Incremental Improvement (<2h)
**Goal**: Eliminate HF API 429s during Surrogate training data loading and make Lightning Studio resilient to idle-stop, cutting Mac→Lightning iteration to <2 minutes with zero breaking changes.

**Why this ships fastest with highest quality impact**
- Removes the most common runtime failure (HF 429) by switching to CDN-only fetches during training.
- Prevents wasted quota and silent failures from Lightning idle-stop.
- Requires only small, targeted changes to data loading and the launcher; no schema or infra changes.

## Implementation Plan

### 1) Pre-list file paths once on Mac (after rate-limit window)
- Run a one-off script (or add to CI) that calls `list_repo_tree(path, recursive=False)` per date folder.
- Save the list to `training/filelist-{date}.json` (list of `{"repo": "...", "path": "..."}`).
- Embed this file in the training container so Lightning never calls HF API for file listing.

**Script** (`scripts/build_filelist.py`)
```python
#!/usr/bin/env python3
"""
Generate filelist JSON for CDN-only training.
Run from Mac after rate-limit window clears.
"""
import json
import os
from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_DATASET_REPO", "datasets/airship-mirror")
OUT_PATH = os.getenv("OUT_PATH", "training/filelist-latest.json")

api = HfApi()
entries = []

# Top-level folders (e.g. by date)
tree = api.list_repo_tree(HF_REPO, recursive=False)
for folder in tree:
    if not folder.path.endswith("/"):
        continue
    # List files in this date folder only (non-recursive)
    files = api.list_repo_tree(HF_REPO, path=folder.path, recursive=False)
    for f in files:
        if f.path.endswith("/"):
            continue
        entries.append({"repo": HF_REPO, "path": f.path})

os.makedirs(os.path.dirname(OUT_PATH), exist_ok=True)
with open(OUT_PATH, "w") as f:
    json.dump(entries, f, indent=2)

print(f"Wrote {len(entries)} files to {OUT_PATH}")
```

### 2) Lightning training: CDN-only data loader
- Replace `load_dataset(streaming=True)` with a lightweight `IterableDataset` that downloads via CDN URLs.
- No Authorization header; uses `https://huggingface.co/datasets/{repo}/resolve/main/{path}`.
- Project to `{prompt, response}` only at parse time.

**Training snippet** (`surrogate/training/data.py`)
```python
import json
import io
import pyarrow as pa
import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset

HF_CDN = "https://huggingface.co/datasets"

class CDNParquetIterable(IterableDataset):
    def __init__(self, filelist_path, repo, transform=None):
        with open(filelist_path) as f:
            self.files = json.load(f)
        self.repo = repo
        self.transform = transform

    def __iter__(self):
        for item in self.files:
            url = f"{HF_CDN}/{item['repo']}/resolve/main/{item['path']}"
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            buf = io.BytesIO(resp.content)
            table = pq.read_table(buf)
            # Project to minimal schema at parse time
            for batch in table.to_batches(columns=["prompt", "response"]):
                for row in zip(batch[0].to_pylist(), batch[1].to_pylist()):
                    out = {"prompt": row[0], "response": row[1]}
                    if self.transform:
                        out = self.transform(out)
                    yield out
```

### 3) Lightning launcher: reuse studio + handle idle-stop
- Before `.run()`, check studio status; if stopped, restart with target machine.
- Reuse running studios to save quota.

**Launcher snippet** (`surrogate/training/launch.py`)
```python
from lightning_sdk import Studio, Machine
import time

def get_or_create_studio(name: str, machine: Machine = Machine.L40S):
    studios = list(Studio.list())
    running = [s for s in studios if s.name == name and s.status == "running"]
    if running:
        print(f"Reusing running studio: {name}")
        return running[0]

    stopped = [s for s in studios if s.name == name and s.status == "stopped"]
    if stopped:
        print(f"Restarting stopped studio: {name}")
        studio = stopped[0]
        studio.start(machine=machine)
        return studio

    print(f"Creating studio: {name}")
    return Studio.create(name=name, machine=machine, create_ok=True)

def run_training_job(studio_name: str, script: str, args: list[str]):
    studio = get_or_create_studio(studio_name)
    # Ensure studio is running before submit
    if studio.status != "running":
        studio.start(machine=Machine.L40S)
        # Wait until running
        for _ in range(30):
            studio.refresh()
            if studio.status == "running":
                break
            time.sleep(10)

    run = studio.run(script, arguments=args)
    print(f"Submitted run: {run.id}")
    return run
```
