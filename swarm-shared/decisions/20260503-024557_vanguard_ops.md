# vanguard / ops

## Final synthesized implementation (correct + actionable)

**Core principle:** pre-list once at build time, embed the manifest, serve everything from CDN/local static files, and never call HF APIs from the browser. Provide a fast local dev loop and a CI-friendly build step.
