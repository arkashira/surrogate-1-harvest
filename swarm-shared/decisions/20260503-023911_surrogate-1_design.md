# surrogate-1 / design

### Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16`, `DATE`, `HF_TOKEN`
- Uses **one** `list_repo_tree` call (per date folder) to enumerate candidate files, then deterministically assigns each file to a shard via `hash(slug) % SHARD_TOTAL`
- Downloads assigned files via **HF CDN bypass** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header (avoids 429 on `/api/`)
- Projects heterogeneous schemas to `{prompt, response}` only at parse time (avoids PyArrow CastError)
- Deduplicates via central `lib/dedup.py` md5 store (same as Space)
- Writes normalized pairs to `batches/public-merged/{DATE}/shard{N}-{HHMMSS}.jsonl`
- Returns exit code 0 on success; logs counts and skipped duplicates

---

### Why this is the highest-value incremental improvement
- Fixes the known `load_dataset(streaming=True)` mixed-schema CastError pattern
- Eliminates HF API rate-limit risk during data load by using CDN-only fetches
- Keeps parallel 16-shard architecture intact while making each worker robust and observable
- Reuses existing dedup logic and filename convention — no downstream changes required

def slug_for_path(path: str) -> str:
    """Stable slug for dedup/sharding (no extension)."""
    return path.rsplit(".", 1)[0].strip("/")

def file_hash(content: bytes) -> str:
    return hashlib.md5(content).hexdigest()

def project_to_pair(raw: Dict[str, Any], path: str) -> Dict[str, str] | None:
    """
    Project heterogeneous file to {prompt, response}.
    Returns None if projection fails or required fields missing.
    """
    prompt = None
    response = None

    # Common patterns
    if "prompt" in raw and "response" in raw:
        prompt, response = raw["prompt"], raw["response"]
    elif "instruction" in raw and "output" in raw:
        prompt, response = raw["instruction"], raw["output"]
    elif "input" in raw and "output" in raw:
        prompt, response = raw["input"], raw["output"]
    elif "text" in raw:
        parts = str(raw["text"]).split("\n\n", 1)
        if len(parts) == 2:
            prompt, response = parts[0], parts[1]
        else:
            prompt = ""
            response = str(raw["text"])
    else:
        str_fields = [v for v in raw.values() if isinstance(v, str)]
        if len(str_fields) >= 2:
            prompt, response = str_fields[0], str_fields[1]
        elif len(str_fields) == 1:
            prompt = ""
            response = str_fields[0]

    if not isinstance(prompt, str) or not isinstance(response, str):
        log.warning("Non-string fields in %s: %s", path, raw)
        return None

    prompt = prompt.strip()
    response = response.strip()
    if not prompt or not response:
        log.warning("Empty prompt/response in %s", path)
        return None

    return {"prompt": prompt, "response": response}
