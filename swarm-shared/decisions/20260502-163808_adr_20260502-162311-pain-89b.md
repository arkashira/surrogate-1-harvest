## Deployment

## v0.1 scope
v0.1 in 2 weeks: Android CLI/background agent (no root, no VPNService) that establishes a hidden tunnel via tun2socks + QUIC, cloaks local SOCKS5/TUN indicators, fetches signed config bundles from Supabase via Cloudflare Workers edge cache, and reports anonymized session counts (WAUS/U) while staying within free-tier limits.

## Open questions
- Which port-reuse and socket-hiding techniques reliably evade app-level VPN detection on stock Android 12+ without root or Knox exemptions?
- Can we sustain low-latency TCP and UDP via tun2socks + QUIC without triggering VPNService flags or battery optimizations that kill background I/O?
- What cover-traffic profile (domain fronting ratios, traffic morphing, packet timing) minimizes fingerprintability while preserving usable throughput on metered networks?
- How will config rotation and revocation be delivered via Workers KV/Queues to avoid single-point blocking of config endpoints?
- What telemetry and privacy-preserving WAUS/U metric can we compute client-side and aggregate via edge logs without storing PII?
