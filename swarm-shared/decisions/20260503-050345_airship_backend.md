### 1. Add manifest generation script (Mac orchestration only)
- Single API call to `list_repo_tree` for a date folder under `batches/mirror-merged/`
- Emit `manifest-{date}.json` containing `{path, size, sha256?, url}` for CDN downloads
- Save to repo root (or `training/manifests/`) and commit

### 2. Update training script (`train.py` or data loader) to use CDN-only
- Load manifest JSON (local file) instead of `load_dataset` or `list_repo_files`
- Use `IterableDataset` that streams via `requests.get(CDN_URL, stream=True)` + `pyarrow.parquet.ParquetFile`
- Project only `{prompt, response}` at parse time; drop all other columns
- Zero HF API calls during training

### 3. Lightning Studio integration
- Reuse running studio if present (`Teamspace.studios` check)
- Pass manifest path as script arg or embed in repo
- On idle stop, restart with `target.start(machine=Machine.L40S)` before next `.run()`

### 4. Fallback / safety
- If manifest missing, fail fast with clear message (do not fall back to API list calls)
- Validate CDN URLs with HEAD (optional) before training start to catch 404s early

## Code Snippets

### manifest_generator.py (run on Mac)
```python
#!/usr/bin/env python3
"""
Generate CDN-only manifest for Surrogate-1 training.
Run from Mac after HF API rate-limit window clears.
"""
import json
import os
import sys
from datetime import datetime, timezone
from huggingface_hub import HfApi

HF_REPO = "datasets/axentx/surrogate-1-mirror"
DATE_FOLDER = datetime.now(timezone.utc).strftime("%Y-%m-%d")  # or pass as arg
OUTPUT_FILE = f"training/manifests/manifest-{DATE_FOLDER}.json"

def main():
    api = HfApi()
    prefix = f"batches/mirror-merged/{DATE_FOLDER}/"
    print(f"Listing {HF_REPO}/{prefix} ...")
    items = api.list_repo_tree(
        repo_id=HF_REPO,
        path=prefix,
        repo_type="dataset",
        recursive=False,
    )
    files = [it for it in items if it.type == "file" and it.path.endswith(".parquet")]
    if not files:
        print("No parquet files found.")
        sys.exit(1)

    manifest = []
    for f in files:
        cdn_url = f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{f.path}"
        manifest.append({
            "path": f.path,
            "size": f.size,
            "cdn_url": cdn_url,
            "date_folder": DATE_FOLDER,
        })

    os.makedirs(os.path.dirname(OUTPUT_FILE), exist_ok=True)
    with open(OUTPUT_FILE, "w") as fp:
        json.dump(manifest, fp, indent=2)
    print(f"Wrote {len(manifest)} entries to {OUTPUT_FILE}")

if __name__ == "__main__":
    main()
```

### cdn_parquet_dataset.py (Lightning training)
```python
import json
import io
import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path, columns=("prompt", "response")):
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.columns = columns

    def __iter__(self):
        for item in self.manifest:
            url = item["cdn_url"]
            resp = requests.get(url, stream=True, timeout=30)
            resp.raise_for_status()
            data = io.BytesIO(resp.content)
            pf = pq.ParquetFile(data)
            for batch in pf.iter_batches(batch_size=1024, columns=self.columns):
                df = batch.to_pandas()
                for _, row in df.iterrows():
                    yield {k: row[k] for k in self.columns}
```

### Lightning reuse + idle restart guard (launcher snippet)
```python
from lightning_sdk import Teamspace, Studio, Machine

team = Teamspace(name="axentx")
studio = None
for s in team.studios():
    if s.name == "surrogate-1-train" and s.status == "Running":
        studio = s
        break

if studio is None:
    studio = team.studios.create(
        name="surrogate-1-train",
        machine=Machine.L40S,
        script="train.py",
        create_ok=True,
    )

# Before each run, ensure studio is running
if studio.status != "Running":
    studio.start(machine=Machine.L40S)

studio.run(
    run_name="surrogate-cdn-20260503",
    config={"manifest": "training/manifests/manifest-2026-05-03.json"},
)
```
