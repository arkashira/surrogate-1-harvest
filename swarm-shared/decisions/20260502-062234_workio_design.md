**High-Value Incremental Improvement for Workio Frontend and Backend**
### Implementation Plan

**Task:** Implement HF CDN Bypass for Training Pipeline and Surrogate-1 Training Pipeline

**Time Estimate:** 1 hour 30 minutes

**Implementation Steps:**

1. **HF CDN Bypass:**
	* Create a new file `cdn-bypass.js` in `workio/server/utils` directory.
	* In `cdn-bypass.js`, implement the HF CDN Bypass logic using the `axios` library to download files from the HF CDN without authentication.
	* Use the `list_repo_tree` API to get the list of files in a repository and then download each file individually using the `resolve/main` endpoint.
	* Save the list of files to a JSON file for future use.
2. **Surrogate-1 Training Pipeline:**
	* Update the `train.py` script to use the `cdn-bypass.js` utility to download files from the HF CDN.
	* Modify the script to project the downloaded files to the `{prompt, response}` schema only at parse time.
	* Use the `hf_hub_download` function to download each file individually and then project it to the desired schema.

**Implementation Code:**

```javascript
// cdn-bypass.js
const axios = require('axios');

const listRepoTree = async (repo, path) => {
  const response = await axios.get(`https://huggingface.co/api/v1/repos/${repo}/tree/${path}`);
  return response.data;
};

const downloadFile = async (repo, path) => {
  const response = await axios.get(`https://huggingface.co/datasets/${repo}/resolve/main/${path}`);
  return response.data;
};

module.exports = { listRepoTree, downloadFile };
```

* `cdn-bypass.js` function: `listRepoTree(repo, path) -> Promise<object>` and `downloadFile(repo, path) -> Promise<string>`
	+ `repo`: string, repository name
	+ `path`: string, file path
	+ Returns: Promise<object> or Promise<string>, list of files or downloaded file contents

**API Endpoint:**

* `/cdn-bypass`: GET, bypass HF CDN for dataset file download

**Notes:**

* This implementation assumes that the `cdn-bypass.js` function is used in conjunction with the `train.py` script.
* The `cdn-bypass.js` function can be used in other scripts or APIs that require HF CDN bypass.
