## Final Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **Add `train_manifest.json` generation** (single API call from Mac orchestrator)  
   - `bin/build_manifest.py`: `list_repo_tree(path, recursive=False)` for today’s folder → write `train_manifest.json` with CDN URLs and expected slugs.  
   - Embed this manifest in the Lightning training repo; data loader uses **only CDN downloads** (`/resolve/main/...`) with zero API calls during training.

2. **Replace `dataset-enrich.sh` core with `worker.py`**  
   - Deterministic shard assignment via `hash(slug) % 16`.  
   - Stream-normalize per file (no `load_dataset(streaming=True)` on mixed schemas).  
   - Project to `{prompt, response}` only at parse time; drop extra cols.  
   - Central dedup via `lib/dedup.py` (SQLite) keyed by md5 hash.

3. **Update GitHub Actions matrix**  
   - Pass `SHARD_ID` and `MANIFEST_URL` (or embed manifest in repo).  
   - Each runner calls `python worker.py --shard $SHARD_ID`.

4. **Lightning training script update**  
   - Read `train_manifest.json`.  
   - Use `hf_hub_download` per file (or direct CDN `requests` with streaming) → yield `{prompt, response}`.  
   - No `datasets.load_dataset` on the full repo during training.

5. **Ops hygiene**  
   - Ensure `SHELL=/bin/bash` in any crontab that invokes wrappers.  
   - Add shebang `#!/usr/bin/env bash` and `chmod +x` to any new wrapper scripts.  
   - Reuse running Lightning Studio instead of create; restart if idle-stopped.

### 1. `bin/build_manifest.py` (run from Mac orchestrator)

```python
#!/usr/bin/env python3
"""
Generate train_manifest.json for today's folder.
Run from Mac (or CI) after rate-limit window clears.
"""
import json
import os
from datetime import datetime, timezone
from huggingface_hub import HfApi

HF_REPO = "datasets/axentx/surrogate-1-training-pairs"
OUT_FILE = "train_manifest.json"

def main():
    api = HfApi(token=os.getenv("HF_TOKEN"))
    today = datetime.now(timezone.utc).strftime("%Y-%m-%d")
    # Single API call: non-recursive per folder
    entries = api.list_repo_tree(
        repo_id=HF_REPO,
        path=today,
        repo_type="dataset",
        recursive=False,
    )
    files = [e.rfilename for e in entries if e.type == "file"]
    manifest = {
        "date": today,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "files": [
            {
                "filename": f,
                "cdn_url": f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{f}",
                "hf_path": f,
            }
            for f in sorted(files)
        ],
    }
    with open(OUT_FILE, "w") as fp:
        json.dump(manifest, fp, indent=2)
    print(f"Wrote {OUT_FILE} with {len(files)} files")

if __name__ == "__main__":
    main()
```

def slug_hash(slug: str) -> int:
    return int(hashlib.md5(slug.encode()).hexdigest(), 16)

def parse_file_cdn(url: str, shard_id: int, total_shards: int, dedup: DedupStore):
    """Stream download + parse; yield normalized records for this shard."""
    # Use requests streaming to avoid loading whole file in memory
    with requests.get(url, stream=True, timeout=120) as r:
        r.raise_for_status()
        # We assume parquet for now; adapt if jsonl/other appear
        # Download to temp file-like buffer in chunks
        import io
        buf = io.BytesIO()
        for chunk in r.iter_content(chunk_size=8192):
            buf.write(chunk)
        buf.seek(0)

        try:
            table = pq.read_table(buf, columns=["prompt", "response"] if "prompt" in pq.read_schema(buf).names else None)
        except (pa.ArrowInvalid, KeyError):
            # Fallback: try reading all and project
            table = pq.read_table(buf)
            if "prompt" not in table.column_names or "response" not in table.column_names:
                # Try common aliases
                col_map = {}
                for c in table.column_names:
                    low = c.lower()
                    if "prompt" in low:
                        col_map[c] = "prompt"
                    elif "response" in low or "completion" in low or "answer" in low:
                        col_map[c] = "response"
                if "prompt" not in col_map or "response" not in col_map:
                    raise ValueError(f"Cannot find prompt/response in {url}")
                table = table.select([c for c in table.column_names if c in col_map])
                table = table.rename_columns([col_map[c] for c in table.column_names])

        df = table.to_pandas()
        for _, row in df.iterrows():
            prompt = str(row.get("prompt", "")).strip()
            response = str(row.get("response", "")).strip()
            if not prompt or not response:
                continue
            # Build canonical slug from filename + row hash
            combined = f"{url}|{prompt}|{response}"
            md5 = hashlib.md5(combined.encode()).hexdigest()
            if dedup.seen(md5):
                continue
            # Deterministic shard
            if slug_hash(md5) % total_shards != shard_id:
                continue
            dedup.add(md5)
            yield {"prompt": prompt, "response": response, "md5": md5}
