## v0.1 scope
v0.1 in 2 weeks: deterministic research→signal→risk→execution workflow runner (FastAPI on CF Workers), Postgres (Supabase) for immutable audit records, Redis (Upstash) for state/streams, one broker connector (sandbox/paper trading), one data-vendor connector (OHLC/snapshot), idempotent run IDs, signed audit logs, and a replay endpoint to re-run a workflow_run deterministically.
