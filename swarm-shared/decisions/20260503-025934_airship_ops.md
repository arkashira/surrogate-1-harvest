## Implementation Plan — Airship Ops (HF CDN Bypass + Lightning Idle Resilience)

**Value**: Eliminates HF API 429s during training, prevents Lightning idle-stop training loss, reduces Mac→Lightning iteration from ~15 min to <2 min per cycle.  
**Scope**: Non-breaking ops improvements only — no schema changes.

---

### 1) HF CDN Bypass for Surrogate Training (eliminate 429s)
- Single `list_repo_tree` call from Mac (after rate-limit window) for one date folder → save `file_list.json`.
- Embed `file_list.json` in Lightning training script; training uses only CDN URLs (`https://huggingface.co/datasets/.../resolve/main/...`) with zero API calls during data load.
- Fallback: if CDN fetch fails, retry with exponential backoff (max 3) before aborting.

### 2) Lightning Studio Reuse + Idle Resilience
- Before `Studio(create_ok=True)`, list `Teamspace.studios` and reuse a running studio with matching name.
- On each `.run()`, check studio status; if stopped, restart with `target.start(machine=Machine.L40S)` (or fallback to free-tier L40S).
- Add short keep-alive ping during long training epochs (optional) to reduce idle-stop risk.

### 3) Mac Orchestration Wrapper (ops script)
- Bash shebang, executable, sets `SHELL=/bin/bash` for cron compatibility.
- Runs: list HF folder → produce `file_list.json` → launch/update Lightning training.

#### `scripts/ops/hf_cdn_filelist.sh`
```bash
#!/usr/bin/env bash
# Mac-only orchestration: list HF folder and emit file_list.json for training
# Usage: HF_REPO="datasets/axentx/surrogate-mirror" HF_FOLDER="batches/mirror-merged/2026-05-03" ./hf_cdn_filelist.sh

set -euo pipefail
SHELL=/bin/bash

HF_REPO="${HF_REPO:-datasets/axentx/surrogate-mirror}"
HF_FOLDER="${HF_FOLDER:-batches/mirror-merged/$(date +%Y-%m-%d)}"
OUTFILE="${OUTFILE:-file_list.json}"

echo "[$(date -u)] Listing HF folder: ${HF_REPO}/${HF_FOLDER}"

# Requires huggingface_hub installed on Mac (or use curl + token if preferred)
python3 - "$HF_REPO" "$HF_FOLDER" "$OUTFILE" <<'PY'
import json, os, sys
from huggingface_hub import HfApi

repo_id = sys.argv[1].replace("datasets/", "")  # strip datasets/ prefix if present
folder = sys.argv[2].lstrip("/")
outfile = sys.argv[3]

api = HfApi()
# Single non-recursive call per folder (avoids heavy pagination)
files = api.list_repo_tree(repo_id=repo_id, path=folder, repo_type="dataset", recursive=False)
file_list = [f.rfilename for f in files if not f.rfilename.endswith("/")]

payload = {
    "repo_id": repo_id,
    "folder": folder,
    "files": sorted(file_list),
    "cdn_prefix": f"https://huggingface.co/datasets/{repo_id}/resolve/main"
}

with open(outfile, "w") as f:
    json.dump(payload, f, indent=2)

print(f"Wrote {len(file_list)} files to {outfile}")
PY

echo "[$(date -u)] Done. CDN prefix: https://huggingface.co/datasets/${HF_REPO}/resolve/main"
```

#### `scripts/training/surrogate_train.py` (excerpt — embed file_list)
```python
import json, os, requests
from pathlib import Path
from torch.utils.data import IterableDataset

HF_REPO = "datasets/axentx/surrogate-mirror"
FILE_LIST_PATH = Path("file_list.json")

def load_file_list():
    if not FILE_LIST_PATH.exists():
        raise FileNotFoundError(f"{FILE_LIST_PATH} not found. Run hf_cdn_filelist.sh first.")
    with open(FILE_LIST_PATH) as f:
        return json.load(f)

class CDNParquetIterable(IterableDataset):
    def __init__(self, file_list_payload, max_retries=3):
        self.prefix = file_list_payload["cdn_prefix"]
        self.files = file_list_payload["files"]
        self.max_retries = max_retries

    def _fetch_parquet(self, file_rel):
        url = f"{self.prefix}/{file_rel}"
        for attempt in range(1, self.max_retries + 1):
            try:
                resp = requests.get(url, timeout=30)
                resp.raise_for_status()
                return resp.content
            except Exception as exc:
                if attempt == self.max_retries:
                    raise
                sleep_sec = 2 ** attempt
                print(f"CDN fetch failed for {file_rel} (attempt {attempt}): {exc}. Retry in {sleep_sec}s")
                time.sleep(sleep_sec)

    def __iter__(self):
        import io, pyarrow.parquet as pq, time
        for rel in self.files:
            if not rel.endswith(".parquet"):
                continue
            raw = self._fetch_parquet(rel)
            table = pq.read_table(io.BytesIO(raw))
            # Project to {prompt, response} only (no extra schema assumptions)
            for batch in table.to_batches(max_chunksize=1024):
                df = batch.to_pandas()
                for _, row in df.iterrows():
                    prompt = row.get("prompt")
                    response = row.get("response")
                    if prompt is not None and response is not None:
                        yield {"prompt": str(prompt), "response": str(response)}
```

#### `scripts/ops/lightning_train_launcher.py`
```python
#!/usr/bin/env python3
"""
Lightning Studio launcher with reuse + idle resilience.
Run from Mac (or cron) after hf_cdn_filelist.sh produces file_list.json.
"""
import os, time, sys
from lightning_sdk import Studio, Teamspace, Machine

SCRIPT_DIR = os.path.dirname(os.path.abspath(__file__))
PROJECT_ROOT = os.path.dirname(SCRIPT_DIR)
os.chdir(PROJECT_ROOT)

STUDIO_NAME = "surrogate-train-l40s"
MACHINE = Machine.L40S
TRAIN_SCRIPT = "scripts/training/surrogate_train.py"

def main():
    teamspace = Teamspace()
    studio = None

    # Reuse running studio if exists
    for s in teamspace.studios:
        if s.name == STUDIO_NAME:
            studio = s
            print(f"Found existing studio: {s.name} (status={s.status})")
            break

    if studio is None:
        print(f"Creating studio: {STUDIO_NAME}")
        studio = teamspace.studios.create(
            name=STUDIO_NAME,
            machine=MACHINE,
            create_ok=True,
        )

    # Ensure running
    if studio.status != "running":
        print(f"Starting studio (was {studio.status})...")
        studio.start(machine=MACHINE)
        # Wait for running
        for _ in range(60):
            studio.refresh()
            if studio.status == "running":
                break
            time.sleep(5)
        else:
            raise RuntimeError("Studio failed to start")

    # Run training (non-blocking run; adapt as needed)
    print("Launching training script in studio...")
    run = studio.run(
        command=[sys.executable, TRAIN_SCRIPT],
        environment={"PYTHONUNBUFFERED": "1"},
    )

    print(f"Run submitted: {run.id}")
    # Optional: stream logs or poll status
    # For ops, we submit and exit; rely on studio reuse on next iteration.

if __name__ == "__main__":
    main()
```
