# Costinel / ops

## Implementation Plan — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only card that identifies the most-connected hub from the knowledge-rag graph and surfaces the top hub with 3 contextual signals.  
**Timebox**: <2h (frontend-only, no backend changes).

### Why
- Follows pattern: review most-connected hub (e.g., "MOC") before planning tasks (#knowledge-rag #graph #hub).
- Adds immediate ops value by surfacing context at the top of the dashboard without touching ingestion or training pipelines.
- Read-only and safe: no writes, no API changes, no infra.

---

### Implementation Plan (1h 30m total)

1. **Locate dashboard layout** (10m)  
   Identify where to inject the card (likely near top of Costinel dashboard or sidebar).

2. **Add mock graph adapter** (20m)  
   Create a lightweight client-side module that simulates knowledge-rag graph lookup (top hub + 3 signals). Uses deterministic fallback when graph unavailable.

3. **Create TopHubSignal component** (30m)  
   - Displays hub name, short description, and 3 contextual signals.
   - Read-only, accessible, matches existing design tokens.
   - Supports loading/error states.

4. **Wire into dashboard** (20m)  
   Mount component in target layout slot. Ensure responsive behavior.

5. **Polish & test** (10m)  
   Verify rendering, copy, keyboard nav, and no console errors.

  // Today: deterministic fallback to avoid infra changes.

    // ignore storage limits
  }

#### 2) TopHubSignal React component
`src/components/TopHubSignal.jsx`
```jsx
import React, { useEffect, useState } from 'react';
import { getTopHub } from '../lib/knowledgeRagGraph';
import './TopHubSignal.css';

export default function TopHubSignal({ compact = false }) {
  const [hub, setHub] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Simulate lightweight async resolution (would be instant in prod)
    const resolve = () => {
      setHub(getTopHub());
      setLoading(false);
    };
    const t = setTimeout(resolve, 120);
    return () => clearTimeout(t);
  }, []);

  if (loading) {
    return (
      <div className="top-hub-skeleton" role="status" aria-label="Loading top hub">
        <div className="skeleton-bar" />
        <div className="skeleton-bar short" />
      </div>
    );
  }

  if (!hub) return null;

  if (compact) {
    return (
      <div className="top-hub-compact" aria-label={`Top hub: ${hub.label}`}>
        <span className="hub-badge">{hub.hub}</span>
        <span className="hub-label">{hub.label}</span>
      </div>
    );
  }

  return (
    <section className="top-hub-signal" aria-labelledby="top-hub-title">
      <header className="top-hub-header">
        <h2 id="top-hub-title" className="top-hub-title">
          <span className="hub-pill">{hub.hub}</span>
          {hub.label}
        </h2>
        <p className="top-hub-desc">{hub.description}</p>
      </header>

      <ul className="signals-list" aria-label="Top contextual signals">
        {hub.signals.map((s) => (
          <li key={s.id} className="signal-item">
            <div className="signal-meta">
              <span className={`signal-severity severity-${s.severity}`}>{s.severity}</span>
            </div>
            <div className="signal-body">
              <strong className="signal-title">{s.title}</strong>
              <p className="signal-desc">{s.body}</p>
              <span className="signal-action">{s.action}</span>
            </div>
          </li>
        ))}
      </ul>
    </section>
  );
}
```
