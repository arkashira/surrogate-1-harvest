# airship / backend

## Analysis & Highest-Value Increment

Based on the README and patterns, the airship project is a DevOps/AI platform with two microservices (Arkship on 8000, Surrogate AI on 8001). The surrogate-1 training pipeline is blocked by HF API rate limits during dataset ingestion.

**Highest-value incremental improvement (<2h)**:  
Implement a **deterministic CDN-only manifest generator** for HuggingFace datasets that bypasses HF API rate limits. This enables surrogate-1 training to proceed without API throttling by pre-computing file lists on the Mac (orchestration) and embedding them in training scripts for Lightning Studio CDN-only fetches.

---

## Implementation Plan

### 1. Create CLI Tool: `hf-manifest` (30 min)
- Single command: `python -m airship.hf_manifest --repo datasets/username/repo --date 2026-04-29 --output manifest.json`
- Uses HF API **once** to list files in a date folder, then generates manifest with CDN URLs
- Output: JSON array of `{ "path": "...", "cdn_url": "...", "sha256": null }`

### 2. Create HTTP Endpoint in Surrogate (45 min)
- POST `/api/v1/hf/manifest` — accepts repo/date, returns manifest JSON
- Cached for 24h to avoid repeated API calls
- Used by Arkship workflows to fetch manifests before training

### 3. Update Surrogate Training Script Template (30 min)
- Embed manifest path or inline manifest in Lightning training script
- Data loader uses `requests.get(cdn_url)` with streaming — **zero HF API calls during training**
- Add retry/backoff for CDN failures

### 4. Add CLI Command to Arkship (15 min)
- `arkship hf generate-manifest` — wraps the Python module
- Outputs to `./manifests/{repo}/{date}.json`

```python
#!/usr/bin/env python3
"""
Deterministic CDN-only manifest generator for HuggingFace datasets.
Bypasses HF API rate limits by listing once, then using CDN URLs.
"""
import json
import hashlib
import argparse
from datetime import datetime
from pathlib import Path
from typing import List, Dict, Optional

try:
    from huggingface_hub import HfApi, list_repo_tree
except ImportError:
    print("Installing huggingface_hub...")
    import subprocess, sys
    subprocess.check_call([sys.executable, "-m", "pip", "install", "huggingface_hub"])
    from huggingface_hub import HfApi, list_repo_tree

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def generate_manifest(repo: str, date_folder: str, output_path: Optional[Path] = None) -> List[Dict]:
    """
    Generate manifest for a specific date folder in a HF dataset repo.
    
    Args:
        repo: e.g., "username/dataset-name"
        date_folder: e.g., "2026-04-29" or "batches/mirror-merged/2026-04-29"
        output_path: Optional path to save JSON manifest
    
    Returns:
        List of file entries with CDN URLs
    """
    api = HfApi()
    
    # List files in the date folder (non-recursive to avoid pagination explosion)
    # We'll handle subfolders by recursive=True only if depth is small
    try:
        tree = list_repo_tree(repo=repo, path=date_folder, recursive=True)
    except Exception as e:
        print(f"Error listing repo tree: {e}")
        print("Falling back to non-recursive listing...")
        tree = list_repo_tree(repo=repo, path=date_folder, recursive=False)
    
    manifest = []
    for item in tree:
        if item.type != "file":
            continue
        
        file_path = item.path
        cdn_url = CDN_TEMPLATE.format(repo=repo, path=file_path)
        
        entry = {
            "path": file_path,
            "cdn_url": cdn_url,
            "size": getattr(item, "size", None),
            "lfs": getattr(item, "lfs", None) is not None,
        }
        manifest.append(entry)
    
    # Sort for deterministic output
    manifest.sort(key=lambda x: x["path"])
    
    # Add metadata
    result = {
        "repo": repo,
        "date_folder": date_folder,
        "generated_at": datetime.utcnow().isoformat() + "Z",
        "total_files": len(manifest),
        "files": manifest,
    }
    
    if output_path:
        output_path.parent.mkdir(parents=True, exist_ok=True)
        with open(output_path, "w") as f:
            json.dump(result, f, indent=2)
        print(f"Manifest saved to {output_path}")
    
    return result

def main():
    parser = argparse.ArgumentParser(description="Generate CDN manifest for HF dataset")
    parser.add_argument("--repo", required=True, help="HF dataset repo (user/name)")
    parser.add_argument("--date", required=True, help="Date folder (YYYY-MM-DD)")
    parser.add_argument("--output", type=Path, help="Output JSON path")
    parser.add_argument("--batches-dir", action="store_true", 
                       help="Use batches/mirror-merged/{date} path")
    
    args = parser.parse_args()
    
    date_folder = args.date
    if args.batches_dir:
        date_folder = f"batches/mirror-merged/{args.date}"
    
    if not args.output:
        args.output = Path(f"manifests/{args.repo.replace('/', '_')}/{args.date}.json")
    
    generate_manifest(args.repo, date_folder, args.output)

if __name__ == "__main__":
    main()
```
