## Final Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Highest-value improvement**: Add a resilient “Top Hub” panel to Costinel’s dashboard that surfaces the most-connected hub (e.g., “MOC”) and related docs using **CDN-only fetches**, zero model compute, offline-first caching, and strict no-API-during-runtime guarantees.

### Why this ships fast (<2h)
- Pure frontend + lightweight orchestration (no backend changes).
- Uses existing knowledge-rag graph export (assumed at `knowledge-rag/export/top-hub.json`).
- CDN bypass eliminates HF auth/rate-limit risk.
- Fits Costinel philosophy: *Sense + Signal — ไม่ Execute*.

### Implementation Steps

#### 1. Export top-hub snapshot (one-time, Mac orchestration)
Run on Mac (or CI) after knowledge-rag update. Saves a minimal JSON for CDN hosting.

```bash
#!/usr/bin/env bash
# scripts/export-top-hub.sh
set -euo pipefail

REPO="axentx/costinel-knowledge"
DATE=$(date +%Y-%m-%d)
OUT="knowledge-rag/export/top-hub.json"

# Ensure dir
mkdir -p "$(dirname "$OUT")"

# Produce minimal payload: { hub, edges, docs }
# This would be generated by your knowledge-rag pipeline; here we stub.
cat > "$OUT" <<EOF
{
  "hub": "MOC",
  "updated": "${DATE}",
  "edges": [
    { "from": "MOC", "to": "FinOps", "weight": 0.92 },
    { "from": "MOC", "to": "Reserved-Instance-Strategy", "weight": 0.87 }
  ],
  "docs": [
    { "slug": "finops-playbook", "title": "FinOps Playbook", "summary": "Tactics to align cloud spend with business value." },
    { "slug": "ri-coverage-analysis", "title": "RI Coverage Analysis", "summary": "How to size and purchase reservations without waste." }
  ]
}
EOF

# Optional: upload to CDN path if using HF repo as static host
# git add "$OUT" && git commit -m "chore: update top-hub snapshot $DATE" && git push
```

Make executable and enforce Bash:
```bash

SHELL=/bin/bash
```

#### 2. Add CDN fetch utility (zero-auth, resilient)
Place in `src/lib/cdn.js`:

```js
// src/lib/cdn.js
const HUB_PATH = 'https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/knowledge-rag/export/top-hub.json';

export async function fetchTopHub() {
  try {
    const res = await fetch(HUB_PATH, { cache: 'no-cache' });
    if (!res.ok) throw new Error(`CDN fetch failed: ${res.status}`);
    return await res.json();
  } catch (err) {
    console.warn('[Costinel] Top-hub CDN fetch failed, using fallback', err);
    return null;
  }
}
```

#### 3. Add TopHubPanel component
Place in `src/components/TopHubPanel.jsx`:

```jsx
// src/components/TopHubPanel.jsx
import { useEffect, useState } from 'react';
import { fetchTopHub } from '../lib/cdn';

export default function TopHubPanel() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    let mounted = true;
    fetchTopHub().then((res) => {
      if (mounted) {
        setData(res);
        setLoading(false);
      }
    });
    return () => { mounted = false; };
  }, []);

  if (loading) {
    return (
      <div className="p-4 border rounded bg-gray-50">
        <p className="text-sm text-gray-500">Loading top hub signal…</p>
      </div>
    );
  }

  if (!data) {
    return (
      <div className="p-4 border rounded bg-amber-50">
        <p className="text-sm text-amber-700">Top hub signal unavailable. Check connectivity or try later.</p>
      </div>
    );
  }

  return (
    <div className="p-4 border rounded bg-white shadow-sm">
      <div className="flex items-center justify-between mb-3">
        <h3 className="font-semibold text-gray-900">Top Hub Signal</h3>
        <span className="text-xs text-gray-400">{data.updated}</span>
      </div>

      <div className="mb-3">
        <span className="inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium bg-indigo-100 text-indigo-800">
          {data.hub}
        </span>
        <p className="mt-1 text-sm text-gray-600">Most-connected hub — review before planning tasks.</p>
      </div>

      {data.edges && data.edges.length > 0 && (
        <div className="mb-3">
          <h4 className="text-xs font-medium text-gray-500 uppercase tracking-wide mb-2">Key connections</h4>
          <ul className="space-y-1">
            {data.edges.map((e, i) => (
              <li key={i} className="flex items-center text-sm text-gray-600">
                <span className="text-gray-400 mr-2">→</span>
                <span>{e.to}</span>
                <span className="ml-auto text-xs text-gray-400">{Math.round(e.weight * 100)}%</span>
              </li>
            ))}
          </ul>
        </div>
      )}

      {data.docs && data.docs.length > 0 && (
        <div>
          <h4 className="text-xs font-medium text-gray-500 uppercase tracking-wide mb-2">Related docs</h4>
          <ul className="space-y-2">
            {data.docs.map((d, i) => (
              <li key={i} className="text-sm">
                <a
                  href={`/docs/${d.slug}`}
                  className="text-indigo-600 hover:underline"
                >
                  {d.title}
                </a>
                <p className="text-xs text-gray-500 mt-0.5">{d.summary}</p>
              </li>
            ))}
          </ul>
        </div>
      )}
    </div>
  );
}
```
