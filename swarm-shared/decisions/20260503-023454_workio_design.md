# workio / design

## Final Synthesis & Action Plan

**Core diagnosis (accepted from both candidates):**  
Duplicate punches are caused by (1) missing idempotency keys, (2) non-atomic upserts under concurrency, (3) no database-level enforcement of one valid `(employee_id, punch_date, punch_type)` row, and (4) frontend double-taps / LINE retries without client-side deduplication.

**Chosen approach (resolve contradictions in favor of correctness + concrete actionability):**  
- Enforce **database uniqueness** as the source of truth.  
- Use **idempotency_key** for safe retries from all clients (web, mobile, LINE).  
- Make punch creation **atomic** (single statement or transaction with unique constraint).  
- Add **frontend optimistic state + disabled buttons** to prevent double submission.  
- Clean existing duplicates **once** before adding the constraint.

### 1. Database schema (server/src/db/schema.sql)

```sql
-- Remove existing duplicates safely (keep earliest)
DELETE FROM punches
WHERE id IN (
  SELECT id
  FROM (
    SELECT id,
           ROW_NUMBER() OVER (
             PARTITION BY employee_id, punch_date, punch_type
             ORDER BY created_at ASC
           ) AS rn
    FROM punches
  ) dup
  WHERE dup.rn > 1
);

-- Add idempotency support
ALTER TABLE punches
  ADD COLUMN IF NOT EXISTS idempotency_key TEXT,
  ADD COLUMN IF NOT EXISTS created_via TEXT NOT NULL DEFAULT 'web';

-- Unique business rule: one punch type per employee per date
CREATE UNIQUE INDEX IF NOT EXISTS uq_punch_employee_date_type
  ON punches (employee_id, punch_date, punch_type);

-- Fast, safe idempotency lookups
CREATE UNIQUE INDEX IF NOT EXISTS uq_punch_idempotency_key
  ON punches (idempotency_key)
  WHERE idempotency_key IS NOT NULL;
```

### 2. Backend service (server/src/services/punchService.ts)

```ts
import { Pool } from 'pg';
import { AppError } from '../utils/errors';

const pool = new Pool({ connectionString: process.env.DATABASE_URL });

export interface CreatePunchInput {
  employeeId: string;
  punchType: 'clock_in' | 'clock_out';
  timestamp: Date;
  location?: { lat: number; lng: number };
  lineUserId?: string;
  idempotencyKey?: string;
  createdVia?: 'web' | 'line' | 'mobile';
}

export async function createPunch(input: CreatePunchInput) {
  const client = await pool.connect();
  try {
    await client.query('BEGIN');

    const punchDate = new Date(input.timestamp).toISOString().split('T')[0];

    // Idempotency fast-path
    if (input.idempotencyKey) {
      const existing = await client.query(
        `SELECT id, employee_id, punch_date, punch_type, created_at
         FROM punches
         WHERE idempotency_key = $1`,
        [input.idempotencyKey]
      );
      if (existing.rows.length > 0) {
        await client.query('ROLLBACK');
        return { ...existing.rows[0], reused: true };
      }
    }

    // Atomic insert; unique constraint protects races
    const result = await client.query(
      `INSERT INTO punches (
         employee_id, punch_type, punch_date, timestamp,
         latitude, longitude, line_user_id, idempotency_key, created_via
       ) VALUES ($1,$2,$3,$4,$5,$6,$7,$8,$9)
       RETURNING *`,
      [
        input.employeeId,
        input.punchType,
        punchDate,
        input.timestamp,
        input.location?.lat ?? null,
        input.location?.lng ?? null,
        input.lineUserId ?? null,
        input.idempotencyKey ?? null,
        input.createdVia ?? 'web',
      ]
    );

    await client.query('COMMIT');
    return { ...result.rows[0], reused: false };
  } catch (err: any) {
    await client.query('ROLLBACK');
    // Unique violation: concurrent duplicate or constraint conflict
    if (err.code === '23505') {
      const conflict = await client.query(
        `SELECT id, employee_id, punch_date, punch_type, created_at
         FROM punches
         WHERE employee_id = $1 AND punch_date = $2 AND punch_type = $3`,
        [input.employeeId, new Date(input.timestamp).toISOString().split('T')[0], input.punchType]
      );
      if (conflict.rows.length > 0) {
        return { ...conflict.rows[0], reused: true };
      }
    }
    throw new AppError('Failed to create punch', 500);
  } finally {
    client.release();
  }
}
```

### 3. API route (server/src/routes/punch.ts)

```ts
import express from 'express';
import { createPunch } from '../services/punchService';

const router = express.Router();

router.post('/', async (req, res) => {
  try {
    const { employeeId, punchType, timestamp, location, idempotencyKey } = req.body;

    if (!employeeId || !punchType || !timestamp) {
      return res.status(400).json({ error: 'Missing required fields' });
    }

    const record = await createPunch({
      employeeId,
      punchType,
      timestamp: new Date(timestamp),
      location,
      idempotencyKey,
      createdVia: 'web',
    });

    return res.status(200).json(record);
  } catch (e) {
    console.error(e);
    return res.status(500).json({ error: 'Internal server error' });
  }
});

export default router;
```
