# Costinel / design

## Highest-Value Incremental Improvement (<2h)

**Goal**: Add a lightweight, machine-readable **discovery + readiness surface** so onboarding, RAG, and training pipelines can auto-detect sources, schemas, and health without trial-and-error.

**Why this ships fastest**:
- Pure additive changes (no breaking changes).
- Reuses existing patterns: `knowledge-rag` (top-hub), file-manifest strategy (HF CDN bypass), and design-time validation probes.
- Single endpoint + one CLI command covers onboarding, RAG, and surrogate-1 training prep.

---

## Implementation Plan

### 1) Add `/discovery` and `/ready` endpoints (FastAPI)
- `GET /discovery`: returns JSON manifest of detected cloud accounts, datasets, top-hub docs, and ingestible file lists.
- `GET /ready`: liveness/readiness probe (checks DB, HF token, Lightning quota).

### 2) Add CLI: `scripts/discover.sh`
- Runs cloud account scan (AWS/GCP/Azure via configured profiles).
- Runs HF manifest fetch (single `list_repo_tree` per date folder) and writes `batches/mirror-merged/{date}/files.json`.
- Queries knowledge-rag for top-hub (e.g., "MOC") and appends summary to discovery payload.
- Exits non-zero if readiness checks fail.

### 3) Training script adaptation (surrogate-1 prep)
- Update `train.py` to accept `--file-list` (JSON from discovery) and use CDN-only fetches (`huggingface.co/datasets/.../resolve/main/...`) with no auth.

### 4) Add design-time validation module
- Lightweight schema validator for incoming datasets (reject mixed schemas before ingestion).
- Emits actionable signals (not execution) per Costinel philosophy.

```python
from fastapi import APIRouter, HTTPException
from typing import Dict, Any, List
import json, os, subprocess
from datetime import datetime

router = APIRouter()

def detect_cloud_accounts() -> List[Dict[str, str]]:
    # Lightweight detection via configured profiles / env vars
    accounts = []
    for cloud in ("AWS", "GCP", "AZURE"):
        if os.getenv(f"{cloud}_ACCOUNT_ID"):
            accounts.append({"cloud": cloud, "account_id": os.getenv(f"{cloud}_ACCOUNT_ID")})
    return accounts

def load_file_manifest(date_folder: str) -> List[str]:
    path = f"batches/mirror-merged/{date_folder}/files.json"
    if os.path.exists(path):
        with open(path) as f:
            return json.load(f)
    return []

def top_hub_summary() -> Dict[str, Any]:
    # Placeholder: in real usage, query knowledge-rag for top hub (e.g., MOC)
    return {"hub": "MOC", "connections": 124, "summary": "Most-connected hub for cost governance patterns"}

@router.get("/discovery")
def discovery() -> Dict[str, Any]:
    today = datetime.utcnow().strftime("%Y-%m-%d")
    return {
        "timestamp": datetime.utcnow().isoformat(),
        "cloud_accounts": detect_cloud_accounts(),
        "datasets": {
            "mirror_merged": {
                "latest_date": today,
                "files": load_file_manifest(today)
            }
        },
        "top_hub": top_hub_summary(),
        "philosophy": "Sense + Signal — ไม่ Execute"
    }

@router.get("/ready")
def ready() -> Dict[str, str]:
    checks = {
        "database": os.getenv("DB_URL") is not None,
        "hf_token": bool(os.getenv("HF_TOKEN")),
        "lightning_quota": True  # placeholder: could ping Lightning Teamspace.studios
    }
    if not all(checks.values()):
        raise HTTPException(status_code=503, detail={"status": "not ready", "checks": checks})
    return {"status": "ready", "checks": checks}
```

### `scripts/discover.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

# Discovery script for Costinel — produces machine-readable manifest
# Usage: bash scripts/discover.sh [--date YYYY-MM-DD]

DATE="${2:-$(date -u +%Y-%m-%d)}"
OUTDIR="batches/mirror-merged/${DATE}"
OUTFILE="${OUTDIR}/files.json"

mkdir -p "${OUTDIR}"

echo "== Detecting cloud accounts =="
# Lightweight detection via env / profiles
for svc in AWS GCP AZURE; do
  if [[ -n "${!svc_ACCOUNT_ID:-}" ]]; then
    echo "Found ${svc}: ${!svc_ACCOUNT_ID}"
  fi
done

echo "== Fetching HF repo file list (single API call) =="
REPO="${HF_REPO:-axentx/costinel-datasets}"
# Use list_repo_tree per folder (non-recursive) to avoid pagination/rate limits
# Requires HF_TOKEN for API; falls back to CDN-only list if unavailable
if [[ -n "${HF_TOKEN:-}" ]]; then
  python3 -c "
import os, json
from huggingface_hub import HfApi
api = HfApi()
try:
    tree = api.list_repo_tree(repo_id='${REPO}', path='${DATE}', recursive=False)
    files = [item.rfilename for item in tree if not item.rfilename.endswith('/')]
except Exception as e:
    files = []
with open('${OUTFILE}', 'w') as f:
    json.dump(files, f, indent=2)
print(f'Wrote {len(files)} entries to ${OUTFILE}')
"
else
  echo "HF_TOKEN not set — skipping API file list (will use CDN-only in training)."
  echo "[]" > "${OUTFILE}"
fi

echo "== Querying knowledge-rag for top-hub =="
# Placeholder: call knowledge-rag CLI or API if available
# Example: knowledge-rag top-hub --format json > "${OUTDIR}/top-hub.json"
# For now produce minimal stub
cat > "${OUTDIR}/top-hub.json" <<EOF
{
  "hub": "MOC",
  "connections": 124,
  "summary": "Most-connected hub for cost governance patterns"
}
EOF

echo "== Running readiness checks =="
if curl -sf http://localhost:8000/ready > /dev/null; then
  echo "Service ready."
else
  echo "Service not ready — check /ready endpoint."
  exit 1
fi

echo "== Discovery complete =="
python3 -c "
import json, sys
with open('${OUTFILE}') as f:
    files = json.load(f)
with open('${OUTDIR}/top-hub.json') as f:
    hub = json.load(f)
payload = {
    'date': '${DATE}',
    'files': files,
    'top_hub': hub
}
print(json.dumps(payload, indent=2))
"
```
