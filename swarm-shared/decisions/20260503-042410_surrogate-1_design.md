# surrogate-1 / design

## Final Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **`bin/dataset-enrich.sh` → `bin/dataset-enrich.py`**
   - Keep same CLI contract (`SHARD_ID`, `SHARD_TOTAL`, `DATE`) so GitHub Actions matrix requires no changes.
   - Single API call to `list_repo_tree(path, recursive=False)` for the target date folder; save manifest JSON.
   - Download files via **CDN bypass** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header.
   - Deterministic shard assignment via `hash(slug) % SHARD_TOTAL == SHARD_ID` to ensure stable, collision-free partitioning.
   - Stream-parse each file; project to `{prompt, response}` only at parse time (avoids pyarrow CastError on mixed schemas); compute md5 for dedup.
   - Write shard output as `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.
   - Exit codes and stderr preserved for Actions logs.

2. **`lib/dedup.py`** (unchanged contract)
   - Keep SQLite-backed md5 store; expose `is_duplicate(md5) -> bool` and `add(md5) -> None`.
   - No schema changes.

3. **`requirements.txt`**
   - Add `requests` (for CDN downloads) and `tqdm` (optional progress). Keep existing deps.

4. **`.github/workflows/ingest.yml`**
   - No changes to matrix strategy; only update the run command from `bash bin/dataset-enrich.sh` to `python bin/dataset-enrich.py`.

5. **Delete `bin/dataset-enrich.sh`** after verifying Python version works.

def hf_tree(date_folder: str) -> List[Dict[str, Any]]:

    items = HF_API.list_repo_tree(

def cdn_url(repo: str, path: str) -> str:
    """CDN bypass URL (no auth, no API rate limit)."""
    return f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def sha256_bytes(b: bytes) -> str:
    return hashlib.sha256(b).hexdigest()

def project_to_pair(raw: Dict[str, Any]) -> Dict[str, str]:
    """
    Normalize heterogeneous schemas to {prompt, response}.
    Heuristic:
      - prompt/response keys preferred
      - input/output fallback
      - question/answer fallback
      - first/second column fallback
    """
    keys = {k.lower(): k for k in raw.keys()}
    prompt = None
    response = None

    for pk, rk in [
        ("prompt", "response"),
        ("input", "output"),
        ("question", "answer"),
    ]:
        if pk in keys and rk in keys:
            prompt = raw[keys[pk]]
            response = raw[keys[rk]]
            break

    if prompt is None:
        cols = list(raw.keys())
        if len(cols) >= 2:
            prompt = raw[cols[0]]
            response = raw[cols[1]]
        elif len(cols) == 1:
            prompt = raw[cols[0]]
            response = ""
        else:
            prompt = ""
            response = ""

    # Ensure strings
    return {
        "prompt": str(prompt) if prompt is not None else "",
        "response": str(response) if response is not None else "",
    }

def stream_parquet(path: str):
    """Stream rows from local parquet file."""
    table = pq.read_table(path)
    for batch in table.to_batches(max_chunksize=10_000):
        for i in range(batch.num_rows):
            row = {k: batch.column(k)[i].as_py() for k in batch.schema.names}
            yield row

def stream_jsonl(path: str):
    with open(path, "r", encoding="utf-8") as f:
        for line in f:
            line = line.strip()
            if not line:
                continue
            yield json.loads(line)

def stream_csv(path: str):
    table = pcsv.read_csv(path)
    for batch in table.to_batches(max_chunksize=10_000):
        for i in range(batch.num_rows):
            row = {k: batch.column(k)[i].as_py() for k in batch.schema.names}
            yield row

def stream_json(path: str):
    table = pj.read_json(path)
    for batch in table.to_batches(max_chunksize=10_000):
        for i in range(batch.num_rows):
            row = {k: batch.column(k)[i].as_py() for k in batch.schema.names}
            yield row

def stream_file(path: str):
    """Dispatch by extension; fallback to line-based JSON."""
    p = Path(path)
    if p.suffix.lower() == ".parquet":
        yield from stream_parquet(path)
    elif p.suffix.lower() == ".jsonl":
        yield from stream_jsonl(path)
    elif p.suffix.lower() == ".csv":
        yield from stream_csv(path)
    elif p.suffix.lower() == ".json":
        yield from stream_json(path)
    else:
        # Try line-delimited JSON
        with open(path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                try:
                    yield json.loads(line)
                except Exception:
                    continue

def download_via_cdn(repo: str, path: str, dest: Path):
    """Download via CDN (bypasses HF API auth/rate limits)."""
    url = cdn_url(repo, path)
    resp = requests.get(url, timeout=60, stream=True)
    resp.raise_for_status()
    with open(dest, "wb") as f:
        for chunk in resp.iter_content(chunk_size=8192):
            f.write(chunk)
