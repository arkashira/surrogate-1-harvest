# Costinel / ops

**Final Implementation Plan – Top-Hub Signal Panel (Costinel Dashboard)**

---

### 1. Scope & Value (resolved)
- **Frontend-only, read-only widget** that surfaces the most-connected hub (default **MOC**) and its top actionable proposals.
- **Zero backend changes** required.
- **Two deployment options** (pick one at build/config time):
  1. **Client-fetch mode**: lightweight runtime fetch to `/api/knowledge-graph/hubs/{hubId}/insights` with graceful fallback to local static JSON.
  2. **Static-embed mode**: import static JSON at build time (or mount from `public/`) to eliminate runtime API calls entirely.
- Ships in **<2h** as a resilient, self-contained component.
- **SSR-safe** and progressively enhanced; degrades gracefully when graph data is unavailable.
- Aligns with existing patterns: review the most-connected hub before planning tasks.

### 4. Data Strategy (resolved contradiction)

- **Primary**: fetch from `/api/knowledge-graph/hubs/{hubId}/insights` (GET, `cache: "no-store"`).
- **Fallback chain**:
  1. Network or API failure → load `public/data/knowledge-graph/moc-hub.json`.
  2. If that fails (or during SSR) → use imported `public/mock/top-hub-mock.json`.
- **Static-embed option**: replace runtime fetch with `import mocData from "../../../public/data/knowledge-graph/moc-hub.json"` and skip the hook fetch entirely (set via env flag or prop).

### 5. Hook (robust + SSR-safe)

**src/lib/hooks/useTopHubInsights.ts**
```ts
import { useEffect, useState, useCallback } from "react";

const loadMock = () =>
  import("../../../public/mock/top-hub-mock.json").then((m) => m.default);
const loadStaticFallback = () =>
  import("../../../public/data/knowledge-graph/moc-hub.json").then((m) => m.default);

export function useTopHubInsights(
  hubId = "MOC",
  refreshIntervalMs = 60000,
  { useStatic = false } = {}
) {
  const [data, setData] = useState<HubInsights | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<Error | null>(null);

  const fetchData = useCallback(async () => {
    try {
      if (useStatic) {
        const staticData = await loadStaticFallback();
        setData(staticData);
        setError(null);
        return;
      }

      const res = await fetch(`/api/knowledge-graph/hubs/${hubId}/insights`, {
        cache: "no-store",
      });

      if (!res.ok) throw new Error(`HTTP ${res.status}`);
      const json = await res.json();
      setData(json);
      setError(null);
    } catch (err) {
      setError(err as Error);
      try {
        const fallback = await loadStaticFallback();
        setData(fallback);
      } catch {
        const mock = await loadMock();
        setData(mock);
      }
    } finally {
      setLoading(false);
    }
  }, [hubId, useStatic]);

  useEffect(() => {
    fetchData();
    const id = setInterval(fetchData, refreshIntervalMs);
    return () => clearInterval(id);
  }, [fetchData, refreshIntervalMs]);

  return { data, loading, error, refetch: fetchData };
}
```
