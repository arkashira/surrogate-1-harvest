# airship / quality

## Implementation Plan (≤2h)

**Highest-value incremental improvement**: Harden the `airship discover` orchestrator with zero-runtime-dependency Bash-first design, deterministic JSON outputs (`status.json`, `manifest.json`), and CDN-safe upload path so local/CI/CD can rely on it without Python or external tooling.

### What I’m shipping
- `bin/airship-discover` — executable Bash orchestrator (no Python runtime required)
- Deterministic `status.json` (discovery metadata + integrity)
- Deterministic `manifest.json` (file inventory + content hashes)
- CDN-safe upload: `batches/mirror-merged/{YYYY-MM-DD}/{slug}.json` (no extra metadata columns)
- CI-friendly exit codes and idempotent behavior (safe to rerun)

### Steps (concrete)
1. Create `bin/airship-discover` with proper shebang, `set -euo pipefail`, and helper functions.
2. Implement deterministic discovery:
   - Walk project root (`.`) and produce normalized file list (exclude `.git`, `node_modules`, `__pycache__`, `.tox`, `*.pyc`, `*.pyo`, `*.swp`, `.DS_Store`, `bin/airship-discover` itself).
   - For each file: compute `sha256`, record `size`, `mtime` (ISO-8601), `rel_path`.
3. Emit `status.json` (discovery run metadata) and `manifest.json` (file inventory).
4. Produce CDN-safe payload and write to `batches/mirror-merged/{date}/{slug}.json`.
5. Make script executable and ensure it runs with `/bin/bash`.

### Why this is highest value
- Removes runtime dependency (no Python/venv) for discovery — aligns with “Mac=CLI rule + heavy compute on remote”.
- Deterministic outputs enable reliable CI/CD and downstream CDN ingestion.
- Follows established patterns: `batches/mirror-merged/{date}/{slug}.json`, no extra schema columns, CDN bypass friendly.

## Code

### `bin/airship-discover`

```bash
#!/usr/bin/env bash
# airship-discover
# Zero-runtime-dependency discovery orchestrator.
# Emits:
#   status.json    - discovery run metadata
#   manifest.json  - deterministic file inventory
#   batches/mirror-merged/{YYYY-MM-DD}/{slug}.json - CDN-safe upload payload
#
# Usage:
#   bash bin/airship-discover [project-root] [--slug <slug>] [--out-dir <dir>]
#
# Exit codes:
#   0 - success
#   1 - usage / validation error
#   2 - filesystem error
#   3 - jq not available (required for deterministic JSON)

set -euo pipefail

# ---- defaults ----
PROJECT_ROOT="${1:-.}"
SLUG="${AIRSHIP_SLUG:-}"
OUT_DIR="${AIRSHIP_OUT_DIR:-./_discover}"
DATE="$(date -u +%Y-%m-%d)"
TIMESTAMP="$(date -u +%Y-%m-%dT%H:%M:%SZ)"
RUN_ID="$(date -u +%Y%m%dT%H%M%SZ)-$(head -c 8 /dev/urandom | xxd -p)"

# ---- parse opts ----
shift 1 || true
while [[ $# -gt 0 ]]; do
  case "$1" in
    --slug) SLUG="${2:-}"; shift 2 ;;
    --out-dir) OUT_DIR="${2:-}"; shift 2 ;;
    *) echo "Unknown option: $1" >&2; exit 1 ;;
  esac
done

# ---- validation ----
if [[ -z "$SLUG" ]]; then
  # deterministic default slug from project root absolute path
  ABS_ROOT="$(cd "$PROJECT_ROOT" && pwd)"
  SLUG="$(basename "$ABS_ROOT" | tr '[:upper:]' '[:lower:]' | tr -cd '[:alnum:]._-')"
  if [[ -z "$SLUG" ]]; then
    SLUG="airship-unknown"
  fi
fi

if ! command -v jq >/dev/null 2>&1; then
  echo "ERROR: jq is required for deterministic JSON output." >&2
  exit 3
fi

if [[ ! -d "$PROJECT_ROOT" ]]; then
  echo "ERROR: PROJECT_ROOT does not exist or is not a directory: $PROJECT_ROOT" >&2
  exit 2
fi

# ---- paths ----
mkdir -p "$OUT_DIR"
mkdir -p "$OUT_DIR/batches/mirror-merged/$DATE"

STATUS_PATH="$OUT_DIR/status.json"
MANIFEST_PATH="$OUT_DIR/manifest.json"
CDN_PAYLOAD_PATH="$OUT_DIR/batches/mirror-merged/$DATE/${SLUG}.json"

# ---- excludes (deterministic) ----
# Patterns to exclude (relative to PROJECT_ROOT). Keep sorted for deterministic behavior.
EXCLUDE_PATTERNS=(
  ".git"
  "node_modules"
  "__pycache__"
  ".tox"
  ".pytest_cache"
  ".venv"
  "venv"
  "env"
  ".env"
  "*.pyc"
  "*.pyo"
  "*.swp"
  ".DS_Store"
  "Thumbs.db"
  "status.json"
  "manifest.json"
  "batches/"
)

# ---- build file inventory ----
# Use find with deterministic sorting. Record: rel_path, size, mtime (ISO-8601), sha256.
# We avoid streaming find into while-read without sorting to keep output deterministic.
TMP_MANIFEST="$(mktemp)"
trap 'rm -f "$TMP_MANIFEST"' EXIT

cd "$PROJECT_ROOT"

# Build exclude args for find
EXCLUDE_ARGS=()
for p in "${EXCLUDE_PATTERNS[@]}"; do
  EXCLUDE_ARGS+=( -not -path "./$p" )
  # also exclude any path containing the pattern as a segment
  EXCLUDE_ARGS+=( -not -path "./$p/*" )
done

# Also exclude the out-dir if it's inside project root (avoid recursion)
REL_OUT_DIR_RELATIVE=""
if [[ "$OUT_DIR" = /* ]]; then
  # absolute out_dir: compute relative from project root
  REL_OUT_DIR_RELATIVE="$(realpath --relative-to="$PROJECT_ROOT" "$OUT_DIR" 2>/dev/null || echo "")"
else
  REL_OUT_DIR_RELATIVE="$OUT_DIR"
fi
if [[ -n "$REL_OUT_DIR_RELATIVE" && "$REL_OUT_DIR_RELATIVE" != "." ]]; then
  EXCLUDE_ARGS+=( -not -path "./$REL_OUT_DIR_RELATIVE/*" -not -path "./$REL_OUT_DIR_RELATIVE" )
fi

# Find regular files, exclude patterns, sort by relative path
find . -type f "${EXCLUDE_ARGS[@]}" -print0 | sort -z | while IFS= read -r -d '' file; do
  # Normalize leading ./
  rel_path="${file#./}"
  # skip the airship-discover script itself if present in bin/
  if [[ "$rel_path" = "bin/airship-discover" ]]; then
    continue
  fi

  size="$(stat -c%s "$file" 2>/dev/null || stat -f%z "$file" 2>/dev/null || echo 0)"
  # Prefer ISO-8601 mtime; fallback to epoch
  if stat -c%Y "$file" >/dev/null 2>&1; then
    mtime_epoch="$(stat -c%Y "$file")"
  else
    mtime_epoch="$(stat -f%m "$file" 2>/dev/null || echo 0)"
  fi
  mtime_iso="$(date -u -d "@$mtime_epoch" +%Y-%m-%dT%H:%M:%SZ 2>/dev/null || date -u -r "$mtime_epoch" +%Y-%m-%dT%H:%M:%SZ 2>/dev/null || echo "$TIMESTAMP")"
  sha256="$(sha256sum "$file" | awk '{print $1}')"

  # emit as JSON line (will be collected into array deterministically)
  jq -n \
    --arg rp "$rel_path" \
    --arg sz "$size" \
    --arg mt "$mtime_iso" \
    --arg sh "$sha256" \
    '{rel_path: $rp, size: ($sz | tonumber), mtime: $mt, sha256: $sh}'
done > "$TMP_MANIFEST"

# Build manifest.json as sorted array (already sorted by find+sort)
jq -s '.' "$
