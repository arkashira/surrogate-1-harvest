# Costinel / design

Below is the **single, merged, production-ready implementation** that keeps every high-value idea from both candidates while removing duplication and resolving contradictions in favor of correctness and concrete actionability.

Key decisions made:
- **Ops script is canonical** (Candidate 2) because it guarantees zero-runtime API cost and strict CDN-bypass.
- **Component is canonical** (Candidate 1) because it is complete, accessible, and ready to drop in.
- **Composable + utils are merged** into one robust, typed composable with safe fallbacks for local dev and Lightning reuse.
- **File layout** uses Candidate 2 paths (`/opt/axentx/Costinel/`) with Candidate 1 component names for clarity.

## 2) Ops script (source of truth for data)

`scripts/ops-top-hub-sync.sh`

```bash
#!/usr/bin/env bash
# Runs on Mac/cron. One HF API call per run. CDN-bypass for all dataset fetches.
set -euo pipefail

REPO="${HF_REPO:-axentx/knowledge}"
DATE_FOLDER="${1:-$(date +%Y-%m-%d)}"
OUT_DIR="$(cd "$(dirname "$0")/../public/data/knowledge-graph" && pwd)"
OUT_FILE="${OUT_DIR}/latest-hub-index.json"
HF_API="https://huggingface.co/api/models"
HF_CDN="https://huggingface.co/datasets"

mkdir -p "$OUT_DIR"

# 1) List tree for date folder (single API call)
echo "Fetching tree for ${REPO} @ ${DATE_FOLDER}..."
TREE=$(curl -fsSL "${HF_API}/${REPO}/tree/main/${DATE_FOLDER}" \
  -H "Authorization: Bearer ${HF_TOKEN:-}" || true)

if [ -z "${TREE}" ] || [ "${TREE}" = "[]" ]; then
  echo "No tree found. Generating minimal fallback."
  echo '{"generatedAt":"'$(date -u +%Y-%m-%dT%H:%M:%SZ)'","hubs":[]}' > "$OUT_FILE"
  exit 0
fi

# 2) Build lightweight hub index (no recursive fetches)
#    Use CDN-bypass URLs for any parquet/graph assets referenced later.
echo "Building hub index..."
HUBS=$(echo "$TREE" | jq -r '
  map(select(.path | endswith(".json") or endswith(".parquet"))) |
  map({
    name: (.path | split("/")[-1] | split(".")[0]),
    path: .path,
    size: .size,
    cdn_url: ("'"${HF_CDN}/${REPO}/resolve/main/${DATE_FOLDER}/"'" + .path)
  })')

cat > "$OUT_FILE" <<EOF
{
  "generatedAt": "$(date -u +%Y-%m-%dT%H:%M:%SZ)",
  "dateFolder": "${DATE_FOLDER}",
  "repo": "${REPO}",
  "hubs": ${HUBS}
}
EOF

echo "Wrote ${OUT_FILE}"

# 3) Optional Lightning reuse (non-blocking)
if command -v node >/dev/null 2>&1; then
  node "$(cd "$(dirname "$0")" && pwd)/lightning-reuse.js" "$OUT_FILE" || true
fi
```

`src/utils/cdnFetcher.js`

```js
// Loads only local/public assets. No runtime HF API calls.
export async function fetchTopHubIndex(publicPath = '/data/knowledge-graph/latest-hub-index.json') {
  try {
    const res = await fetch(publicPath, { cache: 'no-store' });
    if (!res.ok) throw new Error(`Failed to load hub index: ${res.status}`);
    return await res.json();
  } catch (err) {
    console.warn('cdnFetcher fallback:', err.message);
    return { generatedAt: new Date().toISOString(), hubs: [] };
  }
}
```

`src/utils/hubRanker.js`

```js
// Deterministic, read-only ranking for ops card.
export function rankHubs(index) {
  if (!index?.hubs?.length) return null;

  // Prefer larger files + .parquet + names indicating central hubs
  const scored = index.hubs.map((h) => {
    let score = (h.size || 0);
    if (h.path.endsWith('.parquet')) score *= 1.5;
    if (/moc|central|hub|index/i.test(h.name)) score *= 2;
    return { ...h, score };
  });

  scored.sort((a, b) => b.score - a.score);
  return scored[0];
}
```
