# Costinel / discovery

## Final Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h, production-ready)

### Core Principles (resolve contradictions)
- **CDN-first, not API-first**: Use HuggingFace CDN for all runtime fetches. Avoid real-time HF API calls during training/inference to eliminate rate-limit risk.
- **Build-time index, runtime CDN**: One orchestration call (Mac/CI) lists the date folder and persists `index_latest.json` to the repo/CDN. Runtime reads only from CDN.
- **Non-blocking UI**: SSR-friendly, skeleton while loading, hides when no data.
- **Lightning Studio reuse + quota safety**: Reuse running studio; check status before `.run()`; avoid idle-timeout kills.
- **Local fallback**: Bundled fallback JSON for resilience when CDN is unreachable.
- **<2h effort**: Minimal, focused changes; no heavy backend service or new REST API.

---

### Architecture Summary
1. **Orchestration (Mac/CI)**  
   - Runs after rate-limit window.  
   - Calls HF API **once** to list `top_hub/` folders, picks latest `YYYY-MM-DD`, writes `data/top_hub/index_latest.json` and optionally commits.  
   - Produces `items.json` in that folder (can be generated by same script or separate transform).

2. **Runtime (browser/edge)**  
   - Fetch `index_latest.json` from CDN.  
   - Fetch `items.json` from the latest date folder via CDN.  
   - If CDN fails, fall back to local bundled `fallback.json`.  
   - No HF API tokens or auth required at runtime.

3. **UI**  
   - SSR-friendly React component with `"use client"` for client-side hydration.  
   - Skeleton while loading; compact card with top 3 insights.  
   - Mounted in sidebar or top banner; non-blocking (hides when no data).

4. **Lightning Studio integration**  
   - Launcher snippet reuses running studio; checks status before `.run()`; respects idle timeout.

### Step-by-Step Implementation (ordered for fastest delivery)

#### 1. Orchestration: list top-hub folders and generate index (run on Mac/CI)

```bash
# scripts/list-top-hub.sh
#!/usr/bin/env bash
set -euo pipefail

REPO="axentx/costinel-knowledge"
FOLDER="top_hub"
OUT="data/top_hub/index_latest.json"

MAX_RETRIES=3
RETRY=0

until [ $RETRY -ge $MAX_RETRIES ] || python3 -c "
import json, os, sys
from huggingface_hub import list_repo_tree
try:
    files = list_repo_tree(repo_id='$REPO', path='$FOLDER', repo_type='dataset', recursive=False)
    folders = [f.r['path'] for f in files if f.type == 'directory']
    latest = sorted(folders)[-1] if folders else None
    payload = {'latest_folder': latest, 'folders': folders, 'generated_by': 'list-top-hub.sh'}
    os.makedirs(os.path.dirname('$OUT'), exist_ok=True)
    with open('$OUT', 'w') as f:
        json.dump(payload, f, indent=2)
    print(json.dumps(payload))
    sys.exit(0)
except Exception as e:
    print(f'Error: {e}', file=sys.stderr)
    sys.exit(1)
"
do
  RETRY=$((RETRY+1))
  echo "Attempt $RETRY failed. Waiting 60s..."
  sleep 60
done

echo "Top-hub index written to $OUT"
```

#### 2. Runtime fetch module (CDN-first, local fallback)

```ts
// src/lib/topHubSignal.ts
const CDN_ROOT = "https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main";
const LOCAL_FALLBACK = "/data/top_hub/fallback.json";

export interface TopHubItem {
  title: string;
  summary: string;
  hub: string;
  connections: number;
  url?: string;
}

export interface TopHubIndex {
  latest_folder?: string;
  folders: string[];
  generated_by: string;
}

async function fetchJson<T>(url: string): Promise<T> {
  const res = await fetch(url, { cache: "no-store" });
  if (!res.ok) throw new Error(`Fetch failed: ${res.status} ${url}`);
  return res.json();
}

export async function getTopHubSignal(): Promise<TopHubItem[] | null> {
  try {
    // 1) Try CDN index (fast, no auth)
    const index = await fetchJson<TopHubIndex>(`${CDN_ROOT}/top_hub/index_latest.json`);
    if (!index.latest_folder) return null;

    // 2) Fetch latest payload from CDN
    const payloadUrl = `${CDN_ROOT}/top_hub/${index.latest_folder}/items.json`;
    return await fetchJson<TopHubItem[]>(payloadUrl);
  } catch (err) {
    console.warn("[TopHub] CDN fetch failed, using local fallback", err);
    try {
      return await fetchJson<TopHubItem[]>(LOCAL_FALLBACK);
    } catch (err2) {
      console.error("[TopHub] Local fallback failed", err2);
      return null;
    }
  }
}
```

#### 3. UI Panel (SSR-friendly, non-blocking)

```tsx
// src/components/TopHubSignalPanel.tsx
"use client";

import { useEffect, useState } from "react";
import { TopHubItem, getTopHubSignal } from "@/lib/topHubSignal";

export default function TopHubSignalPanel() {
  const [items, setItems] = useState<TopHubItem[] | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    let mounted = true;
    getTopHubSignal()
      .then((res) => {
        if (mounted) {
          setItems(res);
          setLoading(false);
        }
      })
      .catch(() => {
        if (mounted) setLoading(false);
      });
    return () => {
      mounted = false;
    };
  }, []);

  if (loading) {
    return (
      <div className="p-3 border rounded bg-muted/30">
        <div className="h-4 w-32 bg-muted rounded animate-pulse mb-2"></div>
        <div className="space-y-2">
          {[...Array(3)].map((_, i) => (
            <div key={i} className="h-10 bg-muted rounded animate-pulse"></div>
          ))}
        </div>
      </div>
    );
  }

  if (!items || items.length === 0) {
    return null; // non-blocking — hide if no data
  }

  return (
    <div className="p-3 border rounded bg-card">
      <div className="flex items-center justify-between mb-2">
        <h3 className="text-sm font-semibold">Top Hub Signals</h3>
        <span className="text-xs text-muted-foreground">Live</span>
      </div>
      <ul className="space-y-2 text-sm">
        {items.slice(0, 3).map((item, idx) => (
          <li key={idx} className="p-2 rounded bg-muted/50">
            <div className="font-medium">{item.title}</div>
            <div className="text-muted-foreground text-xs">{item.summary}</div>
            <div className="text-xs text-primary mt-1">
              {item.hub} · {item.connections} connections
            </div>
          </li>
        ))}
      </ul>
    </div>
  );
}
```
