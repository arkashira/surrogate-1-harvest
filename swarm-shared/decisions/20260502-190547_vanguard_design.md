# vanguard / design

Candidate 1 is stronger on concrete, copy-paste-ready automation and CDN-bypass details.  
Candidate 2 is stronger on canonical discovery (knowledge-RAG top-hub/MOC) and local schema enforcement via projection to `{prompt,response}` parquet.

I’ll merge the strongest, most actionable parts into one final, minimal, correct solution. Contradictions are resolved in favor of:

- Correctness: enforce schema projection before anything reaches `enriched/`, and use CDN `resolve/main/` URLs to avoid HF API rate limits.  
- Actionability: one executable script + one small Python projection helper + one-line crontab guard; no changes to existing code.

---

## Final merged solution

### 1) What this does (succinct)
- Reuse a running Lightning Studio (L40S) or report what to create.  
- Produce a `train_manifest.json` with HF CDN-bypass `resolve/main/` URLs for a single date folder (zero API calls during training).  
- Assign a deterministic sibling-repo index for commit-cap mitigation.  
- Project local raw files into `{prompt,response}` parquet with safe filenames before any ingestion.  
- Emit a canonical discovery note (MOC-style top-hub insights) to guide planning.

try:

            reused = s
            break

        payload = {"reused": True, "studio_id": reused.id, "name": reused.name, "machine": reused.machine}
    else:
        payload = {"reused": False, "recommend_machine": "L40S", "action": "Create via Lightning UI or launcher."}

    json.dump(payload, f, indent=2)
print("Wrote studio_status.json")
PY

echo "== 2) HF CDN-bypass file list (single folder) =="
python3 - "$HF_DATASET_REPO" "$DATE_FOLDER" "$OUTPUT_DIR" <<'PY'
import os, json, sys
from huggingface_hub import list_repo_tree

repo = sys.argv[1]
folder = sys.argv[2]
out_dir = sys.argv[3]

try:
    tree = list_repo_tree(repo=repo, path=folder, recursive=False)
    files = [f.rfilename for f in tree if f.type == "file"]
    cdn_urls = [f"https://huggingface.co/datasets/{repo}/resolve/main/{f}" for f in files]
except Exception as e:
    files = []
    cdn_urls = []
    print(f"list_repo_tree failed (may be rate-limited): {e}")

manifest = {
    "date_folder": folder,
    "repo": repo,
    "files": files,
    "cdn_urls": cdn_urls,
    "note": "CDN-only fetches; zero API calls during Lightning training"
}
manifest_path = os.path.join(out_dir, "train_manifest.json")
with open(manifest_path, "w") as f:
    json.dump(manifest, f, indent=2)
print(f"Wrote {len(files)} files to train_manifest.json")
PY

echo "== 3) Sibling repo routing (commit-cap mitigation) =="
python3 - "$OUTPUT_DIR" <<'PY'
import os, json, hashlib, sys
out_dir = sys.argv[1] if len(sys.argv) > 1 else os.getcwd()
manifest_path = os.path.join(out_dir, "train_manifest.json")
if not os.path.exists(manifest_path):
    print("No manifest; skipping routing")
    sys.exit(0)

with open(manifest_path) as f:
    m = json.load(f)

slug = m.get("date_folder", "unknown")
sibling_count = 5
idx = int(hashlib.sha256(slug.encode()).hexdigest(), 16) % sibling_count
m["sibling_repo_index"] = idx
m["sibling_repo_hint"] = f"surrogate-1-sib-{idx}"
with open(manifest_path, "w") as f:
    json.dump(m, f, indent=2)
print(f"Assigned {slug} -> sibling repo index {idx}")
PY

echo "== 4) Local data projection guard (schema hygiene) =="
if [ -d "./data/raw" ]; then
    echo "Projecting raw files to {prompt,response} parquet..."
    python3 /opt/axentx/vanguard/discovery/project_local_data.py \
        --raw-dir "./data/raw" \
        --out-dir "batches/mirror-merged/$DATE_FOLDER" \
        --date-slug "$DATE_FOLDER"
else
    echo "No ./data/raw found; skipping local projection."
fi

echo "== Done =="
echo "Outputs in $OUTPUT_DIR:"
ls -l "$OUTPUT_DIR"
