# workio / quality

## Final Synthesized Solution

### Diagnosis (merged, highest-confidence items)
- **No idempotency** on `/punches` write path → duplicates from LINE webhook retries or client retries.
- **Missing DB uniqueness constraint** for “one open punch per user” → multiple concurrent clock-ins or overlapping open punches.
- **Non-transactional or partial writes** → inconsistent punch state under concurrency or failure.
- **Trusts client timestamps** → allows backdated/spoofed punches.
- **No client-side debounce** → easy to trigger duplicate requests via rapid clicks.

---

### Goal
Enforce **exactly one open punch per user** and make `/punches` **idempotent and transactional**, using server time and safe concurrency.

---

### Implementation

#### 1) DB schema (add constraints + idempotency)

```sql
-- server/src/db/schema.sql

-- One open punch per user (clocked_in_at NOT NULL and clocked_out_at IS NULL)
CREATE UNIQUE INDEX IF NOT EXISTS idx_one_open_punch_per_user
ON punches (user_id)
WHERE clocked_in_at IS NOT NULL AND clocked_out_at IS NULL;

-- Idempotency table for deduplication (retain limited time if desired)
CREATE TABLE IF NOT EXISTS punch_idempotency (
  idempotency_key TEXT PRIMARY KEY,
  punch_id        INTEGER NOT NULL,
  created_at      TIMESTAMPTZ DEFAULT NOW()
);

-- Optional: index for faster cleanup/reporting
CREATE INDEX IF NOT EXISTS idx_punches_user_open
ON punches (user_id)
WHERE clocked_in_at IS NOT NULL AND clocked_out_at IS NULL;
```

#### 2) Service layer (transactional + idempotent + server time)

```ts
// server/src/services/punchService.ts
import { Pool } from 'pg';
import { AppError } from '../lib/errors';

const pool = new Pool({ connectionString: process.env.DATABASE_URL });

export interface CreatePunchInput {
  userId: number;
  type: 'IN' | 'OUT';
  latitude?: number;
  longitude?: number;
  idempotencyKey: string;
}

export async function createPunch(input: CreatePunchInput) {
  const client = await pool.connect();
  try {
    await client.query('BEGIN');

    // 1) Idempotency check
    const idem = await client.query(
      'SELECT punch_id FROM punch_idempotency WHERE idempotency_key = $1',
      [input.idempotencyKey]
    );
    if (idem.rows[0]) {
      const punch = await client.query('SELECT * FROM punches WHERE id = $1', [idem.rows[0].punch_id]);
      await client.query('ROLLBACK');
      return punch.rows[0];
    }

    const now = new Date(); // server time anchor

    // 2) Enforce one open punch per user at DB+app level (defensive)
    if (input.type === 'IN') {
      const open = await client.query(
        'SELECT id FROM punches WHERE user_id = $1 AND clocked_in_at IS NOT NULL AND clocked_out_at IS NULL',
        [input.userId]
      );
      if (open.rows[0]) {
        throw new AppError('There is already an open punch for this user', 409);
      }
    }

    // 3) Create punch with server timestamps
    const punchRes = await client.query(
      `INSERT INTO punches (user_id, clocked_in_at, clocked_out_at, latitude, longitude, created_at, updated_at)
       VALUES ($1, $2, $3, $4, $5, $6, $6)
       RETURNING *`,
      [
        input.userId,
        input.type === 'IN' ? now : null,
        input.type === 'OUT' ? now : null,
        input.latitude ?? null,
        input.longitude ?? null,
        now
      ]
    );
    const punch = punchRes.rows[0];

    // 4) Record idempotency
    await client.query(
      'INSERT INTO punch_idempotency (idempotency_key, punch_id) VALUES ($1, $2)',
      [input.idempotencyKey, punch.id]
    );

    await client.query('COMMIT');
    return punch;
  } catch (err) {
    await client.query('ROLLBACK');
    throw err;
  } finally {
    client.release();
  }
}
```

#### 3) Route handler (read idempotency key, validate input)

```ts
// server/src/routes/punches.ts
import express from 'express';
import { createPunch } from '../services/punchService';
import { AppError } from '../lib/errors';

const router = express.Router();

router.post('/', async (req, res, next) => {
  try {
    const idempotencyKey =
      req.get('X-Line-Retry-Id') ||
      req.get('Idempotency-Key') ||
      `${req.body.userId}-${Date.now()}-${Math.random().toString(36).slice(2)}`;

    const { userId, type, latitude, longitude } = req.body;
    if (!userId || !type) {
      return next(new AppError('userId and type are required', 400));
    }
    if (!['IN', 'OUT'].includes(type)) {
      return next(new AppError('type must be "IN" or "OUT"', 400));
    }

    const punch = await createPunch({ userId, type, latitude, longitude, idempotencyKey });
    return res.json({ ok: true, punch });
  } catch (err: any) {
    next(err);
  }
});

export default router;
```

#### 4) Frontend hardening (debounce + disabled states)

```tsx
// frontend/src/components/PunchButton.tsx
import { useState } from 'react';

export function PunchButton({ onPunch }: { onPunch: (type: 'IN' | 'OUT') => Promise<void> }) {
  const [processing, setProcessing] = useState(false);

  const handle = async (type: 'IN' | 'OUT') => {
    if (processing) return;
    setProcessing(true);
    try {
      await onPunch(type);
    } finally {
      setProcessing(false);
    }
  };

  return (
    <div className="flex gap-2">
      <button
        onClick={() => handle('IN')}
        disabled={processing}
        className="btn btn-primary"
      >
        Clock In
      </button>
      <button
        onClick={() => handle('OUT')}
        disabled={processing}
        className="btn btn-secondary"
      >
        Clock Out
      </button>
    </div>
  );
}
```
