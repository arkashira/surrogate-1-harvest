## v0.1 scope
v0.1 ships a CLI + FastAPI service (hostable on CF Workers) that discovers Lambda and Cloud Functions schedules in configured accounts/regions, persists metadata to Supabase Postgres, detects duplicates (identical cron+target), drift (cron changed vs last known), and orphans (schedule exists with no callable target), exports a JSON report and retirements plan (dry-run only) — all runnable locally and via one API endpoint within 2 weeks.
