## Highest-Value Incremental Improvement (<2h)

**CDN-only ingestion guard + deterministic shard routing + Studio lifecycle guard**  
Eliminates HF API 429s and Lightning idle-timeout training loss while adding deterministic, reproducible ingestion sharding. Ships as two small, focused changes:

1. **Ingestion**: replace recursive `list_repo_files` with folder-level `list_repo_tree` + CDN download; project to `{prompt,response}` only; shard by hash into 5 sibling repos (640/hr aggregate cap).
2. **Training**: embed pre-listed file manifest (JSON) produced on Mac; Lightning training uses CDN-only fetches and guards against idle-stop by checking studio status before each run and restarting on L40S if stopped.

## Implementation Plan

### 1. Ingestion changes (`surrogate/ingest/`)

- Add `ingest_cdn.py`:
  - Single API call: `list_repo_tree(repo, path=date_folder, recursive=False)` → save `file_manifest.json`.
  - For each file: download via CDN URL `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth header).
  - Parse to `{prompt, response}`; drop other columns.
  - Deterministic shard: `repo_suffix = hash(slug) % 5`; upload to `enriched/batches/mirror-merged/{date}/{slug}.parquet` in sibling repo `{base}-shard-{i}`.
  - Retry on 429: wait 360s; exponential backoff for other errors.

- Remove/disable any `load_dataset(streaming=True)` or recursive `list_repo_files` usage.

### 2. Training changes (`surrogate/train/`)

- Add `prepare_manifest.py` (run on Mac):
  - Uses HF API (after rate-limit window) to produce `file_manifest.json` for chosen date folder.
  - Embeds manifest into training container or passes path to Lightning run.

- Update `train.py`:
  - Read `file_manifest.json`.
  - Use `hf_hub_download` per file (or direct CDN fetch) in DataLoader; zero HF API calls during training.
  - Before each training session, check studio status; if stopped, restart with `target.start(machine=Machine.L40S)`.

- Add `studio_guard.py`:
  - `def ensure_running(studio_name, machine=Machine.L40S)` that reuses existing running studio or starts new one on correct Lightning account (`lightning-lambda-prod` for H200, else `lightning-public-prod` for L40S).

### 3. Scripts & cron hygiene

- Ensure all wrapper scripts have `#!/usr/bin/env bash`, are `chmod +x`, invoked via `bash script "$@"`.
- Set `SHELL=/bin/bash` in crontab for any cron jobs (active-learning, opus-pr-reviewer patterns).

### surrogate/ingest/ingest_cdn.py
```python
#!/usr/bin/env python3
"""
CDN-first ingestion: list once, download via CDN, deterministic sharding.
"""
import json, hashlib, time, os
from pathlib import Path
import requests
from hugginggingface import HfApi  # pseudo import; use real hf_api package

API = HfApi()
CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"
SIBLINGS = [f"surrogate-enriched-shard-{i}" for i in range(5)]
DATE_FOLDER = os.getenv("INGEST_DATE", "2026-04-29")
REPO = "surrogate/mirror-merged"

def deterministic_shard(slug: str) -> str:
    return SIBLINGS[hash(slug) % len(SIBLINGS)]

def list_and_save_manifest(out_path: Path):
    tree = API.list_repo_tree(repo_id=REPO, path=DATE_FOLDER, recursive=False)
    files = [entry.path for entry in tree if entry.type == "file"]
    manifest = {"date": DATE_FOLDER, "files": files}
    out_path.write_text(json.dumps(manifest, indent=2))
    return files

def download_cdn(url: str, dest: Path, retries: int = 3):
    for attempt in range(retries):
        try:
            r = requests.get(url, timeout=30)
            r.raise_for_status()
            dest.write_bytes(r.content)
            return
        except Exception as e:
            if attempt == retries - 1:
                raise
            wait = 360 if getattr(e, "response", None) and e.response.status_code == 429 else (2 ** attempt)
            time.sleep(wait)

def project_to_pair(raw_bytes: bytes):
    # Placeholder: parse parquet/jsonl and return {"prompt":..., "response":...}
    # Keep only prompt/response fields; drop source/ts/other.
    import pandas as pd
    from io import BytesIO
    df = pd.read_parquet(BytesIO(raw_bytes))
    return {"prompt": df["prompt"].tolist(), "response": df["response"].tolist()}

def main():
    manifest_path = Path("file_manifest.json")
    files = list_and_save_manifest(manifest_path)

    for fpath in files:
        slug = Path(fpath).stem
        url = CDN_TEMPLATE.format(repo=REPO, path=fpath)
        dest = Path("raw") / fpath
        dest.parent.mkdir(parents=True, exist_ok=True)
        download_cdn(url, dest)

        pair = project_to_pair(dest.read_bytes())
        # Write enriched parquet with only prompt/response
        out_df = pd.DataFrame(pair)
        out_path = Path("enriched") / f"batches/mirror-merged/{DATE_FOLDER}/{slug}.parquet"
        out_path.parent.mkdir(parents=True, exist_ok=True)
        out_df.to_parquet(out_path, index=False)

        # Deterministic shard upload (pseudo)
        target_repo = deterministic_shard(slug)
        # API.upload_file(path_in_repo=out_path.relative_to("enriched"), repo_id=target_repo, ...)
        print(f"Shard {target_repo} <- {out_path}")

if __name__ == "__main__":
    main()
```

### surrogate/train/studio_guard.py
```python
#!/usr/bin/env python3
from lightning_sdk import Teamspace, Studio, Machine

def ensure_running(studio_name: str, machine: Machine = Machine.L40S) -> Studio:
    teamspace = Teamspace()
    for s in teamspace.studios:
        if s.name == studio_name and s.status == "running":
            return s
    # reuse stopped studio or create
    studio = Studio(name=studio_name, create_ok=True)
    if studio.status != "running":
        studio.start(machine=machine)
    return studio
```

### prepare_manifest.py (run on Mac)
```bash
#!/usr/bin/env bash
set -euo pipefail
# Run after rate-limit window clears; produces file_manifest.json for training.
python - <<'PY'
from hugginggingface import HfApi
import json, os
api = HfApi()
repo = "surrogate/mirror-merged"
date = os.getenv("INGEST_DATE", "2026-04-29")
tree = api.list_repo_tree(repo, path=date, recursive=False)
files = [e.path for e in tree if e.type == "file"]
with open("file_manifest.json", "w") as f:
    json.dump({"date": date, "files": files}, f, indent=2)
print("Manifest saved:", len(files))
```
