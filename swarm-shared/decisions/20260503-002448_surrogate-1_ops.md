# surrogate-1 / ops

## Implementation Plan — CDN-first snapshot + zero-HF-API ingestion

**Goal**: Eliminate HF API rate-limit risk during training by producing a deterministic file manifest once (on the Mac orchestrator) and having Lightning training fetch exclusively via CDN URLs.

**Scope** (fits <2h):
- Add `bin/snapshot.sh` → produces `manifests/{date}/files.json` (deterministic list of files for one date folder).
- Add `bin/curl-cdn.sh` → small utility to fetch a file via CDN (no auth).
- Update `bin/dataset-enrich.sh` to accept an optional manifest path and use CDN URLs when provided.
- Add `requirements-cdn.txt` note + one-line Python helper to stream from CDN URLs.
- Update README with usage.

### 1) `bin/snapshot.sh`

```bash
#!/usr/bin/env bash
# bin/snapshot.sh
# Usage: bin/snapshot.sh <date-folder> [out-dir]
# Example: bin/snapshot.sh 2026-04-29 manifests
#
# Requires:
#   HF_TOKEN (for single list_repo_tree call)
#   python3 + huggingface_hub
#
# Produces:
#   manifests/<date>/files.json  (sorted list of file paths)

set -euo pipefail
REPO="datasets/axentx/surrogate-1-training-pairs"
DATE="${1:-}"
OUTDIR="${2:-manifests}"

if [[ -z "$DATE" ]]; then
  echo "Usage: $0 <date-folder> [out-dir]"
  exit 1
fi

mkdir -p "${OUTDIR}/${DATE}"
OUT="${OUTDIR}/${DATE}/files.json"

python3 - "$REPO" "$DATE" "$OUT" <<'PY'
import json, os, sys
from huggingface_hub import HfApi

repo_id = sys.argv[1]
date_folder = sys.argv[2]
out_path = sys.argv[3]

api = HfApi(token=os.environ.get("HF_TOKEN"))
# Single non-recursive call per date folder (avoids 100x pagination)
tree = api.list_repo_tree(repo_id=repo_id, path=date_folder, recursive=False)
files = sorted([f.rfilename for f in tree if f.rfilename.endswith(('.jsonl', '.parquet', '.json'))])

with open(out_path, "w") as f:
    json.dump(files, f, indent=2)

print(f"Wrote {len(files)} files to {out_path}")
PY

echo "Snapshot complete: ${OUT}"
```

Make executable:
```bash
chmod +x bin/snapshot.sh
```

### 2) `bin/curl-cdn.sh` (optional helper)

```bash
#!/usr/bin/env bash
# bin/curl-cdn.sh
# Usage: bin/curl-cdn.sh <date-folder> <file> [output]
# Downloads a dataset file via public CDN (no auth).

set -euo pipefail
REPO="axentx/surrogate-1-training-pairs"
DATE="${1:-}"
FILE="${2:-}"
OUT="${3:--}"

if [[ -z "$DATE" || -z "$FILE" ]]; then
  echo "Usage: $0 <date-folder> <file> [output|-]"
  exit 1
fi

URL="https://huggingface.co/datasets/${REPO}/resolve/main/${DATE}/${FILE}"
if [[ "$OUT" == "-" ]]; then
  curl -fL --retry 3 "$URL"
else
  curl -fL --retry 3 -o "$OUT" "$URL"
fi
```

### 3) Python helper for CDN streaming (add to repo)

Create `lib/cdn_stream.py`:

```python
# lib/cdn_stream.py
import requests
from typing import Iterator, BinaryIO

CDN_TEMPLATE = "https://huggingface.co/datasets/axentx/surrogate-1-training-pairs/resolve/main/{date}/{file}"

def cdn_url(date: str, file: str) -> str:
    return CDN_TEMPLATE.format(date=date, file=file)

def stream_cdn(date: str, file: str, chunk_size: int = 8192) -> Iterator[bytes]:
    url = cdn_url(date, file)
    with requests.get(url, stream=True, timeout=30) as r:
        r.raise_for_status()
        for chunk in r.iter_content(chunk_size=chunk_size):
            if chunk:
                yield chunk

def copy_cdn_to_file(date: str, file: str, out_path: str) -> None:
    with open(out_path, "wb") as f:
        for chunk in stream_cdn(date, file):
            f.write(chunk)
```
