# Costinel / design

## Implementation Plan — Top Hub Signal Panel (CDN-first, frontend-only)

**Scope**: Add a lightweight, resilient Top Hub signal card to the Costinel dashboard that surfaces the most-connected hub (e.g., "MOC") with contextual insights.  
**Effort**: ~60–90 minutes (frontend only).  
**Mechanism**:  
- Single orchestration call (Mac/CI) to `list_repo_tree(path, recursive=False)` for one date folder → save `hub-index.json`.  
- Commit `hub-index.json` to repo (or serve via CDN).  
- Frontend fetches `hub-index.json` (CDN URL, no Authorization header) and renders the top hub card.  
- No backend changes, no runtime HF API calls, zero rate-limit exposure during training/dashboard use.

### 2) Add the Top Hub card component (React)

Create: `src/components/TopHubCard.jsx`

```tsx
import React, { useEffect, useState } from 'react';
import './TopHubCard.css';

export default function TopHubCard() {
  const [hub, setHub] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    // CDN-first fetch — no Authorization header required for public datasets/files
    fetch('/data/hub-index.json', { cache: 'no-store' })
      .then((res) => {
        if (!res.ok) throw new Error(`HTTP ${res.status}`);
        return res.json();
      })
      .then((data) => {
        const top = data.hubs?.find((h) => h.id === data.topHub) || data.hubs?.[0] || null;
        setHub({ ...top, generatedAt: data.generatedAt });
        setLoading(false);
      })
      .catch((err) => {
        setError(err.message);
        setLoading(false);
      });
  }, []);

  if (loading) return <div className="top-hub-card loading">Loading top hub…</div>;
  if (error) return <div className="top-hub-card error">Unable to load top hub</div>;
  if (!hub) return <div className="top-hub-card empty">No hub data</div>;

  return (
    <div className="top-hub-card">
      <div className="top-hub-card__header">
        <span className="top-hub-card__label">Top Hub</span>
        <span className="top-hub-card__pill">{hub.id}</span>
      </div>
      <div className="top-hub-card__body">
        <div className="top-hub-card__stat">
          <span className="top-hub-card__value">{hub.connections?.toLocaleString() ?? '—'}</span>
          <span className="top-hub-card__unit">connections</span>
        </div>
        <div className="top-hub-card__stat">
          <span className="top-hub-card__value">{hub.files ?? '—'}</span>
          <span className="top-hub-card__unit">files</span>
        </div>
      </div>
      <div className="top-hub-card__footer">
        <small>Updated {hub.latestFile ? `via ${hub.latestFile}` : ''}</small>
        <small className="top-hub-card__muted">{hub.generatedAt ? new Date(hub.generatedAt).toLocaleDateString() : ''}</small>
      </div>
    </div>
  );
}
```
