# Costinel / design

## Implementation Plan — `/api/v1/sense/top-hub-signal`

**Estimated effort:** <2h  
**Scope:** Add a single backend orchestration endpoint that senses top-hub signals and returns actionable insights without executing changes.

### Architecture (fits Costinel philosophy)
- **Sense**: Query knowledge-rag for the most-connected hub (e.g., "MOC") and related docs.
- **Signal**: Return enriched insights + recommendations payload (no execution).
- **Auditability**: Every request/response logged with trace-id.

### Stack
- Language: **Python 3.11+** (aligns with existing FastAPI patterns and surrogate-1 tooling).
- Framework: **FastAPI** (async, OpenAPI-first).
- RAG client: lightweight wrapper around HF CDN + local graph index (avoids API rate limits).
- Output: JSON with `{ hub, insights, recommendations, provenance }`.

```python
from fastapi import APIRouter, HTTPException
from pydantic import BaseModel
from typing import List, Optional
import httpx
import asyncio
import logging
import uuid
import os
from datetime import datetime

router = APIRouter()
log = logging.getLogger("costinel.top_hub_signal")

# CDN bypass (HF public datasets) — no auth, avoids /api/ rate limits
HF_DATASETS_BASE = "https://huggingface.co/datasets"
HF_REPO = os.getenv("KNOWLEDGE_RAG_REPO", "axentx/knowledge-rag")

class Insight(BaseModel):
    title: str
    summary: str
    score: float
    source: str
    related_paths: List[str]

class TopHubSignalResponse(BaseModel):
    request_id: str
    timestamp: str
    hub: str
    insights: List[Insight]
    recommendations: List[str]
    provenance: dict

async def fetch_cdn_index(date_folder: str = "latest") -> List[str]:
    """
    Pre-listed file paths are preferred; this is a safe fallback that uses CDN only.
    Returns list of parquet paths for the date folder.
    """
    url = f"{HF_DATASETS_BASE}/{HF_REPO}/resolve/main/{date_folder}/"
    try:
        async with httpx.AsyncClient(timeout=10.0) as client:
            r = await client.get(url)
            if r.status_code != 200:
                return []
            # crude HTML parse for .parquet links (CDN directory listing)
            import re
            return [f"{date_folder}/{m}" for m in re.findall(r'href="([^"]+\.parquet)"', r.text)]
    except Exception as exc:
        log.warning("CDN index fetch failed: %s", exc)
        return []

async def query_hub_from_parquet(paths: List[str], limit: int = 5) -> List[dict]:
    """
    Lightweight projection: read only required columns from parquet via CDN.
    Uses pyarrow.dataset to stream project without full schema validation.
    """
    import pyarrow.dataset as ds
    import pyarrow.parquet as pq
    results = []
    for p in paths[:10]:  # bound requests
        url = f"{HF_DATASETS_BASE}/{HF_REPO}/resolve/main/{p}"
        try:
            # project only needed fields; tolerate mixed schemas
            with pq.ParquetFile(url, memory_map=False) as pf:
                schema = pf.schema_arrow
                available = {f.name for f in schema}
                cols = [c for c in ["hub", "score", "title", "summary", "source"] if c in available]
                if "hub" not in available:
                    continue
                table = pf.read(columns=cols)
                for row in table.to_pylist():
                    results.append(row)
                    if len(results) >= limit:
                        return results
        except Exception as exc:
            log.debug("Skipped %s: %s", p, exc)
            continue
    return results

@router.get("/top-hub-signal", response_model=TopHubSignalResponse)
async def top_hub_signal(date_folder: Optional[str] = None):
    request_id = str(uuid.uuid4())
    ts = datetime.utcnow().isoformat() + "Z"
    folder = date_folder or "latest"

    try:
        paths = await fetch_cdn_index(folder)
        if not paths:
            # fallback: use pre-listed file manifest if available (preferred)
            manifest_url = f"{HF_DATASETS_BASE}/{HF_REPO}/resolve/main/{folder}/files.json"
            async with httpx.AsyncClient(timeout=10.0) as client:
                r = await client.get(manifest_url)
                if r.status_code == 200:
                    paths = r.json().get("files", [])

        rows = await query_hub_from_parquet(paths, limit=5)
        if not rows:
            raise HTTPException(status_code=404, detail="No hub signals found")

        top = max(rows, key=lambda x: x.get("score", 0))
        hub_name = top.get("hub", "MOC")

        insights = [
            Insight(
                title=r.get("title", "Untitled"),
                summary=r.get("summary", ""),
                score=float(r.get("score", 0)),
                source=r.get("source", "knowledge-rag"),
                related_paths=r.get("related_paths", [])
            )
            for r in rows[:5]
        ]

        recommendations = [
            f"Review governance policies related to '{hub_name}'",
            "Run cost-anomaly sweep for resources tagged with this hub",
            "Validate RI coverage for workloads in this hub"
        ]

        provenance = {
            "source_repo": HF_REPO,
            "date_folder": folder,
            "files_considered": len(paths),
            "cdn_only": True,
            "schema_projection": True
        }

        res = TopHubSignalResponse(
            request_id=request_id,
            timestamp=ts,
            hub=hub_name,
            insights=insights,
            recommendations=recommendations,
            provenance=provenance
        )

        log.info("top-hub-signal request_id=%s hub=%s", request_id, hub_name)
        return res

    except Exception as exc:
        log.exception("top-hub-signal failed request_id=%s", request_id)
        raise HTTPException(status_code=500, detail=str(exc))
```

### How it satisfies patterns & constraints

- **Sense + Signal (no Execute)**: Endpoint only reads and returns insights; no state change.
- **Knowledge-rag + top-hub**: Queries the most-connected hub and related docs via CDN-projected parquet.
- **HF CDN bypass**: Uses `resolve/main/` public URLs; avoids `/api/` auth and rate limits.
- **Schema projection**: Uses `pyarrow.parquet.ParquetFile` column projection to tolerate mixed schemas (surrogate-1 lesson).
- **Pre-list file paths**: Falls back to `files.json` manifest (single API call from Mac) so Lightning training can embed list and do CDN-only fetches.
- **Auditability**: Every request has `request_id` and structured provenance.
