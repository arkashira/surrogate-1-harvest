## Final synthesized plan (correct + actionable)

### 1. Diagnosis (merged, corrected)
- **No idempotency enforcement** on LINE webhook ingestion → duplicates on retry.
- **Race-prone upsert** (`findOne` → conditional write) → duplicates under concurrency.
- **Missing DB uniqueness constraint** → duplicates possible at the row level.
- **Handler trusts client timestamps** → backdating/spoofing possible.
- **No replay-protection key storage** → no reliable way to deduplicate.
- *(Rejected from Candidate 2)* Using `X-Line-Signature` as a uniqueness key is incorrect; use a stable LINE event identifier or content-derived key plus `tenant_id`.

### 2. One-sentence goal
Make LINE webhook ingestion idempotent and race-safe by adding DB uniqueness constraints and an atomic idempotency check that is immune to retries and concurrent delivery.

### 3. Implementation (single, concrete plan)

#### 3.1 DB schema (run once)

-- Idempotency table for LINE events
CREATE TABLE IF NOT EXISTS processed_webhooks (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  tenant_id       UUID NOT NULL REFERENCES tenants(id) ON DELETE CASCADE,
  line_event_id   TEXT NOT NULL,
  event_type      TEXT NOT NULL,
  processed_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  CONSTRAINT uq_line_event_per_tenant UNIQUE (tenant_id, line_event_id, event_type)

-- Prevent duplicate punches at DB level

ON attendance (user_id, event_type, event_timestamp, tenant_id)
WHERE deleted_at IS NULL;

#### 3.2 Handler (TypeScript) — replace `/opt/axentx/workio/server/src/routes/webhook/line.ts`
Key points:
- Verify signature with raw body.
- Derive stable `line_event_id` from the event (not from signature).
- Use `INSERT ... ON CONFLICT DO NOTHING` into `processed_webhooks` as the atomic idempotency gate.
- Clamp event timestamps to ±5 min server skew; default to server time when out of window.
- Perform attendance insert with `ON CONFLICT DO NOTHING` under the same transaction.

```ts
import { Request, Response } from 'express';
import { pool } from '../../db';
import crypto from 'crypto';

const MAX_CLOCK_SKEW_MS = 5 * 60 * 1000; // 5m

function verifyLineSignature(body: string, signature: string, channelSecret: string): boolean {
  const expected = crypto
    .createHmac('sha256', channelSecret)
    .update(body)
    .digest('base64');
  return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
}

export async function lineWebhookHandler(req: Request, res: Response) {
  const signature = req.get('X-Line-Signature');
  const channelSecret = process.env.LINE_CHANNEL_SECRET!;
  const rawBody = JSON.stringify(req.body);

  if (!signature || !verifyLineSignature(rawBody, signature, channelSecret)) {
    return res.status(401).json({ error: 'Invalid signature' });
  }

  const events = req.body.events;
  if (!Array.isArray(events) || events.length === 0) {
    return res.status(200).json({ ok: true });
  }

  const client = await pool.connect();
  try {
    await client.query('BEGIN');

    for (const ev of events) {
      // Stable, content-derived id (not signature)
      const lineEventId = ev['webhookEventId'] ||
        (ev.source?.userId && ev.timestamp
          ? `${ev.source.userId}:${ev.type}:${ev.timestamp}`
          : crypto.createHash('sha256').update(JSON.stringify(ev)).digest('hex'));

      const eventType = ev.type;
      const tenantId = process.env.DEFAULT_TENANT_ID; // derive properly in prod

      // Atomic idempotency check
      const idem = await client.query(
        `INSERT INTO processed_webhooks (tenant_id, line_event_id, event_type)
         VALUES ($1, $2, $3)
         ON CONFLICT (tenant_id, line_event_id, event_type) DO NOTHING
         RETURNING id`,
        [tenantId, lineEventId, eventType]
      );

      if (idem.rowCount === 0) continue; // already processed

      // Example: clock in/out via postback
      if (eventType === 'postback' && ev.postback?.data) {
        const payload = Object.fromEntries(new URLSearchParams(ev.postback.data));
        if (payload.action === 'clock') {
          const now = new Date();
          const clientTs = payload.ts ? new Date(parseInt(payload.ts, 10)) : now;
          const skew = Math.abs(now.getTime() - clientTs.getTime());
          const eventTime = skew <= MAX_CLOCK_SKEW_MS ? clientTs : now;

          await client.query(
            `INSERT INTO attendance (user_id, tenant_id, event_type, event_timestamp, created_at, source)
             VALUES ($1, $2, $3, $4, NOW(), $5)
             ON CONFLICT (user_id, event_type, event_timestamp, tenant_id) DO NOTHING`,
            [ev.source.userId, tenantId, 'clock', eventTime.toISOString(), 'line']
          );
        }
      }
    }

    await client.query('COMMIT');
    res.status(200).json({ ok: true });
  } catch (err) {
    await client.query('ROLLBACK');
    console.error('LINE webhook handler error:', err);
    res.status(500).json({ error: 'Internal server error' });
  } finally {
    client.release();
  }
}
```
