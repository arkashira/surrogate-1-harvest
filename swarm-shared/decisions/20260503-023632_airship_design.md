# airship / design

## Final Actionable Plan (Synthesized)

**Goal:** Eliminate HF API 429 during Surrogate training within ≤2 hours.  
**Scope:** `/opt/axentx/airship/surrogate` — add ingestion + training orchestration.

---

### 1) Deterministic Ingestion (single HF API call)

- **One listing**: `list_repo_tree(repo, path=f"raw/{date}", recursive=False)` → save `filelist-{date}.json`.
- **Projection**: per file, stream-parse and keep only `{prompt, response}` (tolerate `input/question` → `prompt`, `output/answer` → `response`).
- **Schema hygiene**: do **not** add extra metadata columns to parquet; store attribution in filename/slug only.
- **Write sharding**: deterministic hash(slug) → pick one of 5 sibling repos (`axentx/surrogate-mirror-0..4`) for `batches/mirror-merged/{date}/{slug}.parquet`.
- **Upload**: use HF API once per shard (after local parquet build) to avoid schema mixing and rate-limit churn.

---

### 2) CDN-Only Training Manifest

- Convert each uploaded parquet to **CDN URL**:  
  `https://huggingface.co/datasets/{repo}/resolve/main/batches/mirror-merged/{date}/{slug}.parquet`
- Emit `train_config.json` containing:
  - `date`
  - `shards: [{slug, repo, path, cdn_url}]`
  - `cdn_only: true`
- Training must consume these URLs directly (no HF API calls, no `streaming=True` on mixed schemas).

---

### 3) Lightning Studio Reuse + Idle Guard

- **Reuse first**: enumerate `Teamspace.studios`; if a studio with matching name exists and is running, reuse it.
- **Restart if stopped**: call `studio.start(machine=Machine.L40S)` (L40S as reliable fallback; H200 only if available and preferred).
- **Idle protection**: before long runs, ensure studio is active; wrap `.run()` with status check and restart-if-stopped logic.
- **Cron hygiene**: use `SHELL=/bin/bash` and `chmod +x` on wrapper script; invoke via `bash` to avoid sh/dash surprises.

---

### 4) Files to Add/Modify

- `surrogate/scripts/build_shards.py` — deterministic listing, projection, parquet, upload, manifest.
- `surrogate/train.py` — accept `--config train_config.json`; use CDN URLs via `fsspec` + `pyarrow` (direct parquet reads).
- `surrogate/launch_studio.py` — reuse or start studio with idle/stop guard.
- `surrogate/cron/run_training.sh` — executable wrapper for cron.

#### `surrogate/scripts/build_shards.py`
```python
#!/usr/bin/env python3
import os, json, hashlib
from pathlib import Path
from datetime import datetime

from huggingface_hub import HfApi, hf_hub_download, list_repo_tree
import pyarrow as pa
import pyarrow.parquet as pq
import pandas as pd

HF_REPO = os.getenv("HF_DATASET_REPO", "axentx/surrogate-mirror")
DATE_FOLDER = os.getenv("DATE_FOLDER", datetime.utcnow().strftime("%Y-%m-%d"))
OUTPUT_DIR = Path(__file__).parent.parent / "data" / "mirror-merged" / DATE_FOLDER
OUTPUT_DIR.mkdir(parents=True, exist_ok=True)

SIBLING_REPOS = [f"axentx/surrogate-mirror-{i}" for i in range(5)]
api = HfApi()

def pick_repo(slug: str) -> str:
    h = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    return SIBLING_REPOS[h % len(SIBLING_REPOS)]

def normalize_record(obj) -> dict:
    prompt = obj.get("prompt") or obj.get("input") or obj.get("question") or ""
    response = obj.get("response") or obj.get("output") or obj.get("answer") or ""
    return {"prompt": str(prompt), "response": str(response)}

def build():
    items = list_repo_tree(repo_id=HF_REPO, path=f"raw/{DATE_FOLDER}", recursive=False)
    filelist = []
    shards = []

    for item in items:
        if not item.rfilename.endswith((".json", ".jsonl", ".parquet")):
            continue
        filelist.append(item.rfilename)
        local_path = hf_hub_download(repo_id=HF_REPO, filename=item.rfilename)
        slug = Path(item.rfilename).stem

        rows = []
        if local_path.endswith(".parquet"):
            table = pq.read_table(local_path)
            df = table.to_pandas()
            for _, row in df.iterrows():
                rows.append(normalize_record(row.to_dict()))
        else:
            with open(local_path, "r", encoding="utf-8") as f:
                if local_path.endswith(".jsonl"):
                    for line in f:
                        line = line.strip()
                        if line:
                            rows.append(normalize_record(json.loads(line)))
                else:
                    obj = json.load(f)
                    rows.append(normalize_record(obj))

        if not rows:
            continue

        df_norm = pd.DataFrame(rows)
        out_name = f"{slug}.parquet"
        target_repo = pick_repo(slug)
        rel_path = f"batches/mirror-merged/{DATE_FOLDER}/{out_name}"
        out_file = OUTPUT_DIR / out_name
        df_norm.to_parquet(out_file, index=False)

        api.upload_file(
            path_or_fileobj=str(out_file),
            path_in_repo=rel_path,
            repo_id=target_repo,
        )

        cdn_url = f"https://huggingface.co/datasets/{target_repo}/resolve/main/{rel_path}"
        shards.append({"slug": slug, "repo": target_repo, "path": rel_path, "cdn_url": cdn_url})

    (OUTPUT_DIR.parent / f"filelist-{DATE_FOLDER}.json").write_text(json.dumps(filelist, indent=2))
    train_config = {
        "date": DATE_FOLDER,
        "shards": shards,
        "cdn_only": True,
        "created_at": datetime.utcnow().isoformat() + "Z",
    }
    (OUTPUT_DIR.parent / f"train_config-{DATE_FOLDER}.json").write_text(json.dumps(train_config, indent=2))
    print(f"Built {len(shards)} shards for {DATE_FOLDER}")

if __name__ == "__main__":
    build()
```

#### `surrogate/train.py` (CDN-only dataloader excerpt)
```python
import json
import pyarrow.parquet as pq
from torch.utils.data import IterableDataset, DataLoader
import fsspec

class CDNParquetDataset(IterableDataset):
    def __init__(self, cdn_urls):
        self.cdn_urls = cdn_urls

    def __iter__(self):
        for url in self.cdn_urls:
            with fsspec.open(url, "rb") as f:
                table = pq.read_table(f)
                df = table.to_pandas()
                for _, row in df.iterrows():
                    yield {"prompt": row["prompt"], "response": row["response"]}

def make_dataloader(config_path):
    cfg = json.load(open(config_path))
    urls = [s["cdn_url"] for s in cfg["shards"]]
    ds = CDNParquetDataset(urls)
    return DataLoader(ds, batch_size=8, num_workers=0)
```
