# airship / design

## Final Consolidated Implementation Plan (≤2h)

**Highest-value improvement:** Deterministic CDN file manifest + Lightning Studio lifecycle resilience for Surrogate-1 training.  
**Why:** Eliminates HF API 429s during Surrogate-1 training and prevents quota loss from idle-stop/Studio recreation loops.

---

### 1) Pre-list date folder → JSON manifest (Mac orchestration)
- **Single API call:** `list_repo_tree(path, recursive=False)` per date folder → save `manifests/{date}/file-list.json`.  
- **CDN-only training:** `train.py` reads the manifest and fetches files via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` with **zero** API calls during data loading.  
- **Fallback:** If manifest missing, generate once and cache.

### 2) Lightning Studio lifecycle guard
- **Reuse before create:** Before each `.run()`, check `Teamspace.studios` for an existing **Running** studio by name; reuse if found.  
- **Restart on stop:** If stopped, restart with `target.start(machine=Machine.L40S)` (or fallback to available cloud account).  
- **Auto-recover from idle-stop:** Wrap training entrypoint in a loop that checks status and auto-restarts on idle-stop (max 3 retries).

### 3) Schema-safe ingestion wrapper
- **Per-file download:** Use `hf_hub_download` per file → project to `{prompt, response}` only at parse time.  
- **Output:** Write to `batches/mirror-merged/{date}/{slug}.parquet` (no `source`/`ts` columns).  
- **Deterministic sibling selection:** Hash slug → pick sibling repo index to bypass 128/hr commit cap.

### `scripts/generate_manifest.py` (Mac orchestration)
```python
#!/usr/bin/env python3
"""
Generate deterministic CDN manifest for a date folder.
Run from Mac (or any orchestrator) before Lightning training.
"""
import json, os, sys
from datetime import datetime
from huggingface_hub import HfApi

API = HfApi()
REPO = "axentx/surrogate-datasets"
DATE = sys.argv[1] if len(sys.argv) > 1 else datetime.utcnow().strftime("%Y-%m-%d")
FOLDER = f"batches/mirror-merged/{DATE}"
OUT_DIR = "manifests"
os.makedirs(OUT_DIR, exist_ok=True)

# Single API call: non-recursive tree listing
tree = API.list_repo_tree(REPO, path=FOLDER, recursive=False)
files = [item.path for item in tree if item.type == "file"]

manifest = {
    "repo": REPO,
    "date": DATE,
    "folder": FOLDER,
    "files": sorted(files),
    "generated_at": datetime.utcnow().isoformat() + "Z",
    "cdn_base": f"https://huggingface.co/datasets/{REPO}/resolve/main"
}

out_path = os.path.join(OUT_DIR, f"{DATE}.json")
with open(out_path, "w") as f:
    json.dump(manifest, f, indent=2)
print(f"Manifest written: {out_path} ({len(files)} files)")
```

### `surrogate/train.py` (Lightning training, CDN-only)
```python
import json, os, pyarrow.parquet as pq, pyarrow as pa
from pathlib import Path
from torch.utils.data import IterableDataset, DataLoader
import requests

MANIFEST_PATH = os.getenv("MANIFEST_PATH", "manifests/latest.json")
CDN_BASE = None

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path):
        with open(manifest_path) as f:
            m = json.load(f)
        global CDN_BASE
        CDN_BASE = m["cdn_base"]
        self.files = m["files"]

    def __iter__(self):
        for rel in self.files:
            url = f"{CDN_BASE}/{rel}"
            # Stream download without HF API auth/rate-limit
            with requests.get(url, stream=True, timeout=60) as r:
                r.raise_for_status()
                # project to {prompt, response} only
                tbl = pq.read_table(pa.BufferReader(r.content), columns=["prompt", "response"])
                for batch in tbl.to_batches(max_chunksize=1024):
                    df = batch.to_pydict()
                    for prompt, response in zip(df["prompt"], df["response"]):
                        yield {"prompt": prompt, "response": response}

def get_dataloader(manifest_path=MANIFEST_PATH, batch_size=8, num_workers=4):
    ds = CDNParquetDataset(manifest_path)
    return DataLoader(ds, batch_size=batch_size, num_workers=num_workers)
```

### `surrogate/lightning_launcher.py` (Studio lifecycle guard)
```python
#!/usr/bin/env python3
"""
Lightning Studio launcher with lifecycle resilience.
Reuses running studios; restarts on idle-stop.
"""
import time
import lightning as L
from lightning.pytorch.cli import LightningCLI

SERVICE_NAME = "surrogate-train"
MACHINE = L.Machine.L40S  # fallback: L40S; H200 requires lightning-lambda-prod

def get_or_create_studio():
    teamspace = L.Teamspace()
    for s in teamspace.studios:
        if s.name == SERVICE_NAME:
            if s.status == "running":
                print(f"Reusing running studio: {s.name}")
                return s
            else:
                print(f"Studio stopped ({s.status}). Restarting...")
                s.start(machine=MACHINE)
                return s
    print("Creating new studio...")
    return L.Studio(
        name=SERVICE_NAME,
        machine=MACHINE,
        create_ok=True
    )

def resilient_run(train_script_path, max_restarts=3):
    restarts = 0
    while restarts <= max_restarts:
        studio = get_or_create_studio()
        try:
            # Run training; if studio stops, this will raise/exit
            studio.run(str(train_script_path))
            print("Training completed.")
            break
        except Exception as exc:
            restarts += 1
            print(f"Training interrupted ({exc}). Restart {restarts}/{max_restarts}")
            if restarts > max_restarts:
                raise
            time.sleep(10)

if __name__ == "__main__":
    resilient_run("train.py")
```

#!/usr/bin/env python3
"""
Ingest heterogeneous HF files -> {prompt,response} parquet.
Uses hf_hub_download per file to avoid mixed-schema CastError.
"""
import hashlib, json, os, pyarrow as pa, pyarrow.parquet as pq
from huggingface_hub import hf_hub_download, HfApi
from datetime import datetime

API = HfApi()
REPO = "axentx/surrogate-datasets"
SIBLINGS = 5  # commit-cap bypass

def pick_sibling(slug: str) -> int:
    return int(hashlib.sha256(slug.encode()).hexdigest(), 16) % SIBLINGS

def project_file(local_path):
    tbl = pq.read_table(local_path, columns=["prompt", "response"])
    return tbl.select(["prompt", "response"])
