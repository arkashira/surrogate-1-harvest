## Highest-Value Incremental Improvement (<2h)

**Goal**: Eliminate HF API 429s during Surrogate-1 training by replacing `load_dataset(streaming=True)` with a CDN-only `IterableDataset` that reads from a pre-generated `file_list.json`. This removes pagination, auth-checks, and rate-limit exposure during training.

**Why this ships fast**:
- Single focused change in the surrogate training loader (~150 LoC).
- No infra changes, no GPU quota changes, no new dependencies.
- Uses existing pattern: pre-list once on Mac, embed list, Lightning fetches via CDN only.
- Immediately unlocks long-running training jobs without 429 stalls.

---

## Implementation Plan

1. **Locate training loader**  
   Find where Surrogate-1 loads HF datasets (likely `surrogate/train.py` or `surrogate/data/dataset.py`).

2. **Create CDN-only IterableDataset**  
   - Accept `repo_id`, `folder_path`, and `file_list.json` path.
   - On init: load `file_list.json` (generated once on Mac via `list_repo_tree`).
   - On iteration: stream each file via `urllib`/`requests` from `https://huggingface.co/datasets/{repo}/resolve/main/{path}`.
   - Parse bytes into `{prompt, response}` only at yield time (avoids schema issues).
   - No `load_dataset`, no `streaming=True`, no HF client during training.

3. **Generate file_list.json (one-time, outside training)**  
   - Script: `scripts/generate_file_list.py`
   - Uses HF API **once** (after rate-limit window) to call `list_repo_tree(path, recursive=False)` for a date folder.
   - Saves list of relative paths to `file_list.json` committed or embedded in training config.

4. **Update training script**  
   - Replace `load_dataset(...)` with `CdnIterableDataset(file_list="file_list.json", repo=..., folder=...)`.
   - Ensure `DataLoader` uses `num_workers>=1` and `pin_memory=True` for throughput.

5. **Add resilience**  
   - Retry with exponential backoff on CDN fetch (429/5xx).
   - Validate shard integrity (optional: quick length check on response).
   - If studio stops, do not recreate; reuse running studio (existing pattern).

6. **Test quickly**  
   - Dry-run over 100 files locally (Mac) to verify parsing and no HF auth calls.
   - Launch Lightning Studio (L40S) with updated train.py; confirm zero HF API calls during first epoch.

### 1. CDN-only IterableDataset (surrogate/data/cdn_dataset.py)

```python
# surrogate/data/cdn_dataset.py
import json
import urllib.request
import time
import random
from typing import Iterator, Dict, Any
from torch.utils.data import IterableDataset

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

class CdnIterableDataset(IterableDataset):
    def __init__(
        self,
        repo: str,
        folder: str,
        file_list_path: str,
        max_retries: int = 5,
        backoff_base: float = 1.5,
    ):
        self.repo = repo
        self.folder = folder.rstrip("/")
        with open(file_list_path) as f:
            self.files = [p for p in json.load(f) if p.startswith(self.folder)]
        self.max_retries = max_retries
        self.backoff_base = backoff_base

    def _fetch(self, path: str) -> bytes:
        url = CDN_TEMPLATE.format(repo=self.repo, path=path)
        for attempt in range(self.max_retries):
            try:
                with urllib.request.urlopen(url, timeout=30) as resp:
                    if resp.status == 200:
                        return resp.read()
                    # Non-200 -> retry
            except Exception:
                pass
            sleep = self.backoff_base ** attempt + random.uniform(0, 1)
            time.sleep(sleep)
        raise RuntimeError(f"Failed to fetch {url} after {self.max_retries} attempts")

    def _parse_to_pair(self, raw: bytes) -> Dict[str, str]:
        # Project to {prompt, response} only; adapt to your file format.
        # Example: assume JSON lines with 'prompt' and 'response'
        import io, json as _json
        try:
            obj = _json.loads(raw.decode("utf-8"))
            return {"prompt": obj["prompt"], "response": obj["response"]}
        except Exception:
            # Fallback: best-effort split for plain text pairs
            text = raw.decode("utf-8", errors="replace")
            # Customize this heuristic to your corpus
            if "\n### Response:\n" in text:
                prompt, response = text.split("\n### Response:\n", 1)
                return {"prompt": prompt.strip(), "response": response.strip()}
            return {"prompt": "", "response": text.strip()}

    def __iter__(self) -> Iterator[Dict[str, str]]:
        for path in self.files:
            raw = self._fetch(path)
            yield self._parse_to_pair(raw)
```
