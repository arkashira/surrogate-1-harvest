# Costinel / design

## Final Synthesis — Top-Hub Signal Panel (CDN-first, <2h)

**Chosen approach:**  
A non-blocking Top-Hub Signal Panel that surfaces the most-connected hub (e.g., “MOC”) using CDN-first data baked at build/orchestration time. Runtime dashboard loads make **zero HF API calls**, fail gracefully with local cache + staleness indicator, and emit a lightweight audit trail.

---

### 1) Concrete, timeboxed implementation plan (≤2h)

| Step | Owner | Time | Deliverable |
|------|-------|------|-------------|
| 1 | FE | 15m | Create `TopHubSignalPanel` (React) under `src/components/dashboard/` with skeleton → data → stale indicator |
| 2 | BE | 20m | Add `/api/signals/top-hub` endpoint returning `{signal_id,hub,slug,updated_at,source_file,status}`; reads from baked CDN file; never calls HF API at runtime |

| 5 | FE | 20m | Wire panel into dashboard with lazy load; fallback to `public/data/signals/top-hub.json`; show staleness badge and toast on persistent CDN failure |

### 2) File changes (minimal, production-ready)

#### 2.1) Backend endpoint: `src/routes/signals/topHubRoute.js`

```js
import express from 'express';
import fs from 'fs/promises';
import path from 'path';
import { fileURLToPath } from 'url';

const __dirname = path.dirname(fileURLToPath(import.meta.url));
const router = express.Router();

const LOCAL_PAYLOAD_PATH = path.join(__dirname, '../../../public/data/signals/top-hub.json');
const AUDIT_LOG_PATH = path.join(__dirname, '../../../logs/signals/top-hub-audit.log');

function audit(entry) {
  return fs.appendFile(AUDIT_LOG_PATH, JSON.stringify(entry) + '\n').catch(() => {});
}

// GET /api/signals/top-hub
router.get('/top-hub', async (req, res) => {
  try {
    const raw = await fs.readFile(LOCAL_PAYLOAD_PATH, 'utf8').catch(() => null);
    if (!raw) throw new Error('top-hub payload missing');

    const payload = JSON.parse(raw);
    const signalId = `top-hub:${payload.slug || 'none'}:${Date.now()}`;

    await audit({
      signal_id: signalId,
      hub: payload.hub,
      slug: payload.slug,
      ts: new Date().toISOString(),
      status: 'ok',
      source_file: payload.source_file
    });

    return res.status(200).json({
      signal_id: signalId,
      hub: payload.hub,
      slug: payload.slug,
      updated_at: payload.updated_at,
      source_file: payload.source_file,
      status: 'ok'
    });
  } catch (err) {
    const signalId = `top-hub:error:${Date.now()}`;
    await audit({
      signal_id: signalId,
      hub: null,
      slug: null,
      ts: new Date().toISOString(),
      status: 'error',
      note: err.message
    });

    // Never 5xx — degrade gracefully
    return res.status(200).json({
      signal_id: signalId,
      hub: null,
      slug: null,
      updated_at: null,
      source_file: null,
      status: 'stale',
      note: 'top-hub unavailable'
    });
  }
});

// Health check
router.get('/health/top-hub', async (req, res) => {
  const ok = await fs.readFile(LOCAL_PAYLOAD_PATH).then(() => true).catch(() => false);
  return res.status(200).json({ ok, source: 'local-baked' });
});

export default router;
```
