# airship / backend

## Highest-Value Incremental Improvement (≤2h)

**Goal**: Harden `airship discover` into a deterministic, **CDN-only orchestrator** that eliminates HF API rate limits and guarantees reproducible training inputs.

### Why this is highest-value
- Directly unblocks surrogate training pipeline (HF 429 → CDN bypass)
- Enables Lightning Studio reuse (quota savings)
- One small CLI change + one training script guard → immediate ROI

---

## Implementation Plan

### 1) CLI: `airship discover` → CDN-only file listing
- Single API call to `list_repo_tree` per date folder (non-recursive)
- Persist `file_list.json` with CDN-resolvable paths only
- Exit early if listing already exists and is <24h old (deterministic)

### 2) Training script guard: Lightning Studio reuse + idle restart
- Before `.run()`, check for existing running studio by name
- If stopped, restart with `target.start(machine=Machine.L40S)`
- Embed `file_list.json` so data loader uses CDN URLs only (zero API calls during training)

### 3) Data loader: CDN-only fetch
- Replace `load_dataset(...)` with direct `wget`/`requests` from  
  `https://huggingface.co/datasets/{repo}/resolve/main/{path}`
- Project to `{prompt, response}` at parse time (ignore mixed schemas)

## Code Snippets

### `airship/cli/discover.py` (CDN-only listing)
```python
#!/usr/bin/env python3
import json, os, time, datetime
from pathlib import Path
from huggingface_hub import HfApi

API = HfApi()
REPO = "axentx/surrogate-dataset-mirror"
OUT_DIR = Path("var/discover")
OUT_DIR.mkdir(parents=True, exist_ok=True)

def list_date_folder(date_str: str):
    """List one date folder non-recursively; save CDN file list."""
    out_file = OUT_DIR / f"{date_str}_file_list.json"
    if out_file.exists() and (time.time() - out_file.stat().st_mtime) < 86400:
        print(f"Fresh listing exists: {out_file}")
        return json.loads(out_file.read_text())

    items = API.list_repo_tree(repo_id=REPO, path=date_str, recursive=False)
    files = [
        {
            "cdn_url": f"https://huggingface.co/datasets/{REPO}/resolve/main/{it['path']}",
            "repo_path": it["path"],
            "size": it.get("size", 0),
        }
        for it in items
        if it.get("type") == "file" and it["path"].endswith(".parquet")
    ]
    out_file.write_text(json.dumps(files, indent=2))
    print(f"Saved {len(files)} files -> {out_file}")
    return files

if __name__ == "__main__":
    today = datetime.date.today().isoformat()
    list_date_folder(today)
```

### `surrogate/train.py` (Lightning Studio reuse + CDN fetch)
```python
#!/usr/bin/env python3
import json, os
from pathlib import Path
from lightning import Lightning, Teamspace, Machine, Studio

LIGHTNING = Lightning()
TEAMSPACE = Teamspace.current()
STUDIO_NAME = os.getenv("STUDIO_NAME", "surrogate-train-l40s")
MACHINE = Machine.L40S

def get_or_create_studio():
    for s in TEAMSPACE.studios:
        if s.name == STUDIO_NAME:
            if s.status == "running":
                print(f"Reusing running studio: {s.name}")
                return s
            else:
                print(f"Restarting stopped studio: {s.name}")
                s.start(machine=MACHINE)
                return s
    print(f"Creating new studio: {STUDIO_NAME}")
    return TEAMSPACE.studios.create(
        name=STUDIO_NAME,
        machine=MACHINE,
        create_ok=True,
    )

def load_cdn_file_list():
    listing = Path("var/discover") / "latest_file_list.json"
    if not listing.exists():
        listing = sorted(Path("var/discover").glob("*_file_list.json"))[-1]
    return json.loads(listing.read_text())

def build_cdn_dataset(files):
    # Lightweight generator: CDN-only, zero HF API calls during training
    import pyarrow.parquet as pq
    import requests
    for f in files:
        resp = requests.get(f["cdn_url"], timeout=30)
        resp.raise_for_status()
        tbl = pq.read_table(pq.ParquetFile(resp.content))
        # Project to {prompt, response} only
        yield from tbl.select(["prompt", "response"]).to_pylist()

if __name__ == "__main__":
    studio = get_or_create_studio()
    files = load_cdn_file_list()
    dataset = list(build_cdn_dataset(files))
    print(f"Loaded {len(dataset)} samples via CDN")
    # Continue with Lightning training run...
```
