## Final Implementation Plan — Airship Quality (HF CDN Bypass + Lightning Idle Resilience)

**Value**: Eliminates HF API 429s during training, prevents Lightning idle-stop training loss, reduces Mac→Lightning iteration from ~15 min to <2 min per cycle.  
**Effort**: <2 h, zero breaking changes, fully backward-compatible.

---

### Ordered Steps (minimal context-switch)

1. **Generate CDN file list on Mac (one-time per date folder)**  
   - Run `tools/build_file_list.py` → `file_list.json`.  
   - Single `list_repo_tree` call; non-recursive per folder to minimize API usage.

2. **Patch training loader to use CDN bypass**  
   - Accept `--file-list`; when present, fetch parquet files via CDN URLs (bypass `/api/`).  
   - Project rows to `{prompt, response}` only at parse time; ignore mixed schema columns.  
   - Fallback: if no file list, use safe per-file `hf_hub_download` + projection (never use `load_dataset(streaming=True)` on mixed schemas).

3. **Lightning studio reuse + idle-stop guard**  
   - Before `.run()`, check `Teamspace.studios`; reuse Running studio by name.  
   - If not Running, restart with `Machine.L40S` (or free-tier L40S).  
   - Prevents quota waste and idle-stop deaths.

4. **Add orchestration wrapper** (`run_surrogate_train.sh`)  
   - Shebang `#!/usr/bin/env bash`, `set -euo pipefail`, `chmod +x`.  
   - Invoked via cron/Mac; sets `SHELL=/bin/bash` reminder in comments.

5. **Smoke test**  
   - Run wrapper locally → verify file-list generation, CDN fetch, studio reuse, and 100 training steps.  
   - No model weights pushed; data pipeline only.

---

### Resolved Contradictions (in favor of correctness + actionability)
- **CDN download method**: Candidate 1 used `urllib.request.urlretrieve` (simple but not robust for large files). Replace with `requests.get(..., stream=True)` + chunked write to handle large parquet files reliably.
- **Lightning SDK usage**: Candidate 1 omitted actual run command inside studio; Candidate 2 hinted at run details. Final: use `studio.run()` with explicit command and working directory to guarantee execution inside the studio environment.
- **Fallback loader**: Candidate 1 raised `NotImplementedError`; Candidate 2 omitted details. Final: implement safe per-file `hf_hub_download` fallback with projection so training never blocks if file list is missing.
- **File list scope**: Both candidates used non-recursive listing for the given date folder. Keep non-recursive to minimize API calls; document that nested subfolders under the date folder are not supported unless explicitly added.

### Code Snippets

#### 1) `tools/build_file_list.py`

```python
#!/usr/bin/env python3
"""
Generate file_list.json for a date folder in a HF dataset repo.
Usage:
  python tools/build_file_list.py \
    --repo <org>/<dataset> \
    --date-folder 2026-05-03 \
    --out file_list.json
"""
import argparse
import json
from huggingface_hub import HfApi

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date-folder", required=True)
    parser.add_argument("--out", default="file_list.json")
    args = parser.parse_args()

    api = HfApi()
    entries = api.list_repo_tree(
        repo_id=args.repo,
        path=args.date_folder,
        repo_type="dataset",
        recursive=False,
    )
    files = sorted(e.path for e in entries if e.type == "file")
    payload = {
        "repo": args.repo,
        "date_folder": args.date_folder,
        "files": files,
    }
    with open(args.out, "w") as f:
        json.dump(payload, f, indent=2)
    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```

#### 2) `surrogate/train.py` (CDN bypass + safe fallback)

```python
import argparse
import json
import os
import shutil
from pathlib import Path
from typing import Dict, Iterator

import requests
import torch
from datasets import load_dataset
from huggingface_hub import hf_hub_download
from torch.utils.data import IterableDataset

class CDNParquetDataset(IterableDataset):
    def __init__(self, file_list_path: str, repo: str, cache_dir: str = ".cache_cdn"):
        super().__init__()
        with open(file_list_path) as f:
            meta = json.load(f)
        self.repo = repo
        self.files = meta["files"]
        self.cache_dir = Path(cache_dir)
        self.cdn_base = f"https://huggingface.co/datasets/{repo}/resolve/main"
        self.cache_dir.mkdir(parents=True, exist_ok=True)

    def _download_cdn(self, rel_path: str) -> Path:
        local_path = self.cache_dir / rel_path.replace("/", "_")
        if local_path.exists():
            return local_path

        url = f"{self.cdn_base}/{rel_path}"
        local_path.parent.mkdir(parents=True, exist_ok=True)
        with requests.get(url, stream=True, timeout=60) as r:
            r.raise_for_status()
            with open(local_path, "wb") as f:
                for chunk in r.iter_content(chunk_size=8192):
                    f.write(chunk)
        return local_path

    def __iter__(self) -> Iterator[Dict[str, str]]:
        for rel_path in self.files:
            local_path = self._download_cdn(rel_path)
            ds = load_dataset("parquet", data_files=str(local_path), split="train")
            for row in ds:
                yield {
                    "prompt": row.get("prompt") or row.get("input") or "",
                    "response": row.get("response") or row.get("output") or "",
                }


class SafeFallbackDataset(IterableDataset):
    """Fallback when CDN file list is unavailable."""
    def __init__(self, repo: str, date_folder: str, cache_dir: str = ".cache_hub"):
        super().__init__()
        self.repo = repo
        self.date_folder = date_folder
        self.cache_dir = Path(cache_dir)
        self.cache_dir.mkdir(parents=True, exist_ok=True)

    def _list_and_download(self) -> list:
        # Lightweight: reuse list_repo_tree once, then download per file via CDN.
        from huggingface_hub import HfApi
        api = HfApi()
        entries = api.list_repo_tree(
            repo_id=self.repo,
            path=self.date_folder,
            repo_type="dataset",
            recursive=False,
        )
        files = [e.path for e in entries if e.type == "file"]
        local_paths = []
        for rel in files:
            local = self.cache_dir / rel.replace("/", "_")
            if not local.exists():
                hf_hub_download(
                    repo_id=self.repo,
                    filename=rel,
                    repo_type="dataset",
                    local_dir=self.cache_dir,
                    local_dir_use_symlinks=False,
                )
                # hf_hub_download returns path; resolve it
                local = self.cache_dir / rel.replace("/", "_")
            local_paths.append(local)
        return local_paths

    def __iter__(self) -> Iterator[Dict[str, str]]:
        local_paths = self._list_and_download()
        for local_path in local_paths:
            ds = load_dataset("parquet", data_files=str(local_path), split="train")
            for row in ds:
                yield {
                    "prompt": row.get("prompt") or row.get("input") or "",
                    "response": row.get("response") or row.get("output") or "",
                }


def build_dataset(args) -> IterableDataset:
    if args.file_list and os.path.exists(args.file_list):
        return CDNParquetDataset(file_list_path=args.file_list, repo=args.dataset_repo)
    # Safe fallback: list once and download per file (still avoids /api/ streaming).
    # Requires date_folder; default to today-like folder or derive from file list if possible.
    date_folder = getattr(args, "date_folder", None)
    if not date_folder:
        raise ValueError("Provide --file_list or --date_folder for fallback.")
