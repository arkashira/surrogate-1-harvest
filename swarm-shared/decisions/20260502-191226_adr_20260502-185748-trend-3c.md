## Deployment
hybrid (local-first CLI + optional Cloudflare Workers for auth/licensing + Hugging Face Spaces for optional model hosting/demo)

## v0.1 scope
v0.1 ships in 2 weeks: local CLI that indexes a repo (Python/JS), parses files with tree-sitter, runs a small on-device transformer to surface code smells and hotspots, stores results in SQLite, and outputs a terminal-friendly report plus a minimal JSON export. No cloud dependency for analysis; optional Cloudflare Worker for license check and HF Space for model download only.

## Open questions
- Which on-device model size balances accuracy vs. cold-start latency on typical developer laptops?
- How will incremental analysis detect semantic changes that don't affect file hashes?
- What is the licensing model and how will paid features be enforced locally?
- Will users accept opt-in telemetry to improve models while preserving privacy?
- How to distribute model updates without requiring external servers for core functionality?
