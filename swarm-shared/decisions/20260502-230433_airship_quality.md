### Core Objective (highest-value increment — <2h)
Make `airship discover` **fully deterministic, CDN-safe, and zero-runtime** during training by:
- Hardening the orchestrator script (strict Bash, `set -euo pipefail`, executable, invoked via `bash`)
- Producing a **pre-computed, non-recursive `file-list.json`** for one date folder
- Ensuring Lightning training uses **CDN-only fetches** (zero HF API calls during data load)
- Enforcing cron/systemd hygiene (`SHELL=/bin/bash`, absolute paths, logging)

This eliminates cron/script failures seen in `opus-pr-reviewer` and `active-learning` wrappers and applies the surrogate-1 + HF CDN bypass pattern.

### 1) Hardened orchestrator: `/opt/axentx/airship/discover`

```bash
#!/usr/bin/env bash
# /opt/axentx/airship/discover
# Purpose: Produce CDN-safe file-list for surrogate-1 training.
# Usage:  ./discover <date-folder> [output-json]
# Cron:   SHELL=/bin/bash
#         0 2 * * * /opt/axentx/airship/discover 2026-05-02 /opt/axentx/airship/file-list.json

set -euo pipefail
IFS=$'\n\t'

REPO="${HF_REPO:-HuggingFaceTB/smollm2-135m}"
DATE_FOLDER="${1:-}"
OUT="${2:-file-list.json}"

if [[ -z "$DATE_FOLDER" ]]; then
  echo "Usage: $0 <date-folder> [output-json]" >&2
  exit 1
fi

# Prefer hf CLI when available; fallback to raw CDN-safe listing if needed.
# Non-recursive listing only (avoids API bloat and rate limits).
echo "Listing repo tree: ${REPO} path=${DATE_FOLDER} (non-recursive)..."

if command -v hf >/dev/null 2>&1; then
  hf api tree "$REPO" --path "$DATE_FOLDER" --recursive false 2>/dev/null \
    | jq -r --arg d "$DATE_FOLDER" '
        [ .entries[]
          | select(.type == "file")
          | "\($d)/\(.path)"
        ]
        | sort
        | { date_folder: $d, repo: env.REPO, files: . }
      ' > "$OUT"
else
  # Fallback: use raw CDN directory index (if exposed) or minimal Python shim.
  # Keep this deterministic and non-recursive.
  python3 - "$REPO" "$DATE_FOLDER" "$OUT" <<'PY'
import json, os, sys, urllib.request

repo = sys.argv[1]
date_folder = sys.argv[2]
out_path = sys.argv[3]

# Raw CDN index for the date folder (non-recursive).
# Example: https://huggingface.co/datasets/repo/tree/main/2026-05-02
# If directory listing is not available, produce empty list and warn.
url = f"https://huggingface.co/datasets/{repo}/tree/main/{date_folder}"
try:
    with urllib.request.urlopen(url, timeout=15) as resp:
        # If HTML listing is returned, parsing would be required.
        # For reliability, prefer hf CLI or a small API call.
        # Here we emit minimal valid JSON and rely on explicit population.
        files = []
        result = {"date_folder": date_folder, "repo": repo, "files": files}
        with open(out_path, "w", encoding="utf-8") as f:
            json.dump(result, f, indent=2)
        print("Warning: CDN fallback used; file list may be empty.", file=sys.stderr)
except Exception as e:
    result = {"date_folder": date_folder, "repo": repo, "files": []}
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(result, f, indent=2)
    print(f"CDN fallback failed ({e}); produced empty file list.", file=sys.stderr)
PY
fi

echo "Wrote file-list: $OUT"
```

Make executable:

```bash
chmod +x /opt/axentx/airship/discover
```

### 2) Training-side CDN loader: `surrogate/data/cdn_loader.py`

```python
# surrogate/data/cdn_loader.py
# Loads files from HF CDN using a pre-computed file-list (zero HF API calls).
import json
import os
from pathlib import Path
from typing import List, Dict
import requests
from tqdm import tqdm

HF_CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def load_file_list(file_list_path: str) -> Dict:
    with open(file_list_path, "r", encoding="utf-8") as f:
        return json.load(f)

def stream_cdn_files(file_list_path: str, cache_dir: str = ".cache/cdn") -> List[Dict]:
    meta = load_file_list(file_list_path)
    repo = meta["repo"]
    files = meta["files"]
    os.makedirs(cache_dir, exist_ok=True)

    records = []
    for rel in tqdm(files, desc="CDN fetch"):
        url = HF_CDN_TEMPLATE.format(repo=repo, path=rel)
        cached = Path(cache_dir) / Path(rel).name
        if not cached.is_file():
            r = requests.get(url, timeout=30)
            r.raise_for_status()
            cached.write_bytes(r.content)

        records.append({"path": rel, "local": str(cached), "url": url})
    return records

def project_to_prompt_response(records: List[Dict]) -> List[Dict]:
    # Repo-specific projection: parse only {prompt,response}.
    # Rule: never store mixed-schema fields; move attribution to filename pattern.
    # Example: batches/mirror-merged/{date}/{slug}.parquet
    projected = []
    for r in records:
        # Implement actual parser per dataset shape here.
        projected.append({
            "prompt": "",   # <- parse from r["local"]
            "response": "", # <- parse from r["local"]
            "slug": Path(r["path"]).stem
        })
    return projected
```

### 4) Cron/systemd hygiene

If using cron on the orchestration host (mac/Linux):

```bash
# crontab -e
SHELL=/bin/bash
0 2 * * * /opt/axentx/airship/discover 2026-05-02 /opt/axentx/airship/file-list.json >> /var/log/airship-discover.log 2>&1
```

If using systemd (preferred for servers):

```ini
# /etc/systemd/system/airship-discover.service
[Unit]
Description=Airship CDN-safe file-list generator

[Service]
Type=oneshot
ExecStart=/bin/bash /opt/axentx/airship/discover 2026-05-02 /opt/axentx/airship/file-list.json
Environment=SHELL=/bin/bash
```

Enable/start:

```bash
sudo systemctl enable --now airship-discover.timer  # if using timer
# or
sudo systemctl enable --now airship-discover.service
```
