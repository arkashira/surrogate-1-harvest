## Deployment
cf workers

## v0.1 scope
v0.1 ships a deterministic, replayable workflow engine with one research adapter, one signal generator, one risk-check rule (block on hard limit), and a simulated execution adapter; immutable audit trail persisted to Supabase Postgres, Redis-backed signal cache, Cloudflare Queues for ordered stages, and CLI to replay any run by id — all deployable on free-tier CF Workers + Supabase.
