## Final Synthesis — Best of Both Proposals

**Highest-value incremental improvement (≤2h)**  
**Ship:** A zero-HF-API-during-training data loader + Lightning idle-resilient runner for Surrogate.

- Eliminates HF 429s during training by using CDN-only fetches.  
- Survives Lightning idle-stop without losing progress.  
- Keeps iteration loops fast and quota-safe.  
- Concrete, copy-paste-ready implementation with clear deployment steps.

## Correctness-first decisions (where proposals conflicted)

1. **File listing must be deterministic and versioned per date folder**  
   - Use `list_repo_tree(recursive=False)` once per date folder from Mac orchestration.  
   - Save `train_files.json` into repo (committed or as a build artifact).  
   - Training never calls HF API for file discovery.

2. **CDN-only data loading must avoid auth checks during training**  
   - Use direct `https://huggingface.co/datasets/{repo}/resolve/main/{path}` URLs.  
   - Cache via `hf_hub_download(..., force_download=False)` so repeated epochs hit local cache.  
   - Do not use `load_dataset(streaming=True)` or any token-requiring HF client during training loops.

3. **Lightning idle-stop recovery must resume from exact state**  
   - Use Lightning `ModelCheckpoint` + `resume_from_checkpoint`.  
   - Persist file list and global step/epoch in checkpoint metadata.  
   - Runner auto-restarts studio and re-launches the same command; Lightning resumes.

4. **Runner must be robust, observable, and non-interactive**  
   - Poll studio status; restart if not Running.  
   - Re-launch identical command on restart so Lightning resume works.  
   - Exit cleanly when run finishes/fails; loop only for idle-stop recovery.

## Implementation Plan (merged, concrete)

1. **Mac orchestration: pre-list files once per date folder**  
   - Run script → produce `train_files.json`.  
   - Commit or copy into repo so training can load it locally.

2. **CDN-only dataset loader**  
   - `train.py` loads `train_files.json`.  
   - Builds CDN URLs and downloads via `hf_hub_download` (cached).  
   - Streams rows from parquet (only needed columns).

3. **Lightning idle-resilient runner**  
   - `lightning_runner.py` ensures studio Running.  
   - Starts/stops monitored run.  
   - On idle-stop, restarts studio and re-runs identical command.

4. **Checkpointing & resume**  
   - Use `ModelCheckpoint` with `every_n_epochs=1` and `save_last=True`.  
   - Pass `--resume-from-checkpoint` on restart when checkpoint exists.

5. **Deploy**  
   - Replace launcher with `lightning_runner.py`.  
   - Keep legacy launcher renamed to `train_legacy.py`.  
   - Smoke-test one short run; confirm zero HF API calls during data loading.

### 2. CDN-only dataset loader (zero HF API during training)

```python
# surrogate/data.py
import json
from pathlib import Path
from typing import List, Dict

import pyarrow.parquet as pq
import torch
from torch.utils.data import IterableDataset
from huggingface_hub import hf_hub_download

REPO = "axentx/surrogate-data"

class CDNParquetDataset(IterableDataset):
    def __init__(self, file_list_path: str, cache_dir: str = ".cache/parquet", columns: List[str] = None):
        super().__init__()
        self.cache_dir = Path(cache_dir)
        self.cache_dir.mkdir(parents=True, exist_ok=True)
        self.columns = columns or ["prompt", "response"]

        with open(file_list_path) as f:
            self.files: List[str] = json.load(f)

        if not self.files:
            raise ValueError("No parquet files found in file list")

    def _download_cdn(self, rel_path: str) -> Path:
        local_path = hf_hub_download(
            repo_id=REPO,
            filename=rel_path,
            repo_type="dataset",
            cache_dir=self.cache_dir,
            force_download=False,
            resume_download=True,
        )
        return Path(local_path)

    def _stream_parquet(self, rel_path: str):
        local_path = self._download_cdn(rel_path)
        table = pq.read_table(local_path, columns=self.columns)
        for row in table.to_pylist():
            yield row

    def __iter__(self):
        for rel_path in self.files:
            yield from self._stream_parquet(rel_path)
```
