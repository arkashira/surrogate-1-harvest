# surrogate-1 / ops

## Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16`, `DATE`, `HF_TOKEN`
- Single `list_repo_tree(path=DATE, recursive=False)` → deterministic shard assignment by filename hash
- Downloads assigned files via **HF CDN** (`https://huggingface.co/datasets/.../resolve/main/...`) with **no Authorization header** (bypasses API rate limits)
- Projects each file to `{prompt, response}` only at parse time (avoids pyarrow CastError on mixed schemas)
- Deduplicates via central md5 store (`lib/dedup.py`)
- Writes `batches/public-merged/{DATE}/shard{N}-{HHMMSS}.jsonl` with **only** `{prompt, response}` (no extra cols)
- Exits 0 on success, non-zero on fatal error (so GitHub Actions matrix fails fast)

Add a tiny orchestration wrapper (`bin/run-shard.sh`) with proper Bash shebang + executable bit so cron/actions invoke safely.

"""
CDN-bypass shard worker for surrogate-1 public-dataset ingestion.

Environment:
  SHARD_ID     (int, 0..SHARD_TOTAL-1)
  SHARD_TOTAL  (int, default 16)
  DATE         (YYYY-MM-DD)
  HF_TOKEN     (write token for axentx/surrogate-1-training-pairs)
  HF_REPO      (default: datasets/axentx/surrogate-1-training-pairs)
  OUT_DIR      (default: batches/public-merged)

import json

import datetime
from pathlib import Path

import requests
from huggingface_hub import HfApi, hf_hub_download

# ── config --

SHARD_TOTAL = int(os.getenv("SHARD_TOTAL", "16"))
SHARD_ID = int(os.getenv("SHARD_ID", "0"))
DATE = os.getenv("DATE", datetime.date.today().isoformat())

OUT_DIR = Path(os.getenv("OUT_DIR", "batches/public-merged"))

    sys.exit(1)

if DEDUP_DB_PATH.exists():
    # Import local dedup module (keeps central store)
    import importlib.util

    spec = importlib.util.spec_from_file_location("dedup", DEDUP_DB_PATH)
    dedup_mod = importlib.util.module_from_spec(spec)
    spec.loader.exec_module(dedup_mod)

else:

        def __init__(self):
            self._s = set()

        def exists(self, h: str) -> bool:
            return h in self._s

        def add(self, h: str) -> None:
            self._s.add(h)

    seen = _FallbackSeen()

api = HfApi(token=HF_TOKEN)

def list_date_files(date_folder: str) -> list[str]:
    """
    Single API call: list top-level files in date folder (non-recursive).
    Avoids list_repo_files recursive pagination/rate-limit.
    """
    items = api.list_repo_tree(
        repo_id=HF_REPO,
        path=date_folder,
        repo_type="dataset",
        recursive=False,
    )
    # items can be dict or object depending on hf_hub version
    names = []
    for it in items:
        name = it["path"] if isinstance(it, dict) else getattr(it, "path", str(it))
        # Keep only files directly under date folder
        if name.startswith(date_folder + "/") and not name.endswith("/"):
            names.append(name)
    return sorted(names)

def deterministic_shard(filepath: str, total: int) -> int:
    """Stable shard by filename hash."""
    h = hashlib.sha256(filepath.encode()).hexdigest()
    return int(h, 16) % total

def cdn_download_url(repo: str, filepath: str) -> str:
    """
    Public CDN URL — no Authorization header required.
    Bypasses /api/ rate limits.
    """
    return f"https://huggingface.co/{repo}/resolve/main/{filepath}"

def project_to_pair(raw: Any, filepath: str) -> Optional[Dict[str, str]]:
    """
    Best-effort projection to {prompt, response}.
    Avoids pyarrow CastError by not relying on dataset streaming
    for mixed-schema repos.
    """
    # If it's already a dict-like with prompt/response, use it
    if isinstance(raw, dict):
        prompt = raw.get("prompt") or raw.get("input") or raw.get("question")
        response = raw.get("response") or raw.get("output") or raw.get("answer")
        if prompt is not None and response is not None:
            return {"prompt": str(prompt), "response": str(response)}

    # Try JSON lines
    if isinstance(raw, str):
        try:
            obj = json.loads(raw)
            return project_to_pair(obj, filepath)
        except Exception:
            pass

    # Fallback: produce minimal pair so pipeline keeps moving
    return {"prompt": str(raw), "response": ""}
