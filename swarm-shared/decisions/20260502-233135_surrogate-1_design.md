# surrogate-1 / design

## Final Implementation Plan (≤2h)

**Highest-value change**: Replace runtime `load_dataset(streaming=True)` + recursive `list_repo_files` in `bin/dataset-enrich.sh` with deterministic pre-flight snapshot + CDN-only fetches. This eliminates HF API rate limits (429), pyarrow CastError from mixed schemas, and keeps 16-shard parallelism intact.

### Steps (est. 90 min)

1. Add `bin/snapshot.sh` — one-time script that calls HF API once per date folder, saves `snapshot/<date>/file-list.json` to repo (or passes via workflow artifact). Uses `list_repo_tree(path, recursive=False)` per folder to avoid 100× pagination.
2. Update `bin/dataset-enrich.sh` to accept `SNAPSHOT_FILE` and `DATE` as env vars. Fetch via CDN URLs (`https://huggingface.co/datasets/.../resolve/main/...`). No Authorization header → bypasses `/api/` rate limits.
3. Project to `{prompt, response}` only at parse time; skip loading full parquet schema.
4. Keep shard logic unchanged (`SLURM_ARRAY_TASK_ID` or `SHARD_ID`/`MATRIX_SHARD`). Deterministic assignment by `hash(path) % 16`.
5. Update GitHub Actions to generate/pass the snapshot as an artifact or inline JSON.
6. Add retry/back-off for CDN 429 (CDN tier rarely rate-limits, but be safe).

### 1) Snapshot generator (`bin/snapshot.sh`)

```bash
#!/usr/bin/env bash
# bin/snapshot.sh
# Usage: HF_TOKEN=<token> ./bin/snapshot.sh axentx surrogate-1-training-pairs 2024-05-01
set -euo pipefail

REPO_OWNER="${1:-axentx}"
REPO_NAME="${2:-surrogate-1-training-pairs}"
DATE_FOLDER="${3:-$(date +%Y-%m-%d)}"
API_ROOT="https://huggingface.co/api"
OUTDIR="snapshot/${DATE_FOLDER}"
OUTFILE="${OUTDIR}/file-list.json"

mkdir -p "$OUTDIR"

# List top-level folders under the date (e.g., batches/public-merged/2024-05-01/)
# Use recursive=false to avoid 100× pagination.
list_folder() {
  local path="$1"
  curl -sSf -H "Authorization: Bearer ${HF_TOKEN:-}" \
    "${API_ROOT}/datasets/${REPO_OWNER}/${REPO_NAME}/tree?path=${path}&recursive=false"
}

# If HF_TOKEN is unset, caller must rely on cached file or public CDN only.
if [ -z "${HF_TOKEN:-}" ]; then
  echo "WARN: HF_TOKEN not set — cannot list repo. Provide cached file-list.json manually." >&2
  exit 1
fi

# Target path inside dataset
BASE_PATH="batches/public-merged/${DATE_FOLDER}"

# Emit JSON array of { "path": "...", "type": "file" } for files only
list_folder "${BASE_PATH}" | \
  jq -r --arg base "$BASE_PATH" '
    [.[] | select(.type == "file") | {path: ($base + "/" + .path)}]
  ' > "$OUTFILE"

echo "Snapshot written to $OUTFILE"
```

Make executable:

```bash
chmod +x bin/snapshot.sh
```

### 2) Updated worker (`bin/dataset-enrich.sh`)

Key changes:
- Accept `SNAPSHOT_FILE` and `DATE` as env vars.
- Fetch via CDN (no auth).
- Project to `{prompt,response}` only.
- Deterministic shard assignment by `hash(path) % 16`.
- Keep dedup via `lib/dedup.py`.

```bash
#!/usr/bin/env bash
# bin/dataset-enrich.sh
# Updated: use deterministic file list + CDN-only fetches
set -euo pipefail

# Required env
: "${HF_TOKEN:?HF_TOKEN required for uploads}"
: "${SHARD_ID:?SHARD_ID required (0-15)}"
: "${N_SHARDS:=16}"
: "${DATE_FOLDER:?DATE_FOLDER required (e.g., 2024-05-01)}"
: "${SNAPSHOT_FILE:?SNAPSHOT_FILE required (path to JSON file list)}"
: "${DATASET_REPO:=axentx/surrogate-1-training-pairs}"
: "${OUTPUT_PREFIX:=batches/public-merged}"

REPO_OWNER="${DATASET_REPO%%/*}"
REPO_NAME="${DATASET_REPO#*/}"

WORKDIR="$(cd "$(dirname "$0")/.." && pwd)"
cd "$WORKDIR"

log() { echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] $*"; }

# Validate shard
if ! [[ "$SHARD_ID" =~ ^[0-9]+$ ]] || [ "$SHARD_ID" -lt 0 ] || [ "$SHARD_ID" -ge "$N_SHARDS" ]; then
  log "ERROR: SHARD_ID must be 0..$((N_SHARDS-1))"
  exit 1
fi

# Validate snapshot
if [ ! -f "$SNAPSHOT_FILE" ]; then
  log "ERROR: SNAPSHOT_FILE not found: $SNAPSHOT_FILE"
  log "Generate with: HF_TOKEN=<t> ./bin/snapshot.sh $REPO_OWNER $REPO_NAME $DATE_FOLDER"
  exit 1
fi

# Deterministic shard assignment by hashing path
select_shard_paths() {
  local list_file="$1"
  jq -r --argjson shard "$SHARD_ID" --argjson nshards "$N_SHARDS" '
    .[] | .path as $p
    | ( ($p | @sh | . as $s | $s | hash ) % $nshards ) as $h
    | select($h == $shard)
    | $p
  ' "$list_file"
}

# Download via CDN (no auth) and project to {prompt,response}
process_file() {
  local rel_path="$1"
  local url="https://huggingface.co/datasets/${DATASET_REPO}/resolve/main/${rel_path}"

  # Use a temp file; stream parquet -> json projection via python for safety
  local tmp
  tmp="$(mktemp)"
  if ! curl -sSfL --retry 3 --retry-delay 2 -o "$tmp" "$url"; then
    log "WARN: failed to download $url"
    rm -f "$tmp"
    return 1
  fi

  # Project to {prompt,response} only; ignore other columns to avoid pyarrow schema issues
  python3 - "$tmp" <<'PY'
import sys, json, pyarrow.parquet as pq, pyarrow as pa

def normalize(val):
    if val is None:
        return ""
    if isinstance(val, (str, int, float, bool)):
        return str(val)
    try:
        return json.dumps(val, ensure_ascii=False)
    except Exception:
        return ""

def main(path):
    try:
        tbl = pq.read_table(path, columns=["prompt", "response"])
    except Exception:
        # Fallback: try common aliases
        try:
            tbl = pq.read_table(path)
        except Exception as e:
            sys.stderr.write(f"Cannot read parquet {path}: {e}\n")
            return
    # Select best columns
    cols = tbl.column_names
    prompt_col = next((c for c in cols if c.lower() in ("prompt", "instruction", "input", "question")), None)
    response_col = next((c for c in cols if c.lower() in ("response", "output", "answer", "completion")), None)

    if prompt_col is None or response_col is None:
        # If missing, skip row or use empty
        for i in range(tbl.num_rows):
            row = { "prompt": "", "response": "" }
            sys.stdout.write(json.dumps(row, ensure_ascii=False) + "\n")
        return

    prompts = tbl.column(prompt_col).to_pylist()
    responses = tbl.column(response_col).to_pylist()
    for p, r in zip(prompts, responses):
        row = { "prompt": normalize(p), "response": normalize(r) }
        sys.stdout.write(json.dumps(row, ensure_ascii=False) + "\n")

if __name__ == "__main__":
    main(sys.argv[1])
PY
  local rc=$?
  rm -f "$tmp"
  return $rc
