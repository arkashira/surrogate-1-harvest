# airship / design

## Highest-value incremental improvement (<2h)

**Implement a manifest-driven, CDN-only dataset loader for Surrogate-1 training** that:
1. Pre-lists target date folder once (Mac orchestration) → saves `file-list.json`
2. Embeds that list in training script → Lightning Studio fetches via CDN only (zero API calls during training)
3. Projects each file to `{prompt,response}` at parse time (avoids `pyarrow.CastError` from mixed schemas)
4. Reuses running Studio when possible (saves quota)

This directly applies the **HF CDN bypass**, **pre-list file paths**, **mixed-schema projection**, and **Studio reuse** patterns from the knowledge base.

---

## Implementation plan

1. **Create manifest generator** (`scripts/build_file_list.py`)
   - Uses HF API once per date folder (after rate-limit window)
   - Recursively lists files under `batches/mirror-merged/{date}/`
   - Outputs `file-list.json` with `{"repo": "...", "files": ["path1.parquet", ...]}`

2. **Update training entrypoint** (`surrogate/train.py`)
   - Accepts `--file-list` (path to embedded/copied manifest)
   - Uses `hf_hub_download` per file (CDN URLs, no auth/API during data load)
   - Arrow table → project only `prompt`/`response` columns (ignore extras)
   - Streams via `iterable_dataset` to avoid full RAM load

3. **Lightning launcher** (`surrogate/launch_studio.py`)
   - Lists running studios, reuses if name matches
   - Starts L40S (or falls back to free-tier) with `target.run(...)`
   - Checks status before each run; restarts if idle-stopped

4. **Add cron-safe wrapper** (`scripts/run_training.sh`)
   - Bash shebang, `chmod +x`
   - Invoked via `bash run_training.sh "$@"`
   - Sets `SHELL=/bin/bash` in crontab (per pattern)

## Code snippets

### 1) Manifest generator (run from Mac)

```python
# scripts/build_file_list.py
#!/usr/bin/env python3
import json
import os
import sys
from huggingface_hub import HfApi

API = HfApi()
REPO = os.getenv("HF_DATASET_REPO", "axentx/surrogate-mirror")
DATE = sys.argv[1] if len(sys.argv) > 1 else "2026-04-29"
OUT = sys.argv[2] if len(sys.argv) > 2 else "file-list.json"

root = f"batches/mirror-merged/{DATE}"
entries = API.list_repo_tree(REPO, path=root, recursive=True)
files = sorted(e.rfilename for e in entries if e.rfilename.endswith(".parquet"))

manifest = {"repo": REPO, "date": DATE, "files": files}
with open(OUT, "w") as f:
    json.dump(manifest, f, indent=2)

print(f"Wrote {len(files)} files to {OUT}")
```

Run once per date folder (after rate-limit clears):

```bash
python scripts/build_file_list.py 2026-04-29 file-list.json
```

### 2) CDN-only dataset loader (embedded manifest)

```python
# surrogate/data.py
import json
import pyarrow as pa
import pyarrow.parquet as pq
from pathlib import Path
from huggingface_hub import hf_hub_download
from torch.utils.data import IterableDataset

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path, repo=None, split=None):
        with open(manifest_path) as f:
            m = json.load(f)
        self.repo = repo or m["repo"]
        self.files = m["files"]
        if split:
            # simple deterministic split by hash
            import hashlib
            keep = []
            for p in self.files:
                h = int(hashlib.md5(p.encode()).hexdigest(), 16)
                if h % 10 < 8:
                    keep.append(p)  # 80% train
            self.files = keep

    def __iter__(self):
        for rpath in self.files:
            local = hf_hub_download(
                repo_id=self.repo,
                filename=rpath,
                repo_type="dataset",
            )
            try:
                table = pq.read_table(local, columns=["prompt", "response"])
            except (KeyError, pa.ArrowInvalid):
                # fallback: read all and project
                table = pq.read_table(local)
                cols = [c for c in table.column_names if c in ("prompt", "response")]
                if len(cols) != 2:
                    continue
                table = table.select(cols)
            # rename to canonical
            if "prompt" not in table.column_names:
                continue
            if "response" not in table.column_names:
                continue
            for batch in table.to_batches(max_chunksize=512):
                for i in range(batch.num_rows):
                    yield {
                        "prompt": batch.column("prompt")[i].as_py(),
                        "response": batch.column("response")[i].as_py(),
                    }
```

### 3) Lightning launcher with Studio reuse

```python
# surrogate/launch_studio.py
import lightning as L

def launch_training(
    script: str = "train.py",
    script_args: list | None = None,
    studio_name: str = "surrogate-l40s",
    cloud: str = "lightning-public-prod",
    machine: str = "L40S",
):
    teamspace = L.Teamspace()
    studio = None
    for s in teamspace.studios:
        if s.name == studio_name and s.status == "running":
            studio = s
            print(f"Reusing running studio: {studio_name}")
            break

    if studio is None:
        print(f"Creating studio: {studio_name}")
        studio = L.Studio(
            name=studio_name,
            create_ok=True,
            cloud=cloud,
            machine=machine,
        )

    # ensure running
    if studio.status != "running":
        print("Studio not running; starting...")
        studio.start(machine=machine)

    cmd = ["python", script] + (script_args or [])
    job = studio.run(command=cmd, wait=False)
    print(f"Launched job: {job}")
    return job

if __name__ == "__main__":
    launch_training(
        script="train.py",
        script_args=["--file-list", "file-list.json", "--epochs", "3"],
    )
```

### 4) Cron-safe wrapper

```bash
#!/usr/bin/env bash
# scripts/run_training.sh
set -euo pipefail
export SHELL=/bin/bash

cd /opt/axentx/airship/surrogate

# optional: rebuild manifest if needed
# python scripts/build_file_list.py 2026-04-29 file-list.json

python launch_studio.py --file-list file-list.json --epochs 3
```

Make executable:

```bash
chmod +x scripts/run_training.sh
```

Crontab entry (example):

```cron
SHELL=/bin/bash
0 2 * * * /opt/axentx/airship/surrogate/scripts/run_training.sh >> /var/log/airship_training.log 2>&1
```
