# airship / design

## Highest-value incremental improvement (<2h)
**Deterministic CDN file manifest + Lightning Studio lifecycle resilience for Surrogate-1 training**

- Eliminates HF API 429s during training by pre-listing one date folder and using CDN-only fetches.
- Prevents idle-stop training loss by checking studio status and auto-restarting on stopped/idle studios.
- Reuses running studios to save Lightning quota.
- Fits cleanly into existing surrogate training pipeline and can ship as a small, focused change.

---

## Implementation plan (concrete)

1. **Add manifest generator** (`scripts/generate_cdn_manifest.py`)
   - Runs on Mac (or any orchestration host).
   - Uses HF Hub `list_repo_tree(path, recursive=False)` for a single date folder (e.g., `batches/mirror-merged/2026-05-03/`).
   - Emits `manifest.json` with CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) and file metadata.
   - Commits/embeds manifest into surrogate training config or passes via env.

2. **Update training script** (`surrogate/train.py` or equivalent loader)
   - Accept path to `manifest.json`.
   - During data loading, stream files via CDN URLs using `requests`/`urllib` with retry/backoff.
   - Parse each file and project to `{prompt, response}` only; ignore extra schema columns.
   - No `load_dataset(streaming=True)` on heterogeneous repos.

3. **Add Lightning Studio lifecycle wrapper** (`scripts/run_surrogate_studio.py`)
   - Before `.run()`, list `Teamspace.studios`; reuse running studio with matching name.
   - If studio exists but is stopped, restart with `target.start(machine=Machine.L40S)` (or fallback to free-tier L40S).
   - Wrap training command so idle-stop can be detected and handled on next iteration.

4. **Small operational changes**
   - Ensure HF token is available for `list_repo_tree` (only once per manifest generation).
   - Add retry/backoff for CDN downloads (separate from API rate limits).
   - Log manifest generation time and file count for observability.

## Code snippets

### 1) Manifest generator (scripts/generate_cdn_manifest.py)
```python
#!/usr/bin/env python3
"""
Generate CDN-only manifest for a single date folder in a HuggingFace dataset repo.
Usage:
    HF_TOKEN=... python generate_cdn_manifest.py \
        --repo "org/surrogate-dataset" \
        --date-folder "batches/mirror-merged/2026-05-03" \
        --output manifest.json
"""
import argparse
import json
import os
import sys
from datetime import datetime

from huggingface_hub import HfApi

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate CDN manifest for HF dataset folder")
    parser.add_argument("--repo", required=True, help="HF dataset repo (org/repo)")
    parser.add_argument("--date-folder", required=True, help="Folder path in repo (e.g. batches/mirror-merged/2026-05-03)")
    parser.add_argument("--output", default="manifest.json", help="Output JSON path")
    args = parser.parse_args()

    token = os.environ.get("HF_TOKEN")
    api = HfApi(token=token)

    # Single non-recursive list to minimize API calls
    try:
        files = api.list_repo_tree(
            repo_id=args.repo,
            path=args.date_folder,
            repo_type="dataset",
            recursive=False,
        )
    except Exception as exc:
        print(f"Failed to list repo tree: {exc}", file=sys.stderr)
        sys.exit(1)

    # Filter to parquet (or desired extensions)
    items = []
    for f in files:
        if not f.rfilename:
            continue
        # f.rfilename is relative to repo root
        cdn_url = CDN_TEMPLATE.format(repo=args.repo, path=f.rfilename)
        items.append({
            "repo": args.repo,
            "path": f.rfilename,
            "cdn_url": cdn_url,
            "size": getattr(f, "size", None),
        })

    manifest = {
        "generated_at": datetime.utcnow().isoformat() + "Z",
        "repo": args.repo,
        "folder": args.date_folder,
        "count": len(items),
        "items": items,
    }

    with open(args.output, "w", encoding="utf-8") as fp:
        json.dump(manifest, fp, indent=2)

    print(f"Wrote {len(items)} entries to {args.output}")

if __name__ == "__main__":
    main()
```

### 2) Lightweight CDN dataset loader (surrogate/data/cdn_loader.py)
```python
import json
import logging
from pathlib import Path
from typing import Iterator, Tuple

import requests
from requests.adapters import HTTPAdapter, Retry

logger = logging.getLogger(__name__)

def _make_session() -> requests.Session:
    session = requests.Session()
    retries = Retry(
        total=5,
        backoff_factor=1.0,
        status_forcelist=[429, 500, 502, 503, 504],
    )
    session.mount("https://", HTTPAdapter(max_retries=retries))
    return session

def iter_cdn_parquet(manifest_path: Path) -> Iterator[Tuple[str, bytes]]:
    """
    Yield (cdn_url, raw_bytes) for each parquet file in manifest.
    Training script should project to {prompt, response} after reading bytes.
    """
    manifest = json.loads(manifest_path.read_text())
    session = _make_session()

    for idx, item in enumerate(manifest.get("items", [])):
        url = item["cdn_url"]
        if not url.endswith(".parquet"):
            logger.debug("Skipping non-parquet: %s", url)
            continue

        logger.info("Fetching [%s/%s] %s", idx + 1, manifest["count"], url)
        try:
            resp = session.get(url, timeout=30)
            resp.raise_for_status()
            yield url, resp.content
        except Exception as exc:
            logger.error("Failed to fetch %s: %s", url, exc)
            # Optionally continue or raise depending on tolerance
            continue
```
