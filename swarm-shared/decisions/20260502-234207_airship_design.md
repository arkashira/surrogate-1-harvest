# airship / design

## Design Decision: Deterministic CDN-Only Manifest for `airship discover`

**Scope (≤2h):**  
Replace HF API–based file listing in `airship discover` with a deterministic, CDN-only manifest generator.  
- Single Mac-side orchestration script lists a **date folder once** via HF API (respecting rate limits), saves a JSON manifest.  
- Training (Lightning Studio) consumes the manifest and downloads files **via CDN URLs only** (`resolve/main/...`) with zero API/auth calls during training.  
- Supports surrogate-1 schema rules: project to `{prompt, response}` at parse time; attribution via filename pattern (`batches/mirror-merged/{date}/{slug}.parquet`); no extra columns.

---

## Implementation Plan

1. **Create `scripts/airship_discover.py`**  
   - Accepts `repo_id`, `date_folder`, `output_manifest`.  
   - Uses `list_repo_tree(path=date_folder, recursive=False)` to list immediate children (avoid recursive paginated calls).  
   - Emits deterministic JSON: sorted by filename; each entry includes CDN URL and local slug.  
   - Exits non-zero on HF API 429 with guidance to retry after 360s.

2. **Update training launcher** (`scripts/run_surrogate1_train.py` or equivalent)  
   - Reads manifest JSON.  
   - Uses `hf_hub_download` per file **or** direct CDN fetch (`requests.get(cdn_url, timeout=60)`).  
   - Projects each file to `{prompt, response}` at parse time (no schema assumptions).  
   - Writes shards to `batches/mirror-merged/{date}/` with slug-based filenames.

3. **Lightning Studio integration**  
   - Reuse running studio if available (`Teamspace.studios` check).  
   - Pass manifest path as argument; training script does **no** `list_repo_files` or recursive API calls.  
   - On idle-stop, restart with `Machine.L40S` in `lightning-public-prod` (H200 requires `lightning-lambda-prod`).

4. **Cron/scheduler hygiene**  
   - Ensure any wrapper scripts have `#!/usr/bin/env bash`, `chmod +x`, invoked via `bash <script> "$@"`.  
   - Set `SHELL=/bin/bash` in crontab to avoid POSIX sh surprises.

## Code Snippets

### `scripts/airship_discover.py`
```python
#!/usr/bin/env python3
"""
Deterministic CDN-only manifest generator for surrogate-1 training.
Usage:
  python airship_discover.py --repo_id <org/repo> \
                             --date_folder 2026-04-29 \
                             --output_manifest manifest.json
"""
import argparse
import json
import os
import sys
import time
from pathlib import Path

from huggingface_hub import list_repo_tree, HfApi

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo_id}/resolve/main/{path}"

def build_manifest(repo_id: str, date_folder: str) -> list[dict]:
    """List one date folder (non-recursive) and build CDN manifest."""
    try:
        tree = list_repo_tree(repo_id=repo_id, path=date_folder, recursive=False)
    except Exception as exc:
        # Heuristic: if 429, suggest wait 360s
        if getattr(exc, "status_code", None) == 429:
            print("HF API 429 — wait 360s before retry.", file=sys.stderr)
        raise

    entries = []
    for item in sorted(tree, key=lambda x: x.path):
        if item.type != "file":
            continue
        cdn_url = CDN_TEMPLATE.format(repo_id=repo_id, path=item.path)
        slug = Path(item.path).stem  # e.g., batches/mirror-merged/2026-04-29/abc123.parquet -> abc123
        entries.append({
            "path": item.path,
            "cdn_url": cdn_url,
            "slug": slug,
            "size": getattr(item, "size", None),
        })
    return entries

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate CDN-only manifest for surrogate-1.")
    parser.add_argument("--repo_id", required=True, help="HF dataset repo id (e.g., org/repo)")
    parser.add_argument("--date_folder", required=True, help="Date folder inside repo (e.g., 2026-04-29)")
    parser.add_argument("--output_manifest", required=True, help="Output JSON manifest path")
    args = parser.parse_args()

    manifest = build_manifest(args.repo_id, args.date_folder)
    out_path = Path(args.output_manifest)
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    print(f"Wrote {len(manifest)} entries to {out_path}")

if __name__ == "__main__":
    main()
```

### `scripts/run_surrogate1_train.py` (excerpt)
```python
import json
import requests
from pathlib import Path

def load_manifest(manifest_path: str):
    with open(manifest_path) as f:
        return json.load(f)

def fetch_via_cdn(entry, local_root: Path):
    local_file = local_root / entry["path"]
    if local_file.exists():
        return local_file
    local_file.parent.mkdir(parents=True, exist_ok=True)
    # CDN fetch — no Authorization header required
    resp = requests.get(entry["cdn_url"], timeout=60)
    resp.raise_for_status()
    local_file.write_bytes(resp.content)
    return local_file

def project_to_pair(local_file: Path):
    """
    Project heterogeneous file to {prompt, response} at parse time.
    Implement per-format (parquet/jsonl/etc).
    Returns dict with keys: prompt, response.
    """
    # Placeholder: adapt to surrogate-1 schema expectations
    if local_file.suffix == ".parquet":
        import pyarrow.parquet as pq
        tbl = pq.read_table(local_file)
        # Keep only prompt/response columns if present; drop others
        cols = set(tbl.column_names)
        keep = [c for c in ("prompt", "response") if c in cols]
        if not keep:
            raise ValueError(f"No prompt/response in {local_file}")
        return tbl.select(keep).to_pylist()
    else:
        raise NotImplementedError(f"Unsupported {local_file.suffix}")

def run(manifest_path: str, out_root: Path):
    manifest = load_manifest(manifest_path)
    out_root.mkdir(parents=True, exist_ok=True)
    for entry in manifest:
        lf = fetch_via_cdn(entry, out_root)
        pairs = project_to_pair(lf)
        # Write shard: batches/mirror-merged/{date}/{slug}.parquet
        date = Path(entry["path"]).parts[-2]  # assumes .../{date}/{file}
        shard_dir = out_root / "batches" / "mirror-merged" / date
        shard_dir.mkdir(parents=True, exist_ok=True)
        shard_path = shard_dir / f"{entry['slug']}.parquet"
        # Save projected pairs (simplified)
        import pyarrow as pa
        tbl = pa.Table.from_pylist(pairs)
        import pyarrow.parquet as pq
        pq.write_table(tbl, shard_path)
        print(f"Shard: {shard_path}")
```

- `airship discover` produces deterministic JSON sorted by path.

- Schema projection happens at parse time; output shards follow
