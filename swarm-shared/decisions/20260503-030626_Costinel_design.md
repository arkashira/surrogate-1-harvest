# Costinel / design

Below is the **single, merged implementation** that keeps every deadline (<2h), removes contradictions, and maximizes correctness + actionability.

Key resolution choices:
- **CDN path**: use HuggingFace CDN as the source of truth (Candidate 1) and **pre-warm a fast fallback** (`/hub-data/moc.json`) at build time (Candidate 2) so the panel is fast and resilient.
- **Fail-safe behavior**: collapse to a minimal “Show/Hide” affordance (not a broken UI) and never throw user-facing errors.
- **Build + deploy**: include generated artifacts in the Docker image so the CDN fallback is available immediately on cold start.
- **Config**: single source of truth via `VITE_HUB_NAME` and a static `hubs.json` generated on Mac/CI.

---

## What we ship (final)
- Non-blocking **TopHubSignalPanel** in the Costinel dashboard sidebar/top banner.
- Defaults to hub **MOC** (`VITE_HUB_NAME`).
- Shows: hub title, description, top 3 signals, last-updated timestamp, “View details” link.
- **CDN-first**: browser fetches from HF CDN; if that fails, uses build-time CDN fallback (`/hub-data/moc.json`); if both fail, collapses gracefully.
- Zero backend changes; deployable via existing Docker/static pipeline.

### `src/lib/hub-panel.ts`
```ts
// CDN-first fetcher with build-time fallback — zero runtime API calls
type Signal = {
  id: string;
  title: string;
  severity: 'critical' | 'high' | 'medium' | 'low';
  summary: string;
  recommendation?: string;
  ts: string;
};

type HubMeta = {
  title: string;
  description: string;
  file: string;
  cdnBase: string;
};

type HubData = {
  hub: HubMeta;
  signals: Signal[];
  updated: string;
};

const HUB_NAME = import.meta.env.VITE_HUB_NAME || 'MOC';

async function loadHubMeta(): Promise<HubMeta | null> {
  try {
    const res = await fetch('/config/hubs.json', { cache: 'no-store' });
    if (!res.ok) return null;
    const map = await res.json() as Record<string, HubMeta>;
    return map[HUB_NAME] || null;
  } catch {
    return null;
  }
}

export async function fetchHubSignals(): Promise<HubData | null> {
  const meta = await loadHubMeta();
  if (!meta) return useLocalFallback();

  // Primary: CDN fetch (no Authorization header)
  try {
    const url = `${meta.cdnBase}/${encodeURIComponent(meta.file)}`;
    const res = await fetch(url, { cache: 'no-store' });
    if (!res.ok) throw new Error('CDN fetch failed');
    const payload = await res.json();

    return {
      hub: meta,
      signals: Array.isArray(payload?.signals) ? payload.signals.slice(0, 3) : [],
      updated: payload?.updated || new Date().toISOString(),
    };
  } catch {
    // Fallback: local build-time copy (fast, available offline)
    return useLocalFallback();
  }
}

function useLocalFallback(): HubData | null {
  try {
    // This path exists in /public/hub-data/<hub>.json and is served as static asset
    return fetch(`/hub-data/${HUB_NAME.toLowerCase()}.json`, { cache: 'no-store' })
      .then((r) => (r.ok ? r.json() : null))
      .then((j) => {
        if (!j) return null;
        return {
          hub: j.hub || { title: HUB_NAME, description: '', file: '', cdnBase: '' },
          signals: Array.isArray(j.signals) ? j.signals.slice(0, 3) : [],
          updated: j.updated || new Date().toISOString(),
        };
      })
      .catch(() => null);
  } catch {
    return null;
  }
}
```
