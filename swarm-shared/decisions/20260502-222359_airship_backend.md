# airship / backend

## Decision: ship `airship discover` (backend)

Highest-value incremental improvement (≤2h):  
Add a zero-config, cron-safe discovery/training orchestrator that:

- runs `granite-business-research.sh` (if present) and tags outputs
- executes `knowledge-rag` to surface top-hub insights (e.g., MOC)
- lists HF dataset files once (per date folder) → saves JSON
- produces a CDN-only `train.py` + file-list so Lightning training uses **zero HF API calls** during data load
- embeds quota-safe patterns (Lightning studio reuse, idle-stop guard, surrogate-1 schema projection)
- exposes a single CLI entrypoint: `bin/airship-discover`

This directly applies every pattern in the prompt and unblocks immediate training iteration without breaking existing services.

---

## Implementation plan (≤2h)

1. Add project structure
   - `bin/airship-discover` (CLI entry, executable)
   - `airship/discover/__init__.py`
   - `airship/discover/orchestrator.py`
   - `airship/discover/train_template.py` (rendered into `train.py`)
   - `airship/discover/lightning_utils.py`
   - `airship/discover/hf_utils.py`

2. CLI behavior
   - `bin/airship-discover` (bash wrapper) → `python -m airship.discover --date-folder YYYY-MM-DD [--reuse-studio] [--hf-repo ORG/REPO]`
   - Shebang `#!/usr/bin/env bash`, `set -euo pipefail`, invoke python with proper args
   - If `granite-business-research.sh` exists → run and tag outputs
   - Run `knowledge-rag` (assume available in PATH) and capture top-hub insight

3. HF file listing (once)
   - Use `list_repo_tree(path=date_folder, recursive=False)` per folder strategy
   - Save `file_list.json` locally and embed into generated `train.py`
   - Include CDN fallback URLs (`https://huggingface.co/datasets/.../resolve/main/...`)

4. Training script generation
   - Render `train_template.py` → `train.py`
   - Data loader uses only CDN URLs (no `load_dataset` with API calls)
   - Project mixed-schema files to `{prompt,response}` at parse time (surrogate-1 pattern)
   - Attribution via filename pattern `batches/mirror-merged/{date}/{slug}.parquet`

5. Lightning utilities
   - List studios and reuse running studio when `--reuse-studio`
   - Guard idle-stop: check status before `.run()`; restart with `target.start(machine=Machine.L40S)` if stopped
   - Prefer `lightning-public-prod` (L40S) unless paid account available

6. Cron safety
   - Wrapper scripts enforce `SHELL=/bin/bash` in crontab comments
   - All invoked scripts have shebang and `chmod +x`

7. Logging & tagging
   - Tag outputs with `#business-research #knowledge-rag #graph #hub`
   - Save run metadata (date-folder, top-hub, file count) to `discovery_runs.jsonl`

## Code snippets

### bin/airship-discover
```bash
#!/usr/bin/env bash
# airship-discover — zero-config discovery + training orchestration
# Cron-safe, CDN-only, quota-safe
# Usage: airship-discover [--date-folder YYYY-MM-DD] [--reuse-studio] [--hf-repo ORG/REPO]
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "$SCRIPT_DIR/.." && pwd)"
cd "$PROJECT_ROOT"

export SHELL=/bin/bash

# Defaults
DATE_FOLDER="${AIRSHIP_DATE_FOLDER:-$(date +%Y-%m-%d)}"
HF_REPO="${AIRSHIP_HF_REPO:-datasets/mirror-merged}"
REUSE_STUDIO="${AIRSHIP_REUSE_STUDIO:-false}"

# Parse flags
while [[ $# -gt 0 ]]; do
  case $1 in
    --date-folder) DATE_FOLDER="$2"; shift 2 ;;
    --hf-repo)     HF_REPO="$2";     shift 2 ;;
    --reuse-studio)REUSE_STUDIO="true"; shift ;;
    --help|-h)     echo "Usage: airship-discover [--date-folder YYYY-MM-DD] [--reuse-studio] [--hf-repo ORG/REPO]"; exit 0 ;;
    *) echo "Unknown option $1"; exit 1 ;;
  esac
done

echo "=== airship-discover ==="
echo "date_folder=$DATE_FOLDER"
echo "hf_repo=$HF_REPO"
echo "reuse_studio=$REUSE_STUDIO"

# 1) Business research (if present)
if [[ -x "$PROJECT_ROOT/granite-business-research.sh" ]]; then
  echo "Running granite-business-research.sh..."
  "$PROJECT_ROOT/granite-business-research.sh"
  echo "Tagged: #business-research"
fi

# 2) Knowledge RAG top-hub insight
if command -v knowledge-rag >/dev/null 2>&1; then
  echo "Running knowledge-rag for top-hub insight..."
  knowledge-rag --top-hub > "$PROJECT_ROOT/top_hub_insight.txt" || true
  echo "Top-hub insight saved. Tagged: #knowledge-rag #graph #hub"
fi

# 3) Run orchestrator
python -m airship.discover \
  --date-folder "$DATE_FOLDER" \
  --hf-repo "$HF_REPO" \
  --reuse-studio "$REUSE_STUDIO"

echo "=== Done ==="
```

### airship/discover/cli.py
```python
import argparse
import json
import sys
from pathlib import Path

from .orchestrator import run_discovery

def main() -> None:
    parser = argparse.ArgumentParser(description="Airship discovery orchestrator")
    parser.add_argument("--date-folder", required=True, help="Date folder (YYYY-MM-DD)")
    parser.add_argument("--hf-repo", default="datasets/mirror-merged", help="HF dataset repo")
    parser.add_argument("--reuse-studio", action="store_true", help="Reuse running Lightning studio")
    parser.add_argument("--output-dir", default=".", help="Output directory for train.py and file_list.json")
    args = parser.parse_args()

    try:
        run_discovery(
            date_folder=args.date_folder,
            hf_repo=args.hf_repo,
            reuse_studio=args.reuse_studio,
            output_dir=Path(args.output_dir),
        )
    except Exception as exc:
        print(f"Discovery failed: {exc}", file=sys.stderr)
        sys.exit(1)

if __name__ == "__main__":
    main()
```

import json
import os
import time
from pathlib import Path
from typing import Dict, List

from huggingface_hub import list_repo_tree, hf_hub_download

HF_API_RATE_LIMIT_RETRY = 360  # seconds (per pattern)

def list_date_files(hf_repo: str, date_folder: str, retries: int = 3) -> List[Dict]:
    """
    List files for a date folder using list_repo_tree per folder (paginated-safe).
    Returns list of dicts with 'path' and optional 'size'.
    """
    attempt = 0
    while attempt < retries:
        try:
            tree = list_repo_tree(repo_id=hf_repo, path=date_folder, recursive=False)
            files = [{"path": f.rfilename, "size": getattr(f, "size", None)} for f in tree if f.type == "file"]
            return files
        except Exception as exc:
            attempt += 1
            if attempt >= retries:
                raise
            print(f"HF API error (attempt {attempt}/{retries}): {exc}. Waiting {HF_API_RATE_LIMIT_RETRY}s")
            time.sleep(HF_API_RATE_LIMIT_RETRY)
    return []

def build_cdn_url(hf_repo: str, file_path: str) -> str:
    """Build CDN URL (bypasses API auth checks)."""
    return f"https://huggingface.co/datasets/{hf_repo}/resolve/main/{file_path}"
