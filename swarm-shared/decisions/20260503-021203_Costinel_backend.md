# Costinel / backend

## Final Implementation Plan — Top-Hub Signal Panel (Costinel Dashboard)

**Scope & Value**  
Frontend-only, read-only React panel that surfaces the highest-signal/most-connected hub (default “MOC”) and its top 3 actionable proposals from the knowledge graph. CDN-first data path with zero runtime API calls or secrets.

**Why this ships highest value in <2h**  
- No schema changes, no backend, no auth during runtime.  
- Reuses existing `/public/data/knowledge-rag/` CDN convention.  
- Immediate signal boost for dashboard users.  
- Estimated effort: <2h (frontend only).

### 4) Custom hook for data fetching

`src/hooks/useTopHubSignal.ts`

```ts
import { useEffect, useState, useCallback } from 'react';
import { TopHub } from '../types/topHub';

export function useTopHubSignal(jsonPath = '/data/knowledge-rag/top-hub-moc.json') {
  const [hub, setHub] = useState<TopHub | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  const load = useCallback(async () => {
    setLoading(true);
    setError(null);
    try {
      const res = await fetch(jsonPath, { cache: 'no-cache' });
      if (!res.ok) throw new Error(`Failed to load top-hub data (${res.status})`);
      const data = (await res.json()) as TopHub;
      setHub(data);
    } catch (err) {
      setError(err instanceof Error ? err.message : 'Unknown error');
    } finally {
      setLoading(false);
    }
  }, [jsonPath]);

  useEffect(() => {
    load();
  }, [load]);

  return { hub, loading, error, refetch: load };
}
```
