# airship / design

## Final Implementation — Manifest-Driven, CDN-Only Dataset Loader

**Goal**: Eliminate HF API rate limits and mixed-schema ingestion failures by replacing `load_dataset`/`list_repo_files` with a manifest-driven, CDN-only iterable loader that is deterministic, retry-resilient, and schema-safe.

**Estimated effort**: 1.5–2h

---

### 1) What to change (single source of truth)

- **Add `scripts/build_manifest.py`**  
  - Run once per folder from a non-rate-limited environment (e.g., your Mac) after the rate-limit window clears.  
  - Uses `list_repo_tree(..., recursive=False)` per folder to avoid pagination and API churn.  
  - Produces `manifests/manifest-{date}.json` with `{repo, folder, generated_at, entries:[{path, sha, size, url}]}`.  
  - Validates file extensions and emits warnings for unexpected types.

- **Add `surrogate/data/cdn_dataset.py`**  
  - Implements `IterableDataset` that streams files directly from HF CDN (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with no Authorization header.  
  - Uses `requests` with connection pooling + exponential backoff (max 3 retries) and skips corrupt/unfetchable files with warnings.  
  - Accepts a mandatory `transform: Callable[[bytes, path], Optional[Dict]]` to normalize heterogeneous schemas.  
  - Yields `{prompt, response, _source_file, _source_repo, _source_line}` for traceability.  
  - Supports optional schema validation via `pydantic` or lightweight checks; invalid records are skipped and counted.

- **Update training entrypoints**  
  - Replace all `load_dataset(..., streaming=True)` and recursive `list_repo_files` calls with `CdnDataset(manifest=..., transform=...)`.  
  - Use `torch DataLoader` with `num_workers>=1` and pin_memory where appropriate.  
  - Add a small deterministic shuffle buffer over manifest entries (not full files) when `shuffle=True`.

- **Remove/disable**  
  - Any remaining `load_dataset` or `list_repo_files` usage in surrogate training code.  
  - Implicit schema inference; enforce explicit transforms.

### 2) Code (production-ready)

#### `scripts/build_manifest.py`
```python
#!/usr/bin/env python3
"""
Build a CDN manifest for a HuggingFace dataset repo folder.
Usage:
  python scripts/build_manifest.py \
    --repo=narrativa/medical-reports \
    --folder=2026-05-03 \
    --out=manifests/manifest-2026-05-03.json
"""
import argparse
import json
import time
from pathlib import Path
from typing import Dict, List

from huggingface_hub import HfApi

API = HfApi()
ALLOWED_EXTS = {".jsonl", ".json", ".parquet", ".csv", ".tsv"}

def build_manifest(repo_id: str, folder: str, out_path: Path) -> None:
    tree = API.list_repo_tree(repo_id=repo_id, path=folder, recursive=False)
    entries: List[Dict] = []
    for item in tree:
        if item.type != "file":
            continue
        path = item.path
        if ALLOWED_EXTS and not any(path.endswith(ext) for ext in ALLOWED_EXTS):
            continue
        entries.append({
            "repo": repo_id,
            "path": path,
            "sha": getattr(item, "oid", None) or getattr(item, "sha", None),
            "size": getattr(item, "size", None),
            "url": f"https://huggingface.co/datasets/{repo_id}/resolve/main/{path}"
        })
    out_path.parent.mkdir(parents=True, exist_ok=True)
    manifest = {
        "repo": repo_id,
        "folder": folder,
        "generated_at": time.time(),
        "entries": entries,
    }
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(manifest, f, indent=2, ensure_ascii=False)
    print(f"Wrote {len(entries)} entries to {out_path}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--folder", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()
    build_manifest(args.repo, args.folder, Path(args.out))
```

#### `surrogate/data/cdn_dataset.py`
```python
"""
IterableDataset that streams files from HF CDN using a manifest.
No Authorization header is sent; CDN downloads bypass API rate limits.
"""
import json
import logging
import time
from typing import Any, Callable, Dict, Iterator, Optional

import requests
from torch.utils.data import IterableDataset

logger = logging.getLogger(__name__)

class CdnDataset(IterableDataset):
    def __init__(
        self,
        manifest_path: str,
        transform: Callable[[bytes, str], Optional[Dict[str, str]]],
        max_retries: int = 3,
        backoff_factor: float = 1.5,
        timeout: int = 30,
        shuffle: bool = False,
        seed: int = 42,
    ):
        """
        Args:
            manifest_path: path to manifest JSON.
            transform: fn(raw_bytes, file_path) -> {"prompt":..., "response":...} or None.
            max_retries: max download retries per file.
            backoff_factor: exponential backoff factor.
            timeout: requests timeout in seconds.
            shuffle: whether to shuffle manifest entries (not file contents).
            seed: random seed for shuffling.
        """
        self.manifest_path = manifest_path
        self.transform = transform
        self.max_retries = max_retries
        self.backoff_factor = backoff_factor
        self.timeout = timeout
        self.shuffle = shuffle
        self.seed = seed
        self._manifest = self._load_manifest()
        self._entries = self._manifest.get("entries", [])
        if self.shuffle:
            import random
            self._rng = random.Random(self.seed)
            self._rng.shuffle(self._entries)

    def _load_manifest(self) -> Dict[str, Any]:
        with open(self.manifest_path, "r", encoding="utf-8") as f:
            return json.load(f)

    def _download(self, url: str) -> Optional[bytes]:
        session = requests.Session()
        for attempt in range(1, self.max_retries + 1):
            try:
                resp = session.get(
                    url,
                    headers={"User-Agent": "axentx-airship-surrogate"},
                    timeout=self.timeout,
                    stream=True,
                )
                resp.raise_for_status()
                return resp.content
            except Exception as exc:
                wait = self.backoff_factor ** attempt
                logger.warning(
                    f"Download attempt {attempt}/{self.max_retries} failed for {url}: {exc}. "
                    f"Retrying in {wait:.1f}s"
                )
                time.sleep(wait)
        logger.error(f"Failed to download {url} after {self.max_retries} attempts")
        return None

    def __iter__(self) -> Iterator[Dict[str, Any]]:
        skipped = 0
        for entry in self._entries:
            url = entry["url"]
            raw = self._download(url)
            if raw is None:
                skipped += 1
                continue
            try:
                transformed = self.transform(raw, entry["path"])
            except Exception as exc:
                logger.warning(f"Transform failed for {entry['path']}: {exc}")
                skipped += 1
                continue
            if transformed is None:
                skipped += 1
                continue
            if not isinstance(transformed, dict):
                logger.warning(f"Transform must return dict or None for {entry['path']}")
                skipped += 1
                continue
            required = {"prompt", "response"}
            if not required.issubset(transformed.keys()):
                logger.warning(f"Missing required keys {required} in record from {entry['path']}")
                skipped += 1
                continue
            transformed["_source_file"] = entry["path"]
            transformed["_source_repo"] = entry["repo"]
            yield transformed
        if skipped:
            logger.info(f"Skipped {skipped} files/records during iteration")
