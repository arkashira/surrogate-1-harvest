# workio / quality

## Final consolidated implementation (best parts, corrected + actionable)

**Summary of choices**
- Use **signature verification** (Candidate 1 + 2) with constant-time compare.
- Use **deterministic idempotency** keyed by tenant+employee+date+eventType (Candidate 1) **plus** a unique constraint/index in the DB (Candidate 2) so duplicates are rejected at the persistence layer.
- Guard races with **PostgreSQL advisory lock per intent key** (Candidate 1) — avoids new Redis infra and is transactional; short-circuit concurrent toggles safely.
- Add **structured observability** (Candidate 2): log receipt, validation, dedupe, insert, and latency; expose counters/errors for alerting.
- Keep **atomic transactional write path** and **rollback on error** (Candidate 1).
- Fix Candidate 1’s unsafe “skip on lock failure” by returning **409/429 with clear semantics** and logging; do not silently drop events.
- Fix Candidate 1’s fragile `makeIdempotencyKey` usage for locking by using a stable, reversible hash for advisory lock and a DB unique constraint for idempotency.

### 1) DB schema changes (minimal, safe)

```sql
-- If not already present
ALTER TABLE attendance
  ADD COLUMN line_event_id text,
  ADD COLUMN intent_hash bytea NOT NULL;

-- Deterministic idempotency: one intent per tenant/employee/date/event_type
CREATE UNIQUE INDEX uq_attendance_intent
  ON attendance (tenant_id, employee_id, date, event_type);

-- Fast lookup for duplicate LINE events
CREATE UNIQUE INDEX uq_attendance_line_event
  ON attendance (line_event_id)
  WHERE line_event_id IS NOT NULL;

-- Optional: small table to record processed webhook deliveries (stronger replay guard)
CREATE TABLE IF NOT EXISTS webhook_deliveries (
  delivery_id text PRIMARY KEY,
  tenant_id text NOT NULL,
  received_at timestamptz NOT NULL DEFAULT NOW()
);
```

import { Router, Request, Response } from 'express';
import crypto from 'crypto';
import { pool } from '../../db';
import { validateClockEvent } from '../../utils/validators';
import { Logger } from '../../utils/logger';
import { Counter, Histogram } from '../../utils/metrics';

const router = Router();
const LINE_CHANNEL_SECRET = process.env.LINE_CHANNEL_SECRET || '';

if (!LINE_CHANNEL_SECRET) {
  throw new Error('LINE_CHANNEL_SECRET is required');
}

const log = Logger.child({ service: 'line-webhook' });

const receiptCounter = new Counter({
  name: 'line_webhook_receipt_total',
  help: 'Total LINE webhook receipts',
  labelNames: ['status'],
});

const processingHistogram = new Histogram({
  name: 'line_webhook_processing_seconds',
  help: 'Webhook processing duration',
  buckets: [0.05, 0.1, 0.25, 0.5, 1, 2, 5],
});

function verifySignature(body: Buffer, signature: string): boolean {
  const expected = crypto
    .createHmac('sha256', LINE_CHANNEL_SECRET)
    .update(body)
    .digest('base64');
  return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
}

function intentHash(tenantId: string, employeeId: string, date: string, eventType: string): Buffer {
  return crypto.createHash('sha256')
    .update(`${tenantId}:${employeeId}:${date}:${eventType}`)
    .digest();
}

function advisoryLockId(intentHashBuf: Buffer): bigint {
  // Use first 8 bytes as signed bigint for pg advisory xact lock
  const prefix = intentHashBuf.slice(0, 8);
  const num = BigInt('0x' + prefix.toString('hex'));
  // map into signed 64-bit range
  return (num % BigInt(9223372036854775807n)) - BigInt(9223372036854775807n) / 2n;
}

    log.warn({ signaturePresent: !!signature }, 'Invalid LINE signature');
    return res.status(401).json({ error: 'Invalid signature' });

  let duplicates = 0;
  let errors = 0;

      if (event.type !== 'message' && event.type !== 'postback') continue;

      const { tenantId, employeeId } = await validateClockEvent(event, client);
      if (!tenantId || !employeeId) {
        log.debug({ event }, 'Skipping malformed event');
        continue;

      const today = new Date().toISOString().slice(0, 10);
      const intent = event.postback?.data || event.message?.text || '';

        : intent.includes('OT') ? 'OT'
        : null;

      if (!eventType) {
        log.debug({ intent }, 'Unrecognized intent');
        continue;

      const iHash = intentHash(tenantId, employeeId, today, eventType);
      const lockId = advisoryLockId(iHash);

      // Serialize concurrent intents for same tenant/employee/date/eventType
      const lockRes = await client.query('SELECT pg_try_advisory_xact_lock($1)', [lockId]);
      if (!lockRes.rows[0]?.pg_try_advisory_xact_lock) {
        log.debug({ tenantId, employeeId, today, eventType }, 'Concurrent intent: lock busy');
        receiptCounter.inc({ status: 'conflict' });
        errors++;
        continue;
      }

      // Idempotency: unique constraint will protect, but check first for clearer logs
      const exists = await client.query(
        `SELECT id FROM attendance
         WHERE tenant_id = $1 AND employee_id = $2 AND date = $3 AND event_type = $4`,
        [tenantId, employeeId, today, eventType]
      );

      if (exists.rows.length > 0) {
        log.debug({ tenantId, employeeId, today, eventType }, 'Duplicate intent skipped');
        duplicates++;
        continue;
      }

      const lineEventId = event.message?.id || event.source?.userId || crypto.randomUUID();

      await client.query(
        `INSERT INTO attendance (tenant_id, employee_id, date, event_type, line_event_id, intent_hash, created_at)
         VALUES ($1, $2, $3, $4, $5, $6, NOW())`,
        [tenantId, employeeId, today, eventType, lineEventId, iHash]
      );

      processed++;
      log.info({ tenantId, employeeId, today, eventType, lineEventId }, 'Attendance recorded');
    }

    await client.query('COMMIT');
    receiptCounter.inc({ status: 'ok' });
    res.status(200).json({ ok: true, processed, duplicates });
