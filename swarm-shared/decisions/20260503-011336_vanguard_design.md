# vanguard / design

## Final Synthesis (Best Parts + Correctness + Actionability)

### 1. Diagnosis (Consensus)
- **Problem**: Every page load triggers authenticated `list_repo_tree` calls → burns HF quota and risks 429s.
- **Problem**: Downloads use authenticated API paths instead of public CDN URLs → unnecessary auth overhead and tighter rate limits.
- **Problem**: No persisted `(repo, dateFolder)` manifest → repeated enumeration even when nothing changed.
- **Problem**: No client-side TTL or graceful fallback when HF API is throttled.
- **Problem**: No deterministic file-list embedding for Lightning training runs.

### 2. Solution Overview
Add a lightweight manifest generator + CDN-based fetcher with:
- **Server-side**: `scripts/build_manifest.py` (non-recursive, date-scoped) + FastAPI `/api/refresh-manifest` (rate-limited, token-protected).
- **Client-side**: Prefer persisted manifest from `static/manifest/`; use public CDN URLs; fallback to stale manifest + background refresh.
- **Training**: Deterministic file list export for Lightning runs.

#### `scripts/build_manifest.py`
- **Key improvement**: Non-recursive single call per date folder (avoids pagination/100× calls).
- **Deterministic output**: Sorted file list for reproducible training embeddings.

```python
#!/usr/bin/env python3
"""
Build a lightweight manifest for one repo+date folder.
Usage:
  HF_TOKEN=hf_xxx python scripts/build_manifest.py \
    --repo my-org/my-dataset \
    --date 2026-04-29 \
    --out static/manifest
"""
import argparse
import json
import os
import sys
from datetime import datetime, timezone
from pathlib import Path

try:
    from huggingface_hub import HfApi
except ImportError:
    print("Missing huggingface_hub. Install with: pip install huggingface_hub")
    sys.exit(1)

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def build_manifest(repo: str, date_folder: str, out_dir: Path):
    api = HfApi(token=os.getenv("HF_TOKEN"))
    # Single non-recursive call per date folder (avoids pagination/100x)
    items = api.list_repo_tree(repo=repo, path=date_folder, recursive=False)

    files = []
    for item in items:
        if getattr(item, "type", None) != "file":
            continue
        # item.path is relative to repo root; date_folder already included
        rel_path = item.path  # e.g. "2026-04-29/file.parquet"
        files.append({
            "path": rel_path,
            "cdn": CDN_TEMPLATE.format(repo=repo, path=rel_path),
            "size": getattr(item, "size", None),
        })

    # Deterministic ordering for reproducible training lists
    files.sort(key=lambda f: f["path"])

    manifest = {
        "repo": repo,
        "date": date_folder,
        "generated_at_utc": datetime.now(timezone.utc).isoformat(),
        "files": files,
        "count": len(files),
    }

    out_path = out_dir / repo.replace("/", "_") / f"{date_folder}.json"
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    print(f"Wrote {len(files)} entries to {out_path}")
    return manifest

if __name__ == "__main__":
    p = argparse.ArgumentParser()
    p.add_argument("--repo", required=True, help="e.g. my-org/my-dataset")
    p.add_argument("--date", required=True, help="date folder, e.g. 2026-04-29")
    p.add_argument("--out", default="static/manifest", help="output directory")
    args = p.parse_args()
    build_manifest(args.repo, args.date, Path(args.out))
```

#### `src/lib/hf_cdn.py`
- **Key improvement**: Explicitly strips `Authorization` to avoid leaking tokens to CDN.

```python
import os
import requests
from typing import Optional

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"
TIMEOUT = (3.05, 15)

def cdn_url(repo: str, path: str) -> str:
    return CDN_TEMPLATE.format(repo=repo, path=path)

def fetch_cdn(repo: str, path: str, headers: Optional[dict] = None) -> requests.Response:
    """
    Fetch a dataset file via public CDN (no Authorization header).
    """
    url = cdn_url(repo, path)
    req_headers = headers.copy() if headers else {}
    # Explicitly avoid leaking any HF token to CDN
    req_headers.pop("Authorization", None)
    return requests.get(url, headers=req_headers, timeout=TIMEOUT)
```

#### `api/refresh_manifest.py` (FastAPI router)
- **Key improvement**: Rate-limited in-memory throttle (1/5min per repo:date) to protect HF quota.

```python
from fastapi import APIRouter, HTTPException, Header
from typing import Optional
import os
import json
from pathlib import Path
from datetime import datetime, timezone, timedelta

from scripts.build_manifest import build_manifest

router = APIRouter()

# Simple in-memory throttle: one refresh per repo:date per 5min
_LAST_REFRESH: dict[str, datetime] = {}

@router.post("/refresh-manifest")
async def refresh_manifest(
    repo: str,
    date: str,
    authorization: Optional[str] = Header(None),
):
    # Require HF token for this endpoint (protects list_repo_tree)
    if not authorization and not os.getenv("HF_TOKEN"):
        raise HTTPException(status_code=401, detail="HF token required")

    key = f"{repo}:{date}"
    now = datetime.now(timezone.utc)
    last = _LAST_REFRESH.get(key)
    if last and (now - last) < timedelta(minutes=5):
        raise HTTPException(status_code=429, detail="Refresh too frequent")

    try:
        manifest = build_manifest(repo, date, Path("static/manifest"))
        _LAST_REFRESH[key] = now
        return manifest
    except Exception as exc:
        raise HTTPException(status_code=502, detail=str(exc))
```

#### Frontend Integration (JavaScript)
- **Key improvement**: Prefer persisted manifest; fallback to stale + background refresh; use CDN URLs.

```js
// src/lib/hfClient.js
const MANIFEST_TTL_MS = 5 * 60 * 1000; // 5 minutes

async function loadManifest(repo, date) {
  const encodedRepo = repo.replace(/\//g, "_");
  const manifestUrl = `/static/manifest/${encodedRepo}/${date}.json`;

  // Try cached manifest first
  const cached = await fetch(manifestUrl).then(r => r.ok ? r.json() : null);
  if (cached) {
    const age = Date.now() - new Date(cached.generated_at_utc).getTime();
    if (age < MANIFEST_TTL_MS) {
      return cached; // fresh
    }
    // Stale but usable: background refresh
    refreshManifestInBackground(repo, date).catch(() => {});
    return cached;
  }

  // No manifest: request refresh (debounced by server throttle)
  return await refreshManifestInBackground(repo, date);
}

async function refreshManifestInBackground(repo, date) {
  const res = await fetch("/api/refresh-manifest", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ repo, date }),
  });
  if (!res.ok) throw new Error("Manifest unavailable");
  return res.json();
}

function toCdnUrl(repo, path) {
  return `https://huggingface.co/datasets/${repo}/resolve/main/${path}`;
}
```
