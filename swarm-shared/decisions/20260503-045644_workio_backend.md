# workio / backend

## 1. Diagnosis

- **No idempotency on LINE webhook ingestion**: duplicate webhook deliveries (LINE retries, user double-taps) can create duplicate clock-in/out or leave/OT records and corrupt daily totals.
- **No signature lifetime bound**: LINE signatures are accepted indefinitely; replay attacks possible if signatures are captured and reused.
- **No replay cache**: same `webhookEventId` can be processed multiple times across retries or pod restarts.
- **Clock-in/out endpoint lacks request-scoped idempotency key**: frontend optimistic UI can race with duplicate webhook deliveries and produce double entries.
- **No atomic upsert for daily punch state**: concurrent clock-in attempts (webhook + direct API) can create multiple records for same tenant/employee/date.

## 2. Proposed change

Add backend idempotency + replay protection for LINE webhooks and clock mutations:

- **File**: `workio/server/src/middleware/lineWebhookAuth.ts` (create)
- **File**: `workio/server/src/middleware/idempotency.ts` (create)
- **File**: `workio/server/src/routes/clock.ts` (update)
- **File**: `workio/server/src/db/schema.sql` (add `line_webhook_replays` + idempotency table + unique constraint on `tenant_id, employee_id, date, type` for daily punch state)

### 3.2 LINE webhook auth + replay middleware

`workio/server/src/middleware/lineWebhookAuth.ts`

```ts
import crypto from 'crypto';
import { Request, Response, NextFunction } from 'express';
import { Pool } from 'pg';

const pool = new Pool({ connectionString: process.env.DATABASE_URL });

const LINE_SIGNATURE_HEADER = 'x-line-signature';
const MAX_SIGNATURE_AGE_MS = 5 * 60 * 1000; // 5 minutes lifetime

export function verifyLineSignature(channelSecret: string) {
  return async (req: Request, res: Response, next: NextFunction) => {
    try {
      const signature = req.get(LINE_SIGNATURE_HEADER);
      const body = JSON.stringify(req.body);
      const now = Date.now();

      // Basic checks
      if (!signature || typeof signature !== 'string') {
        return res.status(401).json({ error: 'Missing signature' });
      }

      // Optional: reject very old timestamps if you include a timestamp header or validate via event timestamp downstream.
      // Enforce short signature validity by rejecting replays quickly via cache.

      const expected = crypto
        .createHmac('sha256', channelSecret)
        .update(body)
        .digest('base64');

      if (!crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected))) {
        return res.status(401).json({ error: 'Invalid signature' });
      }

      // Replay protection: ensure each event_id processed once per tenant
      const events = req.body?.events;
      if (!Array.isArray(events) || events.length === 0) {
        return res.status(400).json({ error: 'No events' });
      }

      const tenantId = (req as any).tenantId; // set by tenant resolution middleware earlier
      if (!tenantId) {
        return res.status(400).json({ error: 'Tenant not resolved' });
      }

      const client = await pool.connect();
      try {
        await client.query('BEGIN');
        for (const ev of events) {
          const eventId = ev?.source?.userId && ev?.type ? `${ev.source.userId}:${ev.type}:${ev.timestamp || ''}:${ev.mode || ''}` : ev?.webhookEventId || ev?.type + ':' + Date.now();
          // Use a stable, unique id provided by LINE when available; fallback constructed above.
          const lineEventId = ev.webhookEventId || eventId;
          const exists = await client.query(
            `SELECT 1 FROM line_webhook_replays WHERE tenant_id = $1 AND event_id = $2`,
            [tenantId, lineEventId]
          );
          if (exists.rows.length > 0) {
            await client.query('ROLLBACK');
            // Idempotent success: already processed
            return res.status(200).json({ ok: true });
          }
        }

        // Insert all event_ids for this delivery (use first event id as representative or store each)
        for (const ev of events) {
          const lineEventId = ev.webhookEventId || `${ev.source?.userId || 'unknown'}:${ev.type}:${ev.timestamp || Date.now()}`;
          await client.query(
            `INSERT INTO line_webhook_replays(tenant_id, event_id) VALUES ($1, $2) ON CONFLICT DO NOTHING`,
            [tenantId, lineEventId]
          );
        }

        await client.query('COMMIT');
        next();
      } catch (err) {
        await client.query('ROLLBACK').catch(() => {});
        next(err);
      } finally {
        client.release();
      }
    } catch (err) {
      next(err);
    }
  };
}
```
