# Costinel / backend

## Final Implementation Plan — Top-Hub Signal Panel (Costinel Backend)

**Scope & Value**  
- Backend-only, read-only API endpoint that surfaces the highest-signal / most-connected hub (default “MOC”) and its actionable proposals from the knowledge graph.  
- CDN-first data: a single JSON file (`/data/knowledge-graph/top-hub.json`) generated by the knowledge-rag pipeline and served as a static asset.  
- Endpoint: `GET /api/v1/hubs/top` returns `{ hub, score, signals[], proposals[], updatedAt, generatedAt, version }`.  
- Zero runtime DB queries; safe for high-traffic dashboard.  
- Ships in ~30–90 min: add endpoint + static asset + minimal tests + docs.

---

### Resolved design choices (correctness + actionability)

- **Graceful fallback + boot-time readiness**: fail loudly in dev/test if file missing; in prod, serve a safe default with `_fallback: true` and 200 to keep dashboards alive.  
- **Caching**: `Cache-Control: public, max-age=60, stale-while-revalidate=300` (fast, CDN-friendly).  
- **Schema**: include `score`, `version`, `generatedAt`, `updatedAt`, and lightweight shape validation.  
- **Route mounting**: explicit route handler (not bare `express.static`) for validation, fallback, and headers.  
- **Pipeline handoff**: documented requirement for `knowledge-rag` to overwrite the JSON file on each successful run.  
- **Tests + docs**: minimal Supertest shape test + OpenAPI snippet to ensure correctness and discoverability.

### Route: `src/routes/topHubRoute.js`

```js
const express = require('express');
const fs = require('fs').promises;
const path = require('path');

const router = express.Router();

const TOP_HUB_PATH = path.join(
  process.cwd(),
  'data',
  'knowledge-graph',
  'top-hub.json'
);

const DEFAULT_TOP_HUB = {
  hub: 'MOC',
  score: 0,
  version: '1.0',
  generatedAt: new Date().toISOString(),
  updatedAt: new Date().toISOString(),
  signals: [],
  proposals: [],
  _fallback: true
};

/**
 * Light schema validation
 */
function isValidPayload(p) {
  return (
    p &&
    typeof p.hub === 'string' &&
    typeof p.score === 'number' &&
    Array.isArray(p.signals) &&
    Array.isArray(p.proposals)
  );
}

/**
 * GET /api/v1/hubs/top
 * Returns top-connected hub and actionable signals from knowledge graph.
 * Data is served from static JSON generated by knowledge-rag pipeline.
 */
router.get('/api/v1/hubs/top', async (req, res) => {
  try {
    const raw = await fs.readFile(TOP_HUB_PATH, 'utf8');
    const payload = JSON.parse(raw);

    if (!isValidPayload(payload)) {
      throw new Error('Invalid top-hub schema');
    }

    // Ensure updatedAt exists for clients
    const response = {
      ...payload,
      updatedAt: payload.updatedAt || payload.generatedAt || new Date().toISOString()
    };

    res.set({
      'Cache-Control': 'public, max-age=60, stale-while-revalidate=300',
      'Content-Type': 'application/json'
    });
    return res.json(response);
  } catch (err) {
    // Graceful fallback if file missing/corrupt
    console.warn('[topHubRoute] Could not load top-hub.json, serving fallback:', err.message);

    // In dev/test, consider failing hard via process.exit(1) if desired.
    // For production resilience, return safe default.
    const isProd = process.env.NODE_ENV === 'production';
    const code = isProd ? 200 : 503;
    const payload = isProd ? DEFAULT_TOP_HUB : { error: 'top-hub unavailable', details: err.message };

    res.set({
      'Cache-Control': 'public, max-age=30',
      'Content-Type': 'application/json'
    });
    return res.status(code).json
