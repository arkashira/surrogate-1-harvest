## Deployment
cf workers (free tier) + Supabase (free tier) + optional HF space for config distribution

## v0.1 scope
v0.1 ships a cross-platform CLI daemon (Linux/macOS/Windows) that: 1) starts a local SOCKS5 listener wrapped with uTLS/obfs4-like framing to mask handshake; 2) creates a userspace TUN via boringtun and routes selected traffic through it; 3) enforces a process-level kill-switch that blocks non-tunneled traffic on unexpected interface loss; 4) stores configs in local encrypted SQLite and syncs minimal, hashed session telemetry to Supabase; 5) discovers bridge endpoints from an HF-space-hosted, signed catalog fetched over DoH with randomized fronting domains; 6) provides zero-config install (single binary) and a basic smoke test to confirm WAU-ZB telemetry submission.

## Open questions

- How to implement reliable fail-closed kill-switch across Windows/macOS/Linux without admin/root?
- What minimal telemetry schema preserves privacy while enabling WAU-ZB measurement?
