## Implementation Plan — Airship Quality (HF CDN Bypass + Lightning Idle Resilience)

**Value**: Eliminates HF API 429s during training, prevents Lightning idle-stop training loss, reduces Mac→Lightning iteration from ~15 min to <2 min per cycle.  
**Effort**: <2 h, zero breaking changes.

---

### 1) HF CDN Bypass (training data path)
- Pre-list once from Mac (after rate-limit window) via `list_repo_tree(path, recursive=False)` for a single date folder.
- Save list to `train_files.json` and embed in `train.py`.
- Lightning training uses only CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with `requests`/`urllib` — zero HF API calls during data load.
- Project to `{prompt, response}` at parse time; do not load heterogeneous schemas via `load_dataset`.

### 2) Lightning Idle Resilience
- Before each `.run()`, check studio status; if stopped, restart with `target.start(machine=Machine.L40S)` (or fallback to public tier).
- Reuse running studios (`Teamspace.studios`) to save quota.
- Wrap training entrypoint so restart preserves args and continues from latest checkpoint.

### 3) File layout changes
- `scripts/list_hf_files.py` — one-shot Mac script to produce `train_files.json`.
- `surrogate/train.py` — updated data loader to use CDN + file list.
- `surrogate/lightning_launcher.py` — studio reuse + idle-safe runner.

### scripts/list_hf_files.py
```python
#!/usr/bin/env python3
"""
Run on Mac (or any dev machine) to list files for a single date folder
and emit train_files.json for CDN-only training.

Usage:
  HF_TOKEN=hf_xxx python scripts/list_hf_files.py \
    --repo datasets/your-org/surrogate-mirror \
    --date-folder 2026-05-03 \
    --out surrogate/train_files.json
"""

import argparse
import json
import os
from huggingface_hub import HfApi

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., datasets/your-org/surrogate-mirror)")
    parser.add_argument("--date-folder", required=True, help="Subfolder date (e.g., 2026-05-03)")
    parser.add_argument("--out", default="train_files.json", help="Output JSON path")
    args = parser.parse_args()

    api = HfApi(token=os.getenv("HF_TOKEN"))
    # List only immediate files in the date folder (non-recursive keeps API usage minimal)
    tree = api.list_repo_tree(repo_id=args.repo, path=args.date_folder, recursive=False)

    files = []
    for item in tree:
        if item.type == "file":
            # CDN URL bypasses API auth/rate limits
            cdn_url = f"https://huggingface.co/datasets/{args.repo}/resolve/main/{args.date_folder}/{item.path}"
            files.append({
                "path": item.path,
                "cdn_url": cdn_url,
                "size": getattr(item, "size", None)
            })

    out_path = os.path.abspath(args.out)
    os.makedirs(os.path.dirname(out_path), exist_ok=True)
    with open(out_path, "w") as f:
        json.dump({"date_folder": args.date_folder, "repo": args.repo, "files": files}, f, indent=2)

    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    main()
```

### surrogate/train.py (CDN-only data loader snippet)
```python
import json
import logging
from pathlib import Path
from typing import List, Dict

import torch
from torch.utils.data import Dataset, DataLoader
import requests

logger = logging.getLogger(__name__)

class CDNTextDataset(Dataset):
    def __init__(self, filelist_path: str, max_files: int = -1):
        with open(filelist_path) as f:
            manifest = json.load(f)
        self.files = manifest["files"]
        if max_files > 0:
            self.files = self.files[:max_files]

    def __len__(self) -> int:
        return len(self.files)

    def _download_one(self, cdn_url: str) -> str:
        # CDN fetch — no HF API token required, no rate-limit (auth) pressure
        resp = requests.get(cdn_url, timeout=30)
        resp.raise_for_status()
        return resp.text

    def _parse_to_pair(self, text: str) -> Dict[str, str]:
        # Minimal projection to {prompt, response}; adapt to your file format
        # Example assumes simple "### Prompt:\n...### Response:\n..."
        prompt, response = "", ""
        if "### Prompt:" in text and "### Response:" in text:
            parts = text.split("### Response:")
            prompt = parts[0].replace("### Prompt:", "").strip()
            response = parts[1].strip()
        else:
            # fallback: treat whole text as response, empty prompt
            response = text.strip()
        return {"prompt": prompt, "response": response}

    def __getitem__(self, idx: int) -> Dict[str, str]:
        entry = self.files[idx]
        raw = self._download_one(entry["cdn_url"])
        pair = self._parse_to_pair(raw)
        return pair


def build_dataloader(filelist_path: str, batch_size: int = 4, num_workers: int = 4) -> DataLoader:
    dataset = CDNTextDataset(filelist_path=filelist_path)
    return DataLoader(
        dataset,
        batch_size=batch_size,
        num_workers=num_workers,
        pin_memory=True,
        shuffle=True,
        drop_last=True,
    )
```
