## v0.1 scope
v0.1 ships a cross-platform CLI stealth client (Rust) that: 1) auto-detects and randomizes local SOCKS5/TUN artifacts, 2) tunnels traffic via obfuscated WebSocket-over-Cloudflare Workers to a free-tier worker endpoint, 3) enforces a local kill-switch that blocks non-tunneled traffic on failure, 4) stores minimal encrypted telemetry to Supabase and reports per-session blocked=false for WAU-ZB, 5) updates blocklists/obfuscation profiles from Workers KV — all deployable with zero server control and no admin rights.

## Open questions
- Which obfuscation primitives (obfs4 vs meek vs custom HTTP/QUIC mimicry) maximize reach while remaining implementable client-side only?
- How will client-side DPI probes detect and classify local artifacts (TUN, SOCKS5 ports, packet timing) without server cooperation?
- What is the minimal, privacy-preserving telemetry schema that yields WAU-ZB without deanonymizing users?
- Can we enforce a reliable kill-switch across desktop/mobile without kernel extensions or admin rights?
- How do we distribute and update blocklists/obfuscation profiles on the edge without revealing endpoints to enumerating adversaries?
