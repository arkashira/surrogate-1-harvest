## Deployment
hybrid (local CLI for private analysis; optional Supabase for workspace/team metadata; optional HF Spaces for hosted UI/demo)

## v0.1 scope
v0.1 ships a CLI that performs local, private analysis on Python and JavaScript/TypeScript repos using tree-sitter parsing and lightweight heuristics + local embeddings for semantic findings; outputs severity-ranked findings (security/performance/refactor) to terminal and JSON; supports workspace metadata stored in local SQLite (with optional Supabase sync flag); installable via pip; runs on macOS/Linux; no cloud processing of repository code.
