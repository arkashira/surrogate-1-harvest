# Costinel / design

## Final Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h, telemetry-aware)

### What we ship
- A **non-blocking Top-Hub Signal Panel** on the Costinel dashboard.
- Defaults to hub **MOC** (configurable via `NEXT_PUBLIC_HUB_NAME`).
- Shows **3 actionable, cost-impact proposals** (title, impact, confidence, source).
- **CDN-first data strategy**: pre-generated file list at build time; runtime fetches via `https://huggingface.co/datasets/.../resolve/main/...` (no API auth, bypasses 429).
- **Telemetry-aware**: emits `panel_open`, `proposal_click`, `refresh_failed` via existing telemetry hook.
- **Graceful fallbacks**: local stub data if CDN or file list unavailable; never blocks or crashes dashboard.

---

### Implementation Steps (≤2h)

1. **Add config & env** (5 min)
   - Add `NEXT_PUBLIC_HUB_NAME=MOC` to `.env.example` and runtime env.
   - Add `NEXT_PUBLIC_HUB_DATA_DATE` (default: current date).

2. **Generate file-list artifact** (15 min)
   - One-time script (`scripts/generate-hub-filelist.sh`) to list hub date folder.
   - Save `public/data/top-hub-filelist.json` (committed or CI-generated).
   - Format: `{ "hub": "MOC", "date": "2026-05-03", "files": [...] }`.

3. **Create CDN fetcher** (20 min)
   - `src/lib/topHubApi.ts` — uses `fetch` with CDN URLs.
   - Loads file list, picks first signals JSON, normalizes to `HubProposal[]`.
   - Retries with exponential backoff; falls back to stub data.

4. **Build TopHubSignalPanel component** (40 min)
   - `src/components/TopHubSignalPanel.tsx` — React, non-blocking, skeleton loader.
   - Shows 3 proposal cards; errors do not crash dashboard.

5. **Wire telemetry** (10 min)
   - Emit `panel_open`, `proposal_click`, `refresh_failed` via existing `useTelemetry()`.

6. **Lightning Studio guard** (10 min)
   - If running in Lightning Studio, respect idle/stop lifecycle (no training here; defensive guard only).

7. **Tests & smoke** (20 min)
   - Local dev smoke test; verify CDN fetch and fallback rendering.

#### 2. File-list generator (run on Mac/CI)
```bash
#!/usr/bin/env bash
# scripts/generate-hub-filelist.sh
set -euo pipefail

HUB_NAME="${HUB_NAME:-MOC}"
DATE="${DATE:-$(date +%Y-%m-%d)}"
OUT_DIR="public/data"
OUT_FILE="${OUT_DIR}/top-hub-filelist.json"

mkdir -p "$OUT_DIR"

python3 - <<PY
import os, json
from huggingface_hub import list_repo_tree

hub = os.getenv("HUB_NAME", "MOC")
date = os.getenv("DATE", "$(date +%Y-%m-%d)")
path = f"{hub}/{date}"

try:
    tree = list_repo_tree(repo_id="AXENTX/Costinel-HubData", path=path, recursive=False)
    files = [f.rfilename for f in tree if f.type == "file"]
except Exception:
    files = [f"{path}/signals.json"]

out = {"hub": hub, "date": date, "files": files}
with open(os.getenv("OUT_FILE", "$OUT_FILE"), "w") as fp:
    json.dump(out, fp, indent=2)
PY

echo "Generated $OUT_FILE"
```

#### 3. CDN fetcher (browser)
```ts
// src/lib/topHubApi.ts
import { CDN_BASE, FILE_LIST_PATH } from '@/config/topHub';

export interface HubProposal {
  title: string;
  impact: string;
  confidence: number;
  source: string;
}

async function fetchJsonCdn(url: string): Promise<any> {
  const res = await fetch(url, { cache: 'no-store' });
  if (!res.ok) throw new Error(`CDN fetch failed: ${res.status}`);
  return res.json();
}

export async function fetchTopHubProposals(): Promise<HubProposal[]> {
  try {
    const listRes = await fetch(FILE_LIST_PATH, { cache: 'no-store' });
    if (!listRes.ok) throw new Error('No file list');
    const { files } = await listRes.json();

    const signalsFile = files.find((f: string) => f.includes('signal') || f.endsWith('.json')) || files[0];
    if (!signalsFile) throw new Error('No signals file');

    const dataUrl = `${CDN_BASE}/${signalsFile}`;
    const raw = await fetchJsonCdn(dataUrl);

    const items = Array.isArray(raw) ? raw : (raw.items || raw.proposals || raw.results || []);
    return items.slice(0, 3).map((item: any) => ({
      title: item.title || item.name || 'Untitled proposal',
      impact: item.impact || item.savings || item.recommendation || 'Impact unknown',
      confidence: Number(item.confidence || item.score || 0.5),
      source: item.source || signalsFile,
    }));
  } catch (err) {
    console.warn('Hub CDN fetch failed, using stub:', err);
    return [
      { title: 'Review idle EC2 instances (MOC)', impact: '$8k/mo potential savings', confidence: 0.78, source: 'stub' },
      { title: 'Right-size over-provisioned RDS', impact: '$4k/mo potential savings', confidence: 0.65, source: 'stub' },
      { title: 'Commit to 1yr Savings Plan for steady workloads', impact: '$15k/mo potential savings', confidence: 0.82, source: 'stub' },
    ];
  }
}
```
