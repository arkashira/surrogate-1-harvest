# vanguard / design

**Final synthesized implementation (single, correct, actionable)**

**Root cause**  
- No persisted `(repo, dateFolder)` manifest → every training run re-enumerates via authenticated HF API → quota burn + 429 risk.  
- Data loading couples training to HF API availability instead of using CDN-only fetches.  
- Lightning Studio reuse is not enforced → idle-stop kills training and recreation burns quota.  
- Mixed-schema ingestion artifacts may leak extra metadata and break downstream schema contracts.

**Single change to make**  
Add a manifest layer and a CDN-only training path, plus Lightning Studio reuse with idle resilience, scoped to one repo/date (e.g. `smithery/surrogate-1`, `2026-04-29`).

### 1) Manifest module (single authenticated call, persisted, CDN URLs)

`/opt/axentx/vanguard/design/manifest.py`
```python
import json
from pathlib import Path
from typing import Dict, List
from huggingface_hub import HfApi

HF_API = HfApi()
MANIFEST_ROOT = Path(__file__).parent / "manifests"
MANIFEST_ROOT.mkdir(exist_ok=True, parents=True)

def _repo_slug(repo: str) -> str:
    return repo.replace("/", "_")

def build_manifest(repo: str, date_folder: str, revision: str = "main") -> Dict:
    """
    Single authenticated call to list one date folder (non-recursive).
    Persists manifest JSON with CDN URLs.
    """
    manifest_path = MANIFEST_ROOT / _repo_slug(repo) / f"{date_folder}.json"
    manifest_path.parent.mkdir(parents=True, exist_ok=True)

    if manifest_path.exists():
        with open(manifest_path) as f:
            return json.load(f)

    items = HF_API.list_repo_tree(
        repo=repo,
        path=date_folder,
        revision=revision,
        recursive=False,
    )
    files = [it.rfilename for it in items if it.type == "file"]

    entries: List[Dict] = []
    for f in sorted(files):
        entries.append({
            "file": f,
            "cdn_url": f"https://huggingface.co/datasets/{repo}/resolve/main/{date_folder}/{f}",
        })

    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "revision": revision,
        "entries": entries,
    }

    with open(manifest_path, "w") as f:
        json.dump(manifest, f, indent=2)
    return manifest


def load_manifest(repo: str, date_folder: str) -> Dict:
    manifest_path = MANIFEST_ROOT / _repo_slug(repo) / f"{date_folder}.json"
    if not manifest_path.exists():
        raise FileNotFoundError(f"Manifest missing: {manifest_path}")
    with open(manifest_path) as f:
        return json.load(f)
```

### 2) Training script (CDN-only dataset + Studio reuse + schema projection)

`/opt/axentx/vanguard/design/train.py`
```python
import os
import json
import requests
from pathlib import Path
from typing import Dict, Iterator
from datasets import Dataset, Features, Value
from lightning import Studio, Teamspace

from .manifest import build_manifest, load_manifest

HF_REPO = os.getenv("HF_REPO", "smithery/surrogate-1")
DATE_FOLDER = os.getenv("DATE_FOLDER", "2026-04-29")

# ---- Build or load manifest once per run ----
MANIFEST = build_manifest(HF_REPO, DATE_FOLDER)

# ---- CDN-only iterator; project to surrogate-1 schema ----
def _cdn_iter(manifest: Dict) -> Iterator[Dict[str, str]]:
    for entry in manifest["entries"]:
        url = entry["cdn_url"]
        try:
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()

            # Surrogate-1 contract: keep only {prompt, response}.
            # Assume each file is JSONL with at least these fields.
            for line in resp.text.strip().splitlines():
                if not line.strip():
                    continue
                obj = json.loads(line)
                # Project to contract; drop extra metadata.
                yield {
                    "prompt": obj.get("prompt", ""),
                    "response": obj.get("response", ""),
                }
        except Exception as exc:
            # Log and continue; don't fail entire run on one file.
            print(f"[cdn_iter] failed {url}: {exc}")


def build_dataset(manifest: Dict) -> Dataset:
    features = Features({
        "prompt": Value("string"),
        "response": Value("string"),
    })
    return Dataset.from_generator(
        lambda: _cdn_iter(manifest),
        features=features,
    )


# ---- Lightning Studio reuse with idle resilience ----

            return s
```

**Result**  
- One authenticated HF API call per `(repo, dateFolder)` → quota-safe.  
- Training uses CDN-only URLs → immune to HF API rate limits during data loading.  
- Lightning Studio reuse enforced → no recreation burn.  
- Surrogate-1 schema contract enforced at load time → no metadata leakage.
