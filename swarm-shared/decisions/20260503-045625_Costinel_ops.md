# Costinel / ops

## Final Implementation Plan — Top Hub Signal Panel (CDN-first)

**Scope**: Frontend-only addition to Costinel dashboard.  
**Effort**: ~60–90 minutes.  
**Mechanism**: CDN JSON fetch (no auth, no backend) with local fallback and graceful degradation. Fails open (never blocks dashboard).

- `src/components/TopHubSignalPanel.tsx` — new component (TypeScript)

### 2) Payload contract (CDN JSON)

**CDN path** (public):  
`https://huggingface.co/datasets/AXENTX/Costinel/resolve/main/public/signals/top-hub.json`

**Local fallback** (`public/signals/top-hub.json`):

```json
{
  "hub": "MOC",
  "title": "Most-Connected Hub",
  "insight": "MOC shows the strongest cross-service cost coupling this week. Prioritize RI coverage and idle-rightsizing signals for MOC-linked accounts to unlock the fastest savings.",
  "updated": "2026-05-03T04:51:53Z",
  "ttl_minutes": 1440
}
```

**Rationale**:
- Stored in `public/` so it is served by the app itself as a local fallback.
- Small, cacheable, and easy to update via CI/CD or ops automation.
- `ttl_minutes` lets the frontend decide whether to re-fetch CDN.

### 3) Component implementation

`src/components/TopHubSignalPanel.tsx`

```tsx
import React, { useEffect, useState } from "react";
import "./top-hub-signal.css";

const CDN_URL =
  "https://huggingface.co/datasets/AXENTX/Costinel/resolve/main/public/signals/top-hub.json";
const LOCAL_URL = "/signals/top-hub.json";

type TopHubPayload = {
  hub: string;
  title?: string;
  insight: string;
  updated?: string;
  ttl_minutes?: number;
};

const TopHubSignalPanel: React.FC = () => {
  const [data, setData] = useState<TopHubPayload | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    let cancelled = false;

    async function fetchFromCDNThenLocal() {
      try {
        const res = await fetch(CDN_URL, { cache: "no-store" });
        if (!res.ok) throw new Error("CDN unavailable");
        const json = (await res.json()) as TopHubPayload;
        if (!cancelled) {
          setData(json);
          setLoading(false);
        }
      } catch {
        // CDN failed — fallback to local
        try {
          const res = await fetch(LOCAL_URL, { cache: "no-store" });
          if (!res.ok) throw new Error("Local unavailable");
          const json = (await res.json()) as TopHubPayload;
          if (!cancelled) {
            setData(json);
            setLoading(false);
          }
        } catch {
          // Silent fail — panel will render nothing (fails open)
          if (!cancelled) {
            setData(null);
            setLoading(false);
          }
        }
      }
    }

    fetchFromCDNThenLocal();
    return () => {
      cancelled = true;
    };
  }, []);

  if (loading) {
    return <div className="top-hub-panel loading">Loading insights…</div>;
  }

  if (!data) {
    return null;
  }

  return (
    <div
      className="top-hub-panel"
      title={data.updated ? `Updated ${new Date(data.updated).toLocaleString()}` : undefined}
    >
      <div className="top-hub-badge">{data.hub}</div>
      <div className="top-hub-body">
        {data.title && <div className="top-hub-label">{data.title}</div>}
        <div className="top-hub-insight">{data.insight}</div>
        <div className="top-hub-source">via knowledge-rag</div>
      </div>
    </div>
  );
};

export default TopHubSignalPanel;
```
