# airship / quality

**Final consolidated implementation (best of Candidate 1 + Candidate 2)**

- **Goal**: Eliminate HF API 429s during Surrogate-1 training by replacing `load_dataset(streaming=True)` with a CDN-only `IterableDataset` that reads from a pre-generated `file_list.json`.  
- **Why this ships fast**: single-file dataset addition + one-line training change; reuses existing CDN bypass pattern and `list_repo_tree` workflow; zero API calls during training.

## 1) Add CDN dataset (single file)

Create: `surrogate/data/cdn_dataset.py`

```python
# surrogate/data/cdn_dataset.py
import json
import urllib.request
import io
import torch
from typing import Iterator, Dict, Any
from torch.utils.data import IterableDataset
import pyarrow.parquet as pq

class CdnIterableDataset(IterableDataset):
    """
    CDN-only IterableDataset for Surrogate-1 training.
    Reads parquet files from HF CDN via resolve/main/ URLs.
    file_list.json format: [{"path": "batches/mirror-merged/2026-04-29/slug.parquet"}, ...]
    Yields: {"prompt": str, "response": str}
    """

    def __init__(
        self,
        repo: str,
        file_list_path: str,
        max_retries: int = 3,
        timeout: int = 30,
    ):
        self.repo = repo.rstrip("/")
        self.timeout = timeout
        self.max_retries = max_retries

        with open(file_list_path, "r", encoding="utf-8") as f:
            items = json.load(f)
        self.paths = [item["path"] if isinstance(item, dict) else item for item in items]
        if not self.paths:
            raise ValueError("file_list.json is empty")

    def _cdn_url(self, path: str) -> str:
        return f"https://huggingface.co/datasets/{self.repo}/resolve/main/{path.lstrip('/')}"

    def _fetch_parquet(self, url: str) -> bytes:
        for attempt in range(self.max_retries):
            try:
                req = urllib.request.Request(
                    url, headers={"User-Agent": "axentx-airship-cdn/1.0"}
                )
                with urllib.request.urlopen(req, timeout=self.timeout) as resp:
                    return resp.read()
            except Exception as exc:
                if attempt == self.max_retries - 1:
                    raise RuntimeError(f"Failed to fetch {url} after {self.max_retries} attempts") from exc
        raise RuntimeError("Unreachable")

    def _project_record(self, batch: Dict[str, Any]) -> Iterator[Dict[str, str]]:
        prompts = batch.get("prompt", [])
        responses = batch.get("response", [])
        if not prompts or not responses:
            return
        for p, r in zip(prompts, responses):
            if p is not None and r is not None:
                yield {"prompt": str(p), "response": str(r)}

    def __iter__(self) -> Iterator[Dict[str, str]]:
        worker_info = torch.utils.data.get_worker_info()
        if worker_info is None:
            iter_paths = self.paths
        else:
            per_worker = max(1, len(self.paths) // worker_info.num_workers)
            worker_id = worker_info.id
            iter_paths = self.paths[worker_id * per_worker : (worker_id + 1) * per_worker]

        for path in iter_paths:
            url = self._cdn_url(path)
            try:
                data = self._fetch_parquet(url)
                table = pq.read_table(io.BytesIO(data))
                batch = table.to_pydict()
                yield from self._project_record(batch)
            except Exception as exc:
                import logging
                logging.warning("Skipping %s: %s", path, exc)
                continue
```

## 3) Generate `file_list.json` (one-time, run after rate-limit window clears)

Create: `surrogate/scripts/gen_file_list.py`

```python
#!/usr/bin/env python3
"""
Generate file_list.json for CDN-only training.
Run on Mac after HF rate-limit window clears.

Usage:
  python surrogate/scripts/gen_file_list.py \
    --repo org/surrogate-mirror \
    --date 2026-04-29 \
    --out surrogate/data/file_list.json
"""

import argparse
import json
from pathlib import Path

try:
    from huggingface_hub import HfApi
except ImportError:
    raise RuntimeError("huggingface_hub required to generate file_list.json")

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate CDN file list for Surrogate-1 training")
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g. org/surrogate-mirror)")
    parser.add_argument("--date", required=True, help="Date folder (e.g. 2026-04-29)")
    parser.add_argument("--out", required=True, help="Output JSON path")
    args = parser.parse_args()

    api = HfApi()
    prefix = f"batches/mirror-merged/{args.date}/"
    entries = api.list_repo_tree(repo_id=args.repo, path=prefix, repo_type="dataset", recursive=False)

    paths = [
        {"path": entry.path}
        for entry in entries
        if entry.path.lower().endswith(".parquet")
    ]
    if not paths:
        raise RuntimeError(f"No parquet files found under {prefix}")

    out_path = Path(args.out)
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(paths, indent=2), encoding="utf-8")
    print(f"Wrote {len(paths)} files to {out_path}")

if __name__ == "__main__":
    main()
```

Run once:

```bash
python surrogate/scripts/gen_file_list.py \
  --repo org/surrogate-mirror \
  --date 2026-04-29 \
  --out surrogate/data/file_list.json
```

## 5) Summary of resolved choices

- **Dataset base class**: `IterableDataset` (supports streaming + multi-worker) — chosen over simpler list-based iterators to preserve DataLoader compatibility.
- **CDN URL pattern**: `resolve/main/` (no auth required) — avoids HF API entirely during training.
- **Parquet reading**: `pyarrow.parquet` (fast, schema-safe) with projection to `{prompt, response}` at parse time.
- **Error handling**: skip bad files with warning instead of crashing training.
- **Worker sharding**: implemented per-worker path slicing to avoid duplicate reads in multi-worker DataLoader.
- **Generation script**: guarded import and explicit prefix filtering for safety and clarity.
