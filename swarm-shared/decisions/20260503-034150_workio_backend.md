# workio / backend

## 1. Diagnosis
- **Idempotency missing**: LINE retries on 5xx/timeouts can replay webhooks and corrupt clock-in/out or leave/OT state (duplicate punches, double approvals).
- **Signature verification absent**: Webhook handler does not verify `X-Line-Signature`, allowing spoofed events to mutate state.
- **No client-side subscription layer**: Frontend relies on local React state + manual refresh; multi-tab/devices diverge after LINE webhook commits.
- **No observable telemetry**: Silent failures, retries, and latency for `/webhook/line` are invisible to operators.
- **No structured logging**: Hard to debug signature mismatches or payload issues during incidents.

## 2. Proposed change
File: `/opt/axentx/workio/server/src/routes/webhook/line.ts` (or create if missing).  
Scope:
- Add `X-Line-Signature` HMAC-SHA256 verification.
- Add idempotency guard using `X-Line-Delivery` or event-level dedupe key (event ID + type) with a fast cache (Redis or in-memory LRU) to reject replays within a safe window.
- Emit structured logs + metrics (request/response, signature result, idempotency hit/miss, processing duration).
- Return deterministic 200 quickly (ack), process async to avoid LINE retries.

## 3. Implementation

Create/update `server/src/routes/webhook/line.ts`:
```ts
import express, { Request, Response, NextFunction } from 'express';
import crypto from 'crypto';
import pino from 'pino';
import { LRUCache } from 'lru-cache';

const logger = pino({ name: 'line-webhook' });
const router = express.Router();

// Idempotency cache: event-based dedupe for 24h
const dedupe = new LRUCache<string, boolean>({
  max: 10_000,
  ttl: 1000 * 60 * 60 * 24,
});

const CHANNEL_SECRET = process.env.LINE_CHANNEL_SECRET || '';
if (!CHANNEL_SECRET) {
  logger.warn('LINE_CHANNEL_SECRET not set — signature verification disabled');
}

function verifySignature(body: string, signature: string): boolean {
  if (!CHANNEL_SECRET) return false;
  const expected = crypto
    .createHmac('sha256', CHANNEL_SECRET)
    .update(body, 'utf8')
    .digest('base64');
  return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
}

function buildDedupeKey(event: any): string | null {
  // Prefer delivery id if present; else stable event id
  if (event.deliveryId) return `delivery:${event.deliveryId}`;
  if (event.message?.id && event.type) return `msg:${event.type}:${event.message.id}`;
  if (event.postback?.data && event.timestamp) {
    // postback dedupe: hash of data+timestamp (same postback can be replayed)
    return `pb:${crypto.createHash('sha256').update(event.postback.data + event.timestamp).digest('hex')}`;
  }
  return null;
}

async function handleEventAsync(event: any) {
  // Lightweight dispatcher — keep fast; delegate to domain services
  // Example: clock-in/out, leave/OT requests
  logger.info({ eventType: event.type, source: event.source }, 'processing LINE event async');
  try {
    // TODO: delegate to application services (clockService, leaveService, etc.)
    // Ensure those services are idempotent at domain level too.
  } catch (err) {
    logger.error({ err, eventType: event.type }, 'async event handler failed');
  }
}

router.post('/line', express.raw({ type: 'application/json' }), (req: Request, res: Response, next: NextFunction) => {
  const start = Date.now();
  const signature = req.get('X-Line-Signature') || '';
  const bodyRaw = req.body.toString('utf8');
  const ok = verifySignature(bodyRaw, signature);

  logger.info({
    path: req.path,
    method: req.method,
    signaturePresent: !!signature,
    signatureOk: ok,
    contentLength: req.get('content-length'),
  }, 'webhook received');

  if (!ok) {
    logger.warn({ signature }, 'invalid LINE signature');
    res.status(401).json({ error: 'invalid signature' });
    return;
  }

  let events;
  try {
    events = JSON.parse(bodyRaw).events;
    if (!Array.isArray(events)) throw new Error('events missing or not array');
  } catch (err) {
    logger.error({ err }, 'invalid LINE payload');
    res.status(400).json({ error: 'invalid payload' });
    return;
  }

  // Idempotency check + async dispatch
  let accepted = 0;
  let duplicates = 0;
  for (const ev of events) {
    const key = buildDedupeKey(ev);
    if (key && dedupe.has(key)) {
      duplicates++;
      continue;
    }
    if (key) dedupe.set(key, true);
    accepted++;
    // Fire-and-forget (but catch to avoid unhandled rejections)
    Promise.resolve().then(() => handleEventAsync(ev)).catch((err) => {
      logger.error({ err, eventType: ev.type }, 'async dispatch threw');
    });
  }

  const duration = Date.now() - start;
  logger.info({ accepted, duplicates, durationMs: duration }, 'webhook processed');

  // Always 200 to stop LINE retries; processing continues async
  res.status(200).json({ ok: true, accepted, duplicates });
});

export default router;
```

Wire into main Express app (`server/src/app.ts` or `server/src/index.ts`):
```ts

app.use('/webhook/line', lineWebhook);
```

## 4. Verification
1. Start backend:
   ```bash
   cd /opt/axentx/workio/server
   npm run dev
   ```
2. Send a test payload with curl (simulate LINE):
   ```bash
   BODY='{"events":[{"type":"message","source":{"userId":"U123"},"message":{"id":"12345","type":"text","text":"test"},"timestamp":1712345678901}]}'
   SIG=$(echo -n "$BODY" | openssl dgst -sha256 -hmac "your_channel_secret_here" -binary | base64)
   curl -v -X POST http://localhost:3000/webhook/line \
     -H "Content-Type: application/json" \
     -H "X-Line-Signature: $SIG" \
     -d "$BODY"
   ```
   Expect: `200 OK` and JSON `{"ok":true,"accepted":1,"duplicates":0}`.  
   Resend same payload: expect `accepted:0, duplicates:1`.

3. Check logs for structured entries:
   - `webhook received` with `signatureOk: true`
   - `webhook processed` with `accepted`, `duplicates`, `durationMs`
   - No `invalid signature` warnings for valid requests.

4. Confirm resilience:
   - Stop DB or downstream service temporarily — webhook should still return 200 (async processing may fail but LINE won’t retry).
   - Verify duplicate events (same deliveryId or message id) are rejected within 24h.
