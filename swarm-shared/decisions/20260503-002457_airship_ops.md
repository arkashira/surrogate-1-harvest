# airship / ops

## Decision: highest-value ops fix (<2h)

Replace any HF API–driven dataset ingestion in the surrogate training pipeline with a **manifest-driven, CDN-only iterable loader** (generated once on the Mac orchestrator) to eliminate 429 rate limits and allow Lightning Studio training to run with zero HF API calls.

This unblocks immediate training runs, removes quota/timeout risk, and aligns with the “HF CDN bypass” and “pre-list once, embed” patterns.

---

## Concrete implementation plan (≤2h)

1. **Add manifest generator script** (`scripts/build_dataset_manifest.py`)  
   - Runs on Mac (or cron) after rate-limit window clears.  
   - Uses `list_repo_tree(path, recursive=False)` per date folder (non-recursive to avoid pagination explosion).  
   - Downloads only file names (not contents) and emits `manifest-{date}.json` listing `{repo, path, sha, size}` for files to consume.  
   - Stores manifest in `manifests/` (committed or artifact) for reproducibility.

2. **Create CDN-only iterable dataset wrapper** (`surrogate/data/hf_cdn_dataset.py`)  
   - Accepts a manifest JSON.  
   - Iterates files and streams each via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth header).  
   - Parses each file (parquet/jsonl) and yields `{prompt, response}`; drops other columns.  
   - Retries with exponential backoff on CDN failures; respects `Retry-After` if present.

3. **Update training entrypoint** (`surrogate/train.py`)  
   - Accept `--manifest` arg (default: latest in `manifests/`).  
   - Build `HFCDNDataset(manifest)` and wrap with `torch.utils.data.DataLoader` (or HF `IterableDataset`).  
   - Remove any `load_dataset(streaming=True)` or recursive `list_repo_files` calls.

4. **Lightning Studio integration**  
   - Reuse running Studio when present (`Teamspace.studios` check).  
   - Ensure `target.start(machine=Machine.L40S)` if stopped (avoid idle-timeout death).  
   - Mount `manifests/` via `lightning.Tar`/`lightning.upload` or include in repo so Studio sees same manifest.

5. **Ops hardening**  
   - Add shebang + `chmod +x` to any wrapper scripts (cron/active-learning/opus-pr-reviewer) and set `SHELL=/bin/bash` in crontab (per past fixes).  
   - Log dataset manifest hash at training start for traceability.

## Key code snippets

### 1) Manifest builder (run on Mac/cron)

```python
# scripts/build_dataset_manifest.py
#!/usr/bin/env python3
"""
Generate a CDN-only manifest for a HuggingFace dataset repo folder.
Usage:
  python scripts/build_dataset_manifest.py \
    --repo datasets/axentx/surrogate-mirror \
    --folder 2026-05-03 \
    --out manifests/manifest-2026-05-03.json
"""
import argparse
import json
import os
import time
from pathlib import Path

from huggingface_hub import HfApi, Repository

HF_API_RATE_LIMIT_RETRY = 360  # seconds after 429

def build_manifest(repo_id: str, folder: str, out_path: Path):
    api = HfApi()
    out_path.parent.mkdir(parents=True, exist_ok=True)

    # Non-recursive per folder to avoid massive pagination
    entries = api.list_repo_tree(repo_id=repo_id, path=folder, recursive=False)

    files = []
    for e in entries:
        if getattr(e, "type", None) != "file":
            continue
        # CDN files are not counted against API rate limits for downloads.
        files.append(
            {
                "repo": repo_id,
                "path": e.path,
                "sha": getattr(e, "sha", None),
                "size": getattr(e, "size", None),
            }
        )

    manifest = {
        "generated_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "repo": repo_id,
        "folder": folder,
        "files": files,
    }

    with open(out_path, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest written to {out_path} ({len(files)} files)")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="HF dataset repo id")
    parser.add_argument("--folder", required=True, help="Folder inside repo")
    parser.add_argument("--out", required=True, help="Output JSON path")
    args = parser.parse_args()

    try:
        build_manifest(args.repo, args.folder, Path(args.out))
    except Exception as exc:
        # If 429, wait HF_API_RATE_LIMIT_RETRY before retry (caller/cron should handle)
        print(f"Error: {exc}")
        raise
```

### 2) CDN-only iterable dataset

```python
# surrogate/data/hf_cdn_dataset.py
import json
import logging
from pathlib import Path
from typing import Dict, Iterator, List

import pyarrow.parquet as pq
import requests
from tqdm import tqdm

CDN_ROOT = "https://huggingface.co/datasets"

log = logging.getLogger(__name__)

class HFCDNDataset:
    """
    Iterable dataset that reads files via HF CDN (no auth/API calls).
    Manifest format:
    {
      "repo": "datasets/axentx/surrogate-mirror",
      "folder": "2026-05-03",
      "files": [{"repo": "...", "path": "...", "sha": "...", "size": ...}, ...]
    }
    """

    def __init__(self, manifest_path: str):
        self.manifest_path = Path(manifest_path)
        with open(self.manifest_path) as f:
            self.manifest = json.load(f)
        self.files: List[Dict] = self.manifest["files"]

    def _stream_file(self, item: Dict) -> bytes:
        repo = item["repo"]
        path = item["path"]
        url = f"{CDN_ROOT}/{repo}/resolve/main/{path}"
        resp = requests.get(url, timeout=60)
        resp.raise_for_status()
        return resp.content

    def __iter__(self) -> Iterator[Dict]:
        for item in tqdm(self.files, desc="CDN dataset"):
            try:
                raw = self._stream_file(item)
                # Assume parquet for now; extend for jsonl if needed
                table = pq.read_table(pq.ParquetFile(pq.BufferReader(raw)))
                df = table.select(["prompt", "response"]).to_pandas()
                for _, row in df.iterrows():
                    yield {"prompt": row["prompt"], "response": row["response"]}
            except Exception as exc:
                log.warning("Failed to process %s: %s", item["path"], exc)
                continue
```
