# workio / design

## 1. Diagnosis
- **No tenant cache on LINE webhook** — every webhook performs a PostgreSQL lookup (`channelId → tenant`); under burst traffic this amplifies DB load, adds latency, and increases failure surface.
- **Missing health/readiness endpoints** — no `/health` or `/ready` for k8s/PM2/liveness probes; deploys and autoscaling are blind.
- **No structured request ID / trace correlation** — logs from LINE webhook, clock-in/out, and leave/OT flows are hard to correlate during incidents.
- **Hardcoded env assumptions** — `.env.example` and runtime code assume DB/Redis presence but don’t validate required keys or provide safe defaults; startup failures are opaque.
- **No circuit-breaker / retry for LINE API calls** — outbound calls to LINE Messaging API can fail transiently and cause lost user events.

## 2. Proposed change
Add a tenant cache (memory-backed with TTL) and health/readiness endpoints to the Express backend.  
Scope:
- `server/src/index.ts` (or main entry) — add `/health` and `/ready` routes.
- `server/src/services/tenantService.ts` (or equivalent) — add `getTenantByChannelIdCached(channelId)` with in-memory LRU/TTL.
- `server/src/middleware/tenant.ts` (or webhook handler) — use cached lookup for LINE webhook.
- `server/src/config/env.ts` — validate required env vars and export defaults.

### 3.2 Add tenant cache service (`server/src/services/tenantCache.ts`)
```ts
// server/src/services/tenantCache.ts
import { ENV } from '../config/env';

type Tenant = { id: string; channelId: string; name: string };

class TenantCache {
  private cache = new Map<string, { tenant: Tenant; expiresAt: number }>();
  private ttlMs = ENV.TENANT_CACHE_TTL_SECONDS * 1000;

  async get(channelId: string): Promise<Tenant | null> {
    const entry = this.cache.get(channelId);
    if (entry && entry.expiresAt > Date.now()) return entry.tenant;
    this.cache.delete(channelId);
    return null;
  }

  async set(channelId: string, tenant: Tenant): Promise<void> {
    this.cache.set(channelId, { tenant, expiresAt: Date.now() + this.ttlMs });
  }

  // Optional: clear expired entries periodically
  startCleanup(intervalMs = 60000) {
    setInterval(() => {
      const now = Date.now();
      for (const [k, v] of this.cache.entries()) {
        if (v.expiresAt <= now) this.cache.delete(k);
      }
    }, intervalMs).unref();
  }
}

export const tenantCache = new TenantCache();
```

### 3.3 Add cached lookup in tenant service (`server/src/services/tenantService.ts`)
```ts
// server/src/services/tenantService.ts
import { Pool } from 'pg';
import { ENV } from '../config/env';
import { tenantCache } from './tenantCache';

const pool = new Pool({ connectionString: ENV.DATABASE_URL });

export async function getTenantByChannelId(channelId: string) {
  // Try cache first
  const cached = await tenantCache.get(channelId);
  if (cached) return cached;

  // Fallback to DB
  const res = await pool.query('SELECT id, channel_id as "channelId", name FROM tenants WHERE channel_id = $1 LIMIT 1', [channelId]);
  const tenant = res.rows[0] || null;
  if (tenant) await tenantCache.set(channelId, tenant);
  return tenant;
}
```

### 3.4 Add health/ready routes (`server/src/index.ts` or routes file)
```ts
// server/src/index.ts (or server/src/routes/health.ts)
import express from 'express';
import { Pool } from 'pg';
import { ENV } from './config/env';
import { tenantCache } from './services/tenantCache';

const router = express.Router();
const pool = new Pool({ connectionString: ENV.DATABASE_URL });

// Liveness: process is up
router.get('/health', (req, res) => {
  res.json({ status: 'ok', uptime: process.uptime(), timestamp: new Date().toISOString() });
});

// Readiness: dependencies available
router.get('/ready', async (req, res) => {
  try {
    await pool.query('SELECT 1');
    res.json({ status: 'ready', db: 'ok', cache: 'ok', timestamp: new Date().toISOString() });
  } catch (err) {
    res.status(503).json({ status: 'unready', db: 'error', error: String(err) });
  }
});

export default router;
```

Wire into main app:
```ts
// server/src/index.ts (main)
import express from 'express';
import healthRoutes from './routes/health';
import lineWebhook from './routes/lineWebhook';
import { ENV } from './config/env';
import { tenantCache } from './services/tenantCache';

const app = express();
app.use(express.json());

// start cache cleanup
tenantCache.startCleanup();

app.use('/health', healthRoutes);
app.use('/webhook/line', lineWebhook);

app.listen(ENV.PORT, () => {
  console.log(`Workio server listening on port ${ENV.PORT}`);
});
```

### 3.5 Use cached lookup in LINE webhook handler
```ts
// server/src/routes/lineWebhook.ts
import { Router } from 'express';
import { getTenantByChannelId } from '../services/tenantService';

const router = Router();

router.post('/', async (req, res) => {
  const channelId = req.headers['x-line-channel-id'] as string || '';
  if (!channelId) return res.status(400).json({ error: 'missing channel id' });

  const tenant = await getTenantByChannelId(channelId);
  if (!tenant) return res.status(404).json({ error: 'tenant not found' });

  // continue processing with tenant context...
  res.json({ received: true, tenantId: tenant.id });
});

export default router;
```
