# surrogate-1 / ops

## Final Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **Add `bin/worker.py`** — single-file, manifest-driven worker:
   - Accepts `SHARD_ID` and `TOTAL_SHARDS` from the matrix.
   - On first run (or when `manifest.json` is missing), performs **one** HF API call via `list_repo_tree(recursive=False)` for the target date folder, saves file list to `manifest.json`.
   - All subsequent shard workers (and future runs within the same date) reuse the same `manifest.json` → zero HF API calls during actual ingestion.
   - Downloads each file via **CDN bypass** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header → bypasses `/api/` rate limits entirely.
   - Projects each file to `{prompt, response}` only at parse time; ignores extra columns to avoid `pyarrow.CastError` on heterogeneous schemas.
   - Produces deterministic output: `batches/public-merged/<date>/shard{N}-<HHMMSS>.jsonl`.

2. **Update `bin/dataset-enrich.sh`** → thin wrapper:
   - Sets `PYTHONUNBUFFERED=1`, `SHELL=/bin/bash`.
   - Validates `HF_TOKEN` and `SHARD_ID`/`MATRIX_SIZE`.
   - Invokes `python3 bin/worker.py` with proper args.
   - Makes script executable (`chmod +x`).

3. **Update `.github/workflows/ingest.yml`** (if needed):
   - Ensure matrix uses `SHARD_ID: [0..15]`.
   - Pass `date` as an input or compute from UTC date in the runner.
   - No other changes required if workflow already sets `HF_TOKEN` and matrix.

4. **Keep `lib/dedup.py`** unchanged — worker will import it for central md5 dedup.

### Why this is highest value
- Eliminates HF API rate-limit risk during training data loads (CDN bypass + manifest reuse).
- Prevents `pyarrow.CastError` by projecting schemas late and per-file.
- Cuts redundant HF API calls from 16×parallel runs to **one** per date folder.
- Fits in <2h: single new Python file + small shell wrapper + no infra changes.

### `bin/worker.py`

```python
#!/usr/bin/env python3
"""
Manifest-driven, CDN-bypass ingestion worker for surrogate-1.

Usage:
  SHARD_ID=0 MATRIX_SIZE=16 python3 bin/worker.py \
    --repo axentx/surrogate-1-training-pairs \
    --date 2026-05-03 \
    --out-dir batches/public-merged

Behavior:
- If manifest.json is missing for <date>, list_repo_tree once (HF API) and save it.
- All workers for the same date reuse the same manifest (no further API calls).
- Downloads via CDN URLs (no auth header) to bypass API rate limits.
- Projects each file to {prompt, response} at parse time to avoid mixed-schema errors.
- Writes deterministic shard output: shard{SHARD_ID}-{ts}.jsonl
"""

import json
import os
import sys
import hashlib
import datetime
import time
from pathlib import Path
from typing import List, Dict, Any

import requests
from huggingface_hub import HfApi, hf_hub_download

# Local dedup helper
sys.path.insert(0, str(Path(__file__).parent))
from lib.dedup import DedupStore  # type: ignore

HF_API = HfApi()
CDN_BASE = "https://huggingface.co/datasets"

def _now_str() -> str:
    return datetime.datetime.utcnow().strftime("%Y%m%d-%H%M%S")

def _load_manifest(manifest_path: Path) -> List[str]:
    if manifest_path.exists():
        with open(manifest_path) as f:
            data = json.load(f)
        return data.get("files", [])
    return []

def _save_manifest(manifest_path: Path, files: List[str]) -> None:
    manifest_path.parent.mkdir(parents=True, exist_ok=True)
    with open(manifest_path, "w") as f:
        json.dump({"files": files, "created_utc": _now_str()}, f)

def build_manifest(repo: str, date_folder: str, manifest_path: Path) -> List[str]:
    """Single HF API call to list files in date_folder (non-recursive)."""
    files = _load_manifest(manifest_path)
    if files:
        return files

    # One API call per date folder
    items = HF_API.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    file_paths = [it.rfilename for it in items if it.type == "file"]
    _save_manifest(manifest_path, file_paths)
    return file_paths

def _cdn_url(repo: str, filepath: str) -> str:
    return f"{CDN_BASE}/{repo}/resolve/main/{filepath}"

def _hash_slug(filepath: str) -> str:
    return hashlib.md5(filepath.encode()).hexdigest()

def _pick_shard(filepath: str, total_shards: int) -> int:
    h = _hash_slug(filepath)
    return int(h, 16) % total_shards

def project_to_pair(raw: Dict[str, Any]) -> Dict[str, str]:
    """
    Project heterogeneous file row to {prompt, response}.
    Implement per known schema family; keep minimal to avoid CastError.
    """
    # Common patterns observed in public datasets:
    # - {"prompt": ..., "response": ...}
    # - {"input": ..., "output": ...}
    # - {"question": ..., "answer": ...}
    # - {"text": ...} -> split by last newline or separator if needed
    prompt = raw.get("prompt") or raw.get("input") or raw.get("question") or ""
    response = raw.get("response") or raw.get("output") or raw.get("answer") or ""

    # Fallback for single-column text
    if not prompt and not response:
        text = raw.get("text") or ""
        parts = text.rsplit("\n\n", 1)
        if len(parts) == 2:
            prompt, response = parts[0], parts[1]
        else:
            prompt, response = "", text

    return {"prompt": str(prompt).strip(), "response": str(response).strip()}

def stream_and_process(
    repo: str,
    files: List[str],
    shard_id: int,
    total_shards: int,
    dedup: DedupStore,
    out_file: Path,
) -> int:
    written = 0
    out_file.parent.mkdir(parents=True, exist_ok=True)

    with open(out_file, "w", buffering=8192) as fo:
        for fp in files:
            if _pick_shard(fp, total_shards) != shard_id:
                continue

            url = _cdn_url(repo, fp)
            try:
                resp = requests.get(url, timeout=30)
                resp.raise_for_status()
            except Exception as exc:
                print(f"[shard{shard_id}] WARN failed to download {fp}: {exc}", file=sys.stderr)
                continue

            # Try parquet first (common), fallback to jsonl
            import io
            try:
                import pyarrow.parquet as pq
                table = pq.read_table(io.BytesIO(resp.content))
                rows = table.to_pylist()
            except Exception:
                # Assume jsonl
                rows = []
                for line in resp.content.decode("utf-8", errors="replace").splitlines():
                    line = line.strip()
                    if not line:
                        continue
                    try:
                        rows.append(json.loads(line))
                    except Exception:
                        continue

            for raw in rows:
                pair = project_to_pair(raw)
                if not pair["prompt"] and not pair["response"]:
                    continue

                # Central dedup by content hash
                payload = json.dumps(pair, sort_keys=True, separators=(",", ":"))
                md5 = hashlib.md5(payload.encode()).hexdigest()
                if dedup.seen(md5):
                    continue

                fo.write(json.dumps(pair, ensure_ascii=False) + "\n")
                written += 1

    return
