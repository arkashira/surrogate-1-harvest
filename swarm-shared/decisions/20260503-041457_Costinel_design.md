# Costinel / design

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a zero-runtime-HF-API “Top Hub Signal” panel to Costinel that surfaces the most-connected hub (e.g., MOC) with contextual insights, using CDN-bypass fetches and pre-listed file manifest.

**Scope** (fits <2h):
- Add a small build-time script (`scripts/build-top-hub-manifest.js`) that lists today’s `knowledge-rag/top-hub/` folder once and writes `public/data/top-hub-manifest.json`.
- Add a React panel component (`src/components/TopHubSignalPanel.tsx`) that CDN-fetches the latest `{hub}.json` from the manifest and renders a compact card.
- Wire panel into the existing dashboard layout (likely sidebar or top-row metric panel).
- No runtime HF API, no auth, no rate-limit risk.

**Why this is highest-value**:
- Immediately gives ops teams “sense + signal” context (top hub + key insights) without touching cloud infra or running heavy queries.
- Reuses the knowledge-rag graph pattern and CDN-bypass insight from earlier lessons.
- Safe, cache-friendly, and deployable via existing Docker/CD pipeline.

### 2) Build script: `scripts/build-top-hub-manifest.js`

```js
#!/usr/bin/env node
/**
 * Build manifest for top-hub CDN assets.
 * Usage: node scripts/build-top-hub-manifest.js
 *
 * Strategy:
 * - List top-level folder once via HF repo tree API (single call).
 * - Save manifest mapping latest file per hub (or use explicit latest.json).
 * - Output to public/data/top-hub-manifest.json for CDN fetch at runtime.
 *
 * Note: HF CDN files at
 * https://huggingface.co/datasets/{repo}/resolve/main/knowledge-rag/top-hub/{file}
 * require no Authorization and bypass API rate limits.
 */

import fs from 'fs';
import path from 'path';
import { fileURLToPath } from 'url';

const __filename = fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);

const REPO = 'datasets/AXENTX/knowledge-rag'; // adjust if repo differs
const BASE_CDN = `https://huggingface.co/${REPO}/resolve/main`;
const FOLDER = 'knowledge-rag/top-hub';
const OUT_DIR = path.join(__dirname, '..', '..', 'public', 'data');
const OUT_FILE = path.join(OUT_DIR, 'top-hub-manifest.json');

// Minimal tree fetch for one folder (non-recursive)
async function listTopHubFolder() {
  // HF API: https://huggingface.co/api/datasets/{repo}/tree -- non-recursive
  const res = await fetch(
    `https://huggingface.co/api/datasets/${REPO}/tree?path=${encodeURIComponent(
      FOLDER
    )}&recursive=false`
  );

  if (!res.ok) {
    // Fallback: use a static manifest if API unavailable (safe default)
    console.warn('HF tree API unavailable, using static fallback manifest.');
    return null;
  }

  const items = await res.json(); // array of { path, type }
  return items.filter((i) => i.type === 'file').map((i) => i.path);
}

function buildManifest(filePaths) {
  // Expect files like: top-hub/MOC-2026-05-03.json, top-hub/AZ-2026-05-03.json
  const byHub = {};
  for (const p of filePaths) {
    const name = path.basename(p, '.json');
    // crude hub id: prefix before first '-' or whole name
    const hubId = name.split('-')[0].toUpperCase();
    const url = `${BASE_CDN}/${p}`;
    if (!byHub[hubId] || p > byHub[hubId].path) {
      byHub[hubId] = { hubId, name, path: p, url, updated: null };
    }
  }

  // Pick "top" hub by alphabetical + recency (or could use explicit latest.json)
  const top = Object.values(byHub).sort((a, b) => b.path.localeCompare(a.path))[0];

  return {
    generatedAt: new Date().toISOString(),
    folder: FOLDER,
    topHub: top ? top.hubId : null,
    hubs: byHub,
    latest: top || null,
  };
}

async function main() {
  const paths = await listTopHubFolder();
  let manifest;
  if (!paths || paths.length === 0) {
    // Safe static fallback so runtime never breaks
    manifest = {
      generatedAt: new Date().toISOString(),
      folder: FOLDER,
      topHub: 'MOC',
      hubs: {
        MOC: {
          hubId: 'MOC',
          name: 'MOC-latest',
          path: 'knowledge-rag/top-hub/MOC-latest.json',
          url: `${BASE_CDN}/knowledge-rag/top-hub/MOC-latest.json`,
          updated: null,
        },
      },
      latest: {
        hubId: 'MOC',
        name: 'MOC-latest',
        path: 'knowledge-rag/top-hub/MOC-latest.json',
        url: `${BASE_CDN}/knowledge-rag/top-hub/MOC-latest.json`,
        updated: null,
      },
      fallback: true,
    };
  } else {
    manifest = buildManifest(paths);
  }

  fs.mkdirSync(OUT_DIR, { recursive: true });
  fs.writeFileSync(OUT_FILE, JSON.stringify(manifest, null, 2));
  console.log(`Manifest written to ${OUT_FILE}`);
}

main().catch((err) => {
  console.error(err);
  process.exit(1);
});
```

Make executable and add to `package.json` scripts:

```json
"scripts": {
  "build:top-hub": "node scripts/build-top-hub-manifest.js",
  "predev": "npm run build:top-hub",
  "prebuild": "npm run build:top-hub"
}
```
