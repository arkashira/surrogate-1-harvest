# airship / quality

**Final Chosen Approach: Manifest-Driven CDN-Only Dataset Loader (Synthesized & Hardened)**

- **Core mechanism**: One-time manifest generation per date folder → CDN-only PyTorch Dataset → Lightning-compatible training loop.  
- **Why this wins**: eliminates HF API rate limits (429), avoids `pyarrow.CastError` via strict projection, removes auth/API dependency during training, and is deployable in <2h on Mac/CI/Lightning Studio.

---

### Concrete Implementation Plan (Actionable)

1. **Generate manifest once per date folder**  
   - Run on Mac or CI: `python scripts/generate_manifest.py --repo org/surrogate1-mirror --date 2026-04-29 --out filelist.json`  
   - Commit `filelist.json` to the training repo so Lightning Studio never calls the HF API during data loading.

2. **Use a CDN-only Dataset**  
   - Downloads via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth, no rate limits).  
   - Projects only `{prompt, response}` at parse time; coerces to string and ignores extra columns to prevent `pyarrow.CastError`.

3. **Update training entrypoint**  
   - Accept `--manifest filelist.json`.  
   - Use `torch.utils.data.DataLoader` with the CDN dataset.  
   - Handle Studio stop/start gracefully (re-check status before `.run()`).

4. **Add two small scripts**  
   - `scripts/generate_manifest.py`: produces `filelist.json` for a given `date_slug`.  
   - `scripts/train_surrogate.py`: Lightning-compatible launcher that uses the CDN dataset.

5. **Validate locally (optional but recommended)**  
   - Run dataset over 100 samples to verify projection and download speed.

---

### Resolved Contradictions (Correctness + Actionability)

- **Manifest scope**: non-recursive per date folder (fast, bounded) rather than recursive full-repo (slow, quota-heavy).  
- **Projection safety**: explicit column-name heuristics + string coercion + deterministic fallback (first two text columns) to avoid schema mismatches and `pyarrow.CastError`.  
- **Caching vs memory**: cache parsed Parquet tables in memory per worker to avoid repeated CDN downloads during epoch repeats; do not cache indefinitely across workers to limit RAM.  
- **Lightning integration**: provide a real `SurrogateDataModule` and minimal `SurrogateTrainerModule` with a working optimizer and logging (fixes Candidate 2’s incomplete stub).  
- **Error handling**: raise clearly on failed CDN downloads; validate manifest keys and file extensions up front.

#### 1. `scripts/generate_manifest.py`
```python
#!/usr/bin/env python3
"""
Generate filelist.json for a date folder in a Hugging Face dataset repo.
Usage:
    python scripts/generate_manifest.py \
        --repo "org/surrogate1-mirror" \
        --date "2026-04-29" \
        --out "filelist.json"
"""
import argparse
import json
import os
from huggingface_hub import HfApi

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="HF dataset repo (org/name)")
    parser.add_argument("--date", required=True, help="Date folder (e.g. 2026-04-29)")
    parser.add_argument("--out", default="filelist.json", help="Output JSON path")
    args = parser.parse_args()

    api = HfApi()
    entries = api.list_repo_tree(
        repo_id=args.repo,
        path=args.date,
        recursive=False,
        repo_type="dataset",
    )

    files = []
    for entry in entries:
        if entry.type == "file" and entry.path.lower().endswith((".parquet", ".json", ".jsonl")):
            files.append(entry.path)

    if not files:
        raise RuntimeError(f"No data files found in {args.repo}/{args.date}")

    manifest = {
        "repo": args.repo,
        "date": args.date,
        "files": sorted(files),
    }

    os.makedirs(os.path.dirname(os.path.abspath(args.out)) or ".", exist_ok=True)
    with open(args.out, "w") as f:
        json.dump(manifest, f, indent=2)

    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```

#### 2. `surrogate/data/cdn_dataset.py`
```python
import json
import pyarrow.parquet as pq
import pyarrow as pa
import requests
import io
import torch
from torch.utils.data import Dataset
from typing import Dict, List

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

class CDNParquetDataset(Dataset):
    """
    CDN-only dataset loader for Surrogate-1 training.
    Downloads parquet files via public CDN URLs (no HF API auth).
    Projects only {prompt, response} to avoid mixed-schema CastError.
    """
    def __init__(self, manifest_path: str, repo: str = None):
        with open(manifest_path) as f:
            manifest = json.load(f)
        self.repo = repo or manifest["repo"]
        self.files = manifest["files"]
        if not self.files:
            raise ValueError("Manifest contains no files")
        self._cache: Dict[int, pa.Table] = {}

    def _download_parquet(self, path: str) -> pa.Table:
        url = CDN_TEMPLATE.format(repo=self.repo, path=path)
        resp = requests.get(url, timeout=30)
        resp.raise_for_status()
        return pq.read_table(io.BytesIO(resp.content))

    def _project_row(self, table: pa.Table, idx: int) -> Dict[str, str]:
        prompt_col = None
        response_col = None
        for c in table.column_names:
            cl = c.lower()
            if cl in ("prompt", "instruction", "input", "question"):
                prompt_col = c
            if cl in ("response", "completion", "output", "answer"):
                response_col = c

        text_cols = [c for c in table.column_names if pa.types.is_string(table.schema.field(c).type)]
        if prompt_col is None and text_cols:
            prompt_col = text_cols[0]
        if response_col is None and len(text_cols) > 1:
            response_col = text_cols[1]

        prompt = str(table[prompt_col][idx].as_py()) if prompt_col else ""
        response = str(table[response_col][idx].as_py()) if response_col else ""
        return {"prompt": prompt.strip(), "response": response.strip()}

    def __len__(self) -> int:
        total = 0
        for i, _ in enumerate(self.files):
            if i not in self._cache:
                self._cache[i] = self._download_parquet(self.files[i])
            total += self._cache[i].num_rows
        return total

    def __getitem__(self, idx: int) -> Dict[str, str]:
        current = 0
        for i, path in enumerate(self.files):
            if i not in self._cache:
                self._cache[i] = self._download_parquet(path)
            t = self._cache[i]
            if current <= idx < current + t.num_rows:
                return self._project_row(t, idx - current)
            current += t.num_rows
        raise IndexError(f"Index {idx} out of range for dataset with {current} rows")
```
