## Highest-Value Incremental Improvement (<2h)

**Goal**: Eliminate HF API 429s during Surrogate-1 training by replacing `load_dataset(streaming=True)` with a CDN-only `IterableDataset` that reads from a pre-generated `file_list.json`. This removes pagination, auth-checks, and per-batch API calls — GPUs will pull parquet files directly from `https://huggingface.co/datasets/{repo}/resolve/main/...` with zero Authorization header.

**Scope**: One focused change in `surrogate/` — add `surrogate/data/cdn_dataset.py`, update training entrypoint to use it, and add a small Mac-side script to pre-list files once and emit `file_list.json`. No infra changes; works with existing docker-compose setup.

**Estimated effort**: 90–110 minutes.

---

## Implementation Plan

1. **Create CDN-only IterableDataset** (`surrogate/data/cdn_dataset.py`)
   - Accepts `repo_id`, `file_list_json` (or inline list of `{repo, path}`).
   - Shuffles file order and yields raw bytes; downstream parsing projects to `{prompt, response}`.
   - Uses `urllib.request` with no auth; retries with exponential backoff on 429/5xx.
   - Supports `IterableDataset` so Lightning can use multiple workers.

2. **Add util to build file_list.json** (`surrogate/scripts/build_file_list.py`)
   - Runs on Mac (or CI) after rate-limit window clears.
   - Uses HF API `list_repo_tree(path, recursive=False)` per date folder (non-recursive to avoid pagination explosion).
   - Emits `file_list.json` with `{"repo": "...", "path": "..."}` entries.
   - Optional: deterministic shard assignment via hash(slug) % N for sibling repos (future-proof).

3. **Update training entrypoint** (`surrogate/train.py` or equivalent)
   - Replace `load_dataset(streaming=True, ...)` with `CdnParquetDataset(file_list_json="file_list.json")`.
   - Keep existing preprocessing (select/project columns) unchanged.
   - Ensure `num_workers` > 0 works (each worker independently fetches via CDN).

4. **Studio reuse guard** (small quality win)
   - Before `Studio(create_ok=True)`, list running studios and reuse if name matches.
   - Prevents quota burn during iterations.

5. **Smoke test**
   - Run one training step locally (or in Studio) and confirm:
     - No HF API calls during data loading (check logs for `resolve/main/` URLs).
     - Dataset yields batches.
     - No 429s.

### 1) CDN-only IterableDataset
`surrogate/data/cdn_dataset.py`
```python
# surrogate/data/cdn_dataset.py
import json
import hashlib
import random
import time
import urllib.request
import urllib.error
from typing import List, Dict, Optional, Iterator
import torch
from torch.utils.data import IterableDataset
import pyarrow.parquet as pq
import io

HF_CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

class CdnParquetDataset(IterableDataset):
    """
    CDN-only dataset for HuggingFace parquet files.
    Avoids HF API auth/rate-limits by using public CDN URLs.
    """

    def __init__(
        self,
        file_list: Optional[List[Dict[str, str]]] = None,
        file_list_json: Optional[str] = None,
        repo_id: Optional[str] = None,
        folder_path: Optional[str] = None,
        shuffle: bool = True,
        seed: int = 42,
        retry_wait: float = 360.0,
        max_retries: int = 5,
    ):
        """
        Provide either:
          - file_list: list of dicts [{"repo": "...", "path": "..."}, ...]
          - file_list_json: path to JSON file with same structure
        Or (legacy) repo_id + folder_path to list via HF API (not recommended here).
        """
        super().__init__()
        if file_list_json:
            with open(file_list_json, "r", encoding="utf-8") as f:
                file_list = json.load(f)
        if not file_list:
            raise ValueError("Provide file_list or file_list_json.")

        self.file_list = file_list
        self.shuffle = shuffle
        self.seed = seed
        self.retry_wait = retry_wait
        self.max_retries = max_retries
        self.epoch = 0

    def _fetch_parquet(self, repo: str, path: str) -> bytes:
        url = HF_CDN_TEMPLATE.format(repo=repo, path=path)
        last_exc = None
        for attempt in range(1, self.max_retries + 1):
            try:
                req = urllib.request.Request(url, headers={})
                with urllib.request.urlopen(req, timeout=60) as resp:
                    if resp.status == 200:
                        return resp.read()
                    # Some CDN paths may redirect or return non-200; treat as retryable
                    raise urllib.error.HTTPError(url, resp.status, resp.reason, {}, None)
            except urllib.error.HTTPError as e:
                if e.code == 429 or e.code >= 500:
                    wait = self.retry_wait * (2 ** (attempt - 1))
                    time.sleep(wait)
                    last_exc = e
                    continue
                raise
            except (urllib.error.URLError, TimeoutError) as e:
                wait = self.retry_wait * (2 ** (attempt - 1))
                time.sleep(wait)
                last_exc = e
                continue
        raise RuntimeError(f"Failed to fetch {url} after {self.max_retries} attempts") from last_exc

    def set_epoch(self, epoch: int):
        self.epoch = epoch

    def __iter__(self) -> Iterator[Dict[str, torch.Tensor]]:
        worker_info = torch.utils.data.get_worker_info()
        if worker_info is None:
            num_workers = 1
            worker_id = 0
        else:
            num_workers = worker_info.num_workers
            worker_id = worker_info.id

        # Deterministic shuffle per epoch across workers
        rng = random.Random(self.seed + self.epoch)
        files = list(self.file_list)
        if self.shuffle:
            rng.shuffle(files)

        # Assign files to this worker in a round-robin fashion
        files = [f for i, f in enumerate(files) if i % num_workers == worker_id]

        for item in files:
            repo = item["repo"]
            path = item["path"]
            try:
                raw = self._fetch_parquet(repo, path)
                table = pq.read_table(io.BytesIO(raw))
                # Project to expected schema: prompt/response only (no source/ts)
                # Keep flexible: yield rows as dicts of tensors (or let collate handle)
                df = table.to_pydict()
                # If multiple rows per parquet, yield each row
                n = len(df.get("prompt", df.get("response", [])))
                if n == 0:
                    continue
                # Normalize: ensure prompt/response exist
                prompts = df.get("prompt", [""] * n)
                responses = df.get("response", [""] * n)
                for i in range(n):
                    yield {
                        "prompt": str(prompts[i]),
                        "response": str(responses[i]),
                    }
            except Exception as exc:
                # Log and skip bad files to avoid crashing training
                # In production, route to logger
                print(f"[cdn_dataset] skip {repo}/{path}: {exc}")
                continue
```
