# airship / design

## Final Synthesis — Deterministic CDN-Only Manifest for Surrogate-1

**Chosen scope (≤2h):**  
Create a single, deterministic `airship discover` command that produces a CDN-only file manifest for a given `repo_id` and `date_folder`, eliminating HF API usage during training data loading and sidestepping PyArrow schema errors. No infra changes, no model code changes.

**Why this resolves the blockers:**  
- Removes HF API rate limits from the hot training path (only optional one-time tree listing).  
- Avoids PyArrow schema/cast errors by never calling `load_dataset`/`streaming` during discovery.  
- Produces a reproducible JSONL manifest that `train.py` can consume to download via CDN URLs only.

---

## Concrete Implementation Plan

1. **Add `airship/discover.py`** (can be invoked directly or wired into CLI later).  
   - Inputs: `repo_id`, `date_folder`, `out_manifest` (default `manifests/{repo_safe}-{date}.jsonl`), `--preflight` flag.  
   - Step A: One-time `list_repo_tree(..., recursive=False)` (HF API; rate-limit safe) to enumerate immediate files in `date_folder`.  
   - Step B: Emit one JSONL record per file:  
     `{"repo": repo_id, "date": date_folder, "path": rel_path, "cdn_url": "...", "size": int|null, "sha256": str|null}`  
   - Step C (optional): Parallel HEADs to CDN to populate `size`. Optional SHA256 for small samples (skip by default for speed).  
   - Idempotency: lockfile `.discover.{repo_safe}.{date}.lock` to avoid duplicate API calls. Clean up lock on finish.

2. **Update training launcher contract** (`train.py` or Lightning Studio entrypoint).  
   - Accept `--manifest <path>` (JSONL).  
   - For each record, download via `cdn_url` using `requests`/`aiohttp` with retries.  
   - Parse only required fields (`prompt`, `response`) at load time; do not rely on dataset schema.  
   - If HF token is present, use it only for optional tree listing; never for data streaming.

3. **Add repo-level conveniences**  
   - `Makefile` targets:  
     - `make discover REPO=<repo> DATE=<date> [PREFLIGHT=1]`  
     - `make train-manifest REPO=<repo> DATE=<date>` (runs discover then launches training with manifest)  
   - `.gitignore`: `manifests/` and `*.lock`.

4. **Validation checklist (≤30 min)**  
   - Run `discover.py` on a small date folder; confirm manifest line count equals visible file count.  
   - Run `train.py --dry-run --manifest <file>`; verify CDN URLs are used and no `/api/` or HF dataset streaming calls appear in logs.

## Final Code

### `airship/discover.py`

```python
#!/usr/bin/env python3
"""
Deterministic CDN-only manifest generator for Surrogate-1 training.

Usage:
  python airship/discover.py \
    --repo hf-internal/mirror-merged \
    --date 2026-04-29 \
    --out manifests/mirror-merged-2026-04-29.jsonl \
    --preflight
"""

import argparse
import hashlib
import json
import sys
import time
from concurrent.futures import ThreadPoolExecutor, as_completed
from pathlib import Path
from typing import Any, Dict, List

import requests
from huggingface_hub import HfApi

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"
DEFAULT_MAX_WORKERS = 20


def head_cdn(url: str, timeout: int = 10) -> Dict[str, Any]:
    try:
        r = requests.head(url, allow_redirects=True, timeout=timeout)
        r.raise_for_status()
        return {"size": int(r.headers.get("content-length", 0)), "status": "ok"}
    except Exception as e:
        return {"size": 0, "status": f"error:{e}"}


def sha256_of_url(url: str, chunk_size: int = 8192, max_bytes: int = 1024**3) -> str:
    """
    Compute sha256 for a URL.
    max_bytes caps how much to read (useful to avoid huge downloads during discovery).
    """
    h = hashlib.sha256()
    resp = requests.get(url, stream=True, timeout=30)
    resp.raise_for_status()
    total = 0
    for chunk in resp.iter_content(chunk_size=chunk_size):
        h.update(chunk)
        total += len(chunk)
        if total >= max_bytes:
            break
    return h.hexdigest()


def build_manifest(
    repo_id: str,
    date_folder: str,
    out_path: Path,
    preflight: bool = False,
    compute_sha256: bool = False,
    max_workers: int = DEFAULT_MAX_WORKERS,
    sha256_max_bytes: int = 1024**3,
) -> None:
    out_path.parent.mkdir(parents=True, exist_ok=True)
    safe_name = f"{repo_id.replace('/', '-')}-{date_folder}".replace(" ", "_")
    lock_path = out_path.parent / f".discover.{safe_name}.lock"

    # Idempotency guard
    if lock_path.exists():
        print(f"Lock exists: {lock_path}. Skipping discover.", file=sys.stderr)
        return
    lock_path.touch()

    api = HfApi()
    base_path = date_folder.strip("/")
    print(f"Listing repo tree: {repo_id}/{base_path}", file=sys.stderr)

    try:
        tree = api.list_repo_tree(repo_id=repo_id, path=base_path, recursive=False)
        files = [item.rfilename for item in tree if getattr(item, "type", None) == "file"]
    except Exception as e:
        print(f"HF API error: {e}", file=sys.stderr)
        files = []

    records: List[Dict[str, Any]] = []
    for f in files:
        rel_path = f"{base_path}/{f}" if base_path else f
        cdn_url = CDN_TEMPLATE.format(repo=repo_id, path=rel_path)
        records.append(
            {
                "repo": repo_id,
                "date": date_folder,
                "path": rel_path,
                "cdn_url": cdn_url,
                "size": None,
                "sha256": None,
            }
        )

    if preflight:
        print(f"Preflight HEAD for {len(records)} files...", file=sys.stderr)
        with ThreadPoolExecutor(max_workers=max_workers) as ex:
            future_to_idx = {ex.submit(head_cdn, r["cdn_url"]): i for i, r in enumerate(records)}
            for future in as_completed(future_to_idx):
                idx = future_to_idx[future]
                try:
                    info = future.result()
                    records[idx]["size"] = info["size"]
                except Exception:
                    records[idx]["size"] = 0

    if compute_sha256:
        print(f"Computing sha256 (capped at {sha256_max_bytes}B) for {len(records)} files...", file=sys.stderr)
        with ThreadPoolExecutor(max_workers=max_workers) as ex:
            future_to_idx = {
                ex.submit(sha256_of_url, r["cdn_url"], max_bytes=sha256_max_bytes): i
                for i, r in enumerate(records)
            }
            for future in as_completed(future_to_idx):
                idx = future_to_idx[future]
                try:
                    records[idx]["sha256"] = future.result()
                except Exception:
                    records[idx]["sha256"] = None

    with out_path.open("w", encoding="utf-8") as fh:
        for r in records:
            fh.write(json.dumps(r, ensure_ascii=False) + "\n")

    lock_path.unlink(missing_ok=True)
    print(
        f"Manifest written: {out_path} ({len(records)} files, preflight={preflight})",
        file=sys.stderr,
    )


if __name__ == "__main__":
    parser = argparse.ArgumentParser
