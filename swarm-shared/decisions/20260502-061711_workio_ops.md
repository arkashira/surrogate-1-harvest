**High-Value Incremental Improvement for Workio Discovery**
#### Implementation Plan

Given the current state of the Workio project and the recent commits, the highest-value incremental improvement for Workio discovery can be achieved by implementing the **"LINE OA Setup"** feature.

**Task:** Configure LINE OA Setup for Workio

**Implementation Steps:**

1. **Create LINE Official Account**: Create a new LINE Official Account and enable the Messaging API.
2. **Set webhook URL**: Set the webhook URL to `https://your-domain.com/webhook/line`.
3. **Add Channel Access Token & Secret**: Add the Channel Access Token and Secret to the `.env` file.
4. **Update frontend and backend code**: Update the frontend and backend code to use the new LINE OA Setup configuration.

**Code Snippets:**

**Update `.env` file**:
```makefile
LINE_CHANNEL_ACCESS_TOKEN=YOUR_CHANNEL_ACCESS_TOKEN
LINE_CHANNEL_SECRET=YOUR_CHANNEL_SECRET
```
**Update frontend code**:
```javascript
// workio/frontend/src/utils/line-api.js
import axios from 'axios';

const lineApi = axios.create({
  baseURL: 'https://api.line.me',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${process.env.LINE_CHANNEL_ACCESS_TOKEN}`,
  },
});

export default lineApi;
```
**Update backend code**:
