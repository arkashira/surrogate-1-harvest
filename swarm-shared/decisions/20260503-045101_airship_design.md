# airship / design

## Final Synthesized Implementation (Best of Both Candidates)

**Core Strategy**: *Decouple metadata discovery from data loading.* Perform all HuggingFace API interactions once on the Mac orchestration node, then switch to pure CDN/raw HTTP fetching during GPU training. This eliminates 429s and minimizes Lightning Studio idle time.

### 1. Unified Manifest Generator (`scripts/generate_manifest.py`)
*Merges Candidate 1’s robustness with Candidate 2’s strict file-type filtering.*

```python
#!/usr/bin/env python3
"""
Generate CDN-only manifest for Surrogate training.
Run on Mac orchestration node (single API call per date).
"""
import json
import os
from datetime import datetime
from huggingface_hub import HfApi

API = HfApi()
REPO_ID = "axentx/surrogate-training"  # Unified repo name
DATE_FOLDER = datetime.utcnow().strftime("%Y-%m-%d")
OUT_PATH = f"manifests/manifest-{DATE_FOLDER}.json"

def main(date_override: str = None) -> None:
    target_date = date_override or DATE_FOLDER
    os.makedirs(os.path.dirname(OUT_PATH), exist_ok=True)

    # Single non-recursive call (avoids pagination 429)
    tree = API.list_repo_tree(
        repo_id=REPO_ID,
        path=target_date,
        recursive=False
    )

    files = []
    for item in tree:
        if item.get("type") != "file":
            continue
        # Candidate 2 filter: strict parquet/jsonl only
        if not item["path"].lower().endswith((".parquet", ".jsonl")):
            continue

        rel_path = f"{target_date}/{item['path'].split('/')[-1]}"
        files.append({
            "repo_id": REPO_ID,
            "rel_path": rel_path,
            "cdn_url": f"https://huggingface.co/datasets/{REPO_ID}/resolve/main/{rel_path}",
            "size": item.get("size", 0)
        })

    manifest = {
        "date": target_date,
        "repo_id": REPO_ID,
        "generated_at": datetime.utcnow().isoformat() + "Z",
        "count": len(files),
        "files": files
    }

    with open(OUT_PATH, "w") as f:
        json.dump(manifest, f, indent=2)

    print(f"✅ Manifest: {OUT_PATH} ({len(files)} files)")

if __name__ == "__main__":
    import sys
    main(date_override=sys.argv[1] if len(sys.argv) > 1 else None)
```

### 3. Lightning Studio Launcher (`scripts/launch_lightning.py`)
*Merges Candidate 1’s reuse logic with Candidate 2’s idle-stop protection.*

```bash
#!/usr/bin/env bash
# Resilient launcher: Reuse running studio + idle-stop guard
set -euo pipefail

STUDIO_NAME="surrogate-train-l40s"
MANIFEST_PATH="${1:-manifests/manifest-$(date -u +%Y-%m-%d).json}"

python - <<PY
import os
import lightning as L

STUDIO_NAME = os.environ.get("STUDIO_NAME", "$STUDIO_NAME")
MANIFEST_PATH = os.environ.get("MANIFEST_PATH", "$MANIFEST_PATH")

teamspace = L.Teamspace()
studio = None

# Reuse running studio (saves ~80hr/mo quota)
for s in teamspace.studios:
    if s.name == STUDIO_NAME and s.status == "running":
        studio = s
        print(f"♻️ Reusing running studio: {s.name}")
        break

# Create or restart if stopped
if not studio:
    studio = L.Studio(
        name=STUDIO_NAME,
        machine=L.Machine.L40S,
        create_ok=True
    )

if studio.status != "running":
    print("🔄 Studio stopped — restarting")
    studio.start(machine=L.Machine.L40S)

# Launch training (zero HF API calls during data load)
studio.run(
    "surrogate/train.py",
    arguments=[f"--manifest={MANIFEST_PATH}"],
    watch=False
)
PY
```

### 4. Deployment Steps (≤2 Hours)

1. **Install manifest generator**
   ```bash
   mkdir -p /opt/axentx/airship/scripts
   cp scripts/generate_manifest.py /opt/axentx/airship/scripts/
   chmod +x /opt/axentx/airship/scripts/generate_manifest.py
   ```

2. **Update training pipeline**
   - Replace `load_dataset` calls in `surrogate/train.py` with `stream_dataset` from `surrogate/data.py`
   - Implement `parse_row()` for your specific file format (Parquet/JSONL)

3. **Install launcher**
   ```bash
   cp scripts/launch_lightning.py /opt/axentx/airship/scripts/
   chmod +x /opt/axentx/airship/scripts/launch_lightning.py
   ```

4. **Generate manifest (post-rate-limit window)**
   ```bash
   cd /opt/axentx/airship
   python scripts/generate_manifest.py
   ```

