# Costinel / design

## Final Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

**Core decision**: Use **static JSON in `public/hubs/`** (CDN-first) with **graceful non-blocking fallback**. This is the simplest, fastest, and safest path that avoids API/auth/rate limits and keeps the panel resilient.

---

### What we ship (in <2h)
- A **non-blocking Top-Hub Signal Panel** mounted in the Costinel dashboard (sidebar or top banner).
- Defaults to hub **MOC** (configurable via `VITE_HUB_NAME`).
- Shows: hub title, short summary, top 3 signals (anomalies/recommendations), last updated timestamp, and “View full hub” link.
- **CDN-first**: static JSON files in `public/hubs/{hubName}.json` fetched at runtime (no backend, no auth).
- **Non-blocking**: panel renders nothing (or minimal placeholder) on failure; never blocks dashboard render or user actions.
- **Scope**: one React component, one styles module, config flag, and one sample JSON file.

---

### File changes

1. **Config** — `.env` (or `vite.config.ts`)
   - `VITE_HUB_NAME=MOC`
   - `VITE_HUBS_PATH=/hubs`

2. **Static data** — `public/hubs/MOC.json`
   - Minimal, stable schema (see below).

3. **Component** — `src/components/TopHubSignalPanel/TopHubSignalPanel.tsx`
   - Fetches JSON with timeout/abort.
   - Non-blocking error handling (silent degrade in prod; dev logs).
   - Renders compact card with top 3 signals.

4. **Styles** — `src/components/TopHubSignalPanel/TopHubSignalPanel.module.css`
   - Small, theme-aware styles using CSS variables.

5. **Mount point** — integrate into existing dashboard layout
   - Likely `src/pages/Dashboard/Dashboard.tsx` or sidebar/banner slot.

### Component — `TopHubSignalPanel.tsx`

```tsx
import React, { useEffect, useState, useMemo } from 'react';
import styles from './TopHubSignalPanel.module.css';

interface Signal {
  id: string;
  title: string;
  severity: 'low' | 'medium' | 'high';
  description: string;
  actionLabel?: string;
  actionUrl?: string;
}

interface HubData {
  hubName: string;
  title: string;
  summary: string;
  updatedAt: string;
  sourceUrl?: string;
  signals: Signal[];
}

const DEFAULT_HUB = import.meta.env.VITE_HUB_NAME || 'MOC';
const HUBS_PATH = import.meta.env.VITE_HUBS_PATH || '/hubs';
const FETCH_TIMEOUT = 5000;

export const TopHubSignalPanel: React.FC<{ hubName?: string }> = ({ hubName = DEFAULT_HUB }) => {
  const [data, setData] = useState<HubData | null>(null);
  const [loading, setLoading] = useState<boolean>(true);
  const [error, setError] = useState<Error | null>(null);

  useEffect(() => {
    const controller = new AbortController();
    const timeoutId = setTimeout(() => controller.abort(), FETCH_TIMEOUT);

    fetch(`${HUBS_PATH}/${hubName}.json`, { signal: controller.signal })
      .then((res) => {
        if (!res.ok) throw new Error(`Failed to load hub: ${res.status}`);
        return res.json();
      })
      .then((json) => {
        setData(json);
        setLoading(false);
      })
      .catch((err) => {
        if (err.name !== 'AbortError') {
          setError(err);
          setLoading(false);
          // Non-blocking: silently degrade in production
          if (process.env.NODE_ENV !== 'development') {
            console.warn('[TopHubSignalPanel] CDN fetch failed', err);
          } else {
            console.error('[TopHubSignalPanel] CDN fetch failed', err);
          }
        }
      })
      .finally(() => clearTimeout(timeoutId));

    return () => {
      controller.abort();
      clearTimeout(timeoutId);
    };
  }, [hubName]);

  const topSignals = useMemo(() => data?.signals?.slice(0, 3) || [], [data]);

  // Non-blocking: render nothing on failure so dashboard is unaffected
  if (error || !data) {
    return null;
  }

  if (loading) {
    return (
      <aside className={styles.panel} aria-busy="true">
        <div className={styles.header}>
          <span className={styles.title}>Loading hub...</span>
        </div>
      </aside>
    );
  }

  return (
    <aside className={styles.panel} aria-label={`Top hub: ${data.title}`}>
      <div className={styles.header}>
        <strong className={styles.title}>{data.title}</strong>
        {data.sourceUrl && (
          <a
            href={data.sourceUrl}
            target="_blank"
            rel="noopener noreferrer"
            className={styles.link}
          >
            View full hub
          </a>
        )}
      </div>
      <p className={styles.summary}>{data.summary}</p>

      <ul className={styles.signals} aria-label="Top signals">
        {topSignals.map((s) => (
          <li key={s.id} className={styles.signalItem}>
            <div className={styles.signalHeader}>
              <span className={styles.signalTitle}>{s.title}</span>
              <span className={`${styles.severity} ${styles[s.severity]}`}>{s.severity}</span>
            </div>
            <p className={styles.signalDesc}>{s.description}</p>
            {s.actionLabel && s.actionUrl && (
              <a href={s.actionUrl} className={styles.action}>
                {s.actionLabel}
              </a>
            )}
          </li>
        ))}
      </ul>

      <div className={styles.footer}>
        <small className={styles.updated}>
          Updated {new Date(data.updatedAt).toLocaleDateString()}
        </small>
      </div>
    </aside>
  );
};
```
