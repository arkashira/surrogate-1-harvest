# airship / quality

## Final Actionable Plan (≤2h)

**Goal**: Make `airship discover` produce a deterministic, CDN-only manifest and a schema-projected parquet staging area so surrogate training never touches the HF API at load time.

**Why this wins**:
- Eliminates HF API 429/128-hr rate-limit and commit-cap risks during training.
- Enforces schema hygiene (project to `{prompt,response}`) to prevent `pyarrow.CastError`.
- Uses existing patterns (CDN bypass, pre-list + embed, studio reuse).
- Small, focused: one CLI command + manifest + staging writer + loader + studio guard. No infra or retraining changes.

---

## Implementation (≤2h)

### 1) CLI: `airship discover` → deterministic manifest + staged parquet
- Command:  
  `airship discover hf://{repo}/{path} --date YYYY-MM-DD --out manifest.json --stage ./staging/`
- Single API call: `list_repo_tree(path, recursive=False)` for the date folder.
- Manifest schema:
  - `repo`, `date_path`, `base_url` (CDN), `files[]` with `{key, url, size, etag}`, `sha256` (manifest-level).
  - Deterministic ordering: sort by key.
- Staging:
  - Download each parquet via CDN.
  - Project columns to `{prompt, response}` (drop others; coerce to utf-8).
  - Write projected parquet to `{stage}/{sha256}_{key}`.
  - Append `local_path` to manifest entry for loader.
- Mac-only orchestration; training uses manifest + staging only.

### 2) Training-side loader: staged parquet → batches
- Prefer local staged files (`local_path`) when present; fall back to CDN URLs.
- Use `pyarrow.parquet` for fast columnar reads (no schema surprises).
- Stream batches; validate row schema (`prompt`, `response` present and string-like).
- Retry/backoff on CDN 429; integrity check via `etag`/`size` when downloading.

### 3) Studio reuse guard
- Before `Studio(create_ok=True)`, list `Teamspace.studios` and reuse running studio with matching name.
- If stopped, restart with `Machine.L40S` before `.run()`.
- Ensure idle-stop → start transition before run.

### 4) Commit-cap mitigation (future-proof)
- Hash slug → pick sibling repo (5 siblings) to spread writes (640/hr aggregate) when writing enriched outputs.

```python
#!/usr/bin/env python3
import json, hashlib, sys, os
from pathlib import Path
from typing import Dict, List
import click
import pyarrow as pa
import pyarrow.parquet as pq
import pyarrow.compute as pc
import aiohttp
import asyncio
from huggingface_hub import HfApi, list_repo_tree

CDN_BASE = "https://huggingface.co/datasets"

def _project_to_prompt_response(table: pa.Table) -> pa.Table:
    # Keep only prompt/response; coerce to string; drop nulls in key columns
    needed = {"prompt", "response"}
    present = [c for c in table.column_names if c in needed]
    missing = needed - set(present)
    if missing:
        raise ValueError(f"Missing required columns: {missing}")
    # Select and cast to utf8
    cols = {}
    for c in present:
        col = table.column(c)
        if not pa.types.is_string(col.type):
            col = pc.cast(col, pa.string())
        cols[c] = col
    # Build minimal table with deterministic column order
    return pa.table([cols["prompt"], cols["response"]], names=["prompt", "response"])

async def _download_and_stage(session: aiohttp.ClientSession, url: str, size: int, etag: str, out_path: Path) -> bool:
    try:
        async with session.get(url) as resp:
            if resp.status != 200:
                return False
            data = await resp.read()
            if size and len(data) != size:
                return False
            out_path.parent.mkdir(parents=True, exist_ok=True)
            out_path.write_bytes(data)
            return True
    except Exception:
        return False

def build_manifest_and_stage(repo: str, date_path: str, stage_dir: Path) -> Dict:
    api = HfApi()
    tree = list_repo_tree(repo=repo, path=date_path, recursive=False)
    files = sorted([f for f in tree if f.type == "file"], key=lambda f: f.path)

    entries = []
    for f in files:
        url = f"{CDN_BASE}/{repo}/resolve/main/{f.path}"
        entry = {
            "key": f.path,
            "url": url,
            "size": getattr(f, "size", 0),
            "etag": getattr(f, "etag", ""),
        }
        entries.append(entry)

    manifest = {
        "repo": repo,
        "date_path": date_path,
        "base_url": f"{CDN_BASE}/{repo}/resolve/main",
        "files": entries,
    }
    blob = json.dumps(manifest, sort_keys=True, separators=(",", ":")).encode()
    manifest_sha = hashlib.sha256(blob).hexdigest()
    manifest["sha256"] = manifest_sha

    # Stage projected parquet files (best-effort async download + projection)
    async def stage_all():
        connector = aiohttp.TCPConnector(limit=10)
        async with aiohttp.ClientSession(connector=connector) as session:
            tasks = []
            for entry in entries:
                if not entry["key"].lower().endswith(".parquet"):
                    continue
                out_path = stage_dir / f"{manifest_sha}_{Path(entry['key']).name}"
                tasks.append(
                    (entry, out_path, _download_and_stage(session, entry["url"], entry["size"], entry["etag"], out_path))
                )
            results = await asyncio.gather(*[t[2] for t in tasks])
            for (entry, out_path, _), ok in zip(tasks, results):
                if ok and out_path.exists():
                    try:
                        table = pq.read_table(out_path)
                        projected = _project_to_prompt_response(table)
                        pq.write_table(projected, out_path)
                        entry["local_path"] = str(out_path)
                    except Exception:
                        # If projection fails, remove and skip local_path
                        try:
                            out_path.unlink(missing_ok=True)
                        except Exception:
                            pass
                        entry.pop("local_path", None)
                else:
                    # download failed; ensure no stale local_path
                    entry.pop("local_path", None)

    asyncio.run(stage_all())
    return manifest

@click.command()
@click.argument("hf_path")  # hf://repo/path or repo/path
@click.option("--date", required=True, help="YYYY-MM-DD folder")
@click.option("--out", default="manifest.json", help="Output manifest path")
@click.option("--stage", default="./staging", help="Staging directory for projected parquet")
def discover(hf_path: str, date: str, out: str, stage: str):
    """
    Produce CDN-only manifest and staged, schema-projected parquet files.
    Example: airship discover hf://datasets/company/repo --date 2026-05-02 --out manifest.json --stage ./staging/
    """
    if hf_path.startswith("hf://"):
        hf_path = hf_path[5:]
    repo, *rest = hf_path.split("/", 1)
    subpath = rest[0] if rest else ""
    date_path = f"{subpath}/{date}".strip("/")

    manifest = build_manifest_and_stage(repo, date_path, Path(stage))
    Path(out).write_text(json.dumps(manifest, indent=2))
    click.echo(f"Manifest written to {out} ({len(manifest['files'])} files)")

if __name__ == "__main__":
    discover(sys.argv[1:])
```
