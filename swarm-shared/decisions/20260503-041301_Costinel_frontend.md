# Costinel / frontend

## Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Add a high-value frontend panel that surfaces the most-connected hub (e.g., "MOC") insights from the knowledge-rag graph using CDN-first data (zero runtime HF API calls). Ships in <2h with resilient UX.

### What ships
- New `TopHubSignalPanel` component under `/src/components/TopHubSignalPanel.tsx`
- Build-time data file `/src/data/top-hub.json` (generated by CI/local script)
- Route `/hub` (or mount into existing dashboard) + responsive card UI
- Runtime CDN fetch with local fallback + stale-while-revalidate behavior
- No runtime HF API usage; respects CDN rate limits and quota patterns

### Architecture (CDN-first)
1. **Build/CI step** (run once per deploy or on schedule):
   - `list_repo_tree` once per date folder → save `top-hub.json` to repo or upload to CDN (`/resolve/main/...`)
   - File schema: `{ hubId, title, score, insights: [{ text, evidence }], related, updatedAt }`
2. **Runtime** (browser):
   - Fetch `https://huggingface.co/datasets/{repo}/resolve/main/data/top-hub.json` (no Authorization header)
   - If CDN fails → fallback to bundled `/data/top-hub.json` (imported at build)
   - If both fail → render empty state with retry button
3. **UI**:
   - Compact card with hub title, score, top 3 insights, and "View details" link
   - Skeleton while loading; toast on retry; timestamp for freshness

### Code snippets

#### 1) Reusable CDN fetch hook (`/src/hooks/useCDNResource.ts`)
```tsx
import { useEffect, useState, useCallback } from 'react';

export type TopHubData = {
  hubId: string;
  title: string;
  score: number;
  insights: Array<{ text: string; evidence?: string }>;
  related: Array<{ hubId: string; title: string; score: number }>;
  updatedAt: string;
};

export function useCDNResource<T>({
  cdnUrl,
  localFallback,
  refreshIntervalMs = 0,
}: {
  cdnUrl: string;
  localFallback: T | null;
  refreshIntervalMs?: number;
}) {
  const [data, setData] = useState<T | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<Error | null>(null);

  const fetchData = useCallback(async () => {
    setLoading(true);
    try {
      const res = await fetch(cdnUrl, { cache: 'no-store' });
      if (!res.ok) throw new Error(`CDN fetch failed: ${res.status}`);
      const json = (await res.json()) as T;
      setData(json);
      setError(null);
    } catch (err) {
      // CDN failed — try local fallback
      if (localFallback !== null) {
        setData(localFallback);
        setError(null);
      } else {
        setError(err instanceof Error ? err : new Error(String(err)));
        setData(null);
      }
    } finally {
      setLoading(false);
    }
  }, [cdnUrl, localFallback]);

  useEffect(() => {
    fetchData();
    if (refreshIntervalMs > 0) {
      const id = setInterval(fetchData, refreshIntervalMs);
      return () => clearInterval(id);
    }
  }, [fetchData, refreshIntervalMs]);

  const retry = useCallback(() => fetchData(), [fetchData]);

  return { data, loading, error, retry };
}
```

#### 2) TopHubSignalPanel component (`/src/components/TopHubSignalPanel.tsx`)
```tsx
import React from 'react';
import { useCDNResource, TopHubData } from '../hooks/useCDNResource';
import localTopHub from '../data/top-hub.json';

const CDN_URL =
  'https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/data/top-hub.json';

export function TopHubSignalPanel() {
  const { data, loading, error, retry } = useCDNResource<TopHubData>({
    cdnUrl: CDN_URL,
    localFallback: localTopHub,
    refreshIntervalMs: 0, // no polling by default; rely on deploy updates
  });

  if (loading) {
    return (
      <div className="rounded-lg border bg-card p-4">
        <div className="h-5 w-32 animate-pulse rounded bg-muted" />
        <div className="mt-3 space-y-2">
          <div className="h-4 w-full animate-pulse rounded bg-muted" />
          <div className="h-4 w-5/6 animate-pulse rounded bg-muted" />
        </div>
      </div>
    );
  }

  if (error && !data) {
    return (
      <div className="rounded-lg border bg-card p-4">
        <p className="text-sm text-muted-foreground">Unable to load top hub insights.</p>
        <button
          onClick={() => retry()}
          className="mt-2 text-sm text-primary underline hover:no-underline"
        >
          Retry
        </button>
      </div>
    );
  }

  if (!data) return null;

  return (
    <div className="rounded-lg border bg-card p-4 shadow-sm">
      <div className="flex items-start justify-between">
        <div>
          <h3 className="font-semibold">{data.title}</h3>
          <p className="text-xs text-muted-foreground">
            Hub score: {data.score.toFixed(2)} • Updated {new Date(data.updatedAt).toLocaleDateString()}
          </p>
        </div>
        <span className="inline-flex items-center rounded-full bg-primary/10 px-2.5 py-0.5 text-xs font-medium text-primary">
          Top hub
        </span>
      </div>

      <div className="mt-3 space-y-2">
        {data.insights.slice(0, 3).map((insight, i) => (
          <div key={i} className="flex gap-2 text-sm">
            <span className="mt-1 flex-shrink-0 text-primary">•</span>
            <span className="text-foreground">{insight.text}</span>
          </div>
        ))}
      </div>

      {data.related.length > 0 && (
        <div className="mt-3">
          <p className="text-xs text-muted-foreground mb-1">Related hubs</p>
          <div className="flex flex-wrap gap-1">
            {data.related.slice(0, 4).map((r) => (
              <span
                key={r.hubId}
                className="rounded bg-muted px-2 py-0.5 text-xs text-muted-foreground"
              >
                {r.title}
              </span>
            ))}
          </div>
        </div>
      )}

      <div className="mt-3 flex items-center justify-between">
        <button
          onClick={retry}
          className="text-xs text-muted-foreground underline hover:text-foreground"
        >
          Refresh
        </button>
        <a
          href={`/hub/${data.hubId}`}
          className="text-xs font-medium text-primary hover:underline"
        >
          View details →
        </a>
      </div>
    </div>
  );
}
```
