# airship / discovery

## Highest-Value Incremental Improvement (<2h)

**Goal**: Harden `airship discover` into a deterministic, **CDN-only** orchestrator that:

1. **Eliminates HF API rate limits** by using `https://huggingface.co/datasets/{repo}/resolve/main/...` CDN URLs (no auth, higher limits).
2. **Eliminates PyArrow `CastError`** by never using `load_dataset(streaming=True)` on heterogeneous repos; instead download individual files via CDN and project `{prompt, response}` at parse time.
3. **Produces a reproducible file list** (JSON) on the Mac orchestrator once per date folder, then embeds it in training so Lightning Studio does **zero API calls** during data loading.

**Why this is highest value**:
- Removes the two blockers that stall surrogate-1 training (429 + CastError).
- Fits the “Mac=CLI only, heavy compute remote” pattern.
- Enables deterministic re-runs and caching.
- Can ship in <2h (single orchestrator script + small loader change).

---

## Implementation Plan

### 1) Add `airship/discover/cdn_orchestrator.py`
- Single entrypoint: `python cdn_orchestrator.py --repo <repo> --date-folder <yyyy-mm-dd> --out-file-list <json>`
- Uses `list_repo_tree(path=date_folder, recursive=False)` **once** (after rate-limit window) to list immediate parquet files.
- Validates each file exists on CDN via `HEAD` (lightweight, counts against CDN not API).
- Emits `file_list.json`:
  ```json
  {
    "repo": "...",
    "date_folder": "...",
    "files": [
      {"path": "file-001.parquet", "cdn_url": "https://huggingface.co/datasets/.../resolve/main/...", "size": 12345},
      ...
    ],
    "generated_at": "ISO-8601",
    "generated_by": "airship-discover-cdn-orchestrator"
  }
  ```

### 2) Add `airship/discover/cdn_loader.py`
- Used inside Lightning training script.
- Accepts `file_list.json` (embedded or passed via env).
- Downloads each parquet via CDN URL with `requests.get(..., stream=True)` + `pyarrow.parquet.read_table` from buffer.
- Projects only `{prompt, response}` columns; drops everything else.
- Yields `(prompt, response)` records; never calls HF dataset APIs.

### 3) Update training launcher (Lightning SDK)
- Before `Studio.run()`, ensure `file_list.json` is present (generated by orchestrator on Mac).
- Mount or embed `file_list.json` into the training environment.
- Training script imports `cdn_loader` and iterates records; zero HF API calls during training.

### 4) Cron / scheduling hygiene (apply learned patterns)
- Ensure orchestrator script has `#!/usr/bin/env bash` wrapper if invoked via cron.
- Set `SHELL=/bin/bash` in crontab.
- On 429: wait 360s before retry (only affects initial tree listing).

### `airship/discover/cdn_orchestrator.py`
```python
#!/usr/bin/env python3
"""
CDN-only orchestrator for airship discover.
Generates a deterministic file list for a date folder to avoid HF API rate limits
and PyArrow schema errors during training.
"""

import argparse
import json
import time
from datetime import datetime, timezone
from pathlib import Path
from typing import List, Dict

import requests
from huggingface_hub import HfApi, hf_hub_url

HEADERS = {"User-Agent": "axentx-airship-discover/1.0"}

def list_date_files(repo_id: str, date_folder: str) -> List[Dict]:
    api = HfApi()
    # List immediate files only (no recursion) to minimize API use
    files = api.list_repo_tree(repo_id=repo_id, path=date_folder, recursive=False)
    entries = []
    for f in files:
        if f.rfilename.lower().endswith(".parquet"):
            entries.append(f)
    return entries

def check_cdn(url: str, timeout: int = 10) -> bool:
    try:
        r = requests.head(url, headers=HEADERS, timeout=timeout, allow_redirects=True)
        return r.status_code == 200
    except Exception:
        return False

def build_file_list(repo_id: str, date_folder: str) -> Dict:
    entries = list_date_files(repo_id, date_folder)
    files = []
    for e in entries:
        cdn_url = hf_hub_url(
            repo_id=repo_id,
            filename=e.rfilename,
            repo_type="dataset",
        ).replace("/api/", "/resolve/")  # CDN bypass
        ok = check_cdn(cdn_url)
        if not ok:
            print(f"WARNING: CDN unavailable for {e.rfilename}, skipping")
            continue
        files.append(
            {
                "path": e.rfilename,
                "cdn_url": cdn_url,
                "size": e.size or 0,
            }
        )
    return {
        "repo": repo_id,
        "date_folder": date_folder,
        "files": files,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "generated_by": "airship-discover-cdn-orchestrator",
    }

def main() -> None:
    parser = argparse.ArgumentParser(description="CDN-only file list generator")
    parser.add_argument("--repo", required=True, help="HF dataset repo id")
    parser.add_argument("--date-folder", required=True, help="Date folder path in repo")
    parser.add_argument("--out-file-list", required=True, help="Output JSON file")
    parser.add_argument("--retry-wait", type=int, default=360, help="Wait on 429 (s)")
    args = parser.parse_args()

    out_path = Path(args.out_file_list)
    out_path.parent.mkdir(parents=True, exist_ok=True)

    for attempt in range(3):
        try:
            listing = build_file_list(args.repo, args.date_folder)
            break
        except Exception as exc:
            if "429" in str(exc):
                print(f"Rate limited, waiting {args.retry_wait}s (attempt {attempt+1})")
                time.sleep(args.retry_wait)
                continue
            raise
    else:
        raise RuntimeError("Failed after retries")

    out_path.write_text(json.dumps(listing, indent=2))
    print(f"Written {len(listing['files'])} files to {out_path}")

if __name__ == "__main__":
    main()
```

### `airship/discover/cdn_loader.py`
```python
"""
CDN-only parquet loader for training.
Usage:
    from airship.discover.cdn_loader import CdnParquetIterable
    iterable = CdnParquetIterable(file_list_path="file_list.json")
    for prompt, response in iterable:
        ...
"""

import json
from pathlib import Path
from typing import Iterator, Tuple

import pyarrow.parquet as pq
import requests

class CdnParquetIterable:
    def __init__(self, file_list_path: str):
        data = json.loads(Path(file_list_path).read_text())
        self.files = data.get("files", [])
        self._validate()

    def _validate(self) -> None:
        if not self.files:
            raise ValueError("No files in file_list")
        for f in self.files:
            if "cdn_url" not in f:
                raise ValueError(f"Missing cdn_url in {f}")

    def __iter__(self) -> Iterator[Tuple[str, str]]:
        for f in self.files:
            url = f["cdn_url"]
            resp = requests.get(url, stream=True, timeout=60)
            resp.raise_for_status()
            table = pq.read_table(pq.ParquetFile(pq.ParquetBuffer(resp.content)))
            df = table.to_pandas()
            # Project only required fields; tolerate missing cols
            if "prompt" not in df.columns or "response" not in df.columns:
                continue
            for _, row in df.iterrows():
                prompt = str(row["prompt"])
                response = str(row["response"])
                if prompt.strip() and response.strip():
                    yield prompt, response
```
