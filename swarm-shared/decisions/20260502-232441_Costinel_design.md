# Costinel / design

## Implementation Plan — Costinel Top-Hub Signal Endpoint

**Highest-value incremental improvement (<2h):**  
Expose a **read-only, side-effect-free** `GET /api/v1/cost-anomaly/signal/top-hub` that surfaces the most-connected hub (e.g., `"MOC"`) and related actionable signals for governance review. Aligns with pattern: *top-hub doc insight* and *Costinel = Sense + Signal (ไม่ Execute)*.

- **Method**: `GET`
- **Path**: `/api/v1/cost-anomaly/signal/top-hub`
- **Semantics**: read-only, no mutations, no external writes, no side effects.
- **Response shape**:

```python
from fastapi import APIRouter, Depends, HTTPException
from app.services.cost_anomaly.top_hub_service import TopHubService
from app.schemas.cost_anomaly import TopHubSignalResponse
from app.core.cache import get_cache

router = APIRouter()

@router.get("/cost-anomaly/signal/top-hub", response_model=TopHubSignalResponse)
async def get_top_hub_signal(
    service: TopHubService = Depends(),
    cache = Depends(get_cache)
):
    cache_key = "top_hub_signal"
    cached = cache.get(cache_key)
    if cached:
        return cached

    try:
        result = await service.compute_top_hub_signal()
        cache.set(cache_key, result, ttl=300)
        return result
    except Exception as exc:
        raise HTTPException(status_code=500, detail="Unable to compute top-hub signal") from exc
```

#### Cache dependency (simple in-memory fallback)
```python
# app/core/cache.py
from fastapi import Depends
from typing import Any, Optional
import time

class SimpleCache:
    def __init__(self):
        self._store: dict[str, tuple[Any, float]] = {}

    def get(self, key: str) -> Optional[Any]:
        entry = self._store.get(key)
        if entry and entry[1] > time.time():
            return entry[0]
        self._store.pop(key, None)
        return None

    def set(self, key: str, value: Any, ttl: int = 300) -> None:
        self._store[key] = (value, time.time() + ttl)

_cache = SimpleCache()

def get_cache():
    return _cache
```
