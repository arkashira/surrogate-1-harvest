## Implementation Plan — Costinel top-hub signal endpoint (<2h)

**Goal:** Add a read-only endpoint `GET /api/v1/cost-anomaly/signal/top-hub` (optional `?for_date=YYYY-MM-DD`) that returns the strongest knowledge-graph hub (e.g., "MOC") as a cost-anomaly signal — consistent with Sense + Signal (no execution).

### Scope (ship in <2h)
- Add FastAPI route under `/api/v1/cost-anomaly/signal/top-hub`
- Query knowledge-rag graph for top hub (by degree/weight) for requested date (defaults to today)
- Return JSON signal: `{ hub, score, date, context_docs, tags }`
- No state changes, no side effects (read-only)
- Minimal dependencies: reuse existing graph/RAG utilities if present; otherwise lightweight local query.

#### 2) Knowledge RAG service (`services/knowledge_rag.py`)
```python
import datetime
from typing import List, Tuple, Optional
from models.signal import TopHubSignal

# Lightweight in-memory graph stub; replace with real graph client if available.
# This keeps the change small and functional for immediate ops value.
class KnowledgeGraph:
    def __init__(self):
        # pre-seeded example edges: (source, target, weight, date)
        self.edges = [
            ("MOC", "cost-spike-2026-05-01", 0.92, "2026-05-01"),
            ("MOC", "aws-ec2-ri-gap", 0.87, "2026-05-01"),
            ("Billing", "gcp-commitment", 0.65, "2026-05-01"),
            ("MOC", "azure-vm-idle", 0.81, "2026-05-02"),
        ]

    def top_hub_by_date(self, for_date: str) -> Optional[Tuple[str, float, List[str]]]:
        # Filter edges by date, sum weights per node (as source), pick max
        from collections import defaultdict
        node_score = defaultdict(float)
        node_docs = defaultdict(list)

        for src, tgt, w, d in self.edges:
            if d == for_date:
                node_score[src] += w
                node_docs[src].append(tgt)

        if not node_score:
            # fallback: use latest date available
            latest = max((d for _, _, _, d in self.edges), default=None)
            if latest is None:
                return None
            for src, tgt, w, d in self.edges:
                if d == latest:
                    node_score[src] += w
                    node_docs[src].append(tgt)
            for_date = latest

        hub = max(node_score, key=node_score.get)
        return hub, node_score[hub], node_docs[hub]

_kg = KnowledgeGraph()

def get_top_hub(for_date: Optional[str] = None) -> TopHubSignal:
    if for_date is None:
        for_date = datetime.date.today().isoformat()
    result = _kg.top_hub_by_date(for_date)
    if result is None:
        # graceful fallback
        hub, score, docs = "UNKNOWN", 0.0, []
    else:
        hub, score, docs = result

    return TopHubSignal(
        hub=hub,
        score=round(score, 4),
        date=for_date,
        context_docs=docs,
        tags=["knowledge-rag", "graph", "hub", "cost-anomaly-signal"]
    )
```

#### 3) FastAPI route (`app/api/v1/endpoints/cost_anomaly.py`)
```python
from fastapi import APIRouter, Query
from services.knowledge_rag import get_top_hub
from models.signal import TopHubSignal

router = APIRouter(prefix="/cost-anomaly/signal", tags=["cost-anomaly-signal"])

@router.get("/top-hub", response_model=TopHubSignal)
async def top_hub_signal(
    for_date: str = Query(None, description="Date (YYYY-MM-DD); defaults to today")
):
    """
    Returns the strongest knowledge-graph hub as a cost-anomaly signal.
    Read-only. No execution.
    """
    return get_top_hub(for_date=for_date)
```
