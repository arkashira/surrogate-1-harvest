## Deployment
cf workers

## v0.1 scope
v0.1 in 2 weeks: CLI + FastAPI endpoint that scans configured AWS and GCP projects, persists schedules to Supabase Postgres, flags duplicates (identical cron+target), drifted (expression changed since last run), and orphans (schedule exists but target function missing), exports a CSV/JSON report and retirements list; no UI, no scheduler, no automated remediation.
