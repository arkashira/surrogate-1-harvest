# Costinel / ops

## Final Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

**Core decision**: Use **static JSON in `public/hubs/`** (CDN-first) with **graceful non-blocking fallback**. This is the simplest, fastest, and safest path that aligns with the "Sense + Signal — ไม่ Execute" philosophy and avoids any backend changes or API rate limits.

### Scope (what we ship in <2h)
1. **Static hub data**: `public/hubs/moc.json` (MOC = most-connected hub per 2026-04-27 pattern)
2. **UI component**: `src/components/HubSignalPanel.vue` — non-blocking, dismissible, CDN-first fetch
3. **Config**: env var `VITE_HUB_NAME` (default: `moc`)
4. **Build**: zero new dependencies, zero backend changes, works offline via static file

### Architecture (CDN-first)
- **Primary**: Fetch `/hubs/{hubName}.json` from CDN (public/) — no auth, no rate limit, instant
- **Fallback**: Local minimal dataset bundled in component if CDN fails (offline/dev)
- **Non-blocking**: Panel loads async; UI renders immediately; errors silently ignored
- **Dismissible**: User can close; preference saved to `localStorage`

### File changes
```
public/hubs/
  └── moc.json          # static hub data (generated/updated by ops)
src/components/
  └── HubSignalPanel.vue
src/App.vue             # mount panel (top banner or sidebar)
.env.example            # VITE_HUB_NAME=moc
```

### 2) Component (src/components/HubSignalPanel.vue)
```vue
<template>
  <div v-if="visible && hasData" class="hub-signal-panel" :class="`severity-${panelSeverity}`">
    <div class="hub-signal-panel__inner">
      <div class="hub-signal-panel__header">
        <div class="hub-signal-panel__title">
          <span class="hub-signal-panel__dot" />
          {{ hubData.title }}
        </div>
        <button
          class="hub-signal-panel__close"
          @click="dismiss"
          aria-label="Dismiss"
        >
          ✕
        </button>
      </div>

      <p class="hub-signal-panel__subtitle">{{ hubData.subtitle }}</p>

      <div class="hub-signal-panel__signals">
        <div
          v-for="s in hubData.signals"
          :key="s.id"
          class="hub-signal-panel__signal"
          :class="`signal-${s.severity}`"
        >
          <div class="hub-signal-panel__signal-title">{{ s.title }}</div>
          <div class="hub-signal-panel__signal-desc">{{ s.description }}</div>
          <a
            v-if="s.action"
            :href="s.action.href"
            class="hub-signal-panel__signal-action"
          >
            {{ s.action.label }} →
          </a>
        </div>
      </div>

      <div class="hub-signal-panel__footer">
        <small class="hub-signal-panel__meta">
          Updated {{ formattedDate }}
        </small>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  name: "HubSignalPanel",
  props: {
    hubName: {
      type: String,
      default: () => import.meta.env.VITE_HUB_NAME || "moc"
    },
    dismissible: {
      type: Boolean,
      default: true
    }
  },
  data() {
    return {
      hubData: null,
      visible: true,
      fetchError: null
    };
  },
  computed: {
    hasData() {
      return this.hubData && this.hubData.signals && this.hubData.signals.length > 0;
    },
    formattedDate() {
      if (!this.hubData?.generatedAt) return "";
      try {
        return new Date(this.hubData.generatedAt).toLocaleDateString(undefined, {
          month: "short",
          day: "numeric",
          hour: "2-digit",
          minute: "2-digit"
        });
      } catch {
        return this.hubData.generatedAt;
      }
    },
    panelSeverity() {
      if (!this.hubData?.signals?.length) return "info";
      const hasWarning = this.hubData.signals.some((s) => s.severity === "warning");
      const hasCritical = this.hubData.signals.some((s) => s.severity === "critical");
      return hasCritical ? "critical" : hasWarning ? "warning" : "info";
    }
  },
  async mounted() {
    // Check dismiss preference
    if (this.dismissible) {
      const dismissed = localStorage.getItem(`hub-panel-dismissed-${this.hubName}`);
      if (dismissed === "true") {
        this.visible = false;
        return;
      }
    }

    await this.loadHubData();
  },
  methods: {
    async loadHubData() {
      const url = `/hubs/${this.hubName}.json?t=${Date.now()}`;

      try {
        const res = await fetch(url, { cache: "no-cache" });
        if (!res.ok) throw new Error(`HTTP ${res.status}`);
        const json = await res.json();
        this.hubData = json;
      } catch (err) {
        this.fetchError = err;
        // Graceful fallback: try bundled minimal data (if provided via inline script)
        // For now, silently fail — non-blocking behavior.
        console.debug("[HubSignalPanel] CDN fetch failed, skipping panel", err);
        this.visible = false;
      }
    },
    dismiss() {
      this.visible = false;
      if (this.dismissible) {
        localStorage.setItem(`hub-panel-dismissed-${this.hubName}`, "true");
      }
      this.$emit("dismissed");
    }
  },
  emits: ["dismissed"]
};
</script>
