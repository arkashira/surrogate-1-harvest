# Costinel / design

## Implementation Plan — Top-hub Signal Panel (Costinel Dashboard)

**Scope**  
- Frontend-only, read-only panel.  
- Surface the most-connected hub (e.g., “MOC”) and actionable proposals from the knowledge graph.  
- Resilient to missing backend: graceful fallback UI when graph data is unavailable.  

**Timebox**  
- <2h (design + minimal viable implementation)

**Stack**  
- React (existing Costinel frontend)  
- Tailwind (existing styles)  
- Optional: lightweight fetch to `/api/graph/hubs/top` (if available) — otherwise static fallback.

---

## Design Decisions (best-judgement)

1. **Panel placement**: Add to the main dashboard sidebar/header area as a compact “Top Signal” card so it’s visible but non-blocking.  
2. **Data contract** (frontend-first):  
   - GET `/api/graph/hubs/top` → `{ hub: { id, name, type, connections, score }, proposals: [{ id, title, reason, action }] }`  
   - If 404/5xx → render fallback card with placeholder and “Enable graph insights” call-to-action.  
3. **Graceful degradation**:  
   - Show cached/last-known hub from `localStorage` if available.  
   - If nothing available, show neutral empty state (no errors in UI).  
4. **Accessibility**:  
   - Use semantic `<section aria-label="Top hub signal">`.  
   - Keyboard-focusable proposals list (buttons/links).  
5. **Performance**:  
   - Fetch on mount with `AbortController` and 4s timeout.  
   - Do not block dashboard render; panel can be lazy or skeleton-first.

---

## Code Snippets

### 1) API helper (`src/lib/graphApi.ts`)
```ts
// Lightweight, resilient graph API helper for Top-hub signal
export interface Hub {
  id: string;
  name: string;
  type: string;
  connections: number;
  score: number;
}

export interface Proposal {
  id: string;
  title: string;
  reason: string;
  action?: string; // e.g., "/proposals/{id}"
}

export interface TopHubResponse {
  hub: Hub;
  proposals: Proposal[];
}

const API_PATH = '/api/graph/hubs/top';
const TIMEOUT_MS = 4000;

export async function fetchTopHub(signal?: AbortSignal): Promise<TopHubResponse | null> {
  try {
    const controller = new AbortController();
    const timeout = setTimeout(() => controller.abort(), TIMEOUT_MS);
    const combinedSignal = signal
      ? (() => {
          const combined = new AbortController();
          signal.addEventListener('abort', () => combined.abort());
          controller.signal.addEventListener('abort', () => combined.abort());
          return combined.signal;
        })()
      : controller.signal;

    const res = await fetch(API_PATH, { signal: combinedSignal });
    clearTimeout(timeout);

    if (!res.ok) {
      // 404 or 5xx -> graceful fallback
      return null;
    }
    return await res.json();
  } catch {
    return null;
  }
}
```

  ts: number;
}

function getCached(): TopHubResponse | null {
  try {
    const raw = localStorage.getItem(STORAGE_KEY);
    if (!raw) return null;
    const parsed: Cached = JSON.parse(raw);
    // cache for 6h
    if (Date.now() - parsed.ts > 6 * 60 * 60 * 1000) return null;
    return { hub: parsed.hub, proposals: parsed.proposals };
  } catch {
    return null;
  }
}
