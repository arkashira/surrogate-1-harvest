# vanguard / design

## 1. Diagnosis

- No deterministic **CDN-first manifest** exists; ingestion/training scripts likely still call `list_repo_tree`/`load_dataset` at runtime, risking 429s and non-reproducible runs.
- Missing **content-addressed file list** keyed by date/slug; training jobs cannot pin exact file set and may silently drift across HF updates.
- No **guardrail script** to validate environment (shebang, executable bits, SHELL) before invoking wrappers (cron/lightning/kaggle), repeating prior wrapper exec errors.
- Training/ingestion may still attempt mixed-schema parquet projection at load time instead of projecting to `{prompt,response}` before upload, violating surrogate-1 schema rules.
- No reuse strategy for Lightning Studio (list + reuse running) — wastes quota and risks idle-stop killing training.

## 2. Proposed change

Create `/opt/axentx/vanguard/ops/cdn-manifest.py` and `/opt/axentx/vanguard/ops/guard-runtime.sh`:

- `cdn-manifest.py`: single Mac-side script that (after rate-limit window) calls `list_repo_tree(path, recursive=False)` for one date folder, produces `manifest-{date}.json` with CDN URLs and content hashes, and embeds it for Lightning training (zero API calls during data load).
- `guard-runtime.sh`: wrapper guard to enforce shebang, executable bit, and `SHELL=/bin/bash`; used by cron jobs and local wrappers before invoking `kaggle`, `lightning`, or `active-learning` scripts.

Scope:
- Add `ops/` directory and the two files above.
- Update any existing ingestion/training entrypoints to accept `--manifest` and use CDN-only fetches.

### `/opt/axentx/vanguard/ops/guard-runtime.sh`

```bash
#!/usr/bin/env bash
# guard-runtime.sh — enforce runtime preconditions for cron/wrapper scripts
set -euo pipefail

SCRIPT_PATH="${1:-}"
shift || true

if [[ -z "$SCRIPT_PATH" ]]; then
  echo "Usage: $0 <script> [args...]" >&2
  exit 1
fi

if [[ ! -x "$SCRIPT_PATH" ]]; then
  echo "Fixing executable bit: $SCRIPT_PATH" >&2
  chmod +x "$SCRIPT_PATH"
fi

# Ensure Bash is used
export SHELL=/bin/bash

# If script lacks Bash shebang, warn but still run via bash
if ! head -1 "$SCRIPT_PATH" | grep -qE '^#!/usr/bin/env bash|^#!/bin/bash'; then
  echo "WARN: missing Bash shebang in $SCRIPT_PATH — invoking via bash" >&2
  exec bash "$SCRIPT_PATH" "$@"
else
  exec "$SCRIPT_PATH" "$@"
fi
```

### `/opt/axentx/vanguard/ops/cdn-manifest.py`

```python
#!/usr/bin/env python3
"""
Generate a CDN-first manifest for a Hugging Face dataset repo folder.
Usage:
  python cdn-manifest.py <repo_id> <date_folder> --out manifest-2026-04-29.json
"""
import argparse
import hashlib
import json
import os
import sys
from datetime import datetime
from pathlib import Path

# Prefer local HF_HOME if set; avoid global collisions
HF_HOME = os.getenv("HF_HOME", str(Path.home() / ".cache" / "huggingface"))
os.environ["HF_HOME"] = HF_HOME

try:
    from huggingface_hub import HfApi, hf_hub_download
except ImportError:
    print("ERROR: install huggingface_hub (pip install huggingface_hub)", file=sys.stderr)
    sys.exit(1)

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo_id}/resolve/main/{repo_path}"

def list_date_files(repo_id: str, date_folder: str):
    """
    List files in repo under date_folder (non-recursive by design).
    Returns list[dict] with repo_path, cdn_url, size.
    """
    api = HfApi()
    # list_repo_tree non-recursive to avoid pagination explosion
    tree = api.list_repo_tree(repo_id=repo_id, path=date_folder, repo_type="dataset", recursive=False)
    entries = []
    for item in tree:
        if item.type != "file":
            continue
        repo_path = item.path
        entries.append({
            "repo_path": repo_path,
            "cdn_url": CDN_TEMPLATE.format(repo_id=repo_id, repo_path=repo_path),
            "size": getattr(item, "size", None),
        })
    return entries

def build_manifest(repo_id: str, date_folder: str, out_path: str):
    manifest = {
        "repo_id": repo_id,
        "date_folder": date_folder,
        "created_at": datetime.utcnow().isoformat() + "Z",
        "files": list_date_files(repo_id, date_folder),
    }
    # content-address manifest by hash of canonical file list
    file_list_json = json.dumps([f["repo_path"] for f in manifest["files"]], sort_keys=True)
    manifest["manifest_hash"] = hashlib.sha256(file_list_json.encode()).hexdigest()[:16]

    Path(out_path).write_text(json.dumps(manifest, indent=2))
    print(f"Wrote manifest to {out_path} (hash={manifest['manifest_hash']})")
    return manifest

if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Generate CDN-first manifest for HF dataset folder")
    parser.add_argument("repo_id", help="HF dataset repo id (e.g., 'axentx/surrogate-1')")
    parser.add_argument("date_folder", help="Folder path in repo (e.g., 'batches/mirror-merged/2026-04-29')")
    parser.add_argument("--out", default="manifest.json", help="Output JSON path")
    args = parser.parse_args()

    build_manifest(args.repo_id, args.date_folder, args.out)
```
