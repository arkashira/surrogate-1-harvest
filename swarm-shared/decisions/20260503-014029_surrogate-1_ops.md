# Final Synthesis — CDN-Bypass Ingestion Pipeline

**Chosen improvement** (highest-value, <2h):  
Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion pipeline** that eliminates HF API rate limits (429) and mixed-schema `pyarrow` errors, and produces clean `{prompt,response}` JSONL shards.

**Why this wins**:
- Directly applies **HF CDN bypass** (`resolve/main/...`) → zero auth/rate-limit cost during worker runs.
- Removes runtime HF API calls → deterministic, reliable 16-way parallelism.
- Projects heterogeneous schemas to `{prompt,response}` → prevents `CastError`.
- Minimal refactor: keep existing 16-shard GitHub Actions layout; only swap ingestion logic.
- Ship time <2h: two small scripts + one updated shell entrypoint.

### 2) Replace `bin/dataset-enrich.sh` with CDN-bypass, manifest-driven sharding

`bin/dataset-enrich.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail
export SHELL=/bin/bash

REPO="axentx/surrogate-1-training-pairs"
DATE_PATH="${DATE_PATH:-$(date +%Y-%m-%d)}"
SHARD_ID="${SHARD_ID:-0}"
TOTAL_SHARDS="${TOTAL_SHARDS:-16}"
MANIFEST_FILE="${MANIFEST_FILE:-manifest-${DATE_PATH}.json}"
OUTDIR="batches/public-merged/${DATE_PATH}"
mkdir -p "${OUTDIR}"
TIMESTAMP=$(date +%H%M%S)
OUTFILE="${OUTDIR}/shard${SHARD_ID}-${TIMESTAMP}.jsonl"

echo "[$(date)] Shard ${SHARD_ID}/${TOTAL_SHARDS} starting"

# Load manifest (preferred) or fallback to API (rate-limited)
if [[ -f "${MANIFEST_FILE}" ]]; then
  mapfile -t ALL_FILES < <(jq -r '.[]' "${MANIFEST_FILE}")
else
  echo "Warning: manifest not found, falling back to API (rate-limited)"
  mapfile -t ALL_FILES < <(python3 -c "
import os, json
from huggingface_hub import list_repo_tree
files = list_repo_tree('${REPO}', path='public-merged/${DATE_PATH}', recursive=False)
print('\n'.join([f['path'] for f in files if f['type']=='file']))
")
fi

# Deterministic shard assignment by path hash
mapfile -t MY_FILES < <(
  for f in "${ALL_FILES[@]}"; do
    hash=$(echo -n "$f" | md5sum | awk '{print $1}')
    bucket=$(( 0x${hash:0:8} % TOTAL_SHARDS ))
    if [[ $bucket -eq $SHARD_ID ]]; then
      echo "$f"
    fi
  done
)

echo "Processing ${#MY_FILES[@]} files for shard ${SHARD_ID}"

# Process files via CDN download + schema projection
python3 bin/process_shard.py "${MY_FILES[@]}" --outfile "${OUTFILE}" --repo "${REPO}"

echo "[$(date)] Shard ${SHARD_ID} wrote ${OUTFILE}"
```

### 3) Robust per-file CDN download and schema projection

`bin/process_shard.py`
```python
#!/usr/bin/env python3
import json, sys, io
from urllib.request import urlopen
from urllib.error import URLError
import pyarrow.parquet as pq
import pyarrow as pa

def cdn_url(repo, path):
    return f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def normalize(obj):
    prompt = obj.get("prompt") or obj.get("input") or obj.get("question") or ""
    response = obj.get("response") or obj.get("output") or obj.get("answer") or ""
    if not prompt or not response:
        return None
    return {"prompt": str(prompt), "response": str(response)}

def read_parquet_or_jsonl(data_bytes):
    # Try parquet first
    try:
        table = pq.read_table(io.BytesIO(data_bytes))
        return table.to_pylist()
    except Exception:
        pass
    # Fallback: JSON lines
    lines = data_bytes.decode("utf-8", errors="ignore").strip().splitlines()
    return [json.loads(l) for l in lines if l.strip()]

def main():
    args = sys.argv[1:]
    outfile = None
    repo = None
    files = []

    i = 0
    while i < len(args):
        if args[i] == "--outfile":
            outfile = args[i + 1]
            i += 2
        elif args[i] == "--repo":
            repo = args[i + 1]
            i += 2
        else:
            files.append(args[i])
            i += 1

    if not outfile or not repo:
        print("Usage: process_shard.py file1 file2 ... --outfile X --repo Y")
        sys.exit(1)

    with open(outfile, "w", encoding="utf-8") as fout:
        for rel_path in files:
            try:
                url = f"https://huggingface.co/datasets/{repo}/resolve/main/{rel_path}"
                with urlopen(url, timeout=30) as resp:
                    data = resp.read()
                rows = read_parquet_or_jsonl(data)
                for obj in rows:
                    pair = normalize(obj)
                    if pair:
                        fout.write(json.dumps(pair, ensure_ascii=False) + "\n")
            except (URLError, Exception) as exc:
                print(f"Skipping {rel_path} due to error: {exc}", file=sys.stderr)

if __name__ == "__main__":
    main()
```

## Usage in GitHub Actions (unchanged layout)

```yaml
- name: Generate manifest (once per date)
  run: |
    HF_TOKEN=${{ secrets.HF_TOKEN }} \
    ./bin/gen-manifest.sh axentx/surrogate-1-training-pairs ${{ env.DATE_PATH }} > manifest-${{ env.DATE_PATH }}.json

- name: Run 16 shards in parallel
  run: |
    for SHARD in $(seq 0 15); do
      SHARD_ID=$SHARD TOTAL_SHARDS=16 \
      MANIFEST_FILE=manifest-${{ env.DATE_PATH }}.json \
      DATE_PATH=${{ env.DATE_PATH }} \
      ./bin/dataset-enrich.sh &
    done
    wait
```
