## v0.1 scope
v0.1 in 2 weeks: single deterministic workflow (research/signal ingestion -> risk guardrail check -> simulated execution) with immutable audit trail, Cloudflare Workers edge runtime, Supabase Postgres for event store, Redis cache for risk limits, and replay endpoint to re-run a workflow run deterministically; includes adapter stubs for one broker/OMS and compliance rule engine with block/warn modes.

- What compliance rule set is non-negotiable for launch (e.g., max notional, restricted symbols, pre-trade checks)?
- What latency budget (p99) from signal receipt to execution submit is acceptable for target workflows?
