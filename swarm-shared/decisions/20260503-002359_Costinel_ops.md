# Costinel / ops

## Implementation Plan — Top-Hub Signal Card (Costinel / ops)

**Scope**: A production-ready, read-only ops card that surfaces the most-connected knowledge hub (e.g., “MOC”) with contextual cost-governance signals. Uses CDN-bypass for live cost data and embedded mock fallback for resilience. Ships in <2h.

### 1. Architecture Decisions
- **Component**: `src/components/cards/TopHubSignalCard.vue` (Vue 3, Composition API, `<script setup>`)
- **Data strategy**:
  - Live: CDN-bypass fetch from `https://huggingface.co/datasets/{repo}/resolve/main/batches/mirror-merged/{date}/` (zero API calls during runtime)
  - Fallback: embedded `mockTopHub` when CDN unavailable or during dev
- **Ops signals**: anomaly score, top-cost driver, RI coverage gap, forecast delta, audit trail link
- **No mutations**: strictly read-only (`Sense + Signal`)

### 2. File Changes

#### `src/components/cards/TopHubSignalCard.vue`
```vue
<template>
  <div class="top-hub-signal-card">
    <div class="card-header">
      <div class="hub-badge">
        <span class="hub-icon">🏆</span>
        <span class="hub-name">{{ hub.name }}</span>
        <span class="hub-score">{{ hub.connectivity }} pts</span>
      </div>
      <div class="status-row">
        <span :class="['status-dot', { live: isLive }]"></span>
        <span class="status-text">{{ isLive ? 'Live CDN' : 'Mock Fallback' }}</span>
        <span class="updated-at">{{ lastUpdated }}</span>
      </div>
    </div>

    <div class="signal-grid">
      <div class="signal-item signal-critical" v-if="hub.anomalyScore > 70">
        <div class="signal-label">Anomaly Score</div>
        <div class="signal-value">{{ hub.anomalyScore }}%</div>
        <div class="signal-desc">Unusual spend spike detected</div>
      </div>

      <div class="signal-item">
        <div class="signal-label">Top Cost Driver</div>
        <div class="signal-value">{{ hub.topDriver.service }}</div>
        <div class="signal-desc">
          {{ hub.topDriver.currency }} {{ hub.topDriver.amount }} ({{ hub.topDriver.delta }} WoW)
        </div>
      </div>

      <div class="signal-item" :class="{ 'signal-warning': hub.riCoverage < 60 }">
        <div class="signal-label">RI Coverage</div>
        <div class="signal-value">{{ hub.riCoverage }}%</div>
        <div class="signal-desc">Potential savings {{ hub.riSavings }}</div>
      </div>

      <div class="signal-item">
        <div class="signal-label">Forecast Delta</div>
        <div class="signal-value" :class="hub.forecastDelta > 0 ? 'text-up' : 'text-down'">
          {{ hub.forecastDelta > 0 ? '+' : '' }}{{ hub.forecastDelta }}%
        </div>
        <div class="signal-desc">EoM projection vs plan</div>
      </div>
    </div>

    <div class="card-footer">
      <a :href="hub.auditUrl" target="_blank" rel="noopener" class="audit-link">
        View Audit Trail
      </a>
      <button @click="refresh" class="refresh-btn" :disabled="loading">
        {{ loading ? 'Loading…' : 'Refresh' }}
      </button>
    </div>
  </div>
</template>

<script setup>
import { ref, computed, onMounted } from 'vue'

const props = defineProps({
  repo: { type: String, default: 'datasets/axentx/costinel-mirror' },
  dateFolder: { type: String, default: () => new Date().toISOString().slice(0, 10).replace(/-/g, '') }
})

const loading = ref(false)
const isLive = ref(false)
const rawData = ref(null)
const lastUpdated = ref(new Date().toLocaleTimeString())

const mockTopHub = {
  name: 'MOC',
  connectivity: 94,
  anomalyScore: 78,
  topDriver: { service: 'AmazonEC2', amount: 42780, currency: '$', delta: '+12%' },
  riCoverage: 54,
  riSavings: '$8.4k/mo',
  forecastDelta: 9,
  auditUrl: '/audit/runs/20260503-002247'
}

const hub = computed(() => {
  if (!rawData.value) return mockTopHub
  // CDN payload shape: { hub, signals, timestamp }
  return {
    name: rawData.value.hub?.name || mockTopHub.name,
    connectivity: rawData.value.hub?.connectivity || mockTopHub.connectivity,
    anomalyScore: rawData.value.signals?.anomalyScore || mockTopHub.anomalyScore,
    topDriver: rawData.value.signals?.topDriver || mockTopHub.topDriver,
    riCoverage: rawData.value.signals?.riCoverage || mockTopHub.riCoverage,
    riSavings: rawData.value.signals?.riSavings || mockTopHub.riSavings,
    forecastDelta: rawData.value.signals?.forecastDelta || mockTopHub.forecastDelta,
    auditUrl: rawData.value.hub?.auditUrl || mockTopHub.auditUrl
  }
})

async function fetchFromCDN() {
  // CDN bypass: no Authorization header; public resolve URLs
  const fileUrl = `https://huggingface.co/datasets/${props.repo}/resolve/main/batches/mirror-merged/${props.dateFolder}/top-hub.json`
  try {
    const res = await fetch(fileUrl, { cache: 'no-store' })
    if (!res.ok) throw new Error(`CDN ${res.status}`)
    const data = await res.json()
    rawData.value = data
    isLive.value = true
    lastUpdated.value = new Date().toLocaleTimeString()
  } catch (err) {
    console.warn('CDN unavailable, using fallback:', err.message)
    rawData.value = null
    isLive.value = false
  }
}

async function refresh() {
  loading.value = true
  await fetchFromCDN()
  loading.value = false
}

onMounted(() => {
  fetchFromCDN()
})
</script>
