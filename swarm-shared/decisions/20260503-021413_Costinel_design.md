# Costinel / design

## Final Implementation Plan — Top-Hub Signal Panel (Costinel Dashboard)

**Scope & Value**  
Frontend-only, read-only React panel that surfaces the highest-signal / most-connected hub (default “MOC”) and its top 3 actionable proposals from the knowledge graph. CDN-first data path, zero backend changes, ships in <2h.

---

### 1) Acceptance Criteria (merged & tightened)
- [x] Panel appears on main dashboard under “Top Signal” section.  
- [x] Shows hub badge, name, description, connection count, and signal score.  
- [x] Shows top 3 proposals with title, 1-line rationale, impact, and confidence.  
- [x] Each proposal links to internal detail (`/proposals/:id`) or external doc (respects `external` flag).  
- [x] Graceful fallbacks: loading → empty/error → stale cached copy (if available).  
- [x] CDN-first: static JSON at `/public/data/knowledge-graph/top-hub.json` (env override optional) to avoid API/auth limits.

### 5) Component implementation

`src/components/TopHubSignalPanel/TopHubSignalPanel.tsx`

```tsx
import React, { useEffect, useState } from "react";
import styles from "./TopHubSignalPanel.module.css";
import type { KnowledgeGraphData } from "./types";

const DATA_URL = `${process.env.PUBLIC_URL}/data/knowledge-graph/top-hub.json`;
const STALE_MS = 1000 * 60 * 60 * 24; // 24h

const TopHubSignalPanel: React.FC = () => {
  const [data, setData] = useState<KnowledgeGraphData | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    const load = async () => {
      try {
        const res = await fetch(DATA_URL, { cache: "no-store" });
        if (!res.ok) throw new Error(`Failed to load hub signals: ${res.status}`);
        const json = (await res.json()) as KnowledgeGraphData;
        setData(json);
        setError(null);
      } catch (err: any) {
        setError(err?.message || "Unable to load signals.");
      } finally {
        setLoading(false);
      }
    };

    load();
  }, []);

  // Fallback to stale cache in localStorage if available and not too old
  useEffect(() => {
    if (!loading || data) return;
    try {
      const raw = localStorage.getItem("costinel:top-hub-cache");
      if (!raw) return;
      const cached = JSON.parse(raw) as KnowledgeGraphData;
      const age = Date.now() - new Date(cached.generatedAt).getTime();
      if (age < STALE_MS) {
        setData(cached);
        setError(null);
      }
    } catch {
      // ignore malformed cache
    }
  }, [loading, data]);

  // Persist successful fetch to localStorage
  useEffect(() => {
    if (data) {
      try {
        localStorage.setItem("costinel:top-hub-cache", JSON.stringify(data));
      } catch {
        // ignore storage failures
      }
    }
  }, [data]);

  if (loading && !data) return <div className={styles.panel}>Loading signals…</div>;
  if (error && !data) return <div className={styles.panel}>Unable to load signals.</div>;
  if (!data) return null;

  const { hub, proposals } = data;
  const topThree = proposals.slice(0, 3);

  return (
    <div className={styles.panel} role="region" aria-label="Top hub signals">
      <div className={styles.header}>
        <div>
          <span className={styles.badge}>{hub.label}</span>
          <h3 className={styles.title}>{hub.name}</h3>
          <p className={styles.meta}>
            {hub.connectionsCount} connections · Signal {hub.signalScore}
          </p>
        </div>
      </div>

      <p className={styles.description}>{hub.description}</p>

      <div className={styles.proposals}>
        {topThree.map((p) => (
          <a
            key={p.id}
