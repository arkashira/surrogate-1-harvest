# Costinel / design

## Final Implementation Plan — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only, zero-backend card that reads a lightweight knowledge-rag graph, identifies the highest-degree hub, and surfaces it with 3 contextual signals.  
**Timebox**: ≤2h.  
**Pattern**: Sense + Signal (read-only; no execution).

---

### 1) Highest-value incremental improvement (merged)
Ship a reusable card that:
- Reads a local/artifact graph snapshot (JSON) produced by `knowledge-rag`.
- Computes the most-connected hub client-side (by degree; tie-break by connection weight if available).
- Renders a concise card: hub name, short description, connection count, and 3 contextual signals (neighbor highlights with optional doc links).
- Uses existing design tokens, fits the current dashboard surface, and requires no new backend infra or auth.

**Why this now**: Directly applies “top-hub doc insight” and “knowledge-rag” patterns, delivers immediate visibility, and is buildable in <2h.

import React, { useEffect, useState } from "react";
import "./TopHubSignalCard.css";

type Node = { id: string; label: string; type?: string; description?: string };
type Edge = { source: string; target: string; weight?: number; doc?: string };
type Graph = { nodes: Node[]; edges: Edge[] };

const TopHubSignalCard: React.FC = () => {
  const [topHub, setTopHub] = useState<Node | null>(null);
  const [degree, setDegree] = useState(0);
  const [signals, setSignals] = useState<Array<Node & { doc?: string; weight?: number }>>([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    const load = async () => {
      try {
        const res = await fetch(`${process.env.PUBLIC_URL}/data/knowledge-rag/hubs.json`, {
          cache: "no-store"
        });
        if (!res.ok) throw new Error(`Failed to load graph: ${res.status}`);
        const graph: Graph = await res.json();

        const degreeMap: Record<string, number> = {};
        const neighborsMap: Record<string, Array<{ node: Node; weight: number; doc?: string }>> = {};
        const nodeMap = new Map(graph.nodes.map((n) => [n.id, n]));

        graph.nodes.forEach((n) => {
          degreeMap[n.id] = 0;
          neighborsMap[n.id] = [];
        });

        graph.edges.forEach((e) => {
          degreeMap[e.source] = (degreeMap[e.source] || 0) + 1;
          degreeMap[e.target] = (degreeMap[e.target] || 0) + 1;

          const weight = e.weight || 1;
          if (nodeMap.has(e.target)) {
            neighborsMap[e.source].push({ node: nodeMap.get(e.target)!, weight, doc: e.doc });
          }
          if (nodeMap.has(e.source)) {
            neighborsMap[e.target].push({ node: nodeMap.get(e.source)!, weight, doc: e.doc });
          }
        });

        const topId = Object.entries(degreeMap).reduce((a, b) => (b[1] > a[1] ? b : a))[0];
        const hubNode = nodeMap.get(topId) || graph.nodes[0];
        const hubDegree = degreeMap[topId] || 0;

        const neighborSignals = (neighborsMap[topId] || [])
          .sort((a, b) => b.weight - a.weight)
          .slice(0, 3)
          .map(({ node, weight, doc }) => ({ ...node, weight, doc }));

        setTopHub(hubNode);
        setDegree(hubDegree);
        setSignals(neighborSignals);
      } catch (err: any) {
        setError(err.message || "Unknown error");
      } finally {
        setLoading(false);
      }
    };

    load();
  }, []);

  if (loading) return <div className="top-hub-card loading" aria-busy="true">Loading insights…</div>;
