## Highest-value incremental improvement (<2h)
Replace any remaining recursive HF ingestion or `load_dataset(streaming=True)` in the surrogate training pipeline with a **manifest-driven, CDN-only iterable loader** generated once on the Mac orchestrator. This eliminates HF API rate limits during training and aligns with the surrogate training pattern (single folder listing → JSON manifest → Lightning Studio CDN-only fetches).

---

## Implementation plan (≤2h)

1. **Locate ingestion/training entrypoints**  
   Check `surrogate/` for scripts that call `load_dataset`, `list_repo_files`, or recursive ingestion.

2. **Add Mac-side manifest generator**  
   Create `scripts/build_hf_manifest.py` that:
   - Uses `list_repo_tree(path, recursive=False)` per folder (non-recursive, paginated safely)
   - Walks one date folder (e.g. `batches/mirror-merged/2026-05-03/`)
   - Emits `manifests/{date}_filelist.json` with `{ "repo": "...", "path": "...", "cdn_url": "https://huggingface.co/datasets/.../resolve/main/..." }`

3. **Add CDN-only iterable dataset**  
   Create `surrogate/data/hf_cdn_dataset.py` with a lightweight `IterableDataset` that:
   - Loads the manifest JSON
   - Streams files via `requests.get(cdn_url, timeout=60, stream=True)`
   - Parses only `{prompt, response}` at read time (no schema assumptions)
   - Never calls HF API during training

4. **Wire into training script**  
   Update training launcher to:
   - Accept `--manifest` arg
   - Use `hf_cdn_dataset` as the data source
   - Skip any `load_dataset` calls

5. **Lightning Studio reuse guard**  
   Ensure launcher checks for running studios and reuses them (saves quota).

6. **Test locally (small sample)**  
   Run manifest build → single-epoch train on 10 files to verify CDN fetch + parsing.

### 1) Mac-side manifest builder (`scripts/build_hf_manifest.py`)
```python
#!/usr/bin/env python3
"""
Generate CDN-only file manifest for a HuggingFace dataset folder.
Run from Mac orchestrator after rate-limit window clears.
Usage:
  python scripts/build_hf_manifest.py \
    --repo datasets/axentx/surrogate-mirror \
    --folder batches/mirror-merged/2026-05-03 \
    --out manifests/2026-05-03_filelist.json
"""
import argparse
import json
import os
import time
from pathlib import Path

try:
    from huggingface_hub import HfApi
except ImportError:
    raise SystemExit("pip install huggingface_hub")

API = HfApi()

def build_manifest(repo_id: str, folder: str, out_path: str):
    # folder must be non-recursive listing; we'll walk subfolders client-side safely
    entries = API.list_repo_tree(repo_id=repo_id, path=folder, recursive=False)
    manifest = []

    def _crawl(items, prefix=""):
        for item in items:
            if item.type == "file":
                rel = os.path.join(prefix, item.path) if prefix else item.path
                manifest.append({
                    "repo": repo_id,
                    "path": rel,
                    "cdn_url": f"https://huggingface.co/datasets/{repo_id}/resolve/main/{rel}"
                })
            else:
                # recurse one level (non-recursive API) to avoid deep pagination
                sub = API.list_repo_tree(repo_id=repo_id, path=item.path, recursive=False)
                _crawl(sub, prefix=item.path)

    _crawl(entries)

    out = Path(out_path)
    out.parent.mkdir(parents=True, exist_ok=True)
    with open(out, "w") as f:
        json.dump(manifest, f, indent=2)

    print(f"Wrote {len(manifest)} files to {out}")

if __name__ == "__main__":
    p = argparse.ArgumentParser()
    p.add_argument("--repo", required=True, help="e.g. datasets/axentx/surrogate-mirror")
    p.add_argument("--folder", required=True, help="Folder to list (non-recursive API)")
    p.add_argument("--out", required=True, help="Output JSON path")
    args = p.parse_args()
    build_manifest(args.repo, args.folder, args.out)
```

### 2) CDN-only iterable dataset (`surrogate/data/hf_cdn_dataset.py`)
```python
#!/usr/bin/env python3
import json
import io
from typing import Dict, Any, Optional

import torch
from torch.utils.data import IterableDataset
import requests

# Optional: fast parquet/arrow handling if files are parquet
try:
    import pyarrow.parquet as pq
    import pyarrow as pa
except Exception:  # noqa
    pq = None

class HFCDNDataset(IterableDataset):
    """
    CDN-only dataset that never calls HF API during training.
    Manifest format: [{"repo": "...", "path": "...", "cdn_url": "..."}, ...]
    """
    def __init__(self, manifest_path: str, max_files: Optional[int] = None):
        super().__init__()
        with open(manifest_path) as f:
            self.files = json.load(f)
        if max_files:
            self.files = self.files[:max_files]

    def _stream_parquet_to_rows(self, url: str):
        # Stream download and decode parquet without full schema trust
        resp = requests.get(url, timeout=60)
        resp.raise_for_status()
        buf = io.BytesIO(resp.content)
        table = pq.read_table(buf, columns=["prompt", "response"], use_threads=False)
        df = table.to_pandas()
        for _, row in df.iterrows():
            yield {"prompt": str(row.get("prompt", "")), "response": str(row.get("response", ""))}

    def _stream_text_lines(self, url: str):
        resp = requests.get(url, timeout=60, stream=True)
        resp.raise_for_status()
        for line in resp.iter_lines(decode_unicode=True):
            if not line:
                continue
            line = line.strip()
            if not line:
                continue
            # naive JSON-lines fallback; adapt to your format
            try:
                import json as _json
                obj = _json.loads(line)
                prompt = obj.get("prompt") or obj.get("input") or ""
                response = obj.get("response") or obj.get("output") or ""
                yield {"prompt": str(prompt), "response": str(response)}
            except Exception:
                continue

    def __iter__(self):
        worker_info = torch.utils.data.get_worker_info()
        if worker_info is None:
            files = self.files
        else:
            per_worker = len(self.files) // worker_info.num_workers
            files = self.files[worker_info.id * per_worker : (worker_info.id + 1) * per_worker]

        for item in files:
            url = item["cdn_url"]
            try:
                if url.endswith(".parquet"):
                    yield from self._stream_parquet_to_rows(url)
                else:
                    yield from self._stream_text_lines(url)
            except Exception as exc:
                # Log and skip bad files; don't crash training
                print(f"Skipping {url}: {exc}")
                continue
```
