# Costinel / design

## Implementation Plan — Costinel Top-Hub Signal (Backend)

**Scope:** Highest-value, read-only, <2h  
**Endpoint:** `GET /api/v1/cost-anomaly/signal/top-hub`  
**Guiding principle:** “Sense + Signal — ไม่ Execute” (no side effects, no mutations).

---

### 1. Architecture (backend)
- Add **read-only** endpoint under `CostAnomalySignalController`.
- Use existing `CostAnomalyService` + new lightweight `TopHubSignalService` (or extend service).
- Response shape:
  ```json
  {
    "hub": "MOC",
    "score": 0.94,
    "anomalyCount": 12,
    "totalImpact": 4870.32,
    "currency": "USD",
    "window": "last_24h",
    "signals": [
      {
        "id": "sig-20260503-001",
        "service": "EC2",
        "region": "ap-southeast-1",
        "severity": "high",
        "deltaPct": 210.4,
        "forecastDelta": 1200.00,
        "tags": ["prod", "moc"]
      }
    ],
    "recommendations": [
      {
        "type": "ri_coverage",
        "action": "Increase RI coverage for m6i.xlarge in ap-southeast-1",
        "estimatedSavings": 1200.00,
        "currency": "USD"
      }
    ],
    "generatedAt": "2026-05-03T02:14:00Z"
  }
  ```

---

### 2. Implementation Steps (≤2h)

1. **Add DTOs** (records/POJOs) in `dto/`  
   - `TopHubSignalResponse`
   - `SignalItem`
   - `RecommendationItem`

2. **Service layer**  
   - Create `TopHubSignalService` (or add method to `CostAnomalyService`).  
   - Logic (read-only):
     - Query last 24h anomalies (existing repo).  
     - Group by hub tag (e.g., `tags` contains "moc") or inferred from service/account metadata.  
     - Pick top hub by `impact = sum(abs(delta))`.  
     - Build signals + recommendations (heuristic: if RI coverage < 60% → RI recommendation).  
   - No mutations, no external calls beyond existing cost/usage stores.

3. **Controller**  
   - Add `GET /api/v1/cost-anomaly/signal/top-hub` to `CostAnomalySignalController`.  
   - Inject `TopHubSignalService`.  
   - Return 200 + response; 204 if no data.

4. **Tests** (quick)  
   - One unit test for service (top-hub selection).  
   - One controller test (happy path).

5. **Wire & run**  
   - Ensure Spring context picks up new controller.  
   - Verify endpoint with `curl`.

#### Service (`service/TopHubSignalService.java`)
```java
package com.axentx.costinel.service;

import com.axentx.costinel.dto.*;
import com.axentx.costinel.model.CostAnomaly;
import com.axentx.costinel.repository.CostAnomalyRepository;
import org.springframework.stereotype.Service;

import java.time.Instant;
import java.time.temporal.ChronoUnit;
import java.util.*;
import java.util.stream.Collectors;

@Service
public class TopHubSignalService {

    private final CostAnomalyRepository anomalyRepository;

    public TopHubSignalService(CostAnomalyRepository anomalyRepository) {
        this.anomalyRepository = anomalyRepository;
    }

    public TopHubSignalResponse buildTopHubSignal() {
        Instant since = Instant.now().minus(24, ChronoUnit.HOURS);
        List<CostAnomaly> recent = anomalyRepository.findByDetectedAtAfter(since);

        if (recent.isEmpty()) {
            return new TopHubSignalResponse(
                null, 0.0, 0, 0.0, "USD", "last_24h",
                List.of(), List.of(), Instant.now()
            );
        }

        // Group by hub (simple: pick first tag that looks like a hub, or fallback to account/service)
        Map<String, List<CostAnomaly>> byHub = recent.stream()
            .collect(Collectors.groupingBy(this::inferHub));

        String topHub = byHub.entrySet().stream()
            .max(Comparator.comparingDouble(e -> Math.abs(e.getValue().stream()
                .mapToDouble(CostAnomaly::getDeltaAmount).sum())))
            .map(Map.Entry::getKey)
            .orElse("Unknown");

        List<CostAnomaly> topHubAnomalies = byHub.get(topHub);
        double totalImpact = Math.abs(topHubAnomalies.stream()
            .mapToDouble(CostAnomaly::getDeltaAmount).sum());
        double avgSeverityScore = topHubAnomalies.stream()
            .mapToDouble(a -> Math.min(1.0, Math.abs(a.getDeltaAmount()) / 1000.0)).average().orElse(0.0);

        List<SignalItem> signals = topHubAnomalies.stream()
            .sorted(Comparator.comparing(CostAnomaly::getDetectedAt).reversed())
            .limit(10)
            .map(a -> new SignalItem(
                a.getId(),
                a.getService(),
                a.getRegion(),
                a.getDeltaAmount() > 0 ? "high" : "medium",
                a.getDeltaAmount() != 0 ? (a.getDeltaAmount() / Math.max(1, a.getBaselineAmount())) * 100.0 : 0.0,
                a.getForecastDelta(),
                a.getTags() != null ? a.getTags() : List.of()
            )).collect(Collectors.toList());

        List<RecommendationItem> recommendations = new ArrayList<>();
        if (totalImpact > 1000) {
            recommendations.add(new RecommendationItem(
                "ri_coverage",
                "Increase RI coverage for top-cost instances in " + topHub,
                totalImpact * 0.3,
                "USD"
            ));
        }

        return new TopHubSignalResponse(
            topHub,
            Math.min(1.0, avgSeverityScore),
            topHubAnomalies.size(),
            totalImpact,
            "USD",
            "last_24h",
            signals,
            recommendations,
            Instant.now()
        );
    }

    private String inferHub(CostAnomaly a) {
        if (a.getTags() != null) {
            for (String t : a.getTags()) {
                if (t.toLowerCase().contains("moc") || t.toLowerCase().contains("hub")) {
                    return t.toUpperCase();
                }
            }
        }
        return a.getAccount() != null ? a.getAccount() : a.getService();
    }
}
```
