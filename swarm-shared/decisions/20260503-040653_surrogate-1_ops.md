# surrogate-1 / ops

## Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads.

### Changes
1. Replace `bin/dataset-enrich.sh` with `bin/dataset-enrich.py` (manifest-driven, CDN-bypass).
2. Add `bin/gen-manifest.py` (run once from Mac) to pre-list date folder and emit `manifest.json`.
3. Update `.github/workflows/ingest.yml` to pass `SHARD_ID`/`SHARD_TOTAL` and optional `DATE_FOLDER`/`MANIFEST_URL`.
4. Keep `lib/dedup.py` unchanged (central md5 store).

### Why this matters
- Single Mac-side API call (after rate-limit window) → `list_repo_tree` for one date folder → `manifest.json`.
- Lightning training workers use **CDN-only fetches** (`https://huggingface.co/datasets/.../resolve/main/...`) with **zero API calls** during data load.
- Worker deterministically hashes `slug` → picks shard, projects to `{prompt,response}`, dedups via central store, writes `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.

## 1) New: `bin/gen-manifest.py` (run on Mac)

```python
#!/usr/bin/env python3
"""
Generate manifest for a date folder.
Usage:
  HF_TOKEN=... python bin/gen-manifest.py \
    --repo axentx/surrogate-1-training-pairs \
    --date-folder 2026-05-03 \
    --out manifest.json
"""
import argparse
import json
import os
import sys
from datetime import datetime, timezone

from huggingface_hub import HfApi

CDN_BASE = "https://huggingface.co/datasets"

def build_manifest(repo: str, date_folder: str, out_path: str) -> None:
    api = HfApi(token=os.getenv("HF_TOKEN"))
    # Single API call: non-recursive tree for the date folder
    tree = api.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    files = [f.rfilename for f in tree if f.type == "file"]

    entries = []
    for f in files:
        # Expect files like 2026-05-03/some-slug.parquet
        slug = os.path.splitext(os.path.basename(f))[0]
        entries.append(
            {
                "slug": slug,
                "path": f,
                "cdn_url": f"{CDN_BASE}/{repo}/resolve/main/{f}",
            }
        )

    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "entries": entries,
    }

    with open(out_path, "w", encoding="utf-8") as fp:
        json.dump(manifest, fp, indent=2)

    print(f"Wrote {len(entries)} entries to {out_path}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Generate CDN manifest for date folder.")
    parser.add_argument("--repo", default="axentx/surrogate-1-training-pairs")
    parser.add_argument("--date-folder", default=datetime.utcnow().strftime("%Y-%m-%d"))
    parser.add_argument("--out", default="manifest.json")
    args = parser.parse_args()
    try:
        build_manifest(args.repo, args.date_folder, args.out)
    except Exception as e:
        print(f"Error: {e}", file=sys.stderr)
        sys.exit(1)
```

Make executable:

def shard_for(slug: str, total: int) -> int:
    # Deterministic, stable across runs
    digest = hashlib.sha256(slug.encode("utf-8")).hexdigest()
    return int(digest, 16) % total

def load_manifest(manifest_path: str) -> Dict[str, Any]:
    if manifest_path.startswith("http://") or manifest_path.startswith("https://"):

        resp.raise_for_status()
        return resp.json()
    with open(manifest_path, encoding="utf-8") as f:

def download_parquet_to_temp(cdn_url: str) -> str:
    # Stream to temp file to avoid memory spikes
    resp = requests.get(cdn_url, stream=True, timeout=120)
    resp.raise_for_status()
    fd, path = tempfile.mkstemp(suffix=".parquet")
    try:
        with os.fdopen(fd, "wb") as out:
            for chunk in resp.iter_content(chunk_size=8192):
                out.write(chunk)
        return path
    except Exception:
        os.unlink(path)
        raise

def project_to_pair(batch) -> List[Dict[str, str]]:
    """
    Convert pyarrow table rows to {prompt, response}.
    Handles common schema variants by field-name matching.
    """
    pairs = []
    cols = set(batch.column_names)

    prompt_col = None
    response_col = None

    for c in cols:
        cl = c.lower()
        if cl in ("prompt", "instruction", "question", "input"):
            prompt_col = c
        if cl in ("response", "completion", "answer", "output"):
            response_col = c

    # Fallback: if only one text-like column, treat as prompt and leave response empty
    if prompt_col is None and len(cols) == 1:
        prompt_col = next(iter(cols))

    for row in batch.to_pylist():
        prompt = (row[prompt_col] or "").strip() if prompt_col and row.get(prompt_col) else ""
        response = (row[response_col] or "").strip() if response_col and row.get(response_col) else ""
        if prompt or response:
            pairs.append({"prompt": prompt, "response": response})
    return pairs
