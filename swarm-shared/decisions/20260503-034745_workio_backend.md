# workio / backend

## Final Answer

### 1. Diagnosis (merged)
- **Idempotency gap**: LINE retries on 5xx/timeouts can replay clock-in/out, leave, and OT events, corrupting state (duplicate punches, double approvals).
- **Missing signature verification**: absence of `X-Line-Signature` validation permits spoofed webhooks to mutate attendance/leave/OT state.
- **Observability gaps**: raw payloads, verification outcomes, and processing results are not logged; retries and spoofing are invisible.
- **Stale frontend state**: webhook-driven state changes (clock/leave/OT) are invisible to clients until hard refresh, causing divergence across tabs/devices.
- **No real-time subscription layer**: UI relies on local React state and manual polling/refresh instead of push-based updates.

### 2. Proposed change (merged)
Implement an **idempotent, signature-verified LINE webhook handler** with structured observability and a **tenant-scoped SSE channel** for real-time frontend synchronization.

Scope:
- Backend: `/opt/axentx/workio/server/src/routes/webhook/line.ts` — handler, signature verification, idempotency guard, structured logs.
- Backend: `/opt/axentx/workio/server/src/services/lineWebhookService.ts` — business logic + idempotency store (PostgreSQL).
- Backend: `/opt/axentx/workio/server/src/routes/events/notifications.ts` — SSE endpoint `/events/notifications` for tenant-scoped real-time updates.
- Backend: `/opt/axentx/workio/server/src/lib/events.ts` — typed EventEmitter + `emitTenantEvent` for SSE broadcast.
- Frontend: `/opt/axentx/workio/src/lib/events.ts` — lightweight EventSource wrapper to subscribe to tenant notifications and update React state.
- Frontend: integrate into attendance/leave/OT pages to replace manual refresh with live updates.

#### Backend: line webhook service (idempotency + verification)
```ts
// server/src/services/lineWebhookService.ts
import crypto from 'crypto';
import { db } from '../db';
import { logger } from '../lib/logger';
import { emitTenantEvent } from '../lib/events';

const CHANNEL_SECRET = process.env.LINE_CHANNEL_SECRET || '';

export function verifyLineSignature(rawBody: string, signature: string): boolean {
  if (!CHANNEL_SECRET) return false;
  const expected = crypto
    .createHmac('sha256', CHANNEL_SECRET)
    .update(rawBody)
    .digest('base64');
  return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
}

export async function processLineWebhook(rawBody: string, signature: string, requestId?: string) {
  const logMeta = { requestId, source: 'line-webhook' };

  // 1) Observability: log raw inbound
  logger.info({ ...logMeta, rawBody }, 'LINE webhook received');

  // 2) Signature verification
  if (!verifyLineSignature(rawBody, signature)) {
    logger.warn({ ...logMeta, signature }, 'LINE webhook signature invalid');
    return { ok: false, reason: 'invalid-signature' };
  }

  let events;
  try {
    events = JSON.parse(rawBody).events || [];
  } catch (err) {
    logger.error({ ...logMeta, err }, 'Failed to parse LINE webhook JSON');
    return { ok: false, reason: 'invalid-json' };
  }

  // 3) Idempotency guard per event (dedupe by event.id within tenant)
  for (const ev of events) {
    const { id: eventId, type, source, timestamp } = ev || {};
    const userId = source?.userId;
    if (!eventId || !userId) continue;

    const user = await db.user.findFirst({ where: { lineUserId: userId } });
    const tenantId = user?.tenantId || null;

    const exists = await db.idempotency.findUnique({
      where: { tenant_eventId: { tenantId, eventId } },
    });
    if (exists) {
      logger.info({ ...logMeta, eventId, userId }, 'Duplicate LINE event skipped (idempotent)');
      continue;
    }

    // Record idempotency entry before processing (prevents replay races)
    await db.idempotency.create({
      data: {
        tenantId,
        eventId,
        payload: ev,
        processedAt: new Date(),
      },
    });

    await handleLineEvent(ev, tenantId, userId, logMeta);
  }

  logger.info({ ...logMeta, count: events.length }, 'LINE webhook processed');
  return { ok: true };
}

async function handleLineEvent(event: any, tenantId: string | null, userId: string, logMeta: any) {
  try {
    const { type, message, postback } = event;
    if (type === 'message' && message?.text) {
      const text = message.text.trim().toLowerCase();
      if (text === 'clock in' || text === 'clock out') {
        await processClockAction(tenantId, userId, text === 'clock in' ? 'in' : 'out', logMeta);
      }
    } else if (type === 'postback') {
      // handle leave/OT approvals etc.
      const data = postback?.data || '';
      // parse and route to approval service
    }
  } catch (err) {
    logger.error({ ...logMeta, err, eventId: event.id }, 'Failed to handle LINE event');
  }
}

async function processClockAction(tenantId: string | null, userId: string, action: 'in' | 'out', logMeta: any) {
  const record = await db.attendance.create({
    data: {
      tenantId,
      userId,
      action,
      timestamp: new Date(),
    },
  });

  emitTenantEvent(tenantId, {
    type: 'attendance.updated',
    payload: { record },
  });

  logger.info({ ...logMeta, tenantId, userId, action, recordId: record.id }, 'Attendance recorded');
}
```

#### Backend: webhook route
```ts
// server/src/routes/webhook/line.ts
import express from 'express';
import { processLineWebhook } from '../../services/lineWebhookService';
import { logger } from '../../lib/logger';

const router = express.Router();

router.post('/', express.raw({ type: 'application/json' }), async (req, res) => {
  const signature = req.headers['x-line-signature'] as string;
  const rawBody = req.body.toString('utf8');
  const requestId = crypto.randomUUID();

  const result = await processLineWebhook(rawBody, signature, requestId);

  if (!result.ok) {
    logger.warn({ requestId, reason: result.reason }, 'LINE webhook rejected');
    return res.status(400).json({ error: result.reason });
  }

  res.status(200).json({ ok: true });
});

export default router;
```
