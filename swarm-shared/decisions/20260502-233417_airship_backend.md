# airship / backend

## Final Synthesis (Best Parts + Correctness + Actionability)

**Goal (unchanged):** Harden `airship discover` into a deterministic, CDN-only orchestrator that eliminates HF API rate limits and PyArrow schema errors during Surrogate-1 dataset ingestion.

**Why this ships high value in ≤2h**
- Removes 429/rate-limit flakiness that blocks training pipelines.
- Eliminates PyArrow `CastError` from mixed-schema repos by never using `load_dataset` or `load_dataset(streaming=True)`.
- Produces a reproducible file manifest Lightning training can consume with **zero API calls** during data load.
- Uses only `hf_hub_download` + CDN URLs → bypasses auth/rate limits entirely.
- Fits in a single orchestrator script + small manifest schema; no infra changes.

---

## Implementation Plan (concrete, merged)

1. **Create `airship/discover.py`** (or extend existing CLI)  
   - Accept: `repo_id`, `date_folder` (e.g. `2026-04-29`), `out_manifest`
   - Single `list_repo_tree(recursive=False)` call for the folder → write `files.json`
   - For each file: `hf_hub_download(repo_id, subfolder/file, local_dir=cache)`
   - Emit manifest rows: `{local_path, repo_file, size, sha256, url_cdn}`

2. **Schema-safe projection helper**  
   - Do **not** load datasets; parse only when needed (e.g. parquet → `{prompt,response}`)
   - Keep attribution in filename pattern: `batches/mirror-merged/{date}/{slug}.parquet`
   - No `source` or `ts` columns in projection

3. **CDN-only training contract**  
   - Manifest includes CDN URLs: `https://huggingface.co/datasets/{repo}/resolve/main/{path}`
   - Lightning `train.py` reads manifest and streams via CDN (no HF API)

4. **Idempotent + cache-friendly**  
   - Skip download if local file exists and size/sha matches manifest
   - Deterministic ordering by filename for reproducible epochs

5. **CLI + cron safety**  
   - Shebang `#!/usr/bin/env bash` if wrapper is bash
   - Ensure executable bit; invoke via `bash` in cron with `SHELL=/bin/bash`

### `airship/discover.py`
```python
#!/usr/bin/env python3
"""
airship discover
Deterministic CDN-only dataset manifest generator for Surrogate-1 training.
Usage:
  python discover.py --repo HuggingFaceTB/cosmopedia --folder 2026-04-29 --out manifest.json
"""
import argparse
import hashlib
import json
import os
import sys
from pathlib import Path
from typing import List, Dict

from huggingface_hub import list_repo_tree, hf_hub_download

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"


def sha256_file(path: Path) -> str:
    h = hashlib.sha256()
    with open(path, "rb") as f:
        for chunk in iter(lambda: f.read(8192), b""):
            h.update(chunk)
    return h.hexdigest()


def build_manifest(repo_id: str, folder: str, out_path: Path, cache_dir: Path) -> List[Dict]:
    cache_dir.mkdir(parents=True, exist_ok=True)

    # Single API call: list only top-level of folder (non-recursive)
    tree = list_repo_tree(repo_id=repo_id, path=folder, recursive=False)
    files = [entry for entry in tree if entry.type == "file"]

    manifest = []
    for f in sorted(files, key=lambda x: x.path):
        rel_path = f"{folder}/{f.path}"
        local_file = cache_dir / f"{f.path}"

        # Download via hf_hub_download (uses CDN under the hood)
        downloaded = hf_hub_download(
            repo_id=repo_id,
            filename=rel_path,
            local_dir=cache_dir,
            local_dir_use_symlinks=False,
        )
        local_path = Path(downloaded)

        # Validate presence
        if not local_path.is_file():
            print(f"ERROR: failed to download {rel_path}", file=sys.stderr)
            continue

        entry = {
            "repo_file": rel_path,
            "local_path": str(local_path.resolve()),
            "size": local_path.stat().st_size,
            "sha256": sha256_file(local_path),
            "url_cdn": CDN_TEMPLATE.format(repo=repo_id, path=rel_path),
        }
        manifest.append(entry)
        print(json.dumps(entry))

    out_path.write_text(json.dumps(manifest, indent=2))
    print(f"Manifest written to {out_path} ({len(manifest)} files)", file=sys.stderr)
    return manifest


def main() -> None:
    parser = argparse.ArgumentParser(description="CDN-only dataset manifest generator")
    parser.add_argument("--repo", required=True, help="HF dataset repo id")
    parser.add_argument("--folder", required=True, help="Folder within repo (e.g. 2026-04-29)")
    parser.add_argument("--out", required=True, help="Output manifest JSON path")
    parser.add_argument("--cache", default=".cache/hf", help="Local cache directory")
    args = parser.parse_args()

    try:
        build_manifest(
            repo_id=args.repo,
            folder=args.folder,
            out_path=Path(args.out),
            cache_dir=Path(args.cache),
        )
    except Exception as e:
        print(f"FATAL: {e}", file=sys.stderr)
        sys.exit(1)


if __name__ == "__main__":
    main()
```

### Lightning `train.py` snippet (CDN-only loader, corrected)
```python
import json
import pyarrow.parquet as pq
from torch.utils.data import IterableDataset
import requests
from io import BytesIO

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path):
        with open(manifest_path) as f:
            self.manifest = json.load(f)

    def __iter__(self):
        for entry in self.manifest:
            # Stream from CDN (no HF API)
            url = entry["url_cdn"]
            resp = requests.get(url, timeout=60)
            resp.raise_for_status()
            # Use BytesIO to avoid writing temp files; read via pyarrow
            table = pq.read_table(BytesIO(resp.content))
            # Project only {prompt, response} — ignore mixed schema extras
            for batch in table.to_batches(columns=["prompt", "response"]):
                prompts = batch["prompt"].to_pylist()
                responses = batch["response"].to_pylist()
                for prompt, response in zip(prompts, responses):
                    yield {"prompt": prompt, "response": response}
```
