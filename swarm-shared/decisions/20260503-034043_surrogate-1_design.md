# surrogate-1 / design

## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16` (matrix) and optional `DATE_FOLDER` (defaults to today `YYYY-MM-DD`)
- Uses **manifest-first strategy**: single HF API `list_repo_tree` (non-recursive) for the date folder → saves `manifest.json` → Lightning training uses CDN-only fetches with zero API calls during data load
- Downloads via HF CDN (`https://huggingface.co/datasets/.../resolve/main/...`) — no Authorization header, bypasses `/api/` rate limits
- Projects heterogeneous files to `{prompt, response}` only at parse time (avoids pyarrow CastError from mixed schemas)
- Deterministic shard assignment: `hash(slug) % SHARD_TOTAL`; each worker processes only its shard
- Central dedup via existing `lib/dedup.py` (md5 store)
- Outputs: `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl`
- Retry/backoff on CDN 429 (CDN tier has separate limits) and HF API 429 (wait 360s)
- Idempotent: re-running same date+shard skips already-processed files via dedup

def hf_api_get(path: str, params: Dict[str, Any] = None, retries: int = 3) -> Any:
    """Call HF API with 429 backoff (360s)."""
    url = f"https://huggingface.co/api/{path}"
    headers = {"Authorization": f"Bearer {HF_TOKEN}"}
    for attempt in range(1, retries + 1):
        try:
            resp = requests.get(url, headers=headers, params=params, timeout=30)
            if resp.status_code == 429:
                wait = 360
                log.warning("HF API 429 — waiting %ss (attempt %s/%s)", wait, attempt, retries)
                time.sleep(wait)
                continue
            resp.raise_for_status()
            return resp.json()
        except Exception as exc:
            if attempt == retries:
                log.error("HF API request failed: %s", exc)
                raise
            time.sleep(2**attempt)
    raise RuntimeError("Exhausted retries for HF API")

def list_date_files(date_folder: str) -> List[str]:
    """List top-level files in date folder (non-recursive)."""
    tree = hf_api_get(f"datasets/{DATASET_REPO}/tree", params={"path": date_folder, "recursive": False})
    files = [item["path"] for item in tree if item.get("type") == "file"]
    log.info("Found %d files in %s", len(files), date_folder)
    return files

def deterministic_shard(path: str) -> int:
    return int(hashlib.sha256(path.encode()).hexdigest(), 16) % SHARD_TOTAL

def download_cdn(path: str, dest: Path, max_retries: int = 5) -> Path:
    """Download via HF CDN (no auth header)."""
    url = f"{CDN_BASE}/{path}"
    for attempt in range(1, max_retries + 1):
        try:
            resp = requests.get(url, timeout=60, stream=True)
            if resp.status_code == 429:
                wait = 60 * attempt
                log.warning("CDN 429 — waiting %ss", wait)
                time.sleep(wait)
                continue
            resp.raise_for_status()
            with open(dest, "wb") as f:
                for chunk in resp.iter_content(chunk_size=8192):
                    f.write(chunk)
            return dest
        except Exception as exc:
            if attempt == max_retries:
                log.error("CDN download failed for %s: %s", path, exc)
                raise
            time.sleep(2**attempt)
    raise RuntimeError("CDN download exhausted retries")
