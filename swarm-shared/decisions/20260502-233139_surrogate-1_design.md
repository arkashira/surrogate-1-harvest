# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value change**: Replace runtime `load_dataset(streaming=True)` + recursive `list_repo_tree` in `bin/dataset-enrich.sh` with a **deterministic pre-flight snapshot + CDN-only fetches**. This eliminates HF API rate limits (429), prevents schema-cast errors from heterogeneous repos, and reduces per-run API calls to exactly one (snapshot) while training/data load uses zero-auth CDN URLs.

### Steps (all in `/opt/axentx/surrogate-1`)

1. Add `bin/snapshot-paths.py` — one-shot Mac/CI script that calls `list_repo_tree` per date folder and writes `file-list.json`.
2. Update `bin/dataset-enrich.sh` to:
   - Accept an optional `FILE_LIST` path (default: use snapshot).
   - Filter deterministic shard slice by `md5(slug) % 16 == SHARD_ID`.
   - Download via CDN (`resolve/main/...`) with `curl`/`wget` (no auth).
   - Project to `{prompt,response}` only at parse time; write `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.
3. Make scripts executable and ensure Bash shebang + `SHELL=/bin/bash` for cron/CI.
4. Add lightweight retry/backoff for CDN downloads and 360s sleep on 429 (CDN rarely 429, but safe).

### 1) Snapshot script (run from Mac/CI before workflow)

`bin/snapshot-paths.py`
```python
#!/usr/bin/env python3
"""
Usage:
  python bin/snapshot-paths.py --repo axentx/surrogate-1-training-pairs \
    --out file-list.json [--date-folder 2026-05-02]

Produces:
  {
    "date": "2026-05-02",
    "repo": "axentx/surrogate-1-training-pairs",
    "paths": [
      "batches/public-raw/2026-05-02/abc123.parquet",
      ...
    ],
    "generated_at": "2026-05-02T23:45:00Z"
  }
"""
import argparse
import json
import os
import sys
import time
from datetime import datetime, timezone

from huggingface_hub import HfApi

def main() -> None:
    parser = argparse.ArgumentParser(description="Snapshot repo tree for CDN-only ingestion.")
    parser.add_argument("--repo", default="axentx/surrogate-1-training-pairs")
    parser.add_argument("--out", required=True, help="Output JSON path")
    parser.add_argument("--date-folder", help="Optional date folder (e.g. 2026-05-02)")
    parser.add_argument("--retry", type=int, default=3, help="Retries for API calls")
    args = parser.parse_args()

    api = HfApi()
    paths = []

    # If date-folder provided, snapshot only that folder
    folders = [f"batches/public-raw/{args.date_folder}"] if args.date_folder else [
        # heuristic: list top-level dates by scanning one level
        # fallback: list_repo_tree root and pick date folders
    ]

    # If no date-folder, list top-level date folders non-recursively
    if not args.date_folder:
        try:
            entries = api.list_repo_tree(repo_id=args.repo, path="batches/public-raw", recursive=False)
            date_folders = [e.path for e in entries if e.type == "directory"]
        except Exception as e:
            print(f"Failed to list date folders: {e}", file=sys.stderr)
            sys.exit(1)
    else:
        date_folders = [f"batches/public-raw/{args.date_folder}"]

    for folder in date_folders:
        for attempt in range(1, args.retry + 1):
            try:
                entries = api.list_repo_tree(repo_id=args.repo, path=folder, recursive=False)
                for e in entries:
                    if e.type == "file":
                        paths.append(e.path)
                break
            except Exception as ex:
                if attempt == args.retry:
                    print(f"Failed to list {folder}: {ex}", file=sys.stderr)
                    sys.exit(1)
                wait = 30 * attempt
                print(f"Retry {attempt}/{args.retry} for {folder} in {wait}s...", file=sys.stderr)
                time.sleep(wait)

    snapshot = {
        "repo": args.repo,
        "date": args.date_folder or datetime.now(timezone.utc).strftime("%Y-%m-%d"),
        "paths": sorted(paths),
        "generated_at": datetime.now(timezone.utc).isoformat(),
    }

    os.makedirs(os.path.dirname(os.path.abspath(args.out)), exist_ok=True)
    with open(args.out, "w", encoding="utf-8") as f:
        json.dump(snapshot, f, indent=2)
    print(f"Snapshot written to {args.out} ({len(paths)} files)")

if __name__ == "__main__":
    main()
```

Make executable:
```bash
chmod +x bin/snapshot-paths.py
```
