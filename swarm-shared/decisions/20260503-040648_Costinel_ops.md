# Costinel / ops

**Final consolidated implementation plan (CDN-first Top-Hub Signal Panel)**

**Scope**  
Add a lightweight, non-blocking Top-Hub Signal Panel to Costinel that surfaces the most-connected hub (e.g., `"MOC"`) using CDN-first data. Zero runtime Hugging Face API calls from the frontend. Deterministic, cache-friendly, deployable in <2h. Fits *Sense + Signal — ไม่ Execute*.

---

### Architecture (CDN-first, zero HF API at runtime)
- **Orchestration** (run on Mac/CI, not in app):  
  - Use `list_repo_tree` on `axentx/knowledge-rag` (date folder or `hubs/`) to identify the most-connected hub (degree/references) or use deterministic fallback (`"MOC"`).  
  - Emit a tiny JSON payload and push **one commit** to the repo (or write to `public/data/` if self-hosted).  
  - Expose via HuggingFace CDN (`resolve/main/...`) or local static path. No auth required; CDN limits are much higher.
- **Frontend**:  
  - Fetch the CDN/static JSON with cache-first and short timeout.  
  - Non-blocking: render graceful placeholder on failure/slow fetch.  
  - Display a compact pill/badge with hub name and optional link to hub doc.
- **No runtime HF API calls from the app** — avoids 429 and quota issues.

### Implementation

#### 1) Orchestration (choose one)

**Bash (preferred for minimal deps, cron-friendly)**  
`scripts/update-top-hub.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

# Config
REPO="axentx/knowledge-rag"
DATE_DIR="hubs/$(date +%Y-%m-%d)"
OUT_FILE="top-hub.json"
HUB_NAME="MOC"  # Deterministic fallback; replace with computed if needed
LABEL="Most-Connected Hub"
UPDATED="$(date -u +"%Y-%m-%dT%H:%M:%SZ")"
CDN_URL="https://huggingface.co/datasets/${REPO}/resolve/main/${DATE_DIR}/${HUB_NAME}.json"

# Build payload
cat > "${OUT_FILE}" <<EOF
{
  "hub": "${HUB_NAME}",
  "label": "${LABEL}",
  "updated": "${UPDATED}",
  "url": "${CDN_URL}"
}
EOF

# Copy to repo path (assumes repo cloned at ../knowledge-rag)
TARGET_DIR="../knowledge-rag/${DATE_DIR}"
mkdir -p "${TARGET_DIR}"
cp "${OUT_FILE}" "${TARGET_DIR}/top-hub.json"

# Commit and push (one write to avoid HF cap)
cd ../knowledge-rag
git add "${TARGET_DIR}/top-hub.json"
git commit -m "chore: update top-hub signal for ${DATE_DIR}" || true
git push origin main

echo "Top-hub updated: ${HUB_NAME}"
```
Make executable: `chmod +x scripts/update-top-hub.sh`  
Cron note: ensure `SHELL=/bin/bash` if used in crontab.

**Node (alternative, if JS tooling preferred)**  
`scripts/update-top-hub.js`
```js
#!/usr/bin/env node
import { writeFileSync, mkdirSync, existsSync } from 'fs';
import { execSync } from 'child_process';
import fetch from 'node-fetch';

const REPO = 'axentx/knowledge-rag';
const DATE_DIR = `hubs/${new Date().toISOString().slice(0,10)}`;
const HUB_NAME = 'MOC'; // deterministic fallback
const LABEL = 'Most-Connected Hub';
const UPDATED = new Date().toISOString();
const CDN_URL = `https://huggingface.co/datasets/${REPO}/resolve/main/${DATE_DIR}/${HUB_NAME}.json`;

const payload = {
  hub: HUB_NAME,
  label: LABEL,
  updated: UPDATED,
  url: CDN_URL
};

const targetDir = `../knowledge-rag/${DATE_DIR}`;
if (!existsSync(targetDir)) mkdirSync(targetDir, { recursive: true });
writeFileSync(`${targetDir}/top-hub.json`, JSON.stringify(payload, null, 2));

try {
  execSync(`git -C ../knowledge-rag add "${targetDir}/top-hub.json"`);
  execSync(`git -C ../knowledge-rag commit -m "chore: update top-hub signal for ${DATE_DIR}" || true`);
  execSync('git -C ../knowledge-rag push origin main');
} catch (e) {
  console.warn('Git push failed (may be CI-only)', e.message);
}

console.log('Top-hub updated:', HUB_NAME);
```
Make executable: `chmod +x scripts/update-top-hub.js`

#### 2) Frontend component (React/Next.js)

`components/TopHubSignalPanel.tsx`
```tsx
'use client';

import { useEffect, useState } from 'react';

interface TopHubData {
  hub: string;
  label?: string;
  updated?: string;
  url?: string;
}

export default function TopHubSignalPanel() {
  const [hub, setHub] = useState<TopHubData | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Prefer local static file; fallback to HF CDN
    const LOCAL_URL = '/data/top-hub-latest.json';
    const CDN_URL =
      'https://huggingface.co/datasets/axentx/knowledge-rag/resolve/main/hubs/2026-05-03/top-hub.json';
    const URL = LOCAL_URL; // switch to CDN_URL if not self-hosting

    const controller = new AbortController();
    const timeout = setTimeout(() => controller.abort(), 4000);

    fetch(URL, { signal: controller.signal, cache: 'force-cache' })
      .then((res) => {
        if (!res.ok) throw new Error('Failed to fetch top-hub');
        return res.json();
      })
      .then((data) => setHub(data))
      .catch(() => setHub(null))
      .finally(() => setLoading(false));

    return () => clearTimeout(timeout);
  }, []);

  if (loading) {
    return (
      <div className="inline-flex items-center gap-1 rounded-full bg-gray-100 px-3 py-1 text-xs text-gray-400">
        <span className="h-1.5 w-1.5 rounded-full bg-gray-300 animate-pulse" />
        Loading hub...
      </div>
    );
  }

  if (!hub) return null;

  const href = hub.url || `https://huggingface.co/datasets/axentx/knowledge-rag/tree/main/hubs`;

  return (
    <a
      href={href}
      target="_blank"
      rel="noopener noreferrer"
      className="inline-flex items-center gap-1.5 rounded-full bg-amber-50 border border-amber-200 px-3 py-1 text-xs font-medium text-amber-800 hover:bg-amber-100 transition"
      title={`Top hub — updated ${hub.updated ? new Date(hub.updated).toLocaleDateString() : '—'}`}
    >
      <span className="h-2 w-2 rounded-full bg-amber-500" />
      {hub.hub}
      <span className="sr-only"> — {hub.label || 'Most-connected hub'}</span>
    </a>
  );
}
```
