### 1) Core Goal (non-negotiable)
- **Eliminate HF API 429s** during Surrogate training by using **CDN-only** parquet fetches.

### 2) Concrete Implementation Plan

#### Step 1 — Generate and embed CDN manifest (one-time, run from dev machine)
- **Command** (run once per date folder):
  ```bash
  python scripts/gen_cdn_manifest.py --date 2026-05-03 --repo axentx/surrogate-data --out surrogate/training/cdn-manifest-2026-05-03.json
  ```
- **What it does**:
  - Calls `list_repo_tree(path="batches/mirror-merged/2026-05-03", recursive=False)` **once**.
  - Saves minimal JSON: `[{"path": "...", "size": 12345, "etag": "..."}]`.
  - Commits JSON to repo so training is **fully self-contained** (no API, no auth).

- **Correctness guard**:
  - If manifest missing at runtime → **fail fast** with clear instructions to regenerate.
  - Do **not** fall back to `load_dataset` or any HF API call (prevents 429s).

#### Step 2 — CDN-only `IterableDataset`
- **Key choices**:
  - Use `pyarrow` + `pd.read_parquet(url, columns=["prompt","response"])` for streaming.
  - Build URLs as:  
    `f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"`
  - No `datasets` library, no `list_repo_files`, no headers.

- **Resilience**:
  - Per-file retry with exponential backoff (3 attempts).
  - Skip corrupted/unreadable files and log (do not crash entire epoch).

- **Memory/IO**:
  - Stream row-by-row from each parquet file (do not load full file into memory).
  - Use `IterableDataset` + `DataLoader` with `num_workers=0` for simplicity in Studio.

#### Step 3 — Lightning idle-resilience launcher
- **Correct behavior** (resolves contradictions from candidates):
  1. **Reuse running studio** if present (name match + status `Running`).
  2. If `Stopped` → **restart on same machine type** (preserve config).
  3. If not found → **create new** with explicit machine and dependencies.
  4. **Never assume studio is alive** — always `refresh()` before use.
  5. Pass manifest path + training args via `run.arguments`.

- **Machine mapping** (explicit, no ambiguity):

### 3) Final Code (single coherent version)

#### `scripts/gen_cdn_manifest.py`
```python
#!/usr/bin/env python3
import argparse
import json
from pathlib import Path
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--date", required=True)
    parser.add_argument("--repo", default="axentx/surrogate-data")
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    api = HfApi()
    path = f"batches/mirror-merged/{args.date}"
    files = api.list_repo_tree(repo_id=args.repo, path=path, recursive=False)

    manifest = [
        {"path": f.path, "size": f.size, "etag": getattr(f, "oid", f.path)}
        for f in files if f.type == "file"
    ]

    out_path = Path(args.out)
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    print(f"Saved {len(manifest)} files to {out_path}")

if __name__ == "__main__":
    main()
```

#### `surrogate/lightning_launcher.py`
```python
#!/usr/bin/env python3
import argparse
import time
from pathlib import Path

from lightning_sdk import Machine, Teamspace, Studio

MANIFEST_DIR = Path(__file__).parent / "training"
SCRIPT_PATH = Path(__file__).parent / "train.py"

MACHINE_MAP = {
    "L40S": Machine.L40S,
    "L40": Machine.L40,
    "A100": Machine.A100,
    "H200": Machine.H200,
}

def get_or_create_studio(name: str, machine: Machine) -> Studio:
    teamspace = Teamspace()
    for s in teamspace.studios:
        if s.name == name and s.status == "Running":
            print(f"Reusing running studio: {s.name} ({s.id})")
            return s
    print(f"Creating studio: {name} on {machine}")
    return Studio(
        name=name,
        script=str(SCRIPT_PATH),
        framework="pytorch",
        target=machine,
        dependencies=["pyarrow", "pandas", "torch", "tqdm"],
    ).create()

def ensure_running(studio: Studio, machine: Machine, timeout: int = 60) -> Studio:
    studio.refresh()
    if studio.status == "Stopped":
        print(f"Studio stopped. Restarting on {machine}...")
        studio.target.start(machine=machine)

    # Wait until Running
    for _ in range(timeout // 5):
        studio.refresh()
        if studio.status == "Running":
            return studio
        time.sleep(5)

    raise RuntimeError(f"Studio failed to start within {timeout}s. Status: {studio.status}")

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--date", required=True)
    parser.add_argument("--machine", default="L40S", choices=list(MACHINE_MAP))
    args = parser.parse_args()

    machine = MACHINE_MAP[args.machine]
    studio_name = f"surrogate-train-{args.date}"
    studio = get_or_create_studio(studio_name, machine)
    studio = ensure_running(studio, machine)

    manifest_path = MANIFEST_DIR / f"cdn-manifest-{args.date}.json"
    if not manifest_path.exists():
        raise FileNotFoundError(
            f"Manifest missing: {manifest_path}. Run scripts/gen_cdn_manifest.py first."
        )

    run = studio.run(
        arguments=[
            "--manifest", str(manifest_path),
            "--epochs", "1",
            "--batch-size", "8",
            "--lr", "1e-4",
        ],
        wait=False,
    )
    print(f"Started run: {run.id}")
    print(f"Logs: {run.url}")

if __name__ == "__main__":
    main()
```
