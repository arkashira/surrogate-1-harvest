## 1) Pre-list file manifest (run once on Mac)

**Why this version:**  
- Uses `huggingface_hub` (more reliable than raw HTTP) and writes a clean manifest.  
- Manifest is committed/embedded so training never needs HF API.  
- Path is under `/opt/axentx/airship/surrogate/configs/` to match scope.

```bash
# /opt/axentx/airship/surrogate/scripts/prebuild_manifest.sh
#!/usr/bin/env bash
set -euo pipefail

REPO="axentx/surrogate-mirror"
DATE="${1:-$(date +%Y-%m-%d)}"
OUT="/opt/axentx/airship/surrogate/configs/file_manifest_${DATE}.json"

python3 - "$REPO" "$DATE" "$OUT" <<'PY'
import json, sys, os
from huggingface_hub import list_repo_tree

repo, date_folder, out_path = sys.argv[1], sys.argv[2], sys.argv[3]
tree = list_repo_tree(repo, path=date_folder, recursive=False)
files = sorted(f.rfilename for f in tree if f.rfilename.endswith(".parquet"))

os.makedirs(os.path.dirname(out_path), exist_ok=True)
with open(out_path, "w") as f:
    json.dump({"repo": repo, "date": date_folder, "files": files}, f, indent=2)
print(f"Manifest: {len(files)} files -> {out_path}")
PY
```

**Action:**  
Run after the daily mirror lands (or during off-peak) and commit the manifest into the training package.

## 2) CDN-only IterableDataset (memory-bounded)

**Why this version:**  
- Uses `IterableDataset` to bound memory (critical for large mirrors).  
- Projects only `prompt`/`response`; ignores heterogeneous schemas safely.  
- No `datasets` dependency during training (faster, fewer failure modes).

```python
# /opt/axentx/airship/surrogate/data/cdn_dataset.py
import io
import json
from typing import Dict, Iterable, Iterator

import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset

HF_CDN = "https://huggingface.co/datasets"

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path: str, start: int = 0, end: int | None = None):
        with open(manifest_path) as f:
            cfg = json.load(f)
        self.repo = cfg["repo"]
        self.date = cfg["date"]
        self.paths = cfg["files"][start:end]

    def _stream_parquet(self, path: str) -> Iterator[Dict[str, str]]:
        url = f"{HF_CDN}/{self.repo}/resolve/main/{self.date}/{path}"
        resp = requests.get(url, timeout=60)
        resp.raise_for_status()
        tbl = pq.read_table(io.BytesIO(resp.content))
        if "prompt" not in tbl.column_names or "response" not in tbl.column_names:
            return
        for row in tbl.select(["prompt", "response"]).to_pylist():
            yield {"prompt": row["prompt"], "response": row["response"]}

    def __iter__(self) -> Iterator[Dict[str, str]]:
        for p in self.paths:
            yield from self._stream_parquet(p)
```

**Usage in trainer:**

## 3) Lightning idle-resilient runner (auto-recover)

**Why this version:**  
- Checks studio status before each run and restarts if stopped (survives idle timeout).  
- Runs training in a loop with a 60s heartbeat; reconnects without full rebuild.  
- Uses `Machine.L40S` explicitly (matches scope).

```python
# /opt/axentx/airship/surrogate/train/run_lightning.py
import os
import time
from pathlib import Path

from lightning import Machine, Studio, Teamspace

REPO_ROOT = Path(__file__).parent.parent.parent
MANIFEST = REPO_ROOT / "configs" / "file_manifest_latest.json"  # symlink or dated file

from surrogate.train.trainer import train  # noqa: E402

def ensure_studio(name: str = "surrogate-train", machine: Machine = Machine.L40S) -> Studio:
    ts = Teamspace()
    for s in ts.studios:
        if s.name == name:
            if s.status == "running":
                return s
            s.start(machine=machine)
            return s
    return Studio(name=name, machine=machine, create_ok=True)

def main() -> None:
    if not MANIFEST.exists():
        raise FileNotFoundError(f"Manifest missing: {MANIFEST}")

    while True:
        studio = ensure_studio()
        try:
            # If already running, this is a lightweight reattach; if stopped, restarted above.
            job = studio.run(
                target=train,
                args=(str(MANIFEST),),
                requirements=["pyarrow", "requests", "torch", "lightning"],
            )
            # Wait for completion or stop
            while studio.status == "running":
                time.sleep(60)
        except Exception as exc:
            print(f"Training error: {exc}")
        # If studio stopped or crashed, loop will restart it
        time.sleep(30)

if __name__ == "__main__":
    main()
```
