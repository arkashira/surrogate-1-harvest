## Final Implementation Plan — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only card that identifies the most-connected hub from the knowledge-rag graph and surfaces the top hub with 3 contextual signals.  
**Timebox**: <2h (frontend-only, zero backend changes).  
**Tags**: #knowledge-rag #graph #hub #Costinel

---

### 1) Highest-value incremental improvement
Add a persistent “Top-Hub Signal” card to the dashboard that:
- Reads a static JSON file (`/data/knowledge-rag/top-hub.json`) generated by the ops/knowledge-rag pipeline.
- Displays the most-connected hub (e.g., `"MOC"`) + 3 contextual signals (title, excerpt, link).
- Zero backend: card is purely presentational; ops pipeline updates the JSON.

This gives immediate visibility into the most-connected hub (per pattern #top-hub doc insight) without touching ingestion/training infra.

#### B) Add card component (React + Tailwind)
Path: `/opt/axentx/Costinel/src/components/TopHubSignalCard.jsx`

```jsx
import { useEffect, useState } from "react";
import { ExternalLink, TrendingUp } from "lucide-react";
import { fallbackTopHub } from "../lib/top-hub";

export default function TopHubSignalCard() {
  const [hub, setHub] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch("/data/knowledge-rag/top-hub.json", { cache: "no-store" })
      .then((r) => {
        if (!r.ok) throw new Error("Network response was not ok");
        return r.json();
      })
      .then((data) => {
        setHub(data);
        setLoading(false);
      })
      .catch(() => {
        // Use fallback data if JSON missing/invalid
        setHub(fallbackTopHub);
        setLoading(false);
      });
  }, []);

  if (loading) {
    return (
      <div className="animate-pulse rounded-xl border border-gray-200 bg-white p-5 shadow-sm">
        <div className="h-5 w-32 rounded bg-gray-200" />
        <div className="mt-4 space-y-3">
          <div className="h-4 w-full rounded bg-gray-100" />
          <div className="h-4 w-5/6 rounded bg-gray-100" />
        </div>
      </div>
    );
  }

  if (!hub) {
    return null; // silent fail
  }

  return (
    <div className="rounded-xl border border-gray-200 bg-white p-5 shadow-sm">
      <div className="flex items-center justify-between">
        <div className="flex items-center gap-2">
          <TrendingUp className="h-5 w-5 text-amber-600" />
          <h3 className="font-semibold text-gray-900">Top-Hub Signal</h3>
        </div>
        <span className="rounded bg-amber-50 px-2 py-0.5 text-xs font-medium text-amber-700">
          {hub.hub}
        </span>
      </div>

      <p className="mt-1 text-sm text-gray-500">{hub.label || hub.hub}</p>

      <div className="mt-4 space-y-3">
        {hub.signals.map((s) => (
          <div key={s.id} className="group relative rounded-lg border border-gray-100 bg-gray-50 p-3">
            <div className="flex items-start justify-between gap-2">
              <div>
                <p className="text-sm font-medium text-gray-900 group-hover:text-amber-700">
                  {s.title}
                </p>
                <p className="mt-0.5 text-xs text-gray-600">{s.excerpt}</p>
              </div>
              {s.link && (
                <a
                  href={s.link}
                  className="flex-shrink-0 text-gray-400 hover:text-amber-600"
                  aria-label={`Open ${s.title}`}
                >
                  <ExternalLink className="h-4 w-4" />
                </a>
              )}
            </div>
          </div>
        ))}
      </div>

      <p className="mt-3 text-xs text-gray-400">
        Updated {new Date(hub.updatedAt).toLocaleString()}
      </p>
    </div>
  );
}
```
