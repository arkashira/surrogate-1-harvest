# airship / design

## Final Implementation — `airship discover` (merged & hardened)

**Goal:** Ship a zero-config, cron-safe discovery CLI in <2h that deterministically runs business research + knowledge RAG + top-hub review, avoids HF API rate limits during training, and produces 3 concrete, actionable insights.

**Resolved contradictions in favor of correctness + actionability:**
- Use **`/opt/axentx/airship/lib/`** for library code (not bin) to separate executables from sourced helpers.
- Keep **strict Bash hygiene** (`set -euo pipefail`, `SHELL=/bin/bash` in cron, log rotation, non-root safe).
- **Never run model training locally** — rely on Lightning/Kaggle per architecture; use HF CDN + pre-listed file paths to bypass rate limits.
- Provide **safe shims** for `granite-business-research.sh` and `knowledge-rag` so `airship discover` always works.
- Emit **exactly 3 prioritized, actionable insights** with suggested next commands.

### 1) Main CLI: `bin/airship`

```bash
#!/usr/bin/env bash
# /opt/axentx/airship/bin/airship
# Airship CLI — zero-config discovery and operations
# Usage: airship discover [--cron] [--quiet]

set -euo pipefail

ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
export AIRSHIP_ROOT="${AIRSHIP_ROOT:-$ROOT}"
export PATH="$AIRSHIP_ROOT/bin:$AIRSHIP_ROOT/lib:$PATH"
export TMPDIR="${TMPDIR:-/tmp}"

_log() {
  local ts
  ts="$(date -u +"%Y-%m-%dT%H:%M:%SZ")"
  echo "[$ts] $*" >&2
}

_require_lib() {
  # shellcheck source=/dev/null
  source "$AIRSHIP_ROOT/lib/$1"
}

_cmd_discover() {
  local cron_mode=0 quiet=0
  for arg in "$@"; do
    case "$arg" in
      --cron) cron_mode=1 ;;
      --quiet) quiet=1 ;;
      *) _log "Unknown arg: $arg"; exit 1 ;;
    esac
  done

  if [[ "$cron_mode" -eq 1 ]]; then
    export SHELL=/bin/bash
    mkdir -p "$AIRSHIP_ROOT/logs"
    exec >>"$AIRSHIP_ROOT/logs/discover-$(date -u +%Y%m%d).log" 2>&1
  fi

  if [[ "$quiet" -eq 1 ]]; then
    exec 1>/dev/null
  fi

  _log "Starting airship discover"

  _require_lib "hf-cdn.sh"
  _require_lib "discover.sh"

  run_business_research
  run_knowledge_rag_top_hub
  print_top_hub_insights

  _log "Finished airship discover"
}

case "${1:-help}" in
  discover) shift; _cmd_discover "$@" ;;
  help|--help|-h)
    cat <<'EOF'
Usage: airship <command>

Commands:
  discover    Run business research + knowledge-rag + top-hub review
  help        Show this help
EOF
    ;;
  *)
    _log "Unknown command: $1"
    exit 1
    ;;
esac
```

Make executable:

```bash
chmod +x /opt/axentx/airship/bin/airship
```
