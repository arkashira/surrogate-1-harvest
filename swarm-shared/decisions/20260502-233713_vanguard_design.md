# vanguard / design

**Most Valuable Next Improvement: Persistent File-List Cache with Reused Lightning Studio Lifecycle**

### Diagnosis

* No persistent file-list cache → repeated `list_repo_tree`/`load_dataset` → HF API 429 (1000 req/5min).
* Lightning Studio lifecycle not reused → quota burn on recreation.
* Training = repeated file-listing → no persisted knowledge of existing files.
* No persisted knowledge of existing files → repeated schema probing during training.

### Proposed Change

* Create a persistent file-list cache in `vanguard/vanguard.py` to store the file manifest for each dataset.
* Reuse the Lightning Studio lifecycle by checking if a Studio with the same name and status exists before recreating.

### Implementation

**Step 1: Create a persistent file-list cache**

```python
# vanguard/vanguard.py
import json

class Vanguard:
    def get_file_manifest(self, dataset_name):
        file_manifest_path = f"cache/{dataset_name}.json"
        try:
            with open(file_manifest_path, "r") as f:
                return json.load(f)
        except FileNotFoundError:
            # Initialize the file manifest if it doesn't exist
            file_manifest = {}
            with open(file_manifest_path, "w") as f:
                json.dump(file_manifest, f)
            return file_manifest
```

**Step 2: Update `train.py` to use the persisted file manifest**

```python
# vanguard/train.py
import json

# Load cached file list
cache_file = 'vanguard/data/cache.json'
with open(cache_file, 'r') as f:
    cache = json.load(f)

# Check if file list is up-to-date
if cache['file_list'] != get_file_list():
    # Update cache file
    cache['file_list'] = get_file_list()
    with open(cache_file, 'w') as f:
        json.dump(cache, f)

# Use cached file list
file_list = cache['file_list']
```

**Step 3: Update `get_file_list()` function to populate cache file**

```python
# vanguard/utils.py
import json

def get_file_list():
    # ...
    cache_file = 'vanguard/data/cache.json'
    with open(cache_file, 'r') as f:
        cache = json.load(f)
    cache['file_list'] = file_list
    with open(cache_file, 'w') as f:
        json.dump(cache, f)
    return file_list
```

**Step 4: Reuse Lightning Studio lifecycle**

```python
# vanguard/train.py
# Check if Studio with same name and status exists
studio = Teamspace.studios.get(name='my_studio', status='Running')
if studio:
    # Reuse existing Studio
    target = studio
else:
    # Create new Studio
    target = Studio.create_ok(name='my_studio')
```
