# Costinel / ops

## Final synthesized implementation (best of both)

**Goal**: Add a resilient, zero-runtime-HF-API “Top-Hub Signal” panel to Costinel that surfaces the most-connected hub (e.g., MOC) using a CDN-baked artifact. No backend changes; deployable in <2h.

---

## Recommended approach (why)
- **Zero runtime HF API**: Eliminates auth/rate-limit risk and keeps the runtime simple.
- **CDN-first artifact**: One small JSON file served from CDN (or repo assets) with a local build-time fallback.
- **Frontend-only change**: No backend modifications, no new secrets, no infra work.
- **Actionable + observable**: Includes refresh script, CI hook, cache strategy, and clear deploy steps.

---

## Implementation plan (<2h)

### 1) Generate baked artifact (orchestration/Mac)
- Run `knowledge-rag` (or equivalent) to produce top-hub payload.
- Save to `public/signals/top-hub.json` (served as static asset) **and** commit to repo so it’s available at build time as a fallback.
- Keep artifact minimal and CDN-ready.

**Schema** (`public/signals/top-hub.json`):
```json
{
  "hub": "MOC",
  "connections": 142,
  "insight": "Most-connected hub driving cross-account cost visibility; 37 linked signals this week.",
  "relatedDocs": [
    { "title": "Cost Anomaly Playbook", "url": "/docs/playbooks/cost-anomaly.md" },
    { "title": "Multi-Account Tagging Strategy", "url": "/docs/guides/tagging.md" }
  ],
  "lastUpdated": "2026-05-03T04:30:00Z"
}
```

### 2) Add resilient UI component (frontend-only)
- File: `components/TopHubSignalPanel.tsx`
- Behavior:
  - Fetch `/signals/top-hub.json` (or full CDN URL if you prefer) via `useSWR`.
  - Use build-time fallback imported from `data/top-hub.json` when fetch fails or while loading.
  - Cache successful response in `sessionStorage` to avoid repeat fetches in same session.
  - Skeleton/empty/error states included.
  - Expandable list for related docs.

**Component** (concise, production-ready):
```tsx
'use client';

import useSWR from 'swr';
import { useEffect, useState } from 'react';
import fallback from '@/data/top-hub.json';

const fetcher = (url: string) => fetch(url).then((r) => {
  if (!r.ok) throw new Error(`Failed ${r.status}`);
  return r.json();
});

interface HubDoc {
  title: string;
  url: string;
}

interface TopHub {
  hub: string;
  connections?: number;
  insight: string;
  relatedDocs: HubDoc[];
  lastUpdated: string;
}

export default function TopHubSignalPanel() {
  const { data, error } = useSWR<TopHub>('/signals/top-hub.json', fetcher, {
    revalidateOnMount: true,
    revalidateOnFocus: false,
    shouldRetryOnError: false,
  });
  const [hydrated, setHydrated] = useState<TopHub | null>(null);
  const [expanded, setExpanded] = useState(false);

  // session cache + fallback + hydration
  useEffect(() => {
    try {
      const cached = sessionStorage.getItem('costinel:top-hub');
      const cachedData = cached ? JSON.parse(cached) : null;
      if (cachedData) setHydrated(cachedData);
    } catch {
      // ignore cache errors
    }
  }, []);

  useEffect(() => {
    if (data) {
      try { sessionStorage.setItem('costinel:top-hub', JSON.stringify(data)); } catch {}
      setHydrated(data);
    }
  }, [data]);

  const payload = (error ? null : data) || hydrated || fallback;
  const isLoading = !data && !error && !hydrated;

  return (
    <div className="rounded border bg-white p-4 shadow-sm">
      <div className="flex items-start justify-between">
        <div>
          <h3 className="text-sm font-semibold text-gray-900">Top Hub Signal</h3>
          <p className="mt-1 text-2xl font-bold text-blue-600">{payload.hub}</p>
          {typeof payload.connections === 'number' && (
            <p className="text-xs text-gray-500">{payload.connections} connections</p>
          )}
          <p className="mt-2 text-sm text-gray-600">{payload.insight}</p>
        </div>
        <span className="text-xs text-gray-400 whitespace-nowrap">
          Updated {new Date(payload.lastUpdated).toLocaleDateString()}
        </span>
      </div>

      {payload.relatedDocs && payload.relatedDocs.length > 0 && (
        <div className="mt-3">
          <button
            onClick={() => setExpanded((x) => !x)}
            className="text-xs text-blue-600 hover:underline focus:outline-none"
            aria-expanded={expanded}
          >
            {expanded ? 'Hide' : 'Show'} related docs ({payload.relatedDocs.length})
          </button>
          {expanded && (
            <ul className="mt-2 space-y-1">
              {payload.relatedDocs.map((doc, i) => (
                <li key={i}>
                  <a
                    href={doc.url}
                    className="text-sm text-blue-600 hover:underline"
                    target="_blank"
                    rel="noopener noreferrer"
                  >
                    → {doc.title}
                  </a>
                </li>
              ))}
            </ul>
          )}
        </div>
      )}

      {isLoading && (
        <div className="mt-3 animate-pulse space-y-2">
          <div className="h-3 w-3/4 rounded bg-gray-200" />
          <div className="h-3 w-1/2 rounded bg-gray-200" />
        </div>
      )}

      {error && (
        <div className="mt-3 rounded border border-red-200 bg-red-50 p-2 text-xs text-red-800">
          Could not refresh top-hub signal. Showing cached/fallback data.
        </div>
      )}
    </div>
  );
}
```

**Fallback** (`data/top-hub.json`) — commit this to repo so the UI works even before the first CDN fetch:
```json
{
  "hub": "—",
  "connections": 0,
  "insight": "Loading top hub signal…",
  "relatedDocs": [],
  "lastUpdated": "2026-01-01T00:00:00Z"
}
```
