## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16`, `DATE`, `HF_TOKEN`, `REPO_ID` (default: `axentx/surrogate-1-training-pairs`)
- Uses a **pre-generated manifest** (`manifests/{DATE}/file-list.json`) produced once per date by the orchestrator (Mac/Lightning runner) via a single `list_repo_tree` call — avoids recursive `list_repo_files` and 429s
- Each shard deterministically hashes `slug` → bucket `hash(slug) % SHARD_TOTAL` and only processes its slice
- Downloads files **directly from HF CDN** (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with no Authorization header — bypasses `/api/` rate limits entirely
- Projects each file to `{prompt, response}` at parse time (no schema assumptions), dedups via centralized SQLite md5 store (shared across workers via repo file or external store), writes to `batches/public-merged/{DATE}/shard{N}-{HHMMSS}.jsonl`
- Commits result via HF API (one commit per shard per run)
- Includes retry/backoff for 429 (wait 360s), handles HF commit cap by deterministic repo selection if multi-repo strategy enabled

def hf_api_get(path: str, params: dict = None, is_cdn: bool = False):
    headers = {} if is_cdn else {"Authorization": f"Bearer {HF_TOKEN}"}
    url = f"https://huggingface.co{path}"
    backoff = 1
    while True:
        resp = requests.get(url, headers=headers, params=params, timeout=60)
        if resp.status_code == 429:
            wait = 360
            log.warning("HF 429 rate limit; waiting %ss", wait)
            time.sleep(wait)
            continue
        if resp.status_code >= 500:
            time.sleep(backoff)
            backoff = min(backoff * 2, 60)
            continue
        resp.raise_for_status()
        return resp

def parse_file(content: bytes, path: str) -> List[Dict[str, str]]:
    """Return list of {prompt, response} dicts."""
    out = []
    if path.endswith(".jsonl"):
        for line in content.decode().splitlines():
            line = line.strip()
            if not line:
                continue
            try:
                obj = json.loads(line)
            except json.JSONDecodeError:
                continue
            prompt = obj.get("prompt") or obj.get("input") or obj.get("question")
            response = obj.get("response") or obj.get("output") or obj.get("answer")
            if prompt is not None and response is not None:
                out.append({"prompt": str(prompt), "response": str(response)})
    elif path.endswith(".parquet"):
        try:
            table = pq.read_table(pa.BufferReader(content))
            df = table.to_pandas()
            for _, row in df.iterrows():
                prompt = row.get("prompt") or row.get("input") or row.get("question")
                response = row.get("response") or row.get("output") or row.get("answer")
                if prompt is not None and response is not None:
                    out.append({"prompt": str(prompt), "response": str(response)})
        except Exception as e:
            log.warning("Failed to parse parquet %s: %s", path, e)
    else:
        log.debug("Skipping unsupported file: %s", path)
    return out
