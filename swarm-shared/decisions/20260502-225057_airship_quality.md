# airship / quality

## Implementation Plan — <2h Quality Increment

**Goal**: Ship a frontend-safe `airship discover` orchestrator that produces a static, cacheable status page + CDN-only manifest so the frontend can render health/state without SSR/backend calls.

**Why this scope**:  
- Delivers immediate UX value (status visibility)  
- Avoids backend round-trips → faster, safer, cache-friendly  
- Fits <2h: single orchestrator script + one template + small build tweak  
- Aligns with past patterns: Mac runs orchestration only, remote compute for heavy work; CDN bypass for data fetches; deterministic file lists for reproducibility.

---

### Steps (est. 1h 30m)

1. **Create orchestrator** (`scripts/airship-discover.sh`)  
   - Bash, shebang, executable  
   - Runs `airship` CLI to collect: services, environments, last-deploy hashes, health probes (via local `curl` to `/healthz` endpoints)  
   - Produces:  
     - `public/status.json` (deterministic, sorted keys)  
     - `public/status.html` (static page with auto-refresh meta)  
     - `public/manifest.json` (CDN asset list with ETag-style hashes)  
   - Uses `jq` for JSON, `envsubst` or `cat <<EOF` for HTML template  
   - Exits non-zero on any critical failure (CI-friendly)

2. **Add lightweight HTML template** (`templates/status-page.html`)  
   - Minimal CSS, client-side JS to poll `status.json` every 30s  
   - Color-coded service cards, last-updated timestamp  
   - No SSR, no backend — pure static

3. **Integrate into build**  
   - Add `npm run discover` or `make discover` that runs the script  
   - Ensure `public/` is in `.gitignore` but committed in CI artifacts  
   - Optionally add pre-commit hook to regenerate on infra changes

4. **Quality hardening**  
   - Set `SHELL=/bin/bash` in any crontab entry (per past wrapper fixes)  
   - Make script idempotent: overwrite outputs, never append  
   - Use `set -euo pipefail` for safety  
   - Log to `logs/discover-$(date +%s).log`

5. **CDN manifest strategy** (borrow HF CDN bypass insight)  
   - `manifest.json` lists all static assets with content hashes  
   - Frontend fetches manifest once from CDN, then loads assets directly via `<script src="https://cdn.../asset.js">`  
   - Enables cache-busting without backend coordination

### Code Snippets

#### `scripts/airship-discover.sh`
```bash
#!/usr/bin/env bash
# airship-discover.sh — generate static status + CDN manifest
# Usage: ./airship-discover.sh [--output-dir public]
set -euo pipefail

OUTPUT_DIR="${1:-public}"
TIMESTAMP="$(date -u +"%Y-%m-%dT%H:%M:%SZ")"
LOG_DIR="logs"
mkdir -p "$OUTPUT_DIR" "$LOG_DIR"

LOG_FILE="$LOG_DIR/discover-$(date +%s).log"
exec > >(tee -a "$LOG_FILE") 2>&1

echo "[$TIMESTAMP] Starting airship discovery..."

# 1) Collect service states via airship CLI + local probes
SERVICES_JSON=$(airship service list --format json 2>/dev/null || echo '[]')

# Enrich with local health probes (best-effort)
ENRICHED=$(echo "$SERVICES_JSON" | jq -c '
  map(
    . as $svc |
    {
      name: .name,
      port: .port,
      environment: .environment,
      lastDeploy: .lastDeploy,
      health: (
        if .port then
          (try (("http://localhost:" + (.port|tostring) + "/healthz") as $url |
                { status: "unknown", error: "probe disabled" })
           catch
           (if . then . else
             (try (curl -s -o /dev/null -w "%{http_code}" ("http://localhost:" + (.port|tostring) + "/healthz")) as $code |
              if $code == "200" then { status: "healthy", code: $code }
              else { status: "unhealthy", code: $code } end)
           catch { status: "error", error: . } end))
        else { status: "n/a" } end
      )
    }
  )
')

# 2) Build deterministic status.json
STATUS_PAYLOAD=$(jq -n \
  --arg ts "$TIMESTAMP" \
  --argjson services "$ENRICHED" \
  '{
    generatedAt: $ts,
    services: $services | sort_by(.name)
  }')

echo "$STATUS_PAYLOAD" | jq . > "$OUTPUT_DIR/status.json"
echo "[$TIMESTAMP] Wrote $OUTPUT_DIR/status.json"

# 3) Generate static HTML
cat > "$OUTPUT_DIR/status.html" <<'HTML'
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8" />
  <title>Arkship Status</title>
  <meta name="viewport" content="width=device-width,initial-scale=1" />
  <meta http-equiv="refresh" content="30" />
  <style>
    body{font-family:system-ui,sans-serif;margin:0;padding:2rem;background:#0f172a;color:#e2e8f0}
    .card{background:#1e293b;border-radius:8px;padding:1rem;margin-bottom:1rem}
    .healthy{color:#34d399}.unhealthy{color:#f87171}.unknown{color:#94a3b8}
    .meta{font-size:0.875rem;color:#64748b;margin-top:2rem}
  </style>
</head>
<body>
  <h1>Arkship Platform — Service Status</h1>
  <div id="status"></div>
  <div class="meta" id="updated"></div>
  <script>
    async function load() {
      try {
        const res = await fetch('status.json?t=' + Date.now());
        const data = await res.json();
        const container = document.getElementById('status');
        container.innerHTML = data.services.map(s => `
          <div class="card">
            <strong>${s.name}</strong>
            <span class="${s.health.status}">[${s.health.status}]</span>
            ${s.health.code ? `(code ${s.health.code})` : ''}
            <br><small>env: ${s.environment || '—'} | port: ${s.port || '—'}</small>
          </div>
        `).join('');
        document.getElementById('updated').textContent = 'Updated: ' + data.generatedAt;
      } catch(e) {
        document.getElementById('status').innerHTML = '<p>Failed to load status.</p>';
      }
    }
    load();
    setInterval(load, 30000);
  </script>
</body>
</html>
HTML

echo "[$TIMESTAMP] Wrote $OUTPUT_DIR/status.html"

# 4) CDN manifest (content-hashed filenames recommended in CI; here we list with mtime hashes)
MANIFEST_ENTRIES=$(find "$OUTPUT_DIR" -type f -name "*.json" -o -name "*.html" | sort | while read -r f; do
  rel="${f#./}"
  hash=$(sha256sum "$f" | awk '{print $1}' | cut -c1-12)
  size=$(stat -c%s "$f")
  jq -n --arg p "$rel" --arg h "$hash" --argjson s "$size" '{path: $p, hash: $h, size: $s}'
done | jq -s '{generatedAt: $ts, assets: .}' --arg ts "$TIMESTAMP")

echo "$MANIFEST_ENTRIES" > "$OUTPUT_DIR/manifest.json"
echo "[$TIMESTAMP] Wrote $OUTPUT_DIR/manifest.json"

echo "[$TIMESTAMP] Discovery complete."
```
