### Resolved contradictions (correctness + actionability)
1. **Snapshot location & commit strategy**  
   - Keep snapshot inside repo (`batches/public-merged/<date>/snapshot.json`) so training and workers can load it locally without extra tooling.  
   - Commit snapshots (small JSON) — negligible against commit cap; avoids extra artifact-only flows that complicate local dev.

2. **CDN vs `hf_hub_download`**  
   - Prefer direct CDN fetch for speed and zero auth/API usage.  
   - Fallback to `hf_hub_download` (with `repo_type="dataset"`) when CDN fails or for non-public repos.

3. **Deterministic sharding**  
   - Use `hash(slug) % TOTAL_SHARDS` (Candidate 1) — simple, deterministic, and easy to verify.  
   - Do NOT shard by file count modulo; use file-level hash to avoid skew when file sizes vary.

4. **Lightning Studio reuse**  
   - Reuse running studio by name if status == "running".  
   - If stopped (idle timeout), restart with `Machine.L40S` before `.run()`.  
   - Do NOT rely on `create_ok=True` alone; explicit reuse prevents quota waste.

5. **Rate-limit handling**  
   - Add backoff only where HF API is still used (snapshot creation, fallback enrich).  
   - CDN-only training path should never hit HF API 429s.

#### `train.py` (CDN-first loader with fallback)
```python
import json, os, io, requests, pyarrow.parquet as pq, time, random
from typing import List, Dict
from huggingface_hub import hf_hub_download

HF_DATASET = os.getenv("HF_DATASET", "axentx/surrogate-1-training-pairs")
CDN_ROOT = f"https://huggingface.co/datasets/{HF_DATASET}/resolve/main"

def load_snapshot(date: str) -> List[str]:
    path = f"batches/public-merged/{date}/snapshot.json"
    with open(path) as f:
        return json.load(f)["files"]

def cdn_fetch(rel_path: str, retries: int = 3) -> io.BytesIO:
    url = f"{CDN_ROOT}/{rel_path}"
    for attempt in range(retries):
        try:
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            return io.BytesIO(resp.content)
        except Exception:
            if attempt == retries - 1:
                raise
            time.sleep(1 + random.random())
    raise RuntimeError("Unreachable")

def hf_fallback_fetch(rel_path: str) -> io.BytesIO:
    local = hf_hub_download(
        repo_id=HF_DATASET,
        filename=rel_path,
        repo_type="dataset",
        local_files_only=False,
    )
    return open(local, "rb")

def fetch_one(rel_path: str) -> io.BytesIO:
    try:
        return cdn_fetch(rel_path)
    except Exception:
        return hf_fallback_fetch(rel_path)

def backoff_on_429(f):
    def wrapper(*a, **kw):
        for i in range(5):
            try:
                return f(*a, **kw)
            except Exception as e:
                if getattr(e, "response", None) and e.response.status_code == 429:
                    wait = 360 * (1 + random.random())
                    print(f"HF 429, sleeping {wait:.0f}s")
                    time.sleep(wait)
                    continue
                raise
        raise RuntimeError("Max retries exceeded")
    return wrapper

@backoff_on_429
def load_snapshot_safe(date: str) -> List[str]:
    return load_snapshot(date)

def stream_pairs(date: str):
    files = load_snapshot_safe(date)
    for rel in files:
        buf = fetch_one(rel)
        try:
            table = pq.read_table(buf, columns=["prompt", "response"])
        except Exception:
            continue
        for batch in table.to_batches():
            for row in batch.to_pylist():
                yield row
```

#### `bin/dataset-enrich.sh` (deterministic shard worker)

#!/usr/bin/env bash
set -euo pipefail

SHARD_ID="${SHARD_ID:-0}"
TOTAL_SHARDS="${TOTAL_SHARDS:-16}"
DATE="${DATE:-$(date +%Y-%m-%d)}"
SNAPSHOT="batches/public-merged/${DATE}/snapshot.json"

if [[ -f "${SNAPSHOT}" ]]; then
  mapfile -t FILES < <(jq -r '.files[]' "${SNAPSHOT}")
else
  echo "Snapshot not found: ${SNAPSHOT}" >&2
  exit 1
fi

for rel in "${FILES[@]}"; do
  slug_hash=$(echo -n "${rel}" | md5sum | awk '{print $1}')
  bucket=$((0x${slug_hash:0:8} % TOTAL_SHARDS))
  if [[ "${bucket}" -eq "${SHARD_ID}" ]]; then
    echo "Shard ${SHARD_ID} processing ${rel}"
    # normalize -> dedup -> upload shard output
  fi
done
