## Deployment
cf workers

## v0.1 scope
v0.1 in 2 weeks: CLI + CF Worker scheduler that reads Lambda and Cloud Functions schedules (read-only), persists snapshots to Supabase Postgres, detects duplicates (identical cron+owner), drifted schedules (cron changed since last snapshot), and orphaned entries (platform resource missing). Exposes a minimal FastAPI endpoint to list audits and mark schedules retired. No UI; outputs JSON. Free-tier only.
