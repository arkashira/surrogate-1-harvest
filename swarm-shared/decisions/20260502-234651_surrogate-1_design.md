# surrogate-1 / design

## Final Implementation Plan (≤2h)

**Highest-value improvement**: Add a pre-flight snapshot generator (`bin/snapshot.sh`) that lists all dataset files once, embeds the file manifest into training, and enables CDN-only downloads during parallel shard processing. This eliminates HF API rate limits during ingestion and guarantees deterministic file lists across all 16 shards.

### Plan (1h 45m total)

1. **Create `bin/snapshot.sh`** (25m)  
   - Single `list_repo_tree` call per date folder (non-recursive) to avoid pagination-triggered 429s.  
   - Outputs `snapshot/snapshot-<date>-<ts>.json` with CDN URLs and metadata.  
   - Shebang `#!/usr/bin/env bash`, `set -euo pipefail`, `chmod +x`.

2. **Add `lib/snapshot.py`** (20m)  
   - Deterministic shard assignment via `hashlib.md5(slug) % 16`.  
   - Functions: `load_snapshot`, `files_for_shard`, `cdn_fetch_list`.  
   - No external dependencies beyond `huggingface_hub` (already required).

3. **Update `bin/dataset-enrich.sh`** (20m)  
   - Accept `SNAPSHOT_JSON` env var; if present and valid, use manifest instead of HF API.  
   - Each shard computes deterministic slice from manifest; workers stream from CDN URLs only (zero auth/API during processing).  
   - Fallback to legacy listing if snapshot missing/unusable.

4. **Update GitHub Actions** (20m)  
   - Add `snapshot` job that runs once, uploads artifact.  
   - `ingest` job matrix (16 shards) downloads artifact, passes `SNAPSHOT_JSON` to each runner.  
   - Explicit `needs` dependency ensures snapshot is ready before shards start.

5. **Test & Buffer** (20m)  
   - Local dry-run with mock manifest, verify shard assignment determinism.  
   - CI dry-run on a feature branch, confirm zero HF API calls during shard jobs.

#### 1. `bin/snapshot.sh`
```bash
#!/usr/bin/env bash
# bin/snapshot.sh
# Generate a file manifest for axentx/surrogate-1-training-pairs
# Usage: HF_TOKEN=... bin/snapshot.sh [YYYY-MM-DD]
# Output: snapshot/snapshot-<date>-<ts>.json
set -euo pipefail

REPO="axentx/surrogate-1-training-pairs"
HF_TOKEN="${HF_TOKEN:-}"
DATE="${1:-$(date +%Y-%m-%d)}"
TS=$(date +%Y%m%d-%H%M%S)
OUTDIR="snapshot"
OUTFILE="${OUTDIR}/snapshot-${DATE}-${TS}.json"

mkdir -p "${OUTDIR}"

echo "Fetching repo tree for ${REPO} (date=${DATE})..."
python3 - <<PY > "${OUTFILE}"
import os, json, sys
from huggingface_hub import HfApi

api = HfApi(token=os.environ.get("HF_TOKEN"))
repo = "${REPO}"
path = "${DATE}"

try:
    items = api.list_repo_tree(repo=repo, path=path, recursive=False)
except Exception:
    # fallback: list root and filter by prefix
    items = api.list_repo_tree(repo=repo, path="", recursive=False)
    items = [i for i in items if i.path.startswith(f"{path}/")]

manifest = []
for item in items:
    if item.type != "file":
        continue
    manifest.append({
        "path": item.path,
        "cdn_url": f"https://huggingface.co/datasets/{repo}/resolve/main/{item.path}",
        "size": getattr(item, "size", None)
    })

json.dump({"date": "${DATE}", "generated": "${TS}", "files": manifest}, sys.stdout, indent=2)
PY

echo "Snapshot written to ${OUTFILE}"
echo "${OUTFILE}"
```

#### 2. `lib/snapshot.py`
```python
# lib/snapshot.py
import json
import hashlib
import os
from typing import List, Dict

def load_snapshot(path: str) -> Dict:
    with open(path) as f:
        return json.load(f)

def shard_for_slug(slug: str, n_shards: int = 16) -> int:
    """Deterministic shard assignment consistent with runner matrix."""
    digest = hashlib.md5(slug.encode()).hexdigest()
    return int(digest, 16) % n_shards

def files_for_shard(manifest: Dict, shard_id: int, n_shards: int = 16) -> List[Dict]:
    files = manifest.get("files", [])
    assigned = []
    for f in files:
        slug = os.path.splitext(os.path.basename(f["path"]))[0]
        if shard_for_slug(slug, n_shards) == shard_id:
            assigned.append(f)
    return assigned

def cdn_fetch_list(file_entries: List[Dict]) -> List[str]:
    return [f["cdn_url"] for f in file_entries]
```

  snapshot:
    runs-on: ubuntu-latest

      snapshot-path: ${{ steps.set.outputs.path }}
