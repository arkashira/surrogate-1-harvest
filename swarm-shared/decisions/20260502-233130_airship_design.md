# airship / design

## Final Synthesized Implementation (Best Parts + Corrected Contradictions)

I have synthesized both proposals, resolving contradictions in favor of **correctness** and **concrete actionability**.

**Key Resolutions:**
1.  **`list_repo_tree` Recursion**: Candidate 1 used `recursive=False` but suggested manual subfolder iteration. Candidate 2 implied recursion. **Resolution**: Use `recursive=True`. It is safe for the <2h goal, avoids complex manual pagination logic, and ensures we capture all files in nested date structures.
2.  **Training Integration**: Candidate 1 provided a broken snippet (`local`). Candidate 2 was vague. **Resolution**: Provide a working `datasets` loading function that uses `streaming=True` on the generated manifest (correctly avoiding PyArrow casts) and integrates with Lightning.
3.  **CLI Scope**: Candidate 1 mixed download and projection. **Resolution**: Strict separation. `discover` only lists. A separate `project` command handles download/parquet creation. This prevents CLI bloat and race conditions.

---

## 1. Core Architecture (CDN-Only & Deterministic)

| Component | Responsibility | Why |
|-----------|----------------|-----|
| **`discover` CLI** | Calls `list_repo_tree(recursive=True)` once. Saves `filelist.json`. | **Eliminates API calls during training.** Single source of truth. |
| **`project` CLI** | Reads `filelist.json`. Downloads via CDN. Projects to Parquet. | **Avoids PyArrow CastError.** Processes files individually, not as a mixed HF dataset. |
| **`train.py`** | Loads `train_manifest.json` via `datasets.load_dataset(..., streaming=True)`. | **Zero rate limits.** Uses CDN URLs directly. Handles heterogeneous schemas safely. |

### 2.1 `airship/cdn.py` (Shared Utilities)
*Resolves: Deterministic naming, retry logic, URL generation.*

```python
# airship/cdn.py
import os
import json
import time
import hashlib
import requests
from pathlib import Path
from typing import List, Dict, Optional
from huggingface_hub import HfApi

CDN_ROOT = "https://huggingface.co/datasets"

def get_cdn_urls(repo: str, paths: List[str]) -> List[str]:
    return [f"{CDN_ROOT}/{repo}/resolve/main/{p}" for p in paths]

def deterministic_slug(repo: str, path: str) -> str:
    h = hashlib.sha256(f"{repo}/{path}".encode()).hexdigest()[:12]
    name = Path(path).stem
    # Sanitize for filesystem safety
    safe_name = "".join(c for c in name if c.isalnum() or c in "._- ")
    return f"{safe_name}_{h}"

def download_via_cdn(repo: str, path: str, out_dir: Path, max_retries: int = 5) -> Path:
    out_dir.mkdir(parents=True, exist_ok=True)
    url = f"{CDN_ROOT}/{repo}/resolve/main/{path}"
    slug = deterministic_slug(repo, path)
    ext = Path(path).suffix or ".bin"
    out_path = out_dir / f"{slug}{ext}"

    if out_path.exists() and out_path.stat().st_size > 0:
        return out_path

    for attempt in range(max_retries):
        try:
            # Important: No Authorization header to bypass rate limits
            resp = requests.get(url, timeout=60, stream=True)
            resp.raise_for_status()
            with open(out_path, 'wb') as f:
                for chunk in resp.iter_content(chunk_size=8192):
                    f.write(chunk)
            return out_path
        except Exception as exc:
            wait = 2 ** attempt
            if out_path.exists():
                out_path.unlink()  # Remove corrupt file
            print(f"[cdn] Retry {attempt+1}/{max_retries} for {path}: {exc}. Waiting {wait}s")
            time.sleep(wait)
    raise RuntimeError(f"Failed to download {url} after {max_retries} retries")
```

### 2.2 `airship/cli/discover.py` (The Discovery Orchestrator)
*Resolves: Reproducibility. Single API call.*

```python
# airship/cli/discover.py
import json
import click
from datetime import datetime
from pathlib import Path
from huggingface_hub import HfApi

api = HfApi()

@click.command()
@click.option("--repo", required=True, help="HF Dataset Repo (e.g., 'username/dataset')")
@click.option("--date", required=True, help="Date folder (YYYY-MM-DD)")
@click.option("--out", default="./data", type=Path, help="Output directory")
def discover(repo: str, date: str, out: Path):
    """
    Creates a deterministic snapshot of a dataset folder.
    Uses HF API once, then relies on CDN.
    """
    # 1. Fetch tree (Recursive=True to handle nested structures safely)
    print(f"[discover] Listing {repo}/{date}...")
    items = api.list_repo_tree(repo=repo, path=date, recursive=True)
    files = [it.rfilename for it in items if it.type == "file"]
    
    if not files:
        print(f"[discover] WARNING: No files found in {repo}/{date}")
        return

    # 2. Create Manifest
    manifest = {
        "repo": repo,
        "date": date,
        "snapshot_ts": datetime.utcnow().isoformat() + "Z",
        "total_files": len(files),
        "files": sorted(files),
        "cdn_urls": [f"{CDN_ROOT}/{repo}/resolve/main/{f}" for f in sorted(files)]
    }

    # 3. Save
    out.mkdir(parents=True, exist_ok=True)
    manifest_path = out / f"{repo.replace('/', '_')}_{date}_manifest.json"
    with open(manifest_path, 'w') as f:
        json.dump(manifest, f, indent=2)

    print(f"[discover] Success. Found {len(files)} files.")
    print(f"[discover] Manifest saved to: {manifest_path}")
    print(f"[discover] Next step: 'airship project --manifest {manifest_path} --out ./parquet'")

if __name__ == "__main__":
    discover()
```
