# workio / ops

## Final Synthesis (Best Parts + Correctness + Actionability)

### Diagnosis (merged, de-duplicated)
- **Non-idempotent LINE webhook handler**: LINE retries on 5xx/timeouts; duplicate clock/leave/OT events corrupt state and double-count hours.
- **No real-time subscription layer**: frontend relies on local React state and polling; LINE-driven changes are invisible until hard refresh → perceived latency and silent failures.
- **Multi-tab/device divergence**: actions in one tab/device do not propagate to others until reload, causing stale dashboards and double-submissions.
- **Webhook replay risk**: no dedupe key/idempotency window means retries create duplicate records.
- **Ops observability blind**: webhook delivery/processing lacks structured logs/metrics (latency, duplicates, failures) → hard to detect upstream issues or abuse.

---

### Proposed Change (merged)
Add **idempotent LINE webhook handling** + **SSE invalidation channel** for real-time frontend updates + **cross-tab sync** + **structured observability**.

**Why this wins**:
- Idempotency is mandatory for correctness with LINE retries.
- SSE is the simplest, most compatible server-push for browsers (no WebSocket infra required).
- BroadcastChannel gives zero-cost cross-tab sync without server fan-out.
- Structured logs/metrics are required to operate and debug the system.

---

#### 2. Idempotent webhook handler (with observability + tenant extraction)
```ts
// server/src/routes/lineWebhook.ts
import { Router, Request, Response } from 'express';
import { pool } from '../db';
import { processClockEvent, processLeaveEvent, processOtEvent } from '../services/lineService';
import crypto from 'crypto';
import pino from 'pino';

const logger = pino({ name: 'line-webhook' });
const router = Router();

function computeDeliveryId(event: any): string {
  // Use LINE event.id when available; fallback to hash of event + timestamp
  const id = event.id ? String(event.id) : JSON.stringify(event);
  return crypto.createHash('sha256').update(id).digest('hex');
}

function extractTenantId(event: any): string {
  // Prefer group/room/user id from source; fallback to configured tenant header or 'system'
  return event.source?.groupId || event.source?.roomId || event.source?.userId || 'system';
}

router.post('/line/webhook', async (req: Request, res: Response) => {
  const startedAt = Date.now();
  const signature = req.headers['x-line-signature'] as string;
  const body = req.body;

  if (!body.events || !Array.isArray(body.events)) {
    logger.warn({ durationMs: Date.now() - startedAt }, 'invalid payload');
    return res.status(400).json({ error: 'invalid payload' });
  }

  const client = await pool.connect();
  try {
    await client.query('BEGIN');

    for (const event of body.events) {
      const deliveryId = computeDeliveryId(event);
      const tenantId = extractTenantId(event);

      // Idempotency check
      const exists = await client.query(
        'SELECT 1 FROM webhook_events WHERE delivery_id = $1',
        [deliveryId]
      );
      if (exists.rows.length > 0) {
        logger.info({ deliveryId, tenantId, eventType: event.type }, 'duplicate skipped');
        continue;
      }

      // Record before processing (safe for retries)
      await client.query(
        `INSERT INTO webhook_events (delivery_id, tenant_id, event_type, payload)
         VALUES ($1, $2, $3, $4)`,
        [deliveryId, tenantId, event.type, event]
      );

      // Dispatch by type
      switch (event.type) {
        case 'clock':
          await processClockEvent(event, client);
          break;
        case 'leave':
          await processLeaveEvent(event, client);
          break;
        case 'ot':
          await processOtEvent(event, client);
          break;
        default:
          // noop
          break;
      }
    }

    await client.query('COMMIT');

    // Broadcast invalidation for affected tenants (simple: broadcast all for now)
    broadcastInvalidation({ type: 'line_webhook', at: new Date().toISOString() });

    logger.info({ durationMs: Date.now() - startedAt, count: body.events.length }, 'webhook processed');
    res.status(200).json({ ok: true });
  } catch (err) {
    await client.query('ROLLBACK');
    logger.error({ err, durationMs: Date.now() - startedAt }, 'webhook processing failed');
    res.status(500).json({ error: 'processing_failed' });
  } finally {
    client.release();
  }
});

export default router;
```

#### 3. SSE broadcast service (lightweight + typed)
```ts
// server/src/services/sse.ts
import { Request, Response } from 'express';

type Listener = (data: string) => void;
const listeners = new Set<Listener>();

export function broadcastInvalidation(payload: any) {
  const data = `data: ${JSON.stringify(payload)}\n\n`;
  for (const l of listeners) {
    try { l(data); } catch (e) {/* ignore dead connections */}
  }
}

export function sseHandler(req: Request, res: Response) {
  res.setHeader('Content-Type', 'text/event-stream');
  res.setHeader('Cache-Control', 'no-cache');
  res.setHeader('Connection', 'keep-alive');
  res.flushHeaders();

  const listener = (data: string) => res.write(data);
  listeners.add(listener);

  req.on('close', () => listeners.delete(listener));
}
```

#### 4. Frontend realtime client (cross-tab safe + React Query integration)
```tsx
// workio/src/lib/realtime.ts
class Realtime {
  private source: EventSource | null = null;
  private listeners: Set<(payload: any) => void> = new Set();
  private channel = new BroadcastChannel('workio_realtime');

  constructor(private endpoint: string) {
    this.channel.onmessage = (ev) => {
      this.listeners.forEach((l) => l(ev.data));
    };
  }

  connect() {
    if (this.source) return;
    this.source = new EventSource(this.endpoint);
    this.source.onmessage = (ev) => {
      try {
        const payload = JSON.parse(ev.data);
        this.channel.postMessage(payload); // broadcast to other tabs
        this.listeners.forEach((l) => l(payload));
      } catch (e) {
        // ignore malformed
      }
    };
    this.source.onerror = () => {
      this.close();
      setTimeout(() => this.connect(), 3000);
    };
  }

  onMessage(fn: (payload: any) => void) {
    this.listeners.add(fn);
    return () => this.listeners.delete(fn);
  }

  close() {
    if (this.source) {
      this.source.close();
      this.source = null;
    }
  }
}

export const realtime
