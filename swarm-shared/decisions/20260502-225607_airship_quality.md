# airship / quality

## Final Synthesized Implementation (Best of Both Candidates)

**Goal**: Ship a **CDN-safe, deterministic `airship discover` orchestrator** that eliminates HF API calls during training/inference. Uses a **Bash-first CLI** with an optional **Python helper** for robustness, emitting `status.json` and `manifest.json` for CI/local consumption.

### Why This Wins
- **Eliminates 429 Risk**: Training scripts fetch `status.json`/`manifest.json` via CDN (`https://huggingface.co/datasets/.../resolve/main/...`) instead of runtime `list_repo_tree` calls.
- **Zero Runtime Dependencies (Core)**: Bash orchestrator requires only `curl` and standard POSIX tools; Python helper is optional for complex parsing.
- **Deterministic & CI-Friendly**: Sorted file lists, strict error handling, and scheduled CI updates ensure reproducible training runs.
- **Fits Existing Patterns**: Reuses `discover` entrypoint; heavy compute stays on Lightning/Kaggle; Mac acts as CLI orchestrator.

---

### Implementation Plan (≤2h)

#### 1. Create `bin/airship-discover` (Bash Orchestrator)
- **Shebang**: `#!/usr/bin/env bash` (executable).
- **Inputs**: `REPO_ID`, `DATE` (YYYY-MM-DD), `OUT_DIR` (env vars or flags).
- **Steps**:
  - Fetch repo tree **non-recursively** via HF API (`tree?path=${DATE}&recursive=false`).
  - Retry with **360s backoff on 429**; fallback to cached `file-list.json` if API fails.
  - Generate deterministic `manifest.json` (sorted files, CDN URLs) and `status.json` (metadata).
  - Validate JSON with `jq` (if available) or strict regex fallback.
  - Output to `OUT_DIR/${DATE}/`.

#### 2. Add `tools/hf_tree.py` (Optional Python Helper)
- **Purpose**: Robust tree fetching when `jq` is unavailable or complex filtering is needed.
- **Features**:
  - Calls `list_repo_tree` with retries and 360s backoff on 429.
  - Projects `path`, `type`, `size`; outputs sorted JSON array.
  - Used by `bin/airship-discover` if Python is present (graceful fallback to pure Bash).

#### 3. Update CI/CD (GitHub Actions)
- **Scheduled Job**: Runs `bin/airship-discover` nightly and on push to `main`.
- **Caching**: Commits updated `status.json`/`manifest.json` to repo (or pushes to HF dataset branch) so CDNs cache fresh files.
- **Rate Limit Handling**: Uses HF token only for the single list call; exponential backoff on 429.

#### 4. Consume in Training/Inference
- **Replace Runtime HF API Calls**: Fetch `status.json`/`manifest.json` via CDN URLs.
- **Training Script Helper**: Tiny `tools/read_manifest.py` to parse local `manifest.json` and yield CDN URLs (no HF API).

#### 5. Tests & Validation
- **Bash Checks**: `bash -n`, `shellcheck` (if available).
- **JSON Validation**: `jq` in CI; smoke test CDN URLs with `curl -I`.
- **Python Tests**: Verify `hf_tree.py` handles 429 and returns sorted JSON.

### Key Code Snippets

#### `bin/airship-discover` (Core Bash Orchestrator)
```bash
#!/usr/bin/env bash
set -euo pipefail

REPO_ID="${REPO_ID:-axentx/airship-data}"
DATE="${DATE:-$(date +%Y-%m-%d)}"
OUT_DIR="${OUT_DIR:-./dist}"
HF_TOKEN="${HF_TOKEN:-}"
API_RETRY="${API_RETRY:-3}"
API_WAIT="${API_WAIT:-360}"

OUT_PATH="${OUT_DIR}/${DATE}"
MANIFEST_PATH="${OUT_PATH}/manifest.json"
STATUS_PATH="${OUT_PATH}/status.json"
FILE_LIST="${OUT_PATH}/file-list.json"

mkdir -p "${OUT_PATH}"

log() { echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] $*" >&2; }

# Fetch tree with retries and 429 backoff
fetch_tree() {
  local url="https://huggingface.co/api/datasets/${REPO_ID}/tree?path=${DATE}&recursive=false"
  local attempt=0
  while (( attempt < API_RETRY )); do
    attempt=$((attempt + 1))
    log "Fetching tree (attempt ${attempt})..."
    local headers
    headers=$(curl -sSf -H "Authorization: Bearer ${HF_TOKEN}" -D - "${url}" -o "${FILE_LIST}.tmp" 2>/dev/null || true)
    if grep -q "HTTP/1.1 429" <<< "${headers}"; then
      log "Rate limited. Waiting ${API_WAIT}s..."
      sleep "${API_WAIT}"
      continue
    fi
    if [[ -s "${FILE_LIST}.tmp" ]]; then
      mv "${FILE_LIST}.tmp" "${FILE_LIST}"
      log "Tree saved to ${FILE_LIST}"
      return 0
    fi
  done
  log "ERROR: Failed to fetch tree after ${API_RETRY} attempts"
  return 1
}

# Use cached file-list if fetch fails
if ! fetch_tree && [[ -f "${FILE_LIST}" ]]; then
  log "WARNING: Using cached file-list"
elif ! fetch_tree; then
  echo '[]' > "${FILE_LIST}"
fi

# Generate manifest.json with jq if available
if command -v jq >/dev/null 2>&1; then
  jq --arg date "${DATE}" --arg repo "${REPO_ID}" '
    [ .[] | select(.type == "file") ] | sort_by(.path) |
    map({
      name: .path,
      size: (.size // 0),
      cdn_url: "https://huggingface.co/datasets/\($repo)/resolve/main/\($date)/" + .path
    })
  ' "${FILE_LIST}" > "${OUT_PATH}/files.json"

  jq -n \
    --arg repo "${REPO_ID}" \
    --arg date "${DATE}" \
    --arg generated_at "$(date -u +%Y-%m-%dT%H:%M:%SZ)" \
    --slurpfile files "${OUT_PATH}/files.json" \
    '{
      repo: $repo,
      date: $date,
      generated_at: $generated_at,
      count: ($files[0] | length),
      files: $files[0]
    }' > "${MANIFEST_PATH}"
else
  log "WARNING: jq not installed. Generating minimal manifest."
  printf '{"repo":"%s","date":"%s","generated_at":"%s","count":0,"files":[]}\n' \
    "${REPO_ID}" "${DATE}" "$(date -u +%Y-%m-%dT%H:%M:%SZ)" > "${MANIFEST_PATH}"
fi

# Generate status.json
if [[ -f "${MANIFEST_PATH}" ]] && command -v jq >/dev/null 2>&1; then
  count=$(jq '.count' "${MANIFEST_PATH}")
  latest_file=$(jq -r '.files | map(.name) | max // "none"' "${MANIFEST_PATH}")
  cdn_prefix="https://huggingface.co/datasets/${REPO_ID}/resolve/main/${DATE}"
else
  count=0
  latest_file="none"
  cdn_prefix="https://huggingface.co/datasets/${REPO_ID}/resolve/main/${DATE}"
fi

jq -n \
  --arg ok "true" \
  --arg generated_at "$(date -u +%Y-%m-%dT%H:%M:%SZ)" \
  --arg repo "${REPO_ID}" \
  --arg date "${DATE}" \
  --argjson count "${count}" \
  --arg latest_file "${latest_file}" \
  --arg cdn_prefix "${cdn_prefix}" \
  '{
    ok: ($ok == "true"),
    generated_at: $generated_at,
    repo: $repo,
    date: $date,
    count: $count,
    latest_file: $latest_file,
    cdn_prefix: $cdn_prefix
  }' > "${STATUS_PATH}"

log "Generated ${MANIFEST_PATH} and ${STATUS_PATH}"
```
