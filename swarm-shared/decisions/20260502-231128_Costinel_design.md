# Costinel / design

**Final consolidated implementation**  
*(Best parts merged; contradictions resolved for correctness + concrete actionability)*

---

## 1) Design decisions (resolved)

- **Route**: `GET /api/v1/cost-anomaly/signal/top-hub`  
- **Read-only, no side effects** — confirmed.  
- **`for_date`**: optional, ISO `YYYY-MM-DD`, default **today in UTC**.  
- **Graph reuse**: lightweight adapter around existing knowledge-graph; no recomputation.  
- **Response**: compact JSON, dashboard/alert ready.  
- **Deployability**: single new endpoint file + one small adapter + tests; <2h.

Resolved contradictions:
- Use **Pydantic v2** (`Field`, `field_validator`) for strict validation and future-proofing.
- Validate date in **router layer** (FastAPI `Query` + validator) and **adapter layer** (graceful fallback).
- Use **dependency injection** for `KnowledgeGraph` (testable, replaceable).
- Use **UTC ISO timestamps** consistently (`generated_at`).
- Provide **realistic mock** in adapter so endpoint ships immediately; clearly mark where to wire real store.

```python
from datetime import date, datetime
from pydantic import BaseModel, Field, field_validator
from typing import List, Optional


class RelatedNode(BaseModel):
    node_id: str
    label: str
    node_type: str
    weight: float = Field(..., ge=0, le=1)


class TopHubSignal(BaseModel):
    date: str = Field(..., description="Query date (YYYY-MM-DD)")
    generated_at: str = Field(..., description="UTC ISO timestamp")
    hub_id: str
    hub_label: str
    hub_type: str
    score: float = Field(..., ge=0, description="Connection strength / degree centrality")
    top_related: List[RelatedNode] = Field(..., max_length=10)
    context: Optional[str] = Field(
        None,
        description="Short human-readable insight (Sense + Signal)"
    )

    @field_validator("date")
    @classmethod
    def validate_date_format(cls, v: str) -> str:
        try:
            date.fromisoformat(v)
        except ValueError as exc:
            raise ValueError("date must be YYYY-MM-DD") from exc
        return v
```

### `src/costinel/api/v1/endpoints/cost_anomaly.py`
```python
from datetime import date
from fastapi import APIRouter, Depends, HTTPException, Query
from app.core.knowledge_graph import KnowledgeGraph
from app.models.signals import TopHubSignal

router = APIRouter()


def _validate_date(value: str) -> str:
    try:
        parsed = date.fromisoformat(value)
        return parsed.isoformat()
    except ValueError as exc:
        raise HTTPException(status_code=400, detail="Invalid for_date, expected YYYY-MM-DD") from exc


def get_kg() -> KnowledgeGraph:
    # TODO: wire real store here if needed
    return KnowledgeGraph()


@router.get(
    "/cost-anomaly/signal/top-hub",
    response_model=TopHubSignal,
    tags=["Cost Anomaly"],
    summary="Top knowledge-graph hub as cost-anomaly signal (Sense + Signal)",
)
def get_top_hub_signal(
    for_date: str = Query(
        default_factory=lambda: date.today().isoformat(),
        description="Date to evaluate (YYYY-MM-DD)",
    ),
    kg: KnowledgeGraph = Depends(get_kg),
) -> TopHubSignal:
    normalized = _validate_date(for_date)
    try:
        return kg.top_hub(normalized)
    except Exception as exc:
        # Log locally in production
        raise HTTPException(status_code=500, detail="Unable to generate top-hub signal") from exc
```
