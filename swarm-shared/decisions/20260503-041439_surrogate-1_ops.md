# surrogate-1 / ops

## Final Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **Add `bin/generate_manifest.py`**  
   - Runs once on Mac (or via cron) after rate-limit window clears.  
   - Calls `list_repo_tree(path, recursive=False)` per date folder.  
   - Emits `manifest.json` with `{date, slug, repo, path, sha, size}` for every parquet file.  
   - Embeds manifest into repo (or uploads to dataset repo as `batches/manifest/latest.json`).

2. **Add `bin/worker.py`**  
   - Accepts `SHARD_ID` and `N_SHARDS` (matrix 0..15).  
   - Loads `manifest.json`, deterministically hashes `slug` → bucket `hash(slug) % N_SHARDS`.  
   - Processes only assigned shards.  
   - For each file:  
     - Download via **CDN bypass** (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with no auth header.  
     - Stream parquet with `pyarrow.parquet.ParquetFile` and project only `{prompt, response}` columns at parse time (ignore extra/mixed schema).  
     - Normalize to `{prompt:str, response:str, source:str, ts:iso}`.  
     - Compute md5 of `(prompt+response)` and check central dedup store (`lib/dedup.py`).  
     - Keep non-duplicates.  
   - Batch-write output as `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl` via HF Hub `upload_file` (one commit per shard).  
   - Exponential backoff on 429; wait 360s after HF API rate limit.

3. **Replace `bin/dataset-enrich.sh` with `bin/worker.py`**  
   - Thin wrapper that sets `SHELL=/bin/bash`, validates env, then invokes `python bin/worker.py "$@"`.  
   - Ensure executable bit (`chmod +x`).

4. **Update `.github/workflows/ingest.yml`**  
   - Keep 16-shard matrix.  
   - Add step before matrix to optionally generate+upload manifest (or rely on pre-baked manifest in repo).  
   - Set `env.SHELL=/bin/bash`.  
   - Use `actions/checkout` to get manifest.

5. **Add `requirements.txt` entries**  
   - `pyarrow>=14`, `huggingface_hub`, `requests`, `tqdm`.

`bin/generate_manifest.py`
```python
#!/usr/bin/env python3
"""
Generate manifest for surrogate-1-training-pairs.
Run on Mac or via cron after rate-limit window clears.
"""
import json, os, hashlib
from huggingface_hub import HfApi

REPO = "axentx/surrogate-1-training-pairs"
api = HfApi()

def list_date_folders():
    # non-recursive per folder to avoid pagination explosion
    root = api.list_repo_tree(repo_id=REPO, path="", recursive=False)
    dates = [item.name for item in root if item.type == "directory"]
    return dates

def build_manifest():
    manifest = []
    for date in list_date_folders():
        items = api.list_repo_tree(repo_id=REPO, path=date, recursive=False)
        for item in items:
            if item.type == "file" and item.path.endswith(".parquet"):
                manifest.append({
                    "date": date,
                    "slug": os.path.splitext(item.name)[0],
                    "repo": REPO,
                    "path": item.path,
                    "sha": item.lfs.get("sha256", ""),
                    "size": item.size or 0,
                })
    out = {"generated_by": "generate_manifest.py", "files": manifest}
    with open("manifest.json", "w") as f:
        json.dump(out, f, indent=2)
    print(f"Wrote manifest with {len(manifest)} files")

if __name__ == "__main__":
    build_manifest()
```

import os, sys, json, hashlib, time
from pathlib import Path
import pyarrow.parquet as pq
import requests
from huggingface_hub import HfApi, upload_file
from lib.dedup import is_duplicate, mark_seen

REPO = "axentx/surrogate-1-training-pairs"
N_SHARDS = int(os.getenv("N_SHARDS", 16))
SHARD_ID = int(os.getenv("SHARD_ID", 0))
MANIFEST = Path("manifest.json")

HF_TOKEN = os.getenv("HF_TOKEN")
if not HF_TOKEN:
    print("HF_TOKEN required", file=sys.stderr)
    sys.exit(1)

api = HfApi(token=HF_TOKEN)

def slug_hash(slug: str) -> int:
    return int(hashlib.sha256(slug.encode()).hexdigest(), 16)

def cdn_download(path: str, dest: Path):
    url = f"https://huggingface.co/datasets/{REPO}/resolve/main/{path}"
    # CDN bypass: no Authorization header
    with requests.get(url, stream=True, timeout=60) as r:
        r.raise_for_status()
        with open(dest, "wb") as f:
            for chunk in r.iter_content(chunk_size=8192):
                f.write(chunk)

def process_file(entry, out_lines):
    local = Path("/tmp") / Path(entry["path"]).name
    try:
        cdn_download(entry["path"], local)
        pf = pq.ParquetFile(local)
        # project only prompt/response; ignore mixed schemas
        batch = pf.read_row_group(0, columns=["prompt", "response"])
        prompts = batch.column("prompt").to_pylist()
        responses = batch.column("response").to_pylist()
    except Exception as e:
        print(f"Skip {entry['path']}: {e}", file=sys.stderr)
        return
    finally:
        if local.exists():
            local.unlink()

    for p, r in zip(prompts, responses):
        if not isinstance(p, str) or not isinstance(r, str):
            continue
        key = hashlib.md5((p + r).encode()).hexdigest()
        if is_duplicate(key):
            continue
        mark_seen(key)
        out_lines.append(json.dumps({"prompt": p, "response": r, "source": entry["slug"], "ts": entry["date"]}, ensure_ascii=False))
