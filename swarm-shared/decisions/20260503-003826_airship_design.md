# airship / design

**Final Implementation Plan — Manifest-Driven CDN-Only Dataset Loader**

**Scope**  
Replace `load_dataset`/`list_repo_files` usage in surrogate training with a manifest-driven, CDN-only iterable loader that:

- Eliminates HF API 429/128-commit bottlenecks  
- Prevents `pyarrow.CastError` and mixed-schema ingestion failures by projecting `{prompt, response}` at parse time  
- Enables Lightning Studio reuse (no idle-timeout restarts) by making training CDN-only (zero API calls during training)  
- Fits in <2h: one loader module, small train script changes, and one cron/manifest job on the Mac orchestrator  

---

### 1) Manifest Generator (run once per date folder on Mac orchestrator)

**Behavior**  
- Uses HF API **once** to list a date folder (non-recursive)  
- Produces `file_manifest.json` with `{repo, path, sha, size}`  
- Commits or stores manifest alongside training code  

**Script** (`generate_manifest.sh`)

```bash
#!/usr/bin/env bash
# generate_manifest.sh
# Usage: ./generate_manifest.sh <repo> <date_folder> <out_json>
# Example: ./generate_manifest.sh axentx/surrogate-mirror 2026-05-03 manifests/manifest-2026-05-03.json

set -euo pipefail

REPO="${1:-axentx/surrogate-mirror}"
DATE_FOLDER="${2:-$(date +%Y-%m-%d)}"
OUT="${3:-manifests/manifest-${DATE_FOLDER}.json}"

mkdir -p "$(dirname "$OUT")"

python3 - "$REPO" "$DATE_FOLDER" "$OUT" <<'PY'
import json
import os
import sys
from huggingface_hub import HfApi

repo = sys.argv[1]
date_folder = sys.argv[2]
out_path = sys.argv[3]

api = HfApi()
entries = api.list_repo_tree(repo=repo, path=date_folder, recursive=False)

manifest = []
for e in entries:
    if e.type == "file":
        manifest.append({
            "repo": repo,
            "path": f"{date_folder}/{os.path.basename(e.path)}",
            "sha": getattr(e, "sha", None),
            "size": getattr(e, "size", None),
        })

with open(out_path, "w") as f:
    json.dump(manifest, f, indent=2)
print(f"Wrote {len(manifest)} entries to {out_path}")
PY

echo "Manifest generated: $OUT"
```

Make executable and crontab hygiene:

```bash
chmod +x generate_manifest.sh
```

```cron
SHELL=/bin/bash
0 2 * * * /opt/axentx/airship/generate_manifest.sh axentx/surrogate-mirror "$(date +\%Y-\%m-\%d)" /opt/axentx/airship/manifests/manifest-$(date +\%Y-\%m-\%d).json >> /var/log/airship/manifest.log 2>&1
```

### 2) CDN-Only Iterable Dataset Loader

**Behavior**  
- Reads `file_manifest.json`  
- Streams each file via CDN URL  
- Parses per-extension and projects only `{prompt, response}`  
- Yields normalized dicts; filters malformed rows  
- Supports JSONL, JSON, Parquet, and text fallback  

**Code** (`surrogate/training/cdn_dataset.py`)

```python
import io
import json
import logging
from pathlib import Path
from typing import Dict, Iterator, Optional

import requests

try:
    import pyarrow.parquet as pq
except Exception:  # pragma: no cover
    pq = None

logger = logging.getLogger(__name__)


class CdnIterableDataset:
    """
    Manifest-driven, CDN-only iterable dataset.
    Each file in manifest is downloaded via CDN URL and parsed to {prompt, response}.
    """

    CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

    def __init__(
        self,
        manifest_path: str,
        repo: Optional[str] = None,
        max_retries: int = 3,
        timeout: int = 30,
    ):
        self.manifest_path = Path(manifest_path)
        self.repo = repo
        self.max_retries = max_retries
        self.timeout = timeout

        if not self.manifest_path.is_file():
            raise FileNotFoundError(f"Manifest not found: {self.manifest_path}")

        with self.manifest_path.open("r") as f:
            self.manifest = json.load(f)

        if not self.manifest:
            raise ValueError(f"No entries found in manifest: {self.manifest_path}")

    def _cdn_url(self, entry: Dict) -> str:
        repo = entry.get("repo") or self.repo
        if not repo:
            raise ValueError("repo must be provided in entry or constructor")
        return self.CDN_TEMPLATE.format(repo=repo, path=entry["path"])

    def _extract_prompt_response(self, obj: Dict, path: str) -> tuple:
        """
        Extract and normalize (prompt, response).
        Supports common key variants and rejects malformed rows.
        """
        prompt_keys = {"prompt", "input", "question", "instruction"}
        response_keys = {"response", "output", "answer", "completion"}

        # Normalize keys to lowercase for matching
        obj_lower = {k.lower(): v for k, v in obj.items()}

        prompt = None
        for k in prompt_keys:
            if k in obj_lower:
                prompt = obj_lower[k]
                break

        response = None
        for k in response_keys:
            if k in obj_lower:
                response = obj_lower[k]
                break

        # Coerce to string and reject empty/None
        if isinstance(prompt, str):
            prompt = prompt.strip()
        if isinstance(response, str):
            response = response.strip()

        if not prompt or not response:
            logger.debug("Skipping row in %s: missing prompt or response", path)
            return None, None

        return prompt, response

    def _parse_file(self, content: bytes, path: str) -> Iterator[Dict]:
        path_l = path.lower()

        # JSONL
        if path_l.endswith(".jsonl"):
            for line in io.BytesIO(content):
                line = line.strip()
                if not line:
                    continue
                try:
                    obj = json.loads(line)
                except Exception as exc:
                    logger.warning("Skipping invalid JSONL line in %s: %s", path, exc)
                    continue
                prompt, response = self._extract_prompt_response(obj, path)
                if prompt is not None and response is not None:
                    yield {"prompt": prompt, "response": response}
            return

        # JSON array
        if path_l.endswith(".json"):
            try:
                objs = json.loads(content)
            except Exception as exc:
                logger.warning("Skipping invalid JSON file %s: %s", path, exc)
                return
            if not isinstance(objs, list):
                objs = [objs]
            for obj in objs:
                prompt, response = self._extract_prompt_response(obj, path)
                if prompt is not None and response is not None:
                    yield {"prompt": prompt, "response": response}
            return

        # Parquet
        if path_l.endswith(".parquet"):
            if pq is None:
                raise ImportError("pyarrow is required to read parquet files")
            try:
                table = pq.read_table(io.BytesIO(content))
            except Exception as exc:
                logger.warning("Failed to read parquet %s: %s", path, exc)
                return

            for row in table.to_pylist():
                prompt, response = self._extract_prompt_response(row, path)
                if prompt is not None and response is not None:
                    yield {"prompt": prompt, "response": response}
            return

        # Fallback: treat as plain text (one prompt/response per line or simple heuristic)
        try:
            text = content.decode("utf-8")
        except Exception as exc:
            logger.warning("Cannot decode %s as UTF-8: %s", path, exc)
            return

        for line in text.splitlines():
            line = line.strip()
            if not line:
                continue
