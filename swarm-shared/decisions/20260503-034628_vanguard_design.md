# vanguard / design

### Final consolidated implementation (best of both proposals, contradictions resolved)

**Design + contract** (non-negotiable):  
- Deterministic, content-addressed manifest keyed by `{date}/{slug}` listing **only CDN-resolvable file paths**.  
- Manifest is produced **once per date folder** (Mac/local) via a single `list_repo_tree` call; training uses **zero HF API calls** and fetches via CDN URLs.  
- Training is **reproducible** and **rate-limit-safe** (429 risk minimized).  

**Orchestration boundary** (non-negotiable):  
- Mac/local never run model/data loading that should be remote; Lightning Studio reuse enforced to save quota and prevent idle-stop deaths.  

**Scope** (minimal, concrete):  
- New: `vanguard/design/manifest-cdn-contract.md`  
- New: `vanguard/training/train.py` (CDN-only loader)  
- New: `vanguard/orchestration/lightning_launcher.py` (studio reuse + remote launch)  
- Update: `vanguard/README.md` (runbook summary)

### 1) Design contract (single source of truth)

`/opt/axentx/vanguard/design/manifest-cdn-contract.md`
```markdown
# Manifest + CDN-only Contract (vanguard)

## Goal
Guarantee reproducible, rate-limit-safe dataset ingestion and training for surrogate-1
by eliminating runtime HF API calls during data loading.

## Manifest schema
- Key: `{date}/{slug}`  (date = YYYY-MM-DD; slug = content-addressed short hash)
- Value: list of CDN-resolvable file paths relative to dataset repo root.
- No runtime `list_repo_tree` or `load_dataset(streaming=True)` during training.

Example manifest.json:
{
  "2026-04-29/7f3a9c": [
    "batches/mirror-merged/2026-04-29/7f3a9c_001.parquet",
    "batches/mirror-merged/2026-04-29/7f3a9c_002.parquet"
  ],
  "2026-04-30/a1b2c3d": [
    "batches/mirror-merged/2026-04-30/a1b2c3d_001.parquet"
  ]
}

## Data loading contract (training)
- Mac/local: produce `manifest.json` once per date folder via single API call.
- Training script (Lightning Studio) reads `manifest.json` and fetches files via:
  `https://huggingface.co/datasets/{repo}/resolve/main/{path}`
- No Authorization header required; CDN tier has higher limits.
- Parquet projection: read only `{prompt, response}` at parse time.

## Attribution & schema
- Attribution via filename pattern: `batches/mirror-merged/{date}/{slug}.parquet`
- Do NOT add `source` or `ts` columns to parquet (keep schema minimal).

## Lightning Studio reuse
- Before `.run()`, check `Teamspace.studios` for existing running studio.
- If stopped, restart with target machine (L40S preferred; fallback to available).
- Never create new studio when a running one exists (quota preservation).
```

### 2) Manifest generator (safe ingestion helper)

`/opt/axentx/vanguard/ingestion/list_files.py`
```python
#!/usr/bin/env python3
"""
Generate manifest.json for a date folder with a single list_repo_tree call.
Usage:
  HF_TOKEN=... python list_files.py \
    --repo datasets/your-org/surrogate-1 \
    --date-folder 2026-04-29 \
    --slug 7f3a9c \
    --out manifest.json
"""
import argparse
import json
import os
import requests

HF_API_TREE = "https://huggingface.co/api/datasets/{repo}/tree/main"

def list_files(repo: str, date_folder: str) -> list[str]:
    url = HF_API_TREE.format(repo=repo)
    params = {"path": date_folder, "recursive": 1}
    headers = {}
    if token := os.getenv("HF_TOKEN"):
        headers["Authorization"] = f"Bearer {token}"
    resp = requests.get(url, params=params, headers=headers, timeout=30)
    resp.raise_for_status()
    entries = resp.json()
    # Keep only parquet files under the date folder
    paths = [
        e["path"]
        for e in entries
        if e["type"] == "file" and e["path"].endswith(".parquet")
    ]
    return sorted(paths)

def build_manifest(repo: str, date_folder: str, slug: str) -> dict:
    paths = list_files(repo, date_folder)
    key = f"{date_folder}/{slug}"
    return {key: paths}

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="e.g. datasets/your-org/surrogate-1")
    parser.add_argument("--date-folder", required=True, help="YYYY-MM-DD")
    parser.add_argument("--slug", required=True, help="content-addressed short hash")
    parser.add_argument("--out", default="manifest.json")
    args = parser.parse_args()

    manifest = build_manifest(args.repo, args.date_folder, args.slug)
    with open(args.out, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {args.out} with {sum(len(v) for v in manifest.values())} files")

if __name__ == "__main__":
    main()
```

### 3) Training script (CDN-only, reproducible)

`/opt/axentx/vanguard/training/train.py`
```python
#!/usr/bin/env python3
"""
Minimal reference: CDN-only data loader for surrogate-1 training.
Expects manifest.json (generated on Mac) listing CDN-resolvable parquet paths.
"""
import json
import os
import pyarrow.parquet as pq
import requests
import tempfile
from torch.utils.data import IterableDataset, DataLoader
from typing import Dict, List

HF_DATASET_REPO = os.getenv("HF_DATASET_REPO", "datasets/your-org/surrogate-1")
MANIFEST_PATH = os.getenv("MANIFEST_PATH", "manifest.json")
CDN_BASE = f"https://huggingface.co/{HF_DATASET_REPO}/resolve/main"

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path: str, columns: List[str] = ("prompt", "response")):
        with open(manifest_path) as f:
            manifest: Dict[str, List[str]] = json.load(f)
        # Flatten across date/slug keys
        self.paths = [p for paths in manifest.values() for p in paths]
        self.columns = columns

    def _stream_rows(self, cdn_path: str):
        url = f"{CDN_BASE}/{cdn_path}"
        with requests.get(url, stream=True, timeout=60) as r:
            r.raise_for_status()
            with tempfile.NamedTemporaryFile(suffix=".parquet", delete=False) as tmp:
                for chunk in r.iter_content(chunk_size=8192):
                    tmp.write(chunk)
                tmp_path = tmp.name
        try:
            pf = pq.ParquetFile(tmp_path)
            for batch in pf.iter_batches(batch_size=32, columns=self.columns):
                for i in range(batch.num_rows):
                    yield {col: batch[col][i].as_py() for col in self.columns}
        finally:
            os.unlink(tmp_path)

    def __iter__(self):
        for p in self.paths:
            yield from self._stream_rows(p)

if __name__ == "__main__":
    # Quick smoke test: iterate a few samples
    ds = CDNParquetDataset(MANIFEST_PATH)
    dl = DataLoader(ds, batch_size=8, num_workers=0)
    for i, batch in enumerate(dl):
        print(f"batch {i}: prompts={len(batch['prompt'])}")
        if i >= 2:
            break
```
