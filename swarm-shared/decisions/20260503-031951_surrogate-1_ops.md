## Surrogate-1 Ops
### Highest-Value Incremental Improvement
Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:
* Accepts `SHARD_ID`, `SHARD_TOTAL=16`, `DATE`, `HF_TOKEN` via env
* Single `list_repo_tree(path, recursive=False)` per date folder
* Downloads files from Hugging Face CDN using `https://huggingface.co/datasets/{repo}/resolve/main/{path}`

### Implementation Plan
1. **Create `bin/dataset-enrich.py`**:
```python
import os
import json
import requests
from huggingface_hub import Repository

# Load env variables
SHARD_ID = int(os.environ['SHARD_ID'])
SHARD_TOTAL = int(os.environ['SHARD_TOTAL'])
DATE = os.environ['DATE']
HF_TOKEN = os.environ['HF_TOKEN']

# Define Hugging Face repository and dataset
REPO_ID = 'axentx/surrogate-1-training-pairs'
DATASET_PATH = f'datasets/{REPO_ID}/resolve/main/{DATE}'

# Initialize Hugging Face repository
repo = Repository(local_dir='./', repo_id=REPO_ID, token=HF_TOKEN)

# Get list of files in date folder
file_list = repo.list_repo_tree(path=DATASET_PATH, recursive=False)

# Filter files for shard
shard_files = [file for file in file_list if int(file.split('-')[0]) % SHARD_TOTAL == SHARD_ID]

# Download files from Hugging Face CDN
for file in shard_files:
    file_path = f'https://huggingface.co/datasets/{REPO_ID}/resolve/main/{DATE}/{file}'
    response = requests.get(file_path)
    with open(file, 'wb') as f:
        f.write(response.content)
```
