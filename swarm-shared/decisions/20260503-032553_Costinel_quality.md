# Costinel / quality

## Final Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h, production-ready)

### Scope (single highest-value deliverable)
Add a **non-blocking Top-Hub Signal Panel** to the Costinel dashboard that:
- Detects and surfaces the most-connected hub (default `MOC`) from a lightweight hub-graph index generated by knowledge-rag
- Uses **CDN-first static asset pattern** (served from `public/data/`) for zero-runtime-API and zero-rate-limit impact
- Renders as a dismissible signal card with contextual insights, top tags, and a “Review hub” action
- Records an **audit trail entry** when user clicks “Review hub” (non-blocking, fire-and-forget)
- Falls back gracefully when index unavailable (no dashboard breakage, silent console debug)

### Architecture decisions (resolved)
- **CDN-first static JSON**: `public/data/hub-index.json` (committed or CI-generated). Served as static asset; no backend changes, no DB migrations, no new secrets.
- **Non-blocking load**: panel fetches after paint; fetch failures are silent (console debug only) and do not block UI.
- **Audit on action**: “Review hub” click triggers a best-effort audit log call (non-blocking; failure does not block navigation).
- **UX**: card shows hub name, connection count, summary, top tags, “Review hub” link, and dismiss button. Dismissal persisted 24h in localStorage.

2) **Panel component (frontend)**  
   `src/components/TopHubSignalPanel.jsx`
   ```jsx
   import React, { useEffect, useState } from 'react';

   const STORAGE_KEY = 'costinel-top-hub-dismissed-until';

   function useDismissible(dismissHours = 24) {
     const [dismissed, setDismissed] = useState(false);

     useEffect(() => {
       try {
         const until = localStorage.getItem(STORAGE_KEY);
         if (until && Date.now() < Number(until)) setDismissed(true);
       } catch (e) {
         // ignore storage errors
       }

       const dismiss = () => {
         try {
           localStorage.setItem(STORAGE_KEY, String(Date.now() + dismissHours * 3600_000));
         } catch (e) {
           // ignore
         }
         setDismissed(true);
       };

       return { dismissed, dismiss };
     }, [dismissHours]);

     return { dismissed, dismiss };
   }

   // Non-blocking audit logger (best-effort)
   function recordAudit(eventType, payload) {
     const url = '/api/audit'; // adjust if your audit endpoint differs
     const body = JSON.stringify({ eventType, ...payload, ts: new Date().toISOString() });
     // fire-and-forget with keepalive where available
     if (navigator.sendBeacon) {
       navigator.sendBeacon(url, body);
     } else {
       fetch(url, { method: 'POST', body, headers: { 'Content-Type': 'application/json' }, keepalive: true }).catch(() => {
         // silent fail
       });
     }
   }

   export default function TopHubSignalPanel() {
     const [hub, setHub] = useState(null);
     const [loading, setLoading] = useState(true);
     const { dismissed, dismiss } = useDismissible(24);

     useEffect(() => {
       fetch('/data/hub-index.json', { cache: 'no-cache' })
         .then((r) => {
           if (!r.ok) throw new Error('hub-index unavailable');
           return r.json();
         })
         .then((data) => {
           setHub(data.topHub || null);
         })
         .catch(() => {
           // silent fail — non-blocking
           setHub(null);
         })
         .finally(() => setLoading(false));
     }, []);

     const handleReview = (hub) => {
       // record audit before navigation
       recordAudit('hub_review_clicked', { hubId: hub.id, hubName: hub.name });
       // allow default link navigation to proceed
     };

     if (dismissed || loading || !hub) return null;

     return (
       <div className="rounded-lg border bg-white p-4 shadow-sm ring-1 ring-gray-100">
         <div className="flex items-start justify-between">
           <div className="flex items-start gap-3">
             <div className="flex h-10 w-10 shrink-0 items-center justify-center rounded-full bg-blue-50 text-blue-600">
               <svg className="h-5 w-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                 <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M13 7h8m0 0v8m0-8l-8 8-4-4-6 6" />
               </svg>
             </div>
             <div>
               <h3 className="text-sm font-semibold text-gray-900">Top Hub Signal</h3>
               <p className="text-sm text-gray-500">
                 Most-connected hub: <span className="font-medium text-gray-900">{hub.name}</span>
                 <span className="ml-1 text-xs text-gray-400">({hub.connections} connections)</span>
               </p>
               {hub.summary && <p className="mt-1 text-xs text-gray-500">{hub.summary}</p>}
               {hub.tags && hub.tags.length > 0 && (
                 <div className="mt-2 flex flex-wrap gap-1">
                   {hub.tags.slice(0, 4).map((t) => (
                     <span
                       key={t}
                       className="inline-flex items-center rounded-full bg-gray-100 px-2 py-0.5 text-xs font-medium text-gray-600"
                     >
                       {t}
                     </span>
                   ))}
                 </div>
               )}
             </div>
           </div>
           <div className="flex items-start gap-2">
             {hub.insightsUrl && (
               <a
                 href={hub.insightsUrl}
                 onClick={() => handleReview(hub)}
                 className="text-xs font-medium text-blue-600 hover:underline"
               >
                 Review hub
               </a>
             )}
             <button
               type="button"
               onClick={dismiss}
               className="text-gray-400 hover:text-gray-600"
               aria-label="Dismiss signal"
             >
               <svg className="h-4 w-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                 <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M6 18L18 6M6 6l12 12" />
               </svg>
             </button>
           <//div>
         </div>
       </div>
     );
   }
   ```
