# surrogate-1 / ops

## Final Implementation Plan (≤2h)

**Highest-value change**: Replace runtime `load_dataset(streaming=True)` + recursive `list_repo_files` in `bin/dataset-enrich.sh` with a deterministic pre-flight snapshot + CDN-only fetches. This removes HF API rate-limit exposure (429s) and recursive pagination costs while keeping per-shard memory bounded and reproducible.

---

### Steps (1h 45m total)

1. **Add snapshot generator** (`bin/list-snapshot.sh`) — run once per cron tick on Mac/CI before the matrix starts.  
   - Uses a **single non-recursive** `list_repo_tree` per top-level date folder.  
   - Produces `snapshot/<date>/file-list.json` (flat list of parquet/jsonl paths).  
   - Embeds snapshot commit hash / timestamp so shards are reproducible.

2. **Update `bin/dataset-enrich.sh`**:  
   - Accept snapshot path via `SNAPSHOT_FILE` (or date via `DATE`, default to latest snapshot).  
   - Deterministic shard assignment: `hash(filename) % 16 == SHARD_ID`.  
   - Fetch files via **CDN only** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header.  
   - Stream-parse each file and project to `{prompt, response}` only; drop extra schema columns.  
   - Keep existing dedup via `lib/dedup.py` (or inline fast dedup) and output to `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.

3. **Update workflow** (`.github/workflows/ingest.yml`):  
   - Add a pre-step (or separate job) that runs `bin/list-snapshot.sh` and uploads the snapshot as an artifact.  
   - Matrix jobs download the snapshot artifact and set `SNAPSHOT_FILE`.  
   - Retain the 16-shard matrix; no other behavioral changes.

4. **Validation & rollback**:  
   - Dry-run snapshot locally; verify shard counts are balanced.  
   - Keep old codepath behind a feature flag (`USE_CDN_ONLY=1`) for quick rollback.  
   - Add a small smoke test in CI that checks: snapshot exists, shard outputs non-empty, and no 429s occur.

## Code Snippets

### 1. Snapshot generator (`bin/list-snapshot.sh`)

```bash
#!/usr/bin/env bash
set -euo pipefail

REPO="${HF_DATASET_REPO:-axentx/surrogate-1-training-pairs}"
OUTDIR="${SNAPSHOT_DIR:-snapshot}"
DATE="${1:-$(date +%Y-%m-%d)}"
OUTFILE="${OUTDIR}/${DATE}/file-list.json"

mkdir -p "$(dirname "${OUTFILE}")"

echo "Listing ${REPO} (non-recursive) for date=${DATE}..."
python3 - <<PY
import os, json, sys
from huggingface_hub import HfApi

api = HfApi()
repo = os.environ["REPO"]
date = os.environ["DATE"]
outfile = os.environ["OUTFILE"]

# Non-recursive list for the date folder
entries = api.list_repo_tree(
    repo_id=repo,
    path=date,
    repo_type="dataset",
    recursive=False
)

# Keep only file-like entries (exclude subfolders)
files = sorted(
    e.rstrip("/") for e in entries
    if e.strip() and not e.endswith("/")
)

payload = {
    "date": date,
    "repo": repo,
    "generated_at_utc": __import__("datetime").datetime.utcnow().isoformat() + "Z",
    "files": files
}

with open(outfile, "w", encoding="utf-8") as f:
    json.dump(payload, f, indent=2, ensure_ascii=False)

print(f"Wrote {len(files)} files to {outfile}")
PY
```

```bash
#!/usr/bin/env bash
set -euo pipefail

# Required env
: "${HF_TOKEN:?required}"
: "${SHARD_ID:?required (0-15)}"
: "${SNAPSHOT_FILE:?required}"
: "${DATE:?required (YYYY-MM-DD)}"

REPO="axentx/surrogate-1-training-pairs"
OUTDIR="batches/public-merged/${DATE}"
TS=$(date +%H%M%S)
OUTFILE="${OUTDIR}/shard${SHARD_ID}-${TS}.jsonl"

mkdir -p "${OUTDIR}"

python3 - <<PY
import os, json, hashlib, sys, requests, pyarrow.parquet as pq, tempfile
from pathlib import Path

HF_TOKEN = os.environ["HF_TOKEN"]
REPO = os.environ.get("REPO", "axentx/surrogate-1-training-pairs")
SHARD_ID = int(os.environ["SHARD_ID"])
SNAPSHOT_FILE = os.environ["SNAPSHOT_FILE"]
OUTFILE = os.environ["OUTFILE"]
DATE = os.environ["DATE"]

with open(SNAPSHOT_FILE, encoding="utf-8") as f:
    snapshot = json.load(f)

files = snapshot.get("files", [])
date = snapshot.get("date", DATE)

def shard_for(name: str) -> int:
    return int(hashlib.md5(name.encode()).hexdigest(), 16) % 16

my_files = [f for f in files if shard_for(f) == SHARD_ID]
print(f"Shard {SHARD_ID}: processing {len(my_files)} files")

def cdn_url(path: str) -> str:
    return f"https://huggingface.co/datasets/{REPO}/resolve/main/{date}/{path}"

def project_record(raw: dict) -> dict:
    return {
        "prompt": raw.get("prompt") or raw.get("input") or "",
        "response": raw.get("response") or raw.get("output") or "",
    }

dedup = set()
written = 0
session = requests.Session()
# Use CDN (no auth) and reasonable timeout
session.headers.update({"User-Agent": "axentx-ingest-worker"})

with open(OUTFILE, "w", buffering=8 * 1024 * 1024) as out_f:
    for fname in my_files:
        url = cdn_url(fname)
        resp = session.get(url, timeout=60)
        resp.raise_for_status()

        with tempfile.NamedTemporaryFile(delete=False) as tmp:
            tmp.write(resp.content)
            tmp_path = tmp.name

        try:
            if fname.endswith(".parquet"):
                table = pq.read_table(tmp_path, columns=["prompt", "response"])
                for batch in table.to_batches(max_chunksize=8192):
                    cols = {c: batch.column(c).to_pylist() for c in batch.schema.names}
                    n = len(cols.get("prompt", []))
                    for i in range(n):
                        rec = {k: cols[k][i] for k in cols}
                        pr = project_record(rec)
                        if not pr["prompt"] or not pr["response"]:
                            continue
                        h = hashlib.md5((pr["prompt"] + "\0" + pr["response"]).encode()).hexdigest()

                            continue
                        dedup.add(h)

                        written += 1
