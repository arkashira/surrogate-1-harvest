# airship / design

## Final Consolidated Implementation Plan (≤2h)

**Core objective**: Eliminate HF API 429s during Surrogate training and prevent Lightning idle-stop waste while maintaining schema safety and attribution.

### Unified Architecture (Best of Both Candidates)

1. **CDN-Only Data Loading** (Strongest from Candidate 1 + Schema guard from Candidate 2)
   - Replace all `load_dataset()` and runtime `list_repo_files` calls with a pre-generated CDN manifest
   - Manifest generated via Mac-side orchestration after ingestion completes (avoids rate-limit window)
   - Training uses `https://huggingface.co/datasets/{repo}/resolve/main/{path}` with zero auth/API calls
   - Schema enforcement: project to `{prompt, response}` only; never write mixed schemas to `enriched/`

2. **Lightning Studio Reuse + Idle Guard** (Combined best practices)
   - Reuse Running studio; restart only if status ≠ Running
   - Pre-check before each `.run()` to handle idle-stop deaths
   - Explicit machine type (L40S) to prevent quota waste

3. **Attribution & Storage** (Candidate 2 clarity)
   - Store attribution via filename: `batches/mirror-merged/{date}/{slug}.parquet`
   - Never commit unmerged/mixed-schema files to `enriched/`

### Implementation (≤2h, No New Infra)

#### 1. Manifest Generator (`scripts/build_manifest.py`)
```python
#!/usr/bin/env python3
"""
Generate CDN-only manifest for Surrogate training.
Run on Mac after ingestion batch completes (post rate-limit window).
"""
import json
import argparse
from pathlib import Path
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., myorg/surrogate-mirror)")
    parser.add_argument("--folder", required=True, help="Date folder (e.g., 2026-05-03)")
    parser.add_argument("--out", required=True, help="Output manifest JSON")
    args = parser.parse_args()

    api = HfApi()
    # Non-recursive per top-level to avoid pagination explosion
    entries = api.list_repo_tree(repo_id=args.repo, path=args.folder, recursive=False)

    manifest = []
    for entry in entries:
        if entry.type != "file" or not entry.path.endswith(".parquet"):
            continue

        # CDN URL (no auth required)
        cdn_url = f"https://huggingface.co/datasets/{args.repo}/resolve/main/{entry.path}"
        manifest.append({
            "repo": args.repo,
            "path": entry.path,
            "cdn_url": cdn_url,
            "size": getattr(entry, "size", None),
            "sha256": getattr(entry, "sha256", None),
        })

    out_path = Path(args.out)
    out_path.parent.mkdir(parents=True, exist_ok=True)
    with open(out_path, "w") as f:
        json.dump(manifest, f, indent=2)

    print(f"Wrote {len(manifest)} entries to {out_path}")

if __name__ == "__main__":
    main()
```

#### 2. CDN-Only Training Loader (`surrogate/train.py` snippet)
```python
import json
import pyarrow.parquet as pq
import requests
from io import BytesIO

def load_from_manifest(manifest_path, columns=("prompt", "response")):
    """Zero HF API calls during data loading."""
    with open(manifest_path) as f:
        files = json.load(f)

    for item in files:
        resp = requests.get(item["cdn_url"], timeout=60)
        resp.raise_for_status()
        table = pq.read_table(BytesIO(resp.content), columns=columns)
        yield from table.to_pylist()  # List[Dict[str, Any]]
```

#### 3. Studio Reuse + Idle Guard (`scripts/ensure_studio.py`)
```python
#!/usr/bin/env python3
import lightning as L
from lightning.fabric.utilities.cloud import Machine

STUDIO_NAME = "surrogate-train"
MACHINE = "L40S"

def ensure_studio():
    studio = None
    for s in L.Teamspace.studios:
        if s.name == STUDIO_NAME:
            studio = s
            break

    if studio is not None and studio.status == "Running":
        print(f"Reusing running studio: {studio.name}")
        return studio

    if studio is not None:
        print(f"Studio exists but status={studio.status}. Starting fresh...")

    print(f"Creating studio: {STUDIO_NAME}")
    studio = L.Studio.create_ok(
        name=STUDIO_NAME,
        machine=Machine.L40S,
    )
    return studio

if __name__ == "__main__":
    studio = ensure_studio()
    print(f"STUDIO_ID={studio.id}")
```
