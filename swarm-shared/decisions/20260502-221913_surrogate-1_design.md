# surrogate-1 / design

## Final Implementation Plan (≤2h)

**Highest-value incremental improvement**: Add deterministic pre-flight file listing + CDN-only ingestion path to eliminate HF API rate limits during training and make shard workers resilient to 429s.

### What we’ll change
- Add `bin/list-files-once.py` — runs once (Mac or after rate-limit clears), calls `list_repo_tree` per date folder, saves deterministic `file-list.json`.
- Add `lib/cdn_fetch.py` — lightweight CDN fetcher with retries, zero HF API auth during data load, plus schema-safe `{prompt,response}` projection to avoid pyarrow CastError from mixed schemas.
- Update `bin/dataset-enrich.sh` to accept optional `FILE_LIST`; if provided, workers stream from CDN URLs only (no `load_dataset`/`list_repo_files` during run).
- Update `.github/workflows/ingest.yml` to generate and upload `file-list.json` as an artifact and pass it into the matrix job so all 16 shards use the same deterministic snapshot.
- Add short README note on the HF CDN bypass pattern and deterministic sharding.

Why this matters
- Eliminates repeated `list_repo_files`/`load_dataset` API calls that trigger 429s.
- CDN downloads (`/resolve/main/`) have much higher rate limits and no auth checks.
- Keeps shard workers deterministic, parallel-safe, and schema-resilient.
- Fits in <2h (small scripts + workflow tweak).

### 1) `bin/list-files-once.py`

```python
#!/usr/bin/env python3
"""
list-files-once.py
Usage:
  HF_TOKEN=... python3 list-files-once.py \
    --owner axentx \
    --repo surrogate-1-training-pairs \
    --date-folder 2026-05-02 \
    --output file-list.json

Produces deterministic file list for a date folder (recursive).
"""
import argparse
import json
import os
import sys
from huggingface_hub import HfApi

def main() -> None:
    parser = argparse.ArgumentParser(description="List repo files under a date folder.")
    parser.add_argument("--owner", default="axentx")
    parser.add_argument("--repo", default="surrogate-1-training-pairs")
    parser.add_argument("--date-folder", required=True)
    parser.add_argument("--output", default="file-list.json")
    args = parser.parse_args()

    api = HfApi(token=os.environ.get("HF_TOKEN"))
    repo_id = f"{args.owner}/{args.repo}"

    files = []

    def collect(prefix: str) -> None:
        items = api.list_repo_tree(repo=repo_id, path=prefix, recursive=False)
        for item in items:
            if item.type == "directory":
                collect(item.path)
            else:
                files.append(item.path)

    collect(args.date_folder)
    files.sort()

    out = {
        "repo": repo_id,
        "date_folder": args.date_folder,
        "files": files,
    }
    with open(args.output, "w", encoding="utf-8") as f:
        json.dump(out, f, indent=2, ensure_ascii=False)
    print(f"Wrote {len(files)} files to {args.output}", file=sys.stderr)

if __name__ == "__main__":
    main()
```
