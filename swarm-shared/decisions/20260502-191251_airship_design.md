# airship / design

## Implementation Plan: `airship discover` — Zero-config discovery CLI

**Highest-value incremental improvement (<2h):**  
Ship a single Bash CLI (`airship discover`) that operationalizes past patterns (business research + knowledge-rag + top-hub insight) and surfaces strategic context for Arkship/Surrogate platform decisions. No Python dependencies, zero config, works out-of-the-box.

### Scope
- Single executable: `/opt/axentx/airship/bin/airship` (symlink or wrapper)
- Subcommand: `discover`
- Runs: `granite-business-research.sh` → `knowledge-rag` query (top hub + related docs) → formatted output
- Handles script-execution pitfalls (shebang, executable bit, Bash invocation) per past patterns
- Safe for cron (no interactive prompts, proper exit codes)

### Design Decisions
- **Bash-first**: avoids Python env issues, cron-friendly, portable
- **Fail-fast**: each step checks exit code, logs to stderr, exits non-zero on failure
- **Idempotent**: can run repeatedly; creates tmp dirs safely with `mktemp`
- **No state pollution**: cleans up temp dirs on exit (trap)
- **Lightweight output**: Markdown summary to stdout, structured JSON optional via `--json`

### File Layout (repo-relative)
```
airship/
├── bin/
│   └── airship              # main CLI (executable)
├── lib/
│   └── airship-common.sh    # shared helpers (logging, cleanup)
├── scripts/
│   ├── granite-business-research.sh
│   └── knowledge-rag.sh
└── README.md
```

## Implementation

### 1) Shared helpers (`lib/airship-common.sh`)

```bash
#!/usr/bin/env bash
# lib/airship-common.sh
set -euo pipefail

AIRSHIP_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
export AIRSHIP_ROOT

log() {
  echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] [airship] $*" >&2
}

fail() {
  log "ERROR: $*"
  exit 1
}

cleanup() {
  if [[ -n "${_TMP_DIR:-}" && -d "$_TMP_DIR" ]]; then
    rm -rf "$_TMP_DIR"
  fi
}
trap cleanup EXIT

mkdtemp_safe() {
  _TMP_DIR="$(mktemp -d)"
  chmod 700 "$_TMP_DIR"
  echo "$_TMP_DIR"
}
```

### 3) Main CLI (`bin/airship`)

```bash
#!/usr/bin/env bash
# bin/airship
# Airship CLI — zero-config strategic context
set -euo pipefail

AIRSHIP_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
export AIRSHIP_ROOT

source "$AIRSHIP_ROOT/lib/airship-common.sh"

VERSION="0.1.0"

usage() {
  cat <<EOF
Usage: airship <command> [options]

Commands:
  discover    Run business research + knowledge-rag and summarize strategic context
  help        Show this help
  version     Show version

Examples:
  airship discover
  airship discover --json
EOF
}

discover() {
  local output_format="markdown"
  while [[ $# -gt 0 ]]; do
    case "$1" in
      --json) output_format="json"; shift ;;
      --help|-h) usage; exit 0 ;;
      *) fail "Unknown option: $1" ;;
    esac
  done

  log "Starting discovery (format=$output_format)"

  # Step 1: Business research
  log "Running granite-business-research.sh"
  if ! bash "$AIRSHIP_ROOT/scripts/granite-business-research.sh"; then
    fail "granite-business-research.sh failed"
  fi

  # Step 2: Knowledge RAG — top hub and related docs
  log "Running knowledge-rag (top-hub insight)"
  local rag_output
  rag_output="$(bash "$AIRSHIP_ROOT/scripts/knowledge-rag.sh" --top-hub 2>&1)" || {
    fail "knowledge-rag.sh failed"
  }

  # Parse top hub from output (best-effort; adapt to actual script output)
  local top_hub="MOC"
  if echo "$rag_output" | grep -qE 'top[- ]hub|most.connected|hub:'; then
    top_hub="$(echo "$rag_output" | grep -oE '[A-Z0-9]{2,10}' | head -1 || echo "MOC")"
  fi

  # Step 3: Build summary
  local timestamp
  timestamp="$(date -u +%Y-%m-%dT%H:%M:%SZ)"

  if [[ "$output_format" == "json" ]]; then
    cat <<JSON
{
  "timestamp": "$timestamp",
  "version": "$VERSION",
  "top_hub": "$top_hub",
  "research_status": "completed",
  "rag_output_summary": "$(echo "$rag_output" | head -5 | sed 's/"/\\"/g' | tr '\n' ' ')",
  "recommendations": [
    "Review the most-connected hub ($top_hub) before planning tasks",
    "Use Arkship platform for workflow orchestration and incident management",
    "Use Surrogate AI for DevOps assistance and consensus learning"
  ]
}
JSON
  else
    cat <<MD
# Airship Discovery Report

**Generated**: $timestamp  
**Version**: $VERSION

## Top Hub
**$top_hub** — review the most-connected hub before planning tasks.  
(Pattern: #knowledge-rag #graph #hub)

## Research Status
- Business research: completed
- Knowledge RAG: completed

## Key Insights
$(echo "$rag_output" | head -20)

## Recommendations
- Review the most-connected hub ($top_hub) before planning tasks.
- Use Arkship platform (port 8000) for workflow orchestration and incident management.
- Use Surrogate AI (port 8001) for DevOps assistance and consensus learning.

## Patterns Applied
- #business-research #knowledge-rag #graph
- #knowledge-rag #graph #hub
MD
  fi

  log "Discovery completed successfully"
}

cmd="${1:-help}"
case "$cmd" in
  discover) shift; discover "$@" ;;
  version|--version|-v) echo "airship $VERSION" ;;
  help|--help|-h|*) usage ;;
esac
```

Make it executable:

```bash
chmod +x bin/airship
```
