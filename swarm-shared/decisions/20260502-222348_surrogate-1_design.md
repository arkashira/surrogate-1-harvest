# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value improvement**: Add deterministic pre-flight file listing + CDN-only ingestion to eliminate HF API 429s during training and make shard workers resilient.

### What we’ll change
1. Add `bin/list-files.sh` — one-time Mac/Linux script that calls `list_repo_tree` per date folder and writes `file-list-<date>.json` (path + size + sha256). Embeds list into training script so Lightning workers do **zero API calls** during data load.
2. Update `bin/dataset-enrich.sh` to optionally accept a file-list JSON and fetch via CDN (`resolve/main/...`) instead of `datasets.load_dataset`. Falls back to streaming only if list missing.
3. Add `lib/cdn_stream.py` — lightweight CDN iterator that yields `{prompt, response, _source_file, _row_hash}` with schema projection and no HF API auth.
4. Update `lib/dedup.py` to accept external file-list for deterministic sharding (`hash(slug) % 16`) so workers can skip already-processed files when re-run.
5. Add `README-cdn.md` with usage and rate-limit bypass notes.

### Why this is highest value
- Eliminates HF API 429 during ingestion/training (the #1 failure mode).
- Keeps GitHub Actions shard workers fast and independent.
- Enables reproducible Lightning training with embedded file list (no API quota burn).
- <2h scope: only new scripts + small glue changes; no infra or model changes.

```bash
# list-files.sh
# Usage: HF_TOKEN=... ./bin/list-files.sh axentx surrogate-1-training-pairs 2026-05-01
# Writes file-list-2026-05-01.json

set -euo pipefail
REPO_OWNER="${1:-axentx}"

DATE="${3:-$(date +%Y-%m-%d)}"

python3 - <<PY
import os, json, sys
from huggingface_hub import list_repo_tree, hf_api

token = os.environ.get("HF_TOKEN", "")
if not token:
    print("HF_TOKEN required", file=sys.stderr)
    sys.exit(1)

owner = "$REPO_OWNER"
repo = "$REPO_NAME"
date = "$DATE"

# non-recursive per folder to avoid 100x pagination on big repos
folders = ["batches/public-merged/${date}", "batches/mirror-merged/${date}"]
entries = []
for folder in folders:
    try:
        tree = list_repo_tree(repo=repo, path=folder, repo_type="dataset", token=token)
        for node in tree:
            if node.size is not None and node.path.endswith((".jsonl", ".parquet")):
                entries.append({
                    "path": node.path,
                    "size": node.size,
                    "cdn_url": f"https://huggingface.co/datasets/{owner}/{repo}/resolve/main/{node.path}"
                })
    except Exception as e:
        print(f"Warning: {folder} -> {e}", file=sys.stderr)

with open("$OUT", "w") as f:
    json.dump(entries, f, indent=2)

print(f"Wrote {len(entries)} entries to $OUT")
PY
```

### 2) `lib/cdn_stream.py`
```python
# lib/cdn_stream.py
import json
import hashlib
import pyarrow.parquet as pq
import pyarrow as pa
import requests
from typing import Iterator, Dict, Any

CHUNK_SIZE = 8 * 1024 * 1024  # 8MB

def _hash_row(obj: Dict[str, Any]) -> str:
    return hashlib.md5(json.dumps(obj, sort_keys=True).encode()).hexdigest()

def _project(obj: Dict[str, Any]) -> Dict[str, Any]:
    # Keep only {prompt,response} + attribution in filename
    return {
        "prompt": obj.get("prompt") or obj.get("input") or obj.get("text") or "",
        "response": obj.get("response") or obj.get("output") or obj.get("completion") or "",
        "_source_file": obj.get("_source_file", ""),
        "_row_hash": obj.get("_row_hash", ""),
    }

def stream_cdn_jsonl(url: str) -> Iterator[Dict[str, Any]]:
    resp = requests.get(url, stream=True, timeout=60)
    resp.raise_for_status()
    buf = ""
    for chunk in resp.iter_content(chunk_size=CHUNK_SIZE, decode_unicode=True):
        buf += chunk
        while "\n" in buf:
            line, buf = buf.split("\n", 1)
            line = line.strip()
            if not line:
                continue
            try:
                obj = json.loads(line)
                obj["_source_file"] = url
                obj["_row_hash"] = _hash_row(obj)
                yield _project(obj)
            except json.JSONDecodeError:
                continue
    if buf.strip():
        try:
            obj = json.loads(buf.strip())
            obj["_source_file"] = url
            obj["_row_hash"] = _hash_row(obj)
            yield _project(obj)
        except json.JSONDecodeError:
            pass

def stream_cdn_parquet(url: str) -> Iterator[Dict[str, Any]]:
    # streaming download via requests + pyarrow buffer
    resp = requests.get(url, timeout=120)
    resp.raise_for_status()
    table = pq.read_table(pa.BufferReader(resp.content))
    df = table.to_pandas()
    for _, row in df.iterrows():
        obj = row.to_dict()
        obj["_source_file"] = url
        obj["_row_hash"] = _hash_row(obj)
        yield _project(obj)

def stream_file_list(file_list_path: str, shard_id: int = 0, total_shards: int = 16) -> Iterator[Dict[str, Any]]:
    with open(file_list_path) as f:
        files = json.load(f)
    for entry in files:
        path = entry["path"]
        # deterministic shard assignment by slug+path
        slug = path.split("/")[-1].split(".")[0]
        if hash(slug) % total_shards != shard_id:
            continue
        url = entry["cdn_url"]
        if path.endswith(".jsonl"):
            yield from stream_cdn_jsonl(url)
        elif path.endswith(".parquet"):
            yield from stream_cdn_parquet(url)
        else:
            continue
```
