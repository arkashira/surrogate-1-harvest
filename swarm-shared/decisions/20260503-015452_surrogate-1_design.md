# surrogate-1 / design

## Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Uses a **single `list_repo_tree` snapshot** (JSON manifest) generated once per date on the Mac orchestrator and committed to the repo (or passed via artifact).  
- Each GitHub runner shard reads the manifest, filters its deterministic 1/16 slice by `hash(slug) % 16 == SHARD_ID`, then downloads **only those files via HF CDN** (`resolve/main/...`) with **zero HF API calls during streaming**.  
- Projects heterogeneous schemas to `{prompt, response}` at parse time, dedups via central `lib/dedup.py` md5 store, and writes `batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl`.  
- Eliminates `load_dataset(streaming=True)` on mixed-schema repos and avoids 429 rate limits during training data load.

### Steps (≤2h)

1. Create `bin/dataset-enrich.py` (replaces shell script) — 60 lines, robust, shebang + executable.  
2. Update `.github/workflows/ingest.yml` to pass `MANIFEST_JSON` artifact or repo file and set `SHELL=/bin/bash`.  
3. Add lightweight `bin/gen-manifest.py` (optional, run on Mac) to snapshot one date folder and save `manifests/YYYY-MM-DD.json`.  
4. Smoke-test locally with `HF_TOKEN` and one shard.

def belongs_to_shard(slug: str, shard_id: int, shard_total: int) -> bool:
    digest = hashlib.md5(slug.encode("utf-8")).hexdigest()
    bucket = int(digest, 16) % shard_total
    return bucket == shard_id

def download_cdn(url: str, timeout: int = 30) -> bytes:
    headers = {}
    resp = requests.get(url, headers=headers, timeout=timeout, stream=True)
    if resp.status_code == 429:
        wait = RETRY_WAIT
        print(f"Rate limited 429. Waiting {wait}s...", file=sys.stderr)
        time.sleep(wait)
        resp = requests.get(url, headers=headers, timeout=timeout, stream=True)
    resp.raise_for_status()
    return resp.content

def project_to_pair(raw_bytes: bytes, file_path: str) -> dict | None:
    """
    Project heterogeneous file to {prompt, response}.
    Supports: .jsonl, .parquet, .json
    Returns None if unparseable or missing fields.
    """
    suffix = Path(file_path).suffix.lower()
    try:
        if suffix == ".parquet":
            tbl = pq.read_table(pa.BufferReader(raw_bytes))
            df = tbl.to_pandas()
        elif suffix == ".jsonl":
            lines = raw_bytes.decode("utf-8").strip().splitlines()
            import json as _json
            rows = [_json.loads(ln) for ln in lines if ln.strip()]
            df = pa.Table.from_pylist(rows).to_pandas()
        elif suffix == ".json":
            import json as _json
            obj = _json.loads(raw_bytes.decode("utf-8"))
            if isinstance(obj, list):
                df = pa.Table.from_pylist(obj).to_pandas()
            else:
                df = pa.Table.from_pylist([obj]).to_pandas()
        else:
            return None
    except Exception as exc:
        print(f"Failed to parse {file_path}: {exc}", file=sys.stderr)
        return None

    # Heuristic field names
    prompt_col = next((c for c in df.columns if "prompt" in c.lower()), None)
    response_col = next((c for c in df.columns if "response" in c.lower() or "completion" in c.lower()), None)

    if prompt_col is None or response_col is None:
        # If no clear columns, try first two text cols
        text_cols = [c for c in df.columns if df[c].dtype == "object"]
        if len(text_cols) >= 2:
            prompt_col, response_col = text_cols[0], text_cols[1]
        else:
            return None

    # Emit one row per record; dedup will handle cross-row duplicates
    out = []
    for _, row in df.iterrows():
        prompt = str(row.get(prompt_col, "")).strip()
        response = str(row.get(response_col, "")).strip()
        if prompt and response:
            # Deterministic content hash for dedup
            content = (prompt + "\n" + response).encode("utf-8")
            md5 = hashlib.md5(content).hexdigest()
            out.append({"prompt": prompt, "response": response, "md5": md5, "source_file": file_path})
    # Return first pair for simple streaming; caller will iterate all rows
    return out[0] if out else None
