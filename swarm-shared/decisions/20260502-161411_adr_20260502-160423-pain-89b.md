## v0.1 scope
v0.1 ships a cross-platform CLI client (Linux x64, macOS, Windows) with SOCKS5 stealth proxy mode only: local SOCKS5 listener wrapped by noise-encrypted multiplexer, DPI-evasion via traffic padding and innocuous TLS mimicry, client-side SQLite for ephemeral state, Cloudflare Workers KV for revocation/config, and kill-switch that blocks non-stealth traffic. Includes client telemetry anonymized to Supabase for WAU-ZB, automated integration tests for evasion and fail-closed behavior, and a 2-week build focused on stealth reliability and zero-config UX (single binary, no server setup).

## Open questions
- Which DPI signatures (SNI, packet lengths, timing, TTL) are most reliable to mimic in-region innocuous traffic?
- Can we achieve reliable TUN concealment on Windows without admin prompts that fingerprint the client?
- What is the acceptable latency overhead for padding/muxing while maintaining usability for browsing/media?
- How will we distribute ephemeral egress endpoints to clients without creating a detectable distribution channel?
- What telemetry can we collect for WAU-ZB without risking deanonymization or creating a honeypot dataset?
- How do we implement a deterministic fail-closed kill-switch across platforms (Windows Filtering Platform, nftables, pf) without leaks during boot/updates?
