# airship / backend

## Highest-Value Incremental Improvement (<2h)

**Goal**: Eliminate HF API 429s during Surrogate training and prevent Lightning idle-stop training loss.

**What we ship**:
1. A **deterministic CDN file manifest generator** (single API call → JSON) that lists public dataset files for a date folder so Lightning training uses CDN-only fetches (zero API calls during data load).
2. A **Lightning Studio lifecycle wrapper** that reuses running studios, checks status before `.run()`, and auto-restarts stopped studios to avoid idle-stop training loss.

**Why**:
- Fixes HF API rate-limit (429) by replacing recursive `list_repo_files` with one `list_repo_tree` + CDN downloads (bypasses auth/rate-limit).
- Fixes Lightning idle-stop kills training by reusing studios and restarting when stopped.
- Fits <2h: small, focused changes with no breaking refactors.

## Code Snippets

### `scripts/build_cdn_manifest.py`

```python
#!/usr/bin/env python3
"""
Build a deterministic CDN manifest for a date folder in a HuggingFace dataset repo.
Usage:
  python build_cdn_manifest.py --repo datasets/airship/surrogate-mirror \
    --date 2026-05-03 --out manifests/2026-05-03.json
"""
import argparse
import json
import os
import sys
from pathlib import Path

try:
    from huggingface_hub import HfApi
except ImportError:
    print("Error: huggingface_hub not installed. Install with: pip install huggingface_hub")
    sys.exit(1)

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def build_manifest(repo: str, date_folder: str, out_path: str):
    api = HfApi()
    # Single API call: list tree non-recursive for the date folder
    try:
        tree = api.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    except Exception as exc:
        print(f"Failed to list repo tree for {repo}/{date_folder}: {exc}")
        sys.exit(1)

    entries = []
    for item in tree:
        if item.rfilename:
            full_path = f"{date_folder}/{item.rfilename}"
            entries.append({
                "path": full_path,
                "cdn_url": CDN_TEMPLATE.format(repo=repo, path=full_path),
                "size": getattr(item, "size", None)
            })

    out = Path(out_path)
    out.parent.mkdir(parents=True, exist_ok=True)
    with out.open("w") as f:
        json.dump(entries, f, indent=2)

    print(f"Wrote {len(entries)} entries to {out_path}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Build CDN manifest for HF dataset date folder")
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., datasets/airship/surrogate-mirror)")
    parser.add_argument("--date", required=True, help="Date folder (e.g., 2026-05-03)")
    parser.add_argument("--out", required=True, help="Output JSON path")
    args = parser.parse_args()
    build_manifest(args.repo, args.date, args.out)
```

### `scripts/lightning_studio.py`

```bash
#!/usr/bin/env bash
# Wrapper to reuse or restart a Lightning Studio and run training.
# Usage:
#   bash scripts/lightning_studio.py --name surrogate-train-l40s --train-script train.py --manifest manifests/2026-05-03.json

set -euo pipefail

NAME="surrogate-train-l40s"
TRAIN_SCRIPT=""
MANIFEST=""

while [[ $# -gt 0 ]]; do
  case $1 in
    --name) NAME="$2"; shift ;;
    --train-script) TRAIN_SCRIPT="$2"; shift ;;
    --manifest) MANIFEST="$2"; shift ;;
    *) echo "Unknown option $1"; exit 1 ;;
  esac
  shift
done

if [[ -z "$TRAIN_SCRIPT" || -z "$MANIFEST" ]]; then
  echo "Usage: $0 --name NAME --train-script train.py --manifest manifest.json"
  exit 1
fi

cd /opt/axentx/airship

python3 - <<PY
import sys
from lightning_sdk import Studio, Teamspace, Machine

NAME = "${NAME}"
TRAIN_SCRIPT = "${TRAIN_SCRIPT}"
MANIFEST = "${MANIFEST}"

teamspace = Teamspace()
studio = None
for s in teamspace.studios:
    if s.name == NAME:
        studio = s
        break

if studio is None:
    print(f"Creating studio {NAME}...")
    studio = Studio.create(name=NAME, machine=Machine.L40S)
else:
    print(f"Found studio {NAME} (status={studio.status}).")

if studio.status != "Running":
    print(f"Starting studio {NAME}...")
    studio.start(machine=Machine.L40S)

# Wait for running (simple poll)
import time
for _ in range(30):
    studio.refresh()
    if studio.status == "Running":
        break
    time.sleep(10)
else:
    print("Studio did not become running in time.")
    sys.exit(1)

print(f"Running training: {TRAIN_SCRIPT} --manifest {MANIFEST}")
run = studio.run(command=f"python {TRAIN_SCRIPT} --manifest {MANIFEST}")
print(f"Run submitted: {run.id}")
PY
```
