# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-first, CDN-only Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema `CastError`.

### Changes

1. **Add `bin/manifest.py`** — single Mac-side script that:
   - Uses HF API *once* (after rate-limit window) to `list_repo_tree` for today’s folder.
   - Produces `manifest.json` with `{date, slug, path, size, md5}` entries.
   - Embeds this manifest into the runner via `workflow_dispatch` input or artifact.

2. **Replace `bin/dataset-enrich.sh` with `bin/worker.py`** — GitHub Actions runner script that:
   - Accepts `SHARD_ID`, `TOTAL_SHARDS`, and `MANIFEST` (JSON string or artifact file).
   - Iterates only over manifest entries assigned to its shard (`hash(slug) % TOTAL_SHARDS == SHARD_ID`).
   - Downloads each file via **CDN bypass** (`https://huggingface.co/datasets/.../resolve/main/...`) with no Authorization header.
   - Projects to `{prompt, response}` only at parse time; never loads full heterogeneous parquet schema.
   - Writes `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl` with deterministic slug → repo mapping for commit-cap spreading (5 sibling repos).

3. **Update `.github/workflows/ingest.yml`** — matrix job that:
   - Runs a single “prepare” step (Mac/lightweight) to generate `manifest.json` and upload as artifact.
   - Passes `manifest.json` to each shard runner.
   - Keeps 16-shard parallelism but removes per-shard HF API calls during ingestion.

4. **Add `lib/cdn.py`** — small helper with:
   - `download_via_cdn(path)` using `requests.get` + streaming.
   - Retry/backoff for CDN failures (separate from API 429).
   - `project_to_pair(raw_bytes, file_ext)` handling parquet/jsonl/csv with schema projection and safe casting.

5. **Update `requirements.txt`** — add `requests`, keep `pyarrow`, `numpy`, `pandas`, `datasets` (only for parquet reading, not `load_dataset`).

### Code Snippets

#### `bin/manifest.py`
```python
#!/usr/bin/env python3
"""
Generate manifest for today's folder in axentx/surrogate-1-training-pairs.
Run on Mac (or CI prepare step) once per ingestion cycle.
"""
import json, os, hashlib, datetime, sys
from huggingface_hub import HfApi

REPO = "datasets/axentx/surrogate-1-training-pairs"
DATE = datetime.datetime.utcnow().strftime("%Y-%m-%d")
OUT = "manifest.json"

def main():
    api = HfApi(token=os.getenv("HF_TOKEN"))
    # Single API call: non-recursive tree for today
    tree = api.list_repo_tree(repo_id=REPO, path=DATE, recursive=False)
    entries = []
    for item in tree:
        if item.type != "file":
            continue
        entries.append({
            "date": DATE,
            "slug": item.path.split("/")[-1].replace(".parquet", "").replace(".jsonl", ""),
            "path": f"{DATE}/{item.path.split('/')[-1]}",
            "size": item.size or 0,
        })
    manifest = {"date": DATE, "entries": entries}
    with open(OUT, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Wrote {len(entries)} entries to {OUT}")

if __name__ == "__main__":
    main()
```

#### `lib/cdn.py`
```python
import io, json, pyarrow.parquet as pq, pyarrow as pa, pandas as pd, requests, os, hashlib
from typing import Dict, Any, Optional

CDN_ROOT = "https://huggingface.co/datasets/axentx/surrogate-1-training-pairs/resolve/main"

def download_via_cdn(path: str, max_retries: int = 3) -> bytes:
    url = f"{CDN_ROOT}/{path}"
    for attempt in range(max_retries):
        try:
            resp = requests.get(url, stream=True, timeout=(5, 30))
            resp.raise_for_status()
            buf = io.BytesIO()
            for chunk in resp.iter_content(chunk_size=8192):
                buf.write(chunk)
            return buf.getvalue()
        except Exception as e:
            if attempt == max_retries - 1:
                raise
            continue
    raise RuntimeError("unreachable")

def project_to_pair(raw: bytes, path: str) -> Optional[Dict[str, str]]:
    """Return {prompt, response} or None if invalid."""
    try:
        if path.endswith(".parquet"):
            tbl = pq.read_table(io.BytesIO(raw), columns=["prompt", "response"])
            df = tbl.to_pandas()
        elif path.endswith(".jsonl"):
            df = pd.read_json(io.BytesIO(raw), lines=True)
        else:
            # fallback: try json/csv
            if raw.strip().startswith(b"{"):
                df = pd.read_json(io.BytesIO(raw), lines=True)
            else:
                df = pd.read_csv(io.BytesIO(raw))
        # Ensure columns exist
        if "prompt" not in df.columns or "response" not in df.columns:
            return None
        row = df.iloc[0]  # one file → one pair for surrogate-1 ingestion pattern
        return {"prompt": str(row["prompt"]), "response": str(row["response"])}
    except Exception:
        return None
```

    return int(hashlib.md5(slug.encode()).hexdigest(), 16) % TOTAL_SHARDS
