# Costinel / ops

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

### What we ship
- A **non-blocking Top-Hub Signal Panel** mounted on the Costinel dashboard (sidebar or top banner area).
- Defaults to hub **MOC** (configurable via `VITE_HUB_NAME=MOC`).
- Shows: hub title, short description, top 3 signals (anomalies/recommendations), and a “View in Knowledge Graph” link.
- **CDN-first data strategy**: file list fetched once from Mac (outside rate-limit), embedded as JSON in build; panel loads data via CDN URLs (`/resolve/main/...`) with zero API calls during runtime.
- Non-blocking: panel failure does not block dashboard; graceful fallback to cached/embedded data.

### Why this is highest-value (<2h)
- Applies the **top-hub doc insight** pattern (MOC) immediately.
- Uses **CDN bypass** pattern to avoid HF API rate limits during runtime.
- Reuses existing dashboard layout (low UI friction).
- Delivers contextual ops signals without touching execution paths (fits “Sense + Signal — ไม่ Execute”).

## Concrete Implementation Steps

### 1) Add environment/config
```bash
# .env or Vite config
VITE_HUB_NAME=MOC
VITE_HUB_REPO=axentx/hub-signals          # public dataset/repo
VITE_HUB_PATH=signals/moc                 # folder in repo
```

### 2) Mac-side: pre-list files and generate embed (run once or on schedule)
```bash
# scripts/generate-hub-filelist.sh
#!/usr/bin/env bash
set -euo pipefail

REPO="${HUB_REPO:-axentx/hub-signals}"
PATH_PREFIX="${HUB_PATH:-signals/moc}"
OUT="src/data/hub-filelist.json"

# Requires HF_TOKEN in env for list_repo_tree (only once)
node - <<NODE
import { HfApi } from "huggingface.js";
const api = new HfApi({ token: process.env.HF_TOKEN });
(async () => {
  const tree = await api.listRepoTree({
    repo: "$REPO",
    path: "$PATH_PREFIX",
    recursive: false
  });
  const files = tree
    .filter((t) => t.type === "file")
    .map((t) => t.path)
    .filter((p) => p.endsWith(".json") || p.endsWith(".parquet"));
  console.log(JSON.stringify({ repo: "$REPO", path: "$PATH_PREFIX", files, generatedAt: new Date().toISOString() }, null, 2));
})().catch((e) => { console.error(e); process.exit(1); });
NODE
> "$OUT"
```

Commit `src/data/hub-filelist.json` to repo (or inject at build time). This avoids runtime `list_repo_files` and rate limits.

### 3) Create TopHubSignalPanel component
`src/components/TopHubSignalPanel.tsx`
```tsx
import { useEffect, useState } from "react";
import "./TopHubSignalPanel.css";

type Signal = {
  id: string;
  title: string;
  summary: string;
  severity: "low" | "medium" | "high";
  cdnUrl?: string;
};

type FileList = {
  repo: string;
  path: string;
  files: string[];
  generatedAt: string;
};

export default function TopHubSignalPanel() {
  const hubName = import.meta.env.VITE_HUB_NAME || "MOC";
  const [fileList, setFileList] = useState<FileList | null>(null);
  const [signals, setSignals] = useState<Signal[]>([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  // Load embedded file list (generated by Mac script)
  useEffect(() => {
    fetch("/data/hub-filelist.json")
      .then((r) => r.json())
      .then(setFileList)
      .catch(() => setError("File list unavailable"));
  }, []);

  // CDN-first fetch: pick latest N signal files and fetch via CDN (no auth)
  useEffect(() => {
    if (!fileList?.files?.length) return;

    const signalFiles = fileList.files
      .filter((f) => f.endsWith(".json"))
      .sort()
      .reverse()
      .slice(0, 3);

    if (signalFiles.length === 0) {
      setLoading(false);
      return;
    }

    // Fetch via CDN (no Authorization header) — bypasses HF API rate limits
    Promise.all(
      signalFiles.map(async (filePath) => {
        const cdnUrl = `https://huggingface.co/datasets/${fileList.repo}/resolve/main/${filePath}`;
        try {
          const res = await fetch(cdnUrl, { cache: "no-store" });
          if (!res.ok) throw new Error(`CDN fetch failed: ${res.status}`);
          const json = await res.json();
          return {
            id: filePath,
            title: json.title || json.name || "Signal",
            summary: json.summary || json.description || "",
            severity: json.severity || "medium",
            cdnUrl,
          } as Signal;
        } catch (e) {
          console.warn("Failed CDN fetch for", filePath, e);
          return null;
        }
      })
    )
      .then((results) => setSignals(results.filter(Boolean) as Signal[]))
      .catch((e) => setError(String(e)))
      .finally(() => setLoading(false));
  }, [fileList]);

  if (loading && signals.length === 0 && !error) {
    return (
      <div className="hub-panel hub-panel-loading">
        <span>Loading {hubName} signals…</span>
      </div>
    );
  }

  return (
    <div className="hub-panel">
      <div className="hub-panel-header">
        <strong>{hubName}</strong>
        <span className="hub-panel-sub">Top signals</span>
      </div>

      {error && <div className="hub-panel-error">{error}</div>}

      <div className="hub-signals">
        {signals.map((s) => (
          <div key={s.id} className={`hub-signal severity-${s.severity}`}>
            <div className="hub-signal-title">{s.title}</div>
            <div className="hub-signal-summary">{s.summary}</div>
            {s.cdnUrl && (
              <a
                href={s.cdnUrl}
                target="_blank"
                rel="noopener noreferrer"
                className="hub-signal-link"
              >
                View raw
              </a>
            )}
          </div>
        ))}
      </div>

      <div className="hub-panel-footer">
        <a
          href={`https://huggingface.co/datasets/${fileList?.repo || "axentx/hub-signals"}/tree/main/${fileList?.path || "signals/moc"}`}
          target="_blank"
          rel="noopener noreferrer"
        >
          View in Knowledge Graph
        </a>
      </div>
    </div>
  );
}
```
