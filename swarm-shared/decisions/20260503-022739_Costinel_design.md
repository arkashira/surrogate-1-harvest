# Costinel / design

## Final Synthesis — Top-Hub Signal Panel (Costinel)

**Chosen approach**: A **read-only, CDN-first Signal Panel** with a **lightweight backend proxy** for resilience and future flexibility.  
- Combines Candidate 2’s speed (CDN-first, no auth/rate-limits) with Candidate 1’s reliability (backend fallback + health check).  
- Avoids direct production CDN dependency from the browser while preserving zero-runtime-cost during dev.  
- Fully implements decision “Top-hub signal panel” (20260503-022544) and aligns with knowledge-rag pipeline outputs.

---

## Architecture (resolved)

- **Data source**: Precomputed `hubs/{hubId}.signals.json` produced by knowledge-rag pipeline and deployed to CDN.  
- **Dev path**: `public/data/hubs/{hubId}/signals.json` (static, committed sample).  
- **Prod path**: CDN URL (e.g., `https://cdn.axentx.io/data/hubs/{hubId}/signals.json`).  
- **Backend**: Thin proxy endpoint `/api/hubs/:hubId/signals` that:
  - Tries CDN first (prod) or local static file (dev).
  - Falls back to local file if CDN fails.
  - Returns normalized shape and 404 if unavailable.
  - Requires no auth (public dashboard) and is read-only.
- **Frontend**: `TopHubSignalPanel` (React) fetches `/api/hubs/:hubId/signals`, renders top 3 proposals, with loading/empty/error states.

## Backend implementation

`server.js` (Express, CDN-first with local fallback)

```js
// server.js
const express = require('express');
const fs = require('fs');
const path = require('path');
const fetch = require('node-fetch');

const app = express();
const PORT = process.env.PORT || 3000;
const LOCAL_DATA_ROOT = path.join(__dirname, 'public', 'data', 'hubs');
const CDN_BASE = process.env.CDN_BASE_URL || 'https://cdn.axentx.io/data/hubs';

function safeHubId(hubId) {
  return String(hubId || 'MOC').replace(/[^a-zA-Z0-9_-]/g, '');
}

async function fetchFromCDN(hubId) {
  const url = `${CDN_BASE}/${encodeURIComponent(hubId)}/signals.json?ts=${Date.now()}`;
  const res = await fetch(url, { timeout: 4000 });
  if (!res.ok) throw new Error(`CDN ${res.status}`);
  return res.json();
}

function fetchLocal(hubId) {
  const filePath = path.join(LOCAL_DATA_ROOT, hubId, 'signals.json');
  if (!fs.existsSync(filePath)) return null;
  return JSON.parse(fs.readFileSync(filePath, 'utf8'));
}

function normalize(raw) {
  if (!raw) return null;
  return {
    hubId: raw.hubId || 'MOC',
    title: raw.title || raw.hubId || 'MOC',
    description: raw.description || '',
    updatedAt: raw.updatedAt || null,
    proposals: Array.isArray(raw.proposals) ? raw.proposals.slice(0, 3) : []
  };
}

app.get('/api/hubs/:hubId/signals', async (req, res) => {
  const hubId = safeHubId(req.params.hubId || 'MOC');
  let payload = null;
  let source = 'none';

  // Prefer CDN in prod-like environments; fallback to local
  try {
    if (process.env.NODE_ENV === 'production' || CDN_BASE) {
      payload = await fetchFromCDN(hubId);
      source = 'cdn';
    }
  } catch (err) {
    console.warn('CDN unavailable, falling back to local:', err.message);
  }

  if (!payload) {
    payload = fetchLocal(hubId);
    source = 'local';
  }

  const normalized = normalize(payload);
  if (!normalized || !normalized.proposals.length) {
    return res.status(404).json({ error: 'Hub signals not found', hubId, source });
  }

  res.json({ ...normalized, source });
});

app.get('/health', (req, res) => res.json({ status: 'ok' }));

app.listen(PORT, () => {
  console.log(`Costinel API listening on ${PORT}`);
});
```

Notes:
- Uses `node-fetch` (install if needed).  
- Timeouts and safe fallback prevent blocking UI.  
- `source` field helps debugging.
