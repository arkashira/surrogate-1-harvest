# workio / backend

## Final Synthesized Answer

**Diagnosis (merged, prioritized by severity)**
- **Non-idempotent webhook ingestion** is the root cause: LINE retries on 5xx/timeouts replay clock/leave/OT events and can double-count or corrupt state.
- **No transactional deduplication** (no idempotency key or de-dup window) → race conditions on retries produce duplicate punches, leave/OT approvals, or lost events.
- Missing **structured audit trail** and **centralized error handling/retry budget** → transient failures leak 200 OK and lose events; compliance visibility is weak.
- Frontend lacks real-time channel (not fixed here) → changes remain invisible until refresh; this is acknowledged but out of scope for the backend-only fix.

**Proposed change (highest-value backend-only)**
Add an **idempotent webhook ingestion layer** for LINE events:
- New files: `server/src/routes/lineWebhook.ts`, `server/src/middleware/idempotency.ts`, `server/src/utils/line.ts`.
- DB additions: `webhook_events` table (unique `event_id`, `tenant_id`, `processed_at`, `status`, `payload_hash`) and idempotency guards on `clock_events`/`leave_requests`/`ot_requests`.
- Scope: POST `/webhook/line` only. This prevents duplicates, makes retries safe, and lays groundwork for subscriptions later.

**Implementation (merged strongest + concrete + corrected)**

### Idempotency middleware (`server/src/middleware/idempotency.ts`)
```ts
import { Request, Response, NextFunction } from 'express';
import { pool } from '../db';
import crypto from 'crypto';

function payloadHash(obj: any): string {
  return crypto.createHash('sha256').update(JSON.stringify(obj)).digest('hex');
}

export async function lineIdempotency(
  req: Request,
  res: Response,
  next: NextFunction
) {
  const events = req.body?.events;
  if (!Array.isArray(events) || events.length === 0) return next();

  const client = await pool.connect();
  try {
    await client.query('BEGIN');

    // Accept a stable event identifier from LINE when available; fallback to deterministic local id
    const seen: string[] = [];
    for (const ev of events) {
      // Prefer LINE's event id if provided; otherwise build deterministic id from source+type+timestamp
      const lineEventId = ev?.eventId || (ev?.source?.userId && ev?.type && ev?.timestamp
        ? `${ev.source.userId}:${ev.type}:${ev.timestamp}`
        : null);

      if (!lineEventId) {
        // If we cannot identify the event, skip dedupe but allow processing (avoid blocking)
        continue;
      }

      const hash = payloadHash(ev);
      const { rows } = await client.query(
        `INSERT INTO webhook_events (event_id, tenant_id, event_type, payload_hash, status)
         VALUES ($1, $2, $3, $4, 'processed')
         ON CONFLICT (event_id, tenant_id) DO UPDATE
         SET payload_hash = EXCLUDED.payload_hash,
             updated_at = now()
         WHERE webhook_events.payload_hash <> EXCLUDED.payload_hash
         RETURNING id, (xmax = 0) AS inserted`,
        [lineEventId, req.tenantId || process.env.DEFAULT_TENANT_ID, ev.type || 'unknown', hash]
      );

      const inserted = rows[0]?.inserted === true;
      if (!inserted) {
        // Duplicate - skip processing but acknowledge
        continue;
      }
      seen.push(lineEventId);
    }

    // Attach metadata for downstream handlers
    (req as any)._lineIdempotency = { seen };
    await client.query('COMMIT');
    next();
  } catch (err) {
    await client.query('ROLLBACK');
    next(err);
  } finally {
    client.release();
  }
}
```

### Webhook route (`server/src/routes/lineWebhook.ts`)
```ts
import { Router } from 'express';
import { lineIdempotency } from '../middleware/idempotency';
import { verifyLineSignature } from '../utils/line';
import { handleClockEvent } from '../services/clockService';
import { handleLeaveEvent } from '../services/leaveService';
import { handleOtEvent } from '../services/otService';

const router = Router();

router.post(
  '/line',
  verifyLineSignature,
  lineIdempotency,
  async (req, res) => {
    const events = req.body?.events || [];
    const results: any[] = [];

    for (const ev of events) {
      try {
        // Skip duplicates already filtered by middleware (optional extra guard)
        switch (ev.type) {
          case 'clock':
            results.push(await handleClockEvent(ev, req.tenantId));
            break;
          case 'leave_request':
            results.push(await handleLeaveEvent(ev, req.tenantId));
            break;
          case 'ot_request':
            results.push(await handleOtEvent(ev, req.tenantId));
            break;
          default:
            // ignore unknown but acknowledge
            results.push({ status: 'ignored', type: ev.type });
        }
      } catch (err) {
        // Do not throw: we want to process remaining events and return 200 for LINE retries budget
        console.error('Webhook event processing failed', { event: ev.type, err });
        results.push({ status: 'failed', type: ev.type, error: String(err) });
      }
    }

    // Always return 200 to prevent LINE retries; rely on idempotency for replays
    res.json({ ok: true, processed: results.length, results });
  }
);

export default router;
```

### Signature verification util (`server/src/utils/line.ts`)
```ts
import crypto from 'crypto';

export function verifyLineSignature(req: any, res: any, next: any) {
  const signature = req.headers['x-line-signature'];
  const body = JSON.stringify(req.body);
  const channelSecret = process.env.LINE_CHANNEL_SECRET;

  if (!channelSecret) {
    return res.status(500).json({ error: 'LINE_CHANNEL_SECRET missing' });
  }

  const hash = crypto
    .createHmac('sha256', channelSecret)
    .update(body, 'utf8')
    .digest('base64');

  if (hash !== signature) {
    return res.status(401).json({ error: 'Invalid signature' });
  }
  next();
}
```
