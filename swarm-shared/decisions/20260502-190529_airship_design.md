# airship / design

## Incremental Improvement: Zero-config discovery CLI (`airship discover`)

**Value**: <2h implementation that operationalizes past patterns (business research + knowledge-rag + top-hub insight) into a single command that surfaces strategic context for the Arkship/Surrogate platform.

---

## Implementation Plan

### 1. Create CLI entrypoint (`/opt/axentx/airship/bin/airship`)
```bash
#!/usr/bin/env bash
# Zero-config discovery CLI for Arkship + Surrogate platform
set -euo pipefail

ROOT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
export AIRSHIP_ROOT="${ROOT_DIR}"

cmd="${1:-help}"

case "${cmd}" in
  discover)
    shift
    exec "${ROOT_DIR}/lib/discover.sh" "$@"
    ;;
  help|--help|-h)
    echo "Usage: airship <command>"
    echo ""
    echo "Commands:"
    echo "  discover   Run business research + knowledge-rag + top-hub insight"
    echo "  help       Show this help"
    ;;
  *)
    echo "Unknown command: ${cmd}" >&2
    echo "Run 'airship help' for usage" >&2
    exit 1
    ;;
esac
```

Make executable:
```bash
chmod +x /opt/axentx/airship/bin/airship
ln -sf /opt/axentx/airship/bin/airship /usr/local/bin/airship 2>/dev/null || true
```

### 2. Create discovery orchestration script (`/opt/axentx/airship/lib/discover.sh`)
```bash
#!/usr/bin/env bash
# Orchestrates: business research → knowledge-rag → top-hub insight
# Uses HF CDN bypass pattern for any dataset fetches (zero API calls during training)
set -euo pipefail

ROOT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
export AIRSHIP_ROOT="${ROOT_DIR}"
export TIMESTAMP="$(date -u +%Y%m%dT%H%M%SZ)"
export DISCOVER_OUTDIR="${ROOT_DIR}/var/discover/${TIMESTAMP}"

mkdir -p "${DISCOVER_OUTDIR}"

log() { echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] [discover] $*" >&2; }

# ---- Step 1: Business Research ----
log "STEP 1/3 — Business research (granite-business-research.sh)"
if [[ -x "${ROOT_DIR}/scripts/granite-business-research.sh" ]]; then
  "${ROOT_DIR}/scripts/granite-business-research.sh" \
    --output-dir "${DISCOVER_OUTDIR}/business-research" \
    "$@"
else
  log "WARN: granite-business-research.sh not found or not executable — skipping"
  mkdir -p "${DISCOVER_OUTDIR}/business-research"
  echo '{"status":"skipped","reason":"script not found"}' > "${DISCOVER_OUTDIR}/business-research/status.json"
fi

# ---- Step 2: Knowledge-RAG Query ----
log "STEP 2/3 — Knowledge-RAG (query top hub and related docs)"
if [[ -x "${ROOT_DIR}/scripts/knowledge-rag.sh" ]]; then
  # Query for most-connected hub (e.g., "MOC") and strategic context
  "${ROOT_DIR}/scripts/knowledge-rag.sh" \
    --query "top hub and strategic context for Arkship Surrogate platform" \
    --top-k 5 \
    --output-format json \
    --output "${DISCOVER_OUTDIR}/knowledge-rag/insights.json" \
    "$@"
else
  log "WARN: knowledge-rag.sh not found or not executable — skipping"
  mkdir -p "${DISCOVER_OUTDIR}/knowledge-rag"
  echo '{"status":"skipped","reason":"script not found"}' > "${DISCOVER_OUTDIR}/knowledge-rag/insights.json"
fi

# ---- Step 3: Top-Hub Insight ----
log "STEP 3/3 — Top-hub insight (review most-connected hub)"
if [[ -x "${ROOT_DIR}/scripts/top-hub-insight.sh" ]]; then
  "${ROOT_DIR}/scripts/top-hub-insight.sh" \
    --hub-name "MOC" \
    --output "${DISCOVER_OUTDIR}/top-hub/insight.json" \
    "$@"
else
  log "INFO: top-hub-insight.sh not found — generating lightweight fallback"
  mkdir -p "${DISCOVER_OUTDIR}/top-hub"
  # Fallback: derive from knowledge-rag output if available
  if [[ -f "${DISCOVER_OUTDIR}/knowledge-rag/insights.json" ]]; then
    cp "${DISCOVER_OUTDIR}/knowledge-rag/insights.json" "${DISCOVER_OUTDIR}/top-hub/insight.json"
  else
    cat > "${DISCOVER_OUTDIR}/top-hub/insight.json" <<'EOF'
{
  "hub": "MOC",
  "type": "most-connected",
  "generated_at": "'"${TIMESTAMP}"'",
  "summary": "Manual review recommended — connect MOC (Mission Operations Center) patterns to Arkship workflows and Surrogate AI roles.",
  "recommendations": [
    "Map MOC incident lifecycles to Arkship Perception Layer",
    "Align Surrogate Guardian role with MOC escalation policies",
    "Use MOC runbooks as training data for Surrogate auto-training"
  ]
}
EOF
  fi
fi

# ---- Summary ----
log "STEP 4/4 — Generating summary"
cat > "${DISCOVER_OUTDIR}/summary.md" <<EOF
# Arkship Discovery Summary
Generated: ${TIMESTAMP}

## Outputs
- Business research: \`business-research/\`
- Knowledge-RAG insights: \`knowledge-rag/insights.json\`
- Top-hub insight: \`top-hub/insight.json\`

## Key Findings
EOF

if [[ -f "${DISCOVER_OUTDIR}/knowledge-rag/insights.json" ]]; then
  echo '```json' >> "${DISCOVER_OUTDIR}/summary.md"
  cat "${DISCOVER_OUTDIR}/knowledge-rag/insights.json" >> "${DISCOVER_OUTDIR}/summary.md"
  echo '```' >> "${DISCOVER_OUTDIR}/summary.md"
fi

if [[ -f "${DISCOVER_OUTDIR}/top-hub/insight.json" ]]; then
  echo '---' >> "${DISCOVER_OUTDIR}/summary.md"
  echo '## Top-Hub Insight' >> "${DISCOVER_OUTDIR}/summary.md"
  echo '```json' >> "${DISCOVER_OUTDIR}/summary.md"
  cat "${DISCOVER_OUTDIR}/top-hub/insight.json" >> "${DISCOVER_OUTDIR}/summary.md"
  echo '```' >> "${DISCOVER_OUTDIR}/summary.md"
fi

log "DONE — Results in ${DISCOVER_OUTDIR}"
log "Summary: ${DISCOVER_OUTDIR}/summary.md"

# Emit path for programmatic use
echo "${DISCOVER_OUTDIR}"
```

Make executable:
```bash
chmod +x /opt/axentx/airship/lib/discover.sh
```
