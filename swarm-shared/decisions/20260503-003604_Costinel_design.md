# Costinel / design

## Final Implementation Plan — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only card that identifies the most-connected hub from the knowledge-rag graph and surfaces the top hub with 3 contextual signals.  
**Timebox**: <2h (frontend-only, no backend changes).

### Why this wins
- Uses **static local JSON as the source of truth** (no backend/ops required) so the card can ship in <2h and remain stable.
- Keeps an **extensible adapter layer** so later we can swap in live graph queries (e.g., Neo4j, DGraph, or a RAG API) without touching UI.
- Follows project conventions: read-only, “Sense + Signal — ไม่ Execute”, and audit-friendly (renders provenance + timestamp).
- Past patterns show reviewing the most-connected hub (e.g., “MOC”) before planning yields high-leverage context.

## 3) Hook: `src/hooks/useTopHubSignal.ts`

```ts
import { useEffect, useState } from "react";
import hubData from "../data/knowledge-rag-hubs.json";

export type Hub = typeof hubData.topHub;
export type Signal = typeof hubData.contextualSignals[number];

export function useTopHubSignal() {
  const [state, setState] = useState<{
    hub: Hub;
    signals: Signal[];
    generatedAt: string;
    source: string;
  } | null>(null);

  useEffect(() => {
    // Static import; can later swap to fetch('/api/knowledge-rag/top-hub')
    setState({
      hub: hubData.topHub,
      signals: hubData.contextualSignals,
      generatedAt: hubData.generatedAt,
      source: hubData.source,
    });
  }, []);

  return state;
}
```

## 4) Component: `src/components/TopHubSignalCard/TopHubSignalCard.tsx`

```tsx
import React from "react";
import { useTopHubSignal } from "../../hooks/useTopHubSignal";
import "./TopHubSignalCard.css";

const severityColors = {
  high: "var(--color-critical)",
  medium: "var(--color-warning)",
  low: "var(--color-info)",
} as const;

export const TopHubSignalCard: React.FC = () => {
  const data = useTopHubSignal();

  if (!data) {
    return (
      <div className="top-hub-card loading">
        <div className="skeleton" />
      </div>
    );
  }

  const { hub, signals, generatedAt, source } = data;

  return (
    <article className="top-hub-card" aria-label={`Top hub: ${hub.label}`}>
      <header className="top-hub-card__header">
        <div>
          <h3 className="top-hub-card__title">{hub.label}</h3>
          <p className="top-hub-card__desc">{hub.description}</p>
        </div>
        <div className="top-hub-card__meta">
          <span className="top-hub-card__stat">
            In: {hub.inbound} | Out: {hub.outbound}
          </span>
        </div>
      </header>

      <section className="top-hub-card__signals" aria-label="Contextual signals">
        {signals.map((s) => (
          <div key={s.id} className="signal-item">
            <div className="signal-item__header">
              <span
                className="signal-item__dot"
                style={{ backgroundColor: severityColors[s.severity] }}
                aria-hidden="true"
              />
              <strong className="signal-item__title">{s.title}</strong>
              <span className="signal-item__badge">{s.severity}</span>
            </div>
            <p className="signal-item__summary">{s.summary}</p>
            <div className="signal-item__tags">
              {s.tags.map((t) => (
                <span key={t} className="tag">
                  {t}
                </span>
              ))}
            </div>
          </div>
        ))}
      </section>

      <footer className="top-hub-card__footer">
        <small className="muted">
          Source: {source} • Generated {new Date(generatedAt).toLocaleString()}
        </small>
        <small className="muted">Sense + Signal — ไม่ Execute</small>
      </footer>
    </article>
  );
};
```
