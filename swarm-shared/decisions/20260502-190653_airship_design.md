# Airship — Design: `airship discover` CLI (Zero-config strategic context)

**Estimated effort**: ~90–110 min  
**Value**: One command that operationalizes past patterns (business research + knowledge-rag + top-hub insight) and surfaces strategic context for Arkship/Surrogate platform decisions.

---

## Implementation Plan

1. **Create CLI entrypoint**  
   - `/opt/axentx/airship/bin/airship` (executable) — thin dispatcher.
   - `/opt/axentx/airship/commands/discover.py` — implements `discover` subcommand.

2. **Implement `discover` behavior**  
   - Step A: Run `granite-business-research.sh` (if present) or fallback to lightweight market scan.
   - Step B: Invoke `knowledge-rag` CLI/query to extract top hub and related docs.
   - Step C: Print concise top-hub insight (e.g., MOC) and suggested next actions.
   - Step D: Exit 0 on success; non-zero with clear diagnostics on failure.

3. **Ensure script hygiene** (apply past lessons)  
   - Use `#!/usr/bin/env bash` for any wrapper scripts.
   - Ensure executables: `chmod +x`.
   - Avoid assumptions about environment; set `SHELL=/bin/bash` in any cron usage.

4. **Add to repo README**  
   - One-line usage and example output.

### `/opt/axentx/airship/bin/airship`
```bash
#!/usr/bin/env bash
# Minimal dispatcher for airship CLI
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "$SCRIPT_DIR/.." && pwd)"

export AIRSHIP_ROOT="$PROJECT_ROOT"

case "${1:-}" in
  discover)
    shift
    exec python3 "$PROJECT_ROOT/commands/discover.py" "$@"
    ;;
  *)
    echo "Usage: airship <command>"
    echo ""
    echo "Commands:"
    echo "  discover   Run zero-config discovery (business research + knowledge-rag + top-hub insight)"
    exit 1
    ;;
esac
```

### `/opt/axentx/airship/commands/discover.py`
```python
#!/usr/bin/env python3
"""
airship discover
Zero-config discovery: business research + knowledge-rag + top-hub insight.
"""
import json
import os
import subprocess
import sys
from pathlib import Path

AIRSHIP_ROOT = Path(os.getenv("AIRSHIP_ROOT", Path(__file__).parent.parent.parent))

def run(cmd, cwd=None, capture=True):
    """Run command and return (stdout, stderr, rc)."""
    try:
        result = subprocess.run(
            cmd,
            shell=False,
            cwd=cwd or AIRSHIP_ROOT,
            capture_output=capture,
            text=True,
            timeout=300,
        )
        return result.stdout, result.stderr, result.returncode
    except subprocess.TimeoutExpired:
        return "", "command timed out", 1
    except Exception as e:
        return "", str(e), 1

def run_granite_business_research():
    """Run granite-business-research.sh if available."""
    script = AIRSHIP_ROOT / "scripts" / "granite-business-research.sh"
    if script.is_file():
        print("[discover] Running business research...")
        stdout, stderr, rc = run([str(script)], cwd=AIRSHIP_ROOT)
        if rc != 0:
            print(f"[discover] Business research exited {rc}: {stderr.strip() or stdout.strip()}")
        else:
            print("[discover] Business research completed.")
        return rc == 0
    else:
        print("[discover] No granite-business-research.sh found — skipping business research step.")
        return False

def run_knowledge_rag():
    """Run knowledge-rag query for top hub and related docs."""
    # Try common knowledge-rag CLI patterns
    candidates = [
        AIRSHIP_ROOT / "scripts" / "knowledge-rag.sh",
        AIRSHIP_ROOT / "bin" / "knowledge-rag",
        AIRSHIP_ROOT / "knowledge-rag" / "cli.py",
    ]
    cli = next((c for c in candidates if c.is_file()), None)

    if cli is None:
        print("[discover] knowledge-rag CLI not found — cannot query top hub.")
        return None

    print("[discover] Querying knowledge-rag for top hub and related docs...")
    if cli.suffix == ".py":
        stdout, stderr, rc = run([sys.executable, str(cli), "top-hub", "--json"], cwd=AIRSHIP_ROOT)
    else:
        stdout, stderr, rc = run([str(cli), "top-hub", "--json"], cwd=AIRSHIP_ROOT)

    if rc != 0:
        print(f"[discover] knowledge-rag query failed ({rc}): {stderr.strip() or stdout.strip()}")
        return None

    try:
        return json.loads(stdout.strip())
    except Exception:
        # fallback: return raw text
        return {"raw": stdout.strip()}

def format_insight(rag_result):
    """Produce concise top-hub insight."""
    if not rag_result:
        return "No knowledge-rag result available."

    if "raw" in rag_result:
        lines = [l for l in rag_result["raw"].splitlines() if l.strip()][:6]
        return "Top hub (raw):\n  " + "\n  ".join(lines)

    hub = rag_result.get("hub") or rag_result.get("top_hub") or rag_result.get("name") or "Unknown"
    summary = rag_result.get("summary") or rag_result.get("description") or ""
    related = rag_result.get("related", [])[:3]

    out = [f"Top hub: {hub}"]
    if summary:
        out.append(f"Summary: {summary}")
    if related:
        out.append("Related docs/tools:")
        for r in related:
            if isinstance(r, str):
                out.append(f"  - {r}")
            elif isinstance(r, dict):
                out.append(f"  - {r.get('title', r.get('name', 'item'))}")
    return "\n".join(out)

def main():
    print("=== airship discover ===")
    print("Running zero-config discovery (business research + knowledge-rag + top-hub insight)")
    print("")

    # Step A: business research
    run_granite_business_research()

    # Step B: knowledge-rag query
    rag_result = run_knowledge_rag()

    # Step C: print insight
    print("")
    print("--- Top-hub Insight ---")
    print(format_insight(rag_result))
    print("")
    print("Suggested next actions:")
    print("  - Review the top hub (e.g., MOC) for current priorities.")
    print("  - Use knowledge-rag to explore related docs before planning tasks.")
    print("  - If running cron-based wrappers (opus-pr-reviewer/active-learning), ensure:")
    print("      * Bash shebang (#!/usr/bin/env bash)")
    print("      * Executable (chmod +x)")
    print("      * SHELL=/bin/bash in crontab")
    print("")
    return 0

if __name__ == "__main__":
    sys.exit(main())
```
