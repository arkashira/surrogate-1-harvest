# surrogate-1 / ops

## Implementation Plan (≤2h)

**Highest-value incremental improvement**: Replace runtime `load_dataset(streaming=True)` + recursive `list_repo_files` in `bin/dataset-enrich.sh` with a deterministic pre-flight snapshot + CDN-only fetches. This eliminates HF API rate-limit risk (429), removes per-shard recursive listing, and aligns with the CDN bypass pattern.

### Steps (est. 90–120 min)

1. Add `bin/list-snapshot.sh` — runs on Mac (or once per cron tick) to call `list_repo_tree(path, recursive=False)` for the target date folder and emit `snapshot.json` (array of `{"path":..., "sha":...}`). Commit or pass to workflow via artifact.
2. Update `bin/dataset-enrich.sh`:
   - Accept snapshot file path as optional arg (default: fetch via `list-snapshot.sh` if not provided).
   - Remove `load_dataset(streaming=True)` and recursive `list_repo_files`.
   - Iterate snapshot entries; for each file, download via CDN URL `https://huggingface.co/datasets/axentx/surrogate-1-training-pairs/resolve/main/<path>` with `curl`/`requests` (no auth header).
   - Keep existing per-schema projection to `{prompt, response}`, dedup via `lib/dedup.py`, and output shard file.
3. Update `.github/workflows/ingest.yml`:
   - Add a one-time job step (or separate workflow) to generate `snapshot.json` and upload as artifact for the matrix jobs.
   - Pass `snapshot.json` into each shard job (or regenerate per shard using deterministic date + SHARD_ID if simpler).
4. Add retry/backoff for CDN downloads (exponential, max 5 retries) and respect 429 on CDN tier with 30–60s sleep.
5. Remove any `datasets` streaming usage from the worker; keep only `pyarrow`/`pandas` for local parquet/jsonl parsing.
6. Test locally with a small date folder to verify schema projection and dedup behavior.

### 1) bin/list-snapshot.sh
```bash
#!/usr/bin/env bash
# Usage: bin/list-snapshot.sh <date-folder> [output.json]
# Example: bin/list-snapshot.sh 2026-05-02 snapshot.json
# Requires: huggingface_hub (python) installed in the runner
set -euo pipefail

REPO="axentx/surrogate-1-training-pairs"
DATE_FOLDER="${1:-$(date +%Y-%m-%d)}"
OUTPUT="${2:-snapshot.json}"

python3 - "$REPO" "$DATE_FOLDER" "$OUTPUT" <<'PY'
import json
import sys
from huggingface_hub import HfApi

def main(repo_id: str, folder: str, out_path: str) -> None:
    api = HfApi()
    # Non-recursive listing for the date folder
    entries = api.list_repo_tree(repo_id=repo_id, path=folder, recursive=False)
    files = []
    for e in entries:
        if not e.path.endswith("/"):  # skip subfolders
            files.append({"path": e.path, "sha": getattr(e, "sha", None)})
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(files, f, indent=2)
    print(f"Wrote {len(files)} files to {out_path}")

if __name__ == "__main__":
    main(sys.argv[1], sys.argv[2], sys.argv[3])
PY

### 2) bin/dataset-enrich.sh (updated core loop)
```bash
#!/usr/bin/env bash
# Updated worker: uses snapshot + CDN-only fetches
# Usage: bin/dataset-enrich.sh <shard_id> <total_shards> [snapshot.json]
set -euo pipefail

SHARD_ID="${1:-0}"
TOTAL_SHARDS="${2:-16}"
SNAPSHOT="${3:-snapshot.json}"
REPO="axentx/surrogate-1-training-pairs"
DATE_STR=$(date +%Y-%m-%d)
OUT_DIR="batches/public-merged/${DATE_STR}"
mkdir -p "$OUT_DIR"
OUT_FILE="${OUT_DIR}/shard${SHARD_ID}-$(date +%H%M%S).jsonl"

# Dedup helper (existing)
DEDUPE_PY="lib/dedup.py"

# Temporary working dir
WORKDIR=$(mktemp -d)
cleanup() { rm -rf "$WORKDIR"; }
trap cleanup EXIT

# Select shard slice from snapshot deterministically
mapfile -t ALL_PATHS < <(python3 -c "
import json, sys
with open(sys.argv[1]) as f:
    items = json.load(f)
paths = [i['path'] for i in items]
paths.sort()
for p in paths:
    print(p)
" "$SNAPSHOT")

TOTAL_FILES=${#ALL_PATHS[@]}
if [[ $TOTAL_FILES -eq 0 ]]; then
  echo "No files in snapshot."
  exit 0
fi

# Assign files to shards by hash mod
for i in "${!ALL_PATHS[@]}"; do
  p="${ALL_PATHS[$i]}"
  # Deterministic shard assignment by path hash
  HASH=$(echo -n "$p" | sha256sum | awk '{print $1}')
  HASH_DEC=$((16#${HASH:0:8}))
  if (( HASH_DEC % TOTAL_SHARDS != SHARD_ID )); then
    continue
  fi

  # CDN download (no auth header)
  CDN_URL="https://huggingface.co/datasets/${REPO}/resolve/main/${p}"
  DEST="${WORKDIR}/$(basename "$p")"
  echo "Downloading ${p} -> ${DEST}"

  # Retry with exponential backoff
  for attempt in {1..5}; do
    if curl -fsSL --retry 3 --retry-delay 2 -o "$DEST" "$CDN_URL"; then
      break
    fi
    if [[ $attempt -eq 5 ]]; then
      echo "Failed to download ${p} after 5 attempts"
      continue 2
    fi
    sleep $(( 2 ** attempt ))
  done

  # Project to {prompt,response} using existing per-schema logic
  # (Replace this block with your actual projection script if different)
  python3 - "$DEST" "$WORKDIR/proj-$(basename "$p").jsonl" <<'PY'
import json, sys, os, pyarrow.parquet as pq, pyarrow as pa, pandas as pd

def project_file(in_path: str, out_path: str) -> None:
    ext = os.path.splitext(in_path)[1].lower()
    rows = []
    try:
        if ext == ".parquet":
            tbl = pq.read_table(in_path)
            df = tbl.to_pandas()
        elif ext in (".jsonl", ".json"):
            df = pd.read_json(in_path, lines=(ext == ".jsonl"))
        else:
            # fallback: try line-by-line json
            with open(in_path, encoding="utf-8") as f:
                for line in f:
                    line = line.strip()
                    if not line:
                        continue
                    try:
                        obj = json.loads(line)
                        rows.append(obj)
                    except Exception:
                        continue
            df = pd.DataFrame(rows) if rows else pd.DataFrame()
    except Exception as e:
        print(f"Parse error {in_path}: {e}", file=sys.stderr)
        return

    # Normalize to prompt/response (adjust keys per your schema)
    prompt_key = None
    response_key = None
    for k in df.columns:
        klow = k.lower()
        if "prompt" in klow:
            prompt_key = k
        if "response" in klow or "completion" in klow or "answer" in klow:
            response_key = k

    with open(out_path, "w", encoding="utf-8") as out:
        for _, row in df.iterrows():
            prompt = row.get(prompt_key, "") if prompt_key else ""
            response = row.get(response_key, "") if response_key else ""
            if prompt or response:
                out.write(json.dumps({"prompt": str(prompt), "response": str(response)}, ensure_ascii=False) + "\n")

if __name__ == "__main__":
    project_file(sys.argv[1], sys.argv[2])
PY
