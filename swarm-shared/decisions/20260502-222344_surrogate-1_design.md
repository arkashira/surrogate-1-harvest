# surrogate-1 / design

## Final Implementation Plan (≤2h)

**Highest-value improvement**: Add deterministic pre-flight file listing + CDN-only ingestion to eliminate HF API 429s during training and make shard workers resilient.

### What we’ll change
1. Add `bin/list-files.sh` — one-shot Mac/Linux script that calls `list_repo_tree` per date folder, saves `file-list-<date>.json` to `./cache/`. Embeds the list into the worker so training uses CDN-only fetches (zero API calls during data load).
2. Update `bin/dataset-enrich.sh` to accept an optional file-list JSON. If provided, workers iterate the local list and download via `https://huggingface.co/datasets/.../resolve/main/...` (CDN, no auth/rate-limit). Fallback to `hf_hub_download` if CDN fails.
3. Add `lib/cdn.py` helper: deterministic shard slicing (`slug-hash % 16 == SHARD_ID`), CDN download with retries, and projection to `{prompt, response}` only at parse time (avoids pyarrow CastError on mixed schemas).
4. Add `requirements-dev.txt` with `requests tqdm` and pin `huggingface-hub>=0.24`.

### 1) New: `bin/list-files.sh`
```bash
#!/usr/bin/env bash
# list-files.sh
# Usage: HF_TOKEN=... ./list-files.sh <repo> <date_folder> [output_dir]
# Example: ./list-files.sh axentx/surrogate-1-training-pairs 2026-05-02 cache

set -euo pipefail
REPO="${1:-axentx/surrogate-1-training-pairs}"
DATE="${2:-$(date +%Y-%m-%d)}"
OUTDIR="${3:-cache}"

mkdir -p "$OUTDIR"
OUTFILE="${OUTDIR}/file-list-${DATE}.json"

# Requires: huggingface_hub
python3 - "$REPO" "$DATE" "$OUTFILE" <<'PY'
import json, os, sys
from huggingface_hub import HfApi

repo = sys.argv[1]
date = sys.argv[2]
outfile = sys.argv[3]
api = HfApi()

folders = [
    f"batches/public-raw/{date}",
    f"batches/mirror-merged/{date}",
]

out = []
for folder in folders:
    try:
        items = api.list_repo_tree(repo=repo, path=folder, recursive=False)
    except Exception as e:
        print(f"Warning: {folder} -> {e}", file=sys.stderr)
        continue
    for it in items:
        if getattr(it, "type", None) == "file" or (hasattr(it, "path") and "." in it.path):
            out.append({
                "path": it.path,
                "size": getattr(it, "size", None),
                "sha256": getattr(it, "sha256", None),
            })

with open(outfile, "w") as f:
    json.dump(out, f, indent=2)
print(f"Wrote {len(out)} entries to {outfile}")
PY
```
Make executable:
```bash
chmod +x bin/list-files.sh
```

### 2) New: `lib/cdn.py`
```python
# lib/cdn.py
import hashlib
import json
import time
from pathlib import Path
from typing import Iterator, Tuple

import requests
from tqdm import tqdm

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def cdn_lines(repo: str, file_path: str, max_retries: int = 3, backoff: float = 2.0) -> Iterator[str]:
    url = CDN_TEMPLATE.format(repo=repo, path=file_path)
    for attempt in range(1, max_retries + 1):
        try:
            with requests.get(url, stream=True, timeout=30) as r:
                r.raise_for_status()
                for chunk in r.iter_lines(decode_unicode=True):
                    if chunk is not None:
                        yield chunk
            return
        except Exception as exc:
            if attempt == max_retries:
                raise
            time.sleep(backoff ** attempt)

def project_to_pair(raw_line: str) -> Tuple[str, str]:
    """
    Project heterogeneous schema to (prompt, response).
    Handles JSONL lines from public-raw and mirror-merged.
    """
    line = raw_line.strip()
    if not line:
        return "", ""

    try:
        obj = json.loads(line)
    except Exception:
        return "", ""

    prompt = obj.get("prompt") or obj.get("input") or obj.get("question") or ""
    response = obj.get("response") or obj.get("output") or obj.get("answer") or ""

    prompt = str(prompt).strip()
    response = str(response).strip()
    return prompt, response

def stream_pairs(repo: str, file_list_path: str, shard_id: int = 0, total_shards: int = 16, limit: int = -1) -> Iterator[Tuple[str, str]]:
    file_list = json.loads(Path(file_list_path).read_text())
    count = 0
    for item in tqdm(file_list, desc="CDN streaming"):
        path = item["path"]
        # Deterministic shard assignment by content hash of path (stable across workers)
        h = hashlib.md5(path.encode()).hexdigest()
        if int(h, 16) % total_shards != shard_id:
            continue

        try:
            for raw in cdn_lines(repo, path):
                p, r = project_to_pair(raw)
                if p and r:
                    yield p, r
                    count += 1
                    if limit > 0 and count >= limit:
                        return
        except Exception as exc:
            tqdm.write(f"Skip {path}: {exc}")
            continue
```

#!/usr/bin/env bash
# dataset-enrich.sh
# Existing behavior unchanged. New optional CDN mode:
#   CDN_MODE=1 FILE_LIST=cache/file-list-2026-05-02.json HF_TOKEN=... ./dataset-enrich.sh

FILE_LIST="${FILE_LIST:-}"

mkdir -p "$OUT_DIR"
