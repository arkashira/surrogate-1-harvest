# Costinel / design

## Final Synthesis — Costinel “Top-Hub Signal” Card (read-only)

**Scope**: Frontend-only, zero-backend card that surfaces the highest-degree hub from the knowledge-rag graph and shows 3 contextual signals.  
**Timebox**: ≤2h.  
**Pattern**: Sense + Signal (read-only; no execution).

---

### 1) Highest-value incremental improvement (resolved)
Ship a reusable card that:
- Consumes a **lightweight, cached graph snapshot** (JSON) produced by prior analysis (`granite-business-research.sh` + `knowledge-rag`).  
- Computes **top hub by degree** (ties broken by total edge weight).  
- Renders:
  - Hub name + short description  
  - 3 contextual signals (neighbor labels + optional weight hint)  
  - Last-updated timestamp  
  - Link to full graph view (if available)

**Why this is highest value**:
- Immediate governance visibility into the most-connected context (e.g., “MOC”).  
- Reuses existing knowledge-rag outputs; no infra/backend.  
- Fits “Sense + Signal — ไม่ Execute”.  
- Drop-in ready for dashboards/docs in <2h.

---

   - Handles malformed/missing graph gracefully (returns `null` + error message).

3. **Card component** (`src/components/TopHubSignalCard.vue`)  
   - Props: `graphUrl` (default: `/data/knowledge-rag-graph.json`), `maxSignals` (default: 3).

   - States: `loading`, `error`, `data`.  
   - Fallbacks:
     - Missing graph → friendly empty state with guidance.  
     - Malformed graph → error message + retry option.

4. **Styling & UX**

   - Optional weight hint for signals (e.g., “High impact”).  
   - Timestamp in locale format.

   - Mount in existing dashboard or docs page.

6. **Test & ship**  
   - Verify with sample graph (real or minimal example).  
   - Validate graceful degradation.

### 3) Code snippets (final)

#### Graph utility (`src/lib/graph/topHub.ts`)
```ts
export interface GraphNode {
  id: string
  label?: string
  description?: string
  type?: string
  [k: string]: unknown
}

export interface GraphEdge {
  source: string
  target: string
  weight?: number
  label?: string
  [k: string]: unknown
}

export interface GraphSnapshot {
  nodes: GraphNode[]
  edges: GraphEdge[]
}

export interface Signal {
  node: GraphNode
  weight: number
  edgeLabel?: string
}

export interface TopHubResult {
  hub: GraphNode | null
  signals: Signal[]
  updatedAt: string
}

export function getTopHubAndSignals(
  graph: GraphSnapshot,
  maxSignals = 3
): TopHubResult | null {
  if (!graph?.nodes || !graph?.edges) return null

  const degree: Record<string, number> = {}
  const adjWeight: Record<string, number> = {}
  const nodeMap = new Map<string, GraphNode>()

  graph.nodes.forEach((n) => {
    degree[n.id] = 0
    adjWeight[n.id] = 0
    nodeMap.set(n.id, n)
  })

  graph.edges.forEach((e) => {
    const w = typeof e.weight === 'number' ? e.weight : 1
    degree[e.source] = (degree[e.source] || 0) + 1
    degree[e.target] = (degree[e.target] || 0) + 1
    adjWeight[e.source] = (adjWeight[e.source] || 0) + w
    adjWeight[e.target] = (adjWeight[e.target] || 0) + w
  })

  const hubId = Object.keys(degree).reduce((a, b) => {
    if (degree[a] !== degree[b]) return degree[a] > degree[b] ? a : b
    return adjWeight[a] >= adjWeight[b] ? a : b
  }, Object.keys(degree)[0] ?? null)

  if (!hubId || !nodeMap.has(hubId)) {
    return null
  }

  const hub = nodeMap.get(hubId)!
  const connected = graph.edges
    .filter((e) => e.source === hubId || e.target === hubId)
    .map((e) => ({
      neighborId: e.source === hubId ? e.target : e.source,
      weight: typeof e.weight === 'number' ? e.weight : 1,
      edgeLabel: e.label
    }))
    .filter((c) => nodeMap.has(c.neighborId))

  const signals = connected
    .sort((x, y) => y.weight - x.weight)
    .slice(0, maxSignals)
    .map((c) => ({
      node: nodeMap.get(c.neighborId)!,
      weight: c.weight,
      edgeLabel: c.edgeLabel
    }))

  return {
    hub,
    signals,
    updatedAt: new Date().toISOString()
  }
}
```
