# Costinel / design

## Incremental Improvement: Top-Hub Signal Panel (Costinel Dashboard)

**Value**: Surface the most-connected hub (default “MOC”) and its top 3 actionable proposals from the knowledge graph. Zero runtime API cost (CDN-first), read-only React panel, ships in <2h.

---

## Implementation Plan

1. **Data contract** (5 min)  
   - Single JSON file per hub at `public/knowledge/hubs/{hub}.json` (CDN).  
   - Schema: `{ hub, title, description, proposals: [{id, title, signal, rationale, impact, tags}] }`.

2. **Static asset** (10 min)  
   - Add `public/knowledge/hubs/MOC.json` with top 3 proposals (seeded from knowledge-rag output).

3. **Frontend component** (60–90 min)  
   - New: `src/components/TopHubSignalPanel.tsx`  
   - CDN fetch via `useEffect` (no backend call).  
   - Skeleton → render proposals → error boundary.  
   - Add to dashboard layout (top of sidebar or main pane per UX).

4. **Routing/integration** (10 min)  
   - Mount panel on `/dashboard` (or replace placeholder widget).

5. **Polish & test** (15 min)  
   - Dark/light styles, copy-to-clipboard for rationale, static snapshot fallback.

### src/components/TopHubSignalPanel.tsx
```tsx
import React, { useEffect, useState } from "react";
import "./TopHubSignalPanel.css";

interface Proposal {
  id: string;
  title: string;
  signal: "high" | "medium" | "low";
  rationale: string;
  impact: string;
  tags: string[];
}

interface HubData {
  hub: string;
  title: string;
  description: string;
  proposals: Proposal[];
}

const HUB = "MOC";
const DATA_URL = `/knowledge/hubs/${HUB}.json`;

const signalColor = (s: Proposal["signal"]) => {
  switch (s) {
    case "high":
      return "var(--signal-high, #ef4444)";
    case "medium":
      return "var(--signal-medium, #f59e0b)";
    default:
      return "var(--signal-low, #10b981)";
  }
};

export const TopHubSignalPanel: React.FC = () => {
  const [data, setData] = useState<HubData | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    fetch(DATA_URL, { cache: "force-cache" })
      .then((res) => {
        if (!res.ok) throw new Error(`Failed to load hub data (${res.status})`);
        return res.json();
      })
      .then((json) => {
        setData(json);
        setLoading(false);
      })
      .catch((err) => {
        setError(err.message);
        setLoading(false);
      });
  }, []);

  if (loading) {
    return (
      <div className="top-hub-panel loading">
        <div className="skeleton title"></div>
        <div className="skeleton desc"></div>
        <div className="skeleton list">
          {Array(3)
            .fill(0)
            .map((_, i) => (
              <div key={i} className="skeleton item"></div>
            ))}
        </div>
      </div>
    );
  }

  if (error || !data) {
    return (
      <div className="top-hub-panel error">
        Unable to load hub signals.{" "}
        <button onClick={() => window.location.reload()}>Retry</button>
      </div>
    );
  }

  return (
    <div className="top-hub-panel">
      <header>
        <h2>{data.title}</h2>
        <p className="description">{data.description}</p>
      </header>

      <section className="proposals">
        {data.proposals.map((p) => (
          <article key={p.id} className="proposal-card">
            <div className="proposal-header">
              <span className="proposal-id">{p.id}</span>
              <span
                className="signal-badge"
                style={{ backgroundColor: signalColor(p.signal) }}
              >
                {p.signal}
              </span>
            </div>
            <h3>{p.title}</h3>
            <p className="rationale">{p.rationale}</p>
            <p className="impact">{p.impact}</p>
            <div className="tags">
              {p.tags.map((t) => (
                <span key={t} className="tag">
                  {t}
                </span>
              ))}
            </div>
          </article>
        ))}
      </section>
    </div>
  );
};
```
