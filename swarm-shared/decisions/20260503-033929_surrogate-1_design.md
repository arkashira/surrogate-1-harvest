# surrogate-1 / design

## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16` (matrix) and optional `DATE_FOLDER` (defaults to today `YYYY-MM-DD`)
- Uses **manifest-first strategy**: single `list_repo_tree` call (per date folder) → local JSON manifest → Lightning training does CDN-only fetches with zero API calls during data load
- Downloads via **HF CDN bypass** (`https://huggingface.co/datasets/.../resolve/main/...`) — no Authorization header, avoids 429 API limits
- Projects each file to `{prompt, response}` only at parse time (avoids pyarrow CastError on mixed schemas)
- Dedups via central md5 store (`lib/dedup.py`) and writes to deterministic shard output:
  ```
  batches/public-merged/<date>/shard<N>-<HHMMSS>.jsonl
  ```
- Commits via HF API with deterministic repo selection (hash slug → 1 of 5 sibling repos) to stay under 128 commits/hr/repo
- Reuses running Lightning Studio when present (saves 80hr/mo quota)
- Runs only on GitHub Actions (Mac orchestration only); training → Lightning Studio; ingestion → HF Space

surrogate-1 CDN-bypass ingestion worker (manifest-driven).
Usage (GitHub Actions matrix):

Optional:

import pyarrow.parquet as pq

HF_REPO = os.getenv("HF_REPO", "axentx/surrogate-1-training-pairs")

if not HF_TOKEN:

SHARD_ID = int(os.getenv("SHARD_ID", "0"))
SHARD_TOTAL = int(os.getenv("SHARD_TOTAL", "16"))
DATE_FOLDER = os.getenv("DATE_FOLDER", datetime.date.today().isoformat())

# sibling repos for commit-cap spread (5 siblings = 640/hr aggregate)
SIBLING_REPOS = [
    "axentx/surrogate-1-training-pairs",
    "axentx/surrogate-1-training-pairs-sib1",
    "axentx/surrogate-1-training-pairs-sib2",
    "axentx/surrogate-1-training-pairs-sib3",
    "axentx/surrogate-1-training-pairs-sib4",
]

API = HfApi(token=HF_TOKEN)

def deterministic_shard(key: str, total: int) -> int:
    return int(hashlib.md5(key.encode()).hexdigest(), 16) % total

def pick_sibling_repo(slug: str) -> str:
    idx = deterministic_shard(slug, len(SIBLING_REPOS))
    return SIBLING_REPOS[idx]

def list_date_files(date_folder: str):
    """Single API call: list top-level files in date folder (non-recursive)."""
    try:
        tree = list_repo_tree(
            repo_id=HF_REPO,
            path=date_folder,
            recursive=False,
            token=HF_TOKEN,
        )
    except Exception as e:
        # If folder doesn't exist yet, return empty
        print(f"list_repo_tree failed (maybe empty): {e}", file=sys.stderr)
        return []
    return [t.rfilename for t in tree if t.type == "file"]

def cdn_download(url: str, dest: Path):
    """Download via HF CDN (no auth header) with retry/backoff."""
    for attempt in range(5):
        try:
            r = requests.get(url, timeout=30)
            r.raise_for_status()
            dest.write_bytes(r.content)
            return
        except Exception as e:
            wait = 2 ** attempt
            print(f"CDN download failed (attempt {attempt+1}), retry in {wait}s: {e}", file=sys.stderr)
            time.sleep(wait)
    raise RuntimeError(f"CDN download failed after retries: {url}")

def parse_to_pair(local_path: Path):
    """Project file to {prompt, response} only. Supports parquet/jsonl."""
    suffix = local_path.suffix.lower()
    if suffix == ".parquet":
        tbl = pq.read_table(local_path, columns=["prompt", "response"])
        df = tbl.to_pandas()
        for _, row in df.iterrows():
            yield {"prompt": str(row["prompt"]), "response": str(row["response"])}
    elif suffix == ".jsonl":
        for line in local_path.read_text().splitlines():
            line = line.strip()
            if not line:
                continue
            obj = json.loads(line)
            yield {"prompt": str(obj["prompt"]), "response": str(obj["response"])}
    else:
        # fallback: try to read as json
        obj = json.loads(local_path.read_text())
        yield {"prompt": str(obj["prompt"]), "response": str(obj["response"])}
