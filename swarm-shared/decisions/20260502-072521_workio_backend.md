Use the Lightning Studio reuse and idle stop handling patterns to optimize the training process. Update the `train.py` script to reuse an existing studio and handle idle stops.

```python
import lightning as L

# Define the studio name and machine type
studio_name = "your-studio-name"
machine_type = L.Model.L40S

# Reuse an existing studio
for s in L.Teamspace.studios:
    if s.name == studio_name and s.status == "Running":
        studio = s
        break
else:
    # Create a new studio if none exists
    studio = L.Studio(create_ok=True)

# Handle idle stops
if studio.status == "Stopped":
    studio.start(machine=machine_type)
```
