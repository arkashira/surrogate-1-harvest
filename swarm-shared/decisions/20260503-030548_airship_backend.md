## Final Consolidated Implementation (Best of Both Candidates)

**Goal**: Eliminate HF API 429s during Surrogate training and make Lightning resilient to idle-stop so iteration is **<2 minutes** and never blocked by rate limits.

**Core Strategy**:  
1. **CDN-bypass data loading** — remove all `huggingface_hub` dataset calls during training; use direct CDN URLs only.  
2. **Lightning reuse + auto-restart** — reuse running studios, auto-restart on idle-stop, and add keep-alive to prevent stops.  
3. **Idempotent, observable, rollback-safe** — single-filelist generation, local parquet projection, and a `USE_CDN` flag for instant revert.

### 1) CDN-bypass file list (10–15 min)

- Add retry with exponential backoff for CDN downloads; treat 429 as long sleep (360s) and retry.

### 3) Lightning Studio reuse + idle-stop resilience (30–40 min)
- Before `.run()`, list studios and reuse a running one if present; start stopped ones.  
- On idle-stop, auto-restart training with the same machine type.  
- Add lightweight keep-alive probe (every 5 min) to reduce idle-stop risk.

### 4) Deploy/test (20–30 min)
- Commit changes to `surrogate/train.py` and launcher.  
- Run from Mac: generate filelist, launch/reuse studio, start training.  
- Verify zero HF API calls during data load and resume after simulated stop.

### 5) Rollback guard (10–15 min)
- Keep old `load_dataset` path behind `USE_CDN=False`.  
- Log fetched files and sizes; fail fast on CDN fetch mismatches.

### 2) Surrogate training: CDN-only loader (surrogate/train.py)
```python
import json, os, time, aiohttp, asyncio, pyarrow.parquet as pq, pandas as pd
from pathlib import Path
from tempfile import TemporaryDirectory

HF_DATASET_REPO = "datasets/your-org/surrogate-mirror"
CDN_BASE = f"https://huggingface.co/{HF_DATASET_REPO}/resolve/main"
USE_CDN = os.getenv("USE_CDN", "true").lower() == "true"

if USE_CDN:
    with open("filelist.json") as f:
        FILE_LIST = json.load(f)

    async def download_file(session, rel_path, dest):
        url = f"{CDN_BASE}/{rel_path}"
        for attempt in range(5):
            try:
                async with session.get(url) as resp:
                    if resp.status == 429:
                        wait = 360
                        print(f"429 on CDN (unexpected), waiting {wait}s")
                        await asyncio.sleep(wait)
                        continue
                    resp.raise_for_status()
                    dest.write_bytes(await resp.read())
                    return
            except Exception as e:
                wait = 2 ** attempt
                print(f"Download failed {rel_path}: {e}, retry in {wait}s")
                await asyncio.sleep(wait)
        raise RuntimeError(f"Failed to download {rel_path}")

    async def fetch_all(to_dir):
        to_dir = Path(to_dir)
        to_dir.mkdir(parents=True, exist_ok=True)
        async with aiohttp.ClientSession() as session:
            tasks = []
            for rel in FILE_LIST:
                dest = to_dir / Path(rel).name
                if not dest.exists():
                    tasks.append(download_file(session, rel, dest))
            if tasks:
                await asyncio.gather(*tasks)
            else:
                print("All files already present")

    def load_parquets(data_dir):
        data_dir = Path(data_dir)
        rows = []
        for p in data_dir.glob("*.parquet"):
            tbl = pq.read_table(p)
            df = tbl.select(["prompt", "response"]).to_pandas()
            rows.append(df)
        return pd.concat(rows, ignore_index=True) if rows else pd.DataFrame()

    async def prepare_data():
        with TemporaryDirectory() as td:
            await fetch_all(td)
            df = load_parquets(td)
            return df

else:
    # fallback: original HF streaming path
    from datasets import load_dataset
    def prepare_data():
        ds = load_dataset(HF_DATASET_REPO, streaming=True, split="train")
        rows = []
        for item in ds:
            rows.append({"prompt": item["prompt"], "response": item["response"]})
        return pd.DataFrame(rows)
```

### 3) Lightning Studio reuse + idle-stop resilience (surrogate/lightning_launcher.py)
```python
from lightning_sdk import Teamspace, Studio, Machine
import time, threading

TEAMSPACE = "your-teamspace"
STUDIO_NAME = "surrogate-train-l40s"
MACHINE = Machine.L40S

def get_or_create_studio():
    ts = Teamspace(TEAMSPACE)
    for s in ts.studios:
        if s.name == STUDIO_NAME:
            if s.status == "Running":
                print(f"Reusing running studio: {STUDIO_NAME}")
                return s
            else:
                print(f"Studio exists but stopped ({s.status}). Starting...")
                s.start(machine=MACHINE)
                return s
    print(f"Creating studio: {STUDIO_NAME}")
    return Studio.create(
        teamspace=TEAMSPACE,
        name=STUDIO_NAME,
        machine=MACHINE,
        create_ok=True,
    )

def keep_alive_probe(studio, interval=300):
    def probe():
        while True:
            time.sleep(interval)
            try:
                studio.refresh()
                if studio.status != "Running":
                    print("Studio stopped externally; will restart on next run()")
                    break
            except Exception:
                break
    t = threading.Thread(target=probe, daemon=True)
    t.start()
    return t

def run_training(script_path="train.py"):
    studio = get_or_create_studio()
    if studio.status != "Running":
        studio.start(machine=MACHINE)
    keep_alive_probe(studio)

    # run training; Lightning will execute script inside studio
    run = studio.run(
        command=f"python {script_path}",
        sync=True,
        wait=True,
    )
    print("Run finished:", run.status)
    return run
```
