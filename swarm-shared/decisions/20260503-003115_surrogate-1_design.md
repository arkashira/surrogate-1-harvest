# surrogate-1 / design

## Final Implementation Plan — CDN-first snapshot + zero-HF-API ingestion

**Highest-value incremental improvement (≤2h):**  
Add a Mac-side snapshot script that lists the target date-partition once, emits a deterministic `file_manifest.json`, and patch the Lightning training launcher to consume that manifest and fetch parquet shards exclusively via raw CDN URLs (no `datasets`, no `hf_api` calls during training). This removes HF API calls and 429/128-commit limits from the hot training path while preserving the existing 16-shard ingest pipeline.

### 2) `bin/snapshot_manifest.py` (deterministic, reproducible)

```python
#!/usr/bin/env python3
"""
Generate a deterministic file manifest for a single date partition.
Run on Mac (or any dev host) when rate-limit window is clear.

Usage:
  python bin/snapshot_manifest.py \
    --repo axentx/surrogate-1-training-pairs \
    --date 2026-05-03 \
    --out training/file_manifest.json
"""

from __future__ import annotations

import argparse
import json
import os
import sys
from datetime import datetime, timezone

from huggingface_hub import HfApi


def build_manifest(repo_id: str, date: str, out_path: str) -> None:
    api = HfApi()
    folder = f"batches/public-merged/{date}"

    try:
        entries = api.list_repo_tree(repo_id=repo_id, path=folder, recursive=False)
    except Exception as exc:
        print(f"HF API error listing {repo_id}/{folder}: {exc}", file=sys.stderr)
        sys.exit(1)

    files = sorted(e.rfilename for e in entries if e.rfilename.endswith(".parquet"))

    manifest = {
        "repo_id": repo_id,
        "date": date,
        "generated_at_utc": datetime.now(timezone.utc).isoformat(),
        "files": files,
        "cdn_base": f"https://huggingface.co/datasets/{repo_id}/resolve/main",
    }

    os.makedirs(os.path.dirname(os.path.abspath(out_path)), exist_ok=True)
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(manifest, f, indent=2)
        f.write("\n")

    print(f"Wrote {len(files)} files to {out_path}")


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Create CDN file manifest.")
    parser.add_argument("--repo", required=True, help="HF dataset repo id")
    parser.add_argument("--date", required=True, help="Date partition (YYYY-MM-DD)")
    parser.add_argument("--out", default="training/file_manifest.json", help="Output JSON path")
    args = parser.parse_args()
    build_manifest(args.repo, args.date, args.out)
```

Make executable (optional): `chmod +x bin/snapshot_manifest.py`.

### 3) Patch `training/train.py` (CDN-only, memory-safe streaming)

Replace any `load_dataset(..., streaming=True)` or recursive HF API usage with a simple `IterableDataset` that streams parquet files via CDN. This version avoids writing temp files to disk and uses PyArrow’s native buffer support for zero-copy reads where possible.

```python
from __future__ import annotations

import io
import json
import os
from typing import Dict, Iterator

import pyarrow.parquet as pq
import requests
import torch
from torch.utils.data import IterableDataset

MANIFEST_PATH = os.path.join(os.path.dirname(__file__), "file_manifest.json")


class CDNParquetIterable(IterableDataset):
    """
    Stream parquet shards via CDN URLs with zero HF API calls during training.
    Falls back to local files if manifest is absent (dev convenience).
    """

    def __init__(self, manifest_path: str = MANIFEST_PATH, columns=("prompt", "response")):
        self.manifest_path = manifest_path
        self.columns = columns
        self.manifest = self._load_manifest()

    def _load_manifest(self) -> Dict:
        if os.path.exists(self.manifest_path):
            with open(self.manifest_path, "r", encoding="utf-8") as f:
                return json.load(f)
        return {"files": [], "cdn_base": ""}

    def _cdn_url(self, rfilename: str) -> str:
        base = self.manifest.get("cdn_base", "")
        if not base:
            raise ValueError("cdn_base missing in manifest")
        return f"{base}/{rfilename}"

    def _stream_shard(self, url: str) -> bytes:
        resp = requests.get(url, timeout=60)
        resp.raise_for_status()
        return resp.content

    def __iter__(self) -> Iterator[Dict]:
        files = self.manifest.get("files", [])
        if not files:
            return

        worker_info = torch.utils.data.get_worker_info()
        if worker_info is None:
            iter_files = files
        else:
            per_worker = len(files) // worker_info.num_workers
            worker_id = worker_info.id
            iter_files = files[worker_id * per_worker : (worker_id + 1) * per_worker]

        for rfn in iter_files:
            url = self._cdn_url(rfn)
            buf = io.BytesIO(self._stream_shard(url))
            table = pq.read_table(buf, columns=self.columns)

            for batch in table.to_batches():
                cols = {c: batch.column(c).to_pylist() for c in self.columns}
                for i in range(len(cols["prompt"])):
                    yield {k: v[i] for k, v in cols.items()}
```

Usage in training loop (minimal change):

```python

train_data = CDNParquetIterable()
loader = DataLoader(train_data, batch_size=8, num_workers=2)

for batch in loader:
    # batch = {"prompt": [...], "response": [...]}
    ...
```

### 4) Operational notes

- **Run snapshot** when rate-limit window clears (or after ingest finishes for the day):  
  `python bin/snapshot_manifest.py --repo axentx/surrogate-1-training-pairs --date 2026-05-03 --out training/file_manifest.json`

- **Commit strategy**: either commit manifest alongside training code for reproducibility, or generate ephemerally in CI before `lightning run`.

- **Lightning Studio reuse**: keep existing pattern — list running studios and reuse to avoid quota burn while iterating this change.

- **Fallback**: if manifest missing, trainer can still glob local files (dev) or error fast (prod).

- **Security**: CDN URLs are public; no token required. No change to `HF_TOKEN` requirements for the ingest pipeline.
