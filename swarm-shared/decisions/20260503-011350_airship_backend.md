# airship / backend

**Final Consolidated Plan**  
*Airship Backend — HF CDN Bypass + Deterministic Manifest + Safe Schema Projection + Lightning Studio Reuse*

**Core Goal**  
Eliminate HF API 429s and `pyarrow.CastError` during Surrogate-1 training, reduce Lightning Studio quota burn, and enable deterministic, CDN-only ingestion with zero Authorization headers during training.

**Scope**  
Backend-only changes (`surrogate/`). Deliver:  
- One-shot manifest generator (single `list_repo_tree` call)  
- CDN-first loader that projects only `{prompt, response}` and never uses `load_dataset(streaming=True)` on heterogeneous repos  
- Lightning launcher that reuses or safely restarts studios and passes manifest to training  
- Training entrypoint that consumes manifest and streams via CDN  
- Cron-safe wrapper template

## Implementation Plan (≤2h)

1. **Manifest generator** (`surrogate/scripts/build_file_manifest.py`)  
   - Single non-recursive `list_repo_tree` call for `batches/mirror-merged/{date}`  
   - Output deterministic JSON: `{repo, date, folder, files}` sorted  
   - Exit non-zero on failure; intended to run from Mac orchestration after rate-limit window

2. **Safe HF ingestion helper** (`surrogate/data/hf_cdn_loader.py`)  
   - Never use `load_dataset(streaming=True)` on mixed-schema repos  
   - Default to CDN URLs via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no Authorization)  
   - Fallback to `hf_hub_download` when CDN not viable  
   - Project only `{prompt, response}` at parse time; drop all other columns  
   - Deterministic write-repo selection for any commits: `repo = f"surrogate-1-{hash(slug) % 5}"`  
   - Per-file resilience: log and skip bad files instead of crashing stream

3. **Lightning launcher with studio reuse** (`surrogate/train/launch_studio.py`)  
   - List running studios; reuse exact match by name  
   - If stopped, restart on `Machine.L40S`  
   - Pass manifest path to training so workers perform CDN-only fetches  
   - Avoid unnecessary new studio creation and quota burn

4. **Training script update** (`surrogate/train/train.py`)  
   - Add `--manifest` argument  
   - Use `hf_cdn_loader.stream_files_from_manifest(..., use_cdn=True)`  
   - Project to `{prompt, response}` and drop other columns before tokenization/collation

5. **Cron-safe wrapper** (`surrogate/scripts/run_wrapper.sh`)  
   - Shebang `#!/usr/bin/env bash`, `SHELL=/bin/bash` comment  
   - Exit on error, validate env vars, log stdout/stderr  
   - Idempotent: safe for cron invocation of manifest build + training

## Resolved Contradictions (in favor of correctness + actionability)

- **Streaming vs. per-file fetching**  
  Chose per-file CDN fetching over `load_dataset(streaming=True)` because heterogeneous repos cause schema errors (`pyarrow.CastError`). This is safer and bypasses HF API during training.

- **Schema projection timing**  
  Project `{prompt, response}` at parse time in the loader (not in training loop) to guarantee downstream code never sees unexpected columns.

- **Authorization during training**  
  Prefer public CDN URLs (no Authorization header) during training to eliminate token-related failures and rate limits. Fallback to `hf_hub_download` only when necessary.

- **Studio lifecycle**  
  Reuse running studios first; restart only if stopped. Avoid unconditional new studio creation to reduce quota burn and startup latency.

- **Deterministic repo selection for writes**  
  Use `hash(slug) % 5` to spread HF commit caps across `surrogate-1-{0..4}`. Keeps writes safe and predictable.

### surrogate/scripts/build_file_manifest.py
```python
#!/usr/bin/env python3
"""
Build a deterministic file manifest for a date folder in a HF dataset repo.
Intended to run from Mac orchestration after HF API rate-limit clears.

Usage:
  python build_file_manifest.py \
    --repo "my-org/surrogate-mirror" \
    --date "2026-04-29" \
    --out manifest_2026-04-29.json
"""

import argparse
import json
import os
import sys
from pathlib import Path

from huggingface_hub import HfApi

HF_TOKEN = os.getenv("HF_TOKEN")

def main() -> None:
    parser = argparse.ArgumentParser(description="Build HF file manifest for a date folder.")
    parser.add_argument("--repo", required=True, help="HF dataset repo (org/name)")
    parser.add_argument("--date", required=True, help="Date folder (YYYY-MM-DD)")
    parser.add_argument("--out", required=True, help="Output JSON path")
    parser.add_argument("--path-prefix", default="batches/mirror-merged", help="Prefix inside repo")
    args = parser.parse_args()

    api = HfApi(token=HF_TOKEN)
    folder = f"{args.path_prefix}/{args.date}"
    try:
        # Non-recursive: one API call, paginated safely
        tree = api.list_repo_tree(repo_id=args.repo, path=folder, recursive=False)
    except Exception as exc:
        print(f"Failed to list repo tree for {folder}: {exc}", file=sys.stderr)
        sys.exit(1)

    files = [entry.path for entry in tree if entry.type == "file"]
    manifest = {
        "repo": args.repo,
        "date": args.date,
        "folder": folder,
        "files": sorted(files),
    }

    Path(args.out).write_text(json.dumps(manifest, indent=2))
    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```

### surrogate/data/hf_cdn_loader.py
```python
"""
Safe HF ingestion: CDN-only fetches, no streaming datasets on mixed-schema repos.
Projects to {prompt, response} only at parse time.
"""

import json
import os
from pathlib import Path
from typing import Dict, Iterator, Optional

import requests
from huggingface_hub import hf_hub_download

HF_TOKEN = os.getenv("HF_TOKEN")

def pick_write_repo(slug: str, n_siblings: int = 5) -> str:
    """Deterministic sibling repo selection to spread HF commit cap."""
    idx = hash(slug) % n_siblings
    return f"surrogate-1-{idx}"

def cdn_url(repo: str, path: str) -> str:
    """Public CDN URL (no Authorization header required)."""
    return f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def stream_files_from_manifest(
    manifest_path: str,
    use_cdn: bool = True,
    cache_dir: Optional[str] = None,
) -> Iterator[Dict[str, str]]:
    """
    Yield {prompt, response} items from files listed in manifest.
    Uses CDN URLs when use_cdn=True (bypasses HF API rate limits).
    """
    manifest = json.loads(Path(manifest_path).read_text())
    repo = manifest["repo"]
    files = manifest["files"]

    for fpath in files:
        try:
            if use_cdn:
                url = cdn_url(repo, fpath)
                resp = requests.get(url, timeout=30)
                resp.raise_for_status()
                raw = resp.content
                local_path = None
            else:
                local_path = hf_hub_download(
                    repo_id=repo,
                    filename=fpath,
                    cache_dir=cache_dir or os.getenv("HF_HOME"),
                    token=HF_TOKEN,
                )
                raw = Path(local_path).read_bytes()

            # Project to {prompt, response} only — ignore other columns.
            text = raw.decode("utf-8").strip()
            if not text:
                continue

            # Try line-delimited JSON first
            for line in text.splitlines():
                line = line.strip()
                if not line:
                    continue
                try:
                    obj = json.loads(line)
                    prompt = obj.get("prompt") or obj.get("input") or ""
                    response = obj.get("response") or obj.get("output") or ""
                    if prompt or response:
                        yield {"prompt": str(prompt), "response": str(response)}
