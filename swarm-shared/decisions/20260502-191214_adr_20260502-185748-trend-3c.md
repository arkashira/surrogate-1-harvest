## Deployment
hybrid (local-first CLI; optional Cloudflare Workers for auth/licensing and Supabase for shared rules/catalog)

## v0.1 scope
v0.1 ships in 2 weeks: local CLI (Python) that indexes a repo, parses Python and JS/TS files with tree-sitter, runs rule-based checks (ruff/eslint-style) and AST-based heuristics, stores results in local SQLite, and outputs a terminal report + JSON export. Incremental re-runs on file changes, basic severity/category tagging, and a sample rules catalog. No external servers required; optional Cloudflare Worker for license key validation only.
