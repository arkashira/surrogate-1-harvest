# Costinel / ops

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

### What we ship
- A **non-blocking Top-Hub Signal Panel** mounted on the Costinel dashboard (top banner area).
- Defaults to hub **MOC** (configurable via `VITE_HUB_NAME`).
- Shows: hub title, short description, top 3 insights, and a “View in Knowledge Graph” link.
- **CDN-first data fetch**: uses public CDN URLs to bypass HF API rate limits; no auth, no API calls during render.
- Zero backend changes; pure frontend addition (React + Tailwind). Safe to ship in <2h.

### Why this is highest-value
- Applies the **top-hub doc insight** pattern (MOC) and **knowledge-rag** context directly into Costinel’s ops dashboard.
- Improves decision context for cloud-cost governance without touching execution paths (fits “Sense + Signal — ไม่ Execute”).
- Uses CDN bypass pattern to avoid rate limits and keep ops tooling fast.

---

### Implementation steps (concrete)

1. **Add env var** (optional, defaults to MOC)
   - `VITE_HUB_NAME=MOC` in `.env` or CI.

2. **Create data URL builder**
   - Public CDN path: `https://huggingface.co/datasets/axentx/knowledge-rag/resolve/main/hubs/{hubName}/latest.json`
   - Fallback to local placeholder if CDN fails (non-blocking).

3. **Create React component: `TopHubSignalPanel`**
   - Mounts at top of dashboard layout (below global nav).
   - Fetches hub data on client (useSWR or fetch + abort).
   - Renders minimal card with title, description, top 3 insights, and link.
   - Non-blocking: never throws; errors are silent or collapsed.

4. **Integrate into dashboard layout**
   - Insert `<TopHubSignalPanel />` in `pages/Dashboard.tsx` or main layout.
   - Ensure mobile responsive (stacked on small screens).

5. **Add loading & empty states**
   - Skeleton while loading.
   - Hidden panel if no data (keeps UI clean).

6. **Test locally**
   - Verify CDN fetch works (no auth headers).
   - Verify fallback behavior when offline/CDN 404.

```ts
export function getHubCdnUrl(hubName: string): string {
  // Public CDN — no auth required
  return `https://huggingface.co/datasets/axentx/knowledge-rag/resolve/main/hubs/${encodeURIComponent(
    hubName
  )}/latest.json`;

```

#### 3) React component
```tsx
// src/components/TopHubSignalPanel.tsx
import { useEffect, useState } from "react";
import { getHubCdnUrl } from "@/lib/hub-cdn";

interface Insight {
  title: string;
  summary: string;
}

interface HubData {
  hubName: string;
  title: string;
  description: string;
  insights: Insight[];
  url?: string;
}

export default function TopHubSignalPanel() {
  const hubName = import.meta.env.VITE_HUB_NAME || "MOC";
  const [data, setData] = useState<HubData | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    const controller = new AbortController();
    const url = getHubCdnUrl(hubName);

    fetch(url, { signal: controller.signal, cache: "no-store" })
      .then(async (res) => {
        if (!res.ok) throw new Error(`CDN ${res.status}`);
        return res.json();
      })
      .then((json) => {
        setData({
          hubName,
          title: json.title || hubName,
          description: json.description || "",
          insights: Array.isArray(json.insights) ? json.insights.slice(0, 3) : [],
          url: json.url || `https://huggingface.co/datasets/axentx/knowledge-rag/tree/main/hubs/${hubName}`,
        });
      })
      .catch(() => {
        // Non-blocking: silently fail
        setData(null);
      })
      .finally(() => {
        if (!controller.signal.aborted) setLoading(false);
      });

    return () => controller.abort();
  }, [hubName]);

  if (loading) {
    return (
      <div className="mb-4 animate-pulse rounded-lg border border-gray-200 bg-gray-50 p-4">
        <div className="h-5 w-32 rounded bg-gray-200"></div>
        <div className="mt-2 h-4 w-full rounded bg-gray-200"></div>
      </div>
    );
  }

  if (!data || !data.insights.length) return null;

  return (
    <div className="mb-4 rounded-lg border border-blue-100 bg-blue-50 p-4 shadow-sm">
      <div className="flex items-start justify-between">
        <div>
          <h3 className="text-sm font-semibold text-blue-900">{data.title}</h3>
          {data.description && (
            <p className="mt-1 text-xs text-blue-800">{data.description}</p>
          )}
        </div>
        {data.url && (
          <a
            href={data.url}
            target="_blank"
            rel="noopener noreferrer"
            className="text-xs text-blue-600 hover:underline"
          >
            View in Graph
          </a>
        )}
      </div>

      <ul className="mt-3 space-y-2">
        {data.insights.map((insight, idx) => (
          <li key={idx} className="text-xs text-blue-800">
            <span className="font-medium">{insight.title}</span>
            {insight.summary && ` — ${insight.summary}`}
          </li>
        ))}
      </ul>
    </div>
  );
}
```
