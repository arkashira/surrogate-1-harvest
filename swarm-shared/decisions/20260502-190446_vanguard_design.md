# vanguard / design

## Final synthesized solution

**Core diagnosis (unified)**  
- Missing canonical discovery entrypoint that surfaces top-hub (MOC) insights before planning → breaks `#knowledge-rag #graph #hub` pattern.  
- No CDN-bypass file list for HF datasets → future training will immediately hit HF API rate limits.  
- No Lightning Studio reuse guard → iteration burns quota by recreating studios.  
- No single-pass executable to orchestrate research, hub insights, HF prep, studio reuse, and data projection.  
- No strict local data projection to `{prompt,response}` → mixed-schema files risk pyarrow CastError in surrogate-1.

**Concrete action**  
Create one executable file `/opt/axentx/vanguard/discovery/run_discovery.sh` (≈120 lines, Bash) that performs all five tasks in a single pass, emits actionable artifacts, and changes no external state except file writes.

**Implementation (final)**  

```bash
#!/usr/bin/env bash
# /opt/axentx/vanguard/discovery/run_discovery.sh
# Single-pass discovery: market analysis, top-hub (MOC) insights, HF CDN-bypass file list,
# Lightning Studio reuse, and local data projection to {prompt,response}.
# Usage: ./run_discovery.sh [--hf-repo <repo>] [--date-folder <path>] [--local-data <dir>]
# Tags: #business-research #knowledge-rag #graph #hub #huggingface #cdn #rate-limit-bypass #lightning-ai #quota

set -euo pipefail
IFS=$'\n\t'

# ---- config ----
HF_REPO=""
DATE_FOLDER=""
LOCAL_DATA_DIR=""

# simple arg parsing
while [[ $# -gt 0 ]]; do
  case "$1" in
    --hf-repo) HF_REPO="$2"; shift 2 ;;
    --date-folder) DATE_FOLDER="$2"; shift 2 ;;
    --local-data) LOCAL_DATA_DIR="$2"; shift 2 ;;
    *) echo "Unknown option: $1"; exit 1 ;;
  esac
done

BASE_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
OUT_DIR="${BASE_DIR}/out"
FILE_LIST="${OUT_DIR}/hf_file_list.json"
STUDIO_NAME="vanguard-training"

mkdir -p "${OUT_DIR}"

log() { echo "[$(date -u +'%Y-%m-%dT%H:%M:%SZ')] $*"; }

# ---- 1) business research + knowledge-rag ----
if command -v granite-business-research.sh &>/dev/null; then
  log "Running market analysis..."
  granite-business-research.sh || log "WARN: market analysis exited non-zero"
fi

if command -v knowledge-rag &>/dev/null; then
  log "Querying top-hub (MOC) insights..."
  knowledge-rag --query "top hub MOC and most connected docs" --format concise > "${OUT_DIR}/top_hub_insights.txt" || log "WARN: knowledge-rag query failed"
  log "Top-hub insights saved to ${OUT_DIR}/top_hub_insights.txt"
fi

# ---- 2) HF CDN-bypass file list ----
if [[ -n "${HF_REPO}" && -n "${DATE_FOLDER}" ]]; then
  log "Listing HF repo tree (non-recursive): ${HF_REPO}/${DATE_FOLDER}"
  python3 - <<PY > "${FILE_LIST}" 2>"${OUT_DIR}/hf_list.err"
import json, os, sys
from huggingface_hub import list_repo_tree

repo = os.getenv("HF_REPO")
folder = os.getenv("DATE_FOLDER").lstrip("/")
try:
    items = list_repo_tree(repo=repo, path=folder, recursive=False)
    files = [f.rfilename for f in items if f.type == "file"]
    cdn_urls = [f"https://huggingface.co/datasets/{repo}/resolve/main/{f}" for f in files]
    payload = {"repo": repo, "folder": folder, "files": files, "cdn_urls": cdn_urls}
    json.dump(payload, sys.stdout, indent=2)
except Exception as e:
    sys.stderr.write(str(e))
    sys.exit(1)
PY
  if [[ -s "${FILE_LIST}" ]]; then
    log "HF file list saved to ${FILE_LIST}"
    log "Embed this file in train.py for CDN-only fetches (zero API calls during training)."
  else
    log "WARN: HF file list empty or failed; see ${OUT_DIR}/hf_list.err"
  fi
else
  log "SKIP HF file list (provide --hf-repo and --date-folder to enable)."
fi

# ---- 3) Lightning Studio reuse ----
if command -v lightning &>/dev/null || python3 -c "import lightning as L; print(L.__version__)" &>/dev/null; then
  log "Checking Lightning Studio: ${STUDIO_NAME}..."
  python3 - <<PY | tee "${OUT_DIR}/studio_status.json"
import json, warnings, os
warnings.filterwarnings("ignore")
try:
    from lightning import Studio, Teamspace
    teamspace = Teamspace(name="default")
    running = [s for s in teamspace.studios if s.name == "${STUDIO_NAME}" and s.status == "Running"]
    if running:
        studio = running[0]
        print(json.dumps({"reused": True, "studio_name": studio.name, "status": studio.status, "url": studio.url}))
    else:
        print(json.dumps({"reused": False, "note": "No running studio found with name ${STUDIO_NAME}"}))
except Exception as e:
    print(json.dumps({"error": str(e)}))
PY
  log "Studio status saved to ${OUT_DIR}/studio_status.json"
else
  log "SKIP Lightning Studio check (lightning SDK not available)."
fi

# ---- 4) Local data projection (surrogate-1 guard) ----
if [[ -n "${LOCAL_DATA_DIR}" && -d "${LOCAL_DATA_DIR}" ]]; then
  log "Projecting local data to {prompt,response} only..."
  python3 - <<PY
import os, json, sys, pyarrow as pa, pyarrow.parquet as pq, pyarrow.csv as pcsv
from pathlib import Path

src = Path("${LOCAL_DATA_DIR}")
out_dir = Path("${OUT_DIR}/projected")
out_dir.mkdir(parents=True, exist_ok=True)

results = []
for fp in src.rglob("*"):
    if fp.suffix.lower() not in {".parquet", ".csv", ".json", ".jsonl"}:
        continue
    try:
        tbl = None
        if fp.suffix.lower() == ".parquet":
            meta = pq.read_metadata(fp)
            if "prompt" in meta.schema.names and "response" in meta.schema.names:
                tbl = pq.read_table(fp, columns=["prompt", "response"])
        elif fp.suffix.lower() == ".csv":
            names = pcsv.read_csv(fp).column_names
            if "prompt" in names and "response" in names:
                tbl = pcsv.read_csv(fp, include_columns=["prompt", "response"])
        else:
            rows = []
            for line in fp.read_text().strip().splitlines():
                line = line.strip()
                if not line:
                    continue
                obj = json.loads(line)
                rows.append({"prompt": obj.get("prompt", ""), "response": obj.get("response", "")})
            tbl = pa.Table.from_pylist(rows)
        if tbl is not None and tbl.num_rows:
            out_fp = out_dir / f"{fp.stem}.projected.parquet"
            pq.write_table(tbl.select(["prompt", "response"]), out_fp)
            results.append({"file": str(fp), "projected": str(out_fp), "rows": tbl.num_rows})
        else:
            results.append({"file": str(fp), "skipped": "missing prompt/response columns or empty"})
    except Exception as e:
        results.append({"file": str(fp), "error": str(e)})

Path("${OUT_DIR}/projection_report.json").write_text(json.dumps(results, indent=2))
print(f"Projection report saved ({len(results)} files).")
PY
  log "Projection report saved to ${OUT_DIR}/projection_report.json"
else
  log "SKIP local data projection (no --local-data provided or directory not found)."
fi

log "Discovery complete. Outputs in ${OUT_DIR}/"
```

**Make executable**  
```bash
chmod +x /opt/axentx/vanguard/discovery/run_discovery.sh
```
