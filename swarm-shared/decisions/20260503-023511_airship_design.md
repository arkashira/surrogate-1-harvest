# airship / design

## Final Implementation Plan — CDN-first Surrogate Training  
*(Scope: `/opt/axentx/airship/surrogate`; eliminates HF 429 + Lightning quota waste)*

---

### 1) Core Design Decisions (resolve contradictions)

- **File listing**: use a **deterministic, date-folder filelist JSON** built once per day (HF API `list_repo_tree(recursive=False)`).  
  - *Why*: avoids recursive `list_repo_files` during training and removes 429 risk.  
  - *Contradiction resolved*: prefer `list_repo_tree` (Candidate 1) over per-file `hf_hub_download` metadata calls; it’s faster and cheaper.

- **Training data access**: **CDN-only downloads** (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with **parquet projection** (`prompt`, `response`).  
  - *Why*: zero HF API calls during training; minimal I/O and memory.  
  - *Contradiction resolved*: use direct `requests` + `pyarrow` (Candidate 1) instead of `datasets` loader; avoids hidden API calls and is faster for row projection.

- **Lightning Studio lifecycle**: **reuse running studio**, restart stopped ones, **skip if stopping/idle_timeout**.  
  - *Why*: prevents quota waste from repeated cold starts and failed runs.  
  - *Contradiction resolved*: enforce explicit `idle_timeout` guard and status check before `run()` (both candidates agree; make it mandatory).

- **Ingestion output schema**: **only `prompt`, `response`** in parquet; attribution via filename; no extra columns.  
  - *Why*: simplifies downstream training and storage; deterministic.

- **Actionability**: provide **exact commands and file paths**; all changes fit in `/opt/axentx/airship/surrogate`.

```python
#!/usr/bin/env python3
import argparse
import json
import os
import sys
from huggingface_hub import HfApi

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    token = os.getenv("HF_TOKEN")
    if not token:
        print("HF_TOKEN required", file=sys.stderr)
        sys.exit(1)

    api = HfApi(token=token)
    entries = api.list_repo_tree(
        repo_id=args.repo,
        path=args.date,
        recursive=False,
        repo_type="dataset",
    )

    filelist = []
    for e in entries:
        if e.type != "file":
            continue
        filelist.append(
            {
                "repo": args.repo,
                "path": e.path,
                "sha": e.lfs.get("sha256", None) if e.lfs else None,
                "size": e.size,
            }
        )

    os.makedirs(os.path.dirname(os.path.abspath(args.out)), exist_ok=True)
    with open(args.out, "w") as f:
        json.dump(filelist, f, indent=2)

    print(f"Wrote {len(filelist)} entries to {args.out}")

if __name__ == "__main__":
    main()
```

#### 2.2) `surrogate/training/cdn_dataset.py`  
*Zero-HF-API dataset for training.*

```python
import io
import json
from typing import Dict, Iterator
import pyarrow.parquet as pq
import requests
import torch

CDN_ROOT = "https://huggingface.co/datasets"

class CDNParquetIterable(torch.utils.data.IterableDataset):
    def __init__(self, filelist_path: str, repo: str, start_idx: int = 0, end_idx: int = -1):
        super().__init__()
        with open(filelist_path) as f:
            self.files = json.load(f)
        self.repo = repo
        if end_idx < 0:
            end_idx = len(self.files)
        self.files = self.files[start_idx:end_idx]

    def _download_via_cdn(self, path: str) -> bytes:
        url = f"{CDN_ROOT}/{self.repo}/resolve/main/{path}"
        resp = requests.get(url, timeout=60)
        resp.raise_for_status()
        return resp.content

    def _iter_file(self, item: Dict[str, str]) -> Iterator[Dict[str, str]]:
        content = self._download_via_cdn(item["path"])
        table = pq.read_table(io.BytesIO(content))
        cols = {}
        for col in ("prompt", "response"):
            if col in table.column_names:
                cols[col] = table[col].to_pylist()
            else:
                cols[col] = [None] * table.num_rows
        for i in range(table.num_rows):
            yield {k: v[i] for k, v in cols.items()}

    def __iter__(self) -> Iterator[Dict[str, str]]:
        for item in self.files:
            yield from self._iter_file(item)
```
