# airship / design

## Final Consolidated Implementation Plan (≤2h)

**Core Strategy**: CDN-first ingestion + deterministic sharding + Studio lifecycle guard.  
**Goal**: Eliminate HF API 429s, bypass 128/hr commit cap, and prevent idle-timeout training loss.

---

### 1. CDN-First Ingestion Utility  
**File**: `surrogate/ingest/cdn_shard_ingest.py`  
- **Single list call** from Mac: `list_repo_tree(repo="axentx/raw-mirror", path=date_folder, recursive=False)` → `file_list.json`.  
- **Training script** loads `file_list.json` and streams via **public CDN URLs only** (no auth, no HF API during training):  
  `https://huggingface.co/datasets/{repo}/resolve/main/{path}`  
- **Projection**: parse each file lazily; keep only `{prompt, response}`; drop all other fields.  
- **Attribution**: store merged parquet as `batches/mirror-merged/{date}/{slug}.parquet` (attribution in filename, not columns).

---

### 2. Deterministic Sibling-Repo Sharding  
**Rule**: `REPO_BASE="axentx/surrogate-mirror"` → 5 shards: `{REPO_BASE}-shard0..4`.  
- `pick_repo(slug: str) -> str`: `SIBLINGS[hash(slug) % 5]`.  
- Use HF `upload_file` to chosen shard; distributes writes → **640 commits/hr aggregate** (5×128).  
- All uploads done from Mac (or CI) after projection; training never writes to HF.

---

### 3. Studio Lifecycle Guard  
**File**: `surrogate/train/studio_guard.py`  
- **Reuse**: before `Studio(create_ok=True)`, list `Teamspace.studios`; if matching name exists and is `running`, reuse it.  
- **Restart on stop**: if status is `stopped` (idle-timeout), call `studio.start(machine=Machine.L40S)` and poll until `running`.  
- **Retry on 429**: wrap `studio.run()` in try/except; on 429 wait 360s and retry.  
- **Orchestration**: Mac launches guard → guard ensures studio is running → Mac submits Lightning job to that studio.

---

### 4. Training Script Update  
**File**: `surrogate/train/train.py`  
- Accept `file_list_path` arg; load JSON list of CDN URLs.  
- Use `IterableDataset` that streams via `requests.get(url, timeout=60)` and yields `{prompt, response}`.  
- **Do not** use `load_dataset(streaming=True)` on heterogeneous repo (avoids schema issues and HF API calls).  
- Keep Mac as orchestrator only (Lightning SDK launcher + HF API for ingestion); all compute in Studio.

---

### 5. Cron/Env Hygiene  
- All wrapper scripts: `#!/usr/bin/env bash`, `chmod +x`, invoked via `bash <script> "$@"`.  
- Crontab: set `SHELL=/bin/bash`.  
- Ensure HF token is available only to ingestion/upload scripts (not training).

### surrogate/ingest/cdn_shard_ingest.py
```python
#!/usr/bin/env python3
import json, hashlib
from pathlib import Path
from hugginggingface import HfApi  # use real hfapi

HF_REPO_BASE = "axentx/surrogate-mirror"
SIBLINGS = [f"{HF_REPO_BASE}-shard{i}" for i in range(5)]
CDN_TMPL = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def pick_repo(slug: str) -> str:
    return SIBLINGS[hash(slug) % len(SIBLINGS)]

def list_date(date_folder: str, src_repo="axentx/raw-mirror"):
    api = HfApi()
    items = api.list_repo_tree(src_repo, path=date_folder, recursive=False)
    return [it.r_path for it in items if it.type == "file"]

def save_list(date_folder: str, out: Path):
    out.parent.mkdir(parents=True, exist_ok=True)
    out.write_text(json.dumps(list_date(date_folder), indent=2))

def cdn_url(repo: str, path: str) -> str:
    return CDN_TMPL.format(repo=repo, path=path)

# Usage: save_list("2026-05-03", Path("file_lists/2026-05-03_files.json"))
```

### surrogate/train/studio_guard.py
```python
#!/usr/bin/env python3
from lightning import Studio, Teamspace, Machine, L40S
import time

STUDIO_NAME = "surrogate-train-l40s"
MACHINE = Machine.L40S

def wait_running(studio, timeout=300, interval=15):
    for _ in range(timeout // interval):
        studio.refresh()
        if studio.status == "running":
            return
        time.sleep(interval)
    raise RuntimeError("Studio failed to start")

def get_or_create_studio():
    team = Teamspace()
    for s in team.studios:
        if s.name == STUDIO_NAME:
            if s.status == "running":
                return s
            s.start(machine=MACHINE)
            wait_running(s)
            return s
    return Studio(name=STUDIO_NAME, machine=MACHINE, create_ok=True)

def ensure_running(studio):
    studio.refresh()
    if studio.status != "running":
        studio.start(machine=MACHINE)
        wait_running(studio)

if __name__ == "__main__":
    st = get_or_create_studio()
    ensure_running(st)
```

## Execution Order (≤2h)
1. Run `cdn_shard_ingest.py` to generate `file_list.json` for target date.  
2. Upload projected parquets to sibling shards (deterministic).  
3. Launch `studio_guard.py` on Mac to ensure studio is running.  
4. Submit training job via Lightning SDK pointing to `train.py` + `file_list.json`.  

**Result**: No HF API 429s during training, 5× commit capacity, and training survives idle timeouts.
