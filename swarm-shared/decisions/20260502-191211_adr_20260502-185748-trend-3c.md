## Deployment
hybrid (local-first CLI + optional Cloudflare Workers for licensing/auth and Hugging Face Spaces for optional hosted model inference)

## v0.1 scope
v0.1 ships in 2 weeks: CLI tool (Python) that indexes a local repo, runs fast local static checks (ruff/tree-sitter) and a small OSS code model on CPU, stores results in SQLite, and outputs actionable findings (JSON + terminal summary). Includes sample repo tests and basic install (pipx/pyproject). No remote services required; optional HF Space for model inference behind a flag.

## Open questions
- Which language analyzers to ship in v0.1 (Python only or multi-language via tree-sitter?)
- Local model size vs. quality trade-offs: run small OSS models on CPU or defer to HF Spaces for larger models?
- Pricing and packaging: one-time license vs. subscription for rule updates/models?
- How to handle large repos (monorepos) — incremental analysis and caching strategy?
- Security posture: sandboxing model/inference and verifying third-party rule bundles?
