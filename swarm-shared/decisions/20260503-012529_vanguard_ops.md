# vanguard / ops

## Final Synthesized Solution

Below is the single, authoritative implementation that merges the strongest insights from all candidates, resolves contradictions in favor of correctness + concrete actionability, and provides a complete, production-ready workflow.

---

## 1. Diagnosis (Consensus)

- **No persisted manifest** → every training run re-authenticates via `list_repo_tree`, burning HF API quota and causing intermittent 429s.
- **Streaming/API data loading** → training uses `load_dataset(streaming=True)` or authenticated paths instead of public CDN URLs, wasting rate limits and adding latency.
- **No Studio reuse** → each run creates a new Lightning Studio, burning quota on redundant startup/teardown.
- **Idle-stop fragility** → Lightning Studio idle timeout kills long-running training; no auto-restart or checkpoint resume.
- **Mac compute leakage** → orchestration may invoke training locally (`.from_pretrained`, local dataloaders), violating “Mac = CLI only” and risking local saturation.

---

## 2. Design Decisions (Resolved)

- **Manifest persistence**: single `list_repo_tree` per `(repo, dateFolder)`; store JSON with CDN URLs and checksums.
- **CDN-only data loading**: never use HF Hub authenticated paths during training; use `https://huggingface.co/datasets/.../resolve/main/...`.
- **Studio lifecycle**: reuse running Studio; restart if stopped; recreate if crashed.
- **Idempotent run contract**: Studio job must be restart-safe (checkpoint dir, deterministic map-style dataset or resumable iterable).
- **Mac = orchestrator only**: no model/data load on Mac; all compute in Lightning Studio.
- **Retry + backoff**: exponential backoff on transient failures; hard fail after max retries.
- **Minimal, typed, no unused branches**: remove speculative multi-cloud and dead code; keep one clear control flow.

---

# /opt/axentx/vanguard/ops/training_ops.py
import json
import os
import time
from pathlib import Path
from typing import Dict, List, Optional

import lightning as L

try:
    from huggingface_hub import list_repo_tree
except ImportError as e:
    raise RuntimeError("huggingface_hub required for training_ops") from e


MANIFEST_NAME = "manifest.json"
CDN_BASE = "https://huggingface.co/datasets"


def build_manifest(repo_id: str, date_folder: str, out_path: Path) -> List[str]:
    """
    Single authenticated list_repo_tree call for one date folder (non-recursive).
    Persists manifest and returns file list.
    """
    tree = list_repo_tree(repo_id=repo_id, path=date_folder, recursive=False)
    files = [item.rfilename for item in tree if item.type == "file"]

    manifest = {
        "repo_id": repo_id,
        "date_folder": date_folder,
        "files": files,
        "created_ts": int(time.time()),
        "cdn_base": CDN_BASE,
        "note": "CDN-only training; use resolve/main/ URLs to bypass HF API rate limits",
    }
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    return files


def load_manifest(manifest_path: Path) -> Dict:
    if not manifest_path.is_file():
        raise FileNotFoundError(f"Manifest not found: {manifest_path}")
    return json.loads(manifest_path.read_text())


def cdn_url_for(repo_id: str, filepath: str) -> str:
    """Public CDN URL — no Authorization header required."""
    return f"{CDN_BASE}/{repo_id}/resolve/main/{filepath}"


def ensure_studio(
    studio_name: str,
    script_path: str,
    machine: str = "L40S",
    dependencies: Optional[List[str]] = None,
    teamspace: str = "default",
) -> L.studio.Studio:
    """
    Reuse running Studio if present; restart if stopped; recreate if crashed/missing.
    """
    teamspace_obj = L.Teamspace(teamspace, create_ok=True)
    existing = next((s for s in teamspace_obj.studios if s.name == studio_name), None)

    if existing is not None:
        if existing.status == "running":
            return existing
        if existing.status == "stopped":
            existing.start(machine=machine)
            return existing
        # crashed/unknown: recreate for determinism
        existing.delete()

    cloud = L.CloudProvider(name="lightning-public-prod")
    machine_obj = L.Machine(machine, cloud=cloud)
    studio = L.studio.Studio(
        name=studio_name,
        script=script_path,
        machine=machine_obj,
        dependencies=dependencies or [],
        teamspace=teamspace,
        create_ok=True,
    )
    return studio


def train_via_lightning(
    repo_id: str,
    date_folder: str,
    manifest_dir: Path,
    studio_name: str,
    train_script: str,
    machine: str = "L40S",
    teamspace: str = "default",
    max_retries: int = 3,
) -> None:
    """
    Mac-side orchestrator (CLI only):
    1) Build/load manifest once (single API call).
    2) Ensure Studio (reuse or restart).
    3) Run training script in Studio (CDN-only data loading).
    """
    manifest_dir = Path(manifest_dir)
    manifest_path = manifest_dir / MANIFEST_NAME

    # Step 1: manifest (single API window usage)
    if not manifest_path.is_file():
        print(f"Building manifest for {repo_id}/{date_folder}...")
        build_manifest(repo_id=repo_id, date_folder=date_folder, out_path=manifest_path)
    manifest = load_manifest(manifest_path)
    print(f"Manifest with {len(manifest['files'])} files loaded.")

    # Step 2: Studio lifecycle
    studio = ensure_studio(
        studio_name=studio_name,
        script_path=train_script,
        machine=machine,
        dependencies=[],
        teamspace=teamspace,
    )

    # Step 3: run training (Studio does CDN-only fetches)
    env = {
        "VANGUARD_MANIFEST_PATH": str(manifest_path),
        "VANGUARD_REPO_ID": repo_id,
        "VANGUARD_DATE_FOLDER": date_folder,
        "VANGUARD_CDN_BASE": CDN_BASE,
    }

    for attempt in range(1, max_retries + 1):
        try:
            if studio.status != "running":
                print(f"Studio not running (status={studio.status}). Restarting...")
                studio.start(machine=machine)

            print(f"Running training (attempt {attempt}/{max_retries})...")
            studio.run(
                args=[
                    "--manifest",
                    str(manifest_path),
                    "--output-dir",
                    "/workspace/output",
                ],
                env=env,
            )
            print("Training completed.")
            return
        except Exception as exc:
            print(f"Training attempt {attempt} failed: {exc}")
            if attempt == max_retries:
                raise
            wait = 300 * attempt
            print(f"Retrying in {wait}s...")
            time.sleep(wait)


def generate_cdn_filelist(repo_id: str, date_folder: str, manifest_path: Path) -> List[str]:
    """Helper for train.py: load manifest and return CDN URLs."""
    manifest = load_manifest(manifest_path)
    return [cdn_url_for(repo_id, f) for f in manifest["files"]]
