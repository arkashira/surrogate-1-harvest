# surrogate-1 / design

## Final Implementation Plan (≤2h)

**Highest-value improvement**: Add deterministic pre-flight file listing + CDN-only ingestion to eliminate HF API 429s during training and make shard workers resilient.

### What we’ll change
1. Add `scripts/list_repo_files.py` — single Mac/CI call that lists one date folder via `list_repo_tree(recursive=False)` and writes `file-list.json` (path + size + sha256).  
2. Add `lib/cdn_download.py` — lightweight CDN-only downloader using raw CDN URLs with retries/backoff and no Authorization header.  
3. Update `bin/dataset-enrich.sh` to accept optional `FILE_LIST`; if provided, workers iterate that list and download via CDN only; if absent, fall back to `datasets` library.  
4. Add `bin/train-cdn.sh` launcher for Lightning Studio: loads `file-list.json`, sets `HF_DATASETS_OFFLINE=1`, and uses a custom IterableDataset that only hits CDN.  
5. Update README with new workflow:  
   - Mac: `python scripts/list_repo_files.py --date 2026-05-03 --out file-list.json` (run once after rate-limit window clears).  
   - GitHub Actions: pass `FILE_LIST=file-list.json` to each shard so all 16 workers do zero API calls during data load.  
   - Lightning: reuse running Studio; training script uses CDN-only dataset.

### Acceptance criteria
- Shard workers can run with no HF API auth during data streaming (only CDN GETs).  
- No `load_dataset(streaming=True)` on heterogeneous repos; each file downloaded individually and projected to `{prompt, response}` at parse time.  
- Deterministic shard assignment via `slug-hash % 16 == SHARD_ID` remains unchanged.  
- Total implementation time ≤2h.

### 1) scripts/list_repo_files.py
```python
#!/usr/bin/env python3
"""
List files in a date folder of axentx/surrogate-1-training-pairs (non-recursive)
and emit file-list.json for CDN-only ingestion.

Usage:
  python scripts/list_repo_files.py --date 2026-05-03 --out file-list.json
"""

import argparse
import json
import os
import sys
from datetime import datetime, timezone

from huggingface_hub import HfApi

REPO_ID = "axentx/surrogate-1-training-pairs"

def main() -> None:
    parser = argparse.ArgumentParser()
    parser.add_argument("--date", required=True, help="Date folder, e.g. 2026-05-03")
    parser.add_argument("--out", required=True, help="Output JSON path")
    args = parser.parse_args()

    api = HfApi(token=os.getenv("HF_TOKEN"))
    folder = f"batches/public-merged/{args.date}"
    try:
        tree = api.list_repo_tree(
            repo_id=REPO_ID,
            path=folder,
            recursive=False,
            repo_type="dataset",
        )
    except Exception as e:
        print(f"Failed to list {folder}: {e}", file=sys.stderr)
        sys.exit(1)

    files = []
    for item in tree:
        if item.type != "file":
            continue
        files.append(
            {
                "path": item.path,
                "size": item.size or 0,
                "sha256": getattr(item, "sha256", None),
            }
        )

    payload = {
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "date": args.date,
        "folder": folder,
        "files": files,
    }

    os.makedirs(os.path.dirname(os.path.abspath(args.out)), exist_ok=True)
    with open(args.out, "w") as f:
        json.dump(payload, f, indent=2)

    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```

### 2) lib/cdn_download.py
```python
#!/usr/bin/env python3
"""
Download dataset files via HuggingFace CDN (no Authorization header).
"""

import time
from pathlib import Path
from typing import Optional

import requests
from tqdm import tqdm

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def cdn_download(
    repo: str,
    repo_path: str,
    target: Path,
    chunk_size: int = 8192,
    max_retries: int = 5,
    backoff: float = 1.0,
) -> Path:
    """
    Download a single file via CDN. Raises on failure.
    """
    url = CDN_TEMPLATE.format(repo=repo, path=repo_path)
    target.parent.mkdir(parents=True, exist_ok=True)

    for attempt in range(1, max_retries + 1):
        try:
            with requests.get(url, stream=True, timeout=30) as r:
                r.raise_for_status()
                total = int(r.headers.get("content-length", 0))
                with open(target, "wb") as f, tqdm(
                    desc=repo_path.split("/")[-1],
                    total=total,
                    unit="B",
                    unit_scale=True,
                    disable=total == 0,
                ) as pbar:
                    for chunk in r.iter_content(chunk_size=chunk_size):
                        f.write(chunk)
                        pbar.update(len(chunk))
            return target
        except Exception as exc:
            if attempt == max_retries:
                raise
            sleep_sec = backoff * (2 ** (attempt - 1))
            print(f"Attempt {attempt} failed for {repo_path}: {exc}. Retry in {sleep_sec}s")
            time.sleep(sleep_sec)
    raise RuntimeError("Unreachable")
```
