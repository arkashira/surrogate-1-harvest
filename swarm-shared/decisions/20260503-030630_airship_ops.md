# airship / ops

## Highest-Value Incremental Improvement (≤2h)
**Goal**: Eliminate HF API 429s and Lightning idle-stop kills so Surrogate training iteration is **<2 minutes** and never blocked by rate limits or idle timeouts.

**Why this ships fastest**:
- One-time file-list pre-cache removes recursive `list_repo_files` → stops 429s.
- CDN-only fetches during training remove auth/rate-limit pressure.
- Studio reuse + idle-check prevents quota waste and silent training death.
- All changes are additive/config-level; no model or schema changes.

---

## Concrete Implementation Plan

### 1) Pre-cache repo file list (Mac orchestration)
- Run once (or cron after rate-limit window) for target date folder.
- Use `list_repo_tree(path, recursive=False)` per folder (non-recursive) to avoid 100x pagination.
- Save JSON to `airship/surrogate/configs/file_index_{date}.json`.
- Embed path in train script so Lightning workers do **zero** API calls during data load.

### 2) Switch data loader to CDN-only
- Replace `load_dataset(streaming=True)` with direct `hf_hub_download` (or raw CDN URLs) for listed files.
- Project to `{prompt, response}` only at parse time.
- Skip heterogeneous schema files safely; no CastError.

### 3) Lightning Studio reuse + idle guard
- Before `Studio(create_ok=True)`, list `Teamspace.studios` and reuse running studio.
- Wrap each `.run()` with status check; if stopped, restart with `target.start(machine=Machine.L40S)` (respects free-tier fallback).
- Set short timeouts and non-blocking polls so idle stop doesn’t kill training.

### 4) Cron/env hardening (prevent wrapper exec errors)
- Ensure all wrapper scripts have `#!/usr/bin/env bash` and `chmod +x`.
- Set `SHELL=/bin/bash` in crontab.
- Invoke via `bash script.sh "$@"` for consistency.

## Code Snippets

### 1) Pre-cache file list (run on Mac)
```bash
#!/usr/bin/env bash
# airship/surrogate/scripts/pre_cache_file_list.sh
set -euo pipefail

REPO="datasets/your-org/surrogate-mirror"
DATE_FOLDER="2026-04-29"
OUT="airship/surrogate/configs/file_index_${DATE_FOLDER}.json"

python3 - <<PY
import os, json
from huggingface_hub import HfApi

api = HfApi()
repo = os.getenv("REPO")
date_folder = os.getenv("DATE_FOLDER")

# non-recursive per folder to avoid pagination explosion
folders = [f"{date_folder}"]
file_list = []
for folder in folders:
    items = api.list_repo_tree(repo=repo, path=folder, recursive=False)
    for item in items:
        if item.get("type") == "file":
            file_list.append(item["path"])

os.makedirs(os.path.dirname(os.getenv("OUT")), exist_ok=True)
with open(os.getenv("OUT"), "w") as f:
    json.dump(file_list, f, indent=2)
print(f"Cached {len(file_list)} files -> {os.getenv('OUT')}")
PY
```

Make executable:
```bash
chmod +x airship/surrogate/scripts/pre_cache_file_list.sh
```

Crontab (set SHELL):
```bash
SHELL=/bin/bash
# Run after known HF rate-limit window clears
0 3 * * * /opt/axentx/airship/surrogate/scripts/pre_cache_file_list.sh >> /var/log/airship_pre_cache.log 2>&1
```

import json
import os
from pathlib import Path
from huggingface_hub import hf_hub_download

def load_file_list(date_folder: str):
    cfg = Path(__file__).parent / "configs" / f"file_index_{date_folder}.json"
    with open(cfg) as f:
        return json.load(f)

### 3) Lightning Studio reuse + idle guard
```python
# airship/surrogate/lightning_launcher.py
from lightning import Lightning, Teamspace, Studio, Machine
import time

def get_or_create_studio(name: str, machine: Machine = Machine.L40S):
    teamspace = Teamspace()
    for s in teamspace.studios:
        if s.name == name and s.status == "Running":
            print(f"Reusing running studio: {name}")
            return s
    print(f"Creating studio: {name}")
    return Studio(name=name, create_ok=True).start(machine=machine)

def run_training_with_idle_guard(studio_name: str, script: str, args: list):
    studio = get_or_create_studio(studio_name, Machine.L40S)
    # Ensure it's running (idle stop may have killed it)
    if studio.status != "Running":
        print("Studio stopped; restarting...")
        studio = studio.start(machine=Machine.L40S)

    # Non-blocking run with short poll interval to avoid idle timeout surprises
    run = studio.run(script, arguments=args, wait=False)
    while not run.is_done:
        time.sleep(10)
        run.refresh()
        # Optional: touch a keep-alive log if you need to prevent idle stop
    return run
```
