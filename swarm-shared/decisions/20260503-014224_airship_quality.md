## Final Synthesized Plan (Best of Both Candidates)

**Goal**: Eliminate HF API 429s during Surrogate training by switching to a **CDN-only ingestion pipeline** with a **pre-listed file manifest** and **deterministic sibling-repo sharding** for writes.

**Why this wins**:
- Bypasses `/api/` auth checks and rate limits entirely for reads (CDN tier has much higher limits).
- Single `list_repo_tree` call from Mac (after rate-limit window) → JSON manifest committed to repo → training script does **zero** API calls during data load.
- Deterministic shard selection spreads writes across 5 sibling repos (640/hr aggregate) to avoid 128/hr/repo cap.
- Projects mixed-schema files to `{prompt,response}` only before upload; moves attribution to filename pattern.
- No model code changes; no GPU quota changes; ships in <2h.

---

## Implementation Plan (≤2h)

### 1) Pre-list file paths (run once or per new date folder)
From Mac/CI (after rate-limit window), list one date folder and save manifest.

```bash
# Quick CLI usage (after rate-limit window)
python scripts/list_hf_date_folder.py \
  --repo datas \
  --date 2026-05-03 \
  --out batches/mirror-merged/2026-05-03/manifest.json
```

### 2) Update training data loader
- Remove `load_dataset(streaming=True)`.
- Read manifest JSON.
- For each entry: `hf_hub_download(repo_id, path, repo_type="dataset")` → local temp file.
- Parse only `{prompt, response}` fields; drop `source`, `ts`, extra cols.
- Stream to PyTorch IterableDataset or webdataset loader.

### 3) Add CDN-only download helper
- Use `https://huggingface.co/datasets/{repo}/resolve/main/{path}` with `requests` (no auth header).
- Retry with exponential backoff; respect 360s wait on 429.

### 4) Deterministic sibling-repo sharding for writes
- Hash slug → `repo_index = hash(slug) % 5`.
- Target sibling repos: `{main_repo}-s00` through `{main_repo}-s04`.
- Write parquet to `batches/mirror-merged/{date}/{slug}.parquet` with cols: `prompt`, `response`.

### 5) Lightning Studio reuse + idle handling
- Before `Studio(create_ok=True)`, list `Teamspace.studios` and reuse running studio with matching name.
- On each `.run()`, check status; if stopped, restart with `target.start(machine=Machine.L40S)`.

### 6) Scripts hygiene (apply past patterns)
- Ensure all wrapper scripts have `#!/usr/bin/env bash`, are `chmod +x`, invoked via `bash <script> "$@"`.
- Set `SHELL=/bin/bash` in crontab.

### scripts/list_hf_date_folder.py (run on Mac)
```python
#!/usr/bin/env python3
import argparse
import json
import os
from huggingface_hub import HfApi

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    api = HfApi()
    out_dir = os.path.dirname(args.out)
    os.makedirs(out_dir, exist_ok=True)

    # single call per folder (non-recursive)
    tree = api.list_repo_tree(
        repo_id=args.repo,
        path=f"batches/mirror-merged/{args.date}",
        repo_type="dataset",
        recursive=False
    )

    manifest = []
    for item in tree:
        if item.type == "file":
            manifest.append({
                "slug": os.path.splitext(os.path.basename(item.path))[0],
                "path": item.path,
                "size": getattr(item, "size", None)
            })

    with open(args.out, "w") as f:
        json.dump(manifest, f, indent=2)

    print(f"Manifest saved: {args.out} ({len(manifest)} files)")

if __name__ == "__main__":
    main()
```

### cdn_dataset.py (used in training)
```python
import json
import os
import requests
from torch.utils.data import IterableDataset
from huggingface_hub import hf_hub_download

class CDNDataset(IterableDataset):
    def __init__(self, repo_id, manifest_path, cache_dir=".cache"):
        self.repo_id = repo_id
        self.cache_dir = cache_dir
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        os.makedirs(cache_dir, exist_ok=True)

    def _download_cdn(self, path):
        # CDN bypass (no auth)
        url = f"https://huggingface.co/datasets/{self.repo_id}/resolve/main/{path}"
        out = os.path.join(self.cache_dir, os.path.basename(path))
        if os.path.exists(out):
            return out
        resp = requests.get(url, timeout=30)
        resp.raise_for_status()
        with open(out, "wb") as f:
            f.write(resp.content)
        return out

    def _parse_parquet(self, local_path):
        import pyarrow.parquet as pq
        tbl = pq.read_table(local_path, columns=["prompt", "response"])
        df = tbl.to_pandas()
        for _, row in df.iterrows():
            yield {"prompt": row["prompt"], "response": row["response"]}

    def __iter__(self):
        for item in self.manifest:
            try:
                # fallback to hf_hub_download if CDN fails (rare)
                local_path = hf_hub_download(
                    repo_id=self.repo_id,
                    filename=item["path"],
                    repo_type="dataset",
                    cache_dir=self.cache_dir
                )
            except Exception:
                local_path = self._download_cdn(item["path"])
            yield from self._parse_parquet(local_path)
```

### shard_writer.py (deterministic sibling sharding)
```python
import hashlib
import os
import pyarrow as pa
import pyarrow.parquet as pq

SIBLINGS = [f"your-org/your-dataset-s{i:03d}" for i in range(5)]

def pick_repo(slug):
    h = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    return SIBLINGS[h % len(SIBLINGS)]

def write_shard(slug, prompt, response, date="2026-05-03"):
    repo = pick_repo(slug)
    out_dir = f"batches/mirror-merged/{date}"
    os.makedirs(out_dir, exist_ok=True)
    out_path = os.path.join(out_dir, f"{slug}.parquet")
    tbl = pa.table({"prompt": [prompt], "response": [response]})
    pq.write_table(tbl, out_path)
    print(f"Shard written: {repo} -> {out_path}")
    return repo, out_path
```

### lightning_reuse.py (studio reuse + idle handling)
```python
from lightning_sdk import Teamspace, Studio, Machine

TEAMSPACE = "your-teamspace"
STUDIO_NAME = "surrogate-train-l40s"

def get_or_create_studio():
    team = Teamspace(TEAMSPACE)
    for s in team.studios:
        if s.name == STUDIO_NAME:
            if s.status == "running":
                print(f"Reusing running studio: {s.name}")
                return s
            else:
                print(f"Restarting stopped studio: {s.name}")
                s.start(machine=Machine.L40S)
                return s
    print(f"Creating new studio: {STUDIO_NAME}")
    return Studio.create(
        name=STUDIO_NAME,
        teamspace=TEAMSPACE,
        machine=Machine.L40S,
        create_ok=True
    )

def run_training_job(script_path):
    studio = get_or_create_studio()
    if studio.status != "running":
        studio.start(machine=Machine.L40S)
    run = studio.run(
        script=script_path,
        requirements=["torch", "pyarrow", "requests", "huggingface-hub"]
    )
    return run
```
