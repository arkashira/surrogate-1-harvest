# vanguard / ops

## Final Synthesis (single, actionable plan)

**Core principle:** Eliminate all Hugging Face Hub API calls during training; pin exact data by content-addressed manifest; enforce Lightning-only compute for training; reuse studios and survive idle-stop.

---

## 1. Diagnosis (resolved)
- **No CDN-first manifest** → generate once on orchestrator (Mac/CI) and commit to repo or S3.  
- **Runtime API calls** → forbid `list_repo_tree`/`load_dataset` in training workers; use only CDN URLs.  
- **No studio reuse** → list running studios by name and reuse; never blind-create.  
- **Idle-stop kills jobs** → treat idle-stop as expected; always check status and auto-restart studio before submit; checkpoint training so restart is safe.  
- **Mac compute leakage** → Mac only generates manifests and launches; all training runs on Lightning (L40S default; H200 only if account available).

---

## 2. Proposed change (single file + launcher + training contract)

Create `/opt/axentx/vanguard/ops/cdn_manifest.py` (≤150 LOC) that:
- Accepts HF dataset repo + date folder via CLI.
- Calls `list_repo_tree(..., recursive=True)` once (to capture full snapshot) and writes `manifest-{date}.json` containing:
  - `repo`, `date`, `generated_at_utc`
  - `files`: `{path, size, lfs_oid, commit_oid, sha256?, cdn_url}`
  - `tree_sha256` over deterministic JSON (sorted keys) to pin manifest content.
- Embeds this manifest into training so workers fetch via `https://huggingface.co/datasets/{repo}/resolve/main/{path}` with zero API calls.

Create `/opt/axentx/vanguard/ops/launch_studio.py` that:
- Lists existing studios by name; reuses if `status == "Running"`.
- If not running, (re)starts on L40S (`lightning-public-prod`); falls back to available machine types; never blind-creates new instances when reuse is possible.
- Before submit, polls until `status == "Running"` (with timeout); if stopped/idle, restarts and re-waits.
- Submits job and optionally waits; prints run ID and status.

Add a training contract (`/opt/axentx/vanguard/ops/README.md`) requiring:
- Manifest generated on orchestrator and shipped with code (or fetched from a pinned S3/GCS path).
- Training code imports manifest and streams only via CDN URLs (e.g., `requests.get(cdn_url, timeout=60, stream=True)` or HF `hf_transfer`/`wget`).
- No `load_dataset`, no `list_repo_tree`, no HF Hub API inside workers.
- Checkpointing enabled so studio restart does not lose progress.

# /opt/axentx/vanguard/ops/cdn_manifest.py
#!/usr/bin/env python3
"""
Generate a deterministic CDN-first manifest for a HuggingFace dataset date folder.
Usage:
  python cdn_manifest.py --repo datasets/mycorp/surrogate-1 --date 2026-04-29 --out manifests/
"""
import argparse
import hashlib
import json
import os
import sys
from datetime import datetime, timezone

try:
    from huggingface_hub import HfApi
except ImportError:
    print("Install: pip install huggingface_hub")
    sys.exit(1)

CDN_BASE = "https://huggingface.co/datasets"


def _deterministic_json_bytes(obj):
    return json.dumps(obj, sort_keys=True, separators=(",", ":")).encode()


def build_manifest(repo: str, date_folder: str, out_dir: str, recursive: bool = True):
    api = HfApi()
    try:
        items = api.list_repo_tree(repo=repo, path=date_folder, recursive=recursive)
    except Exception as e:
        print(f"list_repo_tree failed: {e}")
        sys.exit(1)

    files = []
    for item in items:
        if getattr(item, "type", None) != "file":
            continue
        path = item.path
        size = getattr(item, "size", None)
        lfs_oid = getattr(item, "lfs_oid", None)
        commit_oid = getattr(item, "oid", None) or getattr(item, "commit", "")
        sha256 = getattr(item, "sha256", None)
        cdn_url = f"{CDN_BASE}/{repo}/resolve/main/{path}"
        files.append({
            "path": path,
            "size": size,
            "lfs_oid": lfs_oid,
            "commit_oid": commit_oid,
            "sha256": sha256,
            "cdn_url": cdn_url,
        })

    manifest = {
        "repo": repo,
        "date": date_folder,
        "generated_at_utc": datetime.now(timezone.utc).isoformat(),
        "files": files,
    }
    manifest["tree_sha256"] = hashlib.sha256(_deterministic_json_bytes(manifest)).hexdigest()

    os.makedirs(out_dir, exist_ok=True)
    out_path = os.path.join(out_dir, f"manifest-{date_folder}.json")
    with open(out_path, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest written: {out_path}")
    print(f"Files: {len(files)}")
    return out_path


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Generate CDN-first manifest for HF dataset date folder.")
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., datasets/mycorp/surrogate-1)")
    parser.add_argument("--date", required=True, help="Date folder (e.g., 2026-04-29)")
    parser.add_argument("--out", default="manifests", help="Output directory (default: manifests)")
    parser.add_argument("--no-recursive", action="store_true", help="Disable recursive listing (not recommended)")
    args = parser.parse_args()
    build_manifest(args.repo, args.date, args.out, recursive=not args.no_recursive)

def find_running_studio(name: str):
    try:
        for s in Teamspace.studios:
            if getattr(s, "name", None) == name and getattr(s, "status", None) == "Running":
                return s
    except Exception:
        pass
    return None


def ensure_running(studio, max_wait: int = 300, poll_interval: int = 10):
    waited = 0
    while getattr(studio, "status", None) != "Running" and waited < max_wait:
        print(f"Studio status: {getattr(studio, 'status', 'unknown')}, waiting...")
        time.sleep(poll_interval)
        waited += poll_interval
        try:
            studio.refresh()
        except Exception:
            pass

    if getattr(studio, "status", None) != "Running":
        print(f"Studio not running after {max_wait}s: {getattr(studio, 'status', 'unknown')}")
