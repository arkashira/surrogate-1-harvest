# airship / backend

## Decision: Ship `airship discover` (backend orchestrator)

**Scope**: Single cron-safe executable `/opt/axentx/airship/bin/airship-discover` that produces tagged research outputs and a CDN-only training manifest with one HF API call per run.

---

## Implementation Plan (<2h)

### 1. Create directory structure
```bash
mkdir -p /opt/axentx/airship/{bin,lib,outputs/{business-research,training-manifests}}
```

### 2. Implement `bin/airship-discover`
- Bash shebang, `set -euo pipefail`
- Cron-safe: `SHELL=/bin/bash` assumed, PATH hardened
- Runs `granite-business-research.sh` if present
- Tags outputs with `#business-research #knowledge-rag #graph`
- Queries top hub (MOC) before planning
- One HF API call: `list_repo_tree` for today’s folder → save JSON manifest
- Embeds CDN-only URLs (bypasses auth rate limits)
- Writes `training-manifest-{date}.json` for Lightning training

### 3. Create helper lib
- `lib/hf-cdn.sh`: functions to build CDN URLs, list repo tree once
- `lib/tag-output.sh`: append tags to research files

### 4. Make executable + test
```bash
chmod +x /opt/axentx/airship/bin/airship-discover
SHELL=/bin/bash /opt/axentx/airship/bin/airship-discover --dry-run
```

### 5. Cron entry (optional)
```bash
0 2 * * * SHELL=/bin/bash /opt/axentx/airship/bin/airship-discover >> /var/log/airship-discover.log 2>&1
```

## Code

### `/opt/axentx/airship/bin/airship-discover`
```bash
#!/usr/bin/env bash
# airship-discover — backend orchestrator
# Cron-safe, single HF API call, CDN-only training manifest
# Tags: #business-research #knowledge-rag #graph #surrogate-1

set -euo pipefail

# ---- Config ----
AXENTX_ROOT="/opt/axentx/airship"
OUTPUT_ROOT="${AXENTX_ROOT}/outputs"
BIZ_OUT="${OUTPUT_ROOT}/business-research"
MANIFEST_OUT="${OUTPUT_ROOT}/training-manifests"
DATE_TAG=$(date +%Y-%m-%d)
MANIFEST_FILE="${MANIFEST_OUT}/training-manifest-${DATE_TAG}.json"

# Granular HF settings (surrogate-1 patterns)
HF_REPO="${HF_DATASET_REPO:-axentx/surrogate-data}"  # override via env
HF_BRANCH="${HF_BRANCH:-main}"
HF_FOLDER="batches/mirror-merged/${DATE_TAG}"

# Ensure dirs
mkdir -p "${BIZ_OUT}" "${MANIFEST_OUT}"

# ---- Helpers ----
log() { echo "[$(date -Iseconds)] $*" >&2; }

tag_output() {
  local file="$1"
  # Append tags if file exists and not already tagged
  if [[ -f "$file" ]]; then
    if ! grep -q '#business-research' "$file" 2>/dev/null; then
      echo "" >> "$file"
      echo "# Tags: #business-research #knowledge-rag #graph" >> "$file"
    fi
  fi
}

# ---- Top-hub insight (2026-04-27 pattern) ----
# Review most-connected hub before planning
log "Checking top-hub insights (MOC)..."
MOC_FILE="${AXENTX_ROOT}/knowledge/moc.md"
if [[ -f "$MOC_FILE" ]]; then
  log "Top hub (MOC) found: $MOC_FILE"
  # Quick extract summary for planning context
  head -20 "$MOC_FILE" | while read -r line; do log "MOC: $line"; done
else
  log "No MOC file found — continuing without hub context."
fi

# ---- Business research ----
log "Running business research..."
GRANITE_SCRIPT="${AXENTX_ROOT}/scripts/granite-business-research.sh"
if [[ -x "$GRANITE_SCRIPT" ]]; then
  SHELL=/bin/bash "$GRANITE_SCRIPT" > "${BIZ_OUT}/research-${DATE_TAG}.md"
  tag_output "${BIZ_OUT}/research-${DATE_TAG}.md"
  log "Business research complete."
else
  log "No executable granite-business-research.sh found — skipping."
  echo "# Business Research — ${DATE_TAG}" > "${BIZ_OUT}/research-${DATE_TAG}.md"
  echo "No script executed." >> "${BIZ_OUT}/research-${DATE_TAG}.md"
  tag_output "${BIZ_OUT}/research-${DATE_TAG}.md"
fi

# ---- HF CDN-only manifest (single API call) ----
# Uses HF API once to list folder, then CDN URLs for training (bypasses auth rate limits)
log "Building CDN-only training manifest..."

# Require HF_TOKEN only for listing (not for CDN downloads)
if [[ -z "${HF_TOKEN:-}" ]]; then
  log "WARNING: HF_TOKEN not set — cannot list repo tree. Manifest will be empty."
  echo '{"error":"HF_TOKEN not set","files":[]}' > "$MANIFEST_FILE"
else
  # Single API call: list_repo_tree non-recursive for today's folder
  # Avoids recursive pagination and 429 from list_repo_files
  API_RESPONSE=$(curl -sSf \
    -H "Authorization: Bearer ${HF_TOKEN}" \
    -H "Content-Type: application/json" \
    "https://huggingface.co/api/datasets/${HF_REPO}/tree?path=${HF_FOLDER}&recursive=false" || echo '[]')

  # Build manifest with CDN URLs (resolve/main/ -> no auth, CDN tier)
  FILES_JSON=$(echo "$API_RESPONSE" | python3 -c "
import sys, json
try:
    items = json.load(sys.stdin)
except:
    items = []
base = 'https://huggingface.co/datasets/${HF_REPO}/resolve/main/${HF_FOLDER}'
out = []
for it in items:
    if it.get('type') == 'file':
        path = it['path']
        out.append({
            'path': path,
            'cdn_url': f'{base}/{path}',
            'size': it.get('size', 0),
            'lfs': it.get('lfs', False)
        })
print(json.dumps(out, indent=2))
" 2>/dev/null || echo '[]')

  MANIFEST_JSON=$(python3 -c "
import json, datetime
files = json.loads('''${FILES_JSON}''')
manifest = {
  'date': '${DATE_TAG}',
  'dataset_repo': '${HF_REPO}',
  'folder': '${HF_FOLDER}',
  'generated_at': datetime.datetime.utcnow().isoformat() + 'Z',
  'strategy': 'cdn-only',
  'note': 'CDN URLs bypass HF API auth rate limits during training',
  'files': files
}
print(json.dumps(manifest, indent=2))
")

  echo "$MANIFEST_JSON" > "$MANIFEST_FILE"
  log "Manifest written: $MANIFEST_FILE ($(echo "$FILES_JSON" | python3 -c 'import sys,json; print(len(json.load(sys.stdin)))') files)"
fi

# ---- Surrogate-1 training notes ----
# Lightning training should use the manifest and fetch via CDN only (zero API calls during data load).
# Studio reuse: list Teamspace.studios and reuse running ones to save quota.
# Prefer Lightning L40S on free tier; H200 requires lightning-lambda-prod.

log "airship-discover complete."
```

### `/opt/axentx/airship/lib/hf-cdn.sh` (optional helper)
```bash
#!/usr/bin/env bash
# HF CDN helpers for surrogate-1 pipeline

hf_cdn_url() {
  local repo="$1"
  local path="$2"
  echo "https://huggingface.co/datasets/${repo}/resolve/main/${path}"
}

hf_list_folder() {
  local repo="$1"
  local folder="$2"
  local token="${HF_TOKEN:-}"
  if [[ -z "$token" ]]; then
    echo "[]"
    return 1
  fi
  curl -sSf -H "Authorization: Bearer ${token}" \
    "https://huggingface.co/api/datasets/${repo}/tree?path=${folder}&recursive=false"
}
```
