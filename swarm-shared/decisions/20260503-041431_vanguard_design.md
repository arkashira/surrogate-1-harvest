# vanguard / design

## Final Synthesis (Best Parts + Correctness + Actionability)

I merged the strongest, non-redundant insights from both candidates and resolved contradictions in favor of correctness and concrete actionability.

### Diagnosis (merged, tightened)
- No content-addressed manifest → training/frontend hit HF API at runtime (429s, non-reproducible snapshots).
- Ingestion writes mixed-schema files (`source`, `ts` extra columns) → downstream `load_dataset`/PyArrow casts fail (`CastError`).
- Runtime `list_repo_tree`/`load_dataset` instead of CDN-only, deterministic file list → rate-limited, slow, and non-reproducible.
- No deterministic snapshot hash → cannot share, resume, or verify exact dataset state across runs.

### Proposed change (merged + corrected)
Add a content-addressed manifest generator and a CDN-only data loader for Surrogate-1 training:
- New file: `/opt/axentx/vanguard/scripts/build_manifest.py`
- Modify: `/opt/axentx/vanguard/training/train.py` (data loader section)
- Scope: generate `manifests/{date}/{sha256_prefix}.json` listing `{repo}/{path}` for one date folder; embed manifest in training script to enable CDN-only fetches with zero API calls during training.

**Correction on repo path**: Candidate 1 used `batches/mirror-merged/{date}` in the builder but `datasets/mirror-merged` as repo; Candidate 2 used `enriched/{date}`. Use the actual ingestion output location (`enriched/`) for manifests, but keep repo `datasets/mirror-merged`. Builder will list `enriched/{date}` and produce CDN URLs under that path. This matches ingestion output and avoids schema mismatch.

### Implementation (merged + hardened)

#### 3.1 Create manifest builder (hardened)
```bash
# /opt/axentx/vanguard/scripts/build_manifest.py
#!/usr/bin/env python3
"""
Build content-addressed manifest for one date folder (enriched/).
Usage:
  python build_manifest.py --repo datasets/mirror-merged --date 2026-05-03 --out manifests/2026-05-03/manifest.json
"""
import argparse
import hashlib
import json
import os
import sys
from datetime import datetime, timezone
from pathlib import Path

try:
    from huggingface_hub import list_repo_tree
except ImportError:
    print("Install: pip install huggingface_hub")
    sys.exit(1)

def build_manifest(repo: str, date: str, out_path: str):
    # Use enriched/ to match ingestion output schema
    folder_path = f"enriched/{date}"
    print(f"Listing {repo}/{folder_path} ...")

    # Single non-recursive call (avoids pagination/429)
    tree = list_repo_tree(repo=repo, path=folder_path, recursive=False)
    files = [item.rfilename for item in tree if item.type == "file" and item.rfilename.endswith(".parquet")]

    if not files:
        print(f"No parquet files found in {folder_path}")
        sys.exit(1)

    entries = []
    for f in sorted(files):
        # CDN URL (no auth, bypasses /api/ rate limits)
        cdn_url = f"https://huggingface.co/datasets/{repo}/resolve/main/{f}"
        slug = os.path.splitext(os.path.basename(f))[0]
        entries.append({
            "slug": slug,
            "repo": repo,
            "path": f,
            "cdn_url": cdn_url,
            "size": None  # can be populated later if needed
        })

    manifest = {
        "created_at": datetime.now(timezone.utc).isoformat(),
        "repo": repo,
        "date": date,
        "folder": folder_path,
        "sha256": hashlib.sha256(json.dumps(entries, sort_keys=True).encode()).hexdigest(),
        "entries": entries
    }

    out = Path(out_path)
    out.parent.mkdir(parents=True, exist_ok=True)
    with open(out, "w") as fp:
        json.dump(manifest, fp, indent=2)
    print(f"Manifest written to {out} ({len(entries)} entries)")
    return manifest

if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Build CDN-only manifest for a date folder.")
    parser.add_argument("--repo", default="datasets/mirror-merged", help="HF dataset repo")
    parser.add_argument("--date", required=True, help="Date folder (YYYY-MM-DD)")
    parser.add_argument("--out", required=True, help="Output manifest JSON path")
    args = parser.parse_args()
    build_manifest(args.repo, args.date, args.out)
```

Make executable:
```bash
chmod +x /opt/axentx/vanguard/scripts/build_manifest.py
```

#### 3.2 Update training loader to use manifest + CDN-only (robust)
Patch `/opt/axentx/vanguard/training/train.py` (data section):

```python
# Near top
import json
import os
import tempfile
from pathlib import Path
from typing import Dict, List

import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset, DataLoader

# --- CDN-only dataset loader (no HF API calls during training) ---
class CDNParquetIterable(IterableDataset):
    def __init__(self, manifest_path: str, start_index: int = 0, end_index: int = None, columns: List[str] = None):
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.entries = self.manifest["entries"]
        if end_index is None:
            end_index = len(self.entries)
        self.entries = self.entries[start_index:end_index]
        self.columns = columns or ["prompt", "response"]

    def _stream_file(self, entry: Dict):
        # CDN fetch; no Authorization header required for public files
        resp = requests.get(entry["cdn_url"], stream=True, timeout=60)
        resp.raise_for_status()
        # Write to temp file to let pyarrow memory-map efficiently
        with tempfile.NamedTemporaryFile(suffix=".parquet", delete=False) as tmp:
            for chunk in resp.iter_content(chunk_size=8192):
                tmp.write(chunk)
            tmp_path = tmp.name
        try:
            # Explicitly request only the columns we need; coerce schema safely
            table = pq.read_table(tmp_path, columns=self.columns, use_threads=False)
            # Convert to list of dicts per row
            for batch in table.to_batches(max_chunksize=1024):
                for row in batch.to_pylist():
                    # Ensure only requested columns are present
                    yield {k: row[k] for k in self.columns}
        finally:
            try:
                os.unlink(tmp_path)
            except Exception:
                pass

    def __iter__(self):
        for entry in self.entries:
            yield from self._stream_file(entry)

# Usage in training script
def get_dataloader(manifest_path: str, batch_size: int = 8, num_workers: int = 2, columns: List[str] = None):
    dataset = CDNParquetIterable(manifest_path=manifest_path, columns=columns)
    return DataLoader(dataset, batch_size=batch_size, num_workers=num_workers, pin_memory=True)
```

Notes:
- This avoids `load_dataset` and runtime `list_repo_tree` → zero HF API calls after manifest creation.
- Manifest is content-addressed (`sha256` over entries) → reproducible snapshots.
- Explicit column selection and per-row dict projection avoids mixed-schema pyarrow errors.
- Temp file usage ensures pyarrow can mmap efficiently; cleaned up reliably.
