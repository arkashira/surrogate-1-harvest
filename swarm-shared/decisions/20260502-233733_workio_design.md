# workio / design

**Most Valuable Next Improvement for Workio Design**

### Diagnosis

- **Missing idempotency key on punch records** → LINE webhook redeliveries and concurrent deliveries create duplicate clock-in/out rows.
- **Non-atomic read-then-insert for punch state** → races between concurrent webhooks produce multiple open punches or lost state.
- **No transactional boundary around punch state updates** → concurrent or redelivered webhooks can overwrite existing punches.
- **No validation for punch state transitions** → invalid state transitions can occur due to concurrent or redelivered webhooks.

### Proposed Change

- **Add idempotency key and optimistic concurrency control for punch records**

### Implementation

1. **Update `punch` model to include a unique `idempotency_key` field**:

```diff
// server/src/models/punch.ts
export class Punch {
  id: string;
  idempotencyKey: string;
  employeeId: string;
  clockIn: boolean;
  timestamp: Date;
  version: number;
}
```

2. **Update `punch` repository to use idempotency key for updates**:

```diff
// server/src/repositories/punch.ts
import { Punch } from '../models/punch';

export async function updatePunch(punch: Punch) {
  const existingPunch = await this.findOne({ idempotencyKey: punch.idempotencyKey });
  if (existingPunch) {
    // Update existing punch if it exists
    if (existingPunch.version !== punch.version) {
      throw new Error('Optimistic concurrency failure');
    }
    await this.updateOne({ idempotencyKey: punch.idempotencyKey }, punch);
  } else {
    // Create new punch if it doesn't exist
    await this.create(punch);
  }
}
```

3. **Update `punch` service to use optimistic concurrency control**:

```diff
// server/src/services/punch.ts
import { Punch } from '../models/punch';
import { updatePunch } from '../repositories/punch';

export async function clockIn(employeeId: string) {
  const punch = new Punch({
    idempotencyKey: generateIdempotencyKey(),
    employeeId,
    clockIn: true,
    timestamp: new Date(),
    version: 1,
  });
  await updatePunch(punch);
}

export async function clockOut(employeeId: string) {
  const punch = new Punch({
    idempotencyKey: generateIdempotencyKey(),
    employeeId,
    clockIn: false,
    timestamp: new Date(),
    version: 1,
  });
  await updatePunch(punch);
}
```
