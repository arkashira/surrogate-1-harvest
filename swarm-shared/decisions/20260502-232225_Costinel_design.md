# Costinel / design

## Final Synthesized Implementation Plan — Costinel Discovery: Top-Hub Signal Endpoint

**Core principle:**  
Ship a **read-only, side-effect-free** `GET /api/v1/cost-anomaly/signal/top-hub` that surfaces the most-connected hub and actionable triage context in **<2 hours**, using precomputed cache and graceful fallback.  
**Philosophy:** *Sense + Signal — ไม่ Execute*.

---

### 1) Scope & Response Contract (unified)
Return a single, cache-friendly payload with these required fields:

```json
{
  "hubId": "MOC",
  "hubName": "MOC",
  "hubType": "cost-center",
  "score": 42,
  "connections": 42,
  "topSignals": [
    {
      "signalId": "sig-001",
      "type": "compute-spike",
      "severity": "high",
      "description": "Weekday morning compute spikes",
      "affectedAccounts": ["prod-123", "prod-456"]
    }
  ],
  "insights": [
    "Frequent compute spikes on weekday mornings",
    "High cross-account data transfer costs",
    "Underutilized reserved instance coverage (~38%)"
  ],
  "contextDocs": [
    { "docId": "d1", "title": "MOC Cost Patterns", "snippet": "..." }
  ],
  "generatedAt": 1712345678,
  "ttl": 300,
  "source": "knowledge-graph"
}
```

- **No side effects**: no writes, no mutations, no external calls during request.
- **Fast path**: ≤50 ms p95 via precomputed cache (file or Redis).
- **Fallback**: static safe default if cache/graph unavailable.

```python
# src/services/top_hub_signal.py
import time
import json
from pathlib import Path
from typing import Dict, Any

class TopHubSignalService:
    def __init__(self, cache_path: str = "data/top-hub-cache.json", ttl: int = 300):
        self.cache_path = Path(cache_path)
        self.ttl = ttl
        self._static_fallback = {
            "hubId": "MOC",
            "hubName": "MOC",
            "hubType": "cost-center",
            "score": 42,
            "connections": 42,
            "topSignals": [
                {
                    "signalId": "sig-demo",
                    "type": "compute-spike",
                    "severity": "high",
                    "description": "Weekday morning compute spikes",
                    "affectedAccounts": []
                }
            ],
            "insights": [
                "Frequent compute spikes on weekday mornings",
                "High cross-account data transfer costs",
                "Underutilized reserved instance coverage (~38%)"
            ],
            "contextDocs": [],
            "generatedAt": int(time.time()),
            "ttl": self.ttl,
            "source": "fallback"
        }

    def _load_cache(self) -> Dict[str, Any] | None:
        try:
            if not self.cache_path.exists():
                return None
            data = json.loads(self.cache_path.read_text())
            gen = data.get("generatedAt", 0)
            if time.time() - gen > self.ttl:
                return None
            return data
        except Exception:
            return None

    def get_top_hub(self) -> Dict[str, Any]:
        cached = self._load_cache()
        if cached:
            # Normalize field names if needed
            cached.setdefault("connections", cached.get("score", 0))
            cached.setdefault("insights", [])
            cached.setdefault("contextDocs", [])
            cached["source"] = "knowledge-graph"
            return cached
        return self._static_fallback
```

```typescript
// src/services/topHubSignal.service.ts
import fs from 'fs';
import path from 'path';

export class TopHubSignalService {
  private cachePath = path.resolve(__dirname, '../../../data/top-hub-cache.json');
  private ttl = 300; // 5m

  private staticFallback = {
    hubId: 'MOC',
    hubName: 'MOC',
    hubType: 'cost-center',
    score: 42,
    connections: 42,
    topSignals: [
      {
        signalId: 'sig-demo',
        type: 'compute-spike',
        severity: 'high',
        description: 'Weekday morning compute spikes',
        affectedAccounts: []
      }
    ],
    insights: [
      'Frequent compute spikes on weekday mornings',
      'High cross-account data transfer costs',
      'Underutilized reserved instance coverage (~38%)'
    ],
    contextDocs: [],
    generatedAt: Date.now(),
    ttl: this.ttl,
    source: 'fallback'
  };

  private loadCache() {
    try {
      if (!fs.existsSync(this.cachePath)) return null;
      const raw = fs.readFileSync(this.cachePath, 'utf8');
      const data = JSON.parse(raw);
      if (Date.now() - data.generatedAt > this.ttl * 1000) return null;
      return data;
    } catch {
      return null;
    }
  }

  async getTopHub() {
    const cached = this.loadCache();
    if (cached) {
      cached.connections = cached.connections ?? cached.score ?? 0;
      cached.insights = cached.insights ?? [];
      cached.contextDocs = cached.contextDocs ?? [];
      cached.source = 'knowledge-graph';
      return cached;
    }
    return this.staticFallback;
  }
}
```

### 3) Cache & Offline Job (recommended)
- Generate `data/top-hub-cache.json` via a scheduled job (e.g., every 5–15 minutes).
- Job queries your knowledge-graph for top hub by degree/centrality and latest anomaly signals/docs.
- Keep the job **separate** from the API to guarantee fast, side-effect-free reads.
