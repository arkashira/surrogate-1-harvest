# Costinel / design

## 1. Diagnosis

- No machine-readable manifest of ingestible sources (cloud accounts, datasets, schemas) → onboarding requires manual inspection and trial/error.
- No readiness/health probe endpoint or design-time validation → environment misconfigurations surface only at runtime.
- Signal taxonomy is implicit (scattered in code/comments) → hard to compose, extend, or audit which signals exist and their contract.
- No lightweight discovery CLI or design-time query for “what can I connect / what will it emit” → slows architecture reviews and CI checks.
- Missing explicit schema contract for recommendations (RI, orphaned resources, forecast) → downstream consumers can’t validate payloads reliably.

## 2. Proposed change

Add a **design-time discovery layer** with minimal runtime footprint:

- `src/discovery/manifest.yaml` — declarative catalog of ingestible sources, required auth, and emitted signal schemas.
- `src/discovery/schema/` — JSON Schema files for each signal (recommendation, anomaly, forecast).
- `src/discovery/validate.py` — CLI (`python -m costinel.discovery validate`) that checks environment readiness and validates local config against manifest.
- `src/discovery/__init__.py` exports `load_manifest()` and `signal_schema(name)` for runtime use.
- Update `README.md` with a “Discovery” section showing sample usage.

Scope: ~120–180 lines total; no service code changes; safe to ship.

### 3.4 `src/discovery/__init__.py`

```python
from pathlib import Path
import yaml
import json

_ROOT = Path(__file__).parent
_MANIFEST_PATH = _ROOT / "manifest.yaml"
_SCHEMA_DIR = _ROOT / "schema" / "signals"

def load_manifest():
    with open(_MANIFEST_PATH) as f:
        return yaml.safe_load(f)

def signal_schema(name: str):
    path = _SCHEMA_DIR / f"{name}.json"
    if not path.exists():
        raise ValueError(f"Unknown signal schema: {name}")
    with open(path) as f:
        return json.load(f)
```

Usage: python -m costinel.discovery validate [--strict]
"""

    print("Missing pyyaml. Install: pip install pyyaml")
    sys.exit(1)

    missing = [k for k in required if not os.getenv(k)]
