# surrogate-1 / design

## Final Synthesized Answer

**Goal**: Eliminate HF API rate-limit failures (429) and ingestion OOM by switching to CDN-only fetches and per-folder tree listing. This is the highest-value ≤2h improvement.

**Why this wins**:
- Removes recursive `list_repo_files` calls that hit the 1000 req/5min API limit.
- Uses CDN URLs (`resolve/main/...`) which bypass auth and have much higher rate limits.
- Per-folder `list_repo_tree(..., recursive=False)` keeps API usage minimal and deterministic.
- Preserves existing 16-shard matrix and dedup behavior (zero behavioral regression).
- Concrete, actionable, and mergeable in ≤2h.

---

## Implementation Plan (≤2h)

1. **Add CDN-bypass mode flag** to `bin/dataset-enrich.sh` (default: on).
2. **Replace recursive file listing** with per-folder `list_repo_tree` (non-recursive) once per run; save file list to JSON.
3. **Download via CDN URLs** using `requests` with retry/backoff (separate from API limits).
4. **Parse only `{prompt,response}` fields** at read time; drop all other columns to minimize memory.
5. **Keep dedup via `lib/dedup.py`** unchanged.
6. **Update upload path** to `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl` (unchanged).
7. **Smoke test** locally with a small folder slice.

# Determine shard slice by deterministic hash of filename

import json, hashlib, os, sys

files = json.loads(os.environ["FILELIST"])
shard_id = int(os.environ["SHARD_ID"])
total_shards = int(os.environ["TOTAL_SHARDS"])
out_file = os.environ["OUT_FILE"]

def shard_for(fname, total):
    # Deterministic, stable across runs
    h = int(hashlib.sha256(fname.encode()).hexdigest(), 16)
    return h % total

selected = [f for f in files if shard_for(f, total_shards) == shard_id]
print(f"Shard {shard_id}/{total_shards} processing {len(selected)} files")

# Pass selected files to the enrichment script via env
os.environ["SELECTED_FILES"] = json.dumps(selected)

  --date "${DATE}" \
  --shard-id "${SHARD_ID}" \

  --out "${OUT_FILE}" \

#!/usr/bin/env python3
"""
CDN-bypass enrichment worker for surrogate-1.
Avoids load_dataset(streaming=True) and HF API list_repo_files.
Uses precomputed file list and CDN downloads only.
"""
import argparse
import hashlib
import json
import os
import sys
import time
from pathlib import Path
from typing import List, Dict, Any

import pyarrow as pa
import pyarrow.parquet as pq
import requests
from tqdm import tqdm

HF_CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def download_via_cdn(repo: str, path: str, retries: int = 3, backoff: float = 2.0) -> bytes:
    url = HF_CDN_TEMPLATE.format(repo=repo, path=path)
    for attempt in range(1, retries + 1):
        try:
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            return resp.content
        except Exception as exc:
            if attempt == retries:
                raise
            sleep = backoff ** attempt
            print(f"[cdn] Retry {attempt}/{retries} for {path} after {sleep}s: {exc}", file=sys.stderr)
            time.sleep(sleep)

def parse_parquet_to_pairs(content: bytes) -> List[Dict[str, str]]:
    """
    Parse parquet bytes and project to {prompt, response}.
    Handles mixed schemas gracefully: missing fields become empty strings.
    """
    try:
        table = pq.read_table(pa.BufferReader(content))
    except Exception as exc:
        print(f"[parse] Failed to read parquet: {exc}", file=sys.stderr)
        return []

    pairs = []
    df = table.to_pandas()
    for _, row in df.iterrows():
        prompt = str(row.get("prompt", row.get("input", row.get("question", "")))).strip()
        response = str(row.get("response", row.get("output", row.get("answer", "")))).strip()
        if prompt and response:
            pairs.append({"prompt": prompt, "response": response})
    return pairs

def compute_md5(text: str) -> str:
    return hashlib.md5(text.encode()).hexdigest()

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date", required=True)
    parser.add_argument("--shard-id", type=int, required=True)
    parser.add_argument("--total-shards", type=int, required=True)
