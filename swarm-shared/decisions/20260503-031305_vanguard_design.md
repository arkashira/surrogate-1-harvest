# vanguard / design

## 1. Diagnosis

- No deterministic **CDN-first manifest** exists; ingestion/training scripts likely still call Hugging Face API (`list_repo_tree`, `load_dataset`) at runtime, risking 429s and non-reproducible runs.
- Missing **content-addressed file list** keyed by date/slug; training jobs cannot pin exact file set and may silently drift across HF repo changes.
- No **guardrails** to prevent local model loading or heavy compute on Mac; architecture rule “Mac=CLI only” is not enforced.
- No **Lightning Studio reuse** pattern implemented; each run risks quota waste via recreation and idle-stop training loss.
- No **schema projection** step before upload; enriched/ may still contain mixed schema fields (`source`, `ts`) violating surrogate-1 schema discipline.

## 2. Proposed change

Add a small, high-leverage design artifact and two lightweight scripts:

- `/opt/axentx/vanguard/design/cdn-manifest.json` (generated) — deterministic list of CDN-resolved file URLs for a single date folder, produced once on Mac and embedded in training.
- `/opt/axentx/vanguard/scripts/gen-cdn-manifest.py` — one-shot Mac script: uses HF API once (after rate-limit window) to list files for a date folder and writes `cdn-manifest.json` with `https://huggingface.co/datasets/.../resolve/main/...` URLs + sha256.
- `/opt/axentx/vanguard/scripts/run-lightning-studio.sh` — wrapper that:
  - reuses an existing running Studio by name,
  - starts L40S (or falls back to public-tier),
  - mounts `cdn-manifest.json` into the Studio,
  - executes training with zero HF API calls.

Scope: new files + one optional patch to training entrypoint to read `cdn-manifest.json` and fetch via CDN only.

### 3.1 `scripts/gen-cdn-manifest.py`

```python
#!/usr/bin/env python3
"""
Generate a deterministic CDN-first manifest for one date folder.
Run on Mac (or any dev host) after HF API rate-limit window clears.

Usage:
  HF_TOKEN=<token> python gen-cdn-manifest.py \
    --repo <datasets/username/repo> \
    --date-folder 2026-04-29 \
    --out /opt/axentx/vanguard/design/cdn-manifest.json
"""

import argparse
import hashlib
import json
import os
import sys
from typing import List, Dict

import requests

HF_API_BASE = "https://huggingface.co/api"
CDN_BASE = "https://huggingface.co/datasets"

def list_date_folder(repo: str, date_folder: str, token: str) -> List[str]:
    """
    Non-recursive list for one date folder.
    Uses repo tree endpoint (non-recursive) to minimize API calls.
    """
    url = f"{HF_API_BASE}/datasets/{repo}/tree"
    headers = {}
    if token:
        headers["Authorization"] = f"Bearer {token}"

    params = {"path": date_folder, "recursive": "false"}
    resp = requests.get(url, headers=headers, params=params, timeout=30)
    if resp.status_code == 429:
        print("HF API 429 — wait 360s before retry", file=sys.stderr)
        sys.exit(1)
    resp.raise_for_status()
    items = resp.json()
    paths = []
    for item in items:
        if item.get("type") == "file":
            paths.append(f"{date_folder}/{item['path']}")
    return paths

def build_manifest(repo: str, file_paths: List[str]) -> List[Dict]:
    entries = []
    for path in sorted(file_paths):
        url = f"{CDN_BASE}/{repo}/resolve/main/{path}"
        entries.append(
            {
                "repo": repo,
                "path": path,
                "cdn_url": url,
                # Note: sha256 requires HEAD/GET; optional for now.
                # Can be filled lazily during training validation.
                "sha256": None,
            }
        )
    return entries

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate CDN manifest")
    parser.add_argument("--repo", required=True, help="datasets/username/repo")
    parser.add_argument("--date-folder", required=True, help="e.g. 2026-04-29")
    parser.add_argument("--out", required=True, help="Output JSON path")
    args = parser.parse_args()

    token = os.environ.get("HF_TOKEN", "")
    paths = list_date_folder(args.repo, args.date_folder, token)
    if not paths:
        print(f"No files found in {args.repo}/{args.date_folder}", file=sys.stderr)
        sys.exit(1)

    manifest = {
        "generated_by": "gen-cdn-manifest.py",
        "repo": args.repo,
        "date_folder": args.date_folder,
        "files": build_manifest(args.repo, paths),
    }

    os.makedirs(os.path.dirname(os.path.abspath(args.out)), exist_ok=True)
    with open(args.out, "w", encoding="utf-8") as f:
        json.dump(manifest, f, indent=2, sort_keys=True)
    print(f"Wrote {len(paths)} entries to {args.out}")

if __name__ == "__main__":
    main()
```
