# airship / design

## Implementation Plan: `airship discover` — Zero-config discovery CLI

**Highest-value incremental improvement (<2h):**  
Ship a single Bash CLI (`airship discover`) that operationalizes past patterns (business research + knowledge-rag + top-hub/MOC insight) into one command that surfaces actionable context for the Arkship + Surrogate platform.

---

### Design Decisions
- **Single entrypoint**: `/opt/axentx/airship/bin/airship` (symlink or script) with `discover` subcommand.
- **Zero-config**: Uses sensible defaults (project root, logs, temp dirs) and environment overrides.
- **Pattern compliance**:
  - Runs `granite-business-research.sh` (if present) or falls back to lightweight market scan.
  - Executes `knowledge-rag` query for top hub and related docs.
  - Extracts and prints top-hub/MOC insight (most-connected node) for planning.
- **Robustness**: Bash shebang, `set -euo pipefail`, executable bit, `SHELL=/bin/bash` friendly.
- **Outputs**:
  - Human-readable summary to stdout.
  - JSON report to `./reports/discover-<ts>.json` for CI/automation.

---

### File Changes (3 files, ~120 lines total)

#### 1) `/opt/axentx/airship/bin/airship`
```bash
#!/usr/bin/env bash
# airship — zero-config CLI for Arkship + Surrogate platform
# Usage: airship <command> [args...]

set -euo pipefail
SHELL=/bin/bash

ROOT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
REPORTS_DIR="${ROOT_DIR}/reports"
LOG_DIR="${ROOT_DIR}/logs"

mkdir -p "${REPORTS_DIR}" "${LOG_DIR}"

cmd="${1:-help}"
shift || true

case "${cmd}" in
  discover)
    exec "${ROOT_DIR}/scripts/discover.sh" "$@"
    ;;
  help|--help|-h)
    cat <<EOF
airship — Arkship + Surrogate CLI

Commands:
  discover   Run zero-config discovery (business research + knowledge-rag + top-hub insight)
  help       Show this help

Environment:
  AIRSHIP_ROOT        Project root (default: ${ROOT_DIR})
  REPORTS_DIR         Reports output directory (default: ${REPORTS_DIR})
  LOG_DIR             Logs directory (default: ${LOG_DIR})
EOF
    exit 0
    ;;
  *)
    echo "Unknown command: ${cmd}" >&2
    echo "Run 'airship help' for usage." >&2
    exit 1
    ;;
esac
```

#### 2) `/opt/axentx/airship/scripts/discover.sh`
```bash
#!/usr/bin/env bash
# discover.sh — zero-config discovery: business research + knowledge-rag + top-hub insight

set -euo pipefail
SHELL=/bin/bash

ROOT_DIR="${AIRSHIP_ROOT:-$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)}"
REPORTS_DIR="${REPORTS_DIR:-${ROOT_DIR}/reports}"
LOG_DIR="${LOG_DIR:-${ROOT_DIR}/logs}"

TS="$(date -u +%Y%m%dT%H%M%SZ)"
REPORT_FILE="${REPORTS_DIR}/discover-${TS}.json"
LOG_FILE="${LOG_DIR}/discover-${TS}.log"

mkdir -p "${REPORTS_DIR}" "${LOG_DIR}"

log() {
  echo "[$(date -u +'%Y-%m-%dT%H:%M:%SZ')] $*" | tee -a "${LOG_FILE}"
}

run_step() {
  local name="$1"
  shift
  log "START: ${name}"
  if "$@" >> "${LOG_FILE}" 2>&1; then
    log "DONE: ${name}"
    return 0
  else
    log "WARN: ${name} exited non-zero (continuing)"
    return 1
  fi
}

# ---- Step 1: Business research ----
BUSINESS_OUT="${ROOT_DIR}/tmp/business-research-${TS}.json"
mkdir -p "${ROOT_DIR}/tmp"

if [[ -x "${ROOT_DIR}/scripts/granite-business-research.sh" ]]; then
  run_step "Business research (granite-business-research.sh)" \
    "${ROOT_DIR}/scripts/granite-business-research.sh" --output "${BUSINESS_OUT}" || true
elif [[ -x "${ROOT_DIR}/bin/granite-business-research.sh" ]]; then
  run_step "Business research (granite-business-research.sh)" \
    "${ROOT_DIR}/bin/granite-business-research.sh" --output "${BUSINESS_OUT}" || true
else
  log "SKIP: granite-business-research.sh not found or not executable"
  echo '{"summary":"No business research script found","insights":[]}' > "${BUSINESS_OUT}"
fi

# ---- Step 2: Knowledge RAG query (top hub + related docs) ----
# Prefer knowledge-rag CLI if available; otherwise attempt python module or curl to known endpoint.
RAG_OUT="${ROOT_DIR}/tmp/knowledge-rag-${TS}.json"

if [[ -x "${ROOT_DIR}/scripts/knowledge-rag" ]]; then
  run_step "Knowledge RAG (top hub + related)" \
    "${ROOT_DIR}/scripts/knowledge-rag" query --top-hub --limit 10 --output "${RAG_OUT}" || true
elif [[ -x "${ROOT_DIR}/bin/knowledge-rag" ]]; then
  run_step "Knowledge RAG (top hub + related)" \
    "${ROOT_DIR}/bin/knowledge-rag" query --top-hub --limit 10 --output "${RAG_OUT}" || true
elif command -v python3 >/dev/null 2>&1 && python3 -c "import knowledge_rag" 2>/dev/null; then
  run_step "Knowledge RAG (python module)" \
    python3 -m knowledge_rag query --top-hub --limit 10 --output "${RAG_OUT}" || true
else
  log "SKIP: knowledge-rag not available"
  echo '{"top_hub":null,"related_docs":[]}' > "${RAG_OUT}"
fi

# ---- Step 3: Extract top-hub/MOC insight ----
TOP_HUB="unknown"
TOP_HUB_SCORE=0
RELATED_DOCS="[]"

if [[ -f "${RAG_OUT}" ]]; then
  if command -v jq >/dev/null 2>&1; then
    TOP_HUB="$(jq -r '.top_hub // .hub // .most_connected // "unknown"' "${RAG_OUT}" 2>/dev/null || echo "unknown")"
    TOP_HUB_SCORE="$(jq '[.related_docs[]?.score // .weight // 0] | add // 0' "${RAG_OUT}" 2>/dev/null || echo 0)"
    RELATED_DOCS="$(jq '.related_docs // []' "${RAG_OUT}" 2>/dev/null || echo '[]')"
  else
    # fallback: simple grep for common keys
    TOP_HUB="$(grep -i -m1 -o -E '"(top_hub|hub|most_connected)"\s*:\s*"[^"]+"' "${RAG_OUT}" | sed -E 's/.*"([^"]+)"$/\1/' || echo "unknown")"
  fi
fi

# ---- Step 4: Build report ----
BUSINESS_SUMMARY="No business research output"
if [[ -f "${BUSINESS_OUT}" ]] && command -v jq >/dev/null 2>&1; then
  BUSINESS_SUMMARY="$(jq -r '.summary // "No summary"' "${BUSINESS_OUT}" 2>/dev/null || echo "No summary")"
elif [[ -f "${BUSINESS_OUT}" ]]; then
  BUSINESS_SUMMARY="$(head -1 "${BUSINESS_OUT}")"
fi

cat > "${REPORT_FILE}" <<REPORT_EOF
{
  "timestamp": "${TS}",
  "project_root": "${ROOT_DIR}",
  "business_research": {
    "summary": ${BUSINESS_SUMMARY:+$(printf '%s' "${BUSINESS_SUMMARY}" | python3 -c 'import sys,json;print(json.dumps(sys.stdin.read()))' 2>/dev/null || echo "null")},
    "file": "${BUSINESS_OUT}"
  },
  "knowledge_rag": {
    "top_hub": "${TOP_HUB}",
    "top_hub_score": ${TOP_HUB_SCORE},
    "related_docs": ${RELATED_DOCS}
  },
  "insight": {
    "top_hub_moc": "${TOP_HUB}",
    "recommendation":
