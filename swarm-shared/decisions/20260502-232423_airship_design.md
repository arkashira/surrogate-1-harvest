# airship / design

## Final Synthesis (Best Parts + Correctness + Actionability)

**Chosen approach**: Convert `airship discover` into a **deterministic CDN-only orchestrator** that eliminates HF API rate limits and PyArrow schema errors while producing a reproducible file list. This is the highest-value change that can ship in <2h with no infra or training changes.

---

## Implementation Plan (Consolidated)

### 1. CLI contract (5 min)
```bash
airship discover --repo <repo> --date <YYYY-MM-DD> --out <dir>
```
Produces:
- `<out>/file-list.json` (deterministic, sorted)
- `<out>/batches/mirror-merged/<date>/` (parquet shards)

### 2. Non-recursive listing (15 min)
- Use `list_repo_tree(path=date, recursive=False)` per date folder.
- Save deterministic JSON list; embed same list in `train.py`.
- Fallback: CDN HTML directory listing only if tree API fails.

### 3. CDN-only downloader (30 min)
- Worker pool fetching:
  ```
  https://huggingface.co/datasets/{repo}/resolve/main/{date}/{file}
  ```
- **No Authorization header** (bypasses API rate limits).
- Exponential backoff; 360s sleep on 429.
- Stream to temp file; never load full repo into Arrow.

### 4. Schema-safe projection (20 min)
- Parse by extension; extract only `{prompt, response}`.
- **JSON/JSONL**: read with streaming parser; skip if keys missing.
- **Parquet**: `read_columns(["prompt", "response"])`; ignore other columns.
- Missing keys → log and skip (never fail pipeline).

### 5. Parquet shard writer (15 min)
- Batch rows (10k) → `pyarrow.Table` → `pq.write_table`.
- Filename: `batches/mirror-merged/{date}/{slug}.parquet`
- **No extra metadata columns** (attribution via filename only).

### 6. Integration (15 min)
- Add `--file-list` flag to `train.py` to skip discovery when re-running.
- Ensure `train.py` can consume CDN-only parquet shards directly.

### 7. Tests & validation (20 min)
- Mock CDN 429 → verify backoff/sleep.
- Mixed schema repo → verify projection ignores unknown columns.
- Smoke test against small public dataset repo.

## Code (Consolidated)

### `airship/cli/discover.py`
```python
#!/usr/bin/env python3
import argparse
import json
import asyncio
from pathlib import Path
from airship.discovery.worker import discover_repo

def main() -> None:
    parser = argparse.ArgumentParser(description="CDN-only repo discovery")
    parser.add_argument("--repo", required=True, help="HF dataset repo (user/repo)")
    parser.add_argument("--date", required=True, help="YYYY-MM-DD folder")
    parser.add_argument("--out", default="./batches", help="Output directory")
    args = parser.parse_args()

    out_dir = Path(args.out)
    out_dir.mkdir(parents=True, exist_ok=True)

    file_list = asyncio.run(discover_repo(args.repo, args.date, out_dir))
    (out_dir / "file-list.json").write_text(json.dumps(file_list, indent=2))
    print(f"Discovered {len(file_list)} files -> {out_dir}")

if __name__ == "__main__":
    main()
```

### `airship/discovery/worker.py`
```python
import asyncio
import aiohttp
from huggingface_hub import list_repo_tree
from pathlib import Path
import pyarrow as pa
import pyarrow.parquet as pq
import json
from typing import List, Dict

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

async def fetch_file(session: aiohttp.ClientSession, url: str, dest: Path) -> None:
    async with session.get(url) as resp:
        if resp.status == 429:
            await asyncio.sleep(360)
            return await fetch_file(session, url, dest)
        resp.raise_for_status()
        dest.parent.mkdir(parents=True, exist_ok=True)
        with open(dest, "wb") as f:
            async for chunk in resp.content.iter_chunked(64 * 1024):
                f.write(chunk)

def project_to_pair(local_path: Path) -> List[Dict[str, str]]:
    """Extract {prompt, response} only; ignore schema mismatches."""
    if local_path.suffix == ".jsonl":
        import jsonlines
        rows = []
        try:
            with jsonlines.open(local_path) as reader:
                for obj in reader:
                    if "prompt" in obj and "response" in obj:
                        rows.append({"prompt": obj["prompt"], "response": obj["response"]})
        except Exception:
            pass
        return rows
    elif local_path.suffix == ".parquet":
        try:
            table = pq.read_table(local_path, columns=["prompt", "response"])
            return table.to_pylist()
        except Exception:
            return []
    return []

async def discover_repo(repo: str, date: str, out_dir: Path) -> List[str]:
    # 1) Non-recursive tree listing
    try:
        tree = list_repo_tree(repo, path=date, recursive=False)
        files = [f.rfilename for f in tree if f.type == "file"]
    except Exception:
        files = []

    # 2) CDN download (no auth)
    connector = aiohttp.TCPConnector(limit_per_host=8)
    async with aiohttp.ClientSession(connector=connector) as session:
        tasks = []
        raw_dir = out_dir / "raw" / date
        for f in files:
            url = CDN_TEMPLATE.format(repo=repo, path=f"{date}/{f}")
            dest = raw_dir / f
            tasks.append(fetch_file(session, url, dest))
        await asyncio.gather(*tasks, return_exceptions=True)

    # 3) Project & shard
    rows = []
    for f in files:
        local_path = raw_dir / f
        if local_path.exists():
            rows.extend(project_to_pair(local_path))

    if rows:
        table = pa.Table.from_pylist(rows)
        shard_dir = out_dir / "batches" / "mirror-merged" / date
        shard_dir.mkdir(parents=True, exist_ok=True)
        pq.write_table(table, shard_dir / f"{repo.replace('/', '_')}.parquet")

    return files
```

### `train.py` (CDN-only loader)
```python
import json
from pathlib import Path
import pyarrow as pa
import pyarrow.parquet as pq
import glob

def load_shards(file_list_path: Path):
    root = file_list_path.parent
    date = file_list_path.parent.parent.name
    shard = root / "batches" / "mirror-merged" / date / "*.parquet"
    tables = [pq.read_table(p) for p in glob.glob(str(shard))]
    return pa.concat_tables(tables) if tables else pa.Table.from_pylist([])
```
