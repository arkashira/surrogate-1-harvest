## 1) `bin/airship-discover-cdn`

Deterministic discovery that:
- Uses HF API **once** (after rate-limit window) to list one date folder.
- Persists `file-list.json`.
- Embeds CDN URLs (`resolve/main/...`) for runtime fetches (zero API calls during training).
- Avoids `list_repo_files` recursive; uses `list_repo_tree(path, recursive=False)` per folder.

```bash
#!/usr/bin/env bash
# bin/airship-discover-cdn
# Usage: HF_TOKEN=... ./bin/airship-discover-cdn <repo> <date_folder> [out_json]
set -euo pipefail

REPO="${1:-datasets/mirror-merged}"
DATEFOLDER="${2:-$(date +%Y-%m-%d)}"
OUT="${3:-file-list.json}"

python3 - "$REPO" "$DATEFOLDER" "$OUT" <<'PY'
import os, json, sys
from huggingface_hub import list_repo_tree, hf_hub_download

repo = sys.argv[1]
folder = sys.argv[2]
out = sys.argv[3]

# Single API call: non-recursive per folder (avoids 100x pagination)
tree = list_repo_tree(repo, recursive=False, path=folder)
files = [f.rfilename for f in tree if f.type == "file"]

# Build CDN entries (no auth, bypasses /api/ rate limits)
cdn_base = f"https://huggingface.co/datasets/{repo}/resolve/main"
entries = [
    {
        "repo": repo,
        "path": f,
        "cdn_url": f"{cdn_base}/{f}",
        "hf_path": f"{folder}/{f}"
    }
    for f in files
]

with open(out, "w") as fp:
    json.dump(entries, fp, indent=2)

print(f"Wrote {len(entries)} files to {out}")
PY
```

Make executable:
```bash
chmod +x bin/airship-discover-cdn
```

## 2) `bin/project-to-pairwise-parquet`

Projects heterogeneous files to strict `{prompt, response}` schema at parse time. Attribution moved to filename pattern. No `source`/`ts` columns.

```bash
#!/usr/bin/env bash
# bin/project-to-pairwise-parquet
# Usage: ./bin/project-to-pairwise-parquet file-list.json out-dir/
set -euo pipefail

LIST="${1:-file-list.json}"
OUTDIR="${2:-batches/pairwise}"
mkdir -p "$OUTDIR"

python3 - "$LIST" "$OUTDIR" <<'PY'
import json, os, pyarrow as pa, pyarrow.parquet as pq, uuid, datetime

with open(sys.argv[1]) as f:
    entries = json.load(f)

outdir = sys.argv[2]
os.makedirs(outdir, exist_ok=True)

schema = pa.schema([
    pa.field("prompt", pa.string()),
    pa.field("response", pa.string()),
])

rows = []
for e in entries:
    # Heuristic: treat first non-empty line as prompt, remainder as response
    # Replace with your domain-specific parser as needed.
    try:
        import requests
        r = requests.get(e["cdn_url"], timeout=30)
        r.raise_for_status()
        text = r.text
    except Exception as ex:
        print(f"Skip {e['path']}: {ex}")
        continue

    lines = [ln.strip() for ln in text.splitlines() if ln.strip()]
    if len(lines) < 2:
        continue
    prompt = lines[0]
    response = "\n".join(lines[1:])
    rows.append({"prompt": prompt, "response": response})

if rows:
    tbl = pa.Table.from_pylist(rows, schema=schema)
    slug = datetime.datetime.utcnow().strftime("%Y%m%d-%H%M%S")
    fname = f"batches/mirror-merged/{slug}/{uuid.uuid4().hex}.parquet"
    os.makedirs(os.path.dirname(os.path.join(outdir, fname)), exist_ok=True)
    pq.write_table(tbl, os.path.join(outdir, fname))
    print(f"Wrote {len(rows)} rows to {fname}")
else:
    print("No valid rows produced.")
PY
```

Make executable:
```bash
chmod +x bin/project-to-pairwise-parquet
```

## 3) `surrogate/config/ingest.yaml` (sibling repo sharding)

```yaml
ingest:
  hf_dataset_repo: "datasets/mirror-merged"
  sibling_repos:
    - "datasets/mirror-merged"
    - "datasets/mirror-merged-sib1"
    - "datasets/mirror-merged-sib2"
    - "datasets/mirror-merged-sib3"
    - "datasets/mirror-merged-sib4"
    - "datasets/mirror-merged-sib5"
  shard_strategy: "hash_slug_mod"   # deterministic: hash(slug) % len(sibling_repos)
  file_list: "file-list.json"       # produced by airship-discover-cdn
  projection:
    columns: ["prompt", "response"]
  cdn_only: true
```

## 4) Cron Fix (wrapper + env)

```cron

PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin

# Run discovery after HF rate-limit window clears (e.g., 60m after prior 429)
0 2 * * * bash /opt/axentx/airship/bin/airship-discover-cdn datasets/mirror-merged "$(date +\%Y-\%m-\%d)" /opt/axentx/airship/file-list.json >> /var/log/airship-discover.log 2>&1

# Projection (runs after discovery)
10 2 * * * bash /opt/axentx/airship/bin/project-to-pairwise-parquet /opt/axentx/airship/file-list.json /opt/axentx/airship/batches >> /var/log/airship-project.log 2>&1
```

```bash
#!/usr/bin/env bash
# Ensure executed via bash, not sh
set -euo pipefail
exec python3 -m mymodule "$@"
```

and in crontab: `bash /path/to/wrapper.sh "$@"`.
