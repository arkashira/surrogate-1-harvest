## Deployment
cf workers (free-tier Cloudflare Workers + D1 (Postgres-ish) for API; scans run as scheduled Workers; Supabase for durable relational store)

## v0.1 scope
v0.1 in 2 weeks: CLI-first scan (Python) that lists Lambda + Cloud Functions schedules, computes next-run via croniter, flags duplicates (same cron+handler), drifted (cron changed since last run), and orphaned (function deleted). Results written to Supabase Postgres with audit events. Optional lightweight FastAPI read-only API deployed to Cloudflare Workers with D1 + Supabase. No UI; JSON export and simple webhook for retirement proposals.
