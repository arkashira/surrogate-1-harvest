# workio / design

## Final consolidated implementation

**Core problems (both candidates agree)**  
- No idempotency → duplicate `X-Line-Signature`/event IDs corrupt state.  
- No strict signature time-window → replay/stale webhooks accepted.  
- Non-transactional writes → partial updates on failure/concurrency.  
- No per-tenant abuse control → noisy-neighbor risk.  
- Missing immutable audit trail → non-compliance for labor records.

**Chosen approach**  
Adopt Candidate 1’s complete plan (diagnosis + code) and resolve Candidate 2’s truncation by using Candidate 1’s transactional upsert and audit table. Add two hardened, production-ready fixes:

1. **Idempotency on `line_event_id` + `tenant_id`** (unique constraint) with DB-level upsert to guarantee exactly-once processing even under retries/concurrency.  
2. **Short signature validity window (≤5 min)** plus mandatory idempotency for events without timestamps. Reject replays and stale deliveries.  
3. **Transactional write path**: single DB transaction per webhook that (a) records immutable event, (b) applies attendance transition, (c) updates daily summaries/state atomically.  
4. **Per-tenant rate limiting** at middleware to bound webhook volume and prevent noisy-neighbor effects.  
5. **Immutable audit table** for compliance and debugging.

## 1. DB schema (append once)

```sql
-- server/src/db/schema.sql
CREATE TABLE IF NOT EXISTS attendance_events (
  id              BIGSERIAL PRIMARY KEY,
  tenant_id       BIGINT NOT NULL,
  user_id         BIGINT NOT NULL,
  event_type      TEXT NOT NULL CHECK (event_type IN ('clock_in','clock_out','leave_request','leave_approve','ot_request','ot_approve')),
  payload         JSONB NOT NULL,
  line_event_id   TEXT NOT NULL,
  processed_at    TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  UNIQUE(tenant_id, line_event_id)  -- idempotency key
);

CREATE INDEX IF NOT EXISTS idx_attendance_events_tenant_line ON attendance_events(tenant_id, line_event_id);
CREATE INDEX IF NOT EXISTS idx_attendance_events_user ON attendance_events(user_id);
```

## 2. Middleware

`server/src/middleware/lineWebhookGuard.ts`

```ts
import crypto from 'crypto';
import { Request, Response, NextFunction } from 'express';

const SIGNATURE_HEADER = 'x-line-signature';
const MAX_CLOCK_SKEW_MS = 5 * 60 * 1000; // 5 minutes

export function lineWebhookGuard(channelSecret: string) {
  return (req: Request, res: Response, next: NextFunction) => {
    const signature = req.get(SIGNATURE_HEADER);
    const body = JSON.stringify(req.body);
    const now = Date.now();

    if (!signature || typeof signature !== 'string') {
      return res.status(400).json({ error: 'Missing signature' });
    }

    // Optional timestamp check when provided by LINE
    const eventTimestamp = req.body?.timestamp ?? req.body?.events?.[0]?.timestamp;
    if (eventTimestamp && Math.abs(now - Number(eventTimestamp)) > MAX_CLOCK_SKEW_MS) {
      return res.status(400).json({ error: 'Stale event' });
    }

    const expected = crypto
      .createHmac('sha256', channelSecret)
      .update(body, 'utf8')
      .digest('base64');

    if (!crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected))) {
      return res.status(401).json({ error: 'Invalid signature' });
    }

    (req as any).lineValidated = { body: req.body, signature };
    next();
  };
}
```

`server/src/middleware/tenantRateLimiter.ts`

```ts
import { Request, Response, NextFunction } from 'express';
import LRU from 'lru-cache';

const WINDOW_MS = 60 * 1000;
const MAX_REQUESTS = 30; // per tenant per minute

const cache = new LRU<string, { count: number; resetTime: number }>({
  max: 5000,
  ttl: WINDOW_MS,
});

export function tenantRateLimiter(getTenantId: (req: Request) => string | undefined) {
  return (req: Request, res: Response, next: NextFunction) => {
    const tenantId = getTenantId(req);
    if (!tenantId) return next();

    const key = `tenant:${tenantId}`;
    const now = Date.now();
    const record = cache.get(key) || { count: 0, resetTime: now + WINDOW_MS };

    if (now > record.resetTime) {
      record.count = 0;
      record.resetTime = now + WINDOW_MS;
    }

    record.count += 1;
    cache.set(key, record);

    if (record.count > MAX_REQUESTS) {
      return res.status(429).json({ error: 'Too many requests' });
    }

    next();
  };
}
```
