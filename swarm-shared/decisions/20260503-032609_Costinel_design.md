# Costinel / design

## Final Synthesized Design — Top-Hub Signal Panel (CDN-first, <2h)

**Goal:** Add a single, non-blocking Top-Hub Signal Panel to Costinel that surfaces high-cost anomalies and RI/coverage gaps for the most-connected hub, using a CDN-first data strategy with graceful fallback. The panel proposes, never executes.

---

### 1) Scope & Constraints
- **One deliverable:** a collapsible dashboard widget + one lightweight backend route.
- **No DB migrations, no infra changes, no HF API calls at render time.**
- **CDN-first:** pre-generated files served from CDN; backend falls back to local cache.
- **Human-in-the-loop:** signals are surfaced with context links; no automated actions.
- **Target time:** ≤2 hours.

---

### 3) Backend Route (CDN-first + fallback)

**Path:** `app/api/signals/top-hub/route.ts` (Next.js)  
**Strategy:**  
1. Try CDN file list for today (pre-generated).  
2. Fetch individual signal files from CDN in parallel with `Promise.allSettled`.  
3. If CDN fails or returns no signals, fall back to local cached file.  
4. If cache missing, return empty signals (non-blocking).

```typescript
// app/api/signals/top-hub/route.ts
import { NextResponse } from 'next/server';
import fs from 'fs';
import path from 'path';

export async function GET() {
  const today = new Date().toISOString().slice(0, 10).replace(/-/g, '');
  const repo = 'axentx/costinel-data';
  const cdnListPath = path.join(process.cwd(), 'public', 'cdn-files', `top-hub-${today}.json`);

  try {
    // 1) CDN-first: use pre-generated file list
    if (fs.existsSync(cdnListPath)) {
      const list = JSON.parse(fs.readFileSync(cdnListPath, 'utf8'));
      const results = await Promise.allSettled(
        list.files.map((url: string) =>
          fetch(url, { cache: 'no-store', next: { revalidate: 0 } }).then((r) => {
            if (!r.ok) throw new Error(`CDN ${r.status}`);
            return r.json();
          })
        )
      );

      const signals = results
        .filter((r): r is PromiseFulfilledResult<any> => r.status === 'fulfilled')
        .flatMap((r) => r.value.signals || [])
        .sort((a, b) => (b.impact || 0) - (a.impact || 0))
        .slice(0, 5);

      if (signals.length > 0) {
        return NextResponse.json({ signals, source: 'cdn', date: today });
      }
    }
  } catch (e) {
    console.warn('CDN fetch failed, falling back', e);
  }

  // 2) Fallback: local cached signals
  const fallbackPath = path.join(process.cwd(), 'data', 'cached-top-hub.json');
  if (fs.existsSync(fallbackPath)) {
    const cached = JSON.parse(fs.readFileSync(fallbackPath, 'utf8'));
    return NextResponse.json({ signals: cached.signals || [], source: 'cache', date: cached.date });
  }

  // 3) Empty non-blocking response
  return NextResponse.json({ signals: [], source: 'none' });
}
```

### 4) Orchestration: Generate CDN File List (run on Mac)

This is a one-off or scheduled script to create the CDN file list snapshot. It uses a single HuggingFace tree call and writes a local JSON file consumed by the backend.

```bash
# scripts/generate-top-hub-cdn-list.sh
#!/usr/bin/env bash
set -euo pipefail

REPO="axentx/costinel-data"
DATE=$(date +%Y%m%d)
OUT_DIR="$(cd "$(dirname "$0")/../public/cdn-files" && pwd)"
mkdir -p "$OUT_DIR"
OUT_FILE="$OUT_DIR/top-hub-$DATE.json"

# Use HuggingFace Hub to list today's top-hub folder (non-recursive)
node -e "
const { HfApi } = require('@huggingface/hub');
const fs = require('fs');
(async () => {
  const api = new HfApi();
  try {
    const tree = await api.listRepoTree({
      repo: '$REPO',
      path: 'top-hub/$DATE',
      recursive: false
    });
    const files = tree
      .filter(f => f.type === 'file')
      .map(f => \`https://huggingface.co/datasets/$REPO/resolve/main/top-hub/$DATE/\${f.path}\`);
    fs.writeFileSync('$OUT_FILE', JSON.stringify({ date: '$DATE', files, generatedAt: new Date().toISOString() }, null, 2));
    console.log('Generated', '$OUT_FILE');
  } catch (e) {
    console.error('Failed to list repo tree:', e);
    process.exit(1);
  }
})();
"
```
