## Highest-Value Incremental Improvement
**CDN-only ingestion with deterministic sibling-repo sharding** — eliminate HF API 429s by pre-listing once, downloading via public CDN URLs (no auth header), projecting `{prompt,response}` at parse time, and spreading writes across 5 sibling repos to bypass 128/hr commit cap.

## Implementation Plan (<2h)

### 1. Add deterministic repo selector (surrogate/ingest/shard.py)
- Hash slug → pick sibling repo index deterministically (5 repos)
- Deterministic across runs and workers

### 2. Pre-list file paths (surrogate/ingest/list_paths.py)
- Single API call per date folder via `list_repo_tree(path, recursive=False)`
- Save to JSON (e.g., `batches/mirror-merged/2026-04-29/filelist.json`)
- Commit this JSON to repo so training scripts can embed it

### 3. CDN-only downloader (surrogate/ingest/cdn_download.py)
- Use `https://huggingface.co/datasets/{repo}/resolve/main/{path}`
- No Authorization header (bypasses API auth/rate-limit)
- Stream to temp file, parse `{prompt,response}` only
- Drop all other columns; no `source`/`ts` cols

### 4. Training script integration (surrogate/train/train.py)
- Load embedded filelist JSON
- Use CDN URLs directly in DataLoader (zero API calls during training)
- Optional: verify file existence via HEAD before batch load

### 5. Lightning Studio reuse guard (surrogate/train/launcher.py)
- List running studios; reuse if exists and running
- If stopped, restart with `target.start(machine=Machine.L40S)`
- Prevents quota waste and idle-timeout deaths

### 6. Cron/wrapper hygiene
- Ensure all wrapper scripts have `#!/usr/bin/env bash` and `chmod +x`
- Set `SHELL=/bin/bash` in crontab
- Invoke via `bash <script> "$@"` for safety

## Code Snippets

### surrogate/ingest/shard.py
```python
import hashlib

SIBLING_REPOS = [
    "axentx/surrogate-data",
    "axentx/surrogate-data-s1",
    "axentx/surrogate-data-s2",
    "axentx/surrogate-data-s3",
    "axentx/surrogate-data-s4",
]

def shard_repo_for_slug(slug: str) -> str:
    """Deterministic repo selection from slug."""
    digest = hashlib.sha256(slug.encode()).hexdigest()
    idx = int(digest, 16) % len(SIBLING_REPOS)
    return SIBLING_REPOS[idx]
```

### surrogate/ingest/list_paths.py
```python
import json
from huggingface_hub import list_repo_tree

def list_date_folder(date_folder: str, repo: str = "axentx/surrogate-data"):
    """Single API call to list files in a date folder."""
    tree = list_repo_tree(repo, path=date_folder, recursive=False)
    files = [item.rfilename for item in tree if item.type == "file"]
    out_path = f"batches/mirror-merged/{date_folder}/filelist.json"
    with open(out_path, "w") as f:
        json.dump(files, f, indent=2)
    return files

if __name__ == "__main__":
    import sys
    date_folder = sys.argv[1] if len(sys.argv) > 1 else "2026-04-29"
    files = list_date_folder(date_folder)
    print(f"Listed {len(files)} files -> {date_folder}/filelist.json")
```

### surrogate/ingest/cdn_download.py
```python
import requests
import tempfile
import os

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def download_via_cdn(repo: str, path: str) -> str:
    """Download file via public CDN (no auth header). Returns temp filepath."""
    url = CDN_TEMPLATE.format(repo=repo, path=path)
    with tempfile.NamedTemporaryFile(delete=False, suffix=".parquet") as tmp:
        with requests.get(url, stream=True, timeout=60) as r:
            r.raise_for_status()
            for chunk in r.iter_content(chunk_size=8192):
                tmp.write(chunk)
        return tmp.name

def project_prompt_response(parquet_path: str):
    """Load parquet and project to {prompt, response} only."""
    import pyarrow.parquet as pq
    table = pq.read_table(parquet_path, columns=["prompt", "response"])
    return table.to_pandas()
```

### surrogate/train/train.py (excerpt)
```python
import json
import os
from pathlib import Path
from surrogate.ingest.cdn_download import download_via_cdn, project_prompt_response

def load_filelist(date_folder: str):
    list_path = Path(f"batches/mirror-merged/{date_folder}/filelist.json")
    with open(list_path) as f:
        return json.load(f)

def cdn_data_loader(date_folder: str, repo: str = "axentx/surrogate-data"):
    files = load_filelist(date_folder)
    for fname in files:
        tmp_path = download_via_cdn(repo, fname)
        try:
            df = project_prompt_response(tmp_path)
            yield df
        finally:
            os.unlink(tmp_path)
```

### surrogate/train/launcher.py (Lightning reuse)
```python
from lightning import Teamspace, Studio, Machine

def get_or_create_studio(name: str, machine: Machine = Machine.L40S):
    teamspace = Teamspace()
    for s in teamspace.studios:
        if s.name == name and s.status == "Running":
            print(f"Reusing running studio: {name}")
            return s
    print(f"Creating studio: {name}")
    return Studio(name=name, create_ok=True).start(machine=machine)

def safe_run_studio(name: str, script: str):
    studio = get_or_create_studio(name)
    if studio.status != "Running":
        studio.start(machine=Machine.L40S)
    studio.run(script)
```
