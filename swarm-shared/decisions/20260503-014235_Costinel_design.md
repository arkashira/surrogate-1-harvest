# Costinel / design

## Final Implementation Plan — Top-hub Signal Panel (Costinel Dashboard)

**Scope**  
- Frontend-only, read-only panel.  
- Surface the most-connected hub (e.g., “MOC”) and actionable proposals from the knowledge graph.  
- Resilient to missing backend: graceful fallback UI + local sample data + localStorage cache.  
- Timebox: <2h.

**Stack**  
React + TypeScript + Tailwind (existing conventions). Optional lightweight fetch (no new deps).

---

### 1) Design decisions (merged best choices)

- **Placement**: top of dashboard main content (below header) for high visibility and low friction.  
- **Data source**: `GET /api/knowledge/top-hub` if available. If 404/5xx or timeout, fallback in order:  
  1) valid localStorage cache (10m TTL),  
  2) local sample JSON,  
  3) minimal inline static stub.  
- **No writes**; purely read-only signal.  
- **Accessibility**: keyboard-navigable list, clear labels, color-blind-friendly status indicators, `aria-busy`/`aria-label`.  
- **Performance**: client-side fetch with 3s timeout; stale-while-revalidate pattern (cache in `localStorage`).  
- **Actionability**: proposals show impact, estimated savings (when available), due date, and direct links to details.

#### B) API client + types + cache logic
`src/lib/knowledge.ts`
```ts
export interface Proposal {
  id: string;
  title: string;
  summary?: string;
  rationale?: string;
  impact: 'high' | 'medium' | 'low';
  estimatedSavingsUSD?: number;
  dueBy?: string;
  href?: string;
}

export interface TopHubHub {
  id: string;
  label: string;
  description: string;
  importance?: number;
  connections?: number;
}

export interface TopHub {
  hub: TopHubHub;
  proposals: Proposal[];
  updatedAt: string; // ISO
}

const ENDPOINT = '/api/knowledge/top-hub';
const CACHE_KEY = 'costinel:top-hub:v1';
const TTL_MS = 10 * 60 * 1000; // 10m

function isExpired(ts: string) {
  return Date.now() - new Date(ts).getTime() > TTL_MS;
}

async function loadSample(): Promise<TopHub> {
  // Use dynamic import to avoid bundling issues if path differs; fallback to fetch if needed.
  try {
    // If your build setup supports importing JSON as module, this works at runtime in dev & prod.
    const mod = await import('../data/topHubSample.json').catch(() => null);
    if (mod?.default) return mod.default as TopHub;
  } catch {
    // fallback to fetch if JSON is served statically
  }
  const res = await fetch('/data/topHubSample.json', { cache: 'no-store' });
  if (!res.ok) throw new Error('Sample unavailable');
  return res.json();
}

export async function fetchTopHub(signal?: AbortSignal): Promise<TopHub> {
  const controller = new AbortController();
  if (signal) signal.addEventListener('abort', () => controller.abort());
  const timeout = setTimeout(() => controller.abort(), 3000);

  try {
    const res = await fetch(ENDPOINT, {
      method: 'GET',
      headers: { Accept: 'application/json' },
      signal: controller.signal,
    });
    clearTimeout(timeout);

    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    const json = (await res.json()) as TopHub;
    localStorage.setItem(CACHE_KEY, JSON.stringify(json));
    return json;
  } catch {
    clearTimeout(timeout);
    // try cache
    const raw = localStorage.getItem(CACHE_KEY);
    if (raw) {
      try {
        const cached = JSON.parse(raw) as TopHub;
        if (!isExpired(cached.updatedAt)) return cached;
      } catch {
        // ignore malformed cache
      }
    }
    // try sample
    try {
      const sample = await loadSample();
      return sample;
    } catch {
      // final stub
      return {
        hub: { id: 'MOC', label: 'MOC', description: 'Most-connected operational hub' },
        proposals: [
          {
            id: 'stub-1',
            title: 'Reduce idle RDS instances',
            summary: '3 RDS instances in us-east-1 show <5% CPU avg. Consider downsize or schedule stop/start.',
            impact: 'high',
            href: '/proposals/1',
          },
        ],
        updatedAt: new Date().toISOString(),
      };
    }
  }
}
```
