# airship / design

## Highest-Value Incremental Improvement (<2h)

**Implement a CDN-only, reproducible file-list generator for `airship discover`**  
Eliminates HF API rate limits and PyArrow schema errors by:
1. Using `list_repo_tree(..., recursive=False)` per date folder (single API call)
2. Persisting deterministic file list to JSON
3. Embedding that list in training scripts so Lightning workers fetch via CDN URLs only (`resolve/main/...`)
4. Projecting to `{prompt,response}` at parse time (no mixed-schema ingestion)

This directly applies the **HF CDN Bypass** and **Surrogate-1 training pipeline** patterns and yields an immediate, testable artifact (`file-list.json`) that unblocks downstream training.

## Code Snippets

### 1. `airship/discover/list_repo_files.py`
```python
#!/usr/bin/env python3
"""
Deterministic, CDN-friendly repo file lister.
Usage:
  python list_repo_files.py \
    --repo huggingface/datasets \
    --folder batches/mirror-merged/2026-05-01 \
    --out file-list.json
"""
import argparse
import json
import os
import time
from pathlib import Path
from typing import List, Dict

from huggingface_hub import HfApi, Repository

HF_API_RATE_LIMIT_RETRY = 360  # seconds (per pattern)

def list_repo_files_cdn_safe(
    repo_id: str,
    folder: str,
    recursive: bool = False,
    max_retries: int = 3,
) -> List[Dict[str, str]]:
    """
    Single API call per folder (non-recursive) to avoid pagination/rate limits.
    Returns minimal metadata needed to construct CDN URLs.
    """
    api = HfApi()
    for attempt in range(1, max_retries + 1):
        try:
            tree = api.list_repo_tree(
                repo_id=repo_id,
                path=folder,
                recursive=recursive,
            )
            files = [
                {
                    "repo_id": repo_id,
                    "path": node.path,
                    "cdn_url": f"https://huggingface.co/datasets/{repo_id}/resolve/main/{node.path}",
                    "size": getattr(node, "size", None),
                }
                for node in tree
                if node.type == "file"
            ]
            # Deterministic ordering for reproducibility
            files.sort(key=lambda f: f["path"])
            return files
        except Exception as exc:
            if attempt == max_retries:
                raise
            # If 429-like, wait and retry (per HF API rate-limit pattern)
            wait = HF_API_RATE_LIMIT_RETRY
            print(f"[{attempt}/{max_retries}] Retry after {wait}s: {exc}")
            time.sleep(wait)
    return []

def main() -> None:
    parser = argparse.ArgumentParser(description="CDN-safe repo file lister")
    parser.add_argument("--repo", required=True, help="HF dataset repo id")
    parser.add_argument("--folder", required=True, help="Folder path in repo")
    parser.add_argument("--out", default="file-list.json", help="Output JSON path")
    parser.add_argument(
        "--recursive",
        action="store_true",
        help="List recursively (use sparingly; prefer per-folder calls)",
    )
    args = parser.parse_args()

    files = list_repo_files_cdn_safe(args.repo, args.folder, recursive=args.recursive)
    out_path = Path(args.out)
    out_path.write_text(json.dumps(files, indent=2))
    print(f"Wrote {len(files)} files -> {out_path}")

if __name__ == "__main__":
    main()
```

### 3. Lightning training loader (CDN-only)
```python
# surrogate/data/cdn_dataset.py
import json
from pathlib import Path
from typing import List, Dict

import pyarrow as pa
import pyarrow.parquet as pq
import requests
from torch.utils.data import IterableDataset

class CDNParquetDataset(IterableDataset):
    """
    Loads parquet shards via CDN URLs listed in file-list.json.
    Projects only {prompt,response} at parse time to avoid mixed-schema errors.
    """
    def __init__(self, file_list_json: Path):
        self.file_list = json.loads(file_list_json.read_text())
        # Keep only parquet files
        self.shards = [f["cdn_url"] for f in self.file_list if f["path"].endswith(".parquet")]

    def __iter__(self):
        for url in self.shards:
            # CDN download (no HF API auth/rate-limit)
            resp = requests.get(url, timeout=60)
            resp.raise_for_status()
            with pa.BufferReader(resp.content) as reader:
                table = pq.read_table(reader, columns=["prompt", "response"])
                for row in table.to_pylist():
                    # Ensure schema uniformity
                    yield {
                        "prompt": row.get("prompt") or "",
                        "response": row.get("response") or "",
                    }
```

## Notes & Patterns Applied
- **HF CDN Bypass**: `resolve/main/` URLs avoid `/api/` rate limits entirely.
- **Single API call per folder**: `list_repo_tree(recursive=False)` prevents pagination blowup.
- **Schema safety**: project to `{prompt,response}` only at parse time (no mixed-schema PyArrow errors).
- **Deterministic outputs**: sorted file list → reproducible training runs.
- **Lightning quota**: reuse running Studio; this loader keeps data path CDN-only so workers never hit HF API during training.
