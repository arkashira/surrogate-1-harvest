4. **Determine file schema at runtime**:
	* Use `pyarrow.schema` to infer file schema from the first few rows of each file.
	* Store inferred schema in the manifest file.
5. **Update `airship consume` to use reproducible file manifests**:
	* Modify `consume.py` to read manifest files and use the stored file hashes for verification.
	* Update `consume.py` to use the inferred schema for file processing.

import hashlib
import hf_hub_download
import pyarrow.parquet

def discover_dataset(repo, path):
    # Use HF CDN for file downloads
    files = hf_hub_download(repo, path, cdn=True)
    
    # Compute file hashes and store in manifest file
    manifest = {}
    for file in files:
        hash = hashlib.sha256(file).hexdigest()
        manifest[file] = hash
    
    # Store manifest file alongside downloaded files
    with open('manifest.json', 'w') as f:
