# Costinel / ops

## Implementation Plan — Top Hub Signal Panel (CDN-first, frontend-only)

**Scope**: Add a lightweight, resilient Top Hub signal card to the Costinel dashboard that surfaces the most-connected hub (e.g., "MOC") and related contextual insights without backend changes.  
**Effort**: ~60–90 minutes (frontend only).  
**Mechanism**:  
- Build a frontend-only card that fetches a small JSON file from the CDN (bypasses HF API rate limits).  
- JSON is produced by an ops script (run manually or via cron) that queries the knowledge-rag graph and writes `top-hub.json` to the repo’s `public/signals/` folder.  
- Card falls back to cached data if CDN fetch fails and shows a subtle “last updated” timestamp.  
- No backend changes, no new secrets, no infra work.

### 1) Create the ops script (run locally or in cron)

`/opt/axentx/Costinel/scripts/update-top-hub.sh`

```bash
#!/usr/bin/env bash
# update-top-hub.sh
# Queries knowledge-rag for top hub + insights and writes public/signals/top-hub.json
# Cron: 0 6,18 * * * cd /opt/axentx/Costinel && SHELL=/bin/bash bash scripts/update-top-hub.sh

set -euo pipefail

REPO_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
OUT_DIR="${REPO_ROOT}/public/signals"
OUT_FILE="${OUT_DIR}/top-hub.json"

mkdir -p "${OUT_DIR}"

# Placeholder: replace with actual knowledge-rag query for top hub.
# Example using a CLI or curl to your RAG service.
# For now, we simulate a best-effort fetch and fallback.
TOP_HUB="MOC"
CONTEXT_LINKS=(
  "https://huggingface.co/datasets/axentx/knowledge-rag/resolve/main/hubs/MOC.md"
  "https://huggingface.co/datasets/axentx/knowledge-rag/resolve/main/hubs/MOC-context.md"
)

# Try to enrich by fetching short summaries from CDN (no auth, rate-limit bypass).
SUMMARY="Top-connected hub: ${TOP_HUB}. Review hub docs for contextual insights before planning tasks."
for url in "${CONTEXT_LINKS[@]}"; do
  if curl -fsSL --max-time 5 "${url}" | head -20 | grep -q .; then
    SUMMARY+=" (context available)"
    break
  fi
done

cat > "${OUT_FILE}" <<EOF
{
  "hub": "${TOP_HUB}",
  "summary": "${SUMMARY}",
  "links": ${CONTEXT_LINKS[@]+"$(printf '%s\n' "${CONTEXT_LINKS[@]}" | jq -R . | jq -s .)"},
  "updatedAt": "$(date -u +"%Y-%m-%dT%H:%M:%SZ")",
  "source": "knowledge-rag"
}
EOF

echo "Wrote ${OUT_FILE}"
```

Make executable:

```bash
chmod +x /opt/axentx/Costinel/scripts/update-top-hub.sh
```

Run once to generate initial file:

```bash
cd /opt/axentx/Costinel && bash scripts/update-top-hub.sh
```

### 2) Add the signal card component (React example)

`/opt/axentx/Costinel/src/components/TopHubSignalCard.jsx`

```jsx
import { useEffect, useState } from "react";
import "./TopHubSignalCard.css";

const CDN_JSON = "/signals/top-hub.json";
const CACHE_KEY = "costinel_top_hub_cache";
const CACHE_TTL_MS = 1000 * 60 * 60 * 6; // 6h

function useTopHubSignal() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    let stale = null;
    try {
      const cached = localStorage.getItem(CACHE_KEY);
      if (cached) {
        stale = JSON.parse(cached);
        // accept stale while fetching fresh
        if (stale.updatedAt && Date.now() - new Date(stale.updatedAt).getTime() < CACHE_TTL_MS) {
          setData(stale);
        }
      }
    } catch (e) {
      // ignore
    }

    fetch(CDN_JSON, { cache: "no-store" })
      .then((res) => {
        if (!res.ok) throw new Error(`HTTP ${res.status}`);
        return res.json();
      })
      .then((fresh) => {
        setData(fresh);
        try {
          localStorage.setItem(CACHE_KEY, JSON.stringify(fresh));
        } catch (e) {
          // ignore storage errors
        }
      })
      .catch((err) => {
        if (!stale) setError(err.message || "Failed to load top hub signal");
      })
      .finally(() => setLoading(false));
  }, []);

  return { data, loading, error };
}

export default function TopHubSignalCard() {
  const { data, loading, error } = useTopHubSignal();

  if (loading && !data) {
    return (
      <div className="signal-card loading">
        <div className="signal-card-title">Top Hub</div>
        <div className="signal-card-body">Loading...</div>
      </div>
    );
  }

  const hub = data?.hub || "—";
  const summary = data?.summary || "No signal available.";
  const links = Array.isArray(data?.links) ? data.links : [];
  const updated = data?.updatedAt ? new Date(data.updatedAt).toLocaleString() : null;

  return (
    <div className="signal-card">
      <div className="signal-card-title">Top Hub</div>
      <div className="signal-card-body">
        <div className="hub-name">{hub}</div>
        <p className="hub-summary">{summary}</p>
        {links.length > 0 && (
          <ul className="hub-links">
            {links.map((l, i) => (
              <li key={i}>
                <a href={l} target="_blank" rel="noopener noreferrer">
                  {l.replace(/^https?:\/\//, "").replace(/\/resolve\/main\//, "/tree/main/")}
                </a>
              </li>
            ))}
          </ul>
        )}
        {error && <div className="signal-error">Using cached data — {error}</div>}
        {updated && <div className="signal-meta">Updated {updated}</div>}
      </div>
    </div>
  );
}
```
