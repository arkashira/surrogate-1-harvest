# Costinel / ops

## Implementation Plan — Top-Hub Signal Panel (CDN-first, <2h)

### What we ship
- A **non-blocking Top-Hub Signal Panel** mounted in the Costinel dashboard (sidebar or top banner).
- Defaults to hub **MOC** (configurable via `VITE_HUB_NAME`).
- Shows:
  - Hub title + short insight
  - Top 3 related signals (cost anomalies / recommendations)
  - Last updated timestamp
  - Click-to-expand details drawer
- **CDN-first data flow**: `public/hubs/{hubName}.json` served statically; no backend changes; zero API rate limits during runtime.
- **Graceful fallback**: if JSON missing or malformed, panel hides itself silently (no errors in UI).

### Why this is highest-value (<2h)
- Reuses existing knowledge-rag pattern (#knowledge-rag #graph #hub) by surfacing the most-connected hub (MOC) in the ops UI.
- No infra, no DB, no auth changes — pure static asset + lightweight frontend component.
- Aligns with Costinel philosophy: *Sense + Signal (ไม่ Execute)* — panel only signals, never executes changes.
- Fits ops focus: makes hub insights visible to cost-governance users immediately.

#### 3) Top-Hub Signal Panel component
`src/components/TopHubSignalPanel.vue`
```vue
<template>
  <div v-if="enabled && loaded && hubData" class="top-hub-panel">
    <div class="panel-header" @click="toggleExpanded">
      <div class="panel-title">
        <span class="hub-badge">{{ hubData.hub }}</span>
        <span class="hub-title">{{ hubData.title }}</span>
      </div>
      <div class="panel-meta">
        <span class="updated">Updated {{ formattedDate }}</span>
        <svg class="chevron" :class="{ open: expanded }" width="16" height="16" viewBox="0 0 16 16" fill="none" aria-hidden="true">
          <path d="M4 6l4 4 4-4" stroke="currentColor" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round"/>
        </svg>
      </div>
    </div>

    <div class="panel-insight" v-if="!expanded">
      {{ hubData.insight }}
    </div>

    <transition name="slide">
      <div v-if="expanded" class="panel-drawer">
        <div class="panel-insight expanded">
          {{ hubData.insight }}
        </div>
        <ul class="signals-list">
          <li v-for="s in hubData.signals" :key="s.id" class="signal-item" :class="`severity-${s.severity}`">
            <div class="signal-header">
              <span class="signal-badge" :class="`badge-${s.severity}`">{{ s.severity }}</span>
              <span class="signal-title">{{ s.title }}</span>
            </div>
            <div class="signal-desc">{{ s.description }}</div>
            <div class="signal-action">{{ s.action }}</div>
          </li>
        </ul>
      </div>
    </transition>
  </div>
</template>

<script>
export default {
  name: "TopHubSignalPanel",
  data() {
    return {
      enabled: import.meta.env.VITE_HUB_PANEL_ENABLED !== "false",
      hubName: import.meta.env.VITE_HUB_NAME || "MOC",
      hubData: null,
      loaded: false,
      expanded: false,
      loadError: false
    };
  },
  computed: {
    formattedDate() {
      if (!this.hubData?.updatedAt) return "";
      try {
        return new Date(this.hubData.updatedAt).toLocaleDateString(undefined, {
          month: "short",
          day: "numeric",
          hour: "2-digit",
          minute: "2-digit"
        });
      } catch {
        return this.hubData.updatedAt;
      }
    }
  },
  async mounted() {
    if (!this.enabled) return;
    await this.loadHub();
  },
  methods: {
    async loadHub() {
      try {
        // CDN-first: public/ is served statically; no auth, no API rate limits.
        const res = await fetch(`/hubs/${this.hubName}.json`, { cache: "no-store" });
        if (!res.ok) throw new Error("Hub file not found");
        const json = await res.json();
        // Minimal validation
        if (!json.hub || !Array.isArray(json.signals)) throw new Error("Invalid hub payload");
        this.hubData = json;
        this.loaded = true;
      } catch (err) {
        // Graceful non-blocking fallback: hide panel silently.
        console.debug("[TopHubSignalPanel] skipped", err.message);
        this.loadError = true;
        this.loaded = false;
        this.enabled = false;
      }
    },
    toggleExpanded() {
      this.expanded = !this.expanded;
    }
  }
};
</script>
