# airship / design

### Final Synthesis (Best Parts + Correctness + Actionability)

**Highest-value incremental improvement (<2h):**  
Implement a **manifest-driven, CDN-only dataset loader** for Surrogate-1 training that:

1. **Pre-lists the target date folder once** (Mac orchestration) → saves `file-list.json` with `{repo, path, sha?, size}`.  
2. **Embeds that list in training** → Lightning Studio fetches every parquet via **CDN URLs only** (zero HF API calls during training).  
3. **Projects `{prompt, response}` at parse time** (avoids `pyarrow.CastError` from mixed schemas).  
4. **Uses deterministic repo hashing** for commit-cap avoidance (5-sibling spread).  
5. **Reuses existing Studio if running** and falls back to free-tier machines.

### Concrete Implementation Plan (Executable in <2h)

#### 1. Mac orchestration: one-time manifest build  
`scripts/build_file_manifest.py`

- Uses HF API **once** (`list_repo_tree`, non-recursive) for a date folder.  
- Saves `file-list.json` (repo, path, size).  
- Commits to airship repo or uploads to surrogate config bucket for deterministic reuse.

```python
#!/usr/bin/env python3
import json, os
from huggingface_hub import HfApi

REPO = "axentx/surrogate-1-batches"
DATE_FOLDER = os.getenv("DATE_FOLDER", "2025-01-01")  # override per run
OUTPUT = os.getenv("OUTPUT", "file-list.json")

api = HfApi()
files = api.list_repo_tree(repo_id=REPO, path=DATE_FOLDER, recursive=False, repo_type="dataset")
manifest = [{"repo": REPO, "path": f.path} for f in files if f.path.endswith(".parquet")]

with open(OUTPUT, "w") as f:
    json.dump(manifest, f, indent=2)
print(f"Wrote {len(manifest)} files to {OUTPUT}")
```

#### 2. Training script: CDN-only, schema-projected iterable dataset  
`surrogate/train/cdn_dataset.py`

- Loads `file-list.json`.  
- Downloads each parquet via **CDN URL** (`resolve/main/...`).  
- Projects only `["prompt", "response"]` (prevents `pyarrow.CastError`).  
- Deterministic repo selection for sibling spread.

```python
import json, io, os, hashlib
from typing import Iterable, Dict
import torch
from torch.utils.data import IterableDataset
import pyarrow.parquet as pq
import requests

class CDNParquetDataset(IterableDataset):
    def __init__(self, manifest_path: str, sibling_count: int = 5):
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.sibling_count = sibling_count

    def _repo_for_index(self, idx: int) -> str:
        # Deterministic sibling spread: axentx/surrogate-1-batches-0 .. -4
        base = "axentx/surrogate-1-batches"
        suffix = idx % self.sibling_count
        return f"{base}-{suffix}" if suffix > 0 else base

    def _cdn_url(self, repo: str, path: str) -> str:
        return f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"

    def _project_to_pair(self, table) -> Iterable[Dict[str, str]]:
        if "prompt" not in table.column_names or "response" not in table.column_names:
            return
        prompts = table["prompt"].to_pylist()
        responses = table["response"].to_pylist()
        for p, r in zip(prompts, responses):
            if isinstance(p, str) and isinstance(r, str) and p.strip() and r.strip():
                yield {"prompt": p.strip(), "response": r.strip()}

    def __iter__(self):
        worker_info = torch.utils.data.get_worker_info()
        total = len(self.manifest)
        start, end = 0, total
        if worker_info:
            per_worker = total // worker_info.num_workers
            start = worker_info.id * per_worker
            end = start + per_worker if worker_info.id < worker_info.num_workers - 1 else total

        for i in range(start, end):
            item = self.manifest[i]
            repo = self._repo_for_index(i)
            url = self._cdn_url(repo, item["path"])
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            table = pq.read_table(io.BytesIO(resp.content), columns=["prompt", "response"])
            yield from self._project_to_pair(table)
```

Usage in `train_surrogate.py`:

```python
from surrogate.train.cdn_dataset import CDNParquetDataset

dataset = CDNParquetDataset("file-list.json")
loader = DataLoader(dataset, batch_size=None, num_workers=4)
```

#### 3. Lightning Studio launcher: reuse + deterministic start  
`scripts/launch_studio.py`

- Reuse running Studio if exists (filter by name + status).  
- Pass `FILE_MANIFEST_URL` or embed `file-list.json`.  
- Prefer free-tier `L40S`; fallback to `H200` if available.  
- Guard against idle-stop: check status before `.run()`; restart if stopped.

```python
import os, time
from lightning import Studio, Teamspace

STUDIO_NAME = os.getenv("STUDIO_NAME", "surrogate-1-train")
MANIFEST_PATH = os.getenv("MANIFEST_PATH", "file-list.json")

teamspace = Teamspace()
existing = [s for s in teamspace.studios if s.name == STUDIO_NAME and s.status == "running"]

if existing:
    studio = existing[0]
    print(f"Reusing running studio: {studio.name}")
else:
    studio = Studio(
        name=STUDIO_NAME,
        machine="L40S",
        env={"FILE_MANIFEST_PATH": MANIFEST_PATH},
    )
    # Ensure not idle-stopped
    if studio.status == "stopped":
        studio.start()
        time.sleep(10)

studio.run(
    repo="your-org/surrogate",
    script="surrogate/train/train_surrogate.py",
    env={"FILE_MANIFEST_PATH": MANIFEST_PATH},
)
```

#### 4. (Optional) HF ingestion wrapper: safe writes with sibling spread  
`scripts/ingest_mirror_batch.py`

- Writes projected `{prompt, response}` parquet to deterministic sibling repos.  
- On 429: wait 360s, retry.  
- Keeps ingestion within 128/hr cap.

```python
import time, hashlib
from huggingface_hub import HfApi

api = HfApi()

def deterministic_repo(slug: str, sibling_count: int = 5) -> str:
    h = int(hashlib.md5(slug.encode()).hexdigest(), 16)
    suffix = h % sibling_count
    base = "axentx/surrogate-1-batches"
    return f"{base}-{suffix}" if suffix > 0 else base

def safe_upload_parquet(repo, path, data):
    for attempt in range(5):
        try:
            api.upload_file(repo_id=repo, path_in_repo=path, path_or_fileobj=data, repo_type="dataset")
            return
        except Exception as e:
            if "429" in str(e):
                time.sleep(360)
            else:
                raise
```

### Why This Is Correct + Actionable

- **Avoids HF API rate limits during training**: manifest built once; training uses CDN URLs only.  
- **Prevents `pyarrow.CastError`**: explicit column projection (`["prompt","response"]`) at read time.  
- **Deterministic sibling spread**: repo hashing avoids commit-cap bottlenecks.  
- **Studio-friendly**: reuses running instances, uses free-tier where possible, guards against idle-stop.  
- **<2h scope**: focused changes (manifest script, CDN dataset, Studio launcher) with clear code snippets.
