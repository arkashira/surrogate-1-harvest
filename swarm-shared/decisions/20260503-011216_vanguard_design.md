# vanguard / design

### Final synthesized solution (correct + actionable)

**Core diagnosis (merged, de-duplicated)**
- No persisted `(repo, dateFolder)` manifest → every page load triggers authenticated `list_repo_tree`/`/api/` calls → burns HF quota and risks 429s.
- Data fetches use authenticated `/api/` paths instead of public CDN → unnecessary rate-limit pressure and slower downloads.
- No client-side cache layer → repeated enumeration across sessions; no offline/low-bandwidth fallback.
- Training/ingestion uses `load_dataset(streaming=True)` on heterogeneous repos → latent `pyarrow.CastError`.
- No deterministic mapping for writes → ingestion can hit HF commit caps at scale.

**Guiding principles**
- Correctness: avoid `load_dataset(streaming=True)` on heterogeneous repos; use typed, schema-aware reads or per-file streaming with schema enforcement.
- Actionability: implement a small, typed manifest + CDN-bypass layer in the frontend plus a one-time Mac/Linux orchestration script to produce the manifest.
- Rate-limit hygiene: use public CDN (`resolve/main/...`) for reads; keep authenticated calls to manifest generation only (once per dateFolder).

### 1) Frontend manifest + CDN layer (TypeScript, SvelteKit)

`src/lib/data/manifest.ts`
```ts
// Lightweight persisted manifest: repo + dateFolder -> file list
export interface Manifest {
  repo: string;
  dateFolder: string; // e.g. "2026-04-29"
  files: string[];    // relative paths under dateFolder
  generatedAt: number; // unix ms
  ttl: number;        // ms (e.g. 24h)
}

const STORAGE_KEY = "vanguard:manifest";

export function getStoredManifest(): Manifest | null {
  try {
    const raw = localStorage.getItem(STORAGE_KEY);
    if (!raw) return null;
    const m: Manifest = JSON.parse(raw);
    if (Date.now() - m.generatedAt > m.ttl) return null;
    return m;
  } catch {
    return null;
  }
}

export function storeManifest(m: Manifest): void {
  try {
    localStorage.setItem(STORAGE_KEY, JSON.stringify(m));
  } catch {
    // ignore storage limits/quota errors
  }
}

// CDN bypass: public read URLs (no Authorization header)
export function cdnUrl(repo: string, filePath: string): string {
  // Ensure no double slashes and normalize path
  const cleanRepo = repo.replace(/\/+$/, "");
  const cleanPath = filePath.replace(/^\/+/, "");
  return `https://huggingface.co/datasets/${cleanRepo}/resolve/main/${cleanPath}`;
}
```

`src/lib/data/fetcher.ts`
```ts
import { cdnUrl } from "./manifest";

const DATASET_REPO = "axentx/vanguard-data"; // adjust per project

// Fetch raw parquet (or other) bytes via CDN
export async function fetchParquetFile(filePath: string): Promise<ArrayBuffer> {
  const url = cdnUrl(DATASET_REPO, filePath);
  const res = await fetch(url, { cache: "force-cache" });
  if (!res.ok) throw new Error(`CDN fetch failed: ${res.status} ${url}`);
  return res.arrayBuffer();
}

// Optional: fetch remote manifest JSON produced by orchestration
export async function fetchRemoteManifest(dateFolder: string): Promise<Response> {
  const url = cdnUrl(DATASET_REPO, `manifests/${dateFolder}.json`);
  return fetch(url, { cache: "no-cache" });
}
```

`src/routes/+page.svelte` (minimal excerpt)
```svelte
<script lang="ts">
  import { onMount } from "svelte";
  import { getStoredManifest, storeManifest, cdnUrl } from "$lib/data/manifest";
  import { fetchRemoteManifest } from "$lib/data/fetcher";

  let files: string[] = [];
  let loading = true;

  // In real app, drive this from router/UI
  const DATE_FOLDER = "2026-04-29";
  const REPO = "axentx/vanguard-data";

  async function loadManifest() {
    const stored = getStoredManifest();
    if (stored && stored.dateFolder === DATE_FOLDER) {
      files = stored.files;
      return;
    }

    // Try remote manifest first (CDN). Fallback to local /api if needed.
    try {
      const res = await fetchRemoteManifest(DATE_FOLDER);
      if (!res.ok) throw new Error("Remote manifest unavailable");
      const manifest = await res.json();
      files = manifest.files;
      storeManifest({ ...manifest, generatedAt: Date.now(), ttl: 24 * 3600 * 1000 });
      return;
    } catch {
      // fallback: authenticated endpoint (kept for robustness)
      const res = await fetch(`/api/manifest?dateFolder=${DATE_FOLDER}`);
      if (!res.ok) throw new Error("Manifest unavailable");
      const manifest = await res.json();
      files = manifest.files;
      storeManifest({ ...manifest, generatedAt: Date.now(), ttl: 24 * 3600 * 1000 });
    }
  }

  onMount(async () => {
    loading = true;
    try {
      await loadManifest();
    } finally {
      loading = false;
    }
  });
</script>

{#if loading}
  <p>Loading manifest...</p>
{:else}
  <ul>
    {#each files as f}
      <li>
        <a href={cdnUrl(REPO, f)} target="_blank" rel="noreferrer">{f}</a>
      </li>
    {/each}
  </ul>
{/if}
```

### 2) Backend/API helper (optional but recommended)

Expose a lightweight authenticated endpoint to generate/fetch manifest when CDN copy is missing. Keep it minimal to reduce quota use.

`/api/manifest` (pseudo-handler)
- Accept `dateFolder`.
- If local cached manifest exists and fresh → return it.
- Else call HF API **once** (`list_repo_tree` non-recursive) → produce manifest → cache locally (disk/Redis) and return.
- Set `Cache-Control` to reduce repeat calls.

### 3) Mac/Linux orchestration (one-time or scheduled)

`scripts/generate-manifest.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

REPO="axentx/vanguard-data"
DATEFOLDER="${1:-2026-04-29}"
OUT="manifests/${DATEFOLDER}.json"

mkdir -p manifests

# Single non-recursive API call to list top-level of dateFolder
python3 -c "
import json, os, sys
from huggingface_hub import list_repo_tree

repo = os.getenv('REPO')
datefolder = os.getenv('DATEFOLDER')
if not repo or not datefolder:
    sys.exit('REPO and DATEFOLDER env required')

try:
    files = [f.rfilename for f in list_repo_tree(repo, path=datefolder, recursive=False)]
except Exception as e:
    sys.exit(f'Failed to list repo tree: {e}')

manifest = {
    'repo': repo,
    'dateFolder': datefolder,
    'files': sorted(files),
    'generatedAt': 0,   # runtime fill on client
    'ttl': 86400000
}

with open(os.getenv('OUT'), 'w') as f:
    json.dump(manifest, f, indent=2)
print(f'Wrote {len(files)} entries to {os.getenv("OUT")}')
" REPO="$REPO" DATEFOLDER="$DATEFOLDER" OUT="$OUT"

# Optional: commit and push manifest to dataset repo (so CDN serves it)
# git add "$OUT" && git commit -m "Add manifest $DATEFOLDER" && git push
```

Make executable and run:
```bash
chmod +x scripts/generate-manifest.sh
./scripts/generate-manifest.sh 2026-04-29
```
