# Costinel / ops

## Final Implementation — Top-Hub Signal Panel (CDN-first, <2h, production-ready)

### Scope (single highest-value deliverable)
Add a **non-blocking Top-Hub Signal Panel** to the Costinel dashboard that surfaces the most-connected hub (e.g., "MOC") and 3 contextual insights from the knowledge-RAG graph.  
- CDN-only data fetches (HuggingFace `resolve/main/...`) to avoid HF API rate limits.  
- Zero runtime writes; silent failure so core cost views are never affected.  
- Fits into existing `/opt/axentx/Costinel` layout and can be toggled or collapsed.

---

### Architecture (merged, contradiction-resolved)
- **Single source file** in the HF dataset repo:  
  `top-hub.json` (canonical) produced by an **indexer job**.  
  - Keeps CDN URL simple and stable:  
    `https://huggingface.co/datasets/{REPO_ID}/resolve/main/top-hub.json`
- **Indexer job** (Mac/node) lists date-partitioned RAG snapshots, selects the newest non-empty top-hub file, and writes `top-hub.json` to `public/data/` locally, then pushes to HF.
- **Dashboard** fetches via CDN with `stale-while-revalidate` (SWR/React-Query) and **never blocks** rendering. Failures are silent and do not surface errors to users.
- **No Authorization header at runtime**; CDN fetches are public. HF_TOKEN is only used by the indexer.

#### 1) Indexer / Orchestration script
`scripts/update-hub-index.js`
```js
#!/usr/bin/env node
/**
 * Generate top-hub.json (CDN-first) from knowledge-RAG snapshots.
 *
 * Run after RAG snapshots are pushed.
 * Prerequisites:
 * - HF_TOKEN in env (for listing + upload)
 * - node >=18
 * - @huggingface/hub installed
 */

import { config } from 'dotenv';
config();

import { HfApi, HfFileSystem } from '@huggingface/hub';
import fs from 'fs/promises';
import path from 'path';
import { fileURLToPath } from 'url';

const __filename = fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);
const ROOT = path.resolve(__dirname, '..');

const REPO_ID = process.env.HF_RAG_REPO || 'axentx/costinel-rag';
const RAG_PATH = process.env.RAG_PATH || 'knowledge-rag/snapshots';
const OUT_DIR = path.join(ROOT, 'public', 'data');
const OUT_FILE = path.join(OUT_DIR, 'top-hub.json');
const CDN_BASE = `https://huggingface.co/datasets/${REPO_ID}/resolve/main`;

async function main() {
  try {
    const api = new HfApi({ token: process.env.HF_TOKEN || undefined });

    // List top-level date folders (non-recursive)
    const tree = await api.listRepoTree({
      repo: REPO_ID,
      path: RAG_PATH,
      recursive: false,
    });

    const dateFolders = tree
      .filter((t) => t.type === 'directory')
      .map((t) => t.path)
      .sort()
      .reverse();

    if (dateFolders.length === 0) {
      console.warn('No date folders found under', RAG_PATH);
      return;
    }

    // Find newest folder with a top-hub file
    let selected = null;
    for (const folder of dateFolders) {
      const files = await api.listRepoTree({
        repo: REPO_ID,
        path: folder,
        recursive: false,
      });

      const hubFile = files.find(
        (t) =>
          t.type === 'file' &&
          /top-?hub/i.test(t.path) &&
          (t.path.endsWith('.json') || t.path.endsWith('.jsonl'))
      );

      if (hubFile) {
        selected = {
          folder,
          file: hubFile.path,
          name: path.basename(hubFile.path, '.json'),
          size: hubFile.size,
          date: folder.split('/').pop(),
        };
        break;
      }
    }

    if (!selected) {
      console.warn('No top-hub file found in recent snapshots.');
      return;
    }

    // Read selected file via HF filesystem (or fetch) to extract insights
    const hfFs = new HfFileSystem({ token: process.env.HF_TOKEN || undefined });
    const raw = await hfFs.readFile(`${REPO_ID}/${selected.file}`, 'utf-8');
    let parsed;
    try {
      parsed = JSON.parse(raw);
    } catch {
      // If JSONL, take first valid line
      const firstLine = raw.trim().split('\n').find((l) => l.trim());
      parsed = firstLine ? JSON.parse(firstLine) : {};
    }

    // Normalize insights
    const hubs = Array.isArray(parsed.hubs) ? parsed.hubs.slice(0, 3) : [];
    const topHub = hubs[0] || { name: selected.name || 'MOC', connections: 0, summary: '' };

    const payload = {
      generatedAt: new Date().toISOString(),
      repo: REPO_ID,
      source: selected,
      topHub: {
        name: topHub.name || 'MOC',
        connections: Number(topHub.connections) || 0,
        summary: topHub.summary || '',
      },
      insights: hubs.map((h) => ({
        name: h.name || '—',
        connections: Number(h.connections) || 0,
        summary: h.summary || '',
      })),
      cdnUrl: `${CDN_BASE}/data/top-hub.json`,
    };

    await fs.mkdir(OUT_DIR, { recursive: true });
    await fs.writeFile(OUT_FILE, JSON.stringify(payload, null, 2));
    console.log('top-hub.json generated:', OUT_FILE);

    // Optional: push to HF repo (if HF_TOKEN provided)
    if (process.env.HF_TOKEN) {
      try {
        await api.uploadFile({
          repo: { type: 'dataset', repoId: REPO_ID },
          path: 'data/top-hub.json',
          content: JSON.stringify(payload, null, 2),
          commitMessage: `chore: update top-hub.json (${payload.generatedAt})`,
        });
        console.log('top-hub.json pushed to HF dataset.');
      } catch (uploadErr) {
        console.warn('Failed to push top-hub.json to HF:', uploadErr.message);
      }
    }
  } catch (err) {
    console.error('Failed to update top-hub:', err.message);
    process.exit(1);
  }
}

main();
```

```bash
chmod +x scripts/update-hub-index.js
```
