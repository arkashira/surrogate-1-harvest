# workio / ops

## Final Synthesized Implementation

### 1. Diagnosis (merged, prioritized)
- **Signature bypass**: Missing or partial `X-Line-Signature` HMAC-SHA256 verification enables spoofed/replayed events.
- **Idempotency/replay gaps**: No deduplication key or atomic guard allows duplicate `attendance_punch`, leave, and OT rows on LINE retries or replays.
- **Tenant isolation failures**: Relying only on `userId` without tenant-scoped resolution risks cross-tenant misattribution and privilege escalation.
- **Schema/validation gaps**: Accepting malformed LINE payloads can crash handlers or bypass business rules.
- **Observability deficit**: Missing request/event IDs, tenant context, and metrics makes duplicate/replay detection and debugging hard.

### 2. Proposed Change (scope)
File: `/opt/axentx/workio/server/src/routes/webhook/line.ts`  
- Enforce `X-Line-Signature` verification (fail fast).  
- Extract canonical webhook event ID and build tenant-scoped idempotency key (`tenantId:userId:eventId`).  
- Perform atomic idempotency insert and upsert business records under a single transaction.  
- Add strict per-event schema validation and tenant resolution with channel ownership check.  
- Emit structured logs and metrics (request-id, tenant-id, event-id, duplicate/replay counters).

function verifyLineSignature(rawBody: Buffer, signature: string): boolean {
  if (!LINE_CHANNEL_SECRET) return true; // disabled in dev only
  try {
    const expected = crypto
      .createHmac('sha256', LINE_CHANNEL_SECRET)
      .update(rawBody)
      .digest('base64');
    return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
  } catch {
    return false;
  }
}

function extractIdempotencyKey(tenantId: string, event: any): string {
  const eventId = event?.webhookEventId || event?.deliveryId || '';
  const userId = event?.source?.userId || 'unknown';
  if (!eventId) {
    // Deterministic fallback using content hash
    const content = JSON.stringify(event);
    const hash = crypto.createHash('sha256').update(content).digest('hex').slice(0, 16);
    return `${tenantId}:${userId}:sha256_${hash}`;
  }
  return `${tenantId}:${userId}:${eventId}`;
}

async function resolveTenantByChannel(channelId: string, client: any) {
  const res = await client.query(
    `SELECT id FROM tenants WHERE line_channel_id = $1 AND is_active = true LIMIT 1`,
    [channelId]
  );
  return res.rows[0]?.id;
}

function validateEventShape(event: any): { valid: boolean; reason?: string } {
  if (!event || typeof event !== 'object') return { valid: false, reason: 'event_not_object' };
  if (!event.type || typeof event.type !== 'string') return { valid: false, reason: 'missing_type' };
  if (!event.source || typeof event.source !== 'object') return { valid: false, reason: 'missing_source' };
  if (!event.timestamp || typeof event.timestamp !== 'number') return { valid: false, reason: 'missing_timestamp' };
  return { valid: true };
}
