# airship / ops

## Final Implementation Plan — <2h Quality Increment

**Highest-value improvement**: Harden `airship discover` into a **deterministic, CDN-safe, zero-runtime orchestrator** that eliminates HF API rate limits and PyArrow schema errors while keeping Mac=CLI.

**Core strategy** (resolves contradictions in favor of correctness + actionability):
- **Use `list_repo_tree(..., recursive=False)`** (Candidate 1) for a single, deterministic directory listing instead of per-file API calls (Candidate 2). This is faster and avoids 128/hr commit cap.
- **Embed manifest in training package** (Candidate 1) so Lightning training uses **CDN-only fetches** with no HF API calls during data load.
- **Project to strict `{prompt,response}` parquet** (both) with **no mixed-schema writes** and **attribution via filename only** (`batches/mirror-merged/{date}/{slug}.parquet`).
- **Reuse running Lightning Studio** (Candidate 1) to avoid quota churn and idle-stop deaths.
- **Bash orchestrator with strict mode** (both) for Mac-side determinism.

## Concrete Steps (≤2h)

1. **Harden `airship/discover` orchestrator** (10–15 min)
   - Add shebang + `set -euo pipefail`
   - Accept `YYYY-MM-DD` arg; default to today
   - Use `list_repo_tree(path, recursive=False)` per date folder
   - Emit sorted, deterministic `manifest-{date}.json` with CDN prefix

2. **Projection + staging** (30–45 min)
   - Parse each file via CDN fetch; keep only `{prompt,response}`
   - Write to `batches/mirror-merged/{date}/{slug}.parquet`
   - **No `source`, no `ts` columns** in schema (prevents PyArrow CastError)

3. **Embed manifest in training package** (15–20 min)
   - Include `manifest-{date}.json` alongside parquet
   - Update `train.py` to read manifest and fetch via CDN URLs only
   - Pattern: `https://huggingface.co/datasets/{repo}/resolve/main/{path}`

4. **Lightning Studio reuse + idle handling** (30–45 min)
   - List `Teamspace.studios`; reuse running studio if present
   - Before `.run()`, check status; if stopped, restart with `target.start(machine=Machine.L40S)`
   - Prefer `lightning-lambda-prod` (H200) when available; fallback to `lightning-public-prod` (L40S)

5. **Smoke test** (15–20 min)
   - Run `airship discover 2026-05-02`
   - Verify manifest + parquet
   - Trigger short Lightning job; confirm CDN-only fetches (no 429)

### 1. Hardened `airship/discover` (Bash orchestrator)

```bash
#!/usr/bin/env bash
# airship/discover
# Deterministic, CDN-safe manifest generator for surrogate training.
# Usage: ./discover [YYYY-MM-DD]

set -euo pipefail
IFS=$'\n\t'

export SHELL=/bin/bash
export HF_REPO="${HF_REPO:-datasets/axentx/surrogate-mirror}"
export DATE="${1:-$(date +%Y-%m-%d)}"
export OUTDIR="batches/mirror-merged/${DATE}"
export MANIFEST="manifest-${DATE}.json"

mkdir -p "${OUTDIR}"

echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] Listing ${HF_REPO}/${DATE} (non-recursive)..."
python3 - <<PY
import json, os
from huggingface_hub import list_repo_tree
from datetime import datetime, timezone

HF_REPO = os.environ["HF_REPO"]
DATE = os.environ["DATE"]
OUTDIR = os.environ["OUTDIR"]
MANIFEST = os.environ["MANIFEST"]

# Single API call; non-recursive for deterministic directory listing
entries = list_repo_tree(repo_id=HF_REPO, path=DATE, recursive=False)
files = sorted([e.rfilename for e in entries if e.type == "file"])

manifest = {
    "date": DATE,
    "generated_at": datetime.now(timezone.utc).isoformat(),
    "repo": HF_REPO,
    "files": files,
    "cdn_prefix": f"https://huggingface.co/{HF_REPO}/resolve/main/{DATE}"
}

os.makedirs(OUTDIR, exist_ok=True)
manifest_path = os.path.join(OUTDIR, MANIFEST)
with open(manifest_path, "w") as f:
    json.dump(manifest, f, indent=2, sort_keys=True)

print(f"Wrote manifest with {len(files)} files to {manifest_path}")
PY

echo "[$(date -u +%Y-%m-%dT%H:%M:%SZ)] Manifest ready: ${OUTDIR}/${MANIFEST}"
```

### 2. Projection script (Python) — `project_to_parquet.py`

```python
#!/usr/bin/env python3
"""
Project raw mirror files to {prompt,response}-only parquet.
Attribution via filename: batches/mirror-merged/{date}/{slug}.parquet
No source/ts columns in schema.
"""
import json, os, pyarrow as pa, pyarrow.parquet as pq, uuid, requests
from pathlib import Path
from typing import List, Dict

HF_REPO = os.environ.get("HF_REPO", "datasets/axentx/surrogate-mirror")
DATE = os.environ["DATE"]
MANIFEST_PATH = Path(f"batches/mirror-merged/{DATE}/manifest-{DATE}.json")
OUTDIR = Path(f"batches/mirror-merged/{DATE}")
OUTDIR.mkdir(parents=True, exist_ok=True)

def cdn_fetch(path: str) -> bytes:
    url = f"https://huggingface.co/{HF_REPO}/resolve/main/{path}"
    r = requests.get(url, timeout=30)
    r.raise_for_status()
    return r.content

def parse_file(raw: bytes, ext: str) -> List[Dict[str, str]]:
    import json as _json
    if ext == ".jsonl":
        rows = []
        for line in raw.decode("utf-8").splitlines():
            if not line.strip():
                continue
            obj = _json.loads(line)
            rows.append({
                "prompt": obj.get("prompt") or obj.get("input") or "",
                "response": obj.get("response") or obj.get("output") or ""
            })
        return rows
    elif ext == ".json":
        obj = _json.loads(raw)
        if isinstance(obj, list):
            return [{
                "prompt": item.get("prompt") or item.get("input") or "",
                "response": item.get("response") or item.get("output") or ""
            } for item in obj]
        else:
            return [{
                "prompt": obj.get("prompt") or obj.get("input") or "",
                "response": obj.get("response") or obj.get("output") or ""
            }]
    else:
        raise ValueError(f"Unsupported extension: {ext}")

def main():
    with open(MANIFEST_PATH) as f:
        manifest = json.load(f)

    all_rows = []
    for rel in manifest["files"]:
        ext = Path(rel).suffix.lower()
        raw = cdn_fetch(rel)
        rows = parse_file(raw, ext)
        all_rows.extend(rows)

    if not all_rows:
        print("No rows extracted.")
        return

    table = pa.table({
        "prompt": [r["prompt"] for r in all_rows],
        "response": [r["response"] for r in all_rows],
    })

    slug = f"{DATE}-{str(uuid.uuid4())[:8]}"
    out_path = OUTDIR / f"{slug}.parquet"
    pq.write_table(table, out_path)
    print(f"Projected {len(all_rows)} rows -> {out_path}")

    # Update manifest to reference parquet (optional)
    manifest["parquet_slug"] = slug
    manifest["parquet_path"] = str(out_path)
    with open(MANIFEST_PATH, "w") as f:
        json.dump(manifest, f, indent=2, sort_keys=True)

if __name__ == "__main__":
    main()
```
