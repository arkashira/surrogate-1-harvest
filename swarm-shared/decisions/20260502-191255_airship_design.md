# airship / design

## Implementation Plan: `airship discover` — Zero-config strategic discovery CLI

**Highest-value incremental improvement (<2h):**  
Ship a single Bash CLI (`airship discover`) that operationalizes past patterns (business research + knowledge-rag + top-hub insight) and surfaces strategic context for Arkship/Surrogate platform decisions. Zero-config: detects repo root, runs market research, queries knowledge graph, prints top hub insight, and exits with actionable summary.

### Why this now
- Directly applies 3 past patterns: business research + knowledge-rag + top-hub doc insight.
- Small, self-contained script (<200 LOC) that can ship in <2h.
- Immediate value for design/planning tasks (like this one) and onboarding.

---

### Design

- CLI: `airship discover [--no-rag] [--no-research] [--json]`
- Auto-detects repo root via `.git` or `airship` marker.
- Runs `granite-business-research.sh` (if present) or lightweight market search fallback.
- Executes `knowledge-rag` query for top hub and related docs.
- Extracts and prints most-connected hub insight (e.g., MOC).
- Outputs human-readable summary by default; `--json` for tool integration.
- Defensive: graceful no-ops if deps/tools missing.

### Implementation (concrete)

#### 1) `lib/utils.sh` — helpers

```bash
#!/usr/bin/env bash
set -euo pipefail

AIRSHIP_ROOT=""

find_repo_root() {
  local dir
  dir="$(pwd)"
  while [[ "$dir" != "/" ]]; do
    if [[ -d "$dir/.git" ]] || [[ -f "$dir/.airship-root" ]]; then
      AIRSHIP_ROOT="$dir"
      return 0
    fi
    dir="$(dirname "$dir")"
  done
  echo "ERROR: Could not find repo root (no .git or .airship-root)" >&2
  return 1
}

log_info()  { printf "[INFO]  %s\n" "$*"; }
log_warn()  { printf "[WARN]  %s\n" "$*" >&2; }
log_error() { printf "[ERROR] %s\n" "$*" >&2; }

to_json_string() {
  local s="${1:-}"
  s="${s//\\/\\\\}"
  s="${s//\"/\\\"}"
  s="${s//$'\n'/\\n}"
  printf '%s' "$s"
}
```

#### 2) `lib/discover.sh` — core logic

```bash
#!/usr/bin/env bash
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
# shellcheck source=./utils.sh
source "$SCRIPT_DIR/utils.sh"

DO_RESEARCH=1
DO_RAG=1
OUTPUT_FORMAT="text"

while [[ $# -gt 0 ]]; do
  case "$1" in
    --no-rag)     DO_RAG=0; shift ;;
    --no-research) DO_RESEARCH=0; shift ;;
    --json)       OUTPUT_FORMAT="json"; shift ;;
    --help|-h)    cat <<'EOF'
Usage: airship discover [--no-rag] [--no-research] [--json]

Zero-config strategic context discovery for Arkship/Surrogate.
Runs business research and knowledge-rag, then prints top hub insight.
EOF
      exit 0 ;;
    *) log_error "Unknown option: $1"; exit 1 ;;
  esac
done

find_repo_root
cd "$AIRSHIP_ROOT"

research_out=""
rag_out=""
hub_insight=""

# ---- Business research ----
if [[ "$DO_RESEARCH" -eq 1 ]]; then
  if [[ -x "./scripts/granite-business-research.sh" ]]; then
    log_info "Running business research..."
    research_out=$(bash ./scripts/granite-business-research.sh 2>&1) || {
      log_warn "Business research script exited non-zero (continuing)"
      research_out=""
    }
  else
    log_warn "granite-business-research.sh not found or not executable — skipping research"
  fi
fi

# ---- Knowledge RAG + top-hub ----
if [[ "$DO_RAG" -eq 1 ]]; then
  if command -v knowledge-rag >/dev/null 2>&1; then
    log_info "Querying knowledge graph for top hub and related docs..."
    # Query for most-connected hub / high-level context
    rag_out=$(knowledge-rag query "top hub and most connected node for Arkship/Surrogate platform" 2>&1) || {
      log_warn "knowledge-rag query failed (continuing)"
      rag_out=""
    }

    # Try to extract a clear hub insight (heuristic)
    if [[ -n "$rag_out" ]]; then
      hub_insight=$(printf '%s' "$rag_out" | head -30 | grep -i -E 'most.connected|top.?hub|hub|MOC|central' | head -5 || true)
    fi
  else
    log_warn "knowledge-rag not available — skipping RAG"
  fi
fi

# ---- Output ----
if [[ "$OUTPUT_FORMAT" == "json" ]]; then
  printf '{\n'
  printf '  "repo_root": "%s",\n' "$(to_json_string "$AIRSHIP_ROOT")"
  printf '  "research": "%s",\n' "$(to_json_string "$research_out")"
  printf '  "rag_output": "%s",\n' "$(to_json_string "$rag_out")"
  printf '  "hub_insight": "%s"\n' "$(to_json_string "$hub_insight")"
  printf '}\n'
else
  printf '\n===== Arkship — Strategic Context (airship discover) =====\n\n'
  printf 'Repo root: %s\n\n' "$AIRSHIP_ROOT"

  if [[ -n "$research_out" ]]; then
    printf '--- Business Research ---\n%s\n\n' "$research_out"
  fi

  if [[ -n "$rag_out" ]]; then
    printf '--- Knowledge RAG (top hub + docs) ---\n%s\n\n' "$rag_out"
  fi

  if [[ -n "$hub_insight" ]]; then
    printf '--- Top Hub Insight ---\n%s\n\n' "$hub_insight"
  else
    printf '--- Top Hub Insight ---\n(no clear hub extracted)\n\n'
  fi

  printf 'Recommendation: Review the top hub (e.g., MOC) and related docs before planning tasks.\n'
  printf 'Tags: #knowledge-rag #graph #hub\n'
  printf '================================================================\n'
fi
```

#### 3) `bin/airship` — dispatcher (symlink or small wrapper)

```bash
#!/usr/bin/env bash
set -euo pipefail
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
REPO_ROOT="$(cd "$SCRIPT_DIR/.." && pwd)"

# delegate subcommands
case "${1:-}" in
  discover)
    exec bash "$REPO_ROOT/lib/discover.sh" "${@:2}"
    ;;
  *)
    echo "Usage: airship <discover>"
    exit 1
    ;;
esac
```

Make executable and link:

```bash
chmod +x bin/airship lib/discover.sh lib/utils.sh
# Optionally symlink into PATH for convenience during dev:
# ln -sf "$(pwd)/bin/airship" /usr/local/bin/airship
```
