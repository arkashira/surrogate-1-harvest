# workio / quality

## Final consolidated solution  
*(Best parts of both proposals merged; contradictions resolved in favor of correctness + concrete actionability)*

---

### 1. Diagnosis (merged, de-duplicated)
- **Non-idempotent webhook handler**: LINE retries on 5xx/timeouts replay clock/leave/OT events → duplicate punches, double approvals, state drift.
- **Missing `X-Line-Signature` verification**: no cryptographic proof of origin → spoofable webhooks.
- **No real-time subscription layer**: frontend relies on local React state/polling; LINE-driven changes invisible until refresh → perceived latency, silent failures.
- **Multi-tab/device divergence**: actions in one tab not reflected in others → stale UI, conflicting state.
- **No deduplication key on webhook payloads**: repeated delivery of same LINE event creates duplicate rows and approval-state drift.

---

### 2. Scope and high-level plan
- **Core files**  
  - `workio/server/src/routes/webhook/line.ts` (create/modify) — idempotent handler with signature verification.  
  - `workio/server/src/middleware/lineSignature.ts` — reusable signature verification middleware.  
  - `workio/server/src/db/idempotency.ts` — idempotency store (PostgreSQL-first; Redis optional).  
- **Runtime behavior**  
  - Verify `X-Line-Signature` before any processing.  
  - Reject replays with idempotency key + payload-hash check.  
  - Process events sequentially; never mark failed deliveries as processed.  
  - Emit real-time events after successful processing (SSE or existing WebSocket).  
- **Frontend (lightweight)**  
  - Subscribe to real-time events (SSE preferred for simplicity) and update global state (Zustand/Jotai).  
  - Sync across tabs via `BroadcastChannel` or localStorage events.  
  - Keep short polling as temporary fallback.

### 3. Implementation (concrete, production-ready)

#### 3.1 Signature verification middleware
```ts
// workio/server/src/middleware/lineSignature.ts
import crypto from 'crypto';
import { Request, Response, NextFunction } from 'express';

const LINE_CHANNEL_SECRET = process.env.LINE_CHANNEL_SECRET;
if (!LINE_CHANNEL_SECRET) {
  throw new Error('LINE_CHANNEL_SECRET is required');
}

export function verifyLineSignature(req: Request, res: Response, next: NextFunction) {
  const signature = req.headers['x-line-signature'] as string | undefined;
  if (!signature) {
    return res.status(401).json({ error: 'Missing X-Line-Signature' });
  }

  // Use raw body for accurate signature verification when available.
  // If raw body is not available, fall back to JSON stringification (less ideal).
  const body = (req as any).rawBody || JSON.stringify(req.body);
  const expected = crypto
    .createHmac('sha256', LINE_CHANNEL_SECRET)
    .update(body, 'utf8')
    .digest('base64');

  // Use timingSafeEqual to avoid timing attacks; ensure equal-length buffers.
  const sigBuf = Buffer.from(signature);
  const expBuf = Buffer.from(expected);
  const safeEqual = sigBuf.length === expBuf.length && crypto.timingSafeEqual(sigBuf, expBuf);

  if (!safeEqual) {
    return res.status(401).json({ error: 'Invalid signature' });
  }

  (req as any).lineVerified = true;
  next();
}
```
> **Note**: For production, configure your body parser to preserve raw body for this route (e.g., `express.raw` for `/webhook/line`), then `JSON.parse` in handler.

```ts
// workio/server/src/db/idempotency.ts
import { Pool } from 'pg';

const pool = new Pool({ connectionString: process.env.DATABASE_URL });

export async function isDuplicate(key: string, payloadHash: string): Promise<boolean> {
  const client = await pool.connect();
  try {
    const r = await client.query(
      'SELECT 1 FROM line_event_idempotency WHERE idempotency_key = $1 AND payload_hash = $2',
      [key, payloadHash]
    );
    return r.rowCount > 0;
  } finally {
    client.release();
  }
}

export async function markProcessed(key: string, eventType: string, payloadHash: string): Promise<void> {
  const client = await pool.connect();
  try {
    await client.query(
      'INSERT INTO line_event_idempotency (idempotency_key, event_type, payload_hash) VALUES ($1, $2, $3) ON CONFLICT DO NOTHING',
      [key, eventType, payloadHash]
    );
  } finally {
    client.release();
  }
}
```

#### 3.3 Idempotent LINE webhook route
```ts
// workio/server/src/routes/webhook/line.ts
import { Router } from 'express';
import crypto from 'crypto';
import { verifyLineSignature } from '../../middleware/lineSignature';
import { isDuplicate, markProcessed } from '../../db/idempotency';
import { broadcastLineEvents } from '../../lib/events'; // implement as SSE/WS emitter

const router = Router();

function buildIdempotencyKey(body: any): string {
  // Prefer stable LINE identifiers when present
  const deliveryId = body?.deliveryId;
  const timestamp = body?.timestamp;
  const sourceId = body?.source?.userId || body?.source?.groupId || '';
  const type = (body?.events?.[0]?.type || 'unknown') as string;

  if (deliveryId) return `line:deliveryId:${deliveryId}`;

  const payloadHash = crypto.createHash('sha256').update(JSON.stringify(body)).digest('hex');
  return `line:timestamp:${timestamp}:source:${sourceId}:type:${type}:hash:${payloadHash}`;
}

function payloadHash(body: any): string {
  return crypto.createHash('sha256').update(JSON.stringify(body)).digest('hex');
}

}

router.post('/line', verifyLineSignature, express.json(), async (req, res) => {
  const body = req.body;
  const key = buildIdempotencyKey(body);
  const hash = payloadHash(body);

  try {
    if (await isDuplicate(key, hash)) {
      return res.status(200).json({ ok: true, duplicate: true });
    }

    const events = body.events || [];
    for (const ev of events) {
      await handleLineEvent(ev);
    }

    await markProcessed(key, events[0]?.type || 'unknown', hash);

    // Broadcast for real-time frontends
    broadcastLineEvents({ events });

    return res.status(200).json({ ok: true });
  } catch (err) {
    console.error('LINE webhook processing failed', err);
    // Do NOT mark processed on failure — allow LINE retry
    return res.status(500).json({ error: 'Processing failed' });
  }
});

export default router;
```
