## v0.1 scope
v0.1 ships a single-platform (Linux x86_64) CLI-only stealth client that creates a local SOCKS5 listener, routes traffic through a meek-like cover over HTTPS to a free-tier Cloudflare Worker egress (acting as a reversible proxy to origin), with a best-effort userspace TUN fallback; client-side encrypted SQLite for profiles/sessions; kill-switch via local firewall rules (nftables/iptables) where possible; telemetry limited to anonymized session hashes and blocked/ok flags; zero server control required from user.

## Open questions
- Which cover strategy (meek vs shadowed-TLS vs domain-front) provides the best reach/stealth trade-off for target regions given free-tier egress constraints?
- How will we distribute and rotate camouflaged endpoints without centralized infra while avoiding domain/IP takedowns?
- What minimal client-side telemetry schema satisfies WAU-ZB without risking deanonymization if logs are seized?
- How do we implement a reliable fail-closed kill-switch across platforms (Windows/Linux/macOS) without admin/root?
- Can we run TUN in unprivileged userspace on all target OSes, and what fallback (e.g., tun/tap vs raw-socket proxy) is acceptable if not?
