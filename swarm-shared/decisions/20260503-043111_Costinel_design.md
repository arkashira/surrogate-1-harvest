# Costinel / design

## Final Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Ship a resilient “Top Hub” signal panel into Costinel that shows the most-connected hub (e.g., “MOC”) and related docs using CDN-fetched artifacts.  
**Principle**: Strict **Sense + Signal — ไม่ Execute**. Zero runtime HF API calls. No new infra, no secrets, no DB migrations.

---

### Architecture (CDN-first, correct + actionable)

1. **Offline/Mac orchestration** (run once or on schedule):
   - Use `list_repo_tree(path='knowledge-rag/hubs', recursive=True)` to enumerate hub folders and markdown files.
   - Compute hub connectivity (in-degree/out-degree) from link graph stored as `graph-links.json` or parsed from markdown frontmatter.
   - Pick the top hub; produce `top-hub.json` + `hub-docs.json`.
   - Commit artifacts to repo (or push to CDN path) so they’re available at:
     - `https://huggingface.co/datasets/AXENTX/Costinel/resolve/main/knowledge-rag/signals/top-hub.json`
     - `https://huggingface.co/datasets/AXENTX/Costinel/resolve/main/knowledge-rag/signals/hub-docs.json`

2. **Runtime (Costinel frontend)**:
   - Fetch the two JSON files via CDN URLs (no Authorization header).
   - Render a compact “Top Hub” signal card with:
     - Hub name + short description
     - Connection count (degree)
     - 3–5 related docs (title + link)
   - Graceful fallback if CDN fails (cached last-known or empty state).

3. **Caching & resilience**:
   - Frontend: `localStorage` cache + 5-minute stale-while-revalidate.
   - No retries on 429; CDN tier has higher limits; if unavailable, degrade silently.

#### 2) Add a small Python helper (Mac/offline) to generate artifacts

`scripts/generate_top_hub_signal.py`
```python
#!/usr/bin/env python3
"""
Offline helper to generate top-hub signal artifacts for CDN-first consumption.
Run on Mac (or CI) after rate-limit window clears.
"""
import json
import os
from datetime import datetime, timezone
from pathlib import Path

try:
    from huggingface_hub import list_repo_tree
except ImportError:
    print("Install huggingface_hub to run this script (pip install huggingface_hub).")
    exit(0)

REPO_ID = "AXENTX/Costinel"
HUBS_PATH = "knowledge-rag/hubs"
OUTPUT_DIR = Path("knowledge-rag/signals")
OUTPUT_DIR.mkdir(parents=True, exist_ok=True)

def build_hub_graph():
    """
    Lightweight graph builder:
    - Each hub folder is a node.
    - Parse markdown frontmatter or explicit 'links' field if present;
      otherwise count cross-references by scanning for other hub names in content.
    Returns dict of hub -> {degree, doc_count, description, docs[]}
    """
    # For MVP: use folder presence + simple doc counts.
    # Replace with real link parsing if graph-links.json exists.
    hubs = {}
    try:
        tree = list_repo_tree(repo_id=REPO_ID, path=HUBS_PATH, recursive=True)
    except Exception as exc:
        print(f"Could not list repo tree: {exc}")
        return hubs

    for item in tree:
        if item.rfilename.endswith(".md"):
            parts = item.rfilename.split("/")
            if len(parts) >= 2:
                hub_name = parts[1]
                hubs.setdefault(hub_name, {"docs": [], "degree": 0, "description": ""})
                hubs[hub_name]["docs"].append({
                    "title": Path(item.rfilename).stem.replace("-", " ").title(),
                    "path": item.rfilename,
                    "url": f"https://huggingface.co/datasets/{REPO_ID}/resolve/main/{item.rfilename}"
                })

    # Simple degree = doc_count + cross-hub references (placeholder)
    for hub in hubs:
        hubs[hub]["doc_count"] = len(hubs[hub]["docs"])
        hubs[hub]["degree"] = hubs[hub]["doc_count"]  # replace with real graph degree

    return hubs

def pick_top_hub(hubs):
    if not hubs:
        return None
    return max(hubs.items(), key=lambda kv: kv[1]["degree"])

def main():
    hubs = build_hub_graph()
    top = pick_top_hub(hubs)
    if not top:
        print("No hubs found.")
        return

    hub_name, hub_meta = top
    top_payload = {
        "hub": hub_name,
        "title": hub_name.replace("-", " ").title(),
        "description": hub_meta.get("description", "Central knowledge hub."),
        "degree": hub_meta["degree"],
        "type": "hub",
        "updated": datetime.now(timezone.utc).isoformat()
    }

    docs_payload = hub_meta["docs"][:5]
    for d in docs_payload:
        d["hub"] = hub_name
        d["relevance"] = round(1.0 - (docs_payload.index(d) * 0.1), 2)

    (OUTPUT_DIR / "top-hub.json").write_text(json.dumps(top_payload, indent=2))
    (OUTPUT_DIR / "hub-docs.json").write_text(json.dumps(docs_payload, indent=2))
    print(f"Generated top-hub signal: {hub_name} (degree={hub_meta['degree']})")

if __name__ == "__main__":
    main()
```
