# Costinel / design

### Final Consolidated Design (≤2h)

**Goal**  
Ship a single, deterministic discovery CLI that produces machine-readable manifests and readiness checks so onboarding, training, and governance scripts run without repeated API calls or fragile assumptions.

**Scope (≤2h)**  
- One CLI: `scripts/discover.py` → outputs `manifests/sources.json`, `manifests/top-hub.json`, `manifests/readiness.json`  
- Cloud account enumeration (AWS/GCP/Azure) via local profiles + optional env keys (read-only)  
- HF dataset file list snapshot (one CDN/tree call) + rate-limit bypass guidance  
- Top-hub insight from `knowledge-rag` (reuse existing graph; fallback placeholder)  
- Readiness checks (bash/cron, executable bits, Lightning quota, Kaggle auth format)  
- No training code changes; no long-running jobs.

---

### Implementation Plan

1. **Create `scripts/discover.py`**  
   Orchestrates:  
   - `collect_cloud_accounts()`  
   - `collect_hf_dataset_snapshot(repo_id, path)`  
   - `collect_top_hub()`  
   - `run_readiness_checks()`  
   - `write_manifests(out_dir="manifests")`

2. **Cloud accounts**  
   Use boto3/azure-identity/google-auth local profiles; only `Describe*/List*` read calls. Minimal fields: `{cloud, account_id, alias, region, status}`.

3. **HF dataset snapshot**  
   One `list_repo_tree(path=..., recursive=False)` call. Save `{path, cdn_url, size, sha}`. Include comment with rate-limit bypass guidance and CDN-only fetch pattern for training scripts.

4. **Top-hub insight**  
   If knowledge-rag graph file exists (e.g., `graph/top_hub.json` or `rag/hubs.json`), read and pick highest-degree hub; else produce placeholder with guidance to run `knowledge-rag` first.

5. **Readiness checks**  
   Verify:  
   - Bash shebang + executable bit on wrapper scripts (`opus-pr-reviewer`, `active-learning`)  
   - `SHELL=/bin/bash` in current crontab (if crontab present)  
   - Lightning Studio reuse availability (list running studios via SDK if `LIGHTNING_API_KEY` present)  
   - Kaggle auth format (check `KAGGLE_USERNAME/KAGGLE_KEY` and note Bearer requirement for KGAT)

6. **Output**  
   `manifests/sources.json`, `manifests/top-hub.json`, `manifests/readiness.json`.

7. **Add to repo**  
   Lightweight `Makefile` targets: `make discover`, `make readiness`.

#!/usr/bin/env python3
"""
Costinel discovery module.
Produces:
  manifests/sources.json      -> cloud accounts + HF dataset snapshot
  manifests/top-hub.json      -> highest-degree hub from knowledge-rag
  manifests/readiness.json    -> readiness checks for cron/lightning/kaggle/bash

import json
import os

from datetime import datetime, timezone

MANIFESTS_DIR = Path(__file__).parent.parent / "manifests"
MANIFESTS_DIR.mkdir(exist_ok=True)

# ---------- Helpers ----------
def safe_call(fn):
    try:
        return fn(), None
    except Exception as e:
        return None, str(e)

def is_executable(path: Path) -> bool:
    return path.is_file() and (path.stat().st_mode & stat.S_IXUSR) != 0

def has_bash_shebang(path: Path) -> bool:
    try:
        with open(path, "rb") as f:
            first = f.readline(128).strip()
        return first.startswith(b"#!/usr/bin/env bash") or first.startswith(b"#!/bin/bash")
    except Exception:
        return False

def utc_now_iso() -> str:
    return datetime.now(timezone.utc).isoformat()

# ---------- Cloud accounts ----------
def collect_cloud_accounts() -> Dict[str, Any]:
    results = {"aws": [], "gcp": [], "azure": [], "errors": []}

    # AWS
    try:
        import boto3
        from botocore.exceptions import ClientError, NoCredentialsError

        sts = boto3.client("sts")
        out, err = safe_call(lambda: sts.get_caller_identity())
        if out:
            results["aws"].append(
                {
                    "account_id": out.get("Account"),
                    "user_id": out.get("UserId"),
                    "arn": out.get("Arn"),
                    "alias": None,
                    "region": boto3.Session().region_name or os.getenv("AWS_DEFAULT_REGION"),
                    "status": "active",
                }
            )
        else:
            results["errors"].append({"cloud": "aws", "error": err})
    except Exception as e:
        results["errors"].append({"cloud": "aws", "error": str(e)})

    # GCP (placeholder pattern; implement with google-cloud-resource-manager if desired)
    try:
        import google.auth
        from google.auth.exceptions import DefaultCredentialsError

        creds, project = safe_call(lambda: google.auth.default())
        if project:
            results["gcp"].append(
                {
                    "project_id": project,
                    "alias": None,
                    "region": os.getenv("GOOGLE_CLOUD_REGION"),
                    "status": "active",
                }
            )
    except Exception as e:
        results["errors"].append({"cloud": "gcp", "error": str(e)})

    # Azure (placeholder pattern; implement with azure-identity if desired)
    try:
        from azure.identity import AzureCliCredential
        from azure.mgmt.resource import SubscriptionClient

        cred = AzureCliCredential()
        client = SubscriptionClient(cred)
        for sub in client.subscriptions.list():
            results["azure"].append(
                {
                    "subscription_id": sub.subscription_id,
                    "display_name": sub.display_name,
                    "tenant_id": sub.tenant_id,
                    "state": sub.state,
                    "status": "active",
                }
            )
    except Exception as e:
        results["errors"].append({"cloud": "azure", "error": str(e)})

    return results

# ---------- HF dataset snapshot ----------
def collect_hf_dataset_snapshot(repo_id: str, path: str) -> Dict[str, Any]:
    snapshot = {"repo_id": repo_id, "path": path, "files": [], "generated_at": utc_now_iso(), "errors": []}
    try:
        from huggingface_hub import list_repo_tree

        tree, err = safe_call(lambda: list_repo_tree(repo_id=repo_id, path=path, recursive=False))
        if err:
            snapshot["errors"].append(err)
            return snapshot

        for entry in tree:
            if hasattr(entry, "path"):
                snapshot["files"].append(
                    {
                        "path": entry.path,
                        "size": getattr(entry, "size", None),
                        "sha": getattr(entry, "oid", None) or getattr(entry, "sha", None),
                        "cdn_url": f"https://huggingface.co/datasets/{repo_id}/resolve/{entry.path}",
                    }
                )
    except Exception as e:
        snapshot["errors"].append(str(e))

    # Rate-limit bypass guidance (for training scripts)
    snapshot["guidance"] = (
        "Use CDN-only URLs and add exponential backoff + HF token in headers. "
        "Prefer datasets library streaming or local mirror for large snapshots."
    )
    return snapshot
