# workio / frontend

## Final Synthesis (Best Parts + Correctness + Actionability)

I merged the strongest, non-contradictory parts from both proposals and resolved conflicts in favor of **correctness, reliability, and concrete actionability**.

Key decisions:
- **Keep Candidate 1’s flight-dedupe + optimistic update layer** (it’s concrete, testable, and solves double-tap/duplicate requests).
- **Add Candidate 2’s lightweight realtime/event bus** to solve LINE webhook/silent state changes and cross-tab/manager↔employee sync.
- **Do not replace React Query with a custom event bus**; instead, make events update React Query caches so we keep caching, invalidation, and retry semantics.
- **Add connection/online status + clear UX states** so users see whether an action is pending/applied/failed.
- **Keep backend unchanged** (frontend-only changes).

---

## 1. Diagnosis (merged)

- No optimistic UI for clock-in/out → perceived latency and duplicate taps.
- Missing client-side idempotency for clock/leave/OT → rapid taps queue multiple requests.
- Stale dashboard after LINE/webhook or approval actions → UI doesn’t update until manual refresh.
- No visual feedback for GPS/geofence mismatch or network failures.
- Racey dashboard state across components and tabs.
- No centralized realtime/event bus for LINE-originated updates.

---

## 2. Proposed change (merged scope)

Files:
- `src/lib/events.ts` — typed EventEmitter + optional reconnect/WebSocket/SSE adapter (pluggable).
- `src/hooks/useWorkioMutations.ts` — flight-dedupe + optimistic updates integrated with React Query.
- `src/hooks/useOnlineStatus.ts` — connection/online status + last-seen sync indicator.
- Integrations in `src/pages/Dashboard.tsx`, `src/pages/ClockInOut.tsx`, `src/pages/LeaveRequest.tsx`.

Rules:
- Optimistic updates are applied **locally** and rolled back on error.
- All optimistic and server events update React Query caches (not manual component state) so caches remain the source of truth.
- Events from the server (via webhook→push or polling fallback) invalidate/optimistically patch caches so all tabs update.
- Flight dedupe prevents duplicate identical in-flight requests.

### 3.1 Event bus (`src/lib/events.ts`)

```ts
// Minimal, typed event bus. Replace transport adapter as needed (WS/SSE/polling).
type EventMap = {
  clock: { employeeId: string; action: 'in' | 'out'; ts: string; shift?: any };
  leave: { id: string; status: string; employeeId: string };
  ot: { id: string; status: string; employeeId: string };
  approval: { id: string; type: 'leave' | 'ot'; status: string };
  'connectivity:online': boolean;
};

type Listener<T> = (payload: T) => void;

class EventBus {
  private listeners: { [K in keyof EventMap]?: Set<Listener<T>> } = {};

  on<K extends keyof EventMap>(key: K, listener: Listener<EventMap[K]>) {
    if (!this.listeners[key]) this.listeners[key] = new Set();
    this.listeners[key]!.add(listener);
    return () => this.off(key, listener);
  }

  off<K extends keyof EventMap>(key: K, listener: Listener<EventMap[K]>) {
    this.listeners[key]?.delete(listener);
  }

  emit<K extends keyof EventMap>(key: K, payload: EventMap[K]) {
    this.listeners[key]?.forEach((l) => {
      try {
        l(payload);
      } catch (err) {
        console.error('EventBus listener error', key, err);
      }
    });
  }

  // Optional: helper to connect to a WebSocket/SSE endpoint and map server messages to events.
  connectWebSocket(url: string) {
    const ws = new WebSocket(url);
    ws.addEventListener('message', (ev) => {
      try {
        const msg = JSON.parse(ev.data);
        if (msg && msg.type && msg.payload) {
          this.emit(msg.type as keyof EventMap, msg.payload);
        }
      } catch (err) {
        // ignore malformed messages
        console.warn('Bad WS message', ev.data);
      }
    });
    ws.addEventListener('open', () => this.emit('connectivity:online', true));
    ws.addEventListener('close', () => this.emit('connectivity:online', false));
    return () => ws.close();
  }
}

export const eventBus = new EventBus();
```

### 3.2 Flight dedupe + optimistic mutations (`src/hooks/useWorkioMutations.ts`)

```ts
import { useCallback } from 'react';
import { useQueryClient } from '@tanstack/react-query';
import axios from 'axios';
import { eventBus } from '../lib/events';

type MutationKey = string | string[];

const inFlight = new Map<string, Promise<any>>();

function flightKey(key: MutationKey, payload: any): string {
  return JSON.stringify([key, payload]);
}

export function useWorkioMutation<T = any, V = any>(
  key: MutationKey,
  fn: (variables: V) => Promise<T>
) {
  const queryClient = useQueryClient();

  return useCallback(
    async (
      variables: V,
      options?: {
        optimisticData?: any;
        queryKeysToInvalidate?: string[];
        onSuccessEvent?: { type: keyof typeof eventBus['emit']; payload: any };
      }
    ) => {
      const fk = flightKey(key, variables);
      if (inFlight.has(fk)) {
        return inFlight.get(fk);
      }

      // optimistic update
      const previousMap = new Map<string, any>();
      if (options?.optimisticData && options?.queryKeysToInvalidate) {
        options.queryKeysToInvalidate.forEach((k) => {
          previousMap.set(k, queryClient.getQueryData([k]));
          queryClient.setQueryData([k], options.optimisticData);
        });
      }

      const rollback = () => {
        if (options?.optimisticData && options?.queryKeysToInvalidate) {
          options.queryKeysToInvalidate.forEach((k) => {
            queryClient.setQueryData([k], previousMap.get(k));
          });
        }
      };

      const promise = fn(variables)
        .then((res) => {
          if (options?.queryKeysToInvalidate) {
            options.queryKeysToInvalidate.forEach((k) => queryClient.invalidateQueries([k]));
          }
          if (options?.onSuccessEvent) {
            eventBus.emit(options.onSuccessEvent.type, options.onSuccessEvent.payload);
          }
          return res;
        })
        .catch((err) => {
          rollback();
          return Promise.reject(err);
        })
        .finally(() => {
          inFlight.delete(fk);
        });

      inFlight.set(fk, promise);
      return promise;
    },
    [key, fn, queryClient]
  );
}

// Preset mutations
export function useClockInOut() {
  return useWorkioMutation(
    ['clock'],
    (payload: { action: 'in' | 'out'; lat?: number; lng?: number }) =>
      axios.post('/api/clock', payload).then((r) => r.data)
  );
}

export function useLeaveRequest() {
  return useWorkioMutation(
    ['leave-request'],
    (payload: { type: string; start: string; end: string; reason: string }) =>
      axios.post('/api/leave', payload).then((r) => r.data)
  );
}

export function useApprovalAction() {
  return useWorkioMutation(
    ['approval'],
    (payload: { id: string; type: 'leave' | 'ot'; status: 'approved' | 'rejected' }) =>
      axios.post('/api/approval', payload).then((r) => r.data)
  );
}
```
