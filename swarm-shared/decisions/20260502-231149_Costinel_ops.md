# Costinel / ops

## Implementation Plan — Costinel `GET /api/v1/cost-anomaly/signal/top-hub`

**Scope & Constraints**
- Read-only endpoint (Sense + Signal only). No mutations, no side effects.
- Optional `?for_date=YYYY-MM-DD` (default today).
- Reuse existing knowledge-graph assets; avoid recomputing the graph on every request.
- Return compact JSON suitable for dashboard widgets and alerting pipelines.
- Must be deployable in <2h (single endpoint + minimal wiring).

---

### 1) High-level flow
1. **Resolve date** from query or default to UTC today.
2. **Load** precomputed knowledge-graph snapshot (or lightweight runtime summary) for that date.
3. **Select top hub** by highest connection weight / centrality proxy (e.g., degree × edge weight).
4. **Project** to signal payload:
   - hub identifier (e.g., `"MOC"`)
   - hub label / title
   - top 3 connected docs (id, title, snippet, score)
   - anomaly-relevant metadata (cost_entity, delta, severity)
   - provenance (graph_version, snapshot_ts)
5. **Return** `200` with signal payload; `404` if no graph for date; `400` on bad date.

### 3) Implementation (FastAPI example)

File: `/opt/axentx/Costinel/app/api/v1/endpoints/top_hub.py`

```python
from datetime import datetime, timezone, date as dtdate
from typing import Optional, List
from fastapi import APIRouter, HTTPException, Query
from pydantic import BaseModel

router = APIRouter()

# Local imports (adjust to your project structure)
from app.services.knowledge_graph import load_graph_snapshot_for_date, HubNotFoundError
from app.core.config import settings

# ---- Pydantic models ----
class HubConnection(BaseModel):
    doc_id: str
    title: str
    snippet: str
    score: float

class AnomalyContext(BaseModel):
    cost_entity: Optional[str] = None
    delta_pct: Optional[float] = None
    severity: Optional[str] = None

class Provenance(BaseModel):
    graph_version: str
    snapshot_ts: str

class TopHubSignal(BaseModel):
    hub_id: str
    hub_label: str
    date: str
    signal_ts: str
    top_connections: List[HubConnection]
    anomaly_context: AnomalyContext
    provenance: Provenance

# ---- Helper ----
def _utc_today_str() -> str:
    return datetime.now(timezone.utc).strftime("%Y-%m-%d")

# ---- Endpoint ----
@router.get("/api/v1/cost-anomaly/signal/top-hub", response_model=TopHubSignal)
async def get_top_hub_signal(
    for_date: Optional[str] = Query(
        None,
        regex=r"^\d{4}-\d{2}-\d{2}$",
        description="Date (UTC) to evaluate. Defaults to today."
    )
):
    target_date_str = for_date or _utc_today_str()
    try:
        target_date = dtdate.fromisoformat(target_date_str)
    except ValueError:
        raise HTTPException(status_code=400, detail="Invalid for_date format. Use YYYY-MM-DD.")

    # Load snapshot (cached or on-disk). This should be fast (<100ms).
    try:
        graph = load_graph_snapshot_for_date(target_date)
    except HubNotFoundError as exc:
        raise HTTPException(status_code=404, detail=str(exc))
    except FileNotFoundError:
        raise HTTPException(
            status_code=404,
            detail=f"No knowledge-graph snapshot found for date {target_date_str}"
        )

    # Select top hub (degree * avg_weight as simple proxy)
    top_hub = graph.top_hub(limit=1)
    if not top_hub:
        raise HTTPException(status_code=404, detail="No hub found in graph snapshot")

    # Top 3 connected docs
    connections = [
        HubConnection(
            doc_id=c["doc_id"],
            title=c.get("title", ""),
            snippet=c.get("snippet", "")[:200],
            score=float(c.get("score", 0.0))
        )
        for c in top_hub.connections[:3]
    ]

    # Anomaly context: attempt to extract from hub/node metadata
    anomaly_meta = top_hub.metadata.get("anomaly", {})
    anomaly_ctx = AnomalyContext(
        cost_entity=anomaly_meta.get("cost_entity"),
        delta_pct=anomaly_meta.get("delta_pct"),
        severity=anomaly_meta.get("severity")
    )

    signal_ts = datetime.now(timezone.utc).isoformat(timespec="seconds").replace("+00:00", "Z")

    payload = TopHubSignal(
        hub_id=top_hub.hub_id,
        hub_label=top_hub.label,
        date=target_date_str,
        signal_ts=signal_ts,
        top_connections=connections,
        anomaly_context=anomaly_ctx,
        provenance=Provenance(
            graph_version=graph.version,
            snapshot_ts=graph.snapshot_ts.isoformat(timespec="seconds").replace("+00:00", "Z")
        )
    )
    return payload
```
