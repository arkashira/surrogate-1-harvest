# Costinel / ops

## Final Implementation Plan — Top-hub Signal Panel (Costinel Dashboard)

**Scope**  
- Frontend-only, read-only panel.  
- Surface the most-connected hub and its actionable proposals.  
- Resilient to missing backend: graceful fallback UI + local sample data + localStorage cache.  
- No schema changes, no new backend endpoints, no DB migrations.  

**Timebox**: <2h  
**Stack**: React + TypeScript + Tailwind (existing patterns) + lightweight fetch + localStorage cache.

---

### 1) Combined, highest-value changes

Add a **Top-hub Signal Panel** to the Costinel dashboard that:

- Attempts to fetch `/api/knowledge/hubs/top` (if backend exists) with abort + retry behavior.  
- Persists last successful response to `localStorage` and uses it as a secondary cache.  
- Falls back to a bundled sample payload when API and cache are unavailable.  
- Renders a compact, accessible card:
  - Hub name, connection count, and one-line insight.  
  - Actionable proposals (title, brief, impact/effort badges, link).  
  - Keyboard-navigable links and ARIA labels.  
- UX polish:
  - Skeleton loader while fetching.  
  - Auto-refresh on interval (30s) with cancel on unmount.  
  - Clear “using cached/fallback data” badge when not live.  
- Error resilience:
  - Never block UI on API failure.  
  - Keep prior or fallback data in state and surface non-blocking status.

#### `src/api/knowledge.ts`

```ts
import axios from 'axios';

const API = axios.create({ baseURL: '/api' });
const CACHE_KEY = 'costinel:topHub:latest';

export interface Proposal {
  id: string;
  title: string;
  brief: string;
  description: string;
  impact: 'HIGH' | 'MEDIUM' | 'LOW';
  effort: 'HIGH' | 'MEDIUM' | 'LOW';
  link: string;
  tags?: string[];
}

export interface Hub {
  id: string;
  name: string;
  label?: string;
  connections: number;
  description?: string;
  insight?: string;
  tags?: string[];
}

export interface TopHubResponse {
  hub: Hub;
  proposals: Proposal[];
}

export async function fetchTopHub(signal?: AbortSignal): Promise<TopHubResponse> {
  const res = await API.get<TopHubResponse>('/knowledge/hubs/top', { signal });
  return res.data;
}

export function getCachedTopHub(): TopHubResponse | null {
  try {
    const raw = localStorage.getItem(CACHE_KEY);
    return raw ? JSON.parse(raw) : null;
  } catch {
    return null;
  }
}

export function setCachedTopHub(payload: TopHubResponse): void {
  try {
    localStorage.setItem(CACHE_KEY, JSON.stringify(payload));
  } catch {
    // ignore storage failures
  }
}
```

#### `src/hooks/useTopHub.ts`

```ts
import { useEffect, useState, useCallback } from 'react';
import {
  fetchTopHub,
  getCachedTopHub,
  setCachedTopHub,
  TopHubResponse,
} from '../api/knowledge';
import fallback from '../data/fallback/topHub.json';

const REFRESH_MS = 30_000;

export function useTopHub(poll = true) {
  const [data, setData] = useState<TopHubResponse>(fallback as TopHubResponse);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<Error | null>(null);
  const [source, setSource] = useState<'live' | 'cache' | 'fallback'>('fallback');

  const load = useCallback(async (signal?: AbortSignal) => {
    try {
      setLoading(true);
      const res = await fetchTopHub(signal);
      setData(res);
      setCachedTopHub(res);
      setSource('live');
      setError(null);
    } catch (err: any) {
      if (err.name === 'AbortError') return;
      // try cache
      const cached = getCachedTopHub();
      if (cached) {
        setData(cached);
        setSource('cache');
      } else {
        setData(fallback as TopHubResponse);
        setSource('fallback');
      }
      setError(err);
    } finally {
      setLoading(false);
    }
  }, []);

  useEffect(() => {
    const ac = new AbortController();
    load(ac.signal);

    let timer: NodeJS.Timeout;
    if (poll) {
      timer = setInterval(() => load(), REFRESH_MS);
    }

    return () => {
      ac.abort();
      if (timer) clearInterval(timer);
    };
  }, [load, poll]);

  return { data, loading, error, source, refetch: () => load() };
}
```
