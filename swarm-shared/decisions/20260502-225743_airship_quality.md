# airship / quality

## Final Synthesized Implementation (Best Parts + Corrected)

**Goal**: Ship a zero-runtime-dependency, CDN-safe `airship discover` orchestrator (Bash-first) that deterministically emits `status.json` + `manifest.json` for local/CI use and **eliminates HF API calls during training/inference**.

**Core Principles**
- **Deterministic & reproducible**: sorted manifests, stable ETag/sha256, no API calls at training time.
- **CDN-only at training**: `https://huggingface.co/datasets/{repo}/resolve/main/{path}` with optional checksum validation.
- **Bash-first, deps minimal**: `bash`, `curl`, `jq`. Optional Python helper for robust JSON/checksum handling (no install-time deps).
- **Correctness fixes**: use recursive tree API safely (date-scoped), handle auth gracefully, validate CDN reachability, and include sha256/etag in manifest for integrity.

---

## Implementation Plan (≤2h)

1. **Create CLI entrypoint** `bin/airship-discover` (executable, Bash shebang).
2. **Implement `discover` command**:
   - Required args: `--repo`, `--folder`, `--out-dir`.
   - Optional: `--date` to scope to a date folder (recommended).
   - Single recursive tree API call (date-scoped) with fallback/pagination guard; cache to `~/.airship/file-cache-<repo>-<folder>-<date>.json`.
   - Emit:
     - `status.json`: `{repo, folder, date?, ts, file_count, api_calls, status, cdnBase}`.
     - `manifest.json`: sorted list of `{path, size, url, etag?, sha256?}` for CDN fetch + validation.
3. **CDN URL builder**: `https://huggingface.co/datasets/{repo}/resolve/main/{path}`.
4. **Integrate with training script**:
   - Read `manifest.json` and stream files via `curl`/`wget` during data load (zero HF API calls).
   - Validate via etag or sha256 when available; retry on transient CDN failures.
5. **Add studio-reuse guard** (optional):
   - Before training, check for running Lightning studio with matching name; reuse if found.
6. **Smoke test**:
   - Run `bin/airship-discover` against a small repo/folder/date.
   - Verify `manifest.json` entries are fetchable via CDN (HEAD or range request).
   - Confirm training script can iterate manifest without HF API.

## Code Snippets

### `bin/airship-discover`

```bash
#!/usr/bin/env bash
set -euo pipefail

# airship-discover: CDN-safe repo/folder discovery for HF datasets
# Usage: airship-discover --repo <repo> --folder <folder> [--date <YYYY-MM-DD>] --out-dir <dir>
#
# Notes:
# - Uses HF Tree API (rate-limited). Run sparingly from dev/CI.
# - Training should use manifest.json and fetch via CDN (no API calls).
# - Public repos work without auth. For private repos, set HF_TOKEN.

DEFAULT_OUT_DIR="./airship-out"
CACHE_DIR="${HOME}/.airship"
HF_API_BASE="https://huggingface.co/api/datasets"

show_help() {
  cat <<EOF
Usage: $(basename "$0") --repo <repo> --folder <folder> [--date <YYYY-MM-DD>] --out-dir <dir>

Options:
  --repo     HuggingFace dataset repo (user/repo)
  --folder   Folder path inside repo (e.g. batches/mirror-merged)
  --date     Optional date folder to scope discovery (YYYY-MM-DD)
  --out-dir  Output directory for status.json and manifest.json

Outputs:
  status.json   - discovery metadata
  manifest.json - sorted CDN file list [{path,size,url,etag,sha256}]

Examples:
  $(basename "$0") --repo user/mydata --folder batches --date 2024-06-01 --out-dir ./out
EOF
  exit 1
}

REPO=""
FOLDER=""
DATE=""
OUT_DIR="$DEFAULT_OUT_DIR"

while [[ $# -gt 0 ]]; do
  case "$1" in
    --repo) REPO="$2"; shift 2 ;;
    --folder) FOLDER="$2"; shift 2 ;;
    --date) DATE="$2"; shift 2 ;;
    --out-dir) OUT_DIR="$2"; shift 2 ;;
    -h|--help) show_help ;;
    *) echo "Unknown option: $1"; show_help ;;
  esac
done

if [[ -z "$REPO" || -z "$FOLDER" ]]; then
  echo "Error: --repo and --folder are required."
  show_help
fi

mkdir -p "$OUT_DIR" "$CACHE_DIR"

# Deterministic cache key
CACHE_KEY="$(echo -n "${REPO}::${FOLDER}::${DATE}" | tr -cd '[:alnum:]-' | tr '[:upper:]' '[:lower:]')"
CACHE_FILE="${CACHE_DIR}/file-cache-${CACHE_KEY}.json"

TIMESTAMP="$(date -u +"%Y-%m-%dT%H:%M:%SZ")"
API_CALLS=0

echo "Discovering ${REPO}/${FOLDER} ${DATE:+(date=${DATE})} ..."

# Build target path
TARGET_PATH="${FOLDER%/}"
if [[ -n "$DATE" ]]; then
  TARGET_PATH="${FOLDER%/}/${DATE}"
fi

# Prepare auth header (optional)
AUTH_HEADER=()
if [[ -n "${HF_TOKEN:-}" ]]; then
  AUTH_HEADER=(-H "Authorization: Bearer ${HF_TOKEN}")
fi

# Fetch tree recursively for the scoped path.
# For large folders this may hit rate limits; prefer date-scoped runs.
# We use expand=false to reduce payload; we'll compute sha256 later if needed.
echo "Fetching tree (recursive) for ${TARGET_PATH} ..."
RESPONSE=$(curl -sSf "${AUTH_HEADER[@]}" \
  "${HF_API_BASE}/${REPO}/tree?path=${TARGET_PATH}&recursive=true&expand=false" \
  2>/dev/null || echo "")

if [[ -z "$RESPONSE" ]]; then
  if [[ -s "$CACHE_FILE" ]]; then
    echo "Warning: Tree API failed or empty; using cached file if available."
  else
    echo "Error: No cached tree data available and API call failed."
    exit 1
  fi
else
  echo "$RESPONSE" > "$CACHE_FILE"
  API_CALLS=$((API_CALLS + 1))
fi

if [[ ! -s "$CACHE_FILE" ]]; then
  echo "Error: No tree data available. Cannot proceed."
  exit 1
fi

# Build manifest: include only files (type="file") under the target path.
# Normalize paths and build CDN URLs. Include etag when available.
MANIFEST_FILE="${OUT_DIR}/manifest.json"
STATUS_FILE="${OUT_DIR}/status.json"

if ! command -v jq >/dev/null 2>&1; then
  echo "Error: jq is required but not installed."
  exit 1
fi

# Produce sorted manifest with url and etag (if present).
# Note: HF tree API may not provide etag; we include null placeholder.
jq -r '
  map(select(.type == "file"))
  | map({
      path: .path,
      size: (.size // 0),
      url: ("https://huggingface.co/datasets/'${REPO}'/resolve/main/" + .path),
      etag: (.etag // null)
    })
  | sort_by(.path)
' "$CACHE_FILE" > "$MANIFEST_FILE"

FILE_COUNT=$(jq 'length' "$MANIFEST_FILE")

# Status
jq -n \
  --arg repo "$REPO" \
  --arg folder "$FOLDER" \
  --arg date "$DATE" \
  --arg ts "$TIMESTAMP" \
  --argjson file_count "$FILE_COUNT" \
  --argjson api_calls "$API_CALLS" \
  --arg status "ok" \
  --arg cdnBase "https://huggingface.co/datasets/${REPO}/resolve/main" \
  '{
    repo: $repo,
    folder: $folder,
    date: (if $date == "" then null else $date end),
    ts: $ts,
    file_count: $file_count,
    api_calls: $api_calls,
