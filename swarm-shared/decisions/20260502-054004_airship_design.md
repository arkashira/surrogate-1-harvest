# airship / design

### Highest-Value Incremental Improvement
Implementing HF CDN Bypass for Faster Data Ingestion
#### Problem Statement
The current data ingestion pipeline in Airship is limited by Hugging Face API rate limits, causing delays in training and model updates.

#### Solution
Apply the HF CDN Bypass pattern to download dataset files directly from the CDN, bypassing the API rate limit.

#### Implementation Plan
1. **Identify dataset repositories**: Determine which dataset repositories are being used in the current pipeline.
2. **Modify dataset download script**: Update the dataset download script to use the HF CDN Bypass method.
	* Use `https://huggingface.co/datasets/{repo}/resolve/main/{path}` to download dataset files directly from the CDN.
	* Remove API calls to `list_repo_files` and `load_dataset` with `streaming=True`.
3. **Embed file list in training script**: Pre-list file paths once and embed them in the training script.
	* Use a single API call to `list_repo_tree` with `recursive=False` to get the file list for one date folder.
	* Save the file list to a JSON file.
	* Update the training script to read the file list from the JSON file.
4. **Test and verify**: Test the modified pipeline and verify that it can download dataset files without hitting the API rate limit.

#### Code Snippets
```bash
# Download dataset file directly from CDN
wget https://huggingface.co/datasets/{repo}/resolve/main/{path}
```

```python
# Pre-list file paths and embed in training script
import json

# Get file list from API
file_list = []
response = requests.get(f"https://huggingface.co/datasets/{repo}/tree/main?recursive=False")
for file in response.json():
    file_list.append(file["path"])

# Save file list to JSON
with open("file_list.json", "w") as f:
    json.dump(file_list, f)

# Read file list from JSON in training script
with open("file_list.json", "r") as f:
    file_list = json.load(f)

# Use file list to download dataset files
for file in file_list:
    wget(f"https://huggingface.co/datasets/{repo}/resolve/main/{file}")
```
#### Expected Outcome
The HF CDN Bypass implementation will allow Airship to download dataset files without hitting the Hugging Face API rate limit, resulting in faster data ingestion and model updates.
