# airship / design

## Implementation Plan: `airship discover` — Zero-config strategic discovery CLI

**Highest-value incremental improvement (<2h):**  
Ship a single Bash CLI (`airship discover`) that operationalizes past patterns (business research + knowledge-rag + top-hub insight) and surfaces strategic context for Arkship/Surrogate platform decisions.

**Why this first:**  
- Reuses existing `granite-business-research.sh` and `knowledge-rag` tooling  
- No new services, no infra changes — pure orchestration  
- Immediate value: one command that surfaces MOC-like hubs + market context before any design/dev work  
- Fits <2h: ~20 LoC Bash + light error handling + help text

---

### Design

- **Entrypoint:** `/opt/axentx/airship/bin/airship` (symlink or single script)  
- **Subcommand:** `discover`  
- **Behavior:**  
  1. Run `granite-business-research.sh` (if present) to refresh market/competitor signals  
  2. Run `knowledge-rag` query for top hub and related docs (e.g., MOC)  
  3. Print concise strategic context: top hub, key docs, market insights, recommended next actions  
  4. Exit 0 on success; non-zero on failure with actionable stderr

- **Zero-config:**  
  - Auto-detect project root via `.git` or `airship` marker  
  - Use `./scripts/` or `./tools/` for research/rags; fallback to PATH  
  - No env required; optional `AIRSHIP_PROJECT_ROOT` override

- **Safety:**  
  - All commands invoked via `bash` with `set -euo pipefail`  
  - Timeouts for long-running research (600s)  
  - Idempotent — safe to run repeatedly

### Code: `/opt/axentx/airship/bin/airship`

```bash
#!/usr/bin/env bash
# airship — Arkship + Surrogate strategic CLI
# Usage: airship discover

set -euo pipefail

readonly SCRIPT_NAME="airship"
readonly PROJECT_ROOT="${AIRSHIP_PROJECT_ROOT:-$(cd "$(dirname "$(readlink -f "${BASH_SOURCE[0]}")")/.." && pwd)}"
readonly SCRIPTS_DIR="${PROJECT_ROOT}/scripts"
readonly TOOLS_DIR="${PROJECT_ROOT}/tools"

# Logging helpers
log_info()  { echo "[INFO] $*" >&2; }
log_error() { echo "[ERROR] $*" >&2; }
die()       { log_error "$*"; exit 1; }

# Resolve helper script
resolve_script() {
  local name="$1"
  for d in "$SCRIPTS_DIR" "$TOOLS_DIR"; do
    if [[ -x "$d/$name" ]]; then
      echo "$d/$name"
      return 0
    fi
  done
  if command -v "$name" >/dev/null 2>&1; then
    echo "$name"
    return 0
  fi
  return 1
}

cmd_discover() {
  log_info "Starting strategic discovery for Arkship/Surrogate platform..."

  # 1) Business research (optional)
  local research_script
  if research_script=$(resolve_script "granite-business-research.sh"); then
    log_info "Running business research: $research_script"
    if ! timeout 600 bash "$research_script" "$@"; then
      log_error "Business research completed with warnings or failed — continuing..."
    fi
  else
    log_info "No granite-business-research.sh found — skipping business research."
  fi

  # 2) Knowledge RAG — top hub and related docs
  local rag_script
  if rag_script=$(resolve_script "knowledge-rag"); then
    log_info "Querying knowledge graph for top hub and related docs..."
    local rag_output
    if ! rag_output=$(timeout 300 bash "$rag_script" --query "top hub and related docs (e.g., MOC)" 2>&1); then
      log_error "knowledge-rag query failed: $rag_output"
      die "Unable to retrieve knowledge insights."
    fi
    echo
    echo "=== Strategic Context ==="
    echo "$rag_output"
  else
    log_error "knowledge-rag not found — cannot produce strategic context."
    die "Install or link knowledge-rag into scripts/ or tools/."
  fi

  # 3) Summary + recommendations
  echo
  echo "=== Recommendations ==="
  echo "- Review the top hub (e.g., MOC) before planning tasks (#knowledge-rag #hub)"
  echo "- Use insights to prioritize Arkship workflows and Surrogate training focus"
  echo "- Re-run after major research updates to refresh context"
  echo
  log_info "Discovery complete."
}

usage() {
  cat <<EOF
Usage: $SCRIPT_NAME <command>

Commands:
  discover   Run zero-config strategic discovery (business research + knowledge-rag + top-hub insight)

Environment:
  AIRSHIP_PROJECT_ROOT   Override project root (default: auto-detect)

Examples:
  $SCRIPT_NAME discover
EOF
}

main() {
  if [[ $# -lt 1 ]]; then
    usage
    exit 1
  fi

  local cmd="$1"
  shift

  case "$cmd" in
    discover) cmd_discover "$@" ;;
    --help|-h|help) usage ;;
    *) log_error "Unknown command: $cmd"; usage; exit 1 ;;
  esac
}

main "$@"
```
