# vanguard / design

## Final Synthesized Implementation Plan

**Core Problem:** Surrogate-1 training and ingestion hit HF API rate limits (429s) and lose determinism because they repeatedly enumerate repos instead of using content-addressed manifests and CDN-only fetches, while also writing mixed-schema parquet files.

**Single Source of Truth:** Use a **content-addressed manifest per date folder** and **CDN-first dataloader** to eliminate API calls during training. Enforce strict `{prompt,response}` schema projection before writing to `batches/mirror-merged/{date}/{slug}.parquet`. Use deterministic repo mapping to bypass 128 commits/hr/repo limit.

### 1. Manifest Generator (single API call per date)
**File:** `/opt/axentx/vanguard/surrogate1/manifest.py`

- **Action:** `list_repo_tree(recursive=False)` once per date folder.
- **Output:** `manifests/{date}.json` with `{path, sha256, size}` for each file.
- **SHA256 resolution:** Prefer LFS/file metadata; fallback to temp download only if missing (avoid in loops).
- **Idempotent:** Re-run safely; overwrites same date manifest.

```python
#!/usr/bin/env python3
import argparse, hashlib, json, os, sys
from typing import List, Dict
from huggingface_hub import HfApi

API = HfApi()

def list_date_files(repo: str, date_folder: str) -> List[Dict]:
    tree = API.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    out = []
    for entry in tree:
        if entry.type != "file": continue
        out.append({
            "path": entry.path,
            "size": getattr(entry, "size", None),
            "lfs": getattr(entry, "lfs", None),
        })
    return out

def resolve_sha256(repo: str, path: str) -> str:
    try:
        info = API.model_info(repo, path, files_metadata=True)
        if getattr(info, "sha256", None):
            return info.sha256
    except Exception:
        pass
    local = hf_hub_download(repo_id=repo, filename=path, repo_type="dataset")
    with open(local, "rb") as f:
        return hashlib.sha256(f.read()).hexdigest()

def build_manifest(repo: str, date_folder: str, out_path: str) -> None:
    entries = list_date_files(repo, date_folder)
    for e in entries:
        if e.get("lfs") or not e.get("sha256"):
            e["sha256"] = resolve_sha256(repo, e["path"])
    manifest = {"repo": repo, "date_folder": date_folder, "files": entries}
    os.makedirs(os.path.dirname(os.path.abspath(out_path)), exist_ok=True)
    with open(out_path, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest written to {out_path} ({len(entries)} files)")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date", required=True)
    parser.add_argument("--out", default="manifest.json")
    args = parser.parse_args()
    build_manifest(args.repo, args.date, args.out)
```

### 2. CDN-First Dataset Loader (zero API during training)
**File:** `/opt/axentx/vanguard/surrogate1/dataset.py`

- **Action:** Load manifest once; stream files via CDN URLs only.
- **No auth, no `load_dataset`, no `list_repo_files` during training.**
- **Decoupled decoding:** Yield raw bytes; caller projects schema.

```python
import json, requests
from typing import Dict, Iterator

CDN_ROOT = "https://huggingface.co/datasets"

class CdnDataset:
    def __init__(self, manifest_path: str, repo: str):
        self.repo = repo
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.files = [f["path"] for f in self.manifest["files"]]

    def _download_file(self, path: str) -> bytes:
        url = f"{CDN_ROOT}/{self.repo}/resolve/main/{path}"
        resp = requests.get(url, timeout=30)
        resp.raise_for_status()
        return resp.content

    def __iter__(self) -> Iterator[Dict]:
        for path in self.files:
            content = self._download_file(path)
            yield {"path": path, "content": content}
```

### 3. Deterministic Repo Mapping (bypass 128 commits/hr/repo)
**File:** `/opt/axentx/vanguard/surrogate1/repo_map.py`

- **Action:** Map slug deterministically to one of N sibling repos.
- **Use case:** Parallel writes without hitting per-repo commit limits.

```python
import hashlib

def repo_for_slug(slug: str, base: str = "surrogate1", siblings: int = 5) -> str:
    h = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    idx = h % siblings
    if idx == 0:
        return f"org/{base}"
    return f"org/{base}-{idx}"
```

### 4. Schema Projection (enforce Surrogate-1 pattern)
**File:** `/opt/axentx/vanguard/surrogate1/ingest.py`

- **Action:** Before upload, project to `{prompt, response}` only.
- **Output path:** `batches/mirror-merged/{date}/{slug}.parquet`.
- **No extra columns; no mixed schemas.**

```python
import pyarrow as pa
import pyarrow.parquet as pq
import os

def project_and_write(table: pa.Table, out_path: str) -> None:
    keep = [c for c in ("prompt", "response") if c in table.column_names]
    if not keep:
        raise ValueError("No required columns (prompt, response) found in table")
    projected = table.select(keep)
    os.makedirs(os.path.dirname(out_path), exist_ok=True)
    pq.write_table(projected, out_path)
```
