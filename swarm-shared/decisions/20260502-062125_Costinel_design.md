# Costinel / design

**Task:** Implement design improvements for Costinel's dashboard to enhance user experience and visibility.

**Highest-value incremental improvement:**

* Improve Cost Analytics & Visibility feature by adding a "Cost Breakdown by Resource" chart to provide a detailed view of costs by resource type (e.g., EC2 instances, RDS databases, etc.).

**Implementation Plan:**

1. **Gather data:** Modify the existing data ingestion pipeline to extract resource-level cost data from cloud providers (AWS, GCP, Azure).
2. **Design the chart:** Create a new chart component using a library like Chart.js or D3.js to display the cost breakdown by resource.
3. **Integrate with dashboard:** Add the new chart to the Cost Analytics & Visibility dashboard, ensuring it is responsive and visually appealing.
4. **Test and refine:** Validate the chart's functionality and appearance, making any necessary adjustments to ensure a seamless user experience.

**Code Snippet:**
```javascript
// cost-breakdown-chart.js
import Chart from 'chart.js';

const ctx = document.getElementById('cost-breakdown-chart').getContext('2d');
const chart = new Chart(ctx, {
  type: 'bar',
  data: {
    labels: ['EC2 Instances', 'RDS Databases', 'S3 Buckets'],
    datasets: [{
      label: 'Cost Breakdown',
      data: [100, 50, 200],
      backgroundColor: [
        'rgba(255, 99, 132, 0.2)',
        'rgba(54, 162, 235, 0.2)',
        'rgba(255, 206, 86, 0.2)',
      ],
      borderColor: [
        'rgba(255, 99, 132, 1)',
        'rgba(54, 162, 235, 1)',
        'rgba(255, 206, 86, 1)',
      ],
      borderWidth: 1
    }]
  },
  options: {
    title: {
      display: true,
      text: 'Cost Breakdown by Resource'
    },
    scales: {
      yAxes: [{
        ticks: {
          beginAtZero: true
        }
      }]
    }
  }
});
```
