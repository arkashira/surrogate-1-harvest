# Costinel / ops

## Final Production-Ready Implementation  
**Chosen approach**: CDN-first, non-blocking signal panel with graceful fallback. Combines Candidate 1’s robust delivery + Candidate 2’s cost-governance framing and query semantics into a single, actionable plan.

---

### 1) Scope (single highest-value deliverable)
Add a **non-blocking Top-Hub Signal Panel** to the Costinel dashboard that:
- Detects and surfaces the most-connected hub (default `MOC`) from a lightweight hub-graph snapshot produced by `knowledge-rag`.
- Uses a CDN-bypass pattern (HF CDN) for zero API-rate impact and high availability.
- Renders as a dismissible signal card with contextual cost-governance insights and quick-links.
- Falls back gracefully when graph data is unavailable; never blocks dashboard render.

---

### 2) Architecture (fits existing stack)
- **Data source**: `knowledge-rag` produces `hub-graph.json` → mirrored to CDN  
  `https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/{date}/hub-graph.json`
- **Ingestion** (Mac orchestration): cron job downloads latest snapshot to `/tmp/costinel-hub-graph/hub-graph-latest.json`.
- **Backend** (FastAPI): `/api/signals/top-hub` reads local cache, returns `{ hub, score, insights[], links[], source, cached_at }`.
- **Frontend** (React): `TopHubSignalPanel` fetches endpoint, renders dismissible card with skeleton/timeout handling.

### 3) Concrete Implementation

#### 3.1 Orchestration script — `scripts/orchestration/fetch-hub-graph.sh`
```bash
#!/usr/bin/env bash
# Purpose: Fetch latest hub-graph snapshot via CDN bypass (no HF API auth/rate-limit)
# Usage: bash fetch-hub-graph.sh [date-folder]
# Tags: #cdn #rate-limit-bypass #knowledge-rag #graph #hub

set -euo pipefail
SHELL=/bin/bash

REPO="axentx/costinel-knowledge"
DATE_FOLDER="${1:-$(date -u +%Y-%m-%d)}"
FILE_NAME="hub-graph.json"
CDN_BASE="https://huggingface.co/datasets/${REPO}/resolve/main"
URL="${CDN_BASE}/${DATE_FOLDER}/${FILE_NAME}"
CACHE_DIR="/tmp/costinel-hub-graph"
CACHE_FILE="${CACHE_DIR}/hub-graph-latest.json"

mkdir -p "${CACHE_DIR}"

echo "[$(date -u)] Fetching hub-graph via CDN: ${URL}"
if curl -fsSL --retry 3 --retry-delay 5 --max-time 30 "${URL}" -o "${CACHE_FILE}.tmp"; then
  mv "${CACHE_FILE}.tmp" "${CACHE_FILE}"
  echo "[$(date -u)] Saved to ${CACHE_FILE} ($(wc -c <"${CACHE_FILE}" 2>/dev/null || echo 0) bytes)"
else
  echo "[$(date -u)] WARN: CDN fetch failed, preserving existing cache if present"
  rm -f "${CACHE_FILE}.tmp"
  exit 0
fi
```
Make executable:
```bash
chmod +x scripts/orchestration/fetch-hub-graph.sh
```

#### 3.2 Backend endpoint — `backend/api/signals.py`
```python
from fastapi import APIRouter, HTTPException
from pathlib import Path
from typing import List, Optional
import json
import logging
from datetime import datetime, timezone

router = APIRouter()
logger = logging.getLogger("costinel.signals")

CACHE_FILE = Path("/tmp/costinel-hub-graph/hub-graph-latest.json")

class InsightItem(BaseModel):
    text: str
    severity: str = "info"  # info | warning | critical

class HubLink(BaseModel):
    label: str
    href: str
    icon: Optional[str] = None

class TopHubResponse(BaseModel):
    hub: str
    score: float
    insights: List[InsightItem]
    links: List[HubLink]
    source: str = "knowledge-rag"
    cached_at: Optional[str] = None

@router.get("/signals/top-hub", response_model=TopHubResponse, responses={404: {"description": "Hub graph not available"}})
async def get_top_hub():
    """
    Return the most-connected hub (e.g., MOC) and contextual cost-governance insights.
    Non-blocking: fast, cache-first, graceful fallback.
    Tags: #knowledge-rag #graph #hub
    """
    try:
        if not CACHE_FILE.is_file():
            raise HTTPException(status_code=404, detail="Hub graph not available")

        with open(CACHE_FILE, "r", encoding="utf-8") as f:
            graph = json.load(f)

        # Expected shape:
        # { "nodes": [...], "edges": [...], "hubs": [ { "id": "...", "score": ..., "insights": [...] } ] }
        hubs = graph.get("hubs", [])
        if not hubs:
            # Fallback: compute top by degree
            degree = {}
            for e in graph.get("edges", []):
                a, b = e.get("source"), e.get("target")
                if a:
                    degree[a] = degree.get(a, 0) + 1
                if b:
                    degree[b] = degree.get(b, 0) + 1
            if not degree:
                raise HTTPException(status_code=404, detail="No hubs or edges in graph")
            top_id = max(degree, key=degree.get)
            top_score = float(degree[top_id])
            insights = [{"text": f"Top hub by edge degree ({top_score} connections)", "severity": "info"}]
        else:
            top = max(hubs, key=lambda x: x.get("score", 0))
            top_id = top.get("id", "MOC")
            top_score = float(top.get("score", 0))
            insights_raw = top.get("insights", [])
            if isinstance(insights_raw, list):
                insights = [
                    {"text": str(i), "severity": "info"} if isinstance(i, str) else i
                    for i in insights_raw
                ]
            else:
                insights = [{"text": str(insights_raw), "severity": "info"}]

        links = [
            {"label": "Knowledge Graph", "href": "/knowledge-rag", "icon": "graph"},
            {"label": f"{top_id} Details", "href": f"/knowledge-rag/hubs/{top_id}", "icon": "info"},
        ]

        cached_at = None
        try:
            mtime = CACHE_FILE.stat().st_mtime
            cached_at = datetime.fromtimestamp(mtime, tz=timezone.utc).isoformat()
        except Exception:
            pass

        return TopHubResponse(
            hub=top_id,
            score=top_score,
            insights=insights,
            links=links,
            cached_at=cached_at,
        )
    except json.JSONDecodeError as exc:
        logger.warning("Invalid JSON in hub-graph cache: %s", exc)
        raise HTTPException(status_code=500, detail="Invalid hub-graph data") from exc
    except Exception as exc:
        logger.exception("Unexpected error in /signals/top-hub")
        raise HTTPException(status_code=500, detail="Internal error") from exc
```

Add import/router in `backend/main.py` (or app composition):
```python

```
