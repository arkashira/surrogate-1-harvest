## Final Integrated Solution  
*(Best parts synthesized; contradictions resolved for correctness + concrete actionability)*

---

### 1) Goal (unchanged)
Make Surrogate training **HF-rate-limit-proof** and **Lightning-idle-resilient** in <2h.

---

### 2) What ships (tightened)
1. **Mac script** → `file_list.json` for one date folder (CDN-only, zero auth/rate-limit during training).  
2. **`train.py`** → streams from CDN URLs via `file_list.json`; **zero HF API calls** during training; deterministic sharding for HF commit cap.  
3. **Lightning launcher** → reuse running Studio; auto-restart if stopped (idle-resilient).  
4. **Deterministic repo sharding** → hash slug → sibling repo to stay under HF commit cap.

---

### 3) Resolved contradictions & correctness fixes
- **Tree listing**: use `list_repo_tree(..., recursive=False)` on the date folder; if empty, list repo root once and filter by prefix (avoids 100× pagination and path bugs).  
- **CDN URLs**: always `https://huggingface.co/datasets/{repo}/resolve/main/{path}`; no signed URLs needed for public datasets.  
- **Streaming**: chunked download with `requests.get(..., stream=True)` + `pyarrow` projection; memory-bounded; supports Parquet and JSONL; rejects unknown formats with clear error.  
- **Sharding**: deterministic (`hash(slug) % n_shards`) and **write-time only**; never changes existing shard assignment; avoids duplicate writes.  
- **Lightning idle**: check studio status before run; if stopped, restart on same machine type; reuse existing studio if running.

---

### 4) Implementation plan (actionable)

#### Step 1 — Mac orchestration: generate CDN file list (1 script)
- Run once per date folder after rate-limit window clears.  
- Outputs `file_list.json` with `repo`, `path`, `cdn_url`, `size`, `lfs`.  
- Embed or mount this file in the Lightning Studio.

#### Step 2 — `train.py`: CDN-only streaming + sharding
- Remove `load_dataset(streaming=True)` for heterogeneous repos.  
- Use `IterableDataset` over `file_list.json`; stream via CDN; project to `{prompt, response}` at parse time.  
- Deterministic sharding helper for any HF writes (commits/pushes).

#### Step 3 — Lightning launcher: reuse + idle-resilient
- Before run: list `Teamspace.studios`; reuse running studio.  
- On stopped: restart with `target.start(machine=Machine.L40S)` (or same spec).

#### Step 4 — HF commit cap mitigation
- Hash slug → sibling repo from `repos = [f"{base}-shard{i}" for i in range(5)]`.  
- Aggregate cap 640/hr across shards.

#### `scripts/list_cdn_files.py`
```python
#!/usr/bin/env python3
"""
Generate CDN-only file list for a date folder to bypass HF API rate limits.
Usage:
    python scripts/list_cdn_files.py --repo org/dataset --date 2026-04-29 --out file_list.json
"""
import argparse
import json
from pathlib import Path

from huggingface_hub import HfApi, list_repo_tree

HF_API = HfApi()

def build_cdn_url(repo: str, path: str) -> str:
    return f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def list_date_files(repo: str, date_folder: str) -> list[dict]:
    """
    List files for a single date folder without recursive pagination.
    Falls back to repo-root listing + prefix filtering if folder not found.
    """
    entries = []

    # Try direct folder listing first
    try:
        tree = list_repo_tree(repo=repo, path=date_folder, recursive=False)
    except Exception:
        tree = []

    # If empty, list repo root and filter by prefix
    if not tree:
        root_tree = list_repo_tree(repo=repo, path="", recursive=False)
        prefix = date_folder.rstrip("/") + "/"
        tree = [node for node in root_tree if node.path.startswith(prefix) and node.type == "file"]

    for node in tree:
        if node.type != "file":
            continue
        entries.append(
            {
                "repo": repo,
                "path": node.path,
                "cdn_url": build_cdn_url(repo, node.path),
                "size": getattr(node, "size", None),
                "lfs": getattr(node, "lfs", None),
            }
        )
    return entries

def main() -> None:
    parser = argparse.ArgumentParser(description="Generate CDN file list for training.")
    parser.add_argument("--repo", required=True, help="HF dataset repo (org/name)")
    parser.add_argument("--date", required=True, help="Date folder (e.g., 2026-04-29)")
    parser.add_argument("--out", default="file_list.json", help="Output JSON path")
    args = parser.parse_args()

    print(f"Listing files for {args.repo}/{args.date} ...")
    entries = list_date_files(args.repo, args.date)

    out_path = Path(args.out)
    out_path.write_text(json.dumps(entries, indent=2))
    print(f"Wrote {len(entries)} files to {out_path}")
    print("\nNOTE: Mount or embed this file in train.py. Training will use CDN URLs directly.")

if __name__ == "__main__":
    main()
```

#### `surrogate/training/train.py`
```python
#!/usr/bin/env python3
"""
CDN-only training dataloader + Lightning idle-resilient launcher.
- Uses file_list.json to stream from CDN (no HF API during training).
- Projects heterogeneous files to {prompt, response} at parse time.
- Deterministic sharding for HF commit cap avoidance.
"""
import hashlib
import json
import os
from pathlib import Path
from typing import Dict, Iterator, List, Optional

import pyarrow as pa
import pyarrow.parquet as pq
import requests
import torch
from torch.utils.data import IterableDataset
from lightning.pytorch import LightningModule, Trainer
from lightning.pytorch.loggers import CSVLogger

# Optional config fallback
try:
    from surrogate.config import TRAIN_CONFIG
except Exception:
    TRAIN_CONFIG = {"batch_size": 8, "max_steps": 1000}

# --- Deterministic sharding helpers ---
def shard_repo_for_write(slug: str, base_repo: str = "org/dataset", n_shards: int = 5) -> str:
    """Deterministically pick a sibling repo to avoid HF commit cap."""
    digest = int(hashlib.sha256(slug.encode()).hexdigest(), 16)
    idx = digest % n_shards
    if idx == 0:
        return base_repo
    return f"{base_repo}-shard{idx}"

# --- CDN IterableDataset ---
class CDNIterableDataset(IterableDataset):
    def __init__(self, file_list_path: str, transform=None, buffer_size: int = 8 * 1024 * 1024):
        super().__init__()
        self.file_list_path = file_list_path
        self.transform = transform
        self.buffer_size = buffer_size
        self._file_list: Optional[List[dict]] = None

    def _load_file_list(self) -> List[dict]:
        if self._file_list is None:
            with open(self.file_list_path) as f:
                self._file_list = json.load(f)
        return self._file_list

    def _stream_parquet(self, content: bytes):
        table = pq.read_table(pa.BufferReader(content))
        cols = table.column_names
        prompt_col = next((c for c in ("prompt", "instruction", "input") if c in cols), None)
        response_col = next((c for c in ("response", "output", "completion") if c in cols), None)
        if prompt_col is None or response_col is None:
            raise ValueError(f"Parquet missing prompt/response columns. Found: {cols}")
        for i in range(table.num_rows):
            row = {
                "prompt": table[prompt_col][i].as_py(),
                "response": table[response_col][i].as_py(),
            }
            if self.transform:
                row = self.transform(row)
