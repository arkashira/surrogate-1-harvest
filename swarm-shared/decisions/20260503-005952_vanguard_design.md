# vanguard / design

## 1. Diagnosis
- No persisted `(repo, dateFolder)` manifest: every training run re-enumerates via authenticated HF API → quota burn + 429 risk.
- Recursive/heavy enumeration exposes mixed-schema files and wastes I/O during data discovery.
- Training script still relies on `load_dataset(streaming=True)` on heterogeneous repos → PyArrow `CastError` on schema drift.
- Lightning Studio recreation on each run wastes 80hr/mo quota; idle-stop kills training and loses progress.
- Mac/local runs attempt `model.from_pretrained()` or heavy ingestion — violates “Mac=CLI orchestration only” rule and risks OOM/thermal throttling.

## 2. Proposed change
Create `/opt/axentx/vanguard/manifest.py` + `/opt/axentx/vanguard/train.py` (minimal) + update orchestration script to:
- Build and persist a `manifest.json` for one `(repo, dateFolder)` via single `list_repo_tree(recursive=False)` call.
- Embed the manifest in training so Lightning workers fetch via HF CDN only (zero API calls during data load).
- Project to `{prompt, response}` at parse time (ignore mixed schema).
- Reuse running Lightning Studio; restart only if stopped.
- Keep Mac side as pure orchestrator (no model load/ingest compute).

## 3. Implementation

### manifest.py
```python
# /opt/axentx/vanguard/manifest.py
import json
import os
from datetime import datetime
from pathlib import Path
from huggingface_hub import HfApi

HF_REPO = os.getenv("HF_REPO", "datasets/your-org/surrogate-1")
DATE_FOLDER = os.getenv("DATE_FOLDER", datetime.utcnow().strftime("%Y-%m-%d"))
OUT_DIR = Path(__file__).parent

def build_manifest(repo: str, date_folder: str, out_path: Path):
    api = HfApi()
    # Non-recursive, single page to avoid pagination + quota burn
    tree = api.list_repo_tree(repo=repo, path=date_folder, recursive=False)
    files = [
        f.rfilename
        for f in tree
        if f.rfilename.endswith((".jsonl", ".parquet", ".json"))
    ]
    manifest = {
        "repo": repo,
        "date_folder": date_folder,
        "created_utc": datetime.utcnow().isoformat() + "Z",
        "files": sorted(files)
    }
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    return manifest

if __name__ == "__main__":
    manifest = build_manifest(HF_REPO, DATE_FOLDER, OUT_DIR / "manifest.json")
    print(json.dumps(manifest, indent=2))
```

### train.py (Lightning-compatible, CDN-only)
```python
# /opt/axentx/vanguard/train.py
import json
import os
from pathlib import Path
from typing import Dict, Any

import torch
from datasets import load_dataset
from torch.utils.data import DataLoader
from lightning import LightningModule, Trainer

MANIFEST_PATH = Path(__file__).parent / "manifest.json"
HF_REPO = os.getenv("HF_REPO", "datasets/your-org/surrogate-1")

def load_manifest(path: Path) -> Dict[str, Any]:
    return json.loads(path.read_text())

def cdn_urls(manifest):
    base = f"https://huggingface.co/datasets/{manifest['repo']}/resolve/main/{manifest['date_folder']}"
    return [f"{base}/{f}" for f in manifest["files"]]

def project_record(rec: Dict[str, Any]) -> Dict[str, str]:
    # Keep only {prompt, response}; tolerate extra/mixed columns
    return {
        "prompt": str(rec.get("prompt", rec.get("input", rec.get("text", "")))),
        "response": str(rec.get("response", rec.get("output", "")))
    }

class SurrogateData:
    def __init__(self, manifest_path: Path):
        manifest = load_manifest(manifest_path)
        urls = cdn_urls(manifest)
        # Load via CDN URLs — no authenticated HF API calls during training
        self.ds = load_dataset("json", data_files={"train": urls}, split="train")
        self.ds = self.ds.map(project_record, remove_columns=self.ds.column_names)

    def dataloader(self, batch_size: int = 8, num_workers: int = 4) -> DataLoader:
        return DataLoader(
            self.ds.with_format("torch"),
            batch_size=batch_size,
            num_workers=num_workers,
            shuffle=True
        )

class SurrogateModel(LightningModule):
    def __init__(self, lr: float = 1e-4):
        super().__init__()
        self.lr = lr
        # Lightweight placeholder; replace with your model
        self.net = torch.nn.Linear(1024, 1024)

    def training_step(self, batch, batch_idx):
        # Minimal example; adapt to your actual modeling logic
        x = torch.randn(batch["prompt"].shape[0], 1024, device=self.device)
        y = self.net(x)
        loss = torch.nn.functional.mse_loss(y, x)
        self.log("train_loss", loss)
        return loss

    def configure_optimizers(self):
        return torch.optim.AdamW(self.parameters(), lr=self.lr)

def main():
    data = SurrogateData(MANIFEST_PATH)
    model = SurrogateModel()

    # Reuse running studio if available; avoid recreation
    from lightning import Studio, Machine
    studio_name = "vanguard-l40s"
    running = None
    try:
        for s in Studio.list():
            if s.name == studio_name and s.status == "running":
                running = s
                break
    except Exception:
        running = None

    if running:
        studio = running
    else:
        studio = Studio.create(
            name=studio_name,
            machine=Machine.L40S,
            cloud="lightning-public-prod"
        )

    trainer = Trainer(
        max_epochs=1,
        accelerator="gpu",
        devices=1,
        logger=False,
        enable_checkpointing=False
    )
    trainer.fit(model, data.dataloader())

if __name__ == "__main__":
    main()
```

### Orchestration wrapper (Mac-safe)
```bash
#!/usr/bin/env bash
# /opt/axentx/vanguard/run_vanguard.sh
set -euo pipefail
export SHELL=/bin/bash

cd /opt/axentx/vanguard

# 1) Build manifest once (after rate-limit window clears if needed)
python manifest.py
```

## 4. Verification
1. Run manifest builder:
   ```bash
   cd /opt/axentx/vanguard && python manifest.py
   ```
   Confirm `manifest.json` exists and lists files for the date folder without recursive crawl.

2. Validate CDN URLs resolve without auth:
   ```bash
   head -n1 $(python -c "import json; m=json.load(open('manifest.json')); print(m['files'][0])") 2>/dev/null || \
   curl -sI "$(python -c "import json; m=json.load(open('manifest.json')); print('https://huggingface.co/datasets/'+m['repo']+'/resolve/main/'+m['date_folder']+'/'+m['files'][0])")" | head -5
   ```
   Expect HTTP 200 (no 401/403).

3. Dry-run training data load (no GPU):
   ```bash
   python -c "from train import SurrogateData; d=SurrogateData('manifest.json'); print(len(d.dataloader().dataset))"
   ```
   Should print record count without `CastError`.

4. Lightning Studio behavior:
   - If a running studio named `vanguard-l40s` exists, script reuses it.
   - If stopped, script recreates with L4
