# airship / ops

## Final Synthesized Implementation (Best Parts + Corrected Contradictions)

**Goal**: Harden `airship discover` into a deterministic, CDN-only, zero-HF-API-runtime orchestrator that eliminates HF API rate limits and PyArrow schema errors while producing reproducible file manifests and safe ingestion payloads.

**Why this wins**
- Removes runtime HF API calls during training/ingest → eliminates 429s and HF commit-cap contention.
- Avoids `load_dataset(streaming=True)` on heterogeneous repos → eliminates PyArrow `CastError`.
- Single Mac-side orchestration + Lightning CDN-only training → fits Mac=CLI rule and remote compute pattern.
- Produces `file-list.json` and `batches/mirror-merged/{date}/{slug}.parquet` artifacts that downstream training can consume with zero API cost.

## Concrete Implementation Plan (≤2h)

1. **Add `scripts/discover-cdn.sh`** (Bash, executable, proper shebang) to:
   - Accept `REPO`, `DATE`, optional `OUT_DIR`.
   - Use `list_repo_tree` once (non-recursive per folder) to build deterministic file list.
   - Persist `file-list.json` and a small metadata summary.
   - Emit `batches/mirror-merged/{DATE}/{slug}.parquet` with only `{prompt,response}` projection (no mixed schema cols).
   - Use HF CDN URLs (`resolve/main/...`) for actual file downloads (bypasses API auth limits).
   - Include retry/backoff for 429 with 360s wait; respect per-folder pagination limits.

2. **Add `scripts/run-discovery.sh` wrapper**:
   - Shebang `#!/usr/bin/env bash`, `set -euo pipefail`.
   - Validate env (`HF_TOKEN` optional for listing; downloads work without).
   - Invoke `discover-cdn.sh` and capture exit codes; on failure, log and exit non-zero (cron-friendly).

3. **Add lightweight Python helper `tools/project_to_parquet.py`**:
   - Input: local file paths or CDN URLs + JSONL/CSV/JSON content.
   - Output: parquet with `prompt`, `response`, `slug`, `filename`.
   - Drops any other columns to avoid schema mixing.

   - Add a one-off `discover` service to run in CI/local without touching Arkship/Surrogate runtime.

5. **Cron/cron.d entry pattern** (for future automation):

   - `0 2 * * * /opt/axentx/airship/scripts/run-discovery.sh >> /var/log/airship/discovery.log 2>&1`

6. **Smoke test**:
   - Run once locally against a known small repo.

## Code Snippets

### scripts/discover-cdn.sh
```bash
#!/usr/bin/env bash
# Deterministic CDN-only discovery for HF repos.
# Usage: ./discover-cdn.sh <repo> <date> [out_dir]
# Example: ./discover-cdn.sh bigcode/the-stack 2024-01-15 ./out

set -euo pipefail

REPO="${1:?Missing repo (user/repo)}"
DATE="${2:?Missing date (YYYY-MM-DD)}"
OUT_DIR="${3:-./out}"
HF_API="${HF_API:-https://huggingface.co/api}"
REPO_API="${HF_API}/datasets/${REPO}"
CDN_BASE="https://huggingface.co/datasets/${REPO}/resolve/main"

mkdir -p "${OUT_DIR}/batches/mirror-merged/${DATE}"
MANIFEST="${OUT_DIR}/file-list.json"
PARQUET_OUT="${OUT_DIR}/batches/mirror-merged/${DATE}/mirror-merged.parquet"
TMP_DIR=$(mktemp -d)
trap 'rm -rf "$TMP_DIR"' EXIT

log() { echo "[$(date -Iseconds)] $*"; }

# One-time tree listing (non-recursive per top-level folder to avoid huge pagination).
list_folder() {
  local path="$1"
  local url="${REPO_API}/tree?path=${path}&recursive=false"
  curl -sS -H "Authorization: Bearer ${HF_TOKEN:-}" "$url" || true
}

# Build deterministic file list (JSON array of objects with path, size, type).
log "Listing repo tree (non-recursive) for ${REPO}..."
ROOT_JSON=$(list_folder "")
if [ -z "${ROOT_JSON}" ] || [ "${ROOT_JSON}" = "null" ]; then
  log "WARNING: Empty tree or no access; falling back to CDN path guess for known patterns."
  echo '[]' > "${MANIFEST}"
else
  echo "${ROOT_JSON}" | jq -c 'sort_by(.path)' > "${MANIFEST}"
fi

# Produce deterministic slug for this run
SLUG=$(echo -n "${REPO}/${DATE}" | tr '/' '-' | tr -cd '[:alnum:]._-')
log "Run slug: ${SLUG}"

# Download helper using CDN (no auth required).
download_cdn() {
  local src_path="$1"
  local dst="$2"
  local url="${CDN_BASE}/${src_path}"
  curl -sS -f -L --retry 3 --retry-delay 5 "$url" -o "$dst" || {
    log "ERROR: CDN download failed for ${src_path}"
    return 1
  }
}

# Lightweight projection to {prompt,response}.
project_to_pair() {
  local src="$1"
  local filename="$2"
  python3 - "$src" "$filename" <<'PY'
import sys, json, csv, os
from typing import Any, Dict

src, filename = sys.argv[1], sys.argv[2]
slug = os.path.splitext(os.path.basename(filename))[0]

def try_jsonl(path):
    rows = []
    with open(path, "r", encoding="utf-8", errors="ignore") as f:
        for line in f:
            line = line.strip()
            if not line:
                continue
            try:
                obj = json.loads(line)
                rows.append(obj)
            except Exception:
                continue
    return rows

def try_csv(path):
    rows = []
    with open(path, "r", encoding="utf-8", errors="ignore") as f:
        reader = csv.DictReader(f)
        for row in reader:
            rows.append(row)
    return rows

rows = []
if src.endswith(".jsonl"):
    rows = try_jsonl(src)
elif src.endswith(".csv"):
    rows = try_csv(src)
elif src.endswith(".json"):
    with open(src, "r", encoding="utf-8", errors="ignore") as f:
        data = json.load(f)
        rows = data if isinstance(data, list) else [data]

out_rows = []
for r in rows:
    if isinstance(r, dict):
        prompt = r.get("prompt") or r.get("input") or r.get("question") or ""
        response = r.get("response") or r.get("output") or r.get("answer") or ""
        if isinstance(prompt, str) and isinstance(response, str):
            out_rows.append({
                "prompt": prompt,
                "response": response,
                "slug": slug,
                "filename": filename
            })
for o in out_rows:
    print(json.dumps(o, ensure_ascii=False))
PY
}

# Collect projected rows into NDJSON for parquet conversion.
PROJECTED_NDJSON="${TMP_DIR}/projected.ndjson"
> "${PROJECTED_NDJSON}"

SAMPLE_LIMIT=100
count=0
if [ -s "${MANIFEST}" ]; then
  paths=$(jq -r '.[].path // empty' "${MANIFEST}" | sort | head -n "${SAMPLE_LIMIT}")
else
  paths="data/train.jsonl data/train.csv README.md"
fi

for p in ${paths}; do
  count=$((count + 1))
  tmpf="${TMP_DIR}/file_${count}"
  if download_cdn "$p" "$tmpf"; then
    project_to_pair "$tmpf" "$p" >> "${PROJECTED_NDJSON}"
  else
    log "Skipping ${p} (download failed)"
  fi
done
