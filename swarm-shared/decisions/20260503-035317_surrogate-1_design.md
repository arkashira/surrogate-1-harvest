# surrogate-1 / design

## Final Implementation Plan (≤2h)

Replace `bin/dataset-enrich.sh` with a **manifest-driven, CDN-bypass ingestion worker** (`bin/dataset-enrich.py`) that:

- Accepts `SHARD_ID`, `SHARD_TOTAL=16` (matrix) and optional `DATE_FOLDER` (defaults to today `YYYY-MM-DD`).
- Uses a **single pre-listed manifest** (`batches/public-merged/<DATE_FOLDER>/manifest.json`) produced by a lightweight Mac-side script (or the first successful shard) that calls `list_repo_tree` once per date folder and saves the CDN-resolvable file list. This avoids repeated `list_repo_files` pagination and API rate limits during ingestion.
- Each shard deterministically hashes `slug` (or filename) → `bucket = hash(slug) % SHARD_TOTAL`; processes only its slice.
- Downloads via **CDN bypass** (`https://huggingface.co/datasets/axentx/surrogate-1-training-pairs/resolve/main/<path>`) with streaming + `pyarrow` projection to `{prompt, response}` only; no `streaming=True` on heterogeneous repos.
- Deduplicates via the existing `lib/dedup.py` central md5 store (SQLite on HF Space) — unchanged.
- Writes output as `batches/public-merged/<DATE_FOLDER>/shard<N>-<HHMMSS>.jsonl` (same convention) and **optionally** appends to the same-date manifest with the produced shard file entry for downstream training scripts to consume.
- Exits with clear status codes; logs summary (processed, duped, uploaded, bytes).

### Why this is the highest-value incremental improvement
- Eliminates repeated API list calls during every shard run (reduces 429 risk).
- CDN bypass removes auth/rate-limit pressure during heavy data streaming.
- Manifest makes file list deterministic and reusable across training runs (fits “pre-list once, embed in train.py” pattern).
- Keeps architecture Mac-for-orchestration, remote-for-compute; no model loading on Mac.
- Fits <2h: small focused Python script + tiny manifest generator + workflow tweak.

def deterministic_shard(slug: str) -> int:
    h = int(hashlib.sha256(slug.encode("utf-8")).hexdigest(), 16)
    return h % SHARD_TOTAL

def load_manifest() -> list:
    if not MANIFEST_PATH.exists():
        return []
    try:
        with open(MANIFEST_PATH, "r", encoding="utf-8") as f:
            data = json.load(f)
            if isinstance(data, list):
                return data
            return []
    except Exception:
        return []

def download_parquet_to_rows(url: str):
    """Stream parquet via CDN and yield {prompt, response} rows."""
    import io
    resp = requests.get(url, headers=HEADERS, stream=True, timeout=60)
    resp.raise_for_status()
    buf = io.BytesIO()
    for chunk in resp.iter_content(chunk_size=CHUNK_SIZE):
        buf.write(chunk)
    buf.seek(0)

    table = pq.read_table(buf)
    # Project only needed columns; tolerate missing cols by filling nulls
    cols = set(table.column_names)
    prompt_col = "prompt" if "prompt" in cols else None
    response_col = "response" if "response" in cols else None

    # If schema differs, try common aliases
    if not prompt_col:
        for c in ("input", "question", "instruction"):
            if c in cols:
                prompt_col = c
                break
    if not response_col:
        for c in ("output", "answer", "completion"):
            if c in cols:
                response_col = c
                break

    # Build rows
    n = table.num_rows
    prompts = table[prompt_col] if prompt_col and prompt_col in cols else pa.array([None] * n)
    responses = table[response_col] if response_col and response_col in cols else pa.array([None] * n)

    for i in range(n):
        prompt = prompts[i].as_py() if not pa.types.is_null(prompts.type) else None
        response = responses[i].as_py() if not pa.types.is_null(responses.type) else None
        # Normalize to string
        prompt = "" if prompt is None else str(prompt).strip()
        response = "" if response is None else str(response).strip()
        yield {"prompt": prompt, "response": response}

def download_jsonl_to_rows(url: str):
    resp = requests.get(url, headers=HEADERS, stream=True, timeout=60)
    resp.raise_for_status()
    for line in resp.iter_lines(decode_unicode=True):
        if not line:
            continue
        try:
            obj = json.loads(line)
            prompt = obj.get("prompt") or obj.get("input") or obj.get("instruction") or ""
            response = obj.get("response") or obj.get("output") or obj.get("answer") or obj.get("completion") or ""
            yield {"prompt": str(prompt).strip(), "response": str(response).strip()}
        except Exception:
            continue

def process_file(rel_path: str):
    url = HF_CDN_TEMPLATE.format(path=rel_path)
    ext = Path(rel_path).suffix.lower()
    if ext == ".parquet":
        yield from download_parquet_to_rows(url)
    elif ext == ".jsonl":
        yield from download_jsonl_to_rows(url)
    else:
        # skip unsupported
        return
