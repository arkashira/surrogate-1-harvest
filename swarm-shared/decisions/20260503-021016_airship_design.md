## Final Synthesized Implementation

**Goal**: Eliminate HF API 429 errors and Lightning quota waste during Surrogate training by implementing **CDN-first deterministic ingestion + Lightning Studio reuse with idle-stop guard + Arkship resilience**.

**Why this ships fastest**:
- Removes the primary training blocker (HF 429) with CDN bypass pattern already proven
- Saves ~80hr/mo Lightning quota via Studio reuse pattern
- Prevents lost training from idle timeouts (20min safeguard)
- All changes are additive/config-only — no breaking refactors

### 1. CDN-first file list generation (20 min)
Create `scripts/generate-cdn-filelist.py` that:
- Uses **single** `list_repo_tree` per date folder (respects HF rate limits)
- Outputs `training/filelist-{date}.json` containing only CDN URLs
- Embeds this list into `train.py` at runtime (zero API calls during training)

### 2. Lightning Studio reuse + idle guard (25 min)

- Add `find_running_studio(name)` that reuses existing Running studio
- Add `ensure_not_idle(studio)` that pings status and restarts with `Machine.L40S` if stopped
- Set `SHELL=/bin/bash` in any crontab entries

### 3. Arkship idle-stop resilience (5 min)
Add `arkship/docker-compose.override.yml` with healthcheck + restart policy for API containers to survive idle-stop.

### 4. Integration test (40 min)
- Run `python scripts/generate-cdn-filelist.py` → verify JSON
- Launch training via `bash surrogate/training/launch.sh` → verify Studio reuse
- Confirm zero HF API calls during data loading (check logs)

## Code Snippets

### `scripts/generate-cdn-filelist.py`
```python
#!/usr/bin/env python3
"""
Generate CDN-only file list for Surrogate training.
Uses single HF API call per folder → embeds CDN URLs.
"""
import json
import os
from datetime import datetime
from huggingface_hub import HfApi

HF_REPO = "axentx/surrogate-datasets"
OUTPUT_DIR = "surrogate/training/filelists"
os.makedirs(OUTPUT_DIR, exist_ok=True)

def list_date_folder(date_str: str):
    """List one date folder (non-recursive) and return CDN URLs."""
    api = HfApi()
    try:
        files = api.list_repo_tree(
            repo_id=HF_REPO,
            path=date_str,
            repo_type="dataset",
            recursive=False
        )
    except Exception as e:
        print(f"HF API error for {date_str}: {e}")
        return []
    
    cdn_urls = []
    for f in files:
        if f.type == "file" and f.path.endswith((".parquet", ".jsonl")):
            cdn_url = f"https://huggingface.co/datasets/{HF_REPO}/resolve/main/{f.path}"
            cdn_urls.append({
                "cdn_url": cdn_url,
                "path": f.path,
                "size": getattr(f, "size", None)
            })
    return cdn_urls

def main():
    # Use yesterday's folder by default, or pass via env
    date_str = os.getenv("TRAIN_DATE", datetime.now().strftime("%Y-%m-%d"))
    print(f"Listing {date_str}...")
    
    urls = list_date_folder(date_str)
    out_path = os.path.join(OUTPUT_DIR, f"filelist-{date_str}.json")
    with open(out_path, "w") as f:
        json.dump(urls, f, indent=2)
    
    print(f"Wrote {len(urls)} CDN URLs to {out_path}")
    print("Training will use these URLs directly (zero HF API calls during load).")

if __name__ == "__main__":
    main()
```

### `surrogate/training/launch.py`
```python
#!/usr/bin/env bash
# Wrapper script for Surrogate training with Lightning Studio reuse
# Ensures proper Bash execution and idle-stop resilience

set -euo pipefail
SHELL=/bin/bash

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
TRAIN_SCRIPT="${SCRIPT_DIR}/train.py"
STUDIO_NAME="surrogate-l40s-training"

# Lightning SDK reuse + idle guard
python3 - <<PY
import os
from lightning_sdk import Teamspace, Studio, Machine

def find_running_studio(name: str):
    for s in Teamspace.studios:
        if s.name == name and s.status == "Running":
            return s
    return None

def ensure_studio(name: str):
    studio = find_running_studio(name)
    if studio:
        print(f"Reusing running studio: {studio.name} (id={studio.id})")
        return studio
    
    print(f"Starting new studio: {name}")
    return Studio(
        name=name,
        machine=Machine.L40S,
        create_ok=True
    )

def ensure_not_idle(studio):
    if studio.status != "Running":
        print(f"Studio idle (status={studio.status}), restarting...")
        studio.start(machine=Machine.L40S)
        return True
    return False

if __name__ == "__main__":
    studio = ensure_studio("${STUDIO_NAME}")
    ensure_not_idle(studio)
    
    # Export studio env for training script
    print(f"STUDIO_ID={studio.id}")
    print(f"STUDIO_STATUS={studio.status}")
PY

# Run training with CDN filelist
export HF_DATASETS_OFFLINE=1  # Force CDN-only
export TRAIN_DATE="${TRAIN_DATE:-$(date +%Y-%m-%d)}"

echo "Launching Surrogate training (date=${TRAIN_DATE})..."
python3 "${TRAIN_SCRIPT}" \
    --filelist "surrogate/training/filelists/filelist-${TRAIN_DATE}.json" \
    --studio-name "${STUDIO_NAME}"
```

### `arkship/docker-compose.override.yml`
```yaml
# Prevents idle-stop from killing Arkship API during long training runs
version: '3.8'

services:
  arkship-api:
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:8000/health"]
      interval: 30s
      timeout: 10s
      retries: 3
      start_period: 40s
    deploy:
      resources:
        limits:
          memory: 1G
```
