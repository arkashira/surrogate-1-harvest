## Implementation Plan (≤2h)

**Highest-value incremental improvement**: Add deterministic pre-flight file listing + CDN-only ingestion path to eliminate HF API rate limits during training and make shard workers resilient to 429s.

### What we’ll change
- Add `bin/list-files.py` (run once from Mac) → produces `file-list-<date>.json` containing only file paths for one date folder (non-recursive per folder, then flatten).
- Update `bin/dataset-enrich.sh` to accept an optional file-list JSON; if present, workers read paths from the list and download via raw CDN URLs (no `datasets.load_dataset`, no HF API calls during streaming).
- Add lightweight CDN download helper in Python (requests, streaming) that projects to `{prompt, response}` on the fly and outputs NDJSON for the shard.
- Keep existing `lib/dedup.py` unchanged (central md5 store still used for cross-run dedup).
- Add retry/backoff for 429 with 360s sleep (per pattern) and jitter.

### Why this is highest value
- Eliminates HF API rate-limit failures during ingestion and training.
- Makes shard workers independent of `datasets` library schema issues (mixed parquet/JSON).
- Fits within 2h: small, focused, no infra changes.

### 1) `bin/list-files.py`
```python
#!/usr/bin/env python3
"""
Run from Mac (or any machine with HF token) once per date folder.
Produces file-list-<date>.json with CDN-ready paths.

Usage:
  HF_TOKEN=hf_xxx python bin/list-files.py \
    --repo axentx/surrogate-1-training-pairs \
    --date 2026-04-29 \
    --out file-list-2026-04-29.json
"""

import argparse
import json
import os
import time
from huggingface_hub import HfApi, Repository

HF_API_RATE_LIMIT_RETRY = 360  # seconds per pattern

def list_date_files(repo_id: str, date: str, out_path: str, token: str):
    api = HfApi(token=token)
    # List non-recursive in the date folder (per pattern)
    try:
        items = api.list_repo_tree(
            repo_id=repo_id,
            path=date,
            recursive=False,
            repo_type="dataset",
        )
    except Exception as e:
        if "429" in str(e):
            print(f"Rate limited. Sleeping {HF_API_RATE_LIMIT_RETRY}s")
            time.sleep(HF_API_RATE_LIMIT_RETRY)
            return list_date_files(repo_id, date, out_path, token)
        raise

    # items may be dicts or objects; normalize to path strings
    paths = []
    for item in items:
        name = item.get("path") or getattr(item, "path", None)
        if name:
            paths.append(f"{date}/{name}")

    payload = {
        "repo_id": repo_id,
        "date": date,
        "paths": sorted(paths),
        "cdn_prefix": f"https://huggingface.co/datasets/{repo_id}/resolve/main",
    }

    with open(out_path, "w") as f:
        json.dump(payload, f, indent=2)
    print(f"Wrote {len(paths)} paths to {out_path}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True)
    parser.add_argument("--date", required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    token = os.environ.get("HF_TOKEN")
    if not token:
        raise RuntimeError("HF_TOKEN required")

    list_date_files(args.repo, args.date, args.out, token)
```

Make executable:
```bash
chmod +x bin/list-files.py
```

### 2) `bin/lib/cdn_download.py` (new helper)
```python
import json
import os
import time
import requests
from typing import Iterator, Tuple

CDN_TIMEOUT = 60
MAX_RETRIES = 5
RETRY_BACKOFF = [1, 2, 4, 8, 16]
HF_API_RATE_LIMIT_RETRY = 360

def cdn_url(repo_id: str, path: str) -> str:
    return f"https://huggingface.co/datasets/{repo_id}/resolve/main/{path}"

def stream_cdn_lines(repo_id: str, path: str, token: str = None) -> Iterator[bytes]:
    url = cdn_url(repo_id, path)
    headers = {}
    if token:
        headers["Authorization"] = f"Bearer {token}"

    for attempt in range(MAX_RETRIES):
        try:
            with requests.get(url, headers=headers, stream=True, timeout=CDN_TIMEOUT) as r:
                if r.status_code == 429:
                    print(f"CDN 429 for {path}. Sleeping {HF_API_RATE_LIMIT_RETRY}s")
                    time.sleep(HF_API_RATE_LIMIT_RETRY)
                    continue
                r.raise_for_status()
                for chunk in r.iter_content(chunk_size=8192):
                    if chunk:
                        yield chunk
            return
        except (requests.RequestException, requests.Timeout) as e:
            if attempt == MAX_RETRIES - 1:
                raise
            wait = RETRY_BACKOFF[attempt]
            print(f"Retry {attempt+1}/{MAX_RETRIES} for {path} in {wait}s: {e}")
            time.sleep(wait)

def project_to_pair(raw_bytes: bytes, path: str) -> Iterator[dict]:
    """
    Lightweight projection to {prompt, response}.
    Supports newline-delimited JSON (.jsonl) and simple parquet via pyarrow if available.
    Keep this minimal — heavy schema logic belongs elsewhere.
    """
    # Try JSONL first (fast path)
    try:
        for line in raw_bytes.splitlines():
            if not line.strip():
                continue
            obj = json.loads(line)
            prompt = obj.get("prompt") or obj.get("input") or obj.get("text") or ""
            response = obj.get("response") or obj.get("output") or obj.get("completion") or ""
            if prompt or response:
                yield {"prompt": prompt, "response": response}
        return
    except Exception:
        pass

    # Fallback: attempt pyarrow parquet in-memory (optional)
    try:
        import pyarrow.parquet as pq
        import io
        table = pq.read_table(io.BytesIO(raw_bytes))
        df = table.to_pandas()
        for _, row in df.iterrows():
            prompt = row.get("prompt") or row.get("input") or row.get("text") or ""
            response = row.get("response") or row.get("output") or row.get("completion") or ""
            if prompt or response:
                yield {"prompt": prompt, "response": response}
        return
    except Exception:
        pass

    # If nothing works, yield empty pair (will be filtered upstream)
    yield {"prompt": "", "response": ""}
```
