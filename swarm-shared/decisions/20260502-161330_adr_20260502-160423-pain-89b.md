## v0.1 scope
v0.1 ships a cross-platform CLI client (Linux x86_64, Windows, macOS) that creates a TUN interface, tunnels traffic through a shadowsocks-like obfuscated SOCKS5 proxy, hides local port/TUN artifacts from basic DPI probes, and enforces a fail-closed kill-switch; includes encrypted local SQLite profiles, DoH fallback DNS, and minimal session/telemetry logging for WAU-ZB — no server component, no GUI, no account system.

## Open questions
- Which obfuscation profiles (e.g., shadowsocks 2022-ietf AEAD, v2ray XTLS/utls mimic) provide best DPI evasion vs. performance on low-end devices?
- How will we distribute/rotate proxy endpoints and PSKs without centralized infrastructure while preserving deniability?
- What local port/TUN naming and firewall rules minimize fingerprintability across Windows/macOS/Linux?
- How do we implement a reliable kill-switch across platforms that blocks real traffic if stealth path fails (e.g., firewall rules vs. routes)?
- What minimal client-side telemetry schema balances WAU-ZB measurement with privacy and deniability?
