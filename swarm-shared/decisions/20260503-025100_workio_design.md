# workio / design

## Final Synthesis — Correct, Actionable Solution

**Core problem**: duplicate punches caused by (1) LINE webhook retries without idempotency and (2) frontend retries/races without client-side guards.  
**Resolution priority**: server-side idempotency is non-negotiable; client-side guards are required for UX and defense-in-depth. Clock skew and location-acquisition UX must be fixed to make behavior deterministic and trustworthy.

---

## 1. Server-side (required)

**File**: `workio/server/src/routes/punch.ts` (and webhook route)

- Enforce **idempotency** for punch creation using a unique constraint/index on `idempotencyKey` (or `userId, action, date, idempotencyKey`) with a short TTL (24–48h) in Redis or DB.
- Use **atomic upsert / unique constraint** to prevent race conditions between “find latest active punch” and “create new punch” under concurrent deliveries.
- On duplicate `idempotencyKey`, return **200 OK + existing punch record** (do not create new row).
- Add **compensating reconcile endpoint** `POST /api/punch/reconcile` (admin/internal) to detect and merge/void duplicates (idempotent).
- Add **background job** (cron) to detect same-user same-day duplicate punches (by time window) and reconcile them automatically (log + alert).
- **Webhook handler** (`/webhook/line`) must:
  - validate signature,
  - parse `idempotencyKey` from payload (or generate stable one from LINE event ID + userId + action),
  - apply same idempotency enforcement as `/api/punch`.

---

## 2. Client-side (required)

**File**: `workio/src/hooks/usePunch.ts` (create)  
**File**: `workio/src/components/PunchClockButton.tsx` (update)

- Generate and persist **stable idempotencyKey per intent**: `punch:{action}:{YYYY-MM-DD}` stored in `sessionStorage` (or memory + survive soft-navigations).
- **In-flight request dedupe**: per-key in-memory guard (`Set`) to prevent parallel submissions from double-taps.
- **UI lock**: disable button while `isSubmitting || isAcquiringLocation`.
- **Server time**: fetch `/api/time` once per session to compute offset; use offset for timestamps. Fallback to server echo on submit if endpoint fails.
- **Location acquisition state**: expose `isAcquiringLocation`; show clear UX (“กำลังระบุตำแหน่ง…”) and disable button during acquisition.
- **Graceful location failure**: proceed without location if denied/timed out; do not block punch.
- **Optimistic UI**: optional optimistic update **only after** local validation and idempotency key reservation; revert on server error.

**Implementation** (final consolidated):

```ts
// workio/src/hooks/usePunch.ts
import { useCallback, useRef, useState } from 'react';

type PunchAction = 'clock_in' | 'clock_out';
type PunchResponse = { ok: boolean; punch?: any; serverTime?: string };

const inFlight = new Set<string>();
const idempotencyStore = new Map<string, string>();

function makeIdempotencyKey(action: PunchAction, dateStr: string) {
  return `punch:${action}:${dateStr}`;
}

function persistKey(key: string, idempotencyKey: string) {
  try {
    sessionStorage.setItem(key, idempotencyKey);
  } catch {}
  idempotencyStore.set(key, idempotencyKey);
}

function getPersistedKey(key: string): string | null {
  try {
    return sessionStorage.getItem(key);
  } catch {
    return idempotencyStore.get(key) || null;
  }
}

export function usePunch() {
  const [isSubmitting, setIsSubmitting] = useState(false);
  const [isAcquiringLocation, setIsAcquiringLocation] = useState(false);
  const serverTimeOffsetRef = useRef<number | null>(null);

  const getServerTime = useCallback(async (): Promise<string> => {
    if (serverTimeOffsetRef.current !== null) {
      return new Date(Date.now() + serverTimeOffsetRef.current).toISOString();
    }
    try {
      const res = await fetch('/api/time', { method: 'GET', cache: 'no-store' });
      const { serverTime } = await res.json();
      const serverTs = new Date(serverTime).getTime();
      serverTimeOffsetRef.current = serverTs - Date.now();
      return serverTime;
    } catch {
      return new Date().toISOString();
    }
  }, []);

  const punch = useCallback(
    async (action: PunchAction, location?: GeolocationPosition): Promise<PunchResponse> => {
      const dateStr = new Date().toISOString().slice(0, 10);
      const key = makeIdempotencyKey(action, dateStr);
      let idempotencyKey = getPersistedKey(key) || crypto.randomUUID();
      persistKey(key, idempotencyKey);

      const inFlightKey = `${key}:${idempotencyKey}`;
      if (inFlight.has(inFlightKey)) {
        return { ok: true };
      }

      setIsSubmitting(true);
      inFlight.add(inFlightKey);

      try {
        const clientTimestamp = await getServerTime();

        const body = {
          action,
          timestamp: clientTimestamp,
          location: location
            ? {
                lat: location.coords.latitude,
                lng: location.coords.longitude,
                accuracy: location.coords.accuracy,
              }
            : null,
          idempotencyKey,
        };

        const res = await fetch('/api/punch', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify(body),
        });

        if (!res.ok) {
          inFlight.delete(inFlightKey);
          return { ok: false };
        }

        const data = (await res.json()) as PunchResponse;
        return data;
      } finally {
        setIsSubmitting(false);
        inFlight.delete(inFlightKey);
      }
    },
    [getServerTime]
  );

  const punchWithLocation = useCallback(
    async (action: PunchAction) => {
      setIsAcquiringLocation(true);
      try {
        const position = await new Promise<GeolocationPosition>((resolve, reject) => {
          navigator.geolocation.getCurrentPosition(resolve, reject, {
            enableHighAccuracy: true,
            timeout: 10000,
            maximumAge: 30000,
          });
        });
        return await punch(action, position);
      } catch {
        return await punch(action, undefined);
      } finally {
        setIsAcquiringLocation(false);
      }
    },
    [punch]
  );

  return { punch, punchWithLocation, isSubmitting, isAcquiringLocation };
}
```

```tsx
// workio/src/components/PunchClockButton.tsx
import React from 'react';
import { usePunch } from '../hooks/usePunch';

export function PunchClockButton() {
  const { punchWithLocation, isSubmitting, isAcquiringLocation } = usePunch();
  const isWorking = false; // derive from store/context in real app

  return (
    <button
      onClick={() => punchWithLocation(isWorking ? 'clock_out' : 'clock_in')}
      disabled={isSubmitting || isAcquiringLocation}
      className="btn btn-primary w-full disabled:opacity-60"
    >
      {isAcquiringLocation
        ? 'กำลังระบุตำแหน่ง...'
        : isSubmitting
        ? 'กำลังบันทึก...'
        : isWorking
        ? 'ออกงาน'
        : 'เข้างาน'}
    </button>
  );
}
```
