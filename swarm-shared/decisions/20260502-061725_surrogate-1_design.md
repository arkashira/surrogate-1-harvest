# surrogate-1 / design

**Diagnosis**
* The project lacks a robust implementation for handling Hugging Face API rate limits, which can block dataset training.
* There is inadequate reuse of existing Lightning Studio instances, leading to wasted quota and potential downtime.
* The project does not leverage the Hugging Face CDN to bypass API rate limits.
* The project does not implement a strategy to pre-list file paths once and embed them in training scripts.
* The project does not reuse existing Lightning Studio instances.

**Proposed change**
* Implement a strategy to pre-list file paths once and embed them in training scripts.

**Implementation**
1. Create a new file `src/design/hf_api.py` with the following content:
```python
import json
import os
import requests
from huggingface_hub import HfApi

def list_repo_tree(path, recursive=False):
    api = HfApi()
    response = api.list_repo_tree(path, recursive=recursive)
    return response.json()

def save_file_list(file_list, filename):
    with open(filename, 'w') as f:
        json.dump(file_list, f)

def load_file_list(filename):
    with open(filename, 'r') as f:
        return json.load(f)
```
2. Update the `train.py` file to use the `list_repo_tree` function to pre-list file paths and save them to a JSON file:
```python
import os
import json
from src.design.hf_api import list_repo_tree, save_file_list

# ... (rest of the code remains the same)

# Pre-list file paths and save to JSON
file_list = list_repo_tree('https://huggingface.co/datasets/{repo}/resolve/main/{path}', recursive=False)
save_file_list(file_list, 'file_list.json')

# ... (rest of the code remains the same)
```
3. Update the `train.py` file to load the pre-listed file paths from the JSON file:
```python
import os
import json
from src.design.hf_api import load_file_list

# ... (rest of the code remains the same)

# Load pre-listed file paths from JSON
file_list = load_file_list('file_list.json')

# ... (rest of the code remains the same)
```
**Verification**
1. Run the `train.py` script to train the model.
2. Check that the `file_list.json` file is created and contains the pre-listed file paths.
3. Verify that the training process completes successfully without hitting the Hugging Face API rate limits.
4. Check that the model is trained using the pre-listed file paths.
