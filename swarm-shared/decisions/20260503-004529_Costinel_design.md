# Costinel / design

## 1) Highest-value incremental improvement
Ship a **single reusable “Top-Hub Signal” card** on the dashboard that surfaces the most-connected hub (e.g., MOC) with actionable context from knowledge-rag.  
- Pure frontend, zero backend changes.  
- Read-only (Sense + Signal).  
- Deterministic (uses precomputed hub graph or static JSON).  
- Immediately mergeable and runnable in dev/PR.

---

## 2) Implementation plan (≤2h)

| Step | Owner | Time | Details |
|--|--|--|--|
| 1. Add mock hub data | FE | 10m | `src/data/topHub.json` — `{ hub, score, signals[], relatedDocs[] }` |
| 2. Create card component | FE | 30m | `src/components/cards/TopHubSignalCard.tsx` — compact, responsive, accessible |

| Buffer | — | 20m | Polish & PR prep |

### `src/components/cards/TopHubSignalCard.tsx`
```tsx
import React from 'react';
import topHub from '@/data/topHub.json';

type Severity = 'high' | 'medium' | 'low';
const severityColors: Record<Severity, string> = {
  high: 'bg-red-50 border-red-200 text-red-800',
  medium: 'bg-amber-50 border-amber-200 text-amber-800',
  low: 'bg-blue-50 border-blue-200 text-blue-800',
};

export const TopHubSignalCard: React.FC = () => {
  return (
    <article className="rounded-xl border bg-white p-5 shadow-sm ring-1 ring-gray-100 transition hover:shadow-md">
      <div className="mb-4 flex items-start justify-between">
        <div>
          <h2 className="text-base font-semibold text-gray-900">Top Hub</h2>
          <p className="text-2xl font-bold text-gray-950">{topHub.hub}</p>
          <p className="text-xs text-gray-500">{topHub.label}</p>
        </div>
        <span className="inline-flex items-center rounded-full bg-emerald-50 px-2.5 py-0.5 text-xs font-medium text-emerald-700">
          Score {topHub.score}
        </span>
      </div>

      <div className="space-y-3">
        {topHub.signals.map((s) => (
          <div key={s.id} className={`rounded-lg border p-3 ${severityColors[s.severity]}`}>
            <div className="flex items-start justify-between gap-2">
              <div>
                <p className="text-xs font-semibold uppercase tracking-wide">{s.title}</p>
                <p className="text-xs text-gray-600">{s.description}</p>
              </div>
            </div>
            <p className="mt-2 text-xs font-medium text-gray-700">{s.action}</p>
          </div>
        ))}
      </div>

      <div className="mt-4 flex items-center justify-between border-t border-gray-100 pt-4">
        <div className="flex gap-2">
          {topHub.relatedDocs.map((doc) => (
            <a
              key={doc.title}
              href={doc.url}
              className="text-xs text-gray-500 hover:text-gray-700 hover:underline"
            >
              {doc.title}
            </a>
          ))}
        </div>
        <button
          type="button"
          className="text-xs font-medium text-axentx hover:underline"
          onClick={() => {
            // Intent: open knowledge-rag focused on this hub
            window.open(`/knowledge-rag?hub=${encodeURIComponent(topHub.hub)}`, '_blank');
          }}
        >
          Open in Knowledge-RAG
        </button>
      </div>
    </article>
  );
};
```
