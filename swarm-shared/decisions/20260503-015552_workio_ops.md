# workio / ops

## Final Synthesis (best parts, corrected + actionable)

**Core diagnosis (merged, corrected)**
- LINE retries identical payloads/signatures → duplicate punches.
- Read-then-write upsert is racy; concurrent replays or double-taps corrupt state.
- No transactional boundary for punch + audit rows → partial writes lose traceability.
- No short-term replay protection (DB unique constraint or fast cache).
- Clock-skew tolerance and duplicate-second handling are ambiguous.

**Chosen approach (correctness + actionability)**
- Use **DB-level idempotency + unique constraint** (PostgreSQL) so correctness survives restarts, multi-instance, and cache loss.
- Keep **Redis for fast short-window dedupe** to cheaply absorb bursts/retries, but **DB unique constraint is the source of truth**.
- **Single transaction** per webhook: idempotency insert → punch upsert → audit insert; rollback on any failure.
- **Idempotency key**: `HMAC-SHA256(channelSecret, rawBody)` (or signature+hash) to avoid storing secrets in the key and to detect payload tampering.
- **Unique constraint** for punches: `(user_id, event_type, clock_ts_utc)` with a **1-second tolerance window** implemented by truncating to 1s (or application-level coalesce) to prevent duplicates from retries while allowing legitimate separate punches >1s apart.
- **Alerting**: export duplicate/rejection counters and webhook error rates to Prometheus; add simple log-based alert rules.

### 1) DB migrations (PostgreSQL)

File: `server/src/db/migrations/20260503_line_idempotency_and_punches.sql`

```sql
-- Idempotency keys (short TTL, fast lookups)
CREATE TABLE IF NOT EXISTS line_webhook_idempotency (
  idempotency_key TEXT PRIMARY KEY,
  payload_hash     TEXT NOT NULL,
  created_at       TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_line_idempotency_created_at
  ON line_webhook_idempotency (created_at);

-- Punches table (adjust names/columns to your schema)
-- Ensure a unique constraint to prevent duplicates per user+event+1s window
CREATE TABLE IF NOT EXISTS punches (
  id              BIGSERIAL PRIMARY KEY,
  user_id         BIGINT NOT NULL,
  event_type      TEXT   NOT NULL, -- 'clock_in' | 'clock_out'
  clock_ts_utc    TIMESTAMPTZ NOT NULL,
  line_user_id    TEXT,
  source          TEXT NOT NULL DEFAULT 'line',
  created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

-- Unique constraint: one event per user per 1-second window (coalesces retries)
CREATE UNIQUE INDEX IF NOT EXISTS idx_punches_user_event_clock_1s
  ON punches (user_id, event_type, date_trunc('second', clock_ts_utc));

-- Optional audit table (transactional with punch)
CREATE TABLE IF NOT EXISTS punch_audits (
  id         BIGSERIAL PRIMARY KEY,
  punch_id   BIGINT NOT NULL REFERENCES punches(id) ON DELETE CASCADE,
  action     TEXT NOT NULL,
  details    JSONB,
  created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```

import { Pool } from 'pg';
import crypto from 'crypto';

const pool = new Pool({ connectionString: process.env.DATABASE_URL });
const IDEMPOTENCY_TTL_MS = 5 * 60 * 1000; // 5m short TTL

// Optional Redis fast path (best-effort only; DB is source of truth)
let redis: { get: (k: string) => Promise<string | null>; setex: (k: string, ttl: number, v: string) => Promise<void> } | null = null;
export function setRedisClient(client: typeof redis) {
  redis = client;
}

function idempotencyKey(rawBody: Buffer, signature: string): string {
  // Use HMAC with channel secret so key is tamper-evident and not guessable
  const secret = process.env.LINE_CHANNEL_SECRET || '';
  return 'line:' + crypto.createHmac('sha256', secret).update(rawBody).digest('hex');
}

    const cached = await redis.get(key);
    if (cached === '1') return { ok: true, duplicated: true };

  const client = await pool.connect();
  try {
    await client.query('BEGIN');

    // Strong DB idempotency check+insert
    const exists = await client.query(
      `SELECT 1 FROM line_webhook_idempotency WHERE idempotency_key = $1 FOR UPDATE`,
      [key]

      await client.query('ROLLBACK');
      if (redis) await redis.setex(key, IDEMPOTENCY_TTL_MS / 1000, '1').catch(() => {});
      return { ok: true, duplicated: true };

      return { ok: true };
