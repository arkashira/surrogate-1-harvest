# airship / ops

## Implementation Plan — <2h Quality Increment

**Highest-value incremental improvement**: Harden the `airship discover` orchestrator so it is **fully deterministic, CDN-safe, and zero-runtime** by:
- Adding a strict Bash shebang and `set -euo pipefail`
- Making file-execution robust (`chmod +x`, explicit `bash` invocation)
- Embedding a pre-listed file manifest so Lightning training uses **CDN-only fetches** (zero HF API calls during training)
- Guarding against Lightning idle-stop by checking studio status before `.run()` and auto-restarting on L40S if stopped

### Concrete Steps (all <2h)

1. **Locate `airship discover` orchestrator** (likely `./scripts/discover.sh` or `./airship/discover.sh`).
2. **Add strict Bash header + POSIX safety**:
   ```bash
   #!/usr/bin/env bash
   set -euo pipefail
   IFS=$'\n\t'
   ```
3. **Ensure executable bit and safe invocation** in any cron/CI:
   ```bash
   chmod +x ./scripts/discover.sh
   # invoke explicitly
   bash ./scripts/discover.sh "$@"
   ```
   And in crontab:
   ```cron
   SHELL=/bin/bash
   ```
4. **Produce a deterministic file manifest** (JSON) for the training date folder:
   ```bash
   # single API call (rate-limit window cleared on Mac)
   python -c "
   from huggingface_hub import HfApi
   import json, os, datetime
   api = HfApi()
   repo_id='datasets/your-org/surrogate-mirror'
   date_folder=datetime.date.today().isoformat()
   tree = api.list_repo_tree(repo_id, path=date_folder, recursive=False)
   files = [f.rfilename for f in tree if f.rfilename.endswith('.parquet')]
   os.makedirs('manifests', exist_ok=True)
   with open(f'manifests/files-{date_folder}.json','w') as f:
       json.dump({'date': date_folder, 'files': files}, f)
   "
   ```
5. **Embed manifest in `train.py`** and use CDN-only fetches during training:
   ```python
   import json, requests, pyarrow.parquet as pq, io, os

   with open('manifests/files-2026-05-02.json') as f:
       manifest = json.load(f)

   repo = 'datasets/your-org/surrogate-mirror'
   base = f'https://huggingface.co/{repo}/resolve/main'

   def cdn_fetch(path):
       url = f'{base}/{path}'
       resp = requests.get(url, timeout=30)
       resp.raise_for_status()
       return resp.content

   def load_shard(path):
       buf = io.BytesIO(cdn_fetch(path))
       return pq.read_table(buf, columns=['prompt', 'response'])

   # deterministic order
   shards = [load_shard(p) for p in sorted(manifest['files'])]
   ```
6. **Lightning Studio reuse + idle-stop guard** in launcher (`launch_training.py`):
   ```python
   from lightning import Studio, Machine, Teamspace

   studio_name = 'surrogate-train-l40s'
   teamspace = Teamspace()

   studio = None
   for s in teamspace.studios:
       if s.name == studio_name:
           studio = s
           break

   if studio is None:
       studio = teamspace.create_studio(
           name=studio_name,
           machine=Machine.L40S,
           create_ok=True
       )

   # Guard against idle-stop
   if studio.status != 'Running':
       studio.start(machine=Machine.L40S)

   # Run training (zero HF API calls inside train.py)
   studio.run(
       run_name='surrogate-cdn-train',
       entry_point='python train.py',
       config={'manifests/files-2026-05-02.json': 'manifests/files-2026-05-02.json'}
   )
   ```
