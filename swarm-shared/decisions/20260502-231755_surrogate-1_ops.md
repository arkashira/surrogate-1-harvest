# surrogate-1 / ops

## Highest-value incremental improvement (≤2h)

**Deterministic pre-flight snapshot + CDN-only ingestion for surrogate-1-runner**  
Replace runtime `load_dataset`/`list_repo_files` with a single Mac-side snapshot (JSON) that lists exact file paths for one date folder, then have each GitHub Actions shard worker fetch only its slice via CDN URLs (`resolve/main/...`). This eliminates HF API 429s during ingestion, makes shard workers idempotent, and removes schema-mixing risks because workers project `{prompt,response}` at parse time.

### Concrete implementation plan

1. Add Mac-side snapshot script (`bin/make-snapshot.sh`)
   - Inputs: `DATASET_REPO`, `DATE` (YYYY-MM-DD), optional `HF_TOKEN`
   - Uses `huggingface_hub` to call `list_repo_tree(path=f"public-merged/{DATE}", recursive=True)` once
   - Filters to parquet files, normalizes paths, produces `snapshot-{DATE}.json`
   - Commits snapshot to repo (or uploads as workflow artifact) so runners consume it without HF API calls

2. Update `bin/dataset-enrich.sh`
   - Accept `SNAPSHOT_FILE` and `SHARD_ID`/`SHARD_TOTAL`
   - Load snapshot JSON, deterministically assign files to shards by `hash(slug) % SHARD_TOTAL`
   - For each assigned file: download via CDN URL (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) with `curl`/`wget` (no auth header)
   - Stream-parse parquet → project `{prompt, response}` only
   - Dedup via central `lib/dedup.py` md5 store
   - Emit `batches/public-merged/{date}/shard{N}-{HHMMSS}.jsonl`

3. Update GitHub Actions matrix (`ingest.yml`)
   - Add optional `snapshot` input; if absent, fallback to legacy behavior with warning
   - Pass `SHARD_ID: ${{ matrix.shard_id }}` and `SHARD_TOTAL: 16`
   - Cache snapshot as artifact between jobs to avoid re-downloading

4. Add small Python helper (`tools/snapshot.py`)
   - Deterministic shard assignment function
   - CDN download with retries and backoff
   - Parquet projection to `{prompt, response}` (no mixed-schema columns)

5. Validation & rollback
   - Dry-run snapshot for yesterday’s folder and verify shard counts are balanced
   - Keep legacy path functional behind a feature flag (`USE_SNAPSHOT=false`) for quick rollback

Estimated effort: ~1.5h (one small script + one workflow tweak + one bash update).

### Code snippets

#### bin/make-snapshot.sh
```bash
#!/usr/bin/env bash
set -euo pipefail

REPO="${DATASET_REPO:-axentx/surrogate-1-training-pairs}"
DATE="${1:-$(date -I)}"
OUT="${2:-snapshot-${DATE}.json}"

python3 - <<PY
import os, json, sys
from huggingface_hub import HfApi

repo = os.getenv("DATASET_REPO", "axentx/surrogate-1-training-pairs")
date = sys.argv[1]
out = sys.argv[2]

api = HfApi()
prefix = f"public-merged/{date}"
entries = api.list_repo_tree(repo=repo, path=prefix, recursive=True)

files = [e.rfilename for e in entries if e.rfilename.endswith(".parquet")]
files.sort()

snapshot = {
    "repo": repo,
    "date": date,
    "prefix": prefix,
    "files": files,
    "n_files": len(files),
}

with open(out, "w") as f:
    json.dump(snapshot, f, indent=2)
print(f"Snapshot written to {out} ({len(files)} files)")
PY "$DATE" "$OUT"
```

#### tools/snapshot.py
```python
import hashlib
import json
import os
import pyarrow.parquet as pq
import requests
from typing import List, Dict, Any

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def shard_for_file(path: str, n_shards: int) -> int:
    # Deterministic by content slug
    slug = os.path.splitext(os.path.basename(path))[0]
    digest = hashlib.md5(slug.encode()).hexdigest()
    return int(digest, 16) % n_shards

def download_parquet_via_cdn(repo: str, path: str, local_path: str, max_retries: int = 3) -> str:
    url = CDN_TEMPLATE.format(repo=repo, path=path)
    for attempt in range(1, max_retries + 1):
        try:
            with requests.get(url, stream=True, timeout=30) as r:
                r.raise_for_status()
                with open(local_path, "wb") as f:
                    for chunk in r.iter_content(chunk_size=8192):
                        f.write(chunk)
            return local_path
        except Exception as exc:
            if attempt == max_retries:
                raise
            sleep_s = 2 ** attempt
            print(f"Retry {attempt}/{max_retries} for {url}: {exc}; sleeping {sleep_s}s")
            import time
            time.sleep(sleep_s)

def project_parquet_to_pairs(path: str) -> List[Dict[str, str]]:
    # Only read columns we need; tolerate extra schema columns
    try:
        pf = pq.ParquetFile(path)
        cols = pf.schema.names
        has_prompt = "prompt" in cols
        has_response = "response" in cols
        if not (has_prompt and has_response):
            # Best-effort fallback: look for common aliases
            raise ValueError(f"Missing prompt/response in {path}: {cols}")
        table = pf.read(columns=["prompt", "response"])
        df = table.to_pandas()
        pairs = [{"prompt": str(p), "response": str(r)} for p, r in zip(df["prompt"], df["response"])]
        return pairs
    except Exception as exc:
        raise RuntimeError(f"Failed to project {path}: {exc}")

if __name__ == "__main__":
    # Quick test
    import tempfile, shutil
    repo = "axentx/surrogate-1-training-pairs"
    snapshot_file = "snapshot-test.json"
    if not os.path.exists(snapshot_file):
        print("No snapshot-test.json; skipping test")
    else:
        with open(snapshot_file) as f:
            snap = json.load(f)
        for p in snap["files"][:2]:
            with tempfile.NamedTemporaryFile(suffix=".parquet", delete=False) as tmp:
                tmp_path = tmp.name
            try:
                download_parquet_via_cdn(repo, p, tmp_path)
                pairs = project_parquet_to_pairs(tmp_path)
                print(f"{p} -> {len(pairs)} pairs")
            finally:
                os.unlink(tmp_path)
```
