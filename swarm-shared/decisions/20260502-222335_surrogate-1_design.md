# surrogate-1 / design

Below is the **single, merged implementation** that keeps every high-leverage idea from both proposals while resolving conflicts in favor of **correctness, determinism, and concrete actionability**.

---

## Summary of decisions (conflict resolution)

| Conflict | Chosen approach (why) |
|---|---|
| File-list metadata | Use **path + size + sha256** (Candidate 2). Enables integrity checks and change detection; size alone (Candidate 1) is insufficient. |
| CDN fallback behavior | **Retry once via `hf_hub_download`** (Candidate 2). Keeps private/missing/moved files working without breaking CDN-first speed. Candidate 1 had no fallback. |
| Rate-limit backoff on CDN 429 | **360s cap with exponential backoff** (Candidate 2). Matches HF guidance and prevents infinite loops. Candidate 1 used shorter waits but no ceiling. |
| Parquet projection safety | Keep Candidate 1’s column-aware projection **plus** Candidate 2’s fallback when `prompt`/`response` are absent. |
| Determinism & idempotency | Embed file-list JSON into each shard via `env.FILE_LIST` (both). Workers use **modulo sharding on sorted list** so assignments never drift. |
| Code organization | Single `lib/cdn.py` (Candidate 2 filename) containing all fetch + stream helpers. Avoids scattering logic. |

---

## Implementation Plan (≤2h)

**Highest-value improvement**: Deterministic pre-flight file listing + CDN-only ingestion to eliminate HF API 429s during training and make shard workers fully resilient.

### What we’ll change
1. Add `bin/list-files.sh` — one-time Mac/CI script that calls `list_repo_tree(recursive=False)` per date folder and writes `file-list-<date>.json` with `path`, `size`, `sha256`.
2. Add `lib/cdn.py` — CDN fetcher with retries, exponential backoff (cap 360s), and a one-time fallback to `hf_hub_download` on CDN failure.
3. Update `bin/dataset-enrich.sh` to accept optional `FILE_LIST`. If set, workers iterate the local list and stream via CDN; otherwise keep existing API behavior.
4. Update training stub (`train.py`) to load the file-list and use `lib/cdn.py` for deterministic, CDN-only data loading.

### 1. `bin/list-files.sh`
```bash
#!/usr/bin/env bash
# list-files.sh
# Usage: HF_TOKEN=... ./list-files.sh <repo> <date> > file-list-<date>.json
# Example: HF_TOKEN=hf_xxx ./list-files.sh axentx/surrogate-1-training-pairs 2026-05-02

set -euo pipefail
REPO="${1:-axentx/surrogate-1-training-pairs}"
DATE="${2:-$(date +%Y-%m-%d)}"

python3 - "$REPO" "$DATE" <<'PY'
import os, json, hashlib, sys
from huggingface_hub import HfApi

repo = sys.argv[1]
date = sys.argv[2]
api = HfApi(token=os.getenv("HF_TOKEN"))

# Single non-recursive call per date folder
tree = api.list_repo_tree(repo=repo, path=f"batches/public-merged/{date}", recursive=False)
files = []
for node in tree:
    if node.type != "file":
        continue
    if not (node.path.endswith(".jsonl") or node.path.endswith(".parquet")):
        continue
    # Best-effort sha256; if missing we will compute on first CDN miss fallback
    files.append({
        "repo": repo,
        "path": node.path,
        "size": getattr(node, "size", 0),
        "sha256": getattr(node, "sha256", ""),
    })

# Deterministic ordering
files.sort(key=lambda x: x["path"])

out = {
    "repo": repo,
    "date": date,
    "files": files,
}
json.dump(out, sys.stdout, indent=2)
PY
```

Make executable:
```bash
chmod +x bin/list-files.sh
```

# lib/cdn.py
import io
import json
import time
import os
from typing import Dict, Iterator, Optional

import pyarrow.parquet as pq
import requests
from huggingface_hub import hf_hub_download

CDN_ROOT = "https://huggingface.co/datasets"
MAX_RETRIES = 5
BACKOFF_CAP = 360  # seconds

def _backoff(attempt: int) -> float:
    return min(BACKOFF_CAP, (2 ** attempt) + (attempt * 0.5))

def _retry_get(url: str) -> requests.Response:
    for attempt in range(MAX_RETRIES):
        try:
            resp = requests.get(url, timeout=30)
            if resp.status_code == 429:
                wait = int(resp.headers.get("retry-after", _backoff(attempt)))
                time.sleep(wait)
                continue
            resp.raise_for_status()
            return resp
        except requests.RequestException:
            if attempt == MAX_RETRIES - 1:
                raise
            time.sleep(_backoff(attempt))
    raise RuntimeError("unreachable")

def download_via_cdn(repo: str, path: str, dest: Optional[str] = None) -> str:
    """
    Download file via CDN. On CDN failure (404/403/missing), fallback once to hf_hub_download.
    Returns local path.
    """
    url = f"{CDN_ROOT}/{repo}/resolve/main/{path}"
    try:
        resp = _retry_get(url)
    except Exception:
        # CDN failed; fallback to hub download (supports private/missing/moved files)
        return hf_hub_download(repo_id=repo, filename=path, repo_type="dataset")

    if dest is None:
        dest = os.path.basename(path)
    os.makedirs(os.path.dirname(os.path.abspath(dest)), exist_ok=True)
    with open(dest, "wb") as f:
        for chunk in resp.iter_content(chunk_size=8192):
            f.write(chunk)
    return dest

def _project_pair(raw: Dict) -> Dict:
    return {"prompt": raw.get("prompt", ""), "response": raw.get("response", "")}

def stream_cdn_jsonl(repo: str, path: str) -> Iterator[Dict]:
    url = f"{CDN_ROOT}/{repo}/resolve/main/{path}"
    try:
        resp = _retry_get(url)
    except Exception:
        # fallback
        local_path = hf_hub_download(repo_id=repo, filename=path, repo_type="dataset")
        with open(local_path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                try:
                    yield _project_pair(json.loads(line))
                except json.JSONDecodeError:
                    continue
        return

    for line in resp.iter_lines(decode_unicode=True):
        if not line:
            continue
        try:
            yield _project_pair(json.loads(line))
        except json.JSONDecodeError:
            continue
