# vanguard / design

## 1) Diagnosis

- No canonical discovery entrypoint exists to surface top-hub insights (e.g., MOC) before planning — violates established `#knowledge-rag #graph #hub` pattern.
- Missing CDN-bypass file-list strategy for HF datasets; any future training ingestion will immediately hit 429 rate limits.
- No reusable Lightning Studio orchestration stub to avoid quota waste and idle-stop training loss.
- No executable wrapper to run `granite-business-research.sh` + `knowledge-rag` in sequence for contextual insights.
- No clear separation of orchestration (Mac) vs compute (Lightning) boundaries for surrogate-1 training.

## 2) Proposed change

Add a compact, executable design scaffold under `/opt/axentx/vanguard/`:

- `discover.sh` — runs market research + knowledge-rag query for top-hub insights (business research pattern).
- `hf_filelist.py` — one-shot Mac-side script to list a date folder in an HF dataset repo and emit `filelist.json` for CDN-bypass training.
- `train_stub.py` — Lightning training stub that consumes `filelist.json` and downloads via CDN URLs only (zero API calls during data load), with studio reuse + idle-stop handling.
- `lightning_launcher.py` — orchestration wrapper to find/reuse a running studio or start one (L40S priority, fallback) and submit `train_stub.py`.

Scope: new files only; no existing code modified.

# /opt/axentx/vanguard/hf_filelist.py
#!/usr/bin/env python3
"""
One-time Mac-side script to list HF dataset folder and emit filelist.json.
Usage: python hf_filelist.py --repo <datasets/repo> --path <date-folder> --out filelist.json
Tags: #huggingface #cdn #rate-limit-bypass #training
"""
import argparse
import json
import os
import time
from huggingface_hub import HfApi, hf_hub_download

CDN_TEMPLATE = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., datasets/mycorp/surrogate-1)")
    parser.add_argument("--path", required=True, help="Folder path in repo (e.g., batches/mirror-merged/2026-05-02)")
    parser.add_argument("--out", default="filelist.json", help="Output JSON file")
    parser.add_argument("--retry-wait", type=int, default=360, help="Wait on 429 (s)")
    args = parser.parse_args()

    api = HfApi()
    # Non-recursive list to minimize pagination/requests
    for attempt in range(3):
        try:
            items = api.list_repo_tree(repo_id=args.repo, path=args.path, recursive=False)
            break
        except Exception as e:
            if "429" in str(e):
                print(f"Rate limited; waiting {args.retry_wait}s")
                time.sleep(args.retry_wait)
                continue
            raise
    else:
        raise RuntimeError("Failed after retries")

    files = []
    for item in items:
        if getattr(item, "type", None) == "file" or "." in os.path.basename(item.path or ""):
            rpath = item.path
            files.append({
                "repo": args.repo,
                "path": rpath,
                "cdn_url": CDN_TEMPLATE.format(repo=args.repo, path=rpath)
            })

    os.makedirs(os.path.dirname(os.path.abspath(args.out)) if os.path.dirname(args.out) else ".", exist_ok=True)
    with open(args.out, "w") as f:
        json.dump(files, f, indent=2)
    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()

import torch
from torch.utils.data import Dataset, DataLoader
import requests
import io
import pyarrow as pa
import pyarrow.parquet as pq

class CDNParquetDataset(Dataset):
    def __init__(self, filelist):
        self.filelist = filelist

    def __len__(self):
        return len(self.filelist)

    def __getitem__(self, idx):
        entry = self.filelist[idx]
        resp = requests.get(entry["cdn_url"], timeout=30)
        resp.raise_for_status()
        table = pq.read_table(io.BytesIO(resp.content))
        # Project to {prompt, response} only at parse time (schema heterogeneity safe)
        prompt = table["prompt"].to_pylist()[0] if "prompt" in table.column_names else ""
        response = table["response"].to_pylist()[0] if "response" in table.column_names else ""
        return {"prompt": prompt, "response": response}
