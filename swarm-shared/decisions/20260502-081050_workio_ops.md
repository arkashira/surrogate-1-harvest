### Implementation Plan
#### Step 1: Update Training Script to Use HF CDN Bypass
Update the training script to download dataset files from the HF CDN using the `https://huggingface.co/datasets/{repo}/resolve/main/{path}` URL pattern, bypassing the HF API rate limit.

```python
import requests

# Define the dataset repository and file path
repo = "your-dataset-repo"
file_path = "your-file-path"

# Download the file from the HF CDN
url = f"https://huggingface.co/datasets/{repo}/resolve/main/{file_path}"
response = requests.get(url)

# Save the file to a local directory
with open(file_path, "wb") as f:
    f.write(response.content)
```

#### Step 2: Pre-List File Paths and Embed in Training Script
Make a single API call to `list_repo_tree(path, recursive=False)` to retrieve the list of file paths for a given date folder, and save the list to a JSON file. Embed the JSON file in the training script to use the pre-listed file paths.

```python
import json
import requests

# Define the dataset repository and date folder
repo = "your-dataset-repo"
date_folder = "your-date-folder"

# Make a single API call to retrieve the list of file paths
url = f"https://huggingface.co/datasets/{repo}/api/list_repo_tree?path={date_folder}&recursive=False"
response = requests.get(url)

# Save the list of file paths to a JSON file
with open("file_paths.json", "w") as f:
    json.dump(response.json(), f)

# Embed the JSON file in the training script
with open("file_paths.json", "r") as f:
    file_paths = json.load(f)

# Use the pre-listed file paths in the training script
for file_path in file_paths:
    # Download the file from the HF CDN
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{file_path}"
    response = requests.get(url)

    # Save the file to a local directory
    with open(file_path, "wb") as f:
        f.write(response.content)
```

#### Step 3: Implement Studio Reuse and Idle Stop Handling
Reuse existing Lightning Studios to save 80hr/mo quota, and handle idle stop kills by checking the studio status before each `.run()` call and restarting with `target.start(machine=Machine.L40S)` if stopped.

```python
import lightning

# Define the Lightning Studio and machine
studio = lightning.Studio()
machine = lightning.Machine.L40S

# Reuse existing Lightning Studios
for s in lightning.Teamspace.studios:
    if s.name == "your-studio-name" and s.status == "Running":
        studio = s
        break

# Handle idle stop kills
if studio.status == "Stopped":
    studio.start(machine=machine)

# Run the training script
studio.run()
```
