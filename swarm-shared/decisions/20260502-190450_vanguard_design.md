# vanguard / design

## Final Synthesis (Corrected + Actionable)

**Core diagnosis (accepted from both candidates):**  
- No canonical discovery entrypoint → violates `#knowledge-rag #graph #hub` pattern.  
- No CDN-bypass file-list → future HF ingestion will immediately hit API rate limits.  
- No Lightning Studio reuse guard → quota burn and idle-stop risk.  
- No local data projection for mixed-schema files → schema hygiene risk for surrogate-1 pipeline.  
- No single orchestration script to compose discovery → CDN file-list → studio reuse → local projection.

**Chosen approach:**  
Create one executable discovery script plus two small helpers that are correct, minimal, and immediately runnable.

### 1) Canonical entrypoint (executable)

`/opt/axentx/vanguard/discovery/run_discovery.sh`

```bash
#!/usr/bin/env bash
# run_discovery.sh
# Single-pass discovery: top-hub insights, HF CDN-bypass file list,
# Lightning Studio reuse, local heterogeneous projection.
#
# Usage:
#   ./run_discovery.sh [--hf-repo <repo>] [--date-folder <YYYY-MM-DD>] [--local-src <dir>]
#
# Env may also be used: HF_REPO, DATE_FOLDER, LOCAL_SRC

set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "$SCRIPT_DIR/../../" && pwd)"

HF_REPO="${HF_REPO:-}"
DATE_FOLDER="${DATE_FOLDER:-}"
LOCAL_SRC="${LOCAL_SRC:-}"

while [[ $# -gt 0 ]]; do
  case $1 in
    --hf-repo)     HF_REPO="$2";   shift 2 ;;
    --date-folder) DATE_FOLDER="$2"; shift 2 ;;
    --local-src)   LOCAL_SRC="$2"; shift 2 ;;
    *) echo "Unknown option: $1"; exit 1 ;;
  esac
done

log() { echo "[$(date -Iseconds)] $*"; }

mkdir -p "$PROJECT_ROOT/tmp" "$PROJECT_ROOT/data"

# 1) Top-hub insights (MOC) via knowledge-rag if available
log "Step 1: Querying top-hub insights..."
if command -v knowledge-rag >/dev/null 2>&1; then
  knowledge-rag query --top-hub MOC --limit 5 --format md > "$PROJECT_ROOT/tmp/top_hub.md" 2>/dev/null || true
  log "Top-hub insights saved to tmp/top_hub.md"
else
  log "knowledge-rag not found — skipping hub query (install to enable)."
fi

# 2) HF CDN-bypass file-list (single API call)
if [[ -n "$HF_REPO" && -n "$DATE_FOLDER" ]]; then
  log "Step 2: Generating HF CDN-bypass file list for $HF_REPO/$DATE_FOLDER ..."
  python3 "$SCRIPT_DIR/list_hf_files.py" \
    --repo "$HF_REPO" \
    --folder "$DATE_FOLDER" \
    --out "$PROJECT_ROOT/data/file_list.json"
  log "File list saved to data/file_list.json (embed in train.py for CDN-only fetches)."
else
  log "Step 2: Skipped HF file-list (provide --hf-repo and --date-folder)."
fi

# 3) Reuse running Lightning Studio (quota-safe)
log "Step 3: Checking for running Lightning Studio to reuse..."
python3 "$SCRIPT_DIR/reuse_studio.py" --name "vanguard-train" --machine "L40S" || \
  log "No reusable running studio found (training script will create on demand)."

# 4) Project local heterogeneous data to {prompt,response}
if [[ -n "$LOCAL_SRC" && -d "$LOCAL_SRC" ]]; then
  log "Step 4: Projecting local data from $LOCAL_SRC ..."
  python3 "$SCRIPT_DIR/project_local_data.py" \
    --src "$LOCAL_SRC" \
    --out "$PROJECT_ROOT/data/enriched/projected.parquet"
  log "Projected data saved to data/enriched/projected.parquet"
else
  log "Step 4: Skipped local projection (--local-src not provided or not a directory)."
fi

log "Discovery complete."
```

Make executable:

```bash
chmod +x /opt/axentx/vanguard/discovery/run_discovery.sh
```

### 2) HF CDN-bypass file-list helper

`/opt/axentx/vanguard/discovery/list_hf_files.py`

```python
import argparse
import json
import os
import sys
from typing import List, Dict

try:
    from huggingface_hub import list_repo_tree, HfApi
except ImportError:
    sys.stderr.write("huggingface_hub not installed. Install via: pip install huggingface_hub\n")
    sys.exit(1)

def main() -> None:
    parser = argparse.ArgumentParser(description="List HF dataset files (non-recursive) for CDN-bypass.")
    parser.add_argument("--repo", required=True, help="HF dataset repo (e.g., 'datasets/owner/name')")
    parser.add_argument("--folder", required=True, help="Folder path inside repo (e.g., '2026-04-29')")
    parser.add_argument("--out", required=True, help="Output JSON path")
    args = parser.parse_args()

    api = HfApi()
    try:
        tree = list_repo_tree(repo_id=args.repo, path=args.folder, recursive=False)
    except Exception as e:
        sys.stderr.write(f"Failed to list repo tree: {e}\n")
        sys.exit(1)

    files: List[Dict[str, str]] = []
    for entry in tree:
        if entry.type == "file":
            # CDN URL (no Authorization header) — bypasses /api/ rate limits
            cdn_url = f"https://huggingface.co/datasets/{args.repo}/resolve/main/{entry.path}"
            files.append({"path": entry.path, "cdn_url": cdn_url})

    os.makedirs(os.path.dirname(os.path.abspath(args.out)), exist_ok=True)
    with open(args.out, "w", encoding="utf-8") as f:
        json.dump({"repo": args.repo, "folder": args.folder, "files": files}, f, indent=2)

    print(f"Wrote {len(files)} files to {args.out}")

if __name__ == "__main__":
    main()
```
