# surrogate-1 / ops

## Implementation Plan — CDN-first snapshot + zero-HF-API ingestion

**Goal**: Eliminate HF API rate-limit risk during training by producing a deterministic file manifest once (on the Mac orchestrator) and having Lightning training fetch exclusively via CDN URLs.

**Scope** (fits <2h):
- Add `bin/snapshot.sh` → single API call to `list_repo_tree` for one date folder, emit `snapshot/<date>/files.json`
- Add `bin/cdn_urls.py` → convert snapshot to CDN URLs (`https://huggingface.co/datasets/.../resolve/main/...`)
- Update `bin/dataset-enrich.sh` to accept optional snapshot file and skip recursive listing
- Add `training/cdn_loader.py` example showing zero-API dataloader using CDN URLs + `datasets` `load_from_disk` or direct `parquet` reads
- Add `README.md` snippet documenting the pattern and HF API strategy

**Why this is highest value**:
- Removes the 429/1000 req/5min risk during training loops
- Keeps HF API usage to a single orchestrator call (Mac) per date folder
- Enables deterministic, reproducible training runs
- Fits existing patterns: pre-list once, embed in training script; CDN bypass; reuse file list

### 1) `bin/snapshot.sh` — deterministic manifest

```bash
#!/usr/bin/env bash
# bin/snapshot.sh
# Usage: HF_TOKEN=... ./bin/snapshot.sh <date> [repo]
# Produces: snapshot/<date>/files.json  (list of file paths)

set -euo pipefail

DATE="${1:-$(date +%Y-%m-%d)}"
REPO="${2:-axentx/surrogate-1-training-pairs}"
OUTDIR="snapshot/${DATE}"
OUTFILE="${OUTDIR}/files.json"

mkdir -p "${OUTDIR}"

echo "Listing ${REPO} for date=${DATE} ..."
# Single non-recursive call per folder (avoids 100x pagination)
# Uses huggingface_hub via python for convenience; falls back to curl if needed
python3 - "$REPO" "$DATE" "$OUTFILE" <<'PY'
import os, json, sys
from huggingface_hub import list_repo_tree

repo_id, date_folder, out_path = sys.argv[1], sys.argv[2], sys.argv[3]

# List top-level or date folder non-recursive
try:
    tree = list_repo_tree(repo_id, path=date_folder, recursive=False)
except Exception as e:
    # If date folder doesn't exist, list repo root non-recursive and filter
    tree = list_repo_tree(repo_id, recursive=False)
    tree = [t for t in tree if t.path.startswith(date_folder)]

files = [t.path for t in tree if not t.path.endswith("/")]
with open(out_path, "w") as f:
    json.dump({"repo": repo_id, "date": date_folder, "files": files}, f, indent=2)

print(f"Wrote {len(files)} files to {out_path}")
PY

echo "Snapshot saved to ${OUTFILE}"
```

Make executable:
```bash
chmod +x bin/snapshot.sh
```

### 2) `bin/cdn_urls.py` — convert snapshot to CDN URLs

```python
#!/usr/bin/env python3
# bin/cdn_urls.py
# Usage: python3 bin/cdn_urls.py snapshot/2026-05-03/files.json > urls.txt

import json
import sys

def to_cdn_url(repo_id: str, path: str) -> str:
    # Public CDN URL — no Authorization header required
    return f"https://huggingface.co/datasets/{repo_id}/resolve/main/{path}"

def main():
    if len(sys.argv) < 2:
        print("Usage: python3 bin/cdn_urls.py <snapshot.json>")
        sys.exit(1)

    with open(sys.argv[1]) as f:
        data = json.load(f)

    repo = data["repo"]
    for path in data["files"]:
        print(to_cdn_url(repo, path))

if __name__ == "__main__":
    main()
```

Make executable:
```bash
chmod +x bin/cdn_urls.py
```

### 4) `training/cdn_loader.py` — zero-API dataloader example

```python
# training/cdn_loader.py
# Example: Lightning training using CDN URLs only (no HF API during training)

import json
import pyarrow.parquet as pq
import requests
from io import BytesIO
from torch.utils.data import IterableDataset

class CDNParquetDataset(IterableDataset):
    """
    Load parquet shards via CDN URLs (no HF API auth/rate-limit).
    Provide a snapshot files.json produced by bin/snapshot.sh.
    """

    def __init__(self, snapshot_path, repo_id, columns=("prompt", "response")):
        super().__init__()
        with open(snapshot_path) as f:
            data = json.load(f)
        self.repo = repo_id or data["repo"]
        self.files = [f for f in data["files"] if f.endswith(".parquet")]
        self.columns = columns

    @staticmethod
    def _cdn_url(repo, path):
        return f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"

    def __iter__(self):
        for path in self.files:
            url = self._cdn_url(self.repo, path)
            resp = requests.get(url, timeout=30)
            resp.raise_for_status()
            table = pq.read_table(BytesIO(resp.content), columns=self.columns)
            for batch in table.to_batches():
                for row in zip(*[batch.column(c).to_pylist() for c in self.columns]):
                    yield {"prompt": row[0], "response": row[1]}
```

Usage in Lightning `LightningDataModule`:
```python
# In your LightningDataModule
def train_dataloader(self):
    dataset = CDNParquetDataset("snapshot/2026-05-03/files.json", repo_id="axentx/surrogate-1-training-pairs")
    return DataLoader(dataset, batch_size=8, num_workers=4)
```
