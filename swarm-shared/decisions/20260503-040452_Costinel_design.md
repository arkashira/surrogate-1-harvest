# Costinel / design

## Final Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Scope**: Add a lightweight, non-blocking Top-Hub Signal Panel to Costinel frontend that surfaces the most-connected hub (e.g., "MOC") using CDN-first data baked at build/deploy time (zero HF API calls at runtime). Uses the `#knowledge-rag #graph #hub` pattern and CDN bypass to avoid rate limits.

---

### High-value incremental improvement
- **What**: A sticky top panel (desktop + responsive mobile) that shows the current top hub + 3 related signals with one-click drilldown, plus a concise insight/summary.
- **Why**: Immediate governance context for cloud cost decisions (Sense + Signal) without runtime API calls or blocking UI.
- **Value**: Faster decisions, audit trail for signals, zero runtime HF API dependency, <2h to ship.

---

### Data strategy (CDN-first, HF API only at build)
- Build/deploy step (run on Mac/CI) calls HF API **once** to list and fetch the latest top-hub JSON from the knowledge-rag repo.
- Save to `public/data/top-hub.json` (committed or uploaded to CDN).
- Frontend fetches `/data/top-hub.json` (CDN) — no Authorization header, no rate-limit risk.
- Fallback: bundle a default `top-hub.json` in repo so UI works offline.

---

### File changes (concrete)

#### a) Add build-time script (mac/CI) — `scripts/fetch-top-hub.sh`
```bash
#!/usr/bin/env bash
set -euo pipefail

# Config
REPO="datasets/axentx/knowledge-rag"
OUT_DIR="public/data"
OUT_FILE="${OUT_DIR}/top-hub.json"
DATE=$(date +%Y-%m-%d)
CDN_URL="https://huggingface.co/${REPO}/resolve/main/top-hub-${DATE}.json"

mkdir -p "${OUT_DIR}"

# Fetch latest top-hub JSON from CDN (bypass HF API)
if curl -fsSL --retry 3 --max-time 10 "${CDN_URL}" -o "${OUT_FILE}.tmp"; then
  # Validate minimal shape
  if jq -e '.hub and .signals' "${OUT_FILE}.tmp" > /dev/null 2>&1; then
    mv "${OUT_FILE}.tmp" "${OUT_FILE}"
    echo "✅ Updated ${OUT_FILE}"
  else
    echo "⚠️ Invalid payload, keeping existing"
    rm -f "${OUT_FILE}.tmp"
  fi
else
  echo "⚠️ CDN fetch failed, using bundled fallback"
  rm -f "${OUT_FILE}.tmp"
fi
```
Make executable:
```bash
chmod +x scripts/fetch-top-hub.sh
```

#### c) Frontend component — `src/components/TopHubSignalPanel.tsx`
```tsx
import { useEffect, useState } from 'react';
import './TopHubSignalPanel.css';

interface Signal {
  id: string;
  title: string;
  impact: 'low' | 'medium' | 'high';
  action: string;
  context: string;
}

interface TopHubData {
  hub: string;
  score: number;
  updatedAt: string;
  insight: string;
  signals: Signal[];
}

const FALLBACK_DATA: TopHubData = {
  hub: 'MOC',
  score: 0,
  updatedAt: new Date().toISOString(),
  insight: 'Loading signals...',
  signals: []
};

export default function TopHubSignalPanel() {
  const [data, setData] = useState<TopHubData>(FALLBACK_DATA);
  const [open, setOpen] = useState(false);
  const [acks, setAcks] = useState<Set<string>>(() => {
    try {
      return new Set(JSON.parse(localStorage.getItem('signal-acks') || '[]'));
    } catch {
      return new Set();
    }
  });

  useEffect(() => {
    fetch('/data/top-hub.json', { cache: 'no-cache' })
      .then((r) => {
        if (!r.ok) throw new Error('Fetch failed');
        return r.json();
      })
      .then((json) => {
        setData({
          hub: json.hub || FALLBACK_DATA.hub,
          score: Number(json.score || 0),
          updatedAt: json.updatedAt || FALLBACK_DATA.updatedAt,
          insight: json.insight || FALLBACK_DATA.insight,
          signals: Array.isArray(json.signals) ? json.signals.slice(0, 3) : FALLBACK_DATA.signals
        });
      })
      .catch(() => {
        // Silent fail-safe: keep fallback/default
      });
  }, []);

  const handleAck = (signalId: string) => {
    const next = new Set(acks).add(signalId);
    setAcks(next);
    try {
      localStorage.setItem('signal-acks', JSON.stringify(Array.from(next)));
    } catch {}
  };

  const impactColor = (imp: Signal['impact']) => {
    switch (imp) {
      case 'high': return 'var(--impact-high, #ef4444)';
      case 'medium': return 'var(--impact-medium, #f59e0b)';
      default: return 'var(--impact-low, #10b981)';
    }
  };

  return (
    <div className="top-hub-panel">
      {/* Sticky bar */}
      <div className="top-hub-bar" onClick={() => setOpen(!open)} role="button" tabIndex={0} aria-label="Top hub signals">
        <span className="hub-badge">{data.hub}</span>
        <span className="hub-score">{data.score > 0 ? data.score.toFixed(2) : '—'}</span>
        <span className="hub-insight">{data.insight}</span>
      </div>

      {/* Drawer */}
      {open && (
        <div className="top-hub-drawer" role="dialog" aria-label="Signal details">
          <div className="drawer-header">
            <strong>{data.hub}</strong>
            <button className="close-btn" onClick={() => setOpen(false)} aria-label="Close">×</button>
          </div>
          <p className="updated">Updated: {new Date(data.updatedAt).toLocaleString()}</p>
          <ul className="signals-list">
            {data.signals.map((s) => (
              <li key={s.id} className="signal-row">
                <div>
                  <span className="signal-title">{s.title}</span>
                  <span className="signal-context">{s.context}</span>
                </div>
                <div className="signal-meta">
                  <span className="impact" style={{ color: impactColor(s.impact) }}>{s.impact}</span>
                  {!acks.has(s.id) && (
                    <button className="ack-btn" onClick={() => handleAck(s.id)}>Acknowledge</button>
                  )}
                </div>
              </li>
            ))}
          </ul>
        </div>
      )}
    </div>
  );
}
```
