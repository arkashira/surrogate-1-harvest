# workio / backend

## 1. Diagnosis

- Missing **idempotent webhook processing**: LINE retries on 5xx/timeouts; current handler can create duplicate clock-in/out events or leave requests on retry.
- No **request deduplication key** stored (e.g., `line_delivery_id` or `X-Line-Signature`-derived nonce) in DB; duplicate detection is impossible.
- Webhook handler likely performs **side effects before validation** (DB writes) then returns 200 even if business validation fails, causing retry storms.
- Missing **atomic upsert for clock events** per user+date+session; race conditions on rapid clock-in/out can create overlapping or duplicate rows.
- No **audit trail for webhook source + integrity** (store verification result and delivery id) for debugging and replay analysis.

## 2. Proposed change

File scope:
- `server/src/middleware/lineWebhook.ts` — add signature verification + idempotency key extraction.
- `server/src/routes/webhook/line.ts` — enforce idempotency and atomic upsert for clock events; store delivery metadata.
- `server/src/db/schema.sql` — add unique constraint/index for idempotency key and compound unique for user+timestamp-session to prevent duplicates.

### 3.2 Middleware: signature verification + idempotency (`server/src/middleware/lineWebhook.ts`)

```ts
import crypto from 'crypto';
import { Request, Response, NextFunction } from 'express';

const LINE_CHANNEL_SECRET = process.env.LINE_CHANNEL_SECRET || '';

function verifyLineSignature(rawBody: string, signature: string): boolean {
  if (!LINE_CHANNEL_SECRET) return false;
  const expected = crypto
    .createHmac('sha256', LINE_CHANNEL_SECRET)
    .update(rawBody, 'utf8')
    .digest('base64');
  return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
}

export function lineWebhookMiddleware(
  req: Request,
  res: Response,
  next: NextFunction
) {
  const signature = req.headers['x-line-signature'] as string;
  const rawBody = req.rawBody as string;

  if (!rawBody) {
    return res.status(400).json({ error: 'Missing raw body' });
  }

  const verified = verifyLineSignature(rawBody, signature || '');
  (req as any).lineVerified = verified;
  (req as any).lineSignature = signature;
  next();
}
```

> Note: ensure your body parser preserves raw body. Example setup in `server/src/app.ts` or `server.ts`:

```ts
import express from 'express';
import lineWebhookMiddleware from './middleware/lineWebhook';

const app = express();

// Preserve raw body for LINE signature verification
app.use('/webhook/line', (req, res, next) => {
  const chunks: Buffer[] = [];
  req.on('data', (chunk) => chunks.push(chunk));
  req.on('end', () => {
    (req as any).rawBody = Buffer.concat(chunks).toString('utf8');
    next();
  });
});
app.use('/webhook/line', lineWebhookMiddleware);
```

  return evt.message?.id || evt.source?.userId + '-' + evt.timestamp || uuidv4();
