# surrogate-1 / design

## Implementation Plan (≤2h)

**Highest-value improvement**: Replace fragile shell-based ingestion with a **manifest-driven, CDN-bypass Python worker** that eliminates HF API rate limits during training data loads and prevents mixed-schema CastErrors.

### Changes

1. **Add `bin/manifest.py`** — single Mac-side script that:
   - Uses HF API once (after rate-limit window) to `list_repo_tree` for a date folder.
   - Produces `manifest.json` with CDN URLs and expected slugs.
   - Embeds shard assignment logic (deterministic `hash(slug) % 16`).

2. **Replace `bin/dataset-enrich.sh` with `bin/worker.py`** — Python worker that:
   - Accepts `MANIFEST_PATH`, `SHARD_ID`, `OUTPUT_DIR`.
   - Downloads assigned files via **CDN bypass** (`resolve/main/...`, no auth).
   - Projects to `{prompt, response}` only at parse time (avoids pyarrow CastError).
   - Dedups via existing `lib/dedup.py` (central md5 store).
   - Writes `shard-<N>-<HHMMSS>.jsonl` to output.

3. **Update `.github/workflows/ingest.yml`** — workflow now:
   - Runs `python bin/manifest.py` once in a setup step (or reuses cached manifest).
   - Passes manifest + shard matrix to 16 parallel `worker.py` jobs.
   - Uses `actions/upload-artifact` to collect per-shard JSONL outputs.
   - Single upload step pushes concatenated result to HF dataset repo.

4. **Remove shell invocation risks** — no more Bash shebang/exec/cron issues; all workers are pure Python invoked via `python`.

### Code Snippets

#### `bin/manifest.py`
```python
#!/usr/bin/env python3
"""
Generate manifest for a date folder in axentx/surrogate-1-training-pairs.
Run on Mac (or CI) once per ingestion cycle.
"""
import json, os, sys, hashlib
from datetime import datetime, timezone
from huggingface_hub import HfApi

REPO = "datasets/axentx/surrogate-1-training-pairs"
DATE = datetime.now(timezone.utc).strftime("%Y-%m-%d")
OUT = sys.argv[1] if len(sys.argv) > 1 else f"manifest-{DATE}.json"

api = HfApi()
# Single API call: non-recursive per folder to avoid pagination explosion
folders = api.list_repo_tree(repo_id=REPO, path=DATE, recursive=False)

entries = []
for f in folders:
    if f.type != "file":
        continue
    slug = f.rfilename.replace(".parquet", "").replace(".jsonl", "")
    entries.append({
        "slug": slug,
        "cdn_url": f"https://huggingface.co/datasets/{REPO}/resolve/main/{DATE}/{f.rfilename}",
        "shard": int(hashlib.md5(slug.encode()).hexdigest(), 16) % 16,
    })

manifest = {
    "date": DATE,
    "generated_at": datetime.now(timezone.utc).isoformat(),
    "entries": entries,
}

with open(OUT, "w") as fh:
    json.dump(manifest, fh, indent=2)
print(f"Wrote {len(entries)} entries to {OUT}")
```

#### `bin/worker.py`
```python
#!/usr/bin/env python3
"""
CDN-bypass worker for surrogate-1 ingestion.
Usage:
  python bin/worker.py --manifest manifest.json --shard 3 --out ./out
"""
import argparse, json, os, sys, hashlib, requests, pyarrow as pa, pyarrow.parquet as pq
from pathlib import Path

sys.path.insert(0, str(Path(__file__).parent.parent))
from lib.dedup import DedupStore  # central md5 dedup store

def parse_file(file_path, tmp_dir):
    """Project to {prompt, response} only — avoid mixed-schema CastError."""
    try:
        table = pq.read_table(file_path, columns=["prompt", "response"])
    except (pa.ArrowException, KeyError):
        # fallback: try jsonl
        rows = []
        with open(file_path) as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                obj = json.loads(line)
                rows.append({"prompt": obj.get("prompt", ""), "response": obj.get("response", "")})
        table = pa.Table.from_pylist(rows)
    return table.to_pylist()

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--manifest", required=True)
    parser.add_argument("--shard", type=int, required=True)
    parser.add_argument("--out", required=True)
    args = parser.parse_args()

    with open(args.manifest) as f:
        manifest = json.load(f)

    entries = [e for e in manifest["entries"] if e["shard"] == args.shard]
    os.makedirs(args.out, exist_ok=True)
    dedup = DedupStore()

    out_path = Path(args.out) / f"shard-{args.shard}-{manifest['date']}.jsonl"
    written = 0

    with out_path.open("w") as out_f:
        for ent in entries:
            # CDN bypass: no Authorization header
            resp = requests.get(ent["cdn_url"], timeout=30)
            resp.raise_for_status()

            tmp = Path(args.out) / f"tmp_{hashlib.md5(ent['slug'].encode()).hexdigest()}"
            tmp.write_bytes(resp.content)

            try:
                pairs = parse_file(str(tmp), args.out)
                for p in pairs:
                    prompt = (p.get("prompt") or "").strip()
                    response = (p.get("response") or "").strip()
                    if not prompt or not response:
                        continue
                    md5 = hashlib.md5((prompt + "\n" + response).encode()).hexdigest()
                    if dedup.seen(md5):
                        continue
                    dedup.add(md5)
                    out_f.write(json.dumps({"prompt": prompt, "response": response}) + "\n")
                    written += 1
            finally:
                if tmp.exists():
                    tmp.unlink()

    print(f"Shard {args.shard}: wrote {written} pairs to {out_path}")

if __name__ == "__main__":
    main()
```
