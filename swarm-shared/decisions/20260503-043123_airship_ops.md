# airship / ops

## Final Implementation Plan  
**Goal:** Deterministic CDN-only file manifest + Lightning Studio lifecycle resilience for Surrogate-1 training — ship in <2h on macOS orchestration.

---

### 1) Deterministic CDN manifest generator (macOS)
- **Single API call per date folder** via `list_repo_tree(..., recursive=False)` to avoid pagination/rate limits.  
- **Filter to parquet only** and sort for determinism.  
- **Embed CDN base + per-file CDN URLs** so training workers require zero Hugging Face API calls.  
- **429 handling**: exponential backoff (3 retries) instead of fixed 360s; if all retries fail, load last-known-good manifest and continue (fail-soft).  
- **Cron-safe**: Bash shebang, `set -euo pipefail`, `SHELL=/bin/bash` in crontab, executable bit.

---

### 2) Training script: CDN-only, schema-hygienic, resilient
- **Replace `load_dataset(streaming=True)`** with CDN fetches using manifest.  
- **Schema hygiene**: project to `{prompt, response}` at parse time; skip malformed/mixed-schema rows/files without crashing.  
- **Shard-aware downloads** with streaming decompression where possible; validate parquet schema before full ingest.  
- **Lightning integration**: use `Fabric` and deterministic DataLoader; avoid per-worker API calls.

---

### 3) Lightning Studio lifecycle resilience
- **Reuse running studios** to save quota.  
- **Before `.run()`**, ensure studio is `Running`; if stopped/idle, restart on `Machine.L40S` (or H200 where available).  
- **Retry wrapper** (max 3) with 60s backoff; persist minimal checkpoint (epoch/step) to HF dataset shard or local file to avoid recompute on restart.  
- **Idempotent start**: `create_ok=True`, no duplicate studios.

---

### `airship/surrogate/manifest.py`
```python
#!/usr/bin/env python3
"""
Generate deterministic CDN manifest for one date folder.
Usage:
  python manifest.py --repo axentx/surrogate-1-mirror --date 2026-05-03 --out data/manifest.json
"""
import argparse
import json
import os
import time
from pathlib import Path
from huggingface_hub import HfApi

HF_TOKEN = os.getenv("HF_TOKEN")
api = HfApi(token=HF_TOKEN)

def list_parquet_files(repo: str, date: str):
    folder = f"batches/mirror-merged/{date}"
    retries, delay = 3, 10
    for attempt in range(retries):
        try:
            items = api.list_repo_tree(repo_id=repo, path=folder, recursive=False)
            files = sorted(
                it.rfilename for it in items
                if it.rfilename.endswith(".parquet") and not it.rfilename.endswith("/")
            )
            return files
        except Exception as e:
            if attempt == retries - 1:
                raise
            if "429" in str(e):
                wait = delay * (2 ** attempt)
                print(f"HF 429 — retry {attempt+1}/{retries} in {wait}s")
                time.sleep(wait)
            else:
                raise
    return []

def build_manifest(repo: str, date: str, out_path: Path):
    files = list_parquet_files(repo, date)
    base = f"https://huggingface.co/datasets/{repo}/resolve/main"
    manifest = {
        "repo": repo,
        "date": date,
        "files": [
            {
                "path": f"batches/mirror-merged/{date}/{f}",
                "cdn_url": f"{base}/batches/mirror-merged/{date}/{f}",
            }
            for f in files
        ],
        "total": len(files),
    }
    out_path.parent.mkdir(parents=True, exist_ok=True)
    with open(out_path, "w") as f:
        json.dump(manifest, f, indent=2)
    print(f"Manifest written to {out_path} ({len(files)} files)")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--repo", default="axentx/surrogate-1-mirror")
    parser.add_argument("--date", default="2026-05-03")
    parser.add_argument("--out", default="data/manifest.json")
    args = parser.parse_args()
    build_manifest(args.repo, args.date, Path(args.out))
```

def load_from_cdn(manifest_path: Path, max_files=None):
    with open(manifest_path) as f:
        manifest = json.load(f)

    samples = []
    files = manifest["files"]
    if max_files is not None:
        files = files[:max_files]

    for entry in files:
        url = entry["cdn_url"]
        try:
            resp = requests.get(url, timeout=60, stream=True)
            resp.raise_for_status()
            # Stream parquet without full download to memory when possible
            with pq.ParquetFile(pq.ParquetFile(resp.raw)) as pf:
                # Read in row-group chunks to limit memory
                for rg_i in range(pf.num_row_groups):
                    rg = pf.read_row_group(rg_i)

### `airship/surrogate/studio_lifecycle.py`
```python
import time
from lightning import Studio, Teamspace, Machine, L40S

def get_or_start_studio(name: str) -> Studio:
    for s in Teamspace.studios:
        if s.name == name and s.status == "Running":
            print(f"Reusing running studio: {name}")
            return s
    print(f"Starting studio: {name}")
    return Studio(
        name=name,
        machine=Machine.L40S,
        cloud="lightning-lambda-prod",
        create_ok=True,
    )

def run_with_resilience(studio_name: str, target, max_retries=3):
    studio = get_or_start_studio(studio_name)
    for attempt in range(max_retries):
        try:
            if studio.status != "Running":
                print(f"Studio not running (status={studio.status}), restarting...")
                studio = get_or_start_studio(studio_name)
            target.run(studio=studio)
            return
        except Exception as e:
            print(f"Attempt {attempt+1}/{max_retries} failed: {e}")
            if attempt == max_retries - 1:
                raise
            time.sleep(60)
```
