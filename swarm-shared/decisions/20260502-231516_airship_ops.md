## 1) CDN-only manifest generation (orchestrator)
- **Single API call per date folder** using `list_repo_tree(..., recursive=False)` saved to `manifests/YYYY-MM-DD.json`.
- **Embed manifest in training/ingest scripts** so runtime uses only CDN URLs (`resolve/main/...`).  
- **Validation**: no `load_dataset(streaming=True)` on heterogeneous repos; no HF API calls during data loading.

**Code** (`scripts/airship-discover.sh`):
```bash
#!/usr/bin/env bash
set -euo pipefail

# Usage: ./airship-discover.sh <repo> <date_folder>
REPO="${1:-datasets/example}"
DATE="${2:-$(date +%Y-%m-%d)}"
MANIFEST_DIR="manifests"
MANIFEST="${MANIFEST_DIR}/${DATE}.json"

mkdir -p "${MANIFEST_DIR}"

echo "[$(date)] Listing ${REPO}@${DATE} (non-recursive)..."
python3 -c "
import os, json, time, sys
from huggingface_hub import HfApi
api = HfApi()
try:
    tree = api.list_repo_tree(repo_id='${REPO}', path='${DATE}', recursive=False)
    files = [t.path for t in tree if t.type == 'file']
except Exception as e:
    print('Error:', e)
    time.sleep(360)
    tree = api.list_repo_tree(repo_id='${REPO}', path='${DATE}', recursive=False)
    files = [t.path for t in tree if t.type == 'file']
with open('${MANIFEST}', 'w') as f:
    json.dump({'repo':'${REPO}','date':'${DATE}','files':files}, f, indent=2)
print(f'Wrote {len(files)} files to ${MANIFEST}')
"

echo "[$(date)] Manifest ready: ${MANIFEST}"
```

## 2) CDN-only ingestion with surrogate-1 schema hardening
- **Download via `hf_hub_download` (cached) or raw CDN** → parse → **project to `{prompt, response}` only**.
- **Filename pattern**: `batches/mirror-merged/{date}/{slug}.parquet` (no extra `source`/`ts` columns).  
- **Attribution via filename/path** (traceable, schema-clean).  
- **Handle schema variance safely**: if `prompt`/`response` missing, skip file.

**Code** (`scripts/ingest-to-surrogate.py`):
```python
#!/usr/bin/env python3
"""
Ingest files listed in manifests/YYYY-MM-DD.json to surrogate-1 schema.
Uses CDN-only downloads; projects to {prompt, response} only.
"""
import json, os, sys, pyarrow as pa, pyarrow.parquet as pq, uuid
from pathlib import Path
from huggingface_hub import hf_hub_download

MANIFEST = sys.argv[1]  # manifests/2026-05-02.json
OUT_DIR = Path("batches/mirror-merged")
OUT_DIR.mkdir(parents=True, exist_ok=True)

with open(MANIFEST) as f:
    cfg = json.load(f)

repo = cfg["repo"]
date = cfg["date"]
rows = []

for rel_path in cfg["files"]:
    local_path = hf_hub_download(repo_id=repo, filename=rel_path, repo_type="dataset")
    try:
        table = pq.read_table(local_path, columns=["prompt", "response"])
    except Exception:
        table = pq.read_table(local_path)
        if "prompt" not in table.column_names or "response" not in table.column_names:
            print(f"Skipping {rel_path}: missing prompt/response")
            continue
        table = table.select(["prompt", "response"])

    df = table.to_pandas()
    for _, r in df.iterrows():
        rows.append({
            "id": str(uuid.uuid4()),
            "prompt": str(r["prompt"]),
            "response": str(r["response"]),
        })

if rows:
    out_table = pa.Table.from_pylist(rows)
    slug = f"{date}-{uuid.uuid4().hex[:8]}"
    out_path = OUT_DIR / f"{slug}.parquet"
    pq.write_table(out_table, out_path)
    print(f"Wrote {len(rows)} rows to {out_path}")
else:
    print("No rows to write.")
```

## 3) Lightning Studio reuse guard
- **Before `Studio(create_ok=True)`**, list `Teamspace.studios` and reuse any **Running** one with matching name.
- If **stopped**, restart with `target.start(machine=Machine.L40S)` (avoid idle-timeout death).  
- **Deterministic fallback**: create new with same name if needed.

**Code** (`scripts/reuse-lightning-studio.py`):
```python
#!/usr/bin/env python3
"""
Reuse a running Lightning Studio or start a new one deterministically.
Prevents quota waste and idle-timeout deaths.
"""
from lightning_sdk import Teamspace, Studio, Machine
import os, time

TEAMSPACE = os.getenv("LIGHTNING_TEAMSPACE", "default")
STUDIO_NAME = os.getenv("STUDIO_NAME", "surrogate-train")
MACHINE = Machine.L40S  # fallback; H200 only in lightning-lambda-prod

teamspace = Teamspace(name=TEAMSPACE)
running = None
for s in teamspace.studios:
    if s.name == STUDIO_NAME:
        running = s
        break

if running and running.status == "running":
    print(f"Reusing running studio: {running.name} ({running.id})")
    studio = running
else:
    if running and running.status == "stopped":
        print(f"Restarting stopped studio: {running.name}")
        running.start(machine=MACHINE)
        studio = running
    else:
        print(f"Creating new studio: {STUDIO_NAME}")
        studio = teamspace.studios.create(
            name=STUDIO_NAME,
            machine=MACHINE,
            create_ok=True,
        )

if studio.status != "running":
    print("Waiting for studio to be running...")
    studio.wait_until_running()
print(f"Studio ready: {studio.web_url}")
```
