# Costinel / ops

## Decision
Ship a CDN-first Top-Hub Signal Panel (non-blocking, zero HF API at runtime) that surfaces the most-connected hub (e.g., "MOC") with actionable insights. Estimated effort: **~90 minutes**.

---

## Implementation Plan (<2h)

### 1) Build-time: generate top-hub payload (20m)
- Add script `scripts/build-top-hub.js` (Node) that:
  - Uses `list_repo_tree` (non-recursive) on the knowledge-rag folder (single API call).
  - Picks the most-connected hub by filename convention or lightweight metadata (e.g., `MOC.md` or `hub-MOC.json`).
  - Outputs `public/top-hub.json`:
    ```json
    {
      "hub": "MOC",
      "title": "MOC — Master Operations Context",
      "summary": "High-level ops playbook for Costinel governance.",
      "links": [
        { "label": "Dashboard", "href": "/docs/moc/dashboard" },
        { "label": "Runbook", "href": "/docs/moc/runbook" }
      ],
      "updated": "2026-05-03"
    }
    ```
- Add to `package.json` build step: `"build:top-hub": "node scripts/build-top-hub.js"`.

### 2) Frontend: non-blocking signal panel (30m)
- Create `components/TopHubSignalPanel.tsx`:
  - Client-only (dynamic import or `useEffect`) to avoid SSR hydration mismatch.
  - Fetch `/top-hub.json` with `priority={false}` and `cache: 'no-store'` (or static if baked at build).
  - Skeleton fallback while loading; silent fail if 404/timeout.
  - Visual: compact card with hub name, short summary, and 2 quick links.
- Add to dashboard layout (main cost view) near top summary row.

### 3) CDN-first data strategy (15m)
- Ensure `public/top-hub.json` is committed to repo (or generated in CI) so CDN serves it without HF API.
- If repo is private, use GitHub raw CDN (`https://raw.githubusercontent.com/.../main/public/top-hub.json`) — no auth required for public repos; for private, rely on baked asset.

### 4) Ops/cron hygiene (10m)
- If build script is invoked by cron or CI, ensure:
  - Shebang `#!/usr/bin/env node` in JS script.
  - Executable bit set if using a wrapper sh.
  - `SHELL=/bin/bash` in crontab (per pattern).

### 5) Testing & rollback (15m)
- Verify panel renders with baked JSON.
- Simulate missing JSON → panel should hide gracefully.
- Rollback: remove component import and delete JSON.

## Code Snippets

### `scripts/build-top-hub.js`
```js
#!/usr/bin/env node
/**
 * Build-time script to generate public/top-hub.json (CDN-first).
 * Uses HF API minimally (single list_repo_tree call) and falls back to local
 * convention if API unavailable/rate-limited.
 */
import fs from 'fs';
import path from 'path';
import { HfApi } from '@huggingface/hub';

const OUT_PATH = path.resolve('public/top-hub.json');
const HUB_REPO = process.env.HF_HUB_REPO || 'AXENTX/Costinel-knowledge';
const HUB_FOLDER = process.env.HF_HUB_FOLDER || 'hubs';

async function buildFromHF() {
  try {
    const api = new HfApi();
    // Single non-recursive call (avoids pagination/rate issues)
    const tree = await api.listRepoTree(HUB_REPO, HUB_FOLDER, { recursive: false });
    // Pick most-connected hub: convention "hub-<name>.md" or highest weight file
    const hubFile = tree
      .filter((f) => f.type === 'file' && f.path.startsWith(`hubs/hub-`) && f.path.endsWith('.md'))
      .sort((a, b) => (b.size || 0) - (a.size || 0))[0];

    if (!hubFile) throw new Error('No hub file found');

    const name = path.basename(hubFile.path, '.md').replace('hub-', '');
    return {
      hub: name.toUpperCase(),
      title: `${name.toUpperCase()} — Master Operations Context`,
      summary: 'High-level ops playbook for Costinel governance.',
      links: [
        { label: 'Dashboard', href: `/docs/${name}/dashboard` },
        { label: 'Runbook', href: `/docs/${name}/runbook` },
      ],
      updated: new Date().toISOString().split('T')[0],
    };
  } catch (err) {
    console.warn('HF API unavailable, using fallback hub:', err.message);
    // Fallback: local convention baked into repo
    return {
      hub: 'MOC',
      title: 'MOC — Master Operations Context',
      summary: 'High-level ops playbook for Costinel governance.',
      links: [
        { label: 'Dashboard', href: '/docs/moc/dashboard' },
        { label: 'Runbook', href: '/docs/moc/runbook' },
      ],
      updated: new Date().toISOString().split('T')[0],
    };
  }
}

async function main() {
  const payload = await buildFromHF();
  fs.mkdirSync(path.dirname(OUT_PATH), { recursive: true });
  fs.writeFileSync(OUT_PATH, JSON.stringify(payload, null, 2));
  console.log('Generated', OUT_PATH);
}

main().catch((err) => {
  console.error(err);
  process.exit(1);
});
```

### `components/TopHubSignalPanel.tsx`
```tsx
'use client';

import { useEffect, useState } from 'react';

interface HubLink {
  label: string;
  href: string;
}

interface TopHubData {
  hub: string;
  title: string;
  summary: string;
  links: HubLink[];
  updated: string;
}

export default function TopHubSignalPanel() {
  const [data, setData] = useState<TopHubData | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    let mounted = true;
    fetch('/top-hub.json', { cache: 'no-store' })
      .then((res) => {
        if (!res.ok) throw new Error('Failed to load top-hub');
        return res.json();
      })
      .then((json) => {
        if (mounted) setData(json);
      })
      .catch(() => {
        // Silent fail — panel will not render
      })
      .finally(() => {
        if (mounted) setLoading(false);
      });

    return () => {
      mounted = false;
    };
  }, []);

  if (loading || !data) return null;

  return (
    <section aria-label={`Top hub: ${data.hub}`} className="rounded-lg border bg-card p-4 shadow-sm">
      <div className="flex items-start justify-between gap-4">
        <div>
          <h3 className="text-sm font-semibold text-foreground">{data.title}</h3>
          <p className="mt-1 text-sm text-muted-foreground">{data.summary}</p>
          <div className="mt-3 flex gap-3">
            {data.links.map((link) => (
              <a
                key={link.href}
                href={link.href}
                className="text-sm text-primary hover:underline"
              >
                {link.label}
              </a>
            ))}
          </div>
        </div>
        <div className="text-right">
          <span className="inline-flex items-center rounded-full bg-muted px-2 py-1 text-xs font-medium text-muted-foreground">
            {data.hub}
          </span>
          <p className="mt-1 text-xs text-muted-foreground">Updated {data.updated}</p>
        </div>
      </div>
    </section>
  );
}
```
