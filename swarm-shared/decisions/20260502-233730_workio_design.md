**Synthesized Solution**

The synthesized solution combines the strongest insights from the three proposals, resolving contradictions in favor of correctness and concrete actionability.

**Diagnosis**

- **Missing idempotency key on punch records** → LINE webhook redeliveries and concurrent deliveries create duplicate clock-in/out rows.
- **Non-atomic read-then-insert for punch state** → races between concurrent webhooks produce multiple open punches or lost state.
- **No transactional boundary around punch state updates** → concurrent updates can result in inconsistent state.
- **Lack of optimistic concurrency control** → concurrent updates can overwrite each other's changes.
- **No idempotency guard on LINE webhook punches** → redeliveries and concurrent deliveries create duplicate clock-in/out rows.

**Proposed Change**

Implement idempotency key and optimistic concurrency control for punch records.

**Implementation**

1. **Add idempotency key to punch model**:
```diff
// server/src/models/punch.ts
import { Model, Table, Column, PrimaryKey, AutoIncrement } from 'sequelize-typescript';

@Table({
  tableName: 'punches',
  timestamps: true,
  paranoid: true,
})
export class Punch extends Model<unch> {
  @PrimaryKey
  @AutoIncrement
  @Column
  id: number;

  @Column
  idempotencyKey: string;

  @Column
  version: number;

  @Column
  employeeId: number;

  @Column
  punchIn: Date;

  @Column
  punchOut: Date;
}
```
2. **Update punch creation to use idempotency key**:
```diff
// server/src/controllers/punch.ts
import { Punch } from '../models/punch';

export async function createPunch(data: any) {
  const existingPunch = await Punch.findOne({ idempotencyKey: data.idempotencyKey });
  if (existingPunch) {
    // Update existing punch
    await existingPunch.update(data);
  } else {
    // Create new punch
    const punch = await Punch.create(data);
  }
  return punch;
}
```
3. **Implement optimistic concurrency control**:
```diff
// server/src/controllers/punch.ts
import { Punch } from '../models/punch';

export async function updatePunch(id: number, data: any) {
  const punch = await Punch.findByPk(id);
  if (!punch) {
    throw new Error('Punch not found');
  }

  const currentVersion = punch.version;
  const updatedPunch = await punch.update(data, {
    where: {
      id,
      version: currentVersion,
    },
    returning: true,
  });

  if (updatedPunch[0].version !== currentVersion) {
    throw new Error('Concurrent update detected');
  }

  return updatedPunch[0];
}
```
4. **Implement transactional boundary**:
```diff
// server/src/controllers/punch.ts
import { Punch } from '../models/punch';

export async function createPunch(data: any) {
  const session = await db.transaction({
    isolationLevel: 'SERIALIZABLE',
  });

  try {
    const punch = await Punch.create(data, { session });
    await session.commit();
    return punch;
  } catch (error) {
    await session.rollback();
    throw error;
  }
}
```
