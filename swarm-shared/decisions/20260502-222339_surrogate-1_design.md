## Implementation Plan (≤2h)

**Highest-value improvement**: Add deterministic pre-flight file listing + CDN-only ingestion to eliminate HF API 429s during training and make shard workers resilient.

### What we’ll change
1. Add `bin/list-files.sh` — single Mac/CI call that lists one date folder via `list_repo_tree(recursive=False)` and writes `file-list.json`.
2. Update `bin/dataset-enrich.sh` to accept an optional file-list JSON; if provided, workers stream from CDN URLs only (no `load_dataset`, no per-file API calls).
3. Add `lib/cdn_stream.py` — lightweight CDN fetcher with retry/back-off and `hf_hub_download` fallback for private repos.
4. Update `.github/workflows/ingest.yml` to generate the file-list once and pass it to all 16 matrix shards via `env.FILE_LIST`.

### Why this wins
- Eliminates per-file API calls during ingestion → no 429s from `list_repo_files` or per-file metadata.
- Keeps HF API usage to a single deterministic pre-flight call.
- Enables 16 shards to run in parallel with zero contention on HF API during data loading.
- Fits in <2h: ~1 script + ~1 small lib + CI tweak.

### 1. `bin/list-files.sh` (new)
```bash
#!/usr/bin/env bash
# Generate deterministic file list for a date folder in axentx/surrogate-1-training-pairs
# Usage:
#   FILE_LIST=file-list.json ./bin/list-files.sh --date 2026-05-02
set -euo pipefail

HF_REPO="${HF_REPO:-axentx/surrogate-1-training-pairs}"
HF_TOKEN="${HF_TOKEN:-}"
OUT="${FILE_LIST:-file-list.json}"
DATE=""

while [[ $# -gt 0 ]]; do
  case "$1" in
    --date) DATE="$2"; shift 2 ;;
    *) echo "Unknown option: $1"; exit 1 ;;
  esac
done

if [[ -z "$DATE" ]]; then
  echo "Missing --date YYYY-MM-DD"
  exit 1
fi

if ! [[ "$DATE" =~ ^[0-9]{4}-[0-9]{2}-[0-9]{2}$ ]]; then
  echo "Invalid date format. Use YYYY-MM-DD"
  exit 1
fi

FOLDER="batches/public-merged/${DATE}"
echo "Listing ${HF_REPO}/${FOLDER} ..."

# Use Python one-liner with huggingface_hub to list recursively and emit JSON
python3 - <<PY
import json, os, sys
from datetime import datetime, timezone
from huggingface_hub import list_repo_tree

repo = os.getenv("HF_REPO", "$HF_REPO")
folder = "$FOLDER"
out_path = "$OUT"

tree = list_repo_tree(repo_id=repo, path=folder, recursive=True)
files = [
    {
        "path": item.rfilename,
        "cdn_url": f"https://huggingface.co/datasets/{repo}/resolve/main/{item.rfilename}"
    }
    for item in tree
    if item.type == "file"
]
files.sort(key=lambda x: x["path"])

result = {
    "repo": repo,
    "date": "$DATE",
    "folder": folder,
    "generated_at": datetime.now(timezone.utc).isoformat(),
    "count": len(files),
    "files": files,
}

os.makedirs(os.path.dirname(out_path) if os.path.dirname(out_path) else ".", exist_ok=True)
with open(out_path, "w", encoding="utf-8") as f:
    json.dump(result, f, indent=2)

print(f"Wrote {len(files)} files to {out_path}")
PY
```

### 2. `lib/cdn_stream.py` (new)
```python
import time
from pathlib import Path
from typing import Optional

import requests
from tqdm import tqdm

DEFAULT_TIMEOUT = 30
MAX_RETRIES = 5
BACKOFF_FACTOR = 1.5

def stream_cdn_url(url: str, dest: Path, chunk_size: int = 8192, timeout: int = DEFAULT_TIMEOUT) -> bool:
    """
    Stream a public HF dataset file via CDN (no Authorization header).
    Returns True on success.
    """
    dest.parent.mkdir(parents=True, exist_ok=True)
    retries = 0
    delay = 1.0

    while retries <= MAX_RETRIES:
        try:
            with requests.get(url, stream=True, timeout=timeout) as r:
                r.raise_for_status()
                total = int(r.headers.get("content-length", 0))
                with open(dest, "wb") as f, tqdm(
                    desc=dest.name,
                    total=total,
                    unit="B",
                    unit_scale=True,
                    leave=False,
                ) as pbar:
                    for chunk in r.iter_content(chunk_size=chunk_size):
                        f.write(chunk)
                        pbar.update(len(chunk))
            return True
        except Exception as exc:
            retries += 1
            if retries > MAX_RETRIES:
                print(f"Failed to download {url}: {exc}")
                return False
            print(f"Retry {retries}/{MAX_RETRIES} for {url} in {delay:.1f}s: {exc}")
            time.sleep(delay)
            delay *= BACKOFF_FACTOR
    return False


def hf_hub_download_fallback(repo_id: str, filename: str, dest: Path, **kwargs) -> Optional[Path]:
    """
    Fallback to huggingface_hub.hf_hub_download for private repos or CDN failures.
    Returns local path on success, else None.
    """
    try:
        from huggingface_hub import hf_hub_download
        return Path(hf_hub_download(repo_id=repo_id, filename=filename, **kwargs))
    except Exception as exc:
        print(f"hf_hub_download fallback failed for {repo_id}/{filename}: {exc}")
        return None
```
