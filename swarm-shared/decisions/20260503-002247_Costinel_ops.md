# Costinel / ops

## Final Implementation Plan — Top-Hub Signal Card (Costinel Ops)

**Scope**: A read-only, production-ready React card that surfaces the most-connected knowledge hub (e.g., “MOC”) with contextual cost-governance insights. Uses CDN-bypass for live data and embedded mock fallback for dev/demo.

**Value**: Immediate visibility into the highest-leverage knowledge node for cost decisions.  
**Time budget**: ≤2h (deliverable: component + hook + Storybook story).  
**Deliverables**:
- `TopHubSignalCard.tsx`
- `useTopHubSignal.ts` (real + mock)
- `HubNodePill.tsx` (reusable)
- Storybook story
- CDN-bypass edge endpoint (optional but recommended)

---

### Architecture (resolved)
- **Data strategy**: Prefer CDN-bypass static JSON from HF datasets for production (no runtime API keys, low latency). Use mock JSON for local dev and Storybook.
- **Runtime path**:  
  `TopHubSignalCard` → `useTopHubSignal` → `/api/knowledge/hubs` (edge) → HF CDN direct fetch (bypass).  
- **Fallback**: Embedded static mock when CDN fails or in dev without network.
- **Styling**: Reuse Costinel design tokens (shadcn/ui + Tailwind).
- **Location**: `src/components/ops/`

### 1. Edge API: `/api/knowledge/hubs`
Production path uses CDN-bypass with static file list and zero runtime auth. Includes robust fallback.

```typescript
// src/pages/api/knowledge/hubs.ts
import type { NextApiRequest, NextApiResponse } from 'next';

const HUB_FILE_LIST = {
  date: '2026-04-27',
  files: [
    'hubs/moc.top.json',
    'hubs/aws-ri.top.json',
    'hubs/gcp-commits.top.json',
  ],
};

const BASE_URL = 'https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main';

export default async function handler(
  req: NextApiRequest,
  res: NextApiResponse
) {
  try {
    const resFetch = await fetch(
      `${BASE_URL}/hubs/moc.top.json?cacheBust=${Date.now()}`,
      { headers: { Accept: 'application/json' } }
    );

    if (!resFetch.ok) throw new Error(`HF CDN error: ${resFetch.status}`);

    const hubData = await resFetch.json();

    res.status(200).json({
      hub: hubData,
      fileList: HUB_FILE_LIST,
      source: 'cdn-bypass',
      cached: false,
    });
  } catch (err) {
    console.error('Hub fetch error:', err);
    res.status(200).json({
      hub: {
        id: 'moc',
        name: 'MOC',
        connections: 1247,
        insights: [
          'Reserved Instance coverage gaps',
          'Cross-account cost allocation',
          'Idle resource patterns in linked accounts',
        ],
        lastUpdated: '2026-04-27T10:00:00Z',
      },
      fileList: HUB_FILE_LIST,
      source: 'fallback',
      cached: true,
    });
  }
}
```

### 2. Data Hook: `useTopHubSignal`
Supports three modes: `live` (CDN-bypass via edge), `mock` (explicit mock), and `auto` (live in prod, mock in dev/Storybook).

```typescript
// src/components/ops/useTopHubSignal.ts
import useSWR from 'swr';

export type HubData = {
  id: string;
  name: string;
  connections: number;
  insights: string[];
  lastUpdated?: string;
};

export type HubPayload = {
  hub: HubData;
  fileList?: { date: string; files: string[] };
  source: 'cdn-bypass' | 'fallback' | 'mock';
  cached?: boolean;
};

const fetcher = (url: string) => fetch(url).then((r) => r.json());

const MOCK_HUB: HubPayload = {
  source: 'mock',
  cached: false,
  hub: {
    id: 'moc',
    name: 'MOC',
    connections: 1247,
    insights: [
      'Reserved Instance coverage gaps',
      'Cross-account cost allocation',
      'Idle resource patterns in linked accounts',
    ],
    lastUpdated: '2026-04-27T10:00:00Z',
  },
  fileList: {
    date: '2026-04-27',
    files: ['hubs/moc.top.json', 'hubs/aws-ri.top.json', 'hubs/gcp-commits.top.json'],
  },
};

export function useTopHubSignal(mode: 'auto' | 'live' | 'mock' = 'auto') {
  const shouldUseMock = mode === 'mock' || (mode === 'auto' && process.env.NODE_ENV !== 'production');

  const { data, error, isLoading, mutate } = useSWR<HubPayload>(
    shouldUseMock ? null : '/api/knowledge/hubs',
    fetcher,
    {
      refreshInterval: 300000,
      revalidateOnFocus: false,
      dedupingInterval: 60000,
      fallbackData: shouldUseMock ? MOCK_HUB : undefined,
    }
  );

  return {
    hub: (shouldUseMock ? MOCK_HUB : data)?.hub ?? null,
    isLoading,
    isError: !!error,
    source: (shouldUseMock ? MOCK_HUB : data)?.source ?? null,
    mutate,
  };
}
```
