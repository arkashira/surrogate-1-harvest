## v0.1 scope
v0.1 in 2 weeks: a deterministic, replayable workflow engine (FastAPI + Redis Streams) on Cloudflare Workers with Postgres (Supabase) for state and Redis for cache/queue; supports research→signal→risk_check→execution path with one broker connector and one data vendor; immutable audit entries for every transition; pre-trade risk guardrails (notional and frequency limits); idempotent execution submission; API to submit workflows, query runs, and export audit trails; test suite covering orchestration, guardrails, and replay determinism.
