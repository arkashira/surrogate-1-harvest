## Final Implementation Plan (Synthesized)

**Goal**: Deterministic CDN file manifest + Lightning Studio lifecycle resilience for Surrogate-1 training in <2h.

---

### 1) Manifest generator (Mac orchestration) — deterministic, single-call
- **Single API call**: `list_repo_tree(path, recursive=False)` for one date folder (e.g. `batches/mirror-merged/2026-05-03/`).
- **Output**: `manifests/2026-05-03_files.json` containing:
  - `repo_id`, `folder`, `generated_at`, and sorted list of relative file paths under that folder.
- **Retry policy**: on 429, wait 360s and retry once; fail fast on other errors.
- **No auth during training**: manifest enables pure CDN URLs, eliminating API calls while training.

---

### 2) Lightning Studio lifecycle resilience — quota-safe, deterministic reuse
- **Reuse-first policy**:
  - Enumerate studios; if a studio with the target name is `running`, reuse it.
  - If found but stopped, restart with `target.start(machine=Machine.L40S)` (fallback to available cloud/size per priority order).
- **Guarded execution**:
  - Before each `.run()`, verify studio is running; if not, restart and re-run.
  - Avoid recreation to preserve the 80hr/mo quota.
- **Session hygiene**:
  - Set explicit timeouts and idempotent run semantics so transient stops don’t lose progress.

---

### 3) CDN-only data loader — streaming, schema-minimal, deterministic shard mapping
- **Deterministic fetch**:
  - Read manifest JSON.
  - For each entry, download via CDN URL:
    `https://huggingface.co/datasets/{repo_id}/resolve/main/{folder}/{fname}`
  - Use `requests` with streaming and retries; raise on non-2xx.
- **Parse at load time**:
  - Parse parquet/JSONL only at parse time.
  - Project strictly to `{prompt, response}` with no schema assumptions.
- **Deterministic shard mapping** (for any writes):
  - `hash(slug) % N` to choose sibling repo; bypasses HF commit cap and ensures reproducibility.

---

### 4) Cron/script hygiene — portable, strict
- Shebang: `#!/usr/bin/env bash` and `#!/usr/bin/env python3`.
- `chmod +x` on scripts; invoke via `bash <script> "$@"`.
- In crontab: set `SHELL=/bin/bash` and use absolute paths.
- Fail-fast in scripts: `set -euo pipefail`.

### manifest_generator.py
```python
#!/usr/bin/env python3
"""
Generate deterministic CDN manifest for one date folder.
Run from Mac (or any orchestration host) before training.
"""
import json
import os
import sys
import time
from pathlib import Path

from huggingface_hub import HfApi, HfFolder

REPO_ID = os.getenv("HF_DATASET_REPO", "axentx/surrogate-mirror")
DATE_FOLDER = os.getenv("DATE_FOLDER", "batches/mirror-merged/2026-05-03")
OUT_PATH = Path(os.getenv("OUT_MANIFEST", "manifests/2026-05-03_files.json"))
MAX_RETRIES = 3
RETRY_WAIT = 360  # seconds after 429

api = HfApi(token=HfFolder.get_token())

def list_folder_with_retry(path: str):
    for attempt in range(1, MAX_RETRIES + 1):
        try:
            tree = api.list_repo_tree(repo_id=REPO_ID, path=path, recursive=False)
            return sorted(item.rfilename for item in tree if not item.rfilename.endswith("/"))
        except Exception as exc:
            status = getattr(getattr(exc, "response", None), "status_code", None)
            if status == 429 and attempt < MAX_RETRIES:
                print(f"429 rate-limited, waiting {RETRY_WAIT}s (attempt {attempt})")
                time.sleep(RETRY_WAIT)
                continue
            print(f"Failed to list {path}: {exc}", file=sys.stderr)
            raise

def main():
    OUT_PATH.parent.mkdir(parents=True, exist_ok=True)
    files = list_folder_with_retry(DATE_FOLDER)
    manifest = {
        "repo_id": REPO_ID,
        "folder": DATE_FOLDER,
        "files": files,
        "generated_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
    }
    OUT_PATH.write_text(json.dumps(manifest, indent=2))
    print(f"Wrote {len(files)} files to {OUT_PATH}")

if __name__ == "__main__":
    main()
```

import requests
from lightning import LightningWork, LightningStudio, Machine

MANIFEST_PATH = Path(os.getenv("MANIFEST_PATH", "manifests/2026-05-03_files.json"))
CDN_TEMPLATE = "https://huggingface.co/datasets/{repo_id}/resolve/main/{folder}/{fname}"

def load_manifest(path: Path):
    with open(path) as f:
        return json.load(f)

def cdn_fetch(repo_id, folder, fname, timeout=30, max_retries=3):
    url = CDN_TEMPLATE.format(repo_id=repo_id, folder=folder, fname=fname)
    for attempt in range(1, max_retries + 1):
        try:
            resp = requests.get(url, stream=True, timeout=timeout)
            resp.raise_for_status()
            return resp.content
        except Exception as exc:
            if attempt == max_retries:
                raise
            time.sleep(2 ** attempt)
    raise RuntimeError("Unreachable")

class SurrogateTrainer(LightningWork):
    def __init__(self, studio_name="surrogate-trainer", **kwargs):
        super().__init__(**kwargs)
        self.studio_name = studio_name

    def _reuse_or_create_studio(self):

                print(f"Reusing running studio: {self.studio_name}")
                return s
        # create if none running
        print(f"Creating studio: {self.studio_name}")
        return LightningStudio(
            name=self.studio_name,
            machine=Machine.L40S,
            create_ok=True,

        if self.studio is None or self.studio.status != "running":
