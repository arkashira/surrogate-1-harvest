# Costinel / ops

## Final Implementation Plan — CDN-First Top-Hub Signal Panel (<2h)

**Goal**: Ship a resilient “Top Hub” signal panel into Costinel that shows the most-connected hub (e.g., “MOC”) and related docs using CDN-fetched artifacts (zero runtime HF API calls).  
**Principle**: Strict **Sense + Signal — ไม่ Execute**. No runtime secrets, no DB migrations, no infra changes.

---

### 1) Architecture (correct + actionable)

1. **Offline/Mac orchestration** (run once or on schedule via cron):
   - List target date folder in `knowledge-rag`.
   - Fetch `top-hub.json` and `hub-edges.json` via CDN URLs.
   - Produce `public/signals/top-hub.json` (projected, minimal schema) consumed by the frontend.

2. **Runtime (Costinel frontend)**:
   - Static fetch of `/signals/top-hub.json` (no auth, no HF API).
   - Render card: hub name, rank, short insight, related docs (title + link).
   - Graceful fallback if file missing or malformed.

3. **Cron hygiene**:
   - Use `#!/usr/bin/env bash` shebang.
   - `chmod +x` on wrapper.
   - `SHELL=/bin/bash` in crontab.
   - Invoke via `bash script "$@"`.

### 4) Orchestration script (Mac)

`scripts/signals/update-top-hub.sh`
```bash
#!/usr/bin/env bash
# Usage: bash scripts/signals/update-top-hub.sh <date>
# Example: bash scripts/signals/update-top-hub.sh 2026-04-27
# Requires: curl, jq
# Strategy: CDN-only fetches. No HF API at runtime.

set -euo pipefail

REPO="AXENTX/knowledge-rag"
DATE="${1:-$(date +%Y-%m-%d)}"
OUT_DIR="public/signals"
OUT_FILE="${OUT_DIR}/top-hub.json"

mkdir -p "${OUT_DIR}"

# CDN base (no auth, bypasses /api/ rate limits)
CDN="https://huggingface.co/datasets/${REPO}/resolve"

# Fetch raw artifacts from CDN
TOP_HUB_RAW=$(curl -fsSL "${CDN}/${DATE}/top-hub.json" || echo '{}')
EDGES_RAW=$(curl -fsSL "${CDN}/${DATE}/hub-edges.json" || echo '{}')

# Project minimal schema for frontend
jq -n \
  --argjson top "$TOP_HUB_RAW" \
  --argjson edges "$EDGES_RAW" \
  '{
    hub: ($top.hub // {id: "MOC", label: "MOC", description: "Most-connected operational hub", importance: 0.9}),
    related: (
      ($edges.related // []) | sort_by(-.relevance) | .[0:3]
    ),
    generatedAt: ($top.generatedAt // (now | todate)),
    ttlHours: ($top.ttlHours // 24)
  }' > "${OUT_FILE}.tmp"

mv "${OUT_FILE}.tmp" "${OUT_FILE}"
echo "Updated ${OUT_FILE}"
```

Make executable:
```bash
chmod +x scripts/signals/update-top-hub.sh
```

Crontab example (runs daily at 04:30):
```cron
SHELL=/bin/bash
30 4 * * * bash /opt/axentx/Costinel/scripts/signals/update-top-hub.sh >> /var/log/update-top-hub.log 2>&1
```

### 5) Hook: CDN fetch with cache + TTL (no API calls)

`src/hooks/useCDNSignal.ts`
```ts
import { useEffect, useState, useCallback } from 'react';

const CDN_BASE = process.env.PUBLIC_URL + '/signals';

export interface HubSignal {
  hub: {
    id: string;
    label: string;
    description: string;
    importance: number;
  };
  related: Array<{
    id: string;
    label: string;
    type: 'doc' | 'alert' | 'playbook';
    url: string;
    relevance: number;
  }>;
  generatedAt: string;
  ttlHours: number;
}

export function useCDNSignal() {
  const [signal, setSignal] = useState<HubSignal | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  const fetchSignal = useCallback(async () => {
    setLoading(true);
    try {
      const res = await fetch(`${CDN_BASE}/top-hub.json`, { cache: 'no-cache' });
      if (!res.ok) throw new Error(`CDN fetch failed: ${res.status}`);
      const payload: HubSignal = await res.json();

      // TTL guard (client-side)
      const gen = new Date(payload.generatedAt).getTime();
      const now = Date.now();
      const ttlMs = payload.ttlHours * 60 * 60 * 1000;
      if (now - gen > ttlMs) {
        console.warn('CDN signal stale (TTL exceeded)');
      }

      setSignal(payload);
      setError(null);
    } catch (err) {
      setError(err instanceof Error ? err.message : 'Unknown error');
      setSignal(null);
    } finally {
      setLoading(false);
    }
  }, []);

  useEffect(() => {
    fetchSignal();
    // Optional: refresh on visibility change (CDN-friendly)
    const onVisibility = () => {
      if (!document.hidden) fetchSignal();
    };
    document.addEventListener('visibilitychange', onVisibility);
    return () => {
      document.removeEventListener('visibilitychange', onVisibility);
    };
  }, [fetchSignal]);

  return { signal, loading, error, refetch: fetchSignal };
}
```
