## Final Synthesized Solution: CDN-Driven Autonomous Training Pipeline for Surrogate-1

**Core Value**: Eliminates HF API rate limits during training, fixes `pyarrow.CastError` from mixed schemas, enables 24/7 autonomous training on Lightning, and preserves studio quota. Ships in **<2h**.

---

### Unified Implementation Plan

#### 1. Manifest Generator (Mac / Airship) — **Single API Call**
- **When**: Run once after HF rate-limit reset (or via cron at 00:15 UTC).
- **What**: One non-recursive `list_repo_tree` call per date folder → `file_manifest.json`.
- **CDN URLs**: `https://huggingface.co/datasets/{repo}/resolve/main/{path}` (no auth, no API).
- **Schema Guard**: Projects to `{prompt, response}` at parse time; fills missing cols with `""`.

#### 2. Lightning-Compatible `CdnDataset` Class
- **Zero HF API during training**: Uses CDN URLs from manifest.
- **Streaming parquet**: Downloads via `requests` with retry + exponential backoff.
- **Schema safety**: Projects columns explicitly; handles missing `prompt`/`response` gracefully.
- **Lightning integration**: Returns iterable of `{prompt, response}` dicts; compatible with `fabric.launch`.

#### 3. Studio Reuse & Autostart Logic
- **Reuse**: Check `Teamspace.studios` for existing running studio before create.
- **Autostart**: Wrapper script (`start_studio.sh`) ensures `target.start(Machine.L40S)` on reboot/cron.
- **Cron**: Runs every 6h to restart training if studio died; logs to `/var/log/surrogate_studio.log`.

#### 4. Training Script Update
- Replace `load_dataset(streaming=True)` with `CdnDataset(manifest_path)`.
- Use `Fabric` with `bf16-mixed`, `accelerator="cuda"`.
- Ensure model checkpointing survives studio restarts.

#### `surrogate/data/cdn_dataset.py`
```python
import json
import pyarrow.parquet as pq
import requests
from pathlib import Path
from typing import Dict, Iterator, List
import io
import time

class CdnDataset:
    """
    CDN-only dataset loader for Surrogate-1.
    Bypasses HF API rate limits by using resolve/main/ CDN URLs.
    Projects to {prompt, response} only to avoid pyarrow.CastError.
    """
    def __init__(self, manifest_path: str | Path, max_retries: int = 3):
        self.manifest_path = Path(manifest_path)
        with open(self.manifest_path) as f:
            self.manifest = json.load(f)  # [{"cdn_url": "...", "slug": "...", "date": "..."}]
        self.session = requests.Session()
        self.session.headers.update({"User-Agent": "Surrogate-1-CDN-Loader/1.0"})
        self.max_retries = max_retries

    def __len__(self) -> int:
        return len(self.manifest)

    def _download_parquet(self, cdn_url: str) -> bytes:
        for attempt in range(self.max_retries):
            try:
                resp = self.session.get(cdn_url, timeout=30)
                resp.raise_for_status()
                return resp.content
            except Exception as e:
                if attempt == self.max_retries - 1:
                    raise
                wait = 2 ** attempt
                print(f"Retry {attempt+1}/{self.max_retries} for {cdn_url}: {e}. Waiting {wait}s")
                time.sleep(wait)
        raise RuntimeError("Unreachable")

    def _project_to_pair(self, table) -> List[Dict[str, str]]:
        """Extract only prompt/response, fill missing with empty string."""
        df = table.to_pandas()
        pairs = []
        for _, row in df.iterrows():
            pairs.append({
                "prompt": str(row.get("prompt", row.get("input", ""))),
                "response": str(row.get("response", row.get("output", ""))),
            })
        return pairs

    def stream(self) -> Iterator[Dict[str, str]]:
        """Yield {prompt, response} pairs from all parquet files."""
        for item in self.manifest:
            try:
                data = self._download_parquet(item["cdn_url"])
                table = pq.read_table(io.BytesIO(data))
                yield from self._project_to_pair(table)
            except Exception as e:
                print(f"Skipping {item['slug']}: {e}")
                continue

    def to_lightning_dataset(self):
        """Convert to Lightning-compatible iterable dataset."""
        return list(self.stream())
```

#### `surrogate/data/generate_manifest.py`
```python
#!/usr/bin/env python3
"""
Generate training manifest from HF repo tree.
Run on Mac when API limits reset.
"""
import json
from datetime import datetime, timedelta, timezone
from huggingface_hub import HfApi
from pathlib import Path

def generate_manifest(
    repo_id: str = "axentx/surrogate-mirror-merged",
    date_folder: str = None,
    output_path: str = "surrogate/data/file_manifest.json"
):
    api = HfApi()
    if date_folder is None:
        # Default to yesterday (UTC)
        date_folder = (datetime.now(timezone.utc) - timedelta(days=1)).strftime("%Y-%m-%d")
    
    # Single API call: non-recursive per pattern
    tree = api.list_repo_tree(
        repo_id=repo_id,
        path=f"batches/mirror-merged/{date_folder}",
        recursive=False  # paginates 100x if True on big repos
    )
    
    manifest = []
    for item in tree:
        if item.path.endswith(".parquet"):
            cdn_url = f"https://huggingface.co/datasets/{repo_id}/resolve/main/{item.path}"
            manifest.append({
                "cdn_url": cdn_url,
                "slug": item.path,
                "date": date_folder
            })
    
    out_path = Path(output_path)
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    print(f"Generated manifest with {len(manifest)} files -> {out_path}")
    return out_path

if __name__ == "__main__":
    generate_manifest()
```

            name=studio_name,
            machine=Machine.L40S,
            cloud="lightning-public-prod"
        )

    fabric.launch(model, train_dataloader=dataset)

if __name__ == "__main__":
