# airship / design

Candidate 3:
## Highest-Value Incremental Improvement (<2h)

**Improvement**: Harden `airship discover` into a deterministic, **CDN-only** orchestrator that eliminates HF API rate limits and PyArrow schema errors while producing reproducible file lists.

**Why this ships fastest**:
- Pure orchestration (no model training, no infra changes)
- Reuses existing patterns (CDN bypass, file-list pre-generation, schema projection)
- Fits <2h: ~30m design + 60-90m implementation + 15m test

---

## Implementation Plan

### 1. Create `airship/discover/cdn_orchestrator.py`
- Single entrypoint: `build_file_manifest(repo_id, date_folder, out_path)`
- Steps:
  1. **List once** via HF API (`list_repo_tree(path=date_folder, recursive=False)`) from Mac — only when rate-limit window is clear.
  2. **Filter** to target file extensions (`.parquet`, `.jsonl`, `.json`).
  3. **Emit** `manifest.json` with deterministic fields:
     - `cdn_url`: `https://huggingface.co/datasets/{repo_id}/resolve/main/{path}`
     - `sha256`: optional integrity check
     - `size`: optional for progress bars
  4. **Write** to `out_path` (default: `file_list.json`).

### 2. Create `surrogate/data/cdn_dataset.py`
- `CDNParquetDataset(IterableDataset)`:
  - Accepts `manifest_path`
  - Streams parquet via `requests.get(cdn_url)` + `pyarrow.parquet.read_table`
  - Projects to `{prompt, response}` at parse time (avoids schema errors)
  - Yields dicts for Lightning DataLoader

### 3. Add retry/backoff for 429
- If `list_repo_tree` returns 429, wait 360s and retry once.

### 4. Deterministic repo selection (optional sibling fallback)
- Hash slug → 1 of 5 sibling repos to spread load.

### 1. `airship/discover/cdn_orchestrator.py`

```python
#!/usr/bin/env python3
"""
CDN-only file manifest builder for surrogate-1.
Generates file_list.json for Lightning training (zero API calls during load).
"""
import json
import hashlib
import time
from pathlib import Path
from typing import List, Dict

from huggingface_hub import HfApi

HF_REPO = "axentx/surrogate-mirror"
SIBLINGS = [f"axentx/surrogate-mirror-{i}" for i in range(1, 6)]
RETRY_WAIT = 360

api = HfApi()

def pick_repo(slug: str) -> str:
    digest = hashlib.sha256(slug.encode()).hexdigest()
    idx = int(digest, 16) % len(SIBLINGS)
    return SIBLINGS[idx]

def list_date_folder(repo_id: str, date_folder: str) -> List[Dict]:
    entries = api.list_repo_tree(
        repo_id=repo_id,
        path=date_folder,
        repo_type="dataset",
        recursive=False,
    )
    files = []
    for e in entries:
        if e.type != "file":
            continue
        if not e.path.lower().endswith((".parquet", ".jsonl", ".json")):
            continue
        files.append({
            "path": e.path,
            "cdn_url": f"https://huggingface.co/datasets/{repo_id}/resolve/main/{date_folder}/{e.path}",
            "size": getattr(e, "size", None),
        })
    return files

def build_file_manifest(repo_id: str, date_folder: str, out_path: Path):
    print(f"Building manifest for {date_folder} (CDN-only)")
    try:
        files = list_date_folder(repo_id, date_folder)
    except Exception as exc:
        if "429" in str(exc):
            print(f"Rate limited. Waiting {RETRY_WAIT}s")
            time.sleep(RETRY_WAIT)
            files = list_date_folder(repo_id, date_folder)
        else:
            raise

    manifest = {
        "repo": repo_id,
        "date_folder": date_folder,
        "generated_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "files": files,
    }
    out_path.parent.mkdir(parents=True, exist_ok=True)
    out_path.write_text(json.dumps(manifest, indent=2))
    print(f"Wrote {len(files)} files -> {out_path}")
    return manifest

if __name__ == "__main__":
    import sys
    repo = sys.argv[1] if len(sys.argv) > 1 else HF_REPO
    df = sys.argv[2] if len(sys.argv) > 2 else "2024-01-01"
    out = Path(sys.argv[3]) if len(sys.argv) > 3 else Path("file_list.json")
    build_file_manifest(repo, df, out)
```

### 2. `surrogate/data/cdn_dataset.py`

```python
import json
import pyarrow as pa
import pyarrow.parquet as pq
import requests
from pathlib import Path
from torch.utils.data import IterableDataset

class CDNParquetDataset(IterableDataset):
    """
    Lightning-compatible dataset that fetches parquet files via CDN only.
    Expects file_list.json produced by cdn_orchestrator.
    Projects to {prompt, response} at parse time to avoid schema errors.
    """
    def __init__(self, manifest_path: str, columns=("prompt", "response")):
        super().__init__()
        with open(manifest_path) as f:
            self.manifest = json.load(f)
        self.files = self.manifest["files"]
        self.columns = columns

    def __iter__(self):
        for f in self.files:
            resp = requests.get(f["cdn_url"], timeout=30)
            resp.raise_for_status()
            table = pq.read_table(pa.BufferReader(resp.content), columns=self.columns)
            for batch in table.to_batches():
                df = batch.to_pandas()
                for _, row in df.iterrows():
                    yield {
                        "prompt": row["prompt"],
                        "response": row["response"],
                    }
```

### 3. Ingestion wrapper (project schema at write time)

```bash
#!/usr/bin/env bash
# scripts/ingest_mirror.sh
# Projects to {prompt,response} and writes to sibling repo by hash.

set -euo pipefail
SHELL=/bin/bash

HF_REPO_BASE="axentx/surrogate-mirror"
SIBLINGS=("axentx/surrogate-mirror-1" "axentx/surrogate-mirror-2" "axentx/surrogate-mirror-3" "axentx/surrogate-mirror-4" "axentx/surrogate-mirror-5")

DATE_FOLDER="${1:-$(date +%Y-%m-%d)}"
INPUT_DIR="${2:-./raw}"

pick_repo() {
  local slug="$1"
  local digest=$(echo -n "$slug" | sha256sum | cut -c1-16)
  local idx=$((0x$digest % ${#SIBLINGS[@]}))
  echo "${SIBLINGS[$idx]}"
}

for src in "$INPUT_DIR"/*.jsonl; do
  slug=$(basename "$src" .jsonl)
  REPO=$(pick_repo "$slug")
  OUT="batches/mirror-merged/$DATE_FOLDER/${slug}.parquet"
  python - <<PY
import pyarrow as pa, pyarrow.parquet as pq, json, sys
rows = []
with open("$src") as f:
    for line in f:
        obj = json.loads(line)
        rows.append({
            "prompt": obj.get("prompt", ""),
            "response": obj.get("response", ""),
        })
tbl = pa.Table.from_pylist(rows, schema=pa.schema([
    ("prompt", pa.string()),
    ("response", pa.string()),
]))
pq.write_table(tbl, "/tmp/tmp.parquet")
PY
  huggingface-cli upload "$REPO" "/tmp/tmp.parquet" "$OUT" --repo-type dataset
done
```
