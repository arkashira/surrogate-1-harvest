# airship / design

**Final synthesized implementation** (best parts merged, contradictions resolved for correctness + concrete actionability)

---

## Why this now (merged rationale)
- **Highest-value, lowest-effort**: one zero-config CLI (`airship discover`) that operationalizes three proven patterns: `business-research`, `knowledge-rag`, `top-hub`.
- **Fixes prior script-exec lessons**: shebang, `set -euo pipefail`, executable bits, `crontab SHELL=/bin/bash` guidance, and robust root detection.
- **Delivers immediate strategic context** for Arkship/Surrogate platform decisions.
- **Fits <2h**: single dispatcher + one discover script + tiny optional Python helper + docs.

## Core design decisions (resolved contradictions)

1. **Root detection**: prefer `.git` **or** `.airship-root` (exists in either repo style). Fail fast with clear message if neither found.
2. **Script execution**: use `bash <script>` (not `sh`) and require executable bit + shebang. Document crontab requirement: `SHELL=/bin/bash`.
3. **Discover behavior**:
   - Run business research script if present and executable; log + continue if it fails (non-fatal).
   - Query knowledge graph for top hub + insights; multi-tier fallback: Python helper → `cypher-shell`/Neo4j → static defaults.
   - Never hard-fail on missing components; always print actionable defaults.
4. **HF/CDN guidance**: include concrete rate-limit bypass and CDN-only training guidance (resolve/main/ URLs, sibling repos to spread 128/hr cap).
5. **Platform/AI delegation**: avoid local Mac model loads; prefer Lightning/Kaggle for training (per prior patterns).

### lib/common.sh
```bash
#!/usr/bin/env bash
set -euo pipefail

AIRSHIP_ROOT=""

log() { printf "[airship] %s\n" "$*"; }
die() { printf "[airship] ERROR: %s\n" "$*" >&2; exit 1; }

detect_root() {
  local dir="${1:-$PWD}"
  while [[ "$dir" != "/" ]]; do
    if [[ -d "$dir/.git" ]] || [[ -f "$dir/.airship-root" ]]; then
      AIRSHIP_ROOT="$dir"
      return 0
    fi
    dir="$(dirname "$dir")"
  done
  die "Not in an airship repo (no .git or .airship-root found)"
}
```

### lib/airship/discover.sh
```bash
#!/usr/bin/env bash
set -euo pipefail

SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
source "$SCRIPT_DIR/../common.sh"

detect_root
cd "$AIRSHIP_ROOT"

log "Running strategic discovery for airship..."

# 1) Business research (optional)
RESEARCH_SCRIPT="scripts/granite-business-research.sh"
if [[ -x "$RESEARCH_SCRIPT" ]]; then
  log "Running business research..."
  if ! bash "$RESEARCH_SCRIPT"; then
    log "Business research completed with warnings (non-fatal)"
  fi
else
  log "No business research script at $RESEARCH_SCRIPT — skipping (optional)"
fi

# 2) Knowledge RAG: top hub + insights
TOP_HUB=""
declare -a INSIGHTS=()

# Tier 1: Python helper
if [[ -x "$(command -v python3)" ]] && [[ -f "py/knowledge_rag.py" ]]; then
  log "Querying knowledge graph (Python helper)..."
  mapfile -t RESULTS < <(python3 py/knowledge_rag.py --top-hub --insights 3 2>/dev/null || true)
  if [[ ${#RESULTS[@]} -gt 0 ]]; then
    TOP_HUB="${RESULTS[0]}"
    INSIGHTS=("${RESULTS[@]:1}")
  fi
fi

# Tier 2: Neo4j cypher-shell
if [[ -z "$TOP_HUB" ]] && command -v cypher-shell &>/dev/null && [[ -n "${NEO4J_URI:-}" ]]; then
  log "Attempting Neo4j query for top hub..."
  TOP_HUB=$(cypher-shell -u neo4j -p "${NEO4J_PASSWORD:-neo4j}" \
    "MATCH (h:Hub) RETURN h.name AS name, size((h)--()) AS conns ORDER BY conns DESC LIMIT 1" \
    --format plain 2>/dev/null | tail -1 | awk '{print $1}' || true)
fi

# Tier 3: static defaults (always actionable)
if [[ -z "$TOP_HUB" ]]; then
  TOP_HUB="MOC"
  INSIGHTS=(
    "MOC (Mission Operations Center) is the most-connected hub — review runbooks and incident workflows."
    "Check recent HF ingestion commits — stay under 128/hr/repo or use sibling repos to spread commits."
    "Prefer HF CDN bypass for dataset training: use resolve/main/ URLs to avoid API rate limits."
  )
fi

# 3) Report
log "=== Strategic Context ==="
printf "Top hub: %s\n" "$TOP_HUB"
printf "\nInsights:\n"
for i in "${!INSIGHTS[@]}"; do
  printf "  %s. %s\n" "$((i+1))" "${INSIGHTS[$i]}"
done

printf "\nRecommended next actions:\n"
printf "  - Review hub '%s' and related runbooks in Arkship (port 8000).\n" "$TOP_HUB"
printf "  - If training surrogate-1: pre-list HF files and embed in train.py (CDN-only during training).\n"
printf "  - For HF ingestion: spread commits across sibling repos to bypass 128/hr cap.\n"
printf "  - Use Lightning Studio reuse pattern to save quota; avoid idle timeouts.\n"
log "Done."
```

### bin/airship
```bash
#!/usr/bin/env bash
# bin/airship — Airship CLI dispatcher
# Usage: airship <command> [args...]
# Commands: discover, help

set -euo pipefail

AIRSHIP_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
export AIRSHIP_ROOT

usage() {
  cat <<EOF
Usage: airship <command>

Commands:
  discover   Run zero-config strategic discovery (business research + knowledge-rag + top hub)
  help       Show this help
EOF
}

case "${1:-}" in
  discover)
    exec "$AIRSHIP_ROOT/lib/airship/discover.sh"
    ;;
  help|--help|-h)
    usage
    ;;
  *)
    usage
    exit 1
    ;;
esac
```
