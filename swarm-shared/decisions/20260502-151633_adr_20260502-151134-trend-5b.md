## v0.1 scope
v0.1 in 2 weeks: deterministic workflow engine with Redis Streams event sourcing; Postgres (Supabase) for run metadata and audit; one research connector, one signal generator, one risk-check guardrail, and one simulated execution connector; immutable audit log with hash chaining; CF Workers API gateway for low-latency ingress; replay API to re-run any workflow_run deterministically; basic compliance toggle per rule; end-to-end tests and contract tests for connectors.
