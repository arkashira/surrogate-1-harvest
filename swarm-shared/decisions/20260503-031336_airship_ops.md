# airship / ops

**Final synthesized plan (highest-correctness + concrete actionability)**

Goal: run Surrogate training without HF API rate‑limit (429) failures and without losing progress when Lightning/L40S idle stops.

---

## 1) Core design (resolve contradictions)

- **Use CDN-only data loading, never `load_dataset(..., streaming=True)` during training.**  
  - CDN URLs (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`) are public, no token/header required, and bypass API rate limits.  
  - This is strictly safer and more reliable than depending on HF API availability during long training runs.

- **Pre-list files once (Mac/CI), do not list at runtime.**  
  - Avoids repeated `list_repo_tree` API calls during training.  
  - One-off script run after rate-limit window is sufficient; commit or volume-mount the produced `file_list.json`.

- **Make training resilient to idle stops via lightweight checkpointing + restart logic, not via complex orchestration inside the training loop.**  
  - Save a small, reloadable checkpoint every N steps/epochs.  
  - On restart, reload latest checkpoint and resume data iteration deterministically (by filename order + byte offset or by re-creating the DataLoader and skipping already-seen samples if necessary).  
  - Keep retry/backoff for transient failures, but do not rely on in-loop “studio status polling” as primary control flow.

- **Keep changes minimal and non-invasive to existing pipeline.**  
  - Add one dataset module, one pre-list script, and small checkpoint/restore logic in train script.  
  - Do not rewrite the entire training loop.

### 2.1 Pre-list files (run once)

`scripts/pre_list_files.py`
```python
#!/usr/bin/env python3
"""
Pre-list HF dataset files for a given folder and save to JSON.
Run once per folder (e.g., from Mac or CI) after rate-limit window clears.
"""
import json
import os
from huggingface_hub import HfApi

REPO_ID = os.getenv("HF_DATASET_REPO", "my-org/surrogate-data")
FOLDER   = os.getenv("DATASET_FOLDER", "batches/mirror-merged/2026-05-03")
OUT_PATH = os.getenv("OUT_FILE_LIST", "file_list.json")

def main() -> None:
    api = HfApi()
    tree = api.list_repo_tree(repo_id=REPO_ID, path=FOLDER, recursive=False)
    files = [
        f"{FOLDER}/{entry.path.split('/')[-1]}"
        for entry in tree
        if not entry.path.endswith("/")
    ]
    with open(OUT_PATH, "w") as f:
        json.dump(files, f, indent=2)
    print(f"Wrote {len(files)} files to {OUT_PATH}")

if __name__ == "__main__":
    main()
```
- Run: `HF_DATASET_REPO=... DATASET_FOLDER=... python scripts/pre_list_files.py`
- Ship `file_list.json` alongside code or mount via Docker volume. Add it to `.gitignore` if large; include a small example template in repo.

### 2.2 CDN-only IterableDataset

`surrogate/data/cdn_dataset.py`
```python
import json
import os
from typing import Dict, Iterator, Optional

import requests
import torch
from torch.utils.data import IterableDataset

HF_DATASET_REPO = os.getenv("HF_DATASET_REPO", "my-org/surrogate-data")
FILE_LIST_PATH  = os.getenv("FILE_LIST_PATH", "file_list.json")
CDN_BASE        = f"https://huggingface.co/datasets/{HF_DATASET_REPO}/resolve/main"

class CDNIterableDataset(IterableDataset):
    """Zero-HF-API dataset using public CDN downloads."""

    def __init__(self, file_list_path: str = FILE_LIST_PATH, max_files: Optional[int] = None):
        with open(file_list_path) as f:
            self.files = json.load(f)
        if max_files:
            self.files = self.files[:max_files]

    def _stream_file(self, rel_path: str) -> Iterator[Dict]:
        url = f"{CDN_BASE}/{rel_path}"
        with requests.get(url, stream=True, timeout=60) as r:
            r.raise_for_status()
            for line in r.iter_lines(decode_unicode=True):
                if not line:
                    continue
                try:
                    obj = json.loads(line)
                    yield {
                        "prompt": obj.get("prompt") or obj.get("input") or "",
                        "response": obj.get("response") or obj.get("output") or "",
                    }
                except Exception:
                    continue

    def __iter__(self) -> Iterator[Dict]:
        worker_info = torch.utils.data.get_worker_info()
        if worker_info is None:
            files = self.files
        else:
            per_worker = max(1, len(self.files) // worker_info.num_workers)
            start = worker_info.id * per_worker
            end = min(len(self.files), start + per_worker)
            files = self.files[start:end]

        for rel_path in files:
            try:
                yield from self._stream_file(rel_path)
            except Exception as exc:
                # Log and skip bad files to avoid crashing training
                print(f"Skipping {rel_path}: {exc}")
                continue
```
Notes:
- Assumes line-delimited JSON; adapt parsing if format differs.
- No auth header required for public datasets.
- Deterministic per-worker sharding avoids duplicate samples across workers.
