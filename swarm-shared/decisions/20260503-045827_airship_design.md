# airship / design

## Final Synthesis (Best of Both Candidates)

**Highest-Value Incremental Improvement (≤2h)**  
Embed a CDN-only file manifest into the Surrogate-1 training pipeline so Lightning Studio training runs with **zero HF API calls during data loading**, eliminating 429 rate limits and 128-commit caps while keeping Mac as orchestrator-only.

**Why this ships in <2h**  
- Single manifest script (Mac) + small train.py patch (Lightning).  
- No infra changes, no new deps, no schema migrations, no secrets.  
- Reuses existing CDN paths and Lightning Studio reuse patterns.

---

## Implementation Plan

### 1) Manifest generator (Mac orchestrator) — `scripts/build_cdn_manifest.py`
- Run once per date folder after ingestion completes (or on cron after rate-limit window clears).  
- Uses **one** `list_repo_tree(path, recursive=False)` per date folder (avoids pagination).  
- Emits `manifests/{dataset_repo}/{date}/file_manifest.json` containing:
  - `repo_id`, `date_folder`, `generated_at`
  - per-file: `path`, `cdn_url`, `size`, `sha` (when available)
- Validates CDN reachability with lightweight HEAD (optional, fast, with short timeout).  
- Commits manifest (tiny; one commit per date folder).  
- **Fallback on commit-cap hit**: store manifest locally and pass path via env to training.

### 2) Lightning training patch — `surrogate/train.py`
- Accept `--manifest` (path to `file_manifest.json`).  
- If manifest provided:
  - Load via `datasets.load_dataset("json", data_files=cdn_urls, streaming=True)` (or custom `IterableDataset` with `requests.get(cdn_url)` + backoff for non-JSON).  
  - **Zero** calls to `list_repo_files` or `load_dataset(streaming=True)` on the full repo during training.  
- If no manifest:
  - Fallback to legacy behavior with a clear warning (and optional fail-fast flag for CI).

### 3) Reuse running Lightning Studio
- Before `Studio(create_ok=True)`, list `Teamspace.studios` and reuse a running studio named `surrogate-train-{date}`.  
- If stopped, restart with `target.start(machine=Machine.L40S)` (or H200 if preferred and available).

### 4) Wire in orchestration
Update `scripts/run_training.sh` to:
1. `python scripts/build_cdn_manifest.py --repo surrogate-dataset --date 2026-05-03 --out manifests`  
2. `python surrogate/train.py --manifest manifests/surrogate-dataset/2026-05-03/file_manifest.json ...`

## Code Snippets

### `scripts/build_cdn_manifest.py`
```python
#!/usr/bin/env python3
"""
Build a CDN-only manifest for a HuggingFace dataset repo folder.
Usage:
  python build_cdn_manifest.py --repo surrogate-dataset --date 2026-05-03 --out manifests/
"""
import argparse
import json
import time
from pathlib import Path
from typing import List, Dict

import requests
from huggingface_hub import HfApi, hf_hub_url

HF_API_RATE_LIMIT_RETRY = 360  # seconds
HEAD_TIMEOUT = 5.0

def build_manifest(repo_id: str, date_folder: str, out_dir: Path, validate_cdn: bool = True) -> Path:
    api = HfApi()
    out_dir.mkdir(parents=True, exist_ok=True)

    # Single API call: list top-level of date folder (non-recursive)
    try:
        tree = api.list_repo_tree(repo_id=repo_id, path=date_folder, recursive=False)
    except Exception as e:
        print(f"HF API error (likely 429): {e}. Waiting {HF_API_RATE_LIMIT_RETRY}s")
        time.sleep(HF_API_RATE_LIMIT_RETRY)
        tree = api.list_repo_tree(repo_id=repo_id, path=date_folder, recursive=False)

    entries: List[Dict] = []
    for node in tree:
        if node.type != "file":
            continue
        path = f"{date_folder}/{node.path}"
        cdn_url = str(hf_hub_url(repo_id=repo_id, filename=path, repo_type="dataset"))
        entry = {
            "path": path,
            "cdn_url": cdn_url,
            "size": getattr(node, "size", None),
            "sha": getattr(node, "sha", None),
        }

        if validate_cdn:
            try:
                resp = requests.head(cdn_url, timeout=HEAD_TIMEOUT)
                resp.raise_for_status()
            except Exception as e:
                print(f"CDN HEAD failed for {path}: {e}")
                # continue anyway; training will surface failures
        entries.append(entry)

    manifest = {
        "repo_id": repo_id,
        "date_folder": date_folder,
        "generated_at": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
        "files": entries,
    }

    out_file = out_dir / repo_id / f"{date_folder}.json"
    out_file.parent.mkdir(parents=True, exist_ok=True)
    out_file.write_text(json.dumps(manifest, indent=2))
    print(f"Manifest written to {out_file} ({len(entries)} files)")
    return out_file

if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Build CDN manifest for HF dataset folder")
    parser.add_argument("--repo", required=True, help="HF dataset repo id")
    parser.add_argument("--date", required=True, help="Date folder (e.g., 2026-05-03)")
    parser.add_argument("--out", default="manifests", help="Output directory")
    parser.add_argument("--no-validate-cdn", action="store_true", help="Skip CDN HEAD checks")
    args = parser.parse_args()

    build_manifest(args.repo, args.date, Path(args.out), validate_cdn=not args.no_validate_cdn)
```

### `surrogate/train.py` (patch excerpt)
```python
import argparse
import json
from pathlib import Path

from datasets import load_dataset
import lightning as L

def load_from_manifest(manifest_path: str):
    manifest = json.loads(Path(manifest_path).read_text())
    cdn_urls = [f["cdn_url"] for f in manifest["files"]]
    # Adjust format if files are not JSON/JSONL (e.g., parquet, images)
    return load_dataset("json", data_files=cdn_urls, streaming=True)

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--manifest", type=str, default=None, help="Path to file_manifest.json")
    parser.add_argument("--fail-fast", action="store_true", help="Fail if no manifest provided")
    parser.add_argument("--other-args", nargs="*", default=[])
    args = parser.parse_args()

    # Reuse running studio
    teamspace = L.Teamspace()
    studio_name = "surrogate-train"
    running = None
    for s in teamspace.studios:
        if s.name == studio_name and s.status == "running":
            running = s
            break

    if args.manifest:
        print("Loading dataset from CDN manifest (zero HF API during training)")
        dataset = load_from_manifest(args.manifest)
    else:
        msg = "WARNING: No manifest provided; falling back to HF API (may hit rate limits)"
        if args.fail_fast:
            raise RuntimeError(msg)
        print(msg)
        # legacy fallback
        dataset = load_dataset("surrogate-dataset", streaming=True)

    # Training loop continues...
    # trainer.fit(...)

if __name__ == "__main__":
    main()
```

### `scripts/run_training.sh`
```bash
#!/bin/bash
set -euo pipefail
export SHELL=/bin/bash

REPO="surrogate-dataset"
DATE="2026-05-03"
MANIFEST="manifests/${REPO}/${DATE}.json"

# 1) Build manifest (Mac orchestrator)
python scripts/build_cdn_manifest.py --repo "$REPO" --date "$DATE" --out manifests

# 2) Launch training in Lightning Studio (reuse running)
python surrogate/train.py --manifest "$MANIFEST" --other-args "$@"
```
