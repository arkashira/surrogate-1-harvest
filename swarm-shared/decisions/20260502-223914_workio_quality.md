**Guiding principles used**
- Correctness first: atomic state changes, idempotency, and single source-of-truth.
- Concrete actionability: minimal, copy-paste-ready code and exact verification steps.
- Performance: remove repeated tenant lookups with request-scoped caching.
- Reliability: handle LINE redelivery and double-tap safely.

## 1. Diagnosis (merged)
- Repeated `channelId → tenantId` DB lookups on every LINE webhook cause DB thrash and increase timeout/429 risk under load.
- No request-scoped tenant resolution; each handler re-queries tenant instead of reusing a resolved tenant for the request lifecycle.
- Clock state drift between UI and server: no atomic “clock-in if not already in” guard → duplicate/overlapping punches on retries/double-tap.
- No optimistic UI or local-first draft state for leave/OT forms → data loss on navigation/offline and perceived latency.
- Missing idempotency on webhook events (LINE may redeliver) → duplicate punches/leave requests.
- Missing lightweight dev harness to verify tenant resolution and caching behavior.

## 2. Proposed change (merged scope)
- Add tenant cache middleware + LRU cache module.
- Add atomic clock-in upsert with idempotency keyed by `eventId`.
- Wire middleware and idempotency into LINE webhook.
- Add small, high-value optimistic UI toggle for clock-in.
- Add dev verification script to confirm tenant resolution and caching.

### 1) Lightweight LRU cache (bounded, TTL)
```ts
// server/src/lib/cache.ts
type CacheEntry<T> = { value: T; expiresAt: number };

export class LRUCache<K, V> {
  private cache = new Map<K, CacheEntry<V>>();
  constructor(
    private maxSize = 500,
    private ttlMs = 5 * 60 * 1000
  ) {}

  get(key: K): V | undefined {
    const entry = this.cache.get(key);
    if (!entry) return undefined;
    if (Date.now() > entry.expiresAt) {
      this.cache.delete(key);
      return undefined;
    }
    // refresh recency
    this.cache.delete(key);
    this.cache.set(key, entry);
    return entry.value;
  }

  set(key: K, value: V) {
    if (this.cache.size >= this.maxSize) {
      const oldestKey = this.cache.keys().next().value;
      if (oldestKey !== undefined) this.cache.delete(oldestKey);
    }
    this.cache.set(key, { value, expiresAt: Date.now() + this.ttlMs });
  }

  has(key: K) {
    return this.get(key) !== undefined;
  }
}
```

### 2) Tenant cache middleware (request-scoped)
```ts
// server/src/middleware/tenant.ts
import { Request, Response, NextFunction } from 'express';
import { db } from '../db';
import { LRUCache } from '../lib/cache';

const tenantCache = new LRUCache<string, string>(500, 5 * 60 * 1000);

export function tenantMiddleware(req: Request, res: Response, next: NextFunction) {
  // Prefer explicit header; fallback to query/body for dev/testing
  const channelId =
    req.headers['x-line-channel-id']?.toString() ||
    req.query.channelId?.toString() ||
    req.body.channelId?.toString();

  if (!channelId) {
    return next();
  }

  let tenantId = tenantCache.get(channelId);
  if (!tenantId) {
    // Single-row lookup; keep minimal
    tenantId = db
      .prepare('SELECT tenant_id FROM channel_mappings WHERE channel_id = ?')
      .pluck()
      .get(channelId) as string | undefined;

    if (!tenantId) {
      return res.status(404).json({ error: 'tenant not found for channel' });
    }
    tenantCache.set(channelId, tenantId);
  }

  (req as any).tenantId = tenantId;
  (req as any).channelId = channelId;
  next();
}
```

### 3) Atomic clock-in upsert + idempotency
```ts
// server/src/db/queries/clock.ts
import { db } from '../db';

export function clockInEmployee(
  employeeId: string,
  tenantId: string,
  lat: number | null,
  lng: number | null,
  eventId?: string
) {
  // Idempotency guard
  if (eventId) {
    const exists = db
      .prepare('SELECT 1 FROM clock_events WHERE event_id = ? AND tenant_id = ?')
      .pluck()
      .get(eventId, tenantId);
    if (exists) return getCurrentClockState(employeeId, tenantId);
  }

  const now = new Date().toISOString();

  // Close any open punch atomically (defensive)
  db.prepare(
    `UPDATE clock_punches
     SET clock_out = ?, updated_at = ?
     WHERE employee_id = ? AND tenant_id = ? AND clock_out IS NULL`
  ).run(now, now, employeeId, tenantId);

  // Insert new clock-in
  const result = db
    .prepare(
      `INSERT INTO clock_punches (employee_id, tenant_id, clock_in, clock_out, lat, lng, event_id)
       VALUES (?, ?, ?, NULL, ?, ?, ?)`
    )
    .run(employeeId, tenantId, now, lat, lng, eventId || null);

  // Record event idempotency key
  if (eventId) {
    db.prepare(
      `INSERT OR IGNORE INTO clock_events (event_id, tenant_id, created_at)
       VALUES (?, ?, ?)`
    ).run(eventId, tenantId, now);
  }

  return {
    clockIn: now,
    clockOut: null,
    lastAction: 'clock_in',
    punchId: result.lastInsertRowid,
  };
}

export function getCurrentClockState(employeeId: string, tenantId: string) {
  const row = db
    .prepare(
      `SELECT clock_in, clock_out, lat, lng
       FROM clock_punches
       WHERE employee_id = ? AND tenant_id = ?
       ORDER BY clock_in DESC LIMIT 1`
    )
    .get(employeeId, tenantId);

  return {
    clockIn: row?.clock_in || null,
    clockOut: row?.clock_out || null,
    lat: row?.lat || null,
    lng: row?.lng || null,
    status: row && !row.clock_out ? 'in' : 'out',
  };
}
```
