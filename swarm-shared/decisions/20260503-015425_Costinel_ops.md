# Costinel / ops

**Final Implementation Plan — Top-Hub Signal Panel (Costinel Dashboard)**

**Guiding principles**
- Frontend-only, read-only, zero backend changes.
- CDN-first data delivery to bypass HF API limits and avoid runtime model loads.
- Resilient, cache-first UX with graceful fallback and clear empty/error states.
- Concrete, copy-paste-ready code and file layout; deployable in <2h.
- Correctness: severity + impactUSD validated; actions are safe navigation (no inline mutations).

---

### 1. Architecture & Data Flow (10 min)

- **Data source**: ops-generated `top-hub-MOC.json` committed/synced to CDN.  
  - Path: `public/data/knowledge-rag/top-hub-MOC.json` (dev) **or**  
    CDN URL: `https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/top-hub-MOC.json` (prod).
- **Delivery**: Static fetch with `cache: 'no-store'` + localStorage TTL cache (10m).  
- **Client**: Single reusable component (`TopHubSignalPanel`) using framework-native patterns (React or Vue).  
- **No backend** — minimal change surface.

### 2. Data Contract (canonical)

```json
{
  "hub": "MOC",
  "updatedAt": "2026-05-03T02:00:00Z",
  "proposals": [
    {
      "id": "MOC-2026-05-03-01",
      "title": "Reduce idle EKS node pool in ap-southeast-1",
      "severity": "high",
      "impactUSD": 4200,
      "description": "Detected 62% avg CPU idle across m5.xlarge nodes; right-size to m5.large with 40% headroom.",
      "actions": [
        { "label": "Create RI proposal", "href": "/proposals/new?type=ri&hub=MOC" },
        { "label": "View heatmap", "href": "/dashboard/heatmap?region=ap-southeast-1" }
      ],
      "tags": ["eks", "rightsize", "ap-southeast-1"]
    }
  ]
}
```

- Allowed `severity`: `"critical" | "high" | "medium" | "low"`.  
- `impactUSD` optional (number, >=0).  
- `actions` are safe navigational links (no POST/DELETE).  
- `updatedAt` ISO8601; panel shows relative/human time.

### 4. React Implementation (recommended)

#### Hook: `src/hooks/useTopHubSignal.ts`

```ts
// src/hooks/useTopHubSignal.ts
import { useEffect, useState, useCallback } from 'react';

const CDN_URL =
  'https://huggingface.co/datasets/axentx/costinel-knowledge/resolve/main/top-hub-MOC.json';
const LOCAL_JSON = '/data/knowledge-rag/top-hub-MOC.json';
const CACHE_KEY = 'top-hub-MOC:v1';
const CACHE_TTL_MS = 10 * 60 * 1000; // 10m

export interface ProposalAction {
  label: string;
  href: string;
}

export interface Proposal {
  id: string;
  title: string;
  severity: 'critical' | 'high' | 'medium' | 'low';
  impactUSD?: number;
  description: string;
  actions: ProposalAction[];
  tags: string[];
}

export interface TopHubData {
  hub: string;
  updatedAt: string;
  proposals: Proposal[];
}

export function useTopHubSignal(useCDN = true) {
  const [data, setData] = useState<TopHubData | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<Error | null>(null);

  const fetchData = useCallback(async () => {
    try {
      // memory dedupe by module-level promise could be added if needed
      const raw = localStorage.getItem(CACHE_KEY);
      if (raw) {
        const parsed = JSON.parse(raw) as { ts: number; payload: TopHubData };
        if (Date.now() - parsed.ts < CACHE_TTL_MS) {
          setData(parsed.payload);
          setLoading(false);
          return;
        }
      }

      const url = useCDN ? CDN_URL : LOCAL_JSON;
      const res = await fetch(url, { cache: 'no-store' });
      if (!res.ok) throw new Error(`Fetch failed: ${res.status}`);
      const payload: TopHubData = await res.json();

      localStorage.setItem(CACHE_KEY, JSON.stringify({ ts: Date.now(), payload }));
      setData(payload);
    } catch (err) {
      setError(err instanceof Error ? err : new Error(String(err)));
    } finally {
      setLoading(false);
    }
  }, [useCDN]);

  useEffect(() => {
    fetchData();
  }, [fetchData]);

  return { data, loading, error, refetch: fetchData };
}
```
